# <a name="use-workflows-to-manage-employee-information"></a>ใช้เวิร์กโฟลว์เพื่อจัดการข้อมูลพนักงาน

หัวข้อนี้อธิบายถึงวิธีการที่คุณสามารถใช้ความสามารถในการลำดับงานสำหรับทรัพยากรบุคคลเพื่อจัดการข้อมูลพนักงาน ตัวอย่างเช่น คุณสามารถเชื่อมโยงลำดับงานกับตำแหน่งงาน และตั้งค่าคอนฟิกของเวิร์กโฟลว์การอนุมัติที่เริ่มต้นเมื่อพนักงานเปลี่ยนระเบียนของพวกเขา

ความสามารถในการลำดับงานสำหรับทรัพยากรบุคคลให้เวิร์กโฟลว์มากมายสำหรับการจัดการกิจกรรมของฝ่ายทรัพยากรบุคคล นอกจากนี้ ตัวเลือกมากมายจะพร้อมใช้งานเพื่อให้คุณสามารถปรับเปลี่ยนลำดับงานที่เฉพาะเจาะจง และเชื่อมโยงกับลำดับชั้นรายงาน เวิร์กโฟลว์จะพร้อมใช้งานเพื่อช่วยในการจัดการการเปลี่ยนแปลงกับข้อมูลพนักงานชนิดมาตรฐาน คุณสามารถเชื่อมโยงลำดับงานกับตำแหน่งงาน จาก นั้น ถ้าพนักงานเปลี่ยนระเบียนของพนักงาน เวิร์กโฟลว์จะเริ่มต้นใช้งานที่ต้องมีการอนุมัติก่อนทำบันทึกข้อมูลใหม่ เวิร์กโฟลว์ที่กำหนดล่วงหน้าสำหรับชนิดของข้อมูลที่จะช่วยคุณในการจัดการการเปลี่ยนแปลงได้อย่างมีประสิทธิภาพ และเก็บข้อมูลของพนักงานของคุณอย่างถูกต้องต่อไปนี้:

-   หมายเลขรหัส
-   หลักสูตร
-   การศึกษา
-   รูปภาพ
-   รายการที่ยืม
-   ประสบการณ์ทำงาน
-   ประสบการณ์โครงการ
-   ทักษะ
-   ตำแหน่งตัวแทนของบริษัท
-   การดำเนินการของฝ่ายทรัพยากรบุคคล
-   การลงทะเบียนหลักสูตร

เมื่อพนักงานมีการจ้างงาน โอนย้าย ยกเลิก เวิร์กโฟลว์สามารถรวมกระบวนการตรวจทาน ด้วยวิธีนี้ สามารถแก้ไขเอกสาร หรือคุณสามารถกำหนดเงื่อนไขของการดำเนินการเป็นส่วนหนึ่งของเวิร์กโฟลว์ เมื่อกระบวนการตรวจทานเสร็จสมบูรณ์ เอกสารหรือการดำเนินการเสร็จสมบูรณ์แล้ว และเวิร์กโฟลว์ย้ายไปยังขั้นตอนการอนุมัติขั้นสุดท้าย

## <a name="associate-a-workflow-with-a-position-hierarchy"></a>เชื่อมโยงลำดับงานกับลำดับชั้นของตำแหน่ง
คุณสามารถเชื่อมโยงลำดับงานกับลำดับชั้นใด ๆ ที่คุณตั้งค่าคอนฟิก ตัวอย่างเช่น ถ้าตำแหน่งที่เชื่อมโยงกับลำดับชั้นรายงานเมตริกซ์ คุณอาจกำหนดค่าเวิร์กโฟลว์ที่ใช้ในค่าใช้จ่ายสำหรับโครงการระบุไปยังลูกค้าเป้าหมายโครงการแทนที่เป็นผู้จัดการของพนักงานที่เกี่ยวข้องกับตำแหน่งงานนั้น เมื่อต้องการสร้างลำดับงานใหม่ หรือปรับเปลี่ยนเวิร์กโฟลว์ที่มีอยู่ บน**ลำดับงานของฝ่ายทรัพยากรบุคคล**หน้า คลิ**สร้าง** เลือกเวิร์กโฟลว์ในรายการเมื่อต้องการเริ่มการทำงานของโปรแกรมออกแบบลำดับงาน คุณสามารถใช้ designer เพื่อสร้างลำดับงานใหม่ หรือเปลี่ยนแปลงขั้นตอนในเวิร์กโฟลว์ที่มีอยู่ เมื่อคุณเปลี่ยนเวิร์กโฟลว์ที่มีอยู่ การเปลี่ยนแปลงของคุณจะถูกบันทึกเป็นเวอร์ชันใหม่ ดังนั้น คุณสามารถเสมอกลับไปเป็นรุ่นก่อนหน้านี้ถ้าคุณจำเป็นต้อง

## <a name="configure-a-human-resources-workflow"></a>การตั้งค่าคอนฟิกลำดับงานของฝ่ายทรัพยากรบุคคล
เมื่อต้องการตั้งค่าคอนฟิกลำดับงานพื้นฐานที่เริ่มต้นเมื่อพนักงานร้องขอการเปลี่ยนแปลงรหัสประจำตัวของพวกเขา ให้ทำตามขั้นตอนเหล่านี้

1.  ในการ**ลำดับงานของฝ่ายทรัพยากรบุคคล**หน้า คลิ**สร้าง**
2.  ในรายการของกระแสงานที่พร้อมใช้งาน เลือก**หมายเลขประจำตัว**
3.  คลิ**รัน**เพื่อเริ่มการทำงานของโปรแกรมออกแบบลำดับงาน และจากนั้น ป้อนชื่อผู้ใช้และรหัสผ่านของคุณเมื่อคุณได้รับพร้อมท์
4.  ลาก**อนุมัติเลขรหัส**องค์ประกอบจากรายการขององค์ประกอบลำดับงานกับผืนผ้าใบออกแบบไว้
5.  องค์ประกอบการอนุมัติเพื่อเชื่อมต่อ**เริ่ม**และ**จบ**
6.  คลิกสองครั้งที่**องค์ประกอบของการอนุมัติ**แล้วคลิกขวา และเลือก**คุณสมบัติ**
7.  ทำตามขั้นตอนเหล่านี้เพื่อเพิ่มคำแนะนำเกี่ยวกับรายการงาน:
    1.  เลือก**กำหนด**และจากนั้น เลือก**ลำดับชั้น**ภายใต้ชนิดกำหนด
    2.  ภายใต้การ**ลำดับชั้น**เลือก เลือก**ลำดับชั้นโครง**
    3.  เพิ่มเงื่อนไขการหยุด และปิดหน้า

8.  คำแนะนำเพิ่มเติม (ไม่มีคำเตือนเพิ่มเติมควรมีอยู่) ให้เสร็จสมบูรณ์
9.  คลิก **บันทึกและปิด** เรียกใช้เวิร์กโฟลว์ใหม่เมื่อกล่องโต้ตอบเปิดขึ้นในกล่อง และเลือก**เปิดใช้งาน**
10. ไปที่**ทรัพยากรบุคคล**&gt;**ตำแหน่ง**&gt;**วางตำแหน่งชนิดลำดับชั้นของ**
11. เลือก**เมทริกซ์**
12. เพิ่มการ**หมายเลขประจำตัวผู้ปฏิบัติงาน**เวิร์กโฟลว์ลงในรายการ
