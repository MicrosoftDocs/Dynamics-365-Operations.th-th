# <a name="configure-financial-cross-company-data-sharing"></a>ตั้งค่าคอนฟิกการใช้ข้อมูลทางการเงินร่วมกันข้ามบริษัท

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงวิธีการตั้งค่าคอนฟิก เปิดใช้งาน ตรวจสอบความถูกต้อง และแก้ไขความขัดแย้ง เมื่อมีการใช้ข้อมูลร่วมกันระหว่างบริษัท  ใช้บริษัท USMF และข้อมูลทางการเงินซึ่งใช้เท็มเพลตร่วมกัน 



คู่มืองานนี้ต้องการ Dynamics AX แพลตฟอร์ม 7.1 เป็นต้นไป

1. ไปที่การดูแลระบบ > พื้นที่ทำงาน > การจัดการข้อมูล
2. คลิก นำเข้า
3. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
4. ในฟิลด์รูปแบบข้อมูลต้นทาง ให้เลือกชนิดแพคเกจ
    * คลิก อัพโหลด นำทางไปยังสถานที่ของข้อมูลทางการเงินที่ใช้ไฟล์แพคเกจเท็มเพลตร่วมกัน และเลือกไฟล์นั้น  
5. คลิก บันทึก
6. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
    * เลือกข้อมูลที่ใช้นโยบายร่วมกัน ที่เพิ่งถูกสร้างขึ้น  
7. คลิก นำเข้า
8. คลิก ปิด
9. รีเฟรชหน้า
10. ปิดหน้า
11. ปิดหน้า
12. ปิดหน้า
13. ไปที่การจัดการระบบ > การตั้งค่า > ตั้งค่าคอนฟิกลิขสิทธิ์การใช้ข้อมูลร่วมกันข้ามบริษัท
14. ในรายการนี้ ให้ค้นหาและเลือกจำนวนวันการชำระเงิน
15. คลิก เพิ่ม
16. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
17. ในฟิลด์บริษัท ให้พิมพ์ 'USSI'
18. คลิก เพิ่ม
19. ในฟิลด์บริษัท ให้พิมพ์ 'USMF'
20. คลิก บันทึก
21. คลิกเปิดใช้งาน
22. คลิก ใช่
23. คลิกค้นหาปัญหาที่มีร่วมกัน
24. คลิก ใช่
25. คลิกอัพเดตค่าฟิลด์
26. คลิกใช้ค่าจากบริษัท 1
27. รีเฟรชหน้า
28. ปิดหน้า
