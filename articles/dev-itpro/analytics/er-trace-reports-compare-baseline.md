[!include[banner](../includes/banner.md)]

<span data-ttu-id="712e0-104">คุณสามารถติดตามผลลัพธ์ของรูปแบบ ER ที่สร้างเอกสารอิเล็กทรอนิกส์ขาออกได้</span><span class="sxs-lookup"><span data-stu-id="712e0-104">You can trace the results of ER formats that generate outgoing electronic documents.</span></span> <span data-ttu-id="712e0-105">เมื่อมีการเปิดการสร้างการติดตาม (ER ผู้ใช้พารามิเตอร์ **รันในโหมดดีบัก**) เรกคอร์ดการติดตามใหม่จะถูกสร้างขึ้นในล็อกการดำเนินการเกี่ยวกับรูปแบบ ER ทุกครั้งที่รันรายงาน ER</span><span class="sxs-lookup"><span data-stu-id="712e0-105">When trace generation is turned on (ER user parameter **Run in debug mode**), a new trace record is generated in the ER format execution log every time that an ER report is run.</span></span> <span data-ttu-id="712e0-106">มีการจัดเก็บรายละเอียดต่อไปนี้ในการติดตามแต่ละรายการที่สร้างขึ้น:</span><span class="sxs-lookup"><span data-stu-id="712e0-106">The following details are stored in each trace that is generated:</span></span>

- <span data-ttu-id="712e0-107">คำเตือนทั้งหมดที่ถูกสร้างขึ้นโดยกฎการตรวจสอบ</span><span class="sxs-lookup"><span data-stu-id="712e0-107">All warnings that were generated by validation rules</span></span>
- <span data-ttu-id="712e0-108">ข้อผิดพลาดทั้งหมดที่ถูกสร้างขึ้นโดยกฎการตรวจสอบ</span><span class="sxs-lookup"><span data-stu-id="712e0-108">All errors that were generated by validation rules</span></span>
- <span data-ttu-id="712e0-109">ไฟล์ที่สร้างขึ้นทั้งหมดที่จัดเก็บเป็นสิ่งที่แนบของเรกคอร์ดการติดตาม</span><span class="sxs-lookup"><span data-stu-id="712e0-109">All generated files that are stored as attachments of the trace record</span></span>

<span data-ttu-id="712e0-110">คุณสามารถจัดเก็บไฟล์แอพลิเคชันพื้นฐานแต่ละรายการสำหรับรูปแบบ ER ใดๆ ได้</span><span class="sxs-lookup"><span data-stu-id="712e0-110">You can store individual baseline application files for any ER format.</span></span> <span data-ttu-id="712e0-111">ไฟล์จะถูกถือว่าเป็นไฟล์พื้นฐาน เมื่อพวกเขาอธิบายผลลัพธ์ที่คาดไว้ของรายงานที่จะรัน</span><span class="sxs-lookup"><span data-stu-id="712e0-111">Files are considered baseline files when they describe the expected results of reports that are run.</span></span> <span data-ttu-id="712e0-112">ถ้าไฟล์พื้นฐานพร้อมใช้งานสำหรับรูปแบบ ER ที่รัน ขณะที่มีการเปิดใช้งานการสร้างการติดตาม ร้านค้าติดตาม นอกเหนือจากรายละเอียดที่กล่าวถึงก่อนหน้านี้ ผลลัพธ์ของการเปรียบเทียบเอกสารทางอิเล็กทรอนิกส์ที่สร้างขึ้นกับไฟล์พื้นฐาน</span><span class="sxs-lookup"><span data-stu-id="712e0-112">If a baseline file is available for an ER format that was run while trace generation was turned on, the trace stores, in addition to the details that were mentioned earlier, the result of the comparison of the generated electronic document to the baseline file.</span></span> <span data-ttu-id="712e0-113">ในหนึ่งคลิก คุณยังสามารถดูเอกสารทางอิเล็กทรอนิกส์ที่สร้างขึ้นและไฟล์ของพื้นฐานในไฟล์ zip เดียวกันได้ด้วย</span><span class="sxs-lookup"><span data-stu-id="712e0-113">In one click, you can also get the generated electronic document and its baseline file in a single zip file.</span></span> <span data-ttu-id="712e0-114">จากนั้นคุณสามารถทำการเปรียบเทียบโดยละเอียดได้ โดยใช้เครื่องมือภายนอก เช่น Windiff</span><span class="sxs-lookup"><span data-stu-id="712e0-114">You can then do detailed comparison by using an external tool such as Windiff.</span></span>

<span data-ttu-id="712e0-115">คุณสามารถประเมินการติดตามเพื่อวิเคราะห์ว่า เอกสารทางอิเล็กทรอนิกส์ที่ถูกสร้างขึ้นรวมเนื้อหาที่คาดไว้หรือไม่</span><span class="sxs-lookup"><span data-stu-id="712e0-115">You can evaluate the trace to analyze whether the electronic documents that are generated include the expected content.</span></span> <span data-ttu-id="712e0-116">คุณสามารถทำการประเมินนี้ในสภาพแวดล้อมการทดสอบการยอมรับของผู้ใช้ (UAT) เมื่อมีการเปลี่ยนแปลงพื้นฐานรหัส (ตัวอย่างเช่น เมื่อคุณย้ายไปที่อินสแตนซ์ใหม่ของแอพลิเคชัน ติดตั้งแพคเกจโปรแกรมแก้ไขด่วน หรือปรับใช้การปรับเปลี่ยนรหัส)</span><span class="sxs-lookup"><span data-stu-id="712e0-116">You can do this evaluation in a user acceptance testing (UAT) environment when the code base has been changed (for example, when you migrated to a new instance of the application, installed hotfix packages, or deployed code modifications).</span></span> <span data-ttu-id="712e0-117">ด้วยวิธีนี้ คุณสามารถมั่นใจได้ว่า การประเมินจะไม่มีผลกับการดำเนินการของรายงาน ER ที่ใช้</span><span class="sxs-lookup"><span data-stu-id="712e0-117">In this way, you can make sure that the evaluation doesn't affect the execution of ER reports that are used.</span></span> <span data-ttu-id="712e0-118">สำหรับรายงาน ER มากมาย การประเมินสามารถทำได้ในโหมดแบบอัตโนมัติ</span><span class="sxs-lookup"><span data-stu-id="712e0-118">For many ER reports, the evaluation can be done in unattended mode.</span></span>
