# <a name="configure-computations-to-do-counting-and-summing"></a>ตั้งค่าคอนฟิกการคำนวณเพื่อทำการตรวจนับและสรุป

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

ขั้นตอนต่อไปนี้อธิบายวิธีกำหนดผู้ใช้ให้กับผู้ดูแลระบบหรือวิธีการที่บทบาทนักพัฒนาการรายงานทางอิเล็กทรอนิกส์สามารถตั้งค่าคอนฟิกรูปแบบการรายงานทางอิเล็กทรอนิกส์ (ER) เพื่อทำการตรวจนับและสรุปตามข้อมูลของผลลัพธ์ข้อความที่สร้างขึ้นแล้ว  สามารถดำเนินการขั้นตอนเหล่านี้ได้ในบริษัทใดก็ได้ 

เพื่อทำตามขั้นตอนเหล่านี้ อันดับแรกคุณต้องทำตามขั้นตอนในกระบวนงาน "ER ตั้งค่าคอนฟิกรูปแบบเพื่อทำการตรวจนับและสรุป (ส่วนที่ 1: สร้างรูปแบบ)"

กระบวนงานนี้ใช้สำหรับลักษณะการทำงานที่ถูกเพิ่มลงใน Dynamics 365 for Operations รุ่น 1611


## <a name="create-a-format-configuration-to-add-counting-and-summing-details"></a>สร้างการตั้งค่าคอนฟิกรูปแบบเพื่อเพิ่มรายละเอียดการตรวจนับและการสรุป
1. ไปที่การจัดการองค์กร > พื้นที่ทำงาน > การรายงานทางอิเล็กทรอนิกส์
2. คลิก การตั้งค่าคอนฟิกการรายงาน
3. ในแผนภูมิ ขยาย 'Intrastat model'
4. ในแผนภูมิ เลือก 'Intrastat model\Intrastat (DE)'
    * สมมติว่าคุณจำเป็นต้องกำหนดรูปแบบที่ระบุโดย Microsoft โดยการเพิ่มบรรทัดพร้อมกับรายละเอียดสรุปที่ส่วนท้ายของรายงานอินทราสแทต  คุณจะต้องทำเช่นนั้นโดยได้รับอินสแตนซ์ของการตั้งค่าคอนฟิกอินทราสแทตของเราจากอินสแตนซ์ Microsoft เพื่อทำการปรับเปลี่ยน  
5. คลิก สร้างการตั้งค่าคอนฟิก เพื่อเปิดกล่องโต้ตอบการวาง
6. ในฟิลด์ใหม่ ป้อน 'ได้รับมาจากชื่อ: อินทราสแทต (DE), Microsoft'
7. ในฟิลด์ชื่อ พิมพ์ 'อินทราสแทต (DE) พร้อมกับการตรวจนับ & การสรุป'
8. คลิก สร้างการตั้งค่าคอนฟิก
