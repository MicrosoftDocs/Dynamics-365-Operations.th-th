# <a name="design-er-configurations-to-generate-reports-in-word-format"></a>ออกแบบการตั้งค่าคอนฟิก ER เพื่อสร้างรายงานในรูปแบบ Word

[!include [task guide banner](../../includes/task-guide-banner.md)]

ขั้นตอนต่อไปนี้อธิบายวิธีการที่ผู้ใช้ในบทบาทผู้ดูแลระบบหรือนักพัฒนาการรายงานทางอิเล็กทรอนิกส์ อย่างใดอย่างหนึ่ง สามารถตั้งค่าคอนฟิกรูปแบบการรายงานทางอิเล็กทรอนิกส์ (ER) เพื่อสร้างรายงานเป็นไฟล์ Microsoft Word ขั้นตอนเหล่านี้สามารถถูกดำเนินการได้ในบริษัท GBSI

เพื่อทำขั้นตอนเหล่านี้ให้เสร็จสมบูรณ์ คุณต้องทำขั้นตอนอย่างแรกให้เสร็จสมบูรณ์ในคู่มืองาน "สร้างการตั้งค่าคอนฟิก ER สำหรับการสร้างรายงานในรูปแบบ OPENXML" ก่อน นอกจากนี้คุณยังต้องดาวน์โหลดและบันทึกเท็มเพลตต่อไปนี้ล่วงหน้าไว้ภายในเครื่องสำหรับรายงานตัวอย่าง:

- [เท็มเพลตของรายงานการชำระเงิน](https://go.microsoft.com/fwlink/?linkid=862266)
- [เท็มเพลตที่กำหนดขอบเขตของรายงานการชำระเงิน](https://go.microsoft.com/fwlink/?linkid=862266)


กระบวนงานนี้ใช้สำหรับคุณลักษณะทั้ที่ถูกเพิ่มลงใน Microsoft Dynamics 365 for Operations รุ่น 1611


## <a name="select-the-existing-er-report-configuration"></a>เลือกการตั้งค่าคอนฟิกรายงาน ER ที่มีอยู่
1. ไปที่การจัดการองค์กร > พื้นที่ทำงาน > การรายงานทางอิเล็กทรอนิกส์
    * ตรวจสอบให้แน่ใจว่ามีการเลือกให้ผู้ให้บริการการตั้งค่าคอนฟิก 'Litware, Inc.' เปิดใช้งาน  
2. คลิก การตั้งค่าคอนฟิกการรายงาน
    * เราจะใช้การตั้งค่าคอนฟิก ER ที่มีอยู่ ซึ่งแต่เดิมได้รับการออกแบบเพื่อสร้างเอาท์พุทรายงานในรูปแบบ OPENXML อีกครั้ง  
3. ในแผนภูมิ ขยาย 'รูปแบบการชำระเงิน'
4. ในแผนภูมิ เลือก 'แบบจำลองการชำระเงิน\รายงานแผ่นงานตัวอย่าง'
5. คลิก ตัวออกแบบ

## <a name="replace-the-excel-template-with-the-word-template"></a>แทนที่เท็มเพลต Excel ด้วยเท็มเพลต Word
    * ปัจจุบัน เอกสาร Excel จะใช้เป็นเท็มเพลตเพื่อสร้างเอาท์พุทในรูปแบบ OPENXML เราจะนำเข้าเท็มเพลตของรายงานในรูปแบบ Word  
1. คลิกสิ่งที่แนบ
    * แทนที่เท็มเพลต Excel ที่มีอยู่ด้วยเท็มเพลต Word ที่คุณดาวน์โหลดมาก่อนหน้านี้ SampleVendPaymDocReport.docx หมายเหตุ เท็มเพลตนี้ประกอบด้วยโครงร่างของเอกสารที่เราต้องการสร้างเป็นเอาท์พุท ER เท่านั้น  
2. คลิก ลบ
3. คลิก ใช่
4. คลิก สร้าง
5. คลิกไฟล์
6. คลิกเรียกดู นำทางไปยังและเลือก SampleVendPaymDocReport.docx ที่คุณดาวน์โหลดไว้ก่อนหน้านี้ คลิก ตกลง
    * เลือกเท็มเพลตที่คุณดาวน์โหลดไว้แล้วในขั้นตอนก่อนหน้านี้  
7. ในฟิลด์เท็มเพลต ให้ป้อนหรือเลือกค่า

## <a name="extend-the-word-template-by-adding-a-custom-xml-part"></a>ขยายเท็มเพลต Word โดยการเพิ่มส่วน XML แบบกำหนดเอง
1. คลิก บันทึก
    * นอกจากจะเก็บข้อมูลการเปลี่ยนแปลงการตั้งค่าคอนฟิก การดำเนินการบันทึกยังเป็นการอัพเดตเท็มเพลต Word ที่แนบอีกด้วย โครงสร้างของรูปแบบที่ได้รับการออกแบบมาถูกนำไปยังเอกสาร Word ที่แนบเป็นส่วน XML แบบกำหนดเองใหม่ด้วยชื่อ 'รายงาน' โปรดทราบว่า เท็มเพลต Word ที่แนบไม่ได้มีเฉพาะโครงร่างของเอกสารที่เราต้องการสร้างเป็นเอาท์พุท ER แต่ยังประกอบด้วยโครงสร้างของข้อมูลที่ ER จะเติมลงในเท็มเพลตนี้ขณะรันไทม์  
2. คลิกสิ่งที่แนบ
    * ขณะนี้ คุณจำเป็นต้องเชื่อมโยงองค์ประกอบของส่วน XML แบบกำหนดเอง 'รายงาน' กับส่วนเอกสาร Word  
    * ถ้าคุณคุ้นเคยกับเอกสาร Word ที่สามารถออกแบบให้เป็นแบบฟอร์มที่มีตัวควบคุมเนื้อหา ซึ่งถูกกำหนดขอบเขตด้วยองค์ประกอบของส่วน XML แบบกำหนดเอง – เล่นขั้นตอนทั้งหมดของงานย่อยถัดไปเพื่อสร้างเอกสารดังกล่าว สำหรับข้อมูลเพิ่มเติม ดูลิงก์นี้ https://support.office.com/article/Create-forms-that-users-complete-or-print-in-Word-040c5cc1-e309-445b-94ac-542f732c8c8b?ui=en-US&rs=en-US&ad=US หรือข้ามขั้นตอนทั้งหมดในงานย่อยถัดไป  

## <a name="get-word-with-custom-xml-part-to-do-data-bindings"></a>เรียกดู Word ด้วยส่วน XML แบบกำหนดเองเพื่อดำเนินการเชื่อมโยงข้อมูล
    * เปิดเอกสารนี้ใน Word และดำเนินการต่อไปนี้:  - เปิดแท็บนักพัฒนา Word (เลือกกำหนด Ribbon ถ้ายังไม่ได้เปิดใช้งาน)  - เลือกบานหน้าต่างการแม็ป XML  - เลือกส่วน XML แบบกำหนดเอง 'รายงาน' ในการค้นหา  - ทำการแม็ปองค์ประกอบของส่วน XML แบบกำหนดเองที่เลือกและตัวควบคุมเนื้อหาของเอกสาร Word  - บันทึกเอกสาร Word ที่อัพเดตแล้วบนไดรฟ์ในเครื่อง  

## <a name="upload-the-word-template-with-custom-xml-part-bounded-to-content-controls"></a>อัพโหลดเท็มเพลต Word ด้วยส่วน XML แบบกำหนดเองที่ถูกกำหนดขอบเขตไปยังตัวควบคุมเนื้อหา
1. คลิก ลบ
2. คลิก ใช่
    * เพิ่มเท็มเพลตใหม่ ถ้าคุณทำขั้นตอนต่างๆ ในงานย่อยก่อนหน้านี้เสร็จสมบูรณ์แล้ว ให้เลือกเอกสาร Word ที่คุณจัดเตรียม และบันทึกภายในเครื่อง หรือเลือกเท็มเพลต MS Word SampleVendPaymDocReportBounded.docx ที่ดาวน์โหลดมาก่อนหน้านี้  
3. คลิก สร้าง
4. คลิกไฟล์
5. คลิกเรียกดู นำทางไปยังและเลือก SampleVendPaymDocReportBounded.docx ที่คุณดาวน์โหลดไว้ก่อนหน้านี้ คลิก ตกลง
6. ในฟิลด์เท็มเพลต ให้เลือกเอกสารที่คุณดาวน์โหลดไว้แล้วในขั้นตอนก่อนหน้านี้
7. คลิก บันทึก
8. ปิดหน้า

## <a name="execute-the-format-to-create-word-output"></a>ดำเนินการรูปแบบเพื่อสร้างเอาท์พุทของ Word
1. ในบานหน้าต่างการดำเนินการ คลิก การจัดโครงแบบ
2. คลิก พารามิเตอร์ผู้ใช้
3. เลือก ใช่ ในฟิลด์ รันการตั้งค่า
4. คลิก ตกลง
5. คลิก แก้ไข
6. เลือก ใช่ ในฟิลด์ รันฉบับร่าง
7. คลิก บันทึก
8. ปิดหน้า
9. ปิดหน้า
10. ไปที่ > บัญชีเจ้าหนี้ > การชำระเงิน > สมุดรายวันการชำระเงิน
11. คลิก รายการ
12. ในรายการนี้ ทำเครื่องหมายหรือยกเลิกการทำเครื่องหมายทุกแถว
13. คลิกสถานะการชำระเงิน
14. คลิก ไม่มี
15. คลิกสร้างการชำระเงิน
16. คลิก ตกลง
17. คลิก ตกลง
    * วิเคราะห์ผลลัพธ์ที่สร้างขึ้น โปรดทราบว่าผลลัพธ์ที่สร้างขึ้นจะแสดงในรูปแบบ Word และประกอบด้วยรายละเอียดเกี่ยวกับการชำระเงินที่ประมวลผลแล้ว
