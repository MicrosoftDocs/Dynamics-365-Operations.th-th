# <a name="er-import-a-configuration-from-lifecycle-services"></a>ER นำเข้าการตั้งค่าคอนฟิกจาก Lifecycle Services

[!include [task guide banner](../../includes/task-guide-banner.md)]

ขั้นตอนต่อไปนี้อธิบายวิธีที่ผู้ใช้ในบทบาทผู้ดูแลระบบ หรือนักพัฒนาการรายงานทางอิเล็กทรอนิกส์ สามารถนำเข้าการตั้งค่าคอนฟิกการรายงานทางอิเล็กทรอนิกส์ (ER) รุ่นใหม่จาก Microsoft Lifecycle Services (LCS)

ในตัวอย่างนี้ คุณจะสร้างเลือกเวอร์ชันที่ต้องการของการตั้งค่าคอนฟิก ER และนำเข้าสำหรับบริษัทตัวอย่าง ซึ่งคือ Litware, inc ขั้นตอนเหล่านี้สามารถถูกดำเนินการได้ในบริษัทใดๆ เนื่องจากมีการใช้การตั้งค่าคอนฟิก ER ร่วมกันระหว่างบริษัท เมื่อต้องการดำเนินขั้นตอนเหล่านี้ให้เสร็จสมบูรณ์ ลำดับแรกคุณต้องดำเนินงานขั้นตอนในกระบวนงาน "อัพโหลดการตั้งค่าคอนฟิก ER ลงใน Lifecycle Services"  การเข้าถึง LCS ยังต้องการสำหรับการเสร็จสมบูรณ์ของขั้นตอนเหล่านี้อีกด้วย

1. ไปที่การจัดการองค์กร > พื้นที่ทำงาน > การรายงานทางอิเล็กทรอนิกส์
2. คลิกการตั้งค่าคอนฟลิก

## <a name="delete-a-shared-version-of-data-model-configuration"></a>ลบเวอร์ชันที่ใช้ร่วมกันของการตั้งค่าคอนฟิกแบบจำลองข้อมูล
1. ในแผนภูมิ เลือก 'การตั้งค่าคอนฟิกแบบจำลองตัวอย่าง'
    * เวอร์ชันแรกของการตั้งค่าคอนฟิกแบบจำลองข้อมูลตัวอย่างได้ถูกสร้างขึ้น และถูกเผยแพร่ไปยัง LCS ในระหว่างกระบวนงาน 'อัพโหลดการตั้งค่าคอนฟิก ER ลงใน Lifecycle Services'  ในกระบวนงานนี้ คุณจะลบเวอร์ชันนี้ของการตั้งค่าคอนฟิก ER  การตั้งค่าคอนฟิกแบบจำลองข้อมูลตัวอย่างรุ่นนี้จะถูกนำเข้าในภายหลังจาก LCS  
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * เลือกเวอร์ชันของการตั้งค่าคอนฟิกนี้ ซึ่งอยู่ในสถานะ 'ใช้ร่วมกัน'  สถานะนี้ระบุว่าการตั้งค่าคอนฟิกได้ถูกเผยแพร่ไปยัง LCS แล้ว  
3. คลิก เปลี่ยนแปลงสถานะ
4. คลิกยกเลิกการใช้งาน
    * เปลี่ยนสถานะของเวอร์ชันที่เลือกจาก 'ใช้ร่วมกัน' เป็น 'ถูกยกเลิก' เพื่อทำให้พร้อมใช้งานสำหรับการลบ  
5. คลิก ตกลง
6. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * เลือกเวอร์ชันของการตั้งค่าคอนฟิกนี้ ซึ่งมีสถานะเป็น 'ถูกยกเลิก'  
7. คลิก ลบ
8. คลิก ใช่
    * หมายเหตุว่า เฉพาะแบบร่างเวอร์ชัน 2 ของการตั้งค่าคอนฟิกแบบจำลองข้อมูลพร้อมใช้งาน  
9. ปิดหน้า

## <a name="import-a-shared-version-of-data-model-configuration-from-lcs"></a>นำเข้าเวอร์ชันที่ใช้ร่วมกันของการตั้งค่าคอนฟิกแบบจำลองข้อมูลจาก LCS
1. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
    * เปิดรายการของที่เก็บสำหรับผู้ให้บริการการตั้งค่าคอนฟิก ’Litware, Inc.’  
2. คลิก ที่เก็บ
3. คลิก เปิด
    * เลือกที่เก็บ LCS และเปิดขึ้น  
4. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
    * เลือกเวอร์ชันแรกของ 'การตั้งค่าคอนฟิกแบบจำลองตัวอย่าง' ในรายการเวอร์ชัน  
5. คลิก นำเข้า
6. คลิก ใช่
    * ยืนยันการนำเข้าของเวอร์ชันที่เลือกจาก LCS  
    * หมายเหตุว่า ข้อความข้อมูล (เหนือแบบฟอร์ม) ยืนยันความสำเร็จของการนำเข้าเวอร์ชันที่เลือก  
7. ปิดหน้า
8. ปิดหน้า
9. คลิกการตั้งค่าคอนฟลิก
10. ในแผนภูมิ เลือก 'การตั้งค่าคอนฟิกแบบจำลองตัวอย่าง'
11. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * เลือกเวอร์ชันของการตั้งค่าคอนฟิกนี้ ซึ่งมีสถานะเป็น 'ใช้ร่วมกัน'  
    * หมายเหตุว่า เวอร์ชันที่ใช้ร่วมกัน 1 ของการตั้งค่าคอนฟิกแบบจำลองข้อมูลที่เลือก พร้อมใช้งานด้วย ในขณะนี้
