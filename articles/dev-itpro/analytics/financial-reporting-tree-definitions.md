# <a name="reporting-tree-definitions-in-financial-reports"></a>คำนิยามแผนภูมิรายงานในรายงานทางการเงิน

[!include [banner](../includes/banner.md)]

บทความนี้แสดงข้อมูลเกี่ยวกับคำนิยามแผนภูมิรายงาน คำนิยามแผนภูมิรายงานคือส่วนประกอบรายงาน หรือบล็อคส่วนประกอบ ซึ่งช่วยในการกำหนดโครงสร้างและลำดับชั้นขององค์กรของคุณ

การรายงานทางการเงินสนับสนุนการรายงานที่ยืดหยุ่นเพื่อให้ง่ายต่อการเปลี่ยนแปลงตามการเปลี่ยนแปลงโครงสร้างธุรกิจของคุณ รายงานถูกสร้างขึ้นจากส่วนประกอบหรือบล็อคส่วนประกอบต่างๆ หนึ่งในบล็อคส่วนประกอบเหล่านี้คือคำนิยามแผนภูมิรายงาน คำนิยามแผนภูมิรายงานช่วยกำหนดโครงสร้างและลำดับชั้นขององค์กรของคุณ สิ่งนี้คือโครงสร้างลำดับชั้นข้ามมิติที่อาศัยความสัมพันธ์มิติในข้อมูลทางการเงินของคุณ อีกทั้งให้ข้อมูลในระดับหน่วยการรายงาน และระดับสรุปสำหรับหน่วยทั้งหมดในแผนภูมิ คำนิยามแผนภูมิการรายงานสามารถรวมกับคำนิยามคอลัมน์ และข้อกำหนดของรายงาน เพื่อสร้างกลุ่มการสร้างบล็อคที่สามารถใช้โดยบริษัทหลายบริษัท หน่วยการรายงานจะใช้สำหรับแต่ละกล่องในแผนภูมิองค์กร หน่วยการรายงานอาจเป็นแผนกเฉพาะจากข้อมูลทางการเงิน หรืออาจเป็นหน่วยสรุประดับสูงขึ้นที่รวมข้อมูลจากหน่วยการรายงานอื่นๆ สำหรับข้อกำหนดของรายงานที่รวมแผนภูมิการรายงาน รายงานหนึ่งถูกสร้างสำหรับแต่ละหน่วยการรายงาน และสำหรับระดับสรุป รายงานเหล่านี้ใช้คำนิยามแถว และคอลัมน์ที่ระบุไว้ในข้อกำหนดของรายงาน ยกเว้นว่าข้อกำหนดของรายงานระบุว่าจะใช้แผนภูมิการรายงานจากคำนิยามแถว คำนิยามแถว และคอลัมน์เป็นส่วนประกอบที่สำคัญในการออกแบบ และฟังก์ชันของการรายงานทางการเงิน แผนภูมิการรายงานเพิ่มกำลังของส่วนประกอบ และสนับสนุนการรายงานแบบยืดหยุ่นตามการเปลี่ยนแปลงโครงสร้างธุรกิจ รายงานทางการเงินที่ไม่ได้อาศัยแผนภูมิการรายงานใช้ความสามารถบางอย่างของการรายงานทางการเงินเท่านั้น คุณสามารถใช้หลายคำนิยามแผนภูมิการรายงานกับคำนิยามแถว และคอลัมน์เดียวกัน เพื่อดูข้อมูลขององค์กรของคุณในลักษณะต่างๆ

## <a name="reporting-tree-best-practices"></a>แนวทางปฏิบัติสำหรับแผนภูมิการรายงาน
ก่อนคุณสร้างแผนภูมิการรายงาน พิจารณาแนวทางที่พึงปฏิบัติต่อไปนี้:

-   กำหนดมิติรายงานใดที่นิติบุคคลของคุณ หรือบริษัทจำเป็นต้องมี
-   พิจารณาวิธีที่คุณตั้งค่าโครงสร้างของคุณ จากนั้นวาดแผนภูมิองค์กรของบริษัทคุณ แผนผังองค์กรจะช่วยให้คุณเห็นภาพว่าจะจัดกลุ่มหน่วยการรายงานเข้าสู่ลำดับการรายงานเดียวหรือหลายลำดับ
-   เริ่มด้วยระดับพร้อมใช้งานต่ำสุดของรายละเอียด เช่น แผนก และโครงการที่กำหนดไว้ในข้อมูลทางการเงิน เพิ่มกล่องไปยังระดับของรายละเอียดมากเท่าที่จำเป็น เพื่อแสดงระดับที่สูงกว่าส่วน หรือภูมิภาค แต่ละกล่องแสดงถึงหน่วยการรายงานที่เป็นไปได้ในแผนภูมิรายงานใดๆ ที่คุณสร้าง
-   คุณต้องพิจารณาวิธีดีที่สุดเพื่อสร้างแผนภูมิของคุณ คุณสามารถใช้กระบวนการสร้างอัตโนมัติเพื่อสร้างแผนภูมิการรายงาน หรือคุณจะสร้างแผนภูมิการรายงานด้วยตนเอง จำเป็นที่คุณจะต้องเข้าใจวิธีการทั้งสอง ก่อนที่คุณจะออกแบบแผนภูมิของคุณ
-   คุณสามารถใช้หน่วยการรายงานที่กำหนดไว้ในระบบข้อมูลทางการเงินของคุณ เพื่อเพิ่มหน่วยการรายงานไปยังคำนิยามแผนภูมิการรายงาน

## <a name="create-multiple-reporting-trees"></a>สร้างแผนภูมิการรายงานหลายรายการ
คุณสามารถสร้างแผนภูมิการรายงานได้ไม่จำกัดจำนวนเพื่อดูข้อมูลขององค์กรของคุณในลักษณะต่างๆ แต่ละแผนภูมิการรายงานอาจประกอบด้วยชุดของแผนก และหน่วยการสรุปใดๆ ข้อกำหนดของรายงานสามารถประกอบด้วยลิงค์ไปยังแผนภูมิการรายงานเดียวเท่านั้นในแต่ละครั้ง โดยการจัดเรียงโครงสร้างของหน่วยการรายงานใหม่ คุณสามารถสร้างแผนภูมิการรายงานอื่น จากนั้นคุณสามารถใช้แถวและข้อกำหนดคอลัมน์เดียวกันสำหรับแผนภูมิรายงานแต่ละรายการ ในลักษณะนี้ คุณสร้างโครงร่างรายงานทางการเงินที่แตกต่างกันได้อย่างรวดเร็ว ถ้าคุณสร้างแผนภูมิรายงานหลายรายการ คุณสามารถพิมพ์ชุดของงบการเงินแต่ละเดือนที่วิเคราะห์ และนำเสนอการดำเนินการของบริษัทของคุณได้หลายวิธี สำหรับข้อมูลเพิ่มเติม ดูตัวอย่างของหน่วยการรายงานเมื่อสิ้นสุดของบทความนี้

|       % การรวบรวม        | เปอร์เซ็นต์ของหน่วยการรายงานที่ควรถูกปันส่วนให้กับหน่วยหลัก เปอร์เซ็นต์ที่คุณป้อนในคอลัมน์นี้ใช้กับแต่ละแถวของคำนิยามแถว ก่อนค่าในแถวจะถูกเพิ่มเข้าในรายงานหลัก ตัวอย่างเช่น ถ้าหน่วยรองต้องสามารถหารให้เท่ากันระหว่างสองแผนก ยอดเงินในแต่ละแถวจะถูกคูณด้วย 50 เปอร์เซ็นต์ ก่อนที่จะเพิ่มค่าลงในรายงานแผนก หน่วยการรายงานไม่สามารถมีหน่วยหลักมากกว่า 2 หน่วย เพื่อปันส่วนยอดเงินในหน่วยการรายงานไปยังหน่วยหลักสองหน่วย สร้างหน่วยการรายงานอื่นที่มีมิติเดียวกันเพื่อรวม 50 เปอร์เซ็นต์เพิ่มเติม พิมพ์เปอร์เซ็นต์ทั้งหมดโดยไม่มีจุดทศนิยม ตัวอย่างเช่น <strong>25</strong> หมายถึง 25 เปอร์เซ็นต์การปันส่วนไปยังข้อมูลหลัก ถ้าคุณใส่จุดทศนิยม (<strong>.25</strong>) 0.25 เปอร์เซ็นต์จะถูกปันส่วนไปยังข้อมูลหลัก เมื่อต้องการใช้เปอร์เซ็นต์ที่น้อยกว่า 1 เปอร์เซ็นต์ ใช้ตัวเลือก <strong>อนุญาตให้รวม &lt;1%</strong> ในข้อกำหนดของรายงาน ตัวเลือกนี้อยู่บนแท็บ <strong>ตัวเลือกเพิ่มเติม</strong> ในกล่องโต้ตอบ <strong>การตั้งค่ารายงาน</strong> คุณสามารถเข้าถึงกล่องโต้ตอบนี้จากปุ่ม <strong>อื่น ๆ</strong> บนแท็บ <strong>ตั้งค่า</strong> ของข้อกำหนดของรายงาน |
|     หน่วยความปลอดภัย     |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   ข้อจำกัดที่ผู้ใช้ และกลุ่มสามารถเข้าถึงข้อมูลสำหรับหน่วยการรายงาน                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|    ข้อความเพิ่มเติม    |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               ข้อความที่จะถูกรวมในรายงาน                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |

เพื่อสร้างคำนิยามแผนภูมิการรายงาน ต้องทำตามขั้นตอนต่อไปนี้

1.  เปิดโปรแกรมออกแบบรายงาน
2.  คลิก **ไฟล์** &gt; **สร้าง** &gt; **คำนิยามแผนภูมิการรายงาน**
3.  คลิก **แก้ไข** &gt; **แทรกหน่วยการรายงานจากมิติ**
4.  ในกล่องโต้ตอบ **แทรกหน่วยการรายงานจากมิติ** เลือกกล่องสำหรับแต่ละมิติที่จะรวมไว้ในแผนภูมิการรายงาน กล่องโต้ตอบ **แทรกหน่วยการรายงานจากมิติ** จะประกอบด้วยส่วนต่อไปนี้

    | ส่วน                          | คำอธิบาย                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
    |----------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
    | การแบ่งเซกเมนต์มิติการรายงาน | ใช้ปุ่ม **แบ่งเซ็กเมนต์** และ **รวมเซ็กเมนต์** เพื่อเปลี่ยนจำนวนและความยาวของเซ็กเมนต์ **หมายเหตุ:** คุณสามารถรวมเฉพาะเซ็กเมนต์ที่คุณได้แบ่งไว้ เพื่อรวมรวมหลายมิติ ใช้อักขระตัวแทนในค่ามิติของคุณ                                                                                                                                                                                                          |
    | รวม/ตำแหน่งอักขระ       | ส่วนนี้แสดงรายการมิติที่กำหนดไว้ในข้อมูลทางการเงิน และแสดงจำนวนของอักขระในค่ากำหนดที่ยาวที่สุดสำหรับแต่ละมิติ เลือกกล่องกาเครื่องหมายสำหรับมิติเพื่อรวมมิติในลำดับชั้นของแผนภูมิการรายงาน                                                                                                                                                                                           |
    | เซ็กเมนต์ลำดับชั้น และช่วง     | ส่วนนี้แสดงลำดับชั้นของมิติ คุณสามารถย้ายมิติในรายการเพื่อเปลี่ยนใบสั่งการรายงาน ในฟิลด์ **มิติต้นทาง** และ **มิติปลายทาง** คุณสามารถระบุช่วงของค่าภายในแต่ละมิติ ถ้าคุณไม่ได้ระบุช่วง ค่ามิติทั้งหมดจะถูกแทรกลงในแผนภูมิการรายงาน **หมายเหตุ:** ถ้าคุณใช้มิติมากกว่าหนึ่งรายการ เฉพาะชุดมิติที่มีการลงรายการบัญชจะถูกแสดงในผลลัพธ์ |

    สำหรับภาพหน้าจอที่แสดงตัวอย่างของกล่องโต้ตอบ **แทรกหน่วยการรายงานจากมิติ** ให้ดูส่วน "ตัวอย่างของกล่องโต้ตอบแทรกหน่วยการรายงานจากมิติ" ในบทความนี้
5.  สร้างเซ็กเมนต์เพิ่มเติม (ตัวอย่างเช่น โดยการแบ่งหนึ่งเซ็กเมนต์เป็นสองเซ็กเมนต์ที่สั้นกว่า) คลิกตำแหน่งที่ถูกต้องในฟิลด์ **ตำแหน่งอักขระ** จากนั้นคลิก **แบ่งเซ็กเมนต์**
6.  เพื่อผสานสองเซ็กเมนต์เป็นหนึ่งเซ็กเมนต์ คลิกในกล่องเซ็กเมนต์ใดเซ็กเมนต์หนึ่งเพื่อผสาน จากนั้นคลิก **รวมเซ็กเมนต์**
7.  ลำดับชั้นกำหนดวิธีที่รายงานมิติไประหว่างกัน และช่วงสำหรับแต่ละมิติ เพื่อเปลี่ยนลำดับชั้นของมิติในพื้นที่ **ลำดับชั้น และช่วงของเซ็กเมนต์** เลือกมิติที่จะย้าย จากนั้นคลิก **เลื่อนขึ้น** หรือ **เลื่อนลง**
8.  เพื่อระบุค่าช่วงของมิติเพื่อเพิ่มไปยังแผนภูมิการรายงานใหม่ ในพื้นที่ **ลำดับชั้น และช่วงของเซ็กเมนต์** ให้ปฏิบัติตามขั้นตอนต่อไปนี้:
    1.  ในฟิลด์ **มิติต้นทาง** สำหรับมิตินั้น ป้อนค่าแรกในช่วง
    2.  ในฟิลด์ **มิติสิ้นสุด** ป้อนค่าสุดท้ายในช่วงนั้น

9.  ทำซ้ำขั้นตอนที่ 7 ถึง 8 สำหรับแต่ละมิติในพื้นที่ **ลำดับชั้น และช่วงของเซ็กเมนต์**
10. หลังจากคุณเสร็จสิ้นการกำหนดวิธีนำหน่วยการรายงานใส่ในแผนภูมิการรายงานใหม่ คลิก **ตกลง**
11. คลิก **ไฟล์** &gt; **บันทึก** เพื่อบันทึกแผนภูมิการรายงาน ป้อนชื่อที่ไม่ซ้ำ และคำอธิบายสำหรับแผนภูมิการรายงาน และจากนั้นคลิก **ตกลง**

### <a name="open-an-existing-reporting-tree-definition"></a>เปิดคำนิยามแผนภูมิการรายงานที่มีอยู่

1.  ใน Report Designer คลิก **คำนิยามแผนภูมิการรายงาน** ในบานหน้าต่างนำทาง
2.  ดับเบิลคลิกชื่อในรายการแผนภูมิการรายงานเพื่อเปิด
3.  เพื่อดูบล็อคส่วนประกอบใดๆ ที่เชื่อมโยงกับแผนภูมิการรายงาน คลิกขวาที่คำนิยามแผนภูมิการรายงาน จากนั้นเลือก **การเชื่อมโยง**

### <a name="roll-up-data-in-a-reporting-tree"></a>รวมข้อมูลในแผนภูมิการรายงาน

เมื่อคุณใช้แผนภูมิการรายงาน คุณสามารถรวมยอดเงินจากหน่วยการรายงานรองที่ระดับหน่วยการรายงานหลัก การรวมนี้เรียกว่าการรวมข้อมูล กฎต่อไปนี้ใช้เพื่อรวมยอดเงินกับหน่วยหลักในแผนภูมิการรายงาน:

-   ในแผนภูมิการรายงาน หน่วยรองต้องประกอบด้วยมิติ นอกจากว่าแผนภูมิการรายงานเป็นแผนภูมิระดับเดียว หน่วยหลักโดยปกติจะไม่ประกอบด้วยมิติในแผนภูมิการรายงาน **หมายเหตุ:** ถ้าคุณระบุมิติสำหรับทั้งหน่วยรอง และหน่วยหลัก คุณอาจสามารถทำสำเนาของข้อมูลในรายงาน
-   หน่วยการรายงานที่ประกอบด้วยมิติในแผนภูมิการรายงานจะสอดคล้องกับมิติที่ใช้ในคำนิยามแถว และคอลัมน์ ชุดของมิติกำหนดยอดเงินที่ส่งคืนสำหรับหน่วยนั้น ตัวอย่างเช่น ในตัวอย่างที่ 2 ในบทความนี้หลังจากนี้ รายการที่ 6 และ 7 จะส่งคืนเฉพาะค่าสำหรับแผนก 00 และ 01 ตามลำดับ
-   ยอดเงินสำหรับหน่วยการรายงานหลักที่ไม่ประกอบด้วยมิติในแผนภูมิรายงานจะถูกกำหนดจากหน่วยการรายงานรอง และรวมยอดเงินไปยังหน่วยหลักที่ระบุ ตัวอย่างเช่น ถ้าหน่วยหลัก (ดู Contoso USA ในตัวอย่างที่ 2 ของตัวอย่างการรวมข้อมูล) มีสองหน่วยรอง (022 และ 023) และไม่ประกอบด้วยมิติ รายงานจะถูกสร้างสำหรับแต่ละข้อมูลรอง และข้อมูลหลัก ผลรวมหลัก คือ ผลรวมของสองยอดเงินรอง

### <a name="manage-reporting-units"></a>จัดการหน่วยการรายงาน

คำนิยามแผนภูมิรายงานแต่ละรายการจะแสดงในมุมมองที่ไม่ซ้ำกัน มีมุมมองกราฟิกที่แสดงลำดับชั้นหลัก/รอง และมุมมองแผ่นงานที่แสดงข้อมูลที่เฉพาะเจาะจงสำหรับแต่ละหน่วยการรายงาน มุมมองแบบกราฟิก และมุมมองแผ่นงานเชื่อมต่อกัน เมื่อคุณเลือกหน่วยการรายงานในมุมมองเดียว จะถูกเลือกในมุมมองอื่นด้วย คุณสามารถสร้างลำดับชั้นข้ามมิติโดยอาศัยความสัมพันธ์มิติในข้อมูลทางการเงิน เมื่อคุณสร้างคำนิยามแผนภูมิการรายงาน คุณสามารถใช้คำนิยามแถวเดียวกันหลายครั้ง ไม่ว่าคุณกำลังสร้างรายงานการเงินเงินได้ของแผนก หรือรายงานการเงินสรุปเงินได้แบบรวม มิติที่กำหนดไว้ในคำนิยามแถวที่ถูกรวมกับมิติในคำนิยามแผนภูมิการรายงานแสดงมุมมองที่หลากหลายของประสิทธิภาพขององค์กรของคุณ

### <a name="reporting-unit-structure"></a>โครงสร้างหน่วยการรายงาน

ชนิดของหน่วยการรายงานต่อไปนี้จะใช้ในรายงานทางการเงิน:

-   หน่วยรายละเอียดดึงข้อมูลโดยตรงจากข้อมูลทางการเงิน จากแผ่นงาน Excel หรือจากแผ่นงานรายงานทางการเงินอื่น
-   หน่วยสรุปสรุปข้อมูลจากหน่วยระดับล่าง

หน่วยการรายงานหลักเป็นหน่วยสรุปที่รวมข้อมูลสรุปจากหน่วยรายละเอียด หน่วยสรุปสามารถเป็นทั้งหน่วยรายละเอียด และหน่วยสรุป ซึ่งหมายความว่าหน่วยสรุปสามารถดึงข้อมูลจากหน่วยระดับต่ำกว่า ข้อมูลทางการเงิน หรือแผ่นงาน Excel หน่วยหลักสามารถเป็นหน่วยรองของหน่วยหลักที่มีระดับสูงกว่า หน่วยการรายงานรองสามารถเป็นหน่วยรายละเอียดที่ดึงข้อมูลโดยตรงจากข้อมูลทางการเงิน หรือแผ่นงาน Excel หน่วยการรายงานรองยังสามารถเป็นหน่วยสรุประดับกลางได้อีกด้วย กล่าวคืออาจเป็นหน่วยหลักของหน่วยระดับต่ำกว่า และหน่วยรองของหน่วยสรุประดับที่สูงกว่า ในสถานการณ์ทั่วไปสำหรับหน่วยการรายงานจะมีหน่วยหลักพร้อมเซลล์ว่างอยู่ในคอลัมน์ **มิติ** และมีหน่วยรองพร้อมลิงค์ไปที่ชุดมิติเฉพาะเจาะจง หรือชุดมิติอักขระตัวแทน

### <a name="organize-reporting-units"></a>จัดระเบียบหน่วยการรายงาน

คุณสามารถจัดเรียงโครงสร้างองค์กรของคำนิยามแผนภูมิการรายงานด้วยการย้ายหน่วยการรายงานในมุมมองกราฟิก คุณสามารถเลื่อนหน่วยการรายงานไปยังระดับที่สูงกว่าในแผนภูมิการรายงาน หรือลดระดับไปยังระดับต่ำกว่า

1.  ใน Report Designer เปิดคำนิยามแผนภูมิการรายงานเพื่อแก้ไข
2.  ในมุมมองกราฟิกของคำนิยามแผนภูมิการรายงาน เลือกหน่วยการรายงาน
3.  ลากหน่วยไปยังตำแหน่งใหม่ หรือคลิกขวาที่หน่วย และเลือก **เลื่อนหน่วยการรายงาน** หรือ **ลดระดับหน่วยการรายงาน**
4.  คลิก **ไฟล์** &gt; **บันทึก** เพื่อบันทึกการเปลี่ยนแปลงของคุณ

### <a name="add-text-about-a-reporting-unit"></a>เพิ่มข้อความเกี่ยวกับหน่วยการรายงาน

รายการข้อความเพิ่มเติมเป็นสตริงข้อความแบบคงที่ สูงสุดถึง 255 อักขระ ที่เพิ่มข้อมูลไปยังคำนิยามแผนภูมิการรายงาน ตัวอย่างเช่น ข้อความเพิ่มเติมอาจเป็นคำอธิบายย่อของบริษัท คุณสามารถสร้างรายการข้อความเพิ่มเติมสูงสุดถึงสิบสำหรับแต่ละหน่วยการรายงานในคำนิยามแผนภูมิการรายงาน ข้อความเพิ่มเติมปรากฏในรายงานสำหรับหน่วยการรายงานซึ่งข้อความถูกกำหนด คุณสามารถเพิ่มรายการข้อความจากคอลัมน์ **คำอธิบาย** ของคำนิยามแถว และจากแท็บ **หัวกระดาษ และท้ายกระดาษ** ในข้อกำหนดของรายงานได้

1.  ใน Report Designer เปิดคำนิยามแผนภูมิการรายงานเพื่อแก้ไข
2.  ดับเบิลคลิกเซลล์ **ข้อความเพิ่มเติม** สำหรับแถวหน่วยการรายงาน
3.  ในกล่องโต้ตอบ **ข้อความเพิ่มเติม** ในแถวแรกที่ว่าง ป้อนข้อความ แถวแรกที่ประกอบด้วยข้อความที่ถูกอ้างอิงเป็น UnitText1 โดยไม่คำนึงถึงตำแหน่งในกล่องโต้ตอบ **ข้อความเพิ่มเติม**
4.  เพื่อเพิ่มรายการข้อความเพิ่มเติมสำหรับหน่วยการรายงานนี้ ป้อนข้อความในแถวที่ว่าง
5.  คลิก **ตกลง** ระบบจะนำเข้าข้อมูลการชำระเงิน

### <a name="remove-additional-text-from-a-reporting-unit"></a>ลบข้อความเพิ่มเติมจากหน่วยการรายงาน

1.  ใน Report Designer เปิดคำนิยามลำดับการรายงานเพื่อแก้ไข
2.  ดับเบิลคลิกเซลล์ **ข้อความเพิ่มเติม** สำหรับแถวหน่วยการรายงาน
3.  ในกล่องโต้ตอบ **ข้อความเพิ่มเติม** เลือกรายการเพื่อลบ แล้วคลิก **ล้าง** อีกวิธีหนึ่งคือ คลิกขวาที่รายการ และจากนั้น เลือก **ตัด**
4.  คลิก **ตกลง** ระบบจะนำเข้าข้อมูลการชำระเงิน

### <a name="restrict-access-to-a-reporting-unit"></a>จำกัดการเข้าถึงหน่วยการรายงาน

คุณสามารถป้องกันผู้ใช้ และกลุ่มจากการเข้าถึงหน่วยการรายงาน นอกจากนี้คุณยังสามารถกำหนดข้อจำกัดเพื่อที่จะนำไปใช้กับหน่วยการรายงานรองของหน่วยการรายงานได้

1.  ใน Report Designer เปิดคำนิยามแผนภูมิการรายงานเพื่อแก้ไข
2.  ดับเบิลคลิกเซลล์ **ความปลอดภัยของหน่วย** สำหรับแถวหน่วยการรายงานเพื่อจำกัดการเข้าถึง
3.  ในกล่องโต้ตอบ **หน่วยความปลอดภัย** คลิก **ผู้ใช้และกลุ่ม**
4.  เลือกผู้ใช้ หรือกลุ่ม ที่ควรเข้าถึงหน่วยการรายงาน จากนั้นคลิก **ตกลง**
5.  เพื่อจำกัดการเข้าถึงหน่วยการรายงานรอง เลือก **เพิ่มความปลอดภัยไปยังกล่องกาเครื่องหมายหน่วยการรายงานรอง**
6.  คลิก **ตกลง** ระบบจะนำเข้าข้อมูลการชำระเงิน

### <a name="remove-access-to-a-reporting-unit"></a>ลบการเข้าถึงหน่วยการรายงาน

1.  ใน Report Designer เปิดคำนิยามแผนภูมิการรายงานเพื่อแก้ไข
2.  ดับเบิลคลิกเซลล์ **หน่วยความปลอดภัย** สำหรับเเถวหน่วยการรายงานเพื่อลบการเข้าถึง
3.  ในกล่องโต้ตอบ **หน่วยความปลอดภัย** เลือกชื่อ เเละจากนั้นคลิก **ลบ**
4.  คลิก **ตกลง** ระบบจะนำเข้าข้อมูลการชำระเงิน

### <a name="link-to-reports"></a>ลิงค์ไปยังรายงาน

หลังจากที่คุณได้สร้างคอลัมน์ **รายงาน** ในคำนิยามแถว และระบุรายงานเพื่อรวมไว้ในรายงาน คุณต้องอัพเดตแผนภูมิการรายงานด้วยคอลัมน์ที่เชื่อมโยง และข้อมูลเกี่ยวกับรายงานนั้น รายงานสามารถนำเข้าไปยังหน่วยใดๆ ในแผนภูมิการรายงาน

### <a name="identify-the-report-in-a-reporting-tree"></a>ระบุรายงานในแผนภูมิการรายงาน

1.  ใน Report Designer เปิดคำนิยามแผนภูมิการรายงานเพื่อแก้ไข
2.  ในคอลัมน์ **คำนิยามแถว** ข้อมูลในเซลล์อาศัยข้อมูลแถวที่เลือก เนื่องจากคำนิยามแถวเดียวกันต้องใช้ในทุกหน่วยของแผนภูมิการรายงาน ดับเบิลคลิกเซลล์ **คำนิยามแถว** และจากนั้นเลือกคำนิยามแถวที่ประกอบด้วยข้อมูลเกี่ยวกับรายงาน
3.  ในเซลล์ **ลิงค์แผ่นงาน** สำหรับหน่วยการรายงาน เลือกชื่อลิงค์ที่สอดคล้องกับรายงาน
4.  ในเซลล์ **สมุดงาน หรือพาธรายงาน** สำหรับหน่วยการรายงาน ป้อนชื่อของรายงาน หรือเรียกดูเพื่อเลือกรายงาน
5.  เพื่อระบุแผ่นงานในรายงาน ให้ป้อนชื่อของแผ่นงานในเซลล์ **ชื่อแผ่นงาน**
6.  ทำซ้ำขั้นตอนที่ 3 ถึง 5 สำหรับแต่ละหน่วยการรายงานที่ควรได้รับข้อมูลจากรายงาน เพื่อป้องกันข้อมูลที่ไม่ถูกต้องปรากฏในรายงานของคุณ ตรวจสอบให้แน่ใจว่าชื่อรายงาน ที่ถูกต้องปรากฏขึ้นในหน่วยที่สอดคล้องกันของแผนภูมิการรายงาน
