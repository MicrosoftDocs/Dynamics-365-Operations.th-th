# <a name="learning-power-bi-content"></a>เนื้อหาการเรียนรู้ Power BI

[!include[banner](../includes/banner.md)]

หัวข้อนี้อธิบายถึงเนื้อหา **การเรียนรู้** ใน Microsoft Power BI ซึ่งจะอธิบายถึงวิธีการเข้าถึงเนื้อหา และอธิบายถึงแบบจำลองข้อมูลและเอนทิตีที่ใช้ในการสร้างเนื้อหา

## <a name="accessing-the-power-bi-content"></a>การเข้าถึงเนื้อหา Power BI

ถ้าคุณกำลังใช้ Microsoft Dynamics 365 for Finance and Operations, Enterprise edition (กรกฎาคม 2017)คุณสามารถค้นหาเนื้อหา Power BI **การเรียนรู้** ในไลบรารีสินทรัพย์ใช้ร่วมกันใน Microsoft Dynamics Lifecycle Services (LCS) สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการดาวน์โหลดเนื้อหา Power BI และนำไปใช้ในองค์กรของคุณ ให้ดูที่ [เนื้อหา Power BI ใน LCS จาก Microsoft และคู่ค้าของคุณ](power-bi-content-microsoft-partners.md) เมื่อต้องการดูการสาธิตที่แสดงวิธีใช้เนื้อหา Power BI ให้ดูที่ Office Mix [เนื้อหา Power BI จาก Microsoft และคู่ค้าของคุณใน Dynamics Lifecycle Services](https://mix.office.com/watch/9puyb1b2xs1w)

## <a name="reports-that-are-included-in-the-power-bi-content"></a>รายงานที่รวมอยู่ในเนื้อหา Power BI

รายงานที่รวมอยู่ในเนื้อหา Power BI **การเรียนรู้** มีทั้งแผนภูมิและตารางที่ประกอบด้วยข้อมูลเพิ่มเติม ตารางต่อไปนี้ให้คำอธิบายเกี่ยวกับรายงาน

| รายงาน                | เนื้อหา |
|-----------------------|----------|
| ภาพรวมของการเรียนรู้     | สรุปการรายงานอื่น ๆ |
| การวิเคราะห์หลักสูตร       | การลงทะเบียนโดยเรียงตามสถานที่ ผู้เข้าร่วมโดยเรียงตามสถานะ หลักสูตรโดยเรียงตามชนิดต่อบริษัท และการเข้าเรียนในหลักสูตรโดยเรียงตามงาน |
| การวิเคราะห์การลงทะเบียน | รายการลงทะเบียน |
| ชนิดของหลักสูตร          | ชนิดของหลักสูตรตามทักษะ |
| การวิเคราะห์ผู้สอน   | อัตราส่วนของหลักสูตรต่อผู้สอน จำนวนผู้สอน หลักสูตรโดยเรียงตามผู้สอน หลักสูตรต่อผู้สอน และวาระการประชุมโดยเรียงตามผู้สอน |
| หลักสูตรที่นำเสนอ       | รายการหลักสูตร |
| การออกแบบหลักสูตร        | วาระการประชุมของหลักสูตร |

คุณสามารถกรองข้อมูลแผนภูมิและไทล์ในรายงานเหล่านี้ และตรึงแผนภูมิและไทล์ไปยังแดชบอร์ด ดูข้อมูลเพิ่มเติมเกี่ยวกับวิธีการกรองข้อมูลและตรึงใน Power BI ได้ที่ [สร้างและตั้งค่าคอนฟิกแดชบอร์ด](https://powerbi.microsoft.com/en-us/guided-learning/powerbi-learning-4-2-create-configure-dashboards)

## <a name="understanding-the-data-model-and-entities"></a>การทำความเข้าใจเกี่ยวกับแบบจำลองข้อมูลและเอนทิตี้

ข้อมูลดังต่อไปนี้ถูกใช้ในการกรอกข้อมูลรายงานในเนื้อหา Power BI **การเรียนรู้** ตารางนี้แสดงเอนทิตีที่ชุดเนื้อหายึดตาม

| เอนทิตี้           | เนื้อหา                                                         | ความสัมพันธ์กับเอนทิตีอื่น |
|------------------|------------------------------------------------------------------|-----------------------------------|
| ออฟเซ็ตของปฏิทิน  | ปฏิทินออฟเซ็ตเพื่อแบ่งส่วนรายงาน                                | วาระการประชุมของหลักสูตร ผู้เข้าร่วมหลักสูตร |
| บริษัท          | บริษัทสามารถกรองข้อมูลรายงานโดย                                   | วาระการประชุมของหลักสูตร ผู้เข้าร่วมหลักสูตร |
| หลักสูตร           | หลักสูตร คำอธิบาย ชื่อผู้สอน สถานที่ ห้อง และสถานะ | วาระการประชุมของหลักสูตร ผู้เข้าร่วมหลักสูตร ทักษะในหลักสูตร |
| วาระการประชุมของหลักสูตร    | วาระการประชุม หลักสูตร และเวลาเริ่มต้นและสิ้นสุด                          | บริษัท ออฟเซ็ตปฏิทิน วันที่ หลักสูตร |
| ผู้เข้าร่วมหลักสูตร | ชื่อ สถานะ งาน และวันที่ลงทะเบียน                         | บริษัท ออฟเซ็ตของปฏิทิน วันที่ หลักสูตร ข้อมูลประชากร การจ้างงาน หลักสูตร ชื่อพนักงาน ตำแหน่งของพนักงาน งาน ตำแหน่ง |
| ทักษะหลักสูตร     | ทักษะ ชนิดของทักษะ และระดับ                                     | หลักสูตร |
| วัน เดือน             | จำนวนวัน, จำนวนสัปดาห์, จำนวนเดือน และจำนวนปี                                   | วาระการประชุมของหลักสูตร ผู้เข้าร่วมหลักสูตร |
| ข้อมูลประชากร     | วันเกิด เพศ เชื้อชาติ และสถานภาพการสมรส         | วาระการประชุมของหลักสูตร ผู้เข้าร่วมหลักสูตร |
| การจ้างงาน       | วันที่เริ่มต้น วันที่สิ้นสุด และวันที่เปลี่ยน                        | วาระการประชุมของหลักสูตร ผู้เข้าร่วมหลักสูตร |
| งาน              | ฟังก์ชัน ชนิด และตำแหน่งงาน                                        | วาระการประชุมของหลักสูตร ผู้เข้าร่วมหลักสูตร |
| ตำแหน่ง         | ตำแหน่งงาน ชื่อ และเทียบเท่าเต็มเวลา (FTE)                  | วาระการประชุมของหลักสูตร ผู้เข้าร่วมหลักสูตร |
| ชื่อพนักงาน    | ชื่อจริง นามสกุล และชื่อเต็ม                             | ผู้เข้าร่วมหลักสูตร |
| ตำแหน่งของพนักงาน   | ตำแหน่งงานและวันที่อายุงาน                                         | ผู้เข้าร่วมหลักสูตร |

เอนทิตีเหล่านี้ถูกใช้เพื่อสร้างการวัดที่คำนวณในแบบจำลองข้อมูล จากนั้นจะมีการใช้การวัดที่คำนวณแล้วเหล่านี้เพื่อคำนวณตัวบ่งชี้ประสิทธิภาพหลัก (KPI) และรายงานที่ใช้ในเนื้อหา ถ้าคุณต้องการรวมการคำนวณเพิ่มเติมในรายงานและแดชบอร์ดของคุณ คุณสามารถดาวน์โหลด และปรับเปลี่ยนไฟล์ .pbix จาก LCS ไฟล์นี้เป็นแบบจำลองข้อมูลเริ่มต้นที่ใช้ในการสร้างเนื้อหา หลังจากที่คุณทำการเปลี่ยนแปลงแล้ว คุณสามารถสร้างชุดเนื้อหาและแดชบอร์ดขององค์กรที่ประกอบด้วยข้อมูลที่คุณเพิ่มไว้ได้
