<span data-ttu-id="79d92-104">ลำดับหมายเลขถูกใช้ในการสร้างรหัสที่สามารถอ่านได้ รหัสเฉพาะสำหรับเร็กคอร์ดข้อมูลหลักและเรกคอร์ดธุรกรรมที่ต้องการ </span><span class="sxs-lookup"><span data-stu-id="79d92-104">Number sequences are used to generate readable, unique identifiers for master data records and transaction records that require them.</span></span> <span data-ttu-id="79d92-105">ข้อมูลหลักหรือเรกคอร์ดธุรกรรมที่จำเป็นต้องมีตัวระบุจะอ้างเป็นการอ้างอิง</span><span class="sxs-lookup"><span data-stu-id="79d92-105">A master data or transaction record that requires an identifier is referred to as a reference.</span></span> <span data-ttu-id="79d92-106">ก่อนที่คุณจะสามารถสร้างเรกคอร์ดใหม่สำหรับการอ้างอิงได้ คุณต้องตั้งค่าลำดับหมายเลขและเชื่อมโยงกับการอ้างอิงนั้น</span><span class="sxs-lookup"><span data-stu-id="79d92-106">Before you can create new records for a reference, you must set up a number sequence and associate it with the reference.</span></span> <span data-ttu-id="79d92-107">คุณสามารถตั้งค่าลำดับหมายเลขที่ต้องการทั้งหมดในเวลาเดียวกัน โดยการใช้วิซาร์ดการตั้งค่าลำดับหมายเลข หรือคุณสามารถสร้างหรือแก้ไขลำดับหมายเลขแต่ละตัวได้โดยการใช้หน้าลำดับหมายเลข</span><span class="sxs-lookup"><span data-stu-id="79d92-107">You can set up all required number sequences at the same time by using the Set up number sequences wizard, or you can create or modify individual number sequences by using the Number sequences page.</span></span>

    * <span data-ttu-id="79d92-119">บน FastTab เซ็กเมนต์ ให้กำหนดรูปแบบสำหรับลำดับหมายเลขโดยการเพิ่ม การลบ และการจัดเรียงเซ็กเมนต์ใหม่</span><span class="sxs-lookup"><span data-stu-id="79d92-119">On the Segments FastTab, define the format for the number sequence by adding, removing, and rearranging segments.</span></span>
