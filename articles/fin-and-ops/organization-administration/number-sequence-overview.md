<span data-ttu-id="d0675-104">ลำดับหมายเลขที่ใช้ในการสร้างตัวระบุเฉพาะที่สามารถอ่านได้สำหรับเร็กคอร์ดข้อมูลหลักและเรกคอร์ดธุรกรรมที่จำเป็นต้องมีตัวระบุ</span><span class="sxs-lookup"><span data-stu-id="d0675-104">Number sequences are used to generate readable, unique identifiers for master data records and transaction records that require identifiers.</span></span> <span data-ttu-id="d0675-105">เรกคอร์ดข้อมูลหลักหรือเรกคอร์ดธุรกรรมที่จำเป็นต้องมีตัวระบุจะถูกอ้างเป็น *การอ้างอิง*</span><span class="sxs-lookup"><span data-stu-id="d0675-105">A master data record or transaction record that requires an identifier is referred to as a *reference*.</span></span>

## <a name="performance-considerations-for-number-sequences"></a><span data-ttu-id="d0675-182">การพิจารณาประสิทธิภาพสำหรับลำดับหมายเลข</span><span class="sxs-lookup"><span data-stu-id="d0675-182">Performance considerations for number sequences</span></span>

<span data-ttu-id="d0675-183">พิจารณาข้อมูลต่อไปนี้เกี่ยวกับผลกระทบของการตั้งค่าคอนฟิกของลำดับหมายเลขที่มีต่อประสิทธิภาพของระบบก่อนที่คุณจะตั้งค่าลำดับหมายเลข</span><span class="sxs-lookup"><span data-stu-id="d0675-183">Consider the following information about how the configuration of number sequences can affect system performance before you set up number sequences.</span></span>

### <a name="continuous-and-non-continuous-number-sequences"></a><span data-ttu-id="d0675-184">ลำดับหมายเลขที่ต่อเนื่องและไม่ต่อเนื่อง</span><span class="sxs-lookup"><span data-stu-id="d0675-184">Continuous and non-continuous number sequences</span></span>

<span data-ttu-id="d0675-185">ลำดับหมายเลขอาจต่อเนื่องหรือไม่ต่อเนื่อง</span><span class="sxs-lookup"><span data-stu-id="d0675-185">Number sequences can be continuous or non-continuous.</span></span> <span data-ttu-id="d0675-186">ลำดับหมายเลขที่ต่อเนื่องจะไม่ข้ามหมายเลขใดๆ แต่อาจไม่สามารถใช้หมายเลขตามลำดับได้</span><span class="sxs-lookup"><span data-stu-id="d0675-186">A continuous number sequence does not skip any numbers, but numbers may not be used sequentially.</span></span> <span data-ttu-id="d0675-187">หมายเลขจากลำดับหมายเลขที่ไม่ต่อเนื่องจะถูกใช้โดยเรียงลำดับ แต่ลำดับหมายเลขอาจข้ามหมายเลข</span><span class="sxs-lookup"><span data-stu-id="d0675-187">Numbers from a non-continuous number sequence are used sequentially, but the number sequence may skip numbers.</span></span> <span data-ttu-id="d0675-188">ตัวอย่างเช่น ถ้าผู้ใช้ยกเลิกธุรกรรม มีการสร้างหมายเลข แต่ไม่ได้ใช้</span><span class="sxs-lookup"><span data-stu-id="d0675-188">For example, if a user cancels a transaction, a number is generated, but not used.</span></span> <span data-ttu-id="d0675-189">ในลำดับหมายเลขที่ต่อเนื่อง หมายเลขดังกล่าวจะถูกนำกลับมาใช้ใหม่ในภายหลัง</span><span class="sxs-lookup"><span data-stu-id="d0675-189">In a continuous number sequence, that number is recycled later.</span></span> <span data-ttu-id="d0675-190">ในลำดับหมายเลขที่ไม่ต่อเนื่อง หมายเลขนั้นจะไม่ถูกนำมาใช้</span><span class="sxs-lookup"><span data-stu-id="d0675-190">In a non-continuous number sequence, the number is not used.</span></span>
