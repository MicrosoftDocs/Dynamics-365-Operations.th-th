# <a name="configure-the-properties-of-a-workflow"></a>ตั้งค่าคอนฟิกคุณสมบัติของลำดับงาน

[!include[banner](../includes/banner.md)]


หัวข้อนี้อธิบายวิธีการตั้งค่าคอนฟิกคุณสมบัติต่างๆ ของลำดับงาน

เมื่อต้องการตั้งค่าคอนฟิกคุณสมบัติของลำดับงาน เปิดลำดับงานในโปรแกรมแก้ไขลำดับงาน  คลิกผ้าของโปรแกรมแก้ไขลำดับงาน แล้วคลิก **คุณสมบัติ** เพื่อเปิดหน้า **คุณสมบัติ** จากนั้นคุณสามารถใช้กระบวนงานต่อไปนี้เพื่อตั้งค่าคอนฟิกคุณสมบัติต่างๆ ของลำดับงานได้

## <a name="name-the-workflow"></a>การตั้งชื่อลำดับงาน
ทำตามขั้นตอนเหล่านี้ในการป้อนชื่อสำหรับลำดับงาน

1.  ในบานหน้าต่างทางซ้าย ให้คลิก **การตั้งค่าพื้นฐาน**
2.  ในฟิลด์ **ชื่อ** ให้ป้อนชื่อเฉพาะสำหรับลำดับงาน เช่น ถ้าคุณสร้างลำดับงานของใบสั่งซื้อสำหรับแต่ละประเทศหรือภูมิภาคที่คุณดำเนินธุรกิจอยู่ คุณสามารถตั้งชื่อลำดับงานของใบสั่งซื้อ **ใบสั่งซื้อเดนมาร์ก** หรือ **ใบสั่งซื้อสเปน**

## <a name="specify-the-workflow-owner"></a>การระบุเจ้าของลำดับงาน
เจ้าของลำดับงาน คือบุคคลที่จัดการและรักษาลำดับงานนั้นๆ ทำตามขั้นตอนต่อไปนี้ในการระบุเจ้าของลำดับงาน

1.  ในบานหน้าต่างทางซ้าย ให้คลิก **การตั้งค่าพื้นฐาน**
2.  ในรายการ **เจ้าของ** เลือกชื่อของบุคคลที่จะจัดการลำดับงาน

## <a name="select-an-email-template"></a>เลือกเท็มเพลตอีเมล
ทำตามขั้นตอนเหล่านี้เพื่อเลือกเท็มเพลตอีเมลที่ใช้ในการสร้างข้อความแจ้งเตือนเกี่ยวกับลำดับงาน

1.  ในบานหน้าต่างทางซ้าย ให้คลิก **การตั้งค่าพื้นฐาน**
2.  ในรายการ **เท็มเพลตอีเมลสำหรับการแจ้งเตือนลำดับงาน** เลือกเท็มเพลต

## <a name="enter-instructions-for-users"></a>การป้อนคำแนะนำสำหรับผู้ใช้
คุณสามารถให้คำแนะนำแก่ผู้ใช้ที่ส่งเอกสารสำหรับการประมวลผลและการอนุมัติ ผู้ใช้เหล่านี้จะเรียกว่า *ผู้ริเริ่ม* ตัวอย่างเช่น คุณกำลังสร้างลำดับงานใบขอซื้อ และคุณป้อนคำแนะนำ คำแนะนำเหล่านั้นจะปรากฏแก่ผู้ใช้ที่ป้อนใบขอซื้อในหน้า **ใบขอซื้อ** เมื่อต้องการดูคำสั่ง ผู้ริเริ่มจะคลิกไอคอนในแถบข้อความลำดับงาน ทำตามขั้นตอนเหล่านี้ในการป้อนคำแนะนำสำหรับผู้ใช้

1.  ในบานหน้าต่างทางซ้าย ให้คลิก **การตั้งค่าพื้นฐาน**
2.  ในฟิลด์ **คำแนะนำในการส่ง** ป้อนคำแนะนำ
3.  เมื่อต้องการปรับแต่งคำแนะนำ คุณสามารถแทรกตัวยึด ตัวยึดจะถูกแทนที่ด้วยข้อมูลที่เหมาะสมเมื่อมีการแสดงคำสั่งแก่ผู้ใช้ เมื่อต้องการแทรกตัวยึด ให้ทำตามขั้นตอนเหล่านี้:
    1.  คลิกในฟิลด์ **คำแนะนำในการส่ง** เพื่อระบุตำแหน่งที่ต้องการให้ตัวยึดปรากฏ
    2.  คลิก **แทรกตัวยึด**
    3.  ในรายการที่ปรากฏขึ้น ให้เลือกตัวยึดที่จะแทรก
    4.  คลิก **แทรก**

4.  เมื่อต้องการเพิ่มคำแปลของคำแนะนำ ให้ทำตามขั้นตอนเหล่านี้:
    1.  คลิก **คำแปล**
    2.  บนหน้าที่ปรากฏ คลิก **เพิ่ม**
    3.  ในรายการที่ปรากฏ ให้เลือกภาษาที่คุณจะป้อนข้อความ
    4.  ในฟิลด์ **ข้อความที่แปล** ให้ป้อนข้อความ
    5.  เมื่อต้องการปรับแต่งข้อความ คุณสามารถแทรกตัวยึด สำหรับคำแนะนำเกี่ยวกับวิธีการป้อนตัวยึด ดูขั้นตอนที่ 3
    6.  คลิก **ปิด**

## <a name="specify-when-this-workflow-is-used"></a>ระบุว่าใช้ลำดับงานนี้เมื่อใด
คุณสามารถสร้างลำดับงานหลายรายการที่ยึดตามชนิดเดียวกันได้ เช่น คุณสามารถสร้างลำดับงานของใบสั่งซื้อสำหรับแต่ละประเทศหรือภูมิภาคที่คุณดำเนินธุรกิจอยู่ เช่น ใบสั่งซื้อเดนมาร์ก และ ใบสั่งซื้อสเปน เมื่อคุณมีลำดับงานหลายรายการที่ยึดตามชนิดเดียวกัน คุณต้องระบุว่าใช้ลำดับงานแต่ละรายการเมื่อใด สำหรับตัวอย่างข้างต้น คุณระบุเงื่อนไขต่อไปนี้:

-   ใบสั่งซื้อเดนมาร์กถูกนำมาใช้เมื่อ: ประเทศ/ภูมิภาค = DK
-   ใบสั่งซื้อสเปนถูกนำมาใช้เมื่อ: ประเทศ/ภูมิภาค = ES

ทำตามขั้นตอนเหล่านี้ในการระบุว่าใช้ลำดับงานที่คุณกำลังตั้งค่าคอนฟิกเมื่อใด

1.  ในบานหน้าต่างทางซ้าย ให้คลิก **การเรียกใช้**
2.  เลือกกล่องกาเครื่องหมาย **ตั้งค่าเงื่อนไขสำหรับการรันลำดับงานนี้**
3.  คลิก **เพิ่มเงื่อนไข**
4.  ป้อนเงื่อนไข
5.  ป้อนเงื่อนไขเพิ่มเติมใด ๆ ที่จำเป็น
6.  หากต้องการตรวจสอบว่าเงื่อนไขที่คุณป้อนเข้าไปกำหนดไว้ถูกต้องหรือไม่ ให้ทำตามขั้นตอนเหล่านี้:
    1.  คลิก **ทดสอบ**
    2.  ในหน้า **ทดสอบเงื่อนไขลำดับงาน** ในพื้นที่ **ตรวจสอบความถูกต้องของเงื่อนไข** เลือกเรกคอร์ด
    3.  คลิก **ทดสอบ** ระบบจะประเมินเรกคอร์ดเพื่อพิจารณาว่าเป็นไปตามเงื่อนไขที่คุณระบุไว้หรือไม่ ตัวอย่างเช่น ถ้าคุณกำลังสร้างลำดับงานใบขอซื้อสำหรับสเปน พื้นที่ **ตรวจสอบความถูกต้องของเงื่อนไข** ของหน้าจะแสดงรายการของใบขอซื้อ เมื่อคุณคลิก **ทดสอบ**ระบบจะประเมินใบสั่งซื้อที่เลือกเพื่อกำหนดว่าประเทศ/ภูมิภาคคือ ES หรือไม่
    4.  คลิก **ตกลง** หรือ **ยกเลิก** เพื่อกลับไปที่หน้า **คุณสมบัติ**

## <a name="specify-when-notifications-are-sent"></a>ระบุว่าจะส่งการแจ้งเตือนเมื่อใด
เมื่อมีการส่งเอกสารสำหรับการประมวลผล อินสแตนซ์ลำดับงานจะถูกสร้างขึ้น คุณสามารถส่งการแจ้งเตือนให้แก่ผู้ใช้เมื่ออินสแตนซ์ลำดับงานที่ยึดตามลำดับงานเริ่มต้นแล้ว เสร็จสมบูรณ์ หยุดการทำงาน หรือหยุดเนื่องจากมีข้อผิดพลาด ทำตามขั้นตอนเหล่านี้ในการระบุว่าจะส่งการแจ้งเตือนเมื่อใด

1.  ในบานหน้าต่างทางซ้าย ให้คลิก **การแจ้งเตือน**
2.  เลือกกล่องกาเครื่องหมายสำหรับแต่ละเหตุการณ์ที่ควรทริกเกอร์การแจ้งเตือน:
    -   **เริ่มต้นแล้ว** — ส่งการแจ้งเตือนเมื่ออินสแตนซ์ลำดับงานเริ่มต้นขึ้น
    -   **หยุด** — ส่งการแจ้งเตือนเมื่ออินสแตนซ์ลำดับงานหยุดเนื่องจากมีข้อผิดพลาด
    -   **เสร็จสมบูรณ์** — ส่งการแจ้งเตือนเมื่ออินสแตนซ์ลำดับงานเสร็จสมบูรณ์
    -   **ไม่สามารถแก้ไขได้** — ส่งการแจ้งเตือนเมื่ออินสแตนซ์ลำดับงานหยุดเนื่องจากมีข้อผิดพลาดที่ไม่สามารถแก้ไขได้
    -   **หยุดการทำงาน** — ส่งการแจ้งเตือนเมื่ออินสแตนซ์ลำดับงานหยุดการทำงาน

3.  เลือกแถวของเหตุการณ์ที่คุณเลือกในขั้นตอนที่ 2
4.  ในแท็บ **ข้อความแจ้งเตือน** ป้อนข้อความการแจ้งเตือน
5.  เมื่อต้องการปรับแต่งข้อความ คุณสามารถแทรกตัวยึด ตัวยึดจะถูกแทนที่ด้วยข้อมูลที่เหมาะสมเมื่อมีการแสดงข้อความแก่ผู้ใช้ เมื่อต้องการแทรกตัวยึด ให้ทำตามขั้นตอนเหล่านี้:
    1.  คลิกในฟิลด์เพื่อระบุตำแหน่งที่ต้องการให้ตัวยึดตำแหน่งปรากฏ
    2.  คลิก **แทรกตัวยึด**
    3.  ในรายการที่ปรากฏขึ้น ให้เลือกตัวยึดที่จะแทรก
    4.  คลิก **แทรก**

6.  เมื่อต้องการเพิ่มคำแปลของข้อความ ให้ทำตามขั้นตอนเหล่านี้:
    1.  คลิก **คำแปล**
    2.  บนหน้าที่ปรากฏ คลิก **เพิ่ม**
    3.  ในรายการที่ปรากฏ ให้เลือกภาษาที่คุณจะป้อนข้อความ
    4.  ในฟิลด์ **ข้อความที่แปล** ให้ป้อนข้อความ
    5.  เมื่อต้องการปรับแต่งข้อความ คุณสามารถแทรกตัวยึด สำหรับคำแนะนำเกี่ยวกับวิธีการป้อนตัวยึด ดูขั้นตอนที่ 5
    6.  คลิก **ปิด**

7.  บนแท็บ **ผู้รับ** ใช้ตัวเลือกต่อไปนี้เพื่อระบุบุคคลที่ควรได้รับการแจ้งเตือน

## <a name="enter-comments-about-the-changes-that-you-made-to-the-workflow"></a>การป้อนข้อคิดเห็นเกี่ยวกับการเปลี่ยนแปลงที่คุณทำกับลำดับงาน
หากต้องการป้อนข้อคิดเห็นเกี่ยวกับการเปลี่ยนแปลงที่คุณทำกับลำดับงานนี้ ให้ทำตามขั้นตอนเหล่านี้

1.  ในบานหน้าต่างทางซ้าย ให้คลิก **หมายเหตุ**
2.  ในฟิลด์ **ป้อนข้อคิดเห็นเกี่ยวกับลำดับงาน** ป้อนข้อคิดเห็นของคุณ
