# <a name="feature-management-overview"></a>ภาพรวมการจัดการคุณลักษณะ

[!include[banner](../../includes/banner.md)]
[!include [banner](../../includes/preview-banner.md)]

มีการเพิ่มและปรับปรุงคุณลักษณะในการนำออกใช้ทั้งหมดของ Microsoft Dynamics 365 for Finance and Operations ประสบการณ์การจัดการคุณลักษณะให้พื้นที่ทำงานที่คุณสามารถดูรายการของคุณลักษณะที่ได้ถูกจัดส่งในการนำออกใช้แต่ละครั้ง ตามค่าเริ่มต้น คุณลักษณะใหม่จะถูกปิด คุณสามารถใช้พื้นที่ทำงานเพื่อเปิดและดูเอกสารประกอบได้

## <a name="the-feature-management-workspace"></a>พื้นที่ทำงานการจัดการคุณลักษณะ

คุณสามารถเปิดพื้นที่ทำงาน **การจัดการคุณลักษณะ** ด้วยการเลือกไทล์ที่เหมาะสมบนแดชบอร์ด คุณจะเห็นหน้าที่แสดงรายการของคุณลักษณะสำหรับการนำออกใช้ทั้งหมดที่ได้รับการสนับสนุนโดยประสบการณ์การจัดการคุณลักษณะ เมื่อเวลาผ่านไป Microsoft จะปรับปรุงประสบการณ์การจัดการคุณลักษณะ เพื่อให้มีฟังก์ชันเพิ่มเติมเพื่อช่วยคุณในการจัดการคุณลักษณะ

รายการคุณลักษณะประกอบด้วยข้อมูลต่อไปนี้:

- **ชื่อคุณลักษณะ** – คำอธิบายของคุณลักษณะที่ถูกเพิ่ม
- **สถานะที่เปิดใช้งาน** – สัญลักษณ์บ่งชี้ว่ามีการเปิดใช้งานคุณลักษณะ (เครื่องหมายถูก) หรือไม่ถูกเปิดใช้งานอยู่ (ว่างเปล่า) ได้รับการจัดกำหนดการให้เปิดใช้งาน (นาฬิกา) หรือเป็นข้อบังคับที่เปิดใช้งาน (ล็อค) มีการใช้การตั้งค่าที่แสดงไว้ที่นี่สำหรับนิติบุคคลทั้งหมด โปรดทราบว่า แม้ว่าเมื่อคุณลักษณะถูกเปิดใช้งาน จะยังคงถูกควบคุมโดยความปลอดภัย ดังนั้น คุณลักษณะนี้จะพร้อมใช้งานเฉพาะสำหรับผู้ใช้ที่มีสิทธิ์เข้าถึง ตามบทบาทความปลอดภัยของพวกเขา นอกจากนี้ ยังพร้อมใช้งานสำหรับนิติบุคคลที่ผู้ใช้มีสิทธิ์เข้าถึงเท่านั้น
- **เปิดใช้งานวันที่** – วันที่ที่มีการเปิดใช้งานคุณลักษณะ หรือจะได้รับการเปิดใช้งาน ถ้าวันที่เป็นวันที่ในอนาคต
- **เพิ่มคุณลักษณะแล้ว** – วันที่ที่มีการเพิ่มคุณลักษณะไปยังสภาพแวดล้อมของคุณ วันที่นี้จะถูกป้อนโดยอัตโนมัติ เมื่อคุณปรับปรุงสภาพแวดล้อมของคุณในระหว่างรอบการนำออกใช้รายเดือน
- **โมดูล** – โมดูลที่ได้รับผลกระทบจากคุณลักษณะใหม่

เมื่อคุณเลือกคุณลักษณะ ข้อมูลเพิ่มเติมจะปรากฏขึ้นในบานหน้าต่างแสดงรายละเอียดทางด้านขวาของรายการคุณลักษณะ ที่ด้านบนของบานหน้าต่าง คุณจะเห็นชื่อคุณลักษณะ วันที่ที่มีการเพิ่มคุณลักษณะ โมดูลที่ได้รับผลกระทบจากคุณลักษณะ และลิงค์ **เรียนรู้เพิ่มเติม** เลือกลิงค์นี้เพื่อดูเอกสารประกอบสำหรับคุณลักษณะ ถ้าเอกสารไม่พร้อมใช้งาน คุณจะถูกนำไปยังหน้าชั่วคราว บานหน้าต่างรายละเอียดจะรวมฟิลด์ **ข้อคิดเห็น** ที่ซึ่งคุณสามารถเพิ่มข้อคิดเห็นของคุณเองเกี่ยวกับคุณลักษณะได้

พื้นที่ทำงาน **การจัดการคุณลักษณะ** ยังประกอบด้วยแท็บหลายแท็บที่มีรายการของคุณลักษณะในนั้น 
- **ใหม่** - แสดงคุณลักษณะทั้งหมดที่เพิ่มไว้นับตั้งแต่การปรับปรุงรายเดือนครั้งสุดท้าย ถ้าคุณได้ข้ามการปรับปรุงรายเดือนใดๆ จะมีคุณลักษณะใหม่ทั้งหมดตั้งแต่ครั้งล่าสุดที่คุณปรับปรุง คุณลักษณะใหม่ล่าสุดจะปรากฏขึ้นที่ด้านบนของรายการ นอกจากนี้ จำนวนรวมของคุณลักษณะใหม่จะแสดงอยู่ในไทล์ที่ด้านบนของหน้าด้วย
- **ไม่ได้เปิดใช้งาน** - แสดงคุณลักษณะทั้งหมดที่ยังไม่ได้เปิดใช้งาน คุณลักษณะใหม่ล่าสุดจะปรากฏขึ้นที่ด้านบนของรายการ นอกจากนี้ จำนวนรวมของคุณลักษณะใหม่จะแสดงอยู่ในไทล์ที่ด้านบนของหน้าด้วย
- **จัดกำหนดการ** - แสดงคุณลักษณะที่ได้ถูกจัดกำหนดการให้เปิดใช้งานในวันที่ในอนาคต คุณลักษณะที่มีวันที่ที่จัดกำหนดการที่เร็วที่สุด จะปรากฏขึ้นที่ด้านบนของรายการ นอกจากนี้ จำนวนรวมของคุณลักษณะใหม่จะแสดงอยู่ในไทล์ที่ด้านบนของหน้าด้วย
- **ทั้งหมด** - แสดงคุณลักษณะทั้งหมด คุณลักษณะใหม่ล่าสุดจะปรากฏขึ้นที่ด้านบนของรายการ


## <a name="enable-a-feature"></a>เปิดใช้งานคุณลักษณะ

ถ้าไม่ได้เปิดใช้งานคุณลักษณะ ปุ่ม **เปิดใช้งาน** จะปรากฏขึ้นในบานหน้าต่างแสดงรายละเอียด คุณสามารถใช้ปุ่มนี้เพื่อเปิดใช้งานคุณลักษณะ

1. เลือกคุณลักษณะที่คุณต้องการเปิดใช้งาน และจากนั้น ในบานหน้าต่างแสดงรายละเอียด เลือก **เปิดใช้งาน**
2. ตัวเลื่อนจะปรากฏขึ้น ที่ซึ่งคุณสามารถระบุวันที่ที่ควรเปิดใช้งานคุณลักษณะ โดยค่าเริ่มต้น วันที่นี้ถูกตั้งค่าเป็นวันที่ปัจจุบัน
3. เลือก **เปิดใช้งาน** เพื่อเปิดใช้งานคุณลักษณะ

ไม่สามารถปิดใช้งานคุณลักษณะบางอย่างได้ หลังจากที่คุณเปิดใช้งาน ถ้าคุณลักษณะที่คุณกำลังพยายามเปิดใช้งานไม่สามารถปิดใช้งานได้ คุณจะได้รับคำเตือน ณ จุดนี้ คุณสามารถเลือก **ยกเลิก** เพื่อยกเลิกการดำเนินงาน และปล่อยให้คุณลักษณะถูกปิดใช้งาน อย่างไรก็ตาม ถ้าคุณเลือก **เปิดใช้งาน** และเปิดใช้งานคุณลักษณะ คุณจะไม่สามารถปิดใช้งานได้ในภายหลัง

หลังจากเปิดใช้งานคุณลักษณะ ข้อความจะปรากฏขึ้นด้านล่างลิงค์ **เรียนรู้เพิ่มเติม** ในบานหน้าต่างแสดงรายละเอียด ข้อความนี้ระบุว่าคุณลักษณะถูกเปิดใช้งาน หรือบ่งชี้เวลาที่จะเปิดใช้งานคุณลักษณะในอนาคต อย่างใดอย่างหนึ่ง ถ้าคุณใช้วันที่ในอนาคต คุณลักษณะจะแสดงในรายการ **ที่จัดกำหนดการ** ข้อความนี้จะปรากฏขึ้นทุกครั้งที่คุณเลือกคุณลักษณะในรายการคุณลักษณะ คุณลักษณะที่มีการจัดกำหนดการในอนาคตจะเปิดใช้งานในเวลาเที่ยงคืนโดยกระบวนการชุดงาน ตามโซนเวลาที่แสดงอยู่ในวันที่ของระบบ 

## <a name="reschedule-a-feature"></a>จัดกำหนดการคุณลักษณะใหม่

ถ้ามีการเปิดใช้งานคุณลักษณะในอนาคต ปุ่ม **จัดกำหนดการใหม่** จะปรากฏขึ้นในบานหน้าต่างแสดงรายละเอียด คุณสามารถใช้ปุ่มนี้เพื่อเปลี่ยน **เปิดใช้งานวันที่** เป็นวันที่อื่นได้

1. เลือกคุณลักษณะที่จัดกำหนดการที่คุณต้องการจัดกำหนดการใหม่ และจากนั้น ในบานหน้าต่างแสดงรายละเอียด เลือก **จัดกำหนดการใหม่**
2. ตัวเลื่อนจะปรากฏขึ้น ที่ซึ่งคุณสามารถระบุวันที่ที่ควรเปิดใช้งานคุณลักษณะ 
3. เลือก **เปิดใช้งาน** เพื่อจัดกำหนดการคุณลักษณะ หรือ **ปิดใช้งาน** เพื่อยกเลิกกำหนดการ

## <a name="disable-a-feature"></a>ปิดใช้งานคุณลักษณะ

ถ้ามีการเปิดใช้งานคุณลักษณะอยู่แล้ว ปุ่ม **ปิดใช้งาน** จะปรากฏขึ้นในบานหน้าต่างแสดงรายละเอียด คุณสามารถใช้ปุ่มนี้เพื่อปิดใช้งานคุณลักษณะ ปุ่ม **ปิดใช้งาน** จะไม่พร้อมใช้งาน ถ้าไม่สามารถปิดใช้งานคุณลักษณะได้หลังจากที่เปิดใช้งาน

- เลือกคุณลักษณะที่คุณต้องการปิด และจากนั้น ในบานหน้าต่างแสดงรายละเอียด เลือก **ปิดใช้งาน**

- ปิดใช้งานคุณลักษณะ และไม่มีการล้างข้อมูลวันที่

หลังจากปิดใช้งานคุณลักษณะ ข้อความจะปรากฏขึ้นด้านล่างลิงค์ **เรียนรู้เพิ่มเติม** ในบานหน้าต่างแสดงรายละเอียด ข้อความนี้ระบุว่าคุณลักษณะยังไม่ได้ถูกเปิดใช้งาน และจะปรากฏในรายการ **ไม่ได้เปิดใช้งาน** ข้อความจะปรากฏขึ้นทุกครั้งที่คุณเลือกคุณลักษณะในรายการคุณลักษณะ

## <a name="features-that-must-be-enabled"></a>คุณลักษณะที่ต้องเปิดใช้งาน

คุณลักษณะที่สำคัญอาจได้รับการจัดส่งที่ต้องเปิดใช้งานโดยอัตโนมัติ เมื่อคุณทำการปรับปรุง จะมีการเปิดใช้งานโดยอัตโนมัติใน **เปิดใช้งานวันที่** ข้อความจะปรากฏขึ้นด้านล่างลิงค์ **เรียนรู้เพิ่มเติม** ในบานหน้าต่างแสดงรายละเอียด ข้อความนี้จะระบุว่าคุณลักษณะถูกเปิดใช้งาน หรือจะถูกเปิดใช้งานโดยอัตโนมัติใน **เปิดใช้งานวันที่** นี่จะปรากฏขึ้นทุกครั้งที่คุณเลือกคุณลักษณะในรายการคุณลักษณะ

## <a name="assigning-roles"></a>การกำหนดบทบาท

พื้นที่ทำงาน **การจัดการคุณลักษณะ** สามารถเปิดได้โดยผู้ดูแลระบบ และโดยผู้ใช้ที่ได้รับการกำหนดให้กับบทบาทตัวจัดการคุณลักษณะ หรือบทบาทตัวแสดงคุณลักษณะที่สร้างขึ้นเพื่อสนับสนุนประสบการณ์การจัดการคุณลักษณะ ผู้ใช้ในบทบาทผู้จัดการคุณลักษณะสามารถเปิดหรือปิดคุณลักษณะใดๆ ได้ นอกจากนี้ คุณยังสามารถปรับปรุงส่วนความคิดเห็นสำหรับคุณลักษณะได้ด้วย ผู้ใช้ในบทบาทของตัวแสดงคุณลักษณะสามารถดูได้เฉพาะพื้นที่ทำงาน **การจัดการคุณลักษณะ** ผู้ใช้ไม่สามารถเปิดหรือปิดคุณลักษณะได้

บทบาทของตัวจัดการคุณลักษณะ และบทบาทของตัวแสดงคุณลักษณะ ไม่ได้แทนที่ความปลอดภัยที่มีอยู่ซึ่งผู้ใช้มี บทบาทควบคุมการเข้าถึงคุณลักษณะที่เปิดใช้งานเท่านั้น จะไม่ให้การเข้าถึงไปยังคุณลักษณะเอง

## <a name="using-feature-management-to-enable-isv-features-or-custom-features"></a>การใช้การจัดการคุณลักษณะเพื่อเปิดใช้งานคุณลักษณะ ISV หรือคุณลักษณะที่กำหนดเอง

กระบวนการจัดการคุณลักษณะไม่พร้อมใช้งานในขณะนี้ สำหรับคุณลักษณะ ISV หรือคุณลักษณะที่กำหนดเอง เรากำลังเพิ่มฟังก์ชันการทำงานเพิ่มเติมเพื่อปรับปรุงการจัดการคุณลักษณะ และเมื่อการปรับปรุงดังกล่าวสมบูรณ์แล้ว เราจะเปิดใช้งานการจัดการคุณลักษณะสำหรับคุณลักษณะทั้งหมด และให้คำแนะนำเฉพาะเกี่ยวกับวิธีการปรับปรุงคุณลักษณะของคุณเพื่อใช้งาน

## <a name="feature-management-and-flighting"></a>การจัดการคุณลักษณะและการสลับ

การจัดการคุณลักษณะช่วยให้คุณสามารถควบคุมคุณลักษณะที่มีการจัดส่งในการนำออกใช้แต่ละครั้ง การสลับช่วยให้ Microsoft teams สามารถนำออกใช้คุณลักษณะไปยังลูกค้าจำนวนจำกัด เพื่อให้คุณลักษณะสามารถได้รับการทดสอบและได้รับการตรวจสอบความถูกต้องโดยไม่มีผลกระทบต่อลูกค้าทั้งหมด การจัดการคุณลักษณะไม่ได้ควบคุมการสลับของคุณลักษณะใดๆ
