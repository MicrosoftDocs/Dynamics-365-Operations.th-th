# <a name="change-the-banner-or-logo"></a><span data-ttu-id="ab152-103">เปลี่ยนป้ายประกาศหรือรูปภาพสัญลักษณ์</span><span class="sxs-lookup"><span data-stu-id="ab152-103">Change the banner or logo</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="ab152-104">กระบวนงานต่อไปนี้แสดงรายการขั้นตอนที่ผู้ดูแลระบบสามารถใช้เพื่ออัพเดตป้ายประกาศหรือรูปภาพสัญลักษณ์ที่ถูกแสดงสำหรับนิติบุคคล</span><span class="sxs-lookup"><span data-stu-id="ab152-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="ab152-105">ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF</span><span class="sxs-lookup"><span data-stu-id="ab152-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="ab152-106">ไปที่การจัดการองค์กร > องค์กร > นิติบุคคล</span><span class="sxs-lookup"><span data-stu-id="ab152-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="ab152-107">ในรายการทางด้านซ้าย ให้เลือกนิติบุคคลซึ่งคุณต้องการอัพเดตป้ายประกาศหรือโลโก้ </span><span class="sxs-lookup"><span data-stu-id="ab152-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="ab152-108">ถ้ามีการเลือกแล้ว ไปที่ขั้นตอนถัดไป</span><span class="sxs-lookup"><span data-stu-id="ab152-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="ab152-109">ขยายส่วนของรูปภาพ </span><span class="sxs-lookup"><span data-stu-id="ab152-109">Expand the Images section.</span></span> <span data-ttu-id="ab152-110">หากมีการขยายแล้ว ไปที่ขั้นตอนถัดไป</span><span class="sxs-lookup"><span data-stu-id="ab152-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="ab152-111">คลิกแก้ไข</span><span class="sxs-lookup"><span data-stu-id="ab152-111">Click Edit.</span></span>
5. <span data-ttu-id="ab152-112">ในฟิลด์ชนิดรูปภาพของบริษัทในแดชบอร์ด ให้เลือก 'ป้ายประกาศ'</span><span class="sxs-lookup"><span data-stu-id="ab152-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="ab152-113">คลิกเปลี่ยนแปลง</span><span class="sxs-lookup"><span data-stu-id="ab152-113">Click Change.</span></span>
    * <span data-ttu-id="ab152-114">คำแนะนำ: ความละเอียดที่เหมาะสมที่สุดสำหรับรูปป้ายประกาศคือ 1920 x 281 พิกเซล</span><span class="sxs-lookup"><span data-stu-id="ab152-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="ab152-115">ความกว้างที่เหมาะสมที่สุดสำหรับรูปภาพสัญลักษณ์คือ 350 พิกเซล</span><span class="sxs-lookup"><span data-stu-id="ab152-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="ab152-116">เมื่อการอัพโหลดเสร็จสมบูรณ์ คุณควรเห็นป้ายประกาศหรือรูปภาพสัญลักษณ์ใหม่ของคุณในส่วนรูปภาพในแดชบอร์ด</span><span class="sxs-lookup"><span data-stu-id="ab152-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="ab152-117">คลิก บันทึก</span><span class="sxs-lookup"><span data-stu-id="ab152-117">Click Save.</span></span>
8. <span data-ttu-id="ab152-118">ไปที่ ทั่วไป > ทั่วไป > แดชบอร์ดเริ่มต้น</span><span class="sxs-lookup"><span data-stu-id="ab152-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="ab152-119">คุณควรเห็นรูปภาพสัญลักษณ์หรือป้ายประกาศใหม่ในแดชบอร์ด</span><span class="sxs-lookup"><span data-stu-id="ab152-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="ab152-120">คุณอาจจำเป็นต้องเปลี่ยนบริษัทของคุณโดยใช้ตัวเลือกบริษัท ไปยังบริษัทที่ได้มีการอัพโหลดป้ายประกาศให้</span><span class="sxs-lookup"><span data-stu-id="ab152-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
