# <a name="set-up-compensation-grids"></a>ตั้งค่ากริดค่าตอบแทน

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

กริดค่าตอบแทนจะใช้ในการกำหนด และรักษาโครงสร้างค่าตอบแทนสำหรับแผนค่าตอบแทนที่คงที่  กริดค่าตอบแทนสามารถใช้ร่วมกันระหว่างหลายแผน หรือคัดลอกมาเมื่อสร้างแผนค่าตอบแทนใหม่   ก่อนที่จะสร้างกริดค่าตอบแทน ระดับ และจุดอ้างอิงต้องถูกตั้งค่าก่อน  ตัวอย่างนี้จะสร้างชนิดของระดับใหม่ของกริดค่าตอบแทนโดยใช้ข้อมูลสาธิตสำหรับระดับ และจุดอ้างอิง  ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF


## <a name="set-up-information-about-the-compensation-grid"></a>ตั้งค่าข้อมูลเกี่ยวกับกริดค่าตอบแทน
1. ไปที่ทรัพยากรบุคคล > ค่าตอบแทน > ค่าตอบแทนคงที่ > กริดค่าตอบแทน
2. คลิก สร้าง
3. ในฟิลด์กริด ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. ในฟิลด์ชนิด ให้เลือกหนึ่งตัวเลือก
6. ในฟิลด์การตั้งค่าการอ้างอิง ให้ป้อนหรือเลือกค่า
7. ในฟิลด์สกุลเงิน ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
8. ในฟิลด์สกุลเงิน ให้พิมพ์ค่า
9. ในฟิลด์วันที่มีผลบังคับใช้ ให้ป้อนวันที่

## <a name="add-levels-to-the-compensation-structure"></a>เพิ่มระดับในโครงสร้างค่าตอบแทน

2. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก

4. คลิก สร้าง
5. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
6. ในฟิลด์ระดับ ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
7. คลิก สร้าง
8. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
9. ในฟิลด์ระดับ ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
10. คลิก สร้าง
11. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
12. ในฟิลด์ระดับ ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
13. คลิก สร้าง
14. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
15. ในฟิลด์ระดับ ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง

## <a name="fill-in-the-compensation-structure-with-values"></a>กรอกข้อมูลในโครงสร้างค่าตอบแทนด้วยค่าต่างๆ
1. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
    * ในจำแหน่งนี้ มูลค่าค่าตอบแทนสามารถใส่ด้วยตนเองลงไปในแต่ละตารางได้ หรือสามารถใช้ฟังก์ชันการเปลี่ยนแปลงโดยรวมเพื่อกรอกข้อมูลในฟิลด์ต่างๆ ได้อย่างง่ายดาย และทำการคำนวณข้อมูลพื้นฐาน  
2. คลิกที่การเปลี่ยนแปลงโดยรวม
    * สำหรับกริดนี้ เราจะใช้ค่าสำหรับจุดกึ่งกลางระดับแรกของฟิลด์ทั้งหมดในตาราง  นี่จะเป็นข้อมูลพื้นฐานสำหรับเมทริกซ์ค่าตอบแทน  
3. ในฟิลด์ชนิดการปรับปรุง ให้เลือกตัวเลือกหนึ่งตัวเลือก
4. ในฟิลด์ยอดการปรับปรุง ให้ป้อนหมายเลข
5. ในรายการนี้ ทำเครื่องหมายหรือยกเลิกการทำเครื่องหมายทุกแถว
6. คลิกที่นำไปใช้กับกริด
    * ขณะนี้ เราจะใช้ฟังก์ชันการเปลี่ยนแปลงโดยรวมเพื่อเพิ่มยอดเงินในแต่ละระดับในลำดับต่อมาตามเปอร์เซ็นต์หรือยอดเงินที่แน่นอน  ในตัวอย่างนี้ แต่ละระดับจะมีการกระจาย 12.5% ระหว่างจ้างของตน  
7. ในฟิลด์ชนิดการปรับปรุง ให้เลือกตัวเลือกหนึ่งตัวเลือก
8. ในฟิลด์ยอดการปรับปรุง ให้ป้อนหมายเลข
9. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
10. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
11. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
12. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
13. คลิกที่นำไปใช้กับกริด
14. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
15. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
16. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
17. คลิกที่นำไปใช้กับกริด
18. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
19. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
20. คลิกที่นำไปใช้กับกริด
21. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
22. คลิกที่นำไปใช้กับกริด
    * ขณะนี้ เราจะใช้ฟังก์ชันเปลี่ยนแปลงโดยรวมเพื่อปรับปรุงจุดอ้างอิงค่าต่ำสุดและสูงสุดสำหรับแต่ละระดับ  ตัวอย่างนี้จะใช้การกระจาย 50% จุดอ้างอิงค่าต่ำสุดจะถูก ปรับปรุงเป็น -20% และจำนวนสูงสุดจะปรับปรุง +20%  
23. ในฟิลด์ยอดการปรับปรุง ให้ป้อนหมายเลข
24. ในฟิลด์จุดอ้างอิง ให้ป้อนหรือเลือกค่า
