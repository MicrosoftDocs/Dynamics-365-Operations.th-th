# <a name="assign-a-product-lifecycle-state-to-a-released-product-master"></a>มอบหมายสถานะรอบการขายของผลิตภัณฑ์ให้กับผลิตภัณฑ์หลักที่นำออกใช้

[!include [task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงวิธีการกำหนดสถานะรอบการขายของผลิตภัณฑ์ให้กับผลิตภัณฑ์หลักที่นำออกใช้และตัวแปร ข้อกำหนดเบื้องต้น: คุณต้องเล่นคู่มืองาน "สร้างสถานะรอบการขายของผลิตภัณฑ์ใหม่" ก่อน เพื่อให้แน่ใจว่า คุณมีสถานะรอบการขายของผลิตภัณฑ์อย่างน้อยหนึ่งสถานะที่สร้างขึ้นก่อนที่คุณจะสามารถเล่นคู่มืองานนี้ได้


## <a name="find-a-released-product-master"></a>ค้นหาผลิตภัณฑ์หลักที่นำออกใช้
1. ไปที่การจัดการข้อมูลผลิตภัณฑ์ > ผลิตภัณฑ์ > ผลิตภัณฑ์ที่นำออกใช้ 
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ

> [!NOTE]
> ผลิตภัณฑ์หลักมีผลิตภัณฑ์ที่เป็นชนิดย่อยของผลิตภัณฑ์หลัก  

## <a name="update-the-lifecycle-state"></a>อัพเดตสถานะรอบการขาย
1. คลิก แก้ไข
2. ในฟิลด์สถานะรอบการขายของผลิตภัณฑ์ ให้ป้อนหรือเลือกค่า
3. คลิก บันทึก
4. คลิก ใช่

> [!NOTE]
> ถ้ามีการเลือก ใช่ ผลิตภัณฑ์ย่อยที่นำออกใช้ซึ่งเกี่ยวข้องทั้งหมดที่มีสถานะเป็นต้นฉบับเดียวกันกับผลิตภัณฑ์หลักที่นำออกใช้ จะถูกปรับปรุงเป็นสถานะรอบการขายของผลิตภัณฑ์ใหม่อีกด้วย ถ้ามีการเลือก ไม่ ผลิตภัณฑ์ย่อยทั้งหมดจะรักษาสถานะที่แท้จริง ไม่มีการอัพเดตผลิตภัณฑ์ย่อยที่มีสถานะรอบการขายของผลิตภัณฑ์ที่แตกต่างจากข้อมูลหลักของผลิตภัณฑ์ที่นำออกใช้  

## <a name="verify-the-lifecycle-state-of-the-variants"></a>ตรวจสอบสถานะรอบการขายของผลิตภัณฑ์ย่อย
1. คลิก ผลิตภัณฑ์ย่อยที่นำออกใช้

> [!NOTE]
> โปรดทราบว่า ผลิตภัณฑ์ย่อยทั้งหมดมีการสืบทอดสถานะรอบการขายที่เลือกจากข้อมูลหลักของผลิตภัณฑ์ที่นำออกใช้  

2. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
3. ในฟิลด์สถานะรอบการขายของผลิตภัณฑ์ ให้ป้อนหรือเลือกค่า
