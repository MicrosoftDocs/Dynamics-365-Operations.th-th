# <a name="enter-the-gtin-code-for-a-product"></a>ป้อนรหัส GTIN สำหรับผลิตภัณฑ์

[!include [banner](../../includes/banner.md)]

รหัส GTIN จะถูกกำหนดโดยองค์กร GS1  ในตัวอย่างนี้ คุณจะป้อนรหัสสำหรับหมายเลขผลิตภัณฑ์ M0001 ในบริษัทข้อมูลสาธิต USMF

1. คลิก การบำรุงรักษาผลิตภัณฑ์ที่นำออกใช้
2. คลิกผลิตภัณฑ์ต่างๆที่นำออกใช้
3. ขยายกล่องแสดงข้อมูลย่อรูปภาพผลิตภัณฑ์
4. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
5. ในบานหน้าต่างการดำเนินการ คลิกจัดการสินค้าคงคลัง
6. คลิกรหัส GTIN
7. คลิก สร้าง
8. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
9. ในฟิลด์หน่วย ให้ป้อนหรือเลือกค่า
10. ในฟิลด์รหัส GTIN เลือกหนึ่งตัวเลือก
11. ในฟิลด์ GTIN ให้พิมพ์ค่า
12. ปิดหน้า
