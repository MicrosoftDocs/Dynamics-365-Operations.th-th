# <a name="create-a-hierarchy-of-product-classification"></a>สร้างลำดับขั้นของการจัดประภทผลิตภัณฑ์

[!include [task guide banner](../../includes/task-guide-banner.md)]

ขั้นตอนนี้แสดงวิธีการสร้างการจัดประเภทตามลำดับชั้นใหม่และกำหนดชนิดของลำดับชั้นรหัสสินค้า  ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF ขั้นตอนนี้มีไว้สำหรับผู้จัดการประเภท


## <a name="create-the-new-category-hierarchy"></a>สร้างการจัดประเภทตามลำดับชั้นใหม่
1. ไปที่การจัดการข้อมูลผลิตภัณฑ์ > การตั้งค่า > ประเภทและคุณลักษณะ > การจัดประเภทตามลำดับชั้น
2. คลิก สร้าง
3. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. คลิก สร้าง

## <a name="build-the-hierarchy"></a>สร้างลำดับชั้น
1. คลิกโหนดประเภทใหม่
2. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
3. ในฟิลด์รหัส ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ชื่อที่จำง่าย ให้พิมพ์ค่าใดค่าหนึ่ง
5. คลิกโหนดประเภทใหม่
6. ในฟิลด์ชื่อ ให้พิมพ์ค่า
7. ในฟิลด์รหัส ให้พิมพ์ค่าใดค่าหนึ่ง
8. ในฟิลด์ชื่อที่จำง่าย ให้พิมพ์ค่าใดค่าหนึ่ง
9. คลิกโหนดประเภทใหม่
10. ในฟิลด์ชื่อ ให้พิมพ์ค่า
11. ในฟิลด์รหัส ให้พิมพ์ค่าใดค่าหนึ่ง
12. ในฟิลด์ชื่อที่จำง่าย ให้พิมพ์ค่าใดค่าหนึ่ง
13. คลิกโหนดประเภทใหม่
14. ในฟิลด์ชื่อ ให้พิมพ์ค่า
15. ในฟิลด์รหัส ให้พิมพ์ค่าใดค่าหนึ่ง
16. ในฟิลด์ชื่อที่จำง่าย ให้พิมพ์ค่าใดค่าหนึ่ง
17. ปิดหน้า

## <a name="classify-the-hierarchy"></a>จัดประเภทลำดับชั้น
1. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
2. ในบานหน้าต่างการดำเนินการ คลิกการจัดประเภทตามลำดับชั้น
3. คลิกเชื่อมโยงชนิดลำดับชั้น
4. คลิก สร้าง
5. ในฟิลด์ชนิดของการจัดประเภทตามลำดับชั้น ให้เลือกหนึ่งตัวเลือก
    * เลือกชนิดของการจัดประเภทตามลำดับชั้นรหัสสินค้าที่ซื้อขายกันสำหรับการจัดประเภทผลิตภัณฑ์  
6. ในฟิลด์การจัดประเภทตามลำดับชั้น ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
7. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
8. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
9. ปิดหน้า
