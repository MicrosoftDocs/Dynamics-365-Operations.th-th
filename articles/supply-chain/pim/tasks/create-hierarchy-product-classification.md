# <a name="create-a-hierarchy-of-product-classification"></a>สร้างลำดับขั้นของการจัดประภทผลิตภัณฑ์

[!include [banner](../../includes/banner.md)]

ขั้นตอนนี้แสดงวิธีการสร้างการจัดประเภทตามลำดับชั้นใหม่และกำหนดชนิดของลำดับชั้นรหัสสินค้า  ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF ขั้นตอนนี้มีไว้สำหรับผู้จัดการประเภท


## <a name="create-the-new-category-hierarchy"></a>สร้างการจัดประเภทตามลำดับชั้นใหม่
1. ไปที่ **บานหน้าต่างนำทาง > โมดูล > การจัดการข้อมูลผลิตภัณฑ์ > การตั้งค่า > ประเภทและแอททริบิวต์ > การจัดประเภทตามลำดับชั้น**
2. คลิก **สร้าง**
3. ในฟิลด์ **ชื่อ** ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ **คำอธิบาย** ให้พิมพ์ค่าใดค่าหนึ่ง
5. คลิก **สร้าง**

## <a name="build-the-hierarchy"></a>สร้างลำดับชั้น
1. คลิกโหนดประเภท **ใหม่**
2. ในฟิลด์ **ชื่อ** ให้พิมพ์ค่าใดค่าหนึ่ง
3. ในฟิลด์ **รหัส** ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ **ชื่อที่จำง่าย** ให้พิมพ์ค่าใดค่าหนึ่ง
5. คลิกโหนดประเภท **ใหม่**
6. ในฟิลด์ **ชื่อ** ให้พิมพ์ค่าใดค่าหนึ่ง
7. ในฟิลด์ **รหัส** ให้พิมพ์ค่าใดค่าหนึ่ง
8. ในฟิลด์ **ชื่อที่จำง่าย** ให้พิมพ์ค่าใดค่าหนึ่ง
9. คลิกโหนดประเภท **ใหม่**
10. ในฟิลด์ **ชื่อ** ให้พิมพ์ค่าใดค่าหนึ่ง
11. ในฟิลด์ **รหัส** ให้พิมพ์ค่าใดค่าหนึ่ง
12. ในฟิลด์ **ชื่อที่จำง่าย** ให้พิมพ์ค่าใดค่าหนึ่ง
13. คลิกโหนดประเภท **ใหม่**
14. ในฟิลด์ **ชื่อ** ให้พิมพ์ค่าใดค่าหนึ่ง
15. ในฟิลด์ **รหัส** ให้พิมพ์ค่าใดค่าหนึ่ง
16. ในฟิลด์ **ชื่อที่จำง่าย** ให้พิมพ์ค่าใดค่าหนึ่ง
17. ปิดหน้า

## <a name="classify-the-hierarchy"></a>จัดประเภทลำดับชั้น
1. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
2. ใน **บานหน้าต่างการดำเนินการ** คลิก **การจัดประเภทตามลำดับชั้น**
3. คลิก **เชื่อมโยงชนิดลำดับชั้น**
4. คลิก **สร้าง**
5. ในฟิลด์ **ชนิดของการจัดประเภทตามลำดับชั้น** ให้เลือกหนึ่งตัวเลือก เลือก **ชนิดของการจัดประเภทตามลำดับชั้นรหัสโภคภัณฑ์สำหรับการจัดประเภทผลิตภัณฑ์โภคภัณฑ์**  
6. ในฟิลด์ **การจัดประเภทตามลำดับชั้น** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
7. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
8. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
9. ปิดหน้า
