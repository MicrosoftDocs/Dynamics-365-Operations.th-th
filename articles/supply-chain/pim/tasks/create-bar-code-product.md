[!include [banner](../../includes/banner.md)]

<span data-ttu-id="451b6-104">หัวข้อนี้แสดงวิธีการสร้างบาร์โค้ดด้วยตนเองโดยใช้หมายเลขสินค้า M0001 ดังตัวอย่าง</span><span class="sxs-lookup"><span data-stu-id="451b6-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="451b6-105">บริษัทข้อมูลสาธิตที่เคยสร้างกระบวนงานนี้คือ USMF</span><span class="sxs-lookup"><span data-stu-id="451b6-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="451b6-106">เลือก **การบำรุงรักษาผลิตภัณฑ์ที่นำออกใช้** บนหน้าแรก</span><span class="sxs-lookup"><span data-stu-id="451b6-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="451b6-107">ไปที่ **ผลิตภัณฑ์ > ผลิตภัณฑ์ที่นำออกใช้** ภายใต้ส่วน **การเชื่อมโยง**</span><span class="sxs-lookup"><span data-stu-id="451b6-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="451b6-108">ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ</span><span class="sxs-lookup"><span data-stu-id="451b6-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="451b6-109">สำหรับตัวอย่างนี้ ให้เลือก หมายเลขสินค้า **M0001**</span><span class="sxs-lookup"><span data-stu-id="451b6-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="451b6-110">ในบานหน้าต่างการดำเนินการ เลือก **จัดการสินค้าคงคลัง**</span><span class="sxs-lookup"><span data-stu-id="451b6-110">On the Action Pane, select **Manage inventory**.</span></span>

6. <span data-ttu-id="451b6-112">เลือก **ใหม่**</span><span class="sxs-lookup"><span data-stu-id="451b6-112">Select **New**.</span></span>
7. <span data-ttu-id="451b6-113">ทำเครื่องหมายแถวที่เลือกที่สร้างไว้ในรายการด้านล่าง</span><span class="sxs-lookup"><span data-stu-id="451b6-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="451b6-114">ในฟิลด์ **การตั้งค่าบาร์โค้ด** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง</span><span class="sxs-lookup"><span data-stu-id="451b6-114">In the **Barcode setup** field, enter or select a value.</span></span>

12. <span data-ttu-id="451b6-118">เลือก **บันทึก**</span><span class="sxs-lookup"><span data-stu-id="451b6-118">Select **Save**.</span></span>
13. <span data-ttu-id="451b6-119">ปิดหน้า</span><span class="sxs-lookup"><span data-stu-id="451b6-119">Close the page.</span></span>
