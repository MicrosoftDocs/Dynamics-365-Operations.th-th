# <a name="create-a-bar-code-for-a-product"></a>สร้างบาร์โค้ดให้ผลิตภัณฑ์

[!include [banner](../../includes/banner.md)]

หัวข้อนี้แสดงวิธีการสร้างบาร์โค้ดด้วยตนเองโดยใช้หมายเลขสินค้า M0001 ดังตัวอย่าง บริษัทข้อมูลสาธิตที่เคยสร้างกระบวนงานนี้คือ USMF

1. เลือก **การบำรุงรักษาผลิตภัณฑ์ที่นำออกใช้** บนหน้าแรก
2. ไปที่ **ผลิตภัณฑ์ > ผลิตภัณฑ์ที่นำออกใช้** ภายใต้ส่วน **การเชื่อมโยง**
3. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ สำหรับตัวอย่างนี้ ให้เลือก หมายเลขสินค้า **M0001**
4. ในบานหน้าต่างการดำเนินการ เลือก **จัดการสินค้าคงคลัง**
5. เลือก **บาร์โค้ด**
6. เลือก **ใหม่**
7. ทำเครื่องหมายแถวที่เลือกที่สร้างไว้ในรายการด้านล่าง
8. ในฟิลด์ **การตั้งค่าบาร์โค้ด** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
9. ในฟิลด์ **บาร์โค้ด** ให้ป้อนหรือเลือกค่า
10. ในฟิลด์ **บาร์โค้ด** ให้พิมพ์ค่า  
11. ในฟิลด์ **ปริมาณ** ให้ป้อนตัวเลข
12. เลือก **บันทึก**
13. ปิดหน้า
