# <a name="create-a-default-product-lifecycle-state"></a>สร้างสถานะรอบการขายของผลิตภัณฑ์ค่าเริ่มต้น

[!include [banner](../../includes/banner.md)]

กระบวนงานนี้แสดงวิธีการสร้างสถานะวงจรชีวิตผลิตภัณฑ์เริ่มต้น รวมทั้งวิธีการเชื่อมโยงสถานะเริ่มต้นกับผลิตภัณฑ์ที่นำออกใช้ด้วย


## <a name="create-a-default-lifecycle-state"></a>สร้างสถานะรอบการขายค่าเริ่มต้น
1. ไปที่การจัดการข้อมูลผลิตภัณฑ์ > การตั้งค่า > สถานะรอบการขายของผลิตภัณฑ์
2. คลิก สร้าง
3. ในฟิลด์สถานะ ให้พิมพ์ค่าใดค่าหนึ่ง
4. เลือก ใช่ ในค่าเริ่มต้น เมื่อนำออกใช้กับฟิลด์นิติบุคคล
5. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
6. เลือก ไม่ ในฟิลด์ ใช้งานอยู่สำหรับการวางแผน

> [!NOTE]
> ถ้าไม่ควรรวมผลิตภัณฑ์ที่นำออกใช้ใหม่ในการวางแผนหลัก เลือก ไม่ ถ้าควรรวมอยู่ในการวางแผนหลัก ปล่อยตัวควบคุมที่ค่าเริ่มต้น ใช่  

## <a name="create-a-new-released-product"></a>สร้างผลิตภัณฑ์ที่นำออกใช้ใหม่
1. ปิดหน้า
2. ไปที่การจัดการข้อมูลผลิตภัณฑ์ > ผลิตภัณฑ์ > ผลิตภัณฑ์ที่นำออกใช้ 
3. คลิก สร้าง
4. ในฟิลด์หมายเลขผลิตภัณฑ์ ให้พิมพ์ค่า
5. ในฟิลด์ชื่อผลิตภัณฑ์ ให้พิมพ์ค่า
6. ในฟิลด์ชื่อสำหรับค้นหา ให้พิมพ์ค่าใดค่าหนึ่ง
7. ในฟิลด์กลุ่มแบบจำลอง ให้ป้อนหรือเลือกค่า
8. ในฟิลด์กลุ่มสินค้า ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
9. ในฟิลด์กลุ่มแบบจำลองมิติการจัดเก็บ ให้ป้อนหรือเลือกค่า
10. ในฟิลด์กลุ่มมิติการติดตาม ให้ป้อนหรือเลือกค่า
11. คลิก ตกลง

> [!NOTE]
> สถานะวงจรชีวิตของผลิตภัณฑ์เริ่มต้นคือ คำนิยามสากล  

## <a name="change-the-product-to-an-active-state"></a>เปลี่ยนผลิตภัณฑ์เป็นสถานะที่ใช้งานอยู่
1. ในฟิลด์สถานะรอบการขายของผลิตภัณฑ์ ให้ป้อนหรือเลือกค่า

> [!NOTE]
> สมมติว่า คุณได้ตั้งค่าสถานะที่ใช้งานอยู่ ขณะนี้คุณสามารถเลือกสถานะที่ใช้งานอยู่เพื่ออนุญาตให้มีการใช้ผลิตภัณฑ์ในการคำนวณระดับ BOM และการวางแผนหลัก เห็นได้ชัดว่า ควรใช้รายการนี้ก็ต่อเมื่อรายละเอียดผลิตภัณฑ์ทั้งหมดที่จำเป็นสำหรับการวางแผนถูกระบุเท่านั้น
