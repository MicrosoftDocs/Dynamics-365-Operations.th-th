# <a name="create-a-new-product-lifecycle-state"></a>สร้างสถานะรอบการขายของผลิตภัณฑ์ใหม่

[!include [task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้จะแสดงวิธีการสร้างสถานะรอบการขายของผลิตภัณฑ์ใหม่ เอกสารสถานะรอบการขายของผลิตภัณฑ์ สถานะรอบการขายของผลิตภัณฑ์หรือผลิตภัณฑ์ย่อยที่นำออกใช้ สถานะรอบการขายของผลิตภัณฑ์ถูกกำหนดโดยผู้ใช้ โดยทั่วไป ผู้จัดการผลิตภัณฑ์หรือผู้จัดการข้อมูลหลักของผลิตภัณฑ์ กระบวนการทางธุรกิจเฉพาะ ตัวอย่างเช่น การวางแผนหลัก สามารถได้รับผลกระทบโดยสถานะรอบการขายเฉพาะ


## <a name="create-a-product-lifecycle-state"></a>สร้างสถานะรอบการขายของผลิตภัณฑ์
1. ไปที่การจัดการข้อมูลผลิตภัณฑ์ > การตั้งค่า > สถานะรอบการขายของผลิตภัณฑ์
2. คลิก สร้าง
3. ในฟิลด์สถานะ ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
