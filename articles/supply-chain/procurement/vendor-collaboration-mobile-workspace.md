## <a name="overview"></a><span data-ttu-id="a7761-110">ภาพรวม</span><span class="sxs-lookup"><span data-stu-id="a7761-110">Overview</span></span> 
<span data-ttu-id="a7761-111">พื้นที่ทำงานแบบเคลื่อนที่ของ **การทำงานร่วมกันกับผู้จัดจำหน่าย** ช่วยให้ผู้จัดจำหน่ายได้ทราบเกี่ยวกับใบสั่งซื้อใหม่ เพื่อให้พวกเขาสามารถดูใบสั่งซื้อได้ แล้วตอบสนองต่อพวกเขาใน Microsoft Dynamics 365 สำหรับการเงินและการดำเนินงาน รุ่น Enterprise เว็บไคลเอนต์</span><span class="sxs-lookup"><span data-stu-id="a7761-111">The **Vendor collaboration** mobile workspace keeps vendors informed about new purchase orders, so that they can view purchase orders and then respond to them in the Microsoft Dynamics 365 for Finance and Operations, Enterprise edition, web client.</span></span> 

>[!NOTE]
> <span data-ttu-id="a7761-112">หมายเหตุ: ควรใช้พื้นที่ทำงานแบบเคลื่อนที่เป็นส่วนเพิ่มเติมของเว็บไคลเอนต์การทำงานร่วมกันของผู้จัดจำหน่าย แต่ไม่ได้แทนที่</span><span class="sxs-lookup"><span data-stu-id="a7761-112">The mobile workspace should be used as a supplement to the vendor collaboration web interface, not a replacement for it.</span></span> 

<span data-ttu-id="a7761-113">ผู้จัดจำหน่ายของคุณสามารถใช้พื้นที่ทำงานแบบเคลื่อนที่ของ **การทำงานร่วมกันกับผู้จัดจำหน่าย** ได้ เพื่อดูใบสั่งซื้อใหม่ที่ถูกส่งไปเพื่อขออนุมัติ</span><span class="sxs-lookup"><span data-stu-id="a7761-113">Your vendors can use the **Vendor collaboration** mobile workspace to view new purchase orders that are sent to them for approval.</span></span> <span data-ttu-id="a7761-114">ซึ่งจะแสดงข้อมูลใบสั่งซื้อ เช่น ผลิตภัณฑ์ ปริมาณ และวันที่จัดส่งที่ร้องขอ</span><span class="sxs-lookup"><span data-stu-id="a7761-114">It shows purchase order information, such as products, quantities, and requested delivery dates.</span></span> <span data-ttu-id="a7761-115">นอกจากนี้ยังมีข้อมูลราคาด้วย โดยขึ้นอยู่กับการตั้งค่าคอนฟิกของผู้จัดจำหน่ายแต่ละราย</span><span class="sxs-lookup"><span data-stu-id="a7761-115">Price information is also available, depending on the configuration of each vendor.</span></span> 

<span data-ttu-id="a7761-116">ผู้ใช้ที่ลงชื่อเข้าใช้เป็นผู้จัดจำหน่าย จะเห็นว่าใบสั่งซื้อใดได้รับการตอบรับ และใบสั่งซื้อใดยังคงรอการดำเนินการของลูกค้า</span><span class="sxs-lookup"><span data-stu-id="a7761-116">A user who signs in as a vendor will see which purchase orders have been responded to, and which purchase orders are still awaiting customer action.</span></span> <span data-ttu-id="a7761-117">ตัวอย่างเช่น ใบสั่งซื้ออาจกำลังรอการดำเนินการของลูกค้า เพราะผู้จัดจำหน่ายแนะนำวันที่จัดส่งอื่น แต่ลูกค้ายังไม่ได้ยอมรับวันที่นั้น</span><span class="sxs-lookup"><span data-stu-id="a7761-117">For example, a purchase order might be awaiting customer action because the vendor suggested another delivery date, but the customer hasn't yet agreed to that date.</span></span> <span data-ttu-id="a7761-118">ผู้จัดจำหน่ายจะยังเห็นรายการของใบสั่งซื้อที่ได้ยืนยันแล้วแต่ยังไม่ได้ถูกจัดส่งอีกด้วย</span><span class="sxs-lookup"><span data-stu-id="a7761-118">The vendor will also see a list of purchase orders that have been confirmed but haven't yet been delivered.</span></span> 

<span data-ttu-id="a7761-119">เมื่อต้องการตอบสนองต่อใบสั่งซื้อ ผู้จัดจำหน่ายต้องใช้เว็บอินเทอร์เฟสการทำงานร่วมกันกับผู้จัดจำหน่ายที่พร้อมใช้งานในเว็บไคลเอนต์</span><span class="sxs-lookup"><span data-stu-id="a7761-119">To respond to a purchase order, the vendor must use the vendor collaboration web interface that is available in the web client.</span></span> <span data-ttu-id="a7761-120">ที่นั่น ผู้จัดจำหน่ายยังสามารถได้รับข้อมูลเพิ่มเติมเกี่ยวกับใบสั่ง เช่น สิ่งที่แนบของเอกสาร ที่อยู่จัดส่งต่อรายการ และค่าธรรมเนียมที่เกี่ยวข้องกับผู้จัดจำหน่ายอีกด้วย</span><span class="sxs-lookup"><span data-stu-id="a7761-120">There, the vendor can also get more information about the order, such as document attachments, the delivery address per line, and charges that are associated with the vendor.</span></span>

-   <span data-ttu-id="a7761-126">ดูใบสั่งซื้อที่ผู้จัดจำหน่ายได้ตอบสนอง และที่กำลังรอการดำเนินการของลูกค้าอยู่</span><span class="sxs-lookup"><span data-stu-id="a7761-126">View purchase orders that the vendor has responded to, and that are awaiting customer action.</span></span>
-   <span data-ttu-id="a7761-127">ดูใบสั่งซื้อที่ได้รับการยืนยันแล้ว แต่ยังไม่ได้รับทั้งหมด</span><span class="sxs-lookup"><span data-stu-id="a7761-127">View purchase orders that have been confirmed but haven't yet been fully received.</span></span>

<span data-ttu-id="a7761-203">หน้า **กำลังรอการดำเนินการของลูกค้า** ช่วยให้คุณสามารถค้นหาใบสั่งซื้อที่คุณหรือบุคคลอื่นในบริษัทของคุณ ที่มีการเข้าถึงไปยังการทำงานร่วมกันกับผู้จัดจำหน่ายได้ตอบสนองไป</span><span class="sxs-lookup"><span data-stu-id="a7761-203">The **Awaiting customer action** page lets you find purchase orders that you or another person in your company who has access to vendor collaboration has responded to.</span></span> <span data-ttu-id="a7761-204">ใบสั่งซื้อจะปรากฏในรายการนี้เท่านั้น ถ้าลูกค้าต้องการดำเนินการหนึ่งในการดำเนินการต่อไปนี้ในใบสั่งซื้อ:</span><span class="sxs-lookup"><span data-stu-id="a7761-204">The purchase orders are visible in this list only if the customer must take one of the following actions on the purchase order:</span></span>

<span data-ttu-id="a7761-221">โปรดทราบว่า ฟิลด์ **การจัดส่ง** ถูกตั้งค่าเป็น **ใช่** หรือ **ไม่ใช่** เพื่อบ่งชี้ว่ารายการจะถูกจัดส่งหรือไม่</span><span class="sxs-lookup"><span data-stu-id="a7761-221">Note that the **Delivering** field is set to either **Yes** or **No** to indicate whether the lines will be delivered.</span></span> <span data-ttu-id="a7761-222">รายการอาจไม่สามารถถูกจัดส่งได้เนื่องจากสาเหตุต่อไปนี้:</span><span class="sxs-lookup"><span data-stu-id="a7761-222">A line might not be delivered because for the following reasons:</span></span>

- <span data-ttu-id="a7761-224">มีการทำการแทนที่ และรายการต้นฉบับไม่ได้ถูกคาดหวังว่าจะถูกจัดส่งตามที่ร้องขอในใบสั่งที่ได้รับ</span><span class="sxs-lookup"><span data-stu-id="a7761-224">A substitution was made, and the original line isn't expected to be delivered as requested in the received order.</span></span>

<span data-ttu-id="a7761-226">การเปลี่ยนแปลงใด ๆ ที่ทำไว้กับการตอบสนองรายการใบสั่งจะแสดงขึ้น</span><span class="sxs-lookup"><span data-stu-id="a7761-226">Any changes that have been made to the order line response are shown.</span></span> <span data-ttu-id="a7761-227">อย่างไรก็ตาม บันทึกย่อที่อัพโหลดและสิ่งที่แนบมาจะไม่แสดงขึ้น</span><span class="sxs-lookup"><span data-stu-id="a7761-227">However, uploaded notes and attachments aren't shown.</span></span> <span data-ttu-id="a7761-228">เพื่อดูบันทึกย่อและสิ่งที่แนบ คุณต้องใช้เว็บอินเทอร์เฟสการทำงานร่วมกันกับผู้จัดจำหน่ายในเว็บไคลเอนต์</span><span class="sxs-lookup"><span data-stu-id="a7761-228">To view notes and attachments, you must use the vendor collaboration web interface in the web client.</span></span>
