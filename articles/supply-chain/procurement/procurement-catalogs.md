<span data-ttu-id="2e14a-105">บทความนี้อธิบายถึง วิธีที่ผู้เชี่ยวชาญด้านการจัดซื้อสามารถตั้งค่าและรักษาแค็ตตาล็อกการจัดซื้อในระดับสูง</span><span class="sxs-lookup"><span data-stu-id="2e14a-105">This article describes, at a high level, how purchasing professionals can set up and maintain procurement catalogs.</span></span> <span data-ttu-id="2e14a-106">แค็ตตาล็อกการจัดซื้อกำหนดสินค้าและบริการที่พนักงานบริษัทสามารถสั่งสำหรับใช้ภายใน</span><span class="sxs-lookup"><span data-stu-id="2e14a-106">Procurement catalogs define the items and services that company employees can order for internal use.</span></span>

<span data-ttu-id="2e14a-107">ผู้เชี่ยวชาญด้านการจัดซื้อสามารถสร้างและรักษาแค็ตตาล็อกของสินค้าและบริการที่สามารถซื้อสำหรับใช้ภายในองค์กร</span><span class="sxs-lookup"><span data-stu-id="2e14a-107">Purchasing professionals can create and maintain catalogs of the items and services that can be purchased for internal use in an organization.</span></span> <span data-ttu-id="2e14a-108">หลังจากตั้งค่าแค็ตตาล็อก พนักงานสามารถสร้างใบขอซื้อเพื่อสั่งจากแค็ตตาล็อก</span><span class="sxs-lookup"><span data-stu-id="2e14a-108">After catalogs are set up, company employees can create purchase requisitions to order from them.</span></span> <span data-ttu-id="2e14a-109">แค็ตตาล็อกสามารถใช้เพื่อบังคับใช้นโยบายการจัดซื้อ เพื่อให้พนักงานสามารถสั่งสินค้าและบริการที่ได้รับอนุญาตสำหรับนิติบุคคลที่จัดซื้อ</span><span class="sxs-lookup"><span data-stu-id="2e14a-109">The catalogs can be used to enforce purchasing policies, so that employees can order only the items and services that are allowed for their buying legal entity.</span></span> <span data-ttu-id="2e14a-110">เมื่อคุณสร้างแค็ตตาล็อกการจัดซื้อ คุณควรพิจารณาทำงานต่อไปนี้:</span><span class="sxs-lookup"><span data-stu-id="2e14a-110">When you create a procurement catalog, you should consider the following tasks:</span></span>

-   <span data-ttu-id="2e14a-111">กำหนดค่าลำดับชั้นประเภทการจัดซื้อของคุณก่อนที่คุณจะสร้างแค็ตตาล็อก</span><span class="sxs-lookup"><span data-stu-id="2e14a-111">Configure your procurement category hierarchy before you create the catalog.</span></span>
-   <span data-ttu-id="2e14a-112">กำหนดผลิตภัณฑ์ที่คุณต้องการให้พนักงานของคุณสามารถสั่งได้ </span><span class="sxs-lookup"><span data-stu-id="2e14a-112">Determine which products you want your employees to be able to order.</span></span> <span data-ttu-id="2e14a-113">คุณสามารถแสดงหรือซ่อนผลิตภัณฑ์เฉพาะในแค็ตตาล็อ หรือคุณสามารถแสดงหรือซ่อนผลิตภัณฑ์ทั้งหมดในโหนด</span><span class="sxs-lookup"><span data-stu-id="2e14a-113">You can show or hide specific products in a catalog node, or you can show or hide all the products in a node.</span></span>

<span data-ttu-id="2e14a-116">ปัจจัยหลายประการกำหนดผลิตภัณฑ์ที่พนักงานสามารถสั่งและประเภทการจัดซื้อที่พวกเขาสามารถใช้เมื่อสร้างใบขอซื้อ:</span><span class="sxs-lookup"><span data-stu-id="2e14a-116">Several factors determine the products that employees can order and the procurement categories that they can use when they create purchase requisitions:</span></span>

-   <span data-ttu-id="2e14a-119">พนักงานสามารถสั่งผลิตภัณฑ์ได้ถ้าอยู่ในแค็ตตาล็อกการจัดซื้อที่ใช้งานอยู่สำหรับองค์กรของคุณเท่านั้น และถ้ามีการเปิดใช้งานโหนดและไม่ได้ซ่อนไว้</span><span class="sxs-lookup"><span data-stu-id="2e14a-119">Employees can order a product only if it's in the active procurement catalog for your organization, and if it's in an enabled node and not hidden.</span></span> <span data-ttu-id="2e14a-120">ผลิตภัณฑ์ต้องอยู่ในประเภทที่พนักงานแต่ละคนมีสิทธิ์เข้าถึงตามกฎนโยบายการเข้าถึงประเภท</span><span class="sxs-lookup"><span data-stu-id="2e14a-120">The product must also be in a category that a particular employee has access to according to the category access policy rule.</span></span>
-   <span data-ttu-id="2e14a-121">กฎนโยบายแค็ตตาล็อกระบุในแค็ตตาล็อกที่ใช้</span><span class="sxs-lookup"><span data-stu-id="2e14a-121">The catalog policy rule specifies the catalog that is used.</span></span> <span data-ttu-id="2e14a-122">ถ้าไม่มีกฎนโยบายแค็ตตาล็อกถูกกำหนดไว้ กฎการเข้าถึงประเภทเพียงอย่างเดียวจะกำหนดผลิตภัณฑ์ที่พนักงานสามารถสั่งบนใบสั่งได้</span><span class="sxs-lookup"><span data-stu-id="2e14a-122">If no catalog policy rule is defined, the category access rule alone determines the products that an employee can order on the requisition.</span></span>
