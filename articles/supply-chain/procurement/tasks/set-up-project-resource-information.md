# <a name="set-up-project-resource-information"></a>ตั้งค่าข้อมูลทรัพยากรโครงการ

[!include [task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงให้เห็นถึงวิธีการตั้งค่าทรัพยากรโครงการ งานนี้ใช้ชุดข้อมูล USMF

1. ไปที่ การจัดการและการบัญชีโครงการ > ทรัพยากรโครงการ > รายการทรัพยากร
2. คลิก ทรัพยากรใหม่
3. ในฟิลด์ผู้ปฏิบัติงาน ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
4. ในฟิลด์ปฏิทิน ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
5. ในฟิลด์รหัสรอบระยะเวลา ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
6. คลิก สร้าง
7. คลิก ราคาต้นทุน
8. คลิก ราคาต้นทุน - ชั่วโมง
9. คลิก สร้าง
10. ในฟิลด์วันที่มีผลบังคับใช้ ให้ป้อนวันที่
11. ในฟิลด์ราคาต้นทุน ให้ป้อนหมายเลข
12. คลิก บันทึก
13. ปิดหน้า
14. คลิก ราคาขาย
15. คลิก ราคาขาย - ชั่วโมง
16. คลิก สร้าง
17. ในฟิลด์วันที่มีผลบังคับใช้ ให้ป้อนวันที่
18. ในฟิลด์การกำหนดราคา ให้ป้อนตัวเลข
19. ในฟิลด์ มีผลบังคับใช้สำหรับ ให้เลือกหนึ่งตัวเลือก
20. คลิก บันทึก
21. ปิดหน้า
