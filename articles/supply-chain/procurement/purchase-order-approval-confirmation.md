<span data-ttu-id="d4e62-157">หลังจากที่ผู้จัดจำหน่ายยอมรับใบสั่ง ขั้นตอนต่อไปคือการบันทึก PO เป็น กำหนดให้แล้ว</span><span class="sxs-lookup"><span data-stu-id="d4e62-157">After the vendor has agreed to the order, the next step is to record the PO as committed.</span></span> <span data-ttu-id="d4e62-158">คุณสามารถทำขั้นตอนนี้ให้เสร็จสิ้นได้โดยใช้การดำเนินการ **การยืนยัน** หรือการดำเนินการ **ยืนยัน**</span><span class="sxs-lookup"><span data-stu-id="d4e62-158">You can complete this step by using either the **Confirmation** action or the **Confirm** action.</span></span> <span data-ttu-id="d4e62-159">การดำเนินการทั้งสองจะตั้งค่าสถานะการอนุมัติของใบสั่งเป็น **ยืนยันแล้ว**</span><span class="sxs-lookup"><span data-stu-id="d4e62-159">Both these actions set the approval status of the order to **Confirmed**.</span></span> <span data-ttu-id="d4e62-160">การยืนยันใบสั่งเริ่มต้นจากขั้นตอนเพิ่มเติมสองรายการ:</span><span class="sxs-lookup"><span data-stu-id="d4e62-160">Confirmation of an order initiates two additional processes:</span></span>

## <a name="changing-purchase-orders"></a><span data-ttu-id="d4e62-169">การเปลี่ยนแปลงใบสั่งซื้อ</span><span class="sxs-lookup"><span data-stu-id="d4e62-169">Changing purchase orders</span></span>

<span data-ttu-id="d4e62-175">ถ้าปริมาณที่สั่งของรายการ PO ส่วนหนี่งถูกส่ง คุณจะไม่สามารถเปลี่ยนปริมาณที่สั่งได้ เมื่อใบสั่งซื้ออยู่ใน **ร่าง**</span><span class="sxs-lookup"><span data-stu-id="d4e62-175">If part of the ordered quantity for a PO line has been delivered, you can't change the ordered quantity when the purchase order is in **Draft**.</span></span> <span data-ttu-id="d4e62-176">อย่างไรก็ตาม คุณสามารถเปลี่ยนแปลงปริมาณ **ยอดค้างส่ง** ในรายการสำหรับใบสั่งซื้อที่อยู่ในสถานะ **ร่าง**</span><span class="sxs-lookup"><span data-stu-id="d4e62-176">However, you can change the **Deliver remainder** quantity on the line for the purchase order that is in **Draft** status.</span></span>
