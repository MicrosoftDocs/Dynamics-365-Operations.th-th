# <a name="set-up-a-carrier-fuel-index"></a>ตั้งค่าดัชนีเชื้อเพลิงของผู้ขนส่ง

[!include [banner](../../includes/banner.md)]

คำแนะนำนี้แสดงวิธีการสร้างดัชนีเชื้อเพลิงในภูมิภาค, ดัชนีเชื้อเพลิงและดัชนีเชื้อเพลิงของผู้ขนส่ง  ดัชนีเชื้อเพลิงในภูมิภาคระบุว่าภูมิภาคใดควรใช้ดัชนีเชื้อเพลิงแบบใด และดัชนีเชื้อเพลิงระบุราคาในรอบระยะเวลานั้นๆ เพื่อสะท้อนการเปลี่ยนแปลงในราคาของเชื้อเพลิงตลอดช่วงเวลา คุณสามารถเชื่อมโยงหลายๆดัชนีเชื้อเพลิงกับผู้ขนส่ง  งานเหล่านี้โดยทั่วไปถูกทำโดยผู้ประสานงานขนส่ง คุณสามารถใช้ขั้นตอนนี้ในข้อมูลสาธิตของบริษัทUSMF หรือโดยใช้ข้อมูลของคุณเอง


## <a name="create-a-fuel-index-region"></a>สร้างดัชนีเชื้อเพลิงของภูมิภาค
1. ไปที่จัดการการขนส่ง > การตั้งค่า > ดัชนีเชื้อเพลิง > ดัชนีเชื้อเพลิงของภูมิภาค
    * ก่อนอื่น คุณต้องสร้างภูมิภาคที่แตกต่างกัน สถานที่ที่ดำเนินธุรกิจ และคำนวณค่าเชื้อเพลิงเพิ่มเติมต่างๆ  
2. คลิก สร้าง
3. ในฟิลด์ภูมิภาค ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
5. คลิก บันทึก

## <a name="create-a-fuel-index"></a>สร้างดัชนีเชื้อเพลิง
1. ไปที่จัดการการขนส่ง > การตั้งค่า > ดัชนีเชื้อเพลิง > ดัชนีเชื้อเพลิง
    * สำหรับภูมิภาคที่คุณได้ตั้งค่า คุณต้องป้อนราคาเชื้อเพลิงปัจจุบัน  
2. คลิก สร้าง
3. ในฟิลด์ภูมิภาค ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
5. ในฟิลด์ราคาต่อแกลลอน ให้ป้อนตัวเลข
6. ในฟิลด์วันและเวลาเริ่มต้น ให้ป้อนวันที่และเวลา
7. คลิก บันทึก

## <a name="create-a-carrier-fuel-index"></a>สร้างดัชนีเชื้อเพลิงของผู้ขนส่ง
1. ไปที่จัดการการขนส่ง > การตั้งค่า > ดัชนีเชื้อเพลิง > ดัชนีเชื้อเพลิงของผู้ขนส่ง
2. คลิก สร้าง
3. ในฟิลด์ดัชนีเชื้อเพลิงของผู้ขนส่ง ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. คลิก สร้าง
6. ในฟิลด์วันและเวลาเริ่มต้น ให้ป้อนวันที่และเวลา
7. ในฟิลด์PPGต้นทาง ให้ป้อนตัวเลข
    * ในตัวอย่างนี้ คุณสามารถตั้งค่า ppg ในฟิลด์เป็น 1.95  
8. ในฟิลด์PPGปลายทาง ให้ป้อนตัวเลข
    * ในตัวอย่างนี้ คุณสามารถตั้งค่า ppg ในฟิลด์เป็น 2  
9. ในฟิลด์เปอร์เซ็นทร์ ป้อนจำนวน
    * ในตัวอย่างนี้ คุณสามารถตั้งค่า ppg ในฟิลด์เป็น 3%  
10. ในฟิลด์สกุลเงิน ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
11. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
12. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
13. คลิก บันทึก
