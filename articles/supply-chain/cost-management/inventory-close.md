# <a name="inventory-close"></a><span data-ttu-id="ca0c4-103">การปิดบัญชีสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="ca0c4-103">Inventory close</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="ca0c4-104">โดยเป็นส่วนหนึ่งของกระบวนการที่จะชำระธุรกรรมการออกด้วยธุรกรรมการรับสินทรัพย์ คุณสามารถเลือกให้บัญชีแยกประเภททั่วไปถูกอัพเดตเพื่อสะท้อนถึงการปรับปรุงที่ได้ถูกกระทำไปด้วย</span><span class="sxs-lookup"><span data-stu-id="ca0c4-104">As part of the process to settle issue transactions with receipt transactions, you can also choose to have the general ledger updated to reflect the adjustments that have been made.</span></span>

<span data-ttu-id="ca0c4-105">กระบวนการปิดสินค้าคงคลังใน จับคู่ธุรกรรมการตัดสินค้าจากคลังกับธุรกรรมการรับสินค้า โดยยึดตามวิธีการประเมินค่าสินค้าคงคลังที่เลือกไว้ในกลุ่มแบบจำลองสินค้าของสินค้า</span><span class="sxs-lookup"><span data-stu-id="ca0c4-105">The inventory close process settles issue transactions to receipt transactions, based on the inventory valuation method that is selected in the item’s item model group.</span></span> <span data-ttu-id="ca0c4-106">โดยเป็นส่วนหนึ่งของกระบวนการชำระเงิน คุณสามารถระบุว่าควรมีการอัพเดตบัญชีแยกประเภททั่วไป เพื่อสะท้อนถึงการปรับปรุงที่ทำไว้</span><span class="sxs-lookup"><span data-stu-id="ca0c4-106">As part of the settlement process, you can specify that the general ledger should be updated, so that it reflects the adjustments that have been made.</span></span> <span data-ttu-id="ca0c4-107">อย่างไรก็ตาม จนกว่าจะมีการลงบัญชีธุรกรรมการตัดสินค้าจากคลังที่ราคาต้นทุนเฉลี่ยที่รัน การคำนวณอยู่จนกว่าจะรันการปิดบัญชีสินค้าคงคลังหรือการคำนวณสินค้าคงคลังใหม่</span><span class="sxs-lookup"><span data-stu-id="ca0c4-107">However, until inventory close or recalculation has been run, issue transactions are posted at the calculated running average cost price.</span></span> 

<span data-ttu-id="ca0c4-108">หลังจากที่สินค้าคงคลังปิดบัญชี คุณไม่สามารถลงรายการบัญชีในรอบระยะเวลาก่อนวันที่ปิดบัญชีสินค้าคงคลังที่คุณตั้งค่าไว้ เว้นเสียแต่ว่าคุณย้อนกระบวนการปิดบัญชีสินค้าคงคลังที่เสร็จสมบูรณ์กลับ</span><span class="sxs-lookup"><span data-stu-id="ca0c4-108">After inventory close, you can no longer post in periods that are before the inventory closing date that you set, unless you reverse a completed inventory close process.</span></span> <span data-ttu-id="ca0c4-109">ตัวอย่างเช่น เมื่อรันการปิดสินค้าคงคลังสำหรับรอบระยะเวลาที่สิ้นสุดในวันที่ 31 มกราคม คุณจะไม่สามารถลงรายการบัญชีที่มีวันที่ก่อนหน้าวันที่ 31 มกราคม</span><span class="sxs-lookup"><span data-stu-id="ca0c4-109">For example, if inventory close is run for the period that ends on January 31, you can't post transactions that have a date that is earlier than January 31.</span></span> 

<span data-ttu-id="ca0c4-110">สินค้าในสินค้าคงคลังถูกกำหนดเป็นหนึ่งในสองประเภทของสินค้าคงคลัง: สินค้า หรือบริการ</span><span class="sxs-lookup"><span data-stu-id="ca0c4-110">Items in inventory are assigned to one of two inventory types: item or service.</span></span> <span data-ttu-id="ca0c4-111">การปิดบัญชีสินค้าคงคลังดำเนินการฟังก์ชันเดียวกันสำหรับทั้งสองชนิด</span><span class="sxs-lookup"><span data-stu-id="ca0c4-111">Inventory close performs the same functions for both types.</span></span> <span data-ttu-id="ca0c4-112">อย่างไรก็ตาม สำหรับสินค้าบริการ การปิดบัญชีสินค้าคงคลังจะยังคงจับคู่กับการตัดสินค้าจากคลังกับการรับสินค้า</span><span class="sxs-lookup"><span data-stu-id="ca0c4-112">However, for service items, inventory close still settles issues to receipts.</span></span> 

<span data-ttu-id="ca0c4-113">ความถี่ในการรันกระบวนการปิดบัญชีสินค้าคงคลังแตกต่างกันตามบริษัท</span><span class="sxs-lookup"><span data-stu-id="ca0c4-113">How often the inventory close process is run varies by company.</span></span> <span data-ttu-id="ca0c4-114">อย่างไรก็ตาม ปริมาณธุรกรรมควรช่วยกำหนดความถี่ในการตัดสินใจที่จะรันการปิดบัญชีสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="ca0c4-114">However, transaction volume should help determine how often you decide to run inventory close.</span></span> <span data-ttu-id="ca0c4-115">โดยทั่วไป บริษัทส่วนใหญ่ปิดบัญชีสินค้าคงคลังปิดเป็นส่วนหนึ่งของการปิดบัญชีสิ้นเดือนและกระบวนงานการกระทบยอดบัญชี</span><span class="sxs-lookup"><span data-stu-id="ca0c4-115">In general, most companies run inventory close as part of their month-end close and reconciliation procedures.</span></span>

## <a name="inventory-recalculation-and-the-general-ledger"></a><span data-ttu-id="ca0c4-116">การคำนวณสินค้าคงคลังใหม่และบัญชีแยกประเภททั่วไป</span><span class="sxs-lookup"><span data-stu-id="ca0c4-116">Inventory recalculation and the general ledger</span></span>
<span data-ttu-id="ca0c4-117">หากการปรับปรุงสินค้าคงคลังและบัญชีแยกประเภททั่วไปจำเป็นในช่วงระหว่างเดือนหรือรอบระยะเวลาสินค้าคงคลังอื่น คุณสามารถรันการคำนวณสินค้าคงคลังใหม่แทนการปิดบัญชีสินค้าคงคลังได้</span><span class="sxs-lookup"><span data-stu-id="ca0c4-117">If adjustments to inventory and the general ledger are required during a month or other inventory period, you can run inventory recalculation instead of inventory close.</span></span> <span data-ttu-id="ca0c4-118">คำนวณสินค้าคงคลังทำให้การปรับปรุง แต่ไม่ได้ทำการชำระธุรกรรมสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="ca0c4-118">Inventory recalculation makes adjustments but doesn't make settlements to inventory transactions.</span></span> 

<span data-ttu-id="ca0c4-119">ในระหว่างการคำนวณสินค้าคงคลัง ปรับปรุงปริมาณคงคลังคงเหลือ มีการปรับปรุงธุรกรรมสินค้าคงคลัง และรันการปิดสินค้าคงคลังและการคำนวณสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="ca0c4-119">During inventory recalculation, on-hand inventory is adjusted, inventory transactions are adjusted, and inventory recalculations and inventory closes are run.</span></span> <span data-ttu-id="ca0c4-120">งานนี้จะส่งผลกระทบต่อบัญชีแยกประเภทที่เชื่อมโยงกับธุรกรรมสินค้าคงคลังดั้งเดิม</span><span class="sxs-lookup"><span data-stu-id="ca0c4-120">These tasks affect any ledger account that is linked to the original inventory transaction.</span></span>

<span data-ttu-id="ca0c4-125">เมื่อการอัพเดตเสร็จสมบูรณ์แล้ว คุณสามารถตรวจทานใบสำคัญของบัญชีแยกประเภทที่ได้ ซึ่งจะลงลงรายการบัญชีเนื่องจากงานใดงานหนึ่งเหล่านี้</span><span class="sxs-lookup"><span data-stu-id="ca0c4-125">After the update is completed, you can review the ledger voucher that is posted because of one of these tasks.</span></span>

2.  <span data-ttu-id="ca0c4-127">คลิก **รายละเอียด** แล้วเลือก **ใบสำคัญ**</span><span class="sxs-lookup"><span data-stu-id="ca0c4-127">Click **Details**, and then select **Voucher**.</span></span>

## <a name="effects-of-the-inventory-close-process-on-the-general-ledger"></a><span data-ttu-id="ca0c4-128">ผลกระทบของกระบวนการปิดบัญชีสินค้าคงคลังต่อบัญชีแยกประเภททั่วไป</span><span class="sxs-lookup"><span data-stu-id="ca0c4-128">Effects of the inventory close process on the general ledger</span></span>
<span data-ttu-id="ca0c4-129">หลายงานที่คุณสามารถทำได้ในหน้า **การปิดบัญชีและการปรับปรุง** เป็นผลในการอัพเดตบัญชีแยกประเภททั่วไป</span><span class="sxs-lookup"><span data-stu-id="ca0c4-129">Several of the tasks that you can perform on the **Closing and adjustment** page cause an update to general ledger.</span></span> <span data-ttu-id="ca0c4-130">ตัวอย่างเช่น การปรับปรุงในบัญชีแยกประเภททั่วไปจะทำการปรับปรุงเมื่อคุณปรับปรุงสินค้าคงคลังคงเหลือ ปรับปรุรายการความเคลื่อนไหวของสินค้าคงคลัง รันการคำนวณสินค้าคงคลัง และเรียกใช้การปิดบัญชีสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="ca0c4-130">For example, the general ledger is updated when you make inventory on-hand adjustments, make inventory transaction adjustments, run inventory recalculation, and run inventory close.</span></span> 

<span data-ttu-id="ca0c4-131">บัญชีแยกประเภทที่ได้รับการปรับปรุงเนื่องจากภารกิจเหล่านี้คือบัญชีแยกประเภทที่เชื่อมโยงกับธุรกรรมสินค้าคงคลังดั้งเดิม</span><span class="sxs-lookup"><span data-stu-id="ca0c4-131">The ledger accounts that are updated because of these tasks are linked to the original inventory transaction.</span></span> <span data-ttu-id="ca0c4-132">ตัวอย่างเช่น ถ้าใบสั่งขายถูกจับคู่กับใบสั่งซื้อ บัญชีแยกประเภททั่วไปที่ใช้สำหรับต้นฉบับใบสั่งขายจะถูกปรับปรุง</span><span class="sxs-lookup"><span data-stu-id="ca0c4-132">For example, if a sales order is settled to a purchase order, the general ledger accounts that were used for the original sales order are adjusted.</span></span> <span data-ttu-id="ca0c4-133">พฤติกรรมนี้จะเกิดขึ้นแม้ว่าบัญชีแยกประเภทสำหรับกลุ่มสินค้าที่ถูกกำหนดให้กับสินค้านี้ได้เปลี่ยนแปลงเนื่องจากใบสั่งขายถูกลงรายการบัญชีไปแล้ว</span><span class="sxs-lookup"><span data-stu-id="ca0c4-133">This behavior occurs even if the ledger accounts for the item group that is assigned to the item have changed since the sales order was posted.</span></span> <span data-ttu-id="ca0c4-134">หลังจากการปิดบัญชีสินค้าคงคลังสร้างจำนวนการชำระเงิน จำนวนการชำระเงินยังคงลงถูกรายการบัญชีในบัญชีแยกประเภทดั้งเดิม แทนบัญชีแยกประเภทใหม่ที่ถูกกำหนดให้กับสินค้านั้น</span><span class="sxs-lookup"><span data-stu-id="ca0c4-134">After inventory close creates a settlement amount, the settlement amount is still posted to the original ledger accounts, not to the new ledger accounts that are assigned to the item.</span></span> <span data-ttu-id="ca0c4-135">นอกจากนี้บัญชีแยกประเภททั่วไปอาจได้รับการอัพเดตด้วยการกลับรายการปิดบัญชีสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="ca0c4-135">The general ledger might also be updated if you reverse an inventory close.</span></span>

## <a name="the-inventory-close-log"></a><span data-ttu-id="ca0c4-140">การล็อกการปิดบัญชีสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="ca0c4-140">The inventory close log</span></span>

<span data-ttu-id="ca0c4-142">ก่อนที่ข้อความนี้จะแสดง ระบบจะรายงานหมายเลขสินค้าและธุรกรรมที่ได้รับผลกระทบ</span><span class="sxs-lookup"><span data-stu-id="ca0c4-142">Before this message is shown, the system reports the item number and the affected transaction.</span></span> <span data-ttu-id="ca0c4-143">ข้อความนี้จะแจ้งให้คุณทราบจำนวนต้นทุนที่ถูกใช้สำหรับธุรกรรมนี้ที่ยังไม่ได้อัพเดตเนื่องจากการปิดบัญชีสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="ca0c4-143">The message informs you that the cost amount that is used for this transaction wasn't updated because of the inventory close.</span></span> <span data-ttu-id="ca0c4-144">ข้อความนี้จะปรากฏขึ้นเมื่อไม่สามารถชำระธุรกรรมชนิดการตัดสินค้าจากคลังได้</span><span class="sxs-lookup"><span data-stu-id="ca0c4-144">This message appears when a transaction of the issue type can't be settled.</span></span> 

<span data-ttu-id="ca0c4-145">ในระหว่างกระบวนงานการปิดบัญชีสินค้าคงคลัง ระบบจะตรวจสอบมิติทางการเงินแต่ละมิติเพื่อดูว่ามีการออกมากกว่าการรับจนถึงวันปิดบัญชีที่ระบุหรือไม่</span><span class="sxs-lookup"><span data-stu-id="ca0c4-145">During the inventory close process, the system checks each financial dimension to see whether there are more issues than receipts up to the specified closing date.</span></span> <span data-ttu-id="ca0c4-146">ความไม่สมดุลชนิดนี้สามารถเกิดขึ้นได้เมื่อธุรกรรมสินค้าคงคลังจากใบสั่งซื้อยังไม่ถูกลงรายการบัญชีทางการเงินอย่างครบถ้วน เนื่องจากใบแจ้งหนี้ของผู้จัดจำหน่ายยังไม่ถูกรับมา หรือเนื่องจากส่วนประกอบสูตรการผลิต (BOM) ที่เป็นส่วนประกอบในการผลิตในระดับที่สูงกว่ายังไม่ถูกลงรายการบัญชีอย่างครบถ้วน</span><span class="sxs-lookup"><span data-stu-id="ca0c4-146">This type of imbalance can occur when an inventory transaction from a purchase order isn't fully posted financially, either because the vendor invoice hasn't been received, or because bill of materials (BOM) components that are included in a production on a higher level aren't financially posted.</span></span> <span data-ttu-id="ca0c4-147">(การผลิตย่อยไม่ถูกคำนวณต้นทุน) ถ้าเกิดกรณีนี้ขึ้น การปิดบัญชีในเวลาต่อมาจะไม่ปรับปรุงการตัดสินค้าจากคลังทั้งหมดให้เป็นราคาต้นทุนที่ถูกต้องเนื่องจากข้อมูลการรับสินค้ามีไม่เพียงพอ</span><span class="sxs-lookup"><span data-stu-id="ca0c4-147">(The sub-production isn't cost-calculated.) <span data-ttu-id="ca0c4-148">สำหรับแต่ละกระบวนงานการรันการปิดบัญชี ระบบจะบ่งชี้ว่าล็อกที่มีคำเตือนจะถูกจัดเก็บไว้และสามารถดูได้หรือไม่</span><span class="sxs-lookup"><span data-stu-id="ca0c4-148">For each run of the closing procedure, the system indicates whether a log that contains the warnings is stored and can be viewed.</span></span> 

<span data-ttu-id="ca0c4-149">หากคุณได้รับข้อความเตือนหลายข้อความในกล่องข้อความ เราขอแนะนำให้คุณดำเนินการดังต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="ca0c4-149">If you receive many warnings in the message, we recommend that you perform the following actions:</span></span>

-   <span data-ttu-id="ca0c4-151">เลื่อนวันปิดบัญชีไปข้างหน้า</span><span class="sxs-lookup"><span data-stu-id="ca0c4-151">Advance the closing date.</span></span>
-   <span data-ttu-id="ca0c4-152">ประเมินกระบวนงานทางธุรกิจใหม่</span><span class="sxs-lookup"><span data-stu-id="ca0c4-152">Reevaluate the business procedures.</span></span>

<span data-ttu-id="ca0c4-153">อาจมีกรณีที่คุณไม่สามารถทำอะไรเกี่ยวกับคำเตือนได้</span><span class="sxs-lookup"><span data-stu-id="ca0c4-153">In some circumstances, you might not be able to do anything about the warnings.</span></span> <span data-ttu-id="ca0c4-154">ตัวอย่างเช่น เมื่อการทำเครื่องหมายถูกใช้และใบสั่งซื้อที่ทำเครื่องหมายไว้มีวันที่ทางการเงินหลังจากวันปิดบัญชี วันที่ปิดบัญชีจะไม่สามารถเปลี่ยนแปลงได้</span><span class="sxs-lookup"><span data-stu-id="ca0c4-154">For example, if marking is used, and the marked purchase order has a financial date that is after the closing date, the closing date can't be changed.</span></span>

## <a name="reversing-a-completed-inventory-close"></a><span data-ttu-id="ca0c4-155">การย้อนการปิดบัญชีสินค้าคงคลังที่เสร็จสมบูรณ์แล้วกลับ</span><span class="sxs-lookup"><span data-stu-id="ca0c4-155">Reversing a completed inventory close</span></span>
<span data-ttu-id="ca0c4-156">ในบางครั้ง คุณอาจต้องกลับรายการการปิดบัญชีสินค้าคงคลังที่เสร็จสมบูรณ์แล้วเพื่อคืนการชำระไปยังสถานะก่อนทำการปรับปรุง</span><span class="sxs-lookup"><span data-stu-id="ca0c4-156">Occasionally, you might have to reverse a completed inventory close to return settlements to the state that they had before adjustments were made.</span></span> <span data-ttu-id="ca0c4-157">เมื่อคุณย้อนการปิดบัญชีสินค้าคงคลังที่เสร็จสมบูรณ์แล้วกลับ สินค้าคงคลังจะถูกเปิดใหม่เพื่อเปิดใช้งานการลงรายการบัญชีในรอบระยะเวลาที่การปิดสินค้าคงคลังครอบคลุม</span><span class="sxs-lookup"><span data-stu-id="ca0c4-157">When you reverse a completed inventory close, inventory is reopened to enable posting in the period that the inventory close covers.</span></span> <span data-ttu-id="ca0c4-158">การเปลี่ยนแปลงที่เกี่ยวข้องอาจถูกกระทำในบัญชีแยกประเภททั่วไปด้วยเช่นกัน</span><span class="sxs-lookup"><span data-stu-id="ca0c4-158">Related changes might also be made in the general ledger.</span></span> <span data-ttu-id="ca0c4-159">หลังจากที่คุณเสร็จสิ้นการปรับปรุง คุณสามารถรันการปิดบัญชีสินค้าคงคลังอีกครั้งสำหรับรอบระยะเวลาที่คุณกำลังทำงานด้วย</span><span class="sxs-lookup"><span data-stu-id="ca0c4-159">After you've finished making adjustments, you can run inventory close again for the period that you're working with.</span></span>
