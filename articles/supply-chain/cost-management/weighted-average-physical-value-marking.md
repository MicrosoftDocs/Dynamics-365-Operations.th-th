<span data-ttu-id="18e5b-104">ค่าเฉลี่ยถ่วงน้ำหนักเป็นแบบจำลองสินค้าคงคลังตามหลักการของค่าเฉลี่ยถ่วงน้ำหนัก ซึ่งการตัดสินค้าจากคลังจากสินค้าคงคลังจะมีค่าที่ค่าเฉลี่ยของสินค้าที่ได้รับในสินค้าคงคลังระหว่างรอบระยะเวลาการปิดบัญชีสินค้าคงคลัง บวกด้วยปริมาณคงคลังคงเหลือใดๆ จากรอบระยะเวลาก่อนหน้า</span><span class="sxs-lookup"><span data-stu-id="18e5b-104">Weighted average is an inventory model based on the weighted average principle, where issues from inventory are valued at the average value of the items that are received into inventory during the inventory closing period, plus any on-hand inventory from the previous period.</span></span>

<span data-ttu-id="18e5b-105">เมื่อคุณรันการปิดบัญชีสินค้าคงคลัง การรับสินค้าทั้งหมดจะถูกจับคู่กับการนำสินค้าออกใช้เสมือน ซึ่งมีปริมาณและมูลค่าของสินค้าที่ได้รับทั้งหมด </span><span class="sxs-lookup"><span data-stu-id="18e5b-105">When you run an inventory closing, all receipts are settled against a virtual issue, which holds the total received quantity and value.</span></span> <span data-ttu-id="18e5b-106">การนำสินค้าออกใช้เสมือนมีความสอดคล้องกับการรับสินค้าเสมือนจากที่ที่การออกสินค้าจะถูกชำระด้วย</span><span class="sxs-lookup"><span data-stu-id="18e5b-106">This virtual issue has a corresponding virtual receipt from which the issues are settled.</span></span> <span data-ttu-id="18e5b-107">ในลักษณะนี้ การนำสินค้าจากคลังทั้งหมดได้รับต้นทุนเฉลี่ยเดียวกัน</span><span class="sxs-lookup"><span data-stu-id="18e5b-107">In this manner, all issues get the same average cost.</span></span> <span data-ttu-id="18e5b-108">การนำสินค้าออกใช้เสมือนและการรับสินค้าอาจถือว่าเป็นการโอนย้ายเสมือน ที่เรียกว่าการการโอนย้ายการปิดบัญชีสินค้าคงคลังค่าเฉลี่ยถ่วงน้ำหนัก</span><span class="sxs-lookup"><span data-stu-id="18e5b-108">The virtual issue and receipt can be seen as a virtual transfer, which is named the weighted average inventory closing transfer.</span></span>

<span data-ttu-id="18e5b-109">ถ้ามีการรับสินค้าเพียงรายการเดียว การนำสินค้าออกใช้จากทั้งหมดสามารถชำระจากการรับสินค้านั้นและการโอนย้ายเสมือนจะไม่ถูกสร้าง</span><span class="sxs-lookup"><span data-stu-id="18e5b-109">If there is only one receipt, all issues can be settled from it and the virtual transfer will not be created.</span></span> 

<span data-ttu-id="18e5b-110">เมื่อใช้ค่าเฉลี่ยถ่วงน้ำหนัก คุณสามารถทำเครื่องหมายธุรกรรมสินค้าคงคลัง เพื่อให้การรับสินค้าที่ระบุถูกจับคู่กับการนำสินค้าออกใช้ที่ระบุ แทนการใช้กฎค่าเฉลี่ยถ่วงน้ำหนัก</span><span class="sxs-lookup"><span data-stu-id="18e5b-110">When using weighted average, you can mark inventory transactions so that a specific item receipt is settled against a specific issue, instead of using the weighted average rule.</span></span> 

<span data-ttu-id="18e5b-111">ขอแนะนำให้ปิดบัญชีสินค้าคงคลังเป็นรายเดือน เมื่อคุณใช้แบบจำลองสินค้าคงคลังค่าเฉลี่ยถ่วงน้ำหนัก</span><span class="sxs-lookup"><span data-stu-id="18e5b-111">We recommend a monthly inventory closing when you use the weighted average inventory model.</span></span> 

<span data-ttu-id="18e5b-112">วิธีการคิดต้นทุนสินค้าคงคลังค่าเฉลี่ยถ่วงน้ำหนักจะคำนวณโดยสูตรดังต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="18e5b-112">The weighted average inventory costing method is calculated by the following formula:</span></span>

<span data-ttu-id="18e5b-114">ธุรกรรมสินค้าคงคลังทีออกจากการตัดสินค้าจากสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="18e5b-114">Inventory transactions leaving the inventory issues.</span></span> <span data-ttu-id="18e5b-115">ข้อมูลนี้รวมถึงใบสั่งขาย สมุดรายวันสินค้าคงคลัง และใบสั่งผลิต เกิดขึ้น ณ ราคาต้นทุนประเมินในวันลงรายการบัญชี</span><span class="sxs-lookup"><span data-stu-id="18e5b-115">This includes sales orders, inventory journals, and production orders, occur at an estimated cost price on the posting date.</span></span> <span data-ttu-id="18e5b-116">ราคาต้นทุนการประเมินนี้จะเรียกว่า ค่าเฉลี่ยที่รันอยู่</span><span class="sxs-lookup"><span data-stu-id="18e5b-116">This estimated cost price is also referred to as running average.</span></span> <span data-ttu-id="18e5b-117">ในเวลาที่มีการปิดบัญชีสินค้าคงคลัง ระบบจะวิเคราะห์รายการความเคลื่อนไหวของสินค้าคงคลังสำหรับรอบระยะเวลาก่อนหน้าและปัจจุบัน และกำหนดว่าควรจะใช้หลักการการปิดบัญชีใดดังต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="18e5b-117">At the time of inventory close, the system will analyze the inventory transactions for previous and current periods and determine which of the following closing principles should be used.</span></span>
