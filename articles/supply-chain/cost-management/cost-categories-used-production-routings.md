# <a name="cost-categories-used-in-production-routing"></a><span data-ttu-id="7a4fa-103">ประเภทต้นทุนที่ใช้ในกระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="7a4fa-103">Cost categories used in production routing</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="7a4fa-104">บทความนี้แสดงข้อมูลเกี่ยวกับประเภทต้นทุนที่ใช้กับสภาพแวดล้อมการผลิตที่ใช้กระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="7a4fa-104">This article provides information about cost categories that apply to manufacturing environments that use routing.</span></span>

<span data-ttu-id="7a4fa-105">ประเภทต้นทุนใช้กับสภาพแวดล้อมการผลิตที่ใช้กระบวนการผลิต </span><span class="sxs-lookup"><span data-stu-id="7a4fa-105">Cost categories apply to manufacturing environments that use routing.</span></span> <span data-ttu-id="7a4fa-106">มีการกำหนดให้กับทรัพยากรการดำเนินงานและการดำเนินงานกระบวนการผลิต เพื่อกำหนดต้นทุนต่อชั่วโมง และเพื่อจัดสรรเซกเมนต์ต้นทุนในต้นทุนที่คำนวณได้ของสินค้าที่ผลิต </span><span class="sxs-lookup"><span data-stu-id="7a4fa-106">They are assigned to operations resources and routing operations to define hourly costs and to segment cost contributions in a manufactured item’s calculated costs.</span></span> <span data-ttu-id="7a4fa-107">กลุ่มต้นทุนที่ถูกกำหนดให้กับประเภทต้นทุนจะจัดประเภทการจัดสรรต้นทุน โดยขึ้นอยู่กับทรัพยากรการดำเนินงานและชนิดของกิจกรรม เช่น เวลาการตั้งค่า และเวลาที่ใช้ในการรัน </span><span class="sxs-lookup"><span data-stu-id="7a4fa-107">The cost groups that are assigned to cost categories classify manufacturing cost contributions, based on the operation resources and the type of activity, such as setup time and run time.</span></span> <span data-ttu-id="7a4fa-108">ความเฉพาะเจาะจงของการกำหนดกลุ่มต้นเปิดใช้งานค่าโสหุ้ยที่จะคำนวณการผลิตตามข้อมูลกระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="7a4fa-108">The specificity of cost group assignments enables manufacturing overhead to be calculated based on routing information.</span></span> 

<span data-ttu-id="7a4fa-109">**หมายเหตุ:** ในสภาพแวดล้อมการผลิต ประเภทต้นทุนจะเป็นที่รู้จักในชื่อหลายๆชื่อ เช่น รหัสอัตราค่าแรง หรือรหัสอัตราเครื่องจักร</span><span class="sxs-lookup"><span data-stu-id="7a4fa-109">**Note:** In manufacturing environments, cost categories are known by several other names, such as labor rate codes or machine rate codes.</span></span> 

<span data-ttu-id="7a4fa-110">ประเภทต้นทุนแต่ละประเภทมีงเรกคอร์ดต้นทุนและกลุ่มต้นทุนที่กำหนดที่้เชื่อมโยงกัน </span><span class="sxs-lookup"><span data-stu-id="7a4fa-110">Each cost category has associated cost records and an assigned cost group.</span></span> <span data-ttu-id="7a4fa-111">ประเภทต้นทุนต่างๆจะจำเป็นสำหรับวัตถุประสงค์การผลิตที่แตกต่างกัน</span><span class="sxs-lookup"><span data-stu-id="7a4fa-111">Different cost categories are required for different production purposes.</span></span>

-   <span data-ttu-id="7a4fa-112">กำหนดต้นทุนต่อชั่วโมงที่แตกต่างกันตามทรัพยากรการดำเนินงาน </span><span class="sxs-lookup"><span data-stu-id="7a4fa-112">Assign different hourly costs, depending on the operations resource.</span></span> <span data-ttu-id="7a4fa-113">เช่น ต้นทุนสามารถแตกต่างกันได้สำหรับชนิดต่างๆของทักษะแรงงาน เครื่องจักร และเซลล์การผลิตต่างๆ</span><span class="sxs-lookup"><span data-stu-id="7a4fa-113">For example, the costs can differ for various types of labor skills, machines, or manufacturing cells.</span></span>
-   <span data-ttu-id="7a4fa-114">กำหนดต้นทุนต่อชั่วโมงที่แตกต่างกันสำหรับเวลาเซ็ตอัพหรือเวลาที่ใช้ในการผลิตที่เชื่อมโยงกับการดำเนินกระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="7a4fa-114">Assign different hourly costs for the setup time or run time that is associated with a routing operation.</span></span>

-   <span data-ttu-id="7a4fa-116">ให้การแบ่งส่วนกลุ่มต้นทุนของการจัดสรรต้นทุน ไปยังต้นทุนที่คำนวณได้ของสินค้าที่ผลิต </span><span class="sxs-lookup"><span data-stu-id="7a4fa-116">Provide cost group segmentation of cost contributions to a manufactured item’s calculated cost.</span></span> <span data-ttu-id="7a4fa-117">ตัวอย่างเช่น คุณสามารถแบ่งส่วนต้นทุนแรงงานและเครื่องจักรได้</span><span class="sxs-lookup"><span data-stu-id="7a4fa-117">For example, you can segment of labor and machine costs.</span></span>
-   <span data-ttu-id="7a4fa-118">ให้พื้นฐานกลุ่มต้นทุนสำหรับสูตรคำนวณค่าโสหุ้ย เช่น สูตรสำหรับค่าโสหุ้ยที่เกี่ยวข้องกับแรงงาน และค่าโสหุ้ยที่เกี่ยวข้องกับเครื่องจักร หรือค่าโสหุ้ยที่เกี่ยวข้องกับเวลาการตั้งค่าและเวลาที่ใช้ในการผลิต</span><span class="sxs-lookup"><span data-stu-id="7a4fa-118">Provide the cost group basis for overhead calculation formulas, such as formulas for labor-related and machine-related overhead, or overhead that is related to setup and run time.</span></span>
