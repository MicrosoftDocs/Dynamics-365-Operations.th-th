<span data-ttu-id="bed63-106">บทความนี้แสดงข้อมูลเกี่ยวกับการวิเคราะห์ต้นทุนที่คุณสามารถทำได้ สำหรับใบสั่งผลิตที่เสร็จสมบูรณ์และปัจจุบัน </span><span class="sxs-lookup"><span data-stu-id="bed63-106">This article provides information about the cost analysis that you can do for completed and current production orders.</span></span> <span data-ttu-id="bed63-107">คุณสามารถวิเคราะห์ต้นทุนที่ประเมินและต้นทุนจริง โดยการใช้หน้าการคำนวณราคา หรือการประเมินต้นทุน และการคิดต้นทุนรายงาน </span><span class="sxs-lookup"><span data-stu-id="bed63-107">You can analyze the estimated costs and actual costs by using the Price calculation page or the Cost estimates and costings report.</span></span> <span data-ttu-id="bed63-108">คุณสามารถดูข้อมูลเกี่ยวกับต้นทุนที่ประเมิน และที่เกิดขึ้นจริง (และปริมาณ) สำหรับแต่ละสินค้าส่วนประกอบ การดำเนินงาน และต้นทุนทางอ้อม</span><span class="sxs-lookup"><span data-stu-id="bed63-108">You can view information about the estimated and actual costs (and quantity) for each component item, the routing operation, and the indirect cost.</span></span>

<span data-ttu-id="bed63-109">ต้นทุนจริงสำหรับใบสั่งผลิตจะเป็นไปตามการใช้วัสดุและการดำเนินงานกระบวนการผลิตที่ได้รายงาน</span><span class="sxs-lookup"><span data-stu-id="bed63-109">The actual costs for a production order are based on the reported consumption of material and routing operations.</span></span> <span data-ttu-id="bed63-110">คุณสามารถเข้าถึงธุรกรรมโดยละเอียดเกี่ยวกับการรายงานปริมาณการใช้วัสดุ การดำเนินงานกระบวนการผลิต และต้นทุนทางอ้อมสำหรับใบสั่งผลิตในหน้า **การลงรายการบัญชีการผลิต**</span><span class="sxs-lookup"><span data-stu-id="bed63-110">You can access detailed transactions about the reported consumption of material, routing operations, and indirect costs for a production order on the **Production posting** page.</span></span>

## <a name="variance-analysis-for-a-completed-production-order"></a><span data-ttu-id="bed63-111">การวิเคราะห์ผลต่างสำหรับใบสั่งผลิตที่เสร็จสมบูรณ์แล้ว</span><span class="sxs-lookup"><span data-stu-id="bed63-111">Variance analysis for a completed production order</span></span>
<span data-ttu-id="bed63-112">ผลต่างที่แสดงการเปรียบเทียบของกิจกรรมการผลิตที่รายงาน และการคำนวณต้นทุนมาตรฐานสำหรับรายการผลิต </span><span class="sxs-lookup"><span data-stu-id="bed63-112">The variances reflect a comparison of the reported production activities and the calculation of standard costs for the production item.</span></span> <span data-ttu-id="bed63-113">ผลต่างไม่สะท้อนถึงการเปรียบเทียบกับต้นทุนที่ประเมินได้ของใบสั่งผลิต </span><span class="sxs-lookup"><span data-stu-id="bed63-113">The variances don't reflect a comparison to the production order's estimated costs.</span></span> <span data-ttu-id="bed63-114">กิจกรรมการผลิตที่รายงานประกอบด้วย ปริมาณการใช้วัสดุและการดำเนินงานกระบวนการผลิต พร้อมกับต้นทุนทางอ้อมเกี่ยวข้อง และปริมาณที่รายงานเป็น เสร็จแล้ว </span><span class="sxs-lookup"><span data-stu-id="bed63-114">The production activities that are reported include the consumption of material and routing operations, together with the associated indirect costs, and the quantity that is reported as finished.</span></span> <span data-ttu-id="bed63-115">ผลต่างสี่ชนิดดังต่อไปนี้จะถูกคำนวณ:</span><span class="sxs-lookup"><span data-stu-id="bed63-115">The following four types of variance are calculated:</span></span>

-   <span data-ttu-id="bed63-116">ผลต่างขนาดล็อต</span><span class="sxs-lookup"><span data-stu-id="bed63-116">Lot size variance</span></span>
-   <span data-ttu-id="bed63-117">ผลต่างปริมาณการผลิต</span><span class="sxs-lookup"><span data-stu-id="bed63-117">Production quantity variance</span></span>
-   <span data-ttu-id="bed63-118">ผลต่างราคาการผลิต</span><span class="sxs-lookup"><span data-stu-id="bed63-118">Production price variance</span></span>
-   <span data-ttu-id="bed63-119">ผลต่างการทดแทนการผลิต</span><span class="sxs-lookup"><span data-stu-id="bed63-119">Production substitution variance</span></span>

<span data-ttu-id="bed63-120">แผนภาพต่อไปนี้แสดงผลต่างสี่รายการที่อธิบายผลต่างระหว่างต้นทุนจริงของใบสั่งผลิต และต้นทุนที่คำนวณภายในเรกคอร์ดต้นทุนของสินค้า เมื่อสิ้นสุดใบสั่งผลิต </span><span class="sxs-lookup"><span data-stu-id="bed63-120">The following diagram shows the four variances that account for the difference between a production order's actual costs and the calculated costs within the item's cost record when the production order is ended.</span></span> 

![ผลต่างที่ทำให้เกิดความแตกต่างในใบสั่งผลิตที่เสร็จสมบูรณ์](./media/control.jpg) 

<span data-ttu-id="bed63-122">คุณสามารถวิเคราะห์ผลต่างการผลิตได้โดยใช้หน้า **ผลต่าง** หรือรายงาน **ผลต่างการผลิต**</span><span class="sxs-lookup"><span data-stu-id="bed63-122">You can analyze the production variances by using the **Variance** page or the **Production variance** report.</span></span> <span data-ttu-id="bed63-123">ใช้ตัวเลือกการแสดงผลเพื่อดูผลต่างโดยละเอียดตามสินค้าและทรัพยากรการดำเนินงาน หรือตามกลุ่มต้นทุน </span><span class="sxs-lookup"><span data-stu-id="bed63-123">Use the display options to view detailed variances by item and operations resource, or by cost group.</span></span> <span data-ttu-id="bed63-124">นโยบายสำหรับการแบ่งต้นทุนในพารามิเตอร์สินค้าคงคลังกำหนดว่า มีติดตามผลต่างตามกลุ่มต้นทุนหรือไม่ </span><span class="sxs-lookup"><span data-stu-id="bed63-124">The policy for cost breakdown in the inventory parameters determines whether the variances are tracked by cost group.</span></span> <span data-ttu-id="bed63-125">คุณยังสามารถใช้ตัวเลือกการแสด ผล **เดี่ยว** **หลากหลาย** และ **ผลรวม** เพื่อดูผลต่างโดยสรุปได้ </span><span class="sxs-lookup"><span data-stu-id="bed63-125">You can also use the **single**, **multi**, and **total** display options to view summarized variances.</span></span> <span data-ttu-id="bed63-126">ข้อมูลเกี่ยวกับผลต่างโดยละเอียดสามารถช่วยให้คุณเข้าใจแหล่งที่มาของผลต่างของแต่ละรายการ </span><span class="sxs-lookup"><span data-stu-id="bed63-126">The information about detailed variances can help you understand the source of each variance.</span></span> <span data-ttu-id="bed63-127">เพื่อคาดการณ์ผลต่างก่อนคุณสิ้นสุดใบสั่งผลิต ให้วิเคราะห์ข้อมูลโดยละเอียดที่ได้รับในรายงาน **การประเมินต้นทุนและการคิดต้นทุน**</span><span class="sxs-lookup"><span data-stu-id="bed63-127">To predict variances before you end a production order, analyze the detailed information that is provided on the **Cost estimates and costings** report.</span></span>

## <a name="cost-analysis-for-current-production-orders"></a><span data-ttu-id="bed63-128">การวิเคราะห์ต้นทุนสำหรับใบสั่งผลิตปัจจุบัน</span><span class="sxs-lookup"><span data-stu-id="bed63-128">Cost analysis for current production orders</span></span>
<span data-ttu-id="bed63-129">รายงานต่างๆ ที่แยกต่างหากจะมีข้อมูลเกี่ยวกับธุรกรรมแต่ละชนิด </span><span class="sxs-lookup"><span data-stu-id="bed63-129">Separate reports provide information about each type of transaction.</span></span> <span data-ttu-id="bed63-130">ใช้รายงานเหล่านี้ในการวิเคราะห์ต้นทุนสำหรับกิจกรรมการผลิตที่รายงาน </span><span class="sxs-lookup"><span data-stu-id="bed63-130">Use these reports to analyze costs for reported production activities.</span></span> <span data-ttu-id="bed63-131">มีการแสดงข้อมูลเฉพาะสำหรับใบสั่งผลิตปัจจุบันที่มีสถานะเป็น **เริ่มต้นแล้ว** หรือ **รายงานเป็นเสร็จสมบูรณ์แล้ว**</span><span class="sxs-lookup"><span data-stu-id="bed63-131">Information is displayed only for current production orders that have a status of **Started** or **Reported as finished**.</span></span>

-   <span data-ttu-id="bed63-132">**วัตถุดิบที่อยู่ในกระบวนการ** − รายงานจะแสดงรายการธุรกรรมรายการเบิกสินค้าที่ถูกรายงานเปรียบเทียบกับใบสั่งผลิตปัจจุบัน ณ วันที่ทำธุรกรรมที่ระบุไว้</span><span class="sxs-lookup"><span data-stu-id="bed63-132">**Materials in process** − This report lists the picking list transactions that are reported against the current production orders as of a specified transaction date.</span></span> <span data-ttu-id="bed63-133">รายงานระบุปริมาณของส่วนประกอบที่ถูกตัดจากคลัง และยอดต้นทุนสำหรับธุรกรรมแต่ละรายการ </span><span class="sxs-lookup"><span data-stu-id="bed63-133">The report indicates the quantity of a component that was issued and the cost amount for each transaction.</span></span> <span data-ttu-id="bed63-134">ใช้หลักเกณฑ์การเลือกสำหรับสินค้าที่มีส่วนประกอบเดียว </span><span class="sxs-lookup"><span data-stu-id="bed63-134">Use the selection criteria for a single component item.</span></span> <span data-ttu-id="bed63-135">ตัวอย่างเช่น คุณสามารถพิมพ์ข้อมูลเกี่ยวกับปริมาณที่ตัดส่วนประกอบจากคลังเทียบกับใบสั่งผลิตที่ใช้ได้ </span><span class="sxs-lookup"><span data-stu-id="bed63-135">For example, you can print information about the component’s issued quantity against applicable production orders.</span></span> <span data-ttu-id="bed63-136">ปริมาณที่ถูกตัดจากคลังจะไม่ได้รับการอัพเดตตามปริมาณที่ถูกรายงานเป็น เสร็จสิ้นแล้ว สำหรับสินค้าหลัก </span><span class="sxs-lookup"><span data-stu-id="bed63-136">The issued quantity isn't updated by the quantities that are reported as finished for the parent item.</span></span> <span data-ttu-id="bed63-137">ดังนั้น ปริมาณจริงของวัตถุดิบที่อยู่ระหว่างกระบวนการอาจมีเกินจริงได้</span><span class="sxs-lookup"><span data-stu-id="bed63-137">Therefore, the actual quantity of raw materials in process might be overstated.</span></span>
