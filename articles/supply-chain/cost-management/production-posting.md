# <a name="production-posting"></a><span data-ttu-id="21154-103">การผลิตที่ลงรายการบัญชีแล้ว</span><span class="sxs-lookup"><span data-stu-id="21154-103">Production posting</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="21154-104">บทความนี้แสดงข้อมูลเกี่ยวกับชนิดของการลงรายการบัญชีในกระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="21154-104">This article provides information about different types of postings in the production process.</span></span>

<span data-ttu-id="21154-105">กิจกรรมการลงรายการบัญชีการผลิตตามกระบวนการผลิตที่อธิบายไว้ด้านล่าง</span><span class="sxs-lookup"><span data-stu-id="21154-105">Production posting activities follow production processes that are described in the sections below.</span></span>

## <a name="material-consumption"></a><span data-ttu-id="21154-106">ปริมาณการใช้วัสดุ</span><span class="sxs-lookup"><span data-stu-id="21154-106">Material consumption</span></span>
<span data-ttu-id="21154-107">วัสดุถูกลงทะเบียนการใช้ระหว่างการผลิตเมื่อมีการบันทึกลงสมุดรายการรายการเบิกสินค้ารายวันสำหรับการผลิต</span><span class="sxs-lookup"><span data-stu-id="21154-107">Materials are registered as consumed during production when the production picking list journal is posted.</span></span> <span data-ttu-id="21154-108">กระบวนการนี้สร้างธุรกรรมการตัดสินค้าจากคลังโดยลบสินค้าคงคลังคงเหลือ</span><span class="sxs-lookup"><span data-stu-id="21154-108">This process generates issue transactions that deduct the on-hand inventory.</span></span> <span data-ttu-id="21154-109">ในพารามิเตอร์การผลิต คุณสามารถระบุได้ว่า ควรลงมูลค่าของวัตถุดิบที่อยู่ระหว่างดำเนินการ (งานในระหว่างการดำเนินการ \[WIP\]) ในบัญชีแยกประเภท</span><span class="sxs-lookup"><span data-stu-id="21154-109">In the production parameters, you can specify whether the value of raw materials that are in progress (work in process \[WIP\]) should be posted in the ledger.</span></span> <span data-ttu-id="21154-110">ค่าของวัตถุดิบที่อยู่ในระหว่างการดำเนินการ (WIP) มีการลงรายการบัญชีรายการเบิกสินค้าเฉพาะและบัญชีตรงข้ามสำหรับรายการเบิกสินค้าเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="21154-110">The value of raw materials that are in progress (WIP) is then posted to a dedicated Picking list account and a dedicated Picking list offset account.</span></span>

## <a name="time-consumption"></a><span data-ttu-id="21154-111">ปริมาณการใช้เวลา</span><span class="sxs-lookup"><span data-stu-id="21154-111">Time consumption</span></span>
<span data-ttu-id="21154-112">เวลาที่ผู้ปฏิบัติงานใช้ในงานการผลิตจะถูกบันทึกในสมุดบัตรกระบวนการผลิตรายวันหรือสมุดบัตรงานรายวัน</span><span class="sxs-lookup"><span data-stu-id="21154-112">The time that workers spend on production jobs is recorded in the Route card journal or the Job card journal.</span></span> <span data-ttu-id="21154-113">เมื่อลงรายการบัญชีสมุดรายวันเหล่านี้ มีการประมวลผลบัญชีแยกประเภทที่ลงรายการบัญชีเฉพาะสำหรับทรัพยากรที่อยู่ในระหว่างการดำเนินการ (WIP)</span><span class="sxs-lookup"><span data-stu-id="21154-113">When these journals are posted, ledger posting to a dedicated account for resources that are in progress (WIP) is processed.</span></span> <span data-ttu-id="21154-114">ลงรายการบัญชีนี้แสดงถึงมูลค่าของเวลาที่ใช้ในใบสั่งผลิต</span><span class="sxs-lookup"><span data-stu-id="21154-114">This posting represents the value of the time that is spent on the production order.</span></span> <span data-ttu-id="21154-115">หลังจากที่ใบสั่งผลิตลงทะเบียนเป็นสิ้นสุด บัญชี WIP จะถูกจับคู่</span><span class="sxs-lookup"><span data-stu-id="21154-115">After the production order is registered as ended, the WIP accounts are settled.</span></span>

## <a name="reporting-finished-goods-and-error-quantities"></a><span data-ttu-id="21154-116">การรายงานสินค้าสำเร็จรูปและปริมาณของข้อผิดพลาด</span><span class="sxs-lookup"><span data-stu-id="21154-116">Reporting finished goods and error quantities</span></span>
<span data-ttu-id="21154-117">เมื่อใบสั่งผลิตมีรายงานเมื่อเสร็จสมบูรณ์ ปริมาณของสินค้าสำเร็จรูปที่เสร็จสมบูรณ์จะถูกปรับปรุงในการบริหารสินค้าคงคลังโดยใช้การรายงานสมุดรายวันที่เสร็จสิ้นแล้ว</span><span class="sxs-lookup"><span data-stu-id="21154-117">When a production order is reported as finished, the quantity of the finished goods that have been completed is updated in Inventory management through the Report as finished journal.</span></span> <span data-ttu-id="21154-118">ถ้าคุณกำลังใช้บัญชี WIP ซึ่งสามารถตั้งค่าในพารามิเตอร์การผลิต สมุดวันบัญชีแยกประเภทรายทำเพื่อลดบัญชี WIP และเพิ่มสินค้าคงคลังของสินค้าสำเร็จรูป</span><span class="sxs-lookup"><span data-stu-id="21154-118">If you're using WIP accounting, which can be set up in the production parameters, a ledger journal is made to reduce the WIP accounts and increase the inventory of the finished goods.</span></span> <span data-ttu-id="21154-119">สมุดรายวันใช้ต้นทุนมาตรฐานที่กำหนดไว้สำหรับผลิตภัณฑ์</span><span class="sxs-lookup"><span data-stu-id="21154-119">The journal uses the standard cost that is defined for the product.</span></span>

<span data-ttu-id="21154-121">ก่อนสิ้นสุดการผลิต มีการคำนวณต้นทุนจริงสำหรับปริมาณที่มีการผลิต</span><span class="sxs-lookup"><span data-stu-id="21154-121">Before a production order is ended, actual data-ttu-id="21154-123">ทุนสินค้าสำเร็จรูปโดยรวมเป็นเดบิตจากบัญชีการรับสินค้าคงคลังและเครดิตสำหรับบัญชีสำหรับการออกสินค้า</span><span class="sxs-lookup"><span data-stu-id="21154-123">The overall cost **การสิ้นสุดงาน** เมื่อคุณดำเนินการคำนวณต้นทุน สถานะใบสั่งผลิตจะเปลี่ยนเป็น **สิ้นสุดแล้ว**</span><span class="sxs-lookup"><span data-stu-id="21154-124">If you select the **End job** check box when ## <a name="controlling-the-level-of-ledger-posting"></a><span data-ttu-id="21154-128">การควบคุมระดับของการลงรายการบัญชีแยกประเภท</span><span class="sxs-lookup"><span data-stu-id="21154-128">Controlling the level of ledger posting</span></span>

-   <span data-ttu-id="21154-131">**สินค้าและทรัพยากร**– ใช้บัญชีแยกประเภทที่มีตั้งค่ากลุ่มสินค้าสำหรับวัตถุดิบและสินค้าสำเร็จรูป</span><span class="sxs-lookup"><span data-stu-id="21154-131">**Item and resource** – Use the ledger accounts that are set up on the item groups for raw materials and finished goods.</span></span> <span data-ttu-id="21154-132">งานระหว่างการดำเนินการทำสำหรับการใช้เวลาจะได้มาจากทรัพยากรหรือกลุ่มทรัพยากรจากกระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="21154-132">WIP for time consumption is taken from resource or resource group from the route operations.</span></span>
-   <span data-ttu-id="21154-133">**สินค้าและทรัพยากร**– ใช้บัญชีแยกประเภทที่มีตั้งค่ากลุ่มสินค้าสำหรับวัตถุดิบและสินค้าสำเร็จรูป</span><span class="sxs-lookup"><span data-stu-id="21154-133">**Item and category** – Use the ledger accounts that are set up on the item groups for raw materials and finished goods.</span></span> <span data-ttu-id="21154-134">งานระหว่างการดำเนินการทำสำหรับการใช้เวลาจะได้มาจากประเภทต้นทุนที่เกี่ยวข้องกับกระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="21154-134">WIP for time consumption is taken from the cost categories that are associated with the route operations.</span></span>
-   <span data-ttu-id="21154-135">**กลุ่มการผลิต**– ใช้บัญชีแยกประเภทที่ตั้งค่าบนกลุ่มการผลิตสำหรับปริมาณการใช้วัสดุ และการใช้เวลา</span><span class="sxs-lookup"><span data-stu-id="21154-135">**Production groups** – Use the ledger accounts that are set up on the production groups for both material and time consumption.</span></span> <span data-ttu-id="21154-136">กลุ่มการผลิตจะเชื่อมโยงกับผลิตภัณฑ์นำออกใช้ และคัดลอกไปยังใบสั่งผลิตเมื่อสร้างใบสั่งเหล่านั้น</span><span class="sxs-lookup"><span data-stu-id="21154-136">The production groups are associated with the released products and copied to the production orders when those orders are created.</span></span> <span data-ttu-id="21154-137">การลงรายการบัญชีในใบสั่งผลิตจะทำตามกลุ่มการผลิตที่เชื่อมโยงกับใบสั่งผลิต</span><span class="sxs-lookup"><span data-stu-id="21154-137">The posting on the production orders will then follow the production groups that are associated with the production order.</span></span>
