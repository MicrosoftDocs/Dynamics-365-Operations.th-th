<span data-ttu-id="4a5fd-104">วันที่ถัวเฉลี่ยถ่วงน้ำหนักคือ แบบจำลองสินค้าคงคลังที่ใช้หลักการถัวเฉลี่ยถ่วงน้ำหนัก</span><span class="sxs-lookup"><span data-stu-id="4a5fd-104">Weighted average date is an inventory model that is based on the weighted average principle.</span></span> <span data-ttu-id="4a5fd-105">สำหรับหลักการการถัวเฉลี่ยถ่วงน้ำหนัก การตัดสินค้าจากคลังจะถูกประเมินค่าที่ค่าเฉลี่ยของสินค้า ซึ่งได้รับเข้าในคลังสินค้าสำหรับแต่ละวันในรอบระยะเวลาการปิดบัญชีสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="4a5fd-105">For the weighted average principle, issues from inventory are valued at the average value of the items that are received into inventory for each day in the inventory closing period.</span></span> 

<span data-ttu-id="4a5fd-106">เมื่อคุณดำเนินการปิดบัญชีสินค้าคงคลังโดยใช้วันที่ถัวเฉลี่ยถ่วงน้ำหนัก การรับสินค้าประจำวันทั้งหมดจะจับคู่กับการนำสินค้าออกใช้เสมือน</span><span class="sxs-lookup"><span data-stu-id="4a5fd-106">When you run an inventory closing by using weighted average date, all daily receipts are settled against a virtual issue.</span></span> <span data-ttu-id="4a5fd-107">การนำสินค้าออกใช้เสมือนมีผลรวมปริมาณที่ได้รับและมูลค่าสำหรับวันนั้น</span><span class="sxs-lookup"><span data-stu-id="4a5fd-107">This virtual issue holds the total received quantity and value for that day.</span></span> <span data-ttu-id="4a5fd-108">การนำสินค้าออกใช้เสมือนสอดคล้องกับการรับสินค้าเสมือนที่การออกสินค้าจะถูกจับคู่ด้วย</span><span class="sxs-lookup"><span data-stu-id="4a5fd-108">The virtual issue has a corresponding virtual receipt that the issues will be settled against.</span></span> <span data-ttu-id="4a5fd-109">ดังนั้น การตัดสินค้าจากคลังทั้งหมดได้รับต้นทุนเฉลี่ยเดียวกัน</span><span class="sxs-lookup"><span data-stu-id="4a5fd-109">Therefore, all issues receive the same average cost.</span></span> <span data-ttu-id="4a5fd-110">การนำสินค้าออกใช้เสมือนและการรับสินค้าเสมือนอาจถือว่าเป็นการโอนย้ายเสมือนที่เรียกว่าการ *การโอนย้ายการปิดบัญชีสินค้าคงคลังถัวเฉลี่ยถ่วงน้ำหนัก*</span><span class="sxs-lookup"><span data-stu-id="4a5fd-110">The virtual issue and virtual receipt can be considered a virtual transfer that is known as the *weighted average inventory closing transfer*.</span></span> 

<span data-ttu-id="4a5fd-111">ถ้ามีการรับสินค้าเพียงหนึ่งรายการเกิดขึ้น หรือก่อนวันที่ คุณไม่จำเป็นต้องประเมินค่าเฉลี่ย</span><span class="sxs-lookup"><span data-stu-id="4a5fd-111">If only one receipt has occurred on or before the date, you don't have to value the average.</span></span> <span data-ttu-id="4a5fd-112">เนื่องจากการนำออกใช้ทั้งหมดจะถูกจับคู่กับการรับ การโอนย้ายเสมือนจะไม่สามารถถูกสร้างได้</span><span class="sxs-lookup"><span data-stu-id="4a5fd-112">Because all issues are settled from that receipt, the virtual transfer won't be created.</span></span> <span data-ttu-id="4a5fd-113">ในทำนองเดียวกัน ถ้ามีเพียงการนำออกใช้เกิดขึ้นในวันที่ จะไม่มีการประเมินค่าเฉลี่ยจากการรับสินค้า และจะไม่สามารถสร้างการโอนย้ายเสมือนได้</span><span class="sxs-lookup"><span data-stu-id="4a5fd-113">Likewise, if only issues occur on the date, there are no receipts to value the average from, and the virtual transfer won't be created.</span></span> <span data-ttu-id="4a5fd-114">เมื่อใช้วันที่ถัวเฉลี่ยถ่วงน้ำหนัก คุณสามารถทำเครื่องหมายธุรกรรมสินค้าคงคลังได้ เพื่อให้มีการจับคู่การรับสินค้าที่ระบุกับการนำสินค้าออกใช้ที่ระบุ</span><span class="sxs-lookup"><span data-stu-id="4a5fd-114">When you use weighted average date, you can mark inventory transactions so that a specific item receipt is settled against a specific issue.</span></span> <span data-ttu-id="4a5fd-115">ในกรณีนี้ ไม่มีการใช้กฎวันที่ถัวเฉลี่ยถ่วงน้ำหนัก</span><span class="sxs-lookup"><span data-stu-id="4a5fd-115">In this case, you don't use the weighted average date rule.</span></span> <span data-ttu-id="4a5fd-116">ขอแนะนำให้ปิดบัญชีสินค้าคงคลังเป็นรายเดือน เมื่อคุณใช้แบบจำลองสินค้าคงคลังวันที่ถัวเฉลี่ยถ่วงน้ำหนัก</span><span class="sxs-lookup"><span data-stu-id="4a5fd-116">We recommend a monthly inventory closing when you use the weighted average date inventory model.</span></span> 

<span data-ttu-id="4a5fd-117">สูตรต่อไปนี้จะใช้ในการคำนวณวิธีการคิดต้นทุนของวันที่ค่าเฉลี่ยถ่วงน้ำหนัก:</span><span class="sxs-lookup"><span data-stu-id="4a5fd-117">The following formula is used to calculate the weighted average date costing method:</span></span> 

<span data-ttu-id="4a5fd-118">ค่าเฉลี่ยถ่วงน้ำหนัก = (\[Q1 × P1\] + \[Q2 × P2\] + \[Q*n* × P*n*\]) ÷ (Q1 + Q2 + Q*n*)</span><span class="sxs-lookup"><span data-stu-id="4a5fd-118">Weighted average = (\[Q1 × P1\] + \[Q2 × P2\] + \[Q*n* × P*n*\]) ÷ (Q1 + Q2 + Q*n*)</span></span>
