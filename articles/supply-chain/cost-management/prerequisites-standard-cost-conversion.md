# <a name="prerequisites-for-a-standard-cost-conversion"></a><span data-ttu-id="4fcd9-103">ข้อกำหนดเบื้องต้นสำหรับการแปลงต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-103">Prerequisites for a standard cost conversion</span></span>

[!INCLUDE [banner](../includes/banner.md)]

<span data-ttu-id="4fcd9-104">หัวข้อนี้อธิบายงานการเพื่อดำเนินการก่อนที่คุณจะดำเนินการแปลงต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-104">This topic discusses tasks to perform before you run a standard cost conversion.</span></span> 

<span data-ttu-id="4fcd9-105">ก่อนที่คุณดำเนินการแปลงต้นทุนมาตรฐาน ให้ทำตามขั้นตอนเหล่านี้:</span><span class="sxs-lookup"><span data-stu-id="4fcd9-105">Before you run a standard cost conversion, follow these steps:</span></span>

1.  <span data-ttu-id="4fcd9-106">กำหนด **กลุ่มแบบจำลองสินค้า** สำหรับต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-106">Define an **item model group** for standard costs.</span></span> <span data-ttu-id="4fcd9-107">ใช้หน้ากลุ่มแบบจำลองสินค้าเพื่อสร้างกลุ่มแบบจำลองสินค้าที่ใช้แบบจำลองสินค้าคงคลังต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-107">Use the Item model groups page to create an item model group that uses a standard cost inventory model.</span></span> <span data-ttu-id="4fcd9-108">เมื่อมีการตั้งค่าการแปลงต้นทุนมาตรฐาน คุณกำหนดกลุ่มแบบจำลองสินค้านี้ให้กับสินค้าที่จะถูกแปลง</span><span class="sxs-lookup"><span data-stu-id="4fcd9-108">When setting up a standard cost conversion, you assign this item model group to the items that are being converted.</span></span>
2.  <span data-ttu-id="4fcd9-109">กำหนด **กลุ่มต้นทุน** ให้กับสินค้าแต่ละรายการ</span><span class="sxs-lookup"><span data-stu-id="4fcd9-109">Assign a **cost group** to each item.</span></span>
    -   <span data-ttu-id="4fcd9-110">กลุ่มต้นทุนที่ถูกกำหนดให้กับสินค้าที่ถูกซื้อ สามารถทำหน้าที่เป็นข้อมูลพื้นฐานสำหรับการกำหนดบัญชีแยกประเภทที่เกี่ยวข้องกับต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-110">The cost group that is assigned to a purchased item can act as the basis for assigning ledger accounts that are related to standard costs.</span></span> <span data-ttu-id="4fcd9-111">ซึ่งอาจรวมถึงการกำหนดบัญชีแยกประเภทสำหรับผลต่างราคาซื้อ</span><span class="sxs-lookup"><span data-stu-id="4fcd9-111">This can include assigning ledger accounts for purchase price variances.</span></span> <span data-ttu-id="4fcd9-112">ในสภาพแวดล้อมในการผลิต กลุ่มต้นทุนที่ถูกกำหนดให้กับส่วนประกอยที่ถูกซื้อยังให้การแบ่งส่วนต้นทุนในต้นทุนที่ได้รับการคำนวณของสินค้าที่ผลิตอีกด้วย</span><span class="sxs-lookup"><span data-stu-id="4fcd9-112">In a manufacturing environment, the cost group that is assigned to purchased components also provides cost segmentation in the calculated costs of a manufactured item.</span></span>
    -   <span data-ttu-id="4fcd9-113">กลุ่มต้นทุนที่กำหนดให้กับสินค้าที่ผลิตสามารถให้ข้อมูลพื้นฐาน สำหรับการกำหนดบัญชีแยกประเภทที่เกี่ยวข้องกับต้นทุนมาตรฐาน เช่น การกำหนดบัญชีแยกประเภทสำหรับผลต่างการผลิต</span><span class="sxs-lookup"><span data-stu-id="4fcd9-113">The cost group that is assigned to a manufactured item can act as the basis for assigning ledger accounts that are related to standard costs, such as assigning ledger accounts for production variances.</span></span>

3.  <span data-ttu-id="4fcd9-114">กำหนด **ปริมาณการสั่งมาตรฐาน** ให้กับสินค้าที่ผลิตเมื่อมีต้นทุนคงที่</span><span class="sxs-lookup"><span data-stu-id="4fcd9-114">Assign a **standard order quantity** to a manufactured item when it has constant costs.</span></span> <span data-ttu-id="4fcd9-115">ปริมาณการสั่งมาตรฐานสำหรับสินค้าที่ผลิตทำหน้าที่เป็นขนาดล็อตบัญชีสำหรับการตัดจำหน่าย หรือการคิดตามสัดส่วน ต้นทุนคงที่</span><span class="sxs-lookup"><span data-stu-id="4fcd9-115">The standard order quantity for a manufactured item acts as an accounting lot size for amortizing, or prorating, constant costs.</span></span> <span data-ttu-id="4fcd9-116">ซึ่งสามารถมีการตั้งค่าเวลาในการดำเนินงานกระบวนการผลิตหรือปริมาณส่วนประกอบคงที่ในสูตรการผลิต (BOM)</span><span class="sxs-lookup"><span data-stu-id="4fcd9-116">These can include setup times in routing operations or a constant component quantity in a bill of material (BOM).</span></span>
4.  <span data-ttu-id="4fcd9-117">กำหนด **บัญชีแยกประเภททั่วไป** ที่เกี่ยวข้องกับต้นทุนมาตรฐาน โดยเฉพาะผลต่างการประเมินค่าใหม่</span><span class="sxs-lookup"><span data-stu-id="4fcd9-117">Assign **general ledger accounts** that are related to standard costs, especially the revaluation variance.</span></span> <span data-ttu-id="4fcd9-118">ใช้หน้า **การลงรายการบัญชี** (**การบริหารสินค้าคงคลัง** &gt; **การตั้งค่า**) เพื่อกำหนดบัญชีแยกประเภททั่วไปที่เกี่ยวข้องกับต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-118">Use the **Posting** page (**Inventory management** &gt; **Setup**) to assign general ledger accounts that are related to standard costs.</span></span> <span data-ttu-id="4fcd9-119">ตามข้อกำหนดขั้นต่ำสำหรับกระบวนการแปลงต้นทุนมาตรฐาน คุณต้องกำหนดบัญชีสำหรับผลต่างการประเมินค่าใหม่สำหรับสินค้าทั้งหมดและกลุ่มต้นทุนทั้งหมด</span><span class="sxs-lookup"><span data-stu-id="4fcd9-119">As a minimum for the standard cost conversion process, you must assign the account for the revaluation variance for all items and all cost groups.</span></span> <span data-ttu-id="4fcd9-120">ใช้หน้า **ผังบัญชี** เพื่อกำหนดบัญชีแยกประเภททั่วไปที่จำเป็นสำหรับต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-120">Use the **Chart of accounts** page to define the general ledger accounts that will be needed for standard costs.</span></span> <span data-ttu-id="4fcd9-121">ใช้หน้า **การรวมของธุรกรรมต่างๆ** เพื่อให้สามารถใช้ความสัมพันธ์ของต้นทุน (สำหรับตาราง กลุ่ม และทั้งหมด) ก่อนที่คุณจะกำหนดกฎการลงรายการบัญชีสินค้า</span><span class="sxs-lookup"><span data-stu-id="4fcd9-121">Use the **Transaction combinations** page to enable cost relations (for tables, groups, and all) before you define the item posting rules.</span></span>
5.  <span data-ttu-id="4fcd9-122">กำหนดพารามิเตอร์สินค้าคงคลังที่เกี่ยวข้องกับต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-122">Define the inventory parameters that are related to standard costs.</span></span> <span data-ttu-id="4fcd9-123">ใช้แท็บ **ลำดับหมายเลข** ในหน้า **พารามิเตอร์การบริหารสินค้าคงคลังและคลังสินค้า** เพื่อกำหนดลำดับหมายเลขสำหรับใบสำคัญการประเมินค่าใหม่</span><span class="sxs-lookup"><span data-stu-id="4fcd9-123">Use the **Number sequences** tab on the **Inventory and warehouse management parameters** page to assign a number sequence to revaluation vouchers.</span></span> <span data-ttu-id="4fcd9-124">ใบสำคัญการประเมินค่าใหม่จะถูกสร้างขึ้นเมื่อการแปลงต้นทุนมาตรฐานสร้างการปลี่ยนแปลงในมูลค่าสินค้าคงคลังของสินค้า</span><span class="sxs-lookup"><span data-stu-id="4fcd9-124">A revaluation voucher is generated when the standard cost conversion creates a change of an item's inventory value.</span></span> <span data-ttu-id="4fcd9-125">ใช้หน้า **พารามิเตอร์การจัดการสินค้าคงคลังและคลังสินค้า** เพื่อกำหนดพารามิเตอร์การควบคุมต้นทุน (บนแท็บ **การบัญชีสินค้าคงคลัง** ) เพื่อกำหนดพารามิเตอร์สองตัวที่เกี่ยวข้องกับต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-125">Use the **Inventory and warehouse management parameters** page to define Cost Control parameters (on the **Inventory accounting** tab) to define two parameters that are related to standard costs.</span></span>
    -   <span data-ttu-id="4fcd9-126">ใช้ฟิลด์ **การแบ่งต้นทุน** เพื่อเลือก ไม่ หรือบัญชีแยกประเภทย่อย</span><span class="sxs-lookup"><span data-stu-id="4fcd9-126">Use the **Cost breakdown** field to select No or Sub ledger.</span></span> <span data-ttu-id="4fcd9-127">การเลือกบัญชีแยกประเภทย่อยจะเรียกว่าการแบ่งต้นทุนที่ใช้งานอยู่</span><span class="sxs-lookup"><span data-stu-id="4fcd9-127">The selection of Sub ledger is termed an active cost breakdown.</span></span> <span data-ttu-id="4fcd9-128">การแบ่งต้นทุนที่ใช้งานอยู่มีความสำคัญมากต่อการคำนวณ การเก็บรักษา และการดูการแบ่งกลุ่มต้นทุนระหว่างโครงสร้างผลิตภัณฑ์หลายระดับสำหรับสินค้าต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-128">An active cost breakdown is very important for calculating, retaining, and viewing cost group segmentation across a multilevel product structure for standard cost items.</span></span> <span data-ttu-id="4fcd9-129">เมื่อเปิดใช้งานการแบ่งต้นทุน คุณสามารถรายงานและวิเคราะห์งานต่อไปนี้ในระดับเดียว หลายระดับ หรือรูปแบบรวม:</span><span class="sxs-lookup"><span data-stu-id="4fcd9-129">When the cost breakdown is active, you can report and analyze the following in a single level, multi-level, or total format:</span></span>
        1.  <span data-ttu-id="4fcd9-130">สินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="4fcd9-130">Inventory</span></span>
        2.  <span data-ttu-id="4fcd9-131">งานระหว่างทำ (WIP)</span><span class="sxs-lookup"><span data-stu-id="4fcd9-131">Work in process (WIP)</span></span>
        3.  <span data-ttu-id="4fcd9-132">ต้นทุนขาย (COGS) ต่อกลุ่มต้นทุน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-132">Cost of goods sold (COGS) per cost group</span></span>

        <span data-ttu-id="4fcd9-133">การแบ่งต้นทุนที่ใช้งานอยู่หมายความว่า ถ้าคุณเปิดใช้งานต้นทุนของสินค้าที่ผลิต ผลลัพธ์จะถูกเก็บไว้ในการแบ่งกลุ่มต้นทุนในเรกคอร์ดต้นทุนของสินค้า</span><span class="sxs-lookup"><span data-stu-id="4fcd9-133">An active cost breakdown means that if you enable a manufactured item's cost, the result will be stored in the cost group segmentation in the item's cost record.</span></span> <span data-ttu-id="4fcd9-134">ถ้าคุณไม่ใส่ค่าในฟิลด์ **การแบ่งต้นทุน** การแบ่งกลุ่มต้นทุนจะไม่ถูกรักษาไว้สำหรับสินค้าต้นทุนมาตรฐาน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-134">If you put no value in the **Cost breakdown** field, the cost group segmentation will not be maintained for standard cost items.</span></span> <span data-ttu-id="4fcd9-135">กล่าวคือ ต้นทุนมาตรฐานของสินค้าที่ผลิตจะถูกคำนวณและเก็บรักษาไว้เป็นยอดเงินเดียวโดยไม่มีการจัดเซกเมนต์กลุ่มต้นทุน และการจัดสรรต้นทุนของส่วนประกอบในการผลิตจะถูกรวมเป็นยอดเดียว</span><span class="sxs-lookup"><span data-stu-id="4fcd9-135">That is, a manufactured item's standard cost will be calculated and maintained as a single amount without cost group segmentation, and the cost contributions of manufactured components will be aggregated into the single amount.</span></span>
    -   <span data-ttu-id="4fcd9-136">ใช้ฟิลด์ **ผลต่างจากมาตรฐาน** เพื่อเลือกสรุปหรือต่อกลุ่มต้นทุน</span><span class="sxs-lookup"><span data-stu-id="4fcd9-136">Use the **Variances to standard**
