# <a name="view-cost-entries-for-a-cost-object"></a>ดูรายการข้อมูลต้นทุนสำหรับออบเจ็กต์ต้นทุน

[!include [task guide banner](../../includes/task-guide-banner.md)]

กระบวนการนี้แสดงวิธีการดูรายการต้นทุนสำหรับราคาทุนของสิ่งของ  ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF กระบวนการนี้มีไว้สำหรับผู้ควบคุมต้นทุน

1. คลิกการบริหารต้นทุน
2. คลิกผลิตภัณฑ์ต่างๆที่นำออกใช้
3. ใช้ตัวกรองข้อมูลด่วนเพื่อค้นหาเรกคอร์ด  ตัวอย่างเช่น ใช้ตัวกรองฟิลด์หมายเลขสินค้าด้วยค่า 'm0004'
4. ในบานหน้าต่างการดำเนินการ คลิกการจัดการต้นทุน
5. คลิกวัตถุต้นทุน
6. คลิกรายการต้นทุน
7. ใช้ตัวกรองด่วนเพื่อกรองข้อมูลในฟิลด์หมายเลขด้วยค่า 'p000031'
    * ถ้าหากรายการต้นทุนไม่ปรากฏ ตั้งค่าวันเริ่มต้นวันที่ 31 มกราคม 2012 และวันที่สิ้นสุดเป็นวันที่ 31 ธันวาคม 2012
