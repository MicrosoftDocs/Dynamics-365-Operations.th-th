# <a name="view-inventory-statement-and-kpi-by-cost-object"></a>ดูใบแจ้งยอดของสินค้าคงคลังและ KPI ตามออบเจ็กต์ต้นทุน

[!include [task guide banner](../../includes/task-guide-banner.md)]

ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF กระบวนการนี้มีไว้สำหรับผู้ควบคุมต้นทุน

1. คลิกการบริหารต้นทุน
2. คลิกผลิตภัณฑ์ต่างๆที่นำออกใช้
3. ใช้ตัวกรองข้อมูลด่วนเพื่อค้นหาเรกคอร์ด  ตัวอย่างเช่น ใช้ตัวกรองบนฟิลด์หมายเลขสินค้าด้วยค่า 'd0005'
4. ในบานหน้าต่างการดำเนินการ คลิกการจัดการต้นทุน
5. คลิกวัตถุต้นทุน
6. คลิกใบแจ้งยอดของสินค้าคงคลังและ KPI
7. ในฟิลด์วันที่เริ่มต้น ให้ป้อนวันที่
8. ในฟิลด์วันที่สิ้นสุด ให้ป้อนวันที่
