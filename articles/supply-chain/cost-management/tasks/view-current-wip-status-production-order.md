# <a name="view-current-wip-status-on-a-production-order"></a>ดูสถานะปัจจุบันของ WIP ในใบสั่งผลิต

[!include [task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงวิธีการดูรายงานWIPบนใบสั่งผลิต  ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF กระบวนการนี้มีไว้สำหรับผู้ควบคุมต้นทุน

1. คลิกการบริหารต้นทุน
2. คลิกใบสั่งผลิต
3. ใช้ตัวกรองด่วนเพื่อกรองข้อมูลในฟิลด์การผลิต ด้วยค่า 'p000153'
4. ในบานหน้าต่างการดำเนินการ คลิกการจัดการต้นทุน
5. คลิกรายงาน WIP สำหรับการผลิต
6. ในฟิลด์วันที่เริ่มต้น ตั้งค่าวันที่เป็น ' 2012-12-01'
7. ในฟิลด์วันที่สิ้นสุด ตั้งค่าวันที่เป็น ' 2012-12-31'
