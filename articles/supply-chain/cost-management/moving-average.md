# <a name="moving-average"></a><span data-ttu-id="c11e8-105">ค่าเฉลี่ยเคลื่อนที่</span><span class="sxs-lookup"><span data-stu-id="c11e8-105">Moving average</span></span>

[!include[banner](../includes/banner.md)]

<span data-ttu-id="c11e8-106">ค่าเฉลี่ยเคลื่อนที่เป็นวิธีการคิดต้นทุนแบบไม่จำกัดเวลาที่ยึดตามหลักการถัวเฉลี่ย ที่ต้นทุนสำหรับการตัดสินค้าจากคลังของสินค้าคงคลังไม่เปลี่ยน เมื่อต้นทุนการซื้อเปลี่ยน</span><span class="sxs-lookup"><span data-stu-id="c11e8-106">Moving average is a perpetual costing method based on the average principle, where the costs on inventory issues do not change when the purchase cost does.</span></span> <span data-ttu-id="c11e8-107">ผลต่างถูกบันทึก และจะขึ้นอยู่กับการคำนวณตามสัดส่วน</span><span class="sxs-lookup"><span data-stu-id="c11e8-107">The difference is capitalized and is based on a proportional calculation.</span></span> <span data-ttu-id="c11e8-108">จำนวนที่คงเหลือถูกใช้</span><span class="sxs-lookup"><span data-stu-id="c11e8-108">The amount that remains is expensed.</span></span> 

<span data-ttu-id="c11e8-109">เมื่อคุณใช้ค่าเฉลี่ยเคลื่อนที่ การจับคู่สินค้าคงคลัง และทำเครื่องหมายสินค้าคงคลัง จะไม่ได้รับการสนับสนุน</span><span class="sxs-lookup"><span data-stu-id="c11e8-109">When you use moving average, inventory settlements and inventory marking are not supported.</span></span> <span data-ttu-id="c11e8-110">การปิดบัญชีสินค้าคงคลังไม่มีผลกับผลิตภัณฑ์ที่มีค่าเฉลี่ยเคลื่อนที่เป็นกลุ่มแบบจำลองสินค้าคงคลัง และไม่สร้างการจับคู่ใดๆ ระหว่างธุรกรรม</span><span class="sxs-lookup"><span data-stu-id="c11e8-110">Inventory close does not affect products that have moving average as the inventory model group, and it does not generate any settlements between the transactions.</span></span>

<span data-ttu-id="c11e8-111">ต่อไปนี้เป็นข้อกำหนดเบื้องต้นเมื่อคุณใช้ต้นทุนเฉลี่ยเคลื่อนที่เป็นวิธีการคิดต้นทุน</span><span class="sxs-lookup"><span data-stu-id="c11e8-111">The following are prerequisites when you use moving average cost as a costing method.</span></span>

1.  <span data-ttu-id="c11e8-112">ในหน้า **กลุ่มแบบจำลองสินค้า** ให้ตั้งค่ากลุ่มแบบจำลองสินค้าที่มีค่าเฉลี่ยเคลื่อนที่ที่ได้เลือกไว้ในฟิลด์ **แบบจำลองสินค้าคงคลัง**</span><span class="sxs-lookup"><span data-stu-id="c11e8-112">In the **Item model groups** page, set up an item model group that has Moving average selected in the **Inventory model** field.</span></span> <span data-ttu-id="c11e8-113">**หมายเหตุ:** โดยค่าเริ่มต้น เมื่อมีการเลือกค่าเฉลี่ยเคลื่อนที่ ฟิลด์ **ลงรายการบัญชีสินค้าคงคลังทางกายภาพ** และ **ลงรายการบัญชีสินค้าคงคลังทางการเงิน** จะถูกเลือกด้วย</span><span class="sxs-lookup"><span data-stu-id="c11e8-113">**Note:** By default, when Moving average is selected, the **Post physical inventory** and **Post financial inventory** fields are also selected.</span></span> 

2.  <span data-ttu-id="c11e8-114">ในหน้า **การลงรายการบัญชี** กำหนดบัญชีให้กับ **ผลต่างราคาสำหรับค่าเฉลี่ยเคลื่อนที่** และบัญชี **การประเมินค่าต้นทุนสำหรับค่าเฉลี่ยเคลื่อนที่**บนแท็บ **สินค้าคงคลัง** คุณใช้บัญชี **ผลต่างราคาสำหรับค่าเฉลี่ยเคลื่อนที่** เมื่อต้นทุนต้องเป็นการใช้จ่ายตามสัดส่วน</span><span class="sxs-lookup"><span data-stu-id="c11e8-114">In the **Posting** page, assign accounts to the **Price difference for moving average** and the **Cost revaluation for moving average** accounts on the **Inventory** tab. 3.  <span data-ttu-id="c11e8-117">ในหน้า **ผลิตภัณฑ์ที่นำออกใช้** กำหนดกลุ่มแบบจำลองสินค้าค่าเฉลี่ยเคลื่อนที่ให้กับงผลิตภัณฑ์</span><span class="sxs-lookup"><span data-stu-id="c11e8-117">In the **Released products** page, assign the moving average item model group to the product.</span></span> <span data-ttu-id="c11e8-118">**หมายเหตุ:** กระบวนการปิดบัญชีสินค้าคงคลังปิดเฉพาะรอบระยะเวลาทางบัญชีเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="c11e8-118">**Note:** The inventory close process only closes the accounting period.</span></span> <span data-ttu-id="c11e8-119">จะไม่มีผลกับผลิตภัณฑ์ที่มีค่าเฉลี่ยเคลื่อนที่ที่มีการกำหนดให้เป็นกลุ่มแบบจำลองสินค้า</span><span class="sxs-lookup"><span data-stu-id="c11e8-119">It does not affect products that have moving average assigned to them as an item model group.</span></span>

## <a name="convert-to-the-moving-average-costing-method"></a><span data-ttu-id="c11e8-120">วิธีแปลงเป็นการคิดต้นทุนค่าเฉลี่ยเคลื่อนที่</span><span class="sxs-lookup"><span data-stu-id="c11e8-120">Convert to the moving average costing method</span></span>
<span data-ttu-id="c11e8-121">สามารถแปลงผลิตภัณฑ์ที่จะใช้วิธีการประเมินค่าสินค้าคงคลังค่าเฉลี่ยเคลื่อนที่</span><span class="sxs-lookup"><span data-stu-id="c11e8-121">Products can be converted to use the moving average inventory valuation method.</span></span> <span data-ttu-id="c11e8-122">โดยปกติจะทำการแปลงชนิดนี้ ณ สิ้นปี หลังจากมีการปิดเดือนสุดท้ายของปีปัจจุบัน</span><span class="sxs-lookup"><span data-stu-id="c11e8-122">This type of conversion is usually done at the end of the year, after the last month of the current year is closed.</span></span> <span data-ttu-id="c11e8-123">มีการดำเนินงานให้เสร็จสิ้นโดยใช้แบบจำลองการคำนวณต้นทุนปัจจุบันของผลิตภัณฑ์</span><span class="sxs-lookup"><span data-stu-id="c11e8-123">It is done by using the product’s current costing model.</span></span> <span data-ttu-id="c11e8-124">คุณสามารถเปลี่ยนวิธีการคิดต้นทุนสินค้าคงคลังของคุณ จากวิธีการคำนวณต้นทุนที่ขึ้นอยู่กับต้นทุนเฉลี่ยหรือต้นทุนมาตรฐานไปเป็นวิธีการที่เป็นไปตามค่าเฉลี่ยเคลื่อนที่</span><span class="sxs-lookup"><span data-stu-id="c11e8-124">You can change your inventory costing method from a costing method that is based on average cost or standard cost to a method that is based on moving average.</span></span> 

<span data-ttu-id="c11e8-125">ถ้าคุณกำลังเปลี่ยนแปลงวิธีการคิดต้นทุนจากวิธีการคำนวณต้นทุนมาตรฐานเป็นวิธีค่าเฉลี่ยเคลื่อนที่ คุณจะต้องทำภารกิจต่อไปนี้ให้เสร็จสมบูรณ์:</span><span class="sxs-lookup"><span data-stu-id="c11e8-125">If you are changing your costing method from a standard costing method to a moving average method, you have to complete the following tasks:</span></span>

1.  <span data-ttu-id="c11e8-126">ทำการปรับปรุงให้ได้ปริมาณสินค้าคงคลังและค่าลงมาเป็น 0 (ศูนย์)</span><span class="sxs-lookup"><span data-stu-id="c11e8-126">Make adjustments to get inventory quantities and values down to 0 (zero).</span></span>
2.  <span data-ttu-id="c11e8-127">หลังจากมูลค่าสินค้าคงคลังและปริมาณเป็น 0 (ศูนย์) ให้เปลี่ยนกลุ่มแบบจำลองสินค้าเป็นค่าเฉลี่ยเคลื่อนที่</span><span class="sxs-lookup"><span data-stu-id="c11e8-127">After the inventory value and quantity are 0 (zero), change the item model group to moving average.</span></span>

<span data-ttu-id="c11e8-129">คุณไม่สามารถเปลี่ยนวิธีการคิดต้นทุนสินค้าคงคลังของคุณจากวิธีค่าเฉลี่ยเคลื่อนที่ไปเป็นวิธีเข้าก่อนออกก่อน (FIFO) วิธีเข้าหลังออกก่อน (LIFO) หรือวิธีค่าเฉลี่ยถ่วงน้ำหนัก</span><span class="sxs-lookup"><span data-stu-id="c11e8-129">You cannot change your inventory costing method from a moving average method to a First in, First out (FIFO) method, a Last in, First out (LIFO) method, or a weighted average method.</span></span>

<span data-ttu-id="c11e8-131">ตัวอย่างต่อไปนี้แสดงผลของการใช้วิธีการคิดต้นทุนค่าเฉลี่ยเคลื่อนที่</span><span class="sxs-lookup"><span data-stu-id="c11e8-131">The following examples illustrate the effect of using the moving average costing method.</span></span> <span data-ttu-id="c11e8-132">มีการตั้งค่าคอนฟิกสี่แบบ:</span><span class="sxs-lookup"><span data-stu-id="c11e8-132">There are four configurations:</span></span>
