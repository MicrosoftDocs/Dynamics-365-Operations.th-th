# <a name="count-inventory-in-a-warehouse"></a>นับจำนวนสินค้าคงคลังในคลังสินค้า

[!include [task guide banner](../../includes/task-guide-banner.md)]

ขั้นตอนนี้นำคุณไปสู่กระบวนการของการสร้างและการลงรายการบัญชีสมุดรายวันการตรวจนับสินค้าคงคลังในใบสั่งที่นับสินค้าเฉพาะที่สถานที่ในคลังสินค้า  กระบวนงานดังกล่าวใช้กับฟังก์ชัน "คลังสินค้าพื้นฐาน" ซึ่งอยู่ในโมดูลการจัดการสินค้าคงคลัง และไม่ใช้กับฟังก์ชันคลังสินค้าที่อยู่ในโมดูลการจัดการคลังสินค้า คุณสามารถศึกษากระบวนงานนี้ได้ในบริษัทข้อมูลสาธิต USMF หรือใช้ข้อมูลของคุณเอง ถ้าคุณกำลังใช้ข้อมูลของคุณเอง โปรดตรวจสอบให้แน่ใจว่าคุณได้ตั้งค่าผลิตภัณฑ์และสถานที่แล้ว และคุณได้สร้างชื่อสมุดรายวันสินค้าคงคลังสำหรับสมุดรายวันการตรวจนับแล้ว การนับสินค้าคงคลังจะดำเนินการปกติโดยพนักงานคลังสินค้า


## <a name="create-an-inventory-counting-journal"></a>สร้างสมุดรายวันการตรวจนับสินค้าคงคลัง
1. ไปที่ การบริหารสินค้าคงคลัง > รายการสมุดรายวัน > การนับสินค้า > การนับ
2. คลิก สร้าง
3. ในฟิลด์ชื่อ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ในรายการ คลิกชื่อสมุดรายวันการตรวจนับสินค้าคงคลังที่คุณต้องการใช้
    * บางฟิลด์อื่นๆ จะถูกเติมข้อมูลตามการตั้งค่าของชื่อสมุดรายวันการตรวจนับสินค้าคงคลังที่คุณเลือก  
5. ในฟิลด์ผู้ปฏิบัติงาน ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
6. ในรายการ ให้เลือกผู้ปฏิบัติงานที่คุณต้องการใช้
7. คลิกเลือก 
8. คลิก ตกลง

## <a name="create-journal-lines"></a>สร้างรายการสมุดรายวัน
1. คลิก สร้าง
2. ในฟิลด์หมายเลขสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
3. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * ถ้าคุณกำลังใช้บริษัทข้อมูลสาธิต USMF เลือก 'A0001'  
4. ในฟิลด์ไซต์ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
5. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * ถ้าคุณกำลังใช้บริษัทข้อมูลสาธิต USMF เลือกไซต์ '2'  
6. ในฟิลด์คลังสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
7. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * ถ้าคุณกำลังใช้บริษัทข้อมูลสาธิต USMF เลือกคลังสินค้า '24'  
8. ในฟิลด์ตำแหน่ง ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
9. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * ถ้าคุณกำลังใช้บริษัทข้อมูลสาธิต USMF เลือกสถานที่ 'BULK-001'  
10. ในฟิลด์ตรวจนับ ให้ป้อนตัวเลข
    * ถ้าคุณป้อนหมายเลขการตรวจนับที่แตกต่างกับหมายเลขที่แสดงในฟิลด์คงคลังคงเหลือ ฟิลด์ปริมาณจะถูกอัพเดทเพื่อแสดงความขัดแย้ง  
11. คลิก บันทึก

## <a name="post-the-inventory-counting-journal"></a>ลงรายการบัญชีสมุดรายวันการตรวจนับสินค้าคงคลัง
1. คลิก ลงรายการบัญชี
    * เมื่อคุณลงรายการบัญชีสมุดรายวันการตรวจนับสินค้าคงคลัง ถ้าจำนวนที่ตรวจนับแล้วแตกต่างจากจำนวนที่รายงานในฟิลด์สินค้าคงคลังคงเหลือการรับสินค้าสินค้าคงคลัง หรือตัดสินค้าจากคลังได้ลงถูกรายการบัญชี ระดับสินค้าคงคลังและค่าจะถูกเปลี่ยน และบัญชีแยกประเภทธุรกรรมจะถูกสร้างขึ้น  
2. คลิก ตกลง

## <a name="view-inventory-transactions"></a>ดูธุรกรรมสินค้าคงคลัง
1. คลิกสินค้าคงคลัง
2. คลิกธุรกรรม
    * ที่นี่คุณสามารถดูธุรกรรมที่เกี่ยวข้องใดๆ ซึ่งจะถูกสร้างเมื่อคุณลงรายการบัญชีในสมุดรายวันการตรวจนับสินค้าคงคลัง
