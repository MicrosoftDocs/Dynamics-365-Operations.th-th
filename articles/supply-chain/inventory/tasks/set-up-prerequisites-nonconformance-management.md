# <a name="set-up-prerequisites-for-nonconformance-management"></a>ตั้งค่าข้อกำหนดเบื้องต้นสำหรับการจัดการความไม่สอดคล้องกัน

[!include [task guide banner](../../includes/task-guide-banner.md)]

ใช้หัวข้อนี้เพื่อเปิดใช้งานกระบวนการการจัดการความไม่สอดคล้องกัน ความไม่สอดคล้องกันอธิบายกระบวนงานหรือรายการที่มีปัญหาทางคุณภาพ ที่ข้อมูลคำอธิบายรวมแหล่งที่มาและชนิดของปัญหา กระบวนงานนี้ใช้บริษัทข้อมูลสาธิต USMF โดยทั่วไป กระบวนงานนี้จะถูกดำเนินการโดยผู้จัดการด้านคุณภาพ


## <a name="enable-quality-management-processes-within-the-company"></a>เปิดใช้งานกระบวนการจัดการคุณภาพภายในบริษัท
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > การจัดการสินค้าคงคลัง > การตั้งค่า > พารามิเตอร์สินค้าคงคลังและการจัดการคลังสินค้า**
2. เลือกแท็บ **การจัดการคุณภาพ**
3. เลือก **ใช่** ในฟิลด์ **ใช้การจัดการคุณภาพ** เพื่อเปิดใช้งานกระบวนการจัดการคุณภาพสำหรับบริษัท
4. ในฟิลด์ **อัตราต่อชั่วโมง** ป้อนหมายเลขในสกุลเงินท้องถิ่น อัตราต่อชั่วโมงจะถูกใช้ในการคำนวณต้นทุนสำหรับการดำเนินงานที่เกี่ยวข้องกับความไม่สอดคล้อง และไม่ได้สัมพันธ์กับฟังก์ชันอื่น  
5. เลือก **การตั้งค่ารายงาน** เพื่อกำหนดชนิดบันทึกย่อของรายงานคุณภาพที่จะใช้ในรายงานการจัดการคุณภาพชนิดต่างๆ

## <a name="enable-user-for-nonconformance-processing"></a>เปิดใช้งานผู้ใช้สำหรับการประมวลผลความไม่สอดคล้องกัน
1. ในบานหน้าต่างนำทาง ไปยัง **โมดูล > การดูแลระบบ > ผู้ใช้ > ผู้ใช้** 
2. ใช้ตัวกรองด่วนเพื่อค้นหาผู้ใช้ที่กำลังจะอนุมัติหรือปฏิเสธเรกคอร์ดความไม่สอดคล้องกัน ตัวอย่างเช่น ตัวกรองในฟิลด์ **ชื่อ** ที่มีค่า `Ricardo` เพื่อประมวลผลการอนุมัติของความไม่สอดคล้องกัน ผู้ใช้ที่อนุมัติหรือปฏิเสธความไม่สอดคล้องกันจะต้องมีค่า "ชื่อ" ที่กำหนดอยู่ในหน้า **ผู้ใช้** การใช้บันทึกเอกสาร ผู้ใช้ต้องสามารถเรียกใช้ตัวเลือกผู้ใช้ในการจัดการเอกสาร  
3. ทำเครื่องหมายแถวของเรกคอร์ดที่ต้องการ
4. เลือก **ตัวเลือกผู้ใช้**
5. เลือกแท็บ **การกำหนดลักษณะ**
6. เลือก **ใช่** ในฟิลด์ **เปิดใช้งานการจัดการเอกสาร**

## <a name="define-diagnostic-types-for-nonconformance-processing"></a>กำหนดชนิดของการวินิจฉัย สำหรับการประมวลผลความไม่สอดคล้องกัน
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > การจัดการสินค้าคงคลัง > การตั้งค่า > การจัดการคุณภาพ > ชนิดการวินิจฉัย** ใช้หน้า **ชนิดการวินิจฉัย** เพื่อกำหนดการจัดประเภทการดำเนินการวินิจฉัย การแก้ไขระบุชนิดของการดำเนินการวินิจฉัย ซึ่งควรจะดำเนินการกับความไม่สอดคล้องกันที่ได้รับอนุมัติแล้ว บุคคลที่ควรจะเป็นผู้ดำเนินการ และวันที่เสร็จสมบูรณ์ที่ร้องขอและที่วางแผนไว้  
2. เลือก **ใหม่**
3. ในฟิลด์ **การวินิจฉัย** ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ **คำอธิบาย** ให้พิมพ์ค่าใดค่าหนึ่ง

## <a name="define-quality-charges-for-nonconformance-processing"></a>กำหนดค่าธรรมเนียมคุณภาพ สำหรับการประมวลผลความไม่สอดคล้องกัน
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > การจัดการสินค้าคงคลัง > การตั้งค่า > การจัดการคุณภาพ > ค่าธรรมเนียมเกี่ยวกับคุณภาพ** ใช้หน้า **ค่าธรรมเนียมเกี่ยวกับคุณภาพ** เพื่อกำหนดการจัดประเภทของค่าธรรมเนียมที่จะถูกใช้ในการดำเนินการที่เกี่ยวข้องกับความไม่สอดคล้อง  
2. เลือก **ใหม่**
3. ในฟิลด์ **รหัสค่าธรรมเนียม** ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ **คำอธิบาย** ให้พิมพ์ค่าใดค่าหนึ่ง

## <a name="define-the-operations-for-nonconformance-processing"></a>กำหนดการดำเนินงานสำหรับการประมวลผลความไม่สอดคล้องกัน
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > การจัดการสินค้าคงคลัง > การตั้งค่า > การจัดการคุณภาพ > การดำเนินงาน** ใช้หน้า **การดำเนินงาน** เพื่อกำหนดการจัดประเภทของงานที่สามารถดำเนินการได้สำหรับความไม่สอดคล้องที่อนุมัติแล้ว เมื่อคุณเชื่อมโยงการดำเนินงานกับความไม่สอดคล้อง คุณสามารถกำหนดข้อมูลเกี่ยวกับวัสดุที่เกี่ยวข้อง ชั่วโมงแรงงาน และค่าธรรมเนียมเบ็ดเตล็ดที่จำเป็นเพื่อทำการดำเนินงาน ข้อมูลนี้ให้พื้นฐานสำหรับการคำนวณต้นทุนที่ประเมินสำหรับการดำเนินการ  
2. เลือก **ใหม่**
3. ในฟิลด์ **การดำเนินการ** ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ **คำอธิบาย** ให้พิมพ์ค่าใดค่าหนึ่ง

## <a name="define-problem-types-for-nonconformance-processing"></a>กำหนดชนิดปัญหาสำหรับการประมวลผลความไม่สอดคล้องกัน
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > การจัดการสินค้าคงคลัง > การตั้งค่า > การจัดการคุณภาพ > ชนิดของปัญหา** ใช้หน้า **ชนิดของปัญหา** เพื่อกำหนดการจัดประเภทของปัญหาด้านคุณภาพ ซึ่งพบในความไม่สอดคล้องกันชนิดต่างๆ ชนิดความไม่สอดคล้องประกอบด้วย **ภายใน** **ลูกค้า** **ผู้จัดจำหน่าย** **คำขอการบริการ** **การผลิต** และ **การผลิตผลิตภัณฑ์ร่วม** ชนิดปัญหาเดี่ยวสามารถถูกเชื่อมโยงได้กับชนิดความไม่สอดคล้องที่หลากหลาย  
2. เลือก **ใหม่**
3. ในฟิลด์ **ชนิดปัญหา** ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ **คำอธิบาย** ให้พิมพ์ค่าใดค่าหนึ่ง
5. เลือก **ชนิดความไม่สอดคล้องกัน** ใช้หน้า **ชนิดความไม่สอดคล้องกัน** เพื่ออนุมัติการใช้ชนิดของปัญหาสำหรับความไม่สอดคล้องกันหนึ่งชนิดขึ้นไป ตัวอย่างเช่น ชนิดของปัญหาเกี่ยวกับรหัสความบกพร่องสามารถนำไปใช้กับความไม่สอดคล้องกันได้ทุกชนิด ในขณะที่ชนิดของปัญหาเกี่ยวกับคำร้องเรียนของลูกค้าสามารถนำไปใช้กับความไม่สอดคล้องกันชนิดลูกค้าและคำขอการบริการเท่านั้น  
6. เลือก **ใหม่**
7. ในฟิลด์ **ชนิดของความไม่สอดคล้องกัน** ของแถวใหม่ ให้เลือกตัวเลือกหนึ่งตัวเลือก

## <a name="define-quarantine-zones-for-nonconformance-processing"></a>กำหนดโซนการตรวจสอบสินค้า สำหรับการประมวลผลความไม่สอดคล้องกัน
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > การจัดการสินค้าคงคลัง > การตั้งค่า > การจัดการคุณภาพ > โซนการตรวจสอบสินค้า**
2. เลือก **ใหม่**
3. ในฟิลด์ **โซนการตรวจสอบสินค้า** ให้พิมพ์ค่า
4. ในฟิลด์ **คำอธิบาย** ให้พิมพ์ค่าใดค่าหนึ่ง
5. ปิดหน้า
