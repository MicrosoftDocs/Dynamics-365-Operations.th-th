# <a name="set-up-prerequisites-for-nonconformance-management"></a>ตั้งค่าข้อกำหนดเบื้องต้นสำหรับการจัดการความไม่สอดคล้องกัน

[!include [task guide banner](../../includes/task-guide-banner.md)]

ใช้กระบวนงานนี้เพื่อเปิดใช้งานกระบวนการการจัดการความไม่สอดคล้องกัน ความไม่สอดคล้องกันอธิบายกระบวนงานหรือรายการที่มีปัญหาทางคุณภาพ ที่ข้อมูลคำอธิบายรวมแหล่งที่มาและชนิดของปัญหา กระบวนงานนี้ใช้บริษัทข้อมูลสาธิต USMF โดยทั่วไป กระบวนงานนี้จะถูกดำเนินการโดยผู้จัดการด้านคุณภาพ


## <a name="enable-quality-management-processes-within-the-company"></a>เปิดใช้งานกระบวนการจัดการคุณภาพภายในบริษัท
1. ไปที่การจัดการสินค้าคงคลัง > การตั้งค่า > สินค้าคงคลังและพารามิเตอร์การจัดการคลังสินค้า
2. คลิกแท็บการจัดการคุณภาพ
3. เลือก ใช่ ในฟิลด์ใช้การจัดการคุณภาพ
    * เลือกพารามิเตอร์นี้เพื่อเปิดใช้งานกระบวนการจัดการคุณภาพสำหรับบริษัท  
4. ในฟิลด์อัตราต่อชั่วโมง ให้ป้อนตัวเลข
    * ใช้ฟิลด์อัตราต่อชั่วโมง เพื่อป้อนอัตราแรงงานต่อชั่วโมงในในสกุลเงินท้องถิ่น  อัตราต่อชั่วโมงจะถูกใช้ในการคำนวณต้นทุนสำหรับการดำเนินงานที่เกี่ยวข้องกับความไม่สอดคล้อง และไม่ได้สัมพันธ์กับฟังก์ชันอื่น  
5. คลิกการตั้งค่ารายงาน
    * หน้านี้อนุญาตให้คุณกำหนดชนิดหมายเหตุรายงานตรวจสอบคุณภาพที่จะใช้ในรายงานการจัดการคุณภาพชนิดต่างๆ  
6. ปิดหน้า
7. ปิดหน้า

## <a name="enable-user-for-nonconformance-processing"></a>เปิดใช้งานผู้ใช้สำหรับการประมวลผลความไม่สอดคล้องกัน
1. ไปที่การดูแลระบบ > ผู้ใช้ > ผู้ใช้
    * เพื่อประมวลผลการอนุมัติของความไม่สอดคล้องกัน ผู้ใช้ที่อนุมัติหรือปฏิเสธความไม่สอดคล้องกันต้องมีค่า "ชื่อ" ที่กำหนดไว้ในหน้าผู้ใช้  การใช้บันทึกเอกสาร ผู้ใช้ต้องสามารถเรียกใช้ตัวเลือกผู้ใช้ในการจัดการเอกสาร  
2. ใช้ตัวกรองข้อมูลด่วนเพื่อค้นหาเรกคอร์ด  ตัวอย่างเช่น กรองข้อมูลในฟิลด์ชื่อ ด้วยค่า 'Ricardo'
    * ใช้ตัวกรองเพื่อค้นหาผู้ใช้ที่กำลังจะอนุมัติหรือปฏิเสธเรกคอร์ดความไม่สอดคล้องกัน  
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
    * เพื่อประมวลผลการอนุมัติของความไม่สอดคล้องกัน ตรวจสอบให้แน่ใจว่าผู้ใช้ที่อนุมัติหรือปฏิเสธความไม่สอดคล้องกันมีค่า "ชื่อ" ที่กำหนดอยู่ในหน้าผู้ใช้  
4. คลิกตัวเลือกของผู้ใช้
5. คลิกแท็บการกำหนดลักษณะ
6. เลือก ใช่ ในฟิลด์การเปิดใช้งานการจัดการเอกสาร
    * การใช้บันทึกเอกสาร ผู้ใช้ต้องสามารถเรียกใช้ตัวเลือกผู้ใช้ในการจัดการเอกสาร  
7. ปิดหน้า
8. ปิดหน้า
9. ปิดหน้า

## <a name="define-diagnostic-types-for-nonconformance-processing"></a>กำหนดชนิดของการวินิจฉัย สำหรับการประมวลผลความไม่สอดคล้องกัน
1. ไปที่การจัดการคลังสินค้า > การตั้งค่า > การจัดการคุณภาพ > ชนิดการวินิจฉัย
    * ใช้หน้าชนิดการวินิจฉัย เพื่อกำหนดการจัดประเภทของการดำเนินการวินิจฉัย  การแก้ไขระบุชนิดของการดำเนินการวินิจฉัย ซึ่งควรจะดำเนินการกับความไม่สอดคล้องกันที่ได้รับอนุมัติแล้ว บุคคลที่ควรจะเป็นผู้ดำเนินการ และวันที่เสร็จสมบูรณ์ที่ร้องขอและที่วางแผนไว้  
2. คลิก สร้าง
3. ในฟิลด์การวินิจฉัย ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. ปิดหน้า

## <a name="define-quality-charges-for-nonconformance-processing"></a>กำหนดค่าธรรมเนียมคุณภาพ สำหรับการประมวลผลความไม่สอดคล้องกัน
1. ไปที่การจัดการคลังสินค้า > การตั้งค่า > การจัดการคุณภาพ > ค่าธรรมเนียมเกี่ยวกับคุณภาพ
    * ใช้หน้าค่าธรรมเนียมเกี่ยวกับคุณภาพ เพื่อกำหนดการจัดประเภทของค่าธรรมเนียมที่จะถูกใช้ในการดำเนินการที่เกี่ยวข้องกับความไม่สอดคล้อง  
2. คลิก สร้าง
3. ในฟิลด์รหัสค่าธรรมเนียม ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. ปิดหน้า

## <a name="define-the-operations-for-nonconformance-processing"></a>กำหนดการดำเนินงานสำหรับการประมวลผลความไม่สอดคล้องกัน
1. ไปที่การจัดการคลังสินค้า > การตั้งค่า > การจัดการคุณภาพ > การดำเนินการ
    * ใช้หน้าการดำเนินงานเพื่อกำหนดการจัดประเภทของงานที่สามารถดำเนินการสำหรับความไม่สอดคล้องที่อนุมัติแล้ว เมื่อคุณเชื่อมโยงการดำเนินงานกับความไม่สอดคล้อง คุณสามารถกำหนดข้อมูลเกี่ยวกับวัสดุที่เกี่ยวข้อง ชั่วโมงแรงงาน และค่าธรรมเนียมเบ็ดเตล็ดที่จำเป็นเพื่อทำการดำเนินงาน ข้อมูลนี้ให้พื้นฐานสำหรับการคำนวณต้นทุนที่ประเมินสำหรับการดำเนินการ  
2. คลิก สร้าง
3. ในฟิลด์การดำเนินการ ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. ปิดหน้า

## <a name="define-problem-types-for-nonconformance-processing"></a>กำหนดชนิดปัญหาสำหรับการประมวลผลความไม่สอดคล้องกัน
1. ไปที่การจัดการคลังสินค้า > การตั้งค่า > การจัดการคุณภาพ > ชนิดปัญหา
    * ใช้หน้าชนิดปัญหาเพื่อกำหนดการจัดประเภทของปัญหาด้านคุณภาพ ซึ่งพบในความไม่สอดคล้องกันชนิดต่างๆ ชนิดความไม่สอดคล้องประกอบด้วย ภายใน ลูกค้า ผู้จัดจำหน่าย คำขอการบริการ การผลิต และการผลิตสินค้าร่วม ชนิดปัญหาเดี่ยวสามารถถูกเชื่อมโยงได้กับชนิดความไม่สอดคล้องที่หลากหลาย  
2. คลิก สร้าง
3. ในฟิลด์ชนิดปัญหา ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. คลิกชนิดของความไม่สอดคล้องกัน
    * ใช้หน้าชนิดความไม่สอดคล้องกัน เพื่ออนุมัติการใช้ชนิดของปัญหาสำหรับความไม่สอดคล้องกันหนึ่งชนิดขึ้นไป  ตัวอย่างเช่น ชนิดของปัญหาเกี่ยวกับรหัสความบกพร่องสามารถนำไปใช้กับความไม่สอดคล้องกันได้ทุกชนิด ในขณะที่ชนิดของปัญหาเกี่ยวกับคำร้องเรียนของลูกค้าสามารถนำไปใช้กับความไม่สอดคล้องกันชนิดลูกค้าและคำขอการบริการเท่านั้น  
6. คลิก สร้าง
7. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
8. ในฟิลด์ชนิดของความไม่สอดคล้องกัน ให้เลือกตัวเลือกหนึ่งตัวเลือก
9. ปิดหน้า
10. ปิดหน้า

## <a name="define-quarantine-zones-for-nonconformance-processing"></a>กำหนดโซนการตรวจสอบสินค้า สำหรับการประมวลผลความไม่สอดคล้องกัน
1. ไปที่การบริหารสินค้าคงคลัง > การตั้งค่า > การจัดการคุณภาพ > เขตการตรวจสอบสินค้า
2. คลิก สร้าง
3. ในฟิลด์เขตการตรวจสอบสินค้า ให้พิมพ์ค่า
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. ปิดหน้า
