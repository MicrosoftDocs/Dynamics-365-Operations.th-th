# <a name="create-a-consignment-replenishment-order"></a>สร้างใบสั่งการเพิ่มเติมสินค้าที่มีการส่งมอบ

[!include [banner](../../includes/banner.md)]

หัวข้อนี้อธิบายวิธีการสร้างใบสั่งการเพิ่มเติมสินค้าที่มีการส่งมอบที่ซึ่งคุณสามารถติดตามการจัดส่งที่คาดไว้จากผู้จัดจำหน่ายไปยังสินค้าคงคลังที่มีการส่งมอบของคุณ นอกจากนี้ยังแสดงวิธีการบันทึกการรับผลิตภัณฑ์เพื่อลงทะเบียนสินค้าคงคลังที่มีการส่งมอบเป็นปริมาณคงคลังคงเหลือที่ผู้จัดจำหน่ายเป็นเจ้าของ โดยทั่วไป กระบวนงานนี้จะกระทำโดยผู้เชี่ยวชาญด้านการจัดซื้อ คุณสามารถใช้คำแนะนำนี้ในบริษัทข้อมูลสาธิต USMF  กระบวนงานนี้ใช้สำหรับลักษณะการทำงานที่ถูกเพิ่มใน Dynamics 365 for Operations รุ่น 1611

## <a name="create-a-consignment-replenishment-order"></a>สร้างใบสั่งการเพิ่มเติมสินค้าที่มีการส่งมอบ
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > การจัดซื้อและการจัดหา > การส่งมอบ > ใบสั่งการเพิ่มเติมสินค้าที่มีการส่งมอบ**
2. เลือก **ใหม่**
3. ในฟิลด์ **บัญชีผู้จัดจำหน่าย** เลือกผู้จัดจำหน่าย **US-104** (คุณต้องเลือกผู้จัดจำหน่ายที่มีการลงทะเบียนเป็นเจ้าของในหน้า **เจ้าของสินค้าคงคลัง**) 
4. เลือก **ตกลง**
5. เลือก **เพิ่มรายการ**
6. ในฟิลด์ **หมายเลขสินค้า** ให้พิมพ์ `M9211CI` (คุณต้องเลือกสินค้าที่ตั้งค่าไว้สำหรับสินค้าคงคลังการส่งมอบ)
7. ในฟิลด์ **ปริมาณ** ให้ป้อนตัวเลข
8. ในฟิลด์ **วันที่จัดส่งที่ร้องขอ** ให้ป้อนวันที่ วันที่ร้องขอและวันที่ยืนยันจะถูกนำมาใช้โดยกลไกจัดการ MRP สำหรับการมาถึงที่คาดไว้ของสินค้า  
9. ในฟิลด์ **วันที่จัดส่งที่ยืนยัน** ให้ป้อนวันที่
10. ขยายส่วน **รายละเอียดรายการ**
11. เลือกแท็บ **มิติสินค้าคงคลัง**
12. เมื่อต้องการแสดงเจ้าของในฟิลด์ **เจ้าของมิติสินค้าคงคลัง** ให้รีเฟรชหน้า ขณะนี้มีการแสดงรายการ US-104 ของผู้จัดจำหน่ายเป็นเจ้าของ  

## <a name="check-the-inventory-transaction-status"></a>ตรวจสอบสถานะของรายการความเคลื่อนไหวของสินค้าคงคลัง
1. เลือก **สินค้าคงคลัง**
2. เลือก **ธุรกรรม**
3. ในแถวที่ต้องการ โปรดทราบว่าฟิลด์ **ใบเสร็จ** ถูกตั้งค่าเป็น **สั่งแล้ว**  
4. ปิดหน้า

## <a name="receive-items"></a>รับสินค้า
1. เลือก **ใบรับสินค้า**
2. ในฟิลด์ **ใบรับผลิตภัณฑ์ภายนอก** ให้พิมพ์ค่า
3. ในฟิลด์ **ปริมาณ** ให้ป้อนหมายเลขที่ต่ำกว่าหมายเลขที่แสดงอยู่ที่นั่น 
4. เลือก **ตกลง**

## <a name="check-the-on-hand-inventory"></a>ตรวจสอบปริมาณคงคลังคงเหลือ
1. เลือก **สินค้าคงคลัง**
2. เลือก **ปริมาณคงคลังคงเหลือ**
3. เลือก **ภาพรวม** สินค้าที่ได้รับเป็นสินค้าคงคลังที่มีการส่งมอบที่ผู้จัดจำหน่ายเป็นเจ้าของเป็นปริมาณคงคลังคงเหลือที่พร้อมใช้งาน  ปริมาณคงเหลือในใบสั่งการเพิ่มเติมสินค้าที่มีการส่งมอบจะแสดงอยู่ในฟิลด์ **ผลรวมที่สั่ง**  
4. ปิดหน้า
