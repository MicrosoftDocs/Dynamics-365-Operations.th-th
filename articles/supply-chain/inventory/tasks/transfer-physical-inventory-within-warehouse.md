# <a name="transfer-physical-inventory-within-the-warehouse"></a>โอนย้ายสินค้าคงคลังที่มีอยู่จริงภายในคลังสินค้า

[!include [banner](../../includes/banner.md)]

ขั้นตอนนี้อธิบายเกี่ยวกับกระบวนการของการสร้างและการลงรายการบัญชีสมุดรายวันการโอนสินค้าคงคลังในใบสั่งไปยังทะเบียนการเคลื่อนย้ายของสินค้าจากสถานที่หนึ่งในคลังสินค้าอื่น  คุณจำเป็นต้องมีชื่อสมุดรายวันสินค้าคงคลังที่มีการตั้งค่าสำหรับการโอนย้ายสินค้าคงคลังก่อนที่คุณจะเริ่มการทำงานนี้ คุณสามารถศึกษากระบวนงานนี้ได้ในบริษัทข้อมูลสาธิต USMF โดยใช้ค่าตัวอย่างที่แสดง หรือคุณสามารถใช้ข้อมูลของคุณเองได้ถ้าคุณได้ตั้งค่าผลิตภัณฑ์และสถานที่แล้ว งานเหล่านี้จะปกติจะดำเนินการโดยพนักงานคลังสินค้า


## <a name="create-an-inventory-transfer-journal"></a>สร้างสมุดรายวันการโอนสินค้าคงคลัง
1. ใน **บานหน้าต่างนำทาง** ให้ไปที่ **การบริหารสินค้าคงคลัง > รายการสมุดรายวัน > สินค้า > โอนย้าย**
2. คลิก **สร้าง**
3. ในฟิลด์ **ชื่อ** ป้อนหรือเลือกค่า
4. คลิก **ตกลง** มีตัวเลือกในการระบุเป็น 'เริ่มต้น' และ 'สิ้นสุด' มิติสำหรับแต่ละรายการในสมุดรายวัน  สิ่งเหล่านี้มีความจำเป็นสำหรับชนิดสมุดรายวัน คุณสามารถโอนสิ้นค้าไปยังที่ตั้งที่ใช้กฎที่ต่างกัน ในตัวอย่างนี้ พวกเราจะโอนสินค้าภายในคลังสินค้าเดียวกัน จากสถานที่ที่ควบคุมป้ายทะเบียนไปยังสถานที่ที่ไม่ได้ควบคุมป้ายทะเบียน   

## <a name="create-journal-lines"></a>สร้างรายการสมุดรายวัน
1. **fastTab รายการสมุดรายวัน** คลิก **สร้าง**
2. ในฟิลด์ **หมายเลขสินค้า** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก 'A0001'  
3. ในฟิลด์ **ไซต์ต้นทาง** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก '2'  
4. ในฟิลด์ **ไซต์ปลายทาง** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก '2'  
5. ในฟิลด์ **คลังสินค้าต้นทาง** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก '24'  
6. ในฟิลด์ **คลังสินค้าปลายทาง** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก '24'  
7. ในฟิลด์ **สถานที่ต้นทาง** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก 'FL-001'  
8. ในฟิลด์ **สถานที่ปลายทาง** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง ถ้าคุณกำลังใช้ USMF คุณสามารถเลือกสินค้า 'BULK-001'  
9. ในฟิลด์ **ปริมาณ** ให้ป้อนตัวเลข
10. ใน fastTab **รายละเอียดของรายการ** คลิกแท็บ **มิติสินค้าคงคลัง**
11. ใน **จากมิติสินค้าคงคลัง** ในฟิลด์ **ป้ายทะเบียน** ให้ป้อนหรือเลือกค่า ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก '24'  
12. คลิก **บันทึก**

## <a name="post-the-inventory-transfer-journal"></a>ลงรายการบัญชีสมุดรายวันการโอนสินค้าคงคลัง
1. ใน **บานหน้าต่างการดำเนินการ** คลิก **ลงรายการบัญชี**
2. คลิก **ตกลง**

## <a name="view-inventory-transactions"></a>ดูธุรกรรมสินค้าคงคลัง
1. คลิก **สินค้าคงคลัง**
2. คลิก **ธุรกรรม** คุณสามารถดูธุรกรรมที่ถูกสร้างขึ้นเมื่อคุณลงรายการบัญชีสมุดรายวันของคุณ
