# <a name="adjust-stock-levels-in-the-warehouse-basic-warehousing"></a>ปรับปรุงระดับปริมาณสินค้าคงคลังในคลังสินค้า (คลังสินค้าพื้นฐาน)

[!include [banner](../../includes/banner.md)]

กระบวนงานนี้นำคุณผ่านกระบวนการของการสร้างและการลงรายการบัญชีสมุดรายวันการปรับปรุงสินค้าคงคลังเพื่อปรับปรุงระดับสินค้าคงคลังของผลิตภัณฑ์ในคลังสินค้า  คุณจำเป็นต้องมีชื่อสมุดรายวันสินค้าคงคลังที่ตั้งค่าสำหรับการปรับปรุงสินค้าคงคลัง ก่อนที่คุณจะเริ่มต้นนี้  คุณสามารถดำเนินการขั้นตอนนี้ได้ในข้อมูลสาธิตของบริษัท USMF หรือการใช้ข้อมูลของคุณเอง งานเหล่านี้จะปกติจะดำเนินการโดยพนักงานคลังสินค้า


## <a name="create-an-inventory-adjustment-journal"></a>สร้างสมุดรายวันการปรับปรุงสินค้าคงคลัง
1. ไปที่ การบริหารสินค้าคงคลัง > รายการสมุดบันทึกรายวัน > สินค้า > การปรับปรุงสินค้าคงคลัง
2. คลิก สร้าง
3. ในฟิลด์ชื่อ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ในรายการ คลิก ชื่อสมุดรายวันการปรับปรุงสินค้าคงคลังที่คุณต้องการใช้
    * บางฟิลด์อื่นๆ จะถูกเติมข้อมูลตามการตั้งค่าของชื่อสมุดรายวันการปรับปรุงสินค้าคงคลังที่คุณเลือก  
5. คลิก ตกลง

## <a name="create-journal-lines"></a>สร้างรายการสมุดรายวัน
1. คลิก สร้าง
2. ในรายการ ทำเครื่องหมายที่ฟิลด์หมายเลขสินค้า
3. ในฟิลด์หมายเลขสินค้า เลือกสินค้า  ถ้าคุณกำลังใช้ข้อมูลสาธิตของบริษัท USMF ชนิด 'D0001'
4. ในฟิลด์ไซต์ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
5. ในรายการ เลือก ไซต์
6. ในฟิลด์คลังสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
7. ในรายการ เลือก คลังสินค้า
    * ถ้าคุณได้เลือกสินค้าทีมีตำแหน่งที่ตั้งเป็นมิติบังคับ คุณจะต้องระบุตำแหน่งที่ตั้งที่นี่  
8. ในฟิลด์ ปริมาณ ให้ป้อนตัวเลข
    * ฟิลด์ราคาต้นทุนระบุต้นทุนต่อหน่วยสำหรับการรับสต็อก  ถ้าไม่ระบุต้นทุนสำหรับหมายเลขสินค้า หรือถ้าคุณต้องการเปลี่ยนแปลงต้นทุนด้วยตนเอง คุณควรทำ ณ ที่นี่  

## <a name="validate-and-post-the-inventory-adjustment-journal"></a>ตรวจสอบความถูกต้องและลงรายการบัญชีสมุดการปรับปรุงสินค้าคงคลัง
1. คลิก ตรวจสอบความถูกต้อง
2. คลิก ตกลง
3. คลิก ลงรายการบัญชี
    * เมื่อคุณลงรายการบัญชีสมุดรายวันชนิดนี้ การรับสินค้าคงคลังหรือการตัดสินค้าจากคลังจะถูกลงรายการบัญชี มีการเปลี่ยนแปลงระดับและมูลค่าสินค้าคงคลัง และมีการสร้างธุรกรรมบัญชีแยกประเภท  
4. คลิก ตกลง
5. ปิดแบบฟอร์ม
6. ปิดหน้า
