# <a name="inventory-management"></a>การบริหารสินค้าคงคลัง 

[!include[banner](../includes/banner.md)]

คุณสามารถใช้การจัดการสินค้าคงคลังเพื่อทำภารกิจต่อไปนี้:

-  [การดำเนินการขาเข้า](arrival-overview.md)
-  [การรับรองคุณภาพ](quality-management-processes.md)
-  [กิจกรรมสินค้าคงคลัง](inventory-journals.md)
-  [การดำเนินการขาออก](outbound-process.md)
-  [การควบคุมสินค้าคงคลัง](../cost-management/inventory-close.md) 

## <a name="learning-map"></a>แผนผังการเรียนรู้

แผนผังการเรียนรู้ต่อไปนี้แสดงแนวคิดหลักและงานที่สร้างกรอบงานของโมดูลการจัดการสินค้าคงคลัง คลิกที่แผนผังการเรียนรู้ด้านล่างเพื่อขยาย แผนผังการเรียนรู้นี้จะช่วยคุณในการเริ่มต้นใช้งาน

## <a name="additional-resources"></a>ทรัพยากรเพิ่มเติม

### <a name="whats-new-and-in-development"></a>มีอะไรใหม่และอะไรที่กำลังพัฒนา
ไปที่ [แผนการทำงาน Microsoft Dynamics 365](https://roadmap.dynamics.com/) เพื่อดูว่ามีการนำคุณลักษณะใหม่ใดออกใช้ และมีคุณลักษณะใหม่ใดบ้างที่กำลังพัฒนา

### <a name="inventory-accounting"></a>การบัญชีสินค้าคงคลัง 
เมื่อต้องการทราบข้อมูลเพิ่มเติม ดู [การบัญชีสินค้าคงคลังในการจัดการต้นทุน](../cost-management/inventory-close.md)

### <a name="manufacturing-and-supply-chain-management-blogs"></a>บล็อกการผลิตและการจัดการห่วงโซ่อุปทาน
คุณสามารถค้นหาความคิดเห็น ข่าว และข้อมูลอื่นๆ เกี่ยวกับการจัดการสินค้าคงคลังและโซลูชันอื่นๆ ได้ใน [บล็อกทีมวิจัยและพัฒนาการผลิต Dynamics AX](https://blogs.msdn.microsoft.com/axmfg) และ [การบริหารห่วงโซ่อุปทานในบล็อกทีมวิจัยและพัฒนาการผลิต Dynamics AX](https://blogs.msdn.microsoft.com/dynamicsaxscm)

### <a name="task-guides"></a>คู่มืองาน
วิธีใช้เพิ่มเติมพร้อมใช้เป็นคู่มืองานภายใน Finance and Operations ในการเข้าถึงคู่มืองาน ให้คลิกปุ่ม **วิธีใช้** บนหน้าใดๆ
