## <a name="blocking-items-by-creating-a-quality-order"></a><span data-ttu-id="e949a-120">การบล็อคสินค้าโดยการสร้างใบสั่งตรวจสอบคุณภาพ</span><span class="sxs-lookup"><span data-stu-id="e949a-120">Blocking items by creating a quality order</span></span>
<span data-ttu-id="e949a-121">คุณสามารถระบุสินค้าที่ต้องถูกตรวจสอบได้โดยการสร้างใบสั่งตรวจสอบคุณภาพในหน้า **ใบสั่งตรวจสอบคุณภาพ**</span><span class="sxs-lookup"><span data-stu-id="e949a-121">You can specify items that must be inspected by creating a quality order on the **Quality orders** page.</span></span> <span data-ttu-id="e949a-122">เมื่อคุณสร้างใบสั่งตรวจสอบคุณภาพ ปริมาณที่คุณระบุไว้สำหรับสินค้าจะถูกบล็อค</span><span class="sxs-lookup"><span data-stu-id="e949a-122">When you create a quality order, the quantity that you specify for an item is blocked.</span></span> <span data-ttu-id="e949a-123">แผนการสุ่มตัวอย่างที่เชื่อมโยงกับใบสั่งตรวจสอบคุณภาพควบคุมเฉพาะปริมาณของสินค้าที่ต้องถูกตรวจสอบ ไม่ใช่ปริมาณที่ถูกบล็อค</span><span class="sxs-lookup"><span data-stu-id="e949a-123">The sampling plan that is associated with a quality order controls only the quantity of items that must be inspected, not the quantity that is blocked.</span></span> <span data-ttu-id="e949a-124">ปริมาณที่ถูกป้อนบนใบสั่งตรวจสอบคุณภาพคือปริมาณที่ถูกบล็อค โดยไม่คำนึงถึงปริมาณที่การระบุแผนการสุ่มตัวอย่างควรถูกส่งไปเพื่อการตรวจสอบ</span><span class="sxs-lookup"><span data-stu-id="e949a-124">The quantity that is entered on the quality order is the quantity that is blocked, regardless of the quantity that the sampling plan specifies should be sent for inspection.</span></span>

## <a name="blocking-items-by-using-a-process-that-generates-a-quality-order"></a><span data-ttu-id="e949a-128">การบล็อคสินค้าโดยใช้กระบวนการที่สร้างใบสั่งตรวจสอบคุณภาพ</span><span class="sxs-lookup"><span data-stu-id="e949a-128">Blocking items by using a process that generates a quality order</span></span>
<span data-ttu-id="e949a-129">ถ้ากระบวนการตรวจสอบคุณภาพระบุว่าต้องตรวจสอบสินค้า ปริมาณของสินค้าจะถูกบล็อคโดยอัตโนมัติ</span><span class="sxs-lookup"><span data-stu-id="e949a-129">If a quality process specifies that an item must be inspected, a quantity of the item is blocked automatically.</span></span> <span data-ttu-id="e949a-130">ดังนั้น เมื่อมีการสร้างใบสั่งตรวจสอบคุณภาพโดยอัตโนมัติ แผนการสุ่มตัวอย่างสินค้าที่เชื่อมโยงกับใบสั่งตรวจสอบคุณภาพจะควบคุมทั้งปริมาณของสินค้าที่ถูกบล็อคและปริมาณที่ต้องถูกตรวจสอบ</span><span class="sxs-lookup"><span data-stu-id="e949a-130">Therefore, when a quality order is generated automatically, the item sampling plan that is associated with the quality order controls the both quantity of items that is blocked and the quantity that must be inspected.</span></span> <span data-ttu-id="e949a-131">ถ้าตัวเลือก **การบล็อคทั้งหมด** ในหน้า **การสุ่มตัวอย่างสินค้า** ถูกเลือก ปริมาณทั้งหมดของ ตัวอย่างเช่น รายการใบสั่งซื้อจะถูกบล็อคในระหว่างการตรวจสอบ โดยไม่คำนึงถึงปริมาณการสุ่มตัวอย่างสินค้า</span><span class="sxs-lookup"><span data-stu-id="e949a-131">If the **Full blocking** option on the **Item sampling** page is selected, the full quantity of, for example, a purchase order line is blocked during inspection, regardless of the item sampling quantity.</span></span>
