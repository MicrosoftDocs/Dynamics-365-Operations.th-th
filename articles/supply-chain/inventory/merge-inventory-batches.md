# <a name="merge-inventory-batches"></a>ผสานชุดงานสินค้าคงคลัง

[!include[banner](../includes/banner.md)]


บทความข้อนี้แสดงข้อมูลเกี่ยวกับวิธีการรวมบัญชีสองชุดงานสินค้าคงคลังขึ้นไปไปยังชุดงานแบบผสาน

เมื่อคุณผสานชุดงาน การคำนวณสามารถช่วยปรับลักษณะและแอททริบิวต์ของชุดงานในชุดงานแบบผสาน  หลังจากที่คุณเลือกชุดงานต้นทาง คุณสามารถตรวจทานและเปลี่ยนแปลงชุดงานที่ผสานได้ก่อนที่จะลงรายการบัญชี คุณสามารถโอนย้ายข้อมูลการผสานชุดงานไปที่สมุดรายวันสินค้าคงคลังเพื่อรอการอนุมัติ จากนั้นสามารถจองหรือลงรายการบัญชีสินค้าคงคลังได้โดยตรงจากสมุดรายวันสินค้าคงคลัง เมื่อคุณลงรายการบัญชีชุดงานแบบผสาน สินค้าคงคลังมีการปรับปรุงสำหรับชุดงานต้นทางและชุดงานที่ผสาน

## <a name="when-might-i-want-to-merge-batches-of-inventory"></a>ฉันจะต้องการผสานชุดงานของสินค้าคงคลังเมื่อใด
นี่คือบางตัวอย่างของสถานการณ์จำลองที่อาจจะเป็นประโยชน์ในการผสานชุดงาน:

-   ขณะที่ Sammy กำลังเดินไปผ่านคลังสินค้าของเขา เขาสังเกตพบว่า มีหลายชุดงานของสินค้าเดียวกันที่มีปริมาณเหลือน้อย เขาจะคาดว่าจะได้รับการจัดส่งใหม่หลายรายการ และเขาตระหนักว่าเขาสามารถเพิ่มพื้นจัดวางได้ด้วยการผสานปริมาณที่เหลือกับชุดงานใหม่
-   Sammy กำลังรับสินค้าคงคลัง และเขาต้องการรวมชุดงานใหม่กับชุดงานที่เขาได้รับแล้ว เพื่อปรับปรุงค่าแอททริบิวต์ของชุดงานสำหรับชุดงานที่มีอยู่ การกระทำดังกล่าวจะเป็นการสร้างชุดงานใหม่

## <a name="can-i-merge-batches-across-sites-and-legal-entities"></a>ฉันสามารถผสานชุดงานระหว่างไซต์และนิติบุคคลได้หรือไม่
ไม่ได้ คุณสามารถผสานชุดงานที่มีไซต์เดียวกันและมิติการจัดเก็บของคลังสินค้าในนิติบุคคลเดียวเท่านั้น อย่างไรก็ตาม คุณสามารถระบุสถานที่ตั้งอื่นและรหัสแท่นวางสินค้าสำหรับชุดงานที่ผสาน

## <a name="can-i-merge-partial-quantities"></a>ฉันสามารถผสานปริมาณบางส่วนได้หรือไม่
ไม่ได้ คุณสามารถผสานเฉพาะปริมาณทั้งหมดของชุดงาน ฟังก์ชันการผสานชุดงานมีไว้เป็นคุณลักษณะสินค้าคงคลัง ไม่ใช่คุณลักษณะการลิต

## <a name="what-if-the-batches-have-different-batch-attribute-values"></a>จะเป็นอย่างไรถ้าชุดงานมีค่าแอททริบิวต์ของชุดงานที่แตกต่างกัน
เมื่อคุณเลือกชุดงานต้นทางที่จะรวมในชุดงานที่ผสาน Finance and Operations ตรวจสอบว่าชุดงานทั้งหมดมีลักษณะหรือค่าแอททริบิวต์หรือไม่ เมื่อค่าแอตทริบิวต์เหมือนกัน จะมีการแนะนำค่าสำหรับชุดงานแบบผสาน คุณสามารถเปลี่ยนค่าได้ ค่าแอททริบิวต์ที่ไม่เหมือนกันถูกปล่อยว่างไว้สำหรับชุดงานที่ผสาน และคุณสามารถป้อนค่าเหล่านี้ด้วยตนเอง ถ้าชนิดแอททริบิวต์ของชุดงานสำหรับค่าแอททริบิวต์เป็นจำนวนเต็มหรือเศษส่วน และค่าไม่เหมือนกันสำหรับชุดงานต้นทาง ค่าจะถูกคำนวณโดยใช้การคำนวณค่าเฉลี่ยถ่วงน้ำหนัก มูลค่าที่คำนวณได้จะถูกปัดเศษขึ้นหรือลง เพื่อให้เพิ่มขึ้นใกล้ที่สุด ถ้าค่าถูกเว้นว่างไว้สำหรับชุดงานต้นทาง ชุดงานและปริมาณจะไม่ถูกรวมในการคำนวณ **ตัวอย่าง** ตัวอย่างต่อไปนี้แสดงการคำนวณค่าเฉลี่ยถ่วงน้ำหนักสำหรับชุดงานที่ผสาน สองชุดงานต้นทางมีค่าว่างเปล่าสำหรับชนิดแอททริบิวต์ของชุดงานที่เป็นจำนวนเต็ม แอททริบิวต์ต่อไปนี้ถูกกำหนดให้กับชุดงานต้นทาง

## <a name="what-if-the-batches-have-different-batch-dates"></a>จะเป็นอย่างไรถ้าชุดงานมีวันที่ชุดงานแตกต่างกัน
ถ้าชุดงานมีวันที่ชุดงานแตกต่างกัน วันที่บางวันจะถูกคำนวณตามค่าในกลุ่ม **วันที่ชุดงาน** บนแท็บด่วน **ชุดงานที่ผสาน** ของหน้า **การผสานชุดงาน** ระบบจะคำนวณค่าสำหรับฟิลด์ในกลุ่ม **วันที่ชุดงาน** ค่าเหล่านี้รวมถึงวันที่ผลิต วันหมดอายุ วันที่แนะนำการเก็บสินค้า และวันที่ควรใช้ก่อน วันที่จะถูกคำนวณตามการตั้งค่าสำหรับรายการในกลุ่มฟิลด์ **ข้อมูลสินค้า** ของหน้า **รายละเอียดผลิตภัณฑ์ที่นำออกใช้** คุณสามารถเปลี่ยนค่า หรือป้อนด้วยตนเอง สำหรับวันที่อื่นๆ ทั้งหมด จะไม่มีการคำนวณ หลักการเดียวกับที่ใช้สำหรับค่าแอททริบิวต์ของชุดงาน ถ้าวันที่เหมือนกันสำหรับชุดงานต้นทางทั้งหมด วันที่นี้จะถูกแนะนำสำหรับชุดงานที่ผสาน ถ้าวันที่ไม่เหมือนกันสำหรับชุดงานต้นทางทั้งหมด วันที่จะถูกเว้นว่างไว้ในชุดงานที่ผสาน และคุณสามารถป้อนด้วยตนเอง

## <a name="what-if-the-dimensions-are-different-on-the-batches-that-i-want-to-merge"></a>จะเป็นอย่างไรถ้ามิติแตกต่างกันในชุดงานที่ฉันต้องการผสาน
นี่คือวิธีที่มิติของผลิตภัณฑ์ มิติการติดตาม และมิติการจัดเก็บจะได้รับการจัดการ:

-   **มิติของผลิตภัณฑ์** – มิติของผลิตภัณฑ์ทั้งหมดสำหรับสินค้าที่เลือกต้องเหมือนกัน คุณไม่สามารถดำเนินการผสานชุดงานระหว่างมิติของผลิตภัณฑ์ได้
-   **มิติการติดตาม** – หมายเลขชุดงานใหม่ถูกสร้างขึ้นโดยอัตโนมัติถ้ากลุ่มหมายเลขชุดงานถูกระบุสำหรับสินค้า ถ้ากลุ่มหมายเลขชุดงานไม่ได้กำหนดให้กับสินค้า คุณสามารถเลือกมีชุดงานที่มีอยู่ หรือป้อนหมายเลขด้วยตนเอง หมายเลขลำดับประจำสินค้าถูกโอนย้ายจากชุดงานต้นทางไปยังรายการสมุดรายวันสินค้าคงคลังสำหรับชุดงานที่ผสาน
-   **มิติการจัดเก็บ** – มิติการจัดเก็บไซต์และคลังสินค้าต้องเหมือนกันสำหรับชุดงานต้นทางและชุดงานที่ผสานทั้งหมด อย่างไรก็ตาม คุณสามารถระบุสถานที่ตั้งใหม่และรหัสแท่นวางสินค้าสำหรับชุดงานที่ผสาน

## <a name="how-does-posting-work"></a>การลงรายการบัญชีทำงานอย่างไร
ลงรายการบัญชีทำงานในสองลักษณะ ขึ้นอยู่กับว่าคุณกำลังใช้กระบวนการอนุมัติสำหรับสมุดรายวันหรือไม่ คุณสามารถใช้การดำเนินการ **โอนย้ายไปที่สมุดรายวัน** และ **ลงรายการบัญชีผสานชุดงาน** เพื่อโอนย้ายการผสานชุดงานไปยังสมุดรายวันที่ได้รับการตรวจสอบและลงรายการบัญชีแล้ว หรือคุณสามารถลงรายการบัญชีการผสานชุดงานได้โดยตรง ความแตกต่างหลักระหว่างสองการดำเนินการคือ การโอนย้ายไปสมุดรายวันไม่ได้ลงรายการบัญชีการผสานชุดงาน การดำเนินการทั้งสองจะสร้างชุดงานใหม่ถ้าไม่ได้เลือกชุดงานที่มีอยู่ ปรับปรุงรายละเอียดชุดงานทั้งหมดและค่าแอททริบิวต์ และสร้างสมุดรายวันสินค้าคงคลัง

-   **โอนย้ายไปที่สมุดรายวัน** – โอนย้ายรายละเอียดการผสานชุดงานไปยังสมุดรายวันสินค้าคงคลังใหม่ ถ้าคุณได้ตั้งค่าการจองสินค้าโดยอัตโนมัติ ปริมาณในชุดงานต้นทางจะถูกจองไว้ รายละเอียดของการผสานชุดงานไม่สามารถเปลี่ยนได้ เมื่อต้องการแก้ไขการผสานชุดงาน คุณต้องลบสมุดรายวันออก สามารถใช้สมุดรายวันเป็นงานที่พนักงานคนอื่นต้องดำเนินการในภายหลัง การจองปริมาณชุดงานไปยังบรรทัดสมุดรายวันได้รับการป้องกัน การปันส่วนนี้ช่วยให้ผู้วางแผนคุณภาพหรือผู้จัดการคลังสินค้าสามารถสร้างงานสำหรับพนักงานของเขาหรือเธอได้
-   **ลงรายการบัญชีการผสานชุดงาน** – ลงรายการบัญชีชุดงานผสานโดยตรง สามารถดำเนินการนี้หลังจากเกิดการผสานที่มีอยู่จริงแล้ว

คุณสามารถอนุมัติสมุดรายวันสินค้าคงคลังสำหรับการผสานชุดงานจากหน้ารายการ **การผสานชุดงานทั้งหมด** คลิก **สมุดรายวัน** &gt; **ลงรายการบัญชี** หลังจากที่สมุดรายวันมีลงรายการบัญชี คุณไม่สามารถเปลี่ยนรายละเอียดในชุดงานที่ผสาน  หลังจากที่คุณโอนย้ายการผสานชุดงานไปยังสมุดรายวันสินค้าคงคลัง คุณสามารถเปลี่ยนรายละเอียดเฉพาะเมื่อมีการลบสมุดรายวัน

## <a name="after-i-merged-a-catchweight-item-why-cant-i-see-the-catchweight-information-in-the-inventory-journal"></a>หลังจากที่ได้ผสานสินค้าตามน้ำหนักจริง เหตุใดฉันจึงไม่เห็นข้อมูลน้ำหนักจริงในสมุดรายวันสินค้าคงคลัง
คุณสามารถผสานชุดงานของสินค้าตามน้ำหนักจริงได้เหมือนกับสินค้าอื่นๆทั้งหมด  อย่างไรก็ตาม ข้อมูลน้ำหนักจริงจะไม่แสดงบนสมุดรายวันสินค้าคงคลัง เราขอแนะนำให้คุณตรวจสอบข้อมูลน้ำหนักจริง ก่อนที่คุณโอนย้ายการผสานชุดงานไปยังสมุดรายวันสินค้าคงคลัง
