# <a name="nonconformance-management"></a><span data-ttu-id="b4034-104">การจัดการที่ไม่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b4034-104">Nonconformance management</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="b4034-105">บทความนี้อธิบายการตั้งค่าพื้นฐานที่จำเป็นเพื่อใช้ความไม่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b4034-105">This article describes the basic setup that is required in order to use nonconformances.</span></span> <span data-ttu-id="b4034-106">จำเป็นต้องมีการตั้งค่าเพิ่มเติม ถ้าคุณต้องการใช้ใบสั่งตรวจสอบคุณภาพ</span><span class="sxs-lookup"><span data-stu-id="b4034-106">Additional setup is required if you want to use quality orders.</span></span>

<span data-ttu-id="b4034-107">ในการเปิดใช้งานการจัดการความไม่สอดคล้องกัน ให้ทำตามขั้นตอนเหล่านี้</span><span class="sxs-lookup"><span data-stu-id="b4034-107">To enable nonconformance management, follow these steps:</span></span>

1.  <span data-ttu-id="b4034-108">กำหนดพารามิเตอร์การจัดการสินค้าคงคลังและคลังสินค้าที่เกี่ยวข้องกับความไม่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b4034-108">Define the Inventory and warehouse management parameters that are related to nonconformances:</span></span>
    -   <span data-ttu-id="b4034-109">ตั้งค่าตัวเลือก **การจัดการคุณภาพการใช้** เป็น **ใช่**</span><span class="sxs-lookup"><span data-stu-id="b4034-109">Set the **Use quality management** option to **Yes**.</span></span>
    -   <span data-ttu-id="b4034-110">ในฟิลด์ **อัตราต่อชั่วโมง** ป้อนอัตราค่าแรงต่อชั่วโมงในสกุลเงินท้องถิ่น</span><span class="sxs-lookup"><span data-stu-id="b4034-110">In the **Hourly rate** field, enter an hourly labor rate in the local currency.</span></span> <span data-ttu-id="b4034-111">อัตราต่อชั่วโมงจะถูกใช้ในการคำนวณต้นทุนสำหรับการดำเนินงานที่เกี่ยวข้องกับความไม่สอดคล้อง</span><span class="sxs-lookup"><span data-stu-id="b4034-111">The hourly rate is used to calculate costs for operations that are related to a nonconformance.</span></span> <span data-ttu-id="b4034-112">อัตราต่อชั่วโมงและต้นทุนที่คำนวณได้ให้การอ้างอิงข้อมูลสำหรับความไม่สอดคล้อง</span><span class="sxs-lookup"><span data-stu-id="b4034-112">The hourly rate and calculated costs provide reference information for a nonconformance.</span></span> <span data-ttu-id="b4034-113">จะไม่สัมพันธ์กับฟังก์ชันอื่น</span><span class="sxs-lookup"><span data-stu-id="b4034-113">They don't interact with other functionality.</span></span>
    -   <span data-ttu-id="b4034-114">ใช้แท็บ **การจัดการคุณภาพ** บนหน้า **การตั้งค่ารายงาน** เพื่อกำหนดชนิดของเอกสารที่จะพิมพ์</span><span class="sxs-lookup"><span data-stu-id="b4034-114">Use the **Quality management** tab on the **Report setup** page to define the type of document to print.</span></span> <span data-ttu-id="b4034-115">คุณสามารถพิมพ์รายงานความไม่สอดคล้อง ป้ายความไม่สอดคล้อง หรือรายงานการแก้ไข</span><span class="sxs-lookup"><span data-stu-id="b4034-115">You can print a nonconformance report, a nonconformance tag, or a correction report.</span></span> <span data-ttu-id="b4034-116">คุณสามารถกำหนดเรกคอร์ดมากกว่าหนึ่งเรกคอร์ด เพื่อพิมพ์เอกสารชนิดที่แตกต่างกันบนรายงาน หรือ เพื่อพิมพ์บันทึกภายในและภายนอก</span><span class="sxs-lookup"><span data-stu-id="b4034-116">You can define more than one record to print different document types on a report, or to print internal and external notes.</span></span> <span data-ttu-id="b4034-117">คุณอาจพบว่ามีประโยชน์เมื่อใช้หน้า **ชนิดเอกสาร** เพื่อกำหนดชนิดเอกสารเฉพาะสำหรับความไม่สอดคล้องและชนิดเอกสารเฉพาะสำหรับการแก้ไข</span><span class="sxs-lookup"><span data-stu-id="b4034-117">You might find it helpful to use the **Document type** page to define a unique document type for nonconformances and a unique document type for corrections.</span></span> <span data-ttu-id="b4034-118">ตัวอย่างเช่น ต้องการป้อนหมายเหตุเกี่ยวกับความไม่สอดคล้อง โดยการใช้ชนิดเอกสารเฉพาะสำหรับความไม่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b4034-118">For example, want to enter notes about a nonconformance by using the unique document type for nonconformances.</span></span> <span data-ttu-id="b4034-119">ในกรณีนี้ ระบุชนิดเอกสารเฉพาะในตัวเลือกรายงาน</span><span class="sxs-lookup"><span data-stu-id="b4034-119">In this case, identify the unique document type in the report options.</span></span>
    -   <span data-ttu-id="b4034-120">เปิดใช้งานลำดับหมายเลขสำหรับการอ้างอิงความไม่สอดคล้องกันและการแก้ไข</span><span class="sxs-lookup"><span data-stu-id="b4034-120">Enable number sequences for nonconformance and correction references.</span></span>

2.  <span data-ttu-id="b4034-121">เปิดใช้งานการอนุมัติของผู้ใช้สำหรับความไม่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b4034-121">Enable user approval of nonconformances.</span></span> <span data-ttu-id="b4034-122">ใช้ฟิลด์ **ชื่อ** บนหน้า **ผู้ใช้** เพื่อมอบหมายพนักงานให้กับผู้ใช้แต่ละรายที่ต้องอนุมัติความไม่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b4034-122">Use the **Name** field on the **Users** page to assign an employee to each user who must approve a nonconformance.</span></span> <span data-ttu-id="b4034-123">ระบบใช้พนักงานที่เปลี่ยนสถานะของความไม่สอดคล้องในการติดตามประวัติความไม่สอดคล้อง</span><span class="sxs-lookup"><span data-stu-id="b4034-123">The system uses the employees who change the status of a noncomformance to track the nonconformance history.</span></span> <span data-ttu-id="b4034-124">ผู้ใช้ไม่สามารถอนุมัติความไม่สอดคล้องเว้นแต่จะมีการมอบหมายตัวระบุพนักงาน</span><span class="sxs-lookup"><span data-stu-id="b4034-124">Users can't approve a nonconformance unless they have been assigned an employee identifier.</span></span>
3.  <span data-ttu-id="b4034-125">กำหนดชนิดของปัญหาที่จะกำหนดให้กับความไม่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b4034-125">Define the problem types that will be assigned to nonconformances.</span></span> <span data-ttu-id="b4034-126">ใช้หน้า **ชนิดปัญหา** เพื่อกำหนดการจัดประเภทของปัญหาด้านคุณภาพ ซึ่งพบในความไม่สอดคล้องกันชนิดต่างๆ</span><span class="sxs-lookup"><span data-stu-id="b4034-126">Use the **Problem types** page to define a classification of quality problems that are encountered for the various nonconformance types.</span></span> <span data-ttu-id="b4034-127">คุณสามารถตั้งค่าชนิดของความไม่สอดคล้องกันต่อไปนี้: **ภายใน** **ลูกค้า** **ผู้จัดจำหน่าย** **คำร้องขอการบริการ** **การผลิต**และ **การผลิตผลิตภัณฑ์ร่วม**</span><span class="sxs-lookup"><span data-stu-id="b4034-127">You can set up the following nonconformance types: **Internal**, **Customer**, **Vendor**, **Service request**, **Production**, and **Co-product production**.</span></span> <span data-ttu-id="b4034-128">ใช้หน้า **ชนิดความไม่สอดคล้องกัน** เพื่ออนุมัติชนิดของปัญหาที่จะใช้ในอย่างน้อยหนึ่งชนิดความไม่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b4034-128">Use the **Non conformance types** page to authorize a problem type to be used in one or more nonconformance types.</span></span> <span data-ttu-id="b4034-129">ตัวอย่างเช่น ชนิดของปัญหาที่เกี่ยวข้องกับรหัสความบกพร่องอาจนำไปใช้กับชนิดของความไม่สอดคล้องทั้งหมด ในขณะที่ชนิดปัญหาที่เกี่ยวข้องกับคำร้องเรียนของลูกค้าอาจใช้กับเฉพาะชนิดความไม่สอดคล้องของ **ลูกค้า** และ **คำขอการบริการ**</span><span class="sxs-lookup"><span data-stu-id="b4034-129">For example, a problem type that is related to a defect code might apply to all nonconformance types, whereas a problem type that is related to customer complaints might apply only to the **Customer** and **Service request** nonconformance types.</span></span>
4.  <span data-ttu-id="b4034-130">กำหนดโซนตรวจสอบสินค้าเพื่อให้คำแนะนำเกี่ยวกับการจัดการที่ควรทำกับวัสดุที่บกพร่อง</span><span class="sxs-lookup"><span data-stu-id="b4034-130">Define quarantine zones to provide guidance about defective material should be handled.</span></span> <span data-ttu-id="b4034-131">ใช้หน้า **โซนตรวจสอบสินค้า** เพื่อกำหนดโซนที่สามารถกำหนดให้กับความไม่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b4034-131">Use the **Quarantine zones** page to define zones that can be assigned to a nonconformance.</span></span> <span data-ttu-id="b4034-132">ป้ายความไม่สอดคล้องกันที่พิมพ์ไว้จะแสดงโซนตรวจสอบสินค้าที่กำหนดไว้และข้อมูลเกี่ยวกับการใช้งาน เพื่อให้คำแนะนำวิธีการจัดการกับวัสดุที่บกพร่อง</span><span class="sxs-lookup"><span data-stu-id="b4034-132">The printed nonconformance tag will show the assigned quarantine zone and information about usage, to provide guidance about how to handle defective material.</span></span> <span data-ttu-id="b4034-133">โซนอาจสอดคล้องกับสถานที่เก็บสินค้าคงคลังหรือทรัพยากรการดำเนินงาน</span><span class="sxs-lookup"><span data-stu-id="b4034-133">The zones might correspond to inventory locations or operations resources.</span></span>
5.  <span data-ttu-id="b4034-134">กำหนดชนิดของการวินิจฉัยที่จะกำหนดให้กับการแก้ไข</span><span class="sxs-lookup"><span data-stu-id="b4034-134">Define the diagnostic types that will be assigned to corrections.</span></span> <span data-ttu-id="b4034-135">ใช้หน้า **ชนิดการวินิจฉัย** เพื่อกำหนดการจัดประเภทการดำเนินการวินิจฉัย</span><span class="sxs-lookup"><span data-stu-id="b4034-135">Use the **Diagnostic types** page to define a classification of diagnostic actions.</span></span> <span data-ttu-id="b4034-136">การแก้ไขระบุชนิดของการดำเนินการวินิจฉัยซึ่งควรจะดำเนินการสำหรับความไม่สอดคล้องที่ได้รับอนุมัติ และใครที่ควรดำเนินการนั้น</span><span class="sxs-lookup"><span data-stu-id="b4034-136">A correction specifies what type of diagnostic action should be taken for an approved nonconformance, and who should take that action.</span></span> <span data-ttu-id="b4034-137">ยังระบุวันเสร็จสมบูรณ์ที่ร้องขอและวันเสร็จสมบูรณ์ที่วางแผนไว้ด้วย</span><span class="sxs-lookup"><span data-stu-id="b4034-137">It also specifies the requested completion date and the planned completion date.</span></span>
6.  <span data-ttu-id="b4034-138">กำหนดการดำเนินงานที่เกี่ยวข้องซึ่งจะกำหนดให้กับความไม่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b4034-138">Define the related operations that will be assigned to nonconformances.</span></span> <span data-ttu-id="b4034-139">ใช้หน้า **ดำเนินงาน** เพื่อกำหนดการจัดประเภทของงานที่สามารถดำเนินการสำหรับความไม่สอดคล้องที่อนุมัติแล้ว</span><span class="sxs-lookup"><span data-stu-id="b4034-139">Use the **Operations** page to define a classification of the work that can be performed for an approved nonconformance.</span></span> <span data-ttu-id="b4034-140">เมื่อคุณมอบหมายการดำเนินงานที่เกี่ยวข้องให้กับความไม่สอดคล้อง คุณสามารถให้รายละเอียดข้อมูล เช่น ข้อมูลเกี่ยวกับวัสดุที่เกี่ยวข้อง ชั่วโมงแรงงาน และค่าธรรมเนียมเบ็ดเตล็ดที่จำเป็นเพื่อทำการดำเนินงาน</span><span class="sxs-lookup"><span data-stu-id="b4034-140">When you assign a related operation to a nonconformance, you can provide detailed information, such as information about the associated material, labor hours, and miscellaneous charges that are required in order to perform the operation.</span></span> <span data-ttu-id="b4034-141">ข้อมูลนี้จะถูกใช้ในการคำนวณต้นทุนที่ประเมินสำหรับการดำเนินงาน</span><span class="sxs-lookup"><span data-stu-id="b4034-141">This information is used to calculate an estimated cost for the operation.</span></span> <span data-ttu-id="b4034-142">ข้อมูลรายละเอียดและต้นทุนที่ประเมินมีสำหรับการอ้างอิง</span><span class="sxs-lookup"><span data-stu-id="b4034-142">The detailed information and estimated costs are for reference.</span></span> <span data-ttu-id="b4034-143">การดำเนินงานที่เกี่ยวข้องสำหรับคุณภาพจะแตกต่างจากการดำเนินงานที่สามารถกำหนดให้กับกระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="b4034-143">The related operations for quality differ from the operations that can be defined for a production route.</span></span>
