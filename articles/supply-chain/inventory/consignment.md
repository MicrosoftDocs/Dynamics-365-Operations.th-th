# <a name="set-up-consignment"></a><span data-ttu-id="76f62-103">ตั้งค่าปริมาณการใช้วัสดุ</span><span class="sxs-lookup"><span data-stu-id="76f62-103">Set up consignment</span></span>

[!INCLUDE [banner](../includes/banner.md)]

<span data-ttu-id="76f62-104">หัวข้อนี้อธิบายวิธีการใช้กระบวนการสินค้าคงคลังที่มีการส่งมอบขาเข้า</span><span class="sxs-lookup"><span data-stu-id="76f62-104">This topic explains how to use the inbound consignment inventory processes.</span></span>

<span data-ttu-id="76f62-105">สินค้าคงคลังที่มีการส่งมอบคือสินค้าคงคลังที่เป็นเจ้าของโดยผู้จัดจำหน่าย แต่เก็บอยู่ที่ไซต์ของคุณ</span><span class="sxs-lookup"><span data-stu-id="76f62-105">Consignment inventory is inventory that’s owned by a vendor, but stored at your site.</span></span> <span data-ttu-id="76f62-106">เมื่อคุณพร้อมที่จะใช้หรือใช้สินค้าคงคลัง คุณจะต้องเป็นเจ้าของสินค้าคงคลังดังกล่าว</span><span class="sxs-lookup"><span data-stu-id="76f62-106">When you’re ready to consume or use the inventory, you take over the ownership of the inventory.</span></span> <span data-ttu-id="76f62-107">หัวข้อนี้มีข้อมูลเกี่ยวกับวิธีการรับปริมาณคงคลังคงเหลือที่ผู้จัดจำหน่ายเป็นเจ้าของจริงโดยไม่ต้องสร้างธุรกรรมบัญชีแยกประเภททั่วไป วิธีการเริ่มต้นกระบวนการผลิตที่มีสินค้าคงคลังที่ผู้จัดจำหน่ายเป็นเจ้าของที่สามารถจองได้จริง</span><span class="sxs-lookup"><span data-stu-id="76f62-107">This topic includes information about how to physically receive vendor-owned inventory on-hand without creating general ledger transactions, how to start a production process where the vendor-owned inventory can be physically reserved.</span></span> <span data-ttu-id="76f62-108">และวิธีการเปลี่ยนความเป็นเจ้าของของวัตถุดิบเพื่อให้สามารถประมวลผลปริมาณการใช้โดยเป็นส่วนหนึ่งของการประมวลผลใบสั่งผลิต</span><span class="sxs-lookup"><span data-stu-id="76f62-108">and how to change the ownership of the raw material in order to be able to process the consumption as part of the production order processing.</span></span> <span data-ttu-id="76f62-109">นอกจากนี้ยังมีข้อมูลบางอย่างเกี่ยวกับวิธีการที่ผู้จัดจำหน่ายสามารถตรวจสอบปริมาณการใช้ของสินค้าคงคลังของพวกเขาโดยใช้อินเทอร์เฟสการทำงานร่วมกันกับผู้จัดจำหน่าย</span><span class="sxs-lookup"><span data-stu-id="76f62-109">There’s also some information about how vendors can monitor consumption of their inventory using the vendor collaboration interface.</span></span> 

## <a name="overview-of-the-consignment-process"></a><span data-ttu-id="76f62-110">ภาพรวมของกระบวนการส่งมอบ</span><span class="sxs-lookup"><span data-stu-id="76f62-110">Overview of the consignment process</span></span>
<span data-ttu-id="76f62-111">ในสถานการณ์ตัวอย่างนี้ บริษัท USMF มีข้อตกลงการส่งมอบกับผู้จัดจำหน่าย US-104 สำหรับวัตถุดิบ M9211CI</span><span class="sxs-lookup"><span data-stu-id="76f62-111">In this example scenario, company USMF has a consignment agreement with vendor US-104 for the raw material M9211CI.</span></span>

2.  <span data-ttu-id="76f62-114">ผู้จัดจำหน่ายได้รับแจ้งเกี่ยวกับการจัดส่งที่คาดไว้</span><span class="sxs-lookup"><span data-stu-id="76f62-114">The vendor gets informed about the expected delivery.</span></span> <span data-ttu-id="76f62-115">ซึ่งสามารถเกิดขึ้นได้ในหนึ่งในสามวิธี:</span><span class="sxs-lookup"><span data-stu-id="76f62-115">This can happen in one of three ways:</span></span>
    -   <span data-ttu-id="76f62-116">บุคคลที่ทำงานที่ USMF ส่งข้อมูลใบสั่งให้กับผู้จัดจำหน่าย</span><span class="sxs-lookup"><span data-stu-id="76f62-116">Someone working at USMF sends the order information to the vendor.</span></span>
    -   <span data-ttu-id="76f62-117">ผู้จัดจำหน่ายสามารถตรวจสอบปริมาณคงคลังคงเหลือโดยใช้อินเทอร์เฟสการทำงานร่วมกันกับผู้จัดจำหน่าย</span><span class="sxs-lookup"><span data-stu-id="76f62-117">The vendor can monitor the expected inventory on-hand using the vendor collaboration interface.</span></span>
    -   <span data-ttu-id="76f62-118">บุคคลที่ทำงานที่ USMF กรองข้อมูลในหน้า **ปริมาณคงคลังคงเหลือ** เพื่อแสดงเฉพาะเรกคอร์ดสำหรับผู้จัดจำหน่าย US-104 ที่มีสถานะการรับสินค้าเป็น **สั่งแล้ว** แล้วส่งข้อมูลนี้ให้กับผู้จัดจำหน่าย</span><span class="sxs-lookup"><span data-stu-id="76f62-118">Someone working at USMF filters the data on the **On-hand inventory** page to show only the records for vendor US-104, where the receipt status is **Ordered**, and then sends this information to the vendor.</span></span>
3.  <span data-ttu-id="76f62-119">สินค้าคงคลังถูกจัดส่งจาก US-104 ไปยัง USMF</span><span class="sxs-lookup"><span data-stu-id="76f62-119">The inventory is delivered from US-104 to USMF.</span></span>
4.  <span data-ttu-id="76f62-120">เมื่อวัตถุดิบมาถึงที่ USMF จะมีการอัพเดตใบสั่งการเพิ่มเติมสินค้าที่มีการส่งมอบด้วยใบรับสินค้า</span><span class="sxs-lookup"><span data-stu-id="76f62-120">When the material arrives at USMF, the consignment replenishment order is updated with a product receipt.</span></span> <span data-ttu-id="76f62-121">เฉพาะปริมาณทางกายภาพของสินค้าคงคลังที่ผู้จัดจำหน่ายเป็นเจ้าของจะถูกบันทึกไว้</span><span class="sxs-lookup"><span data-stu-id="76f62-121">Only the physical quantities of the vendor-owned inventory are recorded.</span></span> <span data-ttu-id="76f62-122">ไม่มีการสร้างธุรกรรมบัญชีแยกประเภททั่วไปเนื่องจากสินค้าคงคลังยังคงมีผู้จัดจำหน่ายเป็นเจ้าของ</span><span class="sxs-lookup"><span data-stu-id="76f62-122">There are no general ledger transactions created, because the inventory is still owned by the vendor.</span></span>

9.  <span data-ttu-id="76f62-129">กระบวนการผลิตใช้วัตถุดิบผ่านทางรายการเบิกสินค้าสำหรับการผลิต</span><span class="sxs-lookup"><span data-stu-id="76f62-129">The production process consumes the raw material via a production picking list.</span></span> <span data-ttu-id="76f62-130">จะมีการอัพเดตการจองทางกายภาพโดยอัตโนมัติเพื่อแสดงให้เห็นว่าปริมาณคงคลังคงเหลือเป็นของ USMF</span><span class="sxs-lookup"><span data-stu-id="76f62-130">The physical reservation is automatically updated to reflect that the on-hand inventory is owned by USMF.</span></span>
