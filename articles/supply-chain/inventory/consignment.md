<span data-ttu-id="4f274-112">ในสถานการณ์ตัวอย่างนี้ บริษัท USMF มีข้อตกลงการส่งมอบกับผู้จัดจำหน่าย US-104 สำหรับวัตถุดิบ M9211CI</span><span class="sxs-lookup"><span data-stu-id="4f274-112">In this example scenario, company USMF has a consignment agreement with vendor US-104 for the raw material M9211CI.</span></span>

1.  <span data-ttu-id="4f274-113">มีการสร้างใบสั่งการเพิ่มเติมสินค้าที่มีการส่งมอบด้วยตนเองโดยบุคคลอื่นใน USMF ตามความต้องการที่คาดไว้</span><span class="sxs-lookup"><span data-stu-id="4f274-113">A consignment replenishment order is manually created by someone in USMF, based on the expected demand.</span></span> <span data-ttu-id="4f274-114">มีการสร้างใบสั่งสำหรับผู้จัดจำหน่าย US-104 และมีการเพิ่มบรรทัดสำหรับสินค้า MS9211CI</span><span class="sxs-lookup"><span data-stu-id="4f274-114">The order is created for vendor US-104 and a line is added for item MS9211CI.</span></span>
2.  <span data-ttu-id="4f274-115">ผู้จัดจำหน่ายได้รับแจ้งเกี่ยวกับการจัดส่งที่คาดไว้</span><span class="sxs-lookup"><span data-stu-id="4f274-115">The vendor gets informed about the expected delivery.</span></span> <span data-ttu-id="4f274-116">ซึ่งสามารถเกิดขึ้นได้ในหนึ่งในสามวิธี:</span><span class="sxs-lookup"><span data-stu-id="4f274-116">This can happen in one of three ways:</span></span>
    -   <span data-ttu-id="4f274-117">บุคคลที่ทำงานที่ USMF ส่งข้อมูลใบสั่งให้กับผู้จัดจำหน่าย</span><span class="sxs-lookup"><span data-stu-id="4f274-117">Someone working at USMF sends the order information to the vendor.</span></span>
    -   <span data-ttu-id="4f274-118">ผู้จัดจำหน่ายสามารถตรวจสอบปริมาณคงคลังคงเหลือโดยใช้อินเทอร์เฟสการทำงานร่วมกันกับผู้จัดจำหน่าย</span><span class="sxs-lookup"><span data-stu-id="4f274-118">The vendor can monitor the expected inventory on-hand using the vendor collaboration interface.</span></span>
    -   <span data-ttu-id="4f274-119">บุคคลที่ทำงานที่ USMF กรองข้อมูลในหน้า **ปริมาณคงคลังคงเหลือ** เพื่อแสดงเฉพาะเรกคอร์ดสำหรับผู้จัดจำหน่าย US-104 ที่มีสถานะการรับสินค้าเป็น **สั่งแล้ว** แล้วส่งข้อมูลนี้ให้กับผู้จัดจำหน่าย</span><span class="sxs-lookup"><span data-stu-id="4f274-119">Someone working at USMF filters the data on the **On-hand inventory** page to show only the records for vendor US-104, where the receipt status is **Ordered**, and then sends this information to the vendor.</span></span>

3.  <span data-ttu-id="4f274-120">สินค้าคงคลังถูกจัดส่งจาก US-104 ไปยัง USMF</span><span class="sxs-lookup"><span data-stu-id="4f274-120">The inventory is delivered from US-104 to USMF.</span></span>
4.  <span data-ttu-id="4f274-121">เมื่อวัตถุดิบมาถึงที่ USMF จะมีการอัพเดตใบสั่งการเพิ่มเติมสินค้าที่มีการส่งมอบด้วยใบรับสินค้า</span><span class="sxs-lookup"><span data-stu-id="4f274-121">When the material arrives at USMF, the consignment replenishment order is updated with a product receipt.</span></span> <span data-ttu-id="4f274-122">เฉพาะปริมาณทางกายภาพของสินค้าคงคลังที่ผู้จัดจำหน่ายเป็นเจ้าของจะถูกบันทึกไว้</span><span class="sxs-lookup"><span data-stu-id="4f274-122">Only the physical quantities of the vendor-owned inventory are recorded.</span></span> <span data-ttu-id="4f274-123">ไม่มีการสร้างธุรกรรมบัญชีแยกประเภททั่วไปเนื่องจากสินค้าคงคลังยังคงมีผู้จัดจำหน่ายเป็นเจ้าของ</span><span class="sxs-lookup"><span data-stu-id="4f274-123">There are no general ledger transactions created, because the inventory is still owned by the vendor.</span></span>

7.  <span data-ttu-id="4f274-126">เจ้าของของวัตถุดิบสำรองที่จะใช้ในการผลิตของวันนั้นๆ จะถูกเปลี่ยนจาก US-104 เป็น USMF</span><span class="sxs-lookup"><span data-stu-id="4f274-126">The owner of the reserved raw materials that are going to be consumed in today’s production is changed from US-104 to USMF.</span></span> <span data-ttu-id="4f274-127">ซึ่งทำได้โดยใช้สมุดรายวันการเปลี่ยนแปลงความเป็นเจ้าของสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="4f274-127">This is done using an Inventory ownership change journal.</span></span> <span data-ttu-id="4f274-128">กระบวนการนี้จะสร้างใบสั่งซื้อที่มีฟิลด์ **จุดเริ่มต้น** ที่ตั้งค่าเป็น **การส่งมอบ**</span><span class="sxs-lookup"><span data-stu-id="4f274-128">This process creates purchase orders where the **Origin** field is set to **Consignment**.</span></span>
