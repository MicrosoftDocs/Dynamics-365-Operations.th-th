# <a name="delete-a-service-bom"></a><span data-ttu-id="a517a-103">การลบ BOM การบริการ</span><span class="sxs-lookup"><span data-stu-id="a517a-103">Delete a service BOM</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="a517a-104">คลิก **การจัดการบริการ** \> **ทั่วไป** \> **ข้อตกลงการให้บริการ** \> **ข้อตกลงการให้บริการ**</span><span class="sxs-lookup"><span data-stu-id="a517a-104">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span> <span data-ttu-id="a517a-105">เลือกข้อตกลงการให้บริการที่คุณต้องการลบสูตรการผลิตบริการ (BOM)</span><span class="sxs-lookup"><span data-stu-id="a517a-105">Select the service agreement that you want to delete a service bill of materials (BOM) from.</span></span>

2.  <span data-ttu-id="a517a-106">ใน **บานหน้าต่างการดำเนินการ** ให้คลิก **วัตถุที่ให้บริการ**</span><span class="sxs-lookup"><span data-stu-id="a517a-106">On the **Action Pane**, click **Service objects**.</span></span>

3.  <span data-ttu-id="a517a-107">ในแบบฟอร์ม **วัตถุที่ให้บริการ** ให้เลือกวัตถุที่ให้บริการที่คุณต้องการลบ BOM บริการออก</span><span class="sxs-lookup"><span data-stu-id="a517a-107">In the **Service objects** form, select the service object that you want to delete a service BOM from.</span></span> <span data-ttu-id="a517a-108">คลิก **ฟังก์ชัน** และจากนั้นเลือก **ลบ BOM บริการ**</span><span class="sxs-lookup"><span data-stu-id="a517a-108">Click **Functions**, and then select **Delete service BOM**.</span></span>

4.  <span data-ttu-id="a517a-109">คลิก **ใช่** </span><span class="sxs-lookup"><span data-stu-id="a517a-109">Click **Yes**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="a517a-110">หากคุณไม่ได้ใช้ BOM เท็มเพลต คุณก็สามารถลบ BOM บริการจากแบบฟอร์ม <STRONG>BOMs เท็มเพลต</STRONG> ได้ </span><span class="sxs-lookup"><span data-stu-id="a517a-110">If you have not applied a template BOM, the service BOM can be deleted from the <STRONG>Template BOMs</STRONG> form.</span></span> <span data-ttu-id="a517a-111">หากคุณใช้ BOM เท็มเพลต คุณก็ไม่สามารถลบ BOM บริการโดยใช้วิธีนี้ได้</span><span class="sxs-lookup"><span data-stu-id="a517a-111">If you have applied a template BOM, the service BOM cannot be deleted by using this method.</span></span></P>
