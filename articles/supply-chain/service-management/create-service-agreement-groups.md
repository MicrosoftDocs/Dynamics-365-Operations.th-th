# <a name="create-service-agreement-groups"></a><span data-ttu-id="63f2b-103">สร้างกลุ่มข้อตกลงการให้บริการ</span><span class="sxs-lookup"><span data-stu-id="63f2b-103">Create service agreement groups</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="63f2b-104">กลุ่มข้อตกลงการให้บริการมีประโยชน์สำหรับการเรียงลำดับและการกรองข้อมูลข้อตกลงการให้บริการ</span><span class="sxs-lookup"><span data-stu-id="63f2b-104">Service agreement groups are useful for sorting and filtering service agreements.</span></span>

1. <span data-ttu-id="63f2b-105">คลิก **การจัดการบริการ** \> **ตั้งค่า** \> **ข้อตกลงการให้บริการ** \> **กลุ่มข้อตกลงการให้บริการ**</span><span class="sxs-lookup"><span data-stu-id="63f2b-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service agreement groups**.</span></span>
2. <span data-ttu-id="63f2b-106">กด **Alt+N** เพื่อสร้างกลุ่มข้อตกลงการให้บริการใหม่</span><span class="sxs-lookup"><span data-stu-id="63f2b-106">Press **Alt+N** to create a new service agreement group.</span></span>
3. <span data-ttu-id="63f2b-107">ในฟิลด์ **กลุ่มข้อตกลงการให้บริการ** ป้อน ID สำหรับกลุ่มข้อตกลงการให้บริการ</span><span class="sxs-lookup"><span data-stu-id="63f2b-107">In the **Service agreement group** field, enter an ID for the service agreement group.</span></span>
4. <span data-ttu-id="63f2b-108">ในฟิลด์ **คำอธิบาย** ป้อนคำอธิบายสำหรับกลุ่มข้อตกลงการให้บริการ</span><span class="sxs-lookup"><span data-stu-id="63f2b-108">In the **Description** field, enter a description for the service agreement group.</span></span>
5. <span data-ttu-id="63f2b-109">ปิดหน้าเพื่อบันทึกการเปลี่ยนแปลงของคุณ</span><span class="sxs-lookup"><span data-stu-id="63f2b-109">Close the page to save your changes.</span></span>

> [!NOTE]
> <span data-ttu-id="63f2b-110">ถ้ามีการกำหนดกลุ่มข้อตกลงการให้บริการในฟิลด์ **กลุ่มข้อตกลงการให้บริการ** ของหน้า **พารามิเตอร์การจัดการบริการ** โปรแกรมจะแนะนำกลุ่มนี้สำหรับข้อตกลงการให้บริการใหม่ทั้งหมดที่คุณสร้าง</span><span class="sxs-lookup"><span data-stu-id="63f2b-110">If a service agreement group is defined in the **Service agreement group** field of the **Service management parameters** page, the program suggests this group for all new service agreements that you create.</span></span>
