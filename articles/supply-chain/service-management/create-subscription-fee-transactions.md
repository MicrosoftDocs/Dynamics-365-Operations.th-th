# <a name="create-subscription-fee-transactions"></a><span data-ttu-id="06532-102">สร้างธุรกรรมค่าธรรมเนียมสำหรับการบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="06532-102">Create subscription fee transactions</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="06532-103">คลิก **การจัดการงานบริการ** \> **ทั่วไป** \> **การบอกรับเป็นสมาชิกการบริการ** \> **การบอกรับเป็นสมาชิกการบริการทั้งหมด**</span><span class="sxs-lookup"><span data-stu-id="06532-103">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="06532-104">ในหน้ารายการ **การบอกรับเป็นสมาชิกการบริการทั้งหมด** ให้เลือกการบอกรับเป็นสมาชิกซึ่งคุณต้องการสร้างธุรกรรมค่าธรรมเนียมการบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="06532-104">On the **All service subscriptions** list page, select the subscription for which you want to create a subscription fee transaction.</span></span>

3.  <span data-ttu-id="06532-105">ใน **บานหน้าต่างการดำเนินการ** บนแท็บ **การบอกรับเป็นสมาชิก** คลิก **ค่าธรรมเนียมการบอกรับเป็นสมาชิก**</span><span class="sxs-lookup"><span data-stu-id="06532-105">On the **Action Pane**, on the **Subscription** tab, click **Subscription fees**.</span></span>

4.  <span data-ttu-id="06532-106">ในแบบฟอร์ม **ค่าธรรมเนียมการบอกรับเป็นสมาชิก** ให้ป้อนข้อมูลที่เหมาะสมในฟิลด์</span><span class="sxs-lookup"><span data-stu-id="06532-106">On the **Create subscription fee** form, enter the appropriate information in the fields.</span></span>

5.  <span data-ttu-id="06532-107">คลิก **ตกลง** </span><span class="sxs-lookup"><span data-stu-id="06532-107">Click **OK**.</span></span>
