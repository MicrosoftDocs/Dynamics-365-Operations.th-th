# <a name="copy-service-templates-lines"></a><span data-ttu-id="b7ed6-103">การคัดลอกบรรทัดเท็มเพลตการบริการ</span><span class="sxs-lookup"><span data-stu-id="b7ed6-103">Copy service templates lines</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="b7ed6-104">คุณสามารถคัดลอกข้อมูลเท็มเพลตไปไว้ยังข้อตกลงการให้บริการหรือใบสั่งบริการได้</span><span class="sxs-lookup"><span data-stu-id="b7ed6-104">Template information can be copied to either a service agreement or a service order.</span></span>

## <a name="copy-service-template-lines-into-a-service-agreement"></a><span data-ttu-id="b7ed6-105">การคัดลอกบรรทัดเท็มเพลตการบริการไปไว้ยังข้อตกลงการให้บริการ</span><span class="sxs-lookup"><span data-stu-id="b7ed6-105">Copy service template lines into a service agreement</span></span>

1. <span data-ttu-id="b7ed6-106">คลิก **ข้อตกลงการให้บริการ**</span><span class="sxs-lookup"><span data-stu-id="b7ed6-106">Click **Service agreements**.</span></span>
2. <span data-ttu-id="b7ed6-107">ดับเบิลคลิกที่ข้อตกลงการให้บริการเพื่อเปิดหน้า **ข้อตกลงการให้บริการ**</span><span class="sxs-lookup"><span data-stu-id="b7ed6-107">Double-click a service agreement to open the **Service agreements** page.</span></span>
3. <span data-ttu-id="b7ed6-108">คลิก **รายการเท็มเพลต** เพื่อเปิดหน้า **คัดลอกรายการเท็มเพลตการบริการ**</span><span class="sxs-lookup"><span data-stu-id="b7ed6-108">Click **Template lines** to open the **Copy service template lines** page.</span></span>
4. <span data-ttu-id="b7ed6-109">บนแท็บ **การควบคุมแผนภูมิ** ให้ขยายแผนภูมิ ค้นหาเท็มเพลตที่มีรายการที่คุณต้องการจะคัดลอกไปยังข้อตกลงการให้บริการ และจากนั้นเลือกเท็มเพลตการบริการ</span><span class="sxs-lookup"><span data-stu-id="b7ed6-109">On the **Tree control** tab, expand the tree, find the template that contains the lines that you want to copy to the service agreement, and then select the service template.</span></span>
5. <span data-ttu-id="b7ed6-110">บนแท็บ **รายการ** เลือกกล่องกาเครื่องหมาย **เลือก** สำหรับรายการเท็มเพลตการให้บริการที่คุณต้องการคัดลอก</span><span class="sxs-lookup"><span data-stu-id="b7ed6-110">On the **Lines** tab, select the **Select** check box for the service template lines that you want to copy.</span></span>
6. <span data-ttu-id="b7ed6-111">คลิก **ตกลง** </span><span class="sxs-lookup"><span data-stu-id="b7ed6-111">Click **OK**.</span></span>
7. <span data-ttu-id="b7ed6-112">ปิดหน้าเพื่อบันทึกการเปลี่ยนแปลงของคุณ</span><span class="sxs-lookup"><span data-stu-id="b7ed6-112">Close the pages to save your changes.</span></span>

<span data-ttu-id="b7ed6-113">ตอนนี้จะมีการคัดลอกรายการเท็มเพลตการให้บริการไปยังข้อตกลงการให้บริการ พร้อมกับออบเจ็กต์การบริการและความสัมพันธ์ของภารกิจการให้บริการที่แนบมา</span><span class="sxs-lookup"><span data-stu-id="b7ed6-113">The service template lines are now copied to the service agreement, together with the attached service object and service task relations.</span></span>
