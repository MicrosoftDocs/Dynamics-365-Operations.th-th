# <a name="create-a-service-template-group"></a>การสร้างกลุ่มเท็มเพลตการบริการ  

[!include [banner](../includes/banner.md)]

คุณสามารถใช้กลุ่มเท็มเพลตการบริการในการเรียงและกรองข้อมูลเท็มเพลตการบริการ

1. คลิก **การจัดการงานบริการ** \> **ตั้งค่า** \> **ข้อตกลงการให้บริการ** \> **กลุ่มเท็มเพลตการบริการ**
2. สร้างกลุ่มเท็มเพลตการบริการใหม่
3. ป้อน ID กลุ่ม คำอธิบาย และกลุ่มเท็มเพลตการบริการหลัก หากมี
4. บันทึกกลุ่มเท็มเพลตการบริการ
5. ตั้งค่าแถบเลื่อน **ตัวควบคุมแผนภูมิ** เป็น **ใช่** เพื่อเปิดมุมมองแผนภูมิ และดูกลุ่มเท็มเพลตการบริการ เป็นลำดับชั้นในมุมมองแผนภูมิ หรือเป็นภาพรวม อย่างใดอย่างหนึ่ง
