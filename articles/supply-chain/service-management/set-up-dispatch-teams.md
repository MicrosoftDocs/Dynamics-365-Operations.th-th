[!include [banner](../includes/banner.md)]


<span data-ttu-id="9fca8-104">หัวข้อนี้อธิบายวิธีการสร้างทีมงานการจัดส่ง </span><span class="sxs-lookup"><span data-stu-id="9fca8-104">This topic describes how to create a dispatch team.</span></span> <span data-ttu-id="9fca8-105">หลังจากที่คุณสร้างทีมงานการจัดส่ง คุณสามารถกำหนดผู้ปฏิบัติงานกับทีมงาน </span><span class="sxs-lookup"><span data-stu-id="9fca8-105">After you create a dispatch team, you can assign workers to the team.</span></span> <span data-ttu-id="9fca8-106">เมื่อคุณสร้างคำสั่งบริการ คุณสามารถเชื่อมโยงเป็นสมาชิกของทีมงานการจัดส่งกับกิจกรรมในใบสั่งบริการ</span><span class="sxs-lookup"><span data-stu-id="9fca8-106">When you create a service order, you can then relate a member of a dispatch team to an activity in the service order.</span></span>

1.  <span data-ttu-id="9fca8-107">คลิก **การจัดการงานบริการ** \> **ตั้งค่า** \> **ทีมงานการจัดส่ง**</span><span class="sxs-lookup"><span data-stu-id="9fca8-107">Click **Service management** \> **Setup** \> **Dispatch teams**.</span></span>

2.  <span data-ttu-id="9fca8-108">ในแบบฟอร์ม **ทีมงานการจัดส่ง** ให้คลิก **สร้าง** เพื่อสร้างทีมงานใหม่</span><span class="sxs-lookup"><span data-stu-id="9fca8-108">In the **Dispatch teams** form, click **New** to create a new team.</span></span>

4.  <span data-ttu-id="9fca8-110">ในฟิลด์ **เจ้าของ** ให้เลือกพนักงานที่รับผิดชอบทีมงานนี้</span><span class="sxs-lookup"><span data-stu-id="9fca8-110">In the **Owner** field, select the employee who is responsible for this team.</span></span>
    

    > [!NOTE]

5.  <span data-ttu-id="9fca8-113">ไม่จำเป็นต้องระบุ: ในฟิลด์ **คำอธิบาย** ให้ป้อนคำอธิบายแบบย่อสำหรับทีมงานการจัดส่งนี้</span><span class="sxs-lookup"><span data-stu-id="9fca8-113">Optional: In the **Description** field, enter a brief description for this dispatch team.</span></span>
