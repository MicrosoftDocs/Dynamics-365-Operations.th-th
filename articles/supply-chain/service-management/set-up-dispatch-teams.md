# <a name="set-up-dispatch-teams"></a>ตั้งค่าทีมงานการจัดส่ง 

[!include [banner](../includes/banner.md)]


หัวข้อนี้อธิบายวิธีการสร้างทีมงานการจัดส่ง  หลังจากที่คุณสร้างทีมงานการจัดส่ง คุณสามารถกำหนดผู้ปฏิบัติงานกับทีมงาน  เมื่อคุณสร้างคำสั่งบริการ คุณสามารถเชื่อมโยงเป็นสมาชิกของทีมงานการจัดส่งกับกิจกรรมในใบสั่งบริการ

1.  คลิก **การจัดการงานบริการ** \> **ตั้งค่า** \> **ทีมงานการจัดส่ง**

2.  ในแบบฟอร์ม **ทีมงานการจัดส่ง** ให้คลิก **สร้าง** เพื่อสร้างทีมงานใหม่

3.  ในฟิลด์ **ทีมงานการจัดส่ง** ให้ป้อนชื่อที่ไม่ซ้ำกันสำหรับทีมงานการจัดส่ง

4.  ในฟิลด์ **เจ้าของ** ให้เลือกพนักงานที่รับผิดชอบทีมงานนี้
    

    > [!NOTE]
    > <P>คุณสามารถเลือกเฉพาะพนักงานในฟิลด์นี้  คุณไม่สามารถเลือกผู้จัดจำหน่าย</P>



5.  ไม่จำเป็นต้องระบุ: ในฟิลด์ **คำอธิบาย** ให้ป้อนคำอธิบายแบบย่อสำหรับทีมงานการจัดส่งนี้


> [!NOTE]
> <P>คุณสามารถกำหนดผู้ปฏิบัติงานให้กับทีมงานการจัดส่งนี้ได้ในแบบฟอร์ม <STRONG>ผู้ปฏิบัติงาน</STRONG> สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการกำหนดผู้ปฏิบัติงานให้กับทีมงานการจัดส่ง โปรดดู <A href="https://technet.microsoft.com/en-us/library/dn776288(v=ax.60)">ผู้ปฏิบัติงานการจัดส่ง (แบบฟอร์ม)</A></P>
