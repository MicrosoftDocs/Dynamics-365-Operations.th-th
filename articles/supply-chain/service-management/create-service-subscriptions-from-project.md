# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="4e8c9-103">การสร้างการสั่งซื้อโดยบอกรับเป็นสมาชิกการบริการจากโครงการ</span><span class="sxs-lookup"><span data-stu-id="4e8c9-103">Create service subscriptions from a project</span></span>    

[!include [banner](../includes/banner.md)]


<span data-ttu-id="4e8c9-104">คุณสามารถสร้างการบอกรับเป็นสมาชิกการบริการจากโครงการได้ใน **การจัดการและการบัญชีโครงการ**</span><span class="sxs-lookup"><span data-stu-id="4e8c9-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="4e8c9-105">คลิก **การจัดการโครงการและการบัญชี** \> **ทั่วไป** \> **โครงการ** \> **โครงการทั้งหมด**</span><span class="sxs-lookup"><span data-stu-id="4e8c9-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="4e8c9-106">เปิดโครงการที่คุณต้องการบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="4e8c9-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="4e8c9-107">ในฟอร์ม **โครงการ** บน **บานหน้าต่างการดำเนินการ** บนแท็บ **จัดการ** คลิก **บริการ** \> **การบอกรับเป็นสมาชิกการบริการ**</span><span class="sxs-lookup"><span data-stu-id="4e8c9-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

3.  <span data-ttu-id="4e8c9-108">บนฟอร์ม **การบอกรับเป็นสมาชิก** ในฟิลด์ **การบอกรับเป็นสมาชิกการบริการ** ป้อนหมายเลขรหัสเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="4e8c9-108">On the **Subscription** form, in the **Service subscription** field, enter a unique identification number.</span></span> <span data-ttu-id="4e8c9-109">หมายเลขโครงการจะถูกป้อนลงในการบอกรับเป็นสมาชิกสำหรับโครงการที่เลือกโดยอัตโนมัติ</span><span class="sxs-lookup"><span data-stu-id="4e8c9-109">The project number is entered automatically into the subscription for the selected project.</span></span>

4.  <span data-ttu-id="4e8c9-110">ป้อนข้อมูลที่เหลือลงในแบบฟอร์ม</span><span class="sxs-lookup"><span data-stu-id="4e8c9-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="4e8c9-111">ตอนนี้ คุณสามารถสร้างธุรกรรมการบอกรับเป็นสมาชิกสำหรับการสั่งซื้อโดยบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="4e8c9-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="4e8c9-112">ถ้าคุณตั้งค่าลำดับหมายเลขสำหรับกลุ่มการบอกรับเป็นสมาชิกไว้ รหัสการบอกรับเป็นสมาชิกจะได้รับการสร้างขึ้นโดยอัตโนมัติเมื่อคุณระบุกลุ่มการบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="4e8c9-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
