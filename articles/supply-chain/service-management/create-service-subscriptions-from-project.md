# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="f9840-103">การสร้างการสั่งซื้อโดยบอกรับเป็นสมาชิกการบริการจากโครงการ</span><span class="sxs-lookup"><span data-stu-id="f9840-103">Create service subscriptions from a project</span></span>    

<span data-ttu-id="f9840-104">[!รวม [ป้ายประกาศ](../includes/banner.md)</span><span class="sxs-lookup"><span data-stu-id="f9840-104">[!include [banner](../includes/banner.md)</span></span>


<span data-ttu-id="f9840-105">คุณสามารถสร้างการบอกรับเป็นสมาชิกการบริการจากโครงการได้ใน **การจัดการและการบัญชีโครงการ**</span><span class="sxs-lookup"><span data-stu-id="f9840-105">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="f9840-106">คลิก **การจัดการโครงการและการบัญชี** \> **ทั่วไป** \> **โครงการ** \> **โครงการทั้งหมด**</span><span class="sxs-lookup"><span data-stu-id="f9840-106">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="f9840-107">เปิดโครงการที่คุณต้องการบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="f9840-107">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="f9840-108">ในฟอร์ม **โครงการ** บน **บานหน้าต่างการดำเนินการ** บนแท็บ **จัดการ** คลิก **บริการ** \> **การบอกรับเป็นสมาชิกการบริการ**</span><span class="sxs-lookup"><span data-stu-id="f9840-108">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

3.  <span data-ttu-id="f9840-109">บนฟอร์ม **การบอกรับเป็นสมาชิก** ในฟิลด์ **การบอกรับเป็นสมาชิกการบริการ** ป้อนหมายเลขรหัสเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="f9840-109">On the **Subscription** form, in the **Service subscription** field, enter a unique identification number.</span></span> <span data-ttu-id="f9840-110">หมายเลขโครงการจะถูกป้อนลงในการบอกรับเป็นสมาชิกสำหรับโครงการที่เลือกโดยอัตโนมัติ</span><span class="sxs-lookup"><span data-stu-id="f9840-110">The project number is entered automatically into the subscription for the selected project.</span></span>

4.  <span data-ttu-id="f9840-111">ป้อนข้อมูลที่เหลือลงในแบบฟอร์ม</span><span class="sxs-lookup"><span data-stu-id="f9840-111">Enter the remaining information in the form.</span></span>

<span data-ttu-id="f9840-112">ตอนนี้ คุณสามารถสร้างธุรกรรมการบอกรับเป็นสมาชิกสำหรับการสั่งซื้อโดยบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="f9840-112">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="f9840-113">ถ้าคุณตั้งค่าลำดับหมายเลขสำหรับกลุ่มการบอกรับเป็นสมาชิกไว้ รหัสการบอกรับเป็นสมาชิกจะได้รับการสร้างขึ้นโดยอัตโนมัติเมื่อคุณระบุกลุ่มการบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="f9840-113">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
