# <a name="create-service-object-relations"></a>การสร้างความสัมพันธ์ของวัตถุที่ให้บริการ 

[!include [banner](../includes/banner.md)]


บทความนี้อธิบายวิธีการสร้างความสัมพันธ์ของวัตถุที่ให้บริการสำหรับข้อตกลงการให้บริการและใบสั่งบริการ เมื่อคุณสร้างตัวให้บริการความสัมพันธ์ของวัตถุ คุณเชื่อมโยงวัตถุที่ให้บริการกับข้อตกลงการให้บริการหรือใบสั่งบริการ  เมื่อบริการคำขอของลูกค้าสำหรับสินค้าที่มีวัตถุที่ให้บริการ คุณสามารถเลือกวัตถุที่ให้บริการจากรายการของความสัมพันธ์ของวัตถุที่ให้บริการ

## <a name="create-a-service-object-relation-for-a-service-agreement"></a>การสร้างความสัมพันธ์ของบริการออบเจ็กต์สำหรับข้อตกลงการให้บริการ

ใช้ขั้นตอนต่อไปนี้เพื่อสร้างความสัมพันธ์ของบริการออบเจ็กต์สำหรับข้อตกลงการให้บริการ

1.  คลิก **การจัดการบริการ** \> **ข้อตกลงการให้บริการ** \> **ข้อตกลงการให้บริการ**

2.  ในรายการ **ข้อตกลงการให้บริการ** ให้เลือกข้อตกลงการให้บริการที่มีอยู่ หรือคลิก **สร้าง** เพื่อสร้างข้อตกลงการให้บริการใหม่

3.  ในฟอร์ม **ข้อตกลงการให้บริการ** บน **บานหน้าต่างการดำเนินการ** บนแท็บ **ข้อตกลงการให้บริการ** ในกลุ่ม **ความสัมพันธ์** คลิก **วัตถุที่ให้บริการ**

4.  ในแบบฟอร์ม **วัตถุที่ให้บริการ** คลิก **สร้าง** แล้วเลือกวัตถุที่ให้บริการสำหรับข้อตกลงการให้บริการนี้

5.  เมื่อต้องการกำหนดสูตรการผลิตเท็มเพลต (BOM) ให้กับข้อตกลงการให้บริการ คลิก **ฟังก์ชัน** แล้วเลือก **แนบ BOM เท็มเพลต** ในแบบฟอร์ม **เลือก BOM เท็มเพลต** ในฟิลด์ **BOM เท็มเพลต** เลือกเท็มเพลต 

## <a name="create-a-service-object-relation-for-a-service-order"></a>การสร้างความสัมพันธ์ของบริการออบเจ็กต์สำหรับใบสั่งบริการ

ใช้ขั้นตอนต่อไปนี้เพื่อสร้างความสัมพันธ์ของบริการออบเจ็กต์สำหรับใบสั่งบริการ

1.  คลิก **การจัดการบริการ** \> **ใบสั่งบริการ** \> **ใบสั่งบริการ**

2.  ในรายการ **ใบสั่งบริการ** เลือกใบสั่งบริการที่มีอยู่แล้ว หรือสร้างใบสั่งบริการใหม่

3.  ในฟอร์ม **ใบสั่งบริการ** บน **บานหน้าต่างการดำเนินการ** บนแท็บ **ใบสั่งบริการ** ในกลุ่ม **ความสัมพันธ์** คลิก **วัตถุที่ให้บริการ**

4.  ในแบบฟอร์ม **วัตถุที่ให้บริการ** คลิก **สร้าง** แล้วเลือกวัตถุที่ให้บริการสำหรับใบสั่งบริการนี้

5.  เมื่อต้องการกำหนด BOM เท็มเพลตให้กับข้อตกลงการให้บริการ คลิก **ฟังก์ชัน** แล้วเลือก **แนบ BOM เท็มเพลต** ในแบบฟอร์ม **เลือก BOM เท็มเพลต** ในฟิลด์ **BOM เท็มเพลต** เลือกเท็มเพลต
