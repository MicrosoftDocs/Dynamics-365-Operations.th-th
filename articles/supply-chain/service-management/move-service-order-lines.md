# <a name="move-service-order-lines"></a><span data-ttu-id="38543-103">ย้ายรายการใบสั่งบริการ</span><span class="sxs-lookup"><span data-stu-id="38543-103">Move service order lines</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="38543-104">คลิก **การจัดการบริการ** \> **ทั่วไป** \> **ใบสั่งบริการ** \> **ใบสั่งบริการ**</span><span class="sxs-lookup"><span data-stu-id="38543-104">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="38543-105">คลิกสองครั้งที่ใบสั่งบริการที่มีรายการที่คุณต้องการย้ายไปยังใบสั่งบริการอื่น</span><span class="sxs-lookup"><span data-stu-id="38543-105">Double-click the service order that contains the lines that you want to move to another service order.</span></span>

3.  <span data-ttu-id="38543-106">ในฟอร์ม **ใบสั่งบริการ** บนบานหน้าต่างการดำเนินการ คลิกปุ่ม **ย้ายรายการ**</span><span class="sxs-lookup"><span data-stu-id="38543-106">In the **Service orders** form, on the Action Pane, click the **Move lines** button.</span></span>

4.  <span data-ttu-id="38543-107">ในฟอร์ม **ย้ายรายการใบสั่งบริการ** ในฟิลด์ **ใบสั่งบริการ** เลือกใบสั่งบริการที่คุณต้องการย้ายบรรทัดไป</span><span class="sxs-lookup"><span data-stu-id="38543-107">In the **Move service order lines** form, in the **Service order** field, select the service order that you want to move the lines to.</span></span>

5.  <span data-ttu-id="38543-108">เลือกกล่องกาเครื่องหมาย **ลบใบสั่งบริการที่ว่าง** ถ้าคุณต้องการลบใบสั่งบริการที่คุณย้ายรายการใบสั่งบริการมา</span><span class="sxs-lookup"><span data-stu-id="38543-108">Select the **Delete empty service order** check box to delete the service order that you moved the service order lines from.</span></span>

6.  <span data-ttu-id="38543-109">คลิก **ตกลง** </span><span class="sxs-lookup"><span data-stu-id="38543-109">Click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="38543-110">คุณสามารถย้ายรายการใบสั่งบริการได้เฉพาะระหว่างใบสั่งบริการที่แนบกับข้อตกลงการให้บริการเดียวกันเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="38543-110">You can move service order lines only between service orders that are attached to the same service agreement.</span></span></P>
