## <a name="set-up-return-reason-code-groups"></a><span data-ttu-id="c39d8-104">การตั้งค่ากลุ่มรหัสเหตุผลการส่งคืน</span><span class="sxs-lookup"><span data-stu-id="c39d8-104">Set up return reason code groups</span></span> 

1.  <span data-ttu-id="c39d8-105">คลิก **การขายและการตลาด** \> **ตั้งค่า** \> **ใบสั่งขาย** \> **การส่งคืนสินค้า** \> **กลุ่มรหัสเหตุผลการส่งคืน**</span><span class="sxs-lookup"><span data-stu-id="c39d8-105">Click **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Return reason code groups**.</span></span>

2.  <span data-ttu-id="c39d8-106">กด CTRL+N เพื่อสร้างรายการสำหรับกลุ่มรหัสเหตุผลการส่งคืนใหม่</span><span class="sxs-lookup"><span data-stu-id="c39d8-106">Press CTRL+N to create a line for a new return reason code group.</span></span>

3.  <span data-ttu-id="c39d8-107">ในฟิลด์ **กลุ่มรหัสเหตุผลการส่งคืน** ให้พิมพ์ตัวระบุสำหรับกลุ่มนี้</span><span class="sxs-lookup"><span data-stu-id="c39d8-107">In the **Return reason code group** field, type an identifier for this group.</span></span>

4.  <span data-ttu-id="c39d8-108">ในฟิลด์ **คำอธิบาย** ให้พิมพ์วลีย่อเพื่ออธิบายวิธีการใช้กลุ่มนี้</span><span class="sxs-lookup"><span data-stu-id="c39d8-108">In the **Description** field, type a brief phrase to explain how this group will be used.</span></span>

## <a name="set-up-return-reason-codes"></a><span data-ttu-id="c39d8-109">การตั้งค่ารหัสเหตุผลการส่งคืน</span><span class="sxs-lookup"><span data-stu-id="c39d8-109">Set up return reason codes</span></span> 

<span data-ttu-id="c39d8-110">กระบวนงานนี้ช่วยคุณตั้งค่ารหัสเหตุผลการส่งคืนที่คุณสามารถใช้เพื่อบ่งชี้ว่าเหตุใดจึงมีการส่งคืนผลิตภัณฑ์ โดยลูกค้า</span><span class="sxs-lookup"><span data-stu-id="c39d8-110">This procedure helps you set up return reason codes that you can use to indicate why a product was returned by the customer.</span></span>

1.  <span data-ttu-id="c39d8-111">คลิก **การขายและการตลาด** \> **ตั้งค่า** \> **ใบสั่งขาย** \> **การส่งคืนสินค้า** \> **รหัสเหตุผลการส่งคืน**</span><span class="sxs-lookup"><span data-stu-id="c39d8-111">Click **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Return reason codes**.</span></span>

2.  <span data-ttu-id="c39d8-112">กด CTRL + N เพื่อสร้างบรรทัดรหัสเหตุผลการส่งคืนใหม่</span><span class="sxs-lookup"><span data-stu-id="c39d8-112">Press CTRL + N to create a new return reason code line.</span></span>

3.  <span data-ttu-id="c39d8-113">ในฟิลด์ **รหัสเหตุผลการส่งคืน** ให้ป้อนข้อความเพื่อระบุรหัสนี้</span><span class="sxs-lookup"><span data-stu-id="c39d8-113">In the **Return reason code** field, enter text to identify this code.</span></span>

4.  <span data-ttu-id="c39d8-114">ในฟิลด์ **คำอธิบาย** ให้ป้อนข้อความเพื่ออธิบายรหัสเหตุผลการส่งคืน</span><span class="sxs-lookup"><span data-stu-id="c39d8-114">In the **Description** field, enter text to describe the return reason code.</span></span>

5.  <span data-ttu-id="c39d8-115">ใช้รายการแบบหล่นลง **กลุ่มรหัสเหตุผลการส่งคืน** เพื่อเพิ่มรหัสนี้ในรายการกลุ่มรหัสเหตุผลการส่งคืน</span><span class="sxs-lookup"><span data-stu-id="c39d8-115">Use the **Return reason code group** drop-down list to add the code to a return reason code group.</span></span>
