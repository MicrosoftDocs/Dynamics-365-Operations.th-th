## <a name="set-up-return-reason-codes"></a><span data-ttu-id="17af4-109">การตั้งค่ารหัสเหตุผลการส่งคืน</span><span class="sxs-lookup"><span data-stu-id="17af4-109">Set up return reason codes</span></span> 

<span data-ttu-id="17af4-110">กระบวนงานนี้ช่วยคุณตั้งค่ารหัสเหตุผลการส่งคืนที่คุณสามารถใช้เพื่อบ่งชี้ว่าเหตุใดจึงมีการส่งคืนผลิตภัณฑ์ โดยลูกค้า</span><span class="sxs-lookup"><span data-stu-id="17af4-110">This procedure helps you set up return reason codes that you can use to indicate why a product was returned by the customer.</span></span>

1.  <span data-ttu-id="17af4-111">ไปที่ **การขายและการตลาด** \> **ตั้งค่า** \> **ใบสั่งขาย** \> **การส่งคืนสินค้า** \> **รหัสเหตุผลการส่งคืน**</span><span class="sxs-lookup"><span data-stu-id="17af4-111">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Return reason codes**.</span></span>

2.  <span data-ttu-id="17af4-112">กด CTRL + N เพื่อสร้างบรรทัดรหัสเหตุผลการส่งคืนใหม่</span><span class="sxs-lookup"><span data-stu-id="17af4-112">Press CTRL + N to create a new return reason code line.</span></span>

3.  <span data-ttu-id="17af4-113">ในฟิลด์ **รหัสเหตุผลการส่งคืน** ให้ป้อนข้อความเพื่อระบุรหัสนี้</span><span class="sxs-lookup"><span data-stu-id="17af4-113">In the **Return reason code** field, enter text to identify this code.</span></span>

4.  <span data-ttu-id="17af4-114">ในฟิลด์ **คำอธิบาย** ให้ป้อนข้อความเพื่ออธิบายรหัสเหตุผลการส่งคืน</span><span class="sxs-lookup"><span data-stu-id="17af4-114">In the **Description** field, enter text to describe the return reason code.</span></span>

5.  <span data-ttu-id="17af4-115">ใช้รายการแบบหล่นลง **กลุ่มรหัสเหตุผลการส่งคืน** เพื่อเพิ่มรหัสนี้ในรายการกลุ่มรหัสเหตุผลการส่งคืน</span><span class="sxs-lookup"><span data-stu-id="17af4-115">Use the **Return reason code group** drop-down list to add the code to a return reason code group.</span></span>
