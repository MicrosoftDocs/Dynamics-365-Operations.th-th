# <a name="set-up-return-reason-codes"></a>ตั้งค่ารหัสเหตุผลการส่งคืน   

[!include [banner](../includes/banner.md)]


## <a name="set-up-return-reason-code-groups"></a>การตั้งค่ากลุ่มรหัสเหตุผลการส่งคืน 

1.  ไปที่ **การขายและการตลาด** \> **ตั้งค่า** \> **ใบสั่งขาย** \> **การส่งคืนสินค้า** \> **กลุ่มรหัสเหตุผลการส่งคืน**

2.  เลือก **สร้าง** เพื่อสร้างรายการสำหรับกลุ่มรหัสเหตุผลการส่งคืนใหม่

3.  ในฟิลด์ **กลุ่มรหัสเหตุผลการส่งคืน** ให้พิมพ์ตัวระบุสำหรับกลุ่มนี้

4.  ในฟิลด์ **คำอธิบาย** ให้พิมพ์วลีย่อเพื่ออธิบายวิธีการใช้กลุ่มนี้

## <a name="set-up-return-reason-codes"></a>การตั้งค่ารหัสเหตุผลการส่งคืน 

กระบวนงานนี้ช่วยคุณตั้งค่ารหัสเหตุผลการส่งคืนที่คุณสามารถใช้เพื่อบ่งชี้ว่าเหตุใดจึงมีการส่งคืนผลิตภัณฑ์ โดยลูกค้า

1.  ไปที่ **การขายและการตลาด** \> **ตั้งค่า** \> **ใบสั่งขาย** \> **การส่งคืนสินค้า** \> **รหัสเหตุผลการส่งคืน**

2.  กด CTRL + N เพื่อสร้างบรรทัดรหัสเหตุผลการส่งคืนใหม่

3.  ในฟิลด์ **รหัสเหตุผลการส่งคืน** ให้ป้อนข้อความเพื่อระบุรหัสนี้

4.  ในฟิลด์ **คำอธิบาย** ให้ป้อนข้อความเพื่ออธิบายรหัสเหตุผลการส่งคืน

5.  ใช้รายการแบบหล่นลง **กลุ่มรหัสเหตุผลการส่งคืน** เพื่อเพิ่มรหัสนี้ในรายการกลุ่มรหัสเหตุผลการส่งคืน
