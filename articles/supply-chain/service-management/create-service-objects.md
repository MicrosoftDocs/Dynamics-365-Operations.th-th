# <a name="create-service-objects"></a><span data-ttu-id="03e1d-103">การสร้างวัตถุที่ให้บริการ</span><span class="sxs-lookup"><span data-stu-id="03e1d-103">Create service objects</span></span>  

[!include[banner](../includes/banner.md)]
   
<span data-ttu-id="03e1d-104">ออบเจ็กต์บริการคือสินทรัพย์ของลูกค้าและผลิตภัณฑ์ที่คุณสามารถใช้ดำเนินการให้บริการได้ </span><span class="sxs-lookup"><span data-stu-id="03e1d-104">Service objects are a customer’s assets and products for which you can perform a service.</span></span> 

1. <span data-ttu-id="03e1d-105">คลิก **การจัดการบริการ** \> **ตั้งค่า** \> **บริการออบเจ็กต์** \> **บริการออบเจ็กต์**</span><span class="sxs-lookup"><span data-stu-id="03e1d-105">Click **Service management** \> **Setup** \> **Service objects** \> **Service objects**.</span></span>
2. <span data-ttu-id="03e1d-106">บนแถบเครื่องมือ คลิก **สร้าง**</span><span class="sxs-lookup"><span data-stu-id="03e1d-106">On the toolbar, click **New**.</span></span>
3. <span data-ttu-id="03e1d-107">ในฟิลด์ **คำอธิบาย** ให้ป้อนคำอธิบายของออบเจ็กต์</span><span class="sxs-lookup"><span data-stu-id="03e1d-107">In the **Description** field, enter a description of the object.</span></span>
4. <span data-ttu-id="03e1d-108">ในฟิลด์ **กลุ่มออบเจ็กต์ที่ให้บริการ** ให้เลือกกลุ่มที่ออบเจ็กต์เป็นสมาชิกอยู่</span><span class="sxs-lookup"><span data-stu-id="03e1d-108">In the **Service object group** field, select the group that the object belongs to.</span></span> 
5. <span data-ttu-id="03e1d-109">เลือกระบุได้: เลือกสินค้าสำหรับบรรทัดวัตถุที่ให้บริการ หรือเลือกเท็มเพลตสำหรับรายการวัสดุและส่วนประกอบ (BOM)</span><span class="sxs-lookup"><span data-stu-id="03e1d-109">Optional: Select an item for the service object line, or select a template bill of materials (BOM).</span></span>
