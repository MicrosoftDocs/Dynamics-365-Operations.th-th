# <a name="reverse-changes-in-a-dispatch-board"></a>การกลับรายการการเปลี่ยนแปลงในบอร์ดการจัดส่ง 

[!include [banner](../includes/banner.md)]


หัวข้อนี้อธิบายวิธีการกลับรายการที่ยังไม่ได้บันทึกการปรับเปลี่ยนที่คุณทำในบอร์ดการจัดส่ง  ตัวอย่างเช่น คุณกำหนดผู้ปฏิบัติงานให้กับกิจกรรมการบริการ บันทึกข้อมูล และจากนั้นกำหนดผู้ปฏิบัติงานที่แตกต่างกันให้กับกิจกรรมการบริการ  คุณปรับเปลี่ยนผู้ปฏิบัติงานในบอร์ดการจัดส่ง และจากนั้นก่อนที่จะบันทึกการเปลี่ยนแปลง เรียนรู้ว่า ผู้ปฏิบัติงานที่เพิ่งกำหนดให้ไม่พร้อมใช้งาน  คุณสามารถกลับรายการการเปลี่ยนแปลงที่ไม่ได้บันทึกไว้เพื่อให้ผู้ปฏิบัติงานเดียวกันถูกมอบหมายไปยังใบสั่งบริการ

ใช้ขั้นตอนต่อไปนี้เพื่อกลับรายการยังไม่ได้บันทึกการเปลี่ยนแปลงในบอร์ดการจัดส่ง:

1.  คลิก **การจัดการงานบริการ** \> **งานประจำงวด** \> **บอร์ดการจัดส่ง**

2.  ในแบบฟอร์ม **บอร์ดการจัดส่ง** ป้อนข้อมูลที่เกี่ยวข้องในฟิลด์ และจากนั้นคลิก **ตกลง** 

3.  เมื่อต้องการกลับรายการเปลี่ยนแปลงล่าสุดที่ไม่ได้บันทึกไว้ ให้คลิก **เลิกทำ**

4.  เมื่อต้องการกลับรายการชุดของการเปลี่ยนแปลงซึ่งยังไม่ได้บันทึก ให้คลิก **เลิกทำ** ต่อไป จนกว่าการเปลี่ยนแปลงแต่ละรายการที่คุณต้องการยกเลิกจะถูกกลับรายการ
