# <a name="set-up-template-to-represent-objects-at-customer-site"></a><span data-ttu-id="e0e91-103">การตั้งค่าเท็มเพลตเพื่อแสดงถึงออบเจ็กต์ที่ไซต์ลูกค้า</span><span class="sxs-lookup"><span data-stu-id="e0e91-103">Set up template to represent objects at customer site</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="e0e91-104">ใช้กระบวนงานนี้เพื่อตั้งค่าเท็มเพลตที่แสดงถึงออบเจ็กต์ที่ไซต์ลูกค้า</span><span class="sxs-lookup"><span data-stu-id="e0e91-104">Use this procedure to set up a template that represents objects at a customer site.</span></span>

1.  <span data-ttu-id="e0e91-105">คลิก **การจัดการบริการ** \> **ทั่วไป** \> **ข้อตกลงการให้บริการ** \> **ข้อตกลงการให้บริการ**</span><span class="sxs-lookup"><span data-stu-id="e0e91-105">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

2.  <span data-ttu-id="e0e91-106">ดับเบิลคลิกที่ข้อตกลงการให้บริการ หรือคลิก **ข้อตกลงการให้บริการ** เพื่อสร้างข้อตกลงการให้บริการใหม่</span><span class="sxs-lookup"><span data-stu-id="e0e91-106">Double-click a service agreement, or click **Service agreement** to create a new service agreement.</span></span>

3.  <span data-ttu-id="e0e91-107">คลิกแท็บ **การตั้งค่า**</span><span class="sxs-lookup"><span data-stu-id="e0e91-107">Click the **Setup** tab.</span></span>

4.  <span data-ttu-id="e0e91-108">คลิก **วัตถุที่ให้บริการ** เพื่อแนบสูตรการผลิต (BOM) เท็มเพลตกับข้อตกลงการให้บริการ</span><span class="sxs-lookup"><span data-stu-id="e0e91-108">Click **Service objects** to attach a template bill of materials (BOM) to the service agreement.</span></span>

5.  <span data-ttu-id="e0e91-109">ในฟอร์ม **วัตถุที่ให้บริการ** เลือกวัตถุที่ให้บริการ และจากนั้นคลิก **ฟังก์ชัน** \> **แนบ BOM เท็มเพลต**</span><span class="sxs-lookup"><span data-stu-id="e0e91-109">In the **Service objects** form, select a service object, and then click **Functions** \> **Attach template BOM**.</span></span>

6.  <span data-ttu-id="e0e91-110">ในแบบฟอร์ม **เลือก BOM เท็มเพลต** ให้เลือก BOM เท็มเพลตในฟิลด์ **BOM เท็มเพลต** จากนั้นคลิก **ตกลง**</span><span class="sxs-lookup"><span data-stu-id="e0e91-110">In the **Select template BOM** form, select a template BOM in the **Template BOM** field, and then click **OK**.</span></span>

7.  <span data-ttu-id="e0e91-111">ในแบบฟอร์ม **วัตถุที่ให้บริการ** คลิก **ผู้ออกแบบ** เพื่อแก้ไข BOM เท็มเพลต เพื่อให้ BOM แสดงวัตถุต่างๆ ที่คุณต้องการ</span><span class="sxs-lookup"><span data-stu-id="e0e91-111">In the **Service objects** form, click **Designer** to modify the template BOM so that the BOM represents the objects that you want.</span></span>

8.  <span data-ttu-id="e0e91-112">ปิดแบบฟอร์มเพื่อบันทึกการเปลี่ยนแปลงของคุณ</span><span class="sxs-lookup"><span data-stu-id="e0e91-112">Close the forms to save your changes.</span></span>
