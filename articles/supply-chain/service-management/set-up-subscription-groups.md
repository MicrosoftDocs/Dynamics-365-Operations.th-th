# <a name="set-up-subscription-groups"></a>ตั้งค่ากลุ่มการบอกรับเป็นสมาชิก 

[!include [banner](../includes/banner.md)]


1.  ไปที่ **การจัดการงานบริการ** \> **การตั้งค่า** \> **การบอกรับเป็นสมาชิกการบริการ** \> **กลุ่มการบอกรับเป็นสมาชิก**

2.  เลือก **สร้าง** เพื่อสร้างกลุ่มการสมัครใช้งานใหม่

3.  ในฟิลด์ **กลุ่มการบอกรับเป็นสมาชิก** ให้ป้อนชื่อสำหรับกลุ่มการบอกรับเป็นสมาชิกใหม่

4.  ในฟิลด์ **การออกใบแจ้งหนี้** เลือกความถี่ในการออกใบแจ้งหนี้สำหรับการบอกรับเป็นสมาชิก ความถี่ของการออกใบแจ้งหนี้จะระบุตามรหัสรอบระยะเวลา ซึ่งถูกตั้งค่าไว้ในฟอร์ม **ชนิดรอบระยะเวลา**

5.  ในการรับรู้รายได้สำหรับธุรกรรมการบอกรับเป็นสมาชิก เลือกกล่องกาเครื่องหมาย **การตั้งรายได้ค้างรับ** แล้วในฟิลด์ **คงค้าง** เลือกความถี่ ความถี่ของการรับรู้จะขึ้นอยู่กับรหัสรอบระยะเวลา ซึ่งถูกตั้งค่าไว้ในแบบฟอร์ม **รอบระยะเวลา**
