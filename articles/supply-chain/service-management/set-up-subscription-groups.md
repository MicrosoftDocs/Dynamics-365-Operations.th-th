# <a name="set-up-subscription-groups"></a><span data-ttu-id="40334-103">ตั้งค่ากลุ่มการบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="40334-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="40334-104">คลิก **การจัดการงานบริการ** \> **การตั้งค่า** \> **การบอกรับเป็นสมาชิกการบริการ** \> **กลุ่มการบอกรับเป็นสมาชิก**</span><span class="sxs-lookup"><span data-stu-id="40334-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="40334-105">กด Ctrl+N หรือคลิก **สร้าง** เพื่อสร้างกลุ่มการบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="40334-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="40334-106">ในฟิลด์ **กลุ่มการบอกรับเป็นสมาชิก** ให้ป้อนชื่อสำหรับกลุ่มการบอกรับเป็นสมาชิกใหม่</span><span class="sxs-lookup"><span data-stu-id="40334-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="40334-107">ในฟิลด์ **การออกใบแจ้งหนี้** เลือกความถี่ในการออกใบแจ้งหนี้สำหรับการบอกรับเป็นสมาชิก</span><span class="sxs-lookup"><span data-stu-id="40334-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="40334-108">ความถี่ของการออกใบแจ้งหนี้จะระบุตามรหัสรอบระยะเวลา ซึ่งถูกตั้งค่าไว้ในฟอร์ม **ชนิดรอบระยะเวลา**</span><span class="sxs-lookup"><span data-stu-id="40334-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="40334-109">ในการรับรู้รายได้สำหรับธุรกรรมการบอกรับเป็นสมาชิก เลือกกล่องกาเครื่องหมาย **การตั้งรายได้ค้างรับ** แล้วในฟิลด์ **คงค้าง** เลือกความถี่</span><span class="sxs-lookup"><span data-stu-id="40334-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="40334-110">ความถี่ของการรับรู้จะขึ้นอยู่กับรหัสรอบระยะเวลา ซึ่งถูกตั้งค่าไว้ในแบบฟอร์ม **รอบระยะเวลา**</span><span class="sxs-lookup"><span data-stu-id="40334-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
