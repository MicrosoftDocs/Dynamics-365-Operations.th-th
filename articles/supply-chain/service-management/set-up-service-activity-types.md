# <a name="set-up-service-activity-types"></a>การตั้งค่าชนิดของกิจกรรมการบริการ 

[!include [banner](../includes/banner.md)]


1.  คลิก **การขายและการตลาด** \> **ตั้งค่า** \> **กิจกรรม** \> **ชนิดของกิจกรรม**
    
    หรือ
    
    คลิก **การจัดการองค์กร** \> **ตั้งค่า** \> **กิจกรรม** \> **ชนิดของกิจกรรม**

2.  ในแบบฟอร์ม **ชนิดของกิจกรรม** ให้กด CTRL + N เพื่อเพิ่มรายการชนิดของกิจกรรมใหม่

3.  ในฟิลด์ **ชนิด** ให้ป้อนชื่อสำหรับชนิดของกิจกรรม

4.  ในฟิลด์ **คำอธิบาย** ให้ป้อนข้อความเพื่ออธิบายชนิดของกิจกรรมนี้

5.  ปิดแบบฟอร์มเพื่อบันทึกการเปลี่ยนแปลงของคุณ
