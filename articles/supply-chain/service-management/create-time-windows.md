# <a name="create-time-windows"></a><span data-ttu-id="4d7ef-103">การสร้างหน้าต่างเวลา</span><span class="sxs-lookup"><span data-stu-id="4d7ef-103">Create time windows</span></span>   

[!INCLUDE [banner](../includes/banner.md)]

<span data-ttu-id="4d7ef-104">คุณสามารถใช้หน้าต่างเวลาเพื่อเพิ่มประสิทธิภาพการจัดกำหนดการของรายการใบสั่งบริการได้</span><span class="sxs-lookup"><span data-stu-id="4d7ef-104">You can use time windows to optimize the scheduling of service order lines.</span></span>

1. <span data-ttu-id="4d7ef-105">คลิก **การจัดการบริการ** \> **ตั้งค่า** \> **ข้อตกลงการให้บริการ** \> **หน้าต่างเวลา**</span><span class="sxs-lookup"><span data-stu-id="4d7ef-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Time window**.</span></span>
2. <span data-ttu-id="4d7ef-106">กด **Alt+N** เพื่อสร้างหน้าต่างเวลาใหม่</span><span class="sxs-lookup"><span data-stu-id="4d7ef-106">Press **Alt+N** to create a new time window.</span></span>
3. <span data-ttu-id="4d7ef-107">ในฟิลด์ **หน้าต่างเวลา** ให้ป้อนชื่อสำหรับหน้าต่างเวลา</span><span class="sxs-lookup"><span data-stu-id="4d7ef-107">In the **Time window** field, enter a name for the time window.</span></span>
4. <span data-ttu-id="4d7ef-108">ในฟิลด์ **คำอธิบาย** ป้อนคำอธิบายสำหรับหน้าต่างเวลา</span><span class="sxs-lookup"><span data-stu-id="4d7ef-108">In the **Description** field, enter a description for the time window.</span></span>
5. <span data-ttu-id="4d7ef-109">ในฟิลด์ **วิธี** เลือกวิธีที่จะถูกสร้างโดยอัตโนมัติ</span><span class="sxs-lookup"><span data-stu-id="4d7ef-109">In the **Method** field, select the method to be autocreated.</span></span> <span data-ttu-id="4d7ef-110">หากวิธีเป็นของประเภท **กำหนดเอง** ให้ระบุจำนวนวันก่อนหรือหลังวันที่ถูกคำนวณ ซึ่งสามารถย้ายรายการใบสั่งการบริการได้</span><span class="sxs-lookup"><span data-stu-id="4d7ef-110">If the method is of the **Manual** type, specify the number of days before or after the calculated date that the service order line can be moved.</span></span>
