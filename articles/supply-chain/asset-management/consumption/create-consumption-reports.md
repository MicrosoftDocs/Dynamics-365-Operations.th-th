# <a name="create-consumption-reports"></a>สร้างรายงานปริมาณการใช้

[!include [banner](../../includes/banner.md)]

 

เมื่อคุณสร้าง และการลงทะเบียนลงรายการบัญชีปริมาณการใช้บนใบสั่งงานในการจัดการสินทรัพย์ รายงานทั้งสองพร้อมใช้งานเพื่อการเเสดงรายละเอียดปริมาณ


## <a name="asset-consumption-report"></a>รายงานปริมาณการใช้สินทรัพย์

เมื่อคุณลงรายการบัญชีปริมาณการใช้บนใบสั่งงาน คุณสามารถพิมพ์รายงานปริมาณการใช้สินทรัพย์ รายงานจะแสดงชั่วโมง ต้นทุนชั่วโมง ต้นทุนสินค้า และค่าใช้จ่ายที่ลงรายการบัญชีไว้ในสินทรัพย์

1. คลิก **การจัดการสินทรัพย์** > **รายงาน** > **สินทรัพย์** > **ปริมาณการใช้สินทรัพย์**

2. ในกล่องโต้ตอบ **ปริมาณการใช้สินทรัพย์** เลือกพารามิเตอร์ และระดับรายละเอียดที่คุณต้องการดูโดยการเลือก **ใช่** บนปุ่มสลับที่เกี่ยวข้อง และแทรกระดับตำเเหน่งที่ทำงานในส่วน **เเสดง**
    - คุณสามารถใช้ฟิลด์ **ระดับ** เพื่อระบุรายละเอียดที่คุณต้องการให้รายการสินทรัพย์เกี่ยวกับตำเเหน่งที่ทำงานมากเพียงใด 
    
        ตัวอย่างเช่น หากคุณป้อนหมายเลข "1" ในฟิลด์ และคุณมีโครงสร้างตำเเหน่งที่ทำงานหลายระดับ สินทรัพย์ทั้งหมดสำหรับตำเเหน่งที่ทำงานจะถูกเเสดงบนระดับบนสุดดังนั้นชั่วโมงในรายการอาจจะมีการเพิ่มจากตำเเหน่งที่ทำงานที่อยู่ในระดับที่ต่ำกว่า 
        
        หากคุณป้อนหมายเลข "0" ในฟิลด์ **ระดับ** คุณจะเห็นผลลัพธ์โดยละเอียดแสดงสินทรัพย์ทั้งหมดบนระดับตำเเหน่งที่ทำงานทั้งหมดที่เกี่ยวข้อง 
        
    - เลือก **ใช่** ในปุ่มสลับ **ผลรวมบนสินทรัพย์ย่อยทั้งหมด** เพื่อดูผลรวมของสินทรัพย์ย่อยแต่ละรายการในรายงาน

3. เลือกช่วงวันที่ในส่วน **วันที่**

4. บนแท็บด่วน **ปลายทาง** เลือกถ้าคุณต้องการแสดงรายงานบนหน้าจอ ให้พิมพ์ หรือบันทึกเป็นไฟล์ หรืออีเมล

5. ถ้าจำเป็น คุณสามารถเลือกสินทรัพย์เฉพาะที่จะแสดงในรายงานได้ บนเเท็บด่วน **เรกคอร์ดที่จะรวม** คลิก **ตัวกรอง** เเละเพิ่มสินทรัพย์ที่คุณต้องการรวมไว้ในรายงาน

6. คลิก **ตกลง** เพื่อสร้างรายงาน


## <a name="work-order-consumption-report"></a>รายงานปริมาณการใช้ใบสั่งงาน

เมื่อคุณลงรายการบัญชีปริมาณการใช้บนใบสั่งงาน คุณสามารถพิมพ์รายงานปริมาณการใช้ใบสั่งงาน รายงานจะแสดงชั่วโมง ต้นทุนชั่วโมง ต้นทุนสินค้า และค่าใช้จ่ายที่ลงรายการบัญชีไว้ในใบสั่งงาน

1. คลิก **การจัดการสินทรัพย์** > **รายงาน** > **ใบสั่งงาน** > **ปริมาณการใช้ใบสั่งงาน**

2. ในกล่องโต้ตอบ **ปริมาณการใช้ใบสั่งงาน** เลือกพารามิเตอร์ที่คุณต้องการรวมไว้ในรายงานโดยเลือก **ใช่** บนปุ่มสลับที่เกี่ยวข้องในส่วน **เเสดง**

3. เลือกช่วงวันที่ในส่วน **วันที่**

4. บนแท็บด่วน **ปลายทาง** เลือกถ้าคุณต้องการแสดงรายงานบนหน้าจอ ให้พิมพ์ หรือบันทึกเป็นไฟล์ หรืออีเมล

5. ถ้าจำเป็น คุณสามารถเลือกใบสั่งงานเฉพาะที่จะแสดงในรายงานได้ บนเเท็บด่วน **เรกคอร์ดที่จะรวม** คลิก **ตัวกรอง** เเละเพิ่มใบสั่งงานที่คุณต้องการรวมไว้ในรายงาน

6. คลิก **ตกลง** เพื่อสร้างรายงาน


>[!NOTE]
>นอกจากนี้คุณยังสามารถสร้าง [รายงานใบสั่งงาน](../work-orders/work-order-report.md) ซึ่งประกอบด้วยรายละเอียดของใบสั่งงานเพิ่มเติม
