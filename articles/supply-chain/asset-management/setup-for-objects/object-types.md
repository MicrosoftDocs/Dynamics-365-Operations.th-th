<span data-ttu-id="c7e1a-114">ขั้นแรก ให้คุณตั้งค่าชนิดสินทรัพย์ที่ต้องการ</span><span class="sxs-lookup"><span data-stu-id="c7e1a-114">First, you set up the required asset types.</span></span> <span data-ttu-id="c7e1a-115">ถัดไป คุณสร้างแบบจำลองสินทรัพย์ที่ควรเกี่ยวข้องกับชนิดของสินทรัพย์</span><span class="sxs-lookup"><span data-stu-id="c7e1a-115">Next, you create the asset models that should be related to the asset types.</span></span> <span data-ttu-id="c7e1a-116">ในตอนท้าย บนหน้า **ค่าเริ่มต้นของชนิดสินทรัพย์** คุณสามารถสร้างความแตกต่างของชนิดสินทรัพย์ทั้งหมดที่จำเป็นสำหรับอุปกรณ์ของคุณ</span><span class="sxs-lookup"><span data-stu-id="c7e1a-116">Finally, on the **Asset type defaults** page, you create all the variations of asset types that are required for your equipment.</span></span>

3. <span data-ttu-id="c7e1a-120">ในฟิลด์ **ชนิดสินทรัพย์** ให้ป้อนรหัสชนิดสินทรัพย์</span><span class="sxs-lookup"><span data-stu-id="c7e1a-120">In the **Asset type** field, enter an asset type ID.</span></span>
