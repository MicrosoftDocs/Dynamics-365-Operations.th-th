# <a name="work-order-report"></a>รายงานใบสั่งงาน

[!include [banner](../../includes/banner.md)]


คุณสามารถสร้างรายงานใบสั่งงานที่แสดงข้อมูลโดยละเอียดเกี่ยวกับใบสั่งงานที่คุณเลือกหนึ่งรายการขึ้นไป

1. เลือก **การจัดการสินทรัพย์** > **รายงาน** > **ใบสั่งงาน** > **รายงานใบสั่งงาน**

2. เลือก "ใช่" /"ไม่ใช่" บนปุ่มสลับตามที่คุณต้องการ เพื่อกำหนดรายละเอียดที่จะรวมไว้ในรายงาน  
    - ในส่วน **การตั้งค่าการพิมพ์** คุณสามารถเลือกได้ว่าคุณต้องการรวมเอกสารแนบจากการตั้งค่าของชนิดงานบำรุงรักษาที่เกี่ยวข้องหรือไม่

3. บน FastTab **เรกคอร์ดที่จะรวม** คุณสามารถกรองเนื้อหาของรายงานตาม **ใบสั่งงาน** ได้

4. บน FastTab **รันในเบื้องหลัง** คุณสามารถตั้งค่าการสร้างของรายงานใบสั่งงานเป็นชุดงานตามที่คุณต้องการ

5. เลือก **ตกลง** เพื่อสร้างรายงาน

ภาพประกอบต่อไปนี้แสดงตัวอย่างของวิธีการตั้งค่าพารามิเตอร์รายงาน
