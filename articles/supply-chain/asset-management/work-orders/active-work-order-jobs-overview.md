<span data-ttu-id="27210-104">บนหน้ารายการ **งานบำรุงรักษาของใบสั่งงานที่ใช้งานอยู่** คุณสามารถเรียกดูภาพรวมของใบสั่งงาน ในด้านจำนวนใบสั่งงานที่ได้ถูกสร้างขึ้นในสินทรัพย์เฉพาะ ชนิดสินทรัพย์ ผู้ผลิต แบบจำลอง ชนิดงานการบำรุงรักษา และอื่นๆ</span><span class="sxs-lookup"><span data-stu-id="27210-104">On the **Active work order maintenance jobs** list page, you can get an overview of work orders, in terms of the number of work orders that have been created on specific assets, asset types, manufacturers, models, maintenance job types, and so on.</span></span> <span data-ttu-id="27210-105">โดยการเลือกงานใบสั่งงาน และเลือก **แก้ไข** คุณสามารถเปิดใบสั่งงานที่เกี่ยวข้อง</span><span class="sxs-lookup"><span data-stu-id="27210-105">By selecting a work order job and then selecting **Edit**, you can open the related work order.</span></span> <span data-ttu-id="27210-106">โดยการเลือกงานใบสั่งงานและเลือกปุ่มบนแท็บ **งานการบำรุงรักษาใบสั่งงาน** ของบานหน้าต่างการดำเนินการ คุณสามารถดูข้อมูลสำหรับใบสั่งงานที่เกี่ยวของกับงานใบสั่งงาน</span><span class="sxs-lookup"><span data-stu-id="27210-106">By selecting a work order job and then selecting one of the buttons on the **Work order maintenance jobs** tab of the Action Pane, you can view data for the work order that the work order job is related to.</span></span>

<span data-ttu-id="27210-107">หากต้องการเปิดหน้ารายการ ให้เลือก **การจัดการสินทรัพย์** > **ทั่วไป** > **ใบสั่งงาน** > **งานบำรุงรักษาของใบสั่งงานที่ใช้งานอยู่**</span><span class="sxs-lookup"><span data-stu-id="27210-107">To open the list page, select **Asset management** > **Common** > **Work orders** > **Active work order maintenance jobs**.</span></span> <span data-ttu-id="27210-108">หน้ารายการประกอบด้วยงานของใบสั่งงานที่ใช้งานอยู่ทั้งหมด และยังแสดงข้อมูลบางอย่างที่เกี่ยวข้องกับใบสั่งงานหรืองานของใบสั่งงาน</span><span class="sxs-lookup"><span data-stu-id="27210-108">The page lists all active work order jobs, and it also shows some of the information that is related to the work order or work order job.</span></span>

<span data-ttu-id="27210-109">ในคอลัมน์ **%** หมายเลขจะแสดงถึงความสมบูรณ์ของใบสั่งงานเป็นเปอร์เซ็นต์</span><span class="sxs-lookup"><span data-stu-id="27210-109">In the **%** column, a number shows completion of the work order as a percentage.</span></span> <span data-ttu-id="27210-110">ความเสร็จสมบูรณ์จะขึ้นอยู่กับการคำนวณสองรายการ: ชั่วโมงที่ลงรายการบัญชีเมื่อเทียบกับชั่วโมงที่คาดการณ์ และจำนวนของรายการตรวจสอบการบำรุงรักษาที่เสร็จสมบูรณ์</span><span class="sxs-lookup"><span data-stu-id="27210-110">Completion is based on two calculations: posted hours compared to forecasted hours, and the number of maintenance checklists that have been completed.</span></span>
