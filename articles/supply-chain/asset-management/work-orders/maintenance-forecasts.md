# <a name="maintenance-forecasts"></a>การคาดการณ์ในการบำรุงรักษา

[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]


เมื่อคุณสร้างใบสั่งงาน คุณสร้างงานใบสั่งงานกับสินทรัพย์และชนิดงานบำรุงรักษาที่เกี่ยวข้องได้ เมื่อคุณเลือกชนิดงานบำรุงรักษาที่มีการคาดการณ์ในการบำรุงรักษา การคาดการณ์จะถูกคัดลอกไปยังใบสั่งงานโดยอัตโนมัติ

คุณสามารถเพิ่มหรือลบรายการการคาดการณ์บนใบสั่งงานได้ การตั้งค่าสถานะวงจรชีวิตของใบสั่งงานชนิดโครงการที่เกี่ยวข้องและกฎขั้นที่เกี่ยวข้องกับชนิดโครงการจะกำหนดว่าคุณสามารถเพิ่มหรือแก้ไขรายการการคาดการณ์ได้ 

1. คลิก **การจัดการสินทรัพย์** > **ทั่วไป** > **ใบสั่งงาน** > **ใบสั่งงานทั้งหมด** หรือ **ใบสั่งงานที่ใช้งานอยู่**

2. เลือกใบสั่งงานในรายการ และคลิก **การคาดการณ์** ใน **การคาดการณ์ในการบำรุงรักษาใบสั่งงาน** รายการการคาดการณ์จากชนิดงานการบำรุงรักษาที่เลือกในงานใบสั่งงานจะแสดงขึ้น


## <a name="add-hours-forecast-to-a-work-order"></a>เพิ่มการคาดการณ์ชั่วโมงไปที่ใบสั่งงาน

1. เลือกงานใบสั่งงานที่คุณต้องการเพิ่มการคาดการณ์

2. บนแท็บด่วน **ชั่วโมง** คลิก **เพิ่ม** เพื่อสร้างรายการใหม่

3. เลือกประเภทในฟิลด์ **ประเภท**

4. แทรกจำนวนของชั่วโมงการคาดการณ์ในฟิลด์ **ชั่วโมง**

5. ในฟิลด์ **คุณสมบัติของรายการ** ให้เลือกชนิดค่าธรรมเนียมที่จะใช้ในรายการ


## <a name="add-items-forecast-to-a-work-order"></a>เพิ่มการคาดการณ์สินค้าไปที่ใบสั่งงาน

มีสามวิธีในการเพิ่มสินค้าลงในการคาดการณ์ในการบำรุงรักษาใบสั่งงาน: คุณสามารถสร้างรายการสำหรับสินค้า (อะไหล่สำรอง) ที่ไม่ได้รวมอยู่ในรายการอะไหล่หรือ BOM สินทรัพย์ คุณสามารถเลือกอะไหล่สำรองจากรายการอะไหล่สำรองที่ได้รับอนุมัติ และคุณสามารถเลือกสินค้าจาก BOM สินทรัพย์

1. เลือกงานใบสั่งงานที่คุณต้องการเพิ่มการคาดการณ์

2. เลือกแท็บด่วน **สินค้า**

3. คลิก **เพิ่ม** เพื่อสร้างรายการใหม่สำหรับอะไหล่สำรองที่ไม่ได้อยู่ในรายการอะไหล่สำรองหรือรายการ BOM สินทรัพย์

4. เลือกสินค้าในฟิลด์ **หมายเลขสินค้า**

5. แทรกปริมาณในฟิลด์ **ปริมาณการขาย** และเลือกหน่วยปริมาณในฟิลด์ **หน่วย**

6. แทรกราคาต้นทุนและสกุลเงินในฟิลด์ที่เกี่ยวข้อง และเลือก **คุณสมบัติของรายการ**

7. ถ้าคุณต้องการเปลี่ยนรายการของมิติที่แสดงบนรายการสินค้า ให้คลิก **สินค้าคงคลัง** > **แสดงมิติ** เลือกมิติ และเลือก "ใช่" บนปุ่มสลับ **บันทึกการตั้งค่า**

8. ถ้าคุณต้องการเพิ่มอะไหล่สำรองที่ได้รับการอนุมัติให้กับการคาดการณ์ในการบำรุงรักษา คลิก **เพิ่มอะไหล่สำรอง** เลือกอะไหล่สำรอง แก้ไขข้อมูลที่เกี่ยวข้องถ้าจำเป็น และคลิก **ตกลง**

9. ถ้าคุณต้องการเพิ่มสินค้า BOM สินทรัพย์ลงในการคาดการณ์ ให้คลิก **เพิ่มสินค้า BOM** เลือกสินค้า แก้ไขข้อมูลที่เกี่ยวข้องกับการถ้าจำเป็น และคลิก **ตกลง**

10. คลิก **สินค้าที่ใช้** ถ้าต้องการดูภาพรวมของตำแหน่งที่สินค้าในรายการที่เลือกถูกใช้ในการจัดการสินทรัพย์โดยสัมพันธ์กับสินทรัพย์ ค่าเริ่มต้นของชนิดงานการบำรุงรักษา อะไหล่สำรอง และใบสั่งงาน 



## <a name="add-expense-forecast-to-a-work-order"></a>เพิ่มการคาดการณ์ค่าใช้จ่ายไปที่ใบสั่งงาน

1. หัวข้อนี้อธิบายวิธีการเพิ่มการคาดการณ์ค่าใช้จ่ายไปที่ใบสั่งงาน ในด้านซ้ายมือของฟอร์ม ให้เลือกงานใบสั่งงานที่คุณต้องการเพิ่มการคาดการณ์

2. เลือกแท็บด่วน **ค่าใช้จ่าย**

3. คลิก **เพิ่ม** เพื่อสร้างรายการใหม่

4. เลือกประเภทในฟิลด์ **ประเภท**

5. แทรกปริมาณในฟิลด์ **ปริมาณ**

6. แทรกราคาต้นทุน สกุลเงินการขาย และราคาขายในฟิลด์ที่เกี่ยวข้อง

7. ในฟิลด์ **คุณสมบัติของรายการ** ให้เลือกชนิดค่าธรรมเนียมที่จะใช้ในรายการ

>[!NOTE]
>บนแท็บด่วน **ผลรวมการคาดการณ์ในการบำรุงรักษา** คุณสามารถดูภาพรวมของจำนวนรายการที่สร้างขึ้นบนแต่ละแท็บ สำหรับงานใบสั่งงานที่เลือกและสำหรับใบสั่งงาน นอกจากนี้ คุณยังสามารถดูผลรวมของชั่วโมงการทำงานที่คาดการณ์ไว้สำหรับงานใบสั่งงานและสำหรับใบสั่งงานได้ด้วย

## <a name="automatic-update-of-work-order-forecasts"></a>การอัพเดตอัตโนมัติของการคาดการณ์ใบสั่งงาน

ในการจัดการสินทรัพย์ คุณสามารถอัพเดตการเปลี่ยนแปลงใดๆได้โดยอัตโนมัติในการคาดการณ์ใบสั่งงานที่เกี่ยวข้องกับต้นทุนชั่วโมง ต้นทุนสินค้า และค่าใช้จ่าย ที่มีการอัพเดตในโมดูลอื่นๆ การทำเช่นนี้ทำเพื่อให้แน่ใจว่ามีการใช้ราคาต้นทุนล่าสุดอยู่เสมอในการคาดการณ์ใบสั่งงานของคุณ นอกจากนี้คุณยังสามารถทำการอัพเดตที่คล้ายกันสำหรับ [การคาดการณ์ชนิดงานบำรุงรักษา](../setup-for-work-orders/job-groups-and-job-types-variants-trades-and-checklists.md)

1. คลิก **การจัดการสินทรัพย์** > **ประจำงวด** > **การคาดการณ์** > **อัพเดตการคาดการณ์ใบสั่งงาน**

2. ในกล่องโต้ตอบรายการแบบหล่นลง **การอัพเดตการคาดการณ์ใบสั่งงาน** คุณสามารถเพิ่มการเลือกที่เกี่ยวข้องกับใบสั่งงานหรืองานใบสั่งงานเฉพาะได้ ถ้าจำเป็น คลิก **ตัวกรอง** เพื่อทำการเลือกดังกล่าว

3. ถ้าจำเป็น คุณสามารถตั้งค่าการอัพเดตโดยอัตโนมัติเป็นชุดงานได้บนแท็บด่วน **รันในแบบเบื้องหลัง**

4. คลิก **ตกลง** เพื่อเริ่มต้นการอัพเดตการคาดการณ์
