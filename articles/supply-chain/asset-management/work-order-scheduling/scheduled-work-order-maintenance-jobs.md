<span data-ttu-id="1ecfa-104">หน้า **งานบำรุงรักษาของใบสั่งงานที่จัดกำหนดการ** ใช้เพื่อเรียกดูภาพรวมของใบสั่งงานที่จัดสรรให้กับทรัพยากร</span><span class="sxs-lookup"><span data-stu-id="1ecfa-104">The **Scheduled work order maintenance jobs** page is used to get an overview of the work orders allocated to a resource.</span></span> <span data-ttu-id="1ecfa-105">ใบสั่งงานโดยใช้ชนิดทรัพยากร "ทรัพยากรบุคคล" "เครื่องมือ" และ "เครื่องจักร" จะแสดงอยู่ในรายการ</span><span class="sxs-lookup"><span data-stu-id="1ecfa-105">Work orders using resource types "Human resources", "Tool", and "Machine" are shown in the list.</span></span> <span data-ttu-id="1ecfa-106">คุณสามารถใช้รายการนี้เพื่อเรียกดูภาพรวมของใบสั่งงานที่จัดสรรให้กับทรัพยากรที่ระบุ</span><span class="sxs-lookup"><span data-stu-id="1ecfa-106">The list can be used to get an overview of work orders allocated to a specific resource.</span></span> <span data-ttu-id="1ecfa-107">คุณยังสามารถใช้ในการค้นหาใบสั่งงานที่จัดสรรให้กับเจ้าหน้าที่บำรุงรักษาผู้ซึ่ง ยกตัวอย่าง โทรแจ้งการป่วยในช่วงเช้า จากนั้นจึงจัดสรรเจ้าหน้าที่บำรุงรักษาอีกคนหนึ่งให้กับงานนี้อย่างรวดเร็ว</span><span class="sxs-lookup"><span data-stu-id="1ecfa-107">You can also use it to quickly find a work order allocated to a maintenance worker who, for example, called in sick this morning, and then quickly allocate another maintenance worker to the job.</span></span>

## <a name="view-scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="1ecfa-108">ดูงานการบำรุงรักษาของใบสั่งงานที่จัดกำหนดการ</span><span class="sxs-lookup"><span data-stu-id="1ecfa-108">View scheduled work order maintenance jobs</span></span>

1. <span data-ttu-id="1ecfa-109">คลิก **การจัดการสินทรัพย์** > **ทั่วไป** > **ใบสั่งงาน** > **งานบำรุงรักษาของใบสั่งงานที่จัดกำหนดการ**</span><span class="sxs-lookup"><span data-stu-id="1ecfa-109">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="1ecfa-110">คุณสามารถดูรายการของใบสั่งงานทั้งหมดที่กำหนดให้กับสถานะวงจรของใบสั่งงาน "จัดกำหนดการแล้ว" หรือ "กำลังดำเนินการ"</span><span class="sxs-lookup"><span data-stu-id="1ecfa-110">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

2. <span data-ttu-id="1ecfa-111">คุณสามารถเรียงลำดับรายการ ตัวอย่างเช่น โดยเจ้าหน้าที่บำรุงรักษา</span><span class="sxs-lookup"><span data-stu-id="1ecfa-111">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="1ecfa-112">คุณยังสามารถใช้ตัวกรองเพื่อจำกัดรายการเพื่อแสดงใบสั่งงานที่จัดสรรให้กับทรัพยากรหรือเจ้าหน้าที่บำรุงรักษาเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="1ecfa-112">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

3. <span data-ttu-id="1ecfa-113">คุณสามารถดูหมายเหตุเกี่ยวกับใบสั่งงาน และหากจำเป็น เพิ่มหมายเหตุใหม่โดยการเลือกงานใบสั่งงานในรายการ และคลิก **หมายเหตุ**</span><span class="sxs-lookup"><span data-stu-id="1ecfa-113">You can see notes on the work order and, if required, add new notes by selecting the work order job in the list and clicking **Notes**.</span></span>

4. <span data-ttu-id="1ecfa-114">หากคุณต้องการจัดสรรเจ้าหน้าที่บำรุงรักษาหนึ่งคนให้กับใบสั่งงาน ให้เลือกใบสั่งงานในรายการ และคลิก **ใบสั่งงาน**</span><span class="sxs-lookup"><span data-stu-id="1ecfa-114">If you want to allocate one maintenance worker to a work order, select the work order in the list, and click **Work order**.</span></span>

5. <span data-ttu-id="1ecfa-115">หน้า **ใบสั่งงาน** เปิดขึ้น</span><span class="sxs-lookup"><span data-stu-id="1ecfa-115">The **Work order** page opens.</span></span> <span data-ttu-id="1ecfa-116">คลิก **จัดส่ง** เพื่อจัดกำหนดการใบสั่งงานให้กับเจ้าหน้าที่บำรุงรักษาเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="1ecfa-116">Click **Dispatch** to schedule the work order to a specific maintenance worker.</span></span>
