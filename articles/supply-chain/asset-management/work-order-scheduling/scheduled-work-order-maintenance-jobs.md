[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="b7d68-104">หน้า **งานบำรุงรักษาของใบสั่งงานที่จัดกำหนดการ** แสดงภาพรวมของใบสั่งงานที่จัดสรรให้กับทรัพยากร</span><span class="sxs-lookup"><span data-stu-id="b7d68-104">The **Scheduled work order maintenance jobs** page shows an overview of the work orders allocated to a resource.</span></span> <span data-ttu-id="b7d68-105">ใบสั่งงานที่ใช้ชนิดทรัพยากร "ทรัพยากรบุคคล" "เครื่องมือ" และ "เครื่องจักร" จะถูกแสดง</span><span class="sxs-lookup"><span data-stu-id="b7d68-105">Work orders using resource types "Human resources", "Tool", and "Machine" are shown.</span></span> <span data-ttu-id="b7d68-106">ตัวอย่างเช่น ถ้าเจ้าหน้าที่บำรุงรักษาโทรแจ้งการป่วยในช่วงเช้า คุณสามารถใช้หน้านี้ในการค้นหาใบสั่งงานที่จัดสรรให้กับผู้ปฏิบัติงานได้อย่างรวดเร็ว และจากนั้นจึงจัดสรรเจ้าหน้าที่บำรุงรักษาอีกคนหนึ่งให้กับงาน</span><span class="sxs-lookup"><span data-stu-id="b7d68-106">For example, if a maintenance worker calls in sick, you can use this page to quickly find work orders allocated to the worker, and then allocate another maintenance worker to the job.</span></span>

## <a name="view-scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="b7d68-107">ดูงานการบำรุงรักษาของใบสั่งงานที่จัดกำหนดการ</span><span class="sxs-lookup"><span data-stu-id="b7d68-107">View scheduled work order maintenance jobs</span></span>

1. <span data-ttu-id="b7d68-108">คลิก **การจัดการสินทรัพย์** > **ทั่วไป** > **ใบสั่งงาน** > **งานบำรุงรักษาของใบสั่งงานที่จัดกำหนดการ**</span><span class="sxs-lookup"><span data-stu-id="b7d68-108">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="b7d68-109">คุณสามารถดูรายการของใบสั่งงานทั้งหมดที่กำหนดให้กับสถานะวงจรของใบสั่งงาน "จัดกำหนดการแล้ว" หรือ "กำลังดำเนินการ"</span><span class="sxs-lookup"><span data-stu-id="b7d68-109">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

2. <span data-ttu-id="b7d68-110">คุณสามารถเรียงลำดับรายการ ตัวอย่างเช่น โดยเจ้าหน้าที่บำรุงรักษา</span><span class="sxs-lookup"><span data-stu-id="b7d68-110">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="b7d68-111">คุณยังสามารถใช้ตัวกรองเพื่อจำกัดรายการเพื่อแสดงใบสั่งงานที่จัดสรรให้กับทรัพยากรหรือเจ้าหน้าที่บำรุงรักษาเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="b7d68-111">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

3. <span data-ttu-id="b7d68-112">คุณสามารถดูบันทึกย่อในใบสั่งงาน และหากจำเป็น เพิ่มบันทึกย่อใหม่โดยการเลือกงานใบสั่งงาน และจากนั้น คลิก **บันทึกย่อ**</span><span class="sxs-lookup"><span data-stu-id="b7d68-112">You can see notes on the work order and, if required, add new notes by selecting the work order job, and then click **Notes**.</span></span>

4. <span data-ttu-id="b7d68-113">หากคุณต้องการจัดสรรเจ้าหน้าที่บำรุงรักษาหนึ่งคนให้กับใบสั่งงาน ให้เลือกใบสั่งงาน และจากนั้น คลิก **ใบสั่งงาน**</span><span class="sxs-lookup"><span data-stu-id="b7d68-113">If you want to allocate one maintenance worker to a work order, select the work order, and then click **Work order**.</span></span>

5. <span data-ttu-id="b7d68-114">หน้า **ใบสั่งงาน** เปิดขึ้น</span><span class="sxs-lookup"><span data-stu-id="b7d68-114">The **Work order** page opens.</span></span> <span data-ttu-id="b7d68-115">คลิก **จัดส่ง** เพื่อจัดกำหนดการใบสั่งงานให้กับเจ้าหน้าที่บำรุงรักษาเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="b7d68-115">Click **Dispatch** to schedule the work order to a specific maintenance worker.</span></span>
