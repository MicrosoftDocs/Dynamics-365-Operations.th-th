# <a name="troubleshoot-warehouse-setup"></a>แก้ไขปัญหาการตั้งค่าคลังสินค้า

[!include [banner](../includes/banner.md)]

หัวข้อนี้อธิบายวิธีการแก้ไขปัญหาทั่วไปที่คุณอาจพบในระหว่างที่คุณตั้งค่าคลังสินค้าใน Microsoft Dynamics 365 Supply Chain Management

## <a name="i-cant-use-any-role-except-administrator-to-access-the-mobile-device-app-emulator"></a>ฉันไม่สามารถใช้บทบาทใด ๆ ยกเว้นผู้ดูแลระบบเพื่อเข้าถึงตัวเลียนแบบแอปบนอุปกรณ์เคลื่อนที่

### <a name="issue-description"></a>คำอธิบายปัญหา

คุณไม่สามารถใช้บทบาทใด ๆ ยกเว้นผู้ดูแลระบบเพื่อเข้าถึงตัวเลียนแบบแอปบนอุปกรณ์เคลื่อนที่

### <a name="issue-resolution"></a>การแก้ไขปัญหา

ตัวเลียนแบบแอปบนอุปกรณ์เคลื่อนที่มีการตั้งค่าให้ทำงานกับบัญชีผู้ดูแลระบบเท่านั้น สำหรับวัตถุประสงค์ในการทดสอบและการดำเนินการสด เราขอแนะนำให้คุณใช้แอปคลังสินค้าเอง
