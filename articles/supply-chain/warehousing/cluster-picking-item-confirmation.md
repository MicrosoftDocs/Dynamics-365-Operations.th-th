# <a name="product-confirmation-for-cluster-picking"></a>การยืนยันผลิตภัณฑ์สำหรับการเลือกคลัสเตอร์
การเบิกสินค้าคลัสเตอร์ช่วยให้คุณสามารถเบิกสินค้าสำหรับใบสั่งหลายใบพร้อมกัน เมื่อมีการนำการเบิกสินค้าคลัสเตอร์มาใช้ การยืนยันสินค้าเป็นสิ่งสำคัญยิ่งในการตรวจสอบสินค้าที่ถูกเพิ่มลงในคลัสเตอร์ คุณสามารถตรวจสอบสินค้าในการเบิกสินค้าคลัสเตอร์ในระหว่างกระบวนการเบิกสินค้าคลัสเตอร์

## <a name="where-it-applies"></a>ตำแหน่งที่ใช้
การตรวจสอบสินค้าสำหรับการเบิกสินค้าคลัสเตอร์มีลักษณะเดียวกับเมื่อคุณตรวจสอบสินค้าในกระบวนการเบิกสินค้าที่ไม่ใช่คลัสเตอร์ การตั้งค่าขึ้นอยู่กับการตั้งค่าบาร์โค้ดของผลิตภัณฑ์

## <a name="set-up-item-verification-with-cluster-picking"></a>ตั้งค่าการตรวจสอบสินค้าที่มีการเบิกสินค้าคลัสเตอร์
1.  บนรายการเมนูอุปกรณ์เคลื่อนที่ เปิดแบบฟอร์มการตั้งค่าสำหรับการยืนยันงาน: **การบริหารคลังสินค้า** > **การบริหารคลังสินค้า** > **การตั้งค่า** > **อุปกรณ์เคลื่อนที่** > **รายการเมนูของอุปกรณ์เคลื่อนที่**
2.  จากรายการเมนูบนอุปกรณ์เคลื่อนที่ เปิด **การตั้งค่าการยืนยันงาน**

| ตัวเลือก        | คำอธิบาย   | 
| ------------- | ------------- |
|การยืนยันผลิตภัณฑ์ | ช่วยให้คุณสามารถตรวจสอบสินค้าคงคลังแต่ละรายการได้จากอุปกรณ์เคลื่อนที่เมื่อมีการสแกน|
