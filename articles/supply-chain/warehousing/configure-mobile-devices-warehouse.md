# <a name="set-up-mobile-devices-for-warehouse-work"></a>ตั้งค่าอุปกรณ์เคลื่อนที่สำหรับงานคลังสินค้า

[!include[banner](../includes/banner.md)]


บทความนี้อธิบายวิธีการตั้งค่าคอนฟิกรายการเมนู ที่พนักงานคลังสินค้าใช้ในการดำเนินงานบนอุปกรณ์เคลื่อนที่

**หมายเหตุ:** บทความนี้ใช้กับคุณลักษณะในการบริหารคลังสินค้า บทความนี้ใช้ไม่ได้กับคุณลักษณะในการบริหารสินค้าคงคลัง รายการเมนูที่ปรากฏในเมนูบนอุปกรณ์เคลื่อนของคลังสินค้าที่ถูกตั้งค่าคอนฟิกในหน้า **รายการเมนูของอุปกรณ์เคลื่อนที่** เนื่องจากรายการเมนูสามารถวางบนเมนูต่าง ๆ มันง่ายต่อที่จะตั้งค่าคอนฟิกโครงสร้างเมนู เพื่อให้งานที่เฉพาะเท่านั้นถูกแสดงแก่ผู้ใช้เฉพาะ คุณสามารถตั้งค่าคอนฟิกรายการเมนูเพื่อดำเนินการงานต่อไปนี้:

-   กระบวนการการสอบถามหรือดำเนินกิจกรรม เช่น การพิมพ์ป้ายชื่อ การสร้างหมายเลขป้ายทะเบียน การเริ่มต้นใบสั่งผลิต หรือการค้นหาข้อมูลเกี่ยวกับสินค้าในสถานที่อย่างรวดเร็ว
-   สร้างงานที่จะดำเนินการผ่านกระบวนการอื่น ตัวอย่างเช่น การรับสินค้าสำหรับใบสั่งซื้อสามารถสร้างงานสำรองสำหรับผู้ปฏิบัติงานอื่น
-   ดำเนินงานที่สร้างขึ้นโดยกระบวนการอื่น (งานที่มีอยู่) เช่น งานสำรองที่สร้างขึ้นเมื่อสินค้าถูกได้รับสำหรับใบสั่งซื้อ

เมื่อต้องการสร้างรายการเมนูสำหรับกิจกรรมหรือการสอบถาม ตั้งค่าฟิลด์ **โหมด** เป็น **ทางอ้อม** จากนั้น รายการของตัวเลือก **รหัสกิจกรรม** จะกลายเป็นพร้อมใช้งาน เพื่อให้คุณสามารถเลือกชนิดของการสอบถาม หรือกิจกรรมที่ไว้ใช้งานกับรายการเมนู เพื่อสร้างรายการเมนูที่ใช้ในการสร้างงานคลังสินค้า ตั้งค่าฟิลด์ **โหมด** เป็น **งาน** รายการของตัวเลือก **กระบวนการสร้างงาน** จากนั้นจะกลายเป็นพร้อมใช้งาน เพื่อสร้างรายการเมนูเพื่อดำเนินงานคลังสินค้าที่มีอยู่ ตั้งฟิลด์ **โหมด** เพื่อ **งาน** และตั้งตัวเลือก **ใช้งานที่มีอยู่** เพื่อ **ใช่** **สาเหตุ:** ฟิลด์เพิ่มเติมอาจมีอยู่สำหรับรายการเมนู ขึ้นอยู่กับโหมดที่คุณเลือกสำหรับรายการเมนู และขึ้นกับว่ามีการใช้รายการเมนูเพื่อดำเนินงานที่มีอยู่หรือไม่ สำหรับข้อมูลเกี่ยวกับการเลือกฟิลด์เพิ่มเติม ให้ดูส่วน "ตัวเลือกรายการเมนูเพิ่มเติมสินค้า" ในบทความนี้ต่อไป สำหรับข้อมูลเกี่ยวกับการเลือกฟิลด์เพิ่มเติม ดูส่วนชื่อว่า "เพิ่มเติมสินค้าตัวเลือกเมนู" ในตอนท้ายของบทความนี้

## <a name="configure-menu-items-to-create-work-for-another-worker-or-process"></a>ตั้งค่าคอนฟิกรายการเมนูเพื่อสร้างงานสำหรับผู้ปฏิบัติงานหรือกระบวนการอื่น
คุณสามารถตั้งค่ารายการเมนูที่สร้างงานสำหรับผู้ปฏิบัติงานอื่นหลังจากการดำเนินการเริ่มต้นถูกดำเนินการบนอุปกรณ์เคลื่อนที่ ตัวอย่างเช่น เมื่อผู้ปฏิบัติงานหนึ่งใช้อุปกรณ์เคลื่อนที่เพื่อรับสินค้า งานการย้ายถูกสร้างสำหรับผู้ปฏิบัติงานอื่น เพื่อตั้งค่ารายการเมนูที่ใช้สร้างงาน ในหน้า **รายการเมนูของอุปกรณ์เคลื่อนที่** ในฟิลด์ **โหมด** เลือก **งาน** ในตารางต่อไปนี้ ตัวเลือกในฟิลด์ **กระบวนการสร้างงาน** ถูกจัดเรียงตามชนิดลำดับงาน

## <a name="require-workers-to-confirm-the-product-location-or-quantity-when-they-pick-items"></a>ต้องมีผู้ปฏิบัติงานเพื่อยืนยันผลิตภัณฑ์ สถานที่ หรือปริมาณเมื่อเบิกสินค้า
คุณสามารถตั้งค่าการยืนยันงานที่จำเป็นให้ผู้ปฏิบัติงานใช้อุปกรณ์เคลื่อนที่เพื่อลงทะเบียนสถานที่หรือปริมาณ เมื่อผู้ปฏิบัติงานดำเนินการงานในคลังสินค้า การยืนยันงานช่วยรับประกันว่าผู้ปฏิบัติงานอยู่ในสถานที่ที่ถูกต้องหรือกำลังจัดการปริมาณสินค้าที่ถูกต้อง คุณยังสามารถเปิดใช้งาน Microsoft Dynamics 365 for Finance and Operations เพื่อยืนยันการลงทะเบียนผู้ปฏิบัติงานโดยอัตโนมัติ ถ้าคุณเปิดใช้งานการยืนยันอัตโนมัติ คุณไม่สามารถระบุการยืนยันสำหรับสถานที่หรือปริมาณ การยืนยันงานยังรวมผลิตภัณฑ์และผลิตภัณฑ์ย่อย นอกจากนี้ คุณสามารถลงทะเบียนการยืนยันโดยการสแกนบาร์โค้ด เพื่อยืนยันผลิตภัณฑ์และผลิตภัณฑ์ย่อย คุณต้องป้อนรหัสสำหรับผลิตภัณฑ์หรือผลิตภัณฑ์ย่อย รหัสนี้สามารถเป็นรหัสผลิตภัณฑ์ รหัสการค้นหาผลิตภัณฑ์ รหัสภายนอก GTIN หรือบาร์โค้ด หลังจากที่คุณป้อนรหัสหรือสแกนบาร์โค้ด มิติสำหรับผลิตภัณฑ์ย่อยถูกแสดงบนอุปกรณ์เคลื่อนที่ 

ตารางต่อไปนี้อธิบายถึงชนิดงานต่างๆ ที่คุณสามารถใช้การยืนยันงานด้วย

| ตัวเลือก                 | คำอธิบาย                                                                |
|------------------------|----------------------------------------------------------------------------|
| เลือก                   | จำเป็นต้องมีการยืนยันเมื่อสินค้าถูกเบิก                                |
| ส่งสินค้า                    | จำเป็นต้องมีการยืนยันเมื่อสินค้าถูกส่งในที่ตั้ง                     |
| การตรวจนับ               | ต้องมีการยืนยันในระหว่างการตรวจนับตามรอบ                                |
| การปรับปรุง            | จำเป็นต้องมีการยืนยันเมื่อปริมาณสินค้าคงคลังถูกปรับปรุง               |
| กำหนดเอง                 | ต้องมีการยืนยันสำหรับงานที่กำหนดเอง                                      |
| การตรวจสอบสินค้า             | จำเป็นต้องมีการยืนยันเมื่อสินค้าถูกย้ายไปตรวจสอบ                   |
| การสร้างป้ายทะเบียน | จำเป็นต้องมีการยืนยันเมื่อสินค้าถูกรวมเพื่อสร้างป้ายทะเบียน |
| พิมพ์                  | จำเป็นต้องมีการยืนยันเมื่อป้ายชื่อทะเบียนถูกพิมพ์                |
| การเปลี่ยนสถานะ          | จำเป็นต้องมีการยืนยันเมื่อสถานะของสินค้าคงคลังถูกเปลี่ยน              |
