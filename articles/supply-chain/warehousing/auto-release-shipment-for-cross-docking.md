# <a name="auto-release-shipment-for-cross-docking"></a>การจัดส่งแบบนำออกใช้อัตโนมัติสำหรับการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า

[!include [banner](../includes/banner.md)]

หัวข้อนี้อธิบายถึงกลยุทธ์การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า ที่จะช่วยคุณนำใบสั่งซื้อตามความต้องการออกไปยังคลังสินค้า เมื่อใบสั่งผลิตที่จัดหาปริมาณความต้องการมีการรายงานเป็นเสร็จสิ้น วิธีการนี้ ปริมาณที่ต้องการสำหรับการเติมสินค้าของใบสั่งที่ต้องการจะมีการย้ายโดยตรงจากสถานที่เอาท์พุทการผลิตไปยังสถานที่เก็บสินค้าขาออก

การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า เป็นขั้นตอนการจัดการคลังสินค้า ซึ่งปริมาณที่ต้องการเติมสินค้าตามใบสั่งขาออกมีการกำหนดโดยท่าออกของหรือพื้นที่จัดเตรียมจากสถานที่ที่ได้รับใบสั่งขาเข้า (ใบสั่งขาเข้าสามารถเป็นใบสั่งซื้อ ใบสั่งโอนย้าย หรือใบสั่งผลิตก็ได้) ขณะที่คุณลักษณะการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าขั้นสูงสนับสนุนใบสั่งการจัดหาวัสดุและความต้องการทั้งหมด และต้องการความต้องการขาออกที่มีการนำออกใช้ก่อนที่โอกาสของการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าจะมีการระบุ คุณลักษณะการจัดส่งแบบนำออกใช้อัตโนมัติจะมีลักษณะดังนี้:

- สนับสนุนเฉพาะใบสั่งผลิตที่เป็นการจัดหาวัสดุ และเฉพาะใบสั่งขายและใบสั่งโอนย้ายที่เป็นความต้องการ
- การดำเนินการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าสามารถเริ่มได้ แม้ว่าใบสั่งตามความต้องการยังไม่มีการนำออกใช้ไปยังคลังสินค้า ก่อนที่ใบรับการจัดหาสินค้าจะมีการลงทะเบียน (ก่อนที่การผลิตจะมีการรายงานเป็นเสร็จสิ้น)

ฟังก์ชันการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้ามีข้อดีสองประการ:

- การดำเนินการของคลังสินค้าสามารถข้ามขั้นตอนของการส่งปริมาณของสินค้าสำเร็จรูปไปยังพื้นที่จัดเก็บคลังสินค้าปกติ หากปริมาณเหล่านั้นจะมีการเบิกอีกครั้งเพื่อเติมสินค้าของใบสั่งขาออก แทนที่จะทำเช่นนั้น สามารถทำการย้ายปริมาณจากสถานที่ขาออกไปยังสถานที่สุดท้ายได้ในครั้งเดียว วิธีนี้ ฟังก์ชันจะช่วยลดจำนวนครั้งของการจัดการสินค้าคงคลัง และช่วยเพิ่มเวลาและพื้นที่ของการผลิตในคลังสินค้าได้ในที่สุด
- การดำเนินการของคลังสินค้าสามารถเลื่อนการนำออกใช้ของใบสั่งขายและใบสั่งโอนย้ายไปยังคลังสินค้า จนกว่าผลผลิตของสินค้าสำเร็จรูปสำหรับใบสั่งผลิตที่เกี่ยวข้องจะมีการรายงานเป็นเสร็จสิ้น ข้อดีนี้ อาจมีได้ใช้งานในสภาพแวดล้องของการผลิตแบบผลิตตามสั่ง ที่ระยะเวลารอสินค้าของการผลิตอาจนานกว่าระยะเวลารอสินค้าในสภาพแวดล้อมที่ผลิตตามสินค้าคงคลัง

## <a name="prerequisites"></a>ข้อกำหนดเบื้องต้น

| ข้อกำหนดเบื้องต้น | คำอธิบาย |
|---|---|
| รายการ | สินค้าที่จะต้องใช้สำหรับกระบวนการบริหารคลังสินค้า<p>**หมายเหตุ:** สินค้าที่เปิดใช้งานน้ำหนักจริงจะไม่มีการรวมในกระบวนการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า</p> |
| คลังสินค้า | คลังสินค้าต้องถูกเปิดใช้งานสำหรับกระบวนการจัดการคลังสินค้า  |
| แม่แบบการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า | ต้องตั้งค่าอย่างน้อยหนึ่งแม่แบบของการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าที่ใช้นโยบายการนำออกใช้ความต้องการแบบ **ที่ใบรับการจัดหาสินค้า** สำหรับคลังสินค้าที่มีให้ |
| คลาสงาน | ต้องสร้างรหัสคลาสงานการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าสำหรับชนิดใบสั่งงาน **การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า** |
| เท็มเพลตงาน | จำเป็นต้องมีแม่แบบงานของชนิดใบสั่งงาน **การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า** เพื่อสร้างการเบิกและการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า |
| คำสั่งสถานที่ | จำเป็นต้องมีคำสั่งสถานที่ของชนิดใบสั่งงาน **การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า** เพื่อนำทางการส่งสินค้าในสถานสถานที่ที่มีปริมาณใบสั่งขายที่จะมีการบรรจุและทำการส่ง |
| การทำเครื่องหมายระหว่างใบสั่งความตามความต้องการและใบสั่งผลิต | ระบบคลังสินค้าสามารถทริกเกอร์การนำออกใช้อัตโนมัติของการส่งสินค้าตามใบสั่งขาออก และสร้างงานการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าจากสถานที่เอาท์พุทของการดำเนินการรายงานว่าเสร็จสิ้นแล้ว เฉพาะเพื่อใบสั่งขายและใบสั่งโอนย้ายมีการจองและทำเครื่องหมายตามใบสั่งผลิต |

## <a name="example-cross-docking-flow"></a>ตัวอย่างขั้นตอนการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า

ขั้นตอนการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าโดยทั่วไปประกอบด้วยขั้นตอนหลักต่อไปนี้

1. ผู้รับใบสั่งขายสร้างใบสั่งขายสำหรับผลิตภัณฑ์ที่ผลิตตามสั่ง โดยทั่วไปแล้ว ปริมาณที่ต้องการจะไม่มีอยู่ในคลังและต้องผลิตขึ้นมาก่อน
2. ผู้รับใบสั่งขายสร้างใบสั่งผลิตโดยตรงจากรายการใบสั่งขาย วิธีนี้ ผู้รับใบสั่งขายจะจองและทำเครื่องหมายปริมาณใบสั่งขายกับปริมาณใบสั่งผลิต 

    อีกวิธีหนึ่งคือ โปรแกรมวางแผนการผลิตระบุว่าการทำเครื่องหมายต้องได้รับการอัพเดตเมื่อแผนการใบสั่งผลิตกำลังมีการรับรอง

3. ใบสั่งผลิตจะทำตามขั้นตอนต่อไปนี้:

    1. โปรแกรมวางแผนการผลิตจะประเมินและนำออกใช้ใบสั่งผลิต (การประเมินจะรวมถึงการจองวัตถุดิบ และการนำออกใช้ รวมถึงการนำออกใช้ไปยังคลังสินค้า)
    2. ผู้ปฏิบัติงานคลังสินค้าจะเริ่มต้นและสิ้นสุดการเบิกวัตถุดิบจากสถานที่ในการจัดเก็บ ไปยังสถานที่อินพุทการผลิต ตามงานการเบิกสินค้าสำหรับการผลิต
    3. ตัวดำเนินการผลิตเริ่มใบสั่งผลิต
    4. ในการดำเนินการครั้งสุดท้าย ตัวดำเนินการเครื่องจักรใช้อุปกรณ์เคลื่อนที่เพื่อรายงานใบสั่งผลิตเป็นเสร็จสิ้น

4. ระบบจะใช้การตั้งค่าเพื่อระบุเหตุการณ์การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าสำหรับสองใบสั่งที่เชื่อมโยงกัน แล้วทำงานเหล่านี้ให้เสร็จสมบูรณ์

    1. นำออกใช้ใบสั่งขายที่เกี่ยวข้องอัตโนมัติไปยังคลังสินค้าเพื่อสร้างการจัดส่ง
    2. สร้างงานการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าอัตโนมัติที่มีคำแนะนำเพื่อเบิกสินค้าสำเร็จรูปจากสถานที่เอาท์พุทและย้ายไปยังสถานที่ขาออกที่คำสั่งสถานที่การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้ามีการมอบหมายให้กับใบสั่งขาย
    3. พร้อมต์ตัวดำเนินการเครื่องจักรเพื่อทำงานการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าให้เสร็จทันทีหลังจากใบสั่งผลิตมีการรายงานเป็นเสร็จสิ้น

5. หลังจากที่งานการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าเสร็จสิ้น และปริมาณมีการบรรจุบนพาหนะ โปรแกรมวางแผนคลังสินค้าขาออกจะยืนยันการจัดส่งของการขาย

## <a name="example-scenario"></a>ตัวอย่างสถานการณ์จำลอง

สำหรับสถานการณ์จำลองนี้ คุณต้องมีการติดตั้งข้อมูลสาธิต และคุณต้องใช้บริษัทข้อมูลสาธิต **USMF**

### <a name="set-up-cross-docking-that-uses-the-auto-release-shipment-feature"></a>ตั้งค่าการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าที่ใช้คุณลักษณะการนำออกใช้การจัดส่งอัตโนมัติ

#### <a name="cross-docking-template"></a>แม่แบบการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า

1. ไปที่ **การจัดการคลังสินค้า** \> **การตั้งค่า** \> **งาน** \> **แม่แบบการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า**
2. เลือก **ใหม่**
3. ในฟิลด์ **หมายเลขลำดับ** ให้ป้อน **1**
4. ในฟิลด์ **รหัสแม่แบบการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า** ให้ป้อนชื่อ เช่น **XDock\_RAF**
5. ในฟิลด์ **นโยบายการนำออกใช้ความต้องการ** ให้เลือก **ที่ใบรับการจัดหาสินค้า**
6. ในฟิลด์ **คลังสินค้า** ให้ป้อนหมายเลขของคลังสินค้าที่คุณต้องการตั้งค่ากระบวนการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า สำหรับตัวอย่างนี้ ให้เลือก **51**

    > [!NOTE]
    > ทันทีที่คุณเลือก **ที่ใบรับการจัดหาสินค้า** เป็นนโยบายการนำออกใช้ความต้องการสำหรับแม่แบบนี้ ฟิลด์อื่นในเพจจะไม่สามารถใช้งานได้ เช่นเดียวกัน คุณไม่สามารถกำหนดแหล่งที่มาของการจัดหาวัสดุได้ ลักษณะการทำงานนี้เกิดขึ้นเนื่องจากการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าที่ใช้คุณลักษณะการจัดส่งอัตโนมัติ จะสนับสนุนเฉพาะใบสั่งผลิตที่เป็นแหล่งที่มาของการจัดหาวัสดุ และต้องการการทำเครื่องหมายระหว่างใบสั่งขายและใบสั่งผลิต หากคุณเลือก **ก่อนใบรับการจัดหาสินค้า** เป็นนโยบายการนำออกใช้ความต้องการ ฟิลด์บนแท็บ **การวางแผน** และ **แหล่งที่มาของการจัดหาสินค้า** จะมีให้ใช้งานและสามารถแก้ไขได้

#### <a name="work-classes"></a>คลาสงาน

1. ไปที่ **การจัดการคลังสินค้า** \> **การตั้งค่า** \> **งาน** \> **คลาสงาน**
2. เลือก **ใหม่**
3. ในฟิลด์ **รหัสคลาสงาน** ให้ป้อนชื่อ เช่น **CrossDock**
4. ในฟิลด์ **ชนิดใบสั่งงาน** เลือก **การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า**

เมื่อต้องการจำกัดชนิดของสถานที่การจัดเก็บที่สินค้าสำเร็จรูปที่กำลังทำการส่งผ่านศูนย์เปลี่ยนถ่ายสินค้า คุณสามารถระบุชนิดของสถานที่ที่ถูกต้องได้หนึ่งรายการหรือมากกว่า

#### <a name="work-templates"></a>เท็มเพลตงาน

1. ไปที่ **การจัดการคลังสินค้า** \> **การตั้งค่า** \> **งาน** \> **แม่แบบงาน**
2. ในฟิลด์ **ชนิดใบสั่งงาน** เลือก **การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า**
3. เลือก **ใหม่**
4. ในฟิลด์ **หมายเลขลำดับ** ให้ป้อน **1**
5. ในฟิลด์ **แม่แบบงาน** ให้ป้อนชื่อ เช่น **CrossDock\_51**
6. เลือก **บันทึก**
7. ในส่วน **รายละเอียดแม่แบบงาน** เลือก **สร้าง**
8. สำหรับรายการใหม่ ในฟิลด์ **ชนิดงาน** เลือก **การเบิก** ในฟิลด์ **รหัสคลาสงาน** เลือก **CrossDock**
9. เลือก **ใหม่**
10. สำหรับรายการใหม่ ในฟิลด์ **ชนิดงาน** เลือก **การส่งสินค้า** ในฟิลด์ **รหัสคลาสงาน** เลือก **CrossDock**

#### <a name="location-directives"></a>คำสั่งสถานที่

ขั้นตอนการส่งสินค้าแบบมาตรฐานสำหรับสินค้าสำเร็จรูป จำเป็นต้องมีคำสั่งสถานที่ **การส่งสินค้า** เพื่อนำทางการเคลื่อนย้ายของปริมาณการผลิตที่เบิกไปยังพื้นที่การจัดเก็บปกติ เช่นเดียวกัน คุณจำเป็นต้องตั้งค่าคำสั่งสถานที่ **การส่งสินค้า** ผ่านศูนย์เปลี่ยนถ่ายสินค้าที่วางแผนไว้ เพื่อแนะนำงานให้ทำการส่งปริมาณที่เสร็จสิ้นแล้วในสถานที่ขาออกที่แต่งตั้งไว้ที่บริการของการจัดส่งสินค้าที่เกี่ยวข้องกับใบสั่งขาย

สำหรับการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า ที่เป็นการส่งสินค้าสำเร็จรูปปกติ คุณไม่ต้องส้รางคำสั่งสถานที่เพื่อการดำเนินการงานการเบิก เนื่องจากมีการระบุสถานที่เอาท์พุทแล้ว นอกจากนี่ สถานที่เอาท์พุทนี้ควรจะตั้งค่าเป็นสถานที่เอาท์พุทเริ่มต้นในเรกคอร์ดที่เกี่ยวข้องกับทรัพยากร (กล่าวคือ ทรัพยากร ความสัมพันธ์กลุ่มทรัพยากร หรือกลุ่มทรัพยากร) หรือเป็นสถานที่การผลิตสินค้าสำเร็จรูปเริ่มต้นสำหรับคลังสินค้า

1. ไปที่ **การจัดการคลังสินค้า** \> **การตั้งค่า** \> **คำสั่งสถานที่**
2. ในฟิลด์ **ชนิดใบสั่งงาน** เลือก **การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า**
3. เลือก **ใหม่**
4. ในฟิลด์ **หมายเลขลำดับ** ให้ป้อน **1**
5. ในฟิลด์ **ชื่อ** ให้ป้อนชื่อ เช่น **Baydoor**
6. ในฟิลด์ **ประเภทงาน** ให้เลือก **ส่งสินค้า**
7. ในฟิลด์ **ไซต์** ให้เลือก **5**
8. ในฟิลด์  **คลังสินค้า** เลือก **51**
9. บนแท็บด่วน **รายการ** เลือก **สร้าง**
10. ในฟิลด์ **ปริมาณสิ้นสุด** ให้ป้อนปริมาณที่มากที่สุดของช่วงจำนวน เช่น **1000000**
11. เลือก **บันทึก**
12. บนแท็บด่วน **การดำเนินการของคำสั่งสถานที่** เลือก **สร้าง**
13. ในฟิลด์ **ชื่อ** ให้ป้อนชื่อ เช่น **Baydoor**
14. เลือก **บันทึก**
15. คุณสามารถใช้สินเชื่อแบบสอบถามมาตรฐาน เพื่อจำกัดสถานที่ของการส่งเป็นสถานที่ที่ระบุ เลือก **แก้ไขการสอบถาม** แล้วเลือก **51** เป็นเกณฑ์สำหรับฟิลด์ **คลังสินค้า** ในตาราง **สถานที่**

### <a name="cross-dock-finished-goods-to-the-outbound-location"></a>การส่งสินค้าสำเร็จรูปผ่านศูนย์เปลี่ยนถ่ายสินค้าไปยังสถานที่ขาออก

ในการส่งปริมาณสินค้าสำเร็จรูปผ่านศูนย์เปลี่ยนถ่ายสินค้า ไปยังสถานที่ขาออกของใบสั่งขายที่เกี่ยวข้อง ให้ทำตามขั้นตอนต่อไปนี้

1. ไปที่ **การขายและการตลาด** \> **ใบสั่งขาย** \> **ใบสั่งขายทั้งหมด**
2. เลือก **ใหม่**
3. สำหรับหัวข้อของใบสั่งขาย เลือกบัญชีลูกค้า **US-001** และคลังสินค้าที่ตั้งค่าสำหรับการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าที่ใช้คุณลักษณะการจัดส่งแบบนำออกใช้อัตโนมัติ
4. เพิ่มรายการสำหรับผลิตภัณฑ์สำเร็จรูป แล้วป้อนปริมาณเป็น **10**
5. ในส่วน **รายการใบสั่งขาย** เลือก **ผลิตภัณฑ์และการจัดหาวัสดุ** \> **ใบสั่งผลิต**
6. ในกล่องโต้ตอบ **สร้างใบสั่งผลิต** ตรวจทานค่าเริ่มต้น แล้วเลือก **สร้าง** ใบสั่งผลิตใหม่มีถูกสร้างขึ้นและมีการเชื่อมโยงกับใบสั่งขาย (ที่ถูกจองและถูกทำเครื่องหมายไว้)
7. เลือกระบุได้: เปลี่ยนค่าของฟิลด์ **ปริมาณ** เพื่อให้มีค่ามากกว่าที่ต้องการเติมในใบสั่งขาย เมื่อปริมาณการผลิตมีการรายงานเป็นเสร็จสิ้น ระบบจะสร้างงานส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าสำหรับปริมาณที่ทำเครื่องหมายไว้ และส่งงานสำหรับปริมาณที่คงเหลืออยู่ ตามกระบวนการปกติสำหรับการจัดการส่งสินค้าสำเร็จรูป

    ตามที่ได้กล่าวไว้ การทำเครื่องหมายต้องทำระหว่างใบสั่งขายและใบสั่งผลิต มิฉะนั้น การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าจะไม่ทำงาน สามารถสร้างการทำเครื่องหมายได้หลายวิธี:

    - ระบบสามารถสร้างการทำเครื่องหมายโดยอัตโนมัติในสถานการณ์ต่อไปนี้:

        - ใบสั่งผลิตจะถูกสร้างด้วยตัวเองโดยตรงจากรายการใบสั่งขาย (ดูขั้นตอนที่ 6)
        - แผนการใบสั่งผลิตได้รับการยืนยัน และฟิลด์ **อัพเดตการทำเครื่องหมาย** ตั้งค่าเป็น **มาตรฐาน**

    - ผู้ใช้สามารถสร้างการทำเครื่องหมายได้ด้วยตนเองโดยเปิดเพจ **การทำเครื่องหมาย** จากรายการใบสั่งตามความต้องการ

    > [!NOTE]
    > ไม่สามารถสร้างการาทำเครื่องหมายด้วยตนเองสำหรับสินค้าที่ตั้งค่าให้ใช้ค่ามาตรฐานและค่าเฉลี่ยถ่วงน้ำหนักเป็นแบบจำลองสินค้าคงคลัง

8. ในเพจ **ใบสั่งผลิต** บนบานหน้าต่างการดำเนินการ ในแท็บ **ใบสั่งผลิต** ในกลุ่ม **กระบวนการ** เลือก **ประเมิน** แล้วเลือก **ตกลง** ใบสั่งมีการประเมิน และปริมาณวัตถุดิบมีการจองไว้เพื่อการผลิต
9. บนบานหน้าต่างการดำเนินการ ในแท็บ **ใบสั่งผลิต** ในกลุ่ม **กระบวนการ** เลือก **นำออกใช้** แล้วเลือก **ตกลง** งานการเบิกคลังสินค้าถูกสร้างสำหรับวัตถุดิบ
10. เปิดและตรวจทานงาน บนบานหน้าต่างการดำเนินการ ในแท็บ **คลังสินค้า** ในกลุ่ม **ทั่วไป** เลือก **รายละเอียดงาน** จดบันทึกรหัสงาน
11. ลงชื่อเข้าใช้แอปการจัดการคลังสินค้าบนมือถือเพื่อเรียกใช้งานในคลังสินค้า 51
12. ไปที่ **การผลิต** \> **การเบิกการผลิต**
13. ป้อนรหัสงานที่จะเริ่มต้นและทำการเบิกวัตถุดิบให้เสร็จสมบูรณ์ 

    หลังจากที่งานมีการรายงานเป็นเสร็จสิ้น ปริมาณของวัตถุดิบจะมีให้ใช้งานในสถานที่อินพุทการผลิต (**005** ในข้อมูลตัวอย่าง USMF) และการดำเนินการของใบสั่งผลิตสามารถเริ่มทำงานได้

14. ในเพจ **ใบสั่งผลิต** บนบานหน้าต่างการดำเนินการ ในแท็บ **ใบสั่งผลิต** ในกลุ่ม **กระบวนการ** เลือก **เริ่มทำงาน** แล้วเลือก **ตกลง**
15. ในแอป ให้ไปที่ **การผลิต** \> **RAF และการส่งสินค้า**
16. ในฟิลด์ **รหัสผลิตภัณฑ์** ป้อนเลขใบสั่งผลิตและรายละเอียดที่จำเป็นอื่น แล้วเลือก **ตกลง**

โปรดสังเกตเหตุการณ์ที่จะเกิดขึ้นต่อไปนี้:

- การนำออกใช้ไปยังคลังสินค้ามีการทริกเกอร์สำหรับใบสั่งขายที่เชื่อมโยง
- โดยอิงตามการนำออกใช้ จะมีการสร้างงานการจัดส่งและการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า งานนี้จะแนะนำตัวดำเนินการคลังสินค้าให้เบิกปริมาณที่ต้องการเติมในรายการใบสั่งขาย และส่งสินค้าในสถานที่ขาออกที่ระบุไว้ในคำสั่งสถานที่การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า
- หากปริมาณใบสั่งผลิตมีมากกว่าปริมาณที่ต้องการโดยใบสั่งขาย จะมีการสร้างงานการส่งสินค้าปกติ งานนี้จะแนะนำให้ตัวดำเนินการคลังสินค้าทำการเบิกปริมาณของสินค้าสำเร็จรูปที่คงเหลืออยู่หลังจากการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้า และเคลื่อนย้ายไปยังที่จัดเก็บปกติ ตามคำสั่งสถานที่
