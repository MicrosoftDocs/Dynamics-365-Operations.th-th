<span data-ttu-id="bca5b-104">หัวข้อนี้อธิบายวิธีการใช้เท็มเพลตงานและคำสั่งสถานที่ เพื่อกำหนดวิธีและตำแหน่งที่งานจะถูกดำเนินการในคลังสินค้า</span><span class="sxs-lookup"><span data-stu-id="bca5b-104">This topic describes how to use work templates and location directives to determine how and where work is carried out in the warehouse.</span></span>

<span data-ttu-id="bca5b-105">คำแนะนำที่พนักงานในคลังสินค้าได้รับทางอุปกรณ์มือถือถูกกำหนดโดยเท็มเพลตงานที่คุณตั้งค่าใน Dynamics 365 Supply Chain Management เพื่อกำหนดกระบวนการคลังสินค้าและงานที่หลากหลาย</span><span class="sxs-lookup"><span data-stu-id="bca5b-105">The instructions that warehouse workers receive on a mobile device are determined by the Dynamics 365 Supply Chain Management work templates that you set up to define the various warehouse processes and tasks.</span></span> <span data-ttu-id="bca5b-106">เท็มเพลตงานกำหนดวิธีดำเนินงานสำหรับกระบวนการคลังสินค้าแต่ละรายการ</span><span class="sxs-lookup"><span data-stu-id="bca5b-106">Work templates determine how the work is performed for each warehouse process.</span></span> <span data-ttu-id="bca5b-107">ด้วยการเชื่อมโยงคำสั่งสถานที่กับเท็มเพลตงาน คุณสามารถช่วยรับประกันว่างานเกิดขึ้นเฉพาะในพื้นที่จริงของคลังสินค้า</span><span class="sxs-lookup"><span data-stu-id="bca5b-107">By linking a location directive to work templates, you can help guarantee that work occurs in specific physical areas of the warehouses.</span></span>

## <a name="work-templates"></a><span data-ttu-id="bca5b-108">เท็มเพลตงาน</span><span class="sxs-lookup"><span data-stu-id="bca5b-108">Work templates</span></span>

<span data-ttu-id="bca5b-109">หน้า **เท็มเพลตงาน** ช่วยให้คุณสามารถกำหนดการดำเนินงานที่ต้องดำเนินการในคลังสินค้า</span><span class="sxs-lookup"><span data-stu-id="bca5b-109">The **Work templates** page lets you define the work operations that must be performed in the warehouse.</span></span> <span data-ttu-id="bca5b-110">โดยปกติ การดำเนินงานคลังสินค้าประกอบด้วยสองการดำเนินการ: ผู้ปฏิบัติงานคลังสินค้าเลือกสินค้าคงคลังคงเหลือในสถานที่หนึ่งและหลังจากนั้นก็ใสนำสินค้าคงคลังที่เลือกมาไปลงไว้อีกสถานที่หนึ่ง</span><span class="sxs-lookup"><span data-stu-id="bca5b-110">Typically, warehouse work operations consist of a pair of actions: a warehouse worker picks up on-hand inventory in one location and then puts the picked inventory down in another location.</span></span>

<span data-ttu-id="bca5b-116">การตั้งค่าในคำอธิบายหัวข้องานสามารถใช้เพื่อกำหนดว่าเมื่อใดที่งานชิ้นใหม่ควรถูกสร้าง</span><span class="sxs-lookup"><span data-stu-id="bca5b-116">The settings in the work header definition can be used to determine when a new piece of work should be created.</span></span> <span data-ttu-id="bca5b-117">ตัวอย่างเช่น คุณสามารถตั้งค่าจำนวนสูงสุดของการเบิกสินค้าและเวลาที่คาดว่าจะเบิกสินค้า</span><span class="sxs-lookup"><span data-stu-id="bca5b-117">For example, you can set a maximum number of pick lines and a maximum expected pick time.</span></span> <span data-ttu-id="bca5b-118">จากนั้น ถ้างานสำหรับกระบวนการเบิกสินค้าสำหรับใบสั่งขายนั้นเกินอย่างใดอย่างหนึ่งของค่าเหล่านี้ งานนั้นจะถูกแบ่งออกเป็นงานสองชิ้น</span><span class="sxs-lookup"><span data-stu-id="bca5b-118">Then, if the work for a sales order picking process exceeds either of those values, that work is split into two pieces of work.</span></span> 

<span data-ttu-id="bca5b-119">รายการงานแสดงถึงหน้าที่จริงที่จำเป็นในกระบวนการการดำเนินงาน</span><span class="sxs-lookup"><span data-stu-id="bca5b-119">The work lines represent the physical tasks that are required in order to process the work.</span></span> <span data-ttu-id="bca5b-120">ตัวอย่างเช่น สำหรับกระบวนการคลังสินค้าขาออกอาจมีรายการงานสำหรับการเบิกสินค้าภายในคลังสินค้าและอีกรายการหนึ่งสำหรับการย้ายสินค้าเหล่านั้นไปยังพื้นที่จัดเตรียมสินค้า</span><span class="sxs-lookup"><span data-stu-id="bca5b-120">For example, for an outbound warehouse process, there might be a work line for picking up the items within the warehouse and another line for putting those items into a staging area.</span></span> <span data-ttu-id="bca5b-121">จากนั้นสามารถมีรายการเพิ่มเติมสำหรับการเบิกสินค้าจากการจัดเตรียม และอีกรายการหนึ่งสำหรับการย้ายสินค้าไปยังรถบรรทุกที่เป็นส่วนหนึ่งของกระบวนการบรรทุกสินค้า</span><span class="sxs-lookup"><span data-stu-id="bca5b-121">There can then be an additional line for picking the items from staging, and another line for putting the items into a truck as part of the loading process.</span></span> <span data-ttu-id="bca5b-122">คุณสามารถตั้งค่า *รหัสคำสั่ง* บนรายการเท็มเพลตงานได้</span><span class="sxs-lookup"><span data-stu-id="bca5b-122">You can set a *directive code* on work template lines.</span></span> <span data-ttu-id="bca5b-123">รหัสคำสั่งจะเชื่อมโยงกับคำสั่งสถานที่และดังนั้นจึงช่วยรับประกันว่างานคลังสินค้าจะดำเนินการในสถานที่ที่ถูกต้องในคลังสินค้า</span><span class="sxs-lookup"><span data-stu-id="bca5b-123">A directive code is linked to a location directive and therefore helps guarantee that the warehouse work is processed in the correct location in the warehouse.</span></span> 

<span data-ttu-id="bca5b-124">คุณสามารถตั้งค่าการสอบถามเพื่อควบคุมเมื่อเท็มเพลตงานโดยเฉพาะนั้นถูกใช้</span><span class="sxs-lookup"><span data-stu-id="bca5b-124">You can set up a query to control when a particular work template is used.</span></span> <span data-ttu-id="bca5b-125">ตัวอย่างเช่น คุณสามารถกำหนดข้อจำกัดเพื่อเท็มเพลตงโดยเฉพาะนั้นจะถูกใช้สำหรับงานในคลังสินค้าเฉพาะเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="bca5b-125">For example, you can set a limitation so that a particular template can be used for work only in a specific warehouse.</span></span> <span data-ttu-id="bca5b-126">อีกทางหนึ่งคือ คุณอาจมีเท็มเพลตต่างๆที่ใช้ในการสร้างงานสำหรับการประมวลผลใบสั่งขายขาออก ขึ้นอยู่กับช่องทางการขาย</span><span class="sxs-lookup"><span data-stu-id="bca5b-126">Alternatively, you might have several templates that are used to create work for outbound sales order processing, depending on the sales origin.</span></span> <span data-ttu-id="bca5b-127">ระบบใช้ฟิลด์ **หมายเลขลำดับ** เพื่อกำหนดลำดับที่เท็มเพลตงานมีอยู่นั้นถูกประเมิน</span><span class="sxs-lookup"><span data-stu-id="bca5b-127">The system uses the **Sequence number** field to determine the order that the available work templates are assessed in.</span></span> <span data-ttu-id="bca5b-128">ดังนั้น ถ้าคุณมีแบบสอบถามที่ระบุเฉพาะสำหรับเท็มเพลตงานเฉพาะ คุณควรกำหนดหมายเลขลำดับต่ำให้แก่เท็มเพลตงานนั้น</span><span class="sxs-lookup"><span data-stu-id="bca5b-128">Therefore, if you have a very specific query for a particular work template, you should give it a low sequence number.</span></span> <span data-ttu-id="bca5b-129">จากนั้นแบบสอบถามนั้นจะถูกประเมินก่อนที่จะสอบถามอื่นๆ คำสั่งที่ธรรมดาขึ้น</span><span class="sxs-lookup"><span data-stu-id="bca5b-129">That query will then be evaluated before other, more general queries.</span></span> 

<span data-ttu-id="bca5b-130">เพื่อหยุดหรือหยุดกระบวนการงานชั่วคราว คุณสามารถใช้การตั้งค่า **หยุดงาน** บนรายการงานได้</span><span class="sxs-lookup"><span data-stu-id="bca5b-130">To stop or pause a work process, you can use the **Stop work** setting on the work line.</span></span> <span data-ttu-id="bca5b-131">ในกรณีดังกล่าว ผู้ปฏิบัติงานที่กำลังทำงานจะไม่ถูกขอให้ดำเนินการขั้นตอนถัดไปของรายการงาน</span><span class="sxs-lookup"><span data-stu-id="bca5b-131">In that case, the worker who is performing the work won't be asked to perform the next work line step.</span></span> <span data-ttu-id="bca5b-132">เพื่อเลื่อนไปยังขั้นตอนถัดไป ที่ผู้ปฏิบัติงานหรือผู้ปฏิบัติงานผู้อื่นต้องเลือกงานอีกครั้ง</span><span class="sxs-lookup"><span data-stu-id="bca5b-132">To move on to the next step, that worker or another worker must select the work again.</span></span> <span data-ttu-id="bca5b-133">คุณยังสามารถแยกหน้าภายในชิ้นงาน โดยใช้อื่นซึ่งไม่ใช่ *รหัสคลาสงาน* ในรายการเท็มเพลงานงได้</span><span class="sxs-lookup"><span data-stu-id="bca5b-133">You can also separate the tasks within a piece of work by using a different *work class ID* on the work template lines.</span></span>

## <a name="location-directives"></a><span data-ttu-id="bca5b-134">คำสั่งสถานที่</span><span class="sxs-lookup"><span data-stu-id="bca5b-134">Location directives</span></span>

<span data-ttu-id="bca5b-135">คำสั่งสถานที่มีกฎซึ่งช่วยให้ระบุการเบิกสินค้าและกำหนดสถานที่เก็บสำหรับการย้ายสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="bca5b-135">Location directives are rules that help identify pick and put locations for inventory movement.</span></span> <span data-ttu-id="bca5b-136">ตัวอย่างเช่น ในธุรกรรมใบสั่งขาย คำสั่งสถานกำหนดจุดที่จะเบิกสินค้า และจุดที่จะวางสินค้าที่เบิก</span><span class="sxs-lookup"><span data-stu-id="bca5b-136">For example, in a sales order transaction, a location directive determines where the items will be picked, and where the picked items will be put.</span></span> <span data-ttu-id="bca5b-137">คำสั่งสถานที่ประกอบด้วยหัวข้อและรายการที่เกี่ยวข้อง และคุณสร้างหัวข้อและรายการที่เกี่ยวข้องในหน้า **คำสั่งสถานที่**.</span><span class="sxs-lookup"><span data-stu-id="bca5b-137">Location directives consist of a header and associated lines, and you create them on the **Location directives** page.</span></span>
