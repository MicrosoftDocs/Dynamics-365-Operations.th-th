## <a name="create-a-mobile-device-menu-item"></a><span data-ttu-id="e4c95-108">สร้างรายการเมนูบนอุปกรณ์เคลื่อนที่</span><span class="sxs-lookup"><span data-stu-id="e4c95-108">Create a mobile device menu item</span></span>
