## <a name="create-a-mobile-device-menu-item"></a><span data-ttu-id="74a10-108">สร้างรายการเมนูบนอุปกรณ์เคลื่อนที่</span><span class="sxs-lookup"><span data-stu-id="74a10-108">Create a mobile device menu item</span></span>

3. <span data-ttu-id="74a10-111">ในฟิลด์ **ชื่อรายการในเมนู** ให้พิมพ์ค่า</span><span class="sxs-lookup"><span data-stu-id="74a10-111">In the **Menu item name** field, type a value.</span></span> <span data-ttu-id="74a10-112">นี่คือตัวระบุเฉพาะสำหรับรายการเมนูนี้อุปกรณ์เคลื่อนที่ </span><span class="sxs-lookup"><span data-stu-id="74a10-112">This is the unique identifier for this mobile device menu item.</span></span> <span data-ttu-id="74a10-113">ตัวอย่างเช่น คุณสามารถพิมพ์ `My PO registration`</span><span class="sxs-lookup"><span data-stu-id="74a10-113">For example, you could type `My PO registration`.</span></span>
