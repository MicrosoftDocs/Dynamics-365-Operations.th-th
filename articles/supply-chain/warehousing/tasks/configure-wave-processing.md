[!include [banner](../../includes/banner.md)]

5. <span data-ttu-id="de028-116">ในฟิลด์ **ไซต์** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง</span><span class="sxs-lookup"><span data-stu-id="de028-116">In the **Site** field, enter or select a value.</span></span> <span data-ttu-id="de028-117">ถ้าคุณกำลังใช้ USMF คุณสามารถเลือกไซต์ 2 ได้</span><span class="sxs-lookup"><span data-stu-id="de028-117">If you're using USMF, you can select site 2.</span></span>  
6. <span data-ttu-id="de028-118">ในฟิลด์ **คลังสินค้า** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง</span><span class="sxs-lookup"><span data-stu-id="de028-118">In the **Warehouse** field, enter or select a value.</span></span> <span data-ttu-id="de028-119">ถ้าคุณกำลังใช้ USMF คุณสามารถเลือกคลังสินค้า 24 ได้</span><span class="sxs-lookup"><span data-stu-id="de028-119">If you're using USMF, you can select warehouse 24.</span></span>

8. <span data-ttu-id="de028-122">ตั้งค่าตัวเลือก **ประมวลผลเวฟเมื่อนำออกใช้ไปยังคลังสินค้า** เป็น **ใช่**</span><span class="sxs-lookup"><span data-stu-id="de028-122">Set the **Process wave at release to warehouse** option to **Yes**.</span></span> <span data-ttu-id="de028-123">เลือกตัวเลือกนี้เพื่อประมวลผลเวฟโดยอัตโนมัติ และสร้างงานเมื่อบรรทัดถูกนำออกใช้ไปยังคลังสินค้า</span><span class="sxs-lookup"><span data-stu-id="de028-123">Select this option to automatically process the wave and create work when a line is released to the warehouse.</span></span>
