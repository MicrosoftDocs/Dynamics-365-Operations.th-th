# <a name="plan-loads-and-shipments-using-the-load-planning-workbench"></a>วางแผนการบรรทุกโดยใช้เวิร์กเบนช์การวางแผนการบรรทุก

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงวิธีการใช้เวิร์กเบนช์การวางแผนการบรรทุกเพื่อสร้างจำนวนงานในศูนย์การผลิตสำหรับใบสั่งขาย  เป็นข้อกำหนดเบื้องต้น เราจะสร้างใบสั่งขายเป็นอย่างแรก กระบวนงานนี้เป็นส่วนหนึ่งของงานประจำวันสำหรับผู้ประสานงานด้านการขนส่ง ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF


## <a name="create-a-sales-order"></a>สร้างใบสั่งขาย
1. ไปที่บัญชีลูกหนี้ > ใบสั่งซื้อ > ใบสั่งซื้อทั้งหมด
2. คลิก สร้าง
3. ในฟิลด์บัญชีลูกค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. เลือก บัญชี US-004
5. คลิก ตกลง ระบบจะนำเข้าข้อมูลการชำระเงิน
6. ในฟิลด์หมายเลขสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
7. เลือก สินค้า A0001
    * A0001 เปิดใช้งานสำหรับการจัดการการขนส่ง  
8. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
9. ในฟิลด์ ปริมาณ ให้ป้อนตัวเลข
10. ในฟิลด์คลังสินค้า พิมพ์ '24'
    * ในตัวอย่างนี้ เลือกคลังสินค้า 24 คลังสินค้านี้เปิดใช้งานสำหรับการจัดการการขนส่งและการจัดการคลังสินค้าขั้นสูง  
11. คลิก บันทึก
12. ปิดหน้า

## <a name="create-a-new-load"></a>สร้างจำนวนงานในศูนย์การผลิตใหม่
1. ไปที่ การจัดการการขนส่ง > การวางแผน > เวิร์กเบนช์การวางแผนการบรรทุก
2. คลิก แท็บรายการขาย
    * ขณะนี้ คุณจะสร้างจำนวนงานในศูนย์การผลิตสำหรับใบสั่งขายที่คุณเพิ่งสร้างขึ้น  จำนวนงานในศูนย์การผลิตสามารถสร้างตามการจัดหาวัสดุและความต้องการจากใบสั่งซื้อ ใบสั่งโอนย้าย และใบสั่งขาย  
3. บนบานหน้าต่างการดำเนินการ คลิก การจัดหาและความต้องการ
4. คลิก ไปยังจำนวนงานในศูนย์การผลิตใหม่
5. ในฟิลด์รหัสเท็มเพลตจำนวนงานในศูนย์การผลิต คลิก ปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
    * เท็มเพลตจำนวนงานในศูนย์การผลิตกำหนดหน่วยวัดสูงสุดสำหรับน้ำหนักและปริมาตรของจำนวนงานในศูนย์การผลิตทั้งหมด ตัวอย่างเช่น เท็มเพลตจำนวนงานในศูนย์การผลิตอาจแสดงถึงขนาดของตู้บรรจุสินค้าหรือรถบรรทุก  
6. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
7. คลิก ตกลง

## <a name="rate-and-route-the-load"></a>อัตราและกระบวนการผลิตจำนวนงานในศูนย์การผลิต
1. คลิก การจัดอันดับและการกำหนดเส้นทาง
2. คลิก จัดอันดับเวิร์กเบนช์กระบวนการผลิต
3. คลิก จัดอันดับร้านค้า
4. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
5. คลิก มอบหมาย
6. ปิดหน้า
7. ปิดหน้า
