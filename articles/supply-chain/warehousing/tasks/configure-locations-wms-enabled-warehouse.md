# <a name="configure-locations-in-a-wms-enabled-warehouse"></a>ตั้งค่าคอนฟิกของสถานที่ในคลังสินค้า WMS ที่เปิดใช้งาน

[!include [task guide banner](../../includes/task-guide-banner.md)]

คำแนะนำนี้แสดงวิธีการตั้งค่าคอนฟิกการตั้งค่าสถานที่สำหรับการเปิดใช้งานคลังสินค้า WMS ใหม่ (คลังสินค้าที่ใช้กระบวนการจัดการคลังสินค้าขั้นสูง)  โดยทั่วไป กระบวนการนี้จะทำโดยผู้จัดการคลังสินค้า คุณสามารถรับชมคำแนะนำนี้ในข้อมูลสาธิตของบริษัท USMF หรือข้อมูลของคุณเอง เงื่อนไขเบื้องต้นคือ คุณมีอย่างน้อยหนึ่งไซต์ที่ตั้งค่าคอนฟิก


## <a name="create-a-new-warehouse"></a>สร้างคลังสินค้าใหม่
1. ไปที่คลังสินค้า
2. คลิก สร้าง
3. ในฟิลด์คลังสินค้า ให้พิมพ์ค่า
4. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
5. ในฟิลด์ไซต์ ให้พิมพ์ค่า
6. ขยายหรือยุบส่วนคลังสินค้า
7. ตั้งค่าการใช้กระบวนการจัดการคลังสินค้าที่ตัวเลือก YES
    * การตั้งค่านี้ช่วยให้คุณสามารถดำเนินกระบวนการคลังสินค้าขั้นสูง โดยใช้งานในคลังสินค้าและอุปกรณ์เคลื่อนที่  
8. ปิดหน้า

## <a name="define-a-location-format"></a>กำหนดรูปแบบสถานที่
1. ไปที่รูปแบบสถานที่
    * รูปแบบสถานที่มีการตั้งชื่อระบบที่ใช้ในการสร้างชื่อเฉพาะและสอดคล้องกัน สำหรับพิกัดตำแหน่งของช่องเก็บต่างๆที่ใช้ภายในคลังสินค้า  อาจมีประโยชน์ที่จะใช้ตัวแบ่งเป็นส่วนหนึ่งของรูปแบบสถานที่ เพื่อให้การระบุส่วนประกอบของที่ตั้งเช่นหมายเลขที่เก็บง่ายขึ้น ในตัวอย่างนี้เราจะสร้างชื่อด้วย 4 องค์ประกอบ  ตัวอย่างเช่น องค์ประกอบเหล่านี้อาจเป็น ที่เก็บ ชั้นเก็บสินค้า ชั้น และช่องเก็บ  
2. คลิก สร้าง
3. ในฟิลด์รูปแบบสถานที่ ให้พิมพ์ค่า
4. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
5. ในฟิลด์คำอธิบายเซ็กเมนต์ ให้พิมพ์ค่า
    * มีการอธิบายสิ่งที่ส่วนแรกของชื่อตั้งที่แสดงถึง  อาจจะเป็น ตัวอย่างเช่น ที่เก็บ  
6. ในฟิลด์ความยาว ให้ใส่ตัวเลข
    * จำนวนของอักขระที่ส่วนของชื่อสถานที่เก็บนี้ต้องมีจะถูกกำหนด  โปรดทราบว่า ผลรวมของส่วนประกอบทั้งหมดในชื่อ รวมถึงตัวแบ่ง ต้องไม่เกิน 10 อักขระ  
7. ในฟิลด์ตัวแบ่ง ให้พิมพ์ค่า
    * อักขระหรือสัญลักษณ์ที่ใช้ระหว่างส่วนประกอบที่หนึ่งและสองของชื่อจะถูกกำหนด  
8. คลิก สร้าง
9. ในฟิลด์คำอธิบายเซ็กเมนต์ ให้พิมพ์ค่า
10. ในฟิลด์ความยาว ให้ใส่ตัวเลข
11. ในฟิลด์ตัวแบ่ง ให้พิมพ์ค่า
12. คลิก สร้าง
13. ในฟิลด์คำอธิบายเซ็กเมนต์ ให้พิมพ์ค่า
14. ในฟิลด์ความยาว ให้ใส่ตัวเลข
15. ในฟิลด์ตัวแบ่ง ให้พิมพ์ค่า
16. คลิก สร้าง
17. ในฟิลด์คำอธิบายเซ็กเมนต์ ให้พิมพ์ค่า
18. ในฟิลด์ความยาว ให้ใส่ตัวเลข
19. คลิก บันทึก
20. ปิดหน้า

## <a name="define-location-types"></a>กำหนดชนิดสถานที่
1. ไปที่ชนิดสถานที่
    * ชนิดของสถานที่สามารถใช้เป็นตัวเลือกการกรองข้อมูลเพื่อควบคุมกระบวนการจัดการคลังสินค้าต่างๆ  อย่างน้อยที่สุด คุณต้องสร้างขั้นตอนและชนิดสถานที่การจัดส่งขั้นสุดท้ายเพื่อที่จะกำหนดกระบวนการจัดการคลังสินค้าขาออก  
2. คลิก สร้าง
3. ในฟิลด์ชนิดสถานที่ ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. ปิดหน้า

## <a name="define-location-profile"></a>กำหนดโพรไฟล์สถานที่
1. ไปที่โพรไฟล์สถานที่
    * การกำหนดโพรไฟล์สถานที่นั้นมีความสำคัญมาก  สามารถควบคุมกำลังการผลิตของสถานที่ที่เป็นกลุ่มได้ที่นี่ เช่นเดียวกับนโยบายที่เกี่ยวข้องกับสิ่งที่สินค้าคงคลังได้รับจัดเก็บและวิธีการจัดเก็บ ชนิดของสถานที่สามารถใช้เป็นตัวเลือกการกรองข้อมูลเพื่อควบคุมกระบวนการจัดการคลังสินค้าที่แตกต่างกัน อย่างน้อยที่สุด คุณต้องสร้างโพรไฟล์ตำแหน่งผู้ใช้เพื่อเปิดใช้งานกระบวนการจัดการคลังสินค้า  
2. คลิก สร้าง
3. ในฟิลด์รหัสโพรไฟล์สถานที่ ให้พิมพ์ค่า
4. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
5. ในฟิลด์รูปแบบสถานที่ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
6. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
7. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
8. ในฟิลด์ชนิดสถานที่ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
9. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
10. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
11. เลือกหรือยกเลิกการเลือกกล่องกาเครื่องหมายของการอนุญาตสถานะสินค้าคงคลังแบบผสม
    * เปิดใช้งานตัวเลือกนี้ ถ้าคุณต้องการอนุญาตให้มีค่าสถานะของสินค้าคงคลังแบบผสมในสถานที่ที่จะถูกจัดกลุ่มตามโพรไฟล์ตำแหน่ง  
12. เลือกหรือยกเลิกการเลือกกล่องกาเครื่องหมายของการแทนที่กฎสำหรับจำนวนวันของชุดงาน
    * เปิดใช้งานตัวเลือกนี้เพื่อแทนที่กฎสำหรับจำนวนวันของวันหมดอายุของชุดงานสินค้าคงคลังที่อาจแตกต่างกัน เพื่ออนุญาตให้มีการผสมของชุดงานสินค้าคงคลังที่ไม่ปฏิบัติตามกฎนี้  
13. เลือกหรือยกเลิกการเลือกกล่องกาเครื่องหมายของการอนุญาตการตรวจนับตามรอบ
    * เปิดใช้งานตัวเลือกนี้ เพื่ออนุญาตให้มีการดำเนินการการตรวจนับตามรอบในทุกสถานที่ที่จะถูกจัดกลุ่มตามโพรไฟล์ตำแหน่ง  
14. ขยายหรือยุบส่วนมิติ
    * แท็บมิติช่วยให้คุณสามารถกำหนดพารามิเตอร์และวิธีการเปิดใช้งานการคำนวณที่แม่นยำของกำลังการผลิตที่อยู่ภายในแต่ละที่ตั้ง  
15. ปิดหน้า

## <a name="enable-warehouse-management-parameters"></a>เปิดใช้งานพารามิเตอร์การจัดการคลังสินค้า
1. ไปที่พารามิเตอร์การจัดการคลังสินค้า
    * เพื่อให้สามาถดำเนินงานคลังสินค้าได้ ต้องมีการตั้งค่าพารามิเตอร์สำหรับโพรไฟล์ที่ตั้งผู้ใช้ชนิดสถานที่การแบ่งระยะ และชนิดสถานที่จัดส่งขั้นสุดท้าย ทันทีที่กระบวนการขาออกสิ้นสุดที่ชนิดสถานที่จัดส่งขั้นสุดท้ายตามที่คุณกำหนด ธุรกรรมขาออกที่เกี่ยวข้องจะถูกอัพเดตเป็น เบิกสินค้าแล้ว  
2. ขยายหรือยุบส่วนโปรไฟล์สถานที่
3. ในฟิลด์สถานที่ผู้ใช้ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
5. ขยายหรือยุบส่วนชนิดสถานที่
6. ในฟิลด์ชนิดสถานที่การแบ่งระยะ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
7. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
8. ในฟิลด์ชนิดสถานที่จัดส่งสุดท้าย ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
9. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
10. ปิดหน้า

4. ในฟิลด์ชื่อกลุ่มโซน ให้พิมพ์ค่า
5. ปิดหน้า

## <a name="create-locations-manually"></a>สร้างสถานที่ด้วยตนเอง
1. ไปที่สถานที่
    * การสร้างสถานที่ภายในคลังสินค้าสามารถทำได้อย่างง่ายด้วยตนเอง  ชื่อสถานที่เก็บและรหัสโพรไฟล์สถานที่เป็นค่าบังคับ  
2. คลิก สร้าง
3. ในฟิลด์คลังสินค้า ให้พิมพ์ค่า
4. ในฟิลด์ตำแหน่ง ให้พิมพ์ค่า
    * โปรดทราบว่า คุณกำลังสร้างตำแหน่งใหม่ที่นี่ ดังนั้นคุณต้องป้อนชื่อเฉพาะใหม่ แทนที่การเลือกชื่อที่มีอยู่  
5. ในฟิลด์รหัสโพรไฟล์สถานที่ ให้พิมพ์ค่า
6. ปิดหน้า

## <a name="define-pack-size-categories"></a>กำหนดประเภทของขนาดแพ็ค
1. ไปที่ประเภทของขนาดแพ็ค
    * ประเภทของขนาดแพ็คสามารถใช้เพื่อจัดกลุ่มสินค้าที่มีขนาดใกล้เคียงกัน  ในตัวอย่างนี้ จะใช้ประเภทของขนาดแพ็คเพื่อควบคุมกำลังการผลิตที่สถานที่เบิกสินค้าภายในโซนเฉพาะของคลังสินค้า โปรดทราบว่า รหัสประเภทของขนาดแพ็คต้องถูกกำหนดให้กับเอนทิตี้ผลิตภัณฑ์ที่นำออกใช้เพื่อให้สามารถใช้เป็นส่วนหนึ่งของการประมวลผลการเก็บสต็อก  
2. คลิก สร้าง
3. ในฟิลด์รหัสประเภทขนาดของแพ็ค ให้พิมพ์ค่า
4. ในฟิลด์ชื่อประเภทขนาดของแพ็ค ให้พิมพ์ค่า
5. ปิดหน้า

## <a name="define-location-stocking-limits"></a>กำหนดขีดจำกัดการเก็บสต็อกในสถานที่
1. ไปที่ขีดจำกัดการเก็บสต็อกในสถานที่
    * ขีดจำกัดของสถานที่เก็บสินค้าคงคลังช่วยให้มั่นใจได้ว่างานที่ไม่ได้ถูกขอให้สร้าง สินค้าคงคลังถูกจัดเก็บอยู่ในสถานที่ที่ไม่ได้มีกำลังการผลิตอยู่จริงเพื่อให้แน่ใจว่าสถานทีที่สามารถจัดเก็บสินค้าคงคลังได้  
2. คลิก สร้าง
3. ในฟิลด์คลังสินค้า ให้พิมพ์ค่า
4. ในฟิลด์รหัสโพรไฟล์สถานที่ ให้พิมพ์ค่า
5. ในฟิลด์รหัสประเภทขนาดของแพ็ค ให้พิมพ์ค่า
6. ในฟิลด์ ปริมาณ ให้ป้อนตัวเลข
7. คลิก บันทึก
8. ปิดหน้า

## <a name="define-fixed-picking-locations"></a>กำหนดสถานที่คงที่เบิกสินค้าถาวร
1. ไปที่สถานที่คงที่
    * คุณสามารถกำหนดสถานที่ที่จะใช้สำหรับแต่ละผลิตภัณฑ์หรือต่อผลิตภัณฑ์ย่อย  มีความเป็นไปได้ในการสร้างสถานที่คงที่หลายรายการสำหรับผลิตภัณฑ์เดียวกันภายในคลังสินค้าเดียวกัน  
2. คลิก สร้าง
3. ในฟิลด์หมายเลขสินค้า ให้พิมพ์ค่า
4. ในฟิลด์คลังสินค้า ให้พิมพ์ค่า
5. ในฟิลด์ตำแหน่ง ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
6. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
7. ปิดหน้า
