# <a name="system-directed-cluster-picking"></a><span data-ttu-id="2ec1c-103">การเบิกสินค้าคลัสเตอร์ที่สั่งการโดยระบบ</span><span class="sxs-lookup"><span data-stu-id="2ec1c-103">System-directed cluster picking</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="2ec1c-104">การเบิกสินค้าคลัสเตอร์เป็นกระบวนการเบิกสินค้าที่ช่วยให้คุณสามารถเบิกสินค้าสำหรับใบสั่งหลายรายการในเวลาเดียวกัน โดยการจัดคลัสเตอร์ลงในคลัสเตอร์การเบิกสินค้า</span><span class="sxs-lookup"><span data-stu-id="2ec1c-104">Cluster picking is a piece picking process that lets you pick items for multiple orders at the same time by clustering them into pick clusters.</span></span> <span data-ttu-id="2ec1c-105">จากนั้น คุณจะต้องไปที่สถานที่เบิกสินค้าเพียงครั้งเดียวเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="2ec1c-105">You then have to visit the pick location only one time.</span></span> <span data-ttu-id="2ec1c-106">โดยทั่วไป ฟังก์ชันนี้จะใช้กับการเบิกสินค้าตามใบสั่งขนาดเล็ก หรือปริมาณที่น้อยกว่าปริมาณของกรณีและปัญหา</span><span class="sxs-lookup"><span data-stu-id="2ec1c-106">Typically, this functionality is used with small order picking or quantities that are less than case quantities.</span></span>

- <span data-ttu-id="2ec1c-115">**สร้างรหัสคลัสเตอร์** ควบคุมว่ารหัสคลัสเตอร์จะถูกสร้างขึ้นโดยระบบหรือถูกป้อนโดยผู้ใช้</span><span class="sxs-lookup"><span data-stu-id="2ec1c-115">**Generate cluster ID** controls whether the cluster ID will be generated by the system or entered by the user.</span></span>

<span data-ttu-id="2ec1c-117">รายการเมนูของอุปกรณ์เคลื่อนที่ใหม่จะถูกใช้สำหรับการเบิกสินค้าคลัสเตอร์ที่สั่งการโดยระบบ</span><span class="sxs-lookup"><span data-stu-id="2ec1c-117">A new mobile device menu item is used for system-directed cluster picking.</span></span> <span data-ttu-id="2ec1c-118">ต้องระบุ **รหัสโพรไฟล์คลัสเตอร์** สำหรับตัวเลือก **สั่งการโดย** ที่เลือก</span><span class="sxs-lookup"><span data-stu-id="2ec1c-118">The **Cluster profile ID** must be specified for the selected **Directed by** option.</span></span> <span data-ttu-id="2ec1c-119">นอกจากนี้ การสอบถามลำดับงานที่สั่งการโดยระบบสามารถจัดกลุ่มใบสั่งตามเงื่อนไขเฉพาะบริษัทได้</span><span class="sxs-lookup"><span data-stu-id="2ec1c-119">Additionally, the system-directed work sequence queries can group orders based on company-specific criteria.</span></span> <span data-ttu-id="2ec1c-120">ดังนั้น คุณสามารถเพิ่มประสิทธิภาพการกำหนดของใบสั่งงานเพิ่มเติมได้ โดยการระบุเงื่อนไขการเรียงลำดับที่กำหนดเองโดยใช้การสอบถามลำดับงานที่สั่งการโดยระบบ</span><span class="sxs-lookup"><span data-stu-id="2ec1c-120">Therefore, you can further optimize the assignment of work orders by specifying customized sorting criteria using the system-directed work sequence queries.</span></span>

<span data-ttu-id="2ec1c-121">เมื่อมีการเปิดใช้งานการเบิกสินค้าคลัสเตอที่สั่งการโดยระบบ จะมีการแสดงผู้ปฏิบัติงานคลังสินค้าพร้อมกับคลัสเตอร์ที่มีการกำหนดใบสั่งการเบิกสินค้าล่วงหน้าให้กับตำแหน่งของคลัสเตอร์</span><span class="sxs-lookup"><span data-stu-id="2ec1c-121">When system-directed cluster picking is enabled, warehouse workers are presented with a cluster where picking orders have been preassigned to cluster positions.</span></span> <span data-ttu-id="2ec1c-122">ดังนั้น ผู้ปฏิบัติงานสามารถเริ่มต้นเบิกสินค้าสำหรับใบสั่งผลิตหลายรายการโดยการเยี่ยมชมสถานที่เบิกสินค้าเพียงครั้งเดียว</span><span class="sxs-lookup"><span data-stu-id="2ec1c-122">Therefore, workers can start to pick an item for multiple work orders by visiting the pick location only one time.</span></span> <span data-ttu-id="2ec1c-123">กระบวนการเบิกสินค้าสำหรับการเบิกสินค้าคลัสเตอร์ที่สั่งการโดยระบบจะเหมือนกับกระบวนการสำหรับการเบิกสินค้าของคลัสเตอร์ที่สั่งการโดยผู้ใช้</span><span class="sxs-lookup"><span data-stu-id="2ec1c-123">The picking process for system-directed cluster picking is the same as the process for user-directed cluster picking.</span></span>
