# <a name="system-directed-cluster-picking"></a><span data-ttu-id="e4cd5-103">การเบิกสินค้าคลัสเตอร์ที่สั่งการโดยระบบ</span><span class="sxs-lookup"><span data-stu-id="e4cd5-103">System-directed cluster picking</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="e4cd5-104">การเบิกสินค้าคลัสเตอร์เป็นกระบวนการเบิกสินค้าที่ช่วยให้คุณสามารถเบิกสินค้าสำหรับใบสั่งหลายรายการในเวลาเดียวกัน โดยการจัดคลัสเตอร์ลงในคลัสเตอร์การเบิกสินค้า</span><span class="sxs-lookup"><span data-stu-id="e4cd5-104">Cluster picking is a piece picking process that lets you pick items for multiple orders at the same time by clustering them into pick clusters.</span></span> <span data-ttu-id="e4cd5-105">จากนั้น คุณจะต้องไปที่สถานที่เบิกสินค้าเพียงครั้งเดียวเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="e4cd5-105">You then have to visit the pick location only one time.</span></span> <span data-ttu-id="e4cd5-106">โดยทั่วไป ฟังก์ชันนี้จะใช้กับการเบิกสินค้าตามใบสั่งขนาดเล็ก หรือปริมาณที่น้อยกว่าปริมาณของกรณีและปัญหา</span><span class="sxs-lookup"><span data-stu-id="e4cd5-106">Typically, this functionality is used with small order picking or quantities that are less than case quantities.</span></span>

<span data-ttu-id="e4cd5-110">การเบิกสินค้าคลัสเตอร์ที่สั่งการโดยระบบนำเสนอทางเลือกในการสร้างคลัสเตอร์ด้วยตนเอง ที่ซึ่งโปรไฟล์คลัสเตอร์ถูกใช้เพื่อสร้างคลัสเตอร์</span><span class="sxs-lookup"><span data-stu-id="e4cd5-110">System-directed cluster picking offers an alternative to manual cluster building, where a cluster profile is used to create a cluster.</span></span> <span data-ttu-id="e4cd5-111">ต้องกำหนดรายการที่เกี่ยวข้องกับการตั้งค่าหลายรายการในโพรไฟล์คลัสเตอร์ก่อนที่จะใช้:</span><span class="sxs-lookup"><span data-stu-id="e4cd5-111">Several setup-related lines must be defined in the cluster profile before it's used:</span></span>

- <span data-ttu-id="e4cd5-112">**จำนวนของตำแหน่ง** สอดคล้องกับจำนวนของใบสั่งที่จะวางบนคลัสเตอร์</span><span class="sxs-lookup"><span data-stu-id="e4cd5-112">**Number of positions** corresponds to the number of orders that will be put on a cluster.</span></span> <span data-ttu-id="e4cd5-113">ดังนั้น จึงสอดคล้องกับจำนวนของโหลดบรรทุก</span><span class="sxs-lookup"><span data-stu-id="e4cd5-113">Therefore, it corresponds to the number of totes.</span></span>
- <span data-ttu-id="e4cd5-114">**แบ่งคลัสเตอร์** กำหนดว่าคลัสเตอร์ควรจะถูกแบ่งเมื่อใด</span><span class="sxs-lookup"><span data-stu-id="e4cd5-114">**Break cluster** determines when the cluster should be broken.</span></span>
- <span data-ttu-id="e4cd5-115">**สร้างรหัสคลัสเตอร์** ควบคุมว่ารหัสคลัสเตอร์จะถูกสร้างขึ้นโดยระบบหรือถูกป้อนโดยผู้ใช้</span><span class="sxs-lookup"><span data-stu-id="e4cd5-115">**Generate cluster ID** controls whether the cluster ID will be generated by the system or entered by the user.</span></span>
- <span data-ttu-id="e4cd5-116">**ประเภทการยืนยันการเรียงลำดับ** กำหนดว่าจำเป็นต้องมีการยืนยันตำแหน่งหรือไม่</span><span class="sxs-lookup"><span data-stu-id="e4cd5-116">**Sort verification type** determines whether position verification is required.</span></span>

<span data-ttu-id="e4cd5-117">รายการเมนูของอุปกรณ์เคลื่อนที่ใหม่จะถูกใช้สำหรับการเบิกสินค้าคลัสเตอร์ที่สั่งการโดยระบบ</span><span class="sxs-lookup"><span data-stu-id="e4cd5-117">A new mobile device menu item is used for system-directed cluster picking.</span></span> <span data-ttu-id="e4cd5-118">ต้องระบุรหัสโพรไฟล์คลัสเตอร์สำหรับตัวเลือก **สั่งการโดย**</span><span class="sxs-lookup"><span data-stu-id="e4cd5-118">The cluster profile ID must be specified for the **Directed by** option.</span></span> <span data-ttu-id="e4cd5-119">นอกจากนี้ ลำดับการสอบถามที่สั่งการโดยระบบสามารถจัดกลุ่มใบสั่งตามเงื่อนไขเฉพาะของบริษัทได้</span><span class="sxs-lookup"><span data-stu-id="e4cd5-119">Additionally, the system-directed query order can group orders, based on company-specific criteria.</span></span> <span data-ttu-id="e4cd5-120">ดังนั้น คุณสามารถเพิ่มประสิทธิภาพการกำหนดของใบสั่งงานเพิ่มเติมได้โดยการระบุเงื่อนไขการเรียงลำดับที่กำหนดเองบนลำดับการสอบถามที่สั่งการโดยระบบ</span><span class="sxs-lookup"><span data-stu-id="e4cd5-120">Therefore, you can further optimize the assignment of work orders by specifying customized sorting criteria on the system-directed query order.</span></span>

<span data-ttu-id="e4cd5-121">เมื่อทำการเบิกสินค้าคลัสเตอที่สั่งการโดยระบบเสร็จสิ้น จะมีการแสดงผู้ปฏิบัติงานคลังสินค้าพร้อมกับคลัสเตอร์ที่มีการกำหนดใบสั่งการเบิกสินค้าล่วงหน้าให้กับตำแหน่งของคลัสเตอร์</span><span class="sxs-lookup"><span data-stu-id="e4cd5-121">When system-directed cluster picking is done, warehouse workers are presented with a cluster where picking orders have been preassigned to cluster positions.</span></span> <span data-ttu-id="e4cd5-122">ดังนั้น ผู้ปฏิบัติงานสามารถเริ่มต้นเบิกสินค้าสำหรับใบสั่งผลิตหลายรายการโดยการเยี่ยมชมสถานที่เบิกสินค้าเพียงครั้งเดียว</span><span class="sxs-lookup"><span data-stu-id="e4cd5-122">Therefore, workers can start to pick an item for multiple work orders by visiting the pick location only one time.</span></span> <span data-ttu-id="e4cd5-123">กระบวนการเบิกสินค้าสำหรับการเบิกสินค้าคลัสเตอร์ที่สั่งการโดยระบบจะเหมือนกับกระบวนการสำหรับการเบิกสินค้าของคลัสเตอร์ที่สั่งการโดยผู้ใช้</span><span class="sxs-lookup"><span data-stu-id="e4cd5-123">The picking process for system-directed cluster picking is the same as the process for user-directed cluster picking.</span></span>
