# <a name="direct-deliveries"></a><span data-ttu-id="b8987-104">การจัดส่งสินค้าโดยตรง</span><span class="sxs-lookup"><span data-stu-id="b8987-104">Direct deliveries</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="b8987-105">บทความนี้มีข้อมูลเกี่ยวกับการจัดส่งสินค้าโดยตรง</span><span class="sxs-lookup"><span data-stu-id="b8987-105">This article provides information about direct deliveries.</span></span> <span data-ttu-id="b8987-106">การจัดส่งสินค้าโดยตรงคือการจัดส่งที่ส่งโดยตรงจากผู้จัดจำหน่ายให้ลูกค้าของคุณ</span><span class="sxs-lookup"><span data-stu-id="b8987-106">Direct deliveries are deliveries that are sent directly from the vendor to your customer.</span></span>

<span data-ttu-id="b8987-107">จัดส่งสินค้าโดยตรงประหยัดเวลาจัดส่ง และลดต้นทุนที่เกี่ยวข้องกับการถือครองสินค้าคงคลัง เนื่องจากคุณไม่ได้เก็บสินค้าไว้ในคลังสินค้าของคุณก่อนที่คุณจะจัดส่งให้แก่ลูกค้า</span><span class="sxs-lookup"><span data-stu-id="b8987-107">Direct deliveries save delivery time and reduce the costs that are associated with carrying inventory, because you don't hold the products in your warehouse before you ship them to the customer.</span></span>  

<span data-ttu-id="b8987-108">คุณสามารถสร้างการจัดส่งโดยตรงจากหน้า **ใบสั่งขาย**</span><span class="sxs-lookup"><span data-stu-id="b8987-108">You can create direct deliveries from the **Sales order** page.</span></span> <span data-ttu-id="b8987-109">สร้างใบสั่งขาย และรายการใบสั่ง</span><span class="sxs-lookup"><span data-stu-id="b8987-109">First, create a sales order and order lines.</span></span> <span data-ttu-id="b8987-110">จากนั้นบนบานหน้าต่างการดำเนินการ บนแท็บ **ใบสั่งขาย** เลือก **การจัดส่งสินค้าโดยตรง**</span><span class="sxs-lookup"><span data-stu-id="b8987-110">Then, on the Action Pane, on the **Sales order** tab, select **Direct delivery**.</span></span> <span data-ttu-id="b8987-111">ในตอนท้าย ระบุรายการที่ต้องจัดการเป็นการจัดส่งสินค้าโดยตรง</span><span class="sxs-lookup"><span data-stu-id="b8987-111">Finally, specify the lines that must be handled as a direct delivery.</span></span> <span data-ttu-id="b8987-112">ขณะนี้ลิงค์ถูกสร้างระหว่างรายการใบสั่งขายสำหรับการจัดส่งโดยตรง และรายการใบสั่งซื้อที่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="b8987-112">A link is now created between the sales order lines for the direct delivery and the corresponding purchase order lines.</span></span>  

<span data-ttu-id="b8987-113">**หมายเหตุ:** ถ้าบางส่วนของปริมาณที่สั่งได้จัดส่งแล้ว คุณต้องแบ่งปริมาณที่คงเหลือ</span><span class="sxs-lookup"><span data-stu-id="b8987-113">**Note:** If part of the ordered quantity has already been delivered, you must split the remaining quantity.</span></span> <span data-ttu-id="b8987-114">สร้างรายการใหม่สำหรับปริมาณที่ต้องถูกจัดส่งสินค้าโดยตรง และลบปริมาณออกจากปริมาณในรายการเดิม</span><span class="sxs-lookup"><span data-stu-id="b8987-114">Create a new line for the quantity that must be directly delivered, and subtract that quantity from the quantity on the original line.</span></span> <span data-ttu-id="b8987-115">ตัวอย่างเช่น ถ้าปริมาณเดิมคือ 15 และมีการจัดส่งไปห้าแล้ว คุณต้องสร้างรายการใหม่สำหรับปริมาณคงเหลือ 10 จากนั้นลดปริมาณเดิมด้วยจำนวนนั้น</span><span class="sxs-lookup"><span data-stu-id="b8987-115">For example, if the original quantity was 15, and five have been delivered, you must create a new line for the remaining quantity, 10, and then reduce the original quantity by that amount.</span></span>  

<span data-ttu-id="b8987-116">หลังจากที่คุณสร้างการลิงค์จัดส่งสินค้าโดยตรงระหว่างรายการใบสั่งขาย และรายการใบสั่งซื้อ คุณสามารถอัพเดตใบสั่งขายโดยใช้บันทึกการจัดส่ง</span><span class="sxs-lookup"><span <span data-ttu-id="b8987-117">รันอัพเดตบันทึกการจัดส่ง หรืออัพเดตใบแจ้งหนี้จากใบสั่งซื้อ</span><span class="sxs-lookup"><span data-stu-id="b8987-117">Run either a packing slip update or an invoice data-ttu-id="b8987-119">อัพเดตใบแจ้งหนี้ไม่ทำให้ปริมาณในใบสั่งขายเกินกว่าปริมาณที่มีการลงทะเบียนเป็นได้รับ</span><span class="sxs-lookup"><span data-stu-id="b8987-119">An invoice update can't cause the quantity on the sales order to exceed the quantity that has been registered as received.</span></span> <span data-ttu-id="b8987-120">ตัวอย่างเช่น รายการใบสั่งขายมี 10 ชิ้น แต่มีเพียงห้าชิ้นจากรายการใบสั่งขายที่อัพเดตแล้วโดยใช้บันทึกการจัดส่ง</span><span class="sxs-lookup"><span data-stu-id="b8987-120">For example, a sales order line has 10 pieces, but only five pieces from the sales order line have been updated by using a packing slip.</span></span> <span data-ttu-id="b8987-121">ถ้าคุณเลือก **ทั้งหมด** ในรายการ **ปริมาณ** เมื่อคุณอัพเดตใบแจ้งหนี้ของใบสั่งขาย ## <a name="delivery-date"></a><span data-ttu-id="b8987-123">วันที่ส่ง</span><span class="sxs-lookup"><span data-stu-id="b8987-123">Delivery date</span></span>
