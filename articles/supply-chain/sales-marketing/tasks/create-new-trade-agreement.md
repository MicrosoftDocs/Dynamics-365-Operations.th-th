# <a name="create-a-new-trade-agreement"></a>สร้างข้อตกลงทางการค้าใหม่

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

ขั้นตอนนี้แสดงวิธีการสร้างข้อตกลงทางการค้าที่คุณลงทะเบียนราคาขายใหม่ของผลิตภัณฑ์ซึ่งคุณได้ตกลงกับลูกค้าแต่ละราย  คุณสามารถเรียกใช้ขั้นตอนนี้ ในบริษัทข้อมูลสาธิต USMF หรือข้อมูลของคุณเอง ถ้าคุณกำลังใช้ข้อมูลของคุณเอง ก่อนที่คุณเริ่มทำตามคำแนะนำนี้ คุณจำเป็นต้องแน่ใจว่าชื่อสมุดรายวันข้อตกลงทางการค้าปรากฎอยู่ที่ที่ความสัมพันธ์เริ่มต้นถูกตั้งค่าเป็น "ราคา (ขาย)"


## <a name="create-and-post-a-new-trade-agreement-journal"></a>สร้างและลงรายการสมุดรายวันข้อตกลงทางการค้าใหม่
1. ไปยัง การขายและการตลาด > ราคาและส่วนลด > สมุดรายวันข้อตกลงทางการค้า
2. คลิก สร้าง
3. ในฟิลด์ชื่อ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
5. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
6. คลิก รายการ
7. ในฟิลด์รหัสบัญชี เลือก 'ตาราง' 
    * ในตัวอย่างนี้ คุณกำลังอัพเดตราคาสำหรับลูกค้าเฉพาะราย ซึ่งหมายความว่า คุณต้องเลือกตาราง  ถ้าคุณกำลังอัพเดทราคาของรายการผลิตภัณฑ์ คุณจะต้องเลือกทั้งหมด เพื่อให้ราคาใหม่มีผลบังคับใช้สำหรับลูกค้าทั้งหมด ถ้าคุณได้รับความแตกต่างของราคาระหว่างเซ็กเมนต์ลูกค้าอื่น คุณจะต้องเลือกกลุ่ม เมื่อต้องการเลือกกลุ่ม คุณต้องตั้งค่าราคาของกลุ่มลูกค้า  
8. ในฟิลด์การเลือกบัญชี ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
9. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
10. ในฟิลด์รหัสสินค้า ให้เลือก 'ตาราง'
    * เมื่อคุณป้อนข้อตกลงทางการค้าของชนิด 'ราคา (ขาย)' คุณต้องเลือกเฉพาะ 'ตาราง' ในฟิลด์รหัสสินค้า  ทั้งนี้เนื่องจากราคาเป็นค่าจำนวนเต็ม และต้องไม่เหมือนกันสำหรับผลิตภัณฑ์ทั้งหมดหรือกลุ่มของผลิตภัณฑ์  
11. ในฟิลด์ความสัมพันธ์ของสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
12. ในรายการ เลือกผลิตภัณฑ์ที่คุณต้องการรวมไว้ในข้อตกลง
    * ทำบันทึกของผลิตภัณฑ์ที่คุณเลือกไว้  
13. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
14. ในฟิลด์เริ่มต้น ป้อนปริมาณต่ำสุด
    * ถ้าลูกค้าต้องการสั่งสินค้าในปริมาณต่ำสุดก่อนที่จะสามารถมีสิทธิได้รับราคาใหม่ คุณจำเป็นต้องระบุปริมาณที่นี่  
    * ป้อนค่าในฟิลด์ To เพื่อระบุปริมาณสูงสุดสำหรับราคาตามข้อตกลงที่จะไม่ถูกบังคับใช้  หากคุณเสนอราคาและส่วนลดที่ขึ้นอยู่กับหลายตัวแบ่งปริมาณ แล้วระบุแต่ละวงเล็บของปริมาณเป็นคู่ของปริมาณต่ำสุดและสูงสุดในฟิลด์ 'เริ่มต้น' และ 'สิ้นสุด' ตามลำดับ  
15. ในจำนวนในฟิลด์สกุลเงิน ป้อนราคา
16. ในฟิลด์วันเริ่มต้น ป้อนวันเริ่มต้นซึ่งข้อตกลงนี้จะมีผลบังคับใช้
17. คลิก บันทึก
18. คลิก ตรวจสอบความถูกต้อง
19. คลิกตรวจสอบความถูกต้องรายการที่เลือก
20. คลิก ตกลง
21. คลิก ลงรายการบัญชี
22. คลิก ตกลง

## <a name="view-trade-agreements-for-a-product"></a>ดูข้อตกลงทางการค้าที่สำหรับผลิตภัณฑ์
1. ไปที่การจัดการข้อมูลผลิตภัณฑ์ > ผลิตภัณฑ์ > ผลิตภัณฑ์ที่นำออกใช้ 
2. ในรายการ ค้นหาและเลือกผลิตภัณฑ์ที่คุณอัพเดตราคา
3. ในบานหน้าต่างการดำเนินการ คลิกขาย
4. คลิกดูข้อตกลงทางการค้า
    * ตรวจทานรายละเอียดของราคาที่ตกลงทางการค้าที่คุณเพิ่งสร้าง    
5. ปิดหน้า
