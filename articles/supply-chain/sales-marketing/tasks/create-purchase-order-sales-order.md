# <a name="create-a-purchase-order-from-a-sales-order"></a>การสร้างใบสั่งซื้อจากใบสั่งขาย

[!include [banner](../../includes/banner.md)]

กระบวนงานนี้แสดงวิธีการสร้างใบสั่งซื้อที่ขึ้นอยู่กับใบสั่งขาย  ปริมาณของผลิตภัณฑ์ในใบสั่งซื้อได้ถูกกำหนดไว้แล้วเพื่อตอบสนองความต้องการของใบสั่งขายเริ่มต้น  การดำเนินการตามความต้องการขายด้วยวิธีนี้เป็นทางเลือกหนึ่งเพื่อความครอบคลุมและการทำงานได้เต็มประสิทธิภาพของการกระจายการวางแผนความต้องการ  คุณสามารถเรียกใช้ขั้นตอนนี้ ในบริษัทข้อมูลสาธิต USMF หรือข้อมูลของคุณเอง


## <a name="create-a-purchase-order-from-a-sales-order"></a>สร้างใบสั่งซื้อจากใบสั่งขาย
1. ไปยัง **บานหน้าต่างนำทาง > โมดูล > การขายและการตลาด > ใบสั่งขาย > ใบสั่งขายทั้งหมด**
2. คลิก **สร้าง**
3. ในฟิลด์ **บัญชีลูกค้า** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
5. คลิก **ตกลง**
6. ในฟิลด์ **หมายเลขสินค้า** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
7. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก D0001 ได้  
8. ในฟิลด์ **ปริมาณ** ให้ป้อนตัวเลข
9. คลิก **เพิ่มรายการ**
10. ในฟิลด์ **หมายเลขสินค้า** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
11. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก T0020 ได้  
12. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
13. ในฟิลด์ **ปริมาณ** ให้ป้อนตัวเลข
14. คลิก **บันทึก**
15. ใน **บานหน้าต่างการดำเนินการ** ให้คลิก **ใบสั่งขาย**
16. ให้คลิก **ใบสั่งซื้อ** หน้า **สร้างใบสั่งซื้อ** แสดงรายการใบสั่งขายที่เปิดไว้ที่ถูกคัดลอกมาจากใบสั่งขาย คุณสามารถทบทวนรายละเอียดใบสั่งได้ และถ้าจำเป็น คุณสามารถปรับเปลี่ยนรายละเอียดได้เช่น ปริมาณการซื้อและเงื่อนไขการกำหนดราคา ก่อนที่คุณจะสร้างการจัดซื้อ 
17. ให้เลือก **รวมตัวเลือกทั้งหมด**
    - ถ้าคุณต้องการสร้างใบสั่งซื้อสำหรับเฉพาะชุดย่อยของรายการใบสั่งขาย ให้เลือกรายการเหล่านี้ทีละรายการ  
    - ฟิลด์ **บัญชีผู้จัดจำหน่าย** อาจมีการเติมหรืออาจไม่มีการเติมข้อมูลด้วยหมายเลขผู้จัดจำหน่ายอยู่แล้ว ถ้าผู้จัดจำหน่ายเริ่มต้นได้มีการตั้งค่าไว้สำหรับผลิตภัณฑ์ (ในความครอบคลุมสินค้าที่เกี่ยวข้อง) แล้วผู้จัดจำหน่ายนี้จะถูกคัดลอกไปยังรายการ  มิฉะนั้น คุณต้องป้อนผู้จัดจำหน่ายด้วยตนเอง  ในคำแนะนำนี้ ไม่คำนึงว่าฟิลด์ **บัญชีผู้จัดจำหน่าย** จะมีค่าอยู่แล้วหรือไม่ก็ตาม ขั้นตอนต่อไปนี้ จะแนะนำให้คุณเลือกผู้จัดจำหน่ายใหม่ที่แตกต่างกันไปสำหรับในแต่ละรายการ  
18. ในฟิลด์ **บัญชีผู้จัดจำหน่าย** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
19. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
20. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
21. เลือกรายการใบสั่งที่สอง
22. ในฟิลด์ **บัญชีผู้จัดจำหน่าย** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
23. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
24. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
25. คลิก **ตรวจสอบความถูกต้อง**
26. คลิก **ตกลง** ข้อความจะแจ้งให้คุณทราบว่ามีการสร้างใบสั่งซื้อหนึ่งรายการหรือมากกว่า  ระบบจะสร้างใบสั่งซื้อแยกต่างหากสำหรับแต่ละผู้จัดจำหน่ายที่คุณระบุไว้สำหรับรายการใบสั่งขาย ซึ่งหมายความว่า ถ้ามีรายการใบสั่งขายหลายรายการการจัดส่งถูกจัดส่งโดยผู้จัดจำหน่ายเดียวกัน ใบสั่งซื้อใบเดียวที่มีหลายรายการจะถูกสร้าง  

## <a name="review-purchase-orders-created-from-sales-orders"></a>ตรวทานใบสั่งซื้อที่มีการสร้างขึ้นจากใบสั่งขาย
1. บน **หน้าต่างการดำเนินการ** คลิก **ทั่วไป**
2. ให้คลิก **ใบสั่งที่เกี่ยวข้อง** หน้า **ใบสั่งที่เกี่ยวข้อง** แสดงใบสั่งทั้งหมดที่ถูกสร้างขึ้นจากใบสั่งขาย ในตัวอย่างนี้ มีใบสั่งซื้อสองรายการถูกสร้างขึ้นสำหรับสองผู้จัดจำหน่ายที่แตกต่างกันตามลำดับ 
3. ให้คลิกเพื่อติดตามลิงก์ในฟิลด์ **ใบสั่งซื้อ** รายการใบสั่งซื้อแต่ละรายการจะสัมพันธ์กับรายการใบสั่งขายที่นำไปสู่การซื้อ  ความสัมพันธ์กับใบสั่งขายมีการระบุบน **แท็บผลิตภัณฑ์** ในแท็บดวน **รายละเอียดของรายการ** ใน **ประเภทการอ้างอิง** **หมายเลขอ้างอิง** และฟิลด์ **ล็อตการอ้างอิง**  
4. ขยายหรือยุบส่วน **รายละเอียดของรายการ**
5. ให้คลิกแท็บ **ผลิตภัณฑ์**
    - **ล็อตอ้างอิง** ช่วยให้มั่นใจว่าจะมีการคิดต้นทุนจากการซื้อปัจจุบันกับใบสั่งขายที่แนบมา  
    - คุณสามารถไปที่ใบสั่งขายเริ่มต้น โดยการเปิดลิงค์ในฟิลด์ **หมายเลขอ้างอิง**
