# <a name="create-a-purchase-order-from-a-sales-order"></a>การสร้างใบสั่งซื้อจากใบสั่งขาย

[!include [task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงวิธีการสร้างใบสั่งซื้อที่ขึ้นอยู่กับใบสั่งขาย  ปริมาณของผลิตภัณฑ์ในใบสั่งซื้อได้ถูกกำหนดไว้แล้วเพื่อตอบสนองความต้องการของใบสั่งขายเริ่มต้น  การดำเนินการตามความต้องการขายด้วยวิธีนี้เป็นทางเลือกหนึ่งเพื่อความครอบคลุมและการทำงานได้เต็มประสิทธิภาพของการกระจายการวางแผนความต้องการ  คุณสามารถเรียกใช้ขั้นตอนนี้ ในบริษัทข้อมูลสาธิต USMF หรือข้อมูลของคุณเอง


## <a name="create-a-purchase-order-from-a-sales-order"></a>การสร้างใบสั่งซื้อจากใบสั่งขาย
1. ไปยัง การขายและการตลาด > ใบสั่งขาย > ใบสั่งขายทั้งหมด
2. คลิก สร้าง
3. ในฟิลด์บัญชีลูกค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
5. คลิก ตกลง ระบบจะนำเข้าข้อมูลการชำระเงิน
6. ในฟิลด์หมายเลขสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
7. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก D0001 ได้  
8. ในฟิลด์ ปริมาณ ให้ป้อนตัวเลข
9. คลิก เพิ่มรายการ
10. ในฟิลด์หมายเลขสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
11. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * ถ้าคุณกำลังใช้ USMF คุณสามารถเลือก T0020 ได้  
12. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
13. ในฟิลด์ ปริมาณ ให้ป้อนตัวเลข
14. คลิกบันทึก
15. ในบานหน้าต่างการดำเนินการ ให้คลิก ใบสั่งขาย
16. คลิกใบสั่งซื้อ
    * หน้าการสร้างใบสั่งซื้อแสดงรายการใบสั่งขายที่เปิดค้างไว้ทั้งหมดตามที่ได้คัดลอกมาจากใบสั่งขาย  คุณสามารถทบทวนรายละเอียดใบสั่งได้ และถ้าจำเป็น คุณสามารถปรับเปลี่ยนรายละเอียดได้เช่น ปริมาณการซื้อและเงื่อนไขการกำหนดราคา ก่อนที่คุณจะสร้างการจัดซื้อ  
17. เลือกตัวเลือกการรวมทั้งหมด
    * ถ้าคุณต้องการสร้างใบสั่งซื้อสำหรับเฉพาะชุดย่อยของรายการใบสั่งขาย ให้เลือกรายการเหล่านี้ทีละรายการ  
    * ฟิลด์บัญชีผู้จัดจำหน่ายอาจหรืออาจจะไม่ได้ถูกเติมด้วยหมายเลขผู้จัดจำหน่าย  ถ้าผู้จัดจำหน่ายเริ่มต้นได้มีการตั้งค่าไว้สำหรับผลิตภัณฑ์ (ในความครอบคลุมสินค้าที่เกี่ยวข้อง) แล้วผู้จัดจำหน่ายนี้จะถูกคัดลอกไปยังรายการ  มิฉะนั้น คุณต้องป้อนผู้จัดจำหน่ายด้วยตนเอง  ในคู่มือนี้ ไม่สนใจว่าฟิลด์บัญชีผู้จัดจำหน่ายมีค่าบรรจุอยู่แล้วหรือไม่  ขั้นตอนต่อไปนี้จะให้คุณเลือกผู้จัดจำหน่ายใหม่ซึ่งแตกต่างกันในแต่ละรายการ  
18. ในฟิลด์บัญชีผู้จัดจำหน่าย ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
19. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
20. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
21. เลือกรายการใบสั่งที่สอง
22. ในฟิลด์บัญชีผู้จัดจำหน่าย ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
23. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
24. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
25. คลิก ตรวจสอบความถูกต้อง
26. คลิก ตกลง
    * ข้อความจะแจ้งให้คุณทราบว่ามีการสร้างใบสั่งซื้อหนึ่งรายการหรือมากกว่า  ระบบจะสร้างใบสั่งซื้อแยกต่างหากสำหรับแต่ละผู้จัดจำหน่ายที่คุณระบุไว้สำหรับรายการใบสั่งขาย ซึ่งหมายความว่า ถ้ามีรายการใบสั่งขายหลายรายการการจัดส่งถูกจัดส่งโดยผู้จัดจำหน่ายเดียวกัน ใบสั่งซื้อใบเดียวที่มีหลายรายการจะถูกสร้าง  

## <a name="review-purchase-orders-created-from-sales-orders"></a>ตรวทานใบสั่งซื้อที่มีการสร้างขึ้นจากใบสั่งขาย
1. ในบานหน้าต่างการดำเนินการ ให้คลิก ทั่วไป
2. คลิกใบสั่งที่เกี่ยวข้องกับ
    * หน้าใบสั่งที่เกี่ยวข้องแสดงรายการทั้งหมดที่สร้างขึ้นจากใบสั่งขาย  ในตัวอย่างนี้ มีใบสั่งซื้อสองรายการถูกสร้างขึ้นสำหรับสองผู้จัดจำหน่ายที่แตกต่างกันตามลำดับ  
3. คลิกเพื่อติดตามลิงค์ในฟิลด์ใบสั่งซื้อ
    * รายการใบสั่งซื้อแต่ละรายการจะสัมพันธ์กับรายการใบสั่งขายที่นำไปสู่การซื้อ  ความสัมพันธ์กับใบสั่งขายมีการระบุไว้บนแท็บผลิตภัณฑ์ในแท็บด่วนรายละเอียดรายการ ในฟิลด์ชนิดการอ้างอิง หมายเลขอ้างอิง และการอ้างอิงล็อต  
4. ขยายหรือยุบส่วนรายละเอียดของรายการ
5. คลิกแท็บผลิตภัณฑ์
    * ล็อตอ้างอิงยืนช่วยให้มั่นใจว่าจะมีการคิดต้นทุนจากการซื้อปัจจุบันกับใบสั่งขายที่แนบ  
    * คุณสามารถไปที่ใบสั่งขายเริ่มต้น โดยการเปิดลิงค์ในฟิลด์หมายเลขอ้างอิง
