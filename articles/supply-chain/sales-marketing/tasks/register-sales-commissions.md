# <a name="register-sales-commissions"></a>ลงทะเบียนค่าคอมมิชชันการขาย

[!include [task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงวิธีการคำนวณและการลงทะเบียนค่าคอมมิชชันการขาย  คุณสามารถเรียกใช้ขั้นตอนนี้ ในบริษัทข้อมูลสาธิต USMF หรือข้อมูลของคุณเอง ก่อนที่จะเริ่มคู่มือนี้ เรียกใช้คุ่มือที่เรียกว่า "ตั้งค่ากฎค่าคอมมิชชันการขาย" เพื่อให้แน่ใจว่า คุณได้ตั้งค่าการคำนวณค่าคอมมิชชันที่จำเป็นทั้งหมดแล้ว 

จดบันทึกหมายเลขลูกค้าและสินค้าที่คุณได้เลือกไว้สำหรับกระบวนการค่าคอมมิชชัน และใช้เมื่อถูกถามเพื่อสร้างใบสั่งขายในคู่มือนี้


## <a name="invoice-a-sales-order-that-qualifies-a-salesperson-for-a-commission"></a>ออกใบแจ้งหนี้ใบสั่งขายที่ทำให้พนักงานขายมีสิทธิ์ได้รับค่าคอมมิชชัน
1. ไปยัง การขายและการตลาด > ใบสั่งขาย > ใบสั่งขายทั้งหมด
2. คลิก สร้าง
3. ในฟิลด์บัญชีลูกค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
5. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
6. คลิก ตกลง
7. ในบานหน้าต่างการดำเนินการ คลิกที่ตัวเลือก
8. คลิกเปลี่ยนมุมมอง
9. คลิกมุมมองหัวข้อ 
10. ขยายส่วนการตั้งค่า
    * ค่าในฟิลด์กลุ่มการขายแสดงถึงกลุ่มที่มีการกำหนดพนักงานขายหนึ่งคนหรือมากกว่าให้  บุคลากรในกลุ่มจะเป็นคนที่จะได้รับค่าคอมมิชชันเมื่อใบสั่งออกใบแจ้งหนี้ ตามอัตราที่กำหนดไว้ล่วงหน้าและการกระจายสินค้า   ค่าจะถูกคัดลอกจากบัตรลูกค้า แต่คุณสามารถเปลี่ยนถ้าคุณต้องการ  กลุ่มการขายจะได้รับการคัดลอกไปยังรายการใบสั่งขาย คุณสามารถเปลี่ยนได้เพื่อให้แตกต่างจากสิ่งที่อยู่ในส่วนหัวและ/หรือระหว่างรายการ  
    * ค่าในฟิลด์กลุ่มค่าคอมมิชชันแสดงถึง กลุ่มที่คุณสร้างสำหรับลูกค้าอย่างน้อยหนึ่งคนพร้อมวัตถุประสงค์การติดตามค่าคอมมิชชัน    ค่าจะถูกคัดลอกจากบัตรลูกค้า แต่คุณสามารถเปลี่ยนแปลงได้เมื่อต้องการ    
11. ในบานหน้าต่างการดำเนินการ คลิกที่ตัวเลือก
12. คลิกเปลี่ยนมุมมอง
13. คลิกมุมมองรายการ
14. ในฟิลด์หมายเลขสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
15. ในรายการ ให้เลือกสินค้าที่คุณได้ตั้งค่าสำหรับค่าคอมมิชชัน 
16. ในฟิลด์ ปริมาณ ให้ป้อนตัวเลข
    * จดบันทึกยอดเงินสุทธิของรายการ  ซึ่งจะแสดงการรายได้จากการขาย ในตัวอย่างนี้เป็นพื้นฐานสำหรับการคำนวณค่าคอมมิชชัน  
17. คลิก บันทึก
18. ในบานหน้าต่างการดำเนินการ คลิก ใบแจ้งหนี้
19. คลิก ใบแจ้งหนี้
20. ขยายส่วน พารามิเตอร์
21. ในฟิลด์ ปริมาณ ให้เลือก 'ทั้งหมด'
22. เลือก ใช่ ในฟิลด์การลงรายการบัญชี
23. คลิก ตกลง
24. คลิก ตกลง
    * อาจใช้เวลาสักครู่เพื่อลงรายบัญชีธุรกรรม  อนุญาตให้การประมวลผลเสร็จสมบูรณ์ และห้ามปิดหน้า  

## <a name="review-the-registered-sales-commissions"></a>ตรวจทานค่าคอมมิชชันการขายที่ลงทะเบียนแล้ว
1. ในบานหน้าต่างการดำเนินการ คลิก ใบแจ้งหนี้
2. คลิก ใบแจ้งหนี้
3. ในบานหน้าต่างการดำเนินการ คลิก ใบแจ้งหนี้
4. คลิกธุรกรรมค่าคอมมิชชัน
    * แท็บภาพรวมแสดงรายการที่แสดงถึงยอดค่าคอมมิชชันที่ชำระแก่พนักงานขายที่เกี่ยวข้องกับใบสั่งขายที่ออกใบแจ้งหนี้แล้ว  ลองตรวจทานรายละเอียด     
    * ถ้าคุณใช้คู่มือ "ตั้งค่ากฎค่าคอมมิชชันการขาย" เพื่อตั้งค่ากลุ่มการขายที่มีค่าคอมมิชชัน จะมีพนักงานขายสองคนจะได้รับค่าคอมมิชชันการขาย และค่าคอมมิชชันจะถูกแบ่งเท่าๆกันระหว่างสองคน  
    * ในตัวอย่างนี้ มีการคำนวณยอดรวมของค่าคอมมิชชันเป็นเปอร์เซ็นต์ของรายได้การขาย (ยอดเงินสุทธิของรายการใบสั่ง)   
5. ปิดหน้า
6. คลิกใบสำคัญ
    * คุณสามารถตรวจทานธุรกรรมใบสำคัญสำหรับยอดเงินค่าคอมมิชชัน ที่ได้ลงรายการบัญชีค่าใช้จ่ายค่าคอมมิชชันที่กำหนดไว้ล่วงหน้า และบัญชีเจ้าหนี้ค่าคอมมิชชัน
