# <a name="turn-on-sensor-data-intelligence-for-your-system"></a>เปิดระบบข้อมูลอัจฉริยะจากเซนเซอร์สำหรับระบบของคุณ

[!include [banner](../includes/banner.md)]
[!INCLUDE [preview-banner](../includes/preview-banner.md)]

## <a name="video-instructions"></a>คำแนะนำแบบวิดีโอ

วิดีโอต่อไปนี้แสดงวิธีการเปิดคุณลักษณะ Sensor Data Intelligence และ [ปรับใช้ทรัพยากร Azure ที่จำเป็น](sdi-deploy-iot-solution-on-azure.md) ส่วนอื่นๆ ในบทความนี้จะให้คําแนะนําเดียวกันในรูปแบบข้อความ

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE58g3I]

## <a name="procedure"></a>กระบวนงาน

ก่อนที่คุณจะสามารถใช้ระบบข้อมูลอัจฉริยะจากเซนเซอร์ได้ คุณต้องเปิดใช้งานในระบบของคุณ

1. ไปที่ **การดูแลระบบ \> พื้นที่ทำงาน \> การจัดการคุณลักษณะ** (สำหรับข้อมูลเพิ่มเติม ดูที่ [ภาพรวมการจัดการคุณลักษณะ](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md))
1. บนแท็บ **ทั้งหมด** ให้ใช้ฟิลด์ **ตัวกรอง** เพื่อค้นหาคุณลักษณะที่ชื่อ *ระบบข้อมูลอัจฉริยะจากเซนเซอร์*
1. ถ้าคุณลักษณะ *ระบบข้อมูลอัจฉริยะจากเซนเซอร์* เปิดใช้งานอยู่บนระบบของคุณ ให้เลือกคุณลักษณะนั้นในรายการ แล้วเลือก **ปิดใช้งาน** เพื่อปิดใช้งาน คุณไม่สามารถใช้คุณลักษณะเวอร์ชันที่เก่ากว่านี้ร่วมกับเวอร์ชันแสดงตัวอย่างใหม่ที่คุณจะเปิดใช้งานในขั้นตอนถัดไปได้
1. ให้ใช้ฟิลด์ **ตัวกรอง** เพื่อค้นหาคุณลักษณะที่ชื่อ *(ตัวอย่าง) ระบบข้อมูลอัจฉริยะจากเซนเซอร์*
1. เลือกคุณลักษณะในรายการ แล้วเลือก **เปิดใช้งาน** เพื่อเปิดใช้งาน
