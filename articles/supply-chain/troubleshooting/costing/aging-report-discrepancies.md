# <a name="inventory-aging-report-discrepancies"></a>ความขัดแย้งรายงานอายุหนี้ของสินค้าคงคลัง

## <a name="symptoms"></a>อาการ

**รายงานอายุหนี้สินค้าคงคลัง** จะแสดงค่าที่แตกต่างกันเมื่อดูที่มิติการจัดเก็บที่แตกต่างกัน (เช่น ไซต์หรือคลังสินค้า)

## <a name="resolution"></a>การแก้ปัญหา

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับตรรกะการรายงาน ให้ดูที่ [ตัวอย่างและตรรกะของรายงานการแยกอายุหนี้สินค้าคงคลัง](/dynamics365/supply-chain/cost-management/inventory-aging-report.md)
