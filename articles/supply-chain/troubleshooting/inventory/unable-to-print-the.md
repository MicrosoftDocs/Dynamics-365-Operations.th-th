# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>รายงานรายการเบิกสินค้าไม่แสดงบันทึกใหม่ที่แนบกับสมุดรายวัน

หมายเลข KB: 4611660

## <a name="symptoms"></a>อาการ

รายงานรายการเบิกสินค้าไม่แสดงบันทึกใหม่ที่แนบกับสมุดรายวันรายการเบิกสินค้า

## <a name="resolution"></a>ความละเอียด

รายงานรายการเบิกสินค้าจะแสดงเฉพาะบันทึกที่แนบบนบรรทัดใบสั่งขายและส่วนหน้าของใบสั่งขายเท่านั้น ซึ่งจะไม่แสดงบันทึกใหม่ใดๆ ที่เพิ่มในสมุดรายวันใบแจ้งหนี้
