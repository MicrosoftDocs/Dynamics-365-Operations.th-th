# <a name="system-administrators-cant-clear-order-holds-because-they-arent-authorized"></a>ผู้ดูแลระบบไม่สามารถล้างข้อมูลการระงับใบสั่งเนื่องจากไม่ได้รับอนุญาต

หมายเลข KB: 4614642

## <a name="symptoms"></a>อาการ

ผู้ดูแลระบบไม่สามารถล้างข้อมูลการระงับใบสั่งขาย เว้นแต่จะมีบทบาทเฉพาะที่ถูกมอบหมายในรหัสการระงับใบสั่ง

## <a name="resolution"></a>ความละเอียด

การเข้าถึงการดําเนินงานบางอย่าง เช่น การล้างการระงับใบสั่งขาย -จะถูกควบคุมโดยการตั้งค่านโยบายธุรกิจ โดยทั่วไป ผู้ดูแลระบบไม่ได้รับอนุญาตให้ดําเนินงานชนิดนี้ 

บ่อยครั้ง การเข้าถึงเพื่อปฏิบัติงานเฉพาะจะควบคุมโดยนโยบายธุรกิจ และเฉพาะบุคคลที่ระบุในองค์กรเท่านั้นที่ได้รับการอนุมัติให้ปฏิบัติงานนั้น ตัวอย่างการอนุมัติซึ่งเป็นผลมาจากการอนุมัติลำดับงานและงานเฉพาะที่เป็นผลลัพธ์ของการตั้งค่าคอนฟิกลำดับงาน

แม้ว่าไม่มีลำดับงานไม่สัมพันธ์กับการระงับใบสั่ง หลักการนี้จะคล้ายกัน บทบาทที่เกี่ยวข้องระบุกลุ่มบุคคลเฉพาะในองค์กรที่มีสิทธิในการล้างการระงับใบสั่ง ไม่ควรให้สิทธินี้แก่ผู้ดูแลระบบทั้งหมด เนื่องจากวิธีการดังกล่าวละเมิดนโยบายทางธุรกิจที่กําหนด
