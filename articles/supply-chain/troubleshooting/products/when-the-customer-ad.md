# <a name="purchase-orders-dont-show-translated-product-text"></a>ใบสั่งซื้อจะไม่แสดงข้อความผลิตภัณฑ์ที่แปล

หมายเลข KB: 4612098

## <a name="symptoms"></a>อาการ

คุณได้กําหนดการแปลข้อความเป็นชื่อผลิตภัณฑ์แล้ว อย่างไรก็ตาม ชื่อผลิตภัณฑ์จะแสดงในภาษาดั้งเดิมเสมอ โดยไม่พิจารณาการตั้งค่าภาษาในแอป

## <a name="resolution"></a>ความละเอียด

การแปลผลิตภัณฑ์จะแสดงขึ้นบนเอกสารที่เชื่อมต่อกับภายนอกเท่านั้น ไม่แสดงแก่ผู้ใช้ภายใน สำหรับข้อมูลเพิ่มเติม ให้ดูที่ [FAQ เกี่ยวกับการแปลที่เกี่ยวข้องกับผลิตภัณฑ์](../../pim/translations-product-related-information.md#where-can-i-view-the-translated-information)
