# <a name="the-system-cant-find-a-planned-order-during-operations-on-multiple-orders"></a>ระบบไม่พบแผนการใบสั่งในระหว่างการดําเนินงานในใบสั่งหลายใบ

รหัสข้อผิดพลาด: SYS24774

## <a name="symptoms"></a>อาการ

คุณจะได้รับข้อความแสดงข้อผิดพลาดต่อไปนี้ เมื่อคุณพยายามดําเนินงานในแผนการใบสั่งหลายแผนพร้อมกัน และใบสั่งอย่างน้อยหนึ่งใบเป็นของรหัสสินค้าเดียวกัน ตัวอย่างเช่น ข้อผิดพลาดอาจเกิดขึ้นเมื่อมีการยืนยันใบสั่งหรือชนิดใบสั่งมีการเปลี่ยนแปลง

> ไม่มีแผนการใบสั่ง %1 อยู่

## <a name="cause"></a>สาเหตุ

เมื่อระบบยืนยันหรือเปลี่ยนชนิดของใบสั่ง จะต้องพิจารณาแผนการใบสั่งที่มีอยู่ของสินค้าอีกครั้ง เพื่อให้แน่ใจว่าแผนการการจัดหาวัสดุตรงกับความต้องการและการจัดหาวัสดุที่มีอยู่ เนื่องจากส่วนหนึ่งของกระบวนการนี้ ระบบจะสร้างแผนการใบสั่งใหม่ของสินค้า ดังนั้น แผนการใบสั่งที่สองที่ระบบคาดหวังให้มีการประมวลผลไม่มีอยู่อีกต่อไป

## <a name="workaround"></a>วิธีการแก้ไข

อนุมัติใบสั่งก่อนที่คุณจะประมวลผลใบสั่ง ด้วยวิธีนี้ ใบสั่งจะไม่ถูกลบเมื่อระบบประมวลผลใบสั่งแรกของสินค้า
