# <a name="not-enough-capacity-could-be-found"></a>พบกำลังการผลิตที่ไม่เพียงพอ

## <a name="symptoms"></a>อาการ

เมื่อคุณรันการจัดกำหนดการ คุณอาจได้รับข้อความแสดงข้อผิดพลาดต่อไปนี้:

> ไม่สามารถจัดกำหนดการใบสั่งผลิต %1 พบกำลังการผลิตที่ไม่เพียงพอ

มีเหตุผลหลายประการที่กลไกจัดการการจัดกำหนดการอาจล้มเหลวและออกข้อความแสดงข้อผิดพลาดนี้ หัวข้อนี้จะให้ข้อมูลที่จะช่วยคุณค้นหาสาเหตุรากของข้อผิดพลาด เพื่อลดปัญหา

## <a name="resolution"></a>การแก้ปัญหา

มีเหตุผลหลายประการที่กลไกจัดการการจัดกำหนดการอาจล้มเหลวและออกข้อความแสดงข้อผิดพลาด สำหรับคำแนะนำที่จะช่วยคุณค้นหาสาเหตุรากของข้อผิดพลาด แล้วลดปริมาณดังกล่าว โปรดดู [แก้ไขข้อผิดพลาดเกี่ยวกับกลไกจัดการการจัดกำหนดการ "กำลังการผลิตไม่เพียงพอ"](/dynamics365/supply-chain/master-planning/not-enough-capacity-error-resolution)
