# <a name="planned-purchase-order-is-created-when-a-purchase-exists-within-negative-days"></a>มีการสร้างแผนการใบสั่งซื้อเมื่อมีการซื้อภายในจำนวนวันที่ติดลบ

หมายเลข KB: 4614298

## <a name="symptoms"></a>อาการ

ถ้ารหัสความครอบคลุมเป็น *ต่ำสุด/สูงสุด* การเพิ่มประสิทธิภาพการวางแผนจะสร้างแผนการใบสั่งซื้อเมื่อมีการซื้ออยู่ภายในจำนวนวันที่ติดลบ

## <a name="resolution"></a>ความละเอียด

การเพิ่มประสิทธิภาพการวางแผนไม่สนับสนุนจำนวนวันที่ติดลบ อย่างไรก็ตาม การตรวจสอบให้แน่ใจเสมอว่าแผนการใบสั่งจะไม่มีการจัดตารางการผลิตภายในระยะเวลารอคอยสินค้าที่สัมพันธ์กับวันที่ปัจจุบัน ตัวอย่างเช่น ระยะเวลารอคอยสินค้าจากการซื้อคือ 10 วัน และคาดว่าใบสั่งซื้อจะมาถึงแปดวันนับจากวันนี้ ในกรณีนี้ ใบสั่งซื้อจะใช้เป็นการจัดหาวัสดุเพื่อความต้องการที่เป็นห้าวันนับจากวันนี้

จึงขอแนะนำว่าคุณควรปรับปรุงระยะเวลารอคอยสินค้าเพื่อครอบคลุมสถานการณ์ทั้งหมด (หรือเกือบทั้งหมด) ของคุณ
