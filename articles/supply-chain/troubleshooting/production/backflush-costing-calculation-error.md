# <a name="backflush-costing-calculation-error-during-inventory-closing"></a>ข้อผิดพลาดในการคํานวณการคิดต้นทุนแบบย้อนกลับระหว่างการปิดบัญชีสินค้าคงคลัง

## <a name="symptoms"></a>อาการ

ในการนำออกใช้ก่อนรุ่น 10.0.13 ถ้าคุณไม่ได้ใช้โฟลว์การคิดต้นทุนการผลิตแบบ lean คุณจะได้รับข้อความเตือนที่ผิดพลาดต่อไปนี้ระหว่างการปิดบัญชีสินค้าคงคลัง

> คุณกำลังจะดำเนินการปิดบัญชีสินค้าคงคลังที่มีวันที่ %1 ไม่มีการดำเนินการคำนวณการคิดต้นทุนแบบย้อนกลับที่มีวันที่ %1 สิ้นสุดรอบระยะเวลาที่ตรงกันแล้ว โปรดจำไว้ว่าไม่มีการดำเนินการคำนวณการคิดต้นทุนแบบย้อนกลับที่มีวันที่ %1 สิ้นสุดรอบระยะเวลาที่ตรงกันแล้ว การประเมินค่าสินค้าคงคลัง ต้นทุนขาย และผลต่างอาจไม่ถูกต้องในบัญชีแยกประเภทย่อยหรือบัญชีแยกประเภททั่วไป จนกว่าจะมีการดำเนินการนี้

## <a name="resolution"></a>การแก้ปัญหา

มีการแก้ไขปัญหานี้ในรุ่น10.0.13 และในภายหลัง สำหรับข้อมูลเพิ่มเติม ให้ดู [KB 4582468](https://fix.lcs.dynamics.com/Issue/Details?kb=4582468&bugId=468844&dbType=3&qc=fcd64080446a27382cfde3e4c3bdcfb714279185932259cd11ceb0d500617296)
