# <a name="late-selection-isnt-respected-when-production-orders-are-reset-via-a-batch-job"></a>การเลือกล่าช้าไม่ได้รับการยอมรับเมื่อรีเซ็ตใบสั่งผลิตผ่านชุดงาน

หมายเลข KB: 4614634

## <a name="symptoms"></a>อาการ

เมื่อคุณใช้ชุดงานที่เกิดซ้ำเพื่อรีเซ็ตสถานะของใบสั่งผลิต การเลือกล่าช้าจะไม่ได้รับการยอมรับ

## <a name="resolution"></a>ความละเอียด

การออกแบบปัจจุบันไม่สนับสนุนการใช้การเลือกล่าช้าของกระบวนการ *รีเซ็ตสถานะ*
