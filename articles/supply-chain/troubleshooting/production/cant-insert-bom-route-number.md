# <a name="cant-insert-bill-of-material-and-route-when-creating-a-new-production-order"></a>ไม่สามารถแทรกสูตรการผลิตและกระบวนการผลิตเมื่อสร้างใบสั่งผลิตใหม่

## <a name="symptoms"></a>อาการ

เมื่อคุณสร้างใบสั่งผลิตใหม่หลังจากที่คุณป้อนหมายเลขสินค้า ฟิลด์ **ไซต์** และ **คลังสินค้า** จะมีการตั้งค่าให้กับไซต์และคลังสินค้าเริ่มต้นที่กำหนดไว้ในหน้า **การตั้งค่าใบสั่งเริ่มต้น** สำหรับรายการสินค้าสำเร็จรูปโดยอัตโนมัติ นอกจากนี้ ระบบจะป้อนหมายเลข BOM และหมายเลขกระบวนการผลิตที่ใช้งานอยู่โดยอัตโนมัติ คุณจะไม่ได้รับข้อความต่อไปนี้ที่พร้อมต์เตือนให้คุณระบุค่าดังกล่าว

> แทรกรุ่นที่ใช้งานอยู่สำหรับสูตรการผลิตและกระบวนการผลิตหรือไม่

## <a name="resolution"></a>การแก้ปัญหา

คุณไม่ได้รับพร้อมท์ให้แทรก BOM และหมายเลขกระบวนการผลิต ถ้าคุณเลือกผลิตภัณฑ์ที่มีไซต์และคลังสินค้าอยู่แล้ว บนหน้า **การตั้งค่าใบสั่งเริ่มต้น** คุณได้รับข้อความแจ้งเฉพาะเมื่อมีการกำหนดค่าเหล่านั้นสำหรับผลิตภัณฑ์ที่เลือกเท่านั้น
