# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>ไม่สามารถสืบค้นกลับหมายเลขลำดับประจำสินค้าของสินค้าสำเร็จรูปเทียบกับหมายเลขลำดับประจำสินค้าของสินค้าที่บริโภคได้

## <a name="symptoms"></a>อาการ

คุณไม่สามารถสืบค้นกลับหมายเลขลำดับประจำสินค้าของสินค้าสำเร็จรูปเทียบกับหมายเลขลำดับประจำสินค้าของวัสดุที่ใบสั่งผลิตใช้ในการผลิตสินค้าสำเร็จรูปเหล่านั้นได้

## <a name="workaround"></a>วิธีการแก้ไข

สำหรับวิธีการแก้ปัญหา ให้สร้างใบสั่งผลิตสำหรับปริมาณเป็น 1
