# <a name="error-when-the-report-as-finished-journal-is-posted"></a>ข้อผิดพลาดเมื่อลงรายการบัญชีสมุดรายวันรายงานการเสร็จงาน

หมายเลข KB: 4612891

## <a name="symptoms"></a>อาการ

เมื่อคุณลงรายการบัญชีสมุดรายวัน **รายงานการเสร็จงาน** จะเกิดข้อผิดพลาด และคุณจะได้รับข้อความแสดงข้อผิดพลาดต่อไปนี้

> ไม่สามารถลดปริมาณที่สั่งได้ เนื่องจากมีธุรกรรมสินค้าคงคลังเปิดที่มีสถานะ "สั่งแล้ว" ไม่เพียงพอ สินค้าคือ ซื้อ ได้รับแล้ว หรือลงทะเบียน

ข้อผิดพลาดนี้จะเกิดขึ้นเฉพาะเมื่อตั้งค่าฟิลด์ **ปริมาณที่ผิดพลาด** ในบรรทัดแรกของสมุดรายวัน **รายงานการเสร็จงาน** และมีการตั้งค่าฟิลด์ **ปริมาณสินค้าที่ดี** ในบรรทัดสุดท้าย ถ้าตั้งค่าฟิลด์ **ปริมาณสินค้าที่ผิดพลาด** ในบรรทัดสุดท้าย จะไม่เกิดข้อผิดพลาดขึ้น

## <a name="resolution"></a>ความละเอียด

เมื่อต้องการป้องกันข้อผิดพลาดนี้ ให้เปิดหน้า **พารามิเตอร์การควบคุมการผลิต** และจากนั้นบนแท็บ **ทั่วไป** แล้วตั้งค่าตัวเลือก **เพิ่มปริมาณคงเหลือด้วยปริมาณที่ผิดพลาด** เป็น *ใช่*
