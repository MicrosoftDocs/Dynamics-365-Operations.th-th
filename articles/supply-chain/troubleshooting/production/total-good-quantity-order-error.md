# <a name="total-good-quantity-error-when-trying-to-end-a-production-order"></a>ข้อผิดพลาดของปริมาณสินค้าที่ดีทั้งหมด เมื่อพยายามสิ้นสุดใบสั่งผลิต

## <a name="symptoms"></a>อาการ

เมื่อคุณพยายามลงรายการบัญชีสมุดรายวันรายงานการเสร็จงานในใบสั่งผลิต คุณจะได้รับข้อความแสดงข้อผิดพลาดต่อไปนี้:

> ผลรวมปริมาณสินค้าที่ดีที่รายงานเมื่อเสร็จสมบูรณ์สำหรับการผลิตจะเป็น %1 ผลป้อนกลับสำหรับการดำเนินงานหลังสุดเป็น 0.00 ในยอดรวม

## <a name="cause"></a>สาเหตุ

ปัญหานี้จะเกิดขึ้นถ้าปริมาณที่ได้ลงรายการบัญชีไว้ในการดำเนินงานครั้งสุดท้ายไม่ถูกต้อง ตัวอย่างเช่น ถ้ามีการเริ่มต้นการผลิต แต่ปริมาณที่ต้องเริ่มต้นไม่มีการปัน ส่วนสมุดรายวันบัตรกระบวนการผลิตจะถูกลงรายการบัญชีโดยไม่มีรายการใด ๆ เมื่อต้องการยืนยันสถานการณ์ ให้เปิดใบสั่งผลิต และจากนั้น ในบานหน้าต่างการดำเนินการ บนแท็บ **มุมมอง** ให้เลือก **บัตรกระบวนการผลิต**

## <a name="resolution"></a>การแก้ปัญหา

คุณสามารถแก้ไขปัญหานี้ได้ด้วยการลงรายการบัญชีสมุดรายวันเพิ่มเติมสำหรับการดำเนินงานที่มีการลงรายการบัญชีสมุดรายวันไม่ถูกต้อง เริ่มต้นใบสั่งผลิตใหม่ และเลือกเพื่อลงรายการบัญชีสมุดรายวันเพิ่มเติม การดำเนินการนี้จะไม่เริ่มต้นใบสั่งผลิต แต่จะมีการลงรายการบัญชีสมุดรายวัน บัตรกระบวนการผลิตควรแสดงปริมาณที่ลงรายการบัญชีแล้ว และคุณควรจะสามารถจบใบสั่งผลิตได้เสร็จเรียบร้อยแล้ว
