# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>ชื่อทรัพยากรไม่ได้รับการอัพเดต หลังจากการเปลี่ยนชื่อผู้ปฏิบัติงานในสมุดที่อยู่สากล

## <a name="symptoms"></a>อาการ

หากคุณเปลี่ยนชื่อผู้ปฏิบัติงานในสมุดที่อยู่สากล ชื่อทรัพยากรที่ตรงกันไม่ได้รับการอัพเดตในกลุ่มทรัพยากรหลัก

## <a name="resolution"></a>การแก้ปัญหา

ปัจจุบันไม่มีการอัพเดตชื่อทรัพยากรที่ตรงกันโดยอัตโนมัติ เมื่อต้องการแก้ไขปัญหานี้ คุณต้องอัพเดตชื่อทรัพยากรด้วยตนเอง
