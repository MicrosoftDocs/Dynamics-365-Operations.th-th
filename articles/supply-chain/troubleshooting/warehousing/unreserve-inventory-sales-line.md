# <a name="cant-unreserve-inventory-on-a-sales-order-line"></a>ไม่สามารถจองสินค้าคงคลังในบรรทัดใบสั่งขายได้

## <a name="symptoms"></a>อาการ

ถ้ามีงานที่เปิดโดยเทียบกับบรรทัดใบสั่งขาย และคุณพยายามยกเลิกการจองสินค้าคงคลังในรายการนั้น คุณจะได้รับข้อความแสดงข้อผิดพลาดต่อไปนี้:

> ไม่สามารถลบการจองได้ เนื่องจากมีงานที่ถูกสร้างซึ่งอาศัยการจองอยู่

## <a name="resolution"></a>การแก้ปัญหา

ตรวจสอบว่างานกลุ่มการบรรจุเปิดอยู่เพื่อนำสินค้าจากสถานีบรรจุไปยังที่ตั้งอื่น (เช่น *Baydoor*) ตรวจทานเรกคอร์ดในหน้า **ล็อกประวัติการสร้างงาน** และ **รายละเอียดงาน** เพื่อตรวจสอบว่ามีการจองสินค้าคงคลังทางกายภาพอย่างไร แล้วทำให้เสร็จสมบูรณ์หรือลบงานเพื่อเพิ่มความสะดวกในการจอง
