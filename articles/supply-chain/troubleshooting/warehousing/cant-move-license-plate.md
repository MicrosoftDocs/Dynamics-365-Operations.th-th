# <a name="cant-move-license-plate-if-serial-number-has-blank-issue-and-blank-receipt-allowed"></a>ไม่สามารถย้ายป้ายทะเบียนได้ หากหมายเลขลำดับประจำสินค้าสามารถทำการตัดสินค้าที่ว่างเปล่า และการรับสินค้าที่ว่างเปล่าได้

## <a name="symptoms"></a>อาการ

คุณไม่สามารถย้ายป้ายทะเบียนได้ โดยใช้รายการเมนู **การเคลื่อนย้าย** ถ้าหมายเลขลำดับประจำสินค้ามีการตั้งค่า *การตัดสินค้าที่ว่างเปล่าได้* และ *การรับสินค้าที่ว่างเปล่าได้* บนกลุ่มมิติการติดตาม และถ้ามีป้ายทะเบียนหลายใบในสถานที่เดียวกัน บางรายมีหมายเลขลำดับประจำสินค้าและบางอย่างที่ไม่มีหมายเลขลำดับประจำสินค้า

## <a name="resolution"></a>การแก้ปัญหา

ปัญหานี้จะได้รับการแก้ไขโดยการเปลี่ยนแปลงที่มีการปรับใช้ใน [KB 4571546](https://fix.lcs.dynamics.com/Issue/Details?kb=4571546&bugId=467880&dbType=3&qc=5b46d7faa9cc326cebfe9854cb30be8ea30b21ef33d3572c325fbb21202de687) การเปลี่ยนแปลงดังกล่าวจะทำให้ฟิลด์ **หมายเลขประจำสินค้า** ไม่จำเป็นต้องระบุเมื่อ การตัดสินค้าที่ว่างเปล่า และการรับสินค้าที่ว่างเปล่าสามารถทำได้
