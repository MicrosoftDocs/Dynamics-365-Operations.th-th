# <a name="packing-station-doesnt-show-product-notes"></a>สถานีการจัดส่งไม่แสดงบันทึกผลิตภัณฑ์

หมายเลข KB: 4614615

## <a name="symptoms"></a>อาการ

บันทึกการจัดส่งไม่แสดงอยู่ในฟอร์มการจัดส่ง เมื่อคําแนะนําในการจัดส่งถูกเพิ่มเป็นเอกสารแนบกับผลิตภัณฑ์หลักหรือผลิตภัณฑ์ย่อย

## <a name="resolution"></a>ความละเอียด

ระบบมีการทำงานตามที่ออกแบบ

ตรรกะปัจจุบันที่แสดงบันทึกการจัดส่งในฟอร์มการจัดส่งกําหนดให้ต้องเชื่อมโยงบันทึกกับการจัดส่งสินค้า
