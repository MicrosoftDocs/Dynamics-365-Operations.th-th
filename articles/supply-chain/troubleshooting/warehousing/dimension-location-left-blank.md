# <a name="error-when-confirming-shipment-after-creating-a-transfer-order-for-a-serial-item"></a>ข้อผิดพลาดเมื่อยืนยันการจัดส่งหลังจากสร้างใบสั่งโอนย้ายสำหรับชุดสินค้า

## <a name="symptoms"></a>อาการ

ถ้าคุณสร้างใบสั่งโอนย้ายสำหรับชุดสินค้าโดยใช้คลังสินค้าที่เปิดใช้งานสำหรับการจัดการคลังสินค้าขั้นสูง (WMS) แล้ว หลังจากงานเสร็จสมบูรณ์แล้ว พยายามยืนยันการจัดส่ง คุณได้รับข้อความแสดงข้อความแสดงข้อผิดพลาดนี้:

> ที่ตั้งมิติต้องไม่ว่างเปล่าถ้าตั้งค่าหมายเลขลำดับประจำสินค้าของมิติ

## <a name="cause"></a>สาเหตุ

ทั้งนี้เนื่องจากฟิลด์ **สถานที่รับสินค้าเริ่มต้น** จะว่างเปล่าสำหรับคลังสินค้าส่งต่อของคลังสินค้า "เริ่มต้น"

## <a name="resolution"></a>การแก้ปัญหา

เมื่อต้องการแก้ไขปัญหานี้ ให้ระบุสถานที่รับสินค้าเริ่มต้นในคลังสินค้าส่งต่อ ตรวจสอบให้แน่ใจว่าที่ตั้งนี้เป็นป้ายทะเบียนที่ควบคุม
