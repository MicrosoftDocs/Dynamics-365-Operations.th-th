# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>สามารถใช้บทบาทผู้ดูแลระบบเพื่อเข้าถึงตัวเลียนแบบแอปบนอุปกรณ์เคลื่อนที่

## <a name="symptoms"></a>อาการ

เมื่อตั้งค่าคลังสินค้าใน Supply Chain Management คุณไม่สามารถใช้บทบาทใดๆ ยกเว้นบทบาทผู้ดูแลระบบเพื่อเข้าถึงตัวเลียนแบบแอปในอุปกรณ์เคลื่อนที่

## <a name="resolution"></a>การแก้ปัญหา

ตัวเลียนแบบแอปบนอุปกรณ์เคลื่อนที่มีการตั้งค่าให้ทำงานกับบัญชีผู้ดูแลระบบเท่านั้น สำหรับวัตถุประสงค์ในการทดสอบและการดำเนินการสด เราขอแนะนำให้คุณใช้แอปการจัดการคลังสินค้าบนมือถือเอง
