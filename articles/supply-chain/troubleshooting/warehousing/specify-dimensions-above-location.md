# <a name="cant-release-a-load-for-partial-quantity-with-batch-above-reservation-hierarchy"></a>ไม่สามารถนำโหลดออกใช้สำหรับปริมาณบางส่วนที่มีลำดับชั้นการจองเหนือชุดงาน

## <a name="symptoms"></a>อาการ

เมื่อคุณใช้สินค้าที่มีลำดับชั้นการจอง *เหนือชุดงาน* คำสั่ง **นำออกใช้ไปยังคลังสินค้า** บนหน้า **เวิร์กเบนช์การวางแผนการบรรทุก** สำหรับปริมาณบางส่วนไม่ทำงาน หากคุณพยายามนำโหลดออกใช้ คุณจะได้รับข้อความแสดงข้อผิดพลาดต่อไปนี้:

> หากต้องการให้กำหนดให้กับเวฟ รายการโหลดต้องระบุมิติเหนือสถานที่ ถ้าต้องการกำหนดมิติเหล่านี้ ให้จองและสร้างรายการโหลดใหม่

อย่างไรก็ตาม เมื่อคุณใช้สินค้าที่มีลำดับชั้นการจอง *ต่ำกว่าชุดงาน* คุณสามารถนำโหลดออกใช้ได้สำหรับปริมาณบางส่วนจากหน้า **เวิร์กเบนช์การวางแผน**

## <a name="cause"></a>สาเหตุ

เมื่อมิติที่อยู่ด้านบนของมิติ **สถานที่** ในลำดับชั้นการจอง ต้องระบุก่อนการนำออกใช้ไปยังคลังสินค้า สินค้าคงคลังสามารถปันส่วนเสร็จเรียบร้อย เฉพาะเมื่อไม่มีช่องว่างในมิติเหนือสถานที่เท่านั้น

## <a name="resolution"></a>การแก้ปัญหา

ตรวจสอบให้แน่ใจว่ามิติทั้งหมดเหนือ **สถานที่** ได้รับการมอบหมายโดยการจองและสร้างรายการโหลดใหม่
