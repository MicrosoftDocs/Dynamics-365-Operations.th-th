# <a name="invalid-license-plate-error-when-scanning-license-plate-id-in-the-mobile-app"></a>ป้ายทะเบียนไม่ถูกต้องเกิดข้อผิดพลาดเมื่อสแกนรหัสป้ายทะเบียนในแอปบนมือถือ

## <a name="symptoms"></a>อาการ

เมื่อคุณสแกนรหัสป้ายทะเบียนในแอป Warehouse Management บนมือถือ คุณได้อาจรับข้อความแสดงข้อผิดพลาดนี้:

> ป้ายทะเบียนไม่ถูกต้อง

## <a name="resolution"></a>การแก้ปัญหา

ตรวจสอบให้แน่ใจว่ารหัสป้ายทะเบียนมีอยู่ในตารางป้ายทะเบียน และสินค้าและปริมาณบนป้ายทะเบียนพร้อมใช้งาน (กล่าวคือไม่ถูกบล็อค)
