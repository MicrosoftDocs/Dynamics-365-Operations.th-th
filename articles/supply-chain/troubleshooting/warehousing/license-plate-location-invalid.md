# <a name="invalid-license-plate-or-location-error-when-scanning-in-the-mobile-app"></a>ป้ายทะเบียนไม่ถูกต้องหรือสถานที่เกิดข้อผิดพลาดเมื่อสแกนรหัสในแอปบนมือถือ

## <a name="symptoms"></a>อาการ

เมื่อคุณสแกนรหัสป้ายทะเบียนหรือสถานที่ คุณอาจได้รับข้อความแสดงข้อผิดพลาดต่อไปนี้:

> ป้ายทะเบียนหรือสถานที่ไม่ถูกต้อง

## <a name="resolution"></a>การแก้ปัญหา

ตรวจสอบให้แน่ใจว่ารหัสป้ายทะเบียนไม่ถูกจองโดยอย่างอื่น ปัญหานี้จะเกิดขึ้นเมื่อมีการใช้ค่าที่ผู้ใช้สแกนในแอปการจัดการคลังสินค้าบนมือถือเป็นทั้งสถานที่ที่ถูกต้องและรหัสป้ายทะเบียนที่ถูกต้อง อย่างไรก็ตาม ปัญหานี้ได้รับการแก้ไขแล้วในรุ่น10.0.11
