# <a name="cant-move-inventory-to-a-location-that-is-license-plate-controlled"></a>ไม่สามารถย้ายสินค้าคงคลังไปยังสถานที่ที่ควบคุมป้ายทะเบียน

## <a name="symptoms"></a>อาการ

คุณไม่สามารถลดปริมาณที่เบิกสินค้าไว้ในจำนวนงานในศูนย์การผลิตได้

## <a name="resolution"></a>การแก้ปัญหา

ในรุ่นก่อน คุณไม่สามารถลดปริมาณที่เบิกสินค้าไว้ในจำนวนงานในศูนย์การผลิตได้ อย่างไรก็ตาม ตอนนี้คุณสามารถยกเลิกการเบิกไปยังสถานที่ที่ควบคุมป้ายทะเบียน คุณต้องระบุทั้งค่า **สถานที่** และค่า **ป้ายทะเบียน** สำหรับรายการจำนวนงานในศูนย์การผลิตบนหน้า **ลดปริมาณที่เบิกสินค้า**
