# <a name="shipment-for-load-has-been-confirmed-but-no-lines-are-posted"></a>การจัดส่งสำหรับโหลดได้รับการยืนยันแล้ว แต่ไม่มีการลงรายการบัญชีรายการ

## <a name="symptoms"></a>อาการ

เมื่อใช้งานการดําเนินงานคลังสินค้าขาออก คุณอาจได้รับข้อความต่อไปนี้:

> การจัดส่งสำหรับโหลด 1% ได้รับการยืนยันแล้ว

อย่างไรก็ตาม ไม่มีการลงรายการบัญชีเพิ่มเติม

## <a name="resolution"></a>การแก้ปัญหา

นี่เกิดจากการออกแบบ การยืนยันการจัดส่งไม่มีผลกระทบต่อการลงรายการบัญชี อัพเดตสถานะการจัดส่งและโหลดเท่านั้น การลงรายการบัญชีต้องเกิดขึ้นในกระบวนการแยกต่างหาก
