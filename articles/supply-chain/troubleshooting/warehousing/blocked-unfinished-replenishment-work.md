# <a name="picking-work-cant-be-unblocked-because-of-unfinished-replenishment-work"></a>งานการเบิกสินค้าไม่สามารถปลดการบล็อคได้ เนื่องจากงานการเติมสินค้าที่ไม่เสร็จสมบูรณ์

## <a name="symptoms"></a>อาการ

เมื่อใช้การเพิ่มเติมสินค้าตามความต้องการของเวฟ งานการเบิกสินค้าสามารถถูกบล็อคได้ เนื่องจากงานการเติมสินค้าที่มีการเชื่อมโยง ถ้าต้องเติมสถานที่เบิกสินค้าเพื่อตอบสนองความต้องการของใบสั่งต้นทาง ระบบจะสร้างทั้งงานการเติมสินค้าและงานการเบิกสินค้า อย่างไรก็ตาม มีการบล็อคงานการเบิกสินค้าจนกว่างานการเติมสินค้าจะเสร็จสมบูรณ์ และคุณจะได้รับข้อความแสดงข้อผิดพลาดต่อไปนี้:

> ไม่สามารถยกเลิกการบล็อค %1 เนื่องจากมีงานการเพิ่มเติมสินค้าที่ยังไม่เสร็จสิ้น

## <a name="resolution"></a>การแก้ปัญหา

ลักษณะการทำงานนี้มีเจตนา เนื่องจากสถานที่เบิกสินค้าจะไม่มีสินค้าคงคลังเพียงพอ ยกเว้นว่างานการเติมสินค้าเสร็จสมบูรณ์แล้ว ทำงานการเติมสินค้าให้เสร็จสมบูรณ์ แล้วประมวลผลงานการเบิกสินค้า
