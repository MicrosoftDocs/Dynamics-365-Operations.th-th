# <a name="you-cant-add-a-line-to-a-purchase-requisition-after-you-request-a-change"></a>คุณไม่สามารถเพิ่มรายการในใบขอซื้อหลังจากที่คุณร้องขอการเปลี่ยนแปลง

หมายเลข KB: 4611211

## <a name="symptoms"></a>อาการ

ระบบไม่อนุญาตให้คุณเพิ่มรายการในใบขอซื้อหลังจากที่คุณร้องขอการเปลี่ยนแปลง

## <a name="resolution"></a>ความละเอียด

คุณต้องเรียกคืนลำดับงาน หลังจากที่ใบขอซื้ออยู่ในสถานะ *ร่าง* คุณสามารถแก้ไขใบขอซื้อหรือเพิ่มบรรทัดในใบขอซื้อต่อไปได้
