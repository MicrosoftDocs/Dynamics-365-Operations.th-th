# <a name="maintain-planned-orders"></a><span data-ttu-id="2a25d-104">รักษาแผนการใบสั่ง</span><span class="sxs-lookup"><span data-stu-id="2a25d-104">Maintain planned orders</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="2a25d-105">บทความนี้จะแสดงข้อมูลเกี่ยวกับวิธีการจัดการใบสั่งที่วางแผน</span><span class="sxs-lookup"><span data-stu-id="2a25d-105">This article provides information about how to manage planned orders.</span></span> <span data-ttu-id="2a25d-106">โดยอธิบายวิธีการปรับปรุงสถานะของแผนการใบสั่ง ยืนยัน และกรองข้อมูลสำหรับแผนการใบสั่งที่มีสถานะเดียวกันเป็นแผนการใบสั่งที่เลือก</span><span class="sxs-lookup"><span data-stu-id="2a25d-106">It describes how you can update the status of planned orders, firm them, and filter for planned orders that have the same status as a selected planned order.</span></span>

<span data-ttu-id="2a25d-107">คุณสามารถจัดการแผนการใบสั่งจากพื้นที่ทำงาน **การวางแผนหลัก** รายการ **แผนการใบสั่ง** หรือ **แผนการใบสั่งผลิต**, **แผนการใบสั่งซื้อ**และรายการ **แผนการโอนย้าย**</span><span class="sxs-lookup"><span data-stu-id="2a25d-107">You can manage planned orders from the **Master planning** workspace, the **Planned order** list, or the **Planned production orders**, **Planned purchase orders**, and **Planned transfer** lists.</span></span> <span data-ttu-id="2a25d-108">คุณสามารถใช้ฟิลด์ **สถานะ** เพื่อช่วยติดตามความคืบหน้าของคุณ</span><span class="sxs-lookup"><span data-stu-id="2a25d-108">You can use the **Status** field to help track your progress.</span></span> <span data-ttu-id="2a25d-109">ใช้ค่าต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="2a25d-109">The following values are used:</span></span>

-   <span data-ttu-id="2a25d-110">เมื่อการวางแผนหลักสร้างแผนการใบสั่ง แผนการใบสั่งมีสถานะเป็น **ยังไม่ได้ดำเนินการ**</span><span class="sxs-lookup"><span data-stu-id="2a25d-110">When master planning generates planned orders, the planned orders have a status of **Unprocessed**.</span></span>
-   <span data-ttu-id="2a25d-111">ถ้าคุณเลือกที่จะไม่ยืนยันแผนการใบสั่ง คุณสามารถกำหนดสถานะให้กับแผนการใบสั่งนั้นเป็น **เสร็จสมบูรณ์**</span><span class="sxs-lookup"><span data-stu-id="2a25d-111">If you decide not to firm a planned order, you can give it a status of **Completed**.</span></span>
-   <span data-ttu-id="2a25d-112">เมื่อคุณเลือกที่จะยืนยันแผนการใบสั่ง คุณสามารถกำหนดสถานะให้กับแผนการใบสั่งนั้นเป็น **อนุมัติแล้ว**.</span><span class="sxs-lookup"><span data-stu-id="2a25d-112">When you decide to firm a planned order, you can give it a status of **Approved**.</span></span> <span data-ttu-id="2a25d-113">สถานะนี้บ่งชี้ว่าคุณอนุมัติการยืนยันแผนการใบสั่ง แต่ยังไม่ได้ทำการยืนยัน</span><span class="sxs-lookup"><span data-stu-id="2a25d-113">This status indicates that you approve firming of the planned order, but it isn't firmed yet.</span></span>

<span data-ttu-id="2a25d-114">**หมายเหตุ:** แผนการใบสั่งที่อนุมัติแล้วจะถูกโอนย้าย ในสถานะปัจจุบัน ไปยังการคำนวณการวางแผนหลัก</span><span class="sxs-lookup"><span data-stu-id="2a25d-114">**Note:** An approved planned order is transferred, in its current state, to the next master planning calculation.</span></span> <span data-ttu-id="2a25d-115">คุณสามารถยืนยันแผนการใบสั่ง โดยการคลิก **ยืนยัน**</span><span class="sxs-lookup"><span data-stu-id="2a25d-115">You can firm planned orders by clicking **Firm**.</span></span> <span data-ttu-id="2a25d-116">คุณสามารถยืนยันแผนการใบสั่งต่อไปนี้:</span><span class="sxs-lookup"><span data-stu-id="2a25d-116">You can firm the following planned orders:</span></span>

-   <span data-ttu-id="2a25d-117">แผนการใบสั่งที่เลือก</span><span class="sxs-lookup"><span data-stu-id="2a25d-117">The planned order that is selected.</span></span>
-   <span data-ttu-id="2a25d-118">แผนการใบสั่งหลายใบ</span><span class="sxs-lookup"><span data-stu-id="2a25d-118">Multiple planned orders.</span></span>
-   <span data-ttu-id="2a25d-119">แผนการใบสั่งที่สร้างโดยการกระจายจากหน้า **การกระจาย**</span><span class="sxs-lookup"><span data-stu-id="2a25d-119">Planned orders that are generated by an explosion from the **Explosion** page.</span></span> <span data-ttu-id="2a25d-120">คลิก **แผนการใบสั่ง** เลือกแผนการใบสั่ง แล้วคลิก **ยืนยัน**.</span><span class="sxs-lookup"><span data-stu-id="2a25d-120">Click **Planned orders**, select the planned order, and then click **Firm**.</span></span>

<span data-ttu-id="2a25d-121">เมื่อมียืนยันแผนการใบสั่ง แผนการใบสั่งจะถูกย้ายไปยังส่วนของใบสั่งของโมดูลที่เกี่ยวข้อง</span><span class="sxs-lookup"><span data-stu-id="2a25d-121">When a planned order is firmed, it's moved to the orders section of the relevant module.</span></span> <span data-ttu-id="2a25d-122">**หมายเหตุ:** คุณสามารถคลิกขวาที่แผนการใบสั่งที่มีสถานะหนึ่งๆ และกรองข้อมูลแผนการใบสั่งที่มีสถานะเดียวกัน</span><span class="sxs-lookup"><span data-stu-id="2a25d-122">**Note:** You can right-click a planned order that has a particular status and filter for other planned orders that have the same status.</span></span> <span data-ttu-id="2a25d-123">ฟังก์ชันนี้มีประโยชน์ ตัวอย่างเช่น ถ้าคุณต้องการกรองแผนการใบสั่งทั้งหมดที่มีสถานะเป็น **อนุมัติ** เพื่อให้คุณสามารถยืนยันแผนการใบสั่งเหล่านั้นได้</span><span class="sxs-lookup"><span data-stu-id="2a25d-123">This functionality is useful if, for example, you want to filter for all planned orders that have a status of **Approved**, so that you can then firm them.</span></span>
