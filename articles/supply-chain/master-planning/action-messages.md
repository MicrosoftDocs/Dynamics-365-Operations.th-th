# <a name="action-messages"></a><span data-ttu-id="71026-103">ข้อความการดำเนินการ</span><span class="sxs-lookup"><span data-stu-id="71026-103">Action messages</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="71026-104">ข้อความเกี่ยวกับการดำเนินการเป็นคำแนะนำที่ระบบสร้างขึ้นเพื่อเปลี่ยนแปลงใบสั่งเติมสินค้าที่วางแผนไว้หรือมีการยืนยันแล้ว</span><span class="sxs-lookup"><span data-stu-id="71026-104">An action message is a system-generated suggestion to change an existing planned or firmed order.</span></span>

## <a name="introduction"></a><span data-ttu-id="71026-105">คำนำ</span><span class="sxs-lookup"><span data-stu-id="71026-105">Introduction</span></span>

<span data-ttu-id="71026-106">ข้อความการดำเนินการจะถูกสร้างโดยการคำนวณการวางแผนหลักเพื่อตอบสนองความต้องการที่เปลี่ยนแปลงไป</span><span class="sxs-lookup"><span data-stu-id="71026-106">Action messages are generated by the master planning calculation in response to changed requirements.</span></span> <span data-ttu-id="71026-107">ตัวอย่างเช่น ในใบสั่งขายอาจมีการเปลี่ยนแปลงวันที่จัดส่งหรือปริมาณที่คุณได้สร้างใบสั่งซื้อเพื่อตอบสนองความต้องการแล้ว</span><span class="sxs-lookup"><span data-stu-id="71026-107">For example, the ship date or quantity may have changed on a sales order for which you have already created a purchase order to fulfill the demand.</span></span> <span data-ttu-id="71026-108">ในกรณีนี้ หนึ่ง หรือข้อความการดำเนินการหลายอย่างที่สร้างขึ้นโดยการคำนวณการวางแผนหลักเพื่ออัพเดตใบสั่งซื้อ</span><span class="sxs-lookup"><span data-stu-id="71026-108">In this case, one or multiple action messages are generated by the master planning calculation to update the purchase order.</span></span> <span data-ttu-id="71026-109">คุณสามารถตัดสินใจได้ว่าจะทำการเปลี่ยนแปลงที่แนะนำหรือไม่</span><span class="sxs-lookup"><span data-stu-id="71026-109">You decide whether to make the changes that are suggested.</span></span>

<span data-ttu-id="71026-110">คุณสามารถตั้งค่าวิธีการคำนวณข้อความการดำเนินการสำหรับกลุ่มความครอบคลุมที่แนบกับสินค้า</span><span class="sxs-lookup"><span data-stu-id="71026-110">You can set up how action messages are calculated for a coverage group that you attach to an item.</span></span>
