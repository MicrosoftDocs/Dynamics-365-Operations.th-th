<span data-ttu-id="cd4d3-105">เมื่อคุณสร้างการคาดการณ์พื้นฐาน ก่อนอื่น คุณต้องระบุพารามิเตอร์และตัวกรองที่ใช้ในการคำนวณ </span><span class="sxs-lookup"><span data-stu-id="cd4d3-105">When you create a baseline forecast, you must first specify the parameters and filters that are used in the calculation.</span></span> <span data-ttu-id="cd4d3-106">ตัวอย่างเช่น คุณสามารถสร้างการคาดการณ์พื้นฐานที่ประเมินความต้องการตามข้อมูลธุรกรรมจากปีที่ผ่านมา สำหรับบริษัทที่ระบุ สำหรับเดือนที่กำลังจะมาถึง และสำหรับกลุ่มสินค้าที่เลือก</span><span class="sxs-lookup"><span data-stu-id="cd4d3-106">For example, you can create a baseline forecast that estimates demand based on transaction data from the past year for a specific company, for the coming month, and for a selected group of items.</span></span>

<span data-ttu-id="cd4d3-110">จำนวนของกลุ่มในการสร้างการคาดการณ์ถูกตั้งค่าในฟิลด์ **ระดับการคาดการณ์**</span><span class="sxs-lookup"><span data-stu-id="cd4d3-110">The number of buckets to generate a forecast for is set in the **Forecast horizon** field.</span></span>

-   <span data-ttu-id="cd4d3-115">โดยการคัดลอกความต้องการจากไตรมาสก่อนหน้า</span><span class="sxs-lookup"><span data-stu-id="cd4d3-115">By copying the demand from the previous quarter.</span></span>

<span data-ttu-id="cd4d3-116">เพื่อป้องกันความสับสนในแผนการผลิต จำนวนของกลุ่มคาดการณ์บางจำนวนสามารถถูกหยุดการดำเนินการได้</span><span class="sxs-lookup"><span data-stu-id="cd4d3-116">To prevent confusion in the production plans, a certain number of forecast buckets can be frozen.</span></span> <span data-ttu-id="cd4d3-117">หมายเลขนี้ถูกตั้งค่าในฟิลด์ **กรอบเวลาหยุด**</span><span class="sxs-lookup"><span data-stu-id="cd4d3-117">This number is set in the **Freeze time fence** field.</span></span> <span data-ttu-id="cd4d3-118">ในหน้า **การคาดการณ์ความต้องการที่มีการปรับปรุง** เซลล์สำหรับกลุ่มที่ถูกหยุดไว้ถูกปิดการใช้งานเพื่อให้ภาพที่บ่งชี้ว่าค่าเหล่านั้นไม่ควรถูกเปลี่ยน</span><span class="sxs-lookup"><span data-stu-id="cd4d3-118">On the **Adjusted demand forecast** page, the cells for the frozen buckets are disabled, to give a visual indication that those values should not be changed.</span></span>
