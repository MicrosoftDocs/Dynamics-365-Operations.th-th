# <a name="cancel-a-job-that-was-created-using-the-deprecated-master-planning-engine"></a>ยกเลิกงานที่สร้างขึ้นโดยใช้กลไกจัดการการวางแผนหลักที่ไม่สนับสนุน

[!include [banner](../includes/banner.md)]

มีตัวเลือกหลายรายการสำหรับการยกเลิกงานที่สร้างขึ้นโดยใช้กลไกจัดการการวางแผนหลักที่ไม่สนับสนุน ตัวอย่างเช่น คุณอาจต้องการยกเลิกงานการวางแผนหลัก ถ้ามีการเริ่มต้นโดยไม่ได้ตั้งใจหรือทำงานนานกว่าที่คาดไว้และคุณต้องการที่จะสิ้นสุด

วิธีที่ดีที่สุดในการยกเลิกงานการวางแผนคือจากหน้า **กระบวนการวางแผนที่ยังไม่เสร็จสิ้น** ตัวเลือกอื่นจากหน้า **ชุดงาน** และ **ชุดงานที่ปรับปรุง** ควรจะใช้เฉพาะเมื่อยกเลิกงานการวางแผนหลักจาหน้า **กระบวนการวางแผนที่ยังไม่เสร็จสมบูรณ์** ไม่เสร็จภายในไม่กี่นาที

## <a name="preferred-cancel-option"></a>ตัวเลือกการยกเลิกที่ต้องการ

### <a name="cancel-master-planning-job-from-the-unfinished-planning-processes-page"></a>ยกเลิกงานการวางแผนหลักจากหน้า กระบวนการวางแผนที่ยังไม่เสร็จสิ้น

1. ไปยัง **การวางแผนหลัก > การสอบถามและรายงาน > การวางแผนหลัก > กระบวนการวางแผนที่ยังไม่เสร็จสิ้น**
2. เลือกบรรทัดที่กระบวนการวางแผนที่คุณต้องการยกเลิก
3. เลือก **ยกเลิก**

## <a name="additional-cancel-options"></a>ตัวเลือกยกเลิกเพิ่มเติม

รายการเหล่านี้ควรถูกใช้ เมื่อยกเลิกงานการวางแผนหลักจากหน้า **กระบวนการวางแผนที่ยังไม่เสร็จสิ้น** ไม่เสร็จสมบูรณ์ภายในสองสามนาที

### <a name="delete-master-planning-job-from-the-batch-jobs-page"></a>ลบงานการวางแผนหลักออกจากหน้า **ชุดงาน**

1. ไปที่ **การดูแลระบบ > การสอบถาม > ชุดงาน**
2. เลือกบรรทัดที่กระบวนการวางแผนงานที่คุณต้องการลบ
3. เลือก **ลบ**

### <a name="abort-master-planning-job-task-from-the-batch-jobs-enhanced-page"></a>ยกเลิกงานการวางแผนหลักออกจากหน้า **ชุดงานที่ปรับปรุง**

1. ไปที่ **การดูแลระบบ > การสอบถาม > ชุดงาน**
2. ถ้าไม่มีการแสดงรหัสงานในรายการให้คลิก **สลับไปยังแบบฟอร์มที่ปรับปรุงแล้ว** มิฉะนั้นดำเนินการขั้นตอนต่อไป
3. เปิดชุดงาน เลือก **รหัสงาน** สำหรับชุดงานที่มีงานที่คุณต้องการสิ้นสุด
4. ใน **ชุดงาน** เลือกงานที่จะสิ้นสุด
5. เลือก **เปลี่ยนแปลงสถานะ** เลือก **การยกเลิก** และคลิก **ตกลง**
6. บนแท็บ **ชุดงาน** ให้คลิก **ยกเลิก**
