### <a name="static-plan"></a><span data-ttu-id="b8e11-111">แผนแบบคงที่</span><span class="sxs-lookup"><span data-stu-id="b8e11-111">Static plan</span></span>

<span data-ttu-id="b8e11-113">แผนแบบคงที่จะใช้ในการอนุมัติและยืนยันใบสั่งที่แนะนำ</span><span class="sxs-lookup"><span data-stu-id="b8e11-113">The static plan is used to approve and firm the orders that are suggested.</span></span> <span data-ttu-id="b8e11-114">ซึ่งเป็นแผนการดำเนินงานที่บุคลากรต่างๆ ของบริษัท (เช่น ผู้ซื้อ หรือผู้วางแผนการผลิต) สามารถอ้างอิงการตัดสินใจของพวกเขาและใช้ในการดำเนินการงานประจำวันและกิจกรรมของพวกเขาได้</span><span class="sxs-lookup"><span data-stu-id="b8e11-114">It's an operating plan that various company personnel (such as a purchaser or production planner) can base their decisions on and use to perform their daily tasks and activities.</span></span>

<span data-ttu-id="b8e11-118">โดยปกติแผนแบบไดนามิกจะเริ่มต้นเป็นสำเนาของแผนแบบคงที่ แต่สามารถปรับปรุงได้ทุกครั้งที่ข้อมูลหลักเปลี่ยนแปลง</span><span class="sxs-lookup"><span data-stu-id="b8e11-118">The dynamic plan usually starts as a copy of the static plan, but it can be updated every time that master data changes.</span></span> <span data-ttu-id="b8e11-119">ตัวอย่างเช่น ถ้าข้อมูลเปลี่ยนแปลง จะมีการสร้างใบสั่งขายใหม่</span><span class="sxs-lookup"><span data-stu-id="b8e11-119">For example, if the data changes, a new sales order is created.</span></span>

## <a name="strategies-for-using-master-plans"></a><span data-ttu-id="b8e11-126">กลยุทธ์สำหรับการใช้แผนหลัก</span><span class="sxs-lookup"><span data-stu-id="b8e11-126">Strategies for using master plans</span></span>
