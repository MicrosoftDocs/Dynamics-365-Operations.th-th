# <a name="create-a-material-plan-for-co-products"></a>สร้างแผนวัสดุสำหรับผลิตภัณฑ์ร่วม

[!include [banner](../../includes/banner.md)]

ผู้วางแผนการผลิตจะวางแผนความต้องการวัสดุสำหรับสินค้าที่เป็นสูตรผลิตภัณฑ์ร่วม  บริษัทข้อมูลสาธิตที่เคยสร้างขั้นตอนนี้คือ USP2


## <a name="create-requirement-for-a-co-product"></a>สร้างความต้องการสำหรับผลิตภัณฑ์ร่วม
1. ไปที่แดชบอร์ดเริ่มต้น 
2. คลิกที่การประมวลผลและการสอบถามใบสั่งขาย 
3. คลิก สร้าง
4. คลิกที่ใบสั่งขาย 
5. ในฟิลด์บัญชีลูกค้า ให้พิมพ์ค่า 
    * ตัวอย่าง: US-001  
6. คลิก ตกลง
7. ในฟิลด์หมายเลขสินค้า ให้พิมพ์ค่า
    * ตัวอย่าง : P6003  
8. ในฟิลด์ ปริมาณ ให้ป้อนตัวเลข
    * ตัวอย่าง : 50000  
9. คลิก บันทึก

## <a name="create-a-material-plan-for-co-products"></a>สร้างแผนวัสดุสำหรับผลิตภัณฑ์ร่วม
1. ปิดหน้า
2. ปิดหน้า
3. คลิกที่งานการวางแผนหลัก

5. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
    * ตัวอย่าง: แผนหลัก  
6. คลิก เรียกใช้
7. ขยายหรือยุบส่วนของเรกคอร์ดที่จะรวม

    * ตัวอย่าง : P6003  
11. คลิก ตกลง
12. คลิก ตกลง

    * กรองด้วยสูตรสินค้าที่เป็นผลิตภัณฑ์ร่วมของสินค้าที่คุณได้สร้างใบสั่งขายให้

16. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
17. ขยายหรือยุบส่วนความเชื่อมโยงความต้องการกับการจัดซื้อ 
18. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
    * คำสั่งซื้อที่วางแผนไว้จะเชื่อมโยงกับใบสั่งขายสำหรับสินค้าร่วม  
19. ปิดหน้า
