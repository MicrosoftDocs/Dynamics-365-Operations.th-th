# <a name="create-a-schedule-for-a-site"></a>สร้างกำหนดการสำหรับไซต์

[!include [banner](../../includes/banner.md)]

กระบวนงานนี้แสดงวิธีจัดกำหนดการใบสั่งผลิตที่ยังไม่ได้เริ่มต้นสำหรับไซต์   ข้อมูลสาธิตของบริษัท USMF ถูกใช้เพื่อดำเนินกระบวนงานนี้ให้เสร็จสมบูรณ์


## <a name="identify-production-orders-that-are-not-started"></a>ระบุใบสั่งผลิตที่ยังไม่เริ่มต้น
1. ไปที่การควบคุมการผลิต > ใบสั่งผลิต > ใบสั่งผลิตทั้งหมด 
2. ใช้ตัวกรองข้อมูลด่วนเพื่อค้นหาเรกคอร์ด  ตัวอย่างเช่น กรองข้อมูลในฟิลด์ไซต์ ด้วยค่า '1'
    * 1 แสดงถึงไซต์ใน USMF ถ้าคุณไม่ได้กำลังใช้ USMF ให้เลือกไซต์จากบริษัทของคุณเอง  
3. เปิดตัวกรองคอลัมน์สถานะ
4. ใช้ตัวกรองในฟิลด์ "สถานะ" ด้วยค่าของ "จัดกำหนดการแล้ว" โดยใช้ตัวดำเนินการตัวกรอง "ตรงทุกประการ"

## <a name="create-a-schedule"></a>สร้างกำหนดการ
1. ในรายการนี้ ทำเครื่องหมายหรือยกเลิกการทำเครื่องหมายทุกแถว
2. ในบานหน้าต่างการดำเนินการ คลิก กำหนดการ
3. คลิกกำหนดเวลางาน
4. ในฟิลด์คำสั่งการกำหนดเวลา เลือก 'ย้อนหลังจากวันจัดส่ง'
5. เลือก ไม่ใช่ ในฟิลด์กำลังการผลิตที่มีจำกัด
6. เลือก ไม่ใช่ ในฟิลด์วัตถุดิบที่มีจำกัด
7. คลิก ตกลง
    * การดำเนินการนี้อาจใช้เวลาสักครู่  

## <a name="view-the-result-of-scheduled-production-orders"></a>ดูผลลัพธ์ของใบสั่งผลิตที่จัดกำหนดการแล้ว
1. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
    * คุณสามารถทำเครื่องหมายแถวใดๆก็ได้  
2. ในบานหน้าต่างการดำเนินการ ให้คลิก ใบสั่งผลิต
3. คลิกงานทั้งหมด
    * ในหน้านี้ คุณสามารถดูรายการของงานได้  ในแท็บการจัดกำหนดการ คุณสามารถดูวันที่เริ่มต้นและวันที่สิ้นสุดสำหรับงานได้  
4. คลิกวัสดุ
    * ในหน้านี้ คุณสามารถดูปริมาณการใช้วัสดุที่ประเมินได้ สำหรับการดำเนินการในใบสั่งและสินค้าคงคลังที่พร้อมใช้งานปัจจุบัน
