# <a name="generate-a-constrained-plan"></a>จัดทำแผนที่มีข้อจำกัด

[!include [task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงวิธีการสร้างแผนที่พิจารณาทั้งข้อจำกัดของวัสดุและกำลังการผลิต  แผนช่วยให้มั่นใจว่าจะไม่เริ่มการผลิตก่อนที่จะมีวัสดุพร้อมใช้งานและทรัพยากรจะไม่ถูกจองเกินเวลา  

ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF กระบวนงานนี้มีไว้สำหรับผู้วางแผนการผลิต


## <a name="set-up-a-constrained-plan"></a>ตั้งค่าแผนที่มีข้อจำกัด
1. คลิกที่งานการวางแผนหลัก 
2. คลิกที่แผนหลัก
3. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * ตัวอย่างเช่น StaticPlan  
4. เลือกใช่ในฟิลด์กำลังการผลิตที่มีจำกัด
5. ในฟิลด์กรอบกำลังการผลิตที่มีจำกัด ให้ป้อน '30'
6. ขยายกรอบเวลาในส่วนของวัน
7. เลือกใช่ในฟิลด์กำลังการผลิต
8. ในฟิลด์กรอบเวลากำลังการผลิต(วัน) ให้ป้อนตัวเลข
    * ตัวอย่างเช่น 60  
9. เลือกใช่ในฟิลด์ความล่าช้าที่ถูกคำนวณ
10. ในฟิลด์กรอบเวลากำลังการผลิต (วัน) ให้ป้อนตัวเลข
    * ตัวอย่างเช่น 60  
11. ขยายส่วนความล่าช้าที่คำนวณได้
12. เลือกใช่ในฟิลด์เพิ่มความล่าช้าที่คำนวณไปยังวันที่ของความต้องการ
13. เลือกใช่ในฟิลด์เพิ่มความล่าช้าที่คำนวณไปยังวันที่ของความต้องการ
14. เลือกใช่ในฟิลด์เพิ่มความล่าช้าที่คำนวณไปยังวันที่ของความต้องการ
15. ปิดหน้า

## <a name="create-a-constrained-plan"></a>สร้างแผนที่มีข้อจำกัด
1. คลิก เรียกใช้
2. ในฟิลด์แผนหลัก ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
    * เลือกแผนสำหรับข้อจำกัดซึ่งคุณได้ตั้งค่าไว้  
3. คลิก ตกลง
    * การดำเนินการนี้อาจใช้เวลาสักครู่  
4. คลิกที่แผนการใบสั่ง
