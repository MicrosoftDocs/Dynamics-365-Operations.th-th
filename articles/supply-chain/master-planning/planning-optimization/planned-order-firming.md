# <a name="firm-planned-orders"></a><span data-ttu-id="77502-104">ยืนยันแผนการใบสั่ง</span><span class="sxs-lookup"><span data-stu-id="77502-104">Firm planned orders</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="77502-105">แผนการใบสั่งต้อง *ยืนยัน* (นั่นคือ นำออกใช้) โดยเป็นส่วนหนึ่งของกระบวนการวางแผนหลัก</span><span class="sxs-lookup"><span data-stu-id="77502-105">Planned orders must be *firmed* (that is, released) as part of the master planning process.</span></span> <span data-ttu-id="77502-106">เมื่อมีการยืนยันแผนการใบสั่ง ใบสั่งจะกลายเป็นใบสั่งซื้อจริง ใบสั่งโอนย้าย หรือใบสั่งผลิต</span><span class="sxs-lookup"><span data-stu-id="77502-106">When planned orders are firmed, they become actual purchase orders, transfer orders, or production orders.</span></span> <span data-ttu-id="77502-107">ใบสั่งเหล่านี้ยังสามารถเรียกได้ว่า *ใบสั่งนำออกใช้* หรือ *ใบสั่งที่เปิด*</span><span class="sxs-lookup"><span data-stu-id="77502-107">These orders are also known as *released orders* or *open orders*.</span></span>

<span data-ttu-id="77502-108">การยืนยันแผนการใบสั่งมีอยู่สามวิธีดังนี้:</span><span class="sxs-lookup"><span data-stu-id="77502-108">There are three methods for firming planned orders:</span></span>

- <span data-ttu-id="77502-109">**การยืนยันด้วยตนเอง** – เลือกแผนการใบสั่งเฉพาะในรายการ แล้วเริ่มต้นกระบวนการด้วยตนเอง</span><span class="sxs-lookup"><span data-stu-id="77502-109">**Manual firming** – Select specific planned orders in a list, and then manually start the process.</span></span>
- <span data-ttu-id="77502-110">**การยืนยันอัตโนมัติ** – กําหนดกรอบเวลาการยืนยันเริ่มต้นให้กับกลุ่มความครอบคลุม สินค้าแต่ละรายการ และชุดของสินค้าและแผนหลัก</span><span class="sxs-lookup"><span data-stu-id="77502-110">**Auto-firming** – Define a default firming time fence for coverage groups, individual items, and combinations of items and master plans.</span></span> <span data-ttu-id="77502-111">จากนั้นในระหว่างรันการวางแผนหลัก แผนการใบสั่งจะมีการยืนยันโดยอัตโนมัติ ถ้าวันที่ในใบสั่งอยู่ภายในกรอบเวลาที่ระบุเพื่อยืนยัน</span><span class="sxs-lookup"><span data-stu-id="77502-111">Then, during master planning runs, planned orders will automatically be firmed if the order date is within the specified time fence for firming.</span></span>
- <span data-ttu-id="77502-112">**การยืนยันโดยใช้การสอบถาม** – กําหนดการสอบถามเพื่อเลือกแผนการใบสั่งตามคุณสมบัติของใบสั่ง</span><span class="sxs-lookup"><span data-stu-id="77502-112">**Query-based firming** – Define a query to select planned orders based on their properties.</span></span> <span data-ttu-id="77502-113">คุณสามารถตั้งค่าชุดงานเพื่อรันการสอบถามและยืนยันการจับคู่ใบสั่งในการจัดตารางการผลิตปกติได้</span><span class="sxs-lookup"><span data-stu-id="77502-113">You can set up a batch job to run the query and firm matching orders on a regular schedule.</span></span>

<span data-ttu-id="77502-114">หัวข้อนี้จะอธิบายแต่ละวิธีการโดยละเอียด</span><span class="sxs-lookup"><span data-stu-id="77502-114">This topic describes each method in detail.</span></span>
