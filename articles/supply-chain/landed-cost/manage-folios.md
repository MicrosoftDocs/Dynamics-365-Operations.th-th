# <a name="manage-folios"></a>จัดการใบแจ้งรายการ

[!include [banner](../../includes/banner.md)]

ใบแจ้งรายการมักจะกําหนดโดยข้อบังคับของศุลกากร ซึ่งอาจประกอบด้วยสินค้าของผู้จัดจำหน่ายสำหรับหนึ่งองค์กรหรือบริษัทต่อการจัดส่ง สินค้าในใบแจ้งรายการสามารถอยู่ในหนึ่งคอนเทนเนอร์ หรือสามารถกระจายไปหลายคอนเทนเนอร์ได้

เมื่อต้องการเปิดหน้า **ใบแจ้งรายการทั้งหมด** ให้ไปที่ **ต้นทุนแฝง \> ใบแจ้งรายการ \> ใบแจ้งรายการทั้งหมด** หน้านี้แสดงรายการใบแจ้งรายการปัจจุบันทั้งหมด คุณสามารถใช้ปุ่มต่างๆ ในบานหน้าต่างการดำเนินการเพื่อสร้าง ลบ และทำงานกับใบแจ้งรายการได้ เลือกใบแจ้งรายการในรายการเพื่อดูรายละเอียดเกี่ยวกับหน้า **ใบแจ้งรายการ**

## <a name="action-pane"></a>บานหน้าต่างการดำเนินการ

บานหน้าต่างการดำเนินการบนหน้า **ใบแจ้งรายการทั้งหมด** และ **ใบแจ้งรายการ** จะมีปุ่มที่ให้คุณสามารถใช้งานใบแจ้งรายการที่เลือกได้ แต่ละปุ่มจะปฏิบัติการเดียว บานหน้าต่างการดำเนินการยังรวมแท็บต่างๆ ซึ่งจะมีชุดปุ่มที่เกี่ยวข้อง ปุ่มและแท็บทั้งหมดที่อธิบายไว้ในส่วนย่อยต่อไปนี้จะพร้อมใช้งานทั้งในมุมมองรายการ (กล่าวคือ หน้า **ใบแจ้งรายการทั้งหมด** ) และในมุมมองโดยละเอียด (กล่าวคือ หน้า **ใบแจ้งรายการ** )

### <a name="buttons-that-appear-directly-on-the-action-pane"></a>ปุ่มที่จะปรากฏโดยตรงในบานหน้าต่างการดำเนินการ

ตารางต่อไปนี้อธิบายถึงปุ่มที่พร้อมใช้งานบนบานหน้าต่างการดำเนินการโดยตรง

| ปุ่ม | คำอธิบาย |
|---|---|
| สร้าง  | สร้างใบแจ้งรายการ |
| Delete | ลบใบแจ้งรายการที่เปิดหรือใบแจ้งรายการที่เลือก |
| ต้นทุนการเดินทาง | เปิดหน้า **ต้นทุนการเดินทาง** ที่คุณสามารถดูและเพิ่มต้นทุนระดับใบแจ้งรายการให้กับสินค้าทั้งหมดในการเดินทาง เมื่อต้นทุนใบแจ้งรายการถูกเพิ่มไปยังการเดินทางด้วยตนเอง ต้นทุนการเดินทางจะถูกเพิ่มเข้าในหน้าการสอบถามต้นทุนโดยอัตโนมัติ และแบ่งตามสินค้าตามวิธีการที่ระบุในหน้า **ต้นทุนการเดินทาง** |

### <a name="buttons-on-the-manage-tab"></a>ปุ่มบนแท็บจัดการ

ตารางต่อไปนี้อธิบายถึงปุ่มที่พร้อมใช้งานบนแท็บ **จัดการ** ของบานหน้าต่างการดำเนินการ

| ปุ่ม | คำอธิบาย |
|---|---|
| ลงรายการบัญชีรายการรับสินค้า | ลงรายการบัญชีรายการรับสินค้าจากบรรทัดใบสั่งซื้อทั้งหมดในใบแจ้งรายการ ถ้ามีการใช้การรจัดส่งหลายบริษัท กล่องโต้ตอบการลงรายการบัญชีรายการรับสินค้าใหม่จะถูกเปิดขึ้นของแต่ละบริษัท |
| ลงรายการบัญชีใบรับสินค้า | ลงรายการบัญชีใบรับสินค้าจากบรรทัดใบสั่งซื้อทั้งหมดในใบแจ้งรายการ ถ้ามีการใช้การเดินทางหลายบริษัท กล่องโต้ตอบการลงรายการบัญชีใบรับสินค้าใหม่จะถูกเปิดขึ้นของแต่ละบริษัท |
| ลงรายการบัญชีใบแจ้งหนี้ | ลงรายการบัญชีใบแจ้งหนี้จากบรรทัดใบสั่งซื้อทั้งหมดในใบแจ้งรายการ ถ้ามีการใช้การเดินทางหลายบริษัท กล่องโต้ตอบการลงรายการบัญชีใบแจ้งหนี้ใหม่จะถูกเปิดขึ้นของแต่ละบริษัท |
| ใบสั่งโอนย้ายการจัดส่ง | ลงรายการบัญชีใบสั่งโอนย้ายของรายการใบสั่งโอนย้ายทั้งหมดที่เกี่ยวข้องกับใบแจ้งรายการปัจจุบันในการจัดส่งที่เกี่ยวข้อง |
| รับใบสั่งโอนย้าย | ลงรายการบัญชีการรับสินค้าใบสั่งโอนย้ายของรายการใบสั่งโอนย้ายทั้งหมดที่เกี่ยวข้องกับใบแจ้งรายการปัจจุบันในการจัดส่งที่เกี่ยวข้อง |
| รับสินค้าที่อยู่ระหว่างส่งต่อ | รับบรรทัดใบสั่งทั้งหมดที่อยู่ระหว่างการส่งต่อใบแจ้งรายการ |
| เอกสารที่ได้รับ | อัปเดตการตั้งค่าเพื่อให้ตั้งค่าตัวเลือก **เอกสารที่ได้รับ** เป็น *ใช่* คุณสามารถใช้ปุ่มนี้เพื่อล็อคสินค้าและ/หรือรายการซื้อเพื่อให้ไม่สามารถอัปเดตเพิ่มเติมได้ |
| ค้นหาต้นทุนอัตโนมัติ | ค้นหาต้นทุนการเดินทางที่เกี่ยวข้อง หากพบหรืออัปเดตต้นทุนเหล่านี้แล้ว คุณจะได้รับข้อความต่อไปนี้ :"มีบรรทัดต้นทุนที่ยังไม่ได้ออกใบแจ้งหนี้อยู่ คุณต้องการเขียนทับข้อมูลดังกล่าวหรือไม่ โปรดทราบว่าต้นทุนการเดินทางที่แนบกับใบแจ้งรายการและออกใบแจ้งหนี้แล้วจะไม่บันทึกทับ |
| สร้างสมุดรายวันการมาถึง | สร้างสมุดรายวันการมาถึงขององค์กรได้ โดยใช้คุณลักษณะคลังสินค้าขั้นสูง คุณสามารถเลือกคือ **ปริมาณเริ่มต้น** (แนะนำ) และ **สร้างจากสินค้าที่อยู่ระหว่างการส่งต่อ** และ/หรือ **สร้างจากใบสั่งซื้อ** ตัวเลือกสุดท้ายจะขึ้นอยู่กับว่ามีการใช้การประมวลผลสินค้าที่อยู่ระหว่างส่งต่อหรือไม่ |
| ต้นทุนค้างรับ | รับรู้ต้นทุนโดยชนิดต้นทุนมีรหัสบัญชีที่ระบุไว้เป็นเดบิต โดยทั่วไป ปุ่มนี้จะใช้เมื่อสินค้าคงคลังอยู่ระหว่างการส่งต่อ หรือเมื่อได้รับและออกใบแจ้งหนี้สินค้าแล้ว |

### <a name="buttons-on-the-general-tab"></a>ปุ่มบนแท็บทั่วไป

ตารางต่อไปนี้อธิบายถึงปุ่มที่พร้อมใช้งานบนแท็บ **ทั่วไป** ของบานหน้าต่างการดำเนินการ

| ปุ่ม | คำอธิบาย |
|---|---|
| รายการรับสินค้า | ลงรายการบัญชีรายการรับสินค้าจากบรรทัดใบสั่งซื้อทั้งหมดในใบแจ้งรายการ ถ้ามีการใช้การเดินทางหลายบริษัท กล่องโต้ตอบการลงรายการบัญชีรายการรับสินค้าใหม่จะถูกเปิดขึ้นของแต่ละบริษัท |
| ใบรับสินค้า | ดูเรกคอร์ดใบรับสินค้าถ้ามีการใช้อยู่ |
| การมาถึงของสินค้า | ดูสมุดรายวันสินค้าที่มาถึง ถ้ามีการใช้ |
| การสอบถามต้นทุน | เปิดหน้าการสอบถามต้นทุนเพื่อดูต้นทุนทั้งหมดของการเดินทาง รวมถึงคอนเทนเนอร์การจัดส่ง ใบแจ้งรายการ และใบสั่งซื้อ คุณสามารถปรับปรุงมุมมองทั้งหมดของหน้าได้โดยการใช้การดำเนินการดู บนหน้าการสอบถามต้นทุน คุณสามารถดูพื้นที่ใด ๆ รวมทั้งรหัสชนิดสินค้าและต้นทุนได้ โดยการนำสินค้าเหล่านี้ออก คุณสามารถปรับปรุงหน้าด้วยการจัดกลุ่มต้นทุนเข้าด้วยกันได้ ความสามารถนี้มีประโยชน์ถ้าคุณจะใช้ขนาดและสี คุณสามารถเปลี่ยนมิติที่แสดงบนหน้าได้ หน้า **ต้นทุน** แสดงเฉพาะรหัสชนิดของต้นทุนที่มีรายการ **เดบิต** บนแท็บ **การลงรายการบัญชี** ตั้งค่าเป็น *สินค้า* |

## <a name="header-view"></a>มุมมองหัวข้อ

เพื่อเปิดมุมมอง **หัวข้อ** เปิดใบแจ้งรายการ แล้วเลือกแท็บ **หัวข้อ** ในด้านขวาบนของหัวข้อใบแจ้งรายการ

### <a name="settings-on-the-general-fasttab"></a>การตั้งค่าบนแท็บด่วนทั่วไป

ตารางต่อไปนี้อธิบายฟิลด์ที่พร้อมใช้งานบนแท็บด่วน **ทั่วไป** ในมุมมอง **หัวข้อ** ของใบแจ้งรายการ

| ฟิลด์ | คำอธิบาย |
|---|---|
| ใบแจ้งรายการ | ชื่อของใบแจ้งรายการ ชื่อนี้งจะสร้างขึ้นโดยอัตโนมัติเมื่อสร้างใบแจ้งรายการ|
| การเดินทาง | การเดินทางที่เชื่อมโยงกับใบแจ้งรายการ |
| นายหน้าศุลกากร | เลือกนายหน้าศุลกากรของใบแจ้งรายการ นายหน้าศุลกากรจะถูกกําหนดให้กับผู้จัดจำหน่าย ซึ่งช่วยให้สามารถกําหนดต้นทุนที่สร้างขึ้นโดยอัตโนมัติได้ |
| ใบตราส่งสินค้าทางอากาศ/ใบตราส่ง | ระบุใบตราส่งสินค้าทางอากาศหรือใบตราส่งที่ใช้กับใบแจ้งรายการ |
| บริษัท | นิติบุคคล (บริษัท) ที่สัมพันธ์กับใบแจ้งรายการ |
| หมายเลขควบคุมการขนส่งสินค้า | ฟิลด์นี้จะใช้โดยแผนกศุลกากรในบางประเทศหรือภูมิภาค |
| การวัด | ฟิลด์นี้เปิดใช้งานการวัดที่จะระบุในโมดูล **ต้นทุนแฝง** การวัดมักใช้โดยองค์กรที่ไม่ทราบปริมาตรหรือน้ำหนักของสินค้าแต่ละรายการ แต่ต้องมีการปรับตามสัดส่วนที่ถูกต้องมากกว่าตัวเลือก จำนวน และ ปริมาณที่มีให้ ผู้ขนส่งสินค้าจะระบุการวัดน้ำหนักหรือลูกบาศก์ และคุณสามารถใส่ในระดับสินค้าหรือระดับใบสั่งซื้อ คุณสามารถอัปเดตพารามิเตอร์โดยอัตโนมัติได้ถ้าเลือกพารามิเตอร์ไว้ หรือสามารถป้อนพารามิเตอร์ด้วยตนเองได้ |
| หน่วยการวัด | หน่วยที่ใช้กับการวัดที่ระบุ |
| จำนวนกล่อง | จํานวนกล่องกระดาษในใบแจ้งรายการ ฟิลด์นี้สามารถอัปเดตโดยอัตโนมัติได้ ขึ้นอยู่กับการเลือกพารามิเตอร์ |
| บัญชีผู้จัดจำหน่าย | เลือกผู้จัดจำหน่ายที่สัมพันธ์กับใบแจ้งรายการ ฟิลด์นี้ใช้เพื่อให้ข้อมูลเท่านั้น จะไม่มีผลกระทบต่อฟังก์ชันใดๆ |
| ชื่อ | ชื่อของบัญชีผู้จัดจำหน่ายที่เลือก |
| หมายเหตุ | ป้อนข้อมูลเพิ่มเติมใดๆ ที่เกี่ยวข้องกับใบแจ้งรายการ |
| คำอธิบายเกี่ยวกับสินค้า | เลือกข้อความอธิบายสินค้าเพื่อช่วยระบุใบแจ้งรายการ สำหรับข้อมูลเพิ่มเติม โปรดดู [คำอธิบายเกี่ยวกับสินค้า](shipping-information-setup.md#description-of-goods) |
| วันที่ประเมินค่า | ฟิลด์นี้จะเกี่ยวข้องกับหน้ารายการภาษี โมดูล **ต้นทุนแฝง** จะใช้อัตราแลกเปลี่ยนของศุลกากรเป็นวันที่ที่คุณตั้งค่าไว้ที่นี่ ค่าเริ่มต้นคือวันที่ในหน้ารายการภาษี |
| รหัสศุลกากร | ป้อนรหัสศุลกากร แผนกศุลกากรในประเทศหรือภูมิภาคจะระบุรหัสนี้ |
| รหัสภาษี | ป้อนรหัสภาษีที่จะเชื่อมโยงกับใบแจ้งรายการ โดยทั่วไป รหัสนี้จะถูกกําหนด (และกําหนด) ตามประเทศหรือภูมิภาคที่คุณจัดส่งสินค้าไป |

### <a name="settings-on-the-delivery-fasttab"></a>การตั้งค่าบนแท็บด่วนการจัดส่ง

ตารางต่อไปนี้อธิบายการตั้งค่าที่พร้อมใช้งานบนแท็บด่วน **การจัดส่ง** ในมุมมอง **หัวข้อ** ของใบแจ้งรายการ

| ฟิลด์ | คำอธิบาย |
|---|---|
| วันที่ใบแจ้งรายการ | เลือกวันที่ที่จะเชื่อมโยงกับใบแจ้งรายการ ค่าเริ่มต้นคือวันที่การสร้างของการเดินทาง |
| ETA ที่ท่าเรือขนส่งสินค้า | วันที่การมาถึงโดยประมาณ (ETA) ที่ท่าเรือปลายทาง (ท่าเรือ "สุดท้าย") |
| วันที่จัดส่งโดยประมาณ | โดยปกติ เป็นวันที่สินค้าครบกําหนดมาถึงในคลังสินค้า ฟิลด์นี้จะไม่ใช้เมื่อคํานวณวันที่จัดส่งโดยประมาณ (ใช้วันที่จัดส่งโดยประมาณของตัวควบคุมการติดตามแทน) เมื่อต้องการตั้งค่าฟิลด์นี้เพื่อให้ค่าตรงกับวันที่จัดส่งโดยประมาณการควบคุมการติดตาม ให้ใช้ [ศูนย์ควบคุมการติดตาม](delivery-information-setup.md#tracking-control-center) |
| ได้รับเอกสารต้นฉบับแล้ว | วันที่ที่ได้รับเอกสารต้นฉบับ |
| นายหน้าแนะนำแล้ว | วันที่ที่นายหน้าแนะนำ |
| ส่งต้นฉบับใบตราส่งแล้ว | วันที่ที่ส่งใบตราส่งต้นฉบับ |
| นำสินค้าออกใช้แล้ว | วันที่ที่ปล่อยสินค้าออกใช้ |
| การนัดหมายลูกค้า | วันที่นัดหมายของลูกค้า |
| จัดส่งสินค้าที่คลังสินค้าแล้ว | วันที่ที่จัดส่งสินค้าไปยังคลังสินค้า |
| วันที่ตรวจสอบความถูกต้อง | วันที่ตรวจสอบความถูกต้อง |
| คำแนะนำในการจัดส่ง | วันที่ที่ได้รับคำแนะนำในการจัดส่ง |
| จากท่าเรือ | ท่าเรือที่เดินทางมา |
| ไปยังท่าเรือ | ท่าเรือที่เดินทางมาถึง คอนเทนเนอร์การจัดส่งสามารถมีท่าเรือที่แตกต่างกันได้ เนื่องจากการจัดส่งอาจหยุดที่ท่าเรือหลายท่า |

### <a name="settings-on-the-export-fasttab"></a>การตั้งค่าบนแท็บด่วนส่งออก

ตารางต่อไปนี้อธิบายการตั้งค่าที่พร้อมใช้งานบนแท็บด่วน **การส่งออก** ในมุมมอง **หัวข้อ** ของใบแจ้งรายการ

| ฟิลด์ | คำอธิบาย |
|---|---|
| ผู้ส่งออก | ผู้ส่งออกสามารถจัดเก็บในใบแจ้งรายการได้ ในการค้าระหว่างประเทศ คุณอาจส่งใบสั่งซื้อให้กับบริษัทหนึ่ง แต่ได้รับสินค้าจากบริษัทอื่น การติดตามและเอกสารเป็นข้อมูลที่ต้องระบุโดยศุลกากร สามารถจัดเก็บชื่อและที่อยู่ผู้ส่งออกที่นี่ได้ |
| ชื่อ | ชื่อของผู้ส่งออกที่เลือก |

## <a name="lines-view"></a>มุมมองรายการ

เพื่อเปิดมุมมอง **รายการ** เปิดใบแจ้งรายการ แล้วเลือกแท็บ **รายการ** ในด้านขวาบนของหัวข้อใบแจ้งรายการ

### <a name="information-on-the-folio-fasttab"></a>ข้อมูลบนแท็บด่วนใบแจ้งรายการ

แท็บด่วน **ใบแจ้งรายการ** ในมุมมอง **รายการ** แสดงข้อมูลเกี่ยวกับใบแสดงรายการ ข้อมูลนี้ส่วนใหญ่ยังปรากฏในมุมมอง **หัวข้อ** ด้วย ตามที่อธิบายไว้ก่อนหน้านี้ในหัวข้อนี้

### <a name="information-and-buttons-on-the-lines-fasttab"></a>ข้อมูลและปุ่มบนแท็บด่วนรายการ

แท็บด่วน **รายการ** ในมุมมอง **รายการ** แสดงรายละเอียดเกี่ยวกับรายการใบสั่งซื้อแบบเต็มหรือบางส่วนแต่ละรายการ ที่รวมอยู่ในใบแจ้งรายการปัจจุบัน

ตารางต่อไปนี้อธิบายถึงปุ่มที่พร้อมใช้งานบนแท็บด่วน **รายการ** ในมุมมอง **รายการ**

| ปุ่ม | คำอธิบาย |
|---|---|
| ลบออก | นำรายการใบสั่งซื้อที่เลือกออกจากการเดินทาง |
| สินค้าคงคลัง \> ธุรกรรม | ดูรายการความเคลื่อนไหวของสินค้าคงคลังของรายการใบสั่งซื้อที่เลือก หมายเหตุว่า ถ้าคุณไม่ได้ใช้สินค้าที่อยู่ระหว่างส่งต่อ ระบบจะแสดงใบสั่งต้นฉบับและใบสั่งสินค้าที่อยู่ระหว่างส่งต่อด้วย |
| สินค้าคงคลัง \> แสดงมิติ | เปิดกล่องโต้ตอบซึ่งคุณสามารถเลือกมิติสินค้าคงคลังที่แสดงไว้สำหรับธุรกรรมที่คุณดู |
| รีเฟรช | อัปเดตข้อมูลที่เกี่ยวข้องกับยอดเงินต่อรายการ น้ําหนัก หรือปริมาตรของรายการใบสั่งซื้อที่เลือก |

### <a name="information-on-the-lines-details-fasttab"></a>ข้อมูลบนแท็บด่วนรายละเอียดรายการ

แท็บด่วน **รายละเอียดรายการ** ในมุมมอง **รายการ** จะให้รายละเอียดเกี่ยวกับรายการใบสั่งซื้อที่เลือกในปัจจุบันบนแท็บด่วน **รายการ**
