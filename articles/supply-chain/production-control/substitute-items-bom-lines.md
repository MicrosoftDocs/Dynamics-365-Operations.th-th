<span data-ttu-id="2e258-104">หัวข้อนี้อธิบายวิธีการทดแทนของวัตถุดิบในระหว่างกระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="2e258-104">This topic describes how to substitute materials during the production process.</span></span>

## <a name="substituting-material-by-date"></a><span data-ttu-id="2e258-109">วัตถุดิบทดแทนตามวันที่</span><span class="sxs-lookup"><span data-stu-id="2e258-109">Substituting material by date</span></span>
<span data-ttu-id="2e258-110">พิจารณาสถานการณ์จำลองต่อไปนี้: เครื่องจักรที่บริษัทที่มีการผลิตประกอบด้วยส่วนประกอบที่จะหมดอายุจากแค็ตตาล็อกของผู้จัดจำหน่ายในสองเดือน</span><span class="sxs-lookup"><span data-stu-id="2e258-110">Consider following scenario: A machine that a company is manufacturing contains a component that will expire from the vendor's catalog in two months.</span></span> <span data-ttu-id="2e258-111">จากวันหมดอายุเป็นต้นไป ผู้จัดจำหน่ายจะเสนอส่วนประกอบใหม่ที่สามารถใช้ทดแทนสำหรับส่วนประกอบเก่าได้</span><span class="sxs-lookup"><span data-stu-id="2e258-111">From the expiration date onward, the vendor will offer a new component that can be substituted for the old component.</span></span> <span data-ttu-id="2e258-112">วันที่เริ่มต้นและวันที่มีผลบังคับใช้สามารถตั้งค่าได้ในรายการสูตรการผลิต (BOM)</span><span class="sxs-lookup"><span data-stu-id="2e258-112">From and to validity dates can be set up on bill of materials (BOM) lines.</span></span> <span data-ttu-id="2e258-113">ตัวอย่างนี้ คุณตั้งค่าส่วนประกอบเก่าให้หมดอายุโดยการป้อนวันหมดอายุในฟิลด์ **วันที่สิ้นสุด**</span><span class="sxs-lookup"><span data-stu-id="2e258-113">For this example, you set up the old component to expire by entering the expiration date in the **To-date** field.</span></span> <span data-ttu-id="2e258-114">จากนั้น บน BOM คุณตั้งค่าส่วนประกอบที่เปลี่ยนใหม่ เพื่อให้มีผลบังคับใช้ตั้งแต่วันหลังจากส่วนประกอบเก่าหมดอายุ</span><span class="sxs-lookup"><span data-stu-id="2e258-114">Then, on the BOM, you set up the new, replacement component so that it's valid from the day after the old component expires.</span></span> <span data-ttu-id="2e258-115">การทำเช่นนี้ ให้ป้อนวันที่เริ่มต้นในฟิลด์ **วันที่เริ่มต้น**</span><span class="sxs-lookup"><span data-stu-id="2e258-115">To do this, enter the start date in the **From-date** field.</span></span>

## <a name="substituting-material-by-planning"></a><span data-ttu-id="2e258-116">วัตถุดิบทดแทนตามการวางแผน</span><span class="sxs-lookup"><span data-stu-id="2e258-116">Substituting material by planning</span></span>

คุณสร้างรายการสูตรสำหรับสามวัตถุดิบ และเชื่อมโยงรายการสูตรกับกลุ่มแผนเดียวกัน</span><span class="sxs-lookup"><span data-stu-id="2e258-122">For this example, you create formula lines for the three
