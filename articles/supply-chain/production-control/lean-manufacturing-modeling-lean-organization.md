## <a name="production-flows-in-lean-manufacturing"></a><span data-ttu-id="f6dec-108">ขั้นตอนการผลิตในการผลิตแบบ Lean</span><span class="sxs-lookup"><span data-stu-id="f6dec-108">Production flows in lean manufacturing</span></span>
<span data-ttu-id="f6dec-109">ในสถานการณ์จำลองการผลิตตามใบสั่งผลิต วัสดุจะออกให้สำหรับใบสั่งผลิตเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="f6dec-109">In production scenarios that are based on production orders, material is issued to a specific production order.</span></span> <span data-ttu-id="f6dec-110">ในระหว่างลำดับของการดำเนินงาน ขึ้นอยู่กับสูตรการผลิต (BOM) และกระบวนการผลิต ผลิตภัณฑ์จะถูกสร้างและสุดท้ายจะได้รับในสถานที่ที่จัดให้</span><span class="sxs-lookup"><span data-stu-id="f6dec-110">During a sequence of operations that is based on a bill of materials (BOM) and routes, products are created and are finally received at the supplied location.</span></span> <span data-ttu-id="f6dec-111">เวลาที่สามารถประมวลผลได้ของใบสั่งผลิตแตกต่างกันไปตั้งแต่นาทีถึงสัปดาห์</span><span class="sxs-lookup"><span data-stu-id="f6dec-111">The throughput time of production orders varies from minutes to weeks.</span></span> <span data-ttu-id="f6dec-112">ต้นทุนที่เกี่ยวข้องทั้งหมด วัสดุ และแรงงานจะถูกสะสมในใบสั่งผลิต</span><span class="sxs-lookup"><span data-stu-id="f6dec-112">All related cost, material, and labor are accumulated on the production order.</span></span>

<span data-ttu-id="f6dec-116">เพื่อสร้างการผลิตและบริบทต้นทุนใหม่สำหรับสถานการณ์คัมบังที่หลากหลายที่ถูกเสนอ ขั้นตอนการผลิตตามกิจกรรมได้ถูกแนะนำเป็นแกนหลักของ lean manufacturing</span><span class="sxs-lookup"><span data-stu-id="f6dec-116">To re-establish a production and cost context for the various kanban scenarios that are proposed, activity-based production flows have been introduced as the backbone of lean manufacturing.</span></span> <span data-ttu-id="f6dec-117">กฎคัมบังทั้งหมดอ้างอิงถึงโครงสร้างที่กำหนดไว้ล่วงหน้านี้</span><span class="sxs-lookup"><span data-stu-id="f6dec-117">All kanban rules refer to this predefined structure.</span></span> <span data-ttu-id="f6dec-118">แบบจำลองตามกิจกรรมสนับสนุนการตั้งค่าของสถานการณ์จำลองที่หลากหลาย</span><span class="sxs-lookup"><span data-stu-id="f6dec-118">The activity-based model supports the setup of a wide range of scenarios.</span></span> <span data-ttu-id="f6dec-119">อย่างไรก็ตาม รุ่นนี้ไม่เพิ่มความซับซ้อนสำหรับผู้ปฏิบัติงานการผลิต เนื่องจากสถานการณ์จำลองทั้งหมดใช้อินเทอร์เฟสผู้ใช้ตามกิจกรรมเดียวกัน</span><span class="sxs-lookup"><span data-stu-id="f6dec-119">However, this model doesn't add complexity for the shop floor workers, because all scenarios use the same activity-based user interface.</span></span>

## <a name="semi-finished-products-non-bom-levels"></a><span data-ttu-id="f6dec-120">ผลิตภัณฑ์กึ่งสำเร็จรูป (ไม่ใช่ระดับ BOM)</span><span class="sxs-lookup"><span data-stu-id="f6dec-120">Semi-finished products (non-BOM levels)</span></span>
