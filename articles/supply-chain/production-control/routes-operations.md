<span data-ttu-id="82b71-330">ถ้าคุณยังแสดงความต้องการทรัพยากรในแง่ของทักษะและความสามารถ และทำให้กระบวนการผลิตของคุณไม่ขึ้นอยู่กับไซต์ จะช่วยลดการบำรุงรักษาอย่างต่อเนื่องของกระบวนการทางธุรกิจของคุณให้น้อยที่สุดได้</span><span class="sxs-lookup"><span data-stu-id="82b71-330">If you also express resource requirements in terms of skills and capabilities, and make your routes site-independent, you can help keep the ongoing maintenance of your business processes to a minimum.</span></span>  

<span data-ttu-id="82b71-331">เมื่อคุณใช้วิธีการนี้ หน้า **ความสัมพันธ์ของการดำเนินงาน** จะกลายเป็นปลายทางหลักสำหรับการรักษารันไทม์และคุณสมบัติอื่น ๆ ของคุณ</span><span class="sxs-lookup"><span data-stu-id="82b71-331">When you use this approach, the **Operation relations** page becomes your primary destination for maintaining run times and other properties.</span></span>

### <a name="resource-specific-process-times"></a><span data-ttu-id="82b71-332">ระยะเวลาของกระบวนการเฉพาะทรัพยากร</span><span class="sxs-lookup"><span data-stu-id="82b71-332">Resource-specific process times</span></span>

<span data-ttu-id="82b71-333">ถ้าคุณไม่ได้ระบุทรัพยากรการดำเนินงานหรือกลุ่มทรัพยากรเป็นส่วนหนึ่งของความต้องการทรัพยากรสำหรับการดำเนินงาน ทรัพยากรที่ใช้ได้อาจทำงานด้วยความเร็วที่แตกต่างกัน</span><span class="sxs-lookup"><span data-stu-id="82b71-333">If you don't specify an operations resource or resource group as part of the resource requirements for an operation, the applicable resources might operate at different speeds.</span></span> <span data-ttu-id="82b71-334">ดังนั้น เวลาที่จำเป็นต้องใช้ในการประมวลผลการดำเนินงานอาจแตกต่างกันไป</span><span class="sxs-lookup"><span data-stu-id="82b71-334">Therefore, the time that is required in order to process an operation might vary.</span></span> <span data-ttu-id="82b71-335">เมื่อต้องการแก้ไขปัญหานี้ คุณสามารถใช้ฟิลด์ **สูตร** ในความสัมพันธ์ของการดำเนินงานเพื่อระบุวิธีคำนวณเวลาที่ใช้ในกระบวนการ</span><span class="sxs-lookup"><span data-stu-id="82b71-335">To resolve this issue, you can use the **Formula** field on the operation relation to specify how the process time is calculated.</span></span> <span data-ttu-id="82b71-336">มีตัวเลือกดังต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="82b71-336">The following options are available:</span></span>

-   <span data-ttu-id="82b71-337">**มาตรฐาน** – (ตัวเลือกเริ่มต้น) การคำนวณใช้เฉพาะฟิลด์จากความสัมพันธ์การดำเนินงาน และคูณรันไทม์ที่กำหนดด้วยปริมาณในใบสั่ง</span><span class="sxs-lookup"><span data-stu-id="82b71-337">**Standard** – (Default option) The calculation uses only the fields from the operation relation and multiplies the specified run time by the order quantity.</span></span>
-   <span data-ttu-id="82b71-338">**กำลังการผลิต** – การคำนวณรวมฟิลด์ **กำลังการผลิต** จากทรัพยากรการดำเนินงาน</span><span class="sxs-lookup"><span data-stu-id="82b71-338">**Capacity** – The calculation includes the **Capacity** field from the operations resource.</span></span> <span data-ttu-id="82b71-339">ดังนั้น เวลาจะขึ้นอยู่กับทรัพยากร</span><span class="sxs-lookup"><span data-stu-id="82b71-339">Therefore, the time is resource-dependent.</span></span> <span data-ttu-id="82b71-340">ค่าที่ระบุในทรัพยากรการดำเนินงานคือกำลังการผลิตต่อชั่วโมง</span><span class="sxs-lookup"><span data-stu-id="82b71-340">The value that is specified on the operations resource is capacity per hour.</span></span> <span data-ttu-id="82b71-341">ค่านี้ถูกคูณด้วยปริมาณในใบสั่งและค่า **ตัวคูณ** จากความสัมพันธ์ของการดำเนินงาน</span><span class="sxs-lookup"><span data-stu-id="82b71-341">This value is multiplied by the order quantity and the **Factor** value from the operation relation.</span></span>
-   <span data-ttu-id="82b71-342">**ชุด** – กำลังการผลิตชุดงานจะถูกคำนวณโดยใช้ข้อมูลจากความสัมพันธ์ของการดำเนินงาน</span><span class="sxs-lookup"><span data-stu-id="82b71-342">**Batch** – A batch capacity is calculated by using information from the operation relation.</span></span> <span data-ttu-id="82b71-343">หมายเลขชุดงานและเวลาที่ใช้ในกระบวนการจะถูกคำนวณตามปริมาณในใบสั่ง</span><span class="sxs-lookup"><span data-stu-id="82b71-343">The number of batches and, therefore, the process time can then be calculated based on the order quantity.</span></span>
-   <span data-ttu-id="82b71-344">**ชุดงานทรัพยากร** – โดยทั่วไปตัวเลือกนี้จะเหมือนกับตัวเลือก **ชุดงาน**</span><span class="sxs-lookup"><span data-stu-id="82b71-344">**Resource batch** – This option is basically the same as the **Batch** option.</span></span> <span data-ttu-id="82b71-345">อย่างไรก็ตาม การคำนวณรวมฟิลด์ **กำลังการผลิตชุดงาน** จากทรัพยากรการดำเนินงาน</span><span class="sxs-lookup"><span data-stu-id="82b71-345">However, the calculation includes the **Batch capacity** field from the operations resource.</span></span> <span data-ttu-id="82b71-346">ดังนั้น เวลาจะขึ้นอยู่กับทรัพยากร</span><span class="sxs-lookup"><span data-stu-id="82b71-346">Therefore, the time is resource-dependent.</span></span>
