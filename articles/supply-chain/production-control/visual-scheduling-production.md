# <a name="gantt-chart-for-job-scheduling"></a>การใช้แผนภูมิ Gantt สำหรับการจัดตารางการผลิตระดับงาน

[!include [banner](../includes/banner.md)]

แผนภูมิ Gantt มีวัตถุประสงค์เพื่อให้ผู้วางแผนการผลิตสามารถควบคุม และปรับแผนการผลิตให้เหมาะสมได้ แผนภูมิ Gantt ทำให้ขั้นตอนการดำเนินงานโปร่งใส และง่ายที่จะปรับปรุงกำหนดการผลิตในขณะที่พิจารณาการขาดแคลนวัสดุหรือทรัพยากร ซึ่งช่วยทำให้ผู้วางแผนการผลิตใช้ทรัพยากรที่พร้อมใช้งานได้ดีที่สุด ลดงานระหว่างดำเนินการ และเพิ่มประสิทธิภาพสูงสุดให้จำนวนครั้งของปริมาณที่สามารถประมวลผลได้สำหรับใบสั่งผลิต

แผนภูมิ Gantt เป็นการแสดงกิจกรรมที่จัดกำหนดการไว้ภายในช่วงเวลาที่กำหนด กิจกรรมมีการจัดกำหนดการทรัพยากร ที่มีกำลังการผลิตที่กำหนดไว้โดยปฏิทินกำลังการผลิต สามารถแสดงชนิดของกิจกรรมต่อไปนี้ในแผนภูมิ Gantt ได้

-   งานจากใบสั่งผลิตที่เป็นงานที่จัดกำหนดการไว้
-   งานจากการใบสั่งผลิตที่วางแผนไว้
-   งานจัดกำหนดการกิจกรรมโครงการของชนิดการคาดการณ์ชั่วโมงทำงาน

สามารถเปิดแผนภูมิ Gantt ได้ในสองมุมมองที่ต่างกัน **มุมมองใบสั่ง** และ **มุมมองทรัพยากร** ใน **มุมมองใบสั่ง** กิจกรรมจะถูกจัดกลุ่มอยู่ภายใต้ใบสั่งผลิต เช่นนี้อาจมีประโยชน์ ตัวอย่างเช่น ถ้าคุณต้องการรักษาภาพรวมของงานทั้งหมดที่เป็นของใบสั่งเดียวกัน ใน **มุมมองทรัพยากร** งานทั้งหมดจะถูกจัดกลุ่มภายใต้ทรัพยากรแต่ละรายการ มุมมองนี้มีประโยชน์เมื่อเพิ่มประสิทธิภาพแผนในระดับทรัพยากร ตัวอย่างเช่น กลุ่มของเครื่องจักรหรือเครื่องจักร แผนภูมิ Gantt ที่แสดงในภาพประกอบด้านล่างแสดง **มุมมองใบสั่ง** และ **มุมมองทรัพยากร** ที่มีองค์ประกอบหลักเหล่านี้:

1.  กิจกรรมแผนภูมิ Gantt
2.  ไอคอนขาดแคลนวัสดุ
3.  ไอคอนความพร้อมใช้งานของวัสดุ
4.  ไอคอนวันที่จัดส่งสินค้าในใบสั่ง
5.  แถบกำลังการผลิต

## <a name="activities"></a>กิจกรรม
กิจกรรมปรากฏเป็นแถบ และถูกจัดระเบียบในกริดของสเกลเวลาที่มีเวลาเริ่มต้นและเวลาสิ้นสุดที่จัดกำหนดการไว้ ซึ่งทำให้ความยาวของแถบเป็นสัดส่วนกับเวลาที่จำเป็นต้องทำให้กิจกรรมเสร็จสมบูรณ์ กิจกรรมถูกแสดงตามสเกลเวลา คุณสามารถปรับสเกลเวลาบนเมนู ที่ซึ่งคุณเลือกวันที่เริ่มต้นและวันที่สิ้นสุดและหน่วยเวลา ตัวอย่างเช่น ชั่วโมง หรือวัน โดยการปรับสเกลเวลา คุณสามารถตั้งโฟกัสได้ในช่วงเวลาที่คุณต้องการจัดการกิจกรรม 

เพื่อให้ได้ภาพรวมที่ดีขึ้น มีตัวเลือกต่าง ๆ สำหรับการควบคุมสีของกิจกรรม คุณสามารถตั้งค่าคอนฟิกสีแต่ละรายการสำหรับกิจกรรม ใช้สีของชุดรูปแบบที่เป็นชุดรูปแบบสีทั่วไปที่ใช้สำหรับแอพลิเคชัน หรือตั้งค่าสีที่จะถูกควบคุมโดยรหัสสีสำหรับใบสั่งผลิต 

ช่วงเวลาสำหรับกิจกรรมมีการแรเงาพื้นหลัง รอบระยะเวลาที่มีการแรเงาสีขาว บ่งชี้ช่วงเวลาที่มีกำลังการผลิตที่กำหนดไว้ของทรัพยากรสำหรับกิจกรรม ในขณะที่รอบระยะเวลาที่มีการแรเงาสีเทา บ่งชี้ช่วงเวลาที่ไม่มีกำลังการผลิตที่กำหนดไว้ 

ทางด้านซ้ายของแผนภูมิ มีข้อมูลเพิ่มเติมเกี่ยวกับกิจกรรม ตัวอย่างเช่น ทรัพยากรที่ซึ่งมีการจัดกำหนดการกิจกรรมและหมายเลขใบสั่งผลิต มีแสดงการเชื่อมต่อระหว่างงานที่เป็นของใบสั่งเดียวกันด้วยลูกศร 

คุณสามารถเรียกดูข้อมูลเพิ่มเติมเกี่ยวกับกิจกรรมได้ในกล่องโต้ตอบของกิจกรรม เมื่อต้องการเปิดกล่องโต้ตอบ ให้ดับเบิลคลิกกิจกรรม หรือเลือกเมนู **ข้อมูล** ในกล่องโต้ตอบของกิจกรรม คุณสามารถดูวันที่เริ่มต้นและวันที่สิ้นสุดที่ถูกจัดกำหนดไว้ และข้อมูลเวลาเกี่ยวกับวัสดุที่กิจกรรมถูกวางแผนว่าจะใช้ 

กิจกรรมสามารถถูกจัดกลุ่มได้ในระดับกลุ่ม ระดับการจัดกลุ่มเป็นลำดับชั้น และสามารถถูกใช้เพื่อทำการจัดกลุ่มทางตรรกะของกิจกรรมได้ ตัวอย่างเช่น ถ้าคุณมีเค้าโครงที่กิจกรรมการผลิตจะถูกจัดกลุ่มตามไซต์ หน่วยการผลิต กลุ่มทรัพยากร และทรัพยากร คุณสามารถใช้ระดับการจัดกลุ่มเพื่อจัดกลุ่มกิจกรรมตามเค้าโครงนั้นได้ สามารถถขยายและยุบระดับการจัดกลุ่มในระดับการจัดกลุ่มแต่ละคน หรือสำหรับระดับทั้งหมดในแผนภูมิได้ โดยใช้ปุ่ม **ขยายทั้งหมด** และ **ยุบทั้งหมด** บนเมนู คุณยังสามารถตั้งค่าคอนฟิกเพื่อขยาย หรือยุบระดับการจัดกลุ่มเมื่อเปิดแผนภูมิได้ด้วย

### <a name="material-availability"></a>ความพร้อมใช้งานของวัสดุ

แผนภูมิ Gantt สามารถถูกตั้งค่า เพื่อให้ผู้วางแผนได้รับข้อมูลรายละเอียดเกี่ยวกับสถานะของวัสดุสำหรับแต่ละกิจกรรม ตัวอย่างเช่น นี่อาจเป็นประโยชน์ถ้าวัตถุดิบล่าช้า และมีผลกระทบต่อแผนการผลิต ในกรณีนี้ การตัดสินค้าจากคลังวัสดุจะถูกเน้นในแผนภูมิ Gantt เพื่อช่วยให้ผู้วางแผนเข้าใจลำดับและทำการปรับปรุงที่จำเป็น 

งานจะปรากฏขึ้นพร้อมกับไอคอนการขาดแคลนวัสดุ ถ้าวันที่เริ่มต้นตามกำหนดการของงานเป็นวันหลังจากวันที่พร้อมใช้งานของวัตถุดิบสำหรับวัสดุที่ใช้โดยงาน วันที่พร้อมใช้งานของวัตถุดิบจะคำนวณตามข้อมูลการเชื่อมโยงความต้องการกับการจัดซื้อในแผนหลักแบบไดนามิก ตัวอย่างเช่น ไอคอนการขาดแคลนวัสดุจะปรากฏขึ้นในงานที่จะใช้วัสดุที่เชื่อมโยงกับการจัดซื้อกับใบสั่งซื้อที่มีการรับสินค้าที่ล่าช้ากว่าวันเริ่มต้นที่วางแผนไว้ของงาน

### <a name="indicator-of-material-availability-date"></a>ตัวบ่งชี้วันที่พร้อมใช้งานของวัสดุ

เมื่อคุณตั้งค่าแผนภูมิเพื่อแสดงงานที่มีการขาดแคลนวัสดุ ยังสามารถแสดงไอคอนสำหรับการแสดงวันที่พร้อมใช้งานของวัสดุสำหรับงานได้ด้วย ไอคอนนี้จะแสดง ถ้าวันที่พร้อมใช้งานของวัสดุอยู่ภายในช่วงเวลาที่กำหนดของแผนภูมิเท่านั้น ถ้าวันที่พร้อมใช้งานของวัสดุอยู่นอกช่วงเวลาที่กำหนด จากนั้นสามารถดึงข้อมูลความพร้อมของวัสดุอย่างละเอียดเพิ่มเติมได้จากรายการวัสดุในกล่องโต้ตอบงาน ในรายการ ยังมีไอคอนที่แสดงวัสดุที่ล่าช้าสำหรับงานอีกด้วย คุณสามารถจัดกำหนดการงานใหม่ได้ โดยใช้วันที่พร้อมใช้งานของวัสดุเป็นวันที่เริ่มต้น

### <a name="indicator-of-order-delivery-date"></a>ตัวบ่งชี้วันที่จัดส่งของใบสั่ง

ไอคอนนี้บ่งชี้วันที่จัดส่งสำหรับใบสั่งผลิต ไอคอนนี้จะมองเห็นได้ในมุมมองใบสั่งเท่านั้น

### <a name="capacity-bar"></a>แถบกำลังการผลิต

คุณสามารถตั้งค่าคอนฟิกแผนภูมิเพื่อแสดงแถบกำลังการผลิตของทรัพยากรได้ แถบนี้แสดงภาพรวมของกำลังการผลิตทรัพยากรสำหรับกิจกรรมในช่วงเวลาที่กำหนดของแผนภูมิ แถบกำลังการผลิตไม่ถูกแสดงสำหรับรอบระยะเวลาของเวลาที่ทรัพยากรไม่ได้ถูกจอง ในรอบระยะเวลาที่มีการจองทรัพยากรให้กับกำลังการผลิต แถบกำลังการผลิตจะแสดงขึ้นเป็นแถบทึบ ในรอบระยะเวลาที่มีการจองทรัพยากรเกินเวลา แถบจะปรากฏหนาขึ้นและเป็นสีแดง ตัวอย่างเช่น ถ้าสองงานทับซ้อน แถบกำลังการผลิตจะบ่งชี้การจองเกินในช่วงเวลาที่มีการทับซ้อน แถบกำลังการผลิตถูกปรับปรุงแบบไดนามิก เมื่อคุณจัดกำหนดการงาน คุณเปิดใช้งานแถบกำลังการผลิตในเมนู **แสดงแถบกำลังการผลิต** มีการแสดงได้ใน **มุมมองทรัพยากร** เท่านั้น ถ้าคุณต้องการเรียกใช้มุมมองโดยละเอียดเพิ่มเติมของการใช้กำลังการผลิตในทรัพยากร ให้ใช้แผนภูมิ **การใช้กำลังการผลิต** ซึ่งสามารถเปิดได้จากเมนูหรือเมนูบริบทสำหรับกิจกรรมที่เลือก

## <a name="job-scheduling-in-the-gantt-chart"></a>การจัดตารางการผลิตระดับงานในแผนภูมิ Gantt
แผนภูมิ Gantt มีตัวเลือกต่าง ๆ สำหรับการทำการปรับปรุงไปยังแผนการผลิต ในแผนภูมิ Gantt คุณสามารถจัดกำหนดการกิจกรรมใหม่เป็นการโต้ตอบแบบลากและปล่อย หรือจากเมนูกำหนดการ ในกระบวนการวางแผน คุณสามารถพิจารณากำลังการผลิตทรัพยากร ความสามารถของทรัพยากร และข้อจำกัดของวัตถุดิบ

### <a name="schedule-a-job-as-a-drag-and-drop-interaction"></a>จัดกำหนดการงานเป็นการโต้ตอบแบบลากและปล่อย

คุณสามารถจัดกำหนดการงานใหม่ได้ภายในช่วงเวลาที่กำหนดให้เป็นการโต้ตอบแบบลากและปล่อย คุณสามารถจัดกำหนดการงานใหม่ได้ในทรัพยากรเดียวกันเท่านั้น และคุณสามารถจัดกำหนดการได้เพียงหนึ่งงานในแต่ละครั้ง

### <a name="schedule-a-job-from-the-menu"></a>จัดกำหนดการงานจากเมนู

ในเมนู **จัดกำหนดการงาน** คุณสามารถจัดกำหนดการงานที่เลือกได้อย่างน้อยหนึ่งรายการ ในแผนภูมิที่ขึ้นอยู่กับทิศทางการจัดกำหนดการและเวลาของวันที่จัดกำหนดการ มีทิศทางที่พร้อมใช้งานอยู่สามแบบ

-   ไปข้างหน้าจากวันที่กำหนดเวลาไว้
-   ย้อนหลังจากวันที่กำหนดเวลาไว้
-   ส่งต่อจากวันที่มีความพร้อมใช้งานของวัสดุ

ไม่สามารถจัดกำหนดการงานนอกช่วงเวลาที่กำหนดของแผนภูมิ Gantt ได้ ถ้าคุณทำเช่นนั้น จะมีการปล่อยงานให้ไม่ได้จัดกำหนดการ และคุณจะได้รับข้อความแสดงข้อผิดพลาด "ไม่สามารถจัดกำหนดการงานภายในรอบระยะเวลาที่โหลดได้"

### <a name="reassign-activities-from-one-resource-to-another"></a>มอบหมายกิจกรรมจากทรัพยากรรายการหนึ่งไปยังอีกรายการหนึ่งอีกครั้ง

คุณสามารถมอบหมายงานจากทรัพยากรรายการหนึ่งไปยังอีกรายการหนึ่งได้ ซึ่งอาจเป็นประโยชน์ในกรณีที่เครื่องจักรชำรุด หรือถูกจองเกินเวลา และคุณต้องการค้นหาทรัพยากรอื่นที่พร้อมใช้งานที่สามารถทำงานได้

### <a name="reassigning-an-activity-as-a-drag-and-drop-interaction"></a>การกำหนดกิจกรรมอีกครั้งให้เป็นการโต้ตอบแบบลากและปล่อย

ในมุมมอง **ทรัพยากร** คุณสามารถกำหนดกิจกรรมอีกครั้งให้กับทรัพยากรอื่นได้ในแผนภูมิ Gantt ให้เป็นการโต้ตอบแบบลากและปล่อย คุณทำเช่นนั้นโดยการเลือกแถวที่ซึ่งมีการจัดกำหนดการกิจกรรม หลังจากที่มีการเลือกแถว คุณสามารถลากแถวไปที่ทรัพยากรในแผนภูมิที่ถูกจัดกลุ่มภายใต้ระดับการจัดกลุ่มทรัพยากรอื่น

### <a name="reassigning-an-activity-from-the-schedule-jobs-menu"></a>การกำหนดกิจกรรมอีกครั้งจากเมนูงานกำหนดการ

คุณสามารถมอบหมายงานอีกครั้งจากกล่องโต้ตอบ **จัดกำหนดการงาน** ที่เปิดจากเมนู **จัดกำหนดการงาน** ได้ จากเมนูนี้ คุณสามารถมอบหมายงานอีกครั้งให้กับทรัพยากรที่มีการโหลดแผนภูมิ Gantt แล้วเท่านั้น ถ้าคุณเลือกงานเพียงหนึ่งงาน แล้วรายการแบบหล่นลงสำหรับทรัพยากรจะถูกเรียงลำดับตามทรัพยากรที่ใช้ได้ ถ้าคุณเลือกงานเพิ่มเติม จากนั้นจะไม่มีข้อมูลเกี่ยวกับทรัพยากรที่ใช้ได้จากรายการทรัพยากร

## <a name="load-additional-resources-to-the-gantt-chart"></a>โหลดแหล่งข้อมูลเพิ่มเติมลงในแผนภูมิ Gantt
ใน **มุมมองทรัพยากร** คุณสามารถเลือกที่จะโหลดทรัพยากรเพิ่มเติมลงในแผนภูมิ Gantt ได้ ซึ่งอาจเป็นประโยชน์ ถ้าคุณต้องการค้นหาทรัพยากรอื่นสำหรับงานที่ถูกจัดกำหนดการบนเครื่องที่ถูกจองเกินเวลาหรือเสีย ในหน้า **โหลดทรัพยากรเพิ่มเติม** คุณจะได้รับรายการของทรัพยากรที่มีวันที่เริ่มใช้งาน ณ วันที่มีการเปิดรายการ จะมีการแสดงรายการทรัพยากรที่ใช้ได้ ซึ่งสัมพันธ์กับงานเลือกไว้ในแผนภูมิ Gantt เป็นลำดับแรก ถ้าคุณมีหลายงานที่เลือกไว้ ก่อนที่จะเปิดรายการ จะไม่มีการแสดงการระบุทรัพยากรที่ใช้ได้ ในหน้า **โหลดทรัพยากรเพิ่มเติม** คุณสามารถเลือกได้อย่างน้อยหนึ่งทรัพยากร ที่จะถูกโหลดลงในแผนภูมิ Gantt เมื่อคุณยืนยันการเลือกของคุณ ถ้าไม่มีงานที่จัดกำหนดการไว้ในทรัพยากรที่เลือกในช่วงเวลาของแผนภูมิ Gantt แล้วทรัพยากรจะถูกวางไว้ภายใต้ระดับการจัดกลุ่มทรัพยากร ด้านล่างรายการกิจกรรมในแผนภูมิ Gantt
