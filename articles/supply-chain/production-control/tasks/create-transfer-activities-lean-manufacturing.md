# <a name="create-transfer-activities-for-lean-manufacturing"></a>สร้างกิจกรรมการโอนย้ายสำหรับ lean manufacturing

[!include[task guide banner](../../includes/task-guide-banner.md)]

สร้างกิจกรรมการโอนย้ายสำหรับการผลิตแบบ lean  

ข้อกำหนดเบื้องต้น: 

1. 1 ต้องสร้างขั้นตอนการผลิตและเวอร์ชันที่ไม่ได้ใช้งานอยู่ 

2. 2. ต้องสร้างคลังสินค้าเริ่มต้นและสิ้นสุดและสถานที่เก็บ  -อีกอย่างหนึ่งควรสร้างการเติมสินค้าหรือการเติมเซลล์ทำงาน


## <a name="find-the-production-flow-version"></a>ค้นหาเวอร์ชันขั้นตอนการผลิต
1. ไปที่การควบคุมการผลิต > การตั้งค่า > ขั้นตอนการผลิตแบบ lean > ขั้นตอนการผลิต
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * โปรดทราบว่าขั้นตอนการผลิตต้องมีเวอร์ชันในสถานะร่าง  
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก

## <a name="create-a-new-activity"></a>สร้างกิจกรรมใหม่
1. คลิกกิจกรรม
    * ตรวจสอบให้แน่ใจว่าขั้นตอนการผลิตที่เลือกมีเวอร์ชันในแบบร่าง และเลือกเวอร์ชันนั้น  
2. คลิกสร้างกิจกรรมการวางแผนใหม่
3. คลิก ถัดไป
4. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
5. ในฟิลด์ชนิดของกิจกรรม ให้เลือก 'การโอนย้าย'
6. ในฟิลด์ปริมาณกระบวนการ ให้ป้อนตัวเลข
7. คลิก ถัดไป

## <a name="select-the-work-cells"></a>เลือกเซลล์ทำงาน
1. ในฟิลด์การเติมสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
    * เมื่อต้องการใช้ที่ตั้งเอาท์พุทเซลล์ทำงานจากสถานที่ในกิจกรรมการโอนย้าย ให้เลือกเซลล์ทำงาน  เหมือนเดียวกันนั้นสามารถทำให้เสร็จด้วยกานเติมเซลล์ทำงาน ซึ่งกำหนดที่ตั้งเป้าหมายของกิจกรรมการโอนย้าย  
2. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก

## <a name="define-the-inventory-updates"></a>กำหนดการอัพเดตสินค้าคงคลัง
1. ในฟิลด์ชนิดของข้อมูลป้อนเข้า ให้เลือกตัวเลือกหนึ่งตัวเลือก
    * โปรดทราบว่า การโอนย้ายไม่ได้เปลี่ยนแปลงชนิดของผลิตภัณฑ์  คุณสามารถโอนย้ายผลิตภัณฑ์สำเร็จรูปหรือผลิตภัณฑ์กึ่งสำเร็จรูป (การโอนย้ายระหว่างสองกิจกรรมของขั้นตอนการผลิต และอาจรวมถึงขั้นตอนคัมบัง)      เมื่อทำการโอนย้ายผลิตภัณฑ์สำเร็จรูป คุณสามารถเลือกว่าการเบิกหรือการรับจะส่งผลในธุรกรรมสินค้าคงคลัง  

## <a name="define-the-transfer-locations"></a>กำหนดที่ตั้งการโอนย้าย
1. คลิก ถัดไป
2. ในฟิลด์คลังสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
3. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
4. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
5. ในฟิลด์ตำแหน่ง ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
6. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
7. ในฟิลด์ขนส่ง ให้เลือก 'ผู้จัดส่ง'
    * ตัวเลือกมีดังนี้: ผู้จัดส่ง - องค์กรที่ปฏิบัติงานคลังสินค้าจัดส่ง ผู้รับ - องค์กรที่ดำเนินการคลังสินค้าที่รับสินค้า ผู้ขนส่งสินค้า - ผู้จัดจำหน่ายบุคคลที่สาม ถ้าองค์กรที่ปฏิบัติงานเป็นผู้จัดจำหน่าย กิจกรรมการโอนย้ายจำเป็นต้องมีข้อตกลงการรับเหมารายย่อย  
8. คลิก ถัดไป

## <a name="define-the-activity-times"></a>กำหนดเวลากิจกรรม
1. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * คำนิยามของรันไทม์จำเป็นต้องใช้  รันไทม์ถูกใช้เพื่อคำนวณต้นทุนและเวลาสำหรับปริมาณที่สามารถประมวลผลได้ของงานคัมบัง  รันไทม์ไม่ได้ใช้ในการคำนวณปริมาณการใช้กำลังการผลิต ซึ่งจะคำนวณโดยเวลาวงจรที่ได้รับมาจากงานเวอร์ชันขั้นตอนการผลิต  
2. ในฟิลด์เวลา ให้ป้อนตัวเลข
3. ในฟิลด์หน่วย ให้พิมพ์ค่า 
4. เลือกหน่วยเวลา
5. ในฟิลด์ต่อปริมาณ ให้ป้อนตัวเลข
6. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * เวลาในการรอไม่จำเป็น  
7. ในฟิลด์เวลา ให้ป้อนตัวเลข
8. ในฟิลด์หน่วย ให้พิมพ์ค่า 
9. เลือกหน่วยเวลา
10. ในฟิลด์ต่อปริมาณ ให้ป้อนตัวเลข
11. คลิก ถัดไป
12. คลิก Finish
13. ปิดหน้า
