# <a name="move-scheduled-kanban-jobs"></a>ย้ายงานคัมบังที่จัดกำหนด

[!include [task guide banner](../../includes/task-guide-banner.md)]

ขั้นตอนนี้มุ่งเน้นไปที่การย้ายกระบวนงานคัมบังที่วางแผนไว้แล้วไปยังรอบระยะเวลาอื่น  ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF ขั้นตอนนี้มีไว้สำหรับหัวหน้างานผลิตหรือนักวางแผนการผลิตที่ทำงานกับคัมบัง


## <a name="select-scheduled-kanban-jobs"></a>เลือก งานคัมบังที่จัดกำหนดแล้ว
1. Gå til Produktionsstyring > คัมบัง > Tidsplanlægning-งานคัมบัง
2. !MtCMR!ในฟิลด์เซลล์ทำงาน ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา áçêìõý !
3. Markér den valgte række på ฟัง
    * เลือกเซลล์ทำงาน 1250  
4. คลิก เลือก
5. Vælg 'Planlagt' feltet แสดงสถานะของงาน
    * ซึ่งนี่จะกรองรายการงานเพื่อแสดงเฉพาะงานคัมบังที่จัดกำหนดการไว้  

## <a name="move-kanban-jobs-to-a-different-period"></a>ย้ายงานคัมบังไปยังรอบระยะเวลาอื่น
1. ค้นหา vælg den ønskede ไปรษณีย์ på ฟัง
    * เลือกงานที่มีสถานะงานที่วางแผนไว้ ตัวอย่างเช่น งานที่จัดกำหนดการไว้ในวันที่ 20 ธันวาคม 2012 ในฟิลด์รอบระยะเวลาที่วางแผนไว้  จากนั้นย้ายงานไปยังรอบระยะเวลาก่อนหน้านี้  
2. คลิก รอบระยะเวลาก่อนหน้า
3. คลิก สิ้นสุด
    * นี่จะย้ายงานไปตอนท้ายของรายการงานเหมือนงานล่าสุดในรอบระยะเวลาก่อนหน้านี้  
4. ค้นหา vælg den ønskede ไปรษณีย์ på ฟัง
    * เลือกงานที่มีสถานะงานที่วางแผนไว้ ตัวอย่างเช่น งานที่จัดกำหนดการไว้ในวันที่ 18 ธันวาคม 2012 ในฟิลด์รอบระยะเวลาที่วางแผนไว้  จากนั้นย้ายงานไปยังรอบระยะเวลาก่อนหน้านี้  
5. คลิก รอบระยะเวลาถัดไป
6. คลิก เริ่มต้น
    * นี่จะย้ายงานไปตอนต้นของรายการงานเหมือนงานแรกในรอบระยะเวลาก่อนหน้านี้  

## <a name="task-move-a-job-within-a-period"></a>งาน: ย้ายงานภายในรอบระยะเวลา
1. ค้นหา vælg den ønskede ไปรษณีย์ på ฟัง
    * เลือกงานที่มีสถานะงานที่วางแผนไว้ ตัวอย่างเช่น งานที่สองที่จัดกำหนดการไว้ในวันที่ 19 ธันวาคม 2012 ในฟิลด์รอบระยะเวลาที่วางแผนไว้  จากนั้นย้ายงานภายในรอบระยะเวลาที่วางแผนไว้  
2. คลิก ไปหน้า
    * ให้สังเกตว่างานถูกย้ายลงไปหนึ่งบรรทัดในรายการ  
3. คลิก ย้อนหลัง
    * ให้สังเกตว่างานถูกย้ายขึ้นไปหนึ่งบรรทัดในรายการ
