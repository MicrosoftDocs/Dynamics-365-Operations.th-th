# <a name="execute-kanban-process-jobs"></a>ดำเนินการงานกระบวนการคัมบัง

[!include [banner](../../includes/banner.md)]

กระบวนงานนี้มุ่งเน้นไปที่การดำเนินงานกระบวนงานคัมบัง  งานแรกเสร็จสมบูรณ์พร้อมกับปริมาณที่คาดหวังไว้และไม่มีข้อผิดพลาด  งานที่สองเสร็จสมบูรณ์พร้อมกับข้อผิดพลาด ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF กระบวนงานนี้มีไว้สำหรับพนักงานควบคุมเครื่องจักร


## <a name="select-a-kanban-job"></a>เลือกงานคัมบัง
1. ไปที่การควบคุมการผลิต > คัมบัง > บอร์ดคัมบังสำหรับงานกระบวนการ 
2. ในฟิลด์เซลล์ทำงาน ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา 
3. คลิกแถวที่มีกลุ่มทรัพยากร 1250 ซึ่งกรองรายการงานเพื่อแสดงเฉพาะงานสำหรับเซลล์ทำงาน 1250
    * ทำเครื่องหมายแถวที่มีสถานะงานเป็นวางแผนแล้ว  

## <a name="complete-a-job-with-expected-quantity"></a>ดำเนินงานที่มีปริมาณที่คาดไว้ให้เสร็จสมบูรณ์
1. ขยายหรือยุบส่วนรายละเอียด
    * ส่วนนี้แสดงข้อมูลสำคัญเกี่ยวกับหมายเลขบัตร หมายเลขสินค้า ปริมาณที่สั่ง และชื่อกิจกรรม  
2. ขยายหรือยุบส่วนคำแนะนำเกี่ยวกับการผลิต
    * ส่วนนี้แสดงคำแนะนำเกี่ยวกับการผลิตสำหรับกิจกรรม คำแนะนำสามารถเป็นข้อความ รูปภาพ ภาพวาด หรือเอกสารอื่นๆได้  
3. คลิก เริ่มต้น
    * เลือกงานที่ยังไม่เสร็จสิ้น  .ช้ไอคอนสถานะในฟิลด์สถานะงาน เพื่อดูสถานะงาน      
4. คลิกเสร็จสมบูรณ์
    * งานเสร็จสมบูรณ์พร้อมกับปริมาณที่คาดหวังไว้  

## <a name="complete-a-job-with-errors"></a>ดำเนินงานที่มีงานที่มีข้อผิดพลาดให้เสร็จสมบูรณ์
1. คลิก เริ่มต้น
    * เมื่องานเสร็จสมบูรณ์ งานต่อไปบนรายการจะถูกเลือกโดยอัตโนมัติ  นี่เป็นเหตุผลว่าทำไมคุณไม่จำเป็นต้องเลือกงานก่อนที่คุณจะคลิกเริ่ม  
2. ในบานหน้าต่างการดำเนินการ คลิกผลิต
3. คลิกเสร็จสมบูรณ์ (รายละเอียด)
4. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
5. ในฟิลด์ปริมาณสินค้าที่บกพร่อง ให้ป้อนตัวเลข
6. ในฟิลด์ปริมาณสินค้าที่ดี ให้ป้อนตัวเลข
7. คลิก ตกลง
