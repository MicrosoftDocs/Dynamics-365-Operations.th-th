# <a name="add-an-existing-activity-to-a-production-flow-version"></a>เพิ่มกิจกรรมที่มีอยู่ไปยังรุ่นขั้นตอนการผลิต

[!include [task guide banner](../../includes/task-guide-banner.md)]

ในขณะที่สร้างขั้นตอนการผลิตรุ่นใหม่ คุณสามารถเลือกที่จะเพิ่มกิจกรรมที่สร้างขึ้นสำหรับรุ่นที่เก่ากว่าไปยังรุ่นใหม่ได้  กระบวนงานนี้แสดงวิธีการสร้างรุ่นใหม่สำหรับขั้นตอนการผลิตที่มีอยู่โดยไม่ต้องคัดลอกกิจกรรม ในขั้นตอนถัดไป กิจกรรมที่มีอยู่จะถูกเพิ่มไปยังรุ่นใหม่ 

งานนี้จำเป็นต้องใช้ขั้นตอนการผลิตที่มีรุ่นและกิจกรรมที่สร้างขึ้นแล้ว


## <a name="create-a-new-production-flow-version"></a>สร้างเวอร์ชันขั้นตอนการผลิตใหม่
1. ไปที่การควบคุมการผลิต > การตั้งค่า > ขั้นตอนการผลิตแบบ lean > ขั้นตอนการผลิต
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
4. คลิกแก้ไข
5. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
6. ในฟิลด์วันหมดอายุ ให้ป้อนวันที่และเวลา
    * โปรดทราบว่าก่อนที่คุณจะสร้างรุ่นขั้นตอนการผลิตใหม่ ตรวจสอบให้แน่ใจว่าได้ตรวจสอบวันและเวลาหมดอายุของรุ่นที่ใช้งานอยู่  จะมีการสร้างรุ่นใหม่ที่มีวันที่เริ่มต้นที่มีผลบังคับใช้ซึ่งต่อจากวันหมดอายุของรุ่นที่เลือก  
7. คลิก เพิ่ม
8. เลือก ไม่ ในฟิลด์ คัดลอกจากรุ่น
    * เลือก ไม่ เพื่อเริ่มต้นใช้รุ่นที่ว่างอยู่ถ้ากิจกรรมของรุ่นที่คัดลอกส่วนใหญ่จะถูกแทนที่ด้วยกิจกรรมใหม่  เพิ่มกิจกรรมที่ไม่มีการเปลี่ยนแปลงไปยังฟังก์ชัน เพิ่มรายการที่มีอยู่ ในแบบฟอร์มกิจกรรมด้วยตนเอง  
9. เลือก ไม่ ในฟิลด์ ทำซ้ำกฎคัมบัง
    * เมื่อกิจกรรมไม่ได้ถูกคัดลอกไปยังรุ่นใหม่ จะไม่สามารถคัดลอกกฎคัมบังในขณะที่สร้างรุ่นใหม่ได้    แต่คุณจะใช้ฟังก์ชัน สร้างคัมบังการแทนที่ในภายหลังในแบบฟอร์มกฎคัมบัง เพื่อแทนที่กฎคัมบังของรุ่นขั้นตอนการผลิตเดิมที่มีกฎคัมบังที่ใช้กิจกรรมของรุ่นใหม่  
10. คลิก ตกลง
11. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ

## <a name="add-an-existing-activity"></a>เพิ่มกิจกรรมที่มีอยู่
1. คลิกกิจกรรม
2. คลิกเพิ่มรายการที่มีอยู่เพื่อเปิดกล่องโต้ตอบการวาง
    * ค้นหาและเลือกกิจกรรมที่มีอยู่ที่จะเพิ่มในรุ่นขั้นตอนการผลิตใหม่   โปรดทราบว่ารายการจะแสดงกิจกรรมทั้งหมดที่ถูกสร้างขึ้นสำหรับขั้นตอนการผลิตนี้สำหรับรุ่นก่อนหน้านี้ทั้งหมดของขั้นตอน  
3. ในฟิลด์กิจกรรม ให้ป้อนหรือเลือกค่า
4. คลิก ตกลง
