# <a name="create-working-time-scheduling-properties"></a>สร้างคุณสมบัติการจัดกำหนดการเวลาการทำงาน

[!include [banner](../../includes/banner.md)]

คุณสมบัติการจัดกำหนดการของเวลาทำงาน จะใช้เพื่อจัดประเภทช่วงเวลาการทำงานและกำหนดการตั้งค่าที่ผู้ปฏิบัติงานต้องคำนึงถึง  กระบวนงานนี้แสดงวิธีการกำหนดคุณสมบัติการจัดกำหนดการเวลาการทำงานและค่าของคุณสมบัติที่เกี่ยวข้อง  คุณสามารถดำเนินการขั้นตอนนี้ได้ในข้อมูลสาธิตของบริษัท USMF หรือการใช้ข้อมูลของคุณเอง

1. ไปที่พื้นที่ทำงานทั้งหมด > การจัดการรอบการใช้งานทรัพยากร 
2. คลิก คุณสมบัติ 

## <a name="create-property"></a>สร้างคุณสมบัติ
1. คลิก สร้าง
2. ในฟิลด์คุณสมบัติ ให้พิมพ์ค่าใดค่าหนึ่ง 
3. ในฟิลด์ชื่อ ให้พิมพ์ค่า 

## <a name="create-property-values"></a>สร้างค่าคุณสมบัติ
1. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
2. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
3. ในฟิลด์ค่า ให้พิมพ์ค่าใดค่าหนึ่ง 
4. คลิก บันทึก
