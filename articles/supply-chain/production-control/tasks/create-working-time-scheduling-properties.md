# <a name="create-working-time-scheduling-properties"></a>สร้างคุณสมบัติการจัดกำหนดการเวลาการทำงาน

[!include [banner](../../includes/banner.md)]

คุณสมบัติการจัดกำหนดการของเวลาทำงาน จะใช้เพื่อจัดประเภทช่วงเวลาการทำงานและกำหนดการตั้งค่าที่ผู้ปฏิบัติงานต้องคำนึงถึง  กระบวนงานนี้แสดงวิธีการกำหนดคุณสมบัติการจัดกำหนดการเวลาการทำงานและค่าของคุณสมบัติที่เกี่ยวข้อง  คุณสามารถศึกษากระบวนงานนี้ได้ในบริษัทข้อมูลสาธิต USMF หรือใช้ข้อมูลของคุณเอง

1. ไปที่ **พื้นที่ทำงาน \> การจัดการรอบการใช้งานทรัพยากร**
2. เลือก **คุณสมบัติ**

## <a name="create-property"></a>สร้างคุณสมบัติ

1. เลือก **ใหม่**
2. ในฟิลด์ **คุณสมบัติ** ให้พิมพ์ค่าใดค่าหนึ่ง
3. ในฟิลด์ **ชื่อ** ให้พิมพ์ค่าใดค่าหนึ่ง

## <a name="create-property-values"></a>สร้างค่าคุณสมบัติ

1. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
2. ในฟิลด์ **ชื่อ** ให้พิมพ์ค่าใดค่าหนึ่ง
3. ในฟิลด์ **ค่า** ให้พิมพ์ค่าใดค่าหนึ่ง
4. เลือก **บันทึก**
