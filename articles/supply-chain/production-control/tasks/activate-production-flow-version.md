# <a name="activate-a-production-flow-version"></a>เรียกใช้เวอร์ชันขั้นตอนการผลิต

[!include [banner](../../includes/banner.md)]

ก่อนที่จะสามารถสร้างงานคัมบังและสามารถประมวลผลกิจกรรมสำหรับเวอร์ชันขั้นตอนการผลิต คุณควรเรียกใช้เวอร์ชัน  แนวทางปฏิบัติที่ดีที่สุดคือเรียกใช้เวอร์ชันก่อนที่วันที่และเวลาเริ่มต้นที่มีผลบังคับใช้จะหมดอายุ  เพื่อดำเนินการนี้ให้เสร็จสิ้น คุณต้องมีเวอร์ชันขั้นตอนการผลิตที่มีสถานะเป็นแบบร่างเพื่อเรียกใช้งานเวอร์ชัน 


## <a name="find-and-activate-a-production-flow-version"></a>ค้นหาและเรียกใช้เวอร์ชันขั้นตอนการผลิต
1. ไปที่ขั้นตอนการผลิต
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * เลือกเวอร์ชันขั้นตอนการผลิตที่มีแบบร่างในฟิลด์สถานะของแผน  
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
4. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
    * เลือกเวอร์ชันขั้นตอนการผลิตที่มีแบบร่างในฟิลด์สถานะของแผน  
5. คลิกเรียกใช้
6. คลิก ตกลง
