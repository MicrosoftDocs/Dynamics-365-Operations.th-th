# <a name="configure-a-worker-using-the-mobile-job-device"></a>ตั้งค่าคอนฟิกผู้ปฏิบัติงานโดยใช้อุปกรณ์เคลื่อนที่ของงาน

[!include[task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงวิธีการกำหนดบทบาทที่ถูกต้องให้กับบัญชีผู้ปฏิบัติงานและเปิดการใช้งานของผู้ปฏิบัติงานในการลงทะเบียนการผลิต


## <a name="assign-roles-to-user-account"></a>กำหนดบทบาทให้กับบัญชีผู้ใช้งาน
1. ไปที่การดูแลระบบ > ผู้ใช้ > ผู้ใช้
2. ใช้ตัวกรองข้อมูลอย่างรวดเร็วเพื่อกรองข้อมูลเกี่ยวกับชื่อของผู้ปฏิบัติงานที่เป็นบัญชีผู้ใช้ที่สัมพันธ์กับบทบาทของผู้ปฏิบัติงานเครื่องจักร  ในข้อมูลตัวอย่าง ชื่อจะเป็น Shannon
3. เน้นเรกคอร์ดบัญชีผู้ใช้
4. ในรายการ คลิกลิงค์ "ชื่อ" ในแถวที่เลือกเพื่อดูรายละเอียดของบัญชีผู้ใช้
5. ในแผนภูมิ เลือก 'ตัวดำเนินการ Roles\Machine'
6. ปิดหน้ารายละเอียดบัญชีผู้ใช้งาน
7. ปิดหน้า

## <a name="configure-worker-account"></a>ตั้งค่าคอนฟิกบัญชีผู้ปฏิบัติงาน
1. ไปที่ทรัพยากรบุคคล > ผู้ปฏิบัติงาน > ผู้ปฏิบัติงาน
2. ใช้ตัวกรองข้อมูลอย่างรวดเร็วเพื่อกรองข้อมูลเกี่ยวกับชื่อของผู้ปฏิบัติงานที่เป็นบัญชีผู้ใช้ที่สัมพันธ์กับบทบาทของผู้ปฏิบัติงานเครื่องจักร  ในข้อมูลตัวอย่าง ชื่อจะเป็น Shannon
3. เน้นเรกคอร์ดบัญชีผู้ใช้
4. ในรายการ คลิกลิงค์ "ชื่อ" ในแถวที่เลือกเพื่อดูรายละเอียดของบัญชีผู้ใช้
5. คลิกแท็บการจ้างงาน
6. ขยายแท็บด่วนระยะเวลาการลงทะเบียน และคลิกเรียกใช้บนเทอร์มินัลการลงทะเบียน
7. คลิกเรียกใช้บนเทอร์มินัลการลงทะเบียน
8. ในฟิลด์กลุ่มแบบจำลอง ให้ป้อนหรือเลือกค่า
9. ในฟิลด์กลุ่มกลุ่มการคำนวณเริ่มต้น ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
10. ในฟิลด์กลุ่มการอนุมัติ ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
11. ในฟิลด์โพรไฟล์มาตรฐาน ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
12. ในฟิลด์กลุ่มโพรไฟล์ ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
13. คลิก ตกลง ระบบจะนำเข้าข้อมูลการชำระเงิน
14. คลิกแก้ไขเพื่อป้อนหมายเลขป้ายชื่อสำหรับผู้ปฏิบัติงานลงทะเบียนเวลาใหม่
15. ในฟิลด์รหัสป้ายชื่อพนักงาน ให้พิมพ์ค่าใดค่าหนึ่ง
16. คลิกบันทึก
17. ใช้ทางลัด SaveRecord
18. ปิดหน้ารายละเอียดผู้ปฏิบัติงาน
19. ปิดหน้า

## <a name="assign-worker-to-device-group"></a>มอบหมายผู้ปฏิบัติงานให้กับกลุ่มอุปกรณ์
1. ไปที่การควบคุมการผลิต > การตั้งค่า > การดำเนินการผลิต > ตั้งค่าคอนฟิกบัตรงานสำหรับอุปกรณ์
2. คลิก เพิ่ม
3. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
4. คลิก ตกลง
5. คลิก แก้ไข
6. ในฟิลด์หน่วยการผลิต คุณสามารถตั้งค่าตัวกรองเริ่มต้นสำหรับผู้ปฏิบัติงาน  ซึ่งจะกำหนดว่า จะแสดงเฉพาะงานการผลิตสำหรับหน่วยการผลิตที่เลือกเมื่อผู้ปฏิบัติงานที่ล็อกออนที่อุปกรณ์
7. ปิดหน้า
