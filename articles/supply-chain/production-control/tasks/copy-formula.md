# <a name="copy-a-formula"></a>คัดลอกสูตร

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

ขั้นตอนนี้มุ่งเน้นการสร้างสูตรที่รวมส่วนผสมเดียวกับสูตรที่มีอยู่ แต่ มีความแตกต่างเล็กน้อย  เมื่อต้องการสร้างบรรทัดสูตร คุณสามารถใช้ฟังก์ชั่นคัดลอกเพื่อคัดลอกสูตรที่มีอยู่แล้วซึ่งส่วนใหญ่จะเป็นส่วนผสมที่คุณต้องการ  จากนั้นคุณสามารถเปลี่ยนแปลงใดๆก็ได้กับแต่ละบรรทัดในเวอร์ชันใหม่  โดยการใช้ฟังก์ชั่นคัดลอก คุณไม่ต้องสร้างหลายสูตรที่ค่อนข้างจะเหมือนกัน  บริษัทข้อมูลสาธิตที่เคยสร้างงานนี้คือ USP2


## <a name="create-a-formula"></a>สร้างสูตร
1. ไปที่การจัดการข้อมูลผลิตภัณฑ์ > สูตรการผลิตและสูตร > สูตร
2. คลิก สร้าง
3. ในฟิลด์สูตร ให้พิมพ์ค่า
4. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
    * พิมพ์ชื่อที่มีความหมายสำหรับสูตร  
5. ในฟิลด์ไซต์ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
6. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
7. ในฟิลด์กลุ่มสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
8. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
9. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
10. คลิก บันทึก

## <a name="copy-formula-lines"></a>คัดลอกบรรทัดสูตร
1. ในบานหน้าต่างการดำเนินการ คลิกสูตร
2. คลิก คัดลอก
3. ในฟิลด์หมายเลขสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
5. ในฟิลด์เวอร์ชั่นสูตตร ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
6. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
7. คลิก ตกลง

## <a name="adjust-copied-formula-lines"></a>ปรับปรุงบรรทัดสูตรที่คัดลอก
1. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
2. คลิก ลบ
3. คลิก ใช่

## <a name="approve-formula"></a>อนุมัติสูตร
1. ในบานหน้าต่างการดำเนินการ คลิกสูตร
2. คลิกอนุมัติสูตร
3. ในฟิลด์อนุมัติโดย ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา 
4. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
5. คลิกเลือก 
6. คลิก ตกลง
