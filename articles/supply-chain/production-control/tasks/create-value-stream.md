# <a name="create-a-value-stream"></a>สร้างสายธารคุณค่า

[!include [banner](../../includes/banner.md)]

สตรีมค่าคือหน่วยปฏิบัติงาน ที่จัดกลุ่มขั้นตอนการผลิตหนึ่งหรือหลายรายการ  เมื่อต้องการกำหนดขั้นตอนการผลิต คุณต้องกำหนดการสตรีมค่าก่อนเป็นอย่างเเรก


## <a name="open-the-form"></a>เปิดแบบฟอร์ม
1. ไปที่การควบคุมการผลิต > การตั้งค่า > ขั้นตอนการผลิตแบบ lean > สตรีมค่า

## <a name="create-a-new-value-stream"></a>สร้างสตรีมค่าใหม่
1. คลิกที่สร้างเพื่อเปิดกล่องโต้ตอบการวาง
2. ในฟิลด์ชื่อ ให้พิมพ์ค่า 

## <a name="select-a-value-stream-manager"></a>เลือกผู้จัดการสตรีมค่า
1. ในฟิลด์ผู้จัดการ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
    * ผู้จัดการสายธารค่าถูกกำหนดไปยังเอกสารองค์กร  
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก

## <a name="save-and-close"></a>บันทึกและปิด
1. คลิก บันทึก
2. ปิดหน้า
3. รีเฟรชหน้า
