6. <span data-ttu-id="0d466-124">ในฟิลด์ชนิดรอบระยะเวลาการวางแผน ให้เลือกหนึ่งตัวเลือก</span><span class="sxs-lookup"><span data-stu-id="0d466-124">In the Planning period type field, select an option.</span></span>
