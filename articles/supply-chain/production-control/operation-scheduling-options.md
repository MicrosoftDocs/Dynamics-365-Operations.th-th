# <a name="operations-scheduling-options"></a>ตัวเลือกการจัดตารางการผลิตระดับการดำเนินงาน

[!include [banner](../includes/banner.md)]

หัวข้อนี้อธิบายตัวเลือกสำหรับการจัดตารางการผลิตระดับการดำเนินงาน คุณสามารถใช้การจัดตารางการผลิตระดับการดำเนินงานเพื่อให้การประเมินโดยทั่วไปของกระบวนการผลิตตามช่วงเวลา

การจัดตารางการผลิตระดับการดำเนินงานคำนวณข้อมูลต่อไปนี้สำหรับใบสั่งผลิต:

-   วันที่เริ่มต้นและวันที่สิ้นสุดถูกตั้งค่าสำหรับใบสั่งผลิตและการดำเนินงานแต่ละรายการ
-   มีการกำหนดทรัพยากรให้กับการดำเนินงาน

ตั้งค่าต่างๆ กำหนดวิธีคำนวณตารางการผลิต คุณสามารถกำหนดการตั้งค่าเหล่านี้ในหน้า **การจัดตารางการผลิตระดับการดำเนินงาน** ข้อมูลต่อไปนี้อธิบายตัวเลือกของการจัดกำหนดการ

## <a name="operations-scheduling"></a>การจัดตารางการผลิตระดับการดำเนินงาน
### <a name="scheduling-direction"></a>คำสั่งการกำหนดเวลาการดำเนินงาน

ทิศทางการจัดกำหนดการเป็นข้อมูลพื้นฐานของกระบวนการจัดกำหนดการ สามารถจัดกำหนดการผลิตแบบไปข้างหน้าหรือแบบย้อนหลังจากวันที่ใดๆ ขึ้นอยู่กับช่วงเวลาและความต้องการจัดกำหนดการ

-   **การจัดกำหนดการแบบไปข้างหน้า**– คุณสามารถจัดกำหนดการการผลิตเพื่อเริ่มให้เร็วที่สุดที่เป็นไปได้ การผลิตอาจเริ่มต้นวันนี้ วันพรุ่งนี้ หรือวันที่ระบุในอนาคต การผลิตถูกจัดกำหนดการให้เริ่มต้นในวันที่เร็วที่สุดที่เป็นไปได้ และมีการวางแผนแบบไปข้างหน้าเป็นวันที่สิ้นสุดที่เร็วที่สุดที่เป็นไปได้
-   **การจัดกำหนดการแบบย้อนหลัง** – คุณสามารถจัดกำหนดการการผลิตเพื่อเริ่มให้ช้าที่สุดที่เป็นไปได้ กำหนดการจะขึ้นอยู่กับวันที่ที่การผลิตจะต้องเสร็จสมบูรณ์และนับย้อนหลังไปยังวันที่ที่ช้าที่สุดที่เป็นไปได้ที่สามารถเริ่มต้นการผลิตโดยมีกำหนดเวลาสิ้นสุดเป้าหมาย

โดยตัวเลือกที่คุณสามารถเลือกได้มีดังต่อไปนี้

-   **ไปข้างหน้าจากวันวันนี้** – จัดกำหนดการแบบไปข้างหน้าจากวันที่ปัจจุบัน (วันที่ปัจจุบันคือวันที่ของระบบ)
-   **ไปข้างหน้าจากวันที่วางแผนไว้** – จัดกำหนดการแบบไปข้างหน้าจากวันที่เริ่มต้นก่อนหน้า ถ้าไม่มีการจัดตารางการผลิตก่อนหน้านี้ ทิศทางการจัดตารางการผลิตจะเป็นแบบไปข้างหน้าจากวันที่ปัจจุบัน
-   **ไปข้างหน้าจากวันที่จัดกำหนดการ** – จัดกำหนดการแบบไปข้างหน้าจากวันที่ที่ระบุในฟิลด์ **วันที่จัดกำหนดการ** ถ้าคุณไม่ได้ระบุวันที่จัดกำหนดการ ทิศทางการจัดตารางการผลิตจะเป็นแบบไปข้างหน้าจากวันที่ปัจจุบัน
-   **ย้อนหลังจากวันที่จัดส่ง**– จัดกำหนดการแบบย้อนหลังจากวันที่จัดส่งที่ระบุไว้สำหรับใบสั่งผลิต ถ้าคุณเลือกตัวเลือกนี้ แต่ไม่มีการระบุวันที่จัดส่ง วันที่จัดส่งคือวันที่ปัจจุบัน
-   **ย้อนหลังจากวันที่สิ้นสุดที่วางแผนไว้**– จัดกำหนดการแบบย้อนหลังจากวันที่สิ้นสุดที่คำนวณไว้ก่อนหน้านี้ ถ้าไม่มีการจัดตารางการผลิตก่อนหน้านี้ วันที่สิ้นสุดคือวันที่ปัจจุบัน
-   **ย้อนหลังจากวันที่จัดกำหนดการ** – จัดกำหนดการแบบย้อนหลังจากวันที่ที่ระบุในฟิลด์ **วันที่จัดกำหนดการ** ถ้าคุณไม่ได้ระบุวันที่จัดกำหนดการ จะมีการใช้วันที่ปัจจุบัน ย้อนหลังจากวันที่จัดกำหนดการจะถูกคำนวณสำหรับใบสั่งผลิตครั้งล่าสุดที่มีการคำนวณความต้องการ ถ้าไม่ได้ระบุวันที่สำหรับใบสั่งผลิต จะมีการใช้วันที่ระบบปัจจุบัน
-   **ย้อนหลังจากวันที่ในอนาคต** – จัดกำหนดการแบบย้อนหลังจากวันที่ในอนาคตที่คำนวณสำหรับใบสั่งผลิตครั้งล่าสุดที่มีการคำนวณความต้องการ ถ้าไม่ได้ระบุวันที่ในอนาคตสำหรับใบสั่งผลิต จะมีการใช้วันที่ระบบปัจจุบัน
-   **ตามที่จัดกำหนดเวลาครั้งหลังสุด** – สำหรับการจัดตารางการผลิตระดับการดำเนินงานและการจัดตารางการผลิตระดับงาน ทิศทางและวันที่จัดกำหนดการจะถูกบันทึกไว้ ดังนั้น คุณสามารถเลือกตัวเลือกนี้สำหรับการจัดกำหนดการครั้งต่อมาได้ ถ้าไม่มีการจัดตารางการผลิตก่อนหน้านี้ของใบสั่งผลิต การจัดกำหนดการจะเป็นแบบย้อนหลังจากวันที่ระบบปัจจุบัน
-   **ไปข้างหน้าจากวันพรุ่งนี้** – จัดกำหนดการแบบไปข้างหน้าจากวันที่ปัจจุบันบวกหนึ่งวัน
-   **ไปข้างหน้าจากงานก่อนหน้านี้** – ตัวเลือกนี้จะมีความเกี่ยวข้องในการจัดตารางการผลิตระดับงานเท่านั้น ถ้าคุณเลือกทิศทางการจัดกำหนดการนี้สำหรับการจัดตารางการผลิตระดับการดำเนินงาน ใบสั่งผลิตจะถูกจัดกำหนดการแบบไปข้างหน้าจากวันที่ปัจจุบัน ในการจัดตารางการผลิตระดับงาน การจัดกำหนดการจะถูกสร้างขึ้นสำหรับงานหนึ่ง และงานอื่นทั้งหมดสำหรับการผลิตจะถูกจัดกำหนดการตามงานดังกล่าว
-   **ย้อนหลังจากงานก่อนหน้านี้** – ตัวเลือกนี้จะมีความเกี่ยวข้องในการจัดตารางการผลิตระดับงานเท่านั้น ถ้าคุณเลือกทิศทางการจัดกำหนดการนี้สำหรับการจัดตารางการผลิตระดับการดำเนินงาน แผนการใบสั่งจะถูกจัดกำหนดการแบบย้อนหลังจากวันที่ปัจจุบัน ในการจัดตารางการผลิตระดับงาน การจัดกำหนดการจะถูกสร้างขึ้นสำหรับงานหนึ่ง และงานอื่นทั้งหมดสำหรับการผลิตจะถูกจัดกำหนดการตามงานดังกล่าว

### <a name="scheduling-date"></a>วันที่กำหนดเวลาการดำเนินงาน

วันที่นี้จะถูกใช้สำหรับทิศทางการจัดกำหนดการ **แบบไปข้างหน้าจากวันที่จัดกำหนดการ** และ **ย้อนหลังจากวันที่จัดกำหนดการ** การจัดการกำหนดการเป็นแบบย้อนหลังหรือไปข้างหน้าจากวันที่นี้ สำหรับข้อมูลเพิ่มเติม ให้ดูที่ส่วนก่อนหน้า "ทิศทางการจัดกำหนดการ"

### <a name="recalculate-bom-levels"></a>คำนวณระดับ BOM อีกครั้ง

เมื่อคุณเลือก **คำนวณระดับ BOM ใหม่** ระดับของสูตรการผลิต (BOM) ที่เลือกจะถูกคำนวณใหม่เพื่อช่วยรับประกันลำดับการจัดกำหนดการที่ถูกต้อง

## <a name="limitations"></a>การจำกัด
### <a name="finite-capacity"></a>กำลังการผลิตมีจำกัด

การจัดกำหนดการจะขึ้นอยู่กับความพร้อมของทรัพยากรที่จำเป็นในการดำเนินการผลิตให้เสร็จสมบูรณ์ ถ้ากำลังการผลิตมีไม่เพียงพอ อาจทำให้ใบสั่งผลิตล่าช้าหรือหยุดการทำงาน ถ้ามีการใช้กำลังการผลิตมีจำกัดในการจัดตารางการผลิตระดับการดำเนินงาน การจองกำลังการผลิตที่มีอยู่ที่ถูกดำเนินการกับทรัพยากรจะถูกพิจารณา และกำลังการผลิตจะเห็นเป็นไม่พร้อมใช้งาน ใบสั่งผลิตจะถูกจัดกำหนดการโดยขึ้นอยู่กับความพร้อมของกำลังการผลิตของทรัพยากรที่จำเป็น การจัดตารางการผลิตระดับการดำเนินงานใช้ลำดับของการดำเนินงานที่ระบุเพื่อกำหนดลำดับของการดำเนินงานในกระบวนการผลิต การจัดตารางการผลิตระดับการดำเนินงานจะจองกำลังการผลิตของกลุ่มทรัพยากร ขึ้นอยู่กับเวลาการดำเนินงานที่มีกำหนดกระบวนการผลิต กำลังการผลิตของกลุ่มทรัพยากรคือผลรวมของกำลังการผลิตที่พร้อมใช้งานในทรัพยากรทั้งหมดในกลุ่มทรัพยากร

### <a name="finite-material"></a>วัตถุดิบมีจำกัด

นอกจากนี้การจัดกำหนดการยังขึ้นอยู่กับความพร้อมของวัตถุดิบที่จำเป็นในการผลิต ความพร้อมใช้งานของส่วนประกอบที่ไม่เพียงพออาจก่อให้เกิดความล่าช้าในการผลิตได้อีกด้วย นอกจากนี้การจัดกำหนดการยังอาจขึ้นอยู่กับการใช้วัตถุดิบ เพียงแค่ระบุวัตถุดิบที่จะต้องพร้อมใช้งานสำหรับการผลิตแทนวัตถุดิบที่ไม่จำเป็น ชนิดของการจัดกำหนดการนี้เรียกว่าการจัดกำหนดการด้วยวัตถุดิบมีจำกัด เมื่อคุณระบุวัตถุดิบมีจำกัด การผลิตจะถูกจัดกำหนดการโดยขึ้นอยู่กับว่าวัตถุดิบพร้อมใช้งานหรือไม่ **หมายเหตุ:** เมื่อคุณปรับทั้งกำลังการผลิตและวัตถุดิบให้เหมาะสม จะมีการคำนวณการผลิตให้ตรงกับข้อจำกัดทั้งสอง ความพร้อมใช้งานของกำลังการผลิตและวัตถุดิบจะถูกพิจารณา และการดำเนินงานของใบสั่งผลิตจะไม่ถูกจัดกำหนดการให้เริ่มต้นจนกว่ากำลังการผลิตและวัตถุดิบจะพร้อมใช้งานพร้อมกันและในปริมาณที่ต้องการ เลือกกล่องกาเครื่องหมายนี้ถ้าควรคำนึงถึงการมีอยู่จำกัดของวัตถุดิบในระหว่างการจัดตารางการผลิต ถ้าวัสดุมีอยู่จำกัด จะมีการพิจารณาความครอบคลุมของวัตถุดิบสำหรับเวลานั้น กล่าวคือ การจัดกำหนดการจะพิจารณาวันที่ในอนาคตที่จะถูกคำนวณสำหรับสินค้านั้น การจัดกำหนดการจะจองวัตถุดิบและกระจายการผลิตปัจจุบัน ถ้าการจัดกำหนดการเกิดขึ้นหลายครั้ง เฉพาะการจัดกำหนดการแรกที่จะรันการกระจายและทำการจอง ถ้าคุณทำการเปลี่ยนแปลงใน BOM การผลิตหรือกระบวนการผลิต การจัดกำหนดการถัดไปจะรันการกระจาย สำหรับการกระจาย จะถือว่าจำเป็นต้องมีวัตถุดิบในวันเดียวกัน เนื่องจากไม่ได้จัดกำหนดการการผลิตในเวลาของการกระจายกำหนดการหลัก วันที่ปัจจุบันจะเป็นการประเมินที่ดีที่สุดว่าสินค้าจะพร้อมใช้งานเมื่อใด การกระจายจะตรวจสอบว่าสินค้าพร้อมใช้งานหรือไม่ ถ้าสินค้าพร้อมใช้งาน จะสามารถปฏิบัติตามความต้องการของการผลิตได้ ถ้าไม่มีสินค้าคงเหลือในคลังภายในวันที่ปัจจุบัน จะมีการสร้างแผนการใบสั่งขึ้นและมีการเลือกออฟเซ็ตสำหรับแผนการใบสั่ง ถ้าการยืนยันโดยอัตโนมัติถูกตั้งค่าไว้สำหรับแผนการใบสั่ง แผนการใบสั่งจะได้รับการยืนยันโดยอัตโนมัติสำหรับการซื้อหรือการผลิต สถานะการผลิตจะเปลี่ยนโดยอัตโนมัติเป็นสถานะที่ระบุไว้ในฟิลด์ **สถานะการผลิตที่ร้องขอ** ในกล่องโต้ตอบ **กลุ่มความครอบคลุม** ถ้าไม่ได้เลือกกล่องกาเครื่องหมายไว้ การผลิตจะถูกพิจารณาเป็นพร้อมใช้งานเสมอ ดังนั้น การจัดกำหนดการจะไม่พิจารณาว่าวัตถุดิบพร้อมใช้งานในขณะที่ต้องการหรือไม่

### <a name="finite-property"></a>คุณสมบัติแบบจำกัด

เลือกกล่องกาเครื่องหมายนี้ ถ้าควรรวมการจัดตารางการผลิตระดับงานในข้อกำหนดด้านคุณสมบัติ

### <a name="keep-production-unit"></a>รักษาหน่วยการผลิต

เลือกว่ากลไกการจัดกำหนดการควรจัดกำหนดการเฉพาะทรัพยากรที่ระบุในหน่วยการผลิตแล้วหรือไม่

### <a name="keep-warehouse-from-resource"></a>รักษาคลังสินค้าจากทรัพยากร

เลือกว่ากลไกการจัดกำหนดการควรจัดกำหนดการเฉพาะทรัพยากรที่เชื่อมโยงกับคลังสินค้าอินพุทที่ระบุสำหรับทรัพยากรนั้นหรือไม่

## <a name="references"></a>บุคคลอ้างอิง
### <a name="schedule-references"></a>การอ้างอิงกำหนดการ

เมื่อการอ้างอิงขึ้นอยู่กับใบสั่งผลิต จะเรียกอีกอย่างว่าการผลิตย่อย สามารถจัดกำหนดการการผลิตย่อยโดยเป็นส่วนหนึ่งของการจัดกำหนดการใบสั่งผลิต เลือกกล่องกาเครื่องหมายนี้ถ้าการจัดกำหนดการของการผลิตย่อยควรเป็นไปตามการจัดกำหนดการของการผลิตหลัก ตามการผลิตหลัก การผลิตที่อยู่เหนือกว่าจะถูกจัดกำหนดการแบบไปข้างหน้า และการผลิตที่อยู่ภายใต้จะถูกจัดกำหนดการแบบย้อนหลัง คุณสามารถดูการอ้างอิงใบสั่งผลิตได้ในฟิลด์ **ระดับการอ้างอิง** ในหน้า **ใบสั่งผลิต**

### <a name="synchronize-references"></a>ซิงโครไนส์การอ้างอิง

คุณสามารถซิงโครไนส์การอ้างอิงกับใบสั่งผลิต ถ้าเลือกตัวเลือกนี้ วันที่ของการผลิตย่อยจะถูกเปลี่ยนและสอดคล้องเมื่อมีการเปลี่ยนแปลงกำหนดการของใบสั่งผลิต ถ้าใบสั่งผลิตมีการผลิตย่อยอย่างน้อยหนึ่งรายการ คุณอาจต้องการจัดกำหนดการผลิตพร้อมกับการผลิตหลัก ในกรณีนี้ ไม่สามารถเริ่มต้นการผลิตหลักได้จนกว่าการผลิตย่อยจะเสร็จสมบูรณ์ ดังนั้น ให้เลือกกล่องกาเครื่องหมายนี้ ถ้าการจัดกำหนดการของการผลิตย่อยควรเป็นไปตามเวลาเริ่มต้นและสิ้นสุดของการผลิตที่เลือก คุณสามารถเลือกกล่องกาเครื่องหมายนี้เฉพาะถ้ากล่องกาเครื่องหมาย **การอ้างอิงกำหนดการ** ถูกเลือกเช่นกัน

## <a name="cancellation"></a>การยกเลิก
### <a name="cancel-queue-time"></a>ยกเลิกเวลาในการรอ

เลือกกล่องกาเครื่องหมายนี้เพื่อแยกเวลาในการรอออกจากการจัดกำหนดการ

### <a name="cancel-setup"></a>ยกเลิกการตั้งค่า

เลือกกล่องกาเครื่องหมายนี้เพื่อตั้งค่าเวลาจากการจัดกำหนดการ

### <a name="cancel-process"></a>ยกเลิกกระบวนการ

เลือกกล่องกาเครื่องหมายนี้เพื่อรันเวลาจากการจัดกำหนดการ

### <a name="cancel-overlap"></a>ยกเลิกการทับซ้อน

เลือกกล่องกาเครื่องหมายนี้เพื่อทับซ้อนเวลาจากการจัดกำหนดการ

### <a name="cancel-transport"></a>ยกเลิกการขนส่ง

เลือกกล่องกาเครื่องหมายนี้เพื่อส่งต่อเวลาจากการจัดกำหนดการ

## <a name="set-default"></a>กำหนดค่าเริ่มต้น
คุณสามารถบันทึกค่าปัจจุบันเป็นค่าเริ่มต้น มีตัวเลือกสองตัวดังต่อไปนี้

-   ตั้งเป็นค่าเริ่มต้นของฉัน
-   ตั้งเป็นค่าเริ่มต้นสำหรับทุกคน
