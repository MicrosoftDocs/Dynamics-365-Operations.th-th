<span data-ttu-id="9653c-105">หัวข้อนี้อธิบายวิธีการจัดการการดำเนินงานที่ทำสัญญาย่อยใน Microsoft Dynamics 365 for Finance and Operations</span><span class="sxs-lookup"><span data-stu-id="9653c-105">This topic explains how subcontracted operations are managed in Microsoft Dynamics 365 for Finance and Operations.</span></span> <span data-ttu-id="9653c-106">อีกนัยหนึ่ง จะอธิบายวิธีจัดการการดำเนินงานการผลิตที่ปันส่วนให้กับทรัพยากรโดยผู้จัดจำหน่าย</span><span class="sxs-lookup"><span data-stu-id="9653c-106">In other words, it explains how production operations that are allocated to a resource are managed by a vendor.</span></span>

<span data-ttu-id="9653c-107">ใน [กระบวนการผลิต](production-process-overview.md) สามารถทำงานได้โดยใช้ทรัพยากรที่เป็นเจ้าของหรือจัดการโดยผู้จัดจำหน่าย</span><span class="sxs-lookup"><span data-stu-id="9653c-107">In [production processes](production-process-overview.md), work can be done by resources that are owned or administered by vendors.</span></span> <span data-ttu-id="9653c-108">โดยทั่วไป ระบบจะใช้ทรัพยากรผู้จัดจำหน่ายกับอุปสงค์ส่วนเกินเป็นครั้งคราวของระดับที่เกินกำลังการผลิตที่พร้อมใช้งานของทรัพยากรของบริษัทเอง</span><span class="sxs-lookup"><span data-stu-id="9653c-108">Typically, vendor resources are used to level periodic excess demand that surpasses the available capacity of a company's own resources.</span></span> <span data-ttu-id="9653c-109">ผู้จัดจำหน่ายอาจยังสามารถเสนอ [ความสามารถของทรัพยากร](resource-capabilities.md) เฉพาะ หรือทรัพยากรในราคาที่ต่ำกว่าได้ด้วย</span><span class="sxs-lookup"><span data-stu-id="9653c-109">The vendor might also be able to offer specific [resource capabilities](resource-capabilities.md) or resources at a lower price.</span></span>

## <a name="subcontracting-of-route-operations"></a><span data-ttu-id="9653c-135">การรับเหมารายย่อยของขั้นตอนของกระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="9653c-135">Subcontracting of route operations</span></span>

## <a name="material-supply-for-subcontracted-operations"></a><span data-ttu-id="9653c-180">การจัดหาวัสดุสำหรับการดำเนินงานรับเหมารายย่อย</span><span class="sxs-lookup"><span data-stu-id="9653c-180">Material supply for subcontracted operations</span></span>
