# <a name="formula-designer"></a><span data-ttu-id="340c1-103">โปรแกรมออกแบบสูตร</span><span class="sxs-lookup"><span data-stu-id="340c1-103">Formula designer</span></span>

[!include[banner](../includes/banner.md)]

<span data-ttu-id="340c1-104">หัวข้อนี้อธิบายวิธีการใช้โปรแกรมออกแบบสูตรเพื่อวิเคราะห์ และเก็บรักษาสูตรในมุมมองแผนภูมิ</span><span class="sxs-lookup"><span data-stu-id="340c1-104">This topic explains how to use the formula designer to analyze and maintain formulas in a tree view.</span></span>

<span data-ttu-id="340c1-105">เมื่อคุณเปิดหน้า **โปรแกรมการออกแบบสูตร** จากหน้า **ผลิตภัณฑ์ที่นำออกใช้** แผนภูมิในบานหน้าต่างด้านซ้ายจะแสดงรายการของสินค้าร่วมและลำดับชั้นของส่วนผสมสำหรับผลิตภัณฑ์นำออกใช้</span><span class="sxs-lookup"><span data-stu-id="340c1-105">When you open the **Formula designer** page from the **Released products** page, the tree in the left pane shows the list of co-products and the hierarchy of ingredients for the released product.</span></span> <span data-ttu-id="340c1-106">โครงสร้างได้รับมาจากลำดับชั้นของสูตรที่ใช้งานอยู่ และได้รับอนุมัติสำหรับสินค้าที่เลือก และส่วนผสมของโครงสร้าง ไซต์ของใบสั่งสินค้าเริ่มต้น และวันที่จริง</span><span class="sxs-lookup"><span data-stu-id="340c1-106">The structure is derived from the hierarchy of formulas that are active and approved for the selected item and its ingredients, the default order site of the item, and the actual date.</span></span>

<span data-ttu-id="340c1-107">คลิก **ตั้งค่า** เพื่อเลือกการตั้งค่าคอนฟิกและระบุข้อมูลที่ปรากฏในบรรทัดของแผนภูมิ</span><span class="sxs-lookup"><span data-stu-id="340c1-107">Click **Setup** to select different configurations and specify what information appears on the lines of the tree.</span></span>

<span data-ttu-id="340c1-108">คลิก **ตัวกรอง** เพื่อเปลี่ยนการเลือกในมุมมองเริ่มต้น</span><span class="sxs-lookup"><span data-stu-id="340c1-108">Click **Filter** to change the initial selection in the view.</span></span> <span data-ttu-id="340c1-109">ถ้าคุณตั้งค่าหลักการแสดงผลเป็น **ที่เลือก/ใช้งานอยู่** หรือ **ที่เลือก** คุณสามารถเลือกสูตรแต่ละสูตรหรืเวอร์ชันกระบวนการผลิตเพื่อใช้ในมุมมอง</span><span class="sxs-lookup"><span data-stu-id="340c1-109">If you set the display principle to **Selected/Active** or **Selected**, you can select individual formula or route versions to use in the view.</span></span> <span data-ttu-id="340c1-110">คุณสามารถเลือกสูตรของเวอร์ชันที่ไม่ได้รับการอนุมัติและไม่ได้ใช้งานอยู่เพื่อแสดงหรือรักษาในโปรแกรมออกแบบสูตร</span><span class="sxs-lookup"><span data-stu-id="340c1-110">You can select non-approved and non-active formula versions to show or maintain in the formula designer.</span></span>  

> [!NOTE]
> <span data-ttu-id="340c1-111">ถ้าคุณเปิดหน้า **โปรแกรมออกแบบสูตร** จากหน้ารายการ **สูตรการผลิต** จะไม่แสดงข้อมูลกระบวนการผลิต</span><span class="sxs-lookup"><span data-stu-id="340c1-111">If you open the **Formula designer** page from the **Bills of materials** list page, it doesn't show route information.</span></span> <span data-ttu-id="340c1-112">ในปัจจุบัน การเลือกเวอร์ชันสูตรหรือกระบวนการผลิตจะนำไปใช้กับอินสแตนซ์ทั้งหมดของโปรแกรมออกแบบสูตร</span><span class="sxs-lookup"><span data-stu-id="340c1-112">Currently, the selection of a formula or route version applies to all instances of the formula designer.</span></span>
