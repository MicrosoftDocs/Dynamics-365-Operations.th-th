<span data-ttu-id="3b61c-116">การจัดการเงินคืนช่วยให้คุณกําหนดโพรไฟล์การลงบัญชีส่วนสํารอง เงินคืน และการตัดบัญชี</span><span class="sxs-lookup"><span data-stu-id="3b61c-116">Rebate management lets you define posting profiles for provisions, rebates, and write-offs.</span></span> <span data-ttu-id="3b61c-117">นอกจากนี้ คุณยังสามารถกําหนดการลงรายการบัญชีให้กับลูกค้าหรือผู้จัดจำหน่ายเฉพาะรายหนึ่งๆ ได้</span><span class="sxs-lookup"><span data-stu-id="3b61c-117">Furthermore, the postings can be defined for a specific customer or vendor.</span></span> <span data-ttu-id="3b61c-118">ด้วยวิธีนี้ คุณสามารถติดตามข้อตกลงที่จะไม่ใช้กับลูกค้าหรือสถานการณ์สมทบผู้จัดจำหน่ายทั้งหมดผ่านทางการลงรายการบัญชีได้</span><span class="sxs-lookup"><span data-stu-id="3b61c-118">In this way, deals that don't apply to all customer or vendor scenarios can be tracked via postings.</span></span>

<span data-ttu-id="3b61c-119">ธุรกรรมเงินคืนจะถูกสร้างขึ้นโดยอัตโนมัติ ตามวิธีการคํานวณที่เลือกและจัดตารางการผลิตในชุดงาน</span><span class="sxs-lookup"><span data-stu-id="3b61c-119">Rebate transactions are automatically generated, based on the calculation method that is selected and scheduled in batch jobs.</span></span> <span data-ttu-id="3b61c-120">นอกจากนี้ คุณยังสามารถตั้งค่าลำดับงานเพื่อจัดการ ตรวจทาน และอนุมัติข้อตกลงได้</span><span class="sxs-lookup"><span data-stu-id="3b61c-120">Additionally, you can set up workflows to manage, review, and approve agreements.</span></span>
