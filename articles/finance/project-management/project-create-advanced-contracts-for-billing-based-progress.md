# <a name="create-advanced-contracts-for-billing-based-on-progress"></a>สร้างสัญญาขั้นสูงสำหรับการเรียกเก็บเงินตามความคืบหน้า
[!include [banner](../includes/banner.md)]

หัวข้อนี้จะอธิบายถึงวิธีการสร้างสัญญาโครงการ เพื่อให้คุณสามารถสร้างใบแจ้งหนี้สำหรับลูกค้า โดยยึดตามเปอร์เซ็นต์ของงานที่เสร็จสมบูรณ์ ยอดเงินในใบแจ้งหนี้จะถูกคำนวณโดยอัตโนมัติสำหรับประเภทงบประมาณของงานที่คุณตั้งค่าสำหรับโครงการ มีการตั้งค่าช่วงเวลาของใบแจ้งหนี้ เมื่อคุณเจรจาสัญญาโครงการกับลูกค้า

ใช้กระบวนงานในหัวข้อนี้เพื่อตั้งค่าสัญญา โครงการที่เกี่ยวข้อง และกฎการเรียกเก็บเงิน ที่คำนวณยอดเงินในใบแจ้งหนี้สำหรับประเภทงบประมาณของงานที่คุณตั้งค่าสำหรับโครงการ

หลังจากที่คุณได้สร้างสัญญาและโครงการแล้ว คุณสามารถตั้งค่ารายละเอียดของโครงการได้ ตัวอย่างเช่น คุณสามารถกำหนดกิจกรรมและกำหนดผู้ปฏิบัติงานให้แก่โครงการได้

## <a name="example"></a>ตัวอย่าง

องค์กรของคุณเป็นบริษัทพัฒนาซอฟต์แวร์ คุณตกลงที่จะพัฒนาแพ็คเกจการบัญชีค่าจ้างสำหรับลูกค้าสำหรับค่าธรรมเนียมรวม 20,000 ดอลลาร์สหรัฐ (USD) องค์กรของคุณตกลงที่จะออกใบแจ้งหนี้ไปให้ลูกค้า เมื่อคุณดำเนินการเปอร์เซ็นต์ที่ระบุของโครงการแล้วเสร็จ คุณตั้งค่าประเภทโครงการต่อไปนี้สำหรับงาน เพื่อให้คุณสามารถใช้ในกระบวนการเรียกเก็บเงินได้:

- การให้คำปรึกษา
- ออกแบบ
- การติดตั้ง

นอกจากนี้ คุณยังตั้งค่ากฎการเรียกเก็บเงินที่จะคำนวณยอดเงินใบแจ้งหนี้โดยอัตโนมัติ สำหรับเปอร์เซ็นต์ของงานในโครงการที่เสร็จสมบูรณ์ในแต่ละประเภท

ผู้จัดการงบประมาณสร้างงบประมาณสำหรับประเภทโครงการ ปริมาณของงานที่เสร็จสมบูรณ์จะถูกคำนวณเป็นเปอร์เซ็นต์ของงานที่เกิดขึ้นจริงเปรียบเทียบกับยอดเงินที่จัดงบประมาณโดยอัตโนมัติ

## <a name="prerequisites"></a>ข้อกำหนดเบื้องต้น

ก่อนที่คุณจะสร้างโครงการที่ใช้กฎการเรียกเก็บเงิน คุณต้องตั้งค่าลำดับหมายเลขสำหรับกฎการเรียกเก็บเงินและสมุดรายวันค่าธรรมเนียมที่ใช้ในการลงรายการบัญชีการเรียกเก็บเงินตามความคืบหน้า

1. ไปที่ **การจัดการและการบัญชีโครงการ** \> **การตั้งค่า** \> **พารามิเตอร์การจัดการและการบัญชีโครงการ**
2. บนหน้า **พารามิเตอร์การจัดการและการบัญชีโครงการ** บนแท็บ **ลำดับหมายเลข** ให้ตั้งค่าลำดับหมายเลขที่คุณต้องการใช้ เมื่อมีการสร้างกฎการเรียกเก็บเงิน
3. ไปยัง **การจัดการและการบัญชีโครงการ** \> **สมุดรายวัน** \> **ค่าธรรมเนียม**
4. บนหน้า **สมุดรายวันค่าธรรมเนียม** เลือก **สร้าง** แล้วป้อนชื่อสมุดรายวัน

## <a name="create-a-contract-for-progress-billings"></a>สร้างสัญญาสำหรับการเรียกเก็บเงินตามความคืบหน้า

ใช้กระบวนงานนี้เพื่อสร้างสัญญาโครงการสำหรับโครงการที่มีราคาคงที่ คุณสร้างใบแจ้งหนี้โครงการ เมื่องานที่เสร็จสมบูรณ์ในโครงการไปถึงเปอร์เซ็นต์ที่ระบุ

1. ไปที่ **การจัดการและการบัญชีโครงการ** \> **โครงการ** \> **สัญญาโครงการ**
2. ในหน้า **สัญญาโครงการ** เลือก **สร้าง**
3. ในกล่องโต้ตอบ **สัญญาโครงการใหม่** ตั้งค่าฟิลด์ต่อไปนี้:

    - **ชื่อ**
    - **ชนิดของการจัดหาเงินทุน**
    - **แหล่งเงินทุน**
    - **สกุลเงินที่ใช้ในการขาย** – โดยค่าเริ่มต้น สกุลเงินนี้จะใช้สำหรับใบแจ้งหนี้ของลูกค้าที่เชื่อมโยงกับสัญญาโครงการ อย่างไรก็ตาม คุณสามารถเปลี่ยนสกุลเงินที่ใช้ในการขายในใบแจ้งหนี้ของลูกค้าที่ระบุได้

4. เลือก **ตกลง** ข้อมูลจะถูกคัดลอกไปยังส่วนหัวของหน้า **สัญญาโครงการ**
5. บนหน้า **สัญญาโครงการ** ให้กรอกข้อมูลที่จำเป็นส่วนที่เหลือสำหรับโครงการ

## <a name="create-a-project-for-progress-billings"></a>สร้างโครงการสำหรับการเรียกเก็บเงินตามความคืบหน้า

ทำตามขั้นตอนเหล่านี้เพื่อสร้างโครงการและโครงการย่อยใดๆ ที่เชื่อมโยงกับสัญญาโครงการ

1. ไปที่ **การจัดการและการบัญชีโครงการ** \> **โครงการ** \> **โครงการทั้งหมด**
2. ในหน้า **โครงการทั้งหมด** เลือก **สร้าง**
3. ในกล่องโต้ตอบ **โครงการใหม่** ในฟิลด์ **ชนิดโครงการ** ให้เลือก **เวลาและวัสดุ**
4. เลือกกลุ่มโครงการ กลุ่มโครงการกำหนดข้อมูลการลงรายการบัญชีสำหรับโครงการที่ถูกกำหนดให้กับกลุ่ม
5. เลือก **สร้างโครงการ**
6. หลังจากที่มีการสร้างโครงการแล้ว ให้ตั้งค่าขั้นตอนของโครงการเป็น **อยู่ระหว่างดำเนินการ**

## <a name="create-a-budget-for-a-project"></a>สร้างงบประมาณสำหรับโครงการ

ประเภทงบประมาณถูกใช้ในการคำนวณยอดเงินใบแจ้งหนี้สำหรับเปอร์เซ็นต์ของงานที่เสร็จสมบูรณ์สำหรับแต่ละประเภท ทำตามขั้นตอนต่อไปนี้เพื่อสร้างประเภทงบประมาณสำหรับต้นทุนที่ประเมิน

1. ไปที่ **การจัดการและการบัญชีโครงการ** \> **โครงการ** \> **โครงการทั้งหมด**
2. บนหน้า **โครงการทั้งหมด** ให้เลือกและเปิดโครงการที่คุณต้องการ
3. บนหน้า **โครงการ** บนบานหน้าต่างการดำเนินการ บนแท็บ **แผน** ในกลุ่ม **งบประมาณ** ให้เลือก **งบประมาณโครงการ**
4. บนหน้า **งบประมาณโครงการ** ให้ป้อนต้นทุนที่ประเมินสำหรับประเภทแต่ละประเภทในโครงการ

## <a name="create-billing-rules-for-progress-billings"></a>สร้างกฎการเรียกเก็บเงินสำหรับการเรียกเก็บเงินตามความคืบหน้า

1. ไปที่ **การจัดการและการบัญชีโครงการ** \> **โครงการ** \> **สัญญาโครงการ**
2. บนหน้า **สัญญาโครงการ** ให้เลือกและเปิดสัญญาโครงการ
3. บนหน้าสัญญาโครงการ บนแท็บ FastTab **กฎการเรียกเก็บเงิน** ให้เลือก **เพิ่ม**
4. บนหน้า **กฎการเรียกเก็บเงิน** ในฟิลด์ **ชนิดรายการ** เลือก **ความก้าวหน้า**
5. บน FastTab **รายละเอียดของรายการกฎการเรียกเก็บเงิน** ในฟิลด์ **มูลค่าตามสัญญา** ให้ป้อนมูลค่ารวมของสัญญา
6. ในฟิลด์ **ประเภท** ให้เลือกประเภทที่จะลงรายการบัญชีธุรกรรมค่าธรรมเนียม
7. ในฟิลด์ **โครงการ** ให้เลือกโครงการที่ใช้กฎการเรียกเก็บเงินนี้
8. ไม่จำเป็น: กำหนดกฎการเรียกเก็บเงินให้แก่โครงการเพิ่มเติม บน FastTab **โครงการ** ในส่วน **โครงการที่พร้อมใช้งาน** ให้เลือกโครงการ แล้วเลือกปุ่มลูกศรขวาเพื่อเพิ่มโครงการในส่วน **โครงการที่เลือก**
9. ไม่จำเป็น: คำนวณยอดเงินเปอร์เซ็นต์ที่ลูกค้าหักจากเงินที่ชำระในใบแจ้งหนี้ บน FastTab **เงื่อนไขเงินประกันผลงานสำหรับการชำระเงิน** ให้เลือกแหล่งเงินทุน และจากนั้นในฟิลด์ **เปอร์เซ็นต์เงินวางประกัน** ให้ป้อนเปอร์เซ็นต์ของเงินวางประกัน
10. ทำซ้ำขั้นตอนเหล่านี้เพื่อสร้างกฎการเรียกเก็บเงินเพิ่มเติมสำหรับสัญญาโครงการ
