# <a name="create-a-credit-note-on-project-invoices"></a>สร้างใบลดหนี้ในใบแจ้งหนี้ของโครงการ

[!include [banner](../../includes/banner.md)]

1. ไปที่การจัดการโครงการและการบัญชี > โครงการ > โครงการทั้งหมด 
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ 
3. เลือกรหัสโครงการที่ถูกใช้สำหรับใบลดหนี้ ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก 
4. ในบานหน้าต่างการดำเนินการ คลิก จัดการ 
5. คลิกสมุดรายวันใบแจ้งหนี้ 
6. เลือกธุรกรรมสำหรับใบลดหนี้ ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก 
7. คลิกฟังก์ชัน 
8. คลิก เลือกสำหรับใบลดหนี้ 
9. เลือกธุรกรรมสำหรับใบลดหนี้โดยการทำเครื่องหมายกล่องกาเครื่องหมาย
10. คลิก ตกลง 

หลังจากขั้นตอนนี้ ใบลดหนี้จะพร้อมใช้งานสำหรับข้อเสนอใบแจ้งหนี้
