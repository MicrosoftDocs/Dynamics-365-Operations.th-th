# <a name="transfer-a-quotation-to-a-project"></a>โอนย้ายใบเสนอราคาไปยังโครงการ

[!include [banner](../includes/banner.md)]

เมื่อใบเสนอราคาโครงการมีสถานะเป็น **ยืนยันแล้ว** คุณสามารถโอนย้ายบรรทัดใบเสนอราคาบางบรรทัดหรือทั้งหมดไปยังโครงการใหม่หรือโครงการที่มีอยู่ได้ด้วยการใช้วิซาร์ด **โอนย้ายไปยังโครงการ** 

เพื่อใช้วิซาร์ด ให้ทำตามขั้นตอนเหล่านี้

1. ไปที่ **การจัดการและการบัญชีโครงการ** > **ใบเสนอราคา** > **ใบเสนอราคาโครงการ**
2. เลือกใบเสนอราคาที่คุณต้องการโอนย้ายและบนแท็บ **ติดตามผล** ในกลุ่ม **ปรับเปลี่ยน** ให้เลือก **โอนย้ายไปยังโครงการ**
3. ทำตามวิซาร์ด **โอนย้ายใบเสนอราคาไปยังโครงการ** เพื่อเชื่อมโยงและโอนย้ายใบเสนอราคากับโครงการ
