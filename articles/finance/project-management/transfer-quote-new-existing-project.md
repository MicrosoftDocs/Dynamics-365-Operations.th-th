# <a name="transfer-a-quotation-to-a-project"></a><span data-ttu-id="1561e-103">โอนย้ายใบเสนอราคาไปยังโครงการ</span><span class="sxs-lookup"><span data-stu-id="1561e-103">Transfer a quotation to a project</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="1561e-104">เมื่อใบเสนอราคาโครงการมีสถานะเป็น **ยืนยันแล้ว** คุณสามารถโอนย้ายบรรทัดใบเสนอราคาบางบรรทัดหรือทั้งหมดไปยังโครงการใหม่หรือโครงการที่มีอยู่ได้ด้วยการใช้วิซาร์ด **โอนย้ายไปยังโครงการ**</span><span class="sxs-lookup"><span data-stu-id="1561e-104">When a project quotation has a status of **Confirmed**, you can transfer some or all the quotation lines to a new or existing project by using the **Transfer to project** wizard.</span></span> 

<span data-ttu-id="1561e-105">เพื่อใช้วิซาร์ด ให้ทำตามขั้นตอนเหล่านี้</span><span class="sxs-lookup"><span data-stu-id="1561e-105">To use the wizard, follow these steps.</span></span>

1. <span data-ttu-id="1561e-106">ไปที่ **การจัดการและการบัญชีโครงการ** > **ใบเสนอราคา** > **ใบเสนอราคาโครงการ**</span><span class="sxs-lookup"><span data-stu-id="1561e-106">Go to **Project management and accounting** > **Quotations** > **Project quotations**.</span></span>
2. <span data-ttu-id="1561e-107">เลือกใบเสนอราคาที่คุณต้องการโอนย้ายและบนแท็บ **ติดตามผล** ในกลุ่ม **ปรับเปลี่ยน** ให้เลือก **โอนย้ายไปยังโครงการ**</span><span class="sxs-lookup"><span data-stu-id="1561e-107">Select the quotation you want to transfer and on the **Follow up** tab, in the **Modify** group, select **Transfer to project**.</span></span>
3. <span data-ttu-id="1561e-108">ทำตามวิซาร์ด **โอนย้ายใบเสนอราคาไปยังโครงการ** เพื่อเชื่อมโยงและโอนย้ายใบเสนอราคากับโครงการ</span><span class="sxs-lookup"><span data-stu-id="1561e-108">Follow the **Transfer quotation to project** wizard to associate and transfer the quotation with a project.</span></span>
