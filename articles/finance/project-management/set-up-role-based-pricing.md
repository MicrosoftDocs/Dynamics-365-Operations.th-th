# <a name="set-up-role-based-pricing"></a>ตั้งค่าการกำหนดราคาตามบทบาท

[!include [banner](../includes/banner.md)]

ต้นทุน การขาย และการโอนย้ายทั้งหมด สามารถถูกตั้งค่าสำหรับบทบาท ดำเนินการขั้นตอนต่อไปนี้เพื่อเชื่อมโยงราคากับบทบาทเฉพาะ

1. ในหน้า **ราคาขาย (ชั่วโมง)** เลือก **สร้าง** และป้อนวันที่มีผลบังคับใช้
2. ในคอลัมน์ **บทบาท** เลือกบทบาท
3. ในคอลัมน์ **การกำหนดราคา** ให้ป้อนราคาสำหรับบทบาททรัพยากรที่เลือก
