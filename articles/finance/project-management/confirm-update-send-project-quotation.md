<span data-ttu-id="cdb0a-104">หลังจากที่คุณสร้างและส่งใบเสนอราคาโครงการให้กับลูกค้าแล้ว คุณต้องได้รับการยืนยันจากลูกค้าก่อนที่คุณจะอัพเดตสถานะใบเสนอราคาเป็น **ส่งแล้ว**</span><span class="sxs-lookup"><span data-stu-id="cdb0a-104">After you create and send a project quotation to a customer, you must get confirmation from the customer before you update the quotation status to **Sent**.</span></span> <span data-ttu-id="cdb0a-105">สามารถอัพเดตการปรับเปลี่ยนใด ๆ ที่ลูกค้าร้องขอในใบเสนอราคาได้</span><span class="sxs-lookup"><span data-stu-id="cdb0a-105">Any modifications requested by the customer can be updated in the quotation.</span></span> <span data-ttu-id="cdb0a-106">หลังจากคุณอัพเดตสถานะใบเสนอราคาเป็น **ส่งแล้ว** คุณจะไม่สามารถเปลี่ยนแปลงใบเสนอราคาได้</span><span class="sxs-lookup"><span data-stu-id="cdb0a-106">After you update the quotation status to **Sent**, you can’t modify it.</span></span> <span data-ttu-id="cdb0a-107">ขั้นตอนต่อไปนี้จะอธิบายถึงตัวเลือกในการส่งใบเสนอราคาสำหรับ การยืนยัน การทำการอัพเดตโดยใช้คำติชม และส่งใบเสนอราคา</span><span class="sxs-lookup"><span data-stu-id="cdb0a-107">The following procedure describes the options to send a quotation for confirmation, make updates based on feedback, and then send the quotation.</span></span>

## <a name="send-a-project-quotation-confirmation"></a><span data-ttu-id="cdb0a-108">ส่งการยืนยันใบเสนอราคาโครงการ</span><span class="sxs-lookup"><span data-stu-id="cdb0a-108">Send a project quotation confirmation</span></span>  

<span data-ttu-id="cdb0a-109">คุณสามารถส่งใบเสนอราคาโครงการที่มีอยู่ สำหรับการยืนยันผ่านทางอีเมล์หรือเป็นสำเนากระดาษ</span><span class="sxs-lookup"><span data-stu-id="cdb0a-109">You can send an existing project quotation for confirmation through email or as a printed copy.</span></span> 

1. <span data-ttu-id="cdb0a-110">ไปที่ **การจัดการและการบัญชีโครงการ** > **ใบเสนอราคา** > **ใบเสนอราคาทั้งหมด**</span><span class="sxs-lookup"><span data-stu-id="cdb0a-110">Go to **Project management and accounting** > **Quotations** > **Project quotation.**</span></span> 
2. <span data-ttu-id="cdb0a-111">บนหน้า **ใบเสนอราคาโครงการ** ให้เลือกใบเสนอราคาที่คุณต้องการส่งเพื่อยืนยัน</span><span class="sxs-lookup"><span data-stu-id="cdb0a-111">On the **Project quotation** page, select the quotation you want to send for confirmation.</span></span> 
3. <span data-ttu-id="cdb0a-112">บนแท็บ **ติดตามผล** ในกลุ่ม **สร้าง** ให้เลือก **ยืนยัน**</span><span class="sxs-lookup"><span data-stu-id="cdb0a-112">On the **Follow up** tab, in the **Generate** group, select **Confirm**.</span></span> 
4. <span data-ttu-id="cdb0a-113">บนหน้า **ยืนยันใบเสนอราคา** ให้ตั้งค่าพารามิเตอร์ที่จำเป็น</span><span class="sxs-lookup"><span data-stu-id="cdb0a-113">On the **Confirm quotation** page, set the required parameters.</span></span> <span data-ttu-id="cdb0a-114">ตัวอย่างเช่น ถ้าต้องการพิมพ์การยืนยัน ภายใต้ตัวเลือก **พิมพ์** ให้เปิด **การยืนยันการพิมพ์** แล้วเลือก **ตกลง**</span><span class="sxs-lookup"><span data-stu-id="cdb0a-114">For example, to print the confirmation, under **Print** options, turn on **Print confirmation**, and then select **OK**.</span></span>
5. <span data-ttu-id="cdb0a-115">บนหน้า **ส่งใบเสนอราคา** เลือก **ตัวเลือก** และบนหน้า **การตั้งค่าปลายทางการพิมพ์** ให้เลือกว่าคุณต้องการให้ส่งใบเสนอราคาไปยัง **หน้าจอ** **อีเมล** **ไฟล์** **ที่เก็บถาวรของเอกสารที่พิมพ์** หรือ **เครื่องพิมพ์** และทำการปรับปรุงใด ๆ ที่คุณต้องการในพื้นที่ **ข้อมูลจำเพาะ** เพื่อกำหนดเส้นทางใบเสนอราคา</span><span class="sxs-lookup"><span data-stu-id="cdb0a-115">On the **Send quotation** page, select **Options**, and on the **Print destination settings** page, select whether you want the quotation sent to **Screen**, **E-mail**, **File**, **Print archive**, or **Printer**, and then make any adjustments that you want in the **Specification** area to route the quotation.</span></span>
6. <span data-ttu-id="cdb0a-116">บนหน้า **การตั้งค่าปลายทางการพิมพ์** เลือก **ตกลง**</span><span class="sxs-lookup"><span data-stu-id="cdb0a-116">On the **Print destination settings** page, select **OK**.</span></span>  

## <a name="update-a-project-quotation"></a><span data-ttu-id="cdb0a-117">การอัพเดตใบเสนอราคาโครงการ</span><span class="sxs-lookup"><span data-stu-id="cdb0a-117">Update a project quotation</span></span>

<span data-ttu-id="cdb0a-118">เมื่อต้องการปรับเปลี่ยนใบเสนอราคาโครงการที่มีอยู่ สถานะของใบเสนอราคาต้องเป็น **สร้างแล้ว**</span><span class="sxs-lookup"><span data-stu-id="cdb0a-118">To modify an existing project quotation, the quotation status must be **Created**.</span></span> <span data-ttu-id="cdb0a-119">ดำเนินการขั้นตอนต่อไปนี้ให้เสร็จ เพื่ออัพเดตใบเสนอราคาที่มีอยู่</span><span class="sxs-lookup"><span data-stu-id="cdb0a-119">Complete the following steps to update an existing quotation.</span></span> 

1. <span data-ttu-id="cdb0a-120">ไปที่ **การจัดการและการบัญชีโครงการ** > **ใบเสนอราคา** > **ใบเสนอราคาโครงการ**</span><span class="sxs-lookup"><span data-stu-id="cdb0a-120">Go to **Project management and accounting** > **Quotations** > **Project quotations**.</span></span>
2. <span data-ttu-id="cdb0a-121">บนหน้า **ใบเสนอราคาโครงการ** ให้เลือกใบเสนอราคาที่คุณต้องการอัพเดต และในบานหน้าต่างการดำเนินการให้เลือก **แก้ไข**</span><span class="sxs-lookup"><span data-stu-id="cdb0a-121">On the **Project quotations** page, select the quotation you want to update and on the Action Pane, select **Edit**.</span></span>
3. <span data-ttu-id="cdb0a-122">อัพเดตข้อมูลในฟิลด์ที่จำเป็น และในบานหน้าต่างการดำเนินการให้เลือก **บันทึก**</span><span class="sxs-lookup"><span data-stu-id="cdb0a-122">Update the necessary field information, and on the Action Pane, select **Save**.</span></span>

> <span data-ttu-id="cdb0a-127">คุณไม่สามารถแก้ไขใบเสนอราคาโครงการ หลังจากที่มีการเปลี่ยนแปลงสถานะเป็น **ส่งแล้ว**</span><span class="sxs-lookup"><span data-stu-id="cdb0a-127">You can’t modify a project quotation after the status is changed to **Sent**.</span></span>
