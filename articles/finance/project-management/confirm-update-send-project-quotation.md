# <a name="confirm-update-and-send-a-project-quotation"></a>ยืนยัน อัพเดต และส่งใบเสนอราคาโครงการ

[!include [banner](../includes/banner.md)]

หลังจากที่คุณสร้างและส่งใบเสนอราคาโครงการให้กับลูกค้าแล้ว คุณต้องได้รับการยืนยันจากลูกค้าก่อนที่คุณจะอัพเดตสถานะใบเสนอราคาเป็น **ส่งแล้ว** สามารถอัพเดตการปรับเปลี่ยนใด ๆ ที่ลูกค้าร้องขอในใบเสนอราคาได้ หลังจากคุณอัพเดตสถานะใบเสนอราคาเป็น **ส่งแล้ว** คุณจะไม่สามารถเปลี่ยนแปลงใบเสนอราคาได้ ขั้นตอนต่อไปนี้จะอธิบายถึงตัวเลือกในการส่งใบเสนอราคาสำหรับ การยืนยัน การทำการอัพเดตโดยใช้คำติชม และส่งใบเสนอราคา

## <a name="send-a-project-quotation-confirmation"></a>ส่งการยืนยันใบเสนอราคาโครงการ  

คุณสามารถส่งใบเสนอราคาโครงการที่มีอยู่ สำหรับการยืนยันผ่านทางอีเมล์หรือเป็นสำเนากระดาษ 

1. ไปที่ **การจัดการและการบัญชีโครงการ** > **ใบเสนอราคา** > **ใบเสนอราคาทั้งหมด** 
2. บนหน้า **ใบเสนอราคาโครงการ** ให้เลือกใบเสนอราคาที่คุณต้องการส่งเพื่อยืนยัน 
3. บนแท็บ **ติดตามผล** ในกลุ่ม **สร้าง** ให้เลือก **ยืนยัน** 
4. บนหน้า **ยืนยันใบเสนอราคา** ให้ตั้งค่าพารามิเตอร์ที่จำเป็น ตัวอย่างเช่น ถ้าต้องการพิมพ์การยืนยัน ภายใต้ตัวเลือก **พิมพ์** ให้เปิด **การยืนยันการพิมพ์** แล้วเลือก **ตกลง**
5. บนหน้า **ส่งใบเสนอราคา** เลือก **ตัวเลือก** และบนหน้า **การตั้งค่าปลายทางการพิมพ์** ให้เลือกว่าคุณต้องการให้ส่งใบเสนอราคาไปยัง **หน้าจอ** **อีเมล** **ไฟล์** **ที่เก็บถาวรของเอกสารที่พิมพ์** หรือ **เครื่องพิมพ์** และทำการปรับปรุงใด ๆ ที่คุณต้องการในพื้นที่ **ข้อมูลจำเพาะ** เพื่อกำหนดเส้นทางใบเสนอราคา
6. บนหน้า **การตั้งค่าปลายทางการพิมพ์** เลือก **ตกลง**  

## <a name="update-a-project-quotation"></a>การอัพเดตใบเสนอราคาโครงการ

เมื่อต้องการปรับเปลี่ยนใบเสนอราคาโครงการที่มีอยู่ สถานะของใบเสนอราคาต้องเป็น **สร้างแล้ว** ดำเนินการขั้นตอนต่อไปนี้ให้เสร็จ เพื่ออัพเดตใบเสนอราคาที่มีอยู่ 

1. ไปที่ **การจัดการและการบัญชีโครงการ** > **ใบเสนอราคา** > **ใบเสนอราคาโครงการ**
2. บนหน้า **ใบเสนอราคาโครงการ** ให้เลือกใบเสนอราคาที่คุณต้องการอัพเดต และในบานหน้าต่างการดำเนินการให้เลือก **แก้ไข**
3. อัพเดตข้อมูลในฟิลด์ที่จำเป็น และในบานหน้าต่างการดำเนินการให้เลือก **บันทึก**  

## <a name="send-a-project-quotation-to-a-customer"></a>ส่งใบเสนอราคาโครงการไปยังลูกค้า 

1. ไปที่ **ใบเสนอราคาโครงการสำหรับการจัดการโครงการและ** > **ใบเสนอราคา** > **ใบเสนอราคาโครงการ** และเลือกใบเสนอราคาโครงการที่คุณต้องการส่ง
2. บนหน้า **ใบเสนอราคาโครงการ** บนแท็บ **ใบเสนอราคา** ในกลุ่ม **กระบวนการ** ให้เลือก **ส่งใบเสนอราคา** อัพเดตสถานะใบเสนอราคาเป็น **ส่งแล้ว**

> [!NOTE]
> คุณไม่สามารถแก้ไขใบเสนอราคาโครงการ หลังจากที่มีการเปลี่ยนแปลงสถานะเป็น **ส่งแล้ว**
