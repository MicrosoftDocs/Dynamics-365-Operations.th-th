[!include [banner](../includes/banner.md)]

<span data-ttu-id="a4c8e-104">ผู้จัดการโครงการสามารถวางแผนบทบาททรัพยากรที่จำเป็นสำหรับโครงการ</span><span class="sxs-lookup"><span data-stu-id="a4c8e-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="a4c8e-105">ผู้จัดการทรัพยากรจะมองเห็นทรัพยากรที่วางแผนไว้เหล่านี้เป็นการร้องขอบนหน้า **การเติมสินค้าทรัพยากร** และสามารถกำหนดทรัพยากรจริงได้</span><span class="sxs-lookup"><span data-stu-id="a4c8e-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="a4c8e-106">ในหน้า **โครงการทั้งหมด** เลือกโครงการ **การอัปเกรด XYZ ระยะที่ 2**</span><span class="sxs-lookup"><span data-stu-id="a4c8e-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="a4c8e-107">เลือก **โครงการ** และจากนั้นเลือก **แก้ไข**</span><span class="sxs-lookup"><span data-stu-id="a4c8e-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="a4c8e-108">ในแท็บ **ทีมงานและการจัดกำหนดการโครงการ** เลือก **เพิ่ม**</span><span class="sxs-lookup"><span data-stu-id="a4c8e-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="a4c8e-109">ในกล่องโต้ตอบ **เพิ่มบทบาท** เลือกบทบาท **นักพัฒนาซอฟต์แวร์**</span><span class="sxs-lookup"><span data-stu-id="a4c8e-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="a4c8e-110">เลือก **สร้าง** และจากนั้นปิดหน้าโครงการ</span><span class="sxs-lookup"><span data-stu-id="a4c8e-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="a4c8e-111">ในหน้า **การเติมทรัพยากร** เลือก **นักพัฒนาซอฟต์แวร์ 1** สำหรับโครงการ **โครงการการอัปเกรด XYZ ระยะที่ 2**</span><span class="sxs-lookup"><span data-stu-id="a4c8e-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="a4c8e-112">เลือกผู้ปฏิบัติงาน แล้วเลือก **มอบหมาย**</span><span class="sxs-lookup"><span data-stu-id="a4c8e-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="a4c8e-113">ตรวจสอบว่า รายการสำหรับ **นักพัฒนาซอฟต์แวร์ 1** ได้ถูกลบออกสำหรับโครงการ **โครงการการอัปเกรด XYZ ระยะที่ 2**</span><span class="sxs-lookup"><span data-stu-id="a4c8e-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="a4c8e-114">ในแท็บ **ทีมโครงการและการจัดกำหนดการ** สำหรับโครงการ **โครงการการอัปเกรด XYZ ระยะที่ 2** ตรวจสอบว่าผู้ปฏิบัติงานที่คุณเลือกในขั้นตอนก่อนหน้าได้ถูกเพิ่มเป็น **นักพัฒนาซอฟต์แวร์**</span><span class="sxs-lookup"><span data-stu-id="a4c8e-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
