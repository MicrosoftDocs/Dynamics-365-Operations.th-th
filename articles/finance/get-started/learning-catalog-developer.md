# <a name="learning-catalog-for-dynamics-365-finance-developers"></a>แค็ตตาล็อกการเรียนรู้ของนักพัฒนา Dynamics 365 Finance

คุณจำเป็นต้องเขียนรหัสเพื่อรวมกับแหล่งข้อมูลอื่นๆ ขยายการทำงานของระบบหลัก หรือสร้างแอพลิเคชันที่ซับซ้อนหรือไม่

แค็ตตาล็อกต่อไปนี้มีการจัดระเบียบจากความรู้หลักให้กับโดเมนเฉพาะและจากการทำงานพื้นฐานถึงขั้นสูงที่สุด ถ้ามีเนื้อหาอยู่ในรูปแบบเราจะแจ้งให้คุณทราบเพื่อให้คุณสามารถเลือกรูปแบบการฝึกอบรมที่ตรงกับความต้องการของคุณได้ดีที่สุด
