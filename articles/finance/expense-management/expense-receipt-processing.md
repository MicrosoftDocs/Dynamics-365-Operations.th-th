# <a name="expense-receipt-processing"></a>การประมวลผลใบเสร็จค่าใช้จ่าย

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]


มีการปรับปรุงรายการค่าใช้จ่ายโดยใช้การแนะนำในการประมวลผลการรู้จำอักขระด้วยแสง (OCR) สำหรับใบเสร็จรับเงิน คุณลักษณะนี้ได้รับการออกแบบมาเพื่อปรับปรุงประสบการณ์การใช้งานของผู้ใช้ เมื่อมีการสร้างรายงานค่าใช้จ่าย

## <a name="key-features"></a>คุณลักษณะสำคัญ

- ชื่อร้านค้า วันที่ และยอดเงินรวม จะถูกแยกจากใบเสร็จรับเงิน
- คุณลักษณะพยายามจับคู่ใบเสร็จรับเงินที่ยังไม่ได้แนบกับธุรกรรมค่าใช้จ่ายที่ยังไม่ได้แนบ
- ผู้ใช้สามารถสร้างธุรกรรมค่าใช้จ่ายที่ป้อนด้วยตนเองจากใบเสร็จรับเงิน

## <a name="usage-examples"></a>ตัวอย่างการใช้

- **แนบใบเสร็จรับเงินที่มีธุรกรรมบัตรเครดิตโดยอัตโนมัติ เมื่อสร้างรายงานค่าใช้จ่าย**

    1. เปิดพื้นที่ทำงาน **การจัดการค่าใช้จ่าย**
    2. บนแท็บ **ใบเสร็จรับเงิน** ให้ตรวจสอบว่ามีใบเสร็จรับเงินที่ยังไม่ได้แนบอยู่ นอกจากนี้ คุณยังสามารถอัพโหลดใบเสร็จรับเงินในแท็บ **ใบเสร็จรับเงิน**
    3. บนแท็บ **ค่าใช้จ่าย** ให้ตรวจสอบว่ามีค่าใช้จ่ายอยู่หรือไม่ โดยทั่วไป ผู้ดูแลระบบค่าใช้จ่ายจะนำเข้าค่าใช้จ่ายเหล่านี้จากผู้ให้บริการบัตรเครดิต
    4. เลือก **รายงานค่าใช้จ่ายใหม่** โปรดสังเกตว่าคุณสามารถรวมค่าใช้จ่ายและใบเสร็จรับเงินในขณะนี้ได้เช่นกัน เมื่อคุณสร้างรายงานค่าใช้จ่าย ถ้าคุณเพิ่มทั้งค่าใช้จ่ายและใบเสร็จรับเงิน การจับคู่อัตโนมัติของใบเสร็จรับเงินกับค่าใช้จ่ายจะถูกทริกเกอร์

- **สร้างค่าใช้จ่าย หรือจับคู่ค่าใช้จ่ายจากใบเสร็จรับเงิน**

    1. ในรายงานค่าใช้จ่าย บนแท็บ **ใบเสร็จรับเงิน** ให้แนบใบเสร็จรับเงินโดยเลือก **เพิ่มใบเสร็จรับเงิน**
    2. ภายใต้รูปที่อัพโหลดของใบเสร็จรับเงิน ให้สังเกตตัวเลือก **สร้าง** และ **จับคู่**

        - เลือก **สร้าง** เพื่อสร้างธุรกรรมค่าใช้จ่ายที่ป้อนด้วยตนเอง และกรอกค่าที่ถูกแยกจากใบเสร็จรับเงิน
        - ถ้าคุณเลือก **จับคู่** ระบบจะพยายามจับคู่ค่าใช้จ่ายที่มีอยู่กับใบเสร็จรับเงิน

## <a name="installation"></a>การติดตั้ง

คุณลักษณะนี้จะทำงานร่วมกับคุณลักษณะ **รายงานค่าใช้จ่ายที่คิดใหม่** เพื่อช่วยให้ประสบการณ์ค่าใช้จ่ายง่ายขึ้น

เมื่อต้องการใช้ความสามารถในการใช้ค่าใช้จ่ายขั้นสูงเหล่านี้ ให้ติดตั้ง Add-in บริการการจัดการค่าใช้จ่ายสำหรับ Microsoft Dynamics 365 Finance และเปิดใช้งานคุณลักษณะในอินสแตนซ์ของคุณ คุณสามารถเข้าถึง add-in จากโครงการของคุณใน Microsoft Dynamics Lifecycle Services (LCS).

1. ลงชื่อเข้าใช้ LCS และเปิดสภาพแวดล้อมที่ต้องการ
2. ไปที่ **รายละเอียดแบบเต็ม**
3. เลือก **เก็บรักษา** หรือเลื่อนลงไปที่แท็บด่วน **Add-in ของสภาพแวดล้อม**
4. เลือก **การติดตั้ง Add-in ใหม่**
5. เลือก **บริการจัดการค่าใช้จ่าย**
6. ปฏิบัติตามคำแนะนำการติดตั้ง และยอมรับข้อกำหนดและเงื่อนไข
7. เลือก **ติดตั้ง**

ในพื้นที่ทำงาน **การจัดการคุณลักษณะ** เปิดใช้งานคุณลักษณะต่อไปนี้:

- คิดรูปแบบรายงานค่าใช้จ่ายใหม่แล้ว
- จับคู่อัตโนมัติและสร้างค่าใช้จ่ายจากใบเสร็จ

เมื่อคุณเปิดใช้งานคุณลักษณะเหล่านี้ การดำเนินการต่อไปนี้จะเกิดขึ้น:

- พื้นที่ทำงาน **การจัดการค่าใช้จ่าย** ที่มีอยู่จะถูกแทนที่ด้วยพื้นที่ทำงานใหม่
- มีการเพิ่มรายการเมนูใหม่สำหรับการมองเห็นฟิลด์ค่าใช้จ่าย
- คุณยังสามารถเปิดหน้า **รายงานค่าใช้จ่าย** ก่อนหน้านี้ โดยไปที่ **การจัดการค่าใช้จ่าย > ค่าใช้จ่ายของฉัน > รายงานค่าใช้จ่าย**
- ลำดับงานและการอนุมัติใดๆ ยังคงนำคุณไปยังหน้ารายงานค่าใช้จ่ายที่มีอยู่
- ใบเสร็จรับเงินจะถูกประมวลผลโดยผ่านทาง Microsoft Azure Cognitive Services และข้อมูลเมตาจะถูกแยกและเพิ่ม
- มีการเพิ่มตัวเลือกซึ่งช่วยให้คุณสามารถสร้างรายงานค่าใช้จ่ายที่มีใบเสร็จรับเงินที่จับคู่ที่ยังไม่ได้ถูกแนบ
- ตัวเลือกที่บวกเพิ่มเข้าในรายงานค่าใช้จ่ายจะช่วยให้คุณสามารถสร้างรายการค่าใช้จ่ายจากใบเสร็จรับเงิน หรือพยายามจับคู่ใบเสร็จรับเงินที่มีอยู่กับรายการค่าใช้จ่ายที่มีอยู่

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับคุณลักษณะที่คิดรูปแบบรายงานค่าใช้จ่ายใหม่แล้ว ดู [คิดรูปแบบรายงานค่าใช้จ่ายใหม่แล้ว](ExpenseWorkspaceNew.md)

## <a name="frequently-asked-questions"></a>คำถามที่ถามบ่อย

**Microsoft ใช้ข้อมูลของฉันสำหรับแบบจำลองหรือไม่**

ไม่ Microsoft ได้สร้าง Machine Learning ทั่วไปสำหรับบริการการประมวลผลใบเสร็จรับเงิน แบบจำลองนี้ไม่ได้ขึ้นอยู่กับใบเสร็จรับเงินที่คุณอัพโหลด

**คุณลักษณะนี้พร้อมใช้งานและมีการประมวลผลที่ใด**

ในปัจจุบัน ประเทศสหรัฐอเมริกาได้รับการสนับสนุน

**ใบเสร็จรับเงินของฉันอยู่ที่ไหน**

การเงินจะติดต่อ Cognitive Services เพื่อแยกข้อมูลฟิลด์ Cognitive Services จะรักษาสำเนาของใบเสร็จรับเงินของคุณให้นานถึง 24 ชั่วโมง ในขณะที่การประมวลผลเกิดขึ้น หลังจากเสร็จสิ้นการประมวลผลแล้ว Cognitive Services จะลบใบเสร็จรับเงิน ใบเสร็จรับเงินยังคงจัดเก็บอยู่ในการเงิน

สำหรับข้อมูลเพิ่มเติม ให้ดูที่ [เปิดใช้งานความเข้าใจในใบเสร็จรับเงินด้วยความสามารถใหม่ของ Form Recognizer](https://azure.microsoft.com/blog/enable-receipt-understanding-with-form-recognizer-s-new-capability/)
