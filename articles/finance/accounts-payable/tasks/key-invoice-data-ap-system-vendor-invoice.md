# <a name="key-invoice-data-in-ap-using-a-vendor-invoice"></a>ข้อมูลใบแจ้งหนี้ที่สำคัญใน AP ที่ใช้ใบแจ้งหนี้ของผู้จัดจำหน่าย

[!include [banner](../../includes/banner.md)]

คู่มืองานนี้จะช่วยคุณสร้างใบแจ้งหนี้ของผู้จัดจำหน่ายจากใบสั่งซื้อ และดูผลลัพธ์ของการจับคู่ใบสั่งซื้อ ใบรับ และใบแจ้งหนี้ (วิธีการจับคู่ 3 ทาง)


## <a name="create-a-purchase-order"></a>สร้างใบสั่งซื้อ
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > บัญชีเจ้าหนี้ > ใบสั่งซื้อ > ใบสั่งซื้อทั้งหมด**
2. คลิก **สร้าง**
3. ในฟิลด์ **บัญชีผู้จัดจำหน่าย** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ค้นหาผู้จัดจำหน่ายเพื่อเลือก  ตัวอย่างเช่น เลื่อนลงไปที่ US-104
5. เลือกผู้จัดจำหน่าย US-104
6. คลิก **ตกลง**
7. ในฟิลด์ **หมายเลขสินค้า** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
8. เลือกสินค้าในสินค้าคงคลัง  ตัวอย่างเช่น เลือกสินค้าหมายเลข 1000
9. ขยาย fastTab **รายละเอียดรายการ**
10. คลิกแท็บ **การตั้งค่า** คุณสามารถแทนที่นโยบายการจับคู่เพื่อใช้ไม่มีการจับคู่ การจับคู่ 2 ทาง หรือการจับคู่ 3 ทาง  
11. บนบานหน้าต่างการดำเนินการ คลิก **ซื้อ**
12. คลิก **ยืนยัน**

## <a name="receive-the-products"></a>รับผลิตภัณฑ์
1. บนบานหน้าต่างการดำเนินการ คลิก **รับ**
2. คลิก **ใบรับสินค้า**
3. ในฟิลด์ **ใบรับผลิตภัณฑ์** ให้ป้อนหมายเลขใบรับผลิตภัณฑ์ ตัวอย่างเช่น ป้อน PR123
4. คลิก **ตกลง** เพื่อลงรายการบัญชีใบรับผลิตภัณฑ์
5. ปิดหน้า

## <a name="create-a-vendor-invoice"></a>สร้างใบแจ้งหนี้ของผู้จัดจำหน่าย
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > บัญชีเจ้าหนี้ > ใบสั่งซื้อ > ใบสั่งซื้อที่ได้รับแต่ยังไม่ได้ออกใบแจ้งหนี้**
2. เลือกใบสั่งซื้อที่คุณสร้าง
3. บนบานหน้าต่างการดำเนินการ คลิก **ใบแจ้งหนี้**
4. คลิก **ใบแจ้งหนี้**
5. ในฟิลด์ **หมายเลข** ให้ป้อนหมายเลขใบแจ้งหนี้
6. ในฟิลด์ **คำอธิบายใบแจ้งหนี้** ให้พิมพ์ค่า
7. ในฟิลด์ **วันที่ของใบแจ้งหนี้** ให้ป้อนวันที่
8. ในฟิลด์ **ราคาต่อหน่วย** ให้ป้อน 1200
9. คลิก **เพิ่มรายการ**
10. ในฟิลด์ **หมายเลขสินค้า** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
11. ในรายการ ให้ค้นหาหมายเลขสินค้าที่มีค่าธรรมเนียมการติดตั้ง  ตัวอย่างเช่น S0001
12. เลือกหมายเลขสินค้าที่มีค่าธรรมเนียมการติดตั้ง โปรดทราบว่า การจับคู่จะไม่เกิดขึ้นหลังจากที่คุณทำการเปลี่ยนแปลง  
13. คลิก **ปรับปรุงสถานะการจับคู่**
14. บนบานหน้าต่างการดำเนินการ ให้คลิก **ทบทวน**
15. คลิก **รายละเอียดการจับคู่** การบริการรายการใหม่ไม่จำเป็นต้องถูกจับคู่ ดังนั้นสถานะคงอยู่ที่ "ไม่ดำเนินการ"  
16. เลือกใบรับสินค้าสำหรับสินค้าคงคลังที่คุณได้รับ รายการที่มีใบรับสินค้าได้รับการจับคู่ แต่ปริมาณหรือราคาไม่ตรงกัน ดังนั้นการจับคู่จึงไม่สำเร็จ  
17. ในฟิลด์ **ราคาต่อหน่วย** ให้ป้อนหมายเลข หลังจากที่มีการจับคู่ราคาต่อหน่วย สถานะจะถูกอัพเดตเป็นผ่าน ถ้านโยบายของคุณอนุญาตมีความขัดแย้งหรือถ้าการจับคู่เป็นเพียงแค่คำเตือน คุณยังคงสามารถลงรายการใบแจ้งหนี้ได้  
18. ปิดหน้า
19. คลิก **ลงรายการบัญชี**
20. ปิดหน้า 

>[!Note] 
>ใบสั่งซื้อจะไม่ถูกระบุเป็นได้รับแล้วแต่ไม่ได้ออกใบแจ้งหนี้อีกต่อไป
