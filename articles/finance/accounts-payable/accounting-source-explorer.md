# <a name="accounting-source-explorer"></a>โปรแกรมตรวจดูบัญชีต้นทาง

[!include [banner](../includes/banner.md)]

บทความนี้แสดงข้อมูลเกี่ยวกับหน้า **โปรแกรมตรวจดูบัญชีต้นทาง** ซึ่งคุณสามารถใช้สำหรับการวิเคราะห์แหล่งข้อมูลเบื้องหลังรายการบัญชีแยกประเภททั่วไปโดยละเอียด

หน้า **โปรแกรมตรวจดูบัญชีต้นทาง** แสดงข้อมูลแหล่งที่มา คุณสามารถใช้เป็นเครื่องมือแบบสแตนด์อโลน หรือเพื่อวิเคราะห์รายละเอียดหลังรายการบัญชีแยกประเภททั่วไป ตัวอย่างเช่น คุณสามารถใช้หน้านี้เพื่อรับข้อมูลแหล่งที่มาโดยละเอียดมากที่สุด สำหรับยอดดุลในงบทดลอง หรือสำหรับธุรกรรมใบสำคัญ จากนั้น คุณสามารถใช้คุณลักษณะ **Export to MS Excel** เพื่อแยกและย่อยข้อมูลเพิ่มเติมใน Microsoft Excel (ตัวอย่างเช่น ใน PivotTable หรือในรายงาน PivotTable)

หน้า **โปรแกรมตรวจดูบัญชีต้นทาง** แสดงเช่นเดียวกันกับยอดเงินรวมต่อบัญชีแยกประเภทเสมอ ขณะรายการบัญชีแยกประเภททั่วไปแสดงอยู่ (ตัวอย่างเช่น ในงบทดลอง) ดังเช่นในงบทดลอง คุณสามารถแสดงเซ็กเมนต์ในคอลัมน์ที่แยกต่างหาก เพียงแค่เลือกเซ็ตมิติทางการเงินที่เหมาะสม 

คุณสามารถใช้พารามิเตอร์เพื่อกำหนดช่วงวันที่สำหรับการวิเคราะห์ ฟังก์ชันนี้คล้ายกับฟังก์ชันในงบทดลอง

สำหรับเอกสารทั้งหมดที่ใช้ในกรอบงานเอกสารต้นทาง หน้า **โปรแกรมตรวจดูบัญชีต้นทาง** แสดงข้อมูลเพิ่มเติม ตามการกระจายการลงบัญชี และ ถ้ามี โครงการกระจายการลงบัญชี ข้อมูลนี้รวมถึงค่า **ชนิดยอดเงิน** **โครงการ** **กิจกรรม** **ประเภท** และ **คุณสมบัติของรายการ** ต่อไปนี้เป็นตัวอย่างของการวิเคราะห์บางประการที่คุณสามารถทำได้:

- ผลต่างระหว่างใบสั่งซื้อและใบแจ้งหนี้ของผู้จัดจำหน่าย เนื่องจากผลต่างของแต่ละรายการถูกแสดง ด้วยชนิดยอดเงิน เช่นผลต่างค่าธรรมเนียม
- เรียกเก็บเงินได้และชั่วโมงที่ไม่ได้เรียกเก็บเงินและค่าใช้จ่ายสำหรับโครงการแต่ละโครงการ หน่วยธุรกิจ และบัญชีหลัก

สำหรับเอกสารต้นทางที่ใช้แนวคิดข้อมูลเฉพาะตัวอ้างอิงของเอกสารต้นทาง หน้า **โปรแกรมตรวจดูบัญชีต้นทาง** จะแสดงรายละเอียดเพิ่มเติม เช่น **ลูกค้า** **ผู้จัดจำหน่าย** **ผู้ปฏิบัติงาน** **ผลิตภัณฑ์** **ปริมาณ** **ข้อความหน่วย** และ **คำอธิบาย** ต่อไปนี้เป็นตัวอย่างของการวิเคราะห์บางประการที่คุณสามารถทำได้:

- ค่าใช้จ่ายโรงแรมต่อหน่วยธุรกิจและแบรนด์ของโรงแรมสำหรับรอบระยะเวลาทางบัญชี ขึ้นกับรายงานค่าใช้จ่าย
- ส่วนลดสำหรับแต่ละผู้จัดจำหน่าย สินค้า แผนก

สำหรับเอกสารเหล่านี้ คุณสามารถหาเส้นทางไปยังเอกสารต้นทางที่แท้จริงจากหน้า **โปรแกรมตรวจดูบัญชีต้นทาง**

> [!NOTE]
> ตั้งแต่เวอร์ชัน 10.0.31 คุณลักษณะ **การกรองข้อมูลขั้นสูงของ Accounting Source Explorer** ใหม่ จะพร้อมใช้งานในการจัดการคุณลักษณะ คุณลักษณะนี้จะแทนที่ปุ่ม **อัปเดต** เพื่อให้ประสบการณ์การสอบถามขั้นสูงที่มีประสิทธิภาพมากขึ้น ซึ่งคล้ายกับสิ่งที่พร้อมใช้งานบนหน้า **ธุรกรรมใบสำคัญ** ตัวกรองข้อมูลขั้นสูงจะช่วยให้คุณสามารถกรองข้อมูลในฟิลด์ที่คล้ายกันตามสิ่งที่คุณพบบนหน้า **การสอบถามธุรกรรมใบสำคัญ** เช่น **บัญชีแยกประเภท** **หน่วยธุรกิจ** **ศูนย์ต้นทุน** และ **แผนก**
