## <a name="setup-needed-to-view-power-bi-content"></a><span data-ttu-id="969b9-109">จำเป็นต้องตั้งค่าเพื่อดูเนื้อหา Power BI</span><span class="sxs-lookup"><span data-stu-id="969b9-109">Setup needed to view Power BI content</span></span>

<span data-ttu-id="969b9-110">การตั้งค่าต่อไปนี้จำเป็นต้องถูกดำเนินการให้เสร็จสมบูรณ์สำหรับข้อมูลที่จะแสดงในภาพ **การชำระเงินให้แก่ผู้จัดจำหน่าย** Power BI</span><span class="sxs-lookup"><span data-stu-id="969b9-110">The following setup needs to be completed for data to display in **Vendor payments** Power BI visuals.</span></span>

5. <span data-ttu-id="969b9-116">ไปที่ **การจัดการระบบ > การตั้งค่า > ที่จัดเก็บเอนทิตี** เพื่อรีเฟรชการวัดรวม **VendPaymentBIMeasureV2**</span><span class="sxs-lookup"><span data-stu-id="969b9-116">Go to **System administration > Setup > Entity Store** to refresh the **VendPaymentBIMeasureV2** aggregate measurement.</span></span>
