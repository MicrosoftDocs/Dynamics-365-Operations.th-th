# <a name="accounting-distributions"></a><span data-ttu-id="91b31-103">การกระจายการลงบัญชี</span><span class="sxs-lookup"><span data-stu-id="91b31-103">Accounting distributions</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="91b31-104">หัวข้อนี้แสดงข้อมูลเกี่ยวกับการกระจายการลงบัญชี และอธิบายถึงตัวเลือกที่พร้อมใช้งานสำหรับการประมวลผลเหล่านั้น</span><span class="sxs-lookup"><span data-stu-id="91b31-104">This topic provides information about accounting distributions and describes the options that are available for processing them.</span></span> <span data-ttu-id="91b31-105">การกระจายการลงบัญชีจะใช้ในการปันส่วนยอดเงินสำหรับเอกสารต้นทางไปยังบัญชีแยกประเภทเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="91b31-105">Accounting distributions are used to allocate monetary amounts for a source document to specific ledger accounts.</span></span> 

<span data-ttu-id="91b31-106">การกระจายการลงบัญชีมีความสามารถของโปรแกรมที่ถ้วนทั่วในวงกว้างที่จะใช้ และขยาย โดยแต่ละเอกสารต้นทาง, อาทิเช่น ใบสั่งซื้อ ใบแจ้งหนี้ของผู้จัดจำหน่าย รายงานค่าใช้จ่าย และใบแจ้งหนี้ข้อความอิสระ</span><span class="sxs-lookup"><span data-stu-id="91b31-106">Accounting distributions are a program-wide capability that is used and extended by each source document, such as a purchase order, vendor invoice, expense report, and free text invoice.</span></span> <span data-ttu-id="91b31-107">โดยค่าเริ่มต้น, การกระจายการลงบัญชีเริ่มต้นจะถูกสร้างขึ้นสำหรับแต่ละรายการเอกสารต้นทางและยอดเงิน, และจะเปิดใช้งานสำหรับการแก้ไขอย่างมีเงื่อนไข</span><span class="sxs-lookup"><span data-stu-id="91b31-107">By default, a default accounting distribution is generated for each source document line and monetary amount, and is conditionally enabled for modification.</span></span>

> <span data-ttu-id="91b31-108">เอกสารบางรายการยังสนับสนุนยอดเงินที่จัดทำเอกสารส่วนหัว เช่นค่าธรรมเนียมสำหรับใบสั่ง และใบแจ้งหนี้</span><span class="sxs-lookup"><span data-stu-id="91b31-108">Some documents also support header document monetary amounts, such as charges for orders and invoices.</span></span>

    -   <span data-ttu-id="91b31-113">ยอดเงินการกระจายขึ้นอยู่กับสกุลเงินเพิ่มเติมสำหรับเอกสาร</span><span class="sxs-lookup"><span data-stu-id="91b31-113">The distributions amounts are based on the term currency for the document.</span></span> <span data-ttu-id="91b31-114">โดยทั่วไป สกุลเงินนี้คือ สกุลเงินทางธุรกรรม</span><span class="sxs-lookup"><span data-stu-id="91b31-114">Typically, this currency is the transaction currency.</span></span> <span data-ttu-id="91b31-115">การลงบัญชี และการรายงานยอดเงินของสกุลเงินจะถูกสร้างขึ้นให้เป็นส่วนหนึ่งของการลงบัญชีในบัญชีแยกประเภทย่อย</span><span class="sxs-lookup"><span data-stu-id="91b31-115">Accounting and reporting currency amounts are generated as part of subledger accounting.</span></span>
    -   <span data-ttu-id="91b31-116">การกระจายแสดงวันที่ลงบัญชีและเหตุการณ์ทางบัญชี</span><span class="sxs-lookup"><span data-stu-id="91b31-116">The distributions display the accounting date and accounting event.</span></span> <span data-ttu-id="91b31-117">โดยทั่วไป เหตุการณ์ทางบัญชีถูกตั้งค่าเป็น **ไม่มี** จนกว่าเอกสารจะลงรายการบัญชี/ในสมุดรายวัน</span><span class="sxs-lookup"><span data-stu-id="91b31-117">Typically, the accounting event is set to **None** until the document is posted/journalized.</span></span> <span data-ttu-id="91b31-118">ณ จุดที่ เหตุการณ์ทางบัญชีกลายเป็น **ดั้งเดิม**</span><span class="sxs-lookup"><span data-stu-id="91b31-118">At that point, the accounting event becomes **Original**.</span></span> <span data-ttu-id="91b31-119">หลังจากที่มีการลงรายการบัญชีการกระจาย คุณไม่สามารถแก้ไขการกระจายได้</span><span class="sxs-lookup"><span data-stu-id="91b31-119">After the distributions have been posted, you can't modify the distributions.</span></span>
