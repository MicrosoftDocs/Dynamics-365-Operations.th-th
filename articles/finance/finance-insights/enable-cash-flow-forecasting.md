# <a name="enable-cash-flow-forecasting-preview"></a>เปิดใช้งานการคาดการณ์กระแสเงินสด (ตัวอย่าง)

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

หัวข้อนี้จะอธิบายวิธีการเปิดและตั้งค่าคอนฟิกลักษณะการทำงานการคาดการณ์กระแสเงินสดในข้อมูลเชิงลึกทางการเงิน

> [!NOTE]
> เมื่อต้องการใช้การคาดการณ์การชำระเงินในกระแสเงินสด คุณต้องตั้งค่าลักษณะการทำงานการคาดการณ์การชำระเงินของลูกค้าดังที่อธิบายไว้ใน [การเปิดใช้งานการคาดการณ์การชำระเงินของลูกค้า](enable-cust-paymnt-prediction.md)

1. ใช้ข้อมูลจากหน้าสภาพแวดล้อมใน Microsoft Dynamics Lifecycle Services (LCS) เพื่อเชื่อมต่อกับอินสแตนซ์หลักของ AZURE SQL สำหรับสภาพแวดล้อมดังกล่าว รันคำสั่ง Transact-SQL (T-SQL) ต่อไปนี้เพื่อเปิดใช้งานเที่ยวบินสำหรับสภาพแวดล้อม sandbox (คุณอาจต้องเปิดใช้งานการเข้าถึงสำหรับที่อยู่ IP ของคุณใน LCS ก่อนที่คุณจะสามารถเชื่อมต่อระยะไกลกับเซิร์ฟเวอออบเจ็กต์โปรแกรมประยุกต์ \[AOS\])

    `INSERT INTO SYSFLIGHTING (FLIGHTNAME, ENABLED) VALUES ('CashflowInsightsFeature', 1)`

    > [!NOTE]
    > ถ้าการปรับใช้ Microsoft Dynamics 365 Finance ของคุณเป็นการปรับใช้การให้บริการ คุณสามารถข้ามขั้นตอนนี้ได้ ทีมงานในเชิงลึกของการเงินควรมีการเปิดใช้งานเที่ยวบินให้คุณแล้ว ถ้าคุณไม่เห็นคุณลักษณะในพื้นที่ทำงาน **การจัดการลักษณะการทำงาน** หรือหากประสบปัญหาเมื่อคุณพยายามเปิดใช้งาน โปรดติดต่อ <fiap@microsoft.com>
  
2. เปิดพื้นที่ทำงาน **การจัดการลักษณะการทำงาน** และปฏิบัติตามขั้นตอนเหล่านี้:

    1. เลือก **ตรวจหาการอัพเดต**
    2. เปิดใช้งานคุณลักษณะต่อไปนี้

        - ตัวควบคุมเส้นตารางใหม่
        - การจัดกลุ่มในกริด (ตัวอย่าง) 
        - การคาดคะเนการชำระเงินของลูกค้า (พรีวิว)
        - การคาดการณ์กระแสเงินสด (พรีวิว)

3. ไปที่ **การจัดการธนาคารและเงินสด \> การตั้งค่าการคาดการณ์กระแสเงินสด** และเพิ่มบัญชีสภาพคล่องที่ควรรวมไว้ในการคาดการณ์

    > [!NOTE]
    > ถ้าไม่ได้ตั้งค่าบัญชีสภาพคล่องไว้ แสดงว่าไม่สามารถสร้างกระแสเงินสดได้

4. ไปที่ **การจัดการธนาคารและเงินสด \> การตั้งค่า \> ข้อมูลเชิงลึกทางการเงิน (ตัวอย่าง) \> การคาดการณ์กระแสเงินสด (ตัวอย่าง)** และปฏิบัติตามขั้นตอนเหล่านี้:

    1. บนแท็บ **การคาดการณ์กระแสเงินสด** เลือก **เปิดใช้งานลักษณะการทำงาน**
    2. เลือก **สร้างแบบจำลองการคาดการณ์**

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับความสามารถการคาดการณ์กระแสเงินสด ให้ดู [การคาดการณ์กระแสเงินสด](cash-flow-forecast-intro.md)

การแสดงตัวอย่าง (1) อาจใช้วิธีการที่มีความปลอดภัยและความเป็นส่วนตัวที่น้อยลงกว่าบริการ Dynamics 365 Finance and Operations (2) ไม่ถูกรวมอยู่ในข้อตกลงระดับการให้บริการ (SLA) สำหรับบริการนี้ (3) ไม่ควรถูกใช้เพื่อประมวลผลข้อมูลส่วนบุคคลหรือข้อมูลอื่นที่อยู่ภายใต้ข้อกำหนดการปฏิบัติตามกฎหมายหรือระเบียบข้อบังคับ และ (4) มีการสนับสนุนที่จำกัด
