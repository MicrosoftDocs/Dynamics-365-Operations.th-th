# <a name="assign-number-sequences"></a>กำหนดลำดับหมายเลข

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

หัวข้อนี้จะอธิบายถึงวิธีการสร้างลำดับหมายเลขสำหรับรหัสการเช่า และอธิบายวิธีการสร้างรหัสที่ไม่ซ้ำกันที่ใช้ในกระบวนการประเมินค่าใหม่ของดัชนี

1. ไปที่ **การเช่าสินทรัพย์ \> การตั้งค่า \> พารามิเตอร์การเช่าสินทรัพย์**
2. เลือกแท็บด้านข้าง **ลำดับหมายเลข**
3. เลือก **ลำดับหมายเลข** ในแถบด้านข้าง
4. เลือกลำดับหมายเลขสำหรับการอ้างอิง **รหัสการเช่า** ลำดับหมายเลขนี้จะใช้ในการสร้างรหัสเฉพาะสำหรับแต่ละสัญญาเช่า
5. เลือกลำดับหมายเลขสำหรับการอ้างอิง **รหัสกระบวนการ** ลำดับหมายเลขนี้จะใช้ในการสร้างรหัสเฉพาะสำหรับแต่ละการประเมินค่าใหม่ของดัชนี
6. เลือกลำดับหมายเลขสำหรับการอ้างอิง **รหัสข้อเสนอการสิ้นสุด** ลำดับหมายเลขนี้จะใช้ในการสร้างรหัสเฉพาะสำหรับแต่ละข้อเสนอการสิ้นสุด
