# <a name="use-lease-approval-workflows"></a><span data-ttu-id="723a3-103">ใช้ลำดับงานการอนุมัติสัญญาเช่า</span><span class="sxs-lookup"><span data-stu-id="723a3-103">Use lease approval workflows</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="723a3-104">หัวข้อนี้อธิบายวิธีการใช้ลำดับงานเพื่ออนุมัติการเช่าสินทรัพย์ และวิธีติดตามสถานะและประวัติของลำดับงาน</span><span class="sxs-lookup"><span data-stu-id="723a3-104">This topic explains how to use workflows to approve asset leases, and how to track the status and history of the workflows.</span></span> <span data-ttu-id="723a3-105">วิธีใช้ลำดับงานนำความสอดคล้องกับการจัดการการอนุมัติสัญญาเช่าโดยให้ชุดมาตรฐานของขั้นตอนการอนุมัติและการกำหนดผู้ใช้ที่ระบุซึ่งอนุมัติขั้นตอนแต่ละขั้นตอนของกระบวนการ</span><span class="sxs-lookup"><span data-stu-id="723a3-105">Workflows help bring consistency to the management of lease approvals by providing a standard set of approval steps and assigning specific users who approve each step of the process.</span></span> <span data-ttu-id="723a3-106">ผู้อนุมัติสามารถอนุมัติสัญญาเช่า ปฏิเสธ ร้องขอการเปลี่ยนแปลง หรือกำหนดให้กับผู้ใช้รายอื่นสำหรับการอนุมัติ</span><span class="sxs-lookup"><span data-stu-id="723a3-106">An approver can approve a lease, reject it, request a change to it, or assign it to another user for approval.</span></span> <span data-ttu-id="723a3-107">ลำดับงานยังสามารถแสดงให้เห็นถึงกระบวนการอนุมัติมากขึ้นโดยให้คุณสามารถติดตามสถานะและประวัติของตนเองได้</span><span class="sxs-lookup"><span data-stu-id="723a3-107">Workflows can also bring more visibility into the approval process by letting you track their status and history.</span></span> <span data-ttu-id="723a3-108">นอกจากนี้ คุณยังสามารถดูรายการงานส่วนกลางที่แสดงรายการงานและการอนุมัติที่กำหนดให้กับผู้อนุมัติเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="723a3-108">Additionally, you can view a centralized worklist that lists the tasks and approvals that are assigned to specific approvers.</span></span>

<span data-ttu-id="723a3-109">ก่อนที่คุณจะใช้ขั้นตอนนี้ โปรดตรวจสอบให้แน่ใจว่าอย่างน้อยที่สุดมีการสร้างลำดับงานการอนุมัติสัญญาเช่า</span><span class="sxs-lookup"><span data-stu-id="723a3-109">Before you use this procedure, make sure that at least on lease approval workflow has been created.</span></span> <span data-ttu-id="723a3-110">ถ้าไม่มีลำดับงานอยู่ ให้สร้างลำดับงานหนึ่ง</span><span class="sxs-lookup"><span data-stu-id="723a3-110">If no workflow exists, create one.</span></span> <span data-ttu-id="723a3-111">สำหรับข้อมูลเกี่ยวกับวิธีการตั้งค่าลำดับงาน ให้ดูที่ [ตั้งค่าลำดับงานการอนุมัติสัญญาเช่า](set-up-lease-wrkflw.md)</span><span class="sxs-lookup"><span data-stu-id="723a3-111">For information about how to set up a workflow, see [Set up lease approval workflows](set-up-lease-wrkflw.md).</span></span>

1. <span data-ttu-id="723a3-112">ส่งสัญญาเช่าเพื่อการอนุมัติ</span><span class="sxs-lookup"><span data-stu-id="723a3-112">Submit a lease for approval.</span></span> <span data-ttu-id="723a3-113">บนหน้า **รายละเอียดของสมุดบัญชี** สำหรับสัญญาเช่า ให้เลือก **ลำดับงาน** แล้วเลือก **ส่ง**</span><span class="sxs-lookup"><span data-stu-id="723a3-113">On the **Book details** page for the lease, select **Workflow**, and then select **Submit**.</span></span>
2. <span data-ttu-id="723a3-114">ในกล่องโต้ตอบที่ปรากฏขึ้น คุณสามารถเพิ่มข้อคิดเห็นได้</span><span class="sxs-lookup"><span data-stu-id="723a3-114">In the dialog box that appears, you can add a comment.</span></span> <span data-ttu-id="723a3-115">ผู้อนุมัติที่กำหนดจะเห็นข้อคิดเห็นนี้ร่วมกับสัญญาเช่า</span><span class="sxs-lookup"><span data-stu-id="723a3-115">The designated approver will see this comment together with the lease.</span></span> <span data-ttu-id="723a3-116">เมื่อคุณป้อนข้อคิดเห็นเสร็จเรียบร้อยแล้ว ให้เลือก **ส่ง**</span><span class="sxs-lookup"><span data-stu-id="723a3-116">When you've finished entering the comment, select **Submit**.</span></span> <span data-ttu-id="723a3-117">สัญญาเช่าถูกส่งไปยังระบบลำดับงาน และผู้อนุมัติได้รับเพื่อการอนุมัติ</span><span class="sxs-lookup"><span data-stu-id="723a3-117">The lease is submitted to the workflow system, and the approver receives it for approval.</span></span>
3. <span data-ttu-id="723a3-118">เมื่อต้องการดูสัญญาเช่าที่จะกำหนดให้กับการอนุมัติ ให้ไปที่ **โมดูล \> ทั่วไป \> รายการงาน \> รายการงานกำหนดให้กับฉัน**</span><span class="sxs-lookup"><span data-stu-id="723a3-118">To view the leases that they are designated for approval, go to **Modules \> Common \> Work items \> Work items assigned to me**.</span></span>
4. <span data-ttu-id="723a3-119">บนหน้า **รายการงานที่กำหนดให้กับฉัน** ให้เลือกลิงค์ **รหัสสัญญาเช่า** สำหรับสัญญาเช่าที่คุณต้องการดู</span><span class="sxs-lookup"><span data-stu-id="723a3-119">On the **Work items assigned to me** page, select the **Lease ID** link for the lease that you want to view.</span></span> <span data-ttu-id="723a3-120">หน้าซึ่งแสดงขึ้นอยู่กับสมุดบัญชีสัญญาเช่าและรายละเอียดสัญญาเช่าที่คุณสามารถเข้าถึงได้</span><span class="sxs-lookup"><span data-stu-id="723a3-120">The page that appears depends on the lease books and lease details that you have access to.</span></span>
5. <span data-ttu-id="723a3-121">เมื่อคุณดูสัญญาเช่าเสร็จสิ้นแล้ว ให้เลือก **ลำดับงาน** และตัดสินใจว่าควรจะดำเนินการใด</span><span class="sxs-lookup"><span data-stu-id="723a3-121">When you've finish viewing the lease, select **Workflow**, and decide what action should be taken.</span></span> <span data-ttu-id="723a3-122">ตัวเลือกมี **อนุมัติ** **ปฏิเสธ** **ร้องขอการเปลี่ยนแปลง** **มอบหมาย** และ **ยกเลิก**</span><span class="sxs-lookup"><span data-stu-id="723a3-122">The options include **Approve**, **Rejected**, **Request change**, **Delegate**, and **Canceled**.</span></span> <span data-ttu-id="723a3-123">นอกจากนี้คุณยังสามารถเลือก **ดูประวัติ** เพื่อดูประวัติการอนุมัติสำหรับสัญญาเช่าที่เลือก</span><span class="sxs-lookup"><span data-stu-id="723a3-123">You can also select **View history** to view the approval history for the selected lease.</span></span>
6. <span data-ttu-id="723a3-124">หลังจากที่คุณเลือกการดำเนินการแล้ว ให้ป้อนข้อคิดเห็นเพื่ออธิบายการดำเนินการ</span><span class="sxs-lookup"><span data-stu-id="723a3-124">After you've selected an action, enter a comment to describe the action.</span></span> <span data-ttu-id="723a3-125">เมื่อคุณป้อนข้อคิดเห็นเสร็จเรียบร้อยแล้ว ให้เลือกการดำเนินการที่ **ได้รับการอนุมัติแล้ว** ในรายการ</span><span class="sxs-lookup"><span data-stu-id="723a3-125">When you've finished entering the comment, select the **Approved** action in the list.</span></span>
7. <span data-ttu-id="723a3-126">เมื่อต้องการดูการดำเนินการอนุมัติ ให้กลับไปที่หน้า **รายละเอียดสัญญาเช่า** จากหน้า **สรุปสัญญาเช่า** แล้วเลือก **ลำดับงาน \> ดูประวัติ**</span><span class="sxs-lookup"><span data-stu-id="723a3-126">To view the approval actions, go back to the **Lease details** page from the **Lease summary** page, and then select **Workflow \> View history**.</span></span>

    <span data-ttu-id="723a3-127">คุณสามารถดูกิจกรรมลำดับงานบนหน้า **ประวัติลำดับงาน**</span><span class="sxs-lookup"><span data-stu-id="723a3-127">You can view the workflow activities on the **Workflow history** page.</span></span> <span data-ttu-id="723a3-128">หน้านี้แสดงขั้นตอนลำดับงานที่มีการดำเนินการในสัญญาเช่าเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="723a3-128">This page shows the workflow steps that have been taken on the specific lease.</span></span> <span data-ttu-id="723a3-129">นอกจากนี้คุณยังสามารถใช้ฟิลด์ **รายการงาน** เพื่อดูสถานะของรายการงานที่ได้รับมอบหมาย</span><span class="sxs-lookup"><span data-stu-id="723a3-129">You can also use the **Work items** field to view the status of the assigned work items.</span></span>

8. <span data-ttu-id="723a3-130">เมื่อต้องการหยุดลำดับงาน บนหน้า **ประวัติลำดับงาน** ให้เลือก **เรียกคืน**</span><span class="sxs-lookup"><span data-stu-id="723a3-130">To stop a workflow, on the **Workflow history** page, select **Recall**.</span></span> <span data-ttu-id="723a3-131">ในกล่องโต้ตอบที่ปรากฎ ป้อนข้อคิดเห็น จากนั้นเลือก **ตกลง**</span><span class="sxs-lookup"><span data-stu-id="723a3-131">In the dialog box that appears, enter a comment, and then select **OK**.</span></span>
9. <span data-ttu-id="723a3-132">เมื่อต้องการยกเลิกการเรียกใช้ลำดับงาน หรือเรียกใช้งานลำดับงานที่สร้างไว้ก่อนหน้านี้ ให้ไปที่ **การเช่าสินทรัพย์ \> การตั้งค่า \> ลำดับงานสัญญาเช่า**</span><span class="sxs-lookup"><span data-stu-id="723a3-132">To inactivate a workflow, or to activate a workflow that was previously created, go to **Asset leasing \> Setup \> Lease workflow**.</span></span> <span data-ttu-id="723a3-133">บนหน้า **ลำดับงานสัญญาเช่า** ให้เลือก **ลำดับงาน \> รุ่น**</span><span class="sxs-lookup"><span data-stu-id="723a3-133">Then, on the **Lease workflow** page, select **Workflow \> Versions**.</span></span> <span data-ttu-id="723a3-134">เมื่อต้องการทำให้ลำดับงานปัจจุบันไม่ได้ใช้งานอยู่ ให้เลือกสัญญาเช่าที่ใช้งานอยู่ในกล่องโต้ตอบรุ่นสัญญาเช่า แล้วเลือก **ทำให้ไม่มีการใช้งาน**</span><span class="sxs-lookup"><span data-stu-id="723a3-134">To make a current workflow inactive, select the active lease in the lease version dialog box, and then select **Make inactive**.</span></span> <span data-ttu-id="723a3-135">เมื่อต้องการสร้างลำดับงานที่มีอยู่ ให้เลือกลำดับงาน แล้วเลือก **ทำงาน**</span><span class="sxs-lookup"><span data-stu-id="723a3-135">To make an existing workflow active, select the workflow, and then select **Make active**.</span></span>
