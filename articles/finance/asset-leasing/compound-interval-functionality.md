# <a name="compounding-interval-functionality"></a>ฟังก์ชันช่วงเวลาการทบต้น

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

หัวข้อนี้จะให้ข้อมูลซึ่งจะช่วยให้คุณสามารถเลือกช่วงเวลาการทบต้นรายเดือน รายไตรมาส รายครึ่งปี และรายปี ฟังก์ชันช่วงเวลาการทบต้นถูกใช้เพื่อกำหนดจำนวนของรอบระยะเวลาการทบต้นต่อปีในกำหนดการชำระเงินของสัญญาเช่า แต่ละตัวอย่างของตัวอย่างทั้งสี่ในหัวข้อนี้ จะแสดงว่ากำหนดการชำระเงินของสัญญาเช่ามีลักษณะอย่างไรสำหรับช่วงเวลาที่แตกต่างกัน

คุณไม่สามารถเลือกช่วงการทบต้นที่บ่อยน้อยกว่าความถี่ในการชำระเงินของสัญญาเช่า ตัวอย่างเช่น ไม่สามารถใช้ช่วงการทบต้นของรายไตรมาสกับความถี่ในการชำระเงินรายเดือน และไม่สามารถใช้ช่วงการทบต้นรายปีกับความถี่ในการชำระเงินรายครึ่งปีได้ ถ้าคุณพยายามเลือกช่วงการทบต้นที่บ่อยน้อยกว่าความถี่ในการชำระเงินของสัญญาเช่า คุณจะได้รับข้อความแสดงข้อผิดพลาด

> [!NOTE]
> ในตัวอย่างทั้งสี่ในหัวข้อนี้ ช่วงการทบต้นตรงกับความถี่ในการชำระเงิน

## <a name="examples"></a>ตัวอย่างเช่น

### <a name="setup-for-all-four-leases"></a>การตั้งค่าสำหรับสัญญาเช่าสี่ฉบับ

ตารางต่อไปนี้จะแสดงค่าที่กำหนดไว้บนแท็บ **ทั่วไป** และ **บรรทัดกำหนดการชำระเงิน** สำหรับสัญญาเช่าสี่ฉบับที่ใช้ในตัวอย่าง

**แท็บทั่วไป**

| ฟิลด์                      | มูลค่า                        |
|----------------------------|------------------------------|
| อัตราดอกเบี้ยเงินกู้ส่วนเพิ่ม | **5%**                       |
| ชนิดเงินรายปี               | **เงินรายปีสามัญ**         |
| ช่วงเวลาการทบต้น       | โปรดดูตัวอย่างแต่ละตัวอย่าง |
| ความถี่ในการชำระเงิน          | **รายเดือน**                  |
| วันที่เริ่มต้น          | **1/1/2020**                 |

**แท็บรายการกำหนดการชำระเงิน**

| ฟิลด์             | มูลค่า                        |
|-------------------|------------------------------|
| วันที่เริ่มต้น        | **1/1/2020**                 |
| รอบระยะเวลา           | **120**                      |
| ช่วงรอบระยะเวลา   | **เดือน**                   |

| ความถี่ในการชำระเงิน | โปรดดูตัวอย่างแต่ละตัวอย่าง |
