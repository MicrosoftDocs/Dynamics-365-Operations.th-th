# <a name="asset-leasing-reports"></a>รายงานการเช่าสินทรัพย์

[!include [banner](../includes/banner.md)]

หัวข้อนี้จะแสดงรายการและอธิบายรายงานที่มีอยู่ในการเช่าสินทรัพย์โดยสังเขป รายงานส่วนใหญ่แสดงโดยทำตามขั้นตอนเหล่านี้หรือขั้นตอนที่คล้ายกันมาก ตามที่ระบุไว้) 

- ถ้าต้องการดูรายงานการเช่าสินทรัพย์ส่วนใหญ่ ให้ไปที่ **การเช่าสินทรัพย์ > การสอบถามและรายงาน > รายงานการเช่า** แล้วเลือกรายงานที่จะดู สำหรับรายงานที่จำเป็นต้องใช้พาธการเลือกอื่น ขั้นตอนในการเปิดรายงานจะรวมอยู่ในคำอธิบายของรายงานนั้น 
- เมื่อคุณเลือกรายงานที่จะพิมพ์ หน้าพารามิเตอร์จะเปิดซึ่งช่วยให้คุณสามารถกรองข้อมูลที่จะรวมไว้ในรายงานได้ ป้อนเงื่อนไขการกรองข้อมูล แล้วเลือก **ตกลง** เพื่อสร้างรายงาน รายงานที่สร้างขึ้นจะแสดงข้อมูลที่อยู่ภายในตัวกรองข้อมูลที่คุณระบุ

## <a name="asset-movement"></a>การเคลื่อนไหวของสินทรัพย์
รายงานการเคลื่อนย้ายสินทรัพย์ทำหน้าที่เป็นรายงานการกระทบยอด สำหรับยอดดุลสิทธิ์การใช้สินทรัพย์สำหรับแต่ละการเช่า รายงานนี้ช่วยให้คุณสามารถดูธุรกรรมสินทรัพย์ของการเช่าในระหว่างรอบระยะเวลาที่ระบุ รายงานประกอบด้วยฟิลด์ต่อไปนี้

|------------------------------------|--------------------------------------------------------------------------------|
|     วันที่เริ่มต้น              |     วันที่เริ่มต้นของเวอร์ชันแรกสุดของการเช่า                     |   
|     ระยะเวลาของสัญญาเช่า                     |     ระยะเวลาของสัญญาเช่าของเวอร์ชันแรกของการเช่า                            |
|     สัญญาเช่าระยะสั้น               |     ถ้าการเช่าจัดประเภทเป็นสัญญาเช่าระยะสั้น จะแสดงเป็น **ใช่**         |
|     สัญญาเช่ามูลค่าต่ำ                |     ถ้าการเช่าจัดประเภทเป็นสัญญาเช่ามูลค่าต่ำจะแสดงเป็น **ใช่**          |
|     สิทธิ์การใช้สินทรัพย์เริ่มแรก     |     ค่าดั้งเดิมของสิทธิ์การใช้สินทรัพย์จากรายการสมุดรายวันการรับรู้เริ่มต้น      |

|     ค่าใช้จ่ายสำหรับค่าเสื่อมราคาของงวด    |     ยอดเงินค่าเสื่อมราคาที่ถ่ายภายในช่วงวันที่ที่กำหนดไว้สำหรับรายงาน        |

|     การด้อยค่าของสินทรัพย์                     |     มูลค่าของสินทรัพย์ที่ด้อยค่าในช่วงวันที่ที่กำหนดไว้สำหรับรายงาน               |
|     การปรับเปลี่ยน                  |     ยอดเงินของการปรับปรุงสิทธิ์การใช้สินทรัพย์ที่ใช้ระหว่างพารามิเตอร์วันที่                            |
|     มูลค่าตามบัญชีสุทธิ                 |     มูลค่าตามบัญชีสุทธิสิ้นงวดของสิทธิ์การใช้สินทรัพย์ ซึ่งเป็นค่าเสื่อมราคาสะสมสุทธิ ณ **วันที่สิ้นสุด**    |

## <a name="differences-report"></a>รายงานผลต่าง
รายงานความแตกต่าง แสดงความแตกต่างระหว่างข้อมูลที่โหลดไปยังกรอบงานการนำเข้าการเช่า และข้อมูลที่อยู่ในระบบปัจจุบัน การทำเช่นนี้จะช่วยให้คุณสามารถเปรียบเทียบสิ่งที่มีการปรับปรุง อัปเดต หรือเพิ่มไปยังกรอบงานการนำเข้าการเช่าได้  

ค่าในรายงานจะแตกต่างกันไปตามการเช่าที่เลือก รายงานจะแสดงเฉพาะฟิลด์เหล่านั้นที่แตกต่างจากที่อยู่ในระบบในปัจจุบัน และในตารางการแบ่งระยะ ค่าเดิมคือสิ่งที่อยู่ในระบบปัจจุบัน หรือมีอะไรก่อนหน้านี้ในระบบ ทั้งนี้ขึ้นอยู่กับว่ามีการรันกระบวนการนำเข้าหรือไม่ ค่าใหม่จะแสดงค่าที่อยู่ในตารางการแบ่งระยะที่จะแทนที่ค่าเดิม

## <a name="lease-transactions-report"></a>รายงานธุรกรรมการเช่า
การสอบถามเกี่ยวกับธุรกรรมการเช่าแสดงรายการสมุดรายวันทั้งหมดที่สร้างขึ้นโดยการเช่าสินทรัพย์ รายการสมุดรายวันแต่ละรายการจะถูกเชื่อมโยงกับรหัสสมุดบัญชีที่เป็นต้นกำเนิด การทำเช่นนี้จะช่วยให้คุณสามารถเชื่อมโยงรายการสมุดรายวันกับการเช่าที่สอดคล้องกัน ฟังก์ชันการสอบถามเกี่ยวกับธุรกรรมการเช่าในลักษณะที่คล้ายกับกน้า **ธุรกรรมใบสำคัญ** ในบัญชีแยกประเภททั่วไป
