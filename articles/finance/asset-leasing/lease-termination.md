# <a name="propose-a-lease-for-termination"></a>เสนอสัญญาเช่าเพื่อการยกเลิก

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

ถ้ามีการยกเลิกสัญญาเช่าก่อนเวลา การเช่าสินทรัพย์สามารถบันทึกรายการสมุดรายวันการเลิกจ้างเพื่อตัดจ่ายหนี้สินการเช่า สิทธิ์การใช้สินทรัพย์ (ROU) และค่าเสื่อมราคาสะสม และจองกําไรหรือขาดทุน กระบวนการยกเลิกก่อนเวลายกเลิกสัญญาเช่าและสมุดบัญชีสัญญาเช่าที่เกี่ยวข้อง ไม่ได้สิ้นสุดสมุดบัญชีสัญญาเช่ารายเล่ม หัวข้อนี้อธิบายฟังก์ชันที่ช่วยให้คุณสามารถเสนอสัญญาเช่าเพื่อการยกเลิก และประมวลผลรายการสมุดรายวันการยกเลิกสัญญาเช่า

## <a name="propose-a-lease-for-termination"></a>เสนอสัญญาเช่าเพื่อการยกเลิก

1. ไปที่สัญญาเช่าที่ต้องยกเลิก แล้วบนบานหน้าต่างการดําเนินการ ให้เลือก **ข้อเสนอการยกเลิก**

    > [!NOTE]
    > ปุ่ม **ข้อเสนอการยกเลิก** จะไม่พร้อมใช้งาน ถ้ามีรายการสมุดรายวันที่ยังไม่ได้ลงรายการบัญชีใดๆ กับสมุดบัญชีใดๆ ก่อนที่คุณจะสามารถยกเลิกสัญญาเช่าได้ คุณต้องลงรายการบัญชีหรือลบรายการสมุดรายวันใดๆ ที่สร้างขึ้นแล้วกับสัญญาเช่า

2. ในกล่องโต้ตอบที่ปรากฏ ให้ตั้งค่าฟิลด์ **วันที่มีผลบังคับใช้** และ **วันที่ลงรายการบัญชี** ของรายการสมุดรายวันการยกเลิก
3. เลือก **ข้อเสนอการยกเลิก** เพื่อเสนอสัญญาเช่าเพื่อยกเลิก
4. เลือก **ลงรายการบัญชีการยกเลิกสัญญาเช่า** เพื่อลงรายการบัญชีรายการสมุดรายวันการยกเลิกสัญญาเช่าโดยอัตโนมัติ
5. บนหน้า **การยกเลิกสัญญาเช่า** ให้เลือกรหัสสัญญาเช่าของสัญญาเช่าที่คุณเสนอให้ยกเลิกเพื่อดูบรรทัดการยกเลิก บรรทัดการยกเลิกแสดงมูลค่าการดําเนินการของสิทธิ์การใช้สินทรัพย์ หนี้สินสัญญาเช่า ค่าเสื่อมราคาสะสม ค่าเช่าที่รอการตัดบัญชี (ถ้าเกี่ยวข้อง) และกําไรหรือขาดทุนที่ต้องรับรู้เมื่อสิ้นสุดสัญญาเช่า

ขณะนี้สัญญาเช่าพร้อมแล้วที่จะยกเลิก ค่าของฟิลด์ **สถานะการสิ้นสุด** ของสมุดบัญชีการเช่าจะเปลี่ยนเป็น **พร้อมยกเลิก** ในจุดนี้ คุณไม่สามารถลงรายการบัญชีรายการสมุดรายวันกับสัญญาเช่าหรือปรับปรุงหรือลดลงได้อีกต่อไป 

## <a name="process-leases-that-are-ready-for-termination"></a>ประมวลผลสัญญาเช่าที่พร้อมยกเลิก

เมื่อต้องการประมวลผลสัญญาเช่าที่พร้อมรับการสิ้นสุดและลงรายการบัญชีรายการสมุดรายวันการยกเลิก ให้ปฏิบัติตามขั้นตอนเหล่านี้

1. บนหน้า **การยกเลิกสัญญาเช่า** เลือกสัญญาเช่าที่จะดำเนินการ แล้วเลือก **ยกเลิก**
2. ในกล่องโต้ตอบที่ปรากฏ เลือก **ตกลง**

ระบบลงรายการบัญชีรายการสมุดรายวันการยกเลิก ฟิลด์ **สถานะสัญญาเช่า** สำหรับสมุดบัญชีการเช่าถูกตั้งค่าเป็น **ยกเลิกแล้ว** และฟิลด์ **สถานะของข้อเสนอการยกเลิก** ถูกตั้งค่าเป็น **เสร็จสมบูรณ์**

## <a name="reverse-a-lease-termination"></a>การกลับรายการการยกเลิกสัญญาเช่า

เมื่อต้องการกลับรายการสมุดรายวันการยกเลิกสัญญาเช่า และเปิดสัญญาเช่าที่ยกเลิกแล้ว ให้ปฏิบัติตามขั้นตอนนี้

- บนหน้า **การยกเลิกสัญญาเช่า** เลือกสัญญาเช่าที่ยกเลิกเพื่อย้อนกลับ แล้วเลือก **กลับรายการการยกเลิก**

ระบบกลับรายการสมุดรายวันการยกเลิก ฟิลด์ **สถานะสัญญาเช่า** ของสมุดบัญชีการเช่าถูกตั้งค่าเป็น **เปิด** สัญญาเช่าจะไม่ปรากฏขึ้นบนหน้า **การยกเลิกสัญญาเช่า** อีกต่อไป และสามารถนําเสนออีกครั้งเพื่อการยกเลิกได้

## <a name="example-of-a-lease-termination"></a>ตัวอย่างของการยกเลิกสัญญาเช่า

สำหรับตัวอย่างนี้ สัญญาเช่ามีความเกี่ยวข้องกับสินทรัพย์ที่ไม่มีเฉพาะ และไม่ได้โอนความเป็นเจ้าของสินทรัพย์หรือให้ผู้เช่าเลือกตัวเลือกในการซื้อสินทรัพย์

### <a name="steps-for-terminating-the-lease"></a>ขั้นตอนในการยกเลิกสัญญาเช่า

1. หลังจากที่คุณสร้างการเช่าตามที่อธิบายไว้ก่อนหน้านี้ในหัวข้อนี้ ให้ไปที่สมุดบัญชีการเช่า และยืนยันกำหนดการชำระเงิน ลงรายการบัญชีสมุดรายวันการรับรู้เริ่มต้นแล้ว สิทธิ์การใช้สินทรัพย์เริ่มต้นคือ $71,235.81และหนี้สินสัญญาเช่าควรเป็น $70,235.81 สำหรับตัวอย่างนี้ การเช่ามีการจัดประเภทเป็นสัญญาเช่าดำเนินงานภายใต้การเข้ารหัสมาตรฐานทางบัญชี หัวข้อ 842 (ASC 842)
2. รันกระบวนการของสมุดรายวันชุดงานสามครั้งเพื่อจำลองทางสามปีสำหรับการชำระค่าเช่า ดอกเบี้ยจ่าย และค่าเสื่อมราคา
3. หลังจากที่คุณรันชุดงานทั้งสามชุดงานแล้ว ให้กลับไปที่สมุดบัญชีการเช่า และเปิดตารางธุรกรรมสินทรัพย์และหนี้สินเพื่อดูมูลค่าการถือครองปัจจุบันของสิทธิ์การใช้สินทรัพย์และหนี้สินสัญญาเช่า หลังจาก 3 ปี มูลค่าของหนี้สินควรมีมูลค่าประมาณ $-53,893.00 และมูลค่าของสินทรัพย์ควรอยู่ใน $54,593.00 โดยประมาณ

    หลังจากเวลาผ่านไปสามปี ธุรกิจและผู้ให้เช่าร่วมกันตกลงที่จะสิ้นสุดสัญญาเช่า ดังนั้น ตอนนี้คุณต้องสิ้นสุดสัญญาเช่า

4. ไปที่สัญญาเช่าที่ต้องยกเลิก แล้วบนบานหน้าต่างการดําเนินการ ให้เลือก **ข้อเสนอการยกเลิก** 
5. ในกล่องโต้ตอบที่ปรากฏ ในฟิลด์ **วันที่มีผลบังคับใช้** และ **วันที่ลงรายการบัญชี** ให้ป้อน **1/1/2021**
6. เลือก **ข้อเสนอการยกเลิก** เพื่อเสนอสัญญาเช่าเพื่อยกเลิก

    หน้า **การยกเลิกสัญญาเช่า** จะปรากฏขึ้นและแสดงสัญญาเช่าที่จะถูกยกเลิก

7. หากต้องการดูบรรทัดการยกเลิก ให้เลือกรหัสสัญญาเช่าของสัญญาเช่าที่คุณเสนอให้ยกเลิก รายการการยกเลิกแสดงมูลค่าการดําเนินการของสัญญาเช่า ตารางต่อไปนี้แสดงว่าค่าเหล่านี้ควรเป็นค่าอะไรจากตัวอย่างนี้ 

    | ฟิลด์                                                 | มูลค่า      |
    |-------------------------------------------------------|------------|
    | ยอดดุลยกไปของหนี้สินในสกุลเงินของธุรกรรม | $53,892.89 |
    | สิทธิในการใช้สินทรัพย์ในสกุลเงินของธุรกรรม            | $71,235.81 |
    | ค่าเสื่อมราคาสะสมในสกุลเงินของธุรกรรม      | $16,642.92 |
    | กำไร (ขาดทุน) ในสกุลเงินของธุรกรรม                   | $-700.00   |

8. หากต้องการลงรายการบัญชีรายการสมุดรายวันการยกเลิก ให้เลือกสัญญาเช่าบนหน้า **การยกเลิกสัญญาเช่า** แล้วเลือก **ยกเลิก**
9. ในกล่องโต้ตอบที่ปรากฏ เลือก **ตกลง**
10. เมื่อต้องการดูรายการสมุดรายวันการยกเลิกที่มีการสร้างและลงรายการบัญชีแล้ว ให้ไปที่สมุดรายวันทางการเงินของสินทรัพย์ในสมุดบัญชีการเช่า ตารางต่อไปนี้แสดงว่ารายการนี้ควรเป็นค่าอะไรจากตัวอย่างนี้

    | ธุรกรรม                           | เดบิต (Dr.) | เครดิต (Cr.) |
    |---------------------------------------|-------------|--------------|
    | Dr. หนี้สินสัญญาเช่า                   | 53,892.89   |              |
    | ค่าเสื่อมราคาสะสมของ Dr.          | 16,642.92   |              |
    | Dr. กำไร (ขาดทุน) จากการปรับเปลี่ยนสัญญาเช่า | 700.00      |              |
    | Cr. สินทรัพย์สัญญาเช่า                       |             | 71,235.81    |

11. เมื่อต้องการดูผลกระทบสุทธิของการยกเลิก โดยที่สิทธิ์การใช้สินทรัพย์และหนี้สินสัญญาเช่าจะเป็น 0 (ศูนย์) ให้เปิดตารางธุรกรรมหนี้สินและสินทรัพย์

ขณะนี้สถานะสัญญาเช่าควรเป็น **ยกเลิกแล้ว** จะไม่มีการลงรายการบัญชีรายการสมุดรายวันเพิ่มเติมกับสัญญาเช่านี้ เว้นแต่จะกลับรายการการยกเลิก
