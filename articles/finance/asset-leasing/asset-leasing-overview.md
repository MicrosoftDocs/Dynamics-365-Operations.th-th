# <a name="asset-leasing-overview"></a>ภาพรวมของการเช่าสินทรัพย์

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

การเช่าสินทรัพย์เป็นโมดูลขั้นสูงสำหรับการจัดการ การติดตาม และการทำธุรกรรมทางการเงินโดยอัตโนมัติสำหรับสินทรัพย์ที่เช่าใน Microsoft Dynamics 365 Finance โมดูลการเช่าสินทรัพย์สอดคล้องกับมาตรฐานการบัญชีระหว่างประเทศ (IFRS 16) และมาตรฐาน US GAAP (ASC 842) โมดูลการเช่าสินทรัพย์จะรวบรวมและประมวลผลข้อมูลการเช่าหลักและคำนวณรายการสมุดรายวันสำหรับระยะเวลาของการเช่าจากการรับรู้ครั้งแรก รายการสมุดรายวันรายเดือน ไปยังความบกพร่องและการสิ้นสุดการเช่า โมดูลการเช่าสินทรัพย์รวมกับสินทรัพย์ถาวร บัญชีเจ้าหนี้ และบัญชีแยกประเภททั่วไปอย่างราบรื่น

> [!Note] 
> สำหรับข้อมูลเพิ่มเติมเกี่ยวกับมาตรฐานการบัญชี อ้างอิงถึงเอกสารมาตรฐานของ IFRS 16 และ US GAAP ASC 842 

## <a name="asset-leasing-elements"></a>องค์ประกอบของการเช่าสินทรัพย์ 

แผนภาพด้านล่างนี้แสดงให้เห็นถึงองค์ประกอบหลักของกระบวนการทางธุรกิจการเช่า
