# <a name="fixed-asset-disposal-posting-accounts"></a>บัญชีการลงรายการการตัดจำหน่ายสินทรัพย์ถาวร

[!include [banner](../includes/banner.md)]

บทความนี้อธิบายวิธีการตั้งค่าการลงรายการบัญชีของบัญชีแยกประเภททั่วไปเมื่อคุณตัดจำหน่ายสินทรัพย์

เมื่อต้องการตั้งค่าบัญชีการลงรายการบัญชีแยกประเภททั่วไปที่จะใช้เมื่อคุณตัดจําหน่ายสินทรัพย์ ให้เลือก **การตัดจําหน่าย - การขาย** และ **การตัดจําหน่าย - ของเสีย** ในแท็บด่วน **บัญชีแยกประเภท** บนหน้า **โปรไฟล์การลงรายการบัญชีสินทรัพย์ถาวร**

สำหรับทั้งสองชนิดธุรกรรม (การตัดจำหน่ายสินทรัพย์โดยการขายหรือของเสีย) บัญชีแยกประเภทจะถูกเครดิตสำหรับมูลค่าการตัดจำหน่ายของสินทรัพย์ถาวร มีการลงรายการบัญชีเดบิตไปยังบัญชีตรงข้าม ซึ่งอาจเป็นบัญชีธนาคาร (เป็นตัวอย่าง) ถ้ามีการขายสินทรัพย์ถาวรให้แก่ลูกค้า จะมีการใช้บัญชีลูกค้าแทนบัญชีตรงข้าม สำหรับข้อมูลเพิ่มเติม ให้ดู [การตัดจำหน่ายสินทรัพย์ถาวรเป็นเศษซาก](dispose-of-a-fixed-asset-as-scrap.md)

คลิก **การตัดจำหน่าย** และจากนั้นคลิก **การขาย** หรือ **เศษซาก** และจากนั้นตั้งค่ารายละเอียดของบัญชีเพื่อกลับรายการมูลค่าตามบัญชีสุทธิของสินทรัพย์ถาวร คุณยังสามารถป้อนข้อมูลในฟิลด์ **มูลค่าการลงรายการบัญชี** และ **ชนิดมูลค่าการขาย** ในหน้า **พารามิเตอร์การตัดจำหน่าย** 

ธุรกรรมการตัดจำหน่ายสำหรับสินทรัพย์ในกลุ่มสินทรัพย์มูลค่าต่ำ ลดมูลค่าตามบัญชีสุทธิของกลุ่มสินทรัพย์มูลค่าต่ำตามยอดเงินที่ตัดจำหน่ายเท่านั้น อย่างไรก็ตาม เมื่อการขายสินทรัพย์นั้นเกินมูลค่าตามบัญชีสุทธิของกลุ่มสินทรัพย์มูลค่าต่ำ มูลค่าตามบัญชีสุทธิจะถูกลดลงเป็นศูนย์
