# <a name="set-up-and-create-depreciation-profiles"></a>ตั้งค่าและสร้างโพรไฟล์ค่าเสื่อมราคา

[!include [banner](../../includes/banner.md)]

โพรไฟล์การกำหนดชนิดค่าเสื่อมราคาและความถี่ของค่าเสื่อมราคาสำหรับสินทรัพย์    บันทึกนี้ใช้บริษัท USMF สาธิตและบทบาทของนักบัญชี


## <a name="create-a-depreciation-profile"></a>การสร้างโพรไฟล์การคิดค่าเสื่อมราคา
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > สินทรัพย์ถาวร > การตั้งค่า > โพรไฟล์ค่าเสื่อมราคา**
2. คลิก **สร้าง**
3. ในฟิลด์ **โพรไฟล์ค่าเสื่อมราคา** พิมพ์ค่า
4. ในฟิลด์ **ชื่อ** ให้พิมพ์ค่าใดค่าหนึ่ง
5. ในฟิลด์ **วิธีการ** ให้เลือกหนึ่งตัวเลือก ถ้าเลือกยอดดุลที่ลดลง คุณจะต้องป้อนเปอร์เซ็นต์ในฟิลด์ **เปอร์เซ็นต์**  
6. ในฟิลด์ **ปีที่คิดค่าเสื่อมราคา** ให้เลือกหนึ่งตัวเลือก
7. ในฟิลด์ **ความถี่ของรอบระยะเวลา** ให้เลือกหนึ่งตัวเลือก
8. ปิดแบบฟอร์ม
