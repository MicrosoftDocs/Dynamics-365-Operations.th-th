# <a name="create-and-acquire-assets-from-accounts-payable"></a>สร้างและรับสินทรัพย์จากบัญชีเจ้าหนี้

[!include [banner](../../includes/banner.md)]

กระบวนงานนี้จะนำไปสู่การสร้างและการซื้อสินทรัพย์ของสินทรัพย์ถาวรด้วยกระบวนการซื้อ  มีการใช้นักบัญชีและเสมียนบัญชีเจ้าหนี้ดูแลและบริษัทสาธิต USMF


## <a name="set-fixed-assets-parameters"></a>ตั้งค่าพารามิเตอร์สินทรัพย์ถาวร
1. ใน **บานหน้าต่างนำทาง** ไปที่ **โมดูล > สินทรัพย์ถาวร > การตั้งค่า > พารามิเตอร์สินทรัพย์ถาวร**
2. ขยาย fastTab **ใบสั่งซื้อ**
3. ตรวจสอบกล่องกาเครื่องหมาย **อนุญาตให้มีการซื้อสินทรัพย์จากการซื้อ**
4. กล่องกล่องกาเครื่องหมาย **สร้างสินทรัพย์ในระหว่างใบรับผลิตภัณฑ์หรือการลงรายการบัญชีใบแจ้งหนี้**

## <a name="create-a-new-vendor-invoice"></a>สร้างใบแจ้งหนี้ของผู้จัดจำหน่ายใหม่
1. ใน **บานหน้าต่างนำทาง** ไปที่ **โมดูล > บัญชีเจ้าหนี้ >พื้นที่ทำงาน > รายการใบแจ้งหนี้ของผู้จัดจำหน่าย**
2. คลิก **ใบแจ้งหนี้ของผู้จัดจำหน่ายใหม่**
3. ในฟิลด์ **บัญชีใบแจ้งหนี้** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
4. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
5. ในฟิลด์ **หมายเลข** ให้พิมพ์ค่า
6. ในฟิลด์ **วันที่ลงรายการบัญชี** ป้อนวันที่
7. คลิก **เพิ่มรายการ**
8. ในฟิลด์ **หมายเลขสินค้า** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา สินค้าที่ไม่มีการเก็บในคลังหรือประเภทการจัดซื้ออย่างใดอย่างหนึ่่ง สามารถใช้สำหรับการซื้อสินทรัพย์ถาวร  
9. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
10. ในฟิลด์ **ปริมาณ** ให้ป้อนตัวเลข รายการใบแจ้งหนี้หนึ่งรายการจะสร้างสินทรัพย์ถาวรแค่หนึ่งรายการโดยไม่คำนึงถึงปริมาณ  ค่าของฟิลด์ปริมาณใบแจ้งหนี้จะถูกโอนย้ายไปเป็นปริมาณสินทรัพย์ถาวร  
11. ในฟิลด์ **ราคาต่อหน่วย** ให้ป้อนหมายเลข
12. ขยาย fastTab **รายละเอียดรายการ**
13. คลิกที่แท็บ **สินทรัพย์ถาวร**
14. เลือกกล่องกาเครื่องหมาย **สร้างสินทรัพย์ถาวรใหม่**
15. ในฟิลด์ **สินทรัพย์ถาวร** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
16. ในรายการ ให้เลือกสินทรัพย์ถาวรที่จะใช้เมื่อมีการสร้างสินทรัพย์ถาวรใหม่
17. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
18. คลิก **ลงรายการบัญชี** สินทรัพย์ถาวรจะถูกสร้างและซื้อเมื่อมีการลงรายการบัญชีใบแจ้งหนี้
