# <a name="create-a-depreciation-proposal"></a>สร้างข้อเสนอค่าเสื่อมราคา

[!include [banner](../../includes/banner.md)]

หัวข้อนี้อธิบายวิธีการทำงานของข้อเสนอชุดงานค่าเสื่อมราคา และอธิบายวิธีการเสนอค่าเสื่อมราคาสำหรับสินทรัพย์ถาวร งานนี้ใช้บริษัทสาธิต USMF และบทบาทของนักบัญชี


## <a name="create-a-depreciation-proposal"></a>สร้างข้อเสนอค่าเสื่อมราคา
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > สินทรัพย์ถาวร > รายการสมุดรายวัน > สร้างข้อเสนอค่าเสื่อมราคา**
2. ในฟิลด์ **ชื่อของสมุดรายวัน** ให้เลือกตัวเลือกจากเมนูแบบหล่นลง
3. ในฟิลด์ **วันที่สิ้นสุด** ให้ป้อนวันที่

    - เลือกตัวเลือก **สรุปค่าเสื่อมราคา** เพื่อสรุปค่าเสื่อมราคารายเดือนลงในรายการสมุดรายวันหนึ่งรายการ  
    - ตัวอย่างเช่น ถ้าค่าวันที่สิ้นสุดคือวันที่ 31 มีนาคม 2015 คำอธิบายต่อไปนี้จะถูกสร้างขึ้น: "ค่าเสื่อมราคาตั้งแต่วันที่ 31 มกราคม 2015" จากนั้น ฟิลด์ **วันที่** บนรายการสมุดรายวันที่เสนอจะถูกตั้งเป็น 31 มีนาคม 2015  
    - ข้อเสนอค่าเสื่อมราคาสามารถถูกกรองตามสินทรัพย์ กลุ่มสินทรัพย์ หรือเกณฑ์อื่นๆ โดยใช้ตัวเลือก **ตัวกรอง**  
    - เมื่อคุณใช้ฟอร์ม **สร้างข้อเสนอการซื้อสินทรัพย์หรือค่าเสื่อมราคาสำหรับสินทรัพย์ถาวร** คุณสามารถเสนอค่าเสื่อมราคาในชุดงานได้ นี่คือคำแนะนำสำหรับข้อเสนอใหญ่ที่จะใช้ทรัพยากรของระบบมากขึ้น  ถ้าคุณเลือกตัวเลือกชุดงาน คุณยังคงสามารถดำเนินงานอื่นๆ ให้สำเร็จในช่วงเวลานั้น  เมื่อคุณเสนอค่าเสื่อมราคาในลักษณะนี้ จะมีการคำนวณค่าเสื่อมราคาสำหรับรูปแบบมูลค่าสำหรับสินทรัพย์ถาวร  

4. เลือก **สร้างสมุดรายวัน**

## <a name="review-depreciation-entries"></a>ตรวจทานรายการค่าเสื่อมราคา
1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > สินทรัพย์ถาวร > รายการสมุดรายวัน > สมุดรายวันของสินทรัพย์ถาวร**
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
3. เลือก **รายการ**
4. เลือก **ลงรายการบัญชี**
