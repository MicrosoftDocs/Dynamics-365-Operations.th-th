# <a name="create-a-fixed-asset"></a>สร้างสินทรัพย์ถาวร

[!include [banner](../../includes/banner.md)]

บทความนี้จะอธิบายถึงวิธีการสร้างเรกคอร์ดสินทรัพย์ถาวรใหม่จากหน้ารายการ **สินทรัพย์ถาวร**

ระบบจะกำหนดหมายเลขสินทรัพย์ โดยยึดตามลำดับหมายเลขที่กำหนดให้กับกลุ่มสินทรัพย์ถาวร ถ้าคุณใช้เท็มเพลตสินทรัพย์ถาวรเพื่อนำเข้าสินทรัพย์ผ่าน add-in ของ Microsoft Excel หรือถ้าคุณใช้งานนำเข้าอื่น ระบบจะสร้างเรกคอร์ดสินทรัพย์ถาวรและเพิ่มหมายเลขสินทรัพย์โดยอัตโนมัติ

เพื่อสร้างเรกคอร์ดสินทรัพย์ด้วยตนเอง ให้ทำตามขั้นตอนเหล่านี้

1. ไปที่ **บานหน้าต่างนำทาง \> โมดูล \> สินทรัพย์ถาวร \> สินทรัพย์ถาวร \> สินทรัพย์ถาวร**
2. บน **บานหน้าต่างการดำเนินการ** เลือก **สร้าง**
3. ในฟิลด์ **กลุ่มสินทรัพย์ถาวร** ให้ป้อนหรือเลือกค่า ฟิลด์ **หมายเลข** จะใช้ค่าเริ่มต้น ถ้าคุณได้เปิดใช้ **ฟังก์ชันการกำหนดหมายเลขสินทรัพย์ถาวรอัตโนมัติ** ใน **พารามิเตอร์สินทรัพย์ถาวร** และ **กลุ่มสินทรัพย์ถาวร** ถ้าไม่ได้เปิดใช้ คุณต้องป้อนหมายเลขเฉพาะเพื่อระบุสินทรัพย์ถาวร
4. ในฟิลด์ **ชื่อ** ให้ป้อนค่าใดค่าหนึ่ง ป้อนข้อมูลเพิ่มเติมที่ธุรกิจของคุณจำเป็นต้องใช้สำหรับสินทรัพย์นี้
5. บน **บานหน้าต่างการดำเนินการ** เลือก **สมุดบัญชี**
6. ในฟิลด์ **วันที่ซื้อสินทรัพย์** ให้ป้อนวันที่
7. ในฟิลด์ **ราคาที่ซื้อสินทรัพย์** ให้ป้อนหมายเลข

    - ป้อนข้อมูลเพิ่มเติมที่ธุรกิจของคุณจำเป็นต้องใช้สำหรับสมุดบัญชีนี้
    - ป้อนข้อมูลเพิ่มเติมที่ธุรกิจของคุณจำเป็นต้องใช้สำหรับสมุดบัญชีที่เหลือ

8. ปิดหน้า

นอกจากนี้ คุณยังสามารถนำเข้าสินทรัพย์ถาวรโดยใช้ add-in ของ Excel หรือโดยการรันงานการนำเข้าจากพื้นที่ทำงาน **การจัดการข้อมูล** ก่อนที่คุณจะรันการนำเข้า ให้ป้อนค่าสำหรับฟิลด์ที่จำเป็นในเท็มเพลต

ถ้าคุณไม่ได้กำหนดหมายเลขสินทรัพย์ถาวรในเท็มเพลตของ add-in ของ Excel หรือในการจัดการข้อมูล ระบบจะสร้างหมายเลขสินทรัพย์ถาวรสำหรับสินทรัพย์ที่นำเข้าแต่ละรายการ และเพิ่มลำดับหมายเลขสำหรับแต่ละสินทรัพย์ให้แต่ละรายการโดยอัตโนมัติ อย่างไรก็ตาม ถ้าคุณนำเข้าสินทรัพย์และกำหนดหมายเลขสินทรัพย์ในเท็มเพลต ระบบจะ **ไม่** เพิ่มลำดับหมายเลขโดยอัตโนมัติ ในกรณีนี้ ผู้ดูแลระบบอาจต้องอัพเดตลำดับหมายเลขด้วยตนเอง ถ้าคุณได้กำหนดหมายเลขสินทรัพย์ถาวรในเท็มเพลตของ add-in ของ Excel ระบบจะใช้หมายเลขสินทรัพย์ถาวรที่กำหนดและเพิ่มลำดับหมายเลข

> [!NOTE]                                                                                                         
> หลังจากลงรายการบัญชีการคิดค่าเสื่อมราคา ฟิล **เริ่มนำมาใช้งาน** และ **วัวันที่คำนวณค่าเสื่อมราคา** จะมีการล็อคบนหน้า **สมุดบัญชี** นอกจากนี้ ทั้งสองฟิลด์จะถูกอัพเดตจากเอนทิตีข้อมูล

> [!WARNING]
> เรกคอร์ดสินทรัพย์ถาวรจะไม่ถูกลบถ้ามีการลงรายการบัญชีธุรกรรมไปที่สมุดเชื่อมโยง หรือถ้ามีการป้อนสินทรัพย์ถาวรที่สร้างขึ้นใหม่บนบรรทัดสมุดรายวันแต่ยังไม่ได้ลงรายการบัญชี
