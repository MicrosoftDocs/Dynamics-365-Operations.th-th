# <a name="fixed-assets-home-page"></a>โฮมเพจสินทรัพย์ถาวร

[!include [banner](../includes/banner.md)]

สินทรัพย์ถาวรคือสินค้าที่มีมูลค่า เช่น อาคาร พาหนะ ที่ดิน และเครื่องมือ ที่เป็นของบุคคลหรือองค์กร 

คุณสามารถตั้งค่าและป้อนข้อมูลการได้มาสำหรับสินทรัพย์ถาวร แล้วจัดการสินทรัพย์ของคุณโดยการหักค่าเสื่อมราคา และกำหนดขีดจำกัดเงินทุนเพื่อกำหนดค่าเสื่อมราคา คุณยังสามารถคำนวณการปรับปรุงและตัดจำหน่ายสินทรัพย์ถาวรด้วย เมื่อคุณใช้บัญชีแยกประเภททั่วไปกับสินทรัพย์ถาวร คุณสามารถดูมูลค่าปัจจุบันของสินทรัพย์ถาวรทั้งหมดได้ วิธีการจัดการสินทรัพย์ถาวรยังต้องสอดคล้องกับมาตรฐานการลงบัญชีระหว่างประเทศ และกฎหมายการลงบัญชีในแต่ละประเทศ/ภูมิภาค ข้อกำหนดอาจรวมถึงกฎระเบียบสำหรับการจัดทำเรกคอร์ดธุรกรรมการได้มาและการตัดจำหน่าย ค่าเสื่อมราคา อายุการใช้งาน การเพิ่มมูลค่าและการลดมูลค่าของสินทรัพย์ถาวร ฟังก์ชันสินทรัพย์ถาวรประสานรวมมาตรฐานและกฎระเบียบต่างๆ เหล่านี้เข้าด้วยกัน
