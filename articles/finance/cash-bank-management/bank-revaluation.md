# <a name="bank-foreign-currency-revaluation"></a><span data-ttu-id="8c8e5-104">การประเมินค่าใหม่ตามสกุลเงินต่างประเทศของธนาคาร</span><span class="sxs-lookup"><span data-stu-id="8c8e5-104">Bank foreign currency revaluation</span></span>

<span data-ttu-id="8c8e5-105">หัวข้อนี้แสดงภาพรวมของกระบวนการของการประเมินค่าใหม่ตามสกุลเงินต่างประเทศของธนาคาร</span><span class="sxs-lookup"><span data-stu-id="8c8e5-105">This topic provides an overview of the process of bank foreign currency revaluation.</span></span> <span data-ttu-id="8c8e5-106">ซึ่งอธิบายวิธีการตั้งค่าและรันกระบวนการ และให้ข้อมูลเกี่ยวกับการคำนวณสำหรับกระบวนการ</span><span class="sxs-lookup"><span data-stu-id="8c8e5-106">It explains how to set up and run the process, and provides information about the calculation for the process.</span></span> <span data-ttu-id="8c8e5-107">นอกจากนี้ ยังอธิบายวิธีการกลับรายการของธุรกรรมการประเมินค่าใหม่ ถ้าต้องการการกลับรายการ</span><span class="sxs-lookup"><span data-stu-id="8c8e5-107">It also explains how to reverse revaluation transactions, if reversal is required.</span></span>

<span data-ttu-id="8c8e5-108">ในฐานะที่เป็นส่วนหนึ่งของการสิ้นสุดรอบระยะเวลา แบบแผนการบัญชีต้องมีการประเมินค่าใหม่ยอดดุลบัญชีธนาคารในสกุลเงินต่างประเทศโดยใช้ชนิดอัตราแลกเปลี่ยนอื่น (ปัจจุบัน ในอดีต เฉลี่ย และอื่นๆ)</span><span class="sxs-lookup"><span data-stu-id="8c8e5-108">As part of a period end, accounting conventions require that bank account balances in foreign currencies be revalued by using different exchange rate types (current, historical, average, and so on).</span></span> <span data-ttu-id="8c8e5-109">สามารถใช้คุณลักษณะการประเมินค่าใหม่ตามสกุลเงินต่างประเทศของธนาคารเพื่อประเมินค่าบัญชีธนาคารใหม่อย่างน้อยหนึ่งบัญชี</span><span class="sxs-lookup"><span data-stu-id="8c8e5-109">The bank foreign currency revaluation feature can be used to revalue one or more bank accounts.</span></span> <span data-ttu-id="8c8e5-110">นอกจากนี้ คุณลักษณะยังเป็นคุณลักษณะส่วนกลางด้วย</span><span class="sxs-lookup"><span data-stu-id="8c8e5-110">The feature is also a global feature.</span></span> <span data-ttu-id="8c8e5-111">ดังนั้น จากหน้าเดียว คุณสามารถประเมินค่าธนาคารใหม่ในนิติบุคคลทั้งหมดที่คุณมีการเข้าถึง</span><span class="sxs-lookup"><span data-stu-id="8c8e5-111">Therefore, from a single page, you can revalue banks across all the legal entities that you have access to.</span></span>

> [!NOTE]
> <span data-ttu-id="8c8e5-112">เมื่อคุณรันกระบวนการประเมินค่าใหม่ ยอดดุลในบัญชีธนาคารแต่ละบัญชีที่ลงรายการบัญชีในสกุลเงินต่างประเทศจะถูกนำมาประเมินค่าใหม่</span><span class="sxs-lookup"><span data-stu-id="8c8e5-112">When you run the revaluation process, the balance in each bank account that is posted in a foreign currency will be revalued.</span></span> <span data-ttu-id="8c8e5-113">ธุรกรรมกำไรและขาดทุนที่ยังไม่รับรู้ที่ถูกสร้างขึ้นในระหว่างกระบวนการประเมินค่าใหม่จะถูกสร้างขึ้นโดยระบบ</span><span class="sxs-lookup"><span data-stu-id="8c8e5-113">The unrealized gain or loss transactions that are created during the revaluation process are system-generated.</span></span> <span data-ttu-id="8c8e5-114">ธุรกรรมสองรายการอาจถูกสร้างขึ้น รายการหนึ่งสำหรับสกุลเงินทางบัญชีและรายการหนึ่งสำหรับสกุลเงินการรายงาน ถ้าสกุลเงินการรายงานมีความเกี่ยวข้อง</span><span class="sxs-lookup"><span data-stu-id="8c8e5-114">Two transactions might be created, one for the accounting currency and one for the reporting currency, if a reporting currency is relevant.</span></span> <span data-ttu-id="8c8e5-115">รายการบัญชีแต่ละรายการจะถูกลงรายการบัญชีไปยังกำไรและขาดทุนที่ยังไม่รับรู้กำไร และบัญชีหลักที่ถูกประเมินค่าใหม่</span><span class="sxs-lookup"><span data-stu-id="8c8e5-115">Each accounting entry will be posted to the unrealized gain or loss and the main account that is being revalued.</span></span>

## <a name="prepare-to-run-foreign-currency-revaluation"></a><span data-ttu-id="8c8e5-116">เตรียมเพื่อดำเนินการประเมินค่าใหม่ตามสกุลเงินต่างประเทศ</span><span class="sxs-lookup"><span data-stu-id="8c8e5-116">Prepare to run foreign currency revaluation</span></span>

<span data-ttu-id="8c8e5-117">ก่อนที่คุณจะรันกระบวนการประเมินค่าใหม่ จะต้องมีการตั้งค่าต่อไปนี้ก่อน</span><span class="sxs-lookup"><span data-stu-id="8c8e5-117">Before you run the revaluation process, the following setup is required.</span></span>

- <span data-ttu-id="8c8e5-118">ในหน้า **บัญชีแยกประเภท** ให้ระบุชนิดอัตราแลกเปลี่ยน</span><span class="sxs-lookup"><span data-stu-id="8c8e5-118">On the **Ledger** page, specify the exchange rate type.</span></span> <span data-ttu-id="8c8e5-119">ถ้าไม่มีการกำหนดชนิดอัตราแลกเปลี่ยนในบัญชีหลัก ชนิดอัตราแลกเปลี่ยนนี้ถูกใช้ระหว่างการประเมินค่าใหม่ตามสกุลเงินต่างประเทศ</span><span class="sxs-lookup"><span data-stu-id="8c8e5-119">If an exchange rate type isn't defined on the main account, this exchange rate type is used during foreign currency revaluation.</span></span>

- <span data-ttu-id="8c8e5-123">ในหน้า **บัญชีการประเมินค่าใหม่ตามสกุลเงิน** เลือกบัญชีการประเมินค่าใหม่ตามสกุลเงินที่แตกต่างกัน สำหรับสกุลเงินแต่ละสกุลและบริษัทแต่ละแห่งได้</span><span class="sxs-lookup"><span data-stu-id="8c8e5-123">On the **Currency revaluation accounts** page, select different currency revaluation accounts for each currency and company.</span></span> <span data-ttu-id="8c8e5-124">ถ้าไม่มีการกำหนดบัญชี บัญชีจากหน้า **บัญชีแยกประเภท** จะถูกใช้</span><span class="sxs-lookup"><span data-stu-id="8c8e5-124">If no accounts are defined, the accounts from the **Ledger** page are used.</span></span>
