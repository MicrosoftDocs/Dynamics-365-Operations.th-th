# <a name="tax-calculation-data-model"></a>รูปแบบข้อมูลการคํานวณภาษี

บทความนี้มีข้อมูลเกี่ยวกับวิธีการกําหนดค่าฟิลด์ในรูปแบบข้อมูลภาษีในธุรกรรมการคํานวณภาษี

*รูปแบบข้อมูลภาษี* ประกอบด้วยฟิลด์ที่ต้องใช้สำหรับการคํานวณภาษี Microsoft จัดเตรียม *แบบจำลองข้อมูลการคํานวณภาษี* ซึ่งรวมถึงฟิลด์ส่วนหัวและฟิลด์รายการของเอกสารธุรกรรมในแอปการเงินและการดำเนินงาน ฟิลด์ที่ถูกกําหนดในรูปแบบข้อมูลการคํานวณภาษีคือ คอลัมน์ที่พร้อมใช้งานของตารางกฎการใช้งานในการตั้งค่าคอนฟิกของการตั้งค่าคอนฟิกคุณลักษณะการคํานวณภาษี

> [!NOTE] 
> โหนดบางโหนดที่ถูกกําหนดในรูปแบบข้อมูล เช่น **รหัสเรกคอร์ด** และ **รหัสตาราง** ใช้เพื่อวัตถุประสงค์ด้านเทคนิค ไม่มีคอลัมน์อยู่ในการตั้งค่าคอนฟิกคุณลักษณะการคํานวณภาษี

ปฏิบัติตามขั้นตอนต่อไปนี้เพื่อดูรูปแบบข้อมูลการคํานวณภาษี

1. ลงชื่อเข้าใช้ Regulatory Configuration Service (RCS) และไปยัง **การรายงานทางอิเล็กทรอนิกส์** \> **การตั้งค่าคอนฟิกภาษี**
2. เลือก **รูปแบบข้อมูลภาษี** \> **รูปแบบข้อมูลการคํานวณภาษี**
3. บน FastTab **รุ่น** เลือกรุ่น
4. เลือก **ตัวออกแบบ**

    [![ปุ่มโปรแกรมออกแบบ](./media/tax-calculation-model-mapping-1.png)](./media/tax-calculation-model-mapping-1.png)

5. ขยายโหนด **บริการภาษี**
6. ขยายโหนด **ส่วนหัว** เพื่อดูรูปแบบข้อมูลส่วนหัว
7. ขยายโหนด **รายการ** เพื่อดูรูปแบบข้อมูลรายการ

    [![รูปแบบข้อมูล](./media/tax-calculation-model-mapping-2.png)](./media/tax-calculation-model-mapping-2.png)

นอกจากนี้ คุณยังสามารถดูรูปแบบข้อมูลการคํานวณภาษีและคอลัมน์ที่มีอยู่ในตารางกฎการใช้งาน

1. ใน RCS ให้ไปที่ **คุณลักษณะโลกาภิวัตน์** \> **การคํานวณภาษี**
2. ค้นหาคุณลักษณะภาษีที่มีสถานะเป็น **ร่าง** แล้วจากนั้น เลือก **แก้ไข**
3. บนแท็บ **ทั่วไป** ให้เลือกรุ่นการตั้งค่าคอนฟิก โดยยึดตามรุ่นของรูปแบบข้อมูลการคํานวณภาษี ตัวอย่างเช่น เลือก **40.46.212**
4. บนแท็บ **การใช้งานกลุ่มภาษี** ให้เลือก **จัดการคอลัมน์**

    [![จัดการคอลัมน์](./media/tax-calculation-model-mapping-3.png)](./media/tax-calculation-model-mapping-3.png)

แบบจำลองข้อมูลการคํานวณภาษีถูกรวมเข้ากับแอปการเงินและการดำเนินงาน ในระหว่างธุรกรรมแต่ละรายการที่เกี่ยวข้องกับการคํานวณภาษี ค่าของฟิลด์ที่ถูกกําหนดในตารางกฎการใช้งานจะถูกรวบรวมและส่งไปยังบริการคํานวณภาษีเพื่อคํานวณ ในรุ่น 40.46 รูปแบบข้อมูลที่กําหนดล่วงหน้า 57 รายการจะพร้อมใช้งานสำหรับชนิดของธุรกรรมต่อไปนี้: ใบสั่งซื้อ, ใบสั่งขาย, ใบสั่งโอนย้าย, ใบขอซื้อ, การร้องขอใบเสนอราคา, และใบเสนอราคาขาย

## <a name="version-updates-and-restrictions"></a>การอัปเดตรุ่นและข้อจํากัด

- **ใบแจ้งหนี้ข้อความอิสระ:** ได้รับการสนับสนุนในรุ่น 40.50.221 และ Finance รุ่น 10.0.23
- **สมุดรายวันทั่วไป:** ได้รับการสนับสนุนในรุ่น 40.54.234 และ Finance รุ่น 10.0.26

    *ข้อจำกัด:* ไม่รองรับ [ใบสำคัญเดียว](../general-ledger/one-voucher.md#issues-with-one-voucher) และค่าธรรมเนียมการชำระเงิน

- **สมุดรายวันใบแจ้งหนี้ของผู้จัดจำหน่าย:** ได้รับการสนับสนุนในรุ่น 40.54.234 และ Finance รุ่น 10.0.26
- **สมุดรายวันประจำงวด:** ได้รับการสนับสนุนในรุ่น 40.55.236 และ Finance รุ่น 10.0.29
