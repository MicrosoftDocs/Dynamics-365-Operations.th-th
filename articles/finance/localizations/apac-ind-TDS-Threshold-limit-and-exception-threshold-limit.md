# <a name="threshold-limit-and-exception-threshold-limit"></a><span data-ttu-id="2e684-103">ขีดจํากัดและขีดจํากัดข้อยกเว้น</span><span class="sxs-lookup"><span data-stu-id="2e684-103">Threshold limit and exception threshold limit</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="2e684-104">หัวข้อนี้อธิบายขีดจํากัดและขีดจํากัดข้อยกเว้นสำหรับภาษีที่หักที่ต้นทาง (TDS)</span><span class="sxs-lookup"><span data-stu-id="2e684-104">This topic describes the threshold and exception limits for Tax Deducted at Source (TDS).</span></span> <span data-ttu-id="2e684-105">TDS ในใบแจ้งหนี้และการชำระเงินจะถูกคํานวณเสมอ โดยพิจารณาจากขีดจํากัดและขีดจํากัดข้อยกเว้นที่กําหนดไว้สำหรับส่วนประกอบภาษี TDS ในหน้า **ส่วนประกอบภาษีหัก ณ ที่จ่าย**</span><span class="sxs-lookup"><span data-stu-id="2e684-105">The TDS on invoices and payments is always calculated considering the threshold limit and exception threshold limit defined for the TDS tax components on the **Withholding tax components** page.</span></span> <span data-ttu-id="2e684-106">ส่วนประกอบภาษี TDS ถูกแนบอยู่กับรหัสภาษี TDS ซึ่งรวมอยู่ในกลุ่มภาษี TDS</span><span class="sxs-lookup"><span data-stu-id="2e684-106">The TDS tax components are attached to TDS tax codes, which are included in the TDS tax groups.</span></span> <span data-ttu-id="2e684-107">กลุ่มภาษี TDS จะถูกแนบอยู่กับผู้จัดจำหน่ายและลูกค้าเพื่อคํานวณ TDS ในระดับใบแจ้งหนี้หรือระดับการการชำระเงิน</span><span class="sxs-lookup"><span data-stu-id="2e684-107">The TDS tax groups are attached to vendors and customers to calculate TDS at the invoice-level or payment-level.</span></span>

<span data-ttu-id="2e684-108">TDS จะถูกคํานวณ หากยอดเงินสำหรับธุรกรรม หรือธุรกรรมสะสมที่ลงรายการบัญชีด้วยกลุ่ม TDS ที่ระบุสำหรับผู้จัดจำหน่าย เกินขีดจํากัดที่ระบุในหน้า **ส่วนประกอบภาษีหัก ณ ที่จ่าย**</span><span class="sxs-lookup"><span data-stu-id="2e684-108">TDS is calculated if the amount for a transaction or the cumulative transactions posted with a specific TDS group for a vendor exceeds the threshold limit specified on the **Withholding tax components** page.</span></span> <span data-ttu-id="2e684-109">TDS จะไม่ถูกคํานวณจนกว่ายอดเงินธุรกรรมสะสมจะเกินขีดจํากัดที่ระบุ</span><span class="sxs-lookup"><span data-stu-id="2e684-109">TDS will not be calculated until the cumulative transaction amount exceeds the specified threshold limit.</span></span>

<span data-ttu-id="2e684-110">หากมีการระบุขีดจํากัดข้อยกเว้นพร้อมกับขีดจํากัดสำหรับส่วนประกอบ TDS จะมีการคํานวณ TDS เมื่อยอดเงินธุรกรรมเฉพาะเกินกว่าขีดจํากัดข้อยกเว้น แม้ว่ายอดเงินของธุรกรรมสะสมไม่เกินขีดจํากัดที่ระบุ</span><span class="sxs-lookup"><span data-stu-id="2e684-110">If an exception threshold limit is specified along with the threshold limit for a TDS component, TDS is calculated when a specific transaction amount exceeds the exception threshold limit even if the cumulative transaction amount does not exceed the specified threshold limit.</span></span>
