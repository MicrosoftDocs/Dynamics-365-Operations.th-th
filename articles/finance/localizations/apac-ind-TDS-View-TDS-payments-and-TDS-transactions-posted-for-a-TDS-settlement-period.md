# <a name="view-posted-tds-payments-and-transactions-for-a-tds-settlement-period"></a><span data-ttu-id="ba849-103">ดูการชําระเงินและธุรกรรม TDS ที่ลงรายการบัญชีสำหรับรอบระยะเวลาการชําระเงิน TDS</span><span class="sxs-lookup"><span data-stu-id="ba849-103">View posted TDS payments and transactions for a TDS settlement period</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="ba849-104">หัวข้อนี้อธิบายวิธีการดูการชําระเงินภาษีที่หักที่ต้นทาง (TDS) และธุรกรรมที่ลงรายการบัญชีไว้สำหรับรอบระยะเวลาการชําระเงิน</span><span class="sxs-lookup"><span data-stu-id="ba849-104">This topic explains how to view the Tax Deducted at Source (TDS) payments and transactions that were posted for a settlement period.</span></span>

1. <span data-ttu-id="ba849-105">ไปที่ **ภาษี \> ภาษีทางอ้อม \> ภาษีหัก ณ ที่จ่าย \> ระยะเวลาการชำระภาษีหัก ณ ที่จ่าย**</span><span class="sxs-lookup"><span data-stu-id="ba849-105">Go to **Tax \> Indirect taxes \> Withholding tax \> Withholding tax settlement periods**.</span></span>

    <span data-ttu-id="ba849-106">[![หน้ารอบระยะเวลาการชำระภาษีหัก ณ ที่จ่าย](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)</span><span class="sxs-lookup"><span data-stu-id="ba849-106">[![Withholding tax settlement periods page](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)</span></span>

2. <span data-ttu-id="ba849-107">บนหน้า **รอบระยะเวลาการชําระภาษีหัก ณ ที่จ่าย** ให้เลือก **การชําระภาษีหัก ณ ที่จ่าย** เพื่อเปิดหน้า **การชําระภาษีหัก ณ ที่จ่าย** ที่ซึ่งคุณสามารถดูการชําระเงิน TDS ที่ถูกชำระสำหรับรอบระยะเวลาการชําระเงิน TDS เฉพาะ</span><span class="sxs-lookup"><span data-stu-id="ba849-107">On the **Withholding tax settlement periods** page, select **Withholding tax payments** to open the **Withholding tax payments** page, where you can view the TDS settlements that were made for a specific TDS settlement period.</span></span>

    <span data-ttu-id="ba849-108">แท็บ **ภาพรวม** แสดงข้อมูลต่อไปนี้:</span><span class="sxs-lookup"><span data-stu-id="ba849-108">The **Overview** tab shows the following information:</span></span>

    - <span data-ttu-id="ba849-109">**วันที่** – วันที่สำหรับการชำระเงิน TDS</span><span class="sxs-lookup"><span data-stu-id="ba849-109">**Date** – The date of TDS settlement.</span></span>
    - <span data-ttu-id="ba849-110">**ใบสำคัญ** – หมายเลขใบสำคัญของธุรกรรมการชำระเงิน TDS</span><span class="sxs-lookup"><span data-stu-id="ba849-110">**Voucher** – The voucher number of the TDS settlement transaction.</span></span>
    - <span data-ttu-id="ba849-111">**ชนิดภาษี** – ชนิดภาษีที่จะมีการรันกระบวนการชําระเงินให้</span><span class="sxs-lookup"><span data-stu-id="ba849-111">**Tax type** – The tax type that the settlement process is run for.</span></span>
    - <span data-ttu-id="ba849-112">**หมายเลขบัญชีภาษี (TAN)** – หมายเลขบัญชีภาษี (TAN) ของธุรกรรม TDS ที่ชำระแล้ว</span><span class="sxs-lookup"><span data-stu-id="ba849-112">**Tax Account Number (TAN)** – The Tax Account Number (TAN) of the settled TDS transaction.</span></span>
    - <span data-ttu-id="ba849-113">**รอบระยะเวลาการชําระเงิน** – รอบระยะเวลาการชําระเงินที่จะมีการรันกระบวนการชําระเงิน TDS ให้</span><span class="sxs-lookup"><span data-stu-id="ba849-113">**Settlement period** – The settlement period that the TDS settlement process is run for.</span></span>
    - <span data-ttu-id="ba849-114">**วันที่เริ่มต้น** – วันที่เริ่มต้นของรอบระยะเวลาเดอะเซตเทิลเมนต์</span><span class="sxs-lookup"><span data-stu-id="ba849-114">**From date** – The start date of the settlement period.</span></span>
    - <span data-ttu-id="ba849-115">**วันที่สิ้นสุด** – วันที่สิ้นสุดของรอบระยะเวลาเดอะเซตเทิลเมนต์</span><span class="sxs-lookup"><span data-stu-id="ba849-115">**To date** – The end date of the settlement period.</span></span>
    - <span data-ttu-id="ba849-116">**เวอร์ชันการชำระภาษีหัก ณ ที่จ่าย** – เวอร์ชันของการชำระภาษีหัก ณ ที่จ่าย: **ต้นฉบับ**, **การแก้ไขล่าสุด** หรือ **รายการยอดรวม**</span><span class="sxs-lookup"><span data-stu-id="ba849-116">**Withholding tax payment version** – The version of the withholding tax payment: **Original**, **Latest corrections**, or **Total list**.</span></span>

5. <span data-ttu-id="ba849-117">เลือก **ใบสำคัญ** เพื่อดูรายการใบสำคัญสำหรับธุรกรรม TDS</span><span class="sxs-lookup"><span data-stu-id="ba849-117">Select **Voucher** to view the voucher entries for the TDS transaction.</span></span>
6. <span data-ttu-id="ba849-118">เลือก **ธุรกรรมภาษีหัก ณ ที่จ่าย** เพื่อดูรายละเอียดของธุรกรรม TDS ที่ชำระแล้วสำหรับรอบระยะเวลาเฉพาะในระหว่างรอบระยะเวลาการชำระเงิน</span><span class="sxs-lookup"><span data-stu-id="ba849-118">Select **Withholding tax transactions** to view the details of the TDS transactions that were settled for a specific period during a settlement period.</span></span> <span data-ttu-id="ba849-119">เลือก **ใบสำคัญ** เพื่อดูรายการใบสำคัญสำหรับธุรกรรม TDS</span><span class="sxs-lookup"><span data-stu-id="ba849-119">Select **Voucher** to view the voucher entries for the TDS transaction.</span></span> <span data-ttu-id="ba849-120">เลือก **ส่วนประกอบภาษีหัก ณ ที่จ่าย** เพื่อดู TDS ที่คํานวณได้ต่อส่วนประกอบภาษี TDS สำหรับรหัสภาษี TDS เฉพาะ</span><span class="sxs-lookup"><span data-stu-id="ba849-120">Select **Withholding tax components** to view the TDS that was calculated per TDS tax component for a specific TDS tax code.</span></span>
7. <span data-ttu-id="ba849-121">ปิดหน้า **การชำระภาษีหัก ณ ที่จ่าย** เพื่อกลับไปที่หน้า **รอบระยะเวลาการชำระภาษีหัก ณ ที่จ่าย**</span><span class="sxs-lookup"><span data-stu-id="ba849-121">Close the **Withholding tax payments** page to return to the **Withholding tax settlement periods** page.</span></span>
8. <span data-ttu-id="ba849-122">เลือก **ธุรกรรมภาษีหัก ณ ที่จ่าย** เพื่อดูรายละเอียดของธุรกรรม TDS ที่ชำระแล้วสำหรับรอบระยะเวลาเดอะเซตเทิลเมนต์</span><span class="sxs-lookup"><span data-stu-id="ba849-122">Select **Withholding tax transactions** to view the details of the TDS transactions that were settled for the settlement period.</span></span>
