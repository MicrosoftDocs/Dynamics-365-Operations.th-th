# <a name="establish-customer-payment-fees"></a><span data-ttu-id="e4fb4-103">จัดทำค่าธรรมเนียมการชำระเงินของลูกค้า</span><span class="sxs-lookup"><span data-stu-id="e4fb4-103">Establish customer payment fees</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="e4fb4-104">สร้างค่าธรรมเนียมในการชำระเงินสำหรับการชำระเงินของลูกค้า </span><span class="sxs-lookup"><span data-stu-id="e4fb4-104">Create payment fees for customer payments.</span></span>

<span data-ttu-id="e4fb4-105">งานนี้ใช้บริษัทสาธิต USMF </span><span class="sxs-lookup"><span data-stu-id="e4fb4-105">This task uses the USMF demo company.</span></span>

1. <span data-ttu-id="e4fb4-106">ใน **บานหน้าต่างนำทาง** ไปที่ **โมดูล > บัญชีลูกหนี้ > การตั้งค่าการชำระเงิน > ค่าธรรมเนียมการชำระเงิน**</span><span class="sxs-lookup"><span data-stu-id="e4fb4-106">In the **Navigation pane**, go to **Modules > Accounts receivable > Payments setup > Payment fee**.</span></span>
2. <span data-ttu-id="e4fb4-107">คลิก **สร้าง**</span><span class="sxs-lookup"><span data-stu-id="e4fb4-107">Click **New**.</span></span>
3. <span data-ttu-id="e4fb4-108">ในฟิลด์ **รหัสค่าธรรมเนียม** ป้อนรหัสค่าธรรมเนียม</span><span class="sxs-lookup"><span data-stu-id="e4fb4-108">In the **Fee ID** field, enter a Fee ID.</span></span> <span data-ttu-id="e4fb4-109">แสดงรหัสค่าธรรมเนียมในสมุดรายวันการชำระเงิน ดังนั้นให้อธิบายให้เข้าใจถึงค่าธรรมเนียมที่ถูกประเมิน</span><span class="sxs-lookup"><span data-stu-id="e4fb4-109">The Fee ID displays on payment journals, so make it descriptive to understand what fee is being assessed.</span></span>  
4. <span data-ttu-id="e4fb4-110">ในฟิลด์ **ชื่อ** ป้อนชื่อค่าธรรมเนียม</span><span class="sxs-lookup"><span data-stu-id="e4fb4-110">In the **Name** field, enter a fee name.</span></span>
5. <span data-ttu-id="e4fb4-111">ในฟิลด์ **คำอธิบายค่าธรรมเนียม** ป้อนคำอธิบายสำหรับค่าธรรมเนียม</span><span class="sxs-lookup"><span data-stu-id="e4fb4-111">In the **Fee description** field, enter a description for the fee.</span></span>
6. <span data-ttu-id="e4fb4-112">ในฟิลด์  **ค่าธรรมเนียม** เลือกว่าจะเรียกเก็บค่าธรรมเนียมที่ลูกค้าหรือบัญชีแยกประเภท</span><span class="sxs-lookup"><span data-stu-id="e4fb4-112">In the **Charge** field, select whether the fee will be charged to the Customer or a Ledger account.</span></span> <span data-ttu-id="e4fb4-113">ถ้าลูกค้าประเมินค่าธรรมเนียม ให้เลือกลูกค้า </span><span class="sxs-lookup"><span data-stu-id="e4fb4-113">If the customer is assessed the fee, select Customer.</span></span> <span data-ttu-id="e4fb4-114">หากค่าธรรมเนียมจะถูกประเมินเป็นค่าใช้จ่ายขององค์กรของคุณ ให้เลือกบัญชีแยกประเภท</span><span class="sxs-lookup"><span data-stu-id="e4fb4-114">If the fee will be assess to your organization as an expense, select Ledger.</span></span> <span data-ttu-id="e4fb4-115">สำหรับงานนี้เลือกลูกค้า</span><span class="sxs-lookup"><span data-stu-id="e4fb4-115">For this task, select Customer.</span></span>  
7. <span data-ttu-id="e4fb4-116">ในฟิลด์ **ชนิดสมุดรายวัน** ให้เลือกชนิดของสมุดรายวันที่สามารถใช้ค่าธรรมเนียมการชำระเงินนี้ได้</span><span class="sxs-lookup"><span data-stu-id="e4fb4-116">In the **Journal type** field, select the type of journal that can use this payment fee.</span></span> <span data-ttu-id="e4fb4-117">ถ้าค่าธรรมเนียมเหล่านี้จะใช้สำหรับการชำระเงินของลูกค้า ชนิดสมุดรายวันจะมีแนวโน้มเป็นการชำระเงินของลูกค้า</span><span class="sxs-lookup"><span data-stu-id="e4fb4-117">If these fees are used for customer payments, the journal type will likely be Customer payment.</span></span>  
8. <span data-ttu-id="e4fb4-118">คลิก **บันทึก**</span><span class="sxs-lookup"><span data-stu-id="e4fb4-118">Click **Save**.</span></span>
9. <span data-ttu-id="e4fb4-119">คลิก **การตั้งค่าค่าธรรมเนียมการชำระเงิน**</span><span class="sxs-lookup"><span data-stu-id="e4fb4-119">Click **Payment fee setup**.</span></span> <span data-ttu-id="e4fb4-120">การตั้งค่าค่าธรรมเนียมการชำระเงินจะใช้เพื่อกำหนดเกณฑ์สำหรับเมื่อจะถูกประเมินค่า </span><span class="sxs-lookup"><span data-stu-id="e4fb4-120">The Payment fee setup is used to define the criteria for when the payment fee will be assessed.</span></span>  <span data-ttu-id="e4fb4-121">ตัวอย่างเช่น คุณสามารถกำหนดว่าค่าธรรมเนียมจะถูกคำนวณถ้าบัญชีธนาคารเป็น USMF OPER และวิธีการชำระเงินเป็นเช็ค</span><span class="sxs-lookup"><span data-stu-id="e4fb4-121">For example, you can define that the fee will be calculated if the bank account is USMF OPER, and the method of payment is check.</span></span>  
10. <span data-ttu-id="e4fb4-122">ในฟิลด์ **การจัดกลุ่ม** เลือก ตาราง กลุ่ม หรือทั้งหมด เพื่อกำหนดบัญชีธนาคารที่จะถูกประเมินค่าธรรมเนียมนี้</span><span class="sxs-lookup"><span data-stu-id="e4fb4-122">In the **Groupings** field, select either Table, Group or All to define which bank accounts will be assessed this fee.</span></span> <span data-ttu-id="e4fb4-123">ถ้าคุณเลือกทั้งหมด บัญชีธนาคารทั้งหมดจะถูกประเมินค่าธรรมเนียมนี้ </span><span class="sxs-lookup"><span data-stu-id="e4fb4-123">If you select All, all bank accounts could be assessed this fee.</span></span>  <span data-ttu-id="e4fb4-124">ถ้าคุณเลือกตาราง เฉพาะบัญชีธนาคารที่คุณเลือกจะถูกประเมินค่าธรรมเนียมนี้ </span><span class="sxs-lookup"><span data-stu-id="e4fb4-124">If you select Table, only the bank account you select could be assessed this fee.</span></span> <span data-ttu-id="e4fb4-125">ถ้าคุณเลือกกลุ่ม เฉพาะบัญชีธนาคารในกลุ่มธนาคารที่เลือกจะถูกประเมินค่าธรรมเนียมนี้</span><span class="sxs-lookup"><span data-stu-id="e4fb4-125">If you select Group, only the bank accounts in the selected bank group could be assessed this fee.</span></span>  
11. <span data-ttu-id="e4fb4-126">ในฟิลด์ **ความสัมพันธ์ของธนาคาร** ให้เลือกกลุ่มธนาคารหรือบัญชีธนาคาร</span><span class="sxs-lookup"><span data-stu-id="e4fb4-126">In the **Bank relation** field, select either a bank group or a bank account.</span></span> <span data-ttu-id="e4fb4-127">ถ้าคุณเลือกตาราง การค้นหาจะแสดงบัญชีธนาคาร </span><span class="sxs-lookup"><span data-stu-id="e4fb4-127">If you selected Table, the lookup will display bank accounts.</span></span> <span data-ttu-id="e4fb4-128">ถ้าคุณเลือกกลุ่ม การค้นหาจะแสดงกลุ่มธนาคาร</span><span class="sxs-lookup"><span data-stu-id="e4fb4-128">If you selected Group, the lookup will display bank groups.</span></span>  
12. <span data-ttu-id="e4fb4-129">ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก</span><span class="sxs-lookup"><span data-stu-id="e4fb4-129">In the list, click the link in the selected row.</span></span>
13. <span data-ttu-id="e4fb4-130">ในฟิลด์ **วิธีการชำระเงิน** ให้เลือกวิธีการชำระเงินสำหรับค่าธรรมเนียมที่จะถูกประเมิน</span><span class="sxs-lookup"><span data-stu-id="e4fb4-130">In the **Method of payment** field, select the Method of payment for which this fee will be assessed.</span></span> <span data-ttu-id="e4fb4-131">ตัวอย่างเช่น คุณอาจประเมินค่าธรรมเนียมให้กับลูกค้าของคุณถ้าพวกเขาส่งรายการชำระเงินเป็นเช็ค มากกว่าการชำระเงินทางอิเล็กทรอนิกส์</span><span class="sxs-lookup"><span data-stu-id="e4fb4-131">For example, you may assess a fee to your customers if they send payments as a check, rather than as an electronic payment.</span></span>  
14. <span data-ttu-id="e4fb4-132">ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ</span><span class="sxs-lookup"><span data-stu-id="e4fb4-132">In the list, find and select the desired record.</span></span>
15. <span data-ttu-id="e4fb4-133">ถ้าเกี่ยวข้อง ในฟิลด์ **สกุลเงินการชำระเงิน** ให้ป้อนสกุลเงินการชำระเงิน</span><span class="sxs-lookup"><span data-stu-id="e4fb4-133">If relevant, in the **Payment currency** field, enter a payment currency.</span></span> <span data-ttu-id="e4fb4-134">สกุลเงินของการชำระเงินถูกใช้เป็นเกณฑ์เพิ่มเติมว่าค่าธรรมเนียมจะถูกประเมินหรือไม่ </span><span class="sxs-lookup"><span data-stu-id="e4fb4-134">The payment currency is used as an additional criteria for whether the fee will be assessed.</span></span>  <span data-ttu-id="e4fb4-135">ตัวอย่างเช่น ธนาคารของคุณอาจเรียกเก็บค่าธรรมเนียมเพิ่มเติมสำหรับการชำระเงินในสกุลเงิน USD เนื่องจากโดยปกติพวกเขาจะดำเนินการในสกุลเงินยูโรเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="e4fb4-135">For example, your bank may charge an extra fee for payments received in USD currency, since they normally only transact in EUR currency.</span></span>  
16. <span data-ttu-id="e4fb4-136">เลือกว่าค่าธรรมเนียม จะเป็นเปอร์เซ็นต์ ยอดเงิน หรือช่วง</span><span class="sxs-lookup"><span data-stu-id="e4fb4-136">Select whether the fee will be a percent, amount or interval.</span></span>
17. <span data-ttu-id="e4fb4-137">ใน **ฟิลด์เปอร์เซ็นต์/ยอดเงิน** ให้ป้อนเปอร์เซ็นต์ หรือยอดเงินของค่าธรรมเนียม</span><span class="sxs-lookup"><span data-stu-id="e4fb4-137">In the **Percentage/Amount field**, enter either percentage or amount of the fee.</span></span> <span data-ttu-id="e4fb4-138">ถ้าฟิลด์เปอร์เซ็นต์/ยอดเงิน เป็นเปอร์เซ็นต์ ดังนั้นค่าที่ป้อนที่นี่จะเป็นเปอร์เซ็นต์ </span><span class="sxs-lookup"><span data-stu-id="e4fb4-138">If the Percentage/Amount field is Percent, then the value enter here will be a percentage.</span></span> <span data-ttu-id="e4fb4-139">ถ้าฟิลด์เปอร์เซ็นต์/ยอดเงิน เป็นยอดเงิน ค่าคุณป้อนที่นี่จะเป็นยอดเงิน </span><span class="sxs-lookup"><span data-stu-id="e4fb4-139">If the Percentage/Amount field is Amount, then the value you enter here will be an amount.</span></span> <span data-ttu-id="e4fb4-140">ถ้าฟิลด์เปอร์เซ็นต์/ยอดเงิน เป็นช่วง ใช้แท็บช่วงเพื่อกำหนดระดับ</span><span class="sxs-lookup"><span data-stu-id="e4fb4-140">If the Percentage/Amount field is Interval, use the Interval tab to define the tiers.</span></span>  
18. <span data-ttu-id="e4fb4-141">ในฟิลด์ **สกุลเงินค่าธรรมเนียม** เลือกสกุลเงินของค่าธรรมเนียม</span><span class="sxs-lookup"><span data-stu-id="e4fb4-141">In the **Fee currency** field, select the currency of the fee.</span></span> <span data-ttu-id="e4fb4-142">นี่คือสกุลเงินที่จะมีการสร้างค่าธรรมเนียม</span><span class="sxs-lookup"><span data-stu-id="e4fb4-142">This is the currency in which the fee will be created.</span></span>  
19. <span data-ttu-id="e4fb4-143">คลิก **บันทึก**</span><span class="sxs-lookup"><span data-stu-id="e4fb4-143">Click **Save**.</span></span>
