# <a name="set-up-multiple-element-revenue-allocation"></a>ตั้งค่าการปันส่วนรายได้แบบหลายองค์ประกอบ

การปันส่วนรายได้หลายองค์ประกอบทำให้คุณสามารถตั้งค่าเทมเพลตต่างๆ ที่ใช้ในการคำนวณและการปันส่วนรายได้ให้กับหลายรายการ ฟังก์ชันนี้สามารถช่วยให้คุณปฏิบัติตามหัวข้อประมวลมาตรฐานการบัญชี 606 (ASC 606) และ/หรือมาตรฐานการรายงานทางการเงินระหว่างประเทศ 15 (IFRS 15)

## <a name="multiple-element-revenue-allocation-parameters"></a>พารามิเตอร์การปันส่วนรายได้แบบหลายองค์ประกอบ

ใช้หน้า **พารามิเตอร์การปันส่วนรายได้แบบหลายองค์ประกอบ** เพื่อตั้งค่าพารามิเตอร์ให้กับการปันส่วนรายได้หลายองค์ประกอบ

### <a name="standalone-selling-price-origin"></a>ที่มาราคาขายแยกต่างหาก

ฟิลด์ **ที่มาราคาขายแยกต่างหาก** จะระบุจุดเริ่มต้นของราคาขายแยกต่างหาก คุณสามารถอัปเดตค่าในหน้า **ราคาขายแยกต่างหากของรายการ** และในธุรกรรม โดยตัวเลือกที่คุณสามารถเลือกได้มีดังต่อไปนี้

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| จำนวนเงิน | ราคาขายแยกต่างหากคือจํานวนเงินที่คุณระบุว่ามากกว่า 0 (ศูนย์) จํานวนเงินจะถูกแปลงค่าระหว่างสกุลเงินฟังก์ชันและเริ่มต้นตามที่ต้องการ |
| ราคาขายฐาน | ราคาขายแยกต่างหากสอดคล้องกับราคาขายฐานของสินค้า |
| ราคาในใบแจ้งหนี้ | ราคาขายแยกต่างหากสอดคล้องกับราคาในใบแจ้งหนี้ของสินค้า |
| เปอร์เซ็นต์ของรายการ | ราคาขายแยกต่างหากจะถูกระบุเป็นค่าเปอร์เซ็นต์และคํานวณตามราคาของสินค้า ถ้าคุณเลือกตัวเลือกนี้ ให้ระบุเปอร์เซ็นต์เริ่มต้น |
| ปันส่วนจำนวนเงินคงเหลือ | <p>ที่มาของราคาขายแยกต่างหากจะคํานวณเป็น *มูลค่าสัญญารวมของรายการหลัก* – *ราคาขายแยกต่างหากรวมของรายการรอง*:</p><ul><li>*มูลค่าสัญญารวมของรายการหลัก* คือยอดเงินสุทธิหรือยอดเงินที่เรียกเก็บเงิน</li><li>*ราคาขายแยกต่างหากรวมของรายการรอง* เป็นผลรวมของราคาขายแยกต่างหากของสัญญาหรือเพิ่มเติมของรายการรองทั้งหมด ยกเว้นรายการรองที่ใช้ราคาขายแยกต่างหากนี้</li></ul><p>ถ้ายอดเงินที่คํานวณได้เป็นค่าลบ จํานวนเงินจะถูกตั้งค่าเป็น 0 (ศูนย์)</p><p>**หมายเหตุ:** ตัวเลือกนี้สามารถเลือกได้เฉพาะรายการรองเพียงรายการเดียวในการแบ่งส่วนรายได้</p> |
| ไม่มี | ที่มาของราคาขายแยกต่างหากโลนจะอิงอยู่กับรายการรอง ตัวเลือกนี้จะใช้กับสินค้าที่กําหนดเป็นรายการหลักในเทมเพลตการแบ่งส่วนรายได้ ถ้ามีการเลือกกล่องกาเครื่องหมาย **การแบ่งส่วนรายได้** ตัวเลือกนี้จะถูกเลือกโดยอัตโนมัติ และการตั้งค่าไม่สามารถเปลี่ยนแปลงได้ |
| เปอร์เซ็นต์ของราคาในใบแจ้งหนี้หลัก | ที่มาของราคาขายแยกต่างหากคือเปอร์เซ็นต์ของราคาใบแจ้งหนี้ของรายการหลัก คุณสามารถเปลี่ยนค่าเริ่มต้นได้ ตัวเลือกนี้สามารถใช้งานกับรายการรองในเทมเพลตการแบ่งส่วนรายได้เท่านั้น |
| เปอร์เซ็นต์ของราคามาตรฐานหลัก | ที่มาของราคาขายแยกต่างหากคือเปอร์เซ็นต์ของราคามาตรฐานของรายการหลัก คุณสามารถเปลี่ยนค่าเริ่มต้นได้ ตัวเลือกนี้สามารถใช้งานกับรายการรองในเทมเพลตการแบ่งส่วนรายได้เท่านั้น ตัวเลือกนี้เป็นตัวเลือกเริ่มต้นสำหรับรายการรอง เมื่อตัวเลือกสำหรับรายการรองเปลี่ยนจาก **เปอร์เซ็นต์ของราคามาตรฐานหลัก** เป็น **เปอร์เซ็นต์ของราคาในใบแจ้งหนี้หลัก** หรือจาก **เปอร์เซ็นต์ของราคาในใบแจ้งหนี้หลัก** เป็น **เปอร์เซ็นต์ของราคามาตรฐานหลัก** ค่าที่คํานวณได้จะถูกอัปเดตด้วย |

### <a name="account-for-revenue-allocation-rounding-differences"></a>บัญชีสำหรับผลต่างการปัดเศษของการปันส่วนรายได้

ฟิลด์ **บัญชีสำหรับผลต่างการปัดเศษของการปันส่วนรายได้** จะระบุบัญชีที่ใช้ในการบันทึกการปรับปรุงการปัดเศษใดๆ ให้กับจำนวนเงินรายได้ตามสัญญา สามารถใช้งานได้เมื่อมีการใช้การเรียกเก็บเงินตามสัญญาที่เกิดซ้ำ

## <a name="item-standalone-selling-price"></a>ราคาขายแยกต่างหากของรายการ

ใช้หน้า **ราคาขายแยกต่างหากของรายการ** เพื่อระบุที่มาและราคาขายแยกต่างหากของรายการ ค่าที่ระบุบนหน้านี้จะใช้เป็นค่าเริ่มต้นในหน้า **การปันส่วนรายได้** ในการปันส่วนรายได้หลายองค์ประกอบและการเรียกเก็บเงินตามสัญญาที่เกิดซ้ำ

### <a name="specify-item-standalone-selling-price"></a>ระบุราคาขายแยกต่างหากของรายการ

เมื่อต้องการระบุราคาขายแยกต่างหากของรายการ ให้ปฏิบัติตามขั้นตอนเหล่านี้

1. บนหน้า **ราคาขายแยกต่างหากของรายการ** ในฟิลด์ **ที่มาราคาขายแยกต่างหาก** ให้เลือกที่มาของราคาขายแยกต่างหาก
2. ทำตามขั้นตอนเหล่านี้ โดยขึ้นอยู่กับค่าที่คุณเลือกไว้ในฟิลด์ **ที่มาราคาขายแยกต่างหาก**:

    * ถ้าคุณเลือก **เปอร์เซ็นต์ของสินค้า** ให้ระบุเปอร์เซ็นต์ในฟิลด์ **เปอร์เซ็นต์**
    * ถ้าคุณเลือก **จำนวนเงิน** ให้ระบุจำนวนเงินในฟิลด์ **ราคาขายแยกต่างหาก**

2. สำหรับแต่ละรายการที่คุณต้องการเพิ่มลงในรายการ เลือก **เพิ่ม**
3. ในฟิลด์ **รหัสสินค้า** ให้เลือกชนิดสินค้า ในฟิลด์ **ความสัมพันธ์ของสินค้า** ให้เลือกความสัมพันธ์ของสินค้า สำหรับสินค้าที่มีการล้างกล่องกาเครื่องหมาย **การแบ่งส่วนรายได้** ชุดที่เลือกของรหัสสินค้าและความสัมพันธ์ของสินค้าต้องไม่ซ้ำกัน
4. เปลี่ยนค่าเริ่มต้นของฟิลด์ **ที่มาราคาขายแยกต่างหาก**, **ราคาขายแยกต่างหาก** หรือ **เปอร์เซ็นต์** ตามที่คุณต้องการ
5. สำหรับแต่ละรายการหลักที่คุณต้องการเพิ่มลงในรายการ เลือก **เพิ่ม**
6. ในฟิลด์ **รหัสสินค้า** ให้เลือกชนิดสินค้า ในฟิลด์ **ความสัมพันธ์ของสินค้า** ให้เลือกความสัมพันธ์ของสินค้า
7. เลือกกล่องกาเครื่องหมาย **การแบ่งส่วนรายได้**
8. ในฟิลด์ **ความสัมพันธ์ของสินค้า** ให้เลือกความสัมพันธ์ของสินค้า รายการจะถูกอัปเดตด้วยรายการหลักและรายการรองทั้งหมด
9. สำหรับรายการรอง เปลี่ยนค่าเริ่มต้นของฟิลด์ **ที่มาราคาขายแยกต่างหาก**, **เปอร์เซ็นต์ของราคามาตรฐานหลัก**, **เปอร์เซ็นต์ของราคาในใบแจ้งหนี้หลัก** หรือ **ปันส่วนจำนวนเงินคงเหลือ** ตามที่คุณต้องการ
10. เมื่อต้องการเพิ่มหลายรายการในคราวเดียว ให้เลือก **เพิ่มสินค้า**
11. อัปเดตการสอบถามเพื่อเลือกช่วงของรายการที่คุณต้องการเพิ่ม
12. เลือก **ตกลง** ตรวจสอบรายชื่อรายการที่คุณต้องการเพิ่ม และเลือก **ตกลง**

ราคาขายแยกต่างหากสอดคล้องกับราคาในใบแจ้งหนี้ของสินค้า</li><li>**เปอร์เซ็นต์ของรายการ** – ราคาขายแยกต่างหากจะถูกระบุเป็นค่าเปอร์เซ็นต์และคํานวณตามราคาของสินค้า ถ้าคุณเลือกตัวเลือกนี้ ค่าที่คํานวณได้จะถูกอัปเดตด้วย</li></ul> |
| ราคาขายแยกต่างหาก | ระบุที่มาเริ่มต้นของราคาขายแยกต่างหากของรายการ ฟิลด์นี้จะสามารถใช้งานเมื่อตั้งค่าฟิลด์ **ที่มาราคาขายแยกต่างหาก** เป็น **จํานวนเงิน** |
| เปอร์เซ็นต์ | ระบุเปอร์เซ็นต์ของราคาขายแยกต่างหาก ฟิลด์นี้จะสามารถใช้งานเมื่อตั้งค่าฟิลด์ **ที่มาราคาขายแยกต่างหาก** เป็น **เปอร์เซ็นต์ของรายการ**, **เปอร์เซ็นต์ของราคาในใบแจ้งหนี้หลัก** หรือ **เปอร์เซ็นต์ของราคามาตรฐานหลัก** |
| การแบ่งส่วนรายได้ | <p>ระบุว่ารายการใช้การแบ่งส่วนรายได้หรือไม่:</p><ul><li>**เลือกไว้** – เลือกได้เฉพาะรายการที่มีการตั้งค่าเทมเพลตการแบ่งส่วนรายได้ในฟิลด์ **ความสัมพันธ์ของสินค้า** เท่านั้น คุณสามารถเลือกกล่องกาเครื่องหมายนี้สำหรับรายการหลักของเทมเพลตการแบ่งส่วนรายได้เท่านั้น</li><li>**ล้างข้อมูล** – รายการเป็นรายการมาตรฐานที่ไม่ได้ใช้การแบ่งส่วนรายได้</li></ul> |
| ความสัมพันธ์ | สัญลักษณ์ระบุว่ารายการนั้นเป็นรายการหลักหรือรายการย่อยในการแบ่งส่วนรายได้ |
| รหัสรายการ | <p>เลือกรหัสรายการ: **ตาราง** หรือ **กลุ่ม**</p><p>ถ้ามีการเลือกกล่องกาเครื่องหมาย **การแบ่งส่วนรายได้** จะมีการตั้งค่าฟิลด์นี้เป็น **ตาราง** และค่าไม่สามารถเปลี่ยนแปลงได้</p> |
| ความสัมพันธ์ของสินค้า | <p>เลือกหมายเลขสินค้า</p><p>ถ้ามีการเลือกกล่องกาเครื่องหมาย **การแบ่งส่วนรายได้** เฉพาะรายการที่เป็นรายการหลักที่มีการตั้งค่าเทมเพลตการแบ่งส่วนรายได้เท่านั้นที่สามารถเลือกได้ เมื่อเลือกรายการหลัก รายการจะถูกอัปเดตโดยอัตโนมัติด้วยรายการรองของรายการหลักนั้น</p> |
| รายการหลัก | สำหรับรายการหลัก ฟิลด์นี้จะแสดงหมายเลขรายการ สำหรับสินค้าย่อยในการแบ่งส่วนรายได้ จะแสดงหมายเลขสินค้าของสินค้าหลัก |

## <a name="mea-templates"></a>เทมเพลต MEA

ใช้หน้า **เทมเพลต MEA** เพื่อสร้างและแก้ไขรหัสเทมเพลตการจัดการหลายองค์ประกอบ (MEA) รหัสเหล่านี้จะใช้ในหน้า **การปันส่วนรายได้** เพื่อจัดกลุ่มสินค้าเข้าด้วยกัน

### <a name="create-a-template"></a>สร้างเทมเพลต

หากต้องการตั้งค่าเทมเพลต MEA ให้ทำตามขั้นตอนเหล่านี้

1. ในหน้า **เทมเพลต MEA** เลือก **สร้าง**
1. ในฟิลด์ **รหัสเทมเพลต MEA** ให้ป้อนรหัสที่ไม่ซ้ำ
1. ในฟิลด์ **คำอธิบาย** ให้ป้อนคำอธิบาย
1. ในฟิลด์ **บัญชีรายได้ของสัญญารอการตัดบัญชี** ให้เลือกบัญชีที่คุณต้องการใช้เป็นรายการสมุดรายวันเมื่อสร้างใบแจ้งหนี้ของสัญญา MEA ฟิลด์นี้จะสามารถใช้งานเมื่อเปิดใช้งานและใช้การเรียกเก็บเงินตามสัญญาที่เกิดซ้ำ
1. เลือก **บันทึก**
