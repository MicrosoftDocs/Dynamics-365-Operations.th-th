# <a name="create-a-cost-accounting-ledger"></a>สร้างบัญชีแยกประเภทการบัญชีต้นทุน

[!include [banner](../../includes/banner.md)]

บัญชีแยกประเภทการบัญชีต้นทุนแสดงถึงหน่วยการรายงานโดยรวม ซึ่งกำหนดโดยมิติองค์ประกอบต้นทุน มิติทางสถิติ ปฏิทินทางการเงิน และสกุลเงิน ซึ่งไม่ตรงกับแนวคิดของนิติบุคคล นิติบุคคลและข้อมูลของนิติบุคคลสามารถเชื่อมโยงกับบัญชีแยกประเภทการบัญชีต้นทุนหลายบัญชี การบันทึกข้อมูลนี้ใช้บริษัทข้อมูลสาธิต USP2

1. ไปที่ บัญชีต้นทุน > การตั้งค่าบัญชีแยกประเภท > บัญชีแยกประเภทสำหรับการบัญชีต้นทุน
2. คลิก สร้าง
3. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
4. ในฟิลด์มิติองค์ประกอบต้นทุน ให้ป้อนหรือเลือกค่า
5. ในฟิลด์ปฏิทินทางการเงิน ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
6. ในฟิลด์สกุลเงินทางบัญชี ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
7. ในฟิลด์ชนิดอัตราแลกเปลี่ยน ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
8. ในฟิลด์มิติทางสถิติ ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
9. คลิก บันทึก
