--------

<span data-ttu-id="41a62-108">งบประมาณที่ถูกอนุมัติสำหรับนิติบุคคลจะจัดเก็บอยู่ในเอกสารที่เรียกว่า *รายการทะเบียนงบประมาณ*</span><span class="sxs-lookup"><span data-stu-id="41a62-108">The approved budget for a legal entity is maintained in a document that is known as a *budget register entry*.</span></span> <span data-ttu-id="41a62-109">บรรทัดในเอกสารรายการทะเบียนงบประมาณจะถูกเรียกว่ารายการ *บัญชีงบประมาณ* และประกอบด้วยข้อมูลมิติทางการเงิน วันที่ และยอดเงินงบประมาณที่อนุมัติแล้ว</span><span class="sxs-lookup"><span data-stu-id="41a62-109">The lines in a budget register entry document are known as *budget account* entries, and contain financial dimension information, dates, and the amounts of the approved budget.</span></span> <span data-ttu-id="41a62-110">เอกสารรายการทะเบียนงบประมาณถูกรวมเข้ากับรายงานทางการเงินพื้นฐานและหน้าการสอบถามที่ยอดเงินจริงของบัญชีแยกประเภทถูกเปรียบเทียบกับจำนวนเงินงบประมาณ</span><span class="sxs-lookup"><span data-stu-id="41a62-110">The budget register entry document is integrated with basic financial reports and inquiry pages where ledger actual amounts are compared to budget amounts.</span></span> 

<span data-ttu-id="41a62-111">มีหลายวิธีการสำหรับการสร้างรายการทะเบียนงบประมาณ:</span><span class="sxs-lookup"><span data-stu-id="41a62-111">There are multiple methods for creating budget register entries:</span></span>

-   <span data-ttu-id="41a62-116">ถ้าบริษัทของคุณใช้ฟังก์ชันการวางแผนงบประมาณเพื่อจัดเตรียมข้อมูลงบประมาณ คุณสามารถใช้กระบวนการเป็นครั้งคราวที่ **สร้างรายการทะเบียนงบประมาณ** ได้</span><span class="sxs-lookup"><span data-stu-id="41a62-116">If the company uses Budget planning functionality to prepare budget data, you can use the **Generate budget register entry** periodic process.</span></span>

<span data-ttu-id="41a62-117">รายการทะเบียนงบประมาณจะถือว่าเสร็จสมบูรณ์เมื่อยอดดุลงบประมาณถูกอัพเดต</span><span class="sxs-lookup"><span data-stu-id="41a62-117">The budget register entry is considered completed when the budget balances have been updated.</span></span> <span data-ttu-id="41a62-118">ในหน้า **รายการทะเบียนงบประมาณ** คลิก **อัพเดตยอดดุลงบประมาณ** สำหรับงบประมาณลงทะเบียนที่ถูกเลือกหนึ่งรายการหรือหลายรายการ</span><span class="sxs-lookup"><span data-stu-id="41a62-118">On the **Budget register entries** page, click **Update budget balances** for a selected budget register entry or multiple entries.</span></span> <span data-ttu-id="41a62-119">หลังจากที่คุณอัพเดตยอดดุลงบประมาณ สถานะของรายการการลงทะเบียนงบประมาณจะเปลี่ยนเป็น **เสร็จสมบูรณ์**</span><span class="sxs-lookup"><span data-stu-id="41a62-119">After you update the budget balances, the status of the budget register entry changes to **Completed**.</span></span> <span data-ttu-id="41a62-120">รายการทะเบียนงบประมาณที่เสร็จสมบูรณ์ไม่สามารถเปิดใหม่เพื่อแก้ไขได้</span><span class="sxs-lookup"><span data-stu-id="41a62-120">Completed budget register entry can't be re-opened for edits.</span></span> <span data-ttu-id="41a62-121">ดังนั้น ถ้าข้อมูลงบประมาณต้องถูกปรับปรุง คุณต้องสร้างรายการทะเบียนงบประมาณใหม่แทนการแก้ไขข้อมูลในรายการทะเบียนงบประมาณที่เสร็จสมบูรณ์</span><span class="sxs-lookup"><span data-stu-id="41a62-121">Therefore, if the budget data must be adjusted, you must create a new budget register entry instead of correcting data in the completed budget register entry.</span></span>

<span data-ttu-id="41a62-123">เมื่อคุณตั้งค่าคอนฟิกการจัดงบประมาณ เริ่มต้นในหน้า **พารามิเตอร์การจัดทำงบประมาณ**</span><span class="sxs-lookup"><span data-stu-id="41a62-123">When you configure budgeting, start on the **Budgeting parameters** page.</span></span> <span data-ttu-id="41a62-124">ในหน้านี้ คุณต้องกำหนดสมุดรายวันงบประมาณ ลำดับหมายเลขสำหรับรายการทะเบียนงบประมาณ และลักษณะการทำงานเริ่มต้นในพื้นที่ทำงาน</span><span class="sxs-lookup"><span data-stu-id="41a62-124">On this page, you must define the budget journal, the number sequence for budget register entries, and the default behavior in the workspaces.</span></span>
