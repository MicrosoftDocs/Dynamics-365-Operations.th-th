# <a name="view-related-intercompany-voucher-from-journal"></a>ดูใบสำคัญระหว่างบริษัทที่เกี่ยวข้องจากสมุดรายวัน

[!include [banner](../../includes/banner.md)]

หน้าต่างใบสำคัญที่เกี่ยวข้องแสดงใบสำคัญจากบริษัทตรงข้าม เมื่อมีการลงรายการบัญชีธุรกรรมระหว่างบริษัทจากสมุดรายวันทั่วไป


## <a name="post-an-intercompany-journal"></a>ลงรายการบัญชีสมุดรายวันระหว่างบริษัท
1. ไปที่ **สมุดรายวันทั่วไป** คลิก **ใหม่**
2. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
3. ในฟิลด์ **ชื่อ** ให้ป้อนหรือเลือกชื่อสมุดรายวันระหว่างบริษัท
4. คลิก **รายการ** ในรายการ ทำเครื่องหมายแถวที่เลือก
5. ในฟิลด์ **บัญชี** ให้ระบุค่าที่ต้องการ
6. ในฟิลด์ **คำอธิบาย** ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
7. ปิดหน้า
8. ในฟิลด์ **เดบิต** ให้ป้อนตัวเลข
9. ในฟิลด์ **บริษัทตรงข้าม** ให้พิมพ์หรือเลือกบริษัทตรงข้าม
10. ในฟิลด์ **บัญชีตรงข้าม** ให้ระบุค่าที่ต้องการ
11. คลิก **ลงรายการบัญชี**

## <a name="view-related-intercompany-voucher"></a>ดูใบสำคัญระหว่างบริษัทที่เกี่ยวข้อง
1. คลิก **ใบสำคัญ**
2. คลิก **ใบสำคัญที่เกี่ยวข้อง** ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
3. คลิก **ใบสำคัญ**
