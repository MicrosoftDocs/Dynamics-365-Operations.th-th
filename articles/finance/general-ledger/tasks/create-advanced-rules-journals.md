# <a name="create-advanced-rules-for-journals"></a>สร้างกฎขั้นสูงสำหรับสมุดรายวัน

[!include [banner](../../includes/banner.md)]

กระบวนงานนี้ดำเนินการสร้างกฎขั้นสูงสำหรับสมุดรายวัน  ซึ่งรวมถึงการตั้งค่าการควบคุมสมุดรายวันและข้อจำกัดในการลงรายการบัญชีของผู้ใช้  กระบวนงานนี้ใช้บริษัทข้อมูลสาธิต USMF


## <a name="set-up-journal-control"></a>ตั้งค่าการควบคุมสมุดรายวัน
1. ใน **บานหน้าต่างนำทาง** ไปที่ **โมดูล > บัญชีแยกประเภททั่วไป > การตั้งค่าสมุดรายวัน > ชื่อสมุดรายวัน**
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
3. บน **บานหน้าต่างการดำเนินการ** ให้คลิก **การควบคุมสมุดรายวัน**
4. ใน fastTab **ชนิดบัญชีที่สามารถลงรายการบัญชีได้** ให้คลิก **เพิ่ม**
5. ในฟิลด์ **บัญชีบริษัท** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
6. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
7. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
8. ใน fastTab **ค่าเซ็กเมนต์ที่มีผลบังคับใช้สำหรับสมุดรายวันนี้** คลิก **เพิ่ม**
9. ในฟิลด์ **โครงสร้างทางบัญชี** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
10. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
11. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
12. ในฟิลด์ **เซ็กเมนต์** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
13. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
14. ในฟิลด์ **จากค่า** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
15. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
16. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
17. ในฟิลด์ **ไปที่ค่า** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
18. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
19. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก

## <a name="set-up-posting-restrictions"></a>การตั้งค่าข้อจำกัดการลงรายการบัญชี
1. ปิดหน้า
2. คลิก **ข้อจำกัดในการลงรายการบัญชี**
3. ใน **คุณต้องการตั้งค่าข้อจำกัดในการลงรายการบัญชีอย่างไร** ให้เลือก 'โดยกลุ่มผู้ใช้'
4. ในแผนภูมิ ให้ตรวจสอบ 'กลุ่มที่คุณต้องการอนุญาตให้มีการลงรายการบัญชีสำหรับชื่อสมุดรายวัน'
5. คลิก **ตกลง**
