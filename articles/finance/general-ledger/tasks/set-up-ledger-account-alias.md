# <a name="set-up-a-ledger-account-alias"></a>ตั้งค่านามแฝงของบัญชีแยกประเภท

[!include [banner](../../includes/banner.md)]

ขั้นตอนนี้แสดงวิธีการสร้างนามแฝงของบัญชีที่มีทางลัดสำหรับการป้อนหมายเลขบัญชี  ขั้นตอนนี้จะใช้ข้อมูลสาธิตบริษัท USMF

1. ไปที่ **บัญชีแยกประเภททั่วไป > ผังบัญชี > บัญชี > นามแฝงของบัญชีแยกประเภท** คลิก **ใหม่**
2. ในฟิลด์ **นามแฝงของบัญชีแยกประเภท** ให้พิมพ์ค่าใดค่าหนึ่ง
3. ในฟิลด์ **โครงสร้างทางบัญชี** เลือกโครงสร้างที่เป็นของบัญชีและมิติ
4. ในฟิลด์ **บริษัท** ในรายการนี้ ให้หาและเลือกบริษัทที่จะนำนามแฝงไปใช้
5. ในฟิลด์ **ข้อกำหนดนามแฝงของบัญชีแยกประเภท** ให้ระบุบัญชีและมิติ
    * บัญชีและมิติจะถูกเติมเมื่อใช้ทางลัด  
6. ในฟิลด์ **โฟกัสเริ่มต้น** ให้เลือกมิติที่จะมีโฟกัสเมื่อนามแฝงถูกใช้
    * หลังจากที่คุณพิมพ์ลัด และเติมข้อมูลบัญชีและมิติ ฟิลด์ **โฟกัสเริ่มต้น** คือ ตำแหน่งที่เคอร์เซอร์หรือโฟกัสย้ายไป
