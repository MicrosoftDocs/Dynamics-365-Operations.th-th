# <a name="set-up-a-ledger-account-alias"></a>ตั้งค่านามแฝงของบัญชีแยกประเภท

[!include [banner](../../includes/banner.md)]

ขั้นตอนนี้แสดงวิธีการสร้างนามแฝงของบัญชีที่มีทางลัดสำหรับการป้อนหมายเลขบัญชี  ขั้นตอนนี้จะใช้ข้อมูลสาธิตบริษัท USMF

1. ไปที่บัญชีแยกประเภททั่วไป > ผังบัญชี > บัญชี > นามแฝงของบัญชีแยกประเภท
2. คลิก สร้าง
3. ในฟิลด์นามแฝงของบัญชีแยกประเภท ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์โครงสร้างทางบัญชี เลือกโครงสร้างที่เป็นของบัญชีและมิติ
5. ในฟิลด์บริษัท ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
6. ในรายการนี้ ให้หาและเลือกบริษัทที่จะนำนามแฝงไปใช้
7. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
8. ในฟิลด์ข้อกำหนดนามแฝงของบัญชีแยกประเภท ให้ระบุบัญชีและมิติ
    * บัญชีและมิติจะถูกเติมเมื่อใช้ทางลัด  
9. ในฟิลด์โฟกัสเริ่มต้น ให้เลือกมิติที่จะมีโฟกัสเมื่อนามแฝงถูกใช้
    * หลังจากที่คุณพิมพ์ลัด และเติมข้อมูลบัญชีและมิติ ฟิลด์โฟกัสเริ่มต้นคือ ตำแหน่งที่เคอร์เซอร์หรือโฟกัสย้ายไป
