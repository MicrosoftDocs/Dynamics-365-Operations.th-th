# <a name="create-a-main-account"></a>สร้างบัญชีหลัก

[!include [banner](../../includes/banner.md)]

คำแนะนำงานนี้ระบุการเพิ่มบัญชีหลักให้กับผังบัญชีที่มีอยู่  การบันทึกข้อมูลนี้ใช้บริษัทสาธิต USMF  

1. ไปที่ **บานหน้าต่างนำทาง > โมดูล > บัญชีแยกประเภททั่วไป > ผังบัญชี > บัญชี > บัญชีหลัก**
2. คลิก **สร้าง**
3. ในฟิลด์ **บัญชีหลัก** ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ **ชื่อ** ให้พิมพ์ค่าใดค่าหนึ่ง
5. ในฟิลด์ **ชนิดบัญชีหลัก** ให้เลือกชนิดที่แสดงยอดดุลบัญชีและสถานที่ในงบการเงินได้ดีที่สุด
6. ในรายการ ให้เลือกประเภทบัญชีที่เป็นของบัญชีหลัก  ประเภทลูกค้าองค์กรถูกใช้สำหรับรายงานทางการเงินเริ่มต้นและเนื้อหาแดชบอร์ด Power BI  
7. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก เปลี่ยนยอดดุลเดบิตหรือเครดิตเริ่มต้น  
8. ในฟิลด์ **สกุลเงินเริ่มต้น** ให้เลือกค่าจากรายการของสกุลเงิน
9. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
10. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
11. สลับการขยายของส่วน **การแทนที่นิติบุคคล**
12. คลิก **เพิ่ม** เพื่อเลือกนิติบุคคล
13. ในรายการ ให้เลือกนิติบุคคล
14. คลิก **เพิ่ม**
15. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
16. เลือกหรือยกเลิกการเลือกกล่องกาเครื่องหมาย **ระงับแล้ว**
17. ขยายส่วน **การรายงานทางการเงิน**
18. ในฟิลด์ **ชนิดอัตราแลกเปลี่ยน** ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
19. ในรายการ เลือก **ชนิดอัตราแลกเปลี่ยนสำหรับบัญชี**
20. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
21. ในฟิลด์ **ชนิดการแปลงสกุลเงิน** ให้เลือกวิธีสำหรับการคำนวณอัตราแลกเปลี่ยนสำหรับบัญชี
22. ปิดหน้า
