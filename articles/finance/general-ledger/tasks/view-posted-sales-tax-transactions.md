# <a name="view-posted-sales-tax-transactions"></a>ดูธุรกรรมภาษีขายที่ลงรายการบัญชี

[!include [banner](../../includes/banner.md)]

หัวข้อนี้จะอธิบายวิธีการบางอย่างที่จะดูธุรกรรมภาษีขายที่ลงรายการบัญชีแล้ว คุณสามารถดูธุรกรรมภาษีขายที่ลงรายการบัญชีไว้แล้วจากหน้าต่างๆ โดยขึ้นอยู่กับสิ่งที่คุณต้องการดู งานนี้ใช้บริษัทสาธิต USMF 

## <a name="posted-sales-tax-page"></a>หน้าภาษีขายที่ลงรายการบัญชี

1. ในบานหน้าต่างนำทางให้ ไปที่ **โมดูล > ภาษี > การสอบถามและรายงาน > การสอบถามภาษีขาย > ภาษีขายที่ลงรายการบัญชี**
2. เลือก **แสดงตัวกรอง** เพื่อจำกัดมุมมองให้เป็นธุรกรรมที่เฉพาะเจาะจง
3. ปิดหน้า

## <a name="sales-tax-settlement-periods-page"></a>หน้ารอบระยะเวลาการชำระภาษีขาย

1. ในบานหน้าต่างนำทางให้ ไปที่ **โมดูล > ภาษี > ภาษีทางอ้อม > ภาษีขาย > รอบระยะเวลาการชำระภาษีขาย**
2. ขยายส่วน **ช่วงรอบระยะเวลา**
3. เลือกช่วงเวลาที่คุณสนใจ
4. คุณยังสามารถกรองรายการธุรกรรมภาษีขายที่ลงรายการบัญชีไว้แล้ว
5. ปิดหน้า

## <a name="sales-tax-codes-page"></a>หน้ารหัสภาษีขาย

1. ในบานหน้าต่างนำทาง ไปที่ **โมดูล > ภาษี > ภาษีทางอ้อม > ภาษีขาย > รหัสภาษีขาย**
2. บนหน้าต่างการดำเนินการ เลือก **รหัสภาษีขาย**
3. เลือก **ภาษีขายที่ลงรายการบัญชี**
