# <a name="financial-reporting-overview"></a>ภาพรวมของการรายงานทางการเงิน

[!include [banner](../includes/banner.md)]

หัวข้อนี้อธิบายวิธีการจัดเก็บรายงานทางการเงินและวิธีการใช้รายงานทางการเงินอย่างมีประสิทธิภาพ ประกอบด้วยคำอธิบายของรายงานทางการเงินเริ่มต้นที่กำหนดไว้

<a name="accessing-financial-reporting"></a>การเข้าถึงรายงานทางการเงิน
-----------------------------

คุณสามารถค้นหาเมนู **รายงานทางการเงิน** ได้ตามนี้:

-   **บัญชีแยกประเภททั่วไป** &gt; **การสอบถามและรายงาน**
-   **การจัดงบประมาณ** &gt; **การสอบถามและรายงาน** &gt; **การจัดงบประมาณพื้นฐาน**
-   **การจัดงบประมาณ** &gt; **การสอบถามและรายงาน** &gt; **การวางแผนงบประมาณ**
-   **การจัดงบประมาณ** &gt; **การสอบถามและรายงาน** &gt; **การควบคุมงบประมาณ**
-   การรวมบัญชี

เมื่อต้องการสร้างรายงานทางการเงินสำหรับนิติบุคคล คุณต้องตั้งค่าข้อมูลต่อไปนี้สำหรับนิติบุคคลนั้น:

-   ปฏิทินทางการเงิน
-   บัญชีแยกประเภท
-   ผังบัญชี
-   สกุลเงิน

ฟังก์ชันการรายงานทางการเงินพร้อมใช้งานสำหรับผู้ใช้ที่มีสิทธิ์ที่เหมาะสม และถูกกำหนดหน้าที่ให้โดยบทบาทความปลอดภัย ส่วนต่อไป แสดงรายการสิทธิ์และหน้าที่เหล่านี้ ร่วมกับบทบาทเกี่ยวข้อง

## <a name="report-deletions-and-expirations"></a>รายงานการลบและการหมดอายุ
ผู้ใช้ที่สร้างรายงานสามารถลบรายงานของตนเองได้ ผู้ใช้ที่มีหน้าที่ **รักษาความปลอดภัยของการรายงานทางการเงิน** สามารถลบรายงานของผู้อื่นได้ 

โดยเริ่มต้นด้วยรุ่น 10.0.7 แนวคิดของวันหมดอายุได้รับการแนะนำ คุณลักษณะแบบบังคับใหม่จะถูกเปิดใช้งานในพื้นที่ทำงานของการจัดการคุณลักษณะ คุณลักษณะนี้ประกอบด้วยการเปลี่ยนแปลงต่อไปนี้
* รายงานที่สร้างขึ้นใหม่จะถูกทำเครื่องหมายโดยอัตโนมัติว่ามีวันหมดอายุเป็น 90 วันนับจากเวลาที่สร้างขึ้น
* รายงานที่มีอยู่ใดๆ จากก่อนที่จะมีการติดตั้งคุณลักษณะนี้ จะถูกกำหนดระยะเวลาการหมดอายุ 90 วัน วันที่อาจแสดงเป็นว่างเป็นระยะเวลาสั้นๆ จนกว่าจะมีการรันบริการ Financial Reporting จะมีการสร้างรายงาน และการบริการทำการปรับรุ่นเป็นรายงานที่มีอยู่พร้อมด้วยวันหมดอายุที่ว่างเปล่า 
* ผู้ใช้ที่มี **รักษาความปลอดภัยของรายงานทางการเงิน** มีการเข้าถึงฟังก์ชันนี้ ผู้ใช้ใดๆ ในหน้าที่ **รักษารายงานทางการเงิน** ที่ให้สิทธิ์ **รักษาการหมดอายุของรายงานทางการเงิน** ยังจะมีความสามารถในการปรับเปลี่ยนรอบระยะเวลาการหมดอายุด้วย ขณะนี้มีตัวเลือกการเก็บข้อมูลสองตัวเลือกที่พร้อมใช้งาน 
  * การหมดอายุเป็น 90 วัน
  * ตัวเลือกในการตั้งค่ารายงานที่ไม่มีวันหมดอายุ

เมื่อมีการเลือกการหมดอายุ เช่น 90 วัน จะมีการกำหนดให้ 90 วันนับจากวันนี้ ซึ่งเป็นลักษณะการทำงานที่แตกต่างจาก 90 วันจากวันที่สร้างต้นฉบับที่ตั้งค่าในระหว่างการสร้างรายงาน

## <a name="opening-financial-reports"></a>การเปิดรายงานทางการเงิน
เมื่อคุณคลิกเมนู **รายงานทางการเงิน** รายการเริ่มต้นรายงานทางการเงินสำหรับบริษัทจะปรากฏขึ้น คุณสามารถเปิด หรือปรับเปลี่ยนรายงาน เมื่อต้องการเปิดหนึ่งในรายงานเริ่มต้น เลือกชื่อรายงาน ในครั้งแรกที่มีการเปิดรายงาน รายงานจะถูกสร้างโดยอัตโนมัติสำหรับเดือนก่อนหน้านี้ ตัวอย่างเช่น ถ้าคุณเปิดการรายงานเป็นครั้งแรกในเดือน 2016 สิงหาคม รายงานถูกสร้างขึ้นสำหรับวันที่ 31 กรกฎาคม 2016 หลังจากที่มีการเปิดรายงาน คุณสามารถเริ่มสำรวจ โดยการดูรายละเอียดแนวลึกในส่วนของข้อมูลเฉพาะเจาะจง และการเปลี่ยนตัวเลือกรายงาน

## <a name="creating-and-modifying-financial-reports"></a>การสร้างและแก้ไขรายงานทางการเงิน
คุณสามารถสร้างรายงานใหม่ หรือแก้ไขรายงานที่มีอยู่จากรายการรายงานทางการเงิน ถ้าคุณมีสิทธิ์ที่เหมาะสม คุณสามารถสร้างรายงานทางการเงินใหม่ด้วยการคลิก **สร้าง** บนบานหน้าต่างการดำเนินการ โปรแกรมออกแบบรายงานจะถูกดาวน์โหลดไปยังอุปกรณ์ของคุณ หลังจากที่โปรแกรมออกแบบรายงานเริ่มทำงาน คุณสามารถสร้างรายงานใหม่ หลังจากคุณบันทึกรายงานใหม่ จะปรากฏอยู่ในรายการรายงานทางการเงิน รายการจะแสดงเฉพาะรายงานที่สร้างขึ้นสำหรับบริษัทที่คุณกำลังใช้ในทางการเงิน 

> [!NOTE] 
> คอมพิวเตอร์ที่คุณกำลังดาวน์โหลดไคลเอ็นต์ตัวออกแบบรายงาน ต้องมีรุ่น 4.6.2 ของกรอบงาน Microsoft .NET ที่ติดตั้งอยู่ รุ่นนี้ของกรอบงาน Microsoft .NET สามารถดาวน์โหลดและติดตั้งได้จาก [ศูนย์ดาวน์โหลด Microsoft](https://www.microsoft.com/download/details.aspx?id=53345) ถ้าคุณใช้ Chrome คุณต้องติดตั้งส่วนขยาย ClickOnce เพื่อดาวน์โหลดไคลเอ็นต์โปรแกรมออกแบบรายงาน ถ้าคุณกำลังทำงานในโหมด incognito ให้ตรวจสอบให้แน่ใจว่าได้เปิดใช้งานส่วนขยาย ClickOnce สำหรับโหมด incognito แล้ว คุณยังสามารถแก้ไขรายงานที่ปรากฏอยู่ในรายการรายงานทางการเงิน เลือกพื้นที่โดยรอบชื่อรายงานถูก คลิก **แก้ไข** บนบานหน้าต่างการดำเนินการ เริ่มต้นโปรแกรมออกแบบรายงาน
