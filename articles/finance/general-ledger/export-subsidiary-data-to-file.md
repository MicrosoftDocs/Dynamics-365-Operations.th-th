# <a name="export-subsidiary-data-to-files"></a>การส่งออกข้อมูลบริษัทในเครือไปยังไฟล์อื่น

[!include [banner](../includes/banner.md)]

คุณใช้หน้า **การส่งออก** (**การจัดการระบบ \> พื้นที่ทำงาน \> การนําเข้า/ส่งออก**) เพื่อจัดเตรียมการส่งออกข้อมูลบริษัทในเครือไปยังไฟล์ที่สามารถนําเข้าไปยังนิติบุคคลที่รวมบัญชีแล้วได้ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับกระบวนการการนำเข้าและส่งออกข้อมูล ให้ดูที่ [ภาพรวมของงานการนำเข้าและส่งออกข้อมูล](../../fin-ops-core/dev-itpro/data-entities/data-import-export-job.md)

1. สร้างนิติบุคคลสำหรับใช้ในกระบวนการรวมบัญชี สำหรับข้อมูลเกี่ยวกับวิธีการสร้างนิติบุคคล ให้ดูที่ [สร้างนิติบุคคล](../../fin-ops-core/fin-ops/organization-administration/tasks/create-legal-entity.md) สำหรับข้อมูลเพิ่มเติม ให้ดูที่ [จัดเตรียมนิติบุคคลที่จะใช้ในกระบวนการรวมบัญชี](prepare-company-for-consolidation.md) และ [ตั้งค่านิติบุคคลของบริษัทในเครือเพื่อการรวมบัญชี](set-up-subsidiary-company-for-consolidation.md) 

2. ไปที่ **การรวมบัญชี \> ส่งออกยอดดุลของบริษัท** บนหน้า **ส่งออกยอดดุลของบริษัท** บนแท็บ **เงื่อนไข** ให้ระบุรายละเอียดของการรวมบัญชีโดยการตั้งค่าฟิลด์ต่อไปนี้

    | ฟิลด์                             | คำอธิบาย |
    |-----------------------------------|-------|
    | บัญชีหลัก                      | ระบุบัญชีที่จะรวมบัญชี เมื่อต้องการรวมบัญชีทั้งหมด ให้ปล่อยฟิลด์นี้ว่างไว้ |
    | ใช้บัญชีหลักในการรวมบัญชี         | ถ้าคุณระบุบัญชีหลักในการรวมบัญชีไว้ ให้ตั้งค่าตัวเลือกนี้เป็น **ใช่** |
    | เลือกบัญชีการรวมบัญชีจาก | เลือก **บัญชีหลัก** หรือ **บัญชีหลักในการรวมบัญชี** |
    | กลุ่มบัญชีการรวมบัญชี       | เลือกกลุ่มบัญชีหลักในการรวมบัญชีสำหรับบัญชีหลักในการรวมบัญชีที่คุณเลือกไว้ |
    | รอบระยะเวลาการรวมบัญชี              | ระบุวันที่ "จาก" และ "ถึง" สำหรับการรวมบัญชี |
    | รวมยอดเงินจริง            | ตั้งค่าตัวเลือกนี้เป็น **ใช่** เพื่อรวมยอดเงินจริง |
    | รวมยอดเงินงบประมาณ            | ตั้งค่าตัวเลือกนี้เป็น **ใช่** เพื่อรวมยอดเงินงบประมาณในการรวมบัญชี |
    | แบบจำลองงบประมาณ                     | ระบุรูปแบบงบประมาณที่จะรวม |

3. บนแท็บ **มิติทางการเงิน** ระบุรายละเอียดของการรวมบัญชี

    - ระบุข้อมูลมิติทางการเงินที่ควรจะโอนย้ายจากธุรกรรมในบัญชีบริษัทในเครือไปยังธุรกรรมในนิติบุคคลที่รวมบัญชี
    - เลือกมิติทางการเงินในรายการ
    - ระบุข้อมูลจำเพาะระบุที่ถูกต้องสำหรับมิติทางการเงินแต่ละมิติที่มีการรวมบัญชี ตัวเลือกที่พร้อมใช้งานได้แก่ **มิติ** **มิติกลุ่ม** **บัญชีบริษัท** และ **บัญชี**

        > [!NOTE]
        > ตัวเลือก **มิติกลุ่ม** ช่วยให้คุณกําหนดค่ามิติที่ใช้โดยกลุ่มของบริษัทที่จะรวมบัญชีเข้าด้วยกัน

    - ระบุใบสั่งเซ็กเมนต์ที่จะรวมบัญชีไปด้วย

4. บนแท็บ **นิติบุคคล** ให้ปฏิบัติตามขั้นตอนต่อไปนี้เพื่อระบุนิติบุคคลที่คุณส่งออก

    1. เลือก **ใหม่**
    2. ในฟิลด์ **แหล่งที่มานิติบุคคล** ให้ป้อนนิติบุคคล

        หากใช้เงื่อนไขเดียวกันกับบริษัทในเครือหลายบริษัทที่อยู่ในฐานข้อมูลเดียวกัน คุณสามารถโอนย้ายข้อมูลจากบริษัทในเครือเหล่านั้นไปยังไฟล์การส่งออกที่แยกต่างหากในการดำเนินการครั้งเดียวได้

        1. สร้างบรรทัดสำหรับแต่ละนิติบุคคลบริษัทในเครือแต่ละบริษัทที่มีบัญชีซึ่งจะส่งออกไปยังไฟล์ต่างๆ ไฟล์เหล่านี้จะถูกนำเข้าไปรวมบัญชีนิติบุคคลภายหลัง
        2. สำหรับบริษัทในเครือแต่ละบริษัท ให้ป้อนชื่อบริษัทในเครือและชื่อไฟล์ส่งออกที่จะสร้างขึ้นระหว่างงานการส่งออก

    3. ในฟิล์ด **ชนิดบัญชีของความต่างของการแปลง** เลือก **กําไรขาดทุน** หรือ **งบดุล**
    4. ป้อนชื่อไฟล์สำหรับไฟล์การนำออกที่จะสร้างขึ้น

5. เลือก **ตกลง** เพื่อเรียกใช้การนำออก

เมื่อการส่งออกเสร็จสมบูรณ์ คุณจะได้รับข้อความที่แสดงหมายเลขของเรกคอร์ดต่างๆซึ่งถูกบันทึกไว้ในแต่ละไฟล์ จากนั้นคุณจะสามารถนำเข้าไฟล์ต่างๆไปยังนิติบุคคลที่ได้รวมบัญชีไว้
