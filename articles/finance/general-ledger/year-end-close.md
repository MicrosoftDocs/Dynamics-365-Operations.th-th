# <a name="year-end-close"></a><span data-ttu-id="43a40-103">การปิดบัญชีสิ้นปี</span><span class="sxs-lookup"><span data-stu-id="43a40-103">Year-end close</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="43a40-104">หัวข้อนี้อธิบายถึงการตั้งค่าที่จำเป็นและขั้นตอนสำหรับการรันกระบวนการปิดเมื่อสิ้นสุดปีของบัญชีแยกประเภททั่วไป</span><span class="sxs-lookup"><span data-stu-id="43a40-104">This topic describes the required setup and steps for running the general ledger year-end close process.</span></span> 

<span data-ttu-id="43a40-105">เมื่อสิ้นสุดปีบัญชี คุณต้องดำเนินกระบวนการปิดบัญชีสิ้นปีเพื่อโอนย้ายยอดดุลเปิดไปยังปีถัดไป</span><span class="sxs-lookup"><span data-stu-id="43a40-105">At the end of a fiscal year, you must run the year-end close process to transfer opening balances to the new year.</span></span> <span data-ttu-id="43a40-106">องค์กรส่วนใหญ่จะรันกระบวนการปิดบัญชีสิ้นปีหลายครั้ง</span><span class="sxs-lookup"><span data-stu-id="43a40-106">Most organizations will run the year-end close process multiple times.</span></span> <span data-ttu-id="43a40-107">ในครั้งแรกจะเป็นการเรียกดูยอดดุลที่ย้ายไปยังปีบัญชีใหม่</span><span class="sxs-lookup"><span data-stu-id="43a40-107">The first time would be to get the balances moved into the new fiscal year.</span></span> <span data-ttu-id="43a40-108">จากนั้นการปิดบัญชีสิ้นปีสามารถรันอีกครั้ง กี่ครั้งก็ได้ตามที่จำเป็น เพื่อย้ายยอดดุลจากการปรับปรุงรายการไปยังปีบัญชีใหม่</span><span class="sxs-lookup"><span data-stu-id="43a40-108">The year-end close can then be run again, as many times are required, to move the balances from adjusting entries into the new fiscal year.</span></span> 

<span data-ttu-id="43a40-109">ในระหว่างกระบวนการปิดบัญชีสิ้นปี มีการสร้างธุรกรรมที่เป็นไปได้สองชนิด</span><span class="sxs-lookup"><span data-stu-id="43a40-109">During the year-end close process, there are two types of possible transactions created.</span></span> <span data-ttu-id="43a40-110">ธุรกรรมยอดยกมาจะถูกสร้างและใช้ในการสร้างยอดดุลยกมาในปีบัญชีใหม่เสมอ</span><span class="sxs-lookup"><span data-stu-id="43a40-110">An Opening transaction is always generated and is used to create the opening balances in the new fiscal year.</span></span> <span data-ttu-id="43a40-111">ธุรกรรมยอดยกมาแสดงยอดดุลในบัญชีแยกประเภทงบดุลในปีบัญชีใหม่ และยอดดุลจากยอดดุลบัญชีแยกประเภทกำไรและขาดทุนในบัญชีแยกประเภทกำไรสะสมในปีบัญชีใหม่</span><span class="sxs-lookup"><span data-stu-id="43a40-111">The Opening transaction shows the balance sheet ledger account balances in the new fiscal year and balances from the profit and loss ledger account balances in the retained earnings ledger account in the new fiscal year.</span></span> <span data-ttu-id="43a40-112">อีกทางหนึ่งคือมีการสร้างธุรกรรมปิดบัญชีเพื่อให้ยอดดุลของบัญชีกำไรและขาดทุนลดลงเป็นศูนย์ในปีบัญชีที่กำลังจะถูกปิด</span><span class="sxs-lookup"><span data-stu-id="43a40-112">The Closing transaction is optionally created to bring the balances of the profit and loss accounts down to zero in the fiscal year being closed.</span></span>
