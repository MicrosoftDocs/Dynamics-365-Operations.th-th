# <a name="enable-delayed-tax-calculation-on-journal"></a>เปิดใช้งานการคำนวณภาษีที่ล่าช้าในสมุดรายวัน
[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

หัวข้อนี้อธิบายวิธีการใช้ **เปิดใช้งานการคำนวณภาษีที่ล่าช้าในสมุดรายวัน** เพื่อปรับปรุงประสิทธิภาพการคำนวณภาษี เมื่อจำนวนบรรทัดสมุดรายวันมีปริมาณมาก

ลักษณะการทำงานการคำนวณภาษีขายปัจจุบันในสมุดรายวันเป็นแบบ real-time ซึ่งทริกเกอร์เมื่อผู้ใช้อัพเดตฟิลด์ที่เกี่ยวข้องกับภาษี เช่น กลุ่มภาษีขาย/กลุ่มภาษีขายตามประเภทสินค้า การอัพเดตใด ๆ ในสมุดรายวัน จะคำนวณยอดภาษีใหม่ในสมุดรายวันทั้งหมดอีกครั้ง จะช่วยให้ผู้ใช้สามารถดูยอดเงินภาษีที่คำนวณได้แบบเรียลไทม์ แต่อาจนำปัญหาด้านประสิทธิภาพไปใช้ถ้าจำนวนบรรทัดของสมุดรายวันมีปริมาณมาก

ลักษณะการทำงานนี้จะให้ตัวเลือกในการหน่วงเวลาการคำนวณภาษีเพื่อแก้ไขปัญหาประสิทธิภาพการทำงาน ถ้าเปิดใช้งานลักษณะการทำงานนี้ จะมีการคำนวณจำนวนเงินภาษีเมื่อผู้ใช้คลิก "ภาษีขาย" เท่านั้นหรือลงรายการบัญชีสมุดรายวันเท่านั้น

ผู้ใช้สามารถ เปิด/ปิด พารามิเตอร์นี้ได้สามระดับ:
- โดยนิติบุคคล
- โดยชื่อสมุดรายวัน
- โดยส่วนหัวของสมุดรายวัน

ระบบจะใช้ค่าพารามิเตอร์ในส่วนหัวของสมุดรายวันเป็นขั้นสุดท้าย ค่าพารามิเตอร์ในส่วนหัวของสมุดรายวันจะเป็นค่าเริ่มต้นจากชื่อสมุดรายวัน ค่าพารามิเตอร์ในชื่อสมุดรายวันจะเป็นค่าเริ่มต้นจากพารามิเตอร์บัญชีแยกประเภททั่วไป เมื่อมีการสร้างชื่อสมุดรายวัน

"ยอดภาษีขายจริง" และ "ยอดเงินภาษีขายที่คำนวณได้" ในสมุดรายวันจะถูกซ่อนไว้ ถ้ามีการเปิดใช้งานพารามิเตอร์นี้ วัตถุประสงค์เพื่อให้ผู้ใช้ไม่สับสน เนื่องจากค่าของฟิลด์สองฟิลด์เหล่านี้จะแสดง 0 เสมอก่อนที่ผู้ใช้จะทริกเกอร์การคำนวณภาษี

## <a name="enable-delayed-tax-calculation-by-legal-entity"></a>เปิดใช้งานการคำนวณภาษีที่ล่าช้าโดยเรียงตามนิติบุคคล

1. ไปที่ **บัญชีแยกประเภททั่วไป > การตั้งค่าบัญชีแยกประเภท > พารามิเตอร์บัญชีแยกประเภททั่วไป**
2. คลิกแท็บ **ภาษีขาย**
3. ภายใต้แท็บด่วน **ทั่วไป** หาพารามิเตอร์ **การคำนวณภาษีที่ล่าช้า** ให้เปิด/ปิด

![](media/delayed-tax-calculation-gl.png)



## <a name="enable-delayed-tax-calculation-by-journal-name"></a>เปิดใช้งานการคำนวณภาษีที่ล่าช้าตามชื่อสมุดรายวัน

1. ไปที่ **บัญชีแยกประเภททั่วไป > การตั้งค่าสมุดรายวัน > ชื่อสมุดรายวัน**
2. ภายใต้แท็บด่วน **ทั่วไป** หาพารามิเตอร์ **การคำนวณภาษีที่ล่าช้า** ให้เปิด/ปิด

![](media/delayed-tax-calculation-journal-name.png)

## <a name="enable-delayed-tax-calculation-by-journal"></a>เปิดใช้งานการคำนวณภาษีที่ล่าช้า ตามสมุดรายวัน

1. ไปที่ **บัญชีแยกประเภททั่วไป > รายการสมุดรายวัน > สมุดรายวันทั่วไป**
2. คลิก **สร้าง**
3. เลือกชื่อสมุดรายวัน
4. คลิก **ตั้งค่า**
5. หาพารามิเตอร์ **การคำนวณภาษีที่ล่าช้า** ให้เปิด/ปิด

![](media/delayed-tax-calculation-journal-header.png)
