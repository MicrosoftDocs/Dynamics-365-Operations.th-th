<span data-ttu-id="93b08-104">คุณสามารถใช้ตัวออกแบบคลิกเดียวเพื่อออกแบบโครงร่าง Retail Modern POS (MPOS) และ Cloud POS ที่แตกต่างกัน ในโหมดแนวนอนและโหมดแนวตั้ง อย่างใดอย่างหนึ่ง สำหรับร้านค้า การลงทะเบียน พนักงานเก็บเงิน และผู้จัดการ</span><span class="sxs-lookup"><span data-stu-id="93b08-104">You can use the one-click designer to design different Retail Modern POS (MPOS) and Cloud POS layouts, in either Landscape mode or Portrait mode, for stores, registers, cashiers, and managers.</span></span>

<span data-ttu-id="93b08-105">อินเทอร์เฟซการออกแบบกราฟิกสำหรับ MPOS หรือ POS ระบบคลาวด์จะถูกควบคุมโดยโครงร่างลิ้นชักเก็บเงิน</span><span class="sxs-lookup"><span data-stu-id="93b08-105">The graphical design interface for MPOS or Cloud POS is controlled by the till layout.</span></span> <span data-ttu-id="93b08-106">โครงร่างจะควบคุมตำแหน่งของวัตถุต่างๆ</span><span class="sxs-lookup"><span data-stu-id="93b08-106">A layout controls the position of various objects.</span></span> <span data-ttu-id="93b08-107">ตัวอย่างรวมถึงโครงร่างรวม โครงร่างกริดสินค้า โครงร่างของลูกค้า โครงร่างการชำระเงิน และโครงร่างของปุ่มเมนูต่างๆ</span><span class="sxs-lookup"><span data-stu-id="93b08-107">Examples include the total layout, the item grid layout, the customer layout, the payment layout, and the layout of various menu buttons.</span></span> <span data-ttu-id="93b08-108">โครงร่างยังรวมถึงลักษณะโดยรวมของอินเทอร์เฟสการขายที่เสนอให้แก่ผู้ปฏิบัติงาน</span><span class="sxs-lookup"><span data-stu-id="93b08-108">Layouts also include the overall appearance of the sales interface that is presented to workers.</span></span>

## <a name="install-the-one-click-designer"></a><span data-ttu-id="93b08-109">ติดตั้งตัวออกแบบคลิกเดียว</span><span class="sxs-lookup"><span data-stu-id="93b08-109">Install the one-click designer</span></span>

1. <span data-ttu-id="93b08-110">ใน Retail ใช้เมนูในมุมบนซ้ายเพื่อนำทางไปยัง **Retail** **และการค้า** &gt; **การตั้งค่าช่องทาง** &gt; **การตั้งค่า POS** &gt; **POS** &gt; **โครงร่างหน้าจอ**</span><span class="sxs-lookup"><span data-stu-id="93b08-110">In Retail, use the menu in the upper left to navigate to **Retail** **and commerce** &gt; **Channel setup** &gt; **POS setup** &gt; **POS** &gt; **Screen layouts**.</span></span>
2. <span data-ttu-id="93b08-111">เลือกโครงร่างใดๆ ที่มีชนิดแอพลิเคชันของ **Modern POS สำหรับ Windows** หรือ **POS ระบบคลาวด์**แล้วคลิก **ตัวออกแบบโครงร่าง**</span><span class="sxs-lookup"><span data-stu-id="93b08-111">Select any layout that has an application type of **Modern POS for Windows** or **Cloud POS**, and then click **Layout designer**.</span></span>
3. <span data-ttu-id="93b08-112">บนแถบการแจ้งเตือนที่ปรากฏที่ด้านล่างของหน้าต่าง Internet Explorer คลิก **เปิด** เพื่อตั้งตัวออกแบบด้วยคลิกเดียว</span><span class="sxs-lookup"><span data-stu-id="93b08-112">On the notification bar that appears at the bottom of the Internet Explorer window, click **Open** to install the one-click designer.</span></span> <span data-ttu-id="93b08-113">(แถบการแจ้งเตือนอาจปรากฏในตำแหน่งอื่นในเบราเซอร์อื่น)</span><span class="sxs-lookup"><span data-stu-id="93b08-113">(The notification bar might appear in a different place in other browsers.)</span></span>
4. <span data-ttu-id="93b08-114">ในกล่องข้อความ **การรันแอพลิเคชัน - คำเตือนเกี่ยวกับความปลอดภัย** จะปรากฏขึ้น คลิก **รัน**เพื่อติดตั้งโฮสต์ตัวออกแบบขายปลีก</span><span class="sxs-lookup"><span data-stu-id="93b08-114">In the **Application Run - Security Warning** message box that appears, click **Run** to install the Retail designer host.</span></span> <span data-ttu-id="93b08-115">ตัวบ่งชี้ความคืบหน้าแสดงความคืบหน้าของการติดตั้ง</span><span class="sxs-lookup"><span data-stu-id="93b08-115">A progress indicator shows the progress of the installation.</span></span>
5. <span data-ttu-id="93b08-116">หลังจากการติดตั้งเสร็จสิ้น บนหน้า **ลงชือเข้าใช้** ป้อนชื่อผู้ใช้และรหัสผ่าน Retail ของคุณ และจากนั้นคลิก **ลงชื่อเข้าใช้** เพื่อเริ่มต้นตัวออกแบบ</span><span class="sxs-lookup"><span data-stu-id="93b08-116">After the installation is completed, on the **Sign in** page, enter your Retail user name and password, and then click **Sign in** to start the designer.</span></span>
6. <span data-ttu-id="93b08-117">หลังจากที่มีการตรวจสอบข้อมูลประจำตัวของคุณและมีการเริ่มต้นตัวออกแบบ คุณสามารถออกแบบโครงร่างของคุณเองหรือปรับเปลี่ยนโครงร่างที่มีอยู่</span><span class="sxs-lookup"><span data-stu-id="93b08-117">After your credentials are validated and the designer starts, you can design your own layout or modify the existing layout.</span></span>

## <a name="troubleshoot-the-installation-of-the-layout-designer"></a><span data-ttu-id="93b08-119">แก้ไขปัญหาการติดตั้งตัวออกแบบโครงร่าง</span><span class="sxs-lookup"><span data-stu-id="93b08-119">Troubleshoot the installation of the Layout designer</span></span>

- <span data-ttu-id="93b08-120">เมื่อคุณคลิก **ผู้ออกแบบ**พร้อมต์เพื่อดาวน์โหลด (หรือรัน) ตัวติดตั้งจะไม่ปรากฏ หรือการตั้งค่าความปลอดภัยปัจจุบันของคุณจะไม่อนุญาตให้คุณดาวน์โหลดไฟล์ได้</span><span class="sxs-lookup"><span data-stu-id="93b08-120">When you click **Designer**, the prompt to download (or run) the installer doesn't appear, or your current security settings don't allow you to download the file.</span></span> 

    <span data-ttu-id="93b08-121">**โซลูชัน:**</span><span class="sxs-lookup"><span data-stu-id="93b08-121">**Solutions:**</span></span>

    - <span data-ttu-id="93b08-122">ใน Internet Explorer ต้องแน่ใจว่าตัวบล็อกป็อปอัพได้ถูกปิดใช้งานสำหรับไซต์นี้</span><span class="sxs-lookup"><span data-stu-id="93b08-122">In Internet Explorer, make sure that the pop-up blocker is disabled for this site.</span></span> <span data-ttu-id="93b08-123">คลิก **การตั้งค่า** &gt; **ตัวเลือก** &gt; **ความเป็นส่วนตัว** &gt; **ค้นหาตัวบล็อกป๊อปอัพและเปลี่ยนการตั้งค่า** ถ้าจำเป็นต้องมีการเปลี่ยนแปลง</span><span class="sxs-lookup"><span data-stu-id="93b08-123">Click **Settings** &gt; **Options** &gt; **Privacy** &gt; **Find Pop-up Blocker**, and change the setting, if a change is required.</span></span>
    - <span data-ttu-id="93b08-124">ใน Internet Explorer เพิ่ม URL ของ Retail ไปยังไซต์ที่เชื่อถือของคุณ</span><span class="sxs-lookup"><span data-stu-id="93b08-124">In Internet Explorer, add the Retail URL to your trusted sites.</span></span> <span data-ttu-id="93b08-125">คลิก **การตั้งค่า** &gt; **ตัวเลือก** &gt; **ความปลอดภัย** &gt; **ไซต์ที่เชื่อถือได้** &gt; **ไซต์** &gt; **เพิ่ม**</span><span class="sxs-lookup"><span data-stu-id="93b08-125">Click **Settings** &gt; **Options** &gt; **Security** &gt; **Trusted sites** &gt; **Sites** &gt; **Add**.</span></span>

- <span data-ttu-id="93b08-126">โปรแกรมจะไม่เริ่มต้น และระบบจะแนะนำให้คุณติดต่อผู้จัดจำหน่าย</span><span class="sxs-lookup"><span data-stu-id="93b08-126">The program doesn't start, and you're instructed to contact the vendor.</span></span>

    <span data-ttu-id="93b08-127">**โซลูชั่น:** ใน Internet Explorer เพิ่ม URL ของ Retail ไปยังไซต์ที่เชื่อถือของคุณ</span><span class="sxs-lookup"><span data-stu-id="93b08-127">**Solution:** In Internet Explorer, add the Retail URL to your trusted sites.</span></span> <span data-ttu-id="93b08-128">คลิก **การตั้งค่า** &gt; **ตัวเลือก** &gt; **ความปลอดภัย** &gt; **ไซต์ที่เชื่อถือได้** &gt; **ไซต์** &gt; **เพิ่ม**</span><span class="sxs-lookup"><span data-stu-id="93b08-128">Click **Setting** &gt; **Options** &gt; **Security** &gt; **Trusted sites** &gt; **Sites** &gt; **Add**.</span></span>

<span data-ttu-id="93b08-129">**ปัญหาที่ทราบ:** ตัวออกแบบทำงานไม่ถูกต้องในเบราเซอร์ Google Chrome และ Mozilla Firefox</span><span class="sxs-lookup"><span data-stu-id="93b08-129">**Known issue:** The designer doesn't work correctly in the Google Chrome and Mozilla Firefox browsers.</span></span> <span data-ttu-id="93b08-130">เรากำลังดำเนินการเพื่อแก้ไขปัญหานี้</span><span class="sxs-lookup"><span data-stu-id="93b08-130">We are working to fix this issue.</span></span>
