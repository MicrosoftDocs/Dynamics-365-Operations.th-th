# <a name="install-the-retail-pos-layout-designer"></a><span data-ttu-id="0df3e-103">ติดตั้งตัวออกแบบโครงร่าง Retail POS</span><span class="sxs-lookup"><span data-stu-id="0df3e-103">Install the Retail POS Layout designer</span></span>

[!include[banner](includes/banner.md)]


<span data-ttu-id="0df3e-104">คุณสามารถใช้ตัวออกแบบคลิกเดียวเพื่อออกแบบโครงร่าง Modern POS ของการขายปลีก (MPOS) และ POS ระบบคลาวด์ต่างๆ ในโหมดแนวนอนหรือโหมดแนวตั้ง สำหรับร้านค้า เครื่องบันทึกเงินสด พนักงาน และผู้จัดการ</span><span class="sxs-lookup"><span data-stu-id="0df3e-104">You can use the one-click designer to design different Retail Modern POS (MPOS) and Cloud POS layouts, in either Landscape mode or Portrait mode, for stores, registers, cashiers, and managers.</span></span>

<span data-ttu-id="0df3e-105">อินเทอร์เฟซการออกแบบกราฟิกสำหรับ MPOS หรือ POS ระบบคลาวด์จะถูกควบคุมโดยโครงร่างลิ้นชักเก็บเงิน</span><span class="sxs-lookup"><span data-stu-id="0df3e-105">The graphical design interface for MPOS or Cloud POS is controlled by the till layout.</span></span> <span data-ttu-id="0df3e-106">โครงร่างจะควบคุมตำแหน่งของวัตถุต่างๆ</span><span class="sxs-lookup"><span data-stu-id="0df3e-106">A layout controls the position of various objects.</span></span> <span data-ttu-id="0df3e-107">ตัวอย่างรวมถึงโครงร่างรวม โครงร่างกริดสินค้า โครงร่างของลูกค้า โครงร่างการชำระเงิน และโครงร่างของปุ่มเมนูต่างๆ</span><span class="sxs-lookup"><span data-stu-id="0df3e-107">Examples include the total layout, the item grid layout, the customer layout, the payment layout, and the layout of various menu buttons.</span></span> <span data-ttu-id="0df3e-108">โครงร่างยังรวมถึงลักษณะโดยรวมของอินเทอร์เฟสการขายที่เสนอให้แก่ผู้ปฏิบัติงาน</span><span class="sxs-lookup"><span data-stu-id="0df3e-108">Layouts also include the overall appearance of the sales interface that is presented to workers.</span></span>

## <a name="install-the-one-click-designer"></a><span data-ttu-id="0df3e-109">ติดตั้งตัวออกแบบคลิกเดียว</span><span class="sxs-lookup"><span data-stu-id="0df3e-109">Install the one-click designer</span></span>
1.  <span data-ttu-id="0df3e-110">ใน Microsoft Dynamics 365 for Retail ใช้เมนูด้านซ้ายบนเพื่อนำทางไปยัง **การขายปลีก** **และการค้า** &gt; **การตั้งค่าช่องทาง** &gt; **การตั้งค่า POS** &gt; **POS** &gt; **โครงร่างหน้าจอ**</span><span class="sxs-lookup"><span data-stu-id="0df3e-110">In Microsoft Dynamics 365 for Retail, use the menu in the upper left to navigate to **Retail** **and commerce** &gt; **Channel setup** &gt; **POS setup** &gt; **POS** &gt; **Screen layouts**.</span></span>
2.  <span data-ttu-id="0df3e-111">เลือกโครงร่างใดๆ ที่มีชนิดแอพลิเคชันของ **Modern POS สำหรับ Windows** หรือ **POS ระบบคลาวด์**แล้วคลิก **ตัวออกแบบโครงร่าง**</span><span class="sxs-lookup"><span data-stu-id="0df3e-111">Select any layout that has an application type of **Modern POS for Windows** or **Cloud POS**, and then click **Layout designer**.</span></span>
3.  <span data-ttu-id="0df3e-112">บนแถบการแจ้งเตือนที่ปรากฏที่ด้านล่างของหน้าต่าง Internet Explorer คลิก **เปิด** เพื่อติดตั้งตัวออกแบบคลิกเดียว</span><span class="sxs-lookup"><span data-stu-id="0df3e-112">On the notification bar that appears at the bottom of the Internet Explorer window, click **Open** to install the one-click designer.</span></span> <span data-ttu-id="0df3e-113">(แถบการแจ้งเตือนอาจปรากฏในตำแหน่งอื่นในเบราเซอร์อื่น)</span><span class="sxs-lookup"><span data-stu-id="0df3e-113">(The notification bar might appear in a different place in other browsers.)</span></span>
4.  <span data-ttu-id="0df3e-114">ในกล่องข้อความ **การรันแอพลิเคชัน - คำเตือนเกี่ยวกับความปลอดภัย** จะปรากฏขึ้น คลิก **รัน**เพื่อติดตั้งโฮสต์ตัวออกแบบขายปลีก</span><span class="sxs-lookup"><span data-stu-id="0df3e-114">In the **Application Run - Security Warning** message box that appears, click **Run** to install the Retail designer host.</span></span> <span data-ttu-id="0df3e-115">ตัวบ่งชี้ความคืบหน้าแสดงความคืบหน้าของการติดตั้ง</span><span class="sxs-lookup"><span data-stu-id="0df3e-115">A progress indicator shows the progress of the installation.</span></span>
5.  <span data-ttu-id="0df3e-116">หลังจากการติดตั้งเสร็จสมบูรณ์ บนหน้า **ลงชื่อเข้าใช้** ให้ป้อนชื่อผู้ใช้ Microsoft Dynamics 365 for Retail และรหัสผ่านของคุณ และจากนั้นคลิก **ลงชื่อเข้าใช้** เพื่อเริ่มตัวออกแบบบ</span><span class="sxs-lookup"><span data-stu-id="0df3e-116">After the installation is completed, on the **Sign in** page, enter your Microsoft Dynamics 365 for Retail user name and password, and then click **Sign in** to start the designer.</span></span>
6.  <span data-ttu-id="0df3e-117">หลังจากที่มีการตรวจสอบข้อมูลประจำตัวของคุณและมีการเริ่มต้นตัวออกแบบ คุณสามารถออกแบบโครงร่างของคุณเองหรือปรับเปลี่ยนโครงร่างที่มีอยู่</span><span class="sxs-lookup"><span data-stu-id="0df3e-117">After your credentials are validated and the designer starts, you can design your own layout or modify the existing layout.</span></span> <span data-ttu-id="0df3e-118">[![โครงร่างในตัวออกแบบคลิกเดียว](./media/screenlayoutdesign_mposdownload-1024x664.png)](./media/screenlayoutdesign_mposdownload.png)</span><span class="sxs-lookup"><span data-stu-id="0df3e-118">[![Layout in the one-click designer](./media/screenlayoutdesign_mposdownload-1024x664.png)](./media/screenlayoutdesign_mposdownload.png)</span></span>

    -   <span data-ttu-id="0df3e-122">ใน Internet Explorer ตรวจสอบให้แน่ใจว่าปิดใช้งานตัวบล็อกป๊อปอัพสำหรับไซต์นี้แล้ว</span><span class="sxs-lookup"><span data-stu-id="0df3e-122">In Internet Explorer, make sure that the pop-up blocker is disabled for this site.</span></span> <span data-ttu-id="0df3e-123">คลิก **การตั้งค่า** &gt; **ตัวเลือก** &gt; **ความเป็นส่วนตัว** &gt; **ค้นหาตัวบล็อกป๊อปอัพและเปลี่ยนการตั้งค่า** ถ้าจำเป็นต้องมีการเปลี่ยนแปลง</span><span class="sxs-lookup"><span data-stu-id="0df3e-123">Click **Settings** &gt; **Options** &gt; **Privacy** &gt; **Find Pop-up Blocker**, and change the setting, if a change is required.</span></span>
