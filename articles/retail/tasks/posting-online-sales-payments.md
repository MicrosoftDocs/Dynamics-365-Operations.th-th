# <a name="posting-of-online-sales-and-payments"></a>การลงรายการบัญชีการขายออนไลน์และการชำระเงิน

[!include[task guide banner](../includes/task-guide-banner.md)]

กระบวนการนี้นำไปสู่การตั้งค่าคอนฟิกและการรันงานในชุดงานที่เกิดซ้ำ เพื่อการคำนวณใบสั่งขายและการชำระเงินสำหรับธุรกรรมของร้านค้าออนไลน์  กระบวนงานนี้ใช้บริษัท USRT ในข้อมูลสาธิต

1. ไปยังพื้นที่ทำงานทั้งหมด > การเงินของร้านค้าปลีก 
2. คลิกซิงโครไนส์ใบสั่ง
3. ในฟิลด์ลำดับชั้นขององค์กร ให้เลือก 'ร้านค้าปลีกโดยเรียงตามภูมิภาค'
    * เลือกอย่างใดอย่างหนึ่งระหว่างร้านค้าออนไลน์ หรือเลือกโหนด ถ้าคุณต้องการสร้างชุดงานสำหรับกลุ่มร้านค้า  
    * คลิกลูกศรเพื่อเพิ่มสิ่งที่คุณเลือก   
4. คลิกรันในแท็บเบื้องหลัง 
5. เลือกหรือไม่เลือกกล่องกาเครื่องหมายการประมวลผลชุดงาน
6. คลิกการเกิดซ้ำ
7. เลือกตัวเลือกไม่มีวันที่สิ้นสุด 
8. ในฟิลด์ตรวจนับ ให้ป้อนตัวเลข
9. คลิก ตกลง
10. คลิก ตกลง
