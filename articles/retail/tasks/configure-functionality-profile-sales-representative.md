# <a name="configure-functionality-profiles-for-sales-representatives"></a>ตั้งค่าคอนฟิกโพรไฟล์ฟังก์ชันสำหรับพนักงานขาย

[!include [task guide banner](../includes/task-guide-banner.md)]

กระบวนงานนี้อธิบายวิธีการตั้งค่าคอนฟิกการตั้งค่าโพรไฟล์ฟังก์ชันของร้านค้าที่ใช้กับพนักงานขาย  กระบวนงานนี้ใช้บริษัทข้อมูลสาธิต USRT

1. ไปที่การขายปลีกและการค้า > การตั้งค่าช่องทาง > การตั้งค่า POS > โพรไฟล์POS > โพรไฟล์ฟังก์ชัน
2. คลิก แก้ไข
3. ขยายส่วนฟังก์ชัน
    * คุณสามารถใช้การตั้งค่าโพรไฟล์ฟังก์ชันเพื่อตั้งค่าคอนฟิก POS ให้เพิ่มกลุ่มการขายเริ่มต้นของพนักงานเก็บเงินโดยอัตโนมัติ เพื่อแสดงกล่องโต้ตอบสำหรับกลุ่มการขาย และเพื่อกำหนดว่าจำเป็นต้องมีกลุ่มการขาย  
4. ในฟิลด์ พร้อมต์สำหรับพนักงานขาย เลือกตัวเลือก
5. เลือก ใช่ ในฟิลด์ ต้องมีพนักงานขาย
