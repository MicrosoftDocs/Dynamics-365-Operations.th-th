# <a name="download-hardware-stations-by-using-self-service"></a>ดาวน์โหลดสถานีฮาร์ดแวร์โดยใช้การบริการตนเอง

[!include [task guide banner](../includes/task-guide-banner.md)]

กระบวนการนี้นำไปสู่วิธีการดาวน์โหลดสถานีฮาร์ดแวร์ที่กำหนดไว้ล่วงหน้า  ข้อมูลสาธิตที่บริษัทใช้สำหรับกระบวนการนี้คือ USRT  คุณต้องสร้างสถานีโพรไฟล์ฮาร์ดแวร์และสถานีฮาร์ดแวร์ใหม่ก่อนการเริ่มทำงานนี้

1. ไปที่ การขายปลีกและการค้า > ช่องทาง > ร้านค้าปลีกทั้งหมด
2. ในรายการ ให้เลือกร้านค้าHouston
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
4. สลับการขยายของส่วนสถานีฮาร์ดแวร์ 
5. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
6. คลิกดาวน์โหลด
