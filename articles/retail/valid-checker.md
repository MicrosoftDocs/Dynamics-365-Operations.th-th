<span data-ttu-id="278e6-117">กระบวนการชุดงาน **ตรวจสอบความถูกต้องของธุรกรรมของร้านค้า** จะตรวจสอบความสอดคล้องกันของตารางธุรกรรมการขายปลีกในสถานการณ์ต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="278e6-117">The **Validate store transactions** batch process checks the consistency of the retail transaction tables for the following scenarios.</span></span>
