# <a name="set-up-retail-products"></a>ตั้งค่าผลิตภัณฑ์ขายปลีก

[!include[banner](includes/banner.md)]


บทความนี้อธิบายวิธีตั้งค่าผลิตภัณฑ์ขายปลีกใน Microsoft Dynamics 365 for Retail

ก่อนที่คุณสามารถเสนอผลิตภัณฑ์เพื่อการขายต่อในช่องทางการขายปลีก คุณต้องสร้างและตั้งค่าคอนฟิกผลิตภัณฑ์ใน Microsoft Dynamics 365 for Retail ขายปลีกใช้ลักษณะผลิตภัณฑ์ใน Dynamics 365 for Retail เพื่อสร้างผลิตภัณฑ์ทั่วทั้งองค์กรในผลิตภัณฑ์หลัก คุณสามารถสร้างผลิตภัณฑ์ กำหนดคุณสมบัติของผลิตภัณฑ์และแอททริบิวต์ และกำหนดผลิตภัณฑ์ให้กับลำดับชั้นประเภทการขายปลีก การทำให้ผลิตภัณฑ์พร้อมใช้งานสำหรับช่องทางการขายปลีก และเพิ่มลงในการจัดประเภทที่ใช้งานอยู่ คุณต้องนำผลิตภัณฑ์ออกใช้กับนิติบุคคลที่พร้อมใช้งาน การตั้งค่าผลิตภัณฑ์ที่คุณขายโดยใช้ช่องทางการขายปลีก ให้ดำเนินการงานต่อไปนี้ให้เสร็จสมบูรณ์

1.  กำหนดการลำดับชั้นของผลิตภัณฑ์ขายปลีก โดยการใช้ลักษณะการทำงานตามลำดับชั้นประเภทใน Dynamics 365 for Retail คุณสามารถกำหนดลำดับชั้นประเภทการขายปลีกเพื่อการจัดกลุ่ม และการจัดประเภทผลิตภัณฑ์ที่คุณกระจายสินค้าไปยังช่องทางการขายปลีกของคุณ กำหนดโดยผู้ใช้และการแอตทริบิวต์ระบบสามารถกำหนดในระดับประเภท จากนั้น ผลิตภัณฑ์ทั้งหมดที่ถูกกำหนดให้สืบทอดประเภทแอททริบิวต์เหล่านั้น การจัดประเภทตามลำดับชั้นหลายรายการสามารถกำหนดขึ้นได้ และแต่ละผลิตภัณฑ์จะสามารถกำหนดให้กับหลายลำดับชั้น อย่างไรก็ตาม ในลำดับชั้นเดียว แต่ละผลิตภัณฑ์สามารถกำหนดให้กับประเภทเดียวเท่านั้น
2.  เพิ่มผลิตภัณฑ์และผลิตภัณฑ์ย่อยลงในผลิตภัณฑ์หลัก ผลิตภัณฑ์ที่ถูกเพิ่มในผลิตภัณฑ์หลักแสดงรายการสากลของผลิตภัณฑ์ คุณสามารถเพิ่มผลิตภัณฑ์ได้ด้วยตนเอง ครั้งละหนึ่งรายการ หรือคุณสามารถนำเข้าข้อมูลผลิตภัณฑ์จากผู้จัดจำหน่ายของคุณ
3.  นำผลิตภัณฑ์ออกใช้ที่นิติบุคคล เฉพาะผลิตภัณฑ์ที่ถูกนำออกใช้กับนิติบุคคลสามารถทำให้พร้อมใช้งานสำหรับช่องทางการขายปลีกของคุณ เมื่อคุณกำหนดผลิตภัณฑ์ครั้งแรก คุณกำหนดได้ในระดับทั่วทั้งองค์กร จากนั้นคุณสามารถเลือกหนึ่งนิติบุคคลหรือมากกว่านั้นให้นำผลิตภัณฑ์ออกใช้ จากนั้นผลิตภัณฑ์จะพร้อมใช้งานสำหรับช่องทางการขายปลีกหลายรายการไปทั่วทั้งองค์กร คุณสามารถใช้ฟังก์ชันนี้เพื่อสร้างผลิตภัณฑ์หนึ่งครั้ง เพิ่มและอัพเดตคุณลักษณะของผลิตภัณฑ์และคุณสมบัติในที่หนึ่ง และจากนั้นแจกจ่ายผลิตภัณฑ์ทั่วทั้งองค์กรไปยังช่องทางการขายปลีกที่มีอยู่
4.  เพิ่มผลิตภัณฑ์ลงในการจัดประเภท การจัดประเภทแสดงถึงชุดของผลิตภัณฑ์ที่คุณเสนอในช่องทางการขายปลีก คุณสามารถกำหนดการจัดประเภทอย่างน้อยหนึ่งรายการ และแต่ละผลิตภัณฑ์สามารถถูกกำหนดให้กับการจัดประเภทหนึ่งรายการขึ้นไป การกำหนดผลิตภัณฑ์ให้กับช่องทางการขายปลีก ให้กำหนดการจัดประเภทให้ช่องทางการขายปลีกเหล่านั้น เมื่อคุณสร้างการจัดประเภท คุณสามารถเพิ่มผลิตภัณฑ์ที่ยังไม่ได้ถูกนำออกใช้กับนิติบุคคล อย่างไรก็ตาม คุณต้องนำชุดผลิตภัณฑ์ออกใช้กับนิติบุคคล ก่อนที่ผลิตภัณฑ์เหล่านั้นจะสามารถทำให้ผลิตภัณฑ์พร้อมใช้งานสำหรับช่องทางการขายปลีกของคุณได้
5.  เพิ่มผลิตภัณฑ์ไปยังลำดับชั้นการนำทาง ก่อนที่ผลิตภัณฑ์จะสามารถถูกเรียกดูแบบออนไลน์หรือแบบขายหน้าร้าน (POS) พวกเขาต้องจัดประเภทในลำดับชั้นการนำทางการขายปลีก
6.  เพิ่มผลิตภัณฑ์ลงในแค็ตตาล็อก แม้ว่าขั้นตอนนี้ไม่จำเป็นสำหรับ POS แต่ร้านค้าออนไลน์จำเป็นต้องกำหนดว่าผลิตภัณฑ์ถูกรวมอยู่ในอย่างน้อยหนึ่งแค็ตตาล็อก
