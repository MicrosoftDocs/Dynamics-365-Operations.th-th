<span data-ttu-id="16ca1-127">เครื่องบันทึกเงินสดแต่ละเครื่องถูกเปิดเมื่อเริ่มต้นวัน เมื่อพนักงานเก็บเงินเปิดกะใหม่ และแสดงยอดเงินเริ่มต้น</span><span class="sxs-lookup"><span พนักงานเก็บเงินรายนั้นสามารถดำเนินกะของเขาหรือเธอต่อได้ เมื่อหนึ่งในเครื่องบันทึกเงินสดอื่นพร้อมใช้งาน</span><span class="sxs-lookup"><span data-stu-id="16ca1-131">After the first cashier's break has ### <a name="single-user"></a><span data-ttu-id="16ca1-133">ผู้ใช้เดียว</span><span class="sxs-lookup"><span data-stu-id="16ca1-133">Single user</span></span>

<span data-ttu-id="16ca1-134">ผู้ค้าปลีกหลายรายมักจะอนุญาตผู้ใช้รายเดียวเท่านั้นต่อกะ เพื่อช่วยในการรับประกันระดับสูงสุดของการลงบัญชีสำหรับเงินสดในลิ้นชักเงินสด</span><span class="sxs-lookup"><span data-stu-id="16ca1-134">Many retailers prefer to allow only one user per shift, to help guarantee the highest level of accountability for the cash in the cash drawer.</span></span> <span data-ttu-id="16ca1-135">ถ้าผู้ใช้ได้รับอนุญาตเพียงหนึ่งรายให้ใช้ลิ้นชักเก็บเงินที่จะเชื่อมโยงกับกะ ผู้ใช้รายนั้นสามารถรับผิดชอบเฉพาะสำหรับความขัดแย้งใดๆ</span><span class="sxs-lookup"><span data-stu-id="16ca1-135">If only one user is allowed to use the till that is associated with a shift, that user can be held solely responsible for any discrepancies.</span></span> <span data-ttu-id="16ca1-136">ถ้าผู้ใช้มากกว่าหนึ่งรายใช้กะ จะยากในการกำหนดผู้ที่ทำข้อผิดพลาด หรือผู้ที่อาจพยายามขโมยจากลิ้นชักเก็บเงิน</span><span class="sxs-lookup"><span data-stu-id="16ca1-136">If more than one user uses a shift, it's difficult to determine who made an error, or who might be trying to steal from the till.</span></span>
