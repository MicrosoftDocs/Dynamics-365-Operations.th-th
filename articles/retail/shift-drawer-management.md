<span data-ttu-id="564c7-105">บทความนี้อธิบายวิธีการตั้งค่าและการใช้กะการขายหน้าร้านแบบขายปลีก (POS) สองชนิด ซึ่งได้แก่ แบบสแตนด์อโลน และที่ใช้ร่วมกัน</span><span class="sxs-lookup"><span data-stu-id="564c7-105">This article explains how to set up and use the two types of retail point of sale (POS) shifts -  shared and stand-alone.</span></span> <span data-ttu-id="564c7-106">กะที่ใช้ร่วมกันสามารถใช้ได้โดยผู้ใช้หลายคนในสถานที่หลายแห่ง ในขณะที่กะแบบสแตนด์อโลนสามารถใช้ได้โดยผู้ปฏิบัติงานเดียวเท่านั้นในแต่ละครั้ง</span><span class="sxs-lookup"><span data-stu-id="564c7-106">Shared shifts can be used by multiple users in multiple places, whereas stand-alone shifts can be used by only one worker at a time.</span></span>

<span data-ttu-id="564c7-212">**ปิดซ่อนกะ**</span><span class="sxs-lookup"><span data-stu-id="564c7-212">**Blind close shift**</span></span>

<span data-ttu-id="564c7-216">**ปิดกะ**</span><span class="sxs-lookup"><span data-stu-id="564c7-216">**Close shift**</span></span>

<span data-ttu-id="564c7-217">การดำเนินงานนี้คำนวณผลรวมกะ ยอดเงินเกิน/ขาด และดำเนินการขั้นสุดท้ายกับกะที่ปิดซ่อนอยู่หรือใช้งานอยู่</span><span class="sxs-lookup"><span data-stu-id="564c7-217">This operation calculates shift totals, over/short amounts, and then finalizes an active or blind closed shift.</span></span> <span data-ttu-id="564c7-218">กะที่ปิดไม่สามารถดำเนินการต่อหรือปรับเปลี่ยนได้</span><span class="sxs-lookup"><span data-stu-id="564c7-218">Closed shifts cannot be resumed or modified.</span></span>
