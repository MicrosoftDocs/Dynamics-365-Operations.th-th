<span data-ttu-id="c046f-106">Dynamics 365 Commerce ชุดเริ่มต้น เป็นชุดของโมดูลที่สามารถใช้ในการสร้างเว็บไซต์อีคอมเมิร์ซ</span><span class="sxs-lookup"><span data-stu-id="c046f-106">The Dynamics 365 Commerce starter kit is a collection of modules that can be used to build an e-Commerce website.</span></span> <span data-ttu-id="c046f-107">โมดูลมีทั้งด้านอินเทอร์เฟสผู้ใช้ (UI) และลักษณะการทำงาน</span><span class="sxs-lookup"><span data-stu-id="c046f-107">Modules have both user interface (UI) aspects and functional behavior aspects.</span></span>

<span data-ttu-id="c046f-108">ชุดรูปแบบสามารถนำไปใช้กับโมดูลในชุดเริ่มต้น เพื่อเปลี่ยนลักษณะที่แสดง</span><span class="sxs-lookup"><span data-stu-id="c046f-108">Themes can be applied to the modules in the starter kit to change their look and feel.</span></span> <span data-ttu-id="c046f-109">ชุดรูปแบบใช้ Cascading Style Sheets (CSS)</span><span class="sxs-lookup"><span data-stu-id="c046f-109">The themes use Cascading Style Sheets (CSS).</span></span> <span data-ttu-id="c046f-110">ชุดรูปแบบสำหรับไซต์ e-Commerce ที่มีชื่อว่า "Fabrikam" ให้ไว้เป็นส่วนหนึ่งของชุดเริ่มต้น และสามารถใช้เป็นข้อมูลอ้างอิงได้</span><span class="sxs-lookup"><span data-stu-id="c046f-110">A theme for a fictitious e-Commerce site that is named "Fabrikam" is provided as part of the starter kit and can be used as a reference.</span></span>

- <span data-ttu-id="c046f-115">**โมดูลการตลาด** – โมดูลการตลาด ได้แก่ บล็อคเนื้อหา บล็อคข้อความ เครื่องเล่นวิดีโอ และโมดูลแบบวงล้อ</span><span class="sxs-lookup"><span data-stu-id="c046f-115">**Marketing modules** – Marketing modules include content block, text block, video player, and carousel modules.</span></span> <span data-ttu-id="c046f-116">โมดูลเหล่านี้ทั้งหมดสามารถใช้ในการแสดงเนื้อหาได้</span><span class="sxs-lookup"><span data-stu-id="c046f-116">All these modules can be used to showcase content.</span></span> <span data-ttu-id="c046f-117">ซึ่งสามารถนำไปใช้ที่หน้าไหนก็ได้ และจะถูกขับเคลื่อนด้วยข้อมูลจากระบบการจัดการเนื้อหา (CMS)</span><span class="sxs-lookup"><span data-stu-id="c046f-117">They can be put on any page and are driven by data from the content management system (CMS).</span></span>
- <span data-ttu-id="c046f-118">**โมดูลส่วนหัวและส่วนท้าย** – โมดูลส่วนหัวและส่วนท้ายจะปรากฏในส่วนหัวและส่วนท้ายของหน้าไซต์ทั้งหมด</span><span class="sxs-lookup"><span data-stu-id="c046f-118">**Header and footer modules** – Header and footer modules appear in the header and footer of all site pages.</span></span> <span data-ttu-id="c046f-119">คุณสามารถตั้งค่าคอนฟิกโมดูลเหล่านี้ตามต้องการได้โดยใช้คุณสมบัติ</span><span class="sxs-lookup"><span data-stu-id="c046f-119">These modules can be configured as required through properties.</span></span>
- <span data-ttu-id="c046f-120">**โมดูลค้นหา** – ผลิตภัณฑ์จะถูกพบได้โดยใช้โมดูลการค้นหาในส่วนหัว</span><span class="sxs-lookup"><span data-stu-id="c046f-120">**Search modules** – Products can be discovered by using the search module in the header.</span></span> <span data-ttu-id="c046f-121">ผลการค้นหาจะปรากฏขึ้นบนหน้าผลการค้นหา</span><span class="sxs-lookup"><span data-stu-id="c046f-121">Search results appear on the search results page.</span></span> <span data-ttu-id="c046f-122">นอกจากนี้คุณยังสามารถค้นหาผลิตภัณฑ์ในหน้าประเภท ซึ่งเป็นหน้าเฉพาะสำหรับแต่ละประเภทที่ได้รับการสนับสนุน ในลำดับชั้นการนำทางของช่องทาง</span><span class="sxs-lookup"><span data-stu-id="c046f-122">Products can also be discovered on category pages, which are dedicated pages for each category that is supported in the channel navigation hierarchy.</span></span> <span data-ttu-id="c046f-123">นอกจากนี้ คุณยังสามารถใช้โมดูลตัวคัดสรรในการกรองผลการค้นหาและหน้าประเภทได้อีกด้วย</span><span class="sxs-lookup"><span data-stu-id="c046f-123">In addition, refiner modules can be used to further filter results on search results and category pages.</span></span>

- <span data-ttu-id="c046f-133">**โมดูลการจัดการบัญชี** – โมดูลเข้าสู่ระบบของผู้ใช้ ให้ลูกค้าล็อกอินเข้าสู่บัญชีที่มีอยู่ และโมดูลการลงทะเบียนจะช่วยให้ผู้ใช้สามารถสร้างบัญชีใหม่ได้</span><span class="sxs-lookup"><span data-stu-id="c046f-133">**Account management modules** – The sign-in module lets customers sign in to an existing account, and the sign-up module lets them create a new account.</span></span> <span data-ttu-id="c046f-134">หลังจากที่สร้างบัญชีแล้ว คุณสามารถใช้โมดูลบัญชีของใบสั่งซื้อเพื่อดูใบสั่งซื้อล่าสุด และโมดูลรายละเอียดใบสั่งซื้อสามารถใช้เพื่อดูรายละเอียดของใบสั่งซื้อได้</span><span class="sxs-lookup"><span data-stu-id="c046f-134">After an account is created, the order history module can be used to view recent orders, and the order details module can be used to view order details.</span></span>
