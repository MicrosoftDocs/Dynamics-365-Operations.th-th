## <a name="overview"></a><span data-ttu-id="5af31-105">ภาพรวม</span><span class="sxs-lookup"><span data-stu-id="5af31-105">Overview</span></span>

<span data-ttu-id="5af31-106">ไลบรารีโมดูล Dynamics 365 Commerce เป็นชุดของโมดูลที่สามารถใช้ในการสร้างเว็บไซต์อีคอมเมิร์ซ</span><span class="sxs-lookup"><span data-stu-id="5af31-106">The Dynamics 365 Commerce module library is a collection of modules that can be used to build an e-Commerce website.</span></span> <span data-ttu-id="5af31-107">โมดูลมีทั้งด้านอินเทอร์เฟสผู้ใช้ (UI) และลักษณะการทำงาน</span><span class="sxs-lookup"><span data-stu-id="5af31-107">Modules have both user interface (UI) aspects and functional behavior aspects.</span></span>

<span data-ttu-id="5af31-108">ชุดรูปแบบสามารถนำไปใช้กับโมดูลในไลบรารีโมดูล เพื่อเปลี่ยนลักษณะที่แสดง</span><span class="sxs-lookup"><span data-stu-id="5af31-108">Themes can be applied to the modules in the module library to change their look and feel.</span></span> <span data-ttu-id="5af31-109">ชุดรูปแบบใช้ Cascading Style Sheets (CSS)</span><span class="sxs-lookup"><span data-stu-id="5af31-109">The themes use Cascading Style Sheets (CSS).</span></span> <span data-ttu-id="5af31-110">ธีมสำหรับไซต์ e-Commerce แบบสมมติที่มีชื่อว่า "Fabrikam" ให้ไว้เป็นส่วนหนึ่งของไลบรารีโมดูล และสามารถใช้เป็นข้อมูลอ้างอิงได้</span><span class="sxs-lookup"><span data-stu-id="5af31-110">A theme for a fictitious e-Commerce site that is named "Fabrikam" is provided as part of the module library and can be used as a reference.</span></span>

## <a name="module-library-modules"></a><span data-ttu-id="5af31-111">โมดูลไลบรารีของโมดูล</span><span class="sxs-lookup"><span data-stu-id="5af31-111">Module library modules</span></span>

<span data-ttu-id="5af31-112">ชนิดของโมดูลต่อไปนี้มีให้ในไลบรารีโมดูล:</span><span class="sxs-lookup"><span data-stu-id="5af31-112">The following types of modules are provided in the module library:</span></span>

- <span data-ttu-id="5af31-113">**โมดูลคอนเทนเนอร์** – โมดูลคอนเทนเนอร์เป็นโมดูลที่เรียบง่ายที่ทำหน้าที่เป็นโฮสต์สำหรับโมดูลอื่นๆ</span><span class="sxs-lookup"><span data-stu-id="5af31-113">**Container module** – A container module is a simple module that acts as a host for other modules.</span></span> <span data-ttu-id="5af31-114">ซึ่งควบคุมโครงร่างของโมดูลที่อยู่ภายใน</span><span class="sxs-lookup"><span data-stu-id="5af31-114">It controls the layout of the modules that are inside it.</span></span>
- <span data-ttu-id="5af31-115">**โมดูลการตลาด** – โมดูลการตลาด ได้แก่ บล็อคเนื้อหา บล็อคข้อความ เครื่องเล่นวิดีโอ และโมดูลแบบวงล้อ</span><span class="sxs-lookup"><span data-stu-id="5af31-115">**Marketing modules** – Marketing modules include content block, text block, video player, and carousel modules.</span></span> <span data-ttu-id="5af31-116">โมดูลเหล่านี้ทั้งหมดสามารถใช้ในการแสดงเนื้อหาได้</span><span class="sxs-lookup"><span data-stu-id="5af31-116">All these modules can be used to showcase content.</span></span> <span data-ttu-id="5af31-117">ซึ่งสามารถนำไปใช้ที่หน้าไหนก็ได้ และจะถูกขับเคลื่อนด้วยข้อมูลจากระบบการจัดการเนื้อหา (CMS)</span><span class="sxs-lookup"><span data-stu-id="5af31-117">They can be put on any page and are driven by data from the content management system (CMS).</span></span>
- <span data-ttu-id="5af31-118">**โมดูลส่วนหัวและส่วนท้าย** – โมดูลส่วนหัวและส่วนท้ายจะปรากฏในส่วนหัวและส่วนท้ายของหน้าไซต์ทั้งหมด</span><span class="sxs-lookup"><span data-stu-id="5af31-118">**Header and footer modules** – Header and footer modules appear in the header and footer of all site pages.</span></span> <span data-ttu-id="5af31-119">คุณสามารถตั้งค่าคอนฟิกโมดูลเหล่านี้ตามต้องการได้โดยใช้คุณสมบัติ</span><span class="sxs-lookup"><span data-stu-id="5af31-119">These modules can be configured as required through properties.</span></span>
- <span data-ttu-id="5af31-120">**โมดูลค้นหา** – ผลิตภัณฑ์จะถูกพบได้โดยใช้โมดูลการค้นหาในส่วนหัว</span><span class="sxs-lookup"><span data-stu-id="5af31-120">**Search modules** – Products can be discovered by using the search module in the header.</span></span> <span data-ttu-id="5af31-121">ผลการค้นหาจะปรากฏขึ้นบนหน้าผลการค้นหา</span><span class="sxs-lookup"><span data-stu-id="5af31-121">Search results appear on the search results page.</span></span> <span data-ttu-id="5af31-122">นอกจากนี้คุณยังสามารถค้นหาผลิตภัณฑ์ในหน้าประเภท ซึ่งเป็นหน้าเฉพาะสำหรับแต่ละประเภทที่ได้รับการสนับสนุน ในลำดับชั้นการนำทางของช่องทาง</span><span class="sxs-lookup"><span data-stu-id="5af31-122">Products can also be discovered on category pages, which are dedicated pages for each category that is supported in the channel navigation hierarchy.</span></span> <span data-ttu-id="5af31-123">นอกจากนี้ คุณยังสามารถใช้โมดูลตัวคัดสรรในการกรองผลการค้นหาและหน้าประเภทได้อีกด้วย</span><span class="sxs-lookup"><span data-stu-id="5af31-123">In addition, refiner modules can be used to further filter results on search results and category pages.</span></span>
- <span data-ttu-id="5af31-124">**โมดูลหน้ารายละเอียดผลิตภัณฑ์** – หน้ารายละเอียดของผลิตภัณฑ์ใช้หลายโมดูลเพื่อแสดงข้อมูลผลิตภัณฑ์</span><span class="sxs-lookup"><span data-stu-id="5af31-124">**Product details page modules** – Product details pages use several modules to show product information.</span></span> <span data-ttu-id="5af31-125">โมดูลในกล่องการซื้อ ให้ลูกค้าสามารถดูผลิตภัณฑ์และเพิ่มสินค้าลงในรถเข็น</span><span class="sxs-lookup"><span data-stu-id="5af31-125">The buy box module lets customers view products and add them to the cart.</span></span> <span data-ttu-id="5af31-126">โมดูลอื่นๆ เช่น โมดูลข้อมูลจำเพาะด้านเทคนิค แสดงรายละเอียดของผลิตภัณฑ์</span><span class="sxs-lookup"><span data-stu-id="5af31-126">Other modules, such as the tech specs module, show the product details.</span></span> <span data-ttu-id="5af31-127">โมดูลการจัดอันดับ และความคิดเห็นสามารถใช้เพื่อดูและให้ความคิดเห็น</span><span class="sxs-lookup"><span data-stu-id="5af31-127">The ratings and reviews module can used to view and provide reviews.</span></span>
- <span data-ttu-id="5af31-128">**โมดูลซื้อออนไลน์ รับสินค้าที่ร้านค้า** – โมดูลซื้อออนไลน์ รับสินค้าที่ร้านค้าทำงานร่วมกับ Bing Maps</span><span class="sxs-lookup"><span data-stu-id="5af31-128">**Buy online pick up in store module** – The buy online pick up in store module is integrated with Bing Maps.</span></span> <span data-ttu-id="5af31-129">คุณสามารถใช้ในการค้นหาร้านค้าใกล้เคียง ที่ลูกค้าสามารถรับสินค้าที่ซื้อได้</span><span class="sxs-lookup"><span data-stu-id="5af31-129">It can be used to find nearby stores where customers can pick up products that they have purchased.</span></span>
- <span data-ttu-id="5af31-130">**โมดูลการซื้อ** – โมดูลการซื้อรวมถึงโมดูลรถเข็น ซึ่งสามารถใช้เพื่อเพิ่มสินค้าลงในรถเข็น</span><span class="sxs-lookup"><span data-stu-id="5af31-130">**Purchase modules** – Purchase modules include the cart module, which can be used to add items to the cart.</span></span> <span data-ttu-id="5af31-131">โมดูลการชำระเงินจะเก็บที่อยู่ในการจัดส่ง ตัวเลือกการจัดส่ง บัตรของขวัญ โปรแกรมสะสมคะแนน และข้อมูลบัตรเครดิตเพื่อให้สามารถประมวลผลใบสั่งซื้อได้</span><span class="sxs-lookup"><span data-stu-id="5af31-131">The checkout module captures the shipping address, delivery options, and gift card, loyalty program, and credit card information, so that an order can be processed.</span></span> <span data-ttu-id="5af31-132">หลังจากวางใบสั่งซื้อแล้ว คุณสามารถใช้โมดูลการยืนยันใบสั่งเพื่อแสดงรายละเอียดการยืนยันได้</span><span class="sxs-lookup"><span data-stu-id="5af31-132">After an order is placed, the order confirmation module can be used to show the confirmation details.</span></span>
- <span data-ttu-id="5af31-133">**โมดูลการจัดการบัญชี** – โมดูลเข้าสู่ระบบของผู้ใช้ ให้ลูกค้าล็อกอินเข้าสู่บัญชีที่มีอยู่ และโมดูลการลงทะเบียนจะช่วยให้ผู้ใช้สามารถสร้างบัญชีใหม่ได้</span><span class="sxs-lookup"><span data-stu-id="5af31-133">**Account management modules** – The sign-in module lets customers sign in to an existing account, and the sign-up module lets them create a new account.</span></span> <span data-ttu-id="5af31-134">หลังจากที่สร้างบัญชีแล้ว คุณสามารถใช้โมดูลบัญชีของใบสั่งซื้อเพื่อดูใบสั่งซื้อล่าสุด และโมดูลรายละเอียดใบสั่งซื้อสามารถใช้เพื่อดูรายละเอียดของใบสั่งซื้อได้</span><span class="sxs-lookup"><span data-stu-id="5af31-134">After an account is created, the order history module can be used to view recent orders, and the order details module can be used to view order details.</span></span>
- <span data-ttu-id="5af31-135">**โมดูลคำแนะนำ** – ข้อแนะนำแสดงโดยใช้โมดูลการวางผลิตภัณฑ์</span><span class="sxs-lookup"><span data-stu-id="5af31-135">**Recommendations module** – Recommendations are shown by using the product placement module.</span></span> <span data-ttu-id="5af31-136">โมดูลนี้สนับสนุนรายการอัลกอริทึม และรายการที่เลือกโดยบรรณาธิการ ที่สามารถแสดงในหน้าใดก็ได้</span><span class="sxs-lookup"><span data-stu-id="5af31-136">This module supports algorithmic and editorial lists that can be showcased on any page.</span></span>
