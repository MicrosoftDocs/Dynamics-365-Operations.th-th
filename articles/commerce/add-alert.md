# <a name="promo-banner-module"></a>โมดูลแบนเนอร์โปรโมชั่น


[!include [banner](includes/banner.md)]

หัวข้อนี้ครอบคลุมถึงโมดูลแบนเนอร์โปรโมชั่น และอธิบายวิธีการเพิ่มลงในเพจของไซต์ใน Microsoft Dynamics 365 Commerce

## <a name="overview"></a>ภาพรวม

โมดูลแบนเนอร์โปรโมชั่นใช้เพื่อแสดงข้อความที่ให้ข้อมูลแบบอินไลน์บนหน้า ผู้ใช้สามารถใช้เพื่อแสดงโปรโมชันทั้งไซต์ที่ปรากฏในหน้าทั้งหมดของไซต์อีคอมเมิร์ซ 

โมดูลแบนเนอร์โปรโมชั่นสนับสนุนข้อความตัวอักษรและลิงค์ ถ้ามีการเพิ่มข้อความหลายรายการในโมดูลแบนเนอร์โปรโมชั่น โมดูลจะกลายเป็นแบนเนอร์แบบวงล้อที่ให้ลูกค้าวนผ่านข้อความทั้งหมด 

โมดูลแบนเนอร์โปรโมชั่นจะถูกขับเคลื่อนด้วยข้อมูลจากระบบการจัดการเนื้อหา (CMS) และสามารถวางบนหน้าใดก็ได้

## <a name="usage-examples-of-promo-banners-in-e-commerce"></a>ตัวอย่างการใช้งานของแบนเนอร์โปรโมชั่นในอีคอมเมิร์ซ

แบนเนอร์โปรโมชั่นสามารถใช้ในส่วนหัวของไซต์ เพื่อแสดงข้อความหรือข้อความทั่วทั้งไซต์ เช่นเดียวกับในตัวอย่างต่อไปนี้

"ยอดขายประจำปีสิ้นสุดในเวลา10วัน"

"ประหยัดอย่างมากกับโปรโมชัน Back to School ซื้อตอนนี้ "

## <a name="promo-banner-module-properties"></a>คุณสมบัติโมดูลแบนเนอร์โปรโมชั่น

| ชื่อคุณสมบัติ             | Value                              | คำอธิบาย |
|---------------------------|------------------------------------|-------------|
| ข้อความแบนเนอร์           | ข้อความและลิงก์                     | แถวของข้อความและลิงก์ |
| เล่นอัตโนมัติ                  | **จริง** หรือ **เท็จ**              | ค่าที่ระบุว่าข้อความจะถูกวนโดยอัตโนมัติหรือไม่ ถ้ามีการตั้งค่าคอนฟิกข้อความจำนวนมาก |
| ช่วงเวลาของการเปลี่ยนภาพนิ่ง | จำนวนมิลลิวินาที (ms)      | ช่วงเวลาที่ใช้ในการวนผ่านข้อความ |
| อนุญาตให้ปิด             | **จริง** หรือ **เท็จ**              | หากค่าถูกตั้งเป็น **จริง** ลูกค้าสามารถยกเลิกการแจ้งเตือนได้ |
| แสดงตัวเลื่อนวงล้อ     | **จริง** หรือ **เท็จ**              | ค่าที่ระบุว่าควรจะแสดงตัวหมุนวงล้อหรือไม่ เพื่อให้ลูกค้าสามารถหมุนรายการแบนเนอร์หลายรายการด้วยตนเองได้ |
| การปรับแนวข้อความ            | **ขวา** **ซ้าย** หรือ **กึ่งกลาง** | การจัดแนวข้อความในโมดูลแบนเนอร์โปรโมชัน |
| ลิงค์                      | URL                              | URL สำหรับลิงก์ทางเลือก |

## <a name="add-a-promo-banner-module-to-a-page"></a>เพิ่มโมดูลแบนเนอร์โปรโมชั่นไปยังเพจ 

การเพิ่มโมดูลแบนเนอร์โปรโมชั่นไปยังหน้า และตั้งค่าคุณสมบัติที่จำเป็น ให้ทำตามขั้นตอนต่อไปนี้

1. สร้างแม่แบบของหน้าซึ่งชื่อ **แม่แบบแบนเนอร์โปรโมชั่น**
1. ภายใต้ **โครงร่างเพจ** ให้เพิ่ม **Default page** ไปยังช่อง **เนื้อความ** 
1. เช็คอินแม่แบบ และเผยแพร่ 
1. ใช้แม่แบบที่คุณเพิ่งสร้างขึ้นเพื่อสร้างหน้าที่ชื่อ **หน้าแบนเนอร์โปรโมชั่น** 
1. ในช่อง **หลัก** ของหน้าใหม่ ให้เพิ่มโมดูลคอนเทนเนอร์ 
1. ในบานหน้าต่างด้านขวา ให้ตั้งค่า **ความกว้าง** เป็น **เต็มคอนเทนเนอร์**
1. ภายใต้ **โครงร่างเพจ** ให้เพิ่มโมดูลแบนเนอร์โปรโมชั่นไปยังโมดูลตู้บรรจุสินค้า
1. ในการตั้งค่าสำหรับโมดูลแบนเนอร์ ให้เพิ่มข้อความแบนเนอร์หนึ่งข้อขึ้นไป แต่ละข้อความสามารถมีข้อความพร้อมกับลิงก์ได้ คุณสามารถแก้ไขคุณสมบัติอื่น ๆ ได้ เพื่อเลือกกำหนดโมดูลต่อไป
1. บันทึกและแสดงตัวอย่างหน้า ที่ด้านบนของหน้า คุณควรเห็นการแจ้งเตือนที่แสดงข้อความที่คุณเพิ่ม
1. แก้ไขหน้าให้เสร็จสิ้น และเผยแพร่ 

> [!NOTE]
> โดยทั่วไปป้ายโฆษณาแบบโปรโมชั่นจะใช้ในช่องส่วนหัวของหน้า หรือช่องหัวข้อย่อย
