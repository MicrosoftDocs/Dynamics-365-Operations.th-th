# <a name="alert-module"></a>โมดูลการแจ้งเตือน

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

หัวข้อนี้ครอบคลุมถึงโมดูลการแจ้งเตือน และอธิบายวิธีการเพิ่มลงในเพจของไซต์ใน Microsoft Dynamics 365 Commerce

## <a name="overview"></a>ภาพรวม

โมดูลการแจ้งเตือนใช้เพื่อแสดงข้อความที่ให้ข้อมูลแบบอินไลน์บนหน้า โมดูลการแจ้งเตือนสนับสนุนข้อความตัวอักษรและลิงค์ ผู้ใช้สามารถใช้เพื่อแสดงโปรโมชันทั้งไซต์ที่ปรากฏในหน้าทั้งหมดของไซต์อีคอมเมิร์ซ 

โมดูลการแจ้งเตือนจะถูกขับเคลื่อนด้วยข้อมูลจากระบบการจัดการเนื้อหา (CMS) และสามารถวางบนหน้าใดก็ได้

## <a name="examples-of-alert-modules-in-e-commerce"></a>ตัวอย่างของโมดูลการแจ้งเตือนในอีคอมเมิร์ซ

โมดูลการแจ้งเตือนสามารถใช้ในส่วนหัวของไซต์ เพื่อบ่งชี้ว่ามีการส่งเสริมการตลาดหรือข้อความทั่วทั้งไซต์ ยกตัวอย่างเช่น

"ยอดขายประจำปีสิ้นสุดในเวลา10วัน"

"ประหยัดอย่างมากกับโปรโมชัน Back to School ซื้อตอนนี้ "

## <a name="alert-module-properties"></a>คุณสมบัติของโมดูลการแจ้งเตือน

| ชื่อคุณสมบัติ  | มูลค่า                              | คำอธิบาย |
|----------------|------------------------------------|-------------|
| ข้อความ           | ข้อความ                               | ข้อความตัวอักษรที่ปรากฏขึ้นในโมดูลการแจ้งเตือน |
| การปรับแนวข้อความ | **ขวา** **ซ้าย** หรือ **กึ่งกลาง** | ค่าที่กำหนดวิธีการจัดตำแหน่งข้อความไว้ในโมดูลการแจ้งเตือน |
| ยกเลิกการแจ้งเตือน  | **จริง** หรือ **เท็จ**              | หากค่าถูกตั้งเป็น **จริง** ลูกค้าสามารถยกเลิกการแจ้งเตือนได้ |
| ลิงค์           | URL                                | URL สำหรับลิงก์ทางเลือก |

## <a name="add-an-alert-module-to-a-page"></a>เพิ่มโมดูลการแจ้งเตือนไปยังหน้า 

การเพิ่มโมดูลการแจ้งเตือนไปยังหน้า และตั้งค่าคุณสมบัติที่จำเป็น ให้ทำตามขั้นตอนต่อไปนี้

1. สร้างเท็มเพลตของหน้าซึ่งชื่อ **เท็มเพลตการแจ้งเตือน**
1. ในช่อง **หลัก** ของหน้าเริ่มต้น ให้เพิ่มโมดูลการแจ้งเตือน
1. เช็คอินแม่แบบ และเผยแพร่ 
1. ใช้เท็มเพลตการแจ้งเตือนที่คุณเพิ่งสร้างขึ้นเพื่อสร้างหน้าที่ชื่อ **หน้าการแจ้งเตือน** 
1. ในช่อง **หลัก** ของหน้าใหม่ ให้เพิ่มโมดูลการแจ้งเตือน
1. ในการตั้งค่าสำหรับโมดูลการแจ้งเตือน ให้ป้อนข้อความการแจ้งเตือน คุณสามารถแก้ไขคุณสมบัติอื่นๆได้ถ้าคุณต้องการเลือกกำหนดโมดูลการแจ้งเตือนต่อไป
1. บันทึกและแสดงตัวอย่างหน้า ที่ด้านบนของหน้า คุณควรเห็นการแจ้งเตือนที่มีข้อความที่คุณเพิ่ม
1. เช็คอินหน้า และเผยแพร่
