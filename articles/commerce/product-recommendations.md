# <a name="product-recommendations-overview"></a>ภาพรวมของคำแนะนำผลิตภัณฑ์

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Commerce สามารถใช้แสดงคำแนะนำเกี่ยวกับผลิตภัณฑ์บนเว็บไซต์อีคอมเมิร์ซ และอุปกรณ์การขายหน้าร้าน (POS) คำแนะนำผลิตภัณฑ์คือสินค้าที่ลูกค้าอาจสนใจ คำแนะนำเหล่านี้จะขึ้นอยู่กับแนวโน้มของการซื้อของลูกค้ารายอื่นๆ ในร้านค้าออนไลน์และร้านจริง

คำแนะนำผลิตภัณฑ์อนุญาตให้ลูกค้าค้นหาผลิตภัณฑ์ที่พวกเขาต้องการได้อย่างง่ายดายและรวดเร็ว ในขณะเดียวกันก็มีประสบการณ์การใช้งานที่ดี นอกจากนี้ยังสามารถใช้การขายสินค้าชนิดอื่นและการขายสินค้าต่อเนื่อง เพื่อช่วยลูกค้าในการค้นหาผลิตภัณฑ์เพิ่มเติมที่พวกเขาไม่ได้ตั้งใจจะซื้อในตอนแรกได้ เมื่อมีการใช้ข้อแนะนำในการปรับปรุงการค้นหาผลิตภัณฑ์ จะมีการสร้างโอกาสทางการขายในการแปลงเพิ่มเติม ช่วยเพิ่มรายได้จากการขาย และยังช่วยเพิ่มความพึงพอใจของและการบำรุงรักษา

ในอีคอมเมิร์ซ คำแนะนำผลิตภัณฑ์มีการขับเคลื่อนโดยเทคโนโลยี Machine Learning ของ Microsoft Recommendations ในขนาดใหญ่

## <a name="recommendation-service"></a>บริการคำแนะนำ

บริการคำแนะนำผลิตภัณฑ์ใช้เทคโนโลยีปัญญาประดิษฐ์และ machine learning (AI-ML) ในลักษณะต่อไปนี้:

- ข้อมูลในรูปแบบที่บริการคำแนะนำต้องการถูกแยกจากฐานข้อมูลการดำเนินงานเชิงพาณิชย์ และส่งไปที่ Azure Data Lake Storage หรือที่เก็บเอนทิตี
- บริการคำแนะนำจะใช้ข้อมูลที่จัดเก็บในการฝึกอบรมแบบจำลองคำแนะนำสำหรับรายการ **คนชอบสิ่งนี้เช่นกัน** **ซื้อร่วมกันบ่อย** **ใหม่** **ขายดี** และ **ยอดนิยม**

## <a name="scenarios"></a>สถานการณ์

คำแนะนำผลิตภัณฑ์ถูกเปิดใช้งานสำหรับสถานการณ์ต่อไปนี้

- **บนหน้าร้านค้าใดๆ สำหรับการเรียกดูหรือหน้าเริ่มต้นในอีคอมเมิร์ซ:** ถ้าลูกค้าหรือร้านค้าเข้าถึงหน้าร้านค้า โปรแกรมแนะนำสามารถแนะนำผลิตภัณฑ์ในรายการ **สินค้าใหม่** **ขายดี** และ **ที่กำลังนิยม**
- **บนหน้ารายละเอียดของผลิตภัณฑ์:** ถ้าลูกค้าหรือร้านค้าเข้าร่วมหน้า **รายละเอียดผลิตภัณฑ์** โปรแกรมแนะนำจะแนะนำสินค้าเพิ่มเติมที่มีแนวโน้มจะซื้อด้วยเช่นกัน รายการเหล่านี้จะปรากฏในรายชื่อ **คนอื่นชอบสิ่งนี้ด้วยเช่นกัน**
- **บนหน้าธุรกรรมหรือหน้าเช็คเอาท์:** โปรแกรมแนะนำจะแนะนำสินค้าตามรายการสินค้าทั้งหมดในรถเข็น สินค้าเหล่านี้จะปรากฏในรายการ **สินค้าที่มีการซื้อร่วมกันบ่อยๆ**
- **คำแนะนำส่วนบุคคล:** ผู้จัดซื้อสามารถให้ลูกค้าที่ลงชื่อเข้าใช้ **รายการที่เลือกไว้สำหรับคุณ** ส่วนบุคคล นอกเหนือจากฟังก์ชันใหม่ที่อนุญาตให้ใช้รายการสถานการณ์จำลองที่มีอยู่ เพื่อสร้างความเป็นส่วนบุคคลสำหรับลูกค้ารายนั้น เมื่อต้องการเรียนรู้เพิ่มเติม ดูที่ [เปิดใช้งานคำแนะนำที่เป็นแบบส่วนบุคคล](personalized-recommendations.md)

### <a name="types-of-product-recommendations"></a>ชนิดของคำแนะนำผลิตภัณฑ์

ตารางต่อไปนี้อธิบายคำแนะนำเกี่ยวกับผลิตภัณฑ์แบบอัตโนมัติชนิดต่างๆ ที่พร้อมใช้งานสำหรับผู้ขายปลีกเพื่อใช้ในโซลูชัน Dynamics 365 Commerce ของพวกเขาผ่าน [โมดูลการเก็บรวบรวมผลิตภัณฑ์](product-collection-module-overview.md) นอกจากนี้ ผู้ขายปลีกยังสามารถแสดงผลลัพธ์ที่เป็นแบบส่วนบุคคลสำหรับผู้ใช้ที่ลงชื่อเข้าใช้ หากผู้สร้างไซต์เลือกตัวเลือกนั้น

| โมดูลการรวบรวมผลิตภัณฑ์  | ชนิดข้อมูล | คำอธิบาย |
|----------------------------|------|-------------|
| สร้าง                         | อัลกอริทึม | โมดูลนี้แสดงรายการของผลิตภัณฑ์ใหม่ล่าสุดที่มีการจัดประเภทเป็นช่องทางและแค็ตตาล็อกล่าสุด |
| ขายดีที่สุด               | อัลกอริทึม | โมดูลนี้แสดงรายการของผลิตภัณฑ์ที่ถูกจัดอันดับตามจำนวนยอดขายสูงสุด |
| กำลังนิยม                   | อัลกอริทึม | โมดูลนี้แสดงรายการของผลิตภัณฑ์ที่มีประสิทธิภาพสูงสุดสำหรับรอบระยะเวลาที่กำหนด ซึ่งจัดอันดับตามจำนวนยอดขายสูงสุด  |
| สิ่งที่มักซื้อพร้อมกัน | AI-ML | โมดูลนี้จะแนะนำรายการของผลิตภัณฑ์ที่ถูกซื้อโดยทั่วไปพร้อมกับเนื้อหาของรถเข็นปัจจุบันของผู้บริโภค |
| ผู้คนชอบเช่นกัน           | AI-ML | โมดูลนี้แนะนำผลิตภัณฑ์สำหรับผลิตภัณฑ์เมล็ดพันธุ์ที่กำหนดตามรูปแบบการซื้อของผู้บริโภค |
| เลือกมาสำหรับคุณ              | AI-ML | โมดูลนี้จะแนะนำรายการผลิตภัณฑ์ที่เป็นแบบส่วนบุคคลตามรูปแบบการซื้อของผู้ใช้ที่ลงชื่อเข้าใช้ สำหรับผู้ใช้ที่เป็นแขก รายการนี้จะถูกยุบ |
