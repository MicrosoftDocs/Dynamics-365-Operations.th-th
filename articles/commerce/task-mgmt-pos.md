# <a name="task-management-in-pos"></a>การจัดการงานใน POS

[!include [banner](includes/banner.md)]

หัวข้อนี้จะอธิบายถึงการจัดการงานในแอปพลิเคชันการขายหน้าร้าน (POS) ใน Microsoft Dynamics 365 Commerce

แอปพลิเคชัน POS Dynamics 365 Commerce มีลักษณะการทำงานในการจัดการงานที่ให้ผู้จัดการร้านค้าและผู้ปฏิบัติงานจัดการงานและอัปเดตสถานะของงาน ผู้ปฏิบัติงานในร้านค้าสามารถเข้าถึงงานได้ด้วยการเลือกไทล์ **งาน** บนโฮมเพจของ POS หรือโดยการเลือกการแจ้งเตือนของงาน ตามค่าเริ่มต้น จะมีการใช้ผู้ปฏิบัติงานในแท็บ **งานของฉัน** ซึ่งผู้ใช้จะสามารถดูงานที่ได้รับมอบหมายได้ อย่างไรก็ตาม ผู้ใช้สามารถสลับไปยังแท็บ **งานที่พ้นกำหนดงาน** **เปิดงาน** และ **รายการงาน** ได้อย่างง่ายดาย

## <a name="task-operations-for-store-managers"></a>การดำเนินงานสำหรับผู้จัดการร้านค้า

ผู้จัดการร้านค้าสามารถดำเนินงานต่อไปนี้ในแอปพลิเคชัน POS โดยใช้ปุ่มบนแถบคำสั่ง:

- **กำหนด** – กำหนดงานที่เลือกให้กับผู้ปฏิบัติงานในร้านค้า
- **สถานะของงาน** – เปลี่ยนสถานะของงานที่เลือก
- **ตัวกรอง** – โดยค่าเริ่มต้น จะมีการแสดงเฉพาะงานที่ใช้งานอยู่เท่านั้น อย่างไรก็ตาม ด้วยการใช้ตัวกรอง ผู้จัดการสามารถดูงานทั้งหมดได้ แม้กระทั่งงานที่เสร็จสมบูรณ์หรือถูกยกเลิกแล้ว
- **งานใหม่** – สร้างงานภายใต้รายการงานที่มีอยู่หรือสร้างงานวัตถุประสงค์เดียว

ผู้ปฏิบัติงานร้านค้าสามารถดำเนินงานต่อไปนี้ในแอปพลิเคชัน POS โดยใช้ปุ่มบนแถบคำสั่ง:

- **สถานะของงาน** – เปลี่ยนสถานะของงานที่เลือก
- **ตัวกรอง** – โดยค่าเริ่มต้น จะมีการแสดงเฉพาะงานที่ใช้งานอยู่เท่านั้น อย่างไรก็ตาม ด้วยการใช้ตัวกรอง ผู้ปฏิบัติงานสามารถดูงานทั้งหมดได้ แม้กระทั่งงานที่เสร็จสมบูรณ์หรือถูกยกเลิกแล้ว
