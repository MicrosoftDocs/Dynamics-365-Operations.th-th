# <a name="upload-files-other-than-images-and-videos"></a>อัพโหลดไฟล์นอกเหนือจากรูปภาพและวิดีโอ

[!include [banner](includes/banner.md)]

หัวข้อนี้จะอธิบายวิธีการอัพโหลดไฟล์ นอกเหนือจากรูปภาพและวิดีโอในโปรแกรมสร้างไซต์ Microsoft Dynamics 365 Commerce

## <a name="overview"></a>ภาพรวม

ไลบรารีสื่อของโปรแกรมสร้างไซต์ Commerce สนับสนุนการอัพโหลดสินทรัพย์ไบนารี นอกเหนือจากรูปภาพหรือวิดีโอ ตัวอย่างเช่น คุณอาจต้องการอัพโหลด Microsoft Excel Microsoft Word Microsoft PowerPoint หรือไฟล์ PDF

ชนิดเอกสารต่อไปนี้ได้รับการสนับสนุน:
- 7Z
- AVI
- CS
- CSS
- DOC
- DOCX
- EPUB
- GIF
- INDD
- JAR
- JPG
- JPEG
- JS
- MP3
- MP4
- MPEG
- MPG
- ODP
- ODS
- ODT
- PDF
- PNG
- PPT
- PPTX
- PS
- QXP
- RAR
- RTF
- SVG
- TAR
- TGZ
- TXT
- WMV
- XLS
- XLSX
- XML
- ZIP

## <a name="upload-a-file"></a>อัพโหลดไฟล์

หากต้องการอัพโหลดไฟล์ไปยังโปรแกรมสร้างไซต์ Commerce ให้ทำตามขั้นตอนเหล่านี้

1. ในบานหน้าต่างนำทางทางด้านซ้าย ให้เลือก **ไลบรารีสื่อ**
1. บนแถบคำสั่ง เลือก **อัพโหลด \> อัพโหลดรายการสื่อ**
1. ในตัวสำรวจไฟล์ ให้เลือกไฟล์ตั้งแต่หนึ่งไฟล์ขึ้นไป แล้วเลือก **เปิด**
1. ในกล่องโต้ตอบ **อัพโหลดรายการสื่อ** ให้ป้อนชื่อเรื่อง คำอธิบาย และข้อมูลเมตาของคำสำคัญ ตามต้องการ
1. เมื่อต้องการเผยแพร่ไฟล์หลังจากการอัพโหลดโดยทันที ให้เลือกกล่องกาเครื่องหมาย **เผยแพร่รายการสื่อหลังจากการอัพโหลด**
1. เลือก **ตกลง**
