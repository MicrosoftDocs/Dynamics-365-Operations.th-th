# <a name="dynamics-365-commerce-preview-environment-faq"></a>FAQ เกี่ยวกับสภาพแวดล้อมการแสดงตัวอย่างของ Dynamics 365 Commerce

[!include [banner](includes/banner.md)]

หัวข้อนี้ให้คำตอบของคำถามที่ถามบ่อยเกี่ยวกับสภาพแวดล้อมการแสดงตัวอย่าง Microsoft Dynamics 365 Commerce

**ฉันสามารถถ่ายโอนคำเชิญของฉันสำหรับสภาพแวดล้อมการแสดงตัวอย่าง Commerce ไปยังผู้เช่าอื่นได้หรือไม่**

ใช่ สำหรับการถ่ายโอนคำเชิญคุณสามารถใช้ [แบบฟอร์มการโอนตัวอย่าง Commerce](https://aka.ms/Dynamics365CommercePreviewTransferForm)

**การถ่ายโอนคำเชิญใช้เวลานานเท่าใด**

การถ่ายโอนจะใช้เวลาเฉลี่ยประมาณสามถึงห้าวันทำการ อย่างไรก็ตามอาจมีข้อยกเว้น

**สภาพแวดล้อมการแสดงตัวอย่างCommerce ทำงานร่วมกับ Dynamics 365 Finance หรือโครงการ Dynamics 365 Supply Chain หรือไม่**

หมายเลข สภาพแวดล้อมการแสดงตัวอย่างของ Commerce จะทำงานกับโครงการ Dynamics 365 Retail เท่านั้น

**เราสามารถใช้สภาพแวดล้อมการแสดงตัวอย่าง Commerce เป็นหน้าร้านอีคอมเมิร์ซสำหรับลูกค้าที่ใช้ Retail ในปัจจุบันได้หรือไม่**

หมายเลข สภาพแวดล้อมการแสดงตัวอย่าง Commerce เป็นเพียงสภาพแวดล้อมการประเมิน ถ้าคุณต้องการสภาพแวดล้อมสำหรับลูกค้าที่ดำเนินการขายปลีกให้ติดต่อ Microsoft

**สภาพแวดล้อมการแสดงตัวอย่าง Commerce สามารถใช้เพื่อจัดสรรคุณลักษณะอีคอมเมิร์ซที่ด้านบนของแอปพลิเคชัน/สภาพแวดล้อมที่มีการนำการขายปลีกมาใช้งานหรือไม่**

หมายเลข สภาพแวดล้อมการแสดงตัวอย่างของ Commerce อยู่ในขณะนี้พร้อมใช้งานเฉพาะในสภาพแวดล้อมใหม่ที่ถูกปรับใช้บนโครงการหน่วยเก็บสินค้าขายปลีก (SKU) ที่มีข้อมูลสาธิตจากเวอร์ชัน10.0.6 เท่านั้น

**ค่าใช้จ่ายอะไรบ้างที่เกี่ยวข้องในการปรับใช้สภาพแวดล้อมการแสดงตัวอย่างของ Commerce บน Microsoft Azure โดย Microsoft Dynamics Lifecycle Services (LCS)**

Retail คือส่วนประกอบเดียวที่โฮสต์ในการสมัครใช้งานของคุณ ส่วนประกอบอื่นๆ เช่น Retail Cloud Scale Unit (RCSU) และพาณิชย์อิเล็กทรอนิกส์จะถูกโฮสต์ในการสมัครใช้งาน Microsoft คุณสามารถใช้เครื่องคิดเลข [Azure](https://azure.microsoft.com/pricing/calculator/) เพื่อกำหนดราคา

**ภูมิศาสตร์ Azure ใดที่สนับสนุนสำหรับสภาพแวดล้อมการแสดงตัวอย่างของ Commerce ในขณะนี้**

สภาพแวดล้อมการแสดงตัวอย่าง Commerce สามารถปรับใช้ได้เฉพาะในภูมิศาสตร์อเมริกาเหนือเท่านั้น

**มีการดาวน์โหลดบนฮาร์ดดิสก์เสมือน (VHD) ที่มีตัวเลือกเครื่องเสมือนที่สมบูรณ์ OneBox (VM) หรือไม่?**

Dynamics 365 Retail Cloud Scale Unit (RCSU) และอีคอมเมิร์ซเป็นซอฟต์แวร์ที่สมบูรณ์แบบบริการ (SaaS) และต้องเป็นโฮสต์แบบคลาวด์

**สามารถใช้สภาพแวดล้อมการแสดงตัวอย่างของ Commerce ได้นานเท่าใด**

สภาพแวดล้อมการแสดงตัวอย่าง Commerce มีการจำกัดเวลา 30 วัน นับจากวันที่จัดสรรอีคอมเมิร์ซ

**ฉันสามารถขยายขีดจำกัดเวลาสำหรับสภาพแวดล้อมการแสดงตัวอย่างของ Commerce ได้หรือไม่**

ใช่ คุณสามารถติดต่อทีมสนับสนุนโดยใช้ [แบบฟอร์มส่วนขยายการแสดงตัวอย่าง Commerce](https://aka.ms/Dynamics365CommercePreviewExtensionForm)

**เราสามารถทำการร้องขอหลายรายการสำหรับสภาพแวดล้อมการแสดงตัวอย่างของ Commerce ได้หรือไม่**

เรามอบโควตาของสภาพแวดล้อมการแสดงตัวอย่างของ Commerce หนึ่งรายการสำหรับแต่ละคำขอที่ยอมรับ หากคุณต้องการมากกว่าหนึ่งสภาพแวดล้อมการแสดงตัวอย่างโปรดติดต่อ Microsoft สำหรับข้อมูลผู้ติดต่อ ดูที่ส่วนถัดไป

## <a name="dynamics-365-commerce-preview-environment-contact-information"></a>Dynamics 365 Commerce แสดงตัวอย่างข้อมูลผู้ติดต่อของสภาพแวดล้อม

เมื่อต้องการติดต่อ Microsoft หากคุณมีคำถามหรือคำขอที่เกี่ยวข้องกับสภาพแวดล้อมการแสดงตัวอย่าง โปรดไปที่ [กลุ่ม Microsoft Dynamics 365 Commerce Preview Yammer](https://aka.ms/Dynamics365CommercePreviewYammer) เพื่อขอความช่วยเหลือ

ถ้าคุณพบปัญหาเมื่อคุณพยายามเข้าถึงกลุ่ม Yammer คุณสามารถติดต่อ Microsoft ทางอีเมลได้ที่ <Dynamics365Commerce@microsoft.com> ที่อยู่อีเมลนี้ไม่ได้ตรวจสอบอย่างแข็งขัน. ดังนั้นคาดว่าจะมีการหน่วงเวลาในการตอบสนอง
