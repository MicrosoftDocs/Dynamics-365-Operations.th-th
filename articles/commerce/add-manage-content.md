# <a name="ways-to-add-content"></a>วิธีการเพิ่มเนื้อหา

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

หัวข้อนี้ให้ข้อมูลเกี่ยวกับวิธีการเพิ่มและจัดการเนื้อหาบนไซต์ Microsoft Dynamics 365 Commerce ของคุณ

## <a name="overview"></a>ภาพรวม

มีหลายวิธีในการเปลี่ยนรูปลักษณ์ ความรู้สึก และเนื้อหาของเว็บไซต์ของคุณ ทั้งนี้ขึ้นอยู่กับระดับที่จำเป็นของการเลือกกำหนด การเปลี่ยนแปลงเหล่านี้อาจมีการดำเนินการโดยไม่มีนักพัฒนา ตัวอย่างเช่น จะไม่มีการบันทึกรหัสเมื่อต้องการสร้างแม่แบบ เลือกชุดรูปแบบ และเลือกและตั้งค่าคอนฟิกโมดูล ในทางกลับกัน ทักษะการพัฒนาต้องใช้ในการสร้างชุดรูปแบบหรือโมดูลใหม่ เนื่องจากชุดพัฒนาซอฟต์แวร์อีคอมเมิร์ซ (SDK) และ Microsoft Dynamics Lifecycle Services (LCS) ต้องใช้ลำดับงานการปรับใช้

หัวข้อต่อไปนี้ให้ข้อมูลรายละเอียดเกี่ยวกับวิธีการเพิ่มและจัดการเนื้อหาบนไซต์ โดยโฟกัสที่ขอบเขตของไซต์ที่ไม่จำเป็นต้องมีนักพัฒนา ตามความจำเป็น พวกเขาระบุงานที่จำเป็นต้องใช้งาน SDK

- เมื่อต้องการเปลี่ยนแปลงข้อความ รูปภาพ หรือวิดีโอบนหน้าไซต์ที่มีอยู่ ดูที่ [การทำงานกับโมดูล](work-with-modules.md)
- เพื่อช่วยรับประกันความผิดพลาด ประสบการณ์การสร้างรูปแบบแบรนด์สำหรับผู้เขียนเนื้อหาเว็บ ดูที่ [การทำงานกับแม่แบบ](work-with-templates.md)
- เมื่อต้องการจัดเรียงส่วนบนเพจของไซต์ ดูที่ [การทำงานกับโครงร่าง](work-with-layouts.md)
- เมื่อต้องการเปลี่ยนแบบอักษร สี และลักษณะทั่วไปของเพจของไซต์ ดูที่ [การเลือกชุดรูปแบบไซต์](select-site-theme.md)
