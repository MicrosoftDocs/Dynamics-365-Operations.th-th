<span data-ttu-id="822bf-106">โมดูลแกลเลอรี่สื่อแสดงรูปภาพอย่างน้อยหนึ่งรูปภาพในมุมมองแกลเลอรี</span><span class="sxs-lookup"><span data-stu-id="822bf-106">Media gallery modules show one or more images in a gallery view.</span></span> <span data-ttu-id="822bf-107">โมดูลแกลเลอรี่สื่อสนับสนุนภาพขนาดย่อ ซึ่งสามารถจัดเรียงอย่างใดอย่างหนึ่งตามแนวนอน (โดยเป็นแถวด้านล่างรูป) หรือในแนวตั้ง (เป็นคอลัมน์ที่อยู่ถัดจากรูปภาพ)</span><span class="sxs-lookup"><span data-stu-id="822bf-107">Media gallery modules support thumbnail images, which can be arranged either horizontally (as a row below the image) or vertically (as a column next to the image).</span></span> <span data-ttu-id="822bf-108">โมดูลแกลเลอรี่สื่อยังให้ความสามารถที่เปิดใช้งานการซูมภาพ (ขยาย) หรือดูในโหมดเต็มหน้าจอ</span><span class="sxs-lookup"><span data-stu-id="822bf-108">Media gallery modules also provide capabilities that enable images to be zoomed (magnified) or viewed in full-screen mode.</span></span> <span data-ttu-id="822bf-109">เมื่อต้องการแสดงในโมดูลแกลเลอรี่สื่อ รูปภาพต้องมีอยู่ในไลบรารีสื่อของโปรแกรมสร้างไซต์ Commerce</span><span class="sxs-lookup"><span data-stu-id="822bf-109">To be rendered in a media gallery module, an image must be available in the Commerce site builder Media Library.</span></span> <span data-ttu-id="822bf-110">ในปัจจุบันโมดูลแกลเลอรี่สื่อสนับสนุนรูปภาพเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="822bf-110">Currently, media gallery modules support only images.</span></span>
