<span data-ttu-id="e0452-151">โครงร่างคือระดับต่อไปในลำดับชั้นของแบบจำลองหน้าใต้เท็มเพลต</span><span class="sxs-lookup"><span data-stu-id="e0452-151">Layouts are the next level in the page model hierarchy, below templates.</span></span> <span data-ttu-id="e0452-152">ในขณะที่เท็มเพลตจะกำหนดโมดูลทั้งหมดที่ได้รับอนุญาตสำหรับหน้า โครงร่างจะเป็นการเลือกและการจัดเรียงโมดูลที่ชัดแจ้ง</span><span class="sxs-lookup"><span data-stu-id="e0452-152">Whereas a template defines all the modules that are allowed for a page, a layout is an explicit selection and arrangement of modules.</span></span> <span data-ttu-id="e0452-153">หน้าคือระดับต่อไปในลำดับชั้นของแบบจำลองหน้าใต้โครงร่าง</span><span class="sxs-lookup"><span data-stu-id="e0452-153">Pages are the next level in the page model hierarchy, below layouts.</span></span> <span data-ttu-id="e0452-154">มีการกำหนดเนื้อหาที่แปลเป็นภาษาท้องถิ่นสำหรับโมดูลที่เลือกในโครงร่าง</span><span class="sxs-lookup"><span data-stu-id="e0452-154">They define the localized content for the modules that are selected in the layout.</span></span>

สามารถเปลี่ยนเป็นโครงร่างอื่นจากลำดับชั้นเท็มเพลตนี้ได้โดยไม่ต้องกำหนดให้เนื้อหาระดับหน้านั้นถูกตรวจสอบ</span><span class="sxs-lookup"><span data-stu-id="e0452-170">Therefore, the layout for any page โครงร่างมีอีกทางเลือกหนึ่งในการปรับเปลี่ยนองค์ประกอบที่ใช้ร่วมกันสำหรับกลุ่มของหน้าโดยไม่จำเป็นต้องมีการอัพเดตแต่ละหน้า</span><span class="sxs-lookup"><span data-stu-id="e0452-172">Second, layouts ### <a name="preset-and-custom-layouts"></a><span data-ttu-id="e0452-175">โครงร่างที่กำหนดไว้ล่วงหน้าและที่กำหนดเอง</span><span class="sxs-lookup"><span data-stu-id="e0452-175">Preset and custom layouts</span></span>

<span data-ttu-id="e0452-176">โครงร่างบนไซต์ของคุณอาจทั้งแบบ *กำหนดล่วงหน้า* หรือแบบ *กำหนดเอง*</span><span class="sxs-lookup"><span data-stu-id="e0452-176">Layouts on your site can be either *preset* or *custom*:</span></span>
