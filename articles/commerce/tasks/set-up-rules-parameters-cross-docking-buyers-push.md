# <a name="set-up-rules-and-parameters-for-cross-docking-and-buyers-push"></a> ตั้งค่ากฎและพารามิเตอร์สำหรับการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าและการจัดส่งสินค้าแบบผลัก

[!include [banner](../includes/banner.md)]

ขั้นตอนนี้จะสาธิตขั้นตอนสร้างกฎการเพิ่มเติมสินค้า  กฎการเพิ่มเติมสินค้าสามารถใช้เพื่อควบคุมวิธีการกระจายผลิตภัณฑ์ไปยังร้านค้าเมื่อใช้การส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าและการจัดส่งสินค้าแบบผลัก  กฎการเติมสินค้าสามารถถูกตั้งค่าให้กับร้านค้าหรือกลุ่มร้านค้าได้  น้ำหนักที่กำหนดไว้สำหรับแต่ละบรรทัดรายการในกฎจะควบคุมวิธีที่ปริมาณของผลิตภัณฑ์นั้นถูกกระจายระหว่างร้านค้าเมื่อใช้กฎการเพิ่มเติมสินค้าเป็นวิธีการกระจายในการส่งสินค้าผ่านศูนย์เปลี่ยนถ่ายสินค้าและการจัดส่งสินค้าแบบผลัก  ขั้นตอนนี้ใช้บริษัทสาธิต USRT

1. ไปที่กฎการเพิ่มเติมสินค้า 
2. คลิก สร้าง
3. ในฟิลด์กฎการเพิ่มเติมสินค้า ให้พิมพ์ค่าใดค่าหนึ่ง 
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. คลิก บันทึก
6. คลิก เพิ่ม
7. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
    * คุณสามารถเลือกลำดับชั้นการเพิ่มเติมสินค้าหรือช่องทางสำหรับชนิดนี้  ค่าจะควบคุมว่าการเลือกด้วยชื่อจะเป็นลำดับชั้นของช่องทางหรือช่องทางเฉพาะ   สำหรับตัวอย่างนี้ ปล่อยให้ตั้งค่าเป็นลำดับชั้นการเพิ่มเติมสินค้า  
8. ในฟิลด์ชื่อ ให้เลือกค่าใดค่าหนึ่ง
    * ค่าน้ำหนักเริ่มต้นได้มาจากน้ำหนักที่กำหนดไว้ในคลังสินค้า   น้ำหนักนี้สามารถใช้สำหรับกฎการเพิ่มเติมสินค้าหรือคุณสามารถป้อนน้ำหนักใหม่ในฟิลด์น้ำหนัก   
9. ในฟิลด์น้ำหนัก ป้อนตัวเลข 
10. คลิก เพิ่ม
11. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
12. ในฟิลด์ชนิด เลือก 'ช่องทาง' 
13. ในฟิลด์ชื่อ ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
14. ในฟิลด์น้ำหนัก ป้อนตัวเลข 
15. คลิก บันทึก
