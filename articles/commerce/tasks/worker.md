# <a name="configure-a-worker"></a> ตั้งค่าคอนฟิกผู้ปฏิบัติงาน

[!include [banner](../includes/banner.md)]

กระบวนงานนี้อธิบายวิธีการตั้งค่าคอนฟิกผู้ปฏิบัติงานเป็นพนักงานขายที่มีสิทธิ์สำหรับค่าคอมมิชชันในการขายใน POS ขั้นตอนนี้ใช้บริษัทข้อมูลสาธิต USRT


## <a name="create-a-commission-sales-group-for-the-worker"></a>สร้างกลุ่มการขายที่มีค่าคอมมิชชันสำหรับผู้ปฏิบัติงาน
1. ไปที่การขายและการตลาด > ค่าคอมมิชชัน > กลุ่มการขาย
    * สามารถกำหนดผู้ปฏิบัติงานให้กับกลุ่มการขายอย่างน้อยหนึ่งกลุ่ม ใน POS คุณสามารถเลือกกลุ่มการขายใดๆ ที่ประกอบด้วยผู้ปฏิบัติงานจากสมุดที่อยู่ของร้านค้า  
2. คลิก สร้าง
3. ในฟิลด์กลุ่ม ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
5. คลิก บันทึก
6. ในบานหน้าต่างการดำเนินการ ให้คลิก ทั่วไป
7. คลิกพนักงานขาย
    * กลุ่มการขายสามารถประกอบด้วยผู้ปฏิบัติงานมากกว่าหนึ่งคน สามารถแบ่งค่าคอมมิชชันระหว่างผู้ปฏิบัติงานต่างๆ โดยขึ้นอยู่กับวิธีที่คุณกำหนดส่วนแบ่งค่าคอมมิชชัน  
8. ในฟิลด์ชื่อ ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
9. ในฟิลด์ส่วนแบ่งค่าคอมมิชชัน ให้ป้อนตัวเลข
10. คลิก บันทึก
11. ปิดหน้า
12. ปิดหน้า

## <a name="assign-the-workers-default-sales-group"></a>กำหนดกลุ่มการขายเริ่มต้นให้กับผู้ปฏิบัติงาน
1. ไปที่ Retail และ Commerce > พนักงาน > ผู้ปฏิบัติงาน
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
4. คลิกแท็บ Commerce
    * สามารถกำหนดผู้ปฏิบัติงานให้กับกลุ่มการขายเริ่มต้น กลุ่มการขายเริ่มต้นจะถูกเพิ่มลงในบรรทัดการขายใน POS โดยอัตโนมัติ ถ้ามีการเปิดใช้งานตัวเลือกในโพรไฟล์ฟังก์ชันสำหรับร้านค้า  
5. คลิก แก้ไข
6. ในฟิลด์กลุ่มเริ่มต้น ให้ป้อนหรือเลือกค่า
7. คลิก บันทึก
