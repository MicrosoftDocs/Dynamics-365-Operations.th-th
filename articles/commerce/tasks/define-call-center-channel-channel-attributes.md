# <a name="create-call-center-channels-and-define-channel-attributes"></a>กำหนดช่องทางคอลเซ็นเตอร์และกำหนดแอททริบิวต์ช่องทาง

[!include [banner](../includes/banner.md)]

กระบวนการนี้จะแนะนำถึงการสร้างช่องทางการค้าใหม่และกำหนดแอตทริบิวต์ของช่องทาง ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างงานนี้คือ USRT  กระบวนการนี้เจาะจงสำหรับบทบาทไอทีในเชิงพาณิชย์


## <a name="create-new-store"></a>สร้างร้านค้าใหม่
1. ไปที่ พื้นที่ทำงานทั้งหมด > การปรับใช้ช่องทาง
2. คลิก ช่องทางใหม่
3. คลิก ร้านค้า
4. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
5. ในฟิลด์หมายเลขร้านค้า ให้พิมพ์ค่า
6. ในฟิลด์คลังสินค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
7. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
8. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
9. ในฟิลด์เขตเวลาของร้านค้า ให้เลือกหนึ่งตัวเลือก
10. ในฟิลด์โพรไฟล์ช่องทาง ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
11. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
12. ในฟิลด์ภาษา ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
13. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
14. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
15. ในฟิลด์กลุ่มภาษีขาย ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
16. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
17. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
18. ในฟิลด์สมุดที่อยู่ลูกค้า ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา 
    * เลือกสมุดที่อยู่ที่ใช้ในการเชื่อมโยงลูกค้าไปยังร้านค้านี้  
19. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
20. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
21. คลิกเลือก 
22. ในฟิลด์สมุดที่อยู่พนักงาน ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา 
    * เลือกสมุดที่อยู่ที่ใช้ในการเชื่อมโยงพนักงานเก็บเงินไปยังช่องทางนี้  
23. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
24. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
25. คลิกเลือก 
26. ในฟิลด์ลูกค้าเริ่มต้น ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
