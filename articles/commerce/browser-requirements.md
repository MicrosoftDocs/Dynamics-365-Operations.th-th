# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>ข้อกำหนดเบราว์เซอร์สำหรับไซต์ Dynamics 365 Commerce

[!include [banner](includes/banner.md)]

บทความนี้ครอบคลุมถึงข้อกำหนดของเบราเซอร์สำหรับไซต์ของ Microsoft Dynamics 365 Commerce

ไซต์ Dynamics 365 Commerce ได้รับการสนับสนุนอย่างดีที่สุดจากเว็บเบราเซอร์ต่อไปนี้ เมื่อต้องการค้นหารุ่นล่าสุดของแต่ละเว็บเบราเซอร์ ให้ไปที่เว็บไซต์ของผู้ผลิตซอฟต์แวร์

- [Microsoft Edge](https://www.microsoft.com/edge) (จนถึงเวอร์ชันปัจจุบัน ลบ 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (จนถึงเวอร์ชันปัจจุบัน ลบ 1)
- [Google Chrome](https://www.google.com/chrome) (จนถึงเวอร์ชันปัจจุบัน ลบ 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (จนถึงเวอร์ชันปัจจุบัน ลบ 1)

> [!NOTE]
> ไม่ได้รับการสนับสนุนใน Internet Explorer หากคุณใช้ Internet Explorer คุณลักษณะบางอย่างอาจไม่ได้เปิดใช้งาน และผู้ใช้อาจมีประสบการณ์ต่ำกว่ามาตรฐาน
