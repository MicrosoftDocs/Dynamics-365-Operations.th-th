<span data-ttu-id="1b478-104">หัวข้อนี้อธิบายวิธีการเพิ่มข้อความสงวนลิขสิทธิ์ให้กับเว็บไซต์อีคอมเมิร์ซของคุณ</span><span class="sxs-lookup"><span data-stu-id="1b478-104">This topic describes how to add a copyright notice to your e-Commerce website.</span></span>

## <a name="prerequisites"></a><span data-ttu-id="1b478-105">ข้อกำหนดเบื้องต้น</span><span class="sxs-lookup"><span data-stu-id="1b478-105">Prerequisites</span></span>

<span data-ttu-id="1b478-106">ก่อนที่คุณจะสามารถเพิ่มข้อความสงวนลิขสิทธิ์ลงในไซต์ของคุณ คุณจะต้องมีสิ่งต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="1b478-106">Before you can add a copyright notice to your site, you must have the following items:</span></span>

- <span data-ttu-id="1b478-107">แม่แบบที่รวมส่วนที่เป็นส่วนท้ายที่ใช้ร่วมกัน</span><span class="sxs-lookup"><span data-stu-id="1b478-107">A template that includes a shared footer fragment.</span></span>
- <span data-ttu-id="1b478-108">เพจที่ใช้แม่แบบ</span><span class="sxs-lookup"><span data-stu-id="1b478-108">A page that uses that template.</span></span>

## <a name="add-a-copyright-notice"></a><span data-ttu-id="1b478-109">เพิ่มข้อความสงวนลิขสิทธิ์</span><span class="sxs-lookup"><span data-stu-id="1b478-109">Add a copyright notice</span></span>

<span data-ttu-id="1b478-110">เมื่อต้องการเพิ่มข้อความสงวนลิขสิทธิ์ที่ด้านล่างของทุกเพจที่ใช้แม่แบบเฉพาะ ให้ทำตามขั้นตอนต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="1b478-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="1b478-111">ไปที่ **ส่วน** จากนั้น ให้เลือก **ส่วนของเพจใหม่**</span><span class="sxs-lookup"><span data-stu-id="1b478-111">Go to **Fragments**, and then select **New Page Fragment**.</span></span>
1. <span data-ttu-id="1b478-112">ในกล่องโต้ตอบ ให้เลือกโมล **ส่วนท้าย** และตั้งชื่อส่วน</span><span class="sxs-lookup"><span data-stu-id="1b478-112">In the dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="1b478-113">ตัวอย่างเช่น ป้อน **ส่วนท้าย-ลิขสิทธิ์**</span><span class="sxs-lookup"><span data-stu-id="1b478-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="1b478-114">เลือก **ตกลง**</span><span class="sxs-lookup"><span data-stu-id="1b478-114">Select **OK**.</span></span>
1. <span data-ttu-id="1b478-115">ในบานหน้าต่างนำทาง เลือกปุ่มจุดไข่ปลา (**...**) ถัดจาก **ส่วนท้าย** แล้วเลือก **เพิ่มโมดูล**</span><span class="sxs-lookup"><span data-stu-id="1b478-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="1b478-116">ในกล่องโต้ตอบ เลือก **ประเภทของส่วนท้าย** แล้วคลิก **ตกลง**</span><span class="sxs-lookup"><span data-stu-id="1b478-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="1b478-117">ในบานหน้าต่างนำทาง เลือกปุ่มจุดไข่ปลาถัดจาก **ประเภทของส่วนท้าย** แล้วเลือก **เพิ่มโมดูล**</span><span class="sxs-lookup"><span data-stu-id="1b478-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="1b478-118">ในกล่องโต้ตอบ เลือก **บล็อคข้อความ** แล้วคลิก **ตกลง**</span><span class="sxs-lookup"><span data-stu-id="1b478-118">In the dialog box, select **Text block**, and then select **OK**.</span></span>
1. <span data-ttu-id="1b478-119">ในบานหน้าต่างนำทางให้เลือก **บล็อกข้อความ**</span><span class="sxs-lookup"><span data-stu-id="1b478-119">In the navigation pane, select **Text block**.</span></span>
1. <span data-ttu-id="1b478-120">ในบานหน้าต่างคุณสมบัติทางด้านขวา ในฟิลด์ **ย่อหน้า** ให้เพิ่มข้อความลิขสิทธิ์ของคุณ</span><span class="sxs-lookup"><span data-stu-id="1b478-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="1b478-121">ตัวอย่างเช่น: ป้อน **(C) Fabrikam 2019**</span><span class="sxs-lookup"><span data-stu-id="1b478-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="1b478-122">เลือก **บันทึก** เลือก **เช็คอิน** แล้วจากนั้น เลือก **เผยแพร่**</span><span class="sxs-lookup"><span data-stu-id="1b478-122">Select **Save**, select **Check In**, and then select **Publish**.</span></span>
1. <span data-ttu-id="1b478-123">ไปที่ **แม่แบบ** แล้วเลือกเท็มเพลต จากนั้นเลือก **เช็คเอาท์**</span><span class="sxs-lookup"><span data-stu-id="1b478-123">Go to **Templates**, select the template, and then select **Check Out**.</span></span>
1. <span data-ttu-id="1b478-124">ภายใต้ **โครงร่างเพจ** ขยาย **เนื้อหา** แล้วขยาย **เพจเริ่มต้น**</span><span class="sxs-lookup"><span data-stu-id="1b478-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="1b478-125">เลือกปุ่มจุดไข่ปลาถัดจาก **ช่องส่วนท้าย** จากนั้นเลือก **เพิ่มส่วน**</span><span class="sxs-lookup"><span data-stu-id="1b478-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="1b478-126">เลือกส่วนที่คุณสร้างไว้ก่อนหน้านี้ แล้วเลือก **เลือก**</span><span class="sxs-lookup"><span data-stu-id="1b478-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="1b478-127">เช็คอินแม่แบบ และเผยแพร่</span><span class="sxs-lookup"><span data-stu-id="1b478-127">Check in the template, and publish it.</span></span>

<span data-ttu-id="1b478-128">ส่วนท้ายที่มีข้อความสงวนลิขสิทธิ์จะปรากฏขึ้นโดยอัตโนมัติที่ด้านล่างของเพจทั้งหมดที่ใช้แม่แบบที่เลือก</span><span class="sxs-lookup"><span data-stu-id="1b478-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
