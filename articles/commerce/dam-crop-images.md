<span data-ttu-id="b0178-106">ไลบรารีสื่อของโปรแกรมสร้างไซต์ Commerce จะช่วยให้คุณสามารถตัดรูปเพื่อให้เหมาะสมกับชนิดโมดูลและ viewports ต่างๆ</span><span class="sxs-lookup"><span data-stu-id="b0178-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="b0178-107">ตัดรูปภาพ</span><span class="sxs-lookup"><span data-stu-id="b0178-107">Crop an image</span></span>

<span data-ttu-id="b0178-108">เมื่อต้องการตัดรูปภาพในโปรแกรมสร้างไซต์ ให้ทำตามขั้นตอนต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="b0178-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="b0178-109">ในบานหน้าต่างนำทางด้านซ้ายของโปรแกรมสร้างไซต์ Commerce ให้เลือก **ไลบรารีสื่อ**</span><span class="sxs-lookup"><span data-stu-id="b0178-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="b0178-110">ในหน้าต่างหลัก เลือกภาพที่คุณต้องการแก้ไข</span><span class="sxs-lookup"><span data-stu-id="b0178-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="b0178-111">บนแถบคำสั่ง ให้เลือก **แก้ไข**</span><span class="sxs-lookup"><span data-stu-id="b0178-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="b0178-112">เลือกรูปภาพเพื่อป้อน **โหมดแก้ไข**</span><span class="sxs-lookup"><span data-stu-id="b0178-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="b0178-113">ภายใต้ **โหมดแก้ไข** เลือก **แก้ไขมุมมองตามโมดูล**</span><span class="sxs-lookup"><span data-stu-id="b0178-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="b0178-114">จากเมนูแบบหล่นลง **โมดูล** เลือกชนิดโมดูล</span><span class="sxs-lookup"><span data-stu-id="b0178-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="b0178-115">จากเมนูแบบหล่นลง **ชนิดมุมมอง** เลือกชนิดมุมมอง</span><span class="sxs-lookup"><span data-stu-id="b0178-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="b0178-116">จากเมนูแบบหล่นลง **การจัดวาง** เลือกการจัดวางภาพ</span><span class="sxs-lookup"><span data-stu-id="b0178-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="b0178-117">จากเมนูแบบหล่นลง **ช่องมุมมอง** เลือกขนาดช่องมุมมอง</span><span class="sxs-lookup"><span data-stu-id="b0178-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="b0178-118">ภาพถูกวางทับกับพื้นที่ซึ่งแสดงถึงเขตการตัด</span><span class="sxs-lookup"><span data-stu-id="b0178-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="b0178-119">ย้ายและปรับขนาดเขตการตัดตามต้องการ</span><span class="sxs-lookup"><span data-stu-id="b0178-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="b0178-120">อัตราส่วนกว้างยาวจะถูกเก็บรักษาไว้โดยอัตโนมัติ</span><span class="sxs-lookup"><span data-stu-id="b0178-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="b0178-121">เมื่อคุณดำเนินการเสร็จสิ้นแล้ว บนแถบคำสั่ง ให้เลือก **บันทึก** และจากนั้น เลือก **เสร็จสิ้นการแก้ไข**</span><span class="sxs-lookup"><span data-stu-id="b0178-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="b0178-122">หลังจากการตัดแบบกำหนดเองเสร็จสมบูรณ์แล้ว การปรับเปลี่ยนรูปภาพจะมีผลเกือบทันที</span><span class="sxs-lookup"><span data-stu-id="b0178-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
