# <a name="hero-module"></a>โมดูลฮีโร

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

หัวข้อนี้ครอบคลุมถึงโมดูลฮีโร และอธิบายวิธีการเพิ่มลงในเพจของไซต์ใน Microsoft Dynamics 365 Commerce

## <a name="overview"></a>ภาพรวม

โมดูลฮีโรใช้ในการทำตลาดผลิตภัณฑ์ หรือโปรโมชันผ่านการรวมกันของรูปภาพและข้อความ ตัวอย่างเช่น ผู้ค้าปลีกสามารถเพิ่มโมดูลฮีโรไปยังโฮมเพจของไซต์อีคอมเมิร์ซเพื่อส่งเสริมผลิตภัณฑ์ใหม่ และดึงดูดความสนใจของลูกค้า

โมดูลของฮีโรถูกขับเคลื่อนด้วยข้อมูลจากระบบการจัดการเนื้อหา (CMS) โมดูลที่ทำงานแยกต่างหากคือโมดูลที่ไม่ขึ้นอยู่กับบริบทเพจ หรือโมดูลอื่นๆ บนเพจ โมดูลฮีโรสามารถวางบนเพจเว็บไซต์ใดๆ ที่ผู้ค้าปลีกต้องการทำตลาด หรือส่งเสริมบางสิ่งบางอย่าง (เช่น ผลิตภัณฑ์ การขาย หรือคุณลักษณะ)

## <a name="examples-of-hero-module-in-e-commerce"></a>ตัวอย่างของโมดูลฮีโรในอีคอมเมิร์ซ

- คุณสามารถใช้โมดูลฮีโรบนโฮมเพจของไซต์อีคอมเมิร์ซ เพื่อเน้นโปรโมชั่นและผลิตภัณฑ์ใหม่
- โมดูลฮีโรสามารถใช้ในเพจรายละเอียดของผลิตภัณฑ์ เพื่อแสดงข้อมูลผลิตภัณฑ์
- หลายโมดูลฮีโรสามารถใส่ไว้ภายในโมดูลวงล้อเพื่อเน้นผลิตภัณฑ์ หรือโปรโมชั่นต่างๆ

## <a name="hero-module-properties"></a>คุณสมบัติของโมดูลฮีโร

| ชื่อคุณสมบัติ  | มูลค่า | คำอธิบาย |
|----------------|--------|-------------|
| รูปภาพ          | ไฟล์รูปภาพ | สามารถใช้รูปภาพเพื่อแสดงผลิตภัณฑ์ หรือโปรโมชันได้ สามารถอัพโหลดรูปภาพไปที่แกลเลอรีรูปภาพ หรือสามารถใช้รูปภาพที่มีอยู่ได้ |
| หัวเรื่อง        | ข้อความหัวข้อ และแท็กหัวข้อ (**H1**, **H2**, **H3**, **H4**, **H5**, or **H6**) | ทุกโมดูลฮีโรสามารถมีหัวข้อได้ โดยค่าเริ่มต้น แท็กหัวเรื่อง **H2** จะใช้สำหรับส่วนหัว อย่างไรก็ตาม คุณสามารถเปลี่ยนแท็กเพื่อให้ตรงกับความต้องการสำหรับการเข้าถึงได้ |
| ย่อหน้า      | ข้อความย่อหน้า | โมดูลฮีโรสนับสนุนข้อความย่อหน้าในรูปแบบข้อความที่หลากหลาย ความสามารถในการทำงานของข้อความที่สมบูรณ์แบบพื้นฐานบางอย่างจะได้รับการสนับสนุน เช่น ข้อความตัวเอน ขีดเส้นใต้ และแบบตัวหนา และไฮเปอร์ลิงค์ ความสามารถบางอย่างเหล่านี้อาจถูกแทนที่ด้วยชุดรูปแบบของเพจที่ใช้กับโมดูล |
| ลิงค์           | ลิงค์ข้อความ ลิงค์ URL ป้ายชื่อ Accessible Rich Internet Applications (ARIA) และ **เปิดลิงค์ในแท็บใหม่** | โมดูลฮีโรสนับสนุนลิงค์ "การกระตุ้นให้ดำเนินการ" หนึ่งรายการขึ้นไป ถ้ามีการเพิ่มลิงค์ ลิงค์ข้อความ URL และป้ายชื่อ ARIA จำเป็นต้องใช้ ป้ายชื่อ ARIA ควรเป็นคำอธิบายเพื่อให้ตรงตามข้อกำหนดของความสามารถในการเข้าถึง สามารถตั้งค่าคอนฟิกลิงค์ เพื่อให้เปิดบนแท็บใหม่ได้ |
| การจัดวางข้อความ | **ด้านบนซ้าย** **ด้านบนขวา** **ด้านบนตรงกลาง** **ด้านล่างซ้าย** **ด้านล่างขวา** **ด้านล่างตรงกลาง** **ศูนย์กลางซ้าย** **ศูนย์กลางขวา** หรือ **กึ่งกลางศูนย์กลาง** | คุณสมบัตินี้กำหนดตำแหน่งของรูปภาพที่สัมพันธ์กับข้อความ ตัวอย่างเช่น ถ้า **ด้านขวา** ถูกเลือก รูปภาพจะปรากฏทางด้านขวาของข้อความ |
| ชุดรูปแบบของข้อความ     | **สีอ่อน** หรือ **สีเข้ม** | คุณสามารถกำหนดโครงร่างสีสำหรับข้อความได้ ตามรูปภาพพื้นหลัง ตัวอย่างเช่น ถ้ารูปภาพมีพื้นหลังสีเข้ม ชุดรูปแบบสีอ่อนสามารถใช้เพื่อให้สามารถมองเห็นข้อความได้ชัดขึ้น และเพื่อให้ตรงกับอัตราส่วนความคมชัดของสีสำหรับวัตถุประสงค์ในการเข้าถึง |
| ไล่ระดับสี       | **จริง** หรือ **เท็จ** | คุณสามารถใช้การไล่ระดับสีกับรูปภาพ เพื่อให้ตรงกับอัตราส่วนความคมชัดของสีสำหรับวัตถุประสงค์ในการเข้าถึง |

## <a name="add-a-hero-module-to-a-new-page"></a>เพิ่มโมดูลฮีโรไปยังเพจใหม่

การเพิ่มโมดูลฮีโรไปยังหน้าใหม่ และตั้งค่าคุณสมบัติที่จำเป็น ให้ทำตามขั้นตอนต่อไปนี้

1. ไปที่ **แม่แบบ** และสร้างแม่แบบเพจที่ชื่อ **แม่แบบฮีโร**
1. ในช่อง **หลัก** ของหน้าเริ่มต้น ให้เพิ่มโมดูลฮีโร
1. เช็คอินแม่แบบ และเผยแพร่
1. ใช้แม่แบบฮีโรที่คุณเพิ่งสร้างขึ้นเพื่อสร้างเพจที่ชื่อ **เพจฮีโร**
1. ในช่อง **หลัก** ของเพจเริ่มต้น เลือกปุ่มจุดไข่ปลา (**...**) แล้วเลือก **เพิ่มโมดูล**
1. ในกล่องโต้ตอบ **เพิ่มโมดูล** ใต้ **เลือกโมดูล** เลือกโมดูลฮีโร จากนั้นเลือก **ตกลง**
1. ในแผนภูมิเค้าร่างทางด้านซ้าย เลือกโมดูลฮีโร
1. ในบานหน้าต่างคุณสมบัติทางด้านขวา ให้เลือก **เพิ่มรูปภาพ** เลือกรูปภาพที่มีอยู่ หรืออัพโหลดรูปภาพใหม่
1. เลือก **หัวข้อ**
1. ในกล่องโต้ตอบ **ส่วนหัว** ให้เพิ่มข้อความหัวข้อ เลือกระดับหัวข้อ แล้วเลือก **ตกลง**
1. ภายใต้ **Rich Text** ให้เพิ่มข้อความตามที่คุณต้องการ
1. เลือก **เพิ่มลิงค์การดำเนินการ**
1. ในกล่องโต้ตอบ **ลิงค์การดำเนินการ** เพิ่มข้อความลิงค์ URL ลิงค์ และป้ายชื่อ ARIA สำหรับลิงค์ แล้วเลือก **ตกลง**
1. บันทึกเพจ และแสดงตัวอย่างการเปลี่ยนแปลงของคุณ
1. เช็คอินหน้า และเผยแพร่
