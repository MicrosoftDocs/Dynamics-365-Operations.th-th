[!include [banner](includes/banner.md)]

<span data-ttu-id="21be8-104">เมื่อสร้าง [ใบสั่งของลูกค้า](customer-orders-overview.md) เพื่อเบิกสินค้าจากร้านค้า ผู้ใช้ร้านค้าสามารถใช้แอปพลิเคชันการขายหน้าร้าน (POS) เพื่อเริ่มต้นการเบิกสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="21be8-104">When a [customer order](customer-orders-overview.md) is created for store pickup, a store user can use the point of sale (POS) application to start the pickup of inventory.</span></span> <span data-ttu-id="21be8-105">POS จะรันการรวบรวมข้อมูลการจ่ายเงินขั้นสุดท้ายตามที่ต้องการ</span><span class="sxs-lookup"><span data-stu-id="21be8-105">POS will run the final payment capture as required.</span></span> <span data-ttu-id="21be8-106">นอกจากนี้ สินค้าคงคลังและการลงรายการบัญชีทางการเงินยังจะเสร็จสิ้นตามปริมาณที่เบิกสินค้าด้วย</span><span class="sxs-lookup"><span data-stu-id="21be8-106">It will also complete the inventory and financial posting for the quantities that are picked up.</span></span>

- <span data-ttu-id="21be8-109">เมื่อต้องการใช้การดําเนินการ **เรียกคืนใบสั่ง** ให้ค้นหาและเลือกใบสั่งที่จะถูกเบิกสินค้า</span><span class="sxs-lookup"><span data-stu-id="21be8-109">To use the **Recall order** operation, search for and select the order that will be picked up.</span></span>

<span data-ttu-id="21be8-111">ถ้าใบสั่งหรือรายการใบสั่งที่เลือกไม่ได้รับการตั้งค่าคอนฟิกให้เบิกสินค้าที่ร้านค้าที่ระบุ หรือถ้ามีการเบิกสินค้าตามใบสั่งครบแล้ว การดําเนินการ **เบิกสินค้า** จะไม่พร้อมใช้งาน</span><span class="sxs-lookup"><span data-stu-id="21be8-111">If the selected order or order lines aren't configured for pickup at that specific store, or if the order has already been fully picked up, the **Pick up** operation will be unavailable.</span></span>

<span data-ttu-id="21be8-113">ใน Microsoft Dynamics 365 Commerce รุ่น 10.0.17 และใหม่กว่า คุณลักษณะ **ประสบการณ์ของผู้ใช้ที่ปรับปรุงเพื่อการประมวลผลใบสั่งในการขายหน้าร้าน** สามารถเปิดผ่านการจัดการคุณลักษณะในศูนย์ควบคุม Commerce</span><span class="sxs-lookup"><span data-stu-id="21be8-113">In Microsoft Dynamics 365 Commerce version 10.0.17 and later, the **Improved user experience for pick up order processing in Point of Sale** feature can be turned on through Feature management in Commerce headquarters.</span></span> <span data-ttu-id="21be8-114">ถ้าคุณลักษณะนี้ปิดอยู่ ผู้ใช้จะเลือกปริมาณการเบิกสินค้าไม่ได้</span><span class="sxs-lookup"><span data-stu-id="21be8-114">If this feature is turned off, users can't select pickup quantities.</span></span> <span data-ttu-id="21be8-115">ตามค่าเริ่มต้น ปริมาณทั้งหมดที่สั่งไว้ของบรรทัดคือปริมาณที่จะเบิกสินค้า</span><span class="sxs-lookup"><span data-stu-id="21be8-115">By default, the full quantity that was ordered for the line is the quantity that will be picked up.</span></span> <span data-ttu-id="21be8-116">ประสบการณ์นี้อาจเป็นปัญหา เนื่องจากผู้ใช้อาจลืมที่จะเลือกสินค้าบางรายการเพื่อเบิกสินค้าเมื่อเบิกสินค้าผ่านการเติมสินค้าตามใบสั่ง</span><span class="sxs-lookup"><span data-stu-id="21be8-116">This experience can be problematic, because users might forget to select some items for pickup when they perform the pickup through order fulfillment.</span></span>

<span data-ttu-id="21be8-117">คุณลักษณะ **ประสบการณ์ของผู้ใช้ที่ปรับปรุงเพื่อการประมวลผลใบสั่งในอยู่ในการขายหน้าร้าน** ช่วยให้ผู้ใช้สามารถควบคุมการเลือกผลิตภัณฑ์ที่จะเบิกสินค้าได้มากกว่า และปริมาณของผลิตภัณฑ์เหล่านั้นที่จะเบิก</span><span class="sxs-lookup"><span data-stu-id="21be8-117">The **Improved user experience for pick up order processing in Point of Sale** feature gives users more control over the selection of products that will be picked up and the quantity of those products that will be picked up.</span></span> <span data-ttu-id="21be8-118">ผู้ใช้ไม่ต้องเลือกทุกบรรทัดของใบสั่งขายในหน้าการเติมสินค้าตามใบสั่งก่อนที่จะเลือก **เบิกสินค้า**</span><span class="sxs-lookup"><span data-stu-id="21be8-118">Users don't have to select every line of the sales order on the order fulfillment page before they select **Pick up**.</span></span> <span data-ttu-id="21be8-119">สินค้าทั้งหมดที่สามารถเบิกสินค้าได้จะแสดงขึ้น</span><span class="sxs-lookup"><span data-stu-id="21be8-119">All the items that can be picked up will be shown.</span></span> <span data-ttu-id="21be8-120">ผู้ใช้สามารถระบุการเบิกสินค้าได้หลายบรรทัดแม้ว่าจะเลือกรายการผลิตภัณฑ์เพียงรายการเดียว</span><span class="sxs-lookup"><span data-stu-id="21be8-120">Users can specify multiple lines for pickup even if only one product line is selected.</span></span>

<span data-ttu-id="21be8-121">คุณลักษณะ **ประสบการณ์ของผู้ใช้ที่ปรับปรุงเพื่อการประมวลผลใบสั่งในอยู่ในการขายหน้าร้าน** เปิด และคุณเลือกการดำเนินการ **เบิกสินค้า** กล่องโต้ตอบ **การเบิกสินค้า** จะปรากฏขึ้น</span><span class="sxs-lookup"><span data-stu-id="21be8-121">When the **Improve user experience for pick up order processing in Point of Sale** feature is turned on, and you select the **Pick up** operation, the **Pick up** dialog box appears.</span></span> <span data-ttu-id="21be8-122">คุณสามารถเลือกสินค้าและปริมาณที่จะเบิกสินค้าได้</span><span class="sxs-lookup"><span data-stu-id="21be8-122">There, you can select the items and quantities that will be picked up.</span></span> <span data-ttu-id="21be8-123">ตามค่าเริ่มต้น ปริมาณที่สั่งใด ๆ ที่มีสินค้าคงคลังในสถานะเบิกสินค้าหรือที่บรรจุจะถือว่ามีสิทธิ์เบิกสินค้า</span><span class="sxs-lookup"><span data-stu-id="21be8-123">By default, any ordered quantity that has inventory in a picked or packed state is considered eligible for pickup.</span></span> <span data-ttu-id="21be8-124">โดยค่าเริ่มต้น ปริมาณจะถูกตั้งค่าเป็นปริมาณการเบิกสินค้า</span><span class="sxs-lookup"><span data-stu-id="21be8-124">By default, that quantity is set as the pickup quantity.</span></span> <span data-ttu-id="21be8-125">คุณสามารถเปลี่ยนปริมาณที่ป้อนได้ ถ้าปริมาณไม่ใช่ 0 (ศูนย์) และไม่เกินปริมาณเปิดรวม (นั่นคือยังไม่ได้ออกใบแจ้งหนี้) ของบรรทัดที่เลือก</span><span class="sxs-lookup"><span data-stu-id="21be8-125">You can change the quantity that was entered, provided that the quantity isn't 0 (zero) and doesn't exceed the total open (that is, non-invoiced) quantity for the selected line.</span></span>

![กล่องโต้ตอบการเบิกสินค้า](media/pickupselect.png)

<span data-ttu-id="21be8-127">หลังจากที่คุณเลือกปริมาณที่จะเบิกสินค้าแล้วเลือก **เบิกสินค้า** หน้าธุรกรรมจะปรากฏขึ้น</span><span class="sxs-lookup"><span data-stu-id="21be8-127">After you select the quantities that will be picked up and then select **Pick up**, the transaction page appears.</span></span> <span data-ttu-id="21be8-128">หากเปิดคุณลักษณะ [การชำระเงินช่องทาง omni](omni-channel-payments.md) และมีบัตรเครดิตที่ได้รับอนุญาตล่วงหน้าในไฟล์ คุณจะต้องใช้การชำระเงิน</span><span class="sxs-lookup"><span data-stu-id="21be8-128">If the [omni-channel payments](omni-channel-payments.md) feature is turned on, and there are pre-authorized credit card payments on file, you must apply the payment.</span></span>

<span data-ttu-id="21be8-129">ในหน้าธุรกรรม ระบบจะคํานวณจํานวนเงินที่ครบกําหนดโดยการคํานวณยอดรวมที่ครบกําหนดของสินค้าในการเบิกสินค้าที่เลือก แล้วหักเงินฝากใด ๆ ที่ใช้ก่อนหน้านี้ หรือการจ่ายผ่านบัตรเครดิตที่ได้รับอนุญาต</span><span class="sxs-lookup"><span data-stu-id="21be8-129">On the transaction page, the system calculates the amounts that are due by calculating the total that is due for the selected pickup items and then subtracting any previously applied deposits or authorized credit card payments.</span></span> <span data-ttu-id="21be8-130">คุณต้องประมวลผลการเบิกสินค้าเพื่อประมวลผลธุรกรรมการเบิกสินค้าให้เสร็จสมบูรณ์</span><span class="sxs-lookup"><span data-stu-id="21be8-130">You must process payment to complete the pickup transaction.</span></span> <span data-ttu-id="21be8-131">หาก [โครงร่างหน้าจอ](pos-screen-layouts.md) ของหน้าธุรกรรมมีการตั้งค่าคอนฟิกเพื่อให้รวมการดําเนินการ **ธุรกรรมสรุป** และไม่มียอดเงินครบกําหนด คุณสามารถดําเนินธุรกรรมให้เสร็จสมบูรณ์ และไม่มียอดเงินครบกําหนดโดยไม่ได้เลือกวิธีการจ่ายเงิน</span><span class="sxs-lookup"><span data-stu-id="21be8-131">If the [screen layout](pos-screen-layouts.md) of the transaction page is configured so that it includes the **Conclude transaction** operation, and no amount is due, you can complete the transaction without selecting a payment method.</span></span> <span data-ttu-id="21be8-132">ถ้าการดําเนินการ **ธุรกรรมสรุป** ไม่พร้อมใช้งาน คุณสามารถเลือกลิงค์ **จำนวนเงินที่ครบกำหนดชำระ $0.00** ในบานหน้าต่าง **ผลรวม** เพื่อสรุปธุรกรรมได้โดยไม่ต้องเลือกวิธีการจ่ายเงิน</span><span class="sxs-lookup"><span data-stu-id="21be8-132">If the **Conclude transaction** operation isn't available, you can select the **$0.00 amount due** link in the **Totals** pane to conclude the transaction without having to select a payment method.</span></span>

## <a name="changing-pickup-lines-or-quantities"></a><span data-ttu-id="21be8-134">การเปลี่ยนรายการหรือปริมาณการเบิกสินค้า</span><span class="sxs-lookup"><span data-stu-id="21be8-134">Changing pickup lines or quantities</span></span>
