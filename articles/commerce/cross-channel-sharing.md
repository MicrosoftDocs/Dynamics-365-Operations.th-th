<span data-ttu-id="142e6-104">หัวข้อนี้อธิบายวิธีการเปิดใช้งานและใช้คุณลักษณะการใช้ร่วมกันข้ามช่องทางของตัวสร้างไซต์ Microsoft Dynamics 365 Commerce</span><span class="sxs-lookup"><span data-stu-id="142e6-104">This topic describes how to enable and use the cross-channel sharing feature of Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="142e6-105">การใช้ร่วมกันข้ามช่องทางช่วยให้ร้านค้าปลีกของร้านค้านำมาใช้ใหม่และแบ่งปันเนื้อหาระหว่างหลายช่องทางของไซต์</span><span class="sxs-lookup"><span data-stu-id="142e6-105">Cross-channel sharing lets retailers reuse and share content among multiple channels of a site.</span></span> <span data-ttu-id="142e6-106">ความสามารถนี้มีประโยชน์เมื่อช่องทางไซต์มีภาษาพื้นฐานที่เข้ากันได้ หรือเมื่อมีรายการเนื้อหาจำนวนมากเหมือนกัน</span><span class="sxs-lookup"><span data-stu-id="142e6-106">This capability is useful when the site channels have a compatible base language, or when they have numerous content items in common.</span></span>

## <a name="when-to-use-cross-channel-sharing"></a><span data-ttu-id="142e6-110">เมื่อใช้การใช้ร่วมกันข้ามช่องทาง</span><span class="sxs-lookup"><span data-stu-id="142e6-110">When to use cross-channel sharing</span></span>

<span data-ttu-id="142e6-111">การใช้ร่วมกันข้ามช่องทางมีประโยชน์เมื่อมีหลายช่องทางในไซต์เดียวสามารถใช้เนื้อหาร่วมกันได้</span><span class="sxs-lookup"><span data-stu-id="142e6-111">Cross-channel sharing is useful when multiple channels on a single site can share content.</span></span> <span data-ttu-id="142e6-112">ตัวอย่างเช่น ผู้ค้าปลีกที่มีหลายแบรนด์และหน้าร้านซึ่งจัดกลุ่มภายใต้ไซต์เดียวกันสามารถใช้เนื้อหาบางอย่างร่วมกันระหว่างบางหน้าร้านหรือทั้งหมด</span><span class="sxs-lookup"><span data-stu-id="142e6-112">For example, a retailer that has multiple brands and storefronts that are grouped under a single site can share some content among some or all of the storefronts.</span></span> <span data-ttu-id="142e6-113">เนื้อหาที่ใช้ร่วมกันนี้อาจรวมถึงหน้าสำหรับเงื่อนไขและข้อกำหนด เงื่อนไขการชำระเงิน วิธีการจัดส่ง และคำถามที่ถามบ่อย (FAQ)</span><span class="sxs-lookup"><span data-stu-id="142e6-113">This shared content can include pages for terms and conditions, payment terms, shipment methods, and frequently asked questions (FAQ).</span></span>

<span data-ttu-id="142e6-114">การใช้ร่วมกันข้ามช่องทางยังรองรับส่วนต่างๆ</span><span class="sxs-lookup"><span data-stu-id="142e6-114">Cross-channel sharing also supports fragments.</span></span> <span data-ttu-id="142e6-115">ดังนั้น หน้าเนื้อหาที่มีส่วนเฉพาะช่องทางสามารถสร้างเป็นเนื้อหาข้ามช่องทางได้</span><span class="sxs-lookup"><span data-stu-id="142e6-115">Therefore, a content page that contains channel-specific fragments can be created as cross-channel content.</span></span> <span data-ttu-id="142e6-116">ในกรณีนี้ ถึงแม้ว่าเนื้อหาส่วนใหญ่จะมีการใช้ร่วมกันระหว่างช่องทาง ส่วนเฉพาะช่องทางในหน้าข้ามช่องทางจะแสดงเฉพาะเมื่อมีการร้องขอจากช่องทางร้านที่สอดคล้องกันเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="142e6-116">In this case, although most of the content will be shared among channels, channel-specific fragments on a cross-channel page will be rendered only when they are requested from the corresponding storefront channel.</span></span>

<span data-ttu-id="142e6-117">ไซต์ที่มีหนึ่งช่องทาง หรือไซต์ที่มีหลายช่องทางที่ไม่สามารถใช้เนื้อหาร่วมกันได้ จะไม่ได้รับประโยชน์จากการใช้ร่วมกันข้ามช่องทาง</span><span class="sxs-lookup"><span data-stu-id="142e6-117">Sites that have only one channel, or sites that have multiple channels that can't share content, won't benefit from cross-channel sharing.</span></span>
