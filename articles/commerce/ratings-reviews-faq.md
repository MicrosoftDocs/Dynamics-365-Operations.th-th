# <a name="ratings-and-reviews-faq"></a>คำถามที่ถามบ่อยเกี่ยวกับการให้คะแนนและบทวิจารณ์

[!include [banner](includes/banner.md)]

หัวข้อนี้ให้คำตอบของคำถามที่ถามบ่อยเกี่ยวกับโซลูชันการให้คะแนนและบทวิจารณ์ Microsoft Dynamics 365 Commerce

### <a name="how-do-i-hide-the-ratings-refiner-that-appears-on-search-results-and-category-pages-if-my-e-commerce-site-doesnt-use-the-ratings-and-reviews-solution"></a>ฉันจะซ่อนตัวคัดสรรค์การให้คะแนนที่ปรากฏในผลการค้นหาและหน้าประเภทอย่างไร ถ้าไซต์อีคอมเมิร์ซของฉันไม่ได้ใช้โซลูชันการให้คะแนนและบทวิจารณ์ได้

หากต้องการทราบคําแนะนําเกี่ยวกับวิธีการซ่อนตัวคัดสรรค์การให้คะแนนบนหน้าผลการค้นหาและหน้าประเภทในช่องทางอีคอมเมิร์ซที่ไม่ได้ใช้โซลูชันการให้คะแนนและบทวิจารณ์ ให้ดูที่ [ตัวคัดสรรค์การให้คะแนนที่ปรากฏในผลการค้นหาและหน้าประเภทเมื่อไม่เปิดใช้งานโซลูชันการให้คะแนนและบทวิจารณ์](troubleshoot/hide-ratings-refiner.md)

### <a name="how-do-i-enforce-the-manual-publishing-of-ratings-and-reviews"></a>ฉันจะบังคับใช้การเผยแพร่การให้คะแนนและบทวิจารณ์ด้วยตนเองอย่างไร

สำหรับคําแนะนําเกี่ยวกับวิธีการเปิดใช้งานและบังคับใช้การเผยแพร่การให้คะแนนและบทวิจารณ์ด้วยตนเอง เพื่อหลีกเลี่ยงการเผยแพร่โดยอัตโนมัติ ให้ดูที่ [เปิดใช้งานการเผยแพร่การให้คะแนนและบทวิจารณ์ตนเองโดยผู้ดูแล](manual-publish-rating-reviews.md)
