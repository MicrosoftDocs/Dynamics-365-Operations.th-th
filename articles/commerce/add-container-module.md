[!include [banner](includes/banner.md)]

<span data-ttu-id="7302a-104">หัวข้อนี้ครอบคลุมถึงโมดูลคอนเทนเนอร์ และอธิบายวิธีการเพิ่มลงในเพจของไซต์ใน Microsoft Dynamics 365 Commerce</span><span class="sxs-lookup"><span data-stu-id="7302a-104">This topic covers container modules and describes how to add them to site pages in Microsoft Dynamics 365 Commerce.</span></span>

## <a name="overview"></a><span data-ttu-id="7302a-105">ภาพรวม</span><span class="sxs-lookup"><span data-stu-id="7302a-105">Overview</span></span>

<span data-ttu-id="7302a-106">โมดูลคอนเทนเนอร์เป็นโมดูลที่โฮสต์โมดูลอื่นๆไว้ภายใน</span><span class="sxs-lookup"><span data-stu-id="7302a-106">A container module is a module that hosts other modules inside it.</span></span> <span data-ttu-id="7302a-107">วัตถุประสงค์หลักของโมดูลคอนเทนเนอร์ คือการกำหนด โดยใช้คุณสมบัติที่ตั้งค่าไว้สำหรับโครงร่างของโมดูลที่อยู่ภายใน</span><span class="sxs-lookup"><span data-stu-id="7302a-107">The primary purpose of a container module is to define, through the properties that are set for it, the layout of the modules that it contains.</span></span> <span data-ttu-id="7302a-108">ตัวอย่าง เช่น โมดูลเหล่านี้อาจปรากฏอยู่เคียงข้างกันในโครงร่างแบบสองคอลัมน์ สามคอลัมน์ หรือหกคอลัมน์</span><span class="sxs-lookup"><span data-stu-id="7302a-108">For example, those modules can appear side by side in a two-column, three-column, four-column, or six-column layout.</span></span> <span data-ttu-id="7302a-109">นอกจากนี้ยังสามารถจำกัดความกว้างของคอนเทนเนอร์ได้ หรือสามารถให้เต็มหน้าจอ</span><span class="sxs-lookup"><span data-stu-id="7302a-109">They can also be limited to the width of the container, or they can fill the screen.</span></span> <span data-ttu-id="7302a-110">นอกจากนี้คุณยังสามารถเพิ่มส่วนหัวไปยังโมดูลคอนเทนเนอร์ทั้งหมดได้อีกด้วย</span><span class="sxs-lookup"><span data-stu-id="7302a-110">A heading can also be added to every container module.</span></span>

<span data-ttu-id="7302a-111">รองรับโมดูลคอนเทนเนอร์สามชนิด ได้แก่ คอนเทนเนอร์ คอนเทนเนอร์แบบ 2 ช่อง และคอนเทนเนอร์แบบ 3 ช่อง</span><span class="sxs-lookup"><span data-stu-id="7302a-111">Three container modules are supported: container, container with 2-slots, and container with 3-slots.</span></span> <span data-ttu-id="7302a-112">โมดูลชนิดใดสามารถบรรจุภายในคอนเทนเนอร์เหล่านี้ได้</span><span class="sxs-lookup"><span data-stu-id="7302a-112">Modules of any type can be put inside these containers.</span></span> 

> [!NOTE] 
> <span data-ttu-id="7302a-113">เราขอแนะนำให้คุณบรรจุโมดูลในโมดูลคอนเทนเนอร์เสมอ เพื่อให้สามารถจำกัดความกว้างของคอนเทนเนอร์ได้</span><span class="sxs-lookup"><span data-stu-id="7302a-113">We recommend that you always put modules inside a container module, so that they can be limited to the width of the container.</span></span>

<span data-ttu-id="7302a-121">รูปภาพต่อไปนี้แสดงตัวอย่างของโมดูลคอนเทนเนอร์ที่มีโมดูลวงล้อในตัวสร้างไซต์ Commerce</span><span class="sxs-lookup"><span data-stu-id="7302a-121">The following image shows an example of a container module that contains a carousel module in Commerce site builder.</span></span> <span data-ttu-id="7302a-122">ในตัวอย่างนี้คุณสมบัติ **ความกว้าง** ของคอนเทนเนอร์ถูกตั้งค่าเต็ม **เติมหน้าจอ**</span><span class="sxs-lookup"><span data-stu-id="7302a-122">In this example, the **Width** property of the container module is set to **Fill Screen**.</span></span>
