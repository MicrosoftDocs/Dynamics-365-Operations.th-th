[!include [banner](includes/banner.md)]

<span data-ttu-id="661c8-104">หัวข้อนี้ครอบคลุมถึงโมดูลคอนเทนเนอร์และอธิบายวิธีการเพิ่มลงในหน้าของไซต์ใน Microsoft Dynamics 365 Commerce</span><span class="sxs-lookup"><span data-stu-id="661c8-104">This topic covers container modules and describes how to add them to site pages in Microsoft Dynamics 365 Commerce.</span></span>

<span data-ttu-id="661c8-105">โมดูลคอนเทนเนอร์เป็นโมดูลที่โฮสต์โมดูลอื่นๆไว้ภายใน</span><span class="sxs-lookup"><span data-stu-id="661c8-105">A container module is a module that hosts other modules inside it.</span></span> <span data-ttu-id="661c8-106">วัตถุประสงค์หลักของโมดูลคอนเทนเนอร์ คือการกำหนด โดยใช้คุณสมบัติที่ตั้งค่าไว้สำหรับโครงร่างของโมดูลที่อยู่ภายใน</span><span class="sxs-lookup"><span data-stu-id="661c8-106">The primary purpose of a container module is to define, through the properties that are set for it, the layout of the modules that it contains.</span></span> <span data-ttu-id="661c8-107">ตัวอย่าง เช่น โมดูลเหล่านี้อาจปรากฏอยู่เคียงข้างกันในโครงร่างแบบสองคอลัมน์ สามคอลัมน์ หรือหกคอลัมน์</span><span class="sxs-lookup"><span data-stu-id="661c8-107">For example, those modules can appear side by side in a two-column, three-column, four-column, or six-column layout.</span></span> <span data-ttu-id="661c8-108">นอกจากนี้ยังสามารถจำกัดความกว้างของคอนเทนเนอร์ได้ หรือสามารถให้เต็มหน้าจอ</span><span class="sxs-lookup"><span data-stu-id="661c8-108">They can also be limited to the width of the container, or they can fill the screen.</span></span> <span data-ttu-id="661c8-109">นอกจากนี้คุณยังสามารถเพิ่มส่วนหัวไปยังโมดูลคอนเทนเนอร์ทั้งหมดได้อีกด้วย</span><span class="sxs-lookup"><span data-stu-id="661c8-109">A heading can also be added to every container module.</span></span>

<span data-ttu-id="661c8-110">รองรับโมดูลคอนเทนเนอร์สามชนิด ได้แก่ คอนเทนเนอร์ คอนเทนเนอร์แบบ 2 ช่อง และคอนเทนเนอร์แบบ 3 ช่อง</span><span class="sxs-lookup"><span data-stu-id="661c8-110">Three container modules are supported: container, container with 2-slots, and container with 3-slots.</span></span> <span data-ttu-id="661c8-111">โมดูลชนิดใดสามารถบรรจุภายในคอนเทนเนอร์เหล่านี้ได้</span><span class="sxs-lookup"><span data-stu-id="661c8-111">Modules of any type can be put inside these containers.</span></span>
