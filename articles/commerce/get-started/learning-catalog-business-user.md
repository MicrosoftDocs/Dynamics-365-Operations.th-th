# <a name="learning-catalog-for-users-of-dynamics-365-commerce"></a>แค็ตตาล็อกการเรียนรู้ของผู้ใช้ Dynamics 365 Commerce

คุณเพิ่งได้รับแอพลิเคชันใหม่เพื่อใช้ใช่หรือไม่

แค็ตตาล็อกต่อไปนี้มีการจัดระเบียบจากความรู้หลักให้กับโดเมนเฉพาะและจากการทำงานพื้นฐานถึงขั้นสูงที่สุด ถ้ามีเนื้อหาอยู่ในรูปแบบเราจะแจ้งให้คุณทราบเพื่อให้คุณสามารถเลือกรูปแบบการฝึกอบรมที่ตรงกับความต้องการของคุณได้ดีที่สุด

สำหรับการฝึกอบรมเกี่ยวกับลักษณะการทำงานที่ใช้ร่วมกันกับแอป Finance and Operations อื่นๆ โปรดดู [แค็ตตาล็อกการเรียนรู้ Dynamics 365 Finance](../../finance/get-started/learning-catalog-business-user.md)
