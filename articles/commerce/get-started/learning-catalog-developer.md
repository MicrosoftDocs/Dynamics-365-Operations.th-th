# <a name="learning-catalog-for-dynamics-365-commerce-developers"></a>แค็ตตาล็อกการเรียนรู้ของนักพัฒนา Dynamics 365 Commerce

คุณจำเป็นต้องเขียนรหัสเพื่อรวมกับแหล่งข้อมูลอื่นๆ ขยายการทำงานของระบบหลัก หรือสร้างแอปพลิเคชันที่ซับซ้อนหรือไม่

แค็ตตาล็อกต่อไปนี้มีการจัดระเบียบจากความรู้หลักให้กับโดเมนเฉพาะและจากการทำงานพื้นฐานถึงขั้นสูงที่สุด ถ้ามีเนื้อหาอยู่ในรูปแบบเราจะแจ้งให้คุณทราบเพื่อให้คุณสามารถเลือกรูปแบบการฝึกอบรมที่ตรงกับความต้องการของคุณได้ดีที่สุด

สำหรับการฝึกอบรมเกี่ยวกับลักษณะการทำงานที่ใช้ร่วมกันกับแอปการเงินและการดำเนินงานอื่นๆ โปรดดู [แค็ตตาล็อกการเรียนรู้ Dynamics 365 Finance](../../finance/get-started/learning-catalog-developer.md)
