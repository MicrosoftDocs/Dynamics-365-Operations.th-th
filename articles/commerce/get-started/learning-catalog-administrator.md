# <a name="learning-catalog-for-dynamics-365-commerce-administrators"></a>แค็ตตาล็อกการเรียนรู้ของผู้ดูแลระบบ Dynamics 365 Commerce

คุณจำเป็นต้องรักษาระบบและข้อมูลให้โฟลว์ มีการเตรียมใช้งาน และรักษาความปลอดภัยอย่างต่อเนื่องใช่หรือไม่

แค็ตตาล็อกต่อไปนี้มีการจัดระเบียบจากความรู้หลักให้กับโดเมนเฉพาะและจากการทำงานพื้นฐานถึงขั้นสูงที่สุด ถ้ามีเนื้อหาอยู่ในรูปแบบเราจะแจ้งให้คุณทราบเพื่อให้คุณสามารถเลือกรูปแบบการฝึกอบรมที่ตรงกับความต้องการของคุณได้ดีที่สุด แค็ตตาล็อกนี้คุณลักษณะเฉพาะของ Dynamics 365 Commerce สำหรับการฝึกอบรมเกี่ยวกับลักษณะการทำงานที่ใช้ร่วมกันกับแอป Finance and Operations อื่นๆ โปรดดู [แค็ตตาล็อกการเรียนรู้ Dynamics 365 Finance](../../finance/get-started/learning-catalog-administrator.md)
