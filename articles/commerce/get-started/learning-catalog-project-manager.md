# <a name="learning-catalog-for-implementation-project-managers-working-with-dynamics-365-commerce"></a>แค็ตตาล็อกการเรียนรู้ของผู้จัดการโครงการสำหรับการใช้งานที่ทำงานกับ Dynamics 365 Commerce

คุณรับผิดชอบการทำให้แน่ใจว่าการใช้งานของบริษัทของคุณเป็นไปอย่างราบรื่นใช่หรือไม่

แค็ตตาล็อกต่อไปนี้มีการจัดระเบียบจากความรู้หลักให้กับโดเมนเฉพาะและจากการทำงานพื้นฐานถึงขั้นสูงที่สุด ถ้ามีเนื้อหาอยู่ในรูปแบบเราจะแจ้งให้คุณทราบเพื่อให้คุณสามารถเลือกรูปแบบการฝึกอบรมที่ตรงกับความต้องการของคุณได้ดีที่สุด

สำหรับการฝึกอบรมเกี่ยวกับลักษณะการทำงานที่ใช้ร่วมกันกับแอปการเงินและการดำเนินงานอื่นๆ โปรดดู [แค็ตตาล็อกการเรียนรู้ Dynamics 365 Finance](../../finance/get-started/learning-catalog-project-manager.md)
