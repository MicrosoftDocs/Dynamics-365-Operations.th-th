# <a name="learning-catalog-for-commerce-microsoft-partner-sales-and-marketing-roles-for-dynamics-365-commerce"></a>แค็ตตาล็อกการเรียนรู้สำหรับบทบาทการขายและการตลาดของคู่ค้า Commerce Microsoft สำหรับ Dynamics 365 Commerce

คุณมีความรับผิดชอบในการช่วยให้ลูกค้าของคุณซื้อโซลูชันที่เหมาะสมหรือไม่

แค็ตตาล็อกต่อไปนี้มีการจัดระเบียบจากความรู้หลักให้กับโดเมนเฉพาะและจากการทำงานพื้นฐานถึงขั้นสูงที่สุด ถ้ามีเนื้อหาอยู่ในรูปแบบเราจะแจ้งให้คุณทราบเพื่อให้คุณสามารถเลือกรูปแบบการฝึกอบรมที่ตรงกับความต้องการของคุณได้ดีที่สุด

สำหรับการฝึกอบรมเกี่ยวกับลักษณะการทำงานที่ใช้ร่วมกันกับแอป Finance and Operations อื่นๆ โปรดดู [แค็ตตาล็อกการเรียนรู้ Dynamics 365 Finance](../../finance/get-started/learning-catalog-partner-sales.md)
