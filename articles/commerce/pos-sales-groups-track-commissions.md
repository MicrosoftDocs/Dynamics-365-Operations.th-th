# <a name="track-commissions-in-the-point-of-sale-pos-by-using-sales-groups"></a>ติดตามค่าคอมมิชชันในการขายหน้าร้าน (POS) โดยใช้กลุ่ม Sales

[!include [banner](includes/banner.md)]

ถือเป็นวิธีปฏิบัติการขายปลีกทั่วไปในการติดตามการขายโดยสมาคมที่ทำงานกับลูกค้าโดย—การให้ความช่วยเหลือ การขายสินค้าต่อเนื่อง การขายสินค้าชนิดอื่น และการดำเนินการธุรกรรม

การติดตามการขายโดยพนักงานขายเป็นการวัดความสามารถในการขายของสมาคม ในขณะที่การขายโดยพนักงานเก็บเงินเป็นการวัดความเร็วและประสิทธิภาพ นอกจากนี้การขายที่ถูกติดตามโดยพนักงานขายมักจะถูกใช้ในการคำนวณค่าคอมมิชชันหรือสิ่งจูงใจอื่น ๆ

## <a name="configuring-a-worker-to-be-a-sales-representative-in-pos"></a>การตั้งค่าคอนฟิกผู้ปฏิบัติงานเป็นพนักงานขายใน POS

เมื่อมีการเพิ่มผู้ปฏิบัติงานให้กับกลุ่มการขาย ผู้ปฏิบัติงานเหล่านั้นจะกลายเป็นมีสิทธิ์สำหรับค่าคอมมิชชัน และสามารถระบุเป็นพนักงานขายในระบบได้ ผู้ปฏิบัติงานที่ไม่ได้อยู่ในกลุ่มการขายจะไม่มีสิทธิ์สำหรับค่าคอมมิชชัน และจะไม่แสดงอยู่ในรายการเป็นพนักงานขายในแอพขายหน้าร้าน (POS) ใน POS รายชื่อของพนักงานขายได้รับมาจากกลุ่มการขายทั้งหมดที่ประกอบด้วยผู้ปฏิบัติงานอย่างน้อยหนึ่งคนที่กำหนดให้กับร้านค้า รายการที่แสดงใน POS เป็นการรวมกันของรหัสกลุ่มการขายและชื่อ (รหัส: ชื่อ) สามารถกำหนดกลุ่มการขายเริ่มต้นให้กับผู้ปฏิบัติงานเพื่อสนับสนุนสถานการณ์จำลองที่ผู้ค้าปลีกเลือกที่จะกำหนดพนักงานขายในบรรทัด POS โดยอัตโนมัติ ผู้ใช้สามารถเลือกจากกลุ่มการขายใด ๆ ที่ผู้ปฏิบัติงานเป็นสมาชิกอยู่

## <a name="displaying-the-sales-representative-information-on-the-pos-transactions-screen"></a>การแสดงข้อมูลพนักงานขายบนหน้าจอธุรกรรม POS

โครงร่างและเนื้อหาหน้าจอของธุรกรรม POS สามารถตั้งค่าคอนฟิกได้โดยใช้โปรแกรมออกแบบโครงร่างหน้าจอ และโครงร่างหน้าจอที่มีการกำหนดให้กับร้านค้า เครื่องบันทึกเงินสด หรือผู้ปฏิบัติงาน สามารถเพิ่มฟิลด์ **พนักงานขาย** ไปยังแท็บรายการของบานหน้าต่างใบเสร็จได้  นี่จะแสดงรหัสของกลุ่มการขายที่ระบุสำหรับแต่ละรายการบนหน้าจอของธุรกรรม

## <a name="adding-sales-representative-operations-to-pos-button-grids"></a>การเพิ่มการดำเนินงานของพนักงานงานไปยังกริดปุ่ม POS

POS อนุญาตให้ผู้ใช้สามารถตั้งค่าคอนฟิกกริดปุ่มได้ซึ่งรวมอยู่ในโครงร่างหน้าจอเพื่อให้การเข้าถึงไปยังการดำเนินงานของ POS สามารถกำหนดการดำเนินการของ POS ต่อไปนี้ไปยังปุ่มกริดปุ่มที่เกี่ยวข้องกับพนักงานขาย

| การดำเนินงาน                                 | คำอธิบาย |
|-------------------------------------------|-------------|
| ตั้งค่าพนักงานขายในรายการ          | การดำเนินการ POS นี้แสดงรายการของกลุ่มการขายที่เหมาะสม (รหัส: ชื่อ) สำหรับร้านค้า การเลือกกลุ่มการขายจากรายการนี้จะตั้งค่าบนรายการธุรกรรมปัจจุบัน |
| ล้างข้อมูลพนักงานขายในรายการ        | การดำเนินการของ POS นี้เป็นการเอาค่ากลุ่มการขายปัจจุบันออกจากรายการธุรกรรมปัจจุบัน |
| ตั้งค่าพนักงานขายในธุรกรรม   | การดำเนินการ POS นี้แสดงรายการของกลุ่มการขายที่เหมาะสม (รหัส: ชื่อ) สำหรับร้านค้า การเลือกกลุ่มการขายจากรายการนี้จะตั้งค่าเริ่มต้นบนธุรกรรมปัจจุบัน รายการที่มีอยู่ใดๆ โดยไม่มีการกำหนดกลุ่มการขายจะถูกตั้งค่า ตลอดจนรายการใดๆ ที่ถูกเพิ่มในเวลาต่อมา |
| ล้างข้อมูลพนักงานขายในธุรกรรม | การดำเนินการของ POS นี้เป็นการเอาค่ากลุ่มการขายเริ่มต้นปัจจุบันออกจากธุรกรรมปัจจุบัน ซึ่งจะไม่มีผลกระทบต่อรายการใดๆ ที่มีอยู่ในธุรกรรม |

## <a name="calculating-commissions"></a>การคำนวณค่าคอมมิชชัน

ค่าคอมมิชชันจะถูกคำนวณสำหรับผู้ปฏิบัติงานในกลุ่มการขายที่ระบุ ณ เวลาที่ลงรายการบัญชีใบแจ้งยอดหรือลงรายการบัญชีใบสั่งขาย ยอดค่าคอมมิชชันจะถูกกำหนดตามส่วนแบ่งค่าคอมมิชชันของผู้ปฏิบัติงาน ตามที่กำหนดไว้ในกลุ่มการขายและการตั้งค่าการคำนวณค่าคอมมิชชันที่เกี่ยวข้องสำหรับลูกค้าและ/หรือผลิตภัณฑ์บนธุรกรรม
