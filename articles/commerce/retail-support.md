# <a name="get-support-for-dynamics-365-retail"></a>รับการสนับสนุนสำหรับ Dynamics 365 Retail

[!include [banner](../includes/banner.md)]

บทความนี้อธิบายวิธีการรับวิธีใช้ด้วย Dynamics 365 Commerce

## <a name="contact-support"></a>ติดต่อฝ่ายสนับสนุน

ใช้ข้อมูลในบทความนี้เพื่อติดต่อ Microsoft ถ้าคุณมีคำถามทางเทคนิคเกี่ยวกับการค้า

### <a name="commerce-support"></a>การสนับสนุนการค้า

เพื่อรับการสนับสนุนสำหรับการค้า ใช้ทรัพยากรในตารางต่อไปนี้ ทรัพยากรเหล่านี้จะแสดงรายการตามลำดับที่เราแนะนำให้คุณปฏิบัติตามเมื่อคุณกำลังแก้ไขปัญหา
