# <a name="add-product-recommendation-lists-to-e-commerce-pages"></a>เพิ่มรายการคำแนะนำผลิตภัณฑ์ลงในหน้าอีคอมเมิร์ซ

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

หัวข้อนี้อธิบายถึงวิธีการเพิ่มรายการคำแนะนำผลิตภัณฑ์ลงในเพจอีคอมเมิร์ซ

1. จากเพจการสร้างอีคอมเมิร์ซที่มีอยู่ ให้เลือก **เพิ่มโมดูล**
2. ค้นหาโมดูล **การรวบรวมผลิตภัณฑ์**
3. ในวิซาร์ดโมดูล ให้เลือกรายการที่จะเพิ่มลงในเพจ
4. บันทึกการเปลี่ยนแปลง

เมื่อต้องการเรียนรู้เพิ่มเติมเกี่ยวกับชนิดของรายการที่คุณสามารถเพิ่มในเพจได้ ดูที่ [ภาพรวมของคำแนะนำของผลิตภัณฑ์](product-recommendations.md)
