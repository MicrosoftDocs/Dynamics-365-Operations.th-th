[!include [banner](includes/banner.md)]

<span data-ttu-id="fca2b-104">หัวข้อนี้แสดงภาพรวมของโฮมเพจใน Microsoft Dynamics 365 Commerce</span><span class="sxs-lookup"><span data-stu-id="fca2b-104">This topic provides an overview of the home page in Microsoft Dynamics 365 Commerce.</span></span>

<span data-ttu-id="fca2b-105">โฮมเพจจะเป็นหน้าเริ่มต้นที่นักช้อปไปเมื่อเข้าไปที่ไซต์อีคอมเมิร์ซ</span><span class="sxs-lookup"><span data-stu-id="fca2b-105">The home page is the default page that shoppers go to when they visit an e-Commerce site.</span></span> <span data-ttu-id="fca2b-106">โดยทั่วไปหน้านี้แสดงผลิตภัณฑ์และการส่งเสริมการขายโดยใช้ชุดของโมดูลการตลาด</span><span class="sxs-lookup"><span data-stu-id="fca2b-106">Typically, this page showcases products and promotions by using a combination of marketing modules.</span></span> <span data-ttu-id="fca2b-107">โฮมเพจควรจะเต็มไปด้วยรูปภาพและข้อความที่จะทำให้นักช้อปมีส่วนร่วมอย่างเต็มที่</span><span class="sxs-lookup"><span data-stu-id="fca2b-107">The home page should be rich with images and text to keep shoppers engaged.</span></span>

<span data-ttu-id="fca2b-108">ภาพประกอบต่อไปนี้แสดงตัวอย่างของหน้าหลักที่สร้างขึ้นโดยใช้ไลบรารีโมดูลและธีม "Fabrikam"</span><span class="sxs-lookup"><span data-stu-id="fca2b-108">The following illustration shows an example of a home page that was built by using the module library and the "Fabrikam" theme.</span></span>

![ตัวอย่างของโฮมเพจ](./media/Homepage2.PNG)

<span data-ttu-id="fca2b-110">ด้านบนสุดของโฮมเพจ มีส่วนหัวที่แสดงประเภทผลิตภัณฑ์ทั้งหมด และหน้าอื่นๆที่ผู้ค้าปลีกต้องการให้ลูกค้าเรียกดู</span><span class="sxs-lookup"><span data-stu-id="fca2b-110">The top of the home page has a header that shows all the product categories and other pages that the retailer wants customers to browse.</span></span> <span data-ttu-id="fca2b-111">ด้านล่างของโฮมเพจ มีส่วนท้ายที่มีลิงก์ลัดไปยังหัวข้อต่าง ๆ ที่อาจทำให้ลูกค้าสนใจ</span><span class="sxs-lookup"><span data-stu-id="fca2b-111">The bottom of the home page has a footer that contains quick links to various topics that might interest customers.</span></span>

<span data-ttu-id="fca2b-112">ส่วนหลักของโฮมเพจจะสามารถเน้นผลิตภัณฑ์ ประเภท หรือโปรโมชัน โดยใช้โมดูล Dynamics 365 Commerce ต่าง ๆ ดังนี้:</span><span class="sxs-lookup"><span data-stu-id="fca2b-112">The main section of the home page can highlight products, categories, or promotions by using various Dynamics 365 Commerce modules:</span></span>

- <span data-ttu-id="fca2b-113">**ฮีโร่** – โดยทั่วไปรายการแรกที่ด้านบนของส่วนหลักจะแสดงภาพ "ฮีโร่" อย่างน้อยหนึ่งภาพที่เน้นผลิตภัณฑ์และโปรโมชั่นใหม่ในร้านค้า</span><span class="sxs-lookup"><span data-stu-id="fca2b-113">**Hero** – Typically, the first item at the top of the main section shows one or more "hero" images that highlight new products and promotions in the store.</span></span> <span data-ttu-id="fca2b-114">ถ้ามีรูปภาพฮีโร่หลายรูป จะโฮสต์อยู่ในโมดูลแบบหมุนเพื่อให้ผู้ใช้สามารถเรียกดูได้</span><span class="sxs-lookup"><span data-stu-id="fca2b-114">If there are multiple hero images, they are hosted in a carousel module so that users can browse them.</span></span>

    <span data-ttu-id="fca2b-115">ภาพประกอบต่อไปนี้แสดงตัวอย่างของโฮมเพจ ที่มีรายการแรกในส่วนหลักเป็นโครงร่างฮีโร่ของโมดูลบล็อคเนื้อหาที่มีชื่อว่า "มาใหม่"</span><span class="sxs-lookup"><span data-stu-id="fca2b-115">The following illustration shows an example of a home page where the first item in the main section is a hero layout of a content block module that is named "New Arrival."</span></span>

    ![ตัวอย่างของโมดูลฮีโร่](./media/Hero.PNG)

- <span data-ttu-id="fca2b-117">**คุณลักษณะ** – โครงร่างคุณลักษณะของโมดูลบล็อคเนื้อหาใช้ในการทำตลาดผลิตภัณฑ์ หรือโปรโมชันผ่านการรวมกันของรูปภาพและข้อความ</span><span class="sxs-lookup"><span data-stu-id="fca2b-117">**Feature** – A feature layout of a content block module is used to market products or promotions by using a combination of images and text.</span></span> <span data-ttu-id="fca2b-118">โครงร่างคุณลักษณะการทำงานที่สามารถใช้ได้โดยอิสระ หรือโฮสต์เหล่านี้จะอยู่ในโมดูลแบบหมุน</span><span class="sxs-lookup"><span data-stu-id="fca2b-118">Features layouts can be used independently, or they can be hosted in a carousel module.</span></span>

    <span data-ttu-id="fca2b-119">ภาพต่อไปนี้แสดงตัวอย่างของโครงร่างคุณลักษณะของโมดูลบล็อคเนื้อหาบนโฮมเพจ</span><span class="sxs-lookup"><span data-stu-id="fca2b-119">The following illustration shows an example of feature layout of a content block module on a home page.</span></span>

    ![ตัวอย่างของโมดูลของลักษณะการทำงาน](./media/Feature.PNG)

- <span data-ttu-id="fca2b-121">**ไทล์** – โครงร่างไทล์ของโมดูลบล็อคเนื้อหาใช้เพื่อแสดงผลิตภัณฑ์หรือประเภทของผลิตภัณฑ์หลายรายการ โดยใช้การรวมกันของรูปภาพและข้อความในรูปแบบคอลัมน์</span><span class="sxs-lookup"><span data-stu-id="fca2b-121">**Tile** – A tile layout of a content block module is used to showcase multiple products or category of products by using a combination of images and text in a multicolumn layout.</span></span> <span data-ttu-id="fca2b-122">ในภาพประกอบของโฮมเพจที่แสดงไว้ก่อนหน้านี้ในหัวข้อนี้ จะมีการใช้โครงร่างไทล์สำหรับการแสดงสามคอลัมน์ของ **ร้านค้าผู้หญิง** **ร้านค้าผู้ชาย** **ร้านค้าเครื่องประดับ**</span><span class="sxs-lookup"><span data-stu-id="fca2b-122">In the illustration of a home page that appears earlier in this topic, a tile  layout is used for the three-column rendering of the **Shop Women**, **Shop Men**, and **Shop Accessories** items.</span></span>
- <span data-ttu-id="fca2b-123">**เครื่องเล่นวิดีโอ** – โมดูลเครื่องเล่นวิดีโอสามารถใช้ในการแสดงเนื้อหาวิดีโอบนโฮมเพจ</span><span class="sxs-lookup"><span data-stu-id="fca2b-123">**Video player** – A video player module can be used to showcase video content on the home page.</span></span> <span data-ttu-id="fca2b-124">ภาพประกอบของโฮมเพจที่แสดงไว้ก่อนหน้านี้ในหัวข้อนี้รวมถึงโมดูลของโปรแกรมเล่นวิดีโอ</span><span class="sxs-lookup"><span data-stu-id="fca2b-124">The illustration of a home page that appears earlier in this topic includes a video player module.</span></span>
- <span data-ttu-id="fca2b-125">**บล็อกข้อความ** – โมดูลบล็อกเนื้อหาสามารถใช้ในการนำเสนอเนื้อหาของข้อความ บนโฮมเพจในโครงร่างคอลัมน์เดียวหรือหลายคอลัมน์</span><span class="sxs-lookup"><span data-stu-id="fca2b-125">**Text block** – A content rich block module can be used to present text content on the home page in a single-column or multicolumn layout.</span></span>
- <span data-ttu-id="fca2b-126">**คำแนะนำผลิตภัณฑ์** – มีการใช้โมดูลคำแนะนำผลิตภัณฑ์เพื่อแสดงรายการ เช่น **ใหม่** **ยอดนิยม** และ **ขายดี** บนโฮมเพจ</span><span class="sxs-lookup"><span data-stu-id="fca2b-126">**Product recommendations** – Product recommendations modules are used to show lists, such as **New**, **Trending**, and **Best Selling** on the home page.</span></span> <span data-ttu-id="fca2b-127">รายชื่อเหล่านี้แสดงผลิตภัณฑ์ตามแนวโน้มของการซื้อของ และสามารถสร้างโดยใช้อัลกอริทึมหรือจัดทำด้วยตนเองได้</span><span class="sxs-lookup"><span data-stu-id="fca2b-127">These lists showcase products based on shopping trends, and they can be algorithmically generated or manually curated.</span></span> <span data-ttu-id="fca2b-128">ผู้ใช้จะช่วยให้ลูกค้าค้นหาผลิตภัณฑ์ยอดนิยมอย่างรวดเร็วและทำการซื้อต่อไป</span><span class="sxs-lookup"><span data-stu-id="fca2b-128">They help customers quickly discover top products and then continue to shop.</span></span>

    <span data-ttu-id="fca2b-129">ภาพประกอบต่อไปนี้แสดงตัวอย่างของโมดูลคำแนะนำผลิตภัณฑ์บนโฮมเพจ</span><span class="sxs-lookup"><span data-stu-id="fca2b-129">The following illustration shows an example of product recommendations modules on a home page.</span></span>

    ![ตัวอย่างของโมดูลคำแนะนำผลิตภัณฑ์](./media/Recommendations.PNG)

> <span data-ttu-id="fca2b-131">โมดูลทั้งหมดที่แสดงรายการไว้ที่นี่สามารถใช้ได้บนหน้าไซต์ใด ๆ</span><span class="sxs-lookup"><span data-stu-id="fca2b-131">All the modules that are listed here can be used on any site page.</span></span> <span data-ttu-id="fca2b-132">อย่างไรก็ตามการจัดวางบนโฮมเพจมีความสำคัญเพราะหน้าดังกล่าวเป็นที่ที่ลูกค้าโต้ตอบกับไซต์ของคุณเป็นครั้งแรก</span><span class="sxs-lookup"><span data-stu-id="fca2b-132">However, their placement on the home page is important because that page is where customers first interact with your site.</span></span>
