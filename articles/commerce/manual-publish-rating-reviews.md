# <a name="enable-manual-publishing-of-ratings-and-reviews-by-a-moderator"></a>เปิดใช้งานการเผยแพร่การให้คะแนนและบทวิจารณ์ด้วยตนเองโดยผู้ดูแล

[!include [banner](includes/banner.md)]

บทความนี้จะอธิบายวิธีการเปิดใช้งานการเผยแพร่การให้คะแนนและบทวิจารณ์ด้วยตนเองโดยผู้ดูแลใน Microsoft Dynamics 365 Commerce และวิธีการเผยแพร่การให้คะแนนและบทวิจารณ์ด้วยตนเอง

โซลูชันการให้คะแนนและบทวิจารณ์ Dynamics 365 Commerce จะใช้ Azure Cognitive Services เพื่อแก้ไขคำสบถในชื่อและเนื้อหาบทวิจารณ์โดยอัตโนมัติ และเพื่อเผยแพร่การให้คะแนนและบทวิจารณ์ ดังนั้น จึงไม่ต้องใช้การขัดจังหวะด้วยตนเองเพื่อตรวจทานและเผยแพร่การให้คะแนนและบทวิจารณ์ไปยังไซต์อีคอมเมิร์ซของคุณ

อย่างไรก็ตาม บางธุรกิจอาจต้องการให้ผู้ดูแลตรวจทานและอนุมัติบทวิจารณ์ด้วยตนเองก่อนที่จะเผยแพร่ หากต้องการเปิดใช้งานการเผยแพร่การให้คะแนนและบทวิจารณ์ด้วยตนเองโดยผู้ดูแล คุณต้องเปิดใช้งานคุณลักษณะ **ต้องการผู้ดูแลสำหรับการให้คะแนนและบทวิจารณ์** ในโปรแกรมสร้างไซต์ Commerce

## <a name="enable-the-require-moderator-for-ratings-and-reviews-feature-in-commerce-site-builder"></a>เปิดใช้งานคุณลักษณะ ต้องการผู้ดูแลสำหรับการให้คะแนนและบทวิจารณ์ ในโปรแกรมสร้างไซต์ Commerce

หากต้องการเปิดใช้งานคุณลักษณะ **ต้องการผู้ดูแลสำหรับการให้คะแนนและบทวิจารณ์** ในโปรแกรมสร้างไซต์ Commerce ให้ปฏิบัติตามขั้นตอนต่อไปนี้

1. ไปที่ **หน้าแรก \> ความคิดเห็น \> การตั้งค่า**
1. ตั้งค่าตัวเลือก **ต้องการผู้ดูแลสำหรับการให้คะแนนและบทวิจารณ์** เป็น **เปิด**

> [!NOTE]
> โดยการเปิดใช้คุณลักษณะ **ต้องการผู้ดูแลสำหรับการให้คะแนนและบทวิจารณ์** คุณหยุดการเผยแพร่การให้คะแนนและบทวิจารณ์โดยอัตโนมัติ เพื่อให้ขณะนี้ต้องใช้การเผยแพร่ด้วยตนเอง แต่ Azure Cognitive Services จะยังคงแสดงชื่อและเนื้อหาบทวิจารณ์ใหม่ต่อไป

## <a name="publish-ratings-and-reviews"></a>เผยแพร่การให้คะแนนและบทวิจารณ์

เมื่อคุณเปิดใช้งานคุณลักษณะ **ต้องการผู้ดูแลสำหรับการให้คะแนนและบทวิจารณ์** ผู้ดูแลต้องตรวจสอบและเผยแพร่การให้คะแนนและบทวิจารณ์ด้วยตนเองเพื่อให้ปรากฎในไซต์อีคอมเมิร์ซของคุณ

เมื่อต้องการตรวจสอบและเผยแพร่การให้คะแนนและบทวิจารณ์ในโปรแกรมสร้างไซต์ Commerce ให้ทำตามขั้นตอนต่อไปนี้

1. ไปที่ **ความคิดเห็น \> การตรวจสอบ**
1. ในกริด ถ้าฟิลด์ **สถานะ** สำหรับแถวถูกตั้งเป็น **ยังไม่ได้เผยแพร่** การให้คะแนนและบทวิจารณ์ในแถวนั้นจะยังไม่ได้รับการเผยแพร่ เมื่อต้องการดูเฉพาะการให้คะแนนและบทวิจารณ์ที่ไม่ได้เผยแพร่ ให้เลือก **สถานะ: ต้องการบทวิจารณ์** ในตัวกรองสถานะเหนือกริด
1. เลือกการให้คะแนนและบทวิจารณ์อย่างน้อยหนึ่งรายการที่มีสถานะ **ยังไม่ได้เผยแพร่** แล้วเลือก **เผยแพร่** บนแถบใบสั่ง การให้คะแนนและบทวิจารณ์ที่เลือกจะถูกเพิ่มในคิวการเผยแพร่ และจะปรากฏบนไซต์อีคอมเมิร์ซหลังจากที่เผยแพร่แล้ว

> [!NOTE]
> - หลังจากที่เผยแพร่การให้คะแนนและบทวิจารณ์แล้ว ค่าสถานะจะเปลี่ยนจาก **ยังไม่ได้เผยแพร่** เป็นค่า null (ฟิลด์ว่างเปล่า)
> - หากคุณเลือกการให้คะแนนและบทวิจารณ์หลายรายการที่มีสถานะผสม แล้วเลือก **เผยแพร่** การให้คะแนนและบทวิจารณ์ที่ยังไม่ได้เผยแพร่จะยังคง ได้รับการเผยแพร่ แต่การให้คะแนนและบทวิจารณ์ที่ได้รับการเผยแพร่แล้วจะไม่ได้รับการเผยแพร่อีกครั้ง

ในภาพต่อไปนี้จะแสดงตัวอย่างที่มีการเลือกการให้คะแนนและบทวิจารณ์ที่ไม่ได้เผยแพร่สามรายการในหน้า **การตรวจสอบ** ในตัวสร้างไซต์ Commerce
