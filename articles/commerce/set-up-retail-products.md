# <a name="set-up-retail-products"></a><span data-ttu-id="fb67d-103">ตั้งค่าผลิตภัณฑ์ขายปลีก</span><span class="sxs-lookup"><span data-stu-id="fb67d-103">Set up retail products</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="fb67d-104">บทความนี้อธิบายวิธีการตั้งค่าผลิตภัณฑ์ใน Dynamics 365 Commerce</span><span class="sxs-lookup"><span data-stu-id="fb67d-104">This article describes how to set up products in Dynamics 365 Commerce.</span></span>

<span data-ttu-id="fb67d-105">ก่อนที่คุณสามารถเสนอผลิตภัณฑ์สำหรับการขายต่อสินค้าในช่องทางการค้าของคุณ คุณต้องสร้างและตั้งค่าคอนฟิกผลิตภัณฑ์</span><span class="sxs-lookup"><span data-stu-id="fb67d-105">Before you can offer products for resale in your commerce channels, you must create and configure the products.</span></span> <span data-ttu-id="fb67d-106">Commerce สร้างผลิตภัณฑ์ทั่วทั้งองค์กรในผลิตภัณฑ์หลัก</span><span class="sxs-lookup"><span data-stu-id="fb67d-106">Commerce creates organization-wide products in the product master.</span></span> <span data-ttu-id="fb67d-107">คุณสามารถสร้างผลิตภัณฑ์ กำหนดคุณสมบัติของผลิตภัณฑ์และแอททริบิวต์ และกำหนดผลิตภัณฑ์ให้กับการจัดประเภทการค้า</span><span class="sxs-lookup"><span data-stu-id="fb67d-107">You can create the products, define the product properties and attributes, and assign the products to commerce category hierarchies.</span></span> <span data-ttu-id="fb67d-108">ในการทำให้ผลิตภัณฑ์พร้อมใช้งานสำหรับช่องทางของคุณ และเพิ่มลงในการจัดประเภทที่ใช้งานอยู่ คุณต้องนำผลิตภัณฑ์ออกใช้ไปยังนิติบุคคลที่พร้อมใช้งาน</span><span class="sxs-lookup"><span data-stu-id="fb67d-108">To make the products available to your channels and add them to an active assortment, you must release the products to the legal entities where they are available.</span></span> <span data-ttu-id="fb67d-109">ในการตั้งค่าผลิตภัณฑ์ที่คุณขายโดยใช้ช่องทาง ให้ดำเนินการงานต่อไปนี้ให้เสร็จสมบูรณ์</span><span class="sxs-lookup"><span data-stu-id="fb67d-109">To set up the products that you sell by using channels, complete the following tasks.</span></span>

1. <span data-ttu-id="fb67d-110">**กำหนดลำดับชั้นของผลิตภัณฑ์**</span><span class="sxs-lookup"><span data-stu-id="fb67d-110">**Define a product hierarchy.**</span></span> <span data-ttu-id="fb67d-111">โดยการใช้คุณลักษณะลำดับชั้นประเภทใน Commerce คุณสามารถกำหนดลำดับชั้นประเภทเพื่อจัดกลุ่มและจัดประเภทผลิตภัณฑ์ที่คุณกระจายไปยังช่องทางของคุณ</span><span class="sxs-lookup"><span data-stu-id="fb67d-111">By using the category hierarchy features in Commerce, you can define category hierarchies to group and categorize the products that you distribute to your channels.</span></span> <span data-ttu-id="fb67d-112">กำหนดโดยผู้ใช้และการแอตทริบิวต์ระบบสามารถกำหนดในระดับประเภท</span><span class="sxs-lookup"><span data-stu-id="fb67d-112">User-defined and system attributes can be defined at the category level.</span></span> <span data-ttu-id="fb67d-113">จากนั้น ผลิตภัณฑ์ทั้งหมดที่ถูกกำหนดให้สืบทอดประเภทแอททริบิวต์เหล่านั้น</span><span class="sxs-lookup"><span data-stu-id="fb67d-113">Then, all products that are assigned to the category inherit those attributes.</span></span> <span data-ttu-id="fb67d-114">การจัดประเภทตามลำดับชั้นหลายรายการสามารถกำหนดขึ้นได้ และแต่ละผลิตภัณฑ์จะสามารถกำหนดให้กับหลายลำดับชั้น</span><span class="sxs-lookup"><span data-stu-id="fb67d-114">Multiple category hierarchies can be defined, and each product can be assigned to multiple hierarchies.</span></span> <span data-ttu-id="fb67d-115">อย่างไรก็ตาม ในลำดับชั้นเดียว แต่ละผลิตภัณฑ์สามารถกำหนดให้กับประเภทเดียวเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="fb67d-115">However, in a single hierarchy, each product can be assigned to only one category.</span></span>
2. <span data-ttu-id="fb67d-116">**เพิ่มผลิตภัณฑ์และผลิตภัณฑ์ย่อยลงในผลิตภัณฑ์หลัก**</span><span class="sxs-lookup"><span data-stu-id="fb67d-116">**Add products and product variants to the product master.**</span></span> <span data-ttu-id="fb67d-117">ผลิตภัณฑ์ที่ถูกเพิ่มในผลิตภัณฑ์หลักแสดงรายการสากลของผลิตภัณฑ์</span><span class="sxs-lookup"><span data-stu-id="fb67d-117">Products that are added to the product master represent a global list of products.</span></span> <span data-ttu-id="fb67d-118">คุณสามารถเพิ่มผลิตภัณฑ์ได้ด้วยตนเอง ครั้งละหนึ่งรายการ หรือคุณสามารถนำเข้าข้อมูลผลิตภัณฑ์จากผู้จัดจำหน่ายของคุณ</span><span class="sxs-lookup"><span data-stu-id="fb67d-118">You can add products manually, one at a time, or you can import product data from your vendors.</span></span>
3. <span data-ttu-id="fb67d-119">**นำผลิตภัณฑ์ออกใช้ที่นิติบุคคล**</span><span class="sxs-lookup"><span data-stu-id="fb67d-119">**Release the products to legal entities.**</span></span> <span data-ttu-id="fb67d-120">สามารถทำให้ผลิตภัณฑ์ที่ถูกนำออกใช้ไปยังนิติบุคคลพร้อมใช้งานสำหรับช่องทางของคุณได้เท่านั้น</span><span class="sxs-lookup"><span data-stu-id="fb67d-120">Only products that have been released to legal entities can be made available to your channels.</span></span> <span data-ttu-id="fb67d-121">เมื่อคุณกำหนดผลิตภัณฑ์ครั้งแรก คุณกำหนดได้ในระดับทั่วทั้งองค์กร</span><span class="sxs-lookup"><span data-stu-id="fb67d-121">When you first define a product, you define it on an organization-wide level.</span></span> <span data-ttu-id="fb67d-122">จากนั้นคุณสามารถเลือกหนึ่งนิติบุคคลหรือมากกว่านั้นให้นำผลิตภัณฑ์ออกใช้</span><span class="sxs-lookup"><span data-stu-id="fb67d-122">You can then select one or more legal entities to release the product to.</span></span> <span data-ttu-id="fb67d-123">จากนั้น ผลิตภัณฑ์จะพร้อมใช้งานสำหรับช่องทางที่หลากหลายทั่วทั้งองค์กรของคุณ</span><span class="sxs-lookup"><span data-stu-id="fb67d-123">The product then becomes available to multiple channels across your organization.</span></span> <span data-ttu-id="fb67d-124">คุณสามารถใช้ฟังก์ชันนี้เพื่อสร้างผลิตภัณฑ์หนึ่งครั้ง เพิ่มและอัพเดตคุณลักษณะของผลิตภัณฑ์และคุณสมบัติในที่หนึ่ง และจากนั้น แจกจ่ายผลิตภัณฑ์ทั่วทั้งองค์กรของคุณไปยังช่องทางที่พร้อมใช้งาน</span><span class="sxs-lookup"><span data-stu-id="fb67d-124">You can use this functionality to create a product one time, add and update product attributes and properties in one place, and then distribute the product across your organization, to the channels where it's available.</span></span>

5. <span data-ttu-id="fb67d-131">**เพิ่มผลิตภัณฑ์ไปยังลำดับชั้นการนำทาง**</span><span class="sxs-lookup"><span data-stu-id="fb67d-131">**Add products to navigation hierarchies.**</span></span> <span data-ttu-id="fb67d-132">ก่อนที่ผลิตภัณฑ์จะสามารถถูกเรียกดูแบบออนไลน์หรือแบบขายหน้าร้าน (POS) ต้องมีการจัดประเภทในลำดับชั้นการนำทาง Commerce</span><span class="sxs-lookup"><span data-stu-id="fb67d-132">Before products can be browsed online or in point of sale (POS), they must be categorized in a Commerce navigation hierarchy.</span></span>
