## <a name="serial-number-configurations"></a><span data-ttu-id="919cb-109">การตั้งค่าคอนฟิกหมายเลขลำดับประจำสินค้า</span><span class="sxs-lookup"><span data-stu-id="919cb-109">Serial number configurations</span></span>

<span data-ttu-id="919cb-110">สินค้าจะถือเป็นสินค้าที่กำหนดหมายเลขลำดับประจำสินค้า ถ้าถูกกำหนดกลุ่มมิติการติดตามที่ตั้งค่าเพื่อให้ใช้หมายเลขลำดับประจำสินค้าได้</span><span class="sxs-lookup"><span data-stu-id="919cb-110">An item is considered a serialized item if it's assigned a tracking dimension group that is set up to allow for serial numbers.</span></span> <span data-ttu-id="919cb-111">ในศูนย์ควบคุม Commerce บนหน้า **กลุ่มมิติการติดตาม** เลือกตัวเลือก **ใช้งานอยู่** เพื่อเปิดใช้งานหมายเลขลำดับประจำสินค้าสำหรับกระบวนการสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="919cb-111">In Commerce headquarters, on the **Tracking dimension groups** page, select the **Active** option to enable serial numbers for the inventory process.</span></span> <span data-ttu-id="919cb-112">เมื่อต้องการเปิดใช้งานหมายเลขลําดับประจําสินค้าสําหรับกระบวนการขาย ให้เลือกตัวเลือก **ใช้งานอยู่ในกระบวนการขาย**</span><span class="sxs-lookup"><span data-stu-id="919cb-112">To enable serial numbers for the sales process, select the **Active in sales process** option.</span></span>

<span data-ttu-id="919cb-113">บนแท็บด่วน **มิติการติดตาม** ถ้าตัวเลือก **การรับสินค้าที่ว่างเปล่า** เปิดอยู่ หมายเลขลําดับประจําสินค้าเป็นอินพุตที่ไม่จำเป็นต้องระบุระหว่างกระบวนการรับสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="919cb-113">On the **Tracking dimensions** FastTab, if the **Blank receipt allowed** option is turned on, the serial number is an optional input during the inventory receipt process.</span></span> <span data-ttu-id="919cb-114">ถ้าตัวเลือกถูกปิดอยู่ หมายเลขลําดับประจําสินค้าจำเป็นต้องระบุ</span><span class="sxs-lookup"><span data-stu-id="919cb-114">If the option is turned off, the serial number is required.</span></span>

<span data-ttu-id="919cb-115">บนแท็บด่วน **หมายเลขลำดับประจำสินค้า** ถ้าตัวเลือก **การควบคุมหมายเลขลำดับประจำสินค้า** เปิดอยู่ หมายเลขลำดับประจำสินค้าต้องไม่ซ้ำกันต่อหน่วย</span><span class="sxs-lookup"><span data-stu-id="919cb-115">On the **Serial numbers** FastTab, if the **Serial number control** option is turned on, the serial number must be unique per unit.</span></span> <span data-ttu-id="919cb-116">กล่าวอีกนัยหนึ่ง หมายเลขลําดับประจําสินค้าซ้ํากันไม่ได้รับอนุญาต</span><span class="sxs-lookup"><span data-stu-id="919cb-116">In other words, duplicated serial numbers aren't allowed.</span></span>

## <a name="serialized-items-in-the-receiving-process"></a><span data-ttu-id="919cb-117">สินค้าที่กำหนดหมายเลขลำดับประจำสินค้าในกระบวนการรับสินค้า</span><span class="sxs-lookup"><span data-stu-id="919cb-117">Serialized items in the receiving process</span></span>

<span data-ttu-id="919cb-122">เมื่อต้องการเริ่มต้นกระบวนการรับสินค้า ในการดำเนินการ **สินค้าคงคลังขาเข้า** คุณสามารถเริ่มในมุมมอง **การรับตอนนี้** โดยการสแกนบาร์โคดของสินค้า หรือการป้อนรหัสสินค้า</span><span class="sxs-lookup"><span data-stu-id="919cb-122">To begin the receiving process, in the **Inbound inventory** operation, you can start in the **Receiving now** view by scanning the item bar code or entering the item ID.</span></span> <span data-ttu-id="919cb-123">อีกวิธีหนึ่งคือ คุณสามารถเริ่มต้นในมุมมอง **รายการใบสั่งทั้งหมด** โดยการเลือกสินค้าด้วยตนเอง</span><span class="sxs-lookup"><span data-stu-id="919cb-123">Alternatively, you can start in the **Full order list** view by manually selecting the item.</span></span>

<span data-ttu-id="919cb-124">ถ้าสินค้าที่ถูกเลือก หรือได้รับเป็นสินค้าที่กำหนดหมายเลขลำดับประจำสินค้า บานหน้าต่าง **รายละเอียด** สำหรับสินค้ามีลิงค์ **การจัดการหมายเลขลําดับประจําสินค้า** ที่เปิดหน้า **การจัดการหมายเลขลำดับประจําสินค้า**</span><span class="sxs-lookup"><span data-stu-id="919cb-124">If the item that is being selected or received is a serialized item, the **Details** pane for the item contains a **Manage serial number** link that opens the **Serial number management** page.</span></span> <span data-ttu-id="919cb-125">อีกวิธีหนึ่งคือ คุณสามารถเปิดหน้า **การจัดการหมายเลขลำดับประจำสินค้า** โดยการเลือก **หมายเลขลำดับประจำสินค้า** บนแถบแอปของมุมมองรายละเอียดใบสั่ง หรือโดยการเลือก **จัดการหมายเลขลำดับประจำสินค้า** ในกล่องโต้ตอบที่จะทำให้คุณในระหว่างกระบวนการรับ</span><span class="sxs-lookup"><span data-stu-id="919cb-125">Alternatively, you can open the **Serial number management** page either by selecting **Serial number** on the app bar of the order details view or by selecting **Manage serial number** in the dialog box that prompts you during the receiving process.</span></span> <span data-ttu-id="919cb-126">หน้า **การจัดการหมายเลขลําดับประจําสินค้า** จะแสดงรายการหมายเลขลําดับประจําสินค้าที่เปิดอยู่ทั้งหมดที่รอการลงทะเบียนหรือการตรวจสอบความถูกต้อง</span><span class="sxs-lookup"><span data-stu-id="919cb-126">The **Serial number management** page lists all open serial number lines that are pending registration or validation.</span></span> <span data-ttu-id="919cb-127">อาจมีแท็บสองแท็บบนหน้านี้: หนึ่งแท็บสําหรับรายการปัจจุบัน และหนึ่งแท็บสําหรับรายการที่กำหนดหมายเลขลำดับประจำสินค้าตามลําดับ</span><span class="sxs-lookup"><span data-stu-id="919cb-127">There might be two tabs on this page: one for the current item and one for all the serialized items in the order.</span></span>
