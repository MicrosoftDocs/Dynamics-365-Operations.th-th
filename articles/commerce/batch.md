# <a name="improved-handling-of-batch-tracked-items"></a>การปรับปรุงให้มีการจัดการสินค้าที่มีการติดตามแบบชุดงาน

[!include [banner](includes/banner.md)]

หัวข้อนี้อธิบายถึงการจัดการสินค้าที่มีการติดตามแบบชุดงานที่ปรับปรุงระหว่างกระบวนการลงรายการบัญชีใบแจ้งยอดใน Microsoft Dynamics 365 Commerce

ในการขายหน้าร้าน (POS) ของ Dynamics 365 Commerce ไม่สามารถบันทึกหมายเลขชุดงานสำหรับสินค้าที่มีการติดตามแบบชุดงานในเวลาที่ขาย อย่างไรก็ตาม สำหรับการตั้งค่าคอนฟิกเฉพาะ เมื่อมีการลงรายการบัญชียอดขายในศูนย์ควบคุม Commerce ผ่านทางใบสั่งของลูกค้าหรือการลงรายการบัญชีใบแจ้งยอด Commerce คาดว่าจะมีหมายเลขชุดงานที่ถูกต้องของสินค้าที่มีการติดตามแบบชุดงาน และจะใช้หมายเลขดังกล่าวในกระบวนการออกใบแจ้งหนี้

หากมีหมายเลขชุดงานที่ถูกต้องของผลิตภัณฑ์ ทั้งกระบวนการออกใบแจ้งหนี้ใบสั่งของลูกค้าและกระบวนการออกใบแจ้งหนี้ใบสั่งขายจากการลงรายการบัญชีใบแจ้งยอดจะใช้หมายเลขดังกล่าว หากหมายเลขชุดงานที่ถูกต้องไม่สามารถใช้งานกับสินค้า กระบวนการออกใบแจ้งหนี้ใบสั่งของลูกค้าจะไม่สามารถลงรายการบัญชีได้ และผู้ใช้ POS จะได้รับข้อความแสดงข้อผิดพลาด การลงรายการบัญชีใบแจ้งยอดจะเข้าสู่สถานะข้อผิดพลาด แม้ว่าสินค้าคงคลังค่าลบจะเปิดอยู่สำหรับสินค้า

การปรับปรุงของ Commerce จะช่วยรับประกันว่า เมื่อเปิดสินค้าคงคลังติดลบสำหรับสินค้าที่มีการติดตามแบบชุดงาน การออกใบแจ้งหนี้สำหรับใบสั่งของลูกค้าและการออกใบแจ้งหนี้ใบสั่งขายผ่านการลงรายการบัญชีใบแจ้งยอดจะไม่ถูกบล็อคสำหรับสินค้าเหล่านั้นหากสินค้าคงคลังเป็น 0 (ศูนย์) หรือไม่มีหมายเลขชุดงาน ฟังก์ชันที่ปรับปรุงใหม่นี้จะใช้รหัสชุดงานเริ่มต้นกับรายการขายเมื่อไม่มีหมายเลขชุดงาน

## <a name="define-the-default-batch-id-that-is-used-for-customer-orders"></a>กําหนดรหัสชุดงานเริ่มต้นที่ใช้กับใบสั่งของลูกค้า

หากต้องการกําหนดรหัสชุดงานเริ่มต้นที่ใช้กับใบสั่งของลูกค้า ให้ทำตามขั้นตอนเหล่านี้

1. ในศูนย์ควบคุม Commerce ไปที่ **Retail และ Commerce \> การตั้งค่าศูนย์ควบคุม \> พารามิเตอร์ \> พารามิเตอร์ Commerce**
1. บนแท็บ **ใบสั่งของลูกค้า** บนแท็บด่วน **ใบสั่ง** ให้ป้อนค่าในฟิลด์ **รหัสชุดงานเริ่มต้น**

## <a name="define-the-default-batch-id-that-is-used-for-sales-order-invoicing-through-statement-posting"></a>กําหนดรหัสชุดงานเริ่มต้นที่จะใช้ในการออกใบแจ้งหนี้ใบสั่งขายผ่านการลงรายการบัญชีใบแจ้งยอด

หากต้องการกําหนดรหัสชุดงานเริ่มต้นที่จะใช้ในการออกใบแจ้งหนี้ใบสั่งขายผ่านการลงรายการบัญชีใบแจ้งยอด ให้ทำตามขั้นตอนเหล่านี้

1. ในศูนย์ควบคุม Commerce ไปที่ **Retail และ Commerce \> การตั้งค่าศูนย์ควบคุม \> พารามิเตอร์ \> พารามิเตอร์ Commerce**
1. บนแท็บ **การลงรายการบัญชี** บนแท็บด่วน **การอัปเดตสินค้าคงคลัง** ให้ป้อนค่าในฟิลด์ **รหัสชุดงานเริ่มต้น**

> [!NOTE]
> - ฟังก์ชันรหัสชุดงานเริ่มต้นสามารถใช้ได้เฉพาะเมื่อมีการเปิดใช้งานการจัดการคลังสินค้าขั้นสูงสำหรับคลังสินค้าและสินค้าของร้านค้าเฉพาะ สำหรับรุ่นในอนาคต ฟังก์ชันรหัสชุดงานเริ่มต้นจะใช้รองรับสถานการณ์ที่ไม่ได้เปิดใช้งานการจัดการคลังสินค้าขั้นสูงด้วย
> - การสนับสนุนสำหรับการจัดการสินค้าที่มีการติดตามเป็นชุดงานที่ปรับปรุงในระหว่างการลงรายการบัญชีใบแจ้งยอดสำหรับสถานการณ์การจัดการคลังสินค้าที่ไม่ใช่ขั้นสูงมีการนำมาใช้ใน Commerce รุ่น 10.0.5
