# <a name="enrich-a-category-landing-page"></a>ทำให้หน้าเริ่มต้นของประเภทสมบูรณ์

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

หัวข้อนี้จะครอบคลุมการตกแต่งของหน้าประเภทใน Dynamics 365 Commerce

## <a name="overview"></a>ภาพรวม

Commerce จะให้หน้าเริ่มต้นที่จะใช้เมื่อมีการแสดงข้อมูลของประเภท หน้าประเภทเริ่มต้น มีองค์ประกอบที่ต้องการเช่น ตัวปรับปรุง การจัดประเภทการวางสินค้า ตัวเลือกการเรียงลำดับข้อมูล สรุปทางเลือก และการควบคุมการแบ่งหน้า 

อย่างไรก็ตาม แทนที่จะใช้หน้าประเภทแบบเริ่มต้นคุณอาจต้องการใช้หน้าเริ่มต้นที่ "สมบูรณ์" ของประเภท ซึ่งมีเนื้อหาเพิ่มเติมและองค์ประกอบเฉพาะมากขึ้น การตกแต่งโดยทั่วไปอาจเกี่ยวข้องกับการเพิ่มเนื้อหาการตลาดเฉพาะประเภทเข้าในหน้าประเภท เนื้อหานี้อาจรวมถึงการจัดวางผลิตภัณฑ์ข้ามประเภท เพื่อวัตถุประสงค์ในการขายสินค้า รายการที่เลือกโดยบรรณาธิการ รูปภาพ วิดีโอ และข้อความอื่นๆ คุณสามารถปรับเปลี่ยนหน้าประเภทเริ่มต้นหรือกำหนดหน้าประเภทอื่นสำหรับประเภทที่ระบุได้

![หน้าเริ่มต้นของประเภทที่สมบูรณ์แล้ว](./media/CategoryLandingPages.png)

ในเครื่องมือการสร้างหน้า **ผลิตภัณฑ์** จะรวมรายการของประเภทต่าง ๆ จากช่องทางที่กำหนดให้กับไซต์ ถ้ามีสถานะ **สมบูรณ์** สำหรับหน้าประเภทดังกล่าว หมายความว่าหน้าประเภทนั้นสมบูรณ์แล้ว มิฉะนั้น จะมีการใช้หน้าประเภทเริ่มต้นและเนื้อหาสำหรับประเภทนั้น คุณสามารถดูตัวอย่างทั้งหน้าของประเภทที่สมบูรณ์และไม่สมบูรณ์สำหรับประเภท โดยการเลือกชื่อประเภท

เมื่อต้องการทำให้หน้าประเภทสมบูรณ์ ให้ทำตามขั้นตอนต่อไปนี้

1. บนหน้า **ผลิตภัณฑ์** ให้เลือกชื่อของประเภทที่คุณต้องการสร้างหน้าประเภท หน้าประเภทเริ่มต้นสำหรับประเภทที่เลือก เปิดอยู่ในโปรแกรมแก้ไขหน้า
2. เลือก**ทำให้หน้าประเภทสมบูรณ์**
3. เลือกแม่แบบสำหรับหน้าประเภทที่สมบูรณ์ ถ้าคุณกำลังทำการเปลี่ยนแปลงแค่เล็กน้อย คุณสามารถเลือกหน้าประเภทเริ่มต้น อีกทางหนึ่งคือคุณสามารถเลือกแม่แบบของหน้าประเภทเฉพาะได้ เมื่อคุณเลือกแม่แบบ โปรแกรมแก้ไขหน้าเปิดและจะใช้แม่แบบที่เลือกเพื่อสร้างหน้าประเภทใหม่สำหรับประเภทที่เลือก หน้าถูกเช็คเอาท์ไปยังคุณ และขณะนี้คุณสามารถทำการเปลี่ยนแปลงของคุณได้

> [!NOTE]
> โมดูลที่ใช้ข้อมูลจำเพาะของประเภทใช้ข้อมูลจากประเภทที่คุณเลือก
>
> การตั้งค่าของแม่แบบที่คุณเลือกจะกำหนดว่าคุณสามารถทำการเปลี่ยนแปลงที่คุณสามารถทำได้
