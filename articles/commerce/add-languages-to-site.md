# <a name="add-languages-to-your-site"></a>เพิ่มภาษาลงในไซต์ของคุณ

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

หัวข้อนี้อธิบายถึงวิธีการเพิ่มการสนับสนุนสำหรับภาษาเพิ่มเติมให้กับไซต์ Microsoft Dynamics 365 Commerce

## <a name="overview"></a>ภาพรวม

คุณสามารถแปลเว็บไซต์ของคุณเป็นภาษาใดก็ได้ที่ Dynamics 365 Commerce สนับสนุน (รายการภาษาที่สนับสนุนจะปรากฏในภายหลังในหัวข้อนี้) เมื่อต้องการเพิ่มภาษาบนเว็บไซต์ของคุณ คุณต้องเพิ่มภาษานั้นในร้านค้าออนไลน์ที่ผูกกับไซต์ของคุณก่อน

## <a name="add-a-language-to-an-online-store"></a>เพิ่มภาษาลงในร้านค้าออนไลน์

ในการเพิ่มภาษาในร้านค้าออนไลน์ ให้ทำตามขั้นตอนเหล่านี้

1. เปิดสภาพแวดล้อม Dynamics 365 Retail สำหรับไซต์ของคุณ
1. ไปที่ **การขายปลีก \> ช่องทาง \> ร้านค้าออนไลน์** เพื่อเข้าถึงรายการร้านค้าออนไลน์ที่ได้รับการตั้งค่าคอนฟิกสำหรับสภาพแวดล้อมของคุณ หรือ ป้อน **ร้านค้าออนไลน์** เป็นคำที่ใช้ค้นหา
1. เลือกร้านค้าออนไลน์เพื่อเพิ่มภาษา
1. บนแท็บด่วน **ภาษา** ให้เลือก **เพิ่ม**
1. ในฟิลด์ **ภาษา** เลือกภาษาที่จะเพิ่ม

ในขณะนี้ภาษาที่คุณเพิ่มจะพร้อมใช้งาน เพื่อให้คุณสามารถตั้งค่าคอนฟิกไซต์ของคุณเพื่อใช้งานในสภาพแวดล้อมการสร้างไซต์
