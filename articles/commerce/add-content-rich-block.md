# <a name="text-block-module"></a>โมดูลบล็อคข้อความ


[!include [banner](includes/banner.md)]

หัวข้อนี้ครอบคลุมถึงโมดูลคโมดูลบล็อกข้อความ และอธิบายวิธีการเพิ่มลงในเพจของไซต์ใน Microsoft Dynamics 365 Commerce

## <a name="overview"></a>ภาพรวม

โมดูลบล็อคข้อความเป็นโมดูลที่ใช้ในการเพิ่มเนื้อหาแบบข้อความ เนื้อหานี้อาจเป็นการให้ข้อมูล หรือการส่งเสริมการขาย

โมดูลบล็อคข้อความจะถูกขับเคลื่อนด้วยข้อมูลจากระบบการจัดการเนื้อหา (CMS) และสามารถวางบนหน้าใดก็ได้ โดยมีโมดูลที่ทำงานแยกต่างหากที่ไม่ขึ้นอยู่กับบริบทเพจ หรือโมดูลอื่นๆ

## <a name="examples-of-text-block-modules-in-e-commerce"></a>ตัวอย่างของโมดูลบล็อกข้อความในอีคอมเมิร์ซ

โมดูลบล็อกข้อความสามารถใช้ในลักษณะต่อไปนี้:

* เพื่อแสดงคุณลักษณะผลิตภัณฑ์บนเพจรายละเอียดผลิตภัณฑ์
* สำหรับใช้เพื่อให้ข้อมูลบนเพจใดๆ ตัวอย่างเช่น ผู้ใช้สามารถอธิบายประโยชน์ของโปรแกรมตอบแทนลูกค้าสมาชิกได้ อธิบายนโยบายการจัดส่งและการส่งคืนสินค้า คำถามที่ถามบ่อย หรือระบุเนื้อหา "เกี่ยวกับเรา"
* เมื่อต้องการเพิ่มข้อความแบบกำหนดเองบนเพจรายละเอียดผลิตภัณฑ์ (ตัวอย่างเช่น "จัดส่งฟรีสำหรับใบสั่งที่มากกว่า $50")
* สำหรับข้อยกเว้นความรับผิดและรายละเอียดการติดต่อบนเพจรายละเอียดของผลิตภัณฑ์ เพจรถเข็น เพจเช็คเอาท์ และเพจอื่นๆ (ตัวอย่างเช่น "การจัดส่งสินค้าและการส่งคืนอยู่ในนโยบายจัดเก็บ")

## <a name="text-block-module-properties"></a>คุณสมบัติโมดูลบล็อกข้อความ

| ชื่อคุณสมบัติ     | Value                                            | คำอธิบาย |
|-------------------|--------------------------------------------------|-------------|
| ข้อความตกแต่ง         | ข้อความตกแต่ง                                        | ข้อความย่อหน้า ความสามารถในการทำงานของข้อความที่สมบูรณ์แบบพื้นฐานบางอย่างจะได้รับการสนับสนุน เช่น ข้อความตัวเอน ขีดเส้นใต้ และแบบตัวหนา |
| ชื่อคลาส แบบกำหนดเอง | ชื่อคลาส Cascading Sheet Sheet (CSS)        | ชื่อของคลาส CSS ที่กำหนดเองที่นักพัฒนาจัดเตรียมให้กับโมดูลนี้ ชื่อคลาสควรกำหนดในชุดรูปแบบ |
| ขนาดแบบอักษร         | **เล็ก** **กลาง** **ใหญ่** **ใหญ่พิเศษ** | ขนาดแบบอักษรของเนื้อหา |

## <a name="add-a-text-block-module-to-a-page"></a>เพิ่มโมดูลบล็อคข้อความไปที่เพจใหม่

การเพิ่มโมดูลบล็อกข้อความไปยังหน้าใหม่ และตั้งค่าคุณสมบัติที่จำเป็น ให้ทำตามขั้นตอนต่อไปนี้

1. สร้างแม่แบบเพจซึ่งชื่อ **แม่แบบเนื้อหา** 
1. ในช่อง **Body** ให้เพิ่มโมดูล **หน้าเริ่มต้น**
1. แก้ไขแม่แบบให้เสร็จสิ้น และเผยแพร่
1. ใช้แม่แบบเนื้อหาที่คุณเพิ่งสร้างขึ้น เพื่อสร้างเพจที่ชื่อ **เพจเนื้อหา**
1. ในช่อง **หลัก** ของหน้าใหม่ ให้เพิ่มโมดูลคอนเทนเนอร์
1. ในบานหน้าต่างคุณสมบัติสำหรับโมดูลคอนเทนเนอร์ ตั้งค่า **ความกว้าง** เป็น **เต็มคอนเทนเนอร์**
1. เพิ่มโมดูลบล็อคข้อความไปยังโมดูลคอนเทนเนอร์ 
1. ในบานหน้าต่างคุณสมบัติของโมดูลบล็อคข้อความให้เพิ่มข้อความในฟิลด์ **Rich Text**
1. แก้ไขหน้าให้เสร็จสิ้น และเผยแพร่
