# <a name="add-a-favicon"></a><span data-ttu-id="b5df2-103">เพิ่มไอคอนประจำไซต์</span><span class="sxs-lookup"><span data-stu-id="b5df2-103">Add a favicon</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="b5df2-104">หัวข้อนี้อธิบายถึงวิธีการเพิ่มไอคอนประจำไซต์ลงในไซต์ของคุณ</span><span class="sxs-lookup"><span data-stu-id="b5df2-104">This topic explains how to add a favicon to your site.</span></span>

## <a name="overview"></a><span data-ttu-id="b5df2-105">ภาพรวม</span><span class="sxs-lookup"><span data-stu-id="b5df2-105">Overview</span></span>

<span data-ttu-id="b5df2-106">ไอคอนประจำไซต์เป็นไฟล์กราฟิกขนาดเล็กที่แสดงอยู่บนแท็บเว็บเบราเซอ ร์ในแถบที่อยู่ ในประวัติการเรียกดู และในบุ๊คมาร์ค หรือรายการโปรดในสถานที่อื่น</span><span class="sxs-lookup"><span data-stu-id="b5df2-106">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="b5df2-107">เราขอแนะนำให้คุณเพิ่มไอคอนประจำไซต์ให้กับไซต์ของคุณ เนื่องจากจะแสดงและกระตุ้นแบรนด์ของคุณ และช่วยแยกความแตกต่างของไซต์ของคุณจากไซต์อื่นที่ลูกค้าของคุณเข้าถึง</span><span class="sxs-lookup"><span data-stu-id="b5df2-107">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="b5df2-108">ถึงแม้ว่าคุณจะสามารถเพิ่มหลายไอคอนประจำไซต์ของขนาดต่างๆ และชนิดไฟล์ลงในไซต์ของคุณได้ หัวข้อนี้จะแสดงวิธีการเพิ่มไอคอนประจำไซต์เดียว</span><span class="sxs-lookup"><span data-stu-id="b5df2-108">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="b5df2-109">อย่างไรก็ตาม กระบวนการและสถานที่เดียวกันจะใช้เพื่อเพิ่มไอคอนประจำไซต์เพิ่มเติม</span><span class="sxs-lookup"><span data-stu-id="b5df2-109">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="b5df2-110">อัปโหลดไอคอนประจำไซต์ไปที่การเก็บรวบรวมสินทรัพย์ของไซต์ของคุณ</span><span class="sxs-lookup"><span data-stu-id="b5df2-110">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="b5df2-111">เมื่อต้องการอัปโหลดไอคอนประจำไซต์ไปที่การเก็บรวบรวมสินทรัพย์ของไซต์ของคุณ ให้ทำตามขั้นตอนต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="b5df2-111">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="b5df2-112">ในบานหน้าต่างนำทางทางด้านซ้าย ให้เลือก **ไลบรารีสื่อ**</span><span class="sxs-lookup"><span data-stu-id="b5df2-112">In the left navigation pane, select **Media Library**.</span></span>
1. <span data-ttu-id="b5df2-113">บนแถบคำสั่ง เลือก **อัปโหลด \> อัปโหลดรายการสื่อ**</span><span class="sxs-lookup"><span data-stu-id="b5df2-113">On the command bar, select **Upload \> Upload Media Items**.</span></span>
1. <span data-ttu-id="b5df2-114">ในหน้าต่างตัวสำรวจไฟล์ ให้เลือกดูไฟล์รูปไอคอนประจำไซต์ที่คุณต้องการอัปโหลด เลือก แล้วเลือก **เปิด**</span><span class="sxs-lookup"><span data-stu-id="b5df2-114">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>
1. <span data-ttu-id="b5df2-115">ในกล่องโต้ตอบ **อัปโหลดรายการสื่อ** ให้ป้อนชื่อเรื่องและข้อความกำกับที่ต้องการ</span><span class="sxs-lookup"><span data-stu-id="b5df2-115">In the **Upload Media Item** dialog box, enter the required title and alt text.</span></span>
1. <span data-ttu-id="b5df2-116">ถ้าคุณต้องการเผยแพร่รูปภาพหลังจากการอัปโหลดโดยทันที ให้เลือกกล่องกาเครื่องหมาย **เผยแพร่รายการสื่อหลังจากการอัปโหลด**</span><span class="sxs-lookup"><span data-stu-id="b5df2-116">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="b5df2-117">ถ้าคุณไม่เลือกกล่องกาเครื่องหมาย **เผยแพร่รายการสื่อหลังจากการอัปโหลด** คุณต้องกลับไปที่หน้า **รายการสื่อ** และเผยแพร่ไอคอนประจำไซต์ด้วยตนเองในภายหลัง</span><span class="sxs-lookup"><span data-stu-id="b5df2-117">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="b5df2-118">เลือก **ตกลง**</span><span class="sxs-lookup"><span data-stu-id="b5df2-118">Select **OK**.</span></span>
1. <span data-ttu-id="b5df2-119">ในบานหน้าต่างคุณสมบัติทางด้านขวา ให้คัดลอก URL สาธารณะของไอคอนประจำไซต์</span><span class="sxs-lookup"><span data-stu-id="b5df2-119">In the property pane on the right, copy the public URL of the favicon.</span></span> <span data-ttu-id="b5df2-120">คุณจะใช้ URL นี้ในภายหลัง</span><span class="sxs-lookup"><span data-stu-id="b5df2-120">You will use this URL later.</span></span>

## <a name="create-the-html-for-your-favicon"></a><span data-ttu-id="b5df2-121">สร้าง HTML สำหรับไอคอนประจำไซต์ของคุณ</span><span class="sxs-lookup"><span data-stu-id="b5df2-121">Create the HTML for your favicon</span></span>

<span data-ttu-id="b5df2-122">เมื่อต้องการสร้าง HTML สำหรับไอคอนประจำไซต์ ให้ใช้สตริง HTML ต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="b5df2-122">To create the HTML for the favicon, use the following HTML string.</span></span> <span data-ttu-id="b5df2-123">สำหรับแอททริบิวต์ **href** ให้แทนที่ **URL\_สาธารณะ\_สำหรับ\_\_ไอคอนประจำไซต์ของคุณ** ด้วย URL สาธารณะที่คุณคัดลอกมาก่อนหน้านี้</span><span class="sxs-lookup"><span data-stu-id="b5df2-123">For the **href** attribute, replace **Public\_URL\_for\_your\_favicon** with the public URL that you copied earlier.</span></span>

## <a name="create-a-fragment-that-contains-a-metatag-for-your-favicon"></a><span data-ttu-id="b5df2-124">สร้างส่วนย่อยที่มีเมตาแท็กสำหรับไอคอนประจำไซต์ของคุณ</span><span class="sxs-lookup"><span data-stu-id="b5df2-124">Create a fragment that contains a metatag for your favicon</span></span>

<span data-ttu-id="b5df2-125">หากต้องการสร้างส่วนย่อยที่มีเมตาแท็กสำหรับไอคอนประจำไซต์ของคุณ ให้ทำตามขั้นตอนต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="b5df2-125">To create a fragment that contains a metatag for your favicon, follow these steps.</span></span>

1. <span data-ttu-id="b5df2-126">ไปที่ **ส่วนย่อย** และให้เลือก **สร้าง**</span><span class="sxs-lookup"><span data-stu-id="b5df2-126">Go to **Fragments**, and select **New**.</span></span>
1. <span data-ttu-id="b5df2-127">ในกล่องโต้ตอบ **ส่วนย่อยใหม่** ให้เลือก **เมตาแท็ก** เป็นโมดูลที่เป็นพื้นฐานของส่วนย่อย</span><span class="sxs-lookup"><span data-stu-id="b5df2-127">In the **New fragment** dialog box, select **Metatags** as the module that the fragment is based on.</span></span>
1. <span data-ttu-id="b5df2-128">ป้อนชื่อสำหรับส่วนย่อย และเลือก **ตกลง**</span><span class="sxs-lookup"><span data-stu-id="b5df2-128">Enter a name for the fragment, and then select **OK**.</span></span>
1. <span data-ttu-id="b5df2-129">ในแผนภูมิลำดับชั้นของส่วน ให้เลือก **เมตาแท็กเริ่มต้น** รอง</span><span class="sxs-lookup"><span data-stu-id="b5df2-129">In the fragment hierarchy tree, select the **Default metatags** child.</span></span>
1. <span data-ttu-id="b5df2-130">ในบานหน้าต่างด้านขวา ภายใต้ **เมตาแท็ก** ให้เลือก **เพิ่ม** แล้วป้อนสตริง HTML ที่คุณสร้างไว้ก่อนหน้านี้สำหรับไอคอนประจำไซต์</span><span class="sxs-lookup"><span data-stu-id="b5df2-130">In the right pane, under **Meta Tags**, select **Add**, and then enter the HTML string that you created earlier for the favicon.</span></span> 
1. <span data-ttu-id="b5df2-131">เลือก **แก้ไขให้เสร็จสิ้น** และจากนั้น เลือก **เผยแพร่** เพื่อเผยแพร่ส่วนย่อย</span><span class="sxs-lookup"><span data-stu-id="b5df2-131">Select **Finish editing**, and then select **Publish** to publish the fragment.</span></span>

## <a name="add-the-metatag-fragment-to-the-html-head-section-of-your-pages"></a><span data-ttu-id="b5df2-132">เพิ่มส่วนย่อยเมตาแท็กไปที่ส่วนหัว HTML ของหน้าของคุณ</span><span class="sxs-lookup"><span data-stu-id="b5df2-132">Add the metatag fragment to the HTML head section of your pages</span></span>

<span data-ttu-id="b5df2-133">หากต้องการเพิ่มส่วนย่อยเมตาแท็กไปที่ส่วน **หัว** HTML ของหน้าของคุณ ให้ทำตามขั้นตอนต่อไปนี้</span><span class="sxs-lookup"><span data-stu-id="b5df2-133">To add the metatag fragment to the HTML **head** section of your pages, follow these steps.</span></span>

1. <span data-ttu-id="b5df2-134">ไปที่ **แม่แบบ** เปิดแม่แบบสำหรับเพจที่คุณต้องการเพิ่มไอคอนประจำไซต์ของคุณ และเลือก **แก้ไข**</span><span class="sxs-lookup"><span data-stu-id="b5df2-134">Go to **Templates**, open the template for the pages that you want to add your favicon to, and then select **Edit**.</span></span>
1. <span data-ttu-id="b5df2-135">ในแผนภูมิลำดับชั้นเท็มเพลต ให้เลือกปุ่มจุดไข่ปลา (**...**) ที่อยู่ทางด้านขวาของคอนเทนเนอร์ **ส่วนหัว ของ HTML** แล้วเลือก **เพิ่มส่วนย่อย**</span><span class="sxs-lookup"><span data-stu-id="b5df2-135">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add fragment**.</span></span>
1. <span data-ttu-id="b5df2-136">ในกล่องโต้ตอบ **เลือกส่วนย่อย** ให้เลือกส่วนย่อยเมตาแท็กที่คุณสร้างไว้ก่อนหน้า แล้วเลือก **ตกลง**</span><span class="sxs-lookup"><span data-stu-id="b5df2-136">In the **Select fragment** dialog box, select the metatag fragment that you created earlier, and then select **OK**.</span></span>
1. <span data-ttu-id="b5df2-137">เลือก **แก้ไขให้เสร็จสิ้น** และจากนั้น เลือก **เผยแพร่** เพื่อเผยแพร่แม่แบบ</span><span class="sxs-lookup"><span data-stu-id="b5df2-137">Select **Finish editing**, and then select **Publish** to publish the template.</span></span>

> [!NOTE]
> <span data-ttu-id="b5df2-138">ถ้าไซต์ของคุณใช้เท็มเพลตมากกว่าหนึ่งเท็มเพลต คุณต้องเพิ่มส่วนย่อยเมตาแท็กให้กับเท็มเพลตทั้งหมด</span><span class="sxs-lookup"><span data-stu-id="b5df2-138">If your site uses more than one template, you must add the metatags fragment to all of them.</span></span>

<span data-ttu-id="b5df2-139">เมื่อคุณแสดงตัวอย่างหน้าเว็บที่เป็นไปตามเท็มเพลตที่คุณได้เพิ่มส่วนย่อยเมตาแท็กให้กับคุณ ในขณะนี้ คุณควรเห็นไอคอนประจำไซต์บนแท็บเบราว์เซอร์</span><span class="sxs-lookup"><span data-stu-id="b5df2-139">When you preview pages that are based on the template that you added the metatags fragment to, you should now see the favicon on the browser tab.</span></span>
