# <a name="add-a-favicon"></a>เพิ่มไอคอนประจำไซต์


[!include [banner](includes/banner.md)]

หัวข้อนี้อธิบายถึงวิธีการเพิ่มไอคอนประจำไซต์ลงในไซต์ของคุณ

## <a name="overview"></a>ภาพรวม

ไอคอนประจำไซต์เป็นไฟล์กราฟิกขนาดเล็กที่แสดงอยู่บนแท็บเว็บเบราเซอ ร์ในแถบที่อยู่ ในประวัติการเรียกดู และในบุ๊คมาร์ค หรือรายการโปรดในสถานที่อื่น เราขอแนะนำให้คุณเพิ่มไอคอนประจำไซต์ให้กับไซต์ของคุณ เนื่องจากจะแสดงและกระตุ้นแบรนด์ของคุณ และช่วยแยกความแตกต่างของไซต์ของคุณจากไซต์อื่นที่ลูกค้าของคุณเข้าถึง

ถึงแม้ว่าคุณจะสามารถเพิ่มหลายไอคอนประจำไซต์ของขนาดต่างๆ และชนิดไฟล์ลงในไซต์ของคุณได้ หัวข้อนี้จะแสดงวิธีการเพิ่มไอคอนประจำไซต์เดียว อย่างไรก็ตาม กระบวนการและสถานที่เดียวกันจะใช้เพื่อเพิ่มไอคอนประจำไซต์เพิ่มเติม

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a>อัพโหลดไอคอนประจำไซต์ไปที่การเก็บรวบรวมสินทรัพย์ของไซต์ของคุณ

เมื่อต้องการอัพโหลดไอคอนประจำไซต์ไปที่การเก็บรวบรวมสินทรัพย์ของไซต์ของคุณ ให้ทำตามขั้นตอนต่อไปนี้

1. ไปที่ **สินทรัพย์ \> อัพโหลด \> อัพโหลดสินทรัพย์**
1. ค้นหาและเลือกไอคอนประจำไซต์บนระบบไฟล์ในเครื่องของคุณ
1. ป้อนชื่อ และจากนั้นเลือก **ตกลง** 
1. ในบานหน้าต่างคุณสมบัติทางด้านขวา ให้คัดลอก URL สาธารณะของไอคอนประจำไซต์

> [!NOTE]
> ถ้าคุณไม่เลือกตัวเลือก **เผยแพร่สินทรัพย์หลังจากการอัพโหลด** คุณต้องกลับไปที่เพจ **สินทรัพย์** และเผยแพร่ไอคอนประจำไซต์ด้วยตนเองในภายหลัง

## <a name="create-the-html-for-the-favicon"></a>สร้าง HTML สำหรับไอคอนประจำไซต์

เมื่อต้องการสร้าง HTML สำหรับไอคอนประจำไซต์ ให้ใช้ส่วนย่อย HTML ต่อไปนี้ สำหรับแอททริบิวต์ **href** ให้แทนที่ **"URL\_สาธารณะ\_สำหรับ\_\_ไอคอนประจำไซต์ของคุณ"** ด้วย URL สาธารณะที่คุณคัดลอกมาก่อนหน้านี้

`<link rel="shortcut icon" href="Public_URL_for_your_favicon">`

## <a name="add-the-html-for-the-favicon-to-the-head-element-of-your-pages"></a>เพิ่ม HTML สำหรับไอคอนประจำไซต์ไปที่องค์ประกอบ \<ส่วนหัว\> ของเพจของคุณ

เมื่อต้องการเพิ่มไอคอนประจำไซต์ลงในไซต์ของคุณ ให้ใช้ขั้นตอนเดียวกันกับที่ใช้เพื่อเพิ่มชนิดของ HTML หรือสคริปต์ไปที่องค์ประกอบ **\<ส่วนหัว\>** ของเพจของไซต์ของคุณ
