# <a name="product-collection-modules"></a>โมดูลการรวบรวมผลิตภัณฑ์


[!include [banner](includes/banner.md)]

หัวข้อนี้แสดงภาพรวมของโมดูลการรวบรวมผลิตภัณฑ์ ใน Microsoft Dynamics 365 Commerce

## <a name="overview"></a>ภาพรวม

การค้นหาผลิตภัณฑ์เป็นเครื่องมือหลักที่ร้านค้าปลีกใช้ในการมีส่วนร่วมกับลูกค้าของตนบนเว็บไซต์อีคอมเมิร์ซ โมดูลการรวบรวมผลิตภัณฑ์ช่วยให้ผู้ค้าปลีกสร้างประสบการณ์การซื้อสินค้าที่น่าสนใจโดยการจัดหาอินเทอร์เฟสแสดงผลที่ใช้งานง่าย ซึ่งสามารถใช้ในการสร้างการรวบรวมผลิตภัณฑ์อย่างรวดเร็ว

โมดูลการรวบรวมผลิตภัณฑ์แสดงถึงผลิตภัณฑ์และบริการทางกายภาพบนเว็บไซต์ โดยทั่วไปโมดูลการรวบรวมผลิตภัณฑ์จะเชื่อมโยงกับหน้ารายละเอียดที่ลูกค้าสามารถซื้อผลิตภัณฑ์หรือบริการ หรือเรียนรู้เพิ่มเติมเกี่ยวกับข้อมูลนี้ 

แหล่งที่มาสำหรับการรวบรวมผลิตภัณฑ์สามารถแสดงรายการได้สี่ชนิดต่อไปนี้:

- รายการบรรณาธิการของผลิตภัณฑ์ที่ได้รับการกำหนดใน Dynamics 365 Commerce ด้วยตนเองในฐานะผลิตภัณฑ์ที่เกี่ยวข้องกับผลิตภัณฑ์หรือรายการผลิตภัณฑ์
- รายการอัลกอริทึม เช่น รายการของผลิตภัณฑ์ใหม่ ผลิตภัณฑ์ที่ขายดีที่สุด หรือผลิตภัณฑ์ที่กำลังได้รับความนิยม
- รายการคำแนะนำที่ขึ้นอยู่กับการเรียนรู้ของเครื่อง
- รายการการตั้งค่าส่วนบุคคลที่สนับสนุนผลลัพธ์ที่กำหนดเป็นแบบส่วนบุคคลสำหรับลูกค้า ลูกค้าต้องลงชื่อเข้าใช้ไซต์พาณิชย์อิเล็กทรอนิกส์เพื่อดูผลลัพธ์ที่กำหนดเป็นแบบส่วนบุคคล ผู้ใช้ที่เป็นแขกไม่เห็นผลลัพธ์ที่กำหนดเป็นแบบส่วนบุคคล ลูกค้าสามารถเลือกไม่ใช้การตั้งค่าส่วนบุคคลได้จาก [หน้าการจัดการบัญชี](account-management.md)

ภาพประกอบต่อไปนี้จะแสดงชนิดของการรวบรวมผลิตภัณฑ์ต่างๆ ที่ใช้บนไซต์อีคอมเมิร์ซ

## <a name="add-a-product-collection-module-to-a-category-page"></a>เพิ่มโมดูลการรวบรวมผลิตภัณฑ์ให้กับหน้าประเภท

เมื่อต้องการเพิ่มโมดูลการรวบรวมผลิตภภัณฑ์ไปที่หน้าประเภท ให้ทำตามขั้นตอนต่อไปนี้

1. ใน Dynamics 365 Commerce ให้ไปที่ไซต์ของคุณ และสร้างหน้าที่ใช้เท็มเพลตเดียวกันกับหน้าประเภทเริ่มต้นของคุณ
1. ในโครงร่างหน้า ให้เลือกช่องเสียบ **ท้ายหน้ากระดาษส่วนย่อย** เลือกปุ่มจุดไข่ปลา (**...**) แล้วเลือก **เพิ่มโมดูล**
1. ในกล่องโต้ตอบ **เพิ่มโมดูล** ให้เลือก **คอนเทนเนอร์** แล้วเลือก **ตกลง**
1. ในโมดูลคอนเทนเนอร์ เลือกปุ่มจุดไข่ปลา (...) แล้วเลือก **เพิ่มโมดูล**
1. ในกล่องโต้ตอบ **เพิ่มโมดูล** ให้เลือก **การรวบรวมผิตภัณฑ์** แล้วเลือก **ตกลง**  
1. ตั้งค่าคอนฟิกการตั้งค่าโดยเลือกแหล่งที่มาข้อมูลและข้อมูลป้อนเข้าที่เหมาะสมสำหรับการรวบรวมผลิตภัณฑ์
1. ในบานหน้าต่างคุณสมบัติสำหรับโมดูลการรวบรวมผลิตภัณฑ์ ให้เลือก **เพิ่มรายการผลิตภัณฑ์**
1. ในกล่องโต้ตอบ **การตั้งค่าคอนฟิกรายการผลิตภัณฑ์** ให้เลือกชนิดของรายการแล้วป้อนจำนวนของสินค้า และเลือกตัวเลือกอื่นๆ ที่พร้อมใช้งานสำหรับชนิดของรายการ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับชนิดรายการ ให้ดูที่ตารางต่อไปนี้ 
1. เลือก **ตกลง**
1. บันทึกหน้าและเช็คอิน
