# <a name="demo-data-screen-layouts-in-modern-pos-mpos-and-cloud-pos"></a>เค้าโครงหน้าจอข้อมูลการสาธิตใน Modern POS (MPOS) และ Cloud POS

[!include [banner](includes/banner.md)]

หัวข้อนี้ให้ข้อมูลเกี่ยวกับโครงร่างหน้าจอที่ถูกรวมกับชุดข้อมูลสาธิตสำหรับประสบการณ์ point of sale (POS) ใน Dynamics 365 Commerce

## <a name="overview"></a>ภาพรวม

เค้าโครงหน้าจอตัวอย่างที่มาพร้อมกับข้อมูลการสาธิตการค้าจัดเตรียมเนื้อหาที่ปรับให้เหมาะกับเซ็กเมนต์การค้าปลีกต่างๆ บทบาทผู้ปฏิบัติงานร้านค้า และอุปกรณ์ โครงร่างเดี่ยวอาจประกอบด้วยหลายขนาดโครงร่างและชุดของกริดปุ่ม เพื่อช่วยให้มั่นใจถึงความครอบคลุม เมื่อผู้ปฏิบัติงานประจำร้านย้ายที่ระหว่างอุปกรณ์และสถานีของร้านค้า หัวข้อนี้เน้นถึงผลต่างระหว่างโครงร่างเหล่านี้ การดำเนินการที่พวกเขาให้ ประสบการณ์โดยรวมที่พวกเขาจัดส่ง

![โครงร่างข้อมูลสาธิตระหว่างอุปกรณ์](../commerce/media/demo-screen-layouts-fig-1-1.png)

![หน้าเค้าโครงหน้าจอ](../commerce/media/demo-screen-layouts-fig-2-1.png)

3. ลักษณะ

## <a name="layout-sizes"></a>ขนาดโครงร่าง

### <a name="full-vs-compact-layouts"></a>เค้าโครงแบบเต็ม vs. แบบย่อ

เค้าโครงหน้าจอสามารถมีการตั้งค่าคอนฟิกได้สำหรับทั้งอุปกรณ์แบบเต็มและแบบย่อ ดังนั้น ผู้ใช้สามรถถูกมอบหมายให้กับโครงร่างหน้าจอเดี่ยวที่จะทำงานในขนาดและสัดส่วนของแบบฟอร์มที่หลากหลายภายในร้านค้าได้

- **Modern POS - รุ่นเต็ม** - โดยทั่วไปโครงร่างแบบเต็มจะใช้ได้ดีที่สุดสำหรับการแสดงขนาดใหญ่ เช่น จอภาพของคอมพิวเตอร์เดสก์ท็อปหรือแท็บเล็ต ผู้ใช้สามารถเลือกองค์ประกอบ UI ที่มีโครงร่าง ระบุขนาดและตำแหน่งขององค์ประกอบเหล่านั้น และตั้งค่าคอนฟิกคุณสมบัติโดยรายละเอียด เค้าโครงทั้งหมดสนับสนุนการตั้งค่าคอนฟิกทั้งแนวตั้งและแนวนอน
- **Modern POS - รุ่นย่อ** - โดยทั่วไปโครงร่างแบบย่อจะใช้ได้ดีที่สุดสำหรับโทรศัพท์หรือแท็บเล็ตขนาดเล็ก ความสามารถในการออกแบบถูกจำกัดสำหรับอุปกรณ์แบบย่อ ผู้ใช้สามารถตั้งค่าคอนฟิกคอลัมน์และฟิลด์ได้สำหรับบานหน้าต่างการรับสินค้าและบานหน้าต่างผลรวม

### <a name="screen-resolutions-that-are-provided"></a>ความละเอียดหน้าจอที่มีให้

ตารางต่อไปนี้แสดงขนาดของโครงร่างที่กำหนดไว้สำหรับความละเอียดของหน้าจอโดยทั่วไป

| ชนิดของโครงร่าง | ความละเอียด | อัตราส่วนกว้างยาว | การแสดงเป้าหมาย          |
|-------------|------------|--------------|-------------------------|
| อัดแน่น\*   | 480 × 853  | 16:9 น.         | โทรศัพท์                  |
| ทั้งหมด        | 1024 × 768 | 4:3 น.          | แท็บเล็ต                 |
| ทั้งหมด\*      | 1280 × 720 | 16:9 น.         | แท็บเล็ต                 |
| ทั้งหมด        | 1366 × 768 | 16:9 น.         | แท็บเล็ต หน้าจอที่ขนาดใหญ่ขึ้น |

> [!TIP]
> POS เลือกขนาดของโครงร่างโดยอัตโนมัติ ตามขนาดใกล้เคียงที่สุดที่พร้อมใช้งานสำหรับความละเอียดของหน้าจอของหน้าต่างแอพลิเคชันปัจจุบัน หากต้องการค้นหารหัสเค้าโครงหน้าจอและความละเอียดเค้าโครงที่ใช้อยู่ในปัจจุบันใน Modern POS (MPOS) หรือ Retail Cloud POS (CPOS) ให้เปิดหน้า **ตั้งค่า** และดูที่ส่วน **ข้อมูลเซสชัน** . คุณยังสามารถดูความละเอียดหน้าต่างจริงสำหรับแอพลิเคชันหรือเฟรมของเบราเซอร์ปัจจุบันของคุณได้ด้วย หลังจากที่คุณมีข้อมูลนี้ คุณสามารถค้นหาแหล่งที่มาของเนื้อหาโครงร่างได้โดยการไปที่ **การตั้งค่าช่องทาง** \> **การตั้งค่า POS** \> **POS** \> **โครงร่างหน้าจอ**
