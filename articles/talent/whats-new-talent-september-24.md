# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-september-24-2018"></a>มีอะไรใหม่หรือเปลี่ยนแปลงใน Dynamics 365 for Talent Core HR (24 กันยายน 2018)

[!include [banner](includes/banner.md)]

**สร้าง 8.1.1015.0**

หัวข้อนี้อธิบายถึงคุณลักษณะที่ใหม่หรือที่มีการเปลี่ยนแปลง อย่างใดอย่างหนึ่งใน Core HR

## <a name="currency-added-to-benefits"></a>สกุลเงินที่เพิ่มในสวัสดิการ

แผนสวัสดิการได้รับการปรับปรุงเพื่อรวมสกุลเงินของสวัสดิการ นอกจากนี้ ฟิลด์ใหม่นี้จะพร้อมใช้งานในสวัสดิการของผู้ปฏิบัติงานลงทะเบียนไว้ ฟิลด์นี้ใหม่เป็นส่วนหนึ่งของการรักษาสวัสดิการ และการดูรายการของสิทธิ์ความปลอดภัยของผลประโยชน์

## <a name="update-proration-process--leave-and-absence"></a>ปรับปรุงกระบวนการการแบ่งตามสัดส่วน – การลางานและการขาดงาน

องค์กรให้รางวัลเวลาหยุดพักแตกต่างกัน ตามเวลาที่พนักงานเข้า และออกจากบริษัท สำหรับพนักงานที่ออกจากองค์กร บางรายอาจต้องการสิ้นสุดรางวัลในวันที่สิ้นสุดการจ้างงาน ในขณะที่ผู้อื่นขึ้นอยู่กับวันที่สุดท้ายที่ทำงานเพื่อหยุดกระบวนการค้างรับค้างจ่าย การเปลี่ยนแปลงเหล่านี้ทำให้องค์กรสามารถเลือกเวลาสิ้นสุดการลงทะเบียนในระหว่างกระบวนการสิ้นสุดการจ้างงานได้ ตัวเลือกใหม่เหล่านี้เป็นส่วนหนึ่งของสิทธิ์การใช้งานสำหรับการเลิกจ้างผู้ปฏิบัติงาน และการเลิกจ้างผู้ปฏิบัติงานจากระบบบริการตนเองของผู้จัดการ 

## <a name="other-changes"></a>การเปลี่ยนแปลงอื่นๆ

รุ่นนี้ประกอบด้วยการแก้ไขบักเพิ่มเติมหลายรายการ

## <a name="known-issue"></a>ปัญหาที่ทราบ

-   **ปัญหา:** เมื่อเพิ่มสิ่งที่แนบมาใหม่กับผู้ปฏิบัติงาน ปุ่ม **สร้าง** และ **แก้ไข** ถูกแรเงา **วิธีการแก้ไขปัญหา:** ก่อนที่จะเปิดหน้าเอกสารแนบ ให้แน่ใจว่ากล่องแสดงข้อมูลย่อในหน้า **ผู้ปฏิบัติงาน** ถูกปิดแล้ว ถ้ากล่องแสดงข้อมูลย่อถูกปิด เมื่อหน้า **ผู้ปฏิบัติงาน** ถูกโหลด ปุ่มเอกสารแนบจะถูกเปิดใช้งาน (ปัญหานี้จะได้รับการแก้ไขในการปรับปรุงแพลตฟอร์มถัดไป)
