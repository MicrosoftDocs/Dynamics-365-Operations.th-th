# <a name="career-site-functionality-in-attract"></a>ฟังก์ชันไซต์การทำงานใน Attract

[!include [banner](includes/banner.md)]

บทความนี้แสดงภาพรวมของฟังก์ชันการทำงานการเชื่อมต่อกับผู้สมัครใน Microsoft Dynamics 365 for Talent: Attract บทความนี้ยังอธิบายวิธีการตั้งค่าฟังก์ชันนี้อีกด้วย

## <a name="overview"></a>ภาพรวม

Attract ให้ไซต์การทำงานหนึ่งไซต์สำหรับแต่ละสภาพแวดล้อมในผู้เช่า ตัวอย่างเช่น ถ้าองค์กรมีสภาพแวดล้อมการพัฒนาและการทดสอบ ไซต์การทำงานหนึ่งไซต์จะถูกเตรียมใช้งานสำหรับสภาพแวดล้อมการพัฒนา และไซต์การทำงานอีกหนึ่งไซต์จะถูกเตรียมใช้งานสำหรับสภาพแวดล้อมการทดสอบ ไซต์การทำงานแต่ละไซต์เป็นแบบ **แยกต่างหากทั้งหมด** และมีกลไกการรับรองความถูกต้องของตนเอง งานและโพรไฟล์ผู้สมัครไม่ได้ใช้ร่วมกันระหว่างไซต์ทำงาน

โดยค่าเริ่มต้น ไซต์การทำงานเป็นแบบสาธารณะ ดังนั้น ผู้สมัครสามารถดูงานทั้งหมดที่ทำเครื่องหมายเป็นภายนอกได้ โดยไม่ต้องเข้าสู่ระบบ อย่างไรก็ตาม การดำเนินการอื่นๆ ทั้งหมดกำหนดให้ผู้สมัครต้องล็อกอิน

## <a name="career-site-management"></a>การจัดการไซต์อาชีพ

รายการต่อไปนี้บนไซต์การทำงานจะถูกควบคุมโดยการตั้งค่า:

- **ชื่อองค์กร:** ชื่อองค์กรปรากฏขึ้นบนแถบนำทางด้านบนของไซต์การทำงาน โดยการเลือกชื่อองค์กร ผู้สมัครไปยังหน้าซึ่งแสดงรายการงานที่เปิดทั้งหมด
- **โลโก้องค์กร:** รูปโลโก้ขององค์กรปรากฏขึ้นในด้านซ้ายบนของไซต์การทำงาน โดยการเลือกรูปโลโก้ ผู้สมัครไปยังหน้าซึ่งแสดงรายการงานที่เปิดทั้งหมด

ในการตั้งค่าสำหรับชื่อองค์กรและโลโก้ ล็อกอินไปยัง Attract ในฐานะผู้ดูแลระบบ เลือก **ศูนย์การจัดการ** บนเมนู **การตั้งค่า** (สัญลักษณ์เกียร์) และจากนั้นเลือกแท็บ **ข้อมูลของบริษัท**

> [!NOTE]
> รูปโลโก้ที่ปรากฏบนไซต์การทำงานมีความสูงคงที่เป็น 20 พิกเซล (px) รูปภาพที่คุณเพิ่มในศูนย์การจัดการจะถูกปรับขนาดให้พอดี ดังนั้น โดยขึ้นอยู่กับรูปภาพ ความกว้างอาจเปลี่ยนแปลง

## <a name="career-site-url"></a>URL ไซต์การทำงาน

เมื่อคุณ [ลงรายการบัญชีงานภายนอก](./Creating-jobs-Attract.md#postings) เป็นครั้งแรก คุณสามารถคัดลอกลิงค์ **สมัคร** จากใบสมัคร Attract ได้ URL สำหรับการเชื่อมโยงนี้จะอยู่ในรูปแบบต่อไปนี้: `https://jobs.talent.dynamics.com/jobs/<company_name>/<environment_number>/<job_number>/apply`

URL ของไซต์การทำงานเป็นสตริงย่อยของ URL **สมัคร** ซึ่งประกอบด้วยข้อมูลทั้งหมดไปจนถึงชื่อบริษัท ดังนั้น สำหรับ URL **สมัคร** ก่อนหน้า URL ของไซต์การทำงานคือ `https://jobs.talent.dynamics.com/jobs/<company_name>/`

## <a name="authentication-options"></a>ตัวเลือกการรับรองความถูกต้อง

ผู้สมัครมีตัวเลือกการลงชื่อเข้าใช้ดังต่อไปนี้สำหรับไซต์การทำงาน Attract:

- บัญชีส่วนตัว:

    - LinkedIn
    - Microsoft
    - Google
    - Facebook

- บัญชีที่ทำงานหรือที่โรงเรียน:

    - Microsoft Azure Active Directory (Azure AD)

การลงชื่อเข้าใช้ของ Azure AD มีไว้สำหรับผู้สมัครภายในเท่านั้น ดังนั้น จะใช้ได้สำหรับผู้สมัครภายในที่ใช้ข้อมูลประจำตัว Azure AD ของบริษัทของพวกเขาเท่านั้น ตัวอย่างเช่น ผู้สมัครที่ปัจจุปันเป็นพนักงาน Contoso Ltd ต้องการสมัครงานในบริษัทที่ไม่เกี่ยวข้อง ซึ่งคือ Alpine Ski House ในกรณีนี้ การลงชื่อเข้าใช้จะไม่สำเร็จ หากพนักงานพยายามที่จะใช้ข้อมูลประจำตัว Azure AD ของเขาหรือเธอจากบริษัท Contoso ltd

## <a name="create-and-maintain-a-profile"></a>สร้างและรักษาโฟรไฟล์

หลังจากที่ผู้สมัครลงชื่อเข้าใช้ไปยังไซต์การทำงาน พวกเขาจะสามารถเลือก **โพรไฟล์ของฉัน** บนแถบนำทางด้านบนของหน้า เพื่อสร้างและรักษาโพรไฟล์ของพวกเขาได้ โพรไฟล์รวมข้อมูลส่วนบุคคล ข้อมูลเกี่ยวกับประสบการณ์การทำงาน รายละเอียดการศึกษา เอกสาร ลิงค์ และข้อมูลเกี่ยวกับชุดทักษะ หลังจากที่มีการสร้างโพรไฟล์ ก็จะสามารถใช้เพื่อสมัครงานที่ผู้สมัครมีความสนใจได้ โพรไฟล์ยังช่วย Attract แนะนำงานที่ถูกต้องให้กับผู้สมัครได้ด้วย

## <a name="find-the-right-job"></a>ค้นหางานที่เหมาะสม

บนหน้ารายการงาน ผู้สมัครสามารถค้นหางานเฉพาะ โดยการป้อนเงื่อนไขการค้นหา ฟังก์ชันการค้นหาจะค้นหาเงื่อนไขการค้นหาในตำแหน่งงานและคำอธิบายเกี่ยวกับงาน และแสดงงานที่เกี่ยวข้องเป็นผลลัพธ์ ผู้สมัครสามารถกรองผลลัพธ์ได้ตลอดเวลา โดยขึ้นอยู่กับตำแหน่งงานหรือฟังก์ชันงาน

ผู้สมัครยังสามารถดูชุดของงานที่แนะนำได้บนไซต์การทำงาน งานที่แนะนำให้ผู้สมัครขึ้นอยู่กับใบสมัครที่ผ่านมาของผู้สมัคร โพรไฟล์ และประวัติย่อ

> [!NOTE]
> มีการแสดงคำแนะนำเกี่ยวกับงาน ก็ต่อเมื่อมีการลงรายการบัญชีงานอย่างน้อย 10 งานบนไซต์การทำงาน และเมื่อผู้สมัครได้กรอกข้อมูลโพรไฟล์ของเขาหรือเธอเสร็จสิ้น

## <a name="apply-for-jobs"></a>สมัครงาน

หลังจากที่ผู้สมัครพบงานที่เหมาะสม พวกเขาสามารถสมัครได้โดยใช้ปุ่ม **สมัคร** บนหน้ารายละเอียดงาน ณ จุดนี้ ผู้สมัครสามารถสร้างโพรไฟล์ใหม่ หรือตรวจทานข้อมูลในโพรไฟล์ที่มีอยู่ของพวกเขา ผู้สมัครยังสามารถอัพโหลดประวัติย่อได้ตามความจำเป็น แล้วจากนั้น ส่งใบสมัครงาน

## <a name="check-application-status"></a>ตรวจสอบสถานะใบสมัคร

หลังจากที่ผู้สมัครสมัครอย่างน้อยหนึ่งงาน พวกเขาจะสามารถเลือก **ใบสมัคร** บนแถบนำทางด้านบนของหน้า เพื่อดูใบสมัครที่เปิดและปิดของพวกเขาได้ เมื่อผู้สมัครเปิดหนึ่งในใบสมัครของพวกเขา พวกเขาจะเห็นขั้นปัจจุบันและรายการการดำเนินการใดๆ ที่ค้างอยู่ ที่พวกเขาต้องทำให้เสร็จสมบูรณ์ ตัวอย่างเช่น ถ้าผู้สมัครต้องระบุวันที่เป็นไปได้สำหรับการสัมภาษณ์ส่วนบุคคล หน้าจะแสดงตัวเลือกของเขาหรือเธอ

## <a name="internal-jobs"></a>งานภายใน

ในปัจจุบัน งานที่ถูกทำเครื่องหมายเป็นภายใน และลงรายการบัญชีไปยังไซต์การทำงาน Attract ไม่ปรากฏบนไซต์การทำงาน พวกเขาสามารถเข้าถึงได้โดยผ่านทาง URL **สมัคร** โดยตรง ที่คุณสามารถคัดลอกจากใบสมัคร Attract ได้
