# <a name="set-up-your-career-site-in-attract"></a>ตั้งค่าไซต์รับสมัครงานของคุณ ใน Attract

[!include [banner](includes/banner.md)]

หัวข้อนี้แสดงภาพรวมของฟังก์ชันไซต์การทำงานที่เชื่อมต่อกับผู้สมัครใน Microsoft Dynamics 365 Talent: Attract นอกจากนี้ ยังอธิบายวิธีการตั้งค่าฟังก์ชันนี้อีกด้วย

Attract ให้ไซต์การทำงานหนึ่งไซต์สำหรับแต่ละสภาพแวดล้อมในผู้เช่า ตัวอย่างเช่น ถ้าองค์กรมีสภาพแวดล้อมการพัฒนาและสภาพแวดล้อมการทดสอบ ไซต์การทำงานหนึ่งไซต์จะถูกเตรียมใช้งานสำหรับสภาพแวดล้อมการพัฒนา และไซต์การทำงานอีกหนึ่งไซต์จะถูกเตรียมใช้งานสำหรับสภาพแวดล้อมการทดสอบ ไซต์การทำงานแต่ละไซต์เป็นแบบแยกต่างหากทั้งหมด และมีกลไกการรับรองความถูกต้องของตนเอง งานและโพรไฟล์ผู้สมัครไม่ได้ใช้ร่วมกันระหว่างไซต์การทำงาน

โดยค่าเริ่มต้น ไซต์การทำงานเป็นแบบสาธารณะ ดังนั้น ผู้สมัครสามารถดูงานทั้งหมดที่ทำเครื่องหมายเป็นภายนอกได้ โดยไม่ต้องเข้าสู่ระบบ อย่างไรก็ตาม การดำเนินการอื่นๆ ทั้งหมดกำหนดให้ผู้สมัครต้องเข้าสู่ระบบ

## <a name="career-site-management"></a>การจัดการไซต์อาชีพ

ในการตั้งค่าสำหรับรายการต่อไปนี้ ล็อกอินไปยัง Attract ในฐานะผู้ดูแลระบบ เลือก **ศูนย์การจัดการ** บนเมนู **การตั้งค่า** (สัญลักษณ์เกียร์) และจากนั้นเลือกแท็บ **ข้อมูลของบริษัท**

-   **ชื่อองค์กร:** - ชื่อองค์กรปรากฏขึ้นบนแถบนำทางด้านบนของไซต์การทำงาน โดยการเลือกชื่อองค์กร ผู้สมัครไปยังหน้าซึ่งแสดงรายการงานที่เปิดทั้งหมด

-   **โลโก้องค์กร:** - รูปโลโก้ขององค์กรปรากฏขึ้นในด้านซ้ายบนของไซต์การทำงาน โดยการเลือกรูปโลโก้ ผู้สมัครไปยังหน้าซึ่งแสดงรายการงานที่เปิดทั้งหมด

    > [!NOTE] 
    > รูปโลโก้ที่ปรากฏบนไซต์การทำงานมีความสูงคงที่เป็น 20 พิกเซล (px) รูปภาพที่คุณเพิ่มในศูนย์การจัดการจะถูกปรับขนาดให้พอดี ดังนั้น โดยขึ้นอยู่กับรูปภาพ ความกว้างอาจเปลี่ยนแปลง
 
ในการตั้งค่าสำหรับรายการต่อไปนี้ ล็อกอินไปยัง Attract ในฐานะผู้ดูแลระบบ เลือก **ศูนย์การจัดการ** บนเมนู **การตั้งค่า** (สัญลักษณ์เกียร์) และจากนั้นเลือกแท็บ **การจัดการไซต์อาชีพ**

-   **การเพิ่มประสิทธิภาพกลไกจัดการการค้นหา** - เมื่อเปิดใช้งาน งานสาธารณะทั้งหมดที่ลงรายการบัญชีไซต์การทำงาน Attract จะถูกค้นหา โดยใช้โปรแกรมค้นหา เช่น Bing และ Google 

    > [!NOTE] 
    > อาจมีความล่าช้าระหว่างการเปิดใช้การตั้งค่านี้และผลการค้นหาที่ปรากฏ โดยขึ้นอยู่กับโปรแกรมค้นหาที่คุณกำลังใช้
    
-   **ข้อกำหนดและเงื่อนไข** - เมื่อเปิดใช้งาน ผู้สมัครทั้งหมดต้องยอมรับข้อกำหนดและเงื่อนไขขององค์กร เมื่อสมัครงานใดๆ ผู้ดูแล Attract สามารถตั้งค่าคอนฟิกข้อความยินยอมของตนเอง และลิงค์ไปที่หน้าข้อกำหนดและเงื่อนไขของตนเองได้ 

        
## <a name="career-site-urls"></a>URL ไซต์การทำงาน

รายการต่อไปนี้ประกอบด้วย URL ของไซต์การทำงานที่ใช้กันทั่วไปและวิธีการเข้าถึง

-   **URL โฮมเพจของไซต์การทำงาน** - เมื่อต้องการดู URL โฮมเพจของไซต์การทำงาน ล็อกอินไปยัง Attract ในฐานะผู้ดูแลระบบ เลือก **ศูนย์การจัดการ** บนเมนู **การตั้งค่า** และจากนั้นเลือกแท็บ **การจัดการไซต์การทำงาน**

-   **URL การสมัครโพสต์งานแต่ละรายการ** - [เมื่อคุณโพสต์งานภายนอก](Creating-jobs-Attract.md#postings) เป็นครั้งแรก คุณสามารถคัดลอกลิงค์ **สมัคร** จากแอพลิเคชัน Attract ได้ URL สำหรับการเชื่อมโยงนี้จะอยู่ในรูปแบบต่อไปนี้: [https://jobs.talent.dynamics.com/jobs/\<company_name\>/\<environment_number\>/\<job_number\>/apply](https://jobs.talent.dynamics.com/jobs/%3ccompany_name%3e/%3cenvironment_number%3e/%3cjob_number%3e/apply)

-   **URL ของโพสต์งานแต่ละรายการ** - URL ของโพสต์งานเป็นสตริงย่อยของ URL การสมัคร ซึ่งประกอบด้วยข้อมูลทั้งหมดไปจนถึงหมายเลขงาน ดังนั้น สำหรับ URL การสมัครก่อนหน้า URL ของโพสต์งานคือ [https://jobs.talent.dynamics.com/jobs/\<company_name\>/\<environment_number\>/\<job_number\>](https://jobs.talent.dynamics.com/jobs/%3ccompany_name%3e/%3cenvironment_number%3e/%3cjob_number%3e)

## <a name="authentication-options"></a>ตัวเลือกการรับรองความถูกต้อง

ผู้สมัครมีตัวเลือกการลงชื่อเข้าใช้ดังต่อไปนี้สำหรับไซต์การทำงาน Attract:

-   บัญชีส่วนตัว:

    -   LinkedIn

    -   Microsoft

    -   Google

    -   Facebook

-   บัญชีที่ทำงานหรือที่โรงเรียน:

    -   Microsoft Azure Active Directory (Azure AD)

การลงชื่อเข้าใช้ของ Azure AD มีไว้สำหรับผู้สมัครภายในเท่านั้น ดังนั้น จะใช้ได้สำหรับผู้สมัครภายในที่ใช้ข้อมูลประจำตัว Azure AD ของบริษัทของพวกเขาเท่านั้น ตัวอย่างเช่น ผู้สมัครที่ปัจจุปันเป็นพนักงานของ Contoso Ltd ต้องการสมัครงานในบริษัทที่ไม่เกี่ยวข้อง ซึ่งคือ Alpine Ski House ในกรณีนี้ การลงชื่อเข้าใช้จะไม่สำเร็จ หากพนักงานพยายามที่จะใช้ข้อมูลประจำตัว Azure AD ของเขาหรือเธอจากบริษัท Contoso ltd 

ผู้สมัครต้องลงชื่อเข้าใช้โดยใช้ Azure AD ถ้างานที่พวกเขากำลังดู หรือกำลังสมัคร ถูกแสดงรายการเป็นแบบภายในเท่านั้น

## <a name="create-and-maintain-a-profile"></a>สร้างและรักษาโพรไฟล์

หลังจากที่ผู้สมัครลงชื่อเข้าใช้ไปยังไซต์การทำงาน พวกเขาจะสามารถเลือก **โพรไฟล์ของฉัน** บนแถบนำทางที่ด้านบนของหน้า เพื่อสร้างและรักษาโพรไฟล์ของพวกเขาได้
โพรไฟล์รวมข้อมูลส่วนบุคคล ข้อมูลเกี่ยวกับประสบการณ์การทำงาน รายละเอียดการศึกษา เอกสาร ลิงค์ และข้อมูลเกี่ยวกับชุดทักษะ หลังจากที่มีการสร้างโพรไฟล์ ก็จะสามารถใช้เพื่อสมัครงานที่ผู้สมัครมีความสนใจได้ โพรไฟล์ยังช่วย Attract แนะนำงานที่เหมาะสมให้กับผู้สมัครได้ด้วย

> [!NOTE]
> ถ้าผู้สมัครใช้รหัสอีเมลเพื่อล็อกอิน โดยใช้หนึ่งในตัวให้บริการการรับรองความถูกต้องที่แสดงรายการข้างต้น รหัสอีเมลนั้นจะมีค่าเริ่มต้นเป็นรหัสอีเมลของผู้ติดต่อที่เชื่อมโยงกับโพรไฟล์ อย่างไรก็ตาม จะสามารถเปลี่ยนได้ตลอดเวลา และเป็นอิสระโดยสิ้นเชิงจากของเดิม Attract จะใช้รหัสอีเมลของผู้ติดต่อเสมอ เพื่อเชื่อมโยงกับโพรไฟล์ของคุณสำหรับการสื่อสารทางอีเมลทั้งหมด

## <a name="find-the-right-job"></a>ค้นหางานที่เหมาะสม

บนหน้ารายการงาน ผู้สมัครสามารถค้นหางานเฉพาะได้โดยการป้อนเงื่อนไขการค้นหา ฟังก์ชันการค้นหาจะค้นหาเงื่อนไขการค้นหาในตำแหน่งงานและคำอธิบายเกี่ยวกับงาน และแสดงงานที่เกี่ยวข้องเป็นผลลัพธ์ ผู้สมัครสามารถกรองผลลัพธ์ได้ตลอดเวลา โดยขึ้นอยู่กับตำแหน่งงานหรือฟังก์ชันงาน

ผู้สมัครยังสามารถดูชุดของงานที่แนะนำได้บนไซต์การทำงานได้ด้วย งานที่แนะนำให้ผู้สมัครขึ้นอยู่กับใบสมัคร โพรไฟล์ และประวัติย่อ ที่ผ่านมาของผู้สมัคร

> [!NOTE] 
> มีการแสดงคำแนะนำเกี่ยวกับงาน ก็ต่อเมื่อมีการลงรายการบัญชีงานอย่างน้อย 10 งานบนไซต์การทำงาน และเมื่อผู้สมัครได้กรอกข้อมูลโพรไฟล์เสร็จสิ้น

นอกจากนี้ ผู้สมัครภายในยังสามารถดูผู้ที่เป็นผู้จัดการการว่าจ้าง และ/หรือผู้สรรหางานสำหรับงาน ในกรณีที่พวกเขาต้องการติดต่อสมาชิกเหล่านั้นของทีมงานการว่าจ้าง อย่างไรก็ตาม ผู้สมัครภายนอกไม่มีความสามารถในการมองเห็นในสมาชิกของทีมงานว่าจ้างสำหรับงานใดๆ

## <a name="contact-the-hiring-team"></a>ติดต่อทีมงานการจ้างงาน
ผู้สมัครภายในเท่านั้นสามารถติดต่อทีมงานการว่าจ้างได้ ข้อจำกัดนี้ใช้กับงานทั้งหมด ไม่ว่าจะอยู่ภายในเท่านั้น หรือถูกลงรายการบัญชีแบบสาธารณะ

ผู้สมัครอาจต้องการติดต่อทีมงานการว่าจ้างเพื่อแสดงดอกเบี้ยในงานที่มีการลงรายการบัญชี หรือฃเรียนรู้เพิ่มเติมเกี่ยวกับการว่าจ้าง พวกเขาสามารถติดต่อสมาชิกทีมว่าจ้างใดๆ ที่ถูกแสดงรายการ (ผู้จัดการที่ว่าจ้างหรือผู้สรรหา) นอกจากนี้พวกเขาอาจยังสามารถเลือกที่จะแนบประวัติย่อกับข้อความ หรือเลือกประวัติย่อที่มีอยู่ที่อัพโหลดไว้ก่อนหน้านี้เป็นส่วนหนึ่งของโปรไฟล์ของพวกเขาได้

หลังจากที่ผู้สมัครภายในเลือกสมาชิกทีมงานว่าจ้างเพื่อติดต่อ Attract จะส่งอีเมลให้บุคคลเหล่านั้นในนามของผู้สมัคร ในเวลาเดียวกัน โพรไฟล์ของผู้สมัครจะถูกเพิ่มไปยังขั้น **ผู้ที่มีแนวโน้มจะเป็นลูกค้า** หากขั้นดังกล่าวพร้อมใช้งานสำหรับงานนั้นๆ ภายใต้ขั้น **ผู้ที่มีแนวโน้มจะเป็นลูกค้า** ผู้สรรหาหรือผู้จัดการว่าจ้างสามารถดูผู้สมัครที่ได้ติดต่อแล้ว พวกเขาสามารถตรวจทานโพรไฟล์ผู้สมัครและเชิญผู้สมัครที่เป็นไปได้ให้สมัคร

ผู้สมัครสามารถสมัครสำหรับงานที่พวกเขาได้ติดต่อสมาชิกทีมงานการจ้างงานแล้ว หลังจากที่สมัคร ผู้สมัครจะไม่สามารถติดต่อทีมงานการจ้างงานผ่านทางไซต์รับสมัครงานได้อีก

## <a name="apply-for-jobs"></a>สมัครงาน

หลังจากที่ผู้สมัครพบงานที่เหมาะสม พวกเขาสามารถสมัครได้โดยใช้ ปุ่ม  **สมัคร** บนหน้า **รายละเอียดงาน** ณ จุดนี้ ผู้สมัครสามารถสร้างโพรไฟล์ใหม่ หรือตรวจทานข้อมูลในโพรไฟล์ที่มีอยู่ของพวกเขาได้ อย่างใดอย่างหนึ่ง
ผู้สมัครยังสามารถอัพโหลดประวัติย่อได้ตามความจำเป็น แล้วจากนั้น ส่งใบสมัครงาน

### <a name="enable-applying-for-jobs-with-linkedin-profiles"></a>เปิดใช้งานการสมัครสำหรับงานที่มีโพรไฟล์ LinkedIn

คุณสามารถช่วยให้ผู้สมัครสามารถสมัครตำแหน่งของคุณไดง่ายโดยการตั้งค่าคอนฟิก Attract เพื่ออนุญาตให้สมัครผ่าน LinkedIn

> [!NOTE] 
> คุณต้องมีสิทธิ์การใช้งานของผู้สรรหาจาก LinkedIn อย่างน้อยหนึ่งรายการก่อนที่คุณจะสามารถอนุญาตให้ผู้สมัครสมัครกับ LinkedIn

1. ลงชื่อเข้าใช้ Attract เป็นผู้ดูแลระบบ
2. เลือกปุ่ม **การตั้งค่า** (สัญลักษณ์เกียร์) ในมุมขวาบนของหน้า และจากนั้น **ศูนย์การจัดการ**
3. เลือกแท็บ **การรวม LinkedIn** และเชื่อมโยงกับบัญชี LinkedIn Recruiter
4. ในส่วน **การรวม LinkedIn Recruiter System Connect** เลือก **เปิดใช้งานแล้ว** สำหรับการตั้งค่า **สมัครกับ LinkedIn**

หลังจากที่คุณได้เปิดใช้งานการตั้งค่า ผู้สมัครสามารถสมัครโดยใช้ข้อมูลในโพรไฟล์ LinkedIn ที่มีอยู่ได้ เมื่อผู้สมัครสมัครโดยการเลือกปุ่ม **สมัครกับ LinkedIn** ระบบจะขอให้พวกเขารับรองความถูกต้องกับ LinkedIn ถ้าพวกเขาไม่ได้ล็อกอินอยู่ หลังจากที่พวกเขาได้รับรองความถูกต้องแล้ว โพรไฟล์ LinkedIn ของพวกเขาจะแทนข้อมูลในโพรไฟล์ที่มีอยู่ใดๆ ที่แสดงในหน้าใบสมัคร ผู้สมัครสามารถแก้ไขข้อมูลตามความจำเป็นก่อนแล้วส่งใบสมัคร ถ้าผู้สมัครนำทางออกจากหน้านั้นโดยไม่ได้สมัครงาน ข้อมูลในโพรไฟล์ของพวกเขาจะไม่มีการปรับปรุงใน Attract

## <a name="check-application-status"></a>ตรวจสอบสถานะใบสมัคร

หลังจากที่ผู้สมัครสมัครงานอย่างน้อยหนึ่งงาน พวกเขาจะสามารถเลือก **ใบสมัคร** บนแถบนำทางที่ด้านบนของหน้า เพื่อดูใบสมัครที่เปิดและปิดของพวกเขาได้ เมื่อผู้สมัครเปิดหนึ่งในใบสมัครของพวกเขา พวกเขาจะเห็นขั้นตอนปัจจุบันและรายการการดำเนินการใดๆ ที่ค้างอยู่ ที่พวกเขาต้องทำให้เสร็จสมบูรณ์ ตัวอย่างเช่น ถ้าผู้สมัครต้องระบุวันที่ที่เป็นไปได้สำหรับการสัมภาษณ์ส่วนบุคคล หน้าจะแสดงตัวเลือกที่พร้อมใช้งาน

## <a name="internal-jobs"></a>งานภายใน

ในปัจจุบัน งานที่ถูกทำเครื่องหมายเป็นภายใน และถูกลงรายการบัญชีไปยังไซต์การทำงาน Attract ไม่ปรากฏบนไซต์การทำงาน รายการเหล่านั้นสามารถเข้าถึงได้โดยใช้ทาง URL **สมัคร** โดยตรง ที่คุณสามารถคัดลอกจากใบสมัคร Attract ได้
