# <a name="view-the-status-of-onboarding-employees-and-your-tasks"></a>ดูสถานะพนักงานการเตรียมความพร้อมและงานของคุณ

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboard ช่วยให้คุณสามารถดูความคืบหน้าของพนักงานใหม่ของคุณได้ในกิจกรรมการเตรียมความพร้อมของพวกเขา นอกจากนี้คุณยังสามารถดูงานต่างๆที่ได้รับการกำหนดให้กับคุณจากคู่มือการเตรียมความพร้อม

## <a name="view-the-status-of-onboarding-employees"></a>ดูสถานะของพนักงานการเตรียมความพร้อม

1. บนเมนูด้านซ้าย ให้เลือก **คู่มือ** คุณมองเห็นภาพรวมของความคืบหน้าสำหรับคู่มือการเตรียมความพร้อมทั้งหมดที่คุณส่งได้

    ![[ดูสถานะของคู่มือการเตรียมความพร้อม](./media/onboard-guide-status.png)](./media/onboard-guide-status.png)

2. ถ้าต้องการดูรายละเอียดแนวลึก ให้เลือกคู่มือที่คุณต้องการดู
3. เลือกแท็บ **กิจกรรม** เพื่อดูกิจกรรมที่พนักงานใหม่ของคุณทำเสร็จแล้ว

    ![[ดูกิจกรรมที่พนักงานใหม่ทำเสร็จแล้ว](./media/onboard-status-activities.png)](./media/onboard-status-activities.png)

## <a name="view-onboarding-tasks-that-are-assigned-to-you"></a>ดูงานการเตรียมพร้อมที่กำหนดให้กับคุณ

1. บนเมนูด้านซ้าย ให้เลือก **งาน** คุณสามารถดูภาพรวมของงานการเตรียมพร้อมทั้งหมดที่ได้รับการกำหนดให้กับคุณจากคู่มือการเตรียมพร้อมที่ส่งให้แก่พนักงานแล้ว
2. ถ้าต้องการดูรายละเอียดแนวลึก ให้เลือกงานที่ต้องการ
3. ติดตามพนักงานใหม่ตามความจำเป็น
4. เมื่อคุณทำงานเสร็จเรียบร้อยแล้ว ให้ทำเครื่องหมายเป็นเสร็จสมบูรณ์โดยการเพิ่มเครื่องหมายถูกลงในวงกลม
