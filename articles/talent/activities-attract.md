# <a name="activities-in-the-hiring-processes"></a>กิจกรรมในกระบวนการว่าจ้าง

[!include[banner](../includes/banner.md)]

สามารถเพิ่มกิจกรรมเป็นส่วนหนึ่งของกระบวนการว่าจ้างได้ใน Microsoft Dynamics 365 for Talent: Attract กิจกรรมสามารถถูกเพิ่มไปยังเท็มเพลตกระบวนการ หรือสามารถเพิ่มไปยังกระบวนการว่าจ้างในงานนี้ได้โดยตรง เมื่อมีการกำหนดงาน จะมีการเลือกเท็มเพลตกระบวนการ และกิจกรรมที่จะรวมอยู่ในเท็มเพลตที่จะใช้กับงาน ถ้าไม่ได้มีการเลือกเท็มเพลต จะมีการใช้เท็มเพลตเริ่มต้น กระบวนการว่าจ้างยังสามารถปรับเปลี่ยนในงานนี้ได้ หลังจากที่ใช้เท็มเพลต

> [!NOTE] 
> เท็มเพลตกระบวนการจะพร้อมใช้งานโดยมี add-on การว่าจ้างที่ครอบคลุม

## <a name="prospect-activity"></a>กิจกรรมผู้ที่มีแนวโน้มจะเป็นลูกค้า

กิจกรรมผู้ที่มีแนวโน้มจะเป็นลูกค้าควบคุมว่า สามารถเพิ่มลูกค้าให้กับงานได้หรือไม่ โดยค่าเริ่มต้น สามารถเพิ่มผู้ที่มีแนวโน้มจะเป็นลูกค้าไปยังงานได้ เมื่อต้องการปิดกิจกรรมผู้ที่มีแนวโน้มจะเป็นลูกค้า ตั้งค่าตัวเลือก **เปิดใช้งานผู้ที่มีแนวโน้มจะเป็นลูกค้า** เป็น **ปิด** เมื่อเปิดกิจกรรมผู้ที่มีแนวโน้มจะเป็นลูกค้า ผู้จัดการว่าจ้างสามารถเพิ่ม และดูผู้ที่มีแนวโน้มจะเป็นลูกค้า และแท็บ **ผู้ที่มีแนวโน้มจะเป็นลูกค้า** จะแสดงขึ้นบนงานได้

## <a name="application-activity"></a>กิจกรรมใบสมัคร

ต้องมีกิจกรรมใบสมัครในเท็มเพลตกระบวนการว่าจ้าง เพื่อส่งอีเมลไปที่ผู้สมัคร เมื่อพวกเขาส่งใบสมัครของตน หรือถูกเพิ่มลงในขั้นตอนใบสมัคร ตั้งค่าตัวเลือก **ส่งจดหมายไปยังผู้สมัคร** เป็น **เปิด**

## <a name="scheduler-activity"></a>กิจกรรมตัวกำหนดตารางทำงาน

ไม่จำเป็นต้องมีกิจกรรมตัวกำหนดตารางทำงาน กิจกรรมนี้มีส่วนประกอบสองรายการ: ความพร้อมของผู้สมัครและกำหนดการ คอมโพเนนต์ความพร้อมของผู้สมัครช่วยให้คุณสามารถใช้อีเมลเพื่อขอความพร้อมใช้งานของผู้สมัครได้ คอมโพเนนต์กำหนดการช่วยให้มีความสามารถในการจัดกำหนดการสัมภาษณ์กับผู้สมัครและทีมการว่าจ้าง ในกิจกรรมตัวกำหนดตารางทำงาน สามารถตั้งค่าคอนฟิกตัวเลือกต่อไปนี้ได้: **ร้องขอความพร้อมของผู้สมัคร** **การประชุมแบบออนไลน์** และ **ส่งจดหมายไปยังผู้สมัคร**

- เพื่อส่งอีเมลไปที่ผู้สมัครเพื่อร้องขอความพร้อม ตั้งค่าตัวเลือก **ร้องขอความพร้อมของผู้สมัคร** เป็น **เปิด** ถ้าคุณตั้งค่าตัวเลือกเป็น **ปิด** ขั้นตอนนี้จะไม่แสดงขึ้นในกระบวนการว่าจ้างในงาน
- ในการสตรีมสด หรือมีการประชุมทางโทรศัพท์ โดยใช้ Skype for Business ตั้งค่าฟิลด์ **การประชุมแบบออนไลน์** เป็น **Skype for Business** จากนั้น ลิงค์ **ร่วมการประชุม Skype** ที่ถูกต้อง จะถูกเพิ่มลงในคำขอการประชุมสัมภาษณ์ที่ถูกส่งไปยังผู้สัมภาษณ์
- เพื่อส่งอีเมลไปที่ผู้สมัครเพื่อสรุปกำหนดการขั้นสุดท้าย ตั้งค่าตัวเลือก **ส่งจดหมายไปยังผู้สมัคร** เป็น **เปิด** ถ้าคุณตั้งค่าตัวเลือกเป็น **ปิด** ผู้สมัครจะได้รับกำหนดการสัมภาษณ์เฉพาะเมื่อพวกเขาลงชื่อเข้าใช้ไปยังพอร์ทัลผู้สมัคร

## <a name="feedback-activity"></a>กิจกรรมผลป้อนกลับ

ไม่จำเป็นต้องมีกิจกรรมผลป้อนกลับ กิจกรรมนี้อนุญาตให้ผู้เข้าร่วมในการสัมภาษณ์ป้อนคำแนะนำสำหรับผู้สมัคร พวกเขายังสามารถป้อนข้อคิดเห็นเกี่ยวกับผลป้อนกลับที่พวกเขามี ถ้าคุณเปิดตัวเลือก **สืบทอดผู้เข้าร่วมในผลป้อนกลับจากทีมการว่าจ้าง** ผู้สรรหา ผู้จัดการว่าจ้าง และผู้สัมภาษณ์ ถูกป้อนโดยอัตโนมัติในกิจกรรมผลป้อนกลับ องค์กรสามารถอนุญาตให้ผู้สัมภาษณ์สามารุดูผลป้อนกลับของบุคคลอื่น ก่อนที่จะส่งผลป้อนกลับของพวกเขาเอง องค์กรยังสามารถอนุญาตให้ผู้สัมภาษณ์สามารถแก้ไขผลป้อนกลับของพวกเขา หลังจากที่ส่ง

## <a name="interview-activity"></a>กิจกรรมการสัมภาษณ์

ไม่จำเป็นต้องมีกิจกรรมการสัมภาษณ์ กิจกรรมนี้มีส่วนประกอบสามรายการ: ความพร้อมของผู้สมัคร กำหนดการ และผลป้อนกลับ คอมโพเนนต์ความพร้อมของผู้สมัครช่วยให้คุณสามารถใช้อีเมลเพื่อขอความพร้อมใช้งานของผู้สมัครได้ คอมโพเนนต์กำหนดการช่วยให้มีความสามารถในการจัดกำหนดการสัมภาษณ์กับผู้สมัครและทีมการว่าจ้าง ในกิจกรรมตัวกำหนดตารางทำงาน สามารถตั้งค่าคอนฟิกตัวเลือกต่อไปนี้ได้: **ร้องขอความพร้อมของผู้สมัคร** **การประชุมแบบออนไลน์** และ **ส่งจดหมายไปยังผู้สมัคร**

- เพื่อส่งอีเมลไปที่ผู้สมัครเพื่อร้องขอความพร้อม ตั้งค่าตัวเลือก **ร้องขอความพร้อมของผู้สมัคร** เป็น **เปิด** ถ้าคุณตั้งค่าตัวเลือกเป็น **ปิด** ขั้นตอนนี้จะไม่แสดงขึ้นในกระบวนการว่าจ้างในงาน
- ในการสตรีมสด หรือมีการประชุมทางโทรศัพท์ โดยใช้ Skype for Business ตั้งค่าฟิลด์ **การประชุมแบบออนไลน์** เป็น **Skype for Business** จากนั้น ลิงค์ **ร่วมการประชุม Skype** ที่ถูกต้อง จะถูกเพิ่มลงในคำขอการประชุมสัมภาษณ์
- เพื่อส่งอีเมลไปที่ผู้สมัครเพื่อสรุปกำหนดการขั้นสุดท้าย ตั้งค่าตัวเลือก **ส่งจดหมายไปยังผู้สมัคร** เป็น **เปิด** ถ้าคุณตั้งค่าตัวเลือกเป็น **ปิด** ผู้สมัครจะได้รับกำหนดการสัมภาษณ์เฉพาะเมื่อพวกเขาลงชื่อเข้าใช้ไปยังพอร์ทัลผู้สมัคร

>[!NOTE]
> - สำหรับการสัมภาษณ์ 1:1 ทั้งหมด จดหมายเตือนถูกส่งไปยังผู้สัมภาษณ์ทุก 24 ชั่วโมง ถ้าผู้สัมภาษณ์ไม่ตอบสนอง (ยอมรับ หรือปฏิเสธ) ต่อการร้องขอการสัมภาษณ์
> - สำหรับการสัมภาษณ์แผงทั้งหมด จะไม่มีข้อความเตือนอัตโนมัติเพื่อตอบสนองต่อการร้องขอการสัมภาษณ์ ในการทริกเกอร์ข้อความเตือนด้วยตนเอง แก้ไขการสัมภาษณ์ และใช้ตัวเลือก **อัพเดต & ส่ง** เพื่อส่งคำขอไปที่ผู้สัมภาษณ์

คอมโพเนนต์ผลป้อนกลับช่วยให้ผู้ใช้สามารถป้อนคำแนะนำสำหรับผู้สมัคร พวกเขายังสามารถป้อนข้อคิดเห็นเกี่ยวกับผลป้อนกลับที่พวกเขามี ถ้าคุณเปิดตัวเลือก **สืบทอดผู้เข้าร่วมในผลป้อนกลับจากทีมการว่าจ้าง** ผู้สรรหา ผู้จัดการว่าจ้าง และผู้สัมภาษณ์ ถูกป้อนโดยอัตโนมัติในคอมโพเนนต์ผลป้อนกลับ องค์กรสามารถอนุญาตให้ผู้สัมภาษณ์สามารุดูผลป้อนกลับของบุคคลอื่น ก่อนที่จะส่งผลป้อนกลับของพวกเขาเอง องค์กรยังสามารถอนุญาตให้ผู้สัมภาษณ์สามารถแก้ไขผลป้อนกลับของพวกเขา หลังจากที่ส่ง

## <a name="powerapps-activity"></a>กิจกรรม PowerApps

กิจกรรม PowerApps ช่วยให้คุณสามารถฝังแอป Microsoft PowerApps ในกระบวนการว่าจ้างของคุณได้ แอปอาจจำเป็นสำหรับผู้สมัครทั้งหมด ผู้สมัครภายในเท่านั้น ผู้สมัครภายนอกเท่านั้น หรือไม่มีผู้สมัคร ถ้ามีการทำเครื่องหมายแอปตามต้องการ จะต้องทำให้เสร็จสิ้นก่อนที่สามารถเลื่อนขั้นได้ ถ้าไม่ได้ทำเครื่องหมายแอปตามต้องการ กิจกรรมเป็นขั้นตอนที่ไม่จำเป็นต้องระบุ และสามารถเลื่อนขั้นได้ถึงแม้ว่าแอปจะไม่เสร็จสมบูรณ์

ในการบันทึกกิจกรรม PowerApps ไปยังกระบวนการว่าจ้าง คุณต้องป้อนรหัส PowerApps เมื่อต้องการค้นหารหัส PowerApps ไปยัง [PowerApps](https://web.powerapps.com) เลือก **แอป** แล้วจากนั้น เลือก **รายละเอียด**

ถ้าคุณเลือกตัวเลือก **อนุญาตให้เพิ่มผู้เข้าร่วมสำหรับกิจกรรมนี้** สามารถเพิ่มผู้สนับสนุนเพิ่มเติมสำหรับแอพลิเคชันที่ใช้กิจกรรม PowerApps ได้ ตัวอย่างเช่น องค์กรได้สร้างแอป PowerApps ที่เป็นไลบรารีของคำถามสัมภาษณ์สำหรับบทบาททางเทคนิค ขณะนี้ องค์กรว่าจ้างนักพัฒนาซอฟต์แวร์ใหม่ และได้เพิ่มกิจกรรม PowerApps ลงในกระบวนการว่าจ้างสำหรับบทบาทนักพัฒนาซอฟต์แวร์ ถ้าตัวเลือก **อนุญาตให้เพิ่มผู้เข้าร่วมสำหรับกิจกรรมนี้** ผู้จัดการที่ว่าจ้างหรือผู้สรรหาที่ดูผู้สมัครสำหรับบทบาทนักพัฒนาซอฟต์แวร์ สามารถเพิ่มบุคคลลงในกิจกรรม PowerApps ได้ บุคคลเหล่านั้นสามารถดูแอปที่มีคำถามสัมภาษณ์

> [!NOTE]
> กิจกรรม PowerApps จะพร้อมใช้งานเฉพาะกับ add-on การว่าจ้างที่ครอบคลุม

## <a name="youtube-activity"></a>กิจกรรม YouTube

กิจกรรม YouTube ช่วยให้คุณสามารถแชร์วิดีโอ YouTube เป็นส่วนหนึ่งของกระบวนการว่าจ้างของคุณได้ ในการบันทึกกิจกรรม YouTube ไปยังกระบวนการว่าจ้าง คุณต้องระบุ URL ของวิดีโอ YouTube เช่นกันกับกิจกรรม PowerApps คุณสามารถอนุญาตให้ผู้เข้าร่วมสามารถถูกเพิ่มให้กับกิจกรรมได้ ถ้าคุณเลือกตัวเลือก **แสดงเฉพาะกับผู้สมัคร** วิดีโอจะปรากฏเป็นส่วนหนึ่งของประสบการณ์ผู้สมัครเท่านั้น จึงไม่ได้แสดงในกระบวนการว่าจ้างใน Attract

> [!NOTE]
> กิจกรรม YouTube จะพร้อมใช้งานเฉพาะกับadd-on การว่าจ้างที่ครอบคลุม

## <a name="web-content-activity"></a>กิจกรรมเนื้อหาเว็บ

กิจกรรมเนื้อหาเว็บช่วยให้คุณสามารถฝังเนื้อหาออนไลน์ในกระบวนการว่าจ้างของคุณได้ ในการบันทึกกิจกรรมเนื้อหาเว็บไปยังกระบวนการว่าจ้าง คุณต้องระบุ URL ของเนื้อหา เช่นกันกับกิจกรรม PowerApps และ YouTube คุณสามารถอนุญาตให้ผู้เข้าร่วมสามารถถูกเพิ่มให้กับกิจกรรมได้ ถ้าคุณเลือกตัวเลือก **แสดงเฉพาะกับผู้สมัคร** เนื้อหาจะปรากฏเป็นส่วนหนึ่งของประสบการณ์ผู้สมัครเท่านั้น จึงไม่ได้แสดงในกระบวนการว่าจ้างใน Attract คุณสามารถเลือกขนาดของเนื้อหาที่จะแสดงขึ้น

> [!NOTE]
> กิจกรรมเนื้อหาเว็บพร้อมใช้งานเฉพาะกับ add-on การว่าจ้างที่ครอบคลุม

## <a name="microsoft-forms-activity"></a>กิจกรรม Microsoft Forms

กิจกรรม Microsoft Forms ช่วยให้คุณสามารถฝังฟอร์ม Microsoft Forms ในกระบวนการว่าจ้างของคุณได้ Microsoft Forms ช่วยให้คุณสามารถสร้างแบบทดสอบ การสำรวจความคิดเห็น และการสำรวจได้ เมื่อต้องการบันทึกกิจกรรม Microsoft Forms ไปยังกระบวนการว่าจ้าง คุณต้องระบุ URL ของแบบฟอร์ม เช่นกันกับ PowerApps YouTube และกิจกรรมเนื้อหาเว็บ คุณสามารถอนุญาตให้ผู้เข้าร่วมสามารถถูกเพิ่มให้กับกิจกรรมได้ ถ้าคุณเลือกตัวเลือก **แสดงเฉพาะกับผู้สมัคร** แบบฟอร์มจะปรากฏเป็นส่วนหนึ่งของประสบการณ์ผู้สมัครเท่านั้น จึงไม่ได้แสดงในกระบวนการว่าจ้างใน Attract

ใน Microsoft Forms ผู้เขียนสามารถเปลี่ยนการตั้งค่า เพื่อให้ผู้ใช้ภายนอกองค์กรของพวกเขาตอบสนองการสำรวจความคิดเห็นหรือแบบทดสอบของพวกเขา ในกรณีนี้ ผู้ใช้ส่งการตอบสนองโดยไม่ระบุชื่อ ถ้าคุณต้องการดูผู้ที่ได้กรอกแบบสอบถามหรือแบบทดสอบของคุณ คุณอาจต้องการให้ผู้ตอบป้อนชื่อให้เป็นส่วนหนึ่งของการสำรวจความคิดเห็นหรือแบบทดสอบ
