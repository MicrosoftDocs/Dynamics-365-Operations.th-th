# <a name="welcome-to-dynamics-365-talent"></a>ยินดีต้อนรับสู่ Dynamics 365 Talent

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent ช่วยลดขั้นตอนซ้ำซ้อนของงานเก็บเรกคอร์ดประจำและทำให้กระบวนการหลายอย่างที่เกี่ยวข้องกับการจัดหาพนักงานให้กับองค์กรของคุณทำงานโดยอัตโนมัติ กระบวนการเหล่านี้รวมถึงการดูแลรักษาพนักงาน การจัดการสวัสดิการ การฝึกอบรม การตรวจสอบประสิทธิภาพการทำงาน และการจัดการการเปลี่ยนแปลง และยังให้กรอบงานสำหรับพนักงานฝ่ายทรัพยากรบุคคลในการจัดการพื้นที่ควบคุมดูแล

คุณสามารถใช้ Talent เพื่อดำเนินงานเหล่านี้ให้เสร็จสิ้น:

+ จัดการโครงสร้างองค์กร
+ รักษาข้อมูลผู้ปฏิบัติงานที่ครอบคลุมจากการจ้างงานถึงการปลดเกษียณ
+ กำหนดและจัดการแผนสวัสดิการ ลงทะเบียนผู้ปฏิบัติงาน กำหนดความครอบคลุมผู้อยู่ในอุปการะ และแต่งตั้งผู้รับผลประโยชน์
+ สร้างและตรวจสอบนโยบายการขาดงาน
+ ใช้และติดตามการจัดการเวลาตามโพรไฟล์ และสร้างข้อมูลของค่าจ้างที่จะส่งออกไปยังระบบค่าจ้าง
+ จัดการความสามารถของผู้ปฏิบัติงาน
+ ตรวจทานประสิทธิภาพการทำงานและนำเป้าหมายของผู้ปฏิบัติงานมาใช้
+ ตั้งค่า นำเสนอ และวิเคราะห์หลักสูตรการฝึกอบรมที่มีวาระการประชุม เซสชัน และแทร็ค

สำรวจ Dynamics 365 Talent โดยการดูวิดีโอ [วิธีการเริ่มต้นใช้งาน Dynamics 365 Talent](https://www.youtube.com/watch?v=6rg2ByadbN0)


> [!NOTE] 
> Dynamics 365 Talent เกี่ยวข้องกันอย่างใกล้ชิดกับทรัพยากรบุคคลใน Dynamics 365 Finance ฟังก์ชันเดียวกันอาจปรากฏในทั้งสองผลิตภัณฑ์ ดังนั้น ในหัวข้อที่เกี่ยวข้องกับทรัพยากรบุคคลเป็นหลัก ชื่อผลิตภัณฑ์จะเป็น Dynamics 365 Finance ส่วนในหัวข้อที่เกี่ยวข้องกับ Talent เป็นหลัก ชื่อผลิตภัณฑ์จะเป็น Dynamics 365 Talent หัวข้อที่เขียนขึ้นสำหรับหนึ่งผลิตภัณฑ์อาจนำไปใช้กับฟังก์ชันเดียวกันในผลิตภัณฑ์ที่เกี่ยวข้องได้
