## <a name="overview"></a><span data-ttu-id="ea9a5-104">ภาพรวม</span><span class="sxs-lookup"><span data-stu-id="ea9a5-104">Overview</span></span>

<span data-ttu-id="ea9a5-105">องค์กรที่มีพนักงานรายชั่วโมง สามารถให้รางวัลเวลาหยุดพักตามชั่วโมงที่ทำงาน แทนอายุงานกับองค์กร</span><span class="sxs-lookup"><span data-stu-id="ea9a5-105">Organizations with hourly employees can award time off based on hours worked instead of tenure with the organization.</span></span> <span data-ttu-id="ea9a5-106">โดยทั่วไป ข้อมูลชั่วโมงที่ทำงานถูกจัดเก็บในระบบเวลาและการเข้างาน</span><span class="sxs-lookup"><span data-stu-id="ea9a5-106">The hours worked data is typically stored in a time and attendance system.</span></span> <span data-ttu-id="ea9a5-107">ใน Talent: Core HR ชั่วโมงการทำงานล่วงเวลาและปกติเหล่านั้น สามารถนำเข้า และใช้เป็นพื้นฐานสำหรับรางวัลของพนักงาน</span><span class="sxs-lookup"><span data-stu-id="ea9a5-107">In Talent: Core HR, those regular and overtime hours worked can be imported and used as a basis for an employee's award.</span></span>

<span data-ttu-id="ea9a5-111">ในการตั้งค่าแผนการลางานให้ใช้ชั่วโมงที่ทำงาน ให้ทำตามขั้นตอนเหล่านี้:</span><span class="sxs-lookup"><span data-stu-id="ea9a5-111">To set up a leave plan to use hours worked, follow these steps:</span></span>

3. <span data-ttu-id="ea9a5-114">เลือกความถี่ในการรับรู้รายได้รายจ่ายสำหรับแผน</span><span class="sxs-lookup"><span data-stu-id="ea9a5-114">Select the accrual frequency for the plan.</span></span>
