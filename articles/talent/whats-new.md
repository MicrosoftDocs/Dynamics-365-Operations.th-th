# <a name="whats-new-or-changed-in-dynamics-365-for-talent"></a>มีอะไรใหม่หรือการเปลี่ยนแปลงอะไรใน Dynamics 365 for Talent 

[!include [banner](includes/banner.md)]

หน้านี้ให้รายละเอียดเกี่ยวกับวิธีการค้นหาข้อมูลเกี่ยวกับการนำออกใช้ล่าสุดของ Microsoft Dynamics Talent HR Core

แต่ละสัปดาห์จะมีรุ่นใหม่สำหรับ Talent รุ่นเหล่านี้จะแสดงรายการในสารบัญบนไซต์ docs.microsoft.com ([docs.microsoft.com/dynamics365/unified-operations](../index.md)) คุณสามารถสแกนสารบัญ หรือใช้ตัวควบคุมการกรองที่อยู่เหนือสารบัญ อย่างใดอย่างหนึ่ง เพื่อค้นหาหัวข้อล่าสุดที่คุณกำลังค้นหา ชื่อเรื่องหัวข้อจะเริ่มต้นด้วย "มีอะไรใหม่หรือเปลี่ยนแปลงใน Dynamics 365 for Talent…”

สารบัญจะดำเนินการต่อเพื่ออัพเดตในแต่ละสัปดาห์ ดังนั้น ตรวจสอบย้อนหลังเพื่อค้นหาคุณลักษณะใหม่และการเปลี่ยนแปลงทั้งหมดใน Talent รุ่นล่าสุด
