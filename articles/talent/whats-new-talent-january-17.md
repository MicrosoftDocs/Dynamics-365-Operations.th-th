# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-january-17-2019"></a>มีอะไรใหม่หรือการเปลี่ยนแปลงอะไรใน Dynamics 365 for Talent Core HR (17 มกราคม 2019)

[!include [banner](includes/banner.md)]

**สร้าง 8.1.2107**

หัวข้อนี้อธิบายถึงคุณลักษณะที่ใหม่หรือที่มีการเปลี่ยนแปลง อย่างใดอย่างหนึ่งใน Core HR

## <a name="minor-changes-and-bug-fixes-included-in-this-release"></a>การเปลี่ยนแปลงเล็กน้อยและการแก้ไขบักรวมอยู่ในรุ่นนี้

### <a name="new-position-assignment-start-datetime-default-has-been-corrected"></a>ค่าเริ่มต้นของวัน/เวลาเริ่มต้นของการมอบหมายตำแหน่งงานใหม่ได้รับการแก้ไขแล้ว
เมื่อโอนย้ายพนักงาน ระบบจะกำหนดค่าเริ่มต้นวัน/เวลาจนถึงวันที่ที่เลือกและเวลาในช่วงเริ่มต้นของวันใหม่ สำหรับโซนเวลาของผู้ใช้

### <a name="all-employees-have-the-same-position-description-in-the-exited-worker-task-management-list"></a>พนักงานทั้งหมดมีคำอธิบายตำแหน่งที่เหมือนกันในรายการการจัดการงานของผู้ปฏิบัติงานที่ออกแล้ว
มีการอัพเดตในพื้นที่ทำงาน **การจัดการงาน** เพื่อแสดงคำอธิบายตำแหน่งที่ถูกต้องสำหรับพนักงานทั้งหมดที่เพิ่งออกจากบริษัท

### <a name="action-requested-by-field-is-now-populated-on-workers-action-page"></a>ขณะนี้มีการเติมข้อมูลฟิลด์ "การดำเนินการที่ร้องขอโดย" บนหน้าการดำเนินการของผู้ปฏิบัติงาน
ด้วยการเปลี่ยนแปลงนี้ ขณะนี้ค่าเริ่มต้นสำหรับฟิลด์ **การดำเนินการที่ร้องขอโดย** คือผู้ใช้ที่ร้องขอการเปลี่ยนแปลง

### <a name="ideas-portal-updated"></a>พอร์ทัลแนวคิดได้รับการอัพเดตแล้ว
ในลิงก์แอพลิเคชันเพื่อส่งความคิดสำหรับ Talent ได้รับการอัพเดตไปยังพอร์ทัลแนวคิดใหม่แล้ว แนวคิดทั้งหมดที่ป้อนในพอร์ทัลเดิมถูกย้าย ดังนั้นแนวคิดที่ดีทั้งหมดของคุณจะยังคงอยู่ในนั้น และพร้อมใช้งานเพื่อที่จะได้ลงคะแนนโดยบุคคลอื่น
