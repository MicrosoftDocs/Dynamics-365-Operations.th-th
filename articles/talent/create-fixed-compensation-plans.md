## <a name="fixed-compensation-plans"></a><span data-ttu-id="7a99a-130">แผนค่าตอบแทนคงที่</span><span class="sxs-lookup"><span data-stu-id="7a99a-130">Fixed compensation plans</span></span>
<span data-ttu-id="7a99a-131">คุณสามารถออกแบบแผนค่าตอบแทนคงซึ่งรวมส่วนประกอบทั้งหมดที่คุณได้ตั้งค่าคอนฟิก</span><span class="sxs-lookup"><span data-stu-id="7a99a-131">You can design the fixed compensation plan to combine all the components that you've configured.</span></span> <span data-ttu-id="7a99a-132">สร้างแผนค่าตอบแทนคงที่ เปิดหน้า **แผนแผนค่าตอบแทนคงที่** นั้น</span><span class="sxs-lookup"><span data-stu-id="7a99a-132">To create a fixed compensation plan, open the **Fixed compensation plans** page.</span></span> <span data-ttu-id="7a99a-133">ที่นี่ คุณสามารถตั้งชื่อ และอธิบายแผนของคุณ เลือกชนิดของแผนที่มี (ขั้นตอน ระดับ หรือกลุ่ม), เลือกความถี่ในการจ่ายค่าตอบแทนที่จะใช้สำหรับอัตราค่าจ้างพนักงาน (ยอดเงินต่อชั่วโมงยอดเงินต่อปี และอื่น ๆ) และตั้งค่าตัวเลือกบางตัวเลือกที่ควบคุมกระบวนการประมวลผลของค่าตอบแทน</span><span class="sxs-lookup"><span data-stu-id="7a99a-133">Here, you can give your plan a name and description, select what type of plan it is (step, grade, or band), select the pay frequency that you'll use for the employee's pay rate (amount per hour, amount per year, and so on), and set some options that control how compensation is processed.</span></span> 

<span data-ttu-id="7a99a-134">การตั้งค่า **อยู่นอกช่วงค่าเผื่อ** อนุญาตให้คุณระบุวิธีที่รัดกุมที่คุณต้องการทำเพื่อให้แน่ใจว่า มียอดเงินค่าตอบแทนระหว่างยอดเงินต่ำสุด และสูงสุด</span><span class="sxs-lookup"><span data-stu-id="7a99a-134">The **Out of range tolerance** setting lets you specify how strict you want to be about making sure that compensation amounts are between the minimum and maximum amounts.</span></span> <span data-ttu-id="7a99a-135">ค่าเผื่อ **แบบเข้มงวด** ต้องการค่าตอบแทนให้อยู่ภายในช่วงที่กำหนดไว้ในระดับที่ให้ไว้</span><span class="sxs-lookup"><span data-stu-id="7a99a-135">A **Hard** tolerance requires that compensation be within the range that is defined for a given level.</span></span> <span data-ttu-id="7a99a-136">**ค่าเผื่อแบบผ่อนปรน** เตือนคุณเมื่อออกนอกช่วงยอดเงินค่าตอบแทน แต่ยอมให้คุณสามารถดำเนินต่อ</span><span class="sxs-lookup"><span data-stu-id="7a99a-136">A **Soft** tolerance alerts you when the compensation amount is outside the range but allows you to continue.</span></span> <span data-ttu-id="7a99a-137">ถ้าคุณตั้งค่าค่าเผื่อไปยัง **ไม่มี**คุณสามารถป้อนยอดเงินค่าตอบแทนสำหรับพนักงานโดยไม่ได้รับข้อความเตือนหรือข้อผิดพลาดได้</span><span class="sxs-lookup"><span data-stu-id="7a99a-137">If you set the tolerance to **None**, you can enter any compensation amount for an employee without receiving warning or error messages.</span></span>

<span data-ttu-id="7a99a-141">หลังจากที่คุณกำหนดพื้นฐานของแผนค่าตอบแทนคง คุณสามารถตั้งค่าโครงสร้างค่าตอบแทนสำหรับแผนนั้น</span><span class="sxs-lookup"><span data-stu-id="7a99a-141">After you define the basics of the fixed compensation plan, you can set up the compensation structure for the plan.</span></span> <span data-ttu-id="7a99a-142">คลิก **ตั้งค่าค่าตอบแทน**</span><span class="sxs-lookup"><span data-stu-id="7a99a-142">Click **Set up compensation**.</span></span> <span data-ttu-id="7a99a-143">แถบเลื่อนกล่องโต้ตอบเปิดขึ้นเพื่อแสดงสามตัวเลือก:</span><span class="sxs-lookup"><span data-stu-id="7a99a-143">A dialog slider opens that gives you three options:</span></span>
