# <a name="whats-new-or-changed-in-dynamics-365-talent-december-17-2019"></a>มีอะไรใหม่หรือเปลี่ยนไปอย่างไรบ้างใน Dynamics 365 Talent (17 ธันวาคม 2019)

บทความนี้อธิบายถึงคุณลักษณะใหม่หรือที่มีการเปลี่ยนแปลงใน Dynamics 365 Talent

## <a name="changes-in-attract"></a>การเปลี่ยนแปลงใน Attract

รุ่นนี้ประกอบด้วยการแก้ไขบักรองสำหรับ Dynamics 365 Talent: Attract

## <a name="changes-in-onboard"></a>การเปลี่ยนแปลงในการเตรียมความพร้อม

รุ่นนี้ประกอบด้วยการแก้ไขบักรองสำหรับ Dynamics 365 Talent: Onboard

## <a name="changes-in-core-hr"></a>การเปลี่ยนแปลงใน Core HR

การเปลี่ยนแปลงที่อธิบายไว้ในส่วนนี้นำไปใช้กับการสร้างหมายเลข 8.1.2674 ตัวเลขในวงเล็บในส่วนหัวบางส่วนอ้างอิงถึงหมายเลขที่สนับสนุนใน Microsoft Dynamics Lifecycle Services (LCS)

### <a name="print-performance-reviews"></a>พิมพ์การตรวจทานประสิทธิภาพ

ขณะนี้มีการพิมพ์การตรวจทานประสิทธิภาพในทุกสภาพแวดล้อม หากต้องการเปิดใช้งานฟังก์ชันนี้ ให้ใช้พื้นที่ทำงาน **การจัดการคุณลักษณะ** สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการจัดการคุณลักษณะ ดูที่ [ภาพรวมของการจัดการคุณลักษณะ](https://docs.microsoft.com/dynamics365/fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview) สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการพิมพ์การตรวจทานประสิทธิภาพการทำงาน ดู [การตรวจทานประสิทธิภาพการทำงาน](https://docs.microsoft.com/dynamics365/talent/performance-management-overview#performance-reviews)

### <a name="retired-position-with-direct-reports-floats-in-org-diagram-as-an-open-position-385436"></a>ตำแหน่งที่เกษียณอายุที่มีรายงานโดยตรง "ลอย" ในแผนผังองค์กรเป็นตำแหน่งเปิด (385436)

การนำออกใช้ของสัปดาห์นี้จะแก้ไขวิธีการที่ตำแหน่งที่เกษียณอายุที่มีรายงานโดยตรงแสดงอยู่ในลำดับชั้นการรายงาน
 
### <a name="system-unable-to-generate-calendar-days-381195"></a>ระบบไม่สามารถสร้างวันในปฏิทิน (381195) 

การเปลี่ยนแปลงของสัปดาห์นี้แก้ไขปัญหาที่ไม่สามารถสร้างวันที่ในปฏิทินได้

### <a name="current-balance-report-in-leave-pbi-doubles-employee-balance-385581"></a>รายงานยอดดุลปัจจุบันที่เหลือ PBI ทำให้ยอดดุลของพนักงานเป็นสองเท่า (385581)

การเปลี่ยนแปลงนี้แก้ไขรายงานยอดดุลปัจจุบันในสถานการณ์นี้
