# <a name="share-content-with-other-contributors-in-dynamics-365-talent---onboard"></a>แบ่งปันเนื้อหากับผู้สนับสนุนอื่นๆ ใน Dynamics 365 Talent - Onboard

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboard ช่วยให้คุณสามารถแบ่งปันคู่มือและเท็มเพลตการเตรียมความพร้อมกับผู้สนับสนุนรายอื่นๆ เพื่อให้พวกเขาสามารถทบทวนเอกสารและทำการเปลี่ยนแปลงได้

## <a name="share-an-onboarding-guide-with-other-contributors"></a>แบ่งปันคู่มือการเตรียมความพร้อมกับผู้สนับสนุนอื่นๆ

1. บนเมนูด้านซ้าย ให้เลือก **คู่มือ** แล้วจากนั้น เลือกคู่มือที่คุณต้องการแบ่งปัน
2. เลือกปุ่ม **เพิ่มเติม** (จุดแนวตั้งสามจุด) ในมุมขวาบนของหน้า และจากนั้น เลือก **แบ่งปัน**

    ![[การแบ่งปันคู่มือการเตรียมความพร้อมกับผู้สนับสนุนอื่นๆ](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. ภายใต้ **แบ่งปันกับ** ถ้าผู้รับควรจะสามารถแก้ไขคู่มือได้ ให้กำหนดตัวเลือก **ผู้ใช้เหล่านี้สามารถทำการแก้ไขได้หรือไม่** เป็น **เปิด** ถ้าผู้รับควรได้รับการแจ้งเตือนทางอีเมลว่าคุณได้แบ่งปันคู่มือกับพวกเขา ตั้งค่าตัวเลือก **ส่งการแจ้งเตือนทางอีเมลไปยังผู้รับ** เป็น **เปิด**

    ![[การตั้งค่าการตั้งค่าการเข้าถึงและอีเมล์สำหรับคำแนะนำปฐมนิเทศ] (./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. ป้อนชื่อหรือที่อยู่อีเมล และเลือกผู้รับที่ต้องการในรายการแบบหล่นลง
5. บนแท็บ **จัดการการเข้าถึง** คุณสามารถตรวจทานหรือเปลี่ยนสิ่งที่ผู้รับจะทำกับคู่มือได้ เมื่อต้องการเปลี่ยนแปลงการเข้าถึงของผู้รับ ในรายการแบบหล่นลงที่อยู่ถัดจากชื่อของผู้รับ เลือก **สามารถดูได้** **สามารถแก้ไขได้** หรือ **ลบ** ตามความเหมาะสม
6. เมื่อคุณเพิ่มหรือปรับเปลี่ยนผู้รับเสร็จแล้ว เลือก **ปรับปรุง**

## <a name="share-an-onboarding-template-with-other-contributors"></a>แบ่งปันเท็มเพลตการเตรียมความพร้อมกับผู้สนับสนุนอื่นๆ

1. บนเมนูด้านซ้าย ให้เลือก **เท็มเพลต**
2. ภายใต้ **เท็มเพลต** ให้เลือกปุ่ม **เพิ่มเติม** (จุดไข่ปลา \[**...**\]) ที่มุมด้านขวาบนของเท็มเพลตที่คุณต้องการแบ่งปัน แล้วจากนั้น เลือก **แบ่งปัน**

    [![การแบ่งปันเท็มเพลตกับผู้สนับสนุนรายอื่นๆ](./media/onboard-share-template.png)](./media/onboard-share-template.png)

3. ภายใต้ **ผู้รับ** ถ้าผู้รับควรจะสามารถแก้ไขเท็มเพลตได้ ให้กำหนดตัวเลือก **ผู้ใช้เหล่านี้สามารถทำการแก้ไขได้หรือไม่** เป็น **เปิด** ถ้าผู้รับควรได้รับการแจ้งเตือนทางอีเมลว่าคุณได้แบ่งปันเท็มเพลตกับพวกเขา ตั้งค่าตัวเลือก **ส่งการแจ้งเตือนทางอีเมลไปยังผู้รับ** เป็น **เปิด**

    [!การตั้งค่าการกำหนดลักษณะการเข้าถึงและอีเมลสำหรับเท็มเพลต](./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)

4. ป้อนชื่อหรือที่อยู่อีเมล และเลือกผู้รับที่ต้องการในรายการแบบหล่นลง
5. บนแท็บ **จัดการการเข้าถึง** คุณสามารถตรวจทานหรือเปลี่ยนสิ่งที่ผู้รับจะทำกับเท็มเพลตได้ เมื่อต้องการเปลี่ยนแปลงการเข้าถึงของผู้รับ ในรายการแบบหล่นลงที่อยู่ถัดจากชื่อของผู้รับ เลือก **สามารถดูได้** **สามารถแก้ไขได้** หรือ **ลบ** ตามความเหมาะสม
6. เมื่อคุณเพิ่มหรือปรับเปลี่ยนผู้รับเสร็จแล้ว เลือก **ปรับปรุง**
