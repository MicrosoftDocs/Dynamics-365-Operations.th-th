# <a name="manage-preview-features"></a>จัดการคุณลักษณะแสดงตัวอย่าง

[!include[banner](../includes/banner.md)]

เนื่องจากเป็นส่วนหนึ่งของการเปิดตัวความสามารถของการจัดการทุนมนุษย์ (HCM) สำหรับ Microsoft Dynamics 365 Talent เราต้องการให้ลูกค้าพบกับคุณลักษณะใหม่โดยเร็วที่สุด ผู้ดูและระบบสามารถดูและใช้งานคุณลักษณะการแสดงตัวอย่างในสภาพแวดล้อมดังกล่าวได้ คุณลักษณะเหล่านี้ใกล้จะพร้อมสำหรับความพร้อมใช้งานทั่วไป และได้ผ่านการทดสอบอย่างครอบคลุมแล้ว เรากำลังมองหาความคิดเห็นและการตรวจสอบความถูกต้องของลูกค้าในขั้นตอนสุดท้าย ก่อนที่เราจะนำคุณลักษณะเหล่านี้ออกไปใช้งานสำหรับความพร้อมใช้งานทั่วไป

หัวข้อนี้อธิบายวิธีการที่ทำให้คุณสามารถเปิดใช้งานคุณลักษณะแสดงตัวอย่าง และแสดงรายการคุณลักษณะที่พร้อมใช้งานสำหรับการแสดงตัวอย่างในขณะนี้ รายการนี้จะถูกอัพเดต เมื่อคุณลักษณะถูกนำไปใช้ในความพร้อมใช้งานทั่วไป และเมื่อมีการนำคุณลักษณะใหม่ออกใช้เพื่อแสดงตัวอย่าง ไม่มีการแจ้งเตือนให้ทราบ เมื่อมีการนำคุณลักษณะใหม่มาแสดงตัวอย่าง ผู้ใช้จะเริ่มต้นเห็นคุณลักษณะ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับคุณลักษณะใหม่ๆ ใน Talent ดู [มีอะไรใหม่หรือการเปลี่ยนแปลงอะไรใน Dynamics 365 Talent](./whats-new.md) และ [บันทึกย่อประจำรุ่นของ Dynamics 365 และ Power Platform](https://docs.microsoft.com/business-applications-release-notes)

## <a name="enable-or-disable-preview-features"></a>เปิดใช้งาน หรือปิดใช้งานคุณลักษณะการแสดงตัวอย่าง

เมื่อต้องการเข้าถึงคุณลักษณะการแสดงตัวอย่าง คุณต้องเปิดใช้งานในสภาพแวดล้อมของคุณก่อน การเปิดหรือปิดใช้งานคุณลักษณะการแสดงตัวอย่างเป็นแบบเฉพาะกับสภาพแวดล้อม

> [!IMPORTANT]
> เมื่อคุณเปิดการตั้งค่า **คุณลักษณะการแสดงตัวอย่าง** คุณเปิดใช้งานคุณลักษณะการแสดงตัวอย่างสำหรับผู้ใช้ทั้งหมดในองค์กรของคุณที่อยู่ในสภาพแวดล้อมนั้น เมื่อคุณปิดการตั้งค่า คุณจะปิดใช้งานคุณลักษณะการแสดงตัวอย่าง และทำให้ไม่สามารถเข้าถึงผู้ใช้ของคุณได้ คุณลักษณะการแสดงตัวอย่างมีการสนับสนุนที่จำกัดใน Talent อาจใช้ความเป็นส่วนตัวและมาตรการด้านความปลอดภัยน้อยลง และจะไม่ถูกรวมอยู่ในข้อตกลงระดับการบริการ Talent (SLA) คุณไม่ควรใช้คุณลักษณะการแสดงตัวอย่างในการประมวลผลข้อมูลส่วนบุคคล (นั่นคือ ข้อมูลใดๆ ที่สามารถระบุตัวคุณได้) หรือในการประมวลผลข้อมูลอื่นๆ ที่ขึ้นอยู่กับความต้องการปฏิบัติตามกฎหมาย หรือตามข้อบังคับ

### <a name="attract"></a>ดึงดูด

1. ลงชื่อเข้าใช้ Microsoft Dynamics 365 Talent: Attract
2. ในเมนู **ตั้งค่า** (สัญลักษณ์เกียร์) ในมุมบนด้านขวา เลือก **ศูนย์ผู้ดูแลระบบ**
3. บนแท็บ **การจัดการคุณลักษณะ** เลือกตัวเลือกถัดจาก **คุณลักษณะการแสดงตัวอย่าง** เพื่อให้กลายเป็นสีฟ้า และกล่าวว่า **เปิด**

    ![เปิดใช้งานคุณลักษณะการแสดงตัวอย่างใน Attract](./media/attract-enable-preview-features.png)

4. เลือกหรือยกเลิกการเลือกของคุณลักษณะการแสดงตัวอย่างแต่ละรายการ ถ้าคุณไม่ได้ทำอะไรเลย คุณสามารถเปิดใช้งานคุณลักษณะการแสดงตัวอย่างทั้งหมดที่มีอยู่ได้
5. รีเฟรชเบราเซอร์ของคุณเพื่อเริ่มต้นดูคุณลักษณะใหม่ ผู้ใช้ใดๆ ที่เข้าสู่ระบบอยู่แล้ว จะเห็นคุณลักษณะในครั้งถัดไปที่พวกเขาเข้าสู่ระบบ หรือพวกเขาสามารถรีเฟรชเบราเซอร์ของพวกเขาเพื่อดูคุณลักษณะโดยทันทีได้

> [!NOTE]
> คุณลักษณะการแสดงตัวอย่างบางรายการอาจต้องการการตั้งค่าคอนฟิกเพิ่มเติม ทำตามลิงค์ที่อยู่ถัดจากคุณลักษณะการแสดงตัวอย่างเพื่อทำให้การตั้งค่าเสร็จสมบูรณ์

### <a name="core-hr"></a>HR หลัก

1. ลงชื่อเข้าใช้ Talent
2. เลือก **การจัดการระบบ** แล้วเลือกแท็บ **ลิงค์**
3. บนหน้า **การจัดการระบบ** ภายใต้ **การตั้งค่า** ให้เลือก **พารามิเตอร์ระบบ**
4. บนหน้า **พารามิเตอร์ระบบ** ให้เลือกแท็บ **คุณลักษณะการแสดงตัวอย่าง**
5. ตั้งค่าตัวเลือก **เปิดใช้งานโหมดการแสดงตัวอย่างสำหรับผู้ใช้ทั้งหมด** เป็น **ใช่** เพื่อทำให้คุณลักษณะการแสดงตัวอย่างพร้อมใช้งาน

    ![เปิดใช้งานคุณลักษณะการแสดงตัวอย่างใน Core HR](./media/corehr-enable-preview-features.png)

> [!NOTE]
> เพื่อปิดใช้งานคุณลักษณะการแสดงตัวอย่าง ใช้ขั้นตอนเดียวกัน แต่ตั้งค่าตัวเลือก **เปิดใช้งานโหมดการแสดงตัวอย่างสำหรับผู้ใช้ทั้งหมด** เป็น **ไม่** เมื่อคุณปิดใช้งานคุณลักษณะการแสดงตัวอย่าง พวกเขาจะไม่สามารถเข้าถึงผู้ใช้ของคุณได้ และข้อผิดพลาดอาจเกิดขึ้นในกระบวนการที่เกี่ยวข้องกับคุณลักษณะ

### <a name="onboard"></a>เตรียมความพร้อม

ไม่มีคุณลักษณะตัวอย่างที่พร้อมใช้งานสำหรับ Microsoft Dynamics 365 Talent: Onboard

## <a name="features-that-are-currently-in-preview"></a>คุณลักษณะที่อยู่ในการแสดงตัวอย่างในขณะนี้

### <a name="attract"></a>ดึงดูด

- [คำแนะนำสำหรับผู้สมัคร](./intelligent-recommendations.md#candidate-recommendations) – หากมีผู้สมัครมากกว่าสิบรายที่มีประวัติย่อหรือโพรไฟล์ทั้งหมด ผู้สมัครที่ใกล้เคียงข้อกำหนดของงานมากที่สุดปรากฏขึ้นในส่วน **ผู้สมัครที่จะพิจารณา** บนหน้าของงานนั้น
- [คำแนะนำเกี่ยวกับงาน](./intelligent-recommendations.md#job-recommendations) –ถ้ามีการลงรายการบัญชีงานมากกว่าสิบงานบนไซต์อาชีพของคุณ Attract จะให้คำแนะนำเกี่ยวกับงานแก่ผู้ที่มีแนวโน้ม
- [การรวม Broadbean](./posting-jobs-external.md#post-jobs-to-broadbean) – คุณสามารถลงรายการบัญชีงานจาก Attract ไปที่ Broadbean ซึ่งคือไซต์การลงรายการบัญชีงานภายนอก หลังจากที่คุณเปิดใช้งานคุณลักษณะการแสดงตัวอย่างนี้แล้วคุ ณต้องทำการตั้งค่าให้เสร็จสมบูรณ์โดยป้อนชื่อผู้ใช้ Broadbean ของคุณ รหัสไคลเอนต์ และโทเคนการเข้ารหัส
- [การวิเคราะห์](./analytic-reports.md) – ในฮับการวิเคราะห์ ทีมการจ้างงานสามารถดูการวัดที่สำคัญสำหรับงานหนึ่งๆ รวมในการวัดรวมทั่วทั้งงานทั้งหมดได้
- [EEO](./activities-attract.md) – ชนิดของกิจกรรมใหม่อนุญาตให้คุณสามารถใช้ฟอร์มที่กำหนดไว้ล่วงหน้าเพื่อสะสมโอกาสการจ้างงานที่เท่าเทียมกัน (EEO) และข้อมูลสํานักงานโปรแกรมการตรวจสอบการปฏิบัติตามสัญญาของรัฐบาลกลาง (OFCCP) จากผู้สมัคร ไม่สามารถแก้ไขฟอร์มที่กำหนดไว้ล่วงหน้าได้
- [คำแนะนำสำหรับผู้สมัครงาน](./intelligent-recommendations.md#prospect-recommendations) – Attract ตรวจทานผู้สมัครในอดีตและผู้สมัครปัจจุบันเพื่อแสดงรายการของผู้ที่มีแนวโน้มที่เหมาะสมสำหรับงานของคุณ
- [การค้นหาความเกี่ยวข้อง](./attract-talent-pools.md#search-and-view-candidate-profiles) – คุณสามารถค้นหาฐานข้อมูลของผู้สมัครทั้งหมดสำหรับทักษะเฉพาะ ชื่อ หรือภูมิหลังการศึกษาได้ Attract จะค้นหาโพรไฟล์ทั้งหมด และเน้นการจับคู่ทั้งหมดที่พบ นอกจากนี้ Attract จะค้นหาเอกสารทั้งหมดที่พร้อมใช้งานสำหรับผู้สมัครและจัดลำดับผลการค้นหาอย่างชาญฉลาดอีกด้วย
- [ผู้ชมกิจกรรม](./whats-new-talent-march-20.md#setting-the-audience-on-activities) – คุณสามารถตั้งค่าผู้เข้าชมสำหรับกิจกรรมได้ (เช่น การสัมภาษณ์ กำหนดการ หรือความคิดเห็น) ให้กับ **ผู้สมัครทั้งหมด** **ผู้สมัครภายใน** หรือ **ผู้สมัครภายนอก** คุณสามารถจัดส่งกิจกรรมของลูกค้าได้ เช่น วิดีโอ YouTube เนื้อหาเว็บ และ Microsoft Forms ให้กับผู้สมัครทั้งหมด ผู้สมัครภายในเท่านั้น ผู้สมัครภายนอกเท่านั้น หรือทีมว่าจ้าง
- [นำไปใช้กับ LinkedIn](./career-site.md#enable-applying-for-jobs-with-linkedin-profiles) – คุณสามารถตั้งค่าตัวเลือกบนเว็บไซต์อาชีพ Attract ของคุณเพื่อให้ผู้สมัครงานสามารถนำไปใช้ได้โดยใช้ LinkedIn คุณลักษณะนี้จะลดขั้นตอนซ้ำซ้อนของกระบวนการของแอพลิเคชันสำหรับผู้สมัครของคุณ โดยอนุญาตให้พวกเขาใช้โพรไฟล์ LinkedIn ของพวกเขาเพื่อกรอกข้อมูลในแอพลิเคชันของพวกเขาในไซต์อาชีพของคุณ
- [การติดตามแหล่งที่มา](./source-tracking.md) – Attract ติดตามแหล่งที่มาของใบสมัครของผู้สมัครเพื่อให้ข้อมูลที่เป็นประโยชน์ ซึ่งช่วยให้คุณสามารถกำหนดเป้าหมายความพยายามในการสรรหาบุคลากรของคุณได้ นอกจากนี้ คุณยังสามารถเลือกแหล่งที่มาของใบสมัครได้ ขณะที่คุณเพิ่มผู้สมัครไปยังงานหรือกลุ่ม talent
- [ผู้ได้รับเหรียญเงิน](./whats-new-talent-march-20.md#designate-silver-medalists-to-assign-high-value-applicants-for-future-positions) – ถ้าผู้สมัครมีความเหมาะสมอย่างมากสำหรับองค์กรของคุณ แต่คุณไม่ได้ขยายข้อเสนอให้กับพวกเขาสำหรับตำแหน่งปัจจุบันของคุณ คุณสามารถกำหนดพวกเขาให้เป็นผู้ได้รับเหรียญเงินได้ คุณลักษณะนี้ช่วยลดเวลาของคุณในการจ้างงานครั้งถัดไปที่คุณมีตำแหน่งว่างที่คล้ายกัน

### <a name="core-hr"></a>HR หลัก

- [ตรวจสอบความถูกต้องของข้อมูลลำดับชั้นของตำแหน่ง](./whats-new-talent-may-13-2019.md#new-page-to-validate-position-hierarchy-data) – คุณสามารถตรวจสอบความถูกต้องของลำดับชั้นการจัดการสำหรับการอ้างอิงแบบหมุนเวียนใดๆ ที่มีการนำเข้าโดยไม่ได้ตั้งใจ
- [ระบุรหัสเหตุผลในชนิดของการลางาน](./whats-new-talent-may-13-2019.md#specify-reason-codes-on-leave-types) – คุณสามารถระบุรหัสเหตุผลสำหรับชนิดการลางาน
- [ต้องใช้รหัสเหตุผลในการร้องขอเวลาหยุดพัก](./whats-new-talent-may-13-2019.md#require-reason-codes-for-specific-leave-types-on-time-off-requests) – นอกเหนือจากการระบุรหัสเหตุผลสำหรับชนิดการลางาน คุณอาจต้องใช้รหัสเหตุผลสำหรับการร้องขอเวลาหยุดพัก
- [แสดงรายการธุรกรรมการลางานและการขาดงานสำหรับ HR](./whats-new-talent-may-13-2019.md#provide-a-leave-and-absence-transaction-list-for-hr) – คุณสามารถดูรายการของธุรกรรมการลางานและการขาดงานที่จะช่วยในการให้ข้อมูลเชิงลึกของยอดดุลเวลาหยุดพัก

### <a name="onboard"></a>เตรียมความพร้อม

ไม่มีคุณลักษณะการแสดงตัวอย่างที่พร้อมใช้งานในขณะนี้สำหรับ Onboard

## <a name="feedback"></a>ผลป้อนกลับ

เราต้องการได้ยินจากคุณเกี่ยวกับประสบการณ์ของคุณกับคุณลักษณะการแสดงตัวอย่างใดๆ เหล่านี้ เราขอสนับสนุนให้คุณลงรายการบัญชีคำติชมของคุณอย่างสม่ำเสมอบนไซต์ต่อไปนี้ เมื่อคุณใช้รายการเหล่านี้หรือคุณลักษณะอื่นใดๆ :

- [ชุมชน](https://community.dynamics.com/enterprise/f/759?pi53869=0&category=Talent) – ไซต์นี้เป็นทรัพยากรที่สำคัญที่ผู้ใช้สามารถอธิบายกรณีที่ใช้ ถามคำถาม และขอความช่วยเหลือของชุมชน
- แจ้งให้เราทราบเกี่ยวกับคุณลักษณะที่คุณต้องการดูในผลิตภัณฑ์ และแจ้งให้เราทราบเกี่ยวกับการเปลี่ยนแปลงใดๆ ที่คุณคิดว่าเราควรจะทำต่อคุณลักษณะที่มีอยู่ แนะนำแนวคิดเกี่ยวกับผลิตภัณฑ์ในไซต์ต่อไปนี้:

    - [แนวคิด Attract](https://powerusers.microsoft.com/t5/Ideas-for-Attract/idb-p/Attract)
    - [แนวคิด Core HR](https://powerusers.microsoft.com/t5/Ideas-for-Human-Resources/idb-p/HumanResources)
    - [แนวคิด Onboard](https://powerusers.microsoft.com/t5/Ideas-for-Onboard/idb-p/Onboard)

ต้องแน่ใจว่าคุณไม่ได้รวมข้อมูลส่วนบุคคล (ข้อมูลใดๆ ที่สามารถระบุตัวคุณได้) ในความคิดเห็นของคุณ หรือการส่งการตรวจทานผลิตภัณฑ์ ข้อมูลที่ถูกรวบรวมอาจถูกวิเคราะห์เพิ่มเติม และจะไม่ถูกใช้เพื่อตอบสนองคำขอภายใต้กฎหมายความเป็นส่วนตัวที่เกี่ยวข้องได้ ข้อมูลส่วนบุคคลที่ถูกเก็บแยกต่างหากภายใต้โปรแกรมเหล่านี้มีผลต่อ [Microsoft Privacy Statement](https://privacy.microsoft.com/privacystatement)

> [!TIP]
> คั่นหัวข้อนี้ และลตรวจสอบย้อนหลังบ่อยๆ เพื่อคอยติดตามสถานการณ์ปัจจุบันเกี่ยวกับคุณลักษณะการทำงานใหม่ เมื่อเรานำออกใช้
