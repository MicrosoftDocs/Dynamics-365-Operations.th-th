# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-october-19-2018"></a>มีอะไรใหม่หรือการเปลี่ยนแปลงอะไรใน Dynamics 365 for Talent Core HR (19 ตุลาคม 2018)

[!include[banner](includes/banner.md)]

**สร้าง 8.1.1067**

หัวข้อนี้อธิบายถึงคุณลักษณะที่ใหม่หรือที่มีการเปลี่ยนแปลง อย่างใดอย่างหนึ่งใน Core HR

## <a name="allow-managers-to-update-time-off-requests"></a>อนุญาตให้ผู้จัดการสามารถอัพเดตคำขอการลาหยุดได้

คำขอการลาหยุดของพนักงานอาจจำเป็นต้องถูกปรับปรุง หลังจากที่อนุมัติโดยใช้ลำดับงาน ในหลายกรณี ผู้จัดการทำการปรับปรุงเหล่านี้ในนามของพนักงาน คุณลักษณะใหม่นี้มีความสามารถสำหรับผู้จัดการในการอัพเดตคำขอการลาหยุดหรือยกเลิกการลาหยุดในนามของพนักงานของพวกเขา ความสามารถนี้ถูกควบคุมโดยพารามิเตอร์ **ทำงานในนาม** ที่สามารถตั้งค่าในหน้า **พารามิเตอร์ทรัพยากรบุคคล** ได้ 
 
## <a name="allow-hr-to-update-time-off-requests"></a>อนุญาตให้ HR สามารถอัพเดตคำขอการลาหยุดได้

คล้ายกับลักษณะการทำงานข้างต้น คำขอการลาหยุดของพนักงานอาจจำเป็นต้องถูกอัพเดตโดย HR หลังจากที่อนุมัติแล้วก่อนหน้านี้โดยใช้ลำดับงาน คุณลักษณะนี้มีความสามารถสำหรับผู้ใช้ HR เพื่ออัพเดตคำขอการลาหยุด มีการเปิดใช้งานความสามารถในพื้นที่ทำงาน **บุคลากร** และบนหน้า **ผู้ปฏิบัติงาน** โดยใช้ตัวเลือกใหม่ที่เรียกว่า **ดูเวลาหยุดพัก** บนหน้าเหล่านั้น ผู้ใช้ HR สามารถดูคำขอ และอัพเดตธุรกรรมเวลาหยุดพัก ที่คล้ายคลึงกับวิธีที่ผู้จัดการสามารถทำการดำเนินการเหล่านี้

หน้าที่ด้านความปลอดภัยที่ควบคุมการเข้าถึงฟังก์ชันนี้คือ:
- หน้าที่: รักษากระบวนการลางานและการขาดงานสำหรับผู้ปฏิบัติงานเฉพาะ
- สิทธิ์: รักษาคำขอลางานและขาดงานสำหรับผู้ปฏิบัติงานทั้งหมด

## <a name="other-changes"></a>การเปลี่ยนแปลงอื่นๆ
มีการทำการปรับปรุงต่อไปนี้ในรุ่นนี้:
- การเปลี่ยนแปลงต่อการดำเนินการจ้างงานของผู้ปฏิบัติงานเพื่อให้พวกเขาไม่ "ติด" ในสถานะ **ลำดับงานเสร็จสมบูรณ์** อีกต่อไป
- ขณะนี้ คุณสามารถสร้างเรกคอร์ดการจ้างงาน โดยไม่มีวันที่เริ่มต้นการจ้างงานได้
- วันที่ลงทะเบียน Dynamics 365 Talent สำหรับหลักสูตรที่แสดงไว้ในการบริการตนเองของพนักงาน ในขณะนี้ใช้ออฟเซ็ตโซนเวลาจนถึงวันที่
- ไฟล์ Excel สามารถนำเข้าได้หลายครั้ง โดยไม่มีข้อผิดพลาดใดๆ โดยใช้ **เอนทิตีพนักงาน**

## <a name="known-issue"></a>ปัญหาที่ทราบ

- **ปัญหา**: เมื่อเพิ่มสิ่งที่แนบมาใหม่กับผู้ปฏิบัติงาน ปุ่ม **สร้าง** และ **แก้ไข** จะกลายเป็นสีเทา 
- **วิธีการแก้ไขปัญหา:** ก่อนที่จะเปิดหน้าเอกสารแนบ ให้แน่ใจว่าก FactBoxes บนหน้า **ผู้ปฏิบัติงาน** ถูกปิดแล้ว ถ้า FactBoxes ถูกปิด เมื่อหน้า **ผู้ปฏิบัติงาน** ถูกโหลด ปุ่มเอกสารแนบจะถูกเปิดใช้งาน (ปัญหานี้จะได้รับการแก้ไขในการปรับปรุงแพลตฟอร์มถัดไป)
