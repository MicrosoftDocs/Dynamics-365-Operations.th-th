# <a name="get-started-with-onboard"></a>เริ่มต้นใช้งาน Onboard

[!include [banner](includes/banner.md)]

เมื่อใช้ Microsoft Dynamics 365 for Talent: Onboard คุณสามารถช่วยพนักงานใหม่ของคุณให้ดำเนินการได้อย่างมีประสิทธิภาพมากขึ้นและได้รับความรู้สึกการเป็นส่วนหนึ่งในบทบาทใหม่ของตน กระบวนการเตรียมความพร้อมที่มีประสิทธิภาพจะช่วยให้พนักงานใหม่เข้าใจบทบาทใหม่ของตน นอกจากนี้ยังให้การเริ่มต้นในการสร้างความสัมพันธ์อันมีค่าซึ่งจะช่วยให้มั่นใจได้ถึงความสำเร็จ

Onboard ช่วยให้คุณสามารถสร้างคู่มือการเตรียมความพร้อมที่จะต้อนรับพนักงานใหม่ของคุณได้อย่างรวดเร็วและทำให้เกิดความตื่นเต้นกับการเข้าร่วมองค์กรของคุณ

[![คู่มือการเตรียมความพร้อม](./media/onboard-onboarding-guide.png)](./media/onboard-onboarding-guide.png)

เมื่อต้องการเรียนรู้วิธีการรับและใช้แอป Onboard ให้ดูที่หัวข้อต่อไปนี้

- [สร้างเท็มเพลตการเตรียมความพร้อม](./onboard-create-template.md)
- [แก้ไขคู่มือและเท็มเพลตการเตรียมความพร้อม](./onboard-edit-guides-templates.md)
- [แบ่งปันเนื้อหากับผู้สนับสนุนอื่นๆ](./onboard-share-template.md)
- [ดูสถานะของงานและพนักงานการเตรียมความพร้อม](./onboard-view-status.md)
