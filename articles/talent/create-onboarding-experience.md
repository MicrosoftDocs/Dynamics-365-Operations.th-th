# <a name="get-started-with-onboard"></a>เริ่มต้นใช้งาน Onboard

[!include [banner](includes/banner.md)]

ด้วย Microsoft Dynamics 365 Talent: Onboard: คุณสามารถช่วยพนักงานใหม่ของคุณให้ทำงานได้อย่างมีประสิทธิภาพมากขึ้น และให้พวกเขาได้รับรู้ได้ถึงการทำงานในบทบาทใหม่ ๆ กระบวนการเตรียมความพร้อมที่มีประสิทธิภาพจะช่วยให้พนักงานใหม่เข้าใจบทบาทใหม่ของตน นอกจากนี้ยังให้การเริ่มต้นในการสร้างความสัมพันธ์อันมีค่าซึ่งจะช่วยให้มั่นใจได้ถึงความสำเร็จ

Onboard ช่วยให้คุณสามารถสร้างคู่มือการเตรียมความพร้อมที่จะต้อนรับพนักงานใหม่ของคุณได้อย่างรวดเร็วและทำให้เกิดความตื่นเต้นกับการเข้าร่วมองค์กรของคุณ

[![คู่มือการเตรียมความพร้อม](./media/onboard-onboarding-guide.png)](./media/onboard-onboarding-guide.png)
