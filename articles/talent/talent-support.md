# <a name="get-support-for-talent"></a>รับการสนับสนุนสำหรับ Talent

[!include [banner](includes/banner.md)]

หัวข้อนี้อธิบายวิธีการรับความช่วยเหลือเกี่ยวกับ Talent: Core HR Attract และ Onboard

<a name="contact-support"></a>ติดต่อฝ่ายสนับสนุน
---------------

ใช้ข้อมูลในหัวข้อนี้เพื่อติดต่อ Microsoft ถ้าคุณมีคำถามทางเทคนิคเกี่ยวกับ Talent

### <a name="talent-support"></a>การสนับสนุน Talent 

ในการรับการสนับสนุนสำหรับ Talent ใช้ทรัพยากรในตารางต่อไปนี้ ทรัพยากรเหล่านี้จะแสดงรายการตามลำดับที่เราแนะนำให้คุณปฏิบัติตามเมื่อคุณกำลังแก้ไขปัญหา

| **ภารกิจ**                                                | **ข้อมูลเพิ่มเติม**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| สอบถามชุมชน                                      | ไปที่หน้า [สอบถามชุมชน](https://community.dynamics.com/365/talent) เพื่อค้นหาความช่วยเหลือกับคำถามของคุณ และใช้ตัวเลือก "วิธีใช้" เพื่อค้นหาหัวข้อภายใน Core HR                                                                                                                                                                                                                                                                                                                                                                                  |
| ใช้เครื่องมือการสนับสนุนที่ทำงานบน Cloud                     | ใน [Lifecycle Services (LCS)](https://lcs.dynamics.com/) ไทล์ **สนับสนุน** คือเครื่องมือที่ช่วยคุณในการจัดการเหตุการณ์ด้านการสนับสนุน เมื่อต้องการส่งปัญหาไปยัง Microsoft โดยตรง ไปที่ไทล์ **สนับสนุน** ในโครงการ LCS ของคุณ หรือใช้ **สนับสนุนส่วนของตั๋ว** ใน LCS แล้วคลิก **การสนับสนุนที่ทำงานบน Cloud** จากที่นั่น คุณสามารถส่งปัญหาได้สองวิธี 1: เมื่อส่งปัญหาผ่านทาง LCS ในแท็บ **ปัญหาที่ใช้งานอยู่** เลือกปัญหาของคุณ และจากนั้นคลิก **ส่งไปยัง Microsoft** 2: ในแท็บ **ส่งไปยัง Microsoft** คลิก **ส่งเหตุการณ์ใหม่** เพื่อส่งปัญหาใหม่  |
| ในการสนับสนุนแอพลิเคชัน                                 | เลือก **?** ไอคอนในแถบเมนู จากนั้นเลือก **การสนับสนุน (Core HR) รายงานปัญหา (Attract, Onboard)** ปัญหาจะถูกรายงานบนแท็บ **ปัญหาที่เปิดอยู่** ใน LCS โปรดสังเกตว่าถ้าคุณซื้อ Onboard แบบสแตนด์อโลน รายงานปัญหาจะถูกส่งไปยัง Microsoft  |
| รับความช่วยเหลือเกี่ยวกับคำถามเกี่ยวกับการให้สิทธิ์ Talent         | ติดต่อคู่ค้า หรือพนักงานขายของ Microsoft ของคุณ     |
| ร้องขอลักษณะการทำงานและฟังก์ชันใหม่ภายใน Talent | บนเพจ Talent หลักในแอพลิเคชัน เลือก **?** ไอคอน และเลือก **ไอเดีย** หรือใช้การเชื่อมโยง [ไอเดีย](https://experience.dynamics.com/ideas/) ไปยังการดู ค้นหา ลงคะแนน หรือเพิ่มไอเดียใหม่          |
