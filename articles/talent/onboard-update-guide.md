3. <span data-ttu-id="33f50-111">ทำการเปลี่ยนแปลงที่ต้องการทั้งหมด แล้วเลือกปุ่ม **บันทึก** (สัญลักษณ์ดิสก์)</span><span class="sxs-lookup"><span data-stu-id="33f50-111">Make all the desired changes, and then select the **Save** button (the disk symbol).</span></span>

<span data-ttu-id="33f50-113">Onboard จะส่งอีเมลที่ระบุว่าการเปลี่ยนแปลงนั้นมีอะไรบ้างไปยังพนักงานใหม่โดยอัตโนมัติ</span><span class="sxs-lookup"><span data-stu-id="33f50-113">Onboard will automatically send the new hire an email that indicates what the changes are.</span></span> <span data-ttu-id="33f50-114">สำหรับการระบุรหัสประจำตัวอย่างง่าย ป้าย **ใหม่** สีแดงจะปรากฏขึ้นถัดจากการเปลี่ยนแปลงแต่ละรายการ</span><span class="sxs-lookup"><span data-stu-id="33f50-114">For easy identification, a red **New** tag will appear next to each change.</span></span>
