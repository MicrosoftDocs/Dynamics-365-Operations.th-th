# <a name="set-up-ledger-posting-groups-for-sales-tax"></a><span data-ttu-id="0ea54-103">การตั้งค่ากลุ่มการลงรายการบัญชีแยกประเภทสำหรับภาษีขาย</span><span class="sxs-lookup"><span data-stu-id="0ea54-103">Set up ledger posting groups for sales tax</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="0ea54-104">ภาษีขายมีการคำนวณและลงรายการบัญชีไปยังบัญชีหลักที่ระบุไว้ในกลุ่มการลงรายการบัญชีแยกประเภท </span><span class="sxs-lookup"><span data-stu-id="0ea54-104">Sales tax is calculated and posted to main accounts that are specified in the Ledger posting groups.</span></span> <span data-ttu-id="0ea54-105">กลุ่มการลงรายการบัญชีแยกประเภทจะแนบไปกับรหัสภาษีขายแต่ละรหัส </span><span class="sxs-lookup"><span data-stu-id="0ea54-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="0ea54-106">คุณสามารถตั้งค่ากลุ่มการลงบัญชีแยกประเภทแต่ละกลุ่มสำหรับแต่ละรหัสภาษีขาย ใช้กลุ่มการลงบัญชีแยกประเภทหนึ่งกลุ่มสำหรับรหัสภาษีขายทั้งหมดหรือกำหนดกลุ่มการลงรายการบัญชีแยกประเภทหลายกลุ่มให้กับรหัสภาษีขาย </span><span class="sxs-lookup"><span data-stu-id="0ea54-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="0ea54-107">บันทึกนี้ใช้บริษัทสาธิต DEMF </span><span class="sxs-lookup"><span data-stu-id="0ea54-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="0ea54-108">ไปที่ภาษี > การตั้งค่า > ภาษีขาย > กลุ่มการลงรายการบัญชีแยกประเภท</span><span class="sxs-lookup"><span data-stu-id="0ea54-108">Go to Tax > Setup > Sales tax > Ledger posting groups.</span></span>
2. <span data-ttu-id="0ea54-109">คลิก สร้าง</span><span class="sxs-lookup"><span data-stu-id="0ea54-109">Click New.</span></span>
3. <span data-ttu-id="0ea54-110">ในฟิลด์กลุ่มการลงรายการบัญชีแยกประเภท ให้พิมพ์ค่า</span><span class="sxs-lookup"><span data-stu-id="0ea54-110">In the Ledger posting group field, type a value.</span></span>
4. <span data-ttu-id="0ea54-111">ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า</span><span class="sxs-lookup"><span data-stu-id="0ea54-111">In the Description field, type a value.</span></span>
5. <span data-ttu-id="0ea54-112">ในฟิลด์เจ้าหนี้ภาษีขาย ให้เลือกบัญชีหลักสำหรับภาษีขายขาออกที่ชำระแก่หน่วยงานจัดเก็บภาษี</span><span class="sxs-lookup"><span data-stu-id="0ea54-112">In the Sales tax payable field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span>
    * <span data-ttu-id="0ea54-113">ภาษีขายจะถูกรวบรวมในนามของหน่วยงานจัดเก็บภาษีเมื่อคุณขายสินค้าและบริการที่ต้องเสียภาษี</span><span class="sxs-lookup"><span data-stu-id="0ea54-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="0ea54-114">ในฟิลด์ลูกหนี้ภาษีขาย ให้เลือกบัญชีหลักสำหรับภาษีขายขาเข้าที่ได้รับจากหน่วยงานจัดเก็บภาษี</span><span class="sxs-lookup"><span data-stu-id="0ea54-114">In the Sales tax receivable filed, select the main account for incoming taxes that are received from the tax authority.</span></span>
    * <span data-ttu-id="0ea54-115">ผู้จัดจำหน่ายเรียกเก็บภาษีในนามของหน่วยงานจัดเก็บภาษีเมื่อคุณซื้อสินค้าและบริการที่ต้องเสียภาษี </span><span class="sxs-lookup"><span data-stu-id="0ea54-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="0ea54-116">ฟิลด์นี้จะไม่พร้อมใช้งานเมื่อคุณเลือกตัวเลือกการปรับใช้กฎภาษีสำหรับขายในหน้าพารามิเตอร์บัญชีแยกประเภททั่วไป </span><span class="sxs-lookup"><span data-stu-id="0ea54-116">This field is not available if the Apply sales tax taxation rules option is selected in the General ledger parameters page.</span></span> <span data-ttu-id="0ea54-117">ภาษีขายที่ชำระให้แก่ผู้ขายจะมีการหักบัญชีอัตโนมัติไปยังบัญชีเดียวกับการซื้อแทน</span><span class="sxs-lookup"><span data-stu-id="0ea54-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="0ea54-118">ในฟิลด์ค่าใช้จ่ายภาษีนำเข้า ให้เลือกบัญชีหลักสำหรับการลงบัญชีภาษีนำเข้าที่หักลดได้ซึ่งจะไม่มีการถูกขอคืนหรือถูกรายงานไปยังหน่วยงานจัดเก็บภาษีโดยผู้จัดจำหน่าย เนื่องจากเป็นส่วนหนึ่งของ GST/HST การเก็บภาษีย้อนกลับของ EU</span><span class="sxs-lookup"><span data-stu-id="0ea54-118">In the Use tax expense field, select the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span>
    * <span data-ttu-id="0ea54-119">ตัวเลือกภาษีนำเข้าต้องถูกเลือกสำหรับรหัสภาษีขายในกลุ่มภาษีขายที่ถูกใช้ในธุรกรรม </span><span class="sxs-lookup"><span data-stu-id="0ea54-119">The Use tax option needs to be selected for the Sales tax code in the Sales tax group that is used in the transaction.</span></span>  <span data-ttu-id="0ea54-120">ฟิลด์นี้จะไม่พร้อมใช้งานเมื่อคุณเลือกตัวเลือกการปรับใช้กฎภาษีสำหรับขายในหน้าพารามิเตอร์บัญชีแยกประเภททั่วไป </span><span class="sxs-lookup"><span data-stu-id="0ea54-120">This field is not available if the Apply sales tax taxation rules option is selected in the General ledger parameters page.</span></span>   
8. <span data-ttu-id="0ea54-121">ในฟิลด์เจ้าหนี้ภาษีนำเข้า ให้เลือกบัญชีหลักสำหรับการลงบัญชีภาษีนำเข้าขาเข้าที่ชำระแก่หน่วยงานจัดเก็บภาษี</span><span class="sxs-lookup"><span data-stu-id="0ea54-121">In the Use tax payable field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span>
    * <span data-ttu-id="0ea54-122">ตัวเลือกภาษีนำเข้าต้องถูกเลือกในรหัสภาษีขายในกลุ่มภาษีขายเพื่อลงบัญชีภาษีนำเข้า </span><span class="sxs-lookup"><span data-stu-id="0ea54-122">The Use tax option needs to be selected in the Sales tax code in the Sales tax group to post Use tax.</span></span> <span data-ttu-id="0ea54-123">ถ้ามีการเลือกการใช้กฎภาษีสำหรับภาษีขายในพารามิเตอร์บัญชีแยกประเภททั่วไป จะมีการลงบัญชีออฟเซ็ตในบัญชีค่าใช้จ่ายธุรกรรม</span><span class="sxs-lookup"><span data-stu-id="0ea54-123">If Apply sales tax taxation rules option is selected in General ledger parameters, the offset is posted to the transaction’s expense account.</span></span>   
9. <span data-ttu-id="0ea54-124">ในฟิลด์บัญชีการชำระเงิน ให้เลือกบัญชีหลักซึ่งจะมีการลงบัญชีของยอดดุลสุทธิของบัญชีแยกประเภทที่ระบุในฟิลด์เจ้าหนี้ภาษีนำเข้าและลูกหนี้ภาษีขาย</span><span class="sxs-lookup"><span data-stu-id="0ea54-124">In the Settlement account field, select the main account  that the net balance of the ledger accounts specified in the Use tax payable and Sales tax receivable fields will be posted.</span></span>
    * <span data-ttu-id="0ea54-125">จะมีการสร้างยอดดุลเมื่อมีการชำระภาษีขายและมีการลงรายการบัญชี </span><span class="sxs-lookup"><span data-stu-id="0ea54-125">The balance will be created when the Sales tax settle and post job is ran.</span></span>  <span data-ttu-id="0ea54-126">ถ้าหน่วยงานจัดเก็บภาษีสำหรับรอบระยะเวลาการชำระเงินได้เชื่อมโยงกับบัญชีผู้จัดจำหน่าย ยอดดุลจะถูกลงรายการบัญชีไปยังบัญชีผู้จัดจำหน่ายแทน</span><span class="sxs-lookup"><span data-stu-id="0ea54-126">If the Tax authority for the settlement period is associated with a vendor account, the balance is posted to the vendor account instead.</span></span>   
10. <span data-ttu-id="0ea54-127">ในฟิลด์ส่วนลดเงินสดของผู้จัดจำหน่าย ให้เลือกบัญชีหลักที่จะลงรายการบัญชีส่วนลดเงินสดสำหรับรหัสภาษีขายที่เชื่อมโยงกับกลุ่มการลงรายการบัญชีแยกประเภทนี้</span><span class="sxs-lookup"><span data-stu-id="0ea54-127">In the Vendor cash discount field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span>
    * <span data-ttu-id="0ea54-128">บัญชีหลักในรหัสส่วนลดเงินสดจะถูกนำมาใช้เป็นทางเลือกถ้าไม่มีการป้อนบัญชี </span><span class="sxs-lookup"><span data-stu-id="0ea54-128">This is optional and if no account is entered,  the main account on Cash discount codes will be used.</span></span> <span data-ttu-id="0ea54-129">มีการใช้ประโยชน์ในการใช้บัญชีที่แตกต่างกันต่อกลุ่มการลงรายการบัญชีแยกประเภทถ้ามีการใช้ภาษีขายย้อนกลับในตัวเลือกส่วนลดเงินสดในกลุ่มภาษีขาย</span><span class="sxs-lookup"><span data-stu-id="0ea54-129">It can be useful to use different accounts per Ledger posting group if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="0ea54-130">ในฟิลด์ส่วนลดเงินสดของลูกค้า ให้เลือกบัญชีหลักที่จะลงรายการบัญชีส่วนลดเงินสดสำหรับรหัสภาษีขายที่เชื่อมโยงกับกลุ่มการลงรายการบัญชีแยกประเภทนี้</span><span class="sxs-lookup"><span data-stu-id="0ea54-130">In the Customer case discount field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span>
    * <span data-ttu-id="0ea54-131">บัญชีหลักในรหัสส่วนลดเงินสดจะถูกนำมาใช้เป็นทางเลือกถ้าไม่มีการป้อนบัญชี </span><span class="sxs-lookup"><span data-stu-id="0ea54-131">This is optional and if no account is entered, the main account on the Cash discount codes will be used.</span></span> <span data-ttu-id="0ea54-132">มีการใช้ประโยชน์ในการใช้บัญชีที่แตกต่างกันต่อกลุ่มการลงรายการบัญชีแยกประเภทถ้ามีการใช้ภาษีขายย้อนกลับในตัวเลือกส่วนลดเงินสดในกลุ่มภาษีขาย</span><span class="sxs-lookup"><span data-stu-id="0ea54-132">It can be useful to use different accounts per Ledger posting group if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
12. <span data-ttu-id="0ea54-133">คลิก บันทึก</span><span class="sxs-lookup"><span data-stu-id="0ea54-133">Click Save.</span></span>
