# <a name="create-account-structures"></a>สร้างโครงสร้างทางบัญชี

[!include[task guide banner](../../includes/task-guide-banner.md)]

คำแนะนำงานนี้ระบุขั้นตอนการสร้างโครงสร้างทางบัญชี  ขั้นตอนเหล่านี้ใช้ข้อมูลสาธิตบริษัท USMF

1. ไปที่บัญชีแยกประเภททั่วไป > ผังบัญชี > โครงสร้าง > ตั้งค่าคอนฟิกโครงสร้างทางบัญชี
2. คลิกที่สร้างเพื่อเปิดกล่องโต้ตอบการวาง
3. ในฟิลด์โครงสร้างทางบัญชี ให้พิมพ์ชื่ออธิบายถึงวัตถุประสงค์ของโครงสร้างทางบัญชี
4. ในฟิลด์คำอธิบาย ให้พิมพ์คำอธิบายระบุวัตถุประสงค์ของโครงสร้างทางบัญชี
5. คลิก สร้าง
6. คลิก เพิ่มเซ็กเมนต์
7. ในรายการมิติ ให้เลือกมิติเพื่อเพิ่มโครงสร้างทางบัญชี
8. คลิก เพิ่มเซ็กเมนต์
9. คลิก เพิ่มเซ็กเมนต์

16. ในฟิลด์ผู้ปฏิบัติงาน ให้เลือกตัวเลือก เช่น อยู่ระหว่าง และ รวม
