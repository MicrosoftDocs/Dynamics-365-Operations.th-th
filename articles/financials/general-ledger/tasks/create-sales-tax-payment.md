# <a name="create-a-sales-tax-payment"></a>การสร้างการชำระเงินภาษีขาย

[!include [task guide banner](../../includes/task-guide-banner.md)]

งานชำระและลงรายการบัญชีภาษีขายจะชำระยอดภาษีขายในบัญชีภาษีขาย และออฟเซ็ตไปยังบัญชีการชำระภาษีขายสำหรับรอบระยะเวลาที่กำหนด

1. ไปที่ ภาษี > การรายงาน > ภาษีขาย > ชำระและลงรายการภาษีขาย
2. ในฟิลด์รอบระยะเวลาการจ่ายเงิน ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
4. ในฟิลด์วันที่เริ่มต้น ให้ป้อนวันที่
    * ถ้าไม่มีการเลือกตัวเลือกการแก้ไขรวมบนหน้าพารามิเตอร์บัญชีแยกประเภททั่วไป การชำระบัญชีจะมีการประมวลผลเป็นเวอร์ชันต่างๆ  ต้นฉบับเป็นการชำระบัญชีแรกสำหรับช่วงรอบระยะเวลาและมีการประมวลผลเพียงครั้งเดียวสำหรับช่วงรอบระยะเวลา  การแก้ไขครั้งล่าสุดจะเป็นการชำระธุรกรรมภาษีขายซึ่งมีการลงรายการบัญชีไว้หลังจากสร้างเวอร์ชันต้นฉบับแล้ว   
5. ในฟิลด์วันที่ทำธุรกรรม ให้ป้อนวันที่
6. คลิก ตกลง
