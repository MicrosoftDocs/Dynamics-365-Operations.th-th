# <a name="set-up-main-account-categories"></a>ตั้งค่าประเภทบัญชีหลัก

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

ประเภทบัญชีหลักจะใช้สำหรับรายงานเริ่มต้น ในการรายงานทางการเงิน และ ใน Power BI  ประเภทบัญชีหลักที่ถูกสร้างขึ้น ตามค่าเริ่มต้นสามารถเปลี่ยนชื่อ แต่จะไม่ถูกไม่ลบ  ประเภทบัญชีหลักเพิ่มเติมสามารถสร้างขึ้นได้และใช้เพื่อการรายงานและการวิเคราะห์วัตถุประสงค์  งานนี้ใช้บริษัทสาธิต USMF 


## <a name="create-a-main-account-category"></a>สร้างประเภทบัญชีหลัก
1. ไปที่ บัญชีแยกประเภททั่วไป > ผังบัญชี > บัญชี > ประเภทบัญชีหลัก
2. คลิก สร้าง
3. ในฟิล์ประเภทบัญชีหลัก ให้ป้อนชื่อเฉพาะ
4. ในฟิลด์คำอธิบาย ให้ป้อนคำอธิบายสำหรับประเภทบัญชีหลัก
5. ในชนิดบัญชีหลัก ให้เลือกชนิดบัญชีหลักที่จะเชื่อมโยงกับประเภท

## <a name="link-main-accounts-to-account-category"></a>เชื่อมโยงบัญชีหลักกับประเภทบัญชี
1. คลิก เชื่อมโยงบัญชีหลัก
2. ในรายการนี้ เลือกบัญชีหลักที่จะกำหนดประเภทบัญชีหลัก
    * การกำหนดบัญชีหลักให้กับประเภทบัญชีหลักจะรวมยอดดุลของบัญชีเมื่อใช้ประเภทสำหรับการวิเคราะห์และการรายงานทางการเงิน  
3. ให้เลือก หรือล้างตัวเลือกการเชื่อมโยงเพื่อเลือกบัญชีหลัก
4. คลิก ตกลง ระบบจะนำเข้าข้อมูลการชำระเงิน
5. คลิก ใช่
