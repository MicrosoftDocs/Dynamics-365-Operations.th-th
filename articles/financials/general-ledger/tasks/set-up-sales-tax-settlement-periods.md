# <a name="set-up-sales-tax-settlement-periods"></a>ตั้งค่ารอบระยะเวลาการชำระภาษีขาย

[!include [task guide banner](../../includes/task-guide-banner.md)]

รอบระยะเวลาการจ่ายภาษีขาย ประกอบด้วยข้อมูลเกี่ยวกับช่วงรอบระยะเวลาสำหรับที่ภาษีขายจำเป็นต้องถูกรายงานและชำระเงิน  สามารถดำเนินการกระบวนการชำระเงินได้สำหรับรอบระยะเวลาการชำระเงินสำหรับช่วงวันที่เฉพาะ  รหัสภาษีทั้งหมดที่เชื่อมโยงกับรอบระยะเวลาการจ่ายเงินจะถูกตั้งค่า  ภาระภาษีจะถูกลงรายการบัญชีของผู้จัดจำหน่ายหรือบัญชีแยกประเภททั่วไปอย่างใดอย่างหนึ่ง ขึ้นอยู่กับการตั้งค่าของหน่วยงานจัดเก็บภาษีขายที่เกี่ยวข้อง 



งานนี้ใช้บริษัทสาธิต USMF 



1. ไปที่ภาษี > ภาษีทางอ้อม > ภาษีขาย > ระยะเวลาการชำระภาษีขาย
2. คลิก สร้าง
3. ในฟิลด์ระยะเวลาการชำระเงิน ให้พิมพ์ค่า
4. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
5. ในฟิลด์หน่วยงานจัดเก็บภาษี ให้เลือกหน่วยงานจัดเก็บภาษีที่ได้รับรายงานและการชำระเงินที่สร้างขึ้นสำหรับรอบระยะเวลาการชำระเงิน
6. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
7. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
8. ในฟิลด์เงื่อนไขการชำระเงิน ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
    * หน่วยงานจัดเก็บภาษีขายที่เกี่ยวข้องสามารถถูกตั้งค่าให้เป็นผู้จัดจำหน่ายและการชำระภาษีขายจะสร้างใบแจ้งหนี้ของผู้จัดจำหน่ายคงค้าง  เงื่อนไขการชำระเงินจะกำหนดวันครบกำหนดชำระสำหรับใบแจ้งหนี้ของผู้จัดจำหน่ายคงค้าง  
9. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
10. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
11. เลือกชนิดสำหรับช่วงรอบระยะเวลาการชำระเงิน
12. ป้อนหมายเลขของหน่วยช่วงรอบระยะเวลาต่อรอบระยะเวลา  ตัวอย่างเช่น หนึ่งไตรมาสมี 3 เดือน
13. เลือกหรือยกเลิกการใช้การประมวลผลชุดงานสำหรับกล่องกาเครื่องหมายการชำระภาษีขาย
    * กระบวนการชำระสำหรับรอบระยะเวลาการชำระสามารถประมวลผลเป็นชุดงานในเบื้องหลัง  ซึ่งเหมาะสำหรับธุรกรรมภาษีจำนวนมากภายในช่วงรอบระยะเวลาหนึ่ง  
14. ขยายแท็บช่วงรอบระยะเวลา
15. คลิก เพิ่ม
16. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
17. ในฟิลด์วันที่เริ่มต้น ให้ป้อนวันที่
18. ในฟิลด์วันที่สิ้นสุด ให้ป้อนวันที่
19. คลิกช่วงรอบระยะเวลาใหม่
    * เมื่อมีการป้อนรอบระยะเวลาแรก รอบระยะเวลาใหม่จะถูกสร้างโดยอัตโนมัติ  คุณสามารถกลับมาและเพิ่มช่วงรอบระยะเวลาใหม่ตามความต้องการ  
20. ปิดหน้า
