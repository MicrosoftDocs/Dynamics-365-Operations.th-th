<span data-ttu-id="468fd-104">ฟังก์ชันที่ถูกนำมาใช้ใน Microsoft Dynamics 365 for Finance and Operations รุ่น 8.1 (ตุลาคม 2018) เปิดใช้งานสกุลเงินการรายงานให้ปรับเปลี่ยนวัตถุประสงค์และถูกใช้เป็นสกุลเงินทางบัญชีที่สอง</span><span class="sxs-lookup"><span data-stu-id="468fd-104">Functionality that was introduced in Microsoft Dynamics 365 for Finance and Operations version 8.1 (October 2018) enables the reporting currency to be repurposed and used as a second accounting currency.</span></span> <span data-ttu-id="468fd-105">ฟังก์ชันนี้ถูกอ้างอิงเป็น *สกุลเงินแบบคู่*</span><span class="sxs-lookup"><span data-stu-id="468fd-105">This functionality is referred to as *dual currency*.</span></span> <span data-ttu-id="468fd-106">การเปลี่ยนแปลงสำหรับคู่สกุลเงินไม่สามารถปิดได้ โดยใช้คีย์การตั้งค่าคอนฟิกหรือพารามิเตอร์</span><span class="sxs-lookup"><span data-stu-id="468fd-106">The changes for dual currency can't be turned off through a configuration key or parameter.</span></span> <span data-ttu-id="468fd-107">เนื่องจากมีการใช้สกุลเงินการรายงานเป็นสกุลเงินทางบัญชีที่สอง มีการเปลี่ยนแปลงวิธีการที่มีการคำนวณสกุลเงินการรายงานในตรรกะการลงรายการบัญชี</span><span class="sxs-lookup"><span data-stu-id="468fd-107">Because the reporting currency is used as a second accounting currency, the way that the reporting currency is calculated in the posting logic has changed.</span></span>
