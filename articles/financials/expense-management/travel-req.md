# <a name="travel-requisitions"></a><span data-ttu-id="4509e-104">ใบเบิกค่าเดินทาง</span><span class="sxs-lookup"><span data-stu-id="4509e-104">Travel requisitions</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="4509e-105">*ใบเบิกค่าเดินทาง* เป็นเอกสารต้นทางที่แสดงรายการค่าใช้จ่ายที่จะเกิดขึ้นเพื่อวัตถุประสงค์ในการเดินทาง</span><span class="sxs-lookup"><span data-stu-id="4509e-105">A *travel requisition* is a source document that lists the expenses that will be incurred for the purpose of travel.</span></span> <span data-ttu-id="4509e-106">ใบเบิกค่าเดินทางถูกส่งเพื่อตรวจทาน และจากนั้นจะสามารถนำไปใช้เพื่ออนุมัติค่าใช้จ่ายได้</span><span class="sxs-lookup"><span data-stu-id="4509e-106">A travel requisition is submitted for review and can then be used to authorize expenses.</span></span>

<span data-ttu-id="4509e-107">องค์กรของคุณอาจต้องการให้มีการส่งใบเบิกค่าเดินทาง ก่อนที่พนักงานจะทำให้เกิดค่าใช้จ่ายใดๆ ที่จะถูกเรียกเก็บไปยังองค์กร</span><span class="sxs-lookup"><span data-stu-id="4509e-107">Your organization might require that a travel requisition be submitted before employees incur any expense that is charged to the organization.</span></span> <span data-ttu-id="4509e-108">ใช้ข้อกำหนดนี้ โดยไม่คำนึงว่าพนักงานจะคิดค่าใช้จ่ายไปยังบัตรเครดิตของบริษัท ใช้จ่ายเงินสดที่พวกเขาได้รับจากเงินทดรองจ่าย หรือทำให้เกิดค่าใช้จ่ายจริงที่จะชำระเงินคืนโดยองค์กร</span><span class="sxs-lookup"><span data-stu-id="4509e-108">This requirement applies, regardless of whether employees charge expenses to a corporate credit card, spend cash that they received from a cash advance, or incur out-of-pocket expenses that will be reimbursed by the organization.</span></span>

<span data-ttu-id="4509e-109">ใบเบิกค่าเดินทางและนโยบายสามารถใช้เพื่อช่วยในการควบคุมงบประมาณ</span><span class="sxs-lookup"><span data-stu-id="4509e-109">Travel requisitions and policies can be used to help with budget control.</span></span> <span data-ttu-id="4509e-110">ตัวอย่างเช่น ถ้าองค์กรของคุณกำลังทำงานกับโครงการที่มีราคาคงที่ซึ่งต้องมีการเดินทาง ค่าใช้จ่ายในการเดินทางของสมาชิกทีมงานของโครงการต้องพอดีกับงบประมาณสำหรับโครงการ</span><span class="sxs-lookup"><span data-stu-id="4509e-110">For example, if your organization is working on a fixed-price project that requires travel, the travel expenses of the project's team members must fit within the budget for the project.</span></span> <span data-ttu-id="4509e-111">โดยการกำหนดว่าค่าใช้จ่ายในการเดินทางจะได้รับอนุมัติก่อนที่จะเกิดขึ้น องค์กรสามารถช่วยให้มั่นใจว่าโครงการจะยังคงอยู่ในงบประมาณ</span><span class="sxs-lookup"><span data-stu-id="4509e-111">By requiring that travel expenses be approved before they are incurred, the organization can help make sure that the project remains within budget.</span></span>

<span data-ttu-id="4509e-112">คุณสามารถตั้งค่านโยบายสำหรับใบเบิกค่าเดินทางจากหน้า **ใบเบิกค่าเดินทาง** ได้</span><span class="sxs-lookup"><span data-stu-id="4509e-112">You can set up policies for travel requisitions from the **Travel requisition** page.</span></span>
