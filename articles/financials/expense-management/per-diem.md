# <a name="per-diem-rules"></a><span data-ttu-id="ec11c-103">กฎค่าเบี้ยเลี้ยงต่อวัน</span><span class="sxs-lookup"><span data-stu-id="ec11c-103">Per diem rules</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="ec11c-104">*ค่าเบี้ยเลี้ยงต่อวัน* คือค่าเบี้ยเลี้ยงที่ถูกจ่ายให้ผู้ปฏิบัติงานที่เดินทางเพื่อไปทำงาน</span><span class="sxs-lookup"><span data-stu-id="ec11c-104">A *per diem* is an allowance that is paid to a worker who is traveling for work.</span></span> <span data-ttu-id="ec11c-105">ในการจัดการค่าใช้จ่าย คุณสามารถสร้างกฎค่าเบี้ยเลี้ยงต่อวันสำหรับสถานการณ์การเดินทางต่างๆ ได้</span><span class="sxs-lookup"><span data-stu-id="ec11c-105">In Expense management, you can create per diem rules for various travel situations.</span></span> <span data-ttu-id="ec11c-106">อัตราค่าเบี้ยเลี้ยงต่อวันสามารถขึ้นอยู่กับเวลาของปี สถานที่เดินทาง หรือทั้งสองอย่าง</span><span class="sxs-lookup"><span data-stu-id="ec11c-106">Per diem rates can be based on the time of year, the travel location, or both.</span></span> <span data-ttu-id="ec11c-107">เมื่อคุณสร้างกฎค่าเบี้ยเลี้ยงต่อวัน คุณสามารถระบุได้ว่าจะมีการหักเปอร์เซ็นต์ของอัตราค่าเบี้ยเลี้ยงต่อวันหรือไม่ หากผู้ปฏิบัติงานได้รับประทานอาหารหรือได้รับบริการที่ไม่ต้องเสียค่าใช้จ่าย</span><span class="sxs-lookup"><span data-stu-id="ec11c-107">When you create a per diem rule, you can specify that a percentage of the per diem rate will be withheld if a worker receives complimentary meals or services.</span></span> <span data-ttu-id="ec11c-108">นอกจากนี้คุณยังสามารถตั้งจำนวนชั่วโมงต่ำสุดและสูงสุดที่อัตราค่าเบี้ยเลี้ยงต่อวันสามารถใช้ได้กับการเดินทางของพนักงาน</span><span class="sxs-lookup"><span data-stu-id="ec11c-108">You can also set a minimum and maximum number of hours that the per diem rate can apply to a worker's travel.</span></span>

<span data-ttu-id="ec11c-109">การคำนวณจำนวนค่าเบี้ยเลี้ยงต่อวันรวมจะยึดตามวันที่เริ่มต้นและวันที่สิ้นสุดที่คุณกำหนดไว้ในหน้า **ค่าเบี้ยเลี้ยงต่อวัน**</span><span class="sxs-lookup"><span data-stu-id="ec11c-109">The calculation of a total per diem amount is based on the start dates and end dates that you can set on the **Per diems** page.</span></span>
