[!include [banner](../includes/banner.md)]

<span data-ttu-id="a2f21-104">เมื่อคุณรันการสอบถาม เพื่อดูธุรกรรมในบัญชีแยกประเภททั่วไปหรือการจัดซื้อและการจัดหา แบบสอบถามอาจส่งคืนข้อมูลมากเกินไป</span><span class="sxs-lookup"><span data-stu-id="a2f21-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="a2f21-105">ดังนั้น คุณอาจต้องการจำกัดการค้นหาของคุณ</span><span class="sxs-lookup"><span data-stu-id="a2f21-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="a2f21-106">เพื่อให้ได้ข้อมูลที่คุณต้องการ คุณสามารถดูรายละเอียดแนวลึกลงในธุรกรรมเพื่อดูรายงานค่าใช้จ่ายที่ปรากฏบนนั้น</span><span class="sxs-lookup"><span data-stu-id="a2f21-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="a2f21-107">สำหรับรายงานค่าใช้จ่าย คุณสามารถดูรายละเอียดแนวลึกตั้งแต่ธุรกรรมไปจนถึงรายงานต้นฉบับที่ถูกจัดเก็บในการจัดการค่าใช้จ่าย</span><span class="sxs-lookup"><span data-stu-id="a2f21-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="a2f21-108">นอกเหนือจากยอดรวมของค่าใช้จ่าย คุณสามารถดูธุรกรรมที่แสดงรายการ ธุรกรรมที่เหมาะสมสำหรับการขอคืนภาษีมูลค่าเพิ่ม (VAT) และใบเสร็จรับเงินที่แนบกับค่าใช้จ่ายได้</span><span class="sxs-lookup"><span data-stu-id="a2f21-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="a2f21-109">เพื่อดูรายละเอียดแนวลึกลงในรายงานค่าใช้จ่ายต้นฉบับที่ธุรกรรมที่มีปรากฏ เลือกธุรกรรม และจากนั้นเลือก **เอกสารต้นฉบับ**</span><span class="sxs-lookup"><span data-stu-id="a2f21-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="a2f21-110">เมื่อคุณดูธุรกรรมแต่ละรายการ คุณยังสามารถดูข้อมูลเพิ่มเติมใดๆ ที่แนบกับธุรกรรมได้ด้วย</span><span class="sxs-lookup"><span data-stu-id="a2f21-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="a2f21-111">ตัวอย่างเช่น สำหรับค่าใช้จ่ายโรงแรมที่แสดงรายการ คุณสามารถดูรายการค่าใช้จ่ายโดยละเอียดได้ เช่น การเข้าถึงอินเทอร์เน็ต โทรศัพท์ และบริการห้องพัก</span><span class="sxs-lookup"><span data-stu-id="a2f21-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
