# <a name="customer-posting-profiles"></a>โพรไฟล์การลงรายการบัญชีลูกค้า

[!include[banner](../includes/banner.md)]


โพรไฟล์การลงบัญชีของลูกค้าควบคุมการลงรายการบัญชีของธุรกรรมลูกค้าให้บัญชีแยกประเภททั่วไป

<a name="customer-posting-profiles"></a>โพรไฟล์การลงรายการบัญชีลูกค้า
-------------------------

โพรไฟล์การลงรายการบัญชีลูกค้าช่วยให้คุณสามารถกำหนดบัญชีแยกประเภททั่วไปและการตั้งค่าเอกสารให้กับลูกค้าทั้งหมด กลุ่มลูกค้า หรือลูกค้ารายเดียวได้ การตั้งค่าเหล่านี้จะใช้เมื่อคุณสร้างใบสั่งขาย ใบแจ้งหนี้ข้อความอิสระ การชำระเงินสด จดหมายเรียกเก็บเงิน และดอกเบี้ยตั๋วเงิน สำหรับบางธุรกรรม คุณสามารถเลือกโพรไฟล์การลงบัญชีที่แตกต่าง และมีระดับความสำคัญเหนือกว่า โพรไฟล์การลงบัญชีที่ตั้งค่าไว้แล้วสำหรับธุรกรรมในหน้านี้ 

มีกำหนดค่าเริ่มต้นโพรไฟล์การลงรายการบัญชีในแท็บด่วนบัญชีแยกประเภทและภาษี บนหน้าพารามิเตอร์บัญชีลูกหนี้ ค่าเริ่มต้นโพรไฟล์การลงรายการบัญชีจะรวมอยู่ในส่วนหัวของเอกสารใหม่โดยอัตโนมัติ ซึ่งคุณสามารถเปลี่ยนเป็นโพรไฟล์การลงรายการบัญชีอื่นหากจำเป็น

คุณยังสามารถเชื่อมโยงข้อกำหนดการลงรายการบัญชีกับชนิดการลงรายการบัญชีธุรกรรม ในหน้าข้อกำหนดการลงรายการบัญชีของธุรกรรม ข้อกำหนดการลงรายการบัญชีจะควบคุมการลงรายการบัญชีของธุรกรรมลูกค้าไปยังบัญชีแยกประเภททั่วไปแทนข้อกำหนดการลงรายการบัญชี

## <a name="creating-a-posting-profile"></a>การสร้างการลงรายการบัญชี
ระบุบัญชีแยกประเภทที่จะใช้ในการลงรายการบัญชีธุรกรรมที่มีโพรไฟล์การลงบัญชีที่เลือกไว้ เลือกรหัสบัญชี และหากเป็นไปได้ เลือกหมายเลขบัญชีหรือหมายเลขกลุ่มสำหรับโพรไฟล์การลงบัญชีที่เลือกไว้ ในกระบวนการลงรายการบัญชี ข้อมูลโพรไฟล์การลงบัญชีที่เหมาะสมที่สุดสำหรับแต่ละธุรกรรมจะถูกระบุโดยการค้นหาชุดรหัสบัญชี หมายเลขบัญชี หรือส่วนผสมระหว่างกลุ่มและหมายเลขที่เจาะจงที่สุดตามระดับความสำคัญดังต่อไปนี้

### <a name="table-restrictions"></a>**ข้อจำกัด**

สำหรับธุรกรรมที่มีโพรไฟล์การลงบัญชีที่เลือกไว้ ให้ระบุว่าธุรกรรมจะได้รับชำระเงินโดยอัตโนมัติ จะมีการคำนวณดอกเบี้ย และจะออกจดหมายเรียกเก็บเงิน คุณยังสามารถเลือกบัญชีที่จะใช้เมื่อธุรกรรมที่มีโพรไฟล์การลงบัญชีที่เลือกไว้ปิดลง

ระบุค่าต่อไปนี้เพื่อตั้งค่าโพรไฟล์การลงรายการบัญชีของคุณ:

| ฟิลด์                 | คำอธิบาย                                                                                                                                                                                                                                        |
|-----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **การชำระเงิน**        | เลือกการสลับนี้เพื่อเปิดใช้งานการชำระเงินธุรกรรมที่มีโพรไฟล์การลงรายการบัญชีนี้โดยอัตโนมัติ ถ้าไม่ได้เลือกการสลับนี้ คุณต้องชำระธุรกรรมด้วยตนเองโดยใช้หน้าชำระธุรกรรมที่ค้างอยู่ หรือหน้าป้อนการชำระเงินของลูกค้า |
| **ความสนใจ**          | เลือกการสลับนี้ถ้าควรคำนวณดอกเบี้ยตามยอดดุลค้างชำระของลูกค้า สำหรับบัญชีลูกค้าที่ใช้โพรไฟล์นี้ ถ้าไม่ได้เลือกการสลับนี้ จะไม่มีการคำนวณค่าธรรมเนียมดอกเบี้ยสำหรับลูกค้ารายนี้                                           |
| **จดหมายเรียกเก็บเงิน** | เลือกการสลับนี้ถ้าควรสร้างจดหมายเรียกเก็บเงินสำหรับบัญชีลูกค้าที่ใช้โพรไฟล์นี้ ถ้าไม่ได้เลือกการสลับนี้ จะไม่มีการสร้างจดหมายเรียกเก็บเงินสำหรับลูกค้ารายนี้                                                 |
| **ปิดบัญชี**             | เลือกโพรไฟล์การลงรายการบัญชีที่จะเปลี่ยนเมื่อธุรกรรมที่มีโพรไฟล์การลงรายการบัญชีนี้ปิดบัญชีแล้ว ธุรกรรมนี้จะถือว่าเป็นปิดบัญชีแล้ว เมื่อมีการจับคู่ครบถ้วนแล้ว                                                                           |
