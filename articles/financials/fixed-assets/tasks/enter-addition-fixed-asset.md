# <a name="enter-an-addition-to-a-fixed-asset"></a>การป้อนสินทรัพย์ถาวรที่ได้มาเพิ่มลงในสินทรัพย์ถาวร

[!include [task guide banner](../../includes/task-guide-banner.md)]

กระบวนการนี้จะแสดงวิธีการเพิ่มรายการเพิ่มเติมไปยังทรัพย์สินถาวรที่มีอยู่  วัตถุประสงค์ของส่วนเพิ่มเติมของสินทรัพย์ถาวรเพื่อติดตามการเพิ่มรายการ การบำรุงรักษา หรือการปรับปรุงสำหรับสินทรัพย์ และให้ข้อมูลเท่านั้น  การเปลี่ยนแปลใดๆ กับมูลค่าสินทรัพย์ถาวรหรืออายุการใช้งานจะต้องทำแยกต่างหาก    



กระบวนการใช้บทบาทของนักบัญชีและข้อมูลสาธิตสำหรับนิติบุคคล USMF

1. ไปที่ สินทรัพย์ถาวร > สินทรัพย์ถาวร > สินทรัพย์ถาวร
2. ในรายการ ค้นหาและเลือกสินทรัพย์ถาวรสำหรับเพิ่มเติม
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
4. ในบานหน้าต่างการดำเนินการ คลิกสินทรัพย์ถาวร
5. คลิกเพิ่มเติมสินทรัพย์ถาวร
6. คลิก สร้าง
7. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
8. ตั้งค่าวันที่ของการซื้อเพิ่มหรือบริการ
9. ป้อนต้นทุนของสินค้า การบำรุงรักษา หรือการปรับปรุงอื่นสำหรับสินทรัพย์
10. ในฟิลด์ ปริมาณ ให้ป้อนตัวเลข
    * ต้นทุนทั้งหมดจะไม่มีผลกระทบบนค่าของสินทรัพย์ถาวร และการติดตาม และแจ้งให้ทราบเท่านั้น  ถ้าต้นทุนจะถูกบันทึกเป็นสินทรัพย์ ธุรกรรมการปรับปรุงจะต้องเขียนขึ้นและลงรายการบัญชีแยกต่างหาก  
11. คลิกแท็บ ทั่วไป
    * ตั้งค่าการเพิ่มอายุการใช้งาน ถ้าส่วนเพิ่มเติมจะทำให้อายุการใช้งานของสินทรัพย์เพิ่มขึ้น  
    * ฟิลด์นี้มีไว้เพื่อให้ข้อมูลเท่านั้น  เพื่อเพิ่มอายุการใช้งาน แก้ไขอายุการใช้งานบนรูปแบบมูลค่าและ/หรือ สมุดบัญชีค่าเสื่อมราคาสำหรับสินทรัพย์
