# <a name="split-a-fixed-asset"></a>แบ่งสินทรัพย์ถาวร

[!include [task guide banner](../../includes/task-guide-banner.md)]

คู่มืองานนี้จะแบ่งเป็นเปอร์เซ็นต์ของสมุดบัญชีสินทรัพย์เดียวกับสมุดบัญชีสินทรัพย์ใหม่   มีการเข้าถึงข้อมูลบทบาทของนักบัญชีและข้อมูลสาธิต USMF


## <a name="create-a-new-fixed-asset"></a>สร้างสินทรัพย์ถาวรใหม่
1. ไปที่ สินทรัพย์ถาวร > สินทรัพย์ถาวร > สินทรัพย์ถาวร
2. คลิก สร้าง
3. ในฟิลด์กลุ่มสินทรัพย์ถาวร ให้ป้อนหรือเลือกค่า
4. หมายเหตุ หมายเลขสินทรัพย์ถาวรที่จะใช้ในกระบวนการแบ่งในภายหลัง
5. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
6. ปิดแบบฟอร์ม

## <a name="split-a-fixed-asset"></a>แบ่งสินทรัพย์ถาวร
1. ในรายการ ค้นหาและเลือกสินทรัพย์ถาวรที่จะแบ่ง
2. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
3. คลิก สมุดบัญชี
    * เลือกสมุดบัญชีเพื่อแบ่งไปยังสินทรัพย์ใหม่  
4. คลิกฟังก์ชัน
5. คลิกแบ่งสินทรัพย์ถาวร
6. ในฟิลด์สินทรัพย์ถาวรสิ้นสุด ให้ป้อนหรือเลือกค่า
7. ในฟิลด์ไปยังสมุดบัญชี ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
8. ในฟิลด์วันที่ทำธุรกรรม ให้ป้อนวันที่
9. ในฟิลด์เปอร์เซนต์ ให้ป้อนหมายเลข
10. ในฟิลด์ชื่อสมุดรายวัน ให้ป้อนหรือเลือกค่า
11. คลิก ตกลง

## <a name="post-the-journal-transaction"></a>ลงรายการบัญชีธุรกรรมสมุดรายวัน
1. ไปที่สินทรัพย์ถาวร > รายการสมุดรายวัน > สมุดรายวันสินทรัพย์ถาวร
2. ในรายการ ให้เลือกสมุดรายวันที่สร้างขึ้นด้วยกระบวนการแบ่ง
3. คลิก รายการ
    * ตรวจสอบการสร้างรายการสมุดรายวัน   สร้างธุรกรรมการปรับปรุงการซื้อสินทรัพย์สำหรับสินทรัพย์เดิมเพื่อลดมูลค่าตามเปอร์เซ็นต์ที่ระบุไว้ในระหว่างกระบวนการแบ่ง   สร้างธุรกรรมการซื้อสินทรัพย์สำหรับสินทรัพย์ใหม่สำหรับยอดเงินเดียวกัน  
4. คลิก ลงรายการบัญชี
