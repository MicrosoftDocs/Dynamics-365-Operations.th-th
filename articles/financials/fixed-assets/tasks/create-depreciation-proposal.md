# <a name="create-a-depreciation-proposal"></a>สร้างข้อเสนอค่าเสื่อมราคา

[!include [task guide banner](../../includes/task-guide-banner.md)]

ขั้นตอนนี้อธิบายถึงการทำงานของข้อเสนอของชุดค่าเสื่อมราคา และอธิบายวิธีการเสนอค่าเสื่อมราคาสำหรับสินทรัพย์ถาวร  งานนี้ใช้บริษัทสาธิต USMF และบทบาทของนักบัญชี


## <a name="create-depreciation-proposal"></a>สร้างข้อเสนอค่าเสื่อมราคา
1. ไปที่สินทรัพย์ถาวร > รายการสมุดรายวัน > สร้างข้อเสนอค่าเสื่อมราคา
2. ในฟิลด์ชื่อของสมุดรายวัน ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
4. ในฟิลด์วันที่สิ้นสุด ให้ป้อนวันที่
    * เลือกตัวเลือกการสรุปค่าเสื่อมราคาเพื่อสรุปค่าเสื่อมราคารายเดือนเข้าไปในสมุดรายวันหนึ่ง  
    * ตัวอย่างเช่น ถ้าค่าวันที่สิ้นสุดคือวันที่ 31 มีนาคม 2015 คำอธิบายต่อไปนี้จะถูกสร้างขึ้น: "ค่าเสื่อมราคาตั้งแต่วันที่ 31 มกราคม 2015" ฟิลด์วันที่บนสมุดรายวันที่เสนอจะถูกตั้งเป็น 31 มีนาคม 2015  
    * ข้อเสนอค่าเสื่อมราคาสามารถกรองตามสินทรัพย์ กลุ่มสินทรัพย์ หรือเกณฑ์อื่นๆ โดยใช้ตัวเลือกในการกรอง  
    * เมื่อคุณใช้แบบฟอร์มสร้างข้อเสนอการซื้อสินทรัพย์หรือการเสื่อมราคาสำหรับสินทรัพย์ถาวร คุณสามารถเสนอค่าเสื่อมราคาในชุดงาน  นี่คือคำแนะนำสำหรับข้อเสนอใหญ่ที่จะใช้ทรัพยากรของระบบมากขึ้น  ถ้าคุณเลือกตัวเลือกชุดงาน คุณยังคงสามารถดำเนินงานอื่นๆ ให้สำเร็จในช่วงเวลานั้น  เมื่อคุณเสนอค่าเสื่อมราคาในลักษณะนี้ จะมีการคำนวณค่าเสื่อมราคาสำหรับรูปแบบมูลค่าสำหรับสินทรัพย์ถาวร  
5. คลิกสร้างสมุดรายวัน

## <a name="review-depreciation-entries"></a>ตรวจทานรายการค่าเสื่อมราคา
1. ไปที่สินทรัพย์ถาวร > รายการสมุดรายวัน > สมุดรายวันสินทรัพย์ถาวร
2. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
3. คลิก รายการ
4. คลิก ลงรายการบัญชี
