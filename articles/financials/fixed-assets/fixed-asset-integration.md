<span data-ttu-id="46efa-105">สามารถรวมสินทรัพย์ถาวรกับบัญชีแยกประเภททั่วไป การจัดการสินค้าคงคลัง บัญชีลูกหนี้ และบัญชีเจ้าหนี้ได้ </span><span class="sxs-lookup"><span data-stu-id="46efa-105">Fixed assets can be integrated with General ledger, Inventory management, Accounts receivable, and Accounts payable.</span></span> <span data-ttu-id="46efa-106">คุณยังสามารถตั้งค่าสินทรัพย์ถาวร เพื่อให้รวมกับใบสั่งซื้อได้ด้วย</span><span class="sxs-lookup"><span data-stu-id="46efa-106">You can also set up Fixed assets so that it is integrated with purchase orders.</span></span>

--------------

<span data-ttu-id="46efa-108">ในบัญชีแยกประเภททั่วไป โดยปกติจะมีการสรุปมูลค่าของสินทรัพย์ถาวรทั้งหมดในหลายบัญชีหลักที่จำเป็นสำหรับการรายงานทางการเงิน</span><span class="sxs-lookup"><span data-stu-id="46efa-108">In General ledger, the value of all fixed assets is typically summarized in multiple main accounts that are required for financial reporting.</span></span> <span data-ttu-id="46efa-109">อย่างไรก็ตาม ในหน้า **สินทรัพย์ถาวร** คุณสามารถสร้างเรกคอร์ดสินทรัพย์ถาวรได้หลายรายการ</span><span class="sxs-lookup"><span data-stu-id="46efa-109">However, on the **Fixed assets** page, you can create many fixed asset records.</span></span> <span data-ttu-id="46efa-110">เรกคอร์ดเหล่านี้สามารถรวมข้อมูล เช่น ราคาการซื้อสินทรัพย์ ค่าเสื่อมราคา และประเมินค่า</span><span class="sxs-lookup"><span data-stu-id="46efa-110">These records can include information such as the acquisition price, depreciation, and valuation.</span></span> <span data-ttu-id="46efa-111">แต่ละครั้งที่คุณลงรายบัญชีธุรกรรมสำหรับสินทรัพย์ถาวร บัญชีหลักที่เหมาะสมจะถูกอัพเดต</span><span class="sxs-lookup"><span data-stu-id="46efa-111">Each time that you post a transaction for a fixed asset, the appropriate main accounts are updated.</span></span> <span data-ttu-id="46efa-112">บัญชีหลักสำหรับสินทรัพย์ถาวรแสดงมูลค่าอัพเดตของสินทรัพย์ถาวรเสมอ</span><span class="sxs-lookup"><span data-stu-id="46efa-112">The main accounts for fixed assets always show the updated value of the fixed assets.</span></span>

<span data-ttu-id="46efa-113">ในหน้า **โพรไฟล์การลงรายการบัญชีสินทรัพย์ถาวร** คุณต้องกำหนดบัญชีหลักที่มีการลงรายการบัญชีธุรกรรมแบบจำลองมูลค่าสินทรัพย์ถาวร</span><span class="sxs-lookup"><span data-stu-id="46efa-113">On the **Fixed asset posting profiles** page, you define the main accounts that fixed asset book transactions are posted to.</span></span> <span data-ttu-id="46efa-114">คุณยังระบุชนิดของธุรกรรมสินทรัพย์ถาวรซึ่งมีการลงรายการบัญชีไปที่แต่ละบัญชีหลัก</span><span class="sxs-lookup"><span data-stu-id="46efa-114">You also specify the types of fixed asset transactions that are posted to each main account.</span></span> <span data-ttu-id="46efa-115">คุณสามารถสร้างชุดข้อมูลที่หลากหลายของบัญชีหลักต่างๆสำหรับสินทรัพย์ถาวร ขึ้นอยู่กับระดับของรายละเอียดที่คุณต้องการสำหรับสินทรัพย์ถาวรในบัญชีแยกประเภททั่วไป</span><span class="sxs-lookup"><span data-stu-id="46efa-115">You can create various combinations of main accounts for fixed assets, depending on the level of detail that you want for fixed assets in the general ledger.</span></span> <span data-ttu-id="46efa-116">บัญชีหลักสามารถขึ้นอยู่กับชนิดของธุรกรรม สมุดบัญชี และบัญชีหลักอื่นๆ</span><span class="sxs-lookup"><span data-stu-id="46efa-116">Main accounts can be based on transaction types, books, and other main accounts.</span></span>

## <a name="inventory-management"></a><span data-ttu-id="46efa-117">การบริหารสินค้าคงคลัง</span><span class="sxs-lookup"><span data-stu-id="46efa-117">Inventory management</span></span>
<span data-ttu-id="46efa-118">ในสมุดรายวันสินค้าคงคลังสำหรับสินทรัพย์ถาวร คุณสามารถป้อนการซื้อสินทรัพย์ถาวรที่นิติบุคคลได้ผลิตหรือสร้างขึ้นสำหรับนิติบุคคลเอง</span><span class="sxs-lookup"><span data-stu-id="46efa-118">In the inventory journal for fixed assets, you can enter the acquisition of fixed assets that the legal entity has produced or constructed for itself.</span></span> <span data-ttu-id="46efa-119">จากนั้นคุณสามารถโอนย้ายสินค้าคงคลังเป็นสินทรัพย์ถาวรเป็นการซื้อหรือเป็นส่วนหนึ่งของการซื้อ</span><span class="sxs-lookup"><span data-stu-id="46efa-119">You can then transfer inventory items to fixed assets either as an acquisition or as part of an acquisition.</span></span> 

<span data-ttu-id="46efa-120">คุณยังสามารถซื้อสินทรัพย์ได้โดยใช้ใบสั่งซื้อ</span><span class="sxs-lookup"><span data-stu-id="46efa-120">You can also acquire assets by using purchase orders.</span></span> <span data-ttu-id="46efa-121">เมื่อใบสั่งซื้อมีสินค้าในสินค้าคงคลังที่ถูกออกแบบให้เป็นสินทรัพย์ถาวร การตั้งค่าของตัวเลือก **อนุญาตให้ซื้อสินทรัพย์จากการซื้อได้** ในหน้า **พารามิเตอร์สินทรัพย์ถาวร** กำหนดว่าจะมีการลงรายการการซื้อสินทรัพย์สำหรับสินทรัพย์ถาวรนั้นหรือไม่เมื่อมีการลงรายการใบแจ้งหนี้</span><span class="sxs-lookup"><span data-stu-id="46efa-121">When purchase orders contain inventory items that are designated as fixed assets, the setting of the **Allow asset acquisition from Purchasing** option on the **Fixed assets parameters** page determines whether an acquisition is posted for the fixed asset when the invoice is posted.</span></span> <span data-ttu-id="46efa-122">รายการซื้อหนึ่งรายการจะสร้างสินทรัพย์ถาวรหนึ่งรายการ โดยไม่คำนึงถึงปริมาณ </span><span class="sxs-lookup"><span data-stu-id="46efa-122">One purchasing line will create one fixed asset, regardless of the quantity.</span></span> <span data-ttu-id="46efa-123">ผลกระทบซึ่งการซื้อสินทรัพย์ถาวรมีต่อสินค้าคงคลังจะขึ้นอยู่กับการตั้งค่าของนิติบุคคล</span><span class="sxs-lookup"><span data-stu-id="46efa-123">The effect that the acquisition of fixed assets has on inventory depends on the setup of the legal entity.</span></span> 

<span data-ttu-id="46efa-124">เมื่อสินค้าในสินค้าคงคลังกลายเป็นการซื้อสินทรัพย์ถาวรผ่านสมุดรายวันสินค้าคงคลัง ใบสั่งซื้อ หรือข้อเสนอการซื้อสินทรัพย์ ธุรกรรมการซื้อสมุดบัญชีของสินทรัพย์ถาวรจะถูกสร้างขึ้น</span><span class="sxs-lookup"><span data-stu-id="46efa-124">When an inventory item becomes a fixed asset acquisition through the inventory journal, a purchase order, or an acquisition proposal, a fixed asset book acquisition transaction is created.</span></span> <span data-ttu-id="46efa-125">ถ้าการซื้อสินทรัพย์สมุดบัญชีรวมสมุดบัญชีที่ได้รับอยู่ด้วย ธุรกรรมการซื้อสินทรัพย์สมุดบัญชีค่าเสื่อมราคาจะถูกสร้างขึ้น</span><span class="sxs-lookup"><span data-stu-id="46efa-125">If a book acquisition includes a derived book, the derived book acquisition transaction is also created.</span></span> 

<span data-ttu-id="46efa-126">กฎการลงรายการบัญชีที่มีการตั้งค่าในหน้า **การลงรายการบัญชี** ในการบริหารสินค้าคงคลังควบคุมการลดลงของสินค้าคงคลังเมื่อมีการลงรายการบัญชีการซื้อสินทรัพย์</span><span class="sxs-lookup"><span data-stu-id="46efa-126">Posting rules that are set up on the **Posting** page in Inventory management control the decrease in inventory when an acquisition is posted.</span></span> <span data-ttu-id="46efa-127">อย่างไรก็ตาม คุณไม่ลดสินค้าคงคลังเมื่อคุณลงรายการบัญชีใบแจ้งหนี้ที่เกี่ยวข้องกับสินทรัพย์ถาวรเสมอไป</span><span class="sxs-lookup"><span data-stu-id="46efa-127">However, you don’t always decrease inventory when you post invoices that are related to fixed assets.</span></span> <span data-ttu-id="46efa-128">ในบางกรณี อาจซื้อสินทรัพย์ถาวรสำหรับการใช้ภายใน</span><span class="sxs-lookup"><span data-stu-id="46efa-128">In some cases, the fixed assets might be purchased for internal use.</span></span> <span data-ttu-id="46efa-129">ตัวอย่างได้แก่แล็ปท็อปที่ซื้อสำหรับแผนกขาย</span><span class="sxs-lookup"><span data-stu-id="46efa-129">An example is a laptop that is purchased for the sales department.</span></span> <span data-ttu-id="46efa-130">เมื่อคุณทำงานกับใบสั่งซื้อ คุณสามารถใช้สินค้าที่มีการตั้งค่าสำหรับทั้งการขายต่อและการใช้ภายใน</span><span class="sxs-lookup"><span data-stu-id="46efa-130">When you work with purchase orders, you can use items that are set up for both resale and internal use.</span></span> 

<span data-ttu-id="46efa-131">หากคุณใช้บัญชีรับสินค้าและบัญชีออกสินค้าเฉพาะสำหรับสินทรัพย์ถาวรในกลุ่มสินค้า คุณสามารถใช้สินค้าตัวเดียวกันในสินค้าคงคลังสำหรับทั้งการจัดซื้อภายในและสำหรับสต็อกที่จะนำไปขายต่อได้</span><span class="sxs-lookup"><span data-stu-id="46efa-131">If you use specific receipt and issue accounts on item groups for fixed assets, you can use the same inventory item both for internal purchases and as stock for resale.</span></span> 

<span data-ttu-id="46efa-132">สินทรัพย์ถาวรที่มีไว้สำหรับใช้ภายในจะถูกตั้งค่าเพื่อให้มีชนิดบัญชี **การรับสินทรัพย์ถาวร**</span><span class="sxs-lookup"><span data-stu-id="46efa-132">Fixed assets that are for internal use are set up so that they have an account type of **Fixed asset receipt**.</span></span> <span data-ttu-id="46efa-133">ชนิดบัญชีนี้ถูกใช้เพื่อติดตามการรับสินทรัพย์ถาวร</span><span class="sxs-lookup"><span data-stu-id="46efa-133">This account type is used to track the receipt of the fixed asset.</span></span> <span data-ttu-id="46efa-134">เมื่อคุณลงรายการบัญชีใบแจ้งหนี้ของผู้จัดจำหน่าย ใช้บัญชีการรับสินทรัพย์ถาวร ถ้าเงื่อนไขข้อใดข้อหนึ่งต่อไปนี้เป็นจริง:</span><span class="sxs-lookup"><span data-stu-id="46efa-134">When you post a vendor invoice, use the fixed asset receipt account if any of the following conditions is true:</span></span>

-   <span data-ttu-id="46efa-136">กล่องกาเครื่องหมาย **สินทรัพย์ถาวร?** ถูกเลือกสำหรับรายการใบรับสินค้าที่มีการลงรายการบัญชี</span><span class="sxs-lookup"><span data-stu-id="46efa-136">The **New fixed asset?** check box is selected for the product receipt line that is posted.</span></span>
