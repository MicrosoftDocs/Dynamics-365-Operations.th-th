# <a name="fixed-assets-roll-forward-report"></a>รายงานการกระทบยอดสินทรัพย์ถาวร

[!include[banner](../includes/banner.md)]

รายงาน **การกระทบยอดสินทรัพย์ถาวร** แสดง ในรูปแบบ Microsoft Excel ที่อ่านได้อย่างง่ายดาย ข้อมูลสินทรัพย์ถาวรโดยละเอียดที่คุณต้องการสำหรับการปิดรอบระยะเวลา งบการเงิน และการรายงานภาษี รายงานรวมยอดดุลเริ่มต้นและยอดดุลสิ้นสุดสำหรับสินทรัพย์ถาวร รวมทั้งความเคลื่อนไหวของการประเมินค่าสำหรับรอบระยะเวลา และการซื้อสินทรัพย์ใหม่ และการตัดจำหน่ายใดๆ ที่เกิดขึ้นในระหว่างรอบระยะเวลา มีการรายงานข้อมูลสำหรับสินทรัพย์ถาวรแต่ละรายการ และยังมีการสรุปค่าสำหรับกลุ่มสินทรัพย์ถาวรและนิติบุคคลอีกด้วย

รายงาน **การกระทบยอดสินทรัพย์ถาวร** ใช้กรอบงานการรายงานทางอิเล็กทรอนิกส์ (ER) ก่อนที่คุณจะสามารถรันรายงานได้ แบบจำลองของสินทรัพย์ถาวรและการตั้งค่าคอนฟิกการกระทบยอดสินทรัพย์ถาวรต้องถูกนำเข้าจาก Microsoft Dynamics Lifecycle Services (LCS) สำหรับคำแนะนำ ให้ดูที่ [ดาวน์โหลดการตั้งค่าคอนฟิกการรายงานแบบอิเล็กทรอนิกส์จาก Lifecycle Services](https://docs.microsoft.com/en-us/dynamics365/unified-operations/dev-itpro/analytics/download-electronic-reporting-configuration-lcs)

รายงานนี้จะพร้อมใช้งานใน Microsoft Dynamics 365 for Finance and Operations, Enterprise edition 7.3 หรือ เป็นโปรแกรมแก้ไขด่วนสำหรับ Microsoft Dynamics 365 Finance and Operations, Enterprise edition (กรกฏาคม 2017) ต้องใช้โปรแกรมแก้ไขด่วนสามโปรแกรมกับสภาพแวดล้อมที่มีการนำออกใช้เดือนกรกฎาคม 2017:

- **KB 4041754:** การตั้งค่าคอนฟิกการรายงานทางอิเล็กทรอนิกส์ (ER) ไม่สามารถดาวน์โหลดได้จาก LCS เนื่องจากไม่สามารถใช้ได้สำหรับเวอร์ชันของแอพลิเคชันปัจจุบัน หลังจากการใช้แพคเกจการปรับปรุงแพลตฟอร์ม
- **KB 4056107:** การรายงานทางอิเล็กทรอนิกส์ (GER) การอัพเดตแบบสะสม 5
- **KB 4056353:** รายงานใบแจ้งยอดและบันทึกสินทรัพย์ถาวรไม่ตรงกับความต้องการใน GAAP และ IFRS
