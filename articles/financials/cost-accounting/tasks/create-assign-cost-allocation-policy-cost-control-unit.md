# <a name="create-and-assign-a-cost-allocation-policy-to-a-cost-control-unit"></a>สร้างและกำหนดนโยบายการปันส่วนต้นทุนสำหรับหน่วยการควบคุมต้นทุน

[!include[task guide banner](../../includes/task-guide-banner.md)]

ใช้ขั้กระบวนงานนี้เพื่อสร้างและกำหนดนโยบายการปันส่วนต้นทุนและกฎที่สอดคล้องกันกับหน่วยการควบคุมต้นทุน การบันทึกข้อมูลนี้ใช้บริษัทข้อมูลสาธิต USP2


## <a name="create-a-policy"></a>สร้างนโยบาย
1. ไปที่ การบัญชีต้นทุน > นโยบาย > นโยบายการปันส่วนต้นทุน
2. คลิก สร้าง
3. ในฟิลด์ชื่อนโยบาย ให้พิมพ์ค่า
4. ในฟิลด์ลำดับชั้นมิติออบเจ็กต์ต้นทุน ให้ป้อนหรือเลือกค่า
    * เลือกองค์กร  
5. ในฟิลด์มิติทางสถิติ ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
6. คลิก บันทึก

## <a name="create-allocation-rules"></a>สร้างกฎการปันส่วน
1. คลิก สร้าง
2. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
3. ในฟิลด์โหนดลำดับชั้นมิติออบเจ็กต์ต้นทุน ให้ป้อนหรือเลือกค่า
4. ในฟิลด์ พฤติกรรมต้นทุน เลือก 'รวม'
5. ในฟิลด์ฐานการปันส่วน ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
6. คลิก สร้าง
7. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
8. ในฟิลด์โหนดลำดับชั้นมิติออบเจ็กต์ต้นทุน ให้ป้อนหรือเลือกค่า
9. ในฟิลด์ พฤติกรรมต้นทุน เลือก 'รวม'
10. ในฟิลด์ฐานการปันส่วน ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
11. คลิก สร้าง
12. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
13. ในฟิลด์โหนดลำดับชั้นมิติออบเจ็กต์ต้นทุน ให้ป้อนหรือเลือกค่า
14. ในฟิลด์ พฤติกรรมต้นทุน เลือก 'รวม'
15. ในฟิลด์ฐานการปันส่วน ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
    * ทำต่อไปจนกว่าคุณจะได้สร้างกฎทั้งหมด  
16. คลิก บันทึก

## <a name="assign-the-policy-to-a-cost-control-unit"></a>กำหนดนโยบายให้กับหน่วยควบคุมต้นทุน
1. คลิก การกำหนดนโยบายสำหรับหน่วยควบคุมต้นทุน
2. คลิก สร้าง
3. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
4. ในฟิลด์ มีผลบังคับใช้นับจากวันที่ทางการบัญชี ให้ป้อนวันที่
    * กฎมีผลบังคับวันที่ ผู้ใช้หรือระบบอาจยกเลิกการใช้กฎได้ถ้ามีการสร้างเวอร์ชันใหม่  
5. ในฟิลด์หน่วยการควบคุมต้นทุน ให้ป้อนหรือเลือกค่า
6. คลิก บันทึก
