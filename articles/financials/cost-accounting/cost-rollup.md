# <a name="cost-rollup-policy-and-overhead-calculation"></a><span data-ttu-id="93e93-103">นโยบายการรวบรวมต้นทุนและการคำนวณค่าโสหุ้ย</span><span class="sxs-lookup"><span data-stu-id="93e93-103">Cost rollup policy and overhead calculation</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="93e93-104">การบัญชีต้นทุนช่วยให้คุณได้รับข้อมูลเชิงลึกว่า กระแสต้นทุนเกี่ยวข้องกับผลิตภัณฑ์และบริการที่ถูกจัดส่งภายในองค์กรได้อย่างไร</span><span class="sxs-lookup"><span data-stu-id="93e93-104">Cost accounting lets you gain insight into how the cost flow relates to the products and services that are delivered within an organization.</span></span> <span data-ttu-id="93e93-105">เมื่อต้องการดูความโปร่งใสของต้นทุน จำเป็นต้องดำเนินการปันส่วนต้นทุนระหว่างออบเจ็กต์ต้นทุนที่ขึ้นอยู่กับฐานการปันส่วนที่เหมาะสมให้สำเร็จ</span><span class="sxs-lookup"><span data-stu-id="93e93-105">To see the cost transparency, it's crucial to achieve cost allocation between cost objects based on an appropriate allocation basis.</span></span> <span data-ttu-id="93e93-106">โดยค่าเริ่มต้น การปันส่วนต้นทุนได้บรรลุเป้าหมายสำหรับองค์ประกอบต้นทุนหลัก ที่ต้องการในบางสถานการณ์ แต่มีผลกระทบสองสามรายการที่ควรได้รับการพิจารณา</span><span class="sxs-lookup"><span data-stu-id="93e93-106">By default, the cost allocation is achieved for the primary cost element, which is desired in some situations, but it has a few implications that should be considered.</span></span>

-   <span data-ttu-id="93e93-108">ปริมาตรของรายการต้นทุนที่สร้างขึ้นโดยการคำนวณค่าโสหุ้ยอาจสูงมาก</span><span class="sxs-lookup"><span data-stu-id="93e93-108">The volume of cost entries generated by overhead calculation can be very high.</span></span>

-   <span data-ttu-id="93e93-109">ไม่สามารถติดตามกระแสต้นทุนระหว่างออบเจ็กต์ต้นทุนได้</span><span class="sxs-lookup"><span data-stu-id="93e93-109">It is not possible to track the cost flow between cost objects.</span></span>

<span data-ttu-id="93e93-110">เพื่อหลีกเลี่ยงผลกระทบเหล่านี้ การบัญชีต้นทุนช่วยให้คุณสามารถตั้งค่าคอนฟิกการปันส่วนต้นทุนได้ เพื่อให้พอดีกับข้อกำหนดในการรายงานการจัดการองค์กรจัดของคุณ</span><span class="sxs-lookup"><span data-stu-id="93e93-110">To avoid these implications, Cost accounting lets you configure cost allocation to fit into your organization managerial reporting requirements.</span></span> <span data-ttu-id="93e93-111">หัวข้อนี้อธิบายวิธีที่คุณสามารถกำหนดระดับที่ถูกต้องขององค์ประกอบต้นทุนรอง และสร้างกฎการรวบรวมต้นทุนที่พอดีกับการรายงานขององค์กรและความสามารถในการติดตามต้นทุน</span><span class="sxs-lookup"><span data-stu-id="93e93-111">This topic discusses how you can determine the correct level of secondary cost elements and create cost rollup rules that fit into organization reporting and cost traceability.</span></span>

> [!NOTE]
> <span data-ttu-id="93e93-112">คุณสามารถเปลี่ยนการตั้งค่าคอนฟิกได้ ถ้าข้อกำหนดในรายงานเปลี่ยน</span><span class="sxs-lookup"><span data-stu-id="93e93-112">You can change configurations if reporting requirements change.</span></span>
