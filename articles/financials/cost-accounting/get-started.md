# <a name="get-started-with-cost-accounting"></a>เริ่มต้นใช้งานการบัญชีต้นทุน

[!include[banner](../includes/banner.md)]

การเริ่มต้นด้วยการบัญชีต้นทุนมีหลายวิธี แต่ละหัวข้อต่อไปนี้แสดงข้อมูลเฉพาะที่จะช่วยให้คุณสามารถเรียนรู้เพิ่มเติมเกี่ยวกับโมดูลนี้
