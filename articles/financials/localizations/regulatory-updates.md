# <a name="regulatory-updates"></a>การอัพเดตที่เป็นระเบียบบังคับ

[!include [banner](../includes/banner.md)]

หัวข้อนี้แสดงรายการอัพเดตที่จำเป็นที่วางแผนไว้สำหรับการนำออกใช้ในภาษาท้องถิ่นที่รองรับ Microsoft Dynamics 365 for Finance and Operations กำหนดเวลาการจัดส่งอาจมีการเปลี่ยนแปลง และฟังก์ชันที่คาดการณ์ไว้อาจเปลี่ยนแปลงหรืออาจไม่สามารถนำออกใช้ อ่าน [นโยบาย Microsoft](https://go.microsoft.com/fwlink/p/?linkid=2007332) สำหรับข้อมูลเพิ่มเติม 

การอัพเดตที่จำเป็นมีความแตกต่างคุณลักษณะที่เป็นระเบียบบังคับ เมื่อต้องการเรียนรู้เกี่ยวกับคุณลักษณะต่าง ๆ ที่เป็นระเบียบบังคับที่วางแผนไว้หรือนำออกใช้ ให้อ้างอิงทรัพยากรต่อไปนี้

- [Dynamics 365 และบันทึกย่อประจำรุ่นแพลตฟอร์ม Power](https://docs.microsoft.com/business-applications-release-notes/index)
- [มีอะไรใหม่สำหรับ Microsoft Dynamics 365 for Finance and Operations](../../fin-and-ops/get-started/whats-new-changed.md)

เมื่อมีการใช้เดือนในคอลัมน์วันที่ จะมีการวางแผนการอัพเดตที่จำเป็นให้มีการจัดส่งภายในเดือนนั้นๆ วันที่จัดส่งอาจเป็นวันใดๆ ภายในเดือนนั้นก็ได้ ไม่ใช่แค่วันแรกของเดือนก็ได้

|วันที่รีลีสที่วางแผนไว้|นำเวอร์ชันออกใช้|ประเทศ|การอัปเดตที่เป็นระเบียบบังคับ|
|--------------------|---------------|-------|-------|
|      2019 มีนาคม          |   10.0.0      | ออสเตรีย      |   การเปลี่ยนแปลงใน Kammerumlage 2019 (ภาษีซื้อ)    |
|      2019 มีนาคม          |   10.0.0      |   บราซิล    |     การจ่ายเงินสมทบ SPED EFD - โครงร่าง  004 และ 005  |
|      2019 มีนาคม          |   10.0.0      |    มาเลเซีย     |ประเทศมาเลเซียแทน GST ด้วยการขาย SST & ภาษีบริการ        |

## <a name="additional-resources"></a>ทรัพยากรเพิ่มเติม
- ค้นหาการอัพเดตที่เป็นระเบียบบังคับที่นำออกใช้แล้วและวางแผนไว้ทั้งหมดใน [Lifecycle Service - การค้นหาปัญหา](https://lcs.dynamics.com/Logon/Index) (จำเป็นต้องเข้าสู่ระบบ)
- สำหรับรายการของภาษาท้องถิ่นที่ได้รับการสนับสนุนสำหรับ Dynamics 365 for Finance and Operations ดูที่ [คำแนะนำความพร้อมใช้งานระหว่างประเทศ](https://aka.ms/dynamics_365_international_availability_deck)
