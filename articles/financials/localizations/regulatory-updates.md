# <a name="regulatory-updates"></a>การอัพเดตที่เป็นระเบียบบังคับ

[!include [banner](../includes/banner.md)]

หัวข้อนี้แสดงรายการอัพเดตที่จำเป็นที่วางแผนไว้สำหรับการนำออกใช้ในภาษาท้องถิ่นที่รองรับ Microsoft Dynamics 365 for Finance and Operations กำหนดเวลาการจัดส่งอาจมีการเปลี่ยนแปลง และฟังก์ชันที่คาดการณ์ไว้อาจเปลี่ยนแปลงหรืออาจไม่สามารถนำออกใช้ อ่าน [นโยบาย Microsoft](https://go.microsoft.com/fwlink/p/?linkid=2007332) สำหรับข้อมูลเพิ่มเติม 

การอัปเดตที่เป็นระเบียบบังคับคือคุณลักษณะที่นำมาใช้เพื่อสนับสนุนกฎหมายเฉพาะประเทศใหม่หรือที่มีการเปลี่ยนแปลง เมื่อต้องการเรียนรู้เกี่ยวกับคุณลักษณะเฉพาะประเทศที่วางแผนไว้หรือนำออกใช้อื่นๆ ให้อ้างอิงทรัพยากรต่อไปนี้:

- [Dynamics 365 และบันทึกย่อประจำรุ่น Power Platform](https://docs.microsoft.com/business-applications-release-notes/index)
- [มีอะไรใหม่สำหรับ Microsoft Dynamics 365 for Finance and Operations](../../fin-and-ops/get-started/whats-new-changed.md)

Microsoft มุ่งมั่นที่จะใช้ข้อกำหนดที่เป็นระเบียบข้อบังคับใหม่โดยเร็วที่สุด วันที่จัดส่งจริงขึ้นอยู่กับวันที่ประกาศกฎ ความพร้อมของรายละเอียดข้อกำหนดจากหน่วยงานท้องถิ่น ความพร้อมใช้งานของเครื่องมือการตรวจสอบ และขนาดและความซับซ้อนของการเปลี่ยนแปลง 

เราวางแผนที่จะส่งการอัพเดตที่จำเป็นในการอัปเดตบริการแบบหนึ่งเวอร์ชันที่ถูกนำออกใช้ทันเวลาสำหรับลูกค้าที่จะปรับปรุงและพร้อมสำหรับวันบังคับใช้ (สำหรับการอัพเดตที่เป็นระเบียบข้อบังคับเกี่ยวกับธุรกรรม) หรือสำหรับกำหนดเวลาสิ้นสุดการรายงานที่บังคับแรก (สำหรับการอัพเดตที่เป็นระเบียบข้อบังคับที่เกี่ยวข้องกับการรายงาน) ลูกค้าและคู่ค้าสามารถดูตัวอย่างการอัพเดตที่เป็นระเบียบข้อบังคับใหม่ได้ในโปรแกรมการนำการแสดงตัวอย่างก่อนมาใช้ (Preview Early Adoption Program หรือ PEAP)

ในกรณีที่วันที่ประกาศล่าช้า ความพร้อมของรายละเอียดข้อกำหนดหรือเครื่องมือการตรวจสอบล่าช้า และขนาดและความซับซ้อนของการเปลี่ยนแปลงมากเป็นพิเศษ อาจไม่สามารถทำให้การอัพเดตที่เป็นระเบียบข้อบังคับพร้อมใช้งานภายในวันที่ของความพร้อมใช้งานทั่วไปของการอัพเดตรายเดือน ในกรณีดังกล่าว การอัพเดตที่เป็นระเบียบข้อบังคับจะถูกจัดส่งเป็นการแก้ไขใหม่สำหรับการอัพเดตรายเดือนที่พร้อมใช้งานบางรายการ
