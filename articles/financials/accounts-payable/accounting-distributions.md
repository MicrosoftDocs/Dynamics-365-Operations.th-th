# <a name="accounting-distributions"></a><span data-ttu-id="70564-104">การกระจายการลงบัญชี</span><span class="sxs-lookup"><span data-stu-id="70564-104">Accounting distributions</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="70564-105">บทความนี้แสดงข้อมูลเกี่ยวกับการกระจายการลงบัญชี และอธิบายถึงตัวเลือกที่พร้อมใช้งานสำหรับการประมวลผลเหล่านั้น</span><span class="sxs-lookup"><span data-stu-id="70564-105">This article provides information about accounting distributions and describes the options that are available for processing them.</span></span> <span data-ttu-id="70564-106">การกระจายการลงบัญชีจะใช้ในการปันส่วนยอดเงินสำหรับเอกสารต้นทางไปยังบัญชีแยกประเภทเฉพาะ</span><span class="sxs-lookup"><span data-stu-id="70564-106">Accounting distributions are used to allocate monetary amounts for a source document to specific ledger accounts.</span></span> 

<span data-ttu-id="70564-107">การกระจายการลงบัญชีมีความสามารถของโปรแกรมที่ถ้วนทั่วในวงกว้างที่จะใช้ และขยาย โดยแต่ละเอกสารต้นทาง, อาทิเช่น ใบสั่งซื้อ ใบแจ้งหนี้ของผู้จัดจำหน่าย รายงานค่าใช้จ่าย และใบแจ้งหนี้ข้อความอิสระ</span><span class="sxs-lookup"><span data-stu-id="70564-107">Accounting distributions are a program-wide capability that is used and extended by each source document, such as a purchase order, vendor invoice, expense report, and free text invoice.</span></span> <span data-ttu-id="70564-108">โดยค่าเริ่มต้น, การกระจายการลงบัญชีเริ่มต้นจะถูกสร้างขึ้นสำหรับแต่ละรายการเอกสารต้นทางและยอดเงิน, และจะเปิดใช้งานสำหรับการแก้ไขอย่างมีเงื่อนไข</span><span class="sxs-lookup"><span data-stu-id="70564-108">By default, a default accounting distribution is generated for each source document line and monetary amount, and is conditionally enabled for modification.</span></span> 

> [!Note] 
> <span data-ttu-id="70564-109">เอกสารบางรายการยังสนับสนุนยอดเงินที่จัดทำเอกสารส่วนหัว เช่นค่าธรรมเนียมสำหรับใบสั่ง และใบแจ้งหนี้</span><span class="sxs-lookup"><span data-stu-id="70564-109">Some documents also support header document monetary amounts, such as charges for orders and invoices.</span></span> 

<span data-ttu-id="70564-110">ความสามารถในการกระจายการลงบัญชีทั่วไปให้ตัวเลือกดังต่อไปนี้สำหรับการประมวลผลการกระจายการลงบัญชี:</span><span class="sxs-lookup"><span data-stu-id="70564-110">The generic accounting distribution capabilities provide the following options for processing accounting distributions:</span></span>

    -   <span data-ttu-id="70564-112">สำหรับการกระจายยอดเงินสูงสุด (การกระจายหลัก), บัญชีหลักและมิติทางการเงินอาจจะแก้ไขได้โดยตรงในการควบคุมรายการเป็นส่วน ๆ ในกริด</span><span class="sxs-lookup"><span data-stu-id="70564-112">For the top monetary amount distributions (parent distributions), the main account and financial dimensions might be editable directly in the segmented entry control in the grid.</span></span> <span data-ttu-id="70564-113">ราคารวมเป็นตัวอย่างทั่วไปของการกระจายหลักดังกล่าว</span><span class="sxs-lookup"><span data-stu-id="70564-113">The extended price is a typical example of such a parent distribution.</span></span>
    -   <span data-ttu-id="70564-114">ยอดเงินการกระจายขึ้นอยู่กับสกุลเงินเพิ่มเติมสำหรับเอกสาร</span><span class="sxs-lookup"><span data-stu-id="70564-114">The distributions amounts are based on the term currency for the document.</span></span> <span data-ttu-id="70564-115">โดยทั่วไป สกุลเงินนี้คือ สกุลเงินทางธุรกรรม</span><span class="sxs-lookup"><span data-stu-id="70564-115">Typically, this currency is the transaction currency.</span></span> <span data-ttu-id="70564-116">การลงบัญชี และการรายงานยอดเงินของสกุลเงินจะถูกสร้างขึ้นให้เป็นส่วนหนึ่งของการลงบัญชีในบัญชีแยกประเภทย่อย</span><span class="sxs-lookup"><span data-stu-id="70564-116">Accounting and reporting currency amounts are generated as part of subledger accounting.</span></span>
    -   <span data-ttu-id="70564-117">การกระจายแสดงวันที่ลงบัญชีและเหตุการณ์ทางบัญชี</span><span class="sxs-lookup"><span data-stu-id="70564-117">The distributions display the accounting date and accounting event.</span></span> <span data-ttu-id="70564-118">โดยทั่วไป เหตุการณ์ทางบัญชีถูกตั้งค่าเป็น **ไม่มี** จนกว่าเอกสารจะลงรายการบัญชี/ในสมุดรายวัน</span><span class="sxs-lookup"><span data-stu-id="70564-118">Typically, the accounting event is set to **None** until the document is posted/journalized.</span></span> <span data-ttu-id="70564-119">ณ จุดที่ เหตุการณ์ทางบัญชีกลายเป็น **ดั้งเดิม**</span><span class="sxs-lookup"><span data-stu-id="70564-119">At that point, the accounting event becomes **Original**.</span></span> <span data-ttu-id="70564-120">หลังจากที่มีการลงรายการบัญชีการกระจาย คุณไม่สามารถแก้ไขการกระจายได้</span><span class="sxs-lookup"><span data-stu-id="70564-120">After the distributions have been posted, you can't modify the distributions.</span></span>

    -   <span data-ttu-id="70564-123">ปุ่ม**กระจายเท่าๆกัน** สามารถนำมาใช้ร่วมกับ **แบ่ง** เพื่อปันส่วนจำนวนเท่า ๆ กันระหว่างกระจายทั้งหมดได้โดยอัตโนมัติ</span><span class="sxs-lookup"><span data-stu-id="70564-123">The **Distribute equally** button can be used in combination with **Split** to automatically allocate the amount equally across all distributions.</span></span>
    -   <span data-ttu-id="70564-124">ปุ่ม**รีเซ็ต** อาจถูกเปิดใช้งานสำหรับการกระจายหลักเมื่อมีการกระจายมากกว่าหนึ่งครั้ง</span><span class="sxs-lookup"><span data-stu-id="70564-124">**Reset** button might be enabled for parent distributions when more than one distributions exists.</span></span> <span data-ttu-id="70564-125">**รีเซ็ต** กลับรายการการปรับเปลี่ยนการกระจายใด ๆ ด้วยตนเอง โดยการลบการกระจายที่มีอยู่ทั้งหมด และสร้างการกระจายเริ่มต้นอีกครั้ง</span><span class="sxs-lookup"><span data-stu-id="70564-125">**Reset** reverses any manual modification to the distribution by deleting all existing distributions and re-generating the default distributions.</span></span>
