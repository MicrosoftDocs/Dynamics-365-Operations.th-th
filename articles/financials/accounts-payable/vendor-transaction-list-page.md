    <span data-ttu-id="caeee-111">กล่องโต้ตอบ **ดูการจับคู่** เปิดขึ้น และแสดงธุรกรรมที่เลือกและเอกสารทั้งหมดที่จับคู่ไว้กับรายการนั้น</span><span class="sxs-lookup"><span data-stu-id="caeee-111">The **View settlements** dialog box opens and shows the selected transaction and all documents that are settled against it.</span></span> <span data-ttu-id="caeee-112">กล่องโต้ตอบนี้คล้ายคลึงกับมุมมองประวัติการชำระเงิน แต่จะประกอบด้วยเอกสารที่เกี่ยวข้องทั้งหมด</span><span class="sxs-lookup"><span data-stu-id="caeee-112">This dialog box resembles the settlement history view, but it includes all related documents.</span></span>
