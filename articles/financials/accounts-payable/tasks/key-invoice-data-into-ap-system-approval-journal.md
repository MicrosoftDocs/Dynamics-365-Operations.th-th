# <a name="key-invoice-data-into-accounts-payable-using-an-approval-journal"></a>ข้อมูลใบแจ้งหนี้ที่สำคัญในบัญชีเจ้าหนี้โดยใช้สมุดรายวันการอนุมัติ

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

คู่มือของงานนี้จะแสดงวิธีการใช้ทะเบียนใบแจ้งหนี้เพื่อสร้างใบแจ้งหนี้ และใช้สมุดรายวันการอนุมัติสำหรับการอัพเดตบัญชีค่าใช้จ่าย


## <a name="create-and-post-and-invoice"></a>สร้างและลงรายการบัญชีและใบแจ้งหนี้
1. ไปที่บัญชีเจ้าหนี้ > ใบแจ้งหนี้ > ทะเบียนใบแจ้งหนี้
2. คลิก สร้าง
3. เลือกชื่อทะเบียนใบแจ้งหนี้ที่คุณต้องการจะใช้
4. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
5. คลิกที่รายการเพื่อเปิดการลงทะเบียนและป้อนรายการค่าใช้จ่าย
6. เลือกผู้จัดจำหน่าย ตัวอย่างเช่น ป้อนหรือเลือก US-104
7. ในฟิลด์ใบแจ้งหนี้ ให้พิมพ์ค่า
8. ในฟิลด์คำอธิบาย ให้พิมพ์ค่า
9. ในฟิลด์เครดิต ให้ป้อนหมายเลข
10. ในฟิลด์อนุมัติโดย ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
11. ให้เน้นที่ผู้อนุมัติและคลิกที่เลือกเพื่อเลือกผู้อนุมัติ
12. คลิก ลงรายการบัญชี
13. ปิดหน้า
14. ปิดหน้า

## <a name="approve-an-invoice"></a>อนุมัติใบแจ้งหนี้
1. ไปที่บัญชีเจ้าหนี้ > ใบแจ้งหนี้ > การอนุมัติใบแจ้งหนี้
2. คลิก สร้าง
3. เลือกชื่อสมุดรายวันการอนุมัติใบแจ้งหนี้ที่คุณต้องการจะใช้
4. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
5. คลิกที่รายการเพื่อแสดงหน้าที่คุณจะสามารถเลือกใบแจ้งหนี้ที่คุณต้องการอนุมัติ
6. เลือกค้นหาใบสำคัญเพื่อแสดงใบแจ้งหนี้ทั้งหมดที่พร้อมสำหรับการอนุมัติ
7. ทำเครื่องหมายใบแจ้งหนี้ที่คุณสร้างขึ้น
8. คลิกเลือก 
    * ใบสำคัญที่คุณเลือกไว้ด้านบนจะถูกย้ายไปยังรายการนี้หลังจากที่คุณทำการเลือก  
9. คลิก ตกลง
10. คลิกที่ฟิลด์หมายเลขบัญชีเพื่อเพิ่มบัญชีค่าใช้จ่ายในใบแจ้งหนี้
11. ป้อนหมายเลขบัญชีและปิดฟิลด์  ตัวอย่างเช่น ป้อน 600120
12. คลิก ลงรายการบัญชี
13. คลิกใบสำคัญเพื่อดูรายการที่ลงรายการบัญชีไว้
    * บัญชีการอนุมัติใบแจ้งหนี้ที่ค้างอยู่เป็นการกลับรายการและแทนที่ด้วยบัญชีค่าใช้จ่ายจริง
