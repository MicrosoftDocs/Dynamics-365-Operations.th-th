# <a name="project-mobile-applications-overview"></a>ภาพรวมแอพลิเคชันมือถือของโครงการ

[!include [banner](../includes/banner.md)]

แอพลิเคชันบนมือถือสามรายการที่เกี่ยวข้องกับเวลาของโครงการพร้อมใช้งานสำหรับ Microsoft Dynamics 365: Microsoft Dynamics 365 Project Timesheet รายการข้อมูลเวลาของโครงการ และแผ่นเวลาของฉัน/แผ่นเวลาสำหรับการตรวจทานของฉัน (ปรับให้เหมาะกับมือถือ)

## <a name="project-timesheet-mobile-app"></a>แอพบนมือถือของแผ่นเวลาโครงการ

แอพบนมือถือแผ่นเวลาของโครงการ Dynamics 365 ช่วยให้ผู้ใช้สามารถส่งและอนุมัติแผ่นเวลาสำหรับโครงการบนอุปกรณ์เคลื่อนที่ของพวกเขา แอพบนมือถือนี้จัดการฟังก์ชันแผ่นเวลาในการจัดการโครงการและพื้นที่การลงบัญชีของ Microsoft Dynamics 365 for Finance and Operations ซึ่งช่วยปรับปรุงประสิทธิภาพการทำงานและประสิทธิภาพของผู้ใช้ และยังช่วยเกี่ยวกับรายการตามเวลาที่กำหนดและการอนุมัติแผ่นเวลาโครงการอีกด้วย

## <a name="project-time-entry-workspace"></a>พื้นที่ทำงานการป้อนข้อมูลเวลาของโครงการ

พื้นที่ทำงานการป้อนข้อมูลเวลาของโครงการช่วยให้ผู้ใช้สามารถป้อนและประหยัดเวลาโดยเทียบกับโครงการ อย่างไรก็ตาม ไม่ได้อนุญาตให้ผู้ใช้ส่งหรืออนุมัติแผ่นเวลาโครงการ สามารถใช้สำหรับการป้อนข้อมูลเวลาเท่านั้น พื้นที่ทำงานแบบเคลื่อนที่นี้มีจุดมุ่งหมายเพื่อใช้ได้กับแอพลิเคชัน Microsoft Dynamics 365 for Unified Operations Mobile

## <a name="my-timesheetstimesheets-for-my-review"></a>แผ่นเวลาของฉัน/แผ่นเวลาสำหรับการตรวจทานของฉัน

แผ่นเวลา (ปรับให้เหมาะกับมือถือ) และแผ่นเวลาสำหรับการตรวจทานของฉัน (ปรับให้เหมาะกับมือถือ) ประกอบขึ้นเป็นโซลูชันที่ช่วยให้พนักงานสร้าง ส่ง และอนุมัติแผ่นเวลาโครงการบนอุปกรณ์เคลื่อนที่ ผู้ปฏิบัติงานสามารถเข้าถึงฟังก์ชัน Finance and Operations ผ่านทาง URL สำหรับแอพลิเคชัน โซลูชันนี้ไม่จำเป็นต้องใช้แอพลิเคชัน Dynamics 365 for Unified Operations Mobile
