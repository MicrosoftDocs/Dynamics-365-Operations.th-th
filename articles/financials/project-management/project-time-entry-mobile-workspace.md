# <a name="project-time-entry-mobile-workspace"></a>พื้นที่ทำงานแบบเคลื่อนที่สำหรับการป้อนข้อมูลเวลาโครงการ

[!include [banner](../includes/banner.md)]

หัวข้อนี้แสดงข้อมูลเกี่ยวกับพื้นที่ทำงานแบบเคลื่อนที่ของ **การป้อนข้อมูลเวลาของโครงการ** พื้นที่ทำงานนี้ช่วยให้ผู้ใช้สามารถป้อนและประหยัดเวลาในแง่ของโครงการโดยใช้อุปกรณ์เคลื่อนที่ของตน

พื้นที่ทำงานแบบเคลื่อนที่นี้มีจุดมุ่งหมายเพื่อใช้ได้กับแอป Microsoft Dynamics 365 for Unified Operations mobile 

## <a name="overview"></a>ภาพรวม
ในส่วนหนึ่งของการทำงานประจำวัน ทรัพยากรโครงการมักอยู่ที่ไซต์หรืออยู่ในระหว่างทาง พื้นที่ทำงานแบบเคลื่อนที่ของ **การป้อนข้อมูลเวลาของโครงการ** ช่วยให้ผู้ใช้สามารถป้อนเวลาที่เรียกเก็บเงินได้หรือที่เรียกเก็บเงินไม่ได้สำหรับโครงการของตนบนอุปกรณ์เคลื่อนที่ที่ผู้ใช้เลือก ดังนั้น ทรัพยากรโครงการสามารถบันทึกรายการเวลาเมื่อใดก็ได้ และที่ใดก็ได้ และยังสามารถดูรายการเวลาที่มีการบันทึกแล้ว 

กล่าวคือ ในพื้นที่ทำงานแบบเคลื่อนที่ **รายการข้อมูลเวลาของโครงการ** ผู้ใช้สามารถปฏิบัติงานเหล่านี้:

-   สำหรับวันใด ๆ ที่เลือก ให้ป้อนจำนวนชั่วโมงที่คุณใช้ในงานนั้น ๆ
-   ค้นหาตามชื่อของโครงการหรือลูกค้าเพื่อค้นหาโครงการที่จะป้อนเวลา
-   ระบุประเภทและกิจกรรมสำหรับเวลาที่คุณใช้
-   บันทึกเวลาเป็นที่เรียกเก็บเงินได้หรือที่เรียกเก็บเงินไม่ได้สำหรับโครงการ
-   เลือกที่จะป้อนข้อคิดเห็นภายในหรือภายนอก

## <a name="prerequisites"></a>ข้อกำหนดเบื้องต้น
ข้อกำหนดเบื้องต้นแตกต่างกันไป โดยขึ้นอยู่กับรุ่นของ Microsoft Dynamics 365 ที่ได้ถูกปรับใช้สำหรับองค์กรของคุณ

ถ้ามีการปรับใช้ Microsoft Dynamics 365 for Finance and Operations สำหรับองค์กรของคุณ ผู้ดูแลระบบต้องเผยแพร่พื้นที่ทำงานแบบเคลื่อนที่ **รายการเวลาโครงการ** สำหรับคำแนะนำ ให้ดูที่ [เผยแพร่พื้นที่ทำงานแบบเคลื่อนที่](../../dev-itpro/mobile-apps/publish-mobile-workspace.md)

## <a name="download-and-install-the-mobile-app"></a>ดาวน์โหลดและติดตั้งแอพบนมือถือ

ดาวน์โหลดและติดตั้งแอพบนมือถือ Dynamics 365 for Unified Operations

-   [สำหรับโทรศัพท์ Android](https://go.microsoft.com/fwlink/?linkid=850662)
-   [สำหรับ iPhone](https://go.microsoft.com/fwlink/?linkid=850663)

## <a name="sign-in-to-the-mobile-app"></a>ล็อกอินเข้าสู่แอพบนมือถือ
1.  เริ่มการทำงานแอพบนอุปกรณ์เคลื่อนที่ของคุณ
2.  ป้อน URL ของ Dynamics 365
3.  ในครั้งแรกที่คุณเข้าสู่ระบบ ระบบจะขอให้คุณป้อนชื่อผู้ใช้และรหัสผ่านของคุณ ป้อนข้อมูลประจำตัวของคุณ
4.  หลังจากที่คุณลงชื่อเข้าใช้แล้ว พื้นที่ทำงานที่พร้อมใช้งานสำหรับบริษัทของคุณจะแสดงขึ้น หมายเหตุว่าถ้าผู้ดูแลระบบของคุณเผยแพร่พื้นที่ทำงานใหม่ในภายหลัง คุณจะต้องรีเฟรชรายการของพื้นที่ทำงานแบบเคลื่อนที่

[![ดึงเพื่อรีเฟรช](./media/pull-to-refresh-list-of-workspaces-183x300.png)](./media/pull-to-refresh-list-of-workspaces.png)

## <a name="enter-time-by-using-the-project-time-entry-mobile-workspace"></a>ป้อนเวลาโดยใช้พื้นที่ทำงานแบบเคลื่อนที่ของการป้อนข้อมูลเวลาของโครงการ
1.  บนอุปกรณ์เคลื่อนที่ เลือกพื้นที่ทำงาน **การป้อนข้อมูลเวลาของโครงการ**
2.  เลือก **การป้อนข้อมูลเวลา** วันที่ในปฏิทินสำหรับสัปดาห์ปัจจุบันจะแสดงขึ้น
3.  สำหรับวันที่เลือก ให้เลือก **การดำเนินการ** &gt; **รายการใหม่**
4.  ป้อนจำนวนชั่วโมงเพื่อบันทึก
5.  เลือกโครงการที่จะป้อนข้อมูลเวลา รายการแสดงโครงการที่ถูกโหลดลงในแอพของคุณสำหรับการใช้งานแบบออฟไลน์ โดยค่าเริ่มต้น จะมีการโหลดไว้ 50 รายการ แต่นักพัฒนาสามารถเปลี่ยนแปลงจำนวนนี้ได้ สำหรับข้อมูลเพิ่มเติม ดูที่ [แพลตฟอร์มเคลื่อนที่](../../dev-itpro/mobile-apps/platform/mobile-platform-home-page.md)
6.  ถ้าโครงการของคุณไม่ได้อยู่ในรายการ เลือก **ค้นหา** ค้นหาตามชื่อหรือสลับไปยังการค้นหาตามชื่อของโครงการหรือลูกค้า
7.  เลือกประเภท  รายการแสดงประเภทที่ถูกโหลดลงในแอพของคุณสำหรับการใช้งานแบบออฟไลน์ โดยค่าเริ่มต้น จะมีการโหลดไว้ 50 รายการ แต่นักพัฒนาสามารถเปลี่ยนแปลงจำนวนนี้ได้ สำหรับข้อมูลเพิ่มเติม ดูที่ [แพลตฟอร์มเคลื่อนที่](../../dev-itpro/mobile-apps/platform/mobile-platform-home-page.md)
8.  ถ้าประเภทของคุณไม่ได้อยู่ในรายการ เลือก **ค้นหา** ค้นหาตามประเภทหรือสลับไปยังการค้นหาตามชื่อประเภท
9.  เลือกกิจกรรม รายการแสดงกิจกรรมที่ถูกโหลดลงในแอพของคุณสำหรับการใช้งานแบบออฟไลน์ โดยค่าเริ่มต้น จะมีการโหลดไว้ 50 รายการ แต่นักพัฒนาสามารถเปลี่ยนแปลงจำนวนนี้ได้ สำหรับข้อมูลเพิ่มเติม ดูที่ [แพลตฟอร์มเคลื่อนที่](../../dev-itpro/mobile-apps/platform/mobile-platform-home-page.md)
10. ถ้ากิจกรรมของคุณไม่ได้อยู่ในรายการ เลือก **ค้นหา** ค้นหาตามหมายเลขกิจกรรม หรือสลับไปยังการค้นหาตามวัตถุประสงค์

11. เลือกลักษณะของรายการ
12. เลือกระบุได้: ป้อนข้อคิดเห็นภายในและภายนอก
13. เลือก **เสร็จสิ้น**
