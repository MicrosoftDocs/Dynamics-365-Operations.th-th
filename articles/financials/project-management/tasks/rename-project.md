# <a name="rename-or-modify-a-project-stage"></a>เปลี่ยนชื่อหรือแก้ไขระยะโครงการ

[!include [task guide banner](../../includes/task-guide-banner.md)]

ขั้นของโครงการจะทำหน้าที่เป็นการชี้นำวงจรชีวิตของผลิตภัณฑ์ คุณสามารถกำหนดระยะโครงการที่คุณต้องการรวมไว้ในโครงการของคุณได้ ในพื้นที่ระยะโครงการในหน้าพารามิเตอร์การบัญชีและการจัดการโครงการ Dynamics 365 for Finance and Operations ให้ขั้นตอนโครงการที่กำหนดไว้ล่วงหน้าห้าขั้นตอน และขั้นตอนที่กำหนดโดยผู้ใช้สามขั้นตอน คุณสามารถปรับปรุงชื่อระยะโครงการที่คุณใช้ในการติดตามความคืบหน้าของโครงการได้ คุณยังสามารถให้ชื่อสำหรับชื่อสถานะที่กำหนดโดยผู้ใช้ได้อีกด้วย งานนี้แสดงวิธีการเปลี่ยนชื่อหรือปรับเปลี่ยนระยะโครงการ และใช้ชุดข้อมูล USSI

1. ไปที่ การจัดการและการบัญชีโครงการ > การตั้งค่า > พารามิเตอร์การจัดการและการบัญชีโครงการ
2. คลิกแท็บระยะโครงการ
3. ในฟิลด์ภาษา เลือกภาษาที่คุณต้องการใช้สำหรับชื่อระยะโครงการใหม่
4. ในรายการ เลือกขั้นที่กำหนดไว้ล่วงหน้าหรือกำหนดโดยผู้ใช้ที่คุณต้องการแก้ไข 
5. ในฟิลด์ระยะโครงการ ป้อนชื่อใหม่
6. คลิก บันทึก
