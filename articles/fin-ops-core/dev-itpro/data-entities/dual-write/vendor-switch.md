# <a name="switch-between-vendor-designs"></a>สลับการออกแบบของผู้จัดจำหน่าย

[!include [banner](../../includes/banner.md)]



## <a name="vendor-data-flow"></a>โฟลว์ข้อมูลของผู้จัดจำหน่าย 

ถ้าคุณเลือกที่จะใช้เอนทิตี้ **บัญชี** เพื่อจัดเก็บผู้จัดจำหน่ายของชนิด **องค์กร** และเอนทิตี้ **ผู้ติดต่อ** เพื่อจัดเก็บผู้จัดจำหน่ายของชนิด **บุคคล** ให้ตั้งค่าคอนฟิกลำดับงานต่อไปนี้ มิฉะนั้น ไม่จำเป็นต้องมีการตั้งค่าคอนฟิกนี้

## <a name="use-the-extended-vendor-design-for-vendors-of-the-organization-type"></a>ใช้การออกแบบของผู้จัดจำหน่ายแบบขยายสำหรับผู้จัดจำหน่ายชนิดองค์กร

แพคเกจโซลูชัน **Dynamics365FinanceExtended** ประกอบด้วยเท็มเพลตการประมวลผลลำดับงานต่อไปนี้ คุณจะสร้างลำดับงานสำหรับเท็มเพลตแต่ละรายการ

+ สร้างผู้จัดจำหน่ายในเอนทิตี้บัญชี
+ สร้างผู้จัดจำหน่ายในเอนทิตีผู้จัดจำหน่าย
+ ปรับปรุงผู้จัดจำหน่ายในเอนทิตี้บัญชี
+ ปรับปรุงผู้จัดจำหน่ายในเอนทิตีผู้จัดจำหน่าย

เพื่อสร้างกระบวนการลำดับงานใหม่โดยใช้เท็มเพลตการประมวลผลลำดับงาน ให้ทำตามขั้นตอนเหล่านี้

1. สร้างกระบวนการลำดับงานสำหรับเอนทิตี้ **ผู้จัดจำหน่าย** และเลือกเท็มเพลตกระบวนการลำดับงาน **สร้างผู้จัดจำหน่ายในเอนทิตี้บัญชี** จากนั้น เลือก **ตกลง** ลำดับงานนี้จะจัดการสถานการณ์จำลองการสร้างผู้จัดจำหน่าย สำหรับเอนทิตี **บัญชี**

    ![สร้างผู้จัดจำหน่ายในกระบวนการลำดับงานเอนทิตี้บัญชี](media/create_process.png)

2. สร้างกระบวนการลำดับงานสำหรับเอนทิตี้ **ผู้จัดจำหน่าย** และเลือกเท็มเพลตกระบวนการลำดับงาน **ปรับปรุงผู้จัดจำหน่ายในเอนทิตี้บัญชี** จากนั้น เลือก **ตกลง** ลำดับงานนี้จะจัดการสถานการณ์จำลองการอัปเดตผู้จัดจำหน่าย สำหรับเอนทิตี **บัญชี**
3. สร้างกระบวนการลำดับงานสำหรับเอนทิตี้ **บัญชี** และเลือกเท็มเพลตกระบวนการลำดับงาน **สร้างผู้จัดจำหน่ายในเอนทิตี้ผู้จัดจำหน่าย**
4. สร้างกระบวนการลำดับงานสำหรับเอนทิตี้ **บัญชี** และเลือกเท็มเพลตกระบวนการลำดับงาน **ปรับปรุงผู้จัดจำหน่ายในเอนทิตี้ผู้จัดจำหน่าย**
5. คุณสามารถตั้งค่าคอนฟิกลำดับงานเป็นลำดับงานแบบเรียลไทม์ หรือลำดับงานพื้นหลัง อย่างใดอย่างหนึ่ง โดยขึ้นอยู่กับความต้องการของคุณ เมื่อต้องการตั้งค่าคอนฟิกลำดับงานเป็นลำดับงานพื้นหลัง เลือก **แปลงเป็นลำดับงานพื้นหลัง**

    ![ปุ่มแปลงเป็นลำดับงานพื้นหลัง](media/background_workflow.png)

6. เปิดใช้ลำดับงานที่คุณสร้างขึ้นสำหรับเอนทิตี้ **บัญชี** และ **ผู้จัดจำหน่าย** เพื่อเริ่มต้นใช้งานเอนทิตี้ **บัญชี** เพื่อจัดเก็บข้อมูลสำหรับผู้จัดจำหน่ายของชนิด **องค์กร**
