# <a name="data-templates-with-multiple-worksheets"></a>เทมเพลตข้อมูลที่มีเวิร์กชีตหลายรายการ

[!include [banner](../includes/banner.md)]


[!INCLUDE [PEAP](../../../includes/peap-1.md)]

การจัดการข้อมูลในแอปลิเคชั่น สนับสนุนเทมเพลตสำหรับเอนทิตีข้อมูลที่สร้างตาม Microsoft Excel เทมเพลตเหล่านี้ประกอบด้วยแผ่นงานอย่างน้อยหนึ่งแผ่น เทมเพลตที่มีแผ่นงานหลายแผ่นมักถูกใช้ เมื่อสะดวกในการจัดการข้อมูลในแฟ้มเดียว และในการนำเข้าไปยังเอนทิตีข้อมูลหลายรายการ ตัวอย่างอาจเป็นไซต์และคลังสินค้า

## <a name="upload-a-file-once-and-map-it-to-all-entities"></a>อัพโหลดไฟล์ครั้งเดียว และแมปกับเอนทิตีทั้งหมด
ลองดูตัวอย่างที่มีไฟล์ Excel หนึ่งไฟล์ซึ่งมีแผ่นงานที่เรียกว่า **ไซต์** และ **คลังสินค้า** ในการตั้งค่าโครงการนำเข้าข้อมูล คุณก็จะสามารถเพิ่มเอนทิตีข้อมูลแรก **ไซต์** แล้วจากนั้น อัพโหลดไฟล์ คุณจะสามารถเลือก **ไซต์** เป็นแผ่นงานที่จะใช้สำหรับเอนทิตีนี้ได้

ถ้าคุณเพิ่มเอนทิตีที่สอง **คลังสินค้า** โดยไม่ออกจากฟอร์ม **เพิ่มไฟล์** การค้นหาแผ่นงานจะช่วยให้คุณสามารถเลือกแผ่นงาน **คลังสินค้า** ได้ โดยไม่ต้องอัพโหลดไฟล์อีกครั้ง เหตุผลเดียวในการอัพโหลดไฟล์ใหม่คือ ถ้าข้อมูล **คลังสินค้า** อยู่ในไฟล์อื่น

![แผ่นงานหลายแผ่น](./media/AddFileMultipleWorkSheets.png)

## <a name="fix-worksheet-to-entity-mapping"></a>กำหนดแผ่นงานให้การแมปเอนทิตี

คุณสามารถกำหนดการแมปของแผ่นงานให้กับเอนทิตีข้อมูลในงานการนำเข้าจากกริดได้ คอลัมน์ **แผ่นงาน** ในกริด แสดงแผ่นงานจากไฟล์ที่ถูกแมป คุณสามารถเลือกแผ่นงานอื่นจากเมนูแบบหล่นลงได้ ถ้ามีการแมปแผ่นงานที่เลือกไปยังเอนทิตีในโครงการข้อมูล ระบบจะขอให้คุณยืนยันการเปลี่ยนแปลง เราขอแนะนำให้ คุณแก้ไขการแมปทั้งหมดในกริด

![อัปเดตการแมปแผ่นงาน](./media/UpdateMappings.png)

## <a name="re-map-to-a-new-file"></a>แมปไปยังแฟ้มใหม่อีกครั้ง

ในกรณีที่ต้องอัพโหลดเป็นเวอร์ชันใหม่ของไฟล์เดียวกันหรือไฟล์ใหม่ทั้งหมด สำหรับเอนทิตีที่มีอยู่ในโครงการข้อมูล คุณต้องใช้อัปเดต **เพิ่มไฟล์** แล้วเพิ่มเอนทิตีอีกครั้ง ราวกับว่ามีการเพิ่มเป็นครั้งแรก ระบบจะยืนยันว่า คุณต้องการเขียนทับเอนทิตีที่มีอยู่ในโครงการข้อมูลก่อนดำเนินการต่อ เอนทิตีที่ไม่ได้ถูกเพิ่มอีกครั้ง (หรือถูกเขียนทับ) จะยึดการแมปก่อนหน้าจากไฟล์ก่อนหน้านี้ต่อไป

## <a name="upload-a-file-using-run-project"></a>อัพโหลดไฟล์โดยใช้ เรียกใช้โครงการ

คุณสามารถอัปโหลดไฟล์ Excel ได้ในขณะที่ใช้ตัวเลือก **เรียกใช้โครงการ** เพื่อดำเนินการโครงการที่นำเข้าได้ คุณต้องระมัดระวังในการอัพโหลดเฉพาะไฟล์ที่มีแผ่นงานเหมือนกับการแมปที่มีอยู่ในเอนทิตีข้อมูลในโครงการข้อมูล ถ้าไม่พบแผ่นงานในไฟล์ที่เพิ่งอัพโหลด ระบบจะแสดงข้อผิดพลาด และจะหยุดการนำเข้า ถ้าต้องเปลี่ยนการแมปไปยังแผ่นงานสำหรับเอนทิตี การแมปในโครงการข้อมูลจึงต้องมีการอัปเดตจากภายในโครงการข้อมูลเป็นอันดับแรก ก่อนที่จะใช้ไฟล์ในประสบการณ์ **เรียกใช้โครงการ**
