# <a name="use-record-template-to-create-a-new-record"></a>ใช้เท็มเพลตเรกคอร์ดเพื่อสร้างเรกคอร์ดใหม่

[!include [banner](../../includes/banner.md)]

กระบวนงานนี้แสดงวิธีการใช้เท็มเพลตเรกคอร์ดที่กำหนดไว้ก่อนหน้านี้เพื่อสร้างเรกคอร์ดใหม่ เพื่อทำกระบวนงานนี้ให้เสร็จสมบูรณ์ อันดับแรกคุณต้องดำเนินกระบวนงาน "สร้างเท็มเพลตเรกคอร์ดเพื่อช่วยในการป้อนข้อมูล" ให้เสร็จสมบูรณ์

กระบวนงานนี้ใช้บริษัท USMF

1. ใน **บานหน้าต่างนำทาง** ไปที่ **สินทรัพย์ถาวร > สินทรัพย์ถาวร > สินทรัพย์ถาวร**
2. คลิก **สร้าง** ระบบจะขอให้คุณเลือกเท็มเพลต เลือกรายการที่ตรงกับความต้องการทางธุรกิจของคุณ  
3. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
4. คลิก **ตกลง**
