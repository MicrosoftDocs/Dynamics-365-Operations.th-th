# <a name="upper-er-function"></a>ฟังก์ชัน UPPER ER

[!include [banner](../includes/banner.md)]

ฟังก์ชัน `UPPER` ส่งกลับสตริงข้อความที่ระบุเป็นค่า *สตริง* หลังจากที่ถูกแปลงเป็นตัวพิมพ์ใหญ่

## <a name="syntax"></a>ไวยากรณ์

```vb
UPPER (text )
```

## <a name="arguments"></a>อาร์กิวเมนต์

`text`: *สตริง*

พาธที่ถูกต้องของแหล่งข้อมูลของชนิด *สตริง*

## <a name="return-values"></a>ส่งคืนค่า

*สตริง*

ค่าข้อความที่เป็นผลลัพธ์

## <a name="example"></a>ตัวอย่าง

`UPPER ("Sample")` ส่งกลับ **"ตัวอย่าง"**
