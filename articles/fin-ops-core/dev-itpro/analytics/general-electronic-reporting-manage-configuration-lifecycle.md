## <a name="er-configuration-development-lifecycle"></a><span data-ttu-id="fe676-146">วงจรชีวิตพัฒนาการตั้งค่าคอนฟิก ER</span><span class="sxs-lookup"><span data-stu-id="fe676-146">ER configuration development lifecycle</span></span>
