# <a name="file-destination"></a>ปลายทางของไฟล์

[!include [banner](../includes/banner.md)]

คุณสามารถตั้งค่าคอนฟิกปลายทางของไฟล์สำหรับแต่ละโฟลเดอร์หรือส่วนประกอบของไฟล์ของรูปแบบการรายงานทางอิเล็กทรอนิกส์ (ER) ที่มีการตั้งค่าคอนฟิกเพื่อสร้างเอกสารขาออก เอกสารที่สร้างขึ้นพร้อมใช้งานสำหรับการดาวน์โหลดจากเว็บเบราว์เซอร์ ตามการตั้งค่าปลายทาง

บนหน้า **การตั้งค่าปลายทาง** หากคุณตั้งค่า **เปิดใช้งานอยู่** เป็น **ใช่** กล่องโต้ตอบการเปิดหรือบันทึกจะเปิดขึ้นเมื่อการตั้งค่าคอนฟิกรันเสร็จเรียบร้อยแล้ว
