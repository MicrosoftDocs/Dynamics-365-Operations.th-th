# <a name="list-of-er-functions-in-the-mathematical-category"></a>รายการของฟังก์ชั่น ER ในประเภทคณิตศาสตร์

[!include [banner](../includes/banner.md)]

ฟังก์ชันทางคณิตศาสตร์ของการรายงานอิเล็กทรอนิกส์ (ER) สามารถใช้ในการคำนวณคณิตศาสตร์ทั่วไปมากมาย บทความนี้แสดงสรุปของฟังก์ชันเหล่านี้

## <a name="list-of-supported-functions"></a>รายการฟังก์ชันที่ได้รับการสนับสนุน

| ฟังก์ชัน | คำอธิบาย |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | ฟังก์ชันนี้ส่งกลับค่าเงินเต็ม (โมดูล) ของจำนวนที่ระบุเป็นค่า *จริง* กล่าวคือ ส่งคืนหมายเลขโดยไม่มีเครื่องหมาย |
| [กำลัง](er-functions-mathematical-power.md)         | ฟังก์ชันนี้ส่งกลับค่า *จริง* ที่แสดงถึงผลของการเพิ่มจำนวนบวกที่ระบุให้กับพลังงานที่ระบุ |
| [Round](er-functions-mathematical-round.md)         | ฟังก์ชันนี้ส่งคืนหมายเลขที่ระบุเป็นค่า *จริง* หลังจากที่ได้ถูกปัดเศษเป็นจำนวนทศนิยมที่ระบุ |
| [RoundDown](er-functions-mathematical-rounddown.md) | ฟังก์ชันนี้ส่งคืนหมายเลขที่ระบุเป็นค่า *จริง* หลังจากที่ได้ถูกปัดเศษลงเป็นจำนวนทศนิยมที่ระบุ |
| [RoundUp](er-functions-mathematical-roundup.md)     | ฟังก์ชันนี้ส่งคืนหมายเลขที่ระบุเป็นค่า *จริง* หลังจากที่ได้ถูกปัดเศษขึ้นเป็นจำนวนทศนิยมที่ระบุ |
