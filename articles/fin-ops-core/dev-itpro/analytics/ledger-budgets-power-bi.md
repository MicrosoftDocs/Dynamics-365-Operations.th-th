# <a name="actual-vs-budget-power-bi-content"></a>เนื้อหา Power BI ข้อมูลจริงเทียบกับงบประมาณ

[!include [banner](../includes/banner.md)]

หัวข้อนี้อธิบายเนื้อหา Microsoft Power BI **ข้อมูลจริงเทียบกับงบประมาณ** และยังอธิบายถึงวิธีการเข้าถึงรายงาน Power BI และแสดงข้อมูลเกี่ยวกับแบบจำลองข้อมูลและเอนทิตี้ที่ใช้ในการสร้างเนื้อหานี้

## <a name="overview"></a>ภาพรวม

เนื้อหา Power BI **ข้อมูลจริงเทียบกับงบประมาณ** ถูกสร้างสำหรับบุคคลที่รับผิดชอบการตรวจสอบประสิทธิภาพข้อมูลจริงเทียบกับงบประมาณในองค์กรของพวกเขา เนื้อหา Power BI ของ **ข้อมูลจริงเทียบกับงบประมาณ** ให้ความสามารถในการมองเห็นไปยังผลต่างของงบประมาณของคุณ คุณสามารถวิเคราะห์งบประมาณสำหรับปีปัจจุบันโดยเรียงตามประเภทบัญชี รหัสงบประมาณ บัญชีหลัก คำอธิบายเกี่ยวกับบัญชีหลัก หรือรอบระยะเวลาทางบัญชีเพื่อที่จะทำความเข้าใจกับสาเหตุของผลต่างใด ๆ ได้ดีขึ้น

## <a name="accessing-the-power-bi-content"></a>การเข้าถึงเนื้อหา Power BI
รายงานจากเนื้อหา Power BI ของ **ข้อมูลจริงเทียบกับงบประมาณ** ถูกแสดงในพื้นที่ทำงาน **งบประมาณบัญชีแยกประเภทและการคาดการณ์** และ **CFO**

## <a name="reports-that-are-included-in-the-power-bi-content"></a>รายงานที่รวมอยู่ในเนื้อหา Power BI
ตารางต่อไปนี้ให้รายละเอียดเกี่ยวกับการวัดที่ถูกพบในหน้ารายงานแต่ละหน้าในเนื้อหา Power BI ของ **ข้อมูลจริงเทียบกับงบประมาณ**

| รายงาน                      | เมตริก                                                                             |
|-----------------------------|-------------------------------------------------------------------------------------|
| ค่าใช้จ่าย - เกิดขึ้นจริงเปรียบเทียบกับงบประมาณ | <ul><li>ค่าใช้จ่ายรวมปีนี้</li><li>ค่าใช้จ่ายรวมของงบประมาณในปีนี้</li></ul>  |
| รายได้ - ที่เกิดขึ้นจริงเทียบกับงบประมาณ  | <ul><li>รายได้รวมปีนี้</li><li>รายได้รวมของงบประมาณในปีนี้</li><ul>     |
| ค่าใช้จ่าย                     | <ul><li>ค่าใช้จ่ายรวมปีนี้</li><li>เป้าหมายสำหรับค่าใช้จ่ายตามงบประมาณ</li><ul> |
| รายได้                     | <ul><li>รายได้รวมปีนี้</li><li>เป้าหมายสำหรับรายได้ตามงบประมาณ</li><ul>   |
| กำไรสุทธิ                  | <ul><li>กำไรสุทธิปีนี้</li><li>เป้าหมายสำหรับกำไรสุทธิตามงบประมาณ</li><ul>   |

## <a name="understanding-the-data-model-and-entities"></a>การทำความเข้าใจเกี่ยวกับแบบจำลองข้อมูลและเอนทิตี้

| เอนทิตี้                    | เนื้อหา                                                                         |
|---------------------------|----------------------------------------------------------------------------------|
| กิจกรรมของบัญชีแยกประเภททั่วไป | ยอดเงินของธุรกรรมสำหรับบัญชีแยกประเภททั่วไป                                       |
| กิจกรรมของงบประมาณ         | ยอดเงินธุรกรรมสำหรับทะเบียนงบประมาณ                                      |
| บัญชีหลัก             | บัญชีหลักในการกรองข้อมูลรายงานโดย                                               |
| ปฏิทินทางการเงิน          | ปฏิทินทางการเงินเพื่อกรองข้อมูลรายงานโดย                                            |
| บัญชีแยกประเภท                   | บัญชีแยกประเภทที่สามารถนำมาใช้เพื่อกรองข้อมูลรายงานในบัญชีแยกประเภทปัจจุบันได้              |
| รหัสงบประมาณ              | รหัสงบประมาณเพื่อกรองข้อมูลรายงานโดย                                                |
| นิติบุคคล            | นิติบุคคลที่สามารถนำมาใช้เพื่อกรองข้อมูลรายงานในนิติบุคคลปัจจุบันได้ |
