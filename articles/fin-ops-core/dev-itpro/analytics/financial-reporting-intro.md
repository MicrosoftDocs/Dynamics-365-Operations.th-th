# <a name="financial-reporting"></a><span data-ttu-id="bda09-103">การรายงานทางการเงิน</span><span class="sxs-lookup"><span data-stu-id="bda09-103">Financial reporting</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="bda09-104">การรายงานทางการเงินสำหรับแอพพลิเคชั่นยินยอมให้ผู้เชี่ยวชาญทางการเงินและผู้เชี่ยวชาญทางธุรกิจสามารถสร้าง รักษา ปรับใช้ และดูงบการเงินได้</span><span class="sxs-lookup"><span data-stu-id="bda09-104">Financial reporting for the application allows financial and business professionals to create, maintain, deploy, and view financial statements.</span></span> <span data-ttu-id="bda09-105">ซึ่งสามารถใช้งานได้เกินกว่าข้อจำกัดการรายงานดั้งเดิมเพื่อช่วยให้คุณสามารถออกแบบรายงานชนิดต่างๆ ได้อย่างมีประสิทธิภาพ</span><span class="sxs-lookup"><span data-stu-id="bda09-105">It moves beyond traditional reporting constraints to help you efficiently design various types of reports.</span></span>

<span data-ttu-id="bda09-106">รายงานทางการเงินประกอบด้วยการสนับสนุนมิติ</span><span class="sxs-lookup"><span data-stu-id="bda09-106">Financial reporting includes dimension support.</span></span> <span data-ttu-id="bda09-107">ดังนั้น เซ็กเมนต์บัญชีหรือมิติพร้อมใช้งานทันที</span><span class="sxs-lookup"><span data-stu-id="bda09-107">Therefore, account segments or dimensions are immediately available.</span></span> <span data-ttu-id="bda09-108">ไม่จำเป็นต้องมีเครื่องมือเพิ่มเติมหรือขั้นตอนการตั้งค่าคอนฟิก</span><span class="sxs-lookup"><span data-stu-id="bda09-108">No additional tools or configuration steps are required.</span></span>

## <a name="financial-reporting-setup"></a><span data-ttu-id="bda09-109">การตั้งค่าการรายงานทางการเงิน</span><span class="sxs-lookup"><span data-stu-id="bda09-109">Financial reporting setup</span></span>
<span data-ttu-id="bda09-110">หน้า **ตั้งค่าการรายงานทางการเงิน** มีรายการของมิติทางการเงินทั้งหมดในระบบ</span><span class="sxs-lookup"><span data-stu-id="bda09-110">The **Financial reporting setup** page has a list of all financial dimensions in the system.</span></span> <span data-ttu-id="bda09-111">**บัญชีแยกประเภททั่วไป** \> **การตั้งค่าบัญชีแยกประเภท** \> **การตั้งค่าการรายงานทางการเงิน**</span><span class="sxs-lookup"><span data-stu-id="bda09-111">**General ledger** \> **Ledger setup** \> **Financial reporting setup**.</span></span>

<span data-ttu-id="bda09-112">หน้า **การตั้งค่าการรายงานทางการเงิน** มีสองส่วน ซึ่งระบุข้อมูลที่คุณรายงานในการรายงานทางการเงิน:</span><span class="sxs-lookup"><span data-stu-id="bda09-112">The **Financial reporting setup** page has two sections that determine the data you report on in Financial reporting:</span></span>

- <span data-ttu-id="bda09-113">**แท็บมิติ** - เนื่องจากบริษัทต่างๆ ใช้โครงสร้างทางบัญชีและมิติที่ต่างกัน ไม่มีวิธีการกำหนดใบสั่งที่ผู้ใช้ต้องการดูมิติทางการเงินทั้งหมดในรายงาน</span><span class="sxs-lookup"><span data-stu-id="bda09-113">**Dimensions tab** - Because different companies use different dimensions and account structures, there is no way to determine the order in which users want to view all financial dimensions on reports.</span></span> <span data-ttu-id="bda09-114">หน้านี้ช่วยให้คุณสามารถตั้งค่าใบสั่งที่คุณต้องการให้มิติทางการเงินไปปรากฏขึ้น เมื่อคุณสร้าง และดูรายงานในการรายงานทางการเงิน</span><span class="sxs-lookup"><span data-stu-id="bda09-114">This page allows you set the order in which you want financial dimensions to appear when you build and view a report in Financial reporting.</span></span>
- <span data-ttu-id="bda09-115">**แท็บแอททริบิวต์** คือที่ซึ่งคุณสามารถเลือกได้ว่าคุณต้องการความสามารถในการใช้ **ผู้จัดจำหน่าย** และ **ลูกค้า** เป็นแอททริบิวต์สำหรับการออกแบบรายงานและการกรองข้อมูลหรือไม่</span><span class="sxs-lookup"><span data-stu-id="bda09-115">**Attributes tab** is where you can select whether you want the ability to use **Vendors** and **Customers** as attributes for filtering and report design.</span></span> <span data-ttu-id="bda09-116">การรายงานเกี่ยวกับผู้จัดจำหน่ายและลูกค้าจะมีประโยชน์ เฉพาะถ้าคุณไม่ได้ป้อนผู้จัดจำหน่ายหรือลูกค้าหลายรายการในใบสำคัญเดียว เมื่อลงรายการบัญชี</span><span class="sxs-lookup"><span data-stu-id="bda09-116">Reporting on Vendor and Customer will only be valuable if you do not enter multiple vendors or customers in a single voucher when posting transactions.</span></span> <span data-ttu-id="bda09-117">การเลือกผู้จัดจำหน่ายและ/หรือลูกค้า จะเพิ่มเวลาเพิ่มเติมในการรวม</span><span class="sxs-lookup"><span data-stu-id="bda09-117">Choosing Vendor and/or Customer will add additional time to the integration.</span></span>
