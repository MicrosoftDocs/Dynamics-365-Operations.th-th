# <a name="screen-destination"></a>ปลายทางหน้าจอ

[!include [banner](../includes/banner.md)]

คุณสามารถตั้งค่าคอนฟิกปลายทางของหน้าจอสำหรับแต่ละโฟลเดอร์หรือส่วนประกอบของไฟล์ของรูปแบบการรายงานทางอิเล็กทรอนิกส์ (ER) ที่มีการตั้งค่าคอนฟิกเพื่อสร้างเอกสารขาออก มีการเปิดเอกสารที่สร้างขึ้นสำหรับการแสดงตัวอย่างในแท็บเบราว์เซอร์แยกต่างหาก ตามการตั้งค่าปลายทาง

ถ้าคุณตั้งค่า **เปิดใช้งาน** เป็น **ใช่** ระบบจะสร้างการแสดงตัวอย่างผลผลิตขึ้น คุณสามารถดูไฟล์บางชนิด เช่น XML, TXT หรือ PDF ได้โดยตรงในหน้าต่างเบราว์เซอร์ สำหรับชนิดไฟล์อื่นๆ เช่น Microsoft Excel หรือ Word จะมีการใช้บริการ Microsoft Office Online
