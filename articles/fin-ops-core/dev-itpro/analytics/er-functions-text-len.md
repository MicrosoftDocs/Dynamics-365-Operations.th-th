# <a name="len-er-function"></a>ฟังก์ชัน LEN ER

[!include [banner](../includes/banner.md)]

ฟังก์ชัน `LEN` ส่งกลับจำนวนอักขระในสตริงที่ระบุเป็นค่า *จำนวนเต็ม*

## <a name="syntax"></a>ไวยากรณ์

```vb
LEN (text)
```

## <a name="arguments"></a>อาร์กิวเมนต์

`text`: *สตริง*

ค่า *สตริง* ที่ระบุข้อความ

## <a name="return-values"></a>ส่งคืนค่า

*จำนวนเต็ม*

ค่าตัวเลขที่เป็นผลลัพธ์

## <a name="example"></a>ตัวอย่าง

`LEN ("Sample")` ส่งกลับค่า **6**
