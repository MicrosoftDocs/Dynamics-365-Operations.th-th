# <a name="er-import-configurations-from-rcs"></a>(ER) นำเข้าการตั้งค่าคอนฟิกจาก RCS

[!include [banner](../../includes/banner.md)]

ขั้นตอนต่อไปนี้อธิบายวิธีที่ผู้ใช้ในบทบาทผู้ดูแลระบบหรือนักพัฒนาการรายงานทางอิเล็กทรอนิกส์ สามารถนำเข้าการตั้งค่าคอนฟิกการรายงานทางอิเล็กทรอนิกส์ (ER) รุ่นใหม่จาก Microsoft Regulatory Configuration Services (RCS) ในตัวอย่างนี้ คุณจะเลือกรุ่นของการตั้งค่าคอนฟิก ER ที่ได้รับการตั้งค่าคอนฟิกในอินสแตนซ์ RCS และนำเข้าไปยังอินสแตนซ์ปัจจุบันสำหรับบริษัทตัวอย่าง Litware, Inc ขั้นตอนเหล่านี้สามารถดำเนินการได้ในบริษัทใดๆ เนื่องจากมีการใช้การตั้งค่าคอนฟิก ER ร่วมกันระหว่างบริษัท เพื่อทำขั้นตอนเหล่านี้ให้เสร็จสมบูรณ์ อันดับแรกคุณต้องทำขั้นตอนต่างๆ ในหัวข้อให้เสร็จสมบูรณ์ [สร้างผู้ให้บริการการตั้งค่าคอนฟิก และทำเครื่องหมายเป็น ใช้งานอยู่](er-configuration-provider-mark-it-active-2016-11.md) เมื่อต้องการทำขั้นตอนต่อไปนี้ให้เสร็จสมบูรณ์ คุณยังต้องมีสิทธิเข้าถึงอินสแตนซ์ RCS ที่มีการตั้งค่าคอนฟิก ER อย่างน้อยหนึ่งรายการในสถานะ **เสร็จสมบูรณ์** หรือ **ที่ใช้ร่วมกัน**

1. ไปที่ **การจัดการองค์กร** > **พื้นที่ทำงาน** > **การรายงานทางอิเล็กทรอนิกส์** 
2. ตรวจสอบให้แน่ใจว่าผู้ให้บริการการตั้งค่าคอนฟิกสำหรับบริษัทตัวอย่าง Litware, Inc. พร้อมใช้งาน และถูกทำเครื่องหมายเป็น **ใช้งานอยู่** ถ้าคุณไม่เห็นผู้ให้บริการการตั้งค่าคอนฟิกนี้ ให้ดำเนินขั้นตอนต่างๆ ในหัวข้อ [สร้างผู้ให้บริการการตั้งค่าคอนฟิก และทำเครื่องหมายเป็นใช้งานอยู่](er-configuration-provider-mark-it-active-2016-11.md) ให้เสร็จสมบูรณ์ 
3. ถ้าคุณไม่มีสภาพแวดล้อม RCS ที่จัดสรรให้กับบริษัทของคุณ ให้เลือกที่ลิงค์ภายนอก **Regulatory services – การตั้งค่าคอนฟิก** และทำตามคำแนะนำเพื่อจัดเตรียมสภาพแวดล้อม RCS 
4. เลือกที่ **พารามิเตอร์การรายงานทางอิเล็กทรอนิกส์** 
5. เลือกแท็บ **RCS** 
6. ถ้ามีการจัดสรรสภาพแวดล้อม RCS ให้กับบริษัทของคุณแล้ว ให้ใช้รายการที่แสดงบน URLs ของหน้าเพื่อเข้าถึง 
7. ปิดหน้า 

## <a name="register-a-new-er-repository"></a>ลงทะเบียนที่จัดเก็บ ER ใหม่ 
1. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก 
2. เลือกผู้ให้บริการ Litware, Inc. 
3. เลือก ที่เก็บ 
4. เลือก เพิ่ม เพื่อเปิดกล่องโต้ตอบการวาง 
5. ในฟิลด์ชนิดที่เก็บของการตั้งค่าคอนฟิก ป้อน 'RCS' 
6. เลือก สร้างที่เก็บ 
7. ในฟิลด์ชื่อที่แสดงสภาพแวดล้อม RCS ให้ป้อน หรือเลือกค่า 
8. เลือกอินสแตนซ์ RCS ที่ต้องการ คุณสามารถมีได้หลายรายการ 
9. เลือก ตกลง 

## <a name="import-er-configurations-from-rcs-based-repository"></a>นำเข้าการตั้งค่าคอนฟิก ER จากที่เก็บที่ยึดตาม RCS
1. เลือก **แสดงตัวกรอง** 
2. ป้อนค่าตัวกรองของ "RCS" ในฟิลด์ **ชื่อ** โดยใช้ตัวดำเนินการกรอง **เริ่มต้นด้วย** 
3. เมื่อคุณเปิดที่เก็บที่เลือกบนหน้า **เชื่อมต่อกับ Regulatory Configuration Services** เลือกที่ลิงค์ **เลือกที่นี่เพื่อเชื่อมต่อกับ Regulatory Configuration Services** 
4. เลือก **เปิด** 
5. เลือก **ปิด** 
6. เลือกการตั้งค่าคอนฟิก ER รุ่นที่ต้องการ และเลือก **นำเข้า** เพื่อนำมาใช้ในอินสแตนซ์ปัจจุบัน
