# <a name="er-use-document-management-files-in-format-outputs-part-5---modify-and-run-format"></a>ER ใช้ไฟล์การจัดการเอกสารในผลลัพธ์รูปแบบ (ส่วนที่ 5 - แก้ไขและรันรูปแบบ)

[!include [banner](../../includes/banner.md)]

ขั้นตอนต่อไปนี้อธิบายวิธีกำหนดผู้ใช้ให้กับผู้ดูแลระบบหรือวิธีการที่บทบาทนักพัฒนาการรายงานทางอิเล็กทรอนิกส์สามารถตั้งค่าคอนฟิกรูปแบบการรายงานทางอิเล็กทรอนิกส์ (ER) ให้ใช้ไฟล์การจัดการเอกสาร (เอกสารแนบ) ในผลลัพธ์ ER  สามารถดำเนินการขั้นตอนเหล่านี้ได้ในบริษัท DEMF 

เพื่อดำเนินการขั้นตอนเหล่านี้ให้เสร็จสมบูรณ์ อันดับแรกคุณต้องดำเนินการขั้นตอนในกระบวนงาน "ER ใช้ไฟล์การจัดการเอกสารในผลลัพธ์รูปแบบ (ส่วนที่ 4: รันรูปแบบ)" ให้เสร็จสมบูรณ์

กระบวนงานนี้ใช้สำหรับคุณลักษณะทั้ที่ถูกเพิ่มลงใน Dynamics 365 for Operations รุ่น 1611


## <a name="modify-the-format-to-populate-attachments-into-generating-messages-in-binary-format"></a>แก้ไขรูปแบบเพื่อเติมข้อมูลเอกสารแนบลงในข้อความที่สร้างในรูปแบบฐานสอง
1. ไปที่ การจัดการองค์กร > การรายงานทางอิเล็กทรอนิกส์ > การตั้งค่าคอนฟิก
2. ในแผนภูมิ ให้ขยาย 'แบบจำลองใบแจ้งหนี้ของลูกค้า'
3. ในแผนภูมิ ขยาย 'Customer invoice model\Customer invoice model (custom)'
4. ในแผนภูมิ เลือกตัวอย่างข้อความใบแจ้งหนี้ 'Customer invoice model\Customer invoice model (custom)\Electronic'
5. คลิก ตัวออกแบบ
    * คุณจะเติมข้อมูลข้อความใบแจ้งหนี้ในผลลัพธ์ที่สร้างเป็นไฟล์ XML โดยใช้การเข้ารหัส UNICODE  
6. คลิกเพิ่มรากเพื่อเปิดกล่องโต้ตอบการวาง
7. ในแผนภูมิ เลือก 'Common\File'
8. ในฟิลด์ชื่อ พิมพ์ 'ข้อความ Xml'
    * ข้อความ Xml  
9. ในฟิลด์การเข้ารหัส พิมพ์ 'UTF-8'
    * UTF-8  
10. คลิก ตกลง
    * ตั้งค่าคอนฟิกผลลัพธ์ที่สร้างเป็นไฟล์ที่มีการอัด  
11. คลิกเพิ่มรากเพื่อเปิดกล่องโต้ตอบการวาง
12. ในแผนภูมิ เลือก 'Common\Folder'
13. ในฟิลด์ชื่อ พิมพ์ 'ผลลัพธ์รหัสไปรษณีย์'
    * ผลลัพธ์รหัสไปรษณีย์  
14. คลิก ตกลง
15. ในแผนภูมิ ให้เลือก 'ผลลัพธ์รหัสไปรษณีย์'
    * เพิ่มเอกสารแนบกับไฟล์ที่มีการอัดได้ที่สร้างเป็นไฟล์ที่มีชื่อและนามสกุลเดิม  
16. คลิกเพิ่ม เพื่อเปิดกล่องโต้ตอบการวาง
17. ในแผนภูมิ เลือก 'Common\File'
18. ในฟิลด์ชื่อ ให้พิมพ์ 'ไฟล์ที่แนบมา'
    * ไฟล์ที่แนบมา  
19. คลิก ตกลง
20. ในแผนภูมิ ให้เลือก 'Zip output\Attached file'
21. คลิกเพิ่ม เพื่อเปิดกล่องโต้ตอบการวาง
22. ในแผนภูมิ ให้เลือก 'Text\Base64'
23. คลิก ตกลง

## <a name="map-new-format-elements-to-data-model"></a>แม็ปองค์ประกอบรูปแบบใหม่ไปยังแบบจำลองข้อมูล
1. คลิกแท็บ การแม็ป
2. ในแผนภูมิ ให้ขยาย 'model'
3. ในแผนภูมิ ให้ขยาย 'model\Invoice attachments'
4. ในแผนภูมิ ให้เลือก 'Zip output\Attached file\Base64'
5. ในแผนภูมิ ให้เลือก 'model\Invoice attachments\File content'
6. คลิก ผูก
7. ในแผนภูมิ ให้เลือก 'Zip output\Attached file'
8. คลิก แก้ไขชื่อไฟล์
9. ในแผนภูมิ ให้ขยาย 'model'
10. ในแผนภูมิ ให้ขยาย 'model\Invoice attachments'
11. ในแผนภูมิ ให้เลือก 'model\Invoice attachments\File name'
12. คลิก เพิ่มแหล่งข้อมูล
13. คลิกบันทึก
14. ปิดหน้า
15. ในแผนภูมิ ให้เลือก 'model\Invoice attachments'
16. คลิก ผูก
17. คลิก บันทึก
18. ปิดหน้า
