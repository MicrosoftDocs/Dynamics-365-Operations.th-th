# <a name="list-of-er-functions-in-the-record-category"></a>รายการของฟังก์ชั่น ER ในประเภทเรกคอร์ด

[!include [banner](../includes/banner.md)]

ฟังก์ชันเรกคอร์ดการรายงานทางอิเล็กทรอนิกส์ (ER) สามารถใช้เพื่อดึงข้อมูลจากและดำเนินการกับแหล่งข้อมูลของชนิดข้อมูล *คอนเทนเนอร์ (เรกคอร์ด)* หัวข้อนี้แสดงสรุปของฟังก์ชันเหล่านี้

## <a name="list-of-supported-functions"></a>รายการฟังก์ชันที่ได้รับการสนับสนุน

| ฟังก์ชัน | คำอธิบาย |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | ฟังก์ชันนี้ส่งกลับค่า *คอนเทนเนอร์ (เรกคอร์ด)* ที่เป็น null ซึ่งมีโครงสร้างเดียวกันกับรายการเรกคอร์ดหรือเรกคอร์ดที่ระบุ |
| [EmptyRecord](er-functions-record-emptyrecord.md) | ฟังก์ชันนี้ส่งกลับค่า *คอนเทนเนอร์ (เรกคอร์ด)* ที่เป็น null ซึ่งมีโครงสร้างเดียวกันกับรายการเรกคอร์ดหรือเรกคอร์ดที่ระบุ |
