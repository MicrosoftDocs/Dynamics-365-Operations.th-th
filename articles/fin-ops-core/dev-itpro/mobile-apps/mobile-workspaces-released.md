# <a name="available-mobile-workspaces"></a>พื้นที่ทำงานแบบเคลื่อนที่ที่พร้อมใช้งาน

[!include [banner](../includes/banner.md)]
[!include [mobile app deprecation](../includes/mobile-app-deprecation-banner.md)]

บทความนี้แสดงรายการพื้นที่ทำงานแบบเคลื่อนที่ที่พร้อมใช้งานสำหรับการใช้กับแอปการเงินและการดำเนินงาน (Dynamics 365) บนมือถือ ตารางด้านล่างแสดงแผนคิดค่าเสื่อมราคาของพื้นที่เคลื่อนที่แต่ละพื้นที่เหล่านี้ในแง่ของว่ามีการวางแผนประสบการณ์การแทนที่แล้วหรือไม่  


| พื้นที่ทำงานแบบเคลื่อนที่     | คำอธิบาย   | แผนคิดค่าเสื่อมราคา   |
|----------------------|---------------|--------------|
|[ไดเรกทอรีบริษัท](company-directory-mobile-workspace.md)| ช่วยให้คุณสามารถดูและติดต่อพนักงานคนอื่นในองค์กรของคุณ| ถอน |    
|[ทีมงานของฉัน](manager-self-service-mobile-workspace.md)| คุณสามารถดูการรายงานและพนักงานโดยรวมของคุณได้โดยตรง และส่งการชมเชยสำหรับแต่ละบุคคลในกลุ่มของการรายงานของคุณได้| ถอน |     
|[การอนุมัติใบแจ้งหนี้](invoice-approval-mobile-workspace.md)| แสดงรายการของใบแจ้งหนี้ที่มีการกำหนดให้กับคุณผ่านขั้นตอนลำดับงานส่วนหัวของใบแจ้งหนี้ผู้จัดจำหน่าย| แทนที่   |
| [การจัดการค่าใช้จ่าย](/dynamics365/project-operations/prod-exp/expense-management-mobile-workspace) | คุณสามารถเก็บบันทึกและอัพโหลดใบเสร็จรับเงิน เพื่อให้คุณสามารถแนบกับรายงานค่าใช้จ่ายในภายหลังได้ พื้นที่ทำงานแบบเคลื่อนที่ยังช่วยให้คุณสามารถสร้างรายการค่าใช้จ่ายได้อย่างรวดเร็วโดยใช้ใบเสร็จรับเงินที่แนบ | แทนที่ |
| [การอนุมัติใบสั่งซื้อ](../../../supply-chain/procurement/purchase-order-mobile-workspace.md) | ดูและตอบสนองกับใบสั่งซื้อพร้อมกับการดำเนินการ เช่น อนุมัติ หรือปฏิเสธ | แทนที่ |
| [รายการข้อมูลเวลาของโครงการ](/dynamics365/project-operations/prod-pma/project-time-entry-mobile-workspace) | คุณสามารถป้อนและประหยัดเวลาในแง่ของโครงการโดยการใช้อุปกรณ์เคลื่อนที่ของคุณ | <p>มีการเปลี่ยนทดแทนได้</p><p>[Microsoft Dynamics 365 Project Timesheet](/dynamics365/project-operations/prod-pma/project-timesheet)</p> |
| [การควบคุมต้นทุน](../../../finance/cost-accounting/cost-controlling-mobile-workspace.md) | ผู้จัดการศูนย์ต้นทุนสามารถดูประสิทธิภาพของศูนย์ต้นทุนได้  |  ถอน |
| [ปริมาณสินค้าคงคลังคงเหลือ](../../../supply-chain/inventory/inventory-on-hand-mobile-workspace.md) | รับข้อมูลเชิงลึกในสินค้าคงคลังที่จองไว้และพร้อมใช้งานได้ |  แทนที่ |
| [ใบสั่งขาย](../../../supply-chain/sales-marketing/sales-orders-mobile-workspace.md) | คุณสามารถคอยติดตามสถานการณ์ปัจจุบันเกี่ยวกับใบสั่งขายของคุณได้  |  ถอน |
| [การทำงานร่วมกันกับผู้จัดจำหน่าย](../../../supply-chain/procurement/vendor-collaboration-mobile-workspace.md) | ผู้จัดจำหน่ายสามารถคอยติดตามสถานการณ์ปัจจุบันในใบสั่งซื้อที่ถูกส่งไปขอการอนุมัติ นอกจากนี้ยังสามารถดูข้อมูลเกี่ยวกับใบสั่งซื้อและผู้ติดต่อใหม่และที่อัพเดตแล้ว | ถอน  |
| [การจัดการสินทรัพย์](../../../supply-chain/asset-management/asset-management-mobile-workspace.md) | พื้นที่ทำงานนี้จะช่วยให้ผู้ใช้สามารถดูและสร้างคำขอการบำรุงรักษาและใบสั่งงานได้ นอกจากนี้ผู้ใช้ยังสามารถดูงานในใบสั่งงานที่กำหนดไว้ในปฏิทินหรือมุมมองรายการได้อีกด้วย นอกจากนี้คุณยังสามารถดูและค้นหาสินทรัพย์และสถานที่ทำงานได้ด้วย | แทนที่ |
