# <a name="removed-or-deprecated-platform-features"></a>คุณลักษณะแพลตฟอร์มที่ถูกลบออกหรือเลิกสนับสนุน

[!include [banner](../includes/banner.md)]

หัวข้อนี้อธิบายคุณลักษณะที่ถูกลบออก หรือที่วางแผนไว้ว่าจะลบออกจากการอัปเดตแพลตฟอร์มของแอป Finance and Operations

- คุณลักษณะที่ *ถูกลบ* จะไม่พร้อมใช้งานในผลิตภัณฑ์อีกต่อไป
- คุณลักษณะที่ *ถูกยกเลิกการใช้* ไม่ได้อยู่ในการพัฒนาที่ใช้งานอยู่ และอาจถูกลบออกในการปรับปรุงในอนาคต

รายการนี้มีไว้เพื่อช่วยคุณพิจารณาการลบออกและการยกเลิกการใช้งานเหล่านี้สำหรับการวางแผนของคุณเอง 

ข้อมูลโดยละเอียดเกี่ยวกับออบเจ็กต์ต่างๆ ในแอป Finance and Operations พบได้ใน [รายงานอ้างอิงทางเทคนิค](/dynamics/s-e/global/axtechrefrep_61) คุณสามารถเปรียบเทียบเวอร์ชันต่างๆ ของรายงานเหล่านี้ เพื่อเรียนรู้เกี่ยวกับออบเจ็กต์ที่เปลี่ยนแปลงหรือถูกลบออกในแต่ละรุ่นของแอป Finance and Operations

## <a name="feature-removal-effective-october-2021"></a>การลบคุณลักษณะมีผลบังคับเดือนตุลาคม 2021

### <a name="microsoft-azure-sql-reports-in-lifecycle-services-lcs"></a>รายงาน SQL ของ Microsoft Azure ใน Lifecycle Services (LCS)

| &nbsp;  | &nbsp; |
|------------|--------------------|
| **เหตุผลสำหรับการยกเลิกการใช้/การลบ** | กิจกรรมและการตรวจสอบทั้งหมดจะปฏิบัติภายในโดยแพลตฟอร์ม โดยผ่านระบบอัตโนมัติ ซึ่งไม่ต้องการการขัดจังหวะด้วยตนเองใดๆ|
| **แทนที่ด้วยลักษณะการทำงานอื่นหรือไม่**   | ใช่ ขณะนี้มีระบบอัตโนมัติ ซึ่งแสดงความสามารถเหล่านี้ที่ล้าสมัย |
| **พื้นที่ของผลิตภัณฑ์ที่ได้รับผล**         | รายงาน SQL: DTU ปัจจุบัน, รายละเอียด DTU ปัจจุบัน, รับรายละเอียดล็อค, รายการของคู่มือแผนปัจจุบัน, รับรายการของรหัสการสอบถาม, รับแผนการสอบถาม SQL สำหรับรหัสแผนที่ระบุ, รับแผนการสอบถามและสถานะการปฏิบัติการ, รับการตั้งค่าคอนฟิกการควบคุมปริมาณ, รับสถิติการรอ, แสดงรายการการสอบถามที่มีราคาแพงที่สุด |
| **ตัวเลือกการปรับใช้**              | การปรับใช้ระบบคลาวด์: ส่งผลกระทบต่อสภาพแวดล้อมการทำงานจริงที่จัดการโดย Microsoft และสภาพแวดล้อม Sandbox ระดับ 2 ถึงระดับ 5 |
