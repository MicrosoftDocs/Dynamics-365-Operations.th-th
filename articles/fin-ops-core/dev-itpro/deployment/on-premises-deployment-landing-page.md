# <a name="on-premises-deployment-home-page"></a>โฮมเพจตัวเลือกการปรับใช้ในสถานที่

[!include [banner](../includes/banner.md)]

คุณสามารถปรับใช้ Dynamics 365 Finance + Operations (on-premises) เมื่อคุณเลือกชนิดของการปรับใช้ในองค์กร ความต้องการระบบ การกำหนดฮาร์ดแวร์และฟังก์ชันการทำงานจะแตกต่างจากการปรับใช้บริการระบบคลาวด์ หัวข้อนี้แสดงลิงค์ไปยังเนื้อหาที่มีข้อมูลเฉพาะเกี่ยวกับการปรับใช้ในองค์กร
