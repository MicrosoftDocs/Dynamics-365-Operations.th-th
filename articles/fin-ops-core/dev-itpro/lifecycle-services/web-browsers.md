# <a name="supported-web-browsers-for-lifecycle-services"></a>เว็บเบราว์เซอร์ที่ไม่รองรับสำหรับ Lifecycle Services 

[!include[banner](../includes/banner.md)]

ผู้ใช้สามารถเข้าถึง Microsoft Dynamics Lifecyle Services (LCS) ได้โดยใช้เบราว์เซอร์ที่เป็นที่นิยมรุ่นล่าสุดเหล่านี้: 

- Microsoft Edge (แนะนำ: [Chromium-based Edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

เมื่อต้องการค้นหารุ่นล่าสุดของแต่ละเว็บเบราเซอร์ ให้ไปที่เว็บไซต์ของผู้ผลิตซอฟต์แวร์ เพื่อประสิทธิภาพการและประสบการณ์ที่ดีที่สุด เราขอแนะนำให้คุณใช้เบราว์เซอร์สมัยใหม่รุ่นล่าสุด โดยเฉพาะอย่างยิ่ง Microsoft Edge
