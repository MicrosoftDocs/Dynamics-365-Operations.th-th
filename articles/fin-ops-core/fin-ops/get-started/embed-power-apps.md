<span data-ttu-id="b3d6b-196">ตามค่าเริ่มต้น ผู้ใช้สามารถฝังแอปในหน้าใดก็ได้ภายใต้ปุ่มเมนู Power Apps หรือฝังโดยตรงบนหน้าในแท็บใหม่ FastTab แผ่นหรือเป็นส่วนใหม่ในพื้นที่ทำงาน</span><span class="sxs-lookup"><span data-stu-id="b3d6b-196">By default, users can embed apps on any page, either under the Power Apps menu button or directly on the page as a tab, FastTab, blade or as a new section in a workspace.</span></span> <span data-ttu-id="b3d6b-197">อย่างไรก็ตามหากจำเป็น นักพัฒนาสามารถกำหนดค่าคุณลักษณะนี้เพื่ออนุญาตให้ฝังแอปเฉพาะบางหน้าได้โดยใช้วิธีการดังต่อไปนี้:</span><span class="sxs-lookup"><span data-stu-id="b3d6b-197">However, if required, developers can also configure this feature to only allow embedding of apps on certain pages by implementing the following methods:</span></span>

- <span data-ttu-id="b3d6b-198">**isPowerAppPersonalizationEnabled** – หากวิธีนี้คืนค่าเท็จสำหรับหน้าเฉพาะ ปุ่มเมนู Power Apps จะไม่ปรากฏขึ้น และผู้ใช้จะไม่สามารถฝังแอปที่ใดก็ได้ในหน้านี้รวมถึงที่เป็นแท็บ</span><span class="sxs-lookup"><span data-stu-id="b3d6b-198">**isPowerAppPersonalizationEnabled** – If this method returns false for a specific page, then the Power Apps menu button will not be shown, and users will not be able to embed apps anywhere on this page, including as a tab.</span></span>
- <span data-ttu-id="b3d6b-199">**isPowerAppTabPersonalizationEnabled** – หากวิธีนี้คืนค่าเท็จสำหรับหน้าเฉพาะ ผู้ใช้จะไม่สามารถฝังแอปโดยตรงบนหน้าที่เป็นแท็บ FastTab หรือส่วนรูปแบบพาโนรามา</span><span class="sxs-lookup"><span data-stu-id="b3d6b-199">**isPowerAppTabPersonalizationEnabled** – If this method returns false for a specific page, then users will not be able to embed apps directly on the page as a tab, FastTab, or panorama section.</span></span> <span data-ttu-id="b3d6b-200">ผู้ใช้จะยังคงสามารถฝังแอปผ่านปุ่มเมนู Power Apps หากการฝังได้รับอนุญาตบนหน้านั้น</span><span class="sxs-lookup"><span data-stu-id="b3d6b-200">Users will still be able to embed apps through the Power Apps menu button if embedding is allowed on the page.</span></span>

<span data-ttu-id="b3d6b-201">ตัวอย่างต่อไปนี้แสดงคลาสใหม่ที่มีสองวิธีที่จำเป็นในการกำหนดค่าที่ที่สามารถฝังแอปได้</span><span class="sxs-lookup"><span data-stu-id="b3d6b-201">The following example shows a new class with the two methods needed to configure where apps can be embedded.</span></span>

```powerapps
[ExtensionOf(classStr(FormRunConfigurationPowerAppsConfiguration))]

public final class ClassTest_Extension
{
    public static boolean isPowerAppPersonalizationEnabled(str pageName)
    {
        var result = next isPowerAppPersonalizationEnabled(pageName);
        return result;
    }
    
    public static boolean isPowerAppTabPersonalizationEnabled(str pageName)
    {
        var result = next isPowerAppTabPersonalizationEnabled(pageName);
        return result;
    }
}
```
