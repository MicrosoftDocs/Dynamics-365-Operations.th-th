# <a name="functional-application-consultant-learning-catalog"></a>แค็ตตาล็อกการเรียนรู้ของที่ปรึกษาด้านการทำงานแอพลิเคชั่น

แค็ตตาล็อกต่อไปนี้มีการจัดระเบียบจากความรู้หลักให้กับโดเมนเฉพาะและจากการทำงานพื้นฐานถึงขั้นสูงที่สุด ถ้ามีเนื้อหาอยู่ในรูปแบบเราจะแจ้งให้คุณทราบเพื่อให้คุณสามารถเลือกรูปแบบการฝึกอบรมที่ตรงกับความต้องการของคุณได้ดีที่สุด สำหรับข้อมูลเพิ่มเติม ให้ดูที่รูปแบบเนื้อหาของการฝึกอบรม 

สำหรับที่ปรึกษาด้านการทำงาน งานวิจัยของเราที่วิเคราะห์การปฏิบัติงานแสดงให้เห็นว่าทุกคนต้องเข้าใจถึงการกำหนดเนื้อหาหลักรวมถึงพื้นที่โดเมน
