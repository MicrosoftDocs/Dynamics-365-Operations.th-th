# <a name="new-or-updated-task-guides-december-2017"></a>คู่มืองานใหม่หรือที่ปรับปรุงแล้ว (ธันวาคม 2017)

[!include [banner](../includes/banner.md)]

หัวข้อนี้แสดงรายการคู่มืองานใหม่หรือที่ปรับปรุงที่ถูกนำออกใช้ไปยัง Microsoft (ธันวาคม 2016 - ภาษาทั้งหมด) APQC Unified Library สำหรับ Microsoft Dynamics 365 for Operations

## <a name="new-task-guides"></a>คู่มืองานใหม่

- ER ทำรายงานในรูปแบบ MS Office ด้วยรูปภาพที่ฝังอยู่ (ส่วนที่ 1: ตั้งค่าพารามิเตอร์)
- ย้ายงานคัมบังที่จัดกำหนดการ (กุมภาพันธ์ 2016)
- จัดการสถานะรอบการขายของผลิตภัณฑ์
- สร้างสถานะรอบการขายของผลิตภัณฑ์เพื่อแยกผลิตภัณฑ์ออกจากการวางแผนหลัก (ธันวาคม 2017)
- ค้นหาผลิตภัณฑ์ย่อยที่ล้าสมัย และกำหนดสถานะรอบการขายของผลิตภัณฑ์ (ธันวาคม 2017)
- มอบหมายสถานะรอบการขายของผลิตภัณฑ์ให้กับผลิตภัณฑ์หลักที่นำออกใช้ (ธันวาคม 2017)
- มอบหมายสถานะรอบการขายของผลิตภัณฑ์ให้กับผลิตภัณฑ์ที่นำออกใช้ (ธันวาคม 2017)
- สร้างสถานะรอบการขายของผลิตภัณฑ์ใหม่ค่าเริ่มต้น (ธันวาคม 2017)
- สร้างสถานะรอบการขายของผลิตภัณฑ์ใหม่ (ธันวาคม 2017)
- ER ออกแบบการตั้งค่าคอนฟิกเพื่อนำเข้าข้อมูลจากไฟล์ภายนอกในรูปแบบ CSV (7.3 อัพเดต)
- ER ออกแบบนิพจน์เพื่อเรียกใช้วิธีการของคลาสแอพลิเคชันที่มีอาร์กิวเมนต์ค่าที่ถูกกำหนดที่รันไทม์ (7.3 อัพเดต)
- การปรับปรุง ER ในการแยกวิเคราะห์เอกสารขาเข้าเพื่ออัพเดตข้อมูลแอพลิเคชัน (7.3 อัพเดต)
- ER ช่วยเพิ่มประสิทธิภาพในการคำนวณรวมด้วยการดำเนินการเหล่านั้นในระดับฐานข้อมูล (7.3 อัพเดต)
- ER ช่วยเพิ่มประสิทธิภาพในการคำนวณรวมด้วยการดำเนินการเหล่านั้นในระดับฐานข้อมูล (ส่วนที่ 1: จัดเตรียมการตั้งค่าคอนฟิก)
- ER ช่วยเพิ่มประสิทธิภาพในการคำนวณรวมด้วยการดำเนินการเหล่านั้นในระดับฐานข้อมูล (ส่วนที่ 2: ออกแบบการแม็ปแบบจำลอง)
- ตั้งค่าพารามิเตอร์เพื่อคำนวณการคิดค่าเสื่อมราคา (ธันวาคม 2017)
- ตั้งค่าค่าเสื่อมราคาของกลุ่มสินทรัพย์สำหรับสมุดบัญชีค่าเสื่อมราคา (ธันวาคม 2017)
- ตั้งค่าปฏิทินทางการเงินตามวัน (ธันวาคม 2017)
- กำหนดชนิดของกะและอุตสาหกรรมสำหรับสมุดบัญชี และแนบกับสินทรัพย์ถาวร (ธันวาคม 2017)
- กำหนดอัตราค่าเสื่อมราคาของกะ (ธันวาคม 2017)
- สร้างกลุ่มแบบจำลองสินค้าคงคลังสำหรับสินค้าประเภททุน (ธันวาคม 2017)
- แนบปฏิทินตามวันกับสมุดบัญชีสินทรัพย์ถาวร (ธันวาคม 2017)
- IN-การโอนย้ายในสินค้าประเภททุนไปยังสินทรัพย์ถาวร (ธันวาคม 2017)
