<span data-ttu-id="711d5-105">การตั้งค่าส่วนบุคคลมีบทบาทสำคัญในการอนุญาตให้ผู้ใช้และองค์กรสามารถปรับปรุงประสบการณ์การใช้งานของผู้ใช้เพื่อตอบสนองความต้องการของพวกเขา</span><span class="sxs-lookup"><span data-stu-id="711d5-105">Personalization plays an important role in allowing users and organizations to optimize the user experience to meet their needs.</span></span> <span data-ttu-id="711d5-106">สำหรับรายละเอียดเพิ่มเติมเกี่ยวกับการตั้งค่าส่วนบุคคล ดู [ปรับประสบการณ์ผู้ใช้เป็นแบบส่วนตัว](personalize-user-experience.md)</span><span class="sxs-lookup"><span data-stu-id="711d5-106">For more details on personalization, see [Personalize the user experience](personalize-user-experience.md).</span></span>

-    <span data-ttu-id="711d5-113">สามารถเผยแพร่มุมมองไปยังบทบาทความปลอดภัยได้ ซึ่งหมายความว่าผู้ใช้ใดๆ ที่มีบทบาทนั้นจะสามารถเข้าถึงและใช้มุมมองนั้นได้โดยไม่คำนึงถึงความสามารถของผู้ใช้ในการตั้งค่าส่วนบุคคล</span><span class="sxs-lookup"><span data-stu-id="711d5-113">Views can be published to security roles, meaning any user with that role will be able to access and use that view, regardless of the user’s ability to personalize.</span></span> <span data-ttu-id="711d5-114">ความสามารถในการเผยแพร่นี้ทำให้องค์กรสามารถกำหนดมุมมองมาตรฐานขององค์กรที่ได้รับการปรับให้เหมาะสมกับธุรกิจของพวกเขา</span><span class="sxs-lookup"><span data-stu-id="711d5-114">This publish capability allows organizations to define corporate standard views that are optimized for their business.</span></span> <span data-ttu-id="711d5-115">ดูที่ส่วน [การจัดการการตั้งค่าส่วนบุคคลที่ระดับองค์กรกับมุมมอง](saved-views.md#managing-personalizations-at-an-organizational-level-with-views) สำหรับรายละเอียดเพิ่มเติม</span><span class="sxs-lookup"><span data-stu-id="711d5-115">See the [Managing personalizations at an organizational level with views](saved-views.md#managing-personalizations-at-an-organizational-level-with-views) section for more information.</span></span>

-    <span data-ttu-id="711d5-116">ซึ่งแตกต่างจากการตั้งค่าส่วนบุคคลแบบดั้งเดิม มุมมองจะไม่ถูกบันทึกโดยอัตโนมัติ เมื่อผู้ใช้ดำเนินการการตั้งค่าส่วนบุคคลที่ชัดเจนหรือกรองรายการ</span><span class="sxs-lookup"><span data-stu-id="711d5-116">Unlike traditional personalization, views are not automatically saved when a user performs explicit personalizations or filters a list.</span></span> <span data-ttu-id="711d5-117">ต้องมีการบันทึกที่ชัดเจนเพื่อให้ความยืดหยุ่นในการสร้างมุมมองก่อน หรือหลังจากที่มีการทำการเปลี่ยนแปลงที่เกี่ยวข้องกับมุมมองนั้น และเพื่อตรวจสอบให้แน่ใจว่ามีข้อกำหนดมุมมองไม่ถูกสลับโดยไม่ได้ตั้งใจโดยตัวกรอง หรือการตั้งค่าส่วนบุคคลที่ไม่ได้มีไว้สำหรับการใช้งานในระยะยาว</span><span class="sxs-lookup"><span data-stu-id="711d5-117">Explicit saves are required to provide flexibility in creating a view before or after the changes associated with that view have been made and to ensure that view definitions are not unintentionally altered by filters or personalizations that are not intended for long-term use.</span></span>  

## <a name="switching-between-views"></a><span data-ttu-id="711d5-118">การสลับระหว่างมุมมอง</span><span class="sxs-lookup"><span data-stu-id="711d5-118">Switching between views</span></span>
<span data-ttu-id="711d5-119">หลังจากที่เปิดใช้งานมุมมองสำหรับสภาพแวดล้อมแล้ว หน้าใดก็ตามที่สนับสนุนมุมมองจะรวมตัวควบคุมมุมมองที่ยุบแล้วไว้ที่ด้านบนของแบบฟอร์มซึ่งแสดงชื่อของมุมมองปัจจุบัน</span><span class="sxs-lookup"><span data-stu-id="711d5-119">After views have been enabled for an environment, any page that supports views will include a collapsed view selector control at the top of the form that shows the name of the current view.</span></span>  

<span data-ttu-id="711d5-120">มีการเปลี่ยนแปลงขนาดสองรายการไปยังตัวเลือกมุมมอง:</span><span class="sxs-lookup"><span data-stu-id="711d5-120">There are two size variations to the view selector:</span></span> 

-   <span data-ttu-id="711d5-121">**ตัวเลือกมุมมองที่มีขนาดใหญ่**: หน้าที่มีคุณลักษณะรายการอย่างชัดเจนจะมีตัวเลือกมุมมองที่ใหญ่กว่าด้วยเหตุผลสองสามประการ</span><span class="sxs-lookup"><span data-stu-id="711d5-121">**Large view selectors**: Pages that prominently feature a list will have a larger view selector for a few reasons.</span></span> <span data-ttu-id="711d5-122">ที่สำคัญที่สุดคือ ตัวเลือกมุมมองที่ใหญ่ขึ้นจะบ่งชี้หน้าที่ซึ่งมุมมองนั้นสามารถมีตัวกรองที่ผู้ใช้กำหนด</span><span class="sxs-lookup"><span data-stu-id="711d5-122">Most importantly, the larger view selector indicates the pages where the view can include user-defined filters.</span></span> <span data-ttu-id="711d5-123">เนื่องจากตัวกรองถูกรวมไว้ในมุมมอง จะมีการรับประกันขนาดของตัวเลือกที่มีขนาดใหญ่กว่า เนื่องจากชื่อของมุมมองมักจะเป็นคำอธิบายที่ดีที่สุดของข้อมูลที่แสดงบนหน้าจอ และความคาดหวังคือผู้ใช้จะสลับระหว่างมุมมองต่างๆ บ่อยครั้งกว่าบนชนิดหน้าเหล่านี้</span><span class="sxs-lookup"><span data-stu-id="711d5-123">Because filters are included in the views, the larger selector size is also warranted as the view names will often be the best description of the data shown on the screen and the expectation is that users will switch between views more often on these page types.</span></span>  
 
-   <span data-ttu-id="711d5-124">**ตัวเลือกมุมมองขนาดเล็ก**: ฟอร์มแบบเต็มหน้าอื่นๆ ทั้งหมด (ที่มีข้อยกเว้นของพื้นที่ทำงานและแดชบอร์ด) มีตัวเลือกมุมมองที่เล็กลงซึ่งจะปรากฏขึ้นถัดจากคำอธิบายหน้า</span><span class="sxs-lookup"><span data-stu-id="711d5-124">**Small view selectors**: All other full-page forms (with the exception of workspaces and the dashboard) have a smaller view selector that appears next to the page caption.</span></span> <span data-ttu-id="711d5-125">มุมมองบนหน้าเหล่านี้จะรวมการกำหนดเป็นแบบส่วนบุคคลเท่านั้น (และไม่ใช่ตัวกรองที่กำหนดโดยผู้ใช้)</span><span class="sxs-lookup"><span data-stu-id="711d5-125">Views on these pages only include personalizations (and not user-defined filters).</span></span> <span data-ttu-id="711d5-126">ในหน้าเหล่านี้ คำอธิบายแบบฟอร์มหรือชื่อเรกคอร์ดมักจะเป็นข้อมูลที่สำคัญที่สุดที่ด้านบนของแบบฟอร์ม</span><span class="sxs-lookup"><span data-stu-id="711d5-126">On these pages, the form caption or record title is often the most important information at the top of the form.</span></span> <span data-ttu-id="711d5-127">นอกจากนี้ ขนาดที่เล็กลงจะสะท้อนถึงความถี่ที่คาดไว้ต่ำกว่าของการสลับมุมมองไปยังหน้าเหล่านี้</span><span class="sxs-lookup"><span data-stu-id="711d5-127">The smaller size also reflects a lower expected frequency of view switching on these pages.</span></span> 
 
<span data-ttu-id="711d5-128">ถ้าคุณคลิกที่ชื่อมุมมอง ตัวเลือกมุมมองจะเปิดขึ้นและแสดงรายการของมุมมองที่มีอยู่สำหรับหน้านี้</span><span class="sxs-lookup"><span data-stu-id="711d5-128">If you click on the view name, the view selector opens and shows the list of available views for this page</span></span>

-    <span data-ttu-id="711d5-130">**มุมมองส่วนบุคคล**: มุมมองที่ไม่มีแม่กุญแจแสดงมุมมองส่วนบุคคลของคุณ</span><span class="sxs-lookup"><span data-stu-id="711d5-130">**Personal views**: The views without padlocks represent your personal views.</span></span> <span data-ttu-id="711d5-131">รายการเหล่านี้คือมุมมองที่คุณได้สร้างขึ้น หรือผู้ดูแลระบบของคุณได้กำหนดให้กับคุณ</span><span class="sxs-lookup"><span data-stu-id="711d5-131">These are views that either you have created or that an administrator has given to you.</span></span>  
-    <span data-ttu-id="711d5-132">**มุมมองที่ล็อค**: มุมมองบางมุมมอง (เช่น มุมมองแบบคลาสสิกและมุมมองใดๆ ที่เผยแพร่ไปยังบทบาทของคุณ) มีแม่กุญแจอยู่ถัดจากรายการเหล่านั้นในตัวเลือกมุมมอง ซึ่งบ่งชี้ว่าคุณไม่สามารถแก้ไขมุมมองเหล่านั้นได้</span><span class="sxs-lookup"><span data-stu-id="711d5-132">**Locked views**: Some views (like the Classic view and any views published to your role) have a padlock next to them in the view selector, indicating that you cannot edit those views.</span></span> <span data-ttu-id="711d5-133">อย่างไรก็ตาม การทำให้เป็นแบบส่วนบุคคลโดยนัยที่สะท้อนการใช้งานหน้าจะถูกบันทึกโดยอัตโนมัติ เช่น การเปลี่ยนความกว้างของคอลัมน์กริด หรือการขยายหรือการยุบ FastTab</span><span class="sxs-lookup"><span data-stu-id="711d5-133">However, implicit personalizations that reflect page usage are automatically saved, such as changing the width of a grid column or expanding or collapsing a FastTab.</span></span> <span data-ttu-id="711d5-134">อย่างไรก็ตาม คุณสามารถทำมุมมองส่วนบุคคลซึ่งขึ้นกับมุมมองที่ถูกล็อคโดยใช้การดำเนินการ **บันทึกสำเนา** ถ้าคุณมีสิทธิ์ในการตั้งค่าส่วนบุคคล</span><span class="sxs-lookup"><span data-stu-id="711d5-134">You can, however, make a personal view based on a locked view using the **Save a copy** action, if you have personalization privileges.</span></span>
