# <a name="help-system"></a>ระบบวิธีใช้

[!include [banner](../includes/banner.md)]

หัวข้อนี้แสดงภาพรวมขององค์ประกอบของระบบให้ความช่วยเหลือ ระบบวิธีใช้จะแบ่งตามผลิตภัณฑ์ต่อไปนี้ทั้งหมด:

- Dynamics 365 Finance 
- Dynamics 365 Retail
- Dynamics 365 Supply Chain Management
- Dynamics 365 Talent

คุณสามารถเข้าถึงวิธีใช้จากบานหน้าต่างวิธีใช้ในผลิตภัณฑ์ใดก็ตามที่คุณใช้อยู่ 

![บานหน้าต่างวิธีใช้](./media/help-pane-ops-task-guides.png)

## <a name="help-on-docsmicrosoftcom"></a>วิธีใช้เกี่ยวกับ docs.microsoft.com

ไซต์ docs.microsoft.com ([docs.microsoft.com/dynamics365/](https://docs.microsoft.com/dynamics365/) เป็นแหล่งข้อมูลหลักของเอกสารประกอบผลิตภัณฑ์สำหรับแอพพลิเคชั่นที่กล่าวมาแล้วข้างต้น ไซต์มีลักษณะการทำงานดังต่อไปนี้:

- **เข้าถึงเนื้อหาล่าสุด** – ไซต์ทำให้เราสร้าง จัดส่ง และอัพเดตเอกสารผลิตภัณฑ์รวดเร็วและมีความยืดหยุ่นมากขึ้น ดังนั้น จึงจะช่วยให้แน่ใจว่าคุณมีสิทธิ์เข้าถึงข้อมูลทางเทคนิคล่าสุด
- **เนื้อหาที่เขียนโดยผู้เชี่ยวชาญ** – ไซต์แสดงชุดของเอกสารผลิตภัณฑ์ที่สมบูรณ์กว่า ซึ่งสามารถเพิ่มเติมโดยสมาชิกของชุมชนทั้งภายใน และภาย นอก Microsoft

คุณยังสามารถค้นหาเนื้อหาของเราโดยใช้กลไกจัดการค้นหาใด ๆ เราขอแนะนำว่า เพื่อให้ได้ผลลัพธ์ที่ดีที่สุดคุณควรใช้การค้นหาไซต์ เช่น site:docs.microsoft.com dynamics 365 "คำค้นหา"

### <a name="leave-us-feedback"></a>ฝากคำติชมให้พวกเรา

ถ้าคุณมีความคิดเห็นหรือคำถามเกี่ยวกับหัวข้อ ฝากความคิดเห็นให้พวกเราที่ด้านล่างของหน้า

1. คลิก **ความคิดเห็น** เพื่อเข้าถึงข้อคิดเห็นที่ด้านล่างของหน้า และจากนั้น คลิก **ความคิดเห็นต่อผลิตภัณฑ์** หรือ **ลงชื่อเข้าใช้เพื่อให้ความคิดเห็นเกี่ยวกับเอกสาร**

2. เริ่มพิมพ์ข้อคิดเห็นของคุณ แล้วคลิก **ส่งความคิดเห็น**

    ![ลงรายการข้อคิดเห็น](./media/feedback.png)

## <a name="task-guides"></a>คู่มืองาน

คู่มืองานคือคู่มือในการควบคุม ประสบการณ์การโต้ตอบ ที่นำคุณผ่านขั้นตอนของงานหรือกระบวนการทางธุรกิจ คุณสามารถเปิด (เล่น) คู่มืองานจากบานหน้าต่าง วิธีใช้ ได้ เมื่อคุณคลิกคู่มืองานเป็นครั้งแรก บานหน้าต่าง วิธีใช้ จะแสดงคำแนะนำทีละขั้นตอนสำหรับงานนั้น คู่มือการใช้งานที่เป็นภาษาท้องถิ่นพร้อมใช้งานแล้วในขณะนี้

Microsoft จัดส่งไลบรารีคู่มืองานสำหรับการนำออกใช้ในเดือนธันวาคม 2017 สำหรับ Dynamics 365 for Finance and Operations ส่วน [ระบบวิธีใช้](help-overview.md#accessing-task-guides-from-the-help-pane) จะอธิบายวิธีการค้นหาคู่มืองานที่ถูกต้องสำหรับผลิตภัณฑ์ของคุณ

![มุมมองการอ่านคู่มืองาน](./media/task-guide-ops.png)

เพื่อเริ่มการแนะนำ ประสบการณ์โต้ตอบ คลิก **เริ่มต้นคู่มืองาน** ที่ด้านล่างของบานหน้าต่างวิธีใช้ ตัวชี้สีดำเปิดขึ้น และบ่งชี้ถึงการดำเนินการที่คุณจำเป็นต้องทำ ทำตามคำแนะนำที่ปรากฏใน UI และป้อนข้อมูลตามที่กำหนด

![คำแนะนำตามขั้นตอนของคู่มืองาน](./media/task-guide-step-1-ops.png)

> [!IMPORTANT]
> ข้อมูลที่คุณป้อนเมื่อคุณเล่นคู่มืองานเป็นจริง ถ้าคุณอยู่ในสภาพแวดล้อมการผลิต ข้อมูลจะถูกป้อนในบริษัทที่คุณกำลังใช้อยู่ในปัจจุบัน

คุณสามารถใช้ตัวบันทึกงานเพื่อสร้างคู่มืองานแบบที่คุณกำหนดเองได้ สำหรับข้อมูลเพิ่มเติม ดู [สร้างเอกสารหรือการฝึกอบรมโดยการบันทึกงาน](../../dev-itpro/user-interface/task-recorder-training-docs.md)

## <a name="in-product-help"></a>วิธีใช้ภายในผลิตภัณฑ์

ในการเข้าถึงเนื้อหาการให้ความช่วยเหลือ คลิกไอคอน **Help** (**?**) แล้วเลือก ช่วยเหลือ หรือกด press Ctrl+Shift+? ในทั้งสองกรณี บานหน้าต่าง วิธีใช้ จะเปิดขึ้น จากบานหน้าต่างให้ความช่วยเหลือ คุณสามารถเข้าถึงบทความหรือคู่มืองานได้

![บานหน้าต่างวิธีใช้](./media/help-pane-wiki.png)

### <a name="accessing-help-topics-from-the-help-pane"></a>เข้าถึงหัวข้อวิธีการใช้จากบานหน้าต่างวิธีใช้

จากบานหน้าต่าง ให้ความช่วยเหลือ คุณสามารถเข้าถึงบทความที่ใช้ไปยังไคลเอนต์ได้ เมื่อคุณเปิดบานหน้าต่าง ความช่วยเหลือ และคลิกแท็บ **Help** เป็นครั้งแรก คุณจะเห็นบทความที่ใช้กับหน้าที่คุณกำลังเปิดอยู่ ถ้าไม่พบบทความ คุณสามารถป้อนคำสำคัญเพื่อจำกัดการค้นหา เมื่อคุณคลิกบทความในบานหน้าต่าง วิธีใช้ เบราเซอร์ของคุณแท็บจะเปิดแท็บใหม่ และแสดงบทความนั้น

> [!IMPORTANT]
> ส่วนนี้ไม่ได้ใช้กับ Dynamics 365 Talent ระบบวิธีใช้สำหรับ Talent ได้รับการเชื่อมต่อกับคู่มืองานสำหรับผลิตภัณฑ์โดยอัตโนมัติ นอกจากนี้ คุณไม่สามารถสร้างคู่มืองานแบบกำหนดเองสำหรับ Talent


### <a name="accessing-task-guides-from-the-help-pane"></a>เข้าถึงคู่มืองานจากบานหน้าต่างวิธีใช้

ก่อนที่คุณจะสามารถเข้าถึงคู่มืองานจากบานหน้าต่าง ความช่วยเหลือ ผู้ดูแลระบบต้องไปยังหน้า **System parameters** ใน Finance Chain Managment และ การค้าปลีก และการตั้งค่าคอนฟิกบางอย่าง

> [!NOTE]
> - เพื่อกำหนดค่าวิธีใช้ คุณต้องเข้าสู่ระบบด้วยบัญชีผู้ใช้ในชื่อเดียวกันกับชื่อที่ใช้ในแอป
> - คุณจะไม่สามารถเชื่อมต่อไปยัง LCS library จากอินสแตนซ์ของแอปที่ติดตั้งในฮาร์ดไดรฟ์เสมือนเฉพาะที่ (VHD)

![แบบฟอร์มพารามิเตอร์ระบบพร้อมการตั้งค่าวิธีใช้](./media/system-parameters_ops-1024x437.png)

ในหน้า **พารามิเตอร์ระบบ** ให้ปฏิบัติตามขั้นตอนต่อไปนี้:

1. **สิ่งสำคัญ** ในครั้งแรกที่คุณเปิดแท็บวิธีใช้ คุณต้องเชื่อมต่อกับ Lifecycle Services ตรวจสอบให้แน่ใจว่าคลิกลิงค์ในกึ่งกลางของแบบฟอร์ม รอสำหรับการเชื่อมต่อ ปิดกล่องโต้ตอบและคลิก **ตกลง** เพื่อเข้าถึงแบบฟอร์มพารามิเตอร์

    ![เชื่อมต่อกับ LCS](./media/connect-to-lcs-crop-1024x365.png)

2. เลือกโครงการ Lifecycle Services เพื่อเชื่อมต่อ
3. เลือกไลบรารี BPM (ภายในโครงการที่เลือก) เพื่อดึงข้อมูลการบันทึกงาน
4. ตั้งค่าลำดับการแสดงผลของไลบรารี BPM ซึ่งจะกำหนดลำดับว่าบันทึกงานใดจะปรากฏขึ้นในบานหน้าต่าง วิธีใช้

หลังจากที่ผู้ดูแลระบบเสร็จสิ้นขั้นตอนเหล่านี้ คุณสามารถเปิดบานหน้าต่าง ความช่วยเหลือ และคลิกแท็บ **Task guides** คุณจะเห็นคู่มืองานที่ใช้กับหน้าที่คุณกำลังอยู่เปิดใช้อยู่ ถ้าไม่พบคู่มืองาน คุณสามารถป้อนคำสำคัญเพื่อจำกัดการค้นหา หลังจากที่คุณคลิกคู่มืองานในบานหน้าต่าง วิธีใช้ บานหน้าต่าง วิธีใช้ จะแสดงคำแนะนำทีละขั้นตอน และคุณสามารถเล่นคู่มืองานได้

![มุมมองการอ่านคู่มืองาน](./media/task-guide-ops.png)

### <a name="where-are-the-translated-task-guides-for-microsoft-libraries"></a>คู่มืองานที่แปลสำหรับไลบรารี Microsoft อยู่ที่ใด?

คู่มืองานที่แปลแล้วจะถูกนำออกใช้ในไลบรารี "ภาษาทั้งหมด" ในชื่อเรื่อง เมื่อต้องการดูความช่วยเหลือของคู่มืองานที่เป็นภาษาท้องถิ่น ตรวจสอบให้แน่ใจว่าคุณได้เชื่อมต่อไปยังไลบรารีที่เหมาะสมแล้ว ภาษาที่ปรากฏในคู่มืองานมีการควบคุมสำหรับผู้ใช้แต่ละรายโดยการตั้งค่าภาษาภายใต้ **ตัวเลือก** &gt; **การกำหนดลักษณะ**

- ถ้าคู่มืองานได้รับการแปลแล้ว เมื่อคุณเปิดคู่มืองานนั้น ข้อความทั้งหมดของคู่มืองานจะปรากฏในภาษาที่คุณเลือก
- ถ้าคู่มืองานยังไม่ได้รับการแปล เมื่อคุณเปิดคู่มืองานนั้น จะปรากฏข้อความเพียงบางส่วน (ข้อความของการควบคุม) ในภาษาที่คุณเลือก
