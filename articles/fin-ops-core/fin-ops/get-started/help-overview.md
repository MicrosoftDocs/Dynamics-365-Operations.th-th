# <a name="help-system"></a>ระบบวิธีใช้

[!include [banner](../includes/banner.md)]

ผู้ใช้แอปต่อไปนี้สามารถเข้าถึงวิธีใช้ที่สอดคล้องกับบริบทและเนื้อหาอื่นๆ ที่ยึดตามระบบวิธีใช้เดียวกัน:

- Microsoft Dynamics 365 Commerce
- Dynamics 365 Finance
- Dynamics 365 Human Resources
- Dynamics 365 Supply Chain Management

ในแอปเหล่านี้ คุณสามารถเข้าถึงวิธีใช้เฉพาะผลิตภัณฑ์จากบานหน้าต่าง **วิธีใช้**

![บานหน้าต่างวิธีใช้](./media/help-pane-ops-help.png)

## <a name="help-on-docsmicrosoftcom"></a>วิธีใช้เกี่ยวกับ docs.microsoft.com

ไซต์ docs.microsoft.com ([docs.microsoft.com/dynamics365](https://docs.microsoft.com/dynamics365/)) เป็นแหล่งข้อมูลหลักของเอกสารผลิตภัณฑ์สำหรับแอปพลิเคชันที่กล่าวมาแล้วก่อนหน้านี้ ไซต์นี้มีคุณลักษณะดังต่อไปนี้:

- **เข้าถึงเนื้อหาล่าสุด** – ไซต์ทำให้ Microsoft สร้าง จัดส่ง และอัปเดตเอกสารผลิตภัณฑ์ได้รวดเร็วและมีความยืดหยุ่นมากขึ้น ดังนั้น คุณจึงเข้าถึงข้อมูลทางเทคนิคล่าสุดได้ง่าย
- **เนื้อหาที่เขียนโดยผู้เชี่ยวชาญ** – เนื้อหาบนไซต์เปิดให้เข้าร่วมโดยสมาชิกของชุมชนทั้งภายในและภายนอก Microsoft

คุณสามารถค้นหาเนื้อหาบน docs.microsoft.com โดยใช้โปรแกรมค้นหาใดๆ เพื่อให้ได้ผลลัพธ์ที่ดีที่สุดเราขอแนะนำว่าคุณควรใช้การค้นหาไซต์ เช่น **site:docs.microsoft.com dynamics 365 "คำค้นหา"**

### <a name="leave-us-feedback"></a>ฝากคำติชมให้พวกเรา

ถ้าคุณมีความคิดเห็นหรือคำถามเกี่ยวกับหัวข้อ ฝากความคิดเห็นให้พวกเราที่ด้านล่างของหน้า

1. เลือก **คำติชม** เพื่อไปยังข้อคิดเห็นที่ด้านล่างของหน้านั้น จากนั้นเลือก **คำติชมของผลิตภัณฑ์** หรือ **ลงชื่อเข้าใช้เพื่อให้คำติชมของเอกสาร**

2. เริ่มพิมพ์ข้อคิดเห็นของคุณ แล้วเลือก **ส่งคำติชม**

    ![ลงรายการข้อคิดเห็น](./media/feedback.png)

> [!NOTE]
> ถ้าคุณต้องการส่งคำติชมเกี่ยวกับเอกสาร คุณต้องลงชื่อเข้าใช้โดยใช้บัญชี GitHub สำหรับข้อมูลเพิ่มเติม ให้ดูที่ [การตั้งค่าและการจัดการโพรไฟล์ GitHub ของคุณ](https://help.github.com/github/setting-up-and-managing-your-github-profile)

## <a name="task-guides"></a>คู่มืองาน

คำแนะนำของคือคู่มือในการควบคุม ประสบการณ์การโต้ตอบ ที่นำคุณผ่านขั้นตอนของงานหรือกระบวนการทางธุรกิจ คุณสามารถเปิดคู่มืองาน (เล่น) จากบานหน้าต่าง **วิธีใช้** ได้ เมื่อคุณเลือกคู่มืองานเป็นครั้งแรก บานหน้าต่าง **วิธีใช้** จะแสดงคำแนะนำทีละขั้นตอนสำหรับงานนั้น คู่มืองานที่เป็นภาษาท้องถิ่นพร้อมใช้งานแล้วในขณะนี้

ไลบรารีของคู่มืองานที่นำออกใช้ของ Microsoft สำหรับรุ่นผลิตภัณฑ์ โดยผ่านการเผยแพร่ในเดือนธันวาคม 2017 ของ Dynamics 365 for Finance and Operations ส่วน [การเข้าถึงคู่มืองานจากบานหน้าต่างวิธีใช้](#accessing-task-guides-from-the-help-pane) ของหัวข้อนี้จะอธิบายวิธีการค้นหาคู่มืองานที่ถูกต้องสำหรับผลิตภัณฑ์ของคุณ

![มุมมองการอ่านคู่มืองาน](./media/task-guide-ops.png)

หากต้องการเริ่มการแนะนำ ประสบการณ์เชิงโต้ตอบ ให้เลือก **เริ่มต้นคู่มืองาน** ที่ด้านล่างของบานหน้าต่าง **วิธีใช้** ตัวชี้สีดำแสดงตำแหน่งที่จะไปเป็นอันดับแรก ทำตามคำแนะนำที่ปรากฏในส่วนติดต่อผู้ใช้ (UI) และป้อนข้อมูลตามที่กำหนด

![คำแนะนำตามขั้นตอนของคู่มืองาน](./media/task-guide-step-1-ops.png)

> [!IMPORTANT]
> ข้อมูลที่คุณป้อนเมื่อคุณเล่นคู่มืองานเป็นจริง ถ้าคุณอยู่ในสภาพแวดล้อมการผลิต ข้อมูลจะถูกป้อนในบริษัทที่คุณกำลังใช้อยู่ในปัจจุบัน

คุณสามารถใช้ตัวบันทึกงานเพื่อสร้างคำแนะนำของงานแบบกำหนดเองของคุณเอง สำหรับข้อมูลเพิ่มเติม ดู [สร้างเอกสารหรือการฝึกอบรมโดยการบันทึกงาน](../../dev-itpro/user-interface/task-recorder-training-docs.md)

## <a name="in-product-help"></a>วิธีใช้ภายในผลิตภัณฑ์

ฟิลด์บางฟิลด์มีคำอธิบายฟิลด์เพื่อช่วยให้ผู้ใช้ได้รับการปลดบล็อกเมื่อมีความไม่แน่นอนเกี่ยวกับข้อมูลที่ฟิลด์นี้มีอยู่ เป็นต้น นอกจากนี้ บานหน้าต่าง **วิธีใช้ในผลิตภัณฑ์** จะให้การเข้าถึงเนื้อหาที่สอดคล้องกับบริบท ซึ่งช่วยให้ผู้ใช้สามารถเริ่มต้น ปลดบล็อก และเรียนรู้เพิ่มเติม

ถ้าต้องการเข้าถึงเนื้อหาวิธีใช้ ให้เลือกปุ่ม **วิธีใช้** (**?**) แล้วเลือก **วิธีใช้** อีกทางหนึ่งคือ กด **Ctrl+Shift+?** ในทั้งสองกรณี บานหน้าต่าง **วิธีใช้** จะปรากฏขึ้น จากบานหน้าต่าง **วิธีใช้** คุณสามารถเข้าถึงหัวข้อแนวคิดหรือคู่มืองานที่เกี่ยวข้องกับพื้นที่ของผลิตภัณฑ์ที่คุณกำลังอยู่ในปัจจุบัน

![บานหน้าต่างวิธีใช้](./media/help-pane-ops-help.png)

### <a name="accessing-help-topics-from-the-help-pane"></a>เข้าถึงหัวข้อวิธีใช้จากบานหน้าต่างวิธีใช้

จากบานหน้าต่าง **วิธีใช้** คุณสามารถเข้าถึงหัวข้อที่ใช้กับไคลเอนต์ได้ เมื่อคุณเปิดบานหน้าต่าง **วิธีใช้** และคลิกแท็บ **วิธีใช้** เป็นครั้งแรก คุณจะเห็นหัวข้อที่ใช้ได้กับหน้าที่คุณกำลังเปิดอยู่ ถ้าไม่พบหัวข้อ คุณสามารถป้อนคำสำคัญเพื่อจำกัดการค้นหา เมื่อคุณเลือกหัวข้อในบานหน้าต่าง **วิธีใช้** วิธีใช้จะเปิดอยู่บนแท็บใหม่ในเบราเซอร์ของคุณ

> [!IMPORTANT]
> ส่วนนี้ไม่ได้ใช้กับ Dynamics 365 Human Resources ระบบวิธีใช้สำหรับทรัพยากรบุคคลได้รับการเชื่อมต่อกับคู่มืองานสำหรับผลิตภัณฑ์โดยอัตโนมัติ นอกจากนี้ คุณไม่สามารถสร้างคู่มืองานแบบกำหนดเองสำหรับทรัพยากรบุคคลได้

### <a name="accessing-task-guides-from-the-help-pane"></a>เข้าถึงคู่มืองานจากบานหน้าต่างวิธีใช้

ก่อนที่คุณจะสามารถเข้าถึงคู่มืองานจากบานหน้าต่าง **วิธีใช้** ผู้ดูแลระบบต้องตั้งค่าคอนฟิกการตั้งค่าบางอย่างบนหน้า **พารามิเตอร์ระบบ** Finance, Supply Chain Management, หรือ Commerce สำหรับข้อมูลเพิ่มเติม ให้ดูที่ [การเพิ่มคู่มืองาน](help-connect.md#adding-task-guides)

### <a name="where-are-the-translated-task-guides-for-microsoft-libraries"></a>คู่มืองานที่แปลสำหรับไลบรารี Microsoft อยู่ที่ใด

คู่มืองานที่แปลแล้วจะถูกนำออกใช้ในไลบรารีที่มี "ภาษาทั้งหมด" ในชื่อเรื่อง เมื่อต้องการดูวิธีใช้ของคู่มืองานที่เป็นภาษาท้องถิ่น ตรวจสอบให้แน่ใจว่าคุณได้เชื่อมต่อไปยังไลบรารีที่เหมาะสมแล้ว ผู้ใช้แต่ละรายสามารถเปลี่ยนภาษาที่ปรากฏในคู่มืองานโดยการตั้งค่าภาษาภายใต้ **ตัวเลือก** &gt; **การกำหนดลักษณะ**

- ถ้าคู่มืองานได้รับการแปลแล้ว เมื่อคุณเปิดคู่มืองานนั้น ข้อความทั้งหมดของคู่มืองานจะปรากฏในภาษาที่คุณเลือก
- ถ้าคู่มืองานยังไม่ได้รับการแปล เมื่อคุณเปิดคู่มืองานนั้น จะปรากฏข้อความของตัวควบคุมเพียงบางส่วนในภาษาที่คุณเลือก
