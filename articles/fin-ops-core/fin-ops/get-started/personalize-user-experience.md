# <a name="personalize-the-user-experience"></a><span data-ttu-id="a8f87-103">ตั้งค่าประสบการณ์ผู้ใช้แบบส่วนตัว</span><span class="sxs-lookup"><span data-stu-id="a8f87-103">Personalize the user experience</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a8f87-104">หัวข้อนี้จะอธิบายวิธีการตั้งค่าแอปพลิเคชันตามต้องการและครอบคลุมถึงเรื่องต่อไปนี้:</span><span class="sxs-lookup"><span data-stu-id="a8f87-104">This topic explains how you can personalize the app and covers the following subjects:</span></span> 

- <span data-ttu-id="a8f87-105">**ตัวเลือกทั้งระบบ** ตัวเลือกการตั้งค่าส่วนบุคคลเหล่านี้ทำบนหน้าการตั้งค่าและพร้อมใช้งานสำหรับผู้ใช้ทั้งหมด</span><span class="sxs-lookup"><span data-stu-id="a8f87-105">**System-wide options** – These personalization options are made on a setup page and are available to all users.</span></span> <span data-ttu-id="a8f87-106">ตัวอย่างประกอบด้วยชุดรูปแบบสีและโซนเวลา</span><span class="sxs-lookup"><span data-stu-id="a8f87-106">Examples include the color theme and time zone.</span></span> 
- <span data-ttu-id="a8f87-107">**จำกัดการเข้าถึงการตั้งค่าส่วนบุคคล** – ที่ระดับการเข้าถึงนี้ การดำเนินการของผู้ใช้ที่เกี่ยวข้องกับการใช้งานหน้าทั่วไปจะถูกบันทึกโดยอัตโนมัติโดยแอปและเรียกคืนในครั้งถัดไปที่คุณเข้าใช้งานหน้า</span><span class="sxs-lookup"><span data-stu-id="a8f87-107">**Restricted personalization access** – At this access level, user actions that are associated with typical page usage are automatically saved by the app and restored the next time that you visit the page.</span></span> <span data-ttu-id="a8f87-108">ตัวอย่างเช่น แอปจัดเก็บความกว้างของคอลัมน์กริดหากคุณปรับปรุงคอลัมน์เหล่านั้น และสถานะที่ขยายหรือยุบของแท็บด่วน</span><span class="sxs-lookup"><span data-stu-id="a8f87-108">For example, the app stores the width of grid columns if you adjust them, and the expanded or collapsed state of FastTabs.</span></span> 
- <span data-ttu-id="a8f87-109">**การเข้าถึงการตั้งค่าส่วนบุคคลแบบสมบูรณ์** – ที่ระดับการเข้าถึงนี้ ผู้ใช้จะสามารถเข้าถึงความสามารถในการตั้งค่าส่วนบุคคลทั้งหมดในแอปได้</span><span class="sxs-lookup"><span data-stu-id="a8f87-109">**Full personalization access** – At this access level, users have access to all personalization capabilities in the app.</span></span> <span data-ttu-id="a8f87-110">โดยเฉพาะอย่างยิ่ง มีสิทธิ์เข้าถึงแถบเครื่องมือ **การตั้งค่าส่วนบุคคล**</span><span class="sxs-lookup"><span data-stu-id="a8f87-110">In particular, they have access to the **Personalization** toolbar.</span></span> 
- <span data-ttu-id="a8f87-111">**การตั้งค่าส่วนบุคคลที่ใช้ร่วมกัน** – ผู้ใช้ที่มีสิทธิการเข้าถึงการตั้งค่าส่วนบุคคลสมบูรณ์สามารถส่งออกการตั้งค่าหน้าส่วนบุคคลของตนเองและใช้ร่วมกับผู้ใช้คนอื่นได้</span><span class="sxs-lookup"><span data-stu-id="a8f87-111">**Sharing personalizations** – Users who have full personalization access can export their page personalizations and share them with other users.</span></span>
- <span data-ttu-id="a8f87-112">**การจัดการการทำให้เป็นแบบส่วนบุคคล** – ผู้ใช้ที่มีสิทธิ์สามารถเข้าถึงหน้าการจัดการ **การตั้งค่าส่วนบุคคล** เพื่อจัดการการตั้งค่าส่วนบุคคลทั้งหมดที่ระดับองค์กร</span><span class="sxs-lookup"><span data-stu-id="a8f87-112">**Administration of personalizations** – Privileged users can access the **Personalization** administration page to manage all personalizations at an organizational level.</span></span>
