# <a name="new-or-updated-task-guides-may-2016"></a>คู่มืองานใหม่หรือที่ปรับปรุงแล้ว (พฤษภาคม 2016)

[!include [banner](../includes/banner.md)]

บทความนี้แสดงคู่มืองานใหม่หรือที่ปรับปรุงแล้วที่ถูกนำออกใช้ใน Microsoft (พฤษภาคม 2016 - ทุกภาษา) ไลบรารีโดยรวมของ APQC สำหรับ Microsoft Dynamics AX ไม่มีการนำออกใช้การเริ่มต้นใช้งานไลบรารีใหม่ 

## <a name="task-guides-in-the-may-2016---all-languages-apqc-unified-library-for-microsoft-dynamics-ax"></a>คู่มืองานใน (พฤษภาคม 2016 - ทุกภาษา) ไลบรารีโดยรวมของ APQC สำหรับ Microsoft Dynamics AX

(พฤษภาคม 2016) ไลบรารี่โดยรวมของ APQC ประกอบด้วยคู่มืองานที่มีไว้เพื่อใช้เป็นวิธีใช้พร้อมกับการอัพเดต Microsoft Dynamics AX พฤษภาคม 2016 และการนำออกใช้ใน Microsoft Dynamics AX กุมภาพันธ์ 2016 เวอร์ชันที่เป็นภาษาท้องถิ่นของคู่มืองานที่ถูกนำออกใช้ใน (กุมภาพันธ์ 2016) ไลบรารี่โดยรวมของ APQC จะมีอยู่ในไลบรารี่นี้ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการเชื่อมต่อระบบวิธีใช้กับไลบรารีคู่มืองาน ดู [ระบบวิธีใช้](help-overview.md)

> [!NOTE]
> เครื่องหมายดอกจัน (\*) ที่อยู่ถัดจากชื่อเรื่องของรายการตัวทำแบบจำลองกระบวนการทางธุรกิจ (BPM) บ่งชี้ว่าคู่มืองานสำหรับกระบวนการทางธุรกิจถูกสร้างขึ้นสำหรับการอัพเดต Dynamics AX พฤษภาคม 2016 และไม่ได้รับการสนับสนุนในเวอร์ชันก่อนหน้านี้ของ Microsoft Dynamics AX
