## <a name="quickly-navigate-to-a-particular-page"></a><span data-ttu-id="20e85-114">นำทางไปหน้าเฉพาะได้อย่างรวดเร็ว</span><span class="sxs-lookup"><span data-stu-id="20e85-114">Quickly navigate to a particular page</span></span>
