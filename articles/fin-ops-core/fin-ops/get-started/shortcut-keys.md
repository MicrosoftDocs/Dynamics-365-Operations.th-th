# <a name="keyboard-shortcuts"></a>แป้นพิมพ์ลัด

[!include [banner](../includes/banner.md)]

[!INCLUDE [PEAP](../../../includes/peap-1.md)]

แป้นพิมพ์ลัดจะช่วยให้คุณป้อนข้อมูลได้อย่างรวดเร็วและมีประสิทธิภาพในแอปการเงินและการดำเนินงาน 

> [!NOTE]
> แป้นพิมพ์ลัดที่อธิบายไว้ที่นี่อ้างอิงถึงโครงร่างแป้นพิมพ์ของสหรัฐอเมริกา คีย์ในโครงร่างคีย์บอร์ดอื่นๆอาจไม่ตรงกับคีย์บนแป้นพิมพ์ของสหรัฐอเมริกาทุกประการ

ทางลัดบางอย่างในหน้านี้เป็น *คอร์ดคีย์* ซึ่งหมายความว่าต้องการชุดของคีย์สองชุดที่ติดกัน (ซึ่งคั่นด้วยเครื่องหมายจุลภาค) ที่จำเป็นต้องถูกกดค้างไว้โดยอิสระเพื่อทริกเกอร์การดำเนินการที่จำเป็น ตัวอย่างเช่น ทางลัด "Alt+M,A" ต้องให้ผู้ใช้กด "Alt+M" เป็นอันดับแรก ปล่อยคีย์ แล้วจากนั้น กด "A" 

## <a name="finding-a-shortcut"></a>การค้นหาทางลัด

ผู้ใช้สามารถค้นหาทางลัดที่พร้อมใช้งานอยู่ในขณะนี้ได้โดยตรงจากอินเทอร์เฟสผู้ใช้ เพียงคลิกขวาบนตัวควบคุม และเลือก **ดูทางลัด** นี่จะเปิดกล่องโต้ตอบที่แสดงทางลัดที่คุณสามารถใช้ได้ตามที่ที่คุณอยู่บนหน้า หรือคุณสามารถกด Alt+Shift+K เพื่อเปิดกล่องโต้ตอบนี้ (และกด Alt+Shift+K ในครั้งที่สองเพื่อดูทางลัดที่พร้อมใช้งานในปัจจุบันทั้งหมดบนหน้า)

## <a name="focus-management"></a>การจัดการโฟกัส 
การจัดการโฟกัสจะอธิบายถึงวิธีการจัดการโฟกัสของผู้ใช้ในแอปพลิเคชัน แอปการเงินและการดําเนินงานจะมีลักษณะการทำงานโฟกัสเฉพาะที่มีวัตถุประสงค์เพื่อให้ประสิทธิภาพสูงสุดของผู้ใช้ โดยเฉพาะกับสถานการณ์การป้อนข้อมูล ลักษณะการทำงานของโฟกัสได้แก่  

-  เมื่อโหลดหน้า ตัวควบคุมการป้อนข้อมูลที่แก้ไขได้รายการแรกบนหน้าจะโฟกัสและสามารถยอมรับข้อมูลป้อนเข้าของผู้ใช้ได้ทันที ถ้าไม่มีตัวควบคุมการป้อนข้อมูลที่แก้ไขได้บนหน้า ตัวเลือกที่ใช้แทนของโฟกัสเริ่มต้นคือตัวควบคุมการป้อนข้อมูลรายการแรกบนหน้า ตามด้วยตัวควบคุมแรกที่โฟกัสได้บนหน้า   
-  โดยทั่วไปเมื่อเลือกปุ่ม จะมีการดำเนินการและโฟกัสอาจกลับไปที่หน้า เมื่อเกิดกรณีนี้ โฟกัสมีแนวโน้มที่จะกลับไปยังตัวควบคุมปุ่มที่เลือก การดําเนินการนี้จะเป็นจริงในแอปการเงินและการดำเนินงาน **ยกเว้น** เมื่อปุ่มอยู่ในบานหน้าต่างการดําเนินการหรือบนแถบเครื่องมือ ในกรณีดังกล่าว เมื่อการดำเนินการเสร็จสมบูรณ์ โฟกัสจะกลับไปยังตัวควบคุมล่าสุด *ภายนอก* บานหน้าต่างการดำเนินการ (หรือแถบเครื่องมือ) ที่โฟกัสก่อนที่จะเลือกปุ่ม
