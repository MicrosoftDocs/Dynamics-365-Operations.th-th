# <a name="change-the-banner-or-logo"></a><span data-ttu-id="3b591-103">เปลี่ยนป้ายประกาศหรือรูปภาพสัญลักษณ์</span><span class="sxs-lookup"><span data-stu-id="3b591-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="3b591-104">กระบวนงานต่อไปนี้แสดงรายการขั้นตอนที่ผู้ดูแลระบบสามารถใช้เพื่ออัพเดตป้ายประกาศหรือรูปภาพสัญลักษณ์ที่ถูกแสดงสำหรับนิติบุคคล</span><span class="sxs-lookup"><span data-stu-id="3b591-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="3b591-105">บริษัทข้อมูลสาธิตที่เคยสร้างกระบวนงานนี้คือ USMF</span><span class="sxs-lookup"><span data-stu-id="3b591-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="3b591-106">ไปที่ **บานหน้าต่างนำทาง > โมดูล > การจัดการองค์กร > องค์กร > นิติบุคคล**</span><span class="sxs-lookup"><span data-stu-id="3b591-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="3b591-107">ในรายการทางด้านซ้าย ให้เลือกนิติบุคคลซึ่งคุณต้องการอัพเดตป้ายประกาศหรือโลโก้ </span><span class="sxs-lookup"><span data-stu-id="3b591-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="3b591-108">ถ้ามีการเลือกแล้ว ไปที่ขั้นตอนถัดไป</span><span class="sxs-lookup"><span data-stu-id="3b591-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="3b591-109">ขยายแท็บ **รูปภาพของแดชบอร์ด**</span><span class="sxs-lookup"><span data-stu-id="3b591-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="3b591-110">เลือก **เปลี่ยน**</span><span class="sxs-lookup"><span data-stu-id="3b591-110">Select **Change**.</span></span>
    
    <span data-ttu-id="3b591-111">ความละเอียดที่เหมาะสมที่สุดสำหรับรูปป้ายประกาศคือ 1920 x 281 พิกเซล</span><span class="sxs-lookup"><span data-stu-id="3b591-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="3b591-112">ความกว้างที่เหมาะสมที่สุดสำหรับรูปภาพสัญลักษณ์คือ 350 พิกเซล</span><span class="sxs-lookup"><span data-stu-id="3b591-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="3b591-113">เลือก **บันทึก**</span><span class="sxs-lookup"><span data-stu-id="3b591-113">Select **Save**.</span></span>
6. <span data-ttu-id="3b591-114">ไปที่ **บานหน้าต่างนำทาง > โมดูล > ทั่วไป > ทั่วไป > แดชบอร์ดเริ่มต้น**</span><span class="sxs-lookup"><span data-stu-id="3b591-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="3b591-115">คุณควรเห็นรูปภาพสัญลักษณ์หรือป้ายประกาศใหม่ในแดชบอร์ด</span><span class="sxs-lookup"><span data-stu-id="3b591-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="3b591-116">คุณอาจจำเป็นต้องเปลี่ยนบริษัทของคุณโดยใช้ตัวเลือกบริษัท ไปยังบริษัทที่ได้มีการอัพโหลดป้ายประกาศให้</span><span class="sxs-lookup"><span data-stu-id="3b591-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
