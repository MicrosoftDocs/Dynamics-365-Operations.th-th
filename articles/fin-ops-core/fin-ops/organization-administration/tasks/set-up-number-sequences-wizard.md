# <a name="set-up-number-sequences-using-a-wizard"></a>ตั้งค่าลำดับหมายเลขโดยใช้วิซาร์ด

[!include [banner](../../includes/banner.md)]

ลำดับหมายเลขถูกใช้ในการสร้างรหัสที่สามารถอ่านได้ รหัสเฉพาะสำหรับเร็กคอร์ดข้อมูลหลักและเรกคอร์ดธุรกรรมที่ต้องการ  ข้อมูลหลักหรือเรกคอร์ดธุรกรรมที่จำเป็นต้องมีตัวระบุจะอ้างเป็นการอ้างอิง ก่อนที่คุณจะสามารถสร้างเรกคอร์ดใหม่สำหรับการอ้างอิง คุณต้องตั้งค่าลำดับหมายเลขและเชื่อมโยงกับการอ้างอิงนั้น  บทความนี้อธิบายวิธีการตั้งค่าลำดับหมายเลขที่จำเป็นทั้งหมดพร้อมกันโดยใช้วิซาร์ด บริษัทข้อมูลสาธิตที่เคยสร้างกระบวนงานนี้คือ USMF

1. ไปที่ **การนำทาง > โมดูล > การจัดการองค์กร > ลำดับหมายเลข > ลำดับหมายเลข**
2. เลือก **สร้าง**
3. เลือก **ถัดไป**

   - บนหน้านี้คุณจะสามารถแก้ไขรหัสประจำตัว ค่าต่ำสุดและค่าสูงสุดได้ นอกจากนี้คุณยังสามารถระบุได้ว่าลำดับหมายเลขต้องต่อเนื่องกันหรือไม่   
   - อย่าเลือกตัวเลือก **ต่อเนื่อง** ถ้าคุณต้องปันส่วนหมายเลขล่วงหน้าสำหรับลำดับหมายเลข เมื่อต้องการเพิ่มเซ็กเมนต์ขอบเขตไปยังรูปแบบของลำดับหมายเลข ให้เลือกรูปแบบในรายการ และจากนั้น เลือก **รวมขอบเขตในรูปแบบ** ในการลบเซ็กเมนต์ขอบเขตออกจากรูปแบบของลำดับหมายเลข ให้เลือกรูปแบบในรายการ และจากนั้น เลือก **ลบขอบเขตออกจากรูปแบบ** ในการแยกลำดับหมายเลขออกจากการสร้างโดยอัตโนมัติ ให้เลือกลำดับหมายเลขในรายการ และจากนั้น เลือก **ลบ**  

4. เลือก **ถัดไป**
5. เลือก **เสร็จสิ้น**
