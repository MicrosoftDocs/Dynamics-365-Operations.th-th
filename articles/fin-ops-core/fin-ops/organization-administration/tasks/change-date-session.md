# <a name="change-the-date-for-a-session"></a>การเปลี่ยนแปลงวันที่สำหรับรอบเวลา

[!include [banner](../../includes/banner.md)]

หัวข้อนี้จะอธิบายถึงวิธีการเปลี่ยนแปลงวันที่สำหรับรอบเวลา โดยค่าเริ่มต้น วันที่ปัจจุบันจะถูกใช้เมื่อมีการป้อนและลงรายการบัญชีรายการสมุดรายวันหรือเอกสารต้นทาง คุณสามารถเปลี่ยนวันที่ที่ใช้สำหรับรอบเวลาปัจจุบันของคุณ ใช้คุณลักษณะการทำงานนี้เพื่อลงวันที่รายการสมุดรายวันหรือเอกสารต้นทางย้อนหลัง หากจำเป็น

1. ในบานหน้าต่างนำทาง ให้ไปที่ **โมดูล > ทั่วไป > ทั่วไป > วันที่และเวลาของรอบเวลา**
2. ในฟิลด์ **วันที่** ให้ป้อนวันที่
3. เลือก **ตกลง**
