# <a name="configure-address-books"></a>ตั้งค่าคอนฟิกสมุดที่อยู่

[!include [banner](../../includes/banner.md)]

ใช้กระบวนงานนี้และการตัดสินใจที่คุณได้ทำไว้ในการวางแผนและการตั้งค่าคอนฟิกของสมุดที่อยู่สากลและบทความสมุดที่อยู่เพิ่มเติม เพื่อตั้งค่าสมุดที่อยู่เพิ่มเติมสำหรับองค์กรของคุณ

บริษัทข้อมูลสาธิตที่ใช้ในการสร้างงานนี้คือ USMF


## <a name="configure-address-books"></a>ตั้งค่าคอนฟิกสมุดที่อยู่
1. ใน **บานหน้าต่างนำทาง** ไปที่ **โมดูล > การจัดการองค์กร > สมุดที่อยู่สากล > สมุดที่อยู่**
2. คลิก **สร้าง**
3. ในฟิลด์ **ชื่อ** ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ **คำอธิบาย** ให้พิมพ์ค่าใดค่าหนึ่ง
5. คลิก **บันทึก**
6. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
7. คลิกลูกศรเพื่อเพิ่มทีมงานซึ่งพร้อมใช้งานที่ถูกเลือกไปที่สมุดที่อยู่
8. คลิก **บันทึก**
