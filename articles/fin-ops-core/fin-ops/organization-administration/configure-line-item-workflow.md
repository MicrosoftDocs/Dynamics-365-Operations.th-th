# <a name="configure-line-item-workflows"></a>ตั้งค่าคอนฟิกลำดับงานของสินค้าในรายการ

[!include [banner](../includes/banner.md)]

หัวข้อนี้อธิบายวิธีการตั้งค่าคอนฟิกองค์ประกอบลำดับงานของสินค้าในรายการ

หากต้องการตั้งค่าคอนฟิกองค์ประกอบลำดับงานของสินค้าในรายการในโปรแกรมแก้ไขลำดับงาน ให้คลิกขวาที่องค์ประกอบนั้น แล้วคลิก **คุณสมบัติ** เพื่อเปิดหน้า **คุณสมบัติ** จากนั้น ให้ใช้กระบวนงานต่อไปนี้เพื่อตั้งค่าคอนฟิกคุณสมบัติต่างๆ ขององค์ประกอบลำดับงานของสินค้าในรายการ

## <a name="name-the-line-item-workflow-element"></a>ตั้งชื่อองค์ประกอบลำดับงานของสินค้าในรายการ

ทำตามขั้นตอนเหล่านี้ในการป้อนชื่อสำหรับองค์ประกอบลำดับงานของสินค้าในรายการ

1. ในบานหน้าต่างทางซ้าย ให้คลิก **การตั้งค่าพื้นฐาน**
2. ในฟิลด์ **ชื่อ** ให้ป้อนชื่อเฉพาะสำหรับองค์ประกอบลำดับงานของสินค้าในรายการ

## <a name="specify-whether-the-same-workflow-is-used-to-process-all-line-items"></a>ระบุว่าจะใช้ลำดับงานเดียวกันในการประมวลผลสินค้าในรายการทั้งหมดหรือไม่

ทำตามขั้นตอนเหล่านี้เพื่อระบุว่าจะใช้ลำดับงานเดียวกันในการประมวลผลสินค้าในรายการทั้งหมดบนเอกสารเดียวหรือไม่

1. ในบานหน้าต่างทางซ้าย ให้คลิก **การตั้งค่าพื้นฐาน**
2. ถ้าควรใช้ลำดับงานเดียวกันในการประมวลผลสินค้าในรายการทั้งหมดบนเอกสารเดียว ให้คลิก **เรียกลำดับงานเดียวสำหรับสินค้าในรายการทั้งหมด** จากนั้นเลือกลำดับงานที่จะใช้ในการประมวลผลสินค้าในรายการนั้น
3. ถ้าควรใช้ลำดับงานเฉพาะในการประมวลผลสินค้าในรายการที่ตรงกับชุดเงื่อนไขที่เฉพาะเจาะจง ให้คลิก **เรียกลำดับงานสำหรับสินค้าในรายการแต่ละรายการ** จากนั้น ทำตามขั้นตอนเหล่านี้เพื่อกำหนดชุดของเงื่อนไข:

    1. คลิก **เพิ่ม**
    2. เลือกเงื่อนไขในตาราง
    3. บนแท็บ **ชื่อเงื่อนไข** ป้อนชื่อสำหรับชุดของเงื่อนไขที่คุณกำลังกำหนด
    4. คลิก **เพิ่มเงื่อนไข** เพื่อป้อนเงื่อนไข
    5. ป้อนเงื่อนไขเพิ่มเติมใด ๆ ที่จำเป็น
    6. หากต้องการตรวจสอบว่าชุดของเงื่อนไขที่คุณป้อนเข้าไปถูกตั้งค่าคอนฟิกไว้ถูกต้องหรือไม่ ให้คลิก **ทดสอบ** ในหน้า **ทดสอบเงื่อนไขลำดับงาน** ในพื้นที่ **ตรวจสอบความถูกต้องของเงื่อนไข** เลือกเรกคอร์ด แล้วคลิก **ทดสอบ** ระบบจะประเมินเรกคอร์ดเพื่อพิจารณาว่าเป็นไปตามเงื่อนไขที่คุณกำหนดไว้หรือไม่ คลิก **ตกลง** หรือ **ยกเลิก** เพื่อกลับไปที่หน้า **คุณสมบัติ**

    บนแท็บ **ลำดับงาน** เลือกลำดับงานที่จะใช้ในการประมวลผลสินค้าในรายการที่ตรงกับชุดของเงื่อนไขที่คุณกำหนดไว้
