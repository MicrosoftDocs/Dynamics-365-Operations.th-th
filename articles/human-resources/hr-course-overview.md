# <a name="courses-overview"></a>ภาพรวมของหลักสูตร

Microsoft Dynamics 365 Human Resources เป็นโซลูชันที่จะตอบสนองความต้องการด้านการเรียนรู้ขององค์กร โซลูชันนี้มีพาธหลักสูตรการเรียนรู้สองพาธ ได้แก่: *แบบเสมือน* และ *แบบมีผู้สอน*

*การเรียนรู้แบบเสมือน* เป็นประสบการณ์การเรียนรู้ที่ได้รับการปรับปรุงผ่านทรัพยากรออนไลน์ ใน *การฝึกอบรมโดยมีผู้สอน* ผู้สอนจะช่วยอำนวยความสะดวกในการฝึกอบรมของกลุ่มผู้ปฏิบัติงานหรือผู้เรียน

ในโมดูลการเรียนรู้ของเรา ผู้เชี่ยวชาญด้านทรัพยากรบุคคล ผู้ดูแลระบบ ผู้จัดการฝ่ายฝึกอบรม และผู้จัดการสามารถสร้างและกําหนดพาธหลักสูตรที่การเรียนรู้ให้กับผู้ปฏิบัติงานได้

> [!NOTE]
> หากต้องการใช้หลักสูตรเสมือน คุณต้องเปิดใช้งานคุณลักษณะ **การปรับปรุงหลักสูตร** ใน การจัดการคุณลักษณะ

## <a name="set-up-virtual-courses"></a>ตั้งค่าหลักสูตรเสมือน

หากต้องการตั้งค่าคอนฟิกหลักสูตรเสมือน คุณต้องเปิดใช้งานคุณลักษณะ **การปรับปรุงหลักสูตร** ใน การจัดการคุณลักษณะ ไปที่ **หลักสูตร \> สร้าง** และตั้งค่าตัวเลือก **เสมือน** เป็น **ใช่** หลังจากที่เปิดใช้งานคุณลักษณะ ต้องระบุลิงก์หลักสูตร ฟิลด์ **สถานะหลักสูตร** จะตั้งค่าเป็น **เปิด** ตัวเลือก **อนุญาตให้พนักงานลงทะเบียนด้วยตนเอง** จะตั้งค่าเป็น **ใช่** แต่สามารถตั้งค่าเป็น **ไม่** ได้

หลังจากที่เปิดใช้งานคุณลักษณะ **การปรับปรุงหลักสูตร** ใน การจัดการคุณลักษณะ จะมีการเปลี่ยนแปลงต่อไปนี้เกิดขึ้น:

- คุณต้องกําหนดวันที่ครบกําหนดสำหรับการกําหนดหลักสูตร
- ต้องระบุลิงก์หลักสูตร
- **ระบบบริการตนเองของพนักงาน** จะแสดงภาพรวมสำหรับพนักงานใน **หลักสูตร** ผู้ใช้สามารถดูหลักสูตรที่พ้นกําหนด ครบกําหนดเร็วๆ นี้ และกําหนดแล้วได้
- ผู้ปฏิบัติงานสามารถหลักสูตรเสมือนและยืนยันตัวเองให้เสร็จสิ้นได้

## <a name="set-up-instructor-led-courses"></a>ตั้งค่าหลักสูตรแบบมีผู้สอน

หลักสูตรแบบมีผู้สอนไม่ต้องมีการตั้งค่าคอนฟิกก่อนที่จะใช้

ข้อมูลต่อไปนี้คือข้อมูลตัวเลือกที่คุณสามารถระบุสำหรับหลักสูตรหรือไม่ก็ได้ ถ้ามีการป้อนข้อมูลนี้ลงในหลักสูตร ควรตั้งค่าเรกคอร์ดหลักสูตรก่อนที่จะสร้างเรกคอร์ดหลักสูตรดังนี้

- ชนิดของหลักสูตร
- กลุ่มห้องเรียน
- กลุ่มหลักสูตร
- สถานที่จัดหลักสูตร
- ห้องเรียน
- ผู้สอน
- ชนิดของหลักสูตร

คุณสามารถใช้ *ชนิดหลักสูตร* ในการจัดประเภทหลักสูตรตามผู้สอนหรือเนื้อหา คุณสามารถสร้างชนิดหลักสูตรใน **ชนิดหลักสูตร** 

จำนวนผู้เรียนต่ำสุดและสูงสุดที่คุณสามารถลงทะเบียนได้สำหรับหลักสูตรที่มีการกำหนดไว้บนแท็บด่วน **ทั่วไป**  ของหน้า **หลักสูตร** 

### <a name="course-setup-type"></a>ตั้งค่าชนิดของหลักสูตร 

*ชนิดการตั้งค่า* จะกำหนดโครงสร้างของหลักสูตร การตั้งค่าหลักสูตรมีอยู่สามชนิด

| ชนิดการตั้งค่า | คำอธิบาย |
|------|--------|
| มาตรฐาน | หลักสูตรของชนิดการตั้งค่านี้ไม่มีกำหนดการรายวัน เป็นชนิดการตั้งค่าเริ่มต้นเมื่อคุณสร้างหลักสูตรใหม่ |
| กำหนดการ | เลือกการตั้งค่าชนิดนี้เพื่อวางแผนรายละเอียดของแต่ละวันของหลักสูตรที่เกิดขึ้นในช่วงหลายวัน |
| กำหนดการ+รอบเวลา | <p>เลือกการตั้งค่าชนิดนี้สำหรับหลักสูตรที่ซับซ้อนมากขึ้น ตัวอย่างเช่น คุณสามารถแบ่งกำหนดการสำหรับหลักสูตรเป็น *แทร็ก* และ *รอบเวลา*:</p><ul><li>*แทร็ก* เป็นส่วนหัวเรื่องเฉพาะของหลักสูตร</li><li>*รอบเวลา* แบ่งออกเป็นแทร็กและช่วยระบุกระบวนการเฉพาะหรือเทคนิคที่เกี่ยวข้องกับแทร็ก</li></ul> |

### <a name="course-tasks"></a>งานหลักสูตร

สำหรับแต่ละหลักสูตร ให้ทำงานต่อไปนี้:

- ลงทะเบียนผู้เรียน
- ระบุกำหนดเวลาสิ้นสุดของการลงทะเบียน
- กำหนดจำนวนต่ำสุดและสูงสุดของผู้เรียน
- กำหนดสถานที่จัดหลักสูตรและห้องเรียน
- แนะนำโรงแรมให้ผู้เข้าร่วมหลักสูตร
- สร้างคำอธิบายหลักสูตรซึ่งสามารถเผยแพร่บน **ระบบบริการตนเองของพนักงาน**

> [!NOTE]
> คุณสามารถลบหลักสูตรเมื่อไม่มีใครลงทะเบียนเท่านั้น

### <a name="course-statuses"></a>สถานะของหลักสูตร

ตารางต่อไปนี้แสดงรายการสถานะหลักสูตรและการกิจกรรมที่เสร็จสมบูรณ์ของแต่ละหลักสูตร

| สถานะ | การดำเนินการ |
|------|--------|
| สร้างแล้ว | <ul><li>ป้อนและแก้ไขข้อมูลหลักสูตร</li><li>เปลี่ยนสถานะหลักสูตรเป็น  **เปิด** เพื่อให้ผู้ปฏิบัติงานสามารถลงทะเบียนสำหรับหลักสูตร</li></ul> | 
| เปิด | <ul><li>ลงทะเบียนผู้เข้าร่วมสำหรับหลักสูตร</li><li>ลบผู้เข้าร่วมออกจากหลักสูตร</li><li>ยืนยันผู้เข้าร่วมสำหรับหลักสูตร</li><li>เปลี่ยนสถานะหลักสูตรเป็น  **ปิดแล้ว**  หรือ  **ถูกยกเลิก** ถ้าผู้เข้าร่วมที่มีสถานะเป็น  **ยืนยันแล้ว**</li></ul>|
| ปิดแล้ว | คุณสามารถเปิดหลักสูตรอีกครั้ง |
| ถูกยกเลิก | คุณสามารถเปิดหลักสูตรอีกครั้ง |

### <a name="course-participants"></a>ผู้เข้าร่วมหลักสูตร

*ผู้เข้าร่วมหลักสูตร* คือผู้ปฏิบัติงานที่มีส่วนร่วมในหลักสูตรการฝึกอบรมหรือเหตุการณ์ คุณสามารถลงทะเบียนผู้เรียนสำหรับหลักสูตรที่เปิดเท่านั้น

### <a name="workflow"></a>เวิร์กโฟลว์

พนักงานที่ลงทะเบียนสำหรับหลักสูตรโดยผ่านหน้า **ระบบบริการตนเองของพนักงาน**  จะมีเส้นทางการลงทะเบียนผ่านลำดับงานสำหรับการอนุมัติ คุณสามารถกำหนดลำดับงานไปยังหลักสูตรบนแท็บด่วน  **ทั่วไป**  ของหน้า  **หลักสูตร**
