# <a name="buy-and-sell-leave"></a><span data-ttu-id="84b56-103">ซื้อและขายวันลางาน</span><span class="sxs-lookup"><span data-stu-id="84b56-103">Buy and sell leave</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="84b56-104">ใน Dynamics 365 Human Resources คุณสามารถส่งคำขอเพื่อซื้อและขายวันลาตามนโยบายการซื้อและการขายที่ตั้งค่าโดยบริษัทของคุณ</span><span class="sxs-lookup"><span data-stu-id="84b56-104">In Dynamics 365 Human Resources, you can submit requests to buy and sell leave based on the buy and sell leave policies set up by your company.</span></span>  

## <a name="request-to-buy-leave"></a><span data-ttu-id="84b56-105">คำขอเพื่อซื้อการลางาน</span><span class="sxs-lookup"><span data-stu-id="84b56-105">Request to buy leave</span></span>

1. <span data-ttu-id="84b56-106">ในพื้นที่ทำงาน **การบริการตนเองของพนักงาน** ให้เลือก **คำขอเพื่อซื้อการลางาน** ในไทล์ **ยอดดุลเวลาหยุดพัก**</span><span class="sxs-lookup"><span data-stu-id="84b56-106">In the **Employee self service** workspace, select **Buy leave request** in the **Time Off Balances** tile.</span></span> 

2. <span data-ttu-id="84b56-107">เพิ่ม **ชนิดการลางาน** และป้อน **จำนวน** สำหรับจำนวนของการลางานที่คุณต้องการซื้อ</span><span class="sxs-lookup"><span data-stu-id="84b56-107">Add a **Leave type** and enter an **Amount** for the amount of leave you'd like to buy.</span></span> 

3. <span data-ttu-id="84b56-108">เลือก **ส่ง** เมื่อคุณพร้อมที่จะส่งคำขอของคุณ</span><span class="sxs-lookup"><span data-stu-id="84b56-108">Select **Submit** when you're ready to submit your request.</span></span> 

<span data-ttu-id="84b56-109">ยอดคงเหลือของคุณจะอัพเดตโดยอัตโนมัติหรือดำเนินการผ่านกระบวนการอนุมัติก่อนที่จะอัพเดต</span><span class="sxs-lookup"><span data-stu-id="84b56-109">Your balances will either automatically update or go through an approval process before updating.</span></span> <span data-ttu-id="84b56-110">การดำเนินการนี้จะขึ้นอยู่กับวิธีการตั้งค่าคอนฟิกนโยบายการซื้อ</span><span class="sxs-lookup"><span data-stu-id="84b56-110">This depends on how the buy policy has been configured.</span></span>
