# <a name="whats-new-or-changed-in-dynamics-365-human-resources-november-19-2021"></a>มีอะไรใหม่หรือมีอะไรเปลี่ยนแปลงใน Dynamics 365 Human Resources 19 พฤศจิกายน 2021

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

หัวข้อนี้อธิบายถึงคุณลักษณะที่ใหม่ ที่มีการเปลี่ยนแปลง หรือกำลังจะมาถึงในไม่ช้าใน Microsoft Dynamics 365 Human Resources

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับกระบวนการอัปเดตและกำหนดการของพวกเรา ให้ดูที่ [กระบวนการอัปเดต](hr-admin-setup-update-process.md)

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับลักษณะการทำงานใหม่ และวันที่ที่พร้อมใช้งานทั่วไปที่คาดไว้ ให้ดูที่ [Dynamics 365 Human Resources รีลีส 2021 เวฟ 2](/dynamics365-release-plan/2021wave2/human-resources/dynamics365-human-resources/)

## <a name="in-this-release"></a>ในการเผยแพร่นี้

รุ่นนี้การแก้ไขจุดบกพร่องต่อไปนี้ การเปลี่ยนแปลงที่ใช้เพื่อสร้างหมายเลข 8.1.4591

### <a name="bug-fixes"></a>การแก้ไขปัญหา

การแก้ไขปัญหาต่อไปนี้จะรวมอยู่ในการนำออกใช้นี้

> [!NOTE]
> เป้าหมายของเราคือการส่งข้อมูลนี้ให้คุณโดยเร็วที่สุดเท่าที่จะเป็นไปได้ พวกเราอาจมีการอัปเดตในหัวข้อนี้เพื่อรวมการแก้ไขข้อผิดพลาดที่ทำให้เกิดข้อผิดพลาดในการสร้างหลังจากที่มีการเผยแพร่หัวข้อนี้ในครั้งแรก

| หมายเลขปัญหา | ปัญหา | คำอธิบาย |
|---|---|---|
| 626178 | การนําทางหายไปจากไทล์ผู้ปฏิบัติงานใน **ระบบบริการตนเองของผู้จัดการ** | ขณะนี้ปัญหานี้ได้รับการแก้ไขแล้ว การนําทางสามารถใช้เพื่อดูรายละเอียดรายงานใน **ระบบบริการตนเองของผู้จัดการ** |
| 632573 | ไม่มีข้อผิดพลาดในการตรวจสอบความถูกต้องเมื่อบันทึก **หลักสูตร** | ขณะนี้ปัญหานี้ได้รับการแก้ไขแล้ว เมื่อการสร้างหลักสูตรที่มี **จำนวนผู้เข้าร่วมขั้นต่ำ** มากกว่า 0 ยังคงได้รับอนุญาตให้บันทึกแม้เมื่อ **จำนวนผู้เข้าร่วมสูงสุด** เป็น 0 |
| 615955 | ข้อผิดพลาด "ต้องกรอกข้อมูลในฟิลด์ **วัตถุประสงค์**" เมื่อสร้างสถานที่การร้องขอการสรรหาบุคลากรใหม่ | เมื่อสร้างที่อยู่สำหรับสถานที่ของการร้องขอการสรรหาบุคลากรใหม่ คุณได้รับข้อผิดพลาด: "ต้องกรอกข้อมูลในฟิลด์ 'วัตถุประสงค์'" แต่ฟิลด์ **วัตถุประสงค์** ไม่สามารถใช้งานได้ในหน้านั้น |
| 620797 | ข้อผิดพลาดฟิลด์ **เพศ** ว่างเปล่าเป็นความเข้าใจผิด | เมื่อไม่ได้ป้อนเพศให้กับผู้ติดต่อส่วนบุคคล รายงานจะแสดง 'คลิกหรือแตะที่นี่เพื่อป้อนข้อความ' ทําให้ไม่สามารถป้อนสิ่งใดลงในฟิลด์ได้ |
| 620800 | ลิงก์รายงานสวัสดิการถูกซ่อนไว้ | รายงานสวัสดิการไม่สามารถดูโดยค่าเริ่มต้นใน **ระบบบริการตนเองของพนักงาน**  มีการเพิ่มลิงก์ที่ด้านขวาของ **ระบบบริการตนเองของพนักงาน** ในส่วน **ลิงก์** |
| 629778 | มีปัญหาประสิทธิภาพกับการรวม CDS | การร้องขอที่เกี่ยวข้องกับการอนุญาตก่อให้เกิดปัญหาประสิทธิภาพ |
