<span data-ttu-id="b5355-107">แผนแต่ละชนิดจะบ่งชี้ว่าพนักงานสามารถลงทะเบียนในแผนเดียวของชนิดนั้น ๆ หรือหลายแผนได้หรือไม่</span><span class="sxs-lookup"><span data-stu-id="b5355-107">Each plan type indicates whether an employee can enroll in one plan of its type or multiple.</span></span> <span data-ttu-id="b5355-108">ตัวอย่างเช่น พนักงานอาจมีแนวโน้มที่จะสามารถลงทะเบียนได้ทั้งในนโยบายการประกันตนขั้นพื้นฐานและการประกันตนส่วนเพิ่มของแผนชนิด Life</span><span class="sxs-lookup"><span data-stu-id="b5355-108">For example, an employee would likely be able to enroll in both the Basic life and the Supplemental life policies of plan type Life.</span></span> <span data-ttu-id="b5355-109">พนักงานอาจได้รับอนุญาตให้ลงทะเบียนในนโยบายการแพทย์ชนิดเดียวเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="b5355-109">An employee would likely be allowed to enroll in only one policy of type Medical.</span></span>

<span data-ttu-id="b5355-110">ถ้าชนิดของแผนเกี่ยวข้องกับผู้ติดต่อ ชนิดของแผนจะบ่งชี้ว่าผู้ติดต่อเป็นผู้รับผลประโยชน์หรือผู้อยู่ในอุปการะหรือไม่</span><span class="sxs-lookup"><span data-stu-id="b5355-110">If a plan type involves contacts, the plan type indicates whether contacts are beneficiaries or dependents.</span></span> <span data-ttu-id="b5355-111">ตัวอย่างเช่น ชนิดของแผนประกันตนขั้นพื้นฐานจะมีผู้รับผลประโยชน์ ขณะที่ชนิดของแผนการแพทย์ขั้นพื้นฐานจะมีผู้อยู่ในอุปการะ</span><span class="sxs-lookup"><span data-stu-id="b5355-111">For example, a Basic life plan type would have beneficiaries, while a Basic medical plan type would have dependents.</span></span> <span data-ttu-id="b5355-112">ในบางกรณี แผนอาจไม่มีผู้ติดต่อส่วนบุคคลเลย</span><span class="sxs-lookup"><span data-stu-id="b5355-112">In some cases, a plan may not have any personal contacts.</span></span> <span data-ttu-id="b5355-113">ตัวอย่างเช่น บัญชีการใช้จ่ายแบบยืดหยุ่นหรือที่จอดรถ</span><span class="sxs-lookup"><span data-stu-id="b5355-113">For example, a Flexible Spending Account or Parking allowance.</span></span>
