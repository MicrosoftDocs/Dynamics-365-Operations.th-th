# <a name="set-up-employment-types"></a>ตั้งค่าชนิดการจ้างงาน

[!include [banner](includes/preview-feature.md)]

ตั้งค่าประเภทการจ้างงานใน Dynamics 365 Human Resources เพื่อให้ตรงกับผู้ปฏิบัติงานของคุณ

1. ในพื้นที่ทำงาน **การจัดการสิทธิประโยชน์** ภายใต้ **การตั้งค่า** ให้เลือก **ประเภทการจ้างงาน**

2. เลือก **ใหม่**

3. ระบุค่าสำหรับฟิลด์ต่อไปนี้

   | ฟิลด์ | คำอธิบาย |
   | --- | --- |
   | ชนิดการจ้างงาน | รหัสเฉพาะของประเภทการจ้างงาน |
   | คำอธิบาย | คำอธิบายเกี่ยวกับประเภทการจ้างงาน |

4. เลือก **บันทึก**
