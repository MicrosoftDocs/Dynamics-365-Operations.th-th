# <a name="marital-status"></a>สถานภาพการสมรส


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

บทความนี้อธิบายชุดตัวเลือกสถานภาพการสมรสสำหรับ Dynamics 365 Human Resources

ชื่อทางกายภาพ: mshr_dirpersonmaritalstatus

การแจงนับให้ชุดตัวเลือกสำหรับสถานภาพการสมรสของเรกคอร์ดบุคคล

| มูลค่า | ป้ายชื่อ | คำอธิบาย |
| --- | --- | --- |
| 200000000 | None | ไม่ได้เลือกค่า
| 200000001 | โสด | โสด |
| 200000002 | สมรส | สมรส |
| 200000003 | หย่าร้าง | หย่าร้าง |
| 200000004 | หม้าย | ม่าย |
