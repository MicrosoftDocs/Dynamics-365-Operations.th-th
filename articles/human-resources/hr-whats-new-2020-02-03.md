# <a name="whats-new-or-changed-in-dynamics-365-human-resources-february-3-2020"></a>มีอะไรใหม่หรือการเปลี่ยนแปลงอะไรใน Dynamics 365 Human Resources (3 กุมภาพันธ์ 2020)

หัวข้อนี้อธิบายถึงคุณลักษณะที่ใหม่หรือที่มีการเปลี่ยนแปลง อย่างใดอย่างหนึ่ง ใน Dynamics 365 Human Resources การเปลี่ยนแปลงที่ใช้เพื่อสร้างหมายเลข 8.1.2809 ตัวเลขในวงเล็บในส่วนหัวบางส่วนอ้างอิงถึงหมายเลขที่สนับสนุนใน Microsoft Dynamics Lifecycle Services (LCS)

## <a name="cant-remove-activities-from-performance-review-form-403542"></a>ไม่สามารถลบกิจกรรมออกจากการตรวจทานประสิทธิภาพจาก (403542)

ขณะนี้คุณสามารถลบกิจกรรมที่เลือกได้โดยใช้ปุ่ม **ลบ** บนแบบฟอร์ม **การตรวจทานประสิทธิภาพ**

## <a name="in-preview"></a>ในการแสดงตัวอย่าง

ตัวอย่างคุณลักษณะต่อไปนี้จะพร้อมใช้งานในวันที่ 3 กุมภาพันธ์ 2020:

- **ตัวอย่างคุณลักษณะการลางานและขาดงาน** - สำหรับข้อมูลเพิ่มเติมโปรดดู [ตัวอย่างคุณลักษณะการลางานและขาดงาน](hr-leave-and-absence-overview.md?leave-and-absence-preview-features).

- **ตัวอย่างคุณลักษณะการจัดการสวัสดิการ** - สำหรับข้อมูลเพิ่มเติมรวมทั้งปัญหาที่ทราบโปรดดู [ภาพรวมการจัดการสวัสดิการ](hr-benefits-management-overview.md)

## <a name="coming-soon"></a>เร็วๆ นี้

โซลูชัน Common Data Service ใหม่จะพร้อมใช้งานเร็วๆ นี้ โดยมีการเปลี่ยนแปลงต่อไปนี้:

| คำอธิบาย | เปลี่ยน |
| --- | --- |
| เอนทิตี **งาน/ตำแหน่ง** เปลี่ยนแปลง | <ul><li>เพิ่ม **ภูมิภาคของค่าตอบแทน** แล้ว</li><li>เพิ่ม **มิติทางการเงิน** แล้ว</li></ul> |
| เอนทิตี **ผู้ปฏิบัติงาน** เปลี่ยนแปลง | <ul><li>เพิ่ม **ลำดับชื่อ** แล้ว</li><li>เพิ่ม **การทำงานจากที่บ้าน** แล้ว</li><li>เพิ่ม **ภาษา** แล้ว</li><li>เพิ่ม **วันที่ของอายุงาน** แล้ว</li><li>เพิ่ม **วันที่ครบรอบปี** แล้ว</li><li>เพิ่ม **วันที่จ้างงานเดิม** แล้ว</li></ul> |
| เอนทิตี **การจ้างงาน** เปลี่ยนแปลง | <ul><li>เพิ่ม **มิติทางการเงิน** แล้ว</li><li>เพิ่ม **เหตุผลการสิ้นสุดการจ้างงาน** แล้ว</li><li>**วันที่สิ้นสุดการจ้างงาน** เปลี่ยนชื่อจาก **วันที่เปลี่ยนแปลง**</li><li>เพิ่ม **วันที่ทดลองงาน** แล้ว</li></ul> |
| เอนทิตี **ที่อยู่ผู้ปฏิบัติงาน** เปลี่ยนแปลง | <ul><li>เพิ่ม **ที่อยู่ถนน** แล้ว</li><li>**บรรทัดที่อยู่ 1** **บรรทัดที่อยู่ 2** และ **บรรทัดที่อยู่ 3** ที่ทำเครื่องหมายสำหรับค่าเสื่อมราคา</li></ul> |
| เอนทิตีการตั้งค่าค่าตอบแทนผันแปรใหม่ | <ul><li>**ชนิดแผนตัวแปรค่าตอบแทน**</li><li>**แผนค่าตอบแทนผันแปร**</li><li>**กฎสิทธิพึงได้**</li><li>**ระดับแผนตัวแปรค่าตอบแทน**</li></ul> |
| เอนทตี **การจ้างงานในปฏิทินของผู้ปฏิบัติงาน** ใหม่ | <ul><li>เพิ่ม **เอนทิตีปฏิทินงาน** แล้ว</li></ul> |
