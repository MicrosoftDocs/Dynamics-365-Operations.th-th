# <a name="set-up-tier-codes"></a>ตั้งค่ารหัสชั้น

[!include [banner](includes/preview-feature.md)]

Dynamics 365 Human Resources ใช้รหัสระดับที่มีอัตราสวัสดิการซึ่งชำระโดยพนักงานและนายจ้าง คุณสามารถกำหนดจำนวนของรหัสเทียร์และตั้งค่าการคำนวณอัตราได้ไม่จำกัดเพื่อให้ตรงกับความต้องการของบริษัทของคุณ

1. ในพื้นที่ทำงาน **การจัดการสวัสดิการ** ภายใต้ **การตั้งค่า** ให้เลือก **รหัสระดับ**

2. เลือก **ใหม่**

3. ระบุค่าสำหรับฟิลด์ต่อไปนี้

   | ฟิลด์ | คำอธิบาย |
   | --- | --- |
   | รหัสชั้น | รหัสระดับหรือหมายเลขของสวัสดิการเฉพาะเพื่อระบุรหัสระดับ |
   | คำอธิบาย | คำอธิบายย่อเกี่ยวกับชนิดของรหัสระดับสวัสดิการ |

4. เลือก **บันทึก**
