<span data-ttu-id="2c954-104">คุณสามารถระงับการลางานสำหรับพนักงานเพื่อหยุดกระบวนการของการค้างรับการลางานสำหรับชนิดการลางานที่เลือก</span><span class="sxs-lookup"><span data-stu-id="2c954-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="2c954-105">ระงับการลางานและการขาดงานสำหรับพนักงาน</span><span class="sxs-lookup"><span data-stu-id="2c954-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="2c954-106">บนเรกคอร์ดของพนักงาน ให้เลือก **ลา**</span><span class="sxs-lookup"><span data-stu-id="2c954-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="2c954-107">เลือก **ระงับการลางาน**</span><span class="sxs-lookup"><span data-stu-id="2c954-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="2c954-108">เลือก **ใหม่**</span><span class="sxs-lookup"><span data-stu-id="2c954-108">Select **New**.</span></span>

4. <span data-ttu-id="2c954-109">ในกล่องโต้ตอบ **ระงับการค้างรับ** ให้เลือก **ชนิดการลางาน** พร้อมกับ **วันที่เริ่มต้น** และ **วันที่สิ้นสุด** สำหรับการระงับ</span><span class="sxs-lookup"><span data-stu-id="2c954-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

5. <span data-ttu-id="2c954-110">อีกทางหนึ่งคือ คุณสามารถเพิ่ม **ข้อคิดเห็น** สำหรับการระงับได้</span><span class="sxs-lookup"><span data-stu-id="2c954-110">Optionally, you can add a **Comment** for the suspension.</span></span>
