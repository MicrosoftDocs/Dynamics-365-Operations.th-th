# <a name="create-employment-categories"></a>สร้างประเภทการจ้างงาน

[!include [banner](includes/preview-feature.md)]

ตั้งค่าหมวดหมู่การจ้างงานใน Dynamics 365 Human Resources เพื่อให้ตรงกับผู้ปฏิบัติงานของคุณ

1. ในพื้นที่ทำงาน **การจัดการสิทธิประโยชน์** ภายใต้ **การตั้งค่า** ให้เลือก **หมวดหมู่การจ้างงาน**

2. เลือก **ใหม่**

3. ระบุค่าสำหรับ **ชนิดผู้ปฏิบัติงาน** **ประเภทการจ้างงาน** และ **คำอธิบาย**

4. เลือก **บันทึก**
