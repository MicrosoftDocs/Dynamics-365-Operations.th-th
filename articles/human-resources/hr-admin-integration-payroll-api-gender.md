# <a name="payroll---gender-option-set"></a>ค่าจ้าง - ชุดตัวเลือกเพศ


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

หัวข้อนี้อธิบายชุดตัวเลือกเพศสำหรับ Dynamics 365 Human Resources

## <a name="option-set-gender-option"></a>ชุดตัวเลือก - ตัวเลือกเพศ

การแจงนับนี้จะให้ชุดตัวเลือกของเพศสำหรับพนักงาน ซึ่งพร้อมใช้งานในชุดตัวเลือก mshr_hcmpersongender

| มูลค่า | ป้ายชื่อ | คำอธิบาย |
| --- | --- | --- |
| 200000000 | None | ไม่ได้ระบุเพศ |
| 200000001 | เพศชาย | เพศชาย |
| 200000002 | เพศหญิง | เพศหญิง |
| 200000003 | ไม่เจาะจง | การเลือกเพศที่ไม่ระบุเพศ |
