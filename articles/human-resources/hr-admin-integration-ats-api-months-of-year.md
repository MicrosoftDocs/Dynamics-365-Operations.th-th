# <a name="months-of-year"></a>เดือนต่างๆของปี


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

บทความนี้อธิบายการกำหนดตัวเลือกเดือนต่างๆของปีสำหรับ Dynamics 365 Human Resources

ชื่อทางกายภาพ: mshr_monthsofyear

การแจงนับนี้จะมีการกำหนดตัวเลือกสำหรับเดือนต่างๆของปี

| มูลค่า | ป้ายชื่อ | คำอธิบาย |
| --- | --- | --- |
| 200000000 | None | ไม่ได้เลือกค่า |
| 200000001 | มกราคม | มกราคม |
| 200000002 | กุมภาพันธ์ | กุมภาพันธ์ |
| 200000003 | มีนาคม | มีนาคม |
| 200000004 | เมษายน | เมษายน |
| 200000005 | พ.ค. | พฤษภาคม |
| 200000006 | มิถุนายน | มิถุนายน |
| 200000007 | กรกฎาคม | กรกฎาคม |
| 200000008 | สิงหาคม | สิงหาคม |
| 200000009 | กันยายน | กันยายน |
| 200000010 | ตุลาคม | ตุลาคม |
| 200000011 | พฤศจิกายน | พฤศจิกายน |
| 200000012 | ธันวาคม | ธันวาคม |
