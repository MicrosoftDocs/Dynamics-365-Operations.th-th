# <a name="payroll---no-yes-option-set"></a>บัญชีเงินเดือน - การกำหนดตัวเลือก ไม่ใช่ ใช่


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

บทความนี้อธิบายการกำหนดตัวเลือก ไม่ใช่ ใช่ สำหรับ Dynamics 365 Human Resources

## <a name="option-set-no-yes"></a>การกำหนดตัวเลือก: ไม่ใช่ ใช่

ชื่อทางกายภาพ: mshr_noyes

การแจงนับนี้จะให้การกำหนดตัวเลือกสำหรับคุณสมบัติแบบบูลีนทั่วไปในเอนทิตี้เสมือนของทรัพยากรบุคคล

| มูลค่า | ป้ายชื่อ | คำอธิบาย |
| --- | --- | --- |
| 200000000 | ไม่ | ไม่ |
| 200000001 | ใช่ | ใช่ |
