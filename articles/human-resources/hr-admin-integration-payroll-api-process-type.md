# <a name="process-type"></a>ชนิดของกระบวนการ

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

หัวข้อนี้อธิบายการกำหนดตัวเลือก ไม่ใช่ ใช่ สำหรับ Dynamics 365 Human Resources

## <a name="option-set-process-type"></a>ชุดตัวเลือก: ชนิดของกระบวนการ

ชื่อทางกายภาพ: mshr_hrmCompProcessType

การแจงนับนี้จะให้การกำหนดตัวเลือกสำหรับชนิดของกระบวนการในเอนทิตีเสมือนของทรัพยากรบุคคล

| มูลค่า | ป้ายชื่อ | คำอธิบาย |
| --- | --- | --- |
| 200000000 | None | ไม่มี |
| 200000001 | รายปี | รายปี |
| 200000002 | วันครบรอบปี | วันครบรอบปี |
| 200000003 | ไม่หมุนเวียน | ไม่หมุนเวียน |
| 200000004 | ปี ½ | ทุกครึ่งปี |
| 200000005 | รายไตรมาส | รายไตรมาส |
| 200000006 | รายเดือน | รายเดือน |
