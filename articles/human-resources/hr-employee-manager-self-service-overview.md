# <a name="employee-and-manager-self-service-overview"></a>ภาพรวมของระบบบริการตนเองของพนักงานและผู้จัดการ


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

บทความนี้แสดงภาพรวมเกี่ยวกับพื้นที่ทำงานของระบบบริการตนเองของพนักงานและผู้จัดการ

## <a name="edit-personal-details"></a>แก้ไขรายละเอียดส่วนบุคคล

ถ้าคุณต้องการเพิ่มหรือเปลี่ยนข้อมูลส่วนบุคคลใดๆ ให้ดูที่ [แก้ไขข้อมูลส่วนบุคคล](hr-employee-manager-self-service-edit-personal-information.md)

## <a name="user-not-assigned-to-a-worker-record"></a>ไม่ได้กำหนดผู้ใช้ให้กับเรกคอร์ดผู้ปฏิบัติงาน

ถ้าคุณไม่ได้เชื่อมโยงผู้ใช้ของคุณกับเรกคอร์ด **ผู้ปฏิบัติงาน** ในหน้า **ผู้ใช้** ข้อความต่อไปนี้จะแสดงขึ้น:

**รหัสผู้ใช้ของคุณไม่เชื่อมโยงกับเรกคอร์ดพนักงานในระบบของคุณ คุณจะไม่สามารถดูหรืออัปเดตข้อมูลของคุณได้จนกว่าจะมีการเชื่อมโยง ติดต่อผู้จัดการหรือทีมงานสนับสนุนของคุณเพื่อขอความช่วยเหลือ**

เมื่อต้องการเชื่อมโยงผู้ใช้กับเรกคอร์ด **ผู้ปฏิบัติงาน** ไปที่ **ผู้ใช้** และเลือกผู้ใช้ เลือก **แก้ไข** เพิ่มผู้ปฏิบัติงานที่เกี่ยวข้องในฟิลด์ **บุคคล** ในหน้าและเลือก **บันทึก** ขณะนี้คุณควรมีสิทธิ์เข้าถึง **ระบบบริการตนเองของพนักงาน**

## <a name="security-requirements-for-employee-and-manager-self-service"></a>ข้อกำหนดด้านความปลอดภัยของระบบบริการตนเองสำหรับพนักงานและผู้จัดการ

ระบบบริการตนเองสำหรับพนักงานและผู้จัดการต้องมีบทบาทความปลอดภัยสองอย่าง:

- พนักงานจำเป็นต้องมีบทบาทของพนักงาน
- ผู้จัดการต้องใช้ทั้งบทบาทของพนักงานและผู้จัดการ

>[!NOTE]
>นอกจากนี้ คุณยังสามารถใช้บทบาทที่กำหนดเองเพื่อเข้าถึงระบบบริการตนเองของพนักงานและผู้จัดการได้ตราบเท่าที่ได้รับอนุญาตให้เข้าถึงพื้นที่ทำงานของพนักงานและผู้จัดการ<br>
>การเข้าถึงข้อมูลพนักงานของผู้จัดการจะขึ้นอยู่กับลำดับชั้นของสายตำแหน่งปัจจุบันที่กำหนดไว้ใน Human Resources

## <a name="employee-self-service"></a>ระบบบริการตนเองของพนักงาน

แท็บ **ข้อมูลของฉัน** จะแสดงข้อมูลต่อไปนี้สำหรับ **ระบบบริการตนเองของพนักงาน**  

### <a name="summary"></a>สรุป

**รายการงานที่กำหนดให้กับฉัน** จะแสดงการอนุมัติและรายการลำดับงานทั้งหมดที่กำหนดให้กับพนักงาน คุณสามารถตั้งค่าคอนฟิกรายการลำดับงานเพื่อส่งอีเมลไปยังผู้ใช้

**แบบสอบถามที่กำหนดให้กับฉัน** แสดงแบบสอบถามที่จัดกำหนดการทั้งหมดที่กำหนดให้กับพนักงานหรือกลุ่มโดยตรง

**ไดเรกทอรีของบริษัท** อนุญาตให้พนักงานสามารถค้นหาข้อมูลที่เกี่ยวข้องกับบุคคลในองค์กร ข้อมูลผู้ติดต่อสาธารณะจะพร้อมใช้งานสำหรับพนักงานทั้งหมด ไดเรกทอรีของบริษัทถูกจำกัดให้กับบริษัทที่พนักงานได้ลงชื่อเข้าใช้แล้ว

**ปฏิทินทีมงาน** แสดงข้อมูลปฏิทินของทีมงานของคุณ สำหรับข้อมูลเพิ่มเติม ให้ดูที่ [ดูทีมงานและปฏิทินของบริษัท](hr-employee-self-service-calendar.md)

### <a name="my-career-information"></a>ข้อมูลอาชีพของฉัน

ส่วน **ข้อมูลอาชีพของฉัน** ของ **ระบบบริการตนเองของพนักงาน** แสดงไทล์ที่เกี่ยวข้องกับการลางานและการขาดงาน การจัดการประสิทธิภาพ ความสามารถ สวัสดิการงาน และสิ่งที่แนบมา

ไทล์ **ยอดดุลเวลาหยุดพัก** แสดงยอดดุลสำหรับแผนใดๆ ที่ลงทะเบียน ไทล์นี้คาดการณ์ยอดดุลของคุณตามวิธีการรับรู้ของคุณ คุณสามารถป้อนและส่งคำขอเวลาหยุดพัก ซึ่งจากนั้นจะผ่านกระบวนการลำดับงานการอนุมัติ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการขาดงานและการลางาน ดู [ภาพรวมการลางานและการขาดงาน](hr-leave-and-absence-overview.md)

ไทล์ **งาน** จะแสดงงานที่ได้รับการกำหนดให้กับคุณ และช่วยให้คุณสามารถดูและจัดการได้

**หลักสูตรที่ลงทะเบียนถัดไป** จะแสดงหลักสูตรถัดไปที่คุณลงทะเบียนไว้ คุณสามารถดูและลงทะเบียนสำหรับหลักสูตรที่เปิดใดๆ หลักสูตรทั้งหมดที่เปิดสำหรับการลงทะเบียนมีสถานะเป็น **เริ่มต้นแล้ว** และอนุญาตให้พนักงานลงทะเบียนด้วยตนเองจะปรากฏ โดยขึ้นอยู่กับการตั้งค่าขององค์กรของคุณ การลงทะเบียนหลักสูตรของคุณอาจผ่านกระบวนการอนุมัติ

ไทล์ **ใบรับรอง** จะแสดงใบรับรองและวันหมดอายุของใบรับรองที่ใกล้เคียงกับวันที่ปัจจุบันมากที่สุด คุณสามารถปรับปรุง เพิ่ม หรือลบใบรับรอง โดยขึ้นอยู่กับการตั้งค่าขององค์กรของคุณ การปรับปรุงใบรับรองอาจผ่านกระบวนการอนุมัติ

**การตรวจทานที่จัดกำหนดการถัดไป** แสดงการตรวจทานประสิทธิภาพถัดไปของคุณ คุณสามารถเริ่มต้นการตรวจทานใหม่จากไทล์นี้ได้ ผู้จัดการของคุณหรือตัวแทน HR ก็สามารถเริ่มต้นการตรวจทานได้ด้วย ทั้งนี้ขึ้นอยู่กับการตั้งค่าขององค์กรของคุณ คุณอาจสามารถดู ปรับปรุง และส่งการตรวจทานทางออกได้

คุณสามารถจัดการเป้าหมายของคุณได้ด้วย **เป้าหมายประสิทธิภาพการทำงาน** ไทล์นี้แสดงจำนวนของเป้าหมายที่คุณมีในแต่ละสถานะ (**ไม่ได้เริ่มต้น** **ระหว่างการติดตาม** และ **จำเป็นต้องมีการปรับปรุง**) คุณสามารถสร้าง ปรับปรุง และลบเป้าหมายได้ ทั้งนี้ขึ้นอยู่กับการรักษาความปลอดภัยตามบทบาทที่กำหนดของคุณ ถ้าคุณต้องการ คุณสามารถเพิ่มเป้าหมายใหม่จากกลุ่มหรือเทมเพลตได้ นอกจากนี้ ผู้จัดการและ HR ยังสามารถสร้างเป้าหมายในนามของพนักงานและกำหนดรายละเอียดของแต่ละเป้าหมาย ผู้จัดการและพนักงานสามารถทำงานร่วมกันบนเป้าหมาย และปรับปรุงกิจกรรม การวัด และสถานะ นอกจากนี้ คุณยังสามารถรวมสิ่งที่แนบมาด้วย

คุณสามารถดูทักษะของคุณที่มีอยู่บนไทล์ **ทักษะรวม** คุณสามารถปรับปรุงทักษะ เพิ่มรายการใหม่ หรือลบรายการใดๆ ที่ไม่เกี่ยวข้องอีกต่อไป โดยขึ้นอยู่กับการตั้งค่าขององค์กรของคุณ การเปลี่ยนแปลงไปยังทักษะของคุณอาจผ่านกระบวนการอนุมัติ

คุณสามารถดูค่าตอบแทนปัจจุบันของคุณบน **ค่าตอบแทน** ได้ เลือก **แสดง** เพื่อดูการชำระค่าจ้างรายปีและยอดเงินที่เพิ่มขึ้นครั้งล่าสุดของคุณ ถ้าคุณได้รับการว่าจ้างในบริษัทมากกว่าหนึ่งบริษัท ยอดเงินรายปีแต่ละรายการจะแสดงขึ้น เมื่อต้องการดูประวัติค่าตอบแทนโดยละเอียด ให้เลือก **ยอดเงินเดือนรายปี** เพื่อเปิดหน้า **ประวัติค่าตอบแทนคงที่และผันแปร** ค่าตอบแทนในอนาคตไม่แสดงในหน้านี้ ถ้าคุณมีการจ้างงานมากกว่าหนึ่งรายการ คุณสามารถสลับระหว่างบริษัทต่างๆ ภายในหน้านี้เพื่อดูประวัติค่าตอบแทนของคุณได้ โดยไม่ต้องล็อกอินเข้าสู่แต่ละบริษัท

ดูและจัดการเอกสารโดยใช้ไทล์ **เอกสารแนบ** คุณสามารถจัดการเอกสารแนบ **ภายนอก** ทั้งหมดได้ ทั้ง HR และพนักงานสามารถเพิ่มเอกสารแนบผ่านทาง **ระบบบริการตนเองของพนักงาน** หรือหน้า **ผู้ปฏิบัติงาน** เอกสารแนบถูกตั้งค่าเป็น **ภายนอก** โดยค่าเริ่มต้น

### <a name="additional-information"></a>ข้อมูลเพิ่มเติม

ส่วนนี้จะให้ลิงก์ไปยังพื้นที่ **ระบบบริการตนเองของพนักงาน** คล้ายกับส่วน **ข้อมูลอาชีพของฉัน**

ลงชื่อสมัครรับสวัสดิการผ่านทางลิงก์ **สวัสดิการ** สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการจัดการสวัสดิการ ให้ดูที่ [ภาพรวมของสวัสดิการ](hr-benefits-management-overview.md)

ภายใต้ **ประสิทธิภาพ** คุณสามารถเลือก **สมุดรายวันประสิทธิภาพ** เพื่อสร้างรายการสมุดรายวันประสิทธิภาพเพื่อใช้ทั้งเป้าหมายประสิทธิภาพและบทวิจารณ์ คุณสามารถเลือก **ส่งผลป้อนกลับ** เพื่อให้ผลป้อนกลับสำหรับพนักงานรายอื่นๆ ภายในองค์กรของคุณ ทั้งนี้ขึ้นอยู่กับการตั้งค่าขององค์กรของคุณ อาจมีการส่งอีเมลไปยังผู้รับ ผู้ส่ง และผู้จัดการ คุณสามารถส่งผลป้อนกลับให้กับพนักงานทั้งหมดภายในองค์กรได้ การส่งผลป้อนกลับไม่ถูกจำกัดโดยบริษัท

ภายใต้ **ความสามารถ** คุณสามารถทำการเปลี่ยนแปลงไปยัง **หลักสูตร** **การศึกษา** **ตำแหน่งตัวแทนของบริษัท** และ **ประสบการณ์การทำงาน** โดยขึ้นอยู่กับการตั้งค่าขององค์กรของคุณ การปรับปรุงไปยังความสามารถเหล่านี้อาจผ่านกระบวนการอนุมัติ

คุณสามารถดูรายละเอียดงานภายใต้ **องค์กร** รายละเอียดงานได้แก่ ทักษะ ใบรับรอง และพื้นที่ของความรับผิดชอบ สำหรับตำแหน่งหลักของคุณ นอกจากนี้ คุณยังสามารถดูอุปกรณ์ที่ให้ยืมใดๆ ที่เช็คเอาท์ไปยังคุณได้ โดยขึ้นอยู่กับการตั้งค่าขององค์กรของคุณ การเปลี่ยนแปลงไปยังอุปกรณ์ที่ให้ยืมอาจผ่านกระบวนการอนุมัติ

ภายใต้ **แบบสอบถาม** คุณจะเห็นแบบสอบถามที่เสร็จสมบูรณ์แล้ว นอกจากนี้ คุณยังสามารถดูแบบสอบถามทั้งบริษัทที่ยังไม่เสร็จสมบูรณ์ได้ด้วย คุณสามารถเลือกที่จะกรอกแบบสอบถามได้ตลอดเวลา ผู้เขียนแบบสอบถามสามารถกำหนดกรอบเวลาและผู้ที่สามารถใช้แบบสอบถามได้

คุณสามารถตั้งค่าคอนฟิกลิงก์ที่กำหนดโดยผู้ใช้ใน **พารามิเตอร์ทรัพยากรบุคคล** ตัวอย่างเช่น คุณสามารถกำหนดลิงก์ไปยังใบแจ้งยอดการชำระ เอกสารสิ้นปี หรือโซลูชันภายนอก ลิงก์เหล่านี้จะแสดงขึ้นที่ด้านล่างของส่วนนี้ แต่คุณสามารถย้ายลิงก์เหล่านี้ได้โดยใช้การตั้งค่าส่วนบุคคล

นอกจากนี้ คุณยังสามารถสร้างแท็บเพิ่มเติมได้โดยการฝัง Power Apps ไว้ภายในพื้นที่ทำงานของ **ระบบบริการตนเองของพนักงาน** ใช้เมนู **การตั้งค่า** เพื่อกำหนดหน้าให้เป็นส่วนตัวด้วย Power Apps ใดๆ ในเมนู **การตั้งค่า** คุณสามารถเลือกที่จะเพิ่ม Power App ให้กรอกรายละเอียด และแทรกแอป ตามค่าเริ่มต้น Power Apps จะปรากฏเป็นแท็บแรกในลำดับ คุณสามารถเปลี่ยนลำดับโดยใช้การตั้งค่าส่วนบุคคลมาตรฐาน

## <a name="my-team"></a>ทีมงานของฉัน

แท็บ **ทีมงานของฉัน** จะแสดงข้อมูลต่อไปนี้สำหรับ **ระบบบริการตนเองของผู้จัดการ** เฉพาะผู้จัดการเท่านั้นที่สามารถเข้าถึงแท็บ **ทีมงานของฉัน**

### <a name="personnel-actions"></a>การดำเนินการด้านบุคลากร

การดำเนินการด้านบุคลากรแสดงตามตัวเลือกการตั้งค่าคอนฟิกภายใน **พารามิเตอร์ที่ใช้ร่วมกันของทรัพยากรบุคคล** และ **พารามิเตอร์ทรัพยากรบุคคล** เมื่อเปิดใช้งานสำหรับ **ผู้ปฏิบัติงาน** การดำเนินการด้านบุคลากรจะเปิดใช้งานตัวเลือกเมนูใหม่ ซึ่งรวมถึง:

- **ร้องขอพนักงานใหม่**
- **ร้องขอผู้รับเหมาใหม่**
- **ร้องขอการกำหนดผู้ปฏิบัติงานซ้ำ**
- **ร้องขอการยกเลิกการจ้างงาน**
- **ส่งคำขอการเปลี่ยนแปลงค่าตอบแทนคงที่**

คุณสามารถตั้งค่าคอนฟิกตัวเลือกเหล่านี้เพื่อไปที่ลำดับงานการตรวจทานและการอนุมัติที่เลือกกำหนดได้

การเปิดใช้งาน **การดำเนินการของตำแหน่ง** จะเปิดใช้งานตัวเลือกต่อไปนี้:

- **ร้องขอตำแหน่งใหม่**
- **ร้องขอการเปลี่ยนแปลงรายละเอียดตำแหน่งงาน**

นอกจากนี้ คุณยังสามารถตั้งค่าคอนฟิกตัวเลือกเหล่านี้เพื่อไปที่ลำดับงานการตรวจทานและการอนุมัติที่เลือกกำหนดได้

### <a name="summary"></a>สรุป

ข้อมูลในส่วน **สรุป** จะขึ้นอยู่กับตัวเลือกที่ HR เลือกไว้ใน **พารามิเตอร์ทรัพยากรบุคคล** บนแท็บ **ระบบบริการตนเองของผู้จัดการ** ของหน้า **พารามิเตอร์ทรัพยากรบุคคล** คุณสามารถตั้งค่าคอนฟิกตัวเลือกสำหรับการแสดงเรกคอร์ดที่หมดอายุและตำแหน่งที่เปิด การเปิดใช้งานตัวเลือกเหล่านี้จะกำหนดผู้จัดการที่สามารถมองเห็นได้ในส่วน **สรุป**

คุณสามารถตั้งค่าคอนฟิกไทล์ต่อไปนี้สำหรับผู้จัดการ:

- **ใบรับรองสำหรับทีมงานของฉันกำลังจะหมดอายุ**
- **หมายเลขรหัสที่กำลังจะหมดอายุสำหรับทีมงานของฉัน**
- **ช่วงทดลองงานที่กำลังจะหมดอายุสำหรับทีมงานของฉัน**
- **การคัดเลือกที่กำลังจะหมดอายุสำหรับทีมงานของฉัน**
- **การทดสอบที่กำลังจะหมดอายุสำหรับทีมงานของฉัน**
- **ตำแหน่งที่เปิดสำหรับรายงานโดยตรงและ/หรือแบบขยาย**
- **คำขอปิดเวลาที่ค้างอยู่สำหรับทีมงานของฉัน**
- **การประเมินทักษะของทีมงาน**
- **การวิเคราะห์ช่องว่างของทักษะ**
- **สมุดรายวันประสิทธิภาพของทีมงาน**
- **เป้าหมายประสิทธิภาพของทีมงาน**
- **การทบทวนประสิทธิภาพของทีมงาน**
- **ผู้ปฏิบัติงานที่ลาออก**

คุณสามารถตั้งค่าคอนฟิกตัวเลือกต่อไปนี้สำหรับผู้จัดการ เพื่อทำการเปลี่ยนแปลงหรือเพิ่มคำขอลางานในนามของรายงานโดยตรงของพวกเขา:

- ประกาศนียบัตร
- หลักสูตร
- สินค้าที่ให้กู้ยืม
- ทักษะ
- คำขอการลางานและการขาดงาน

### <a name="my-team-information"></a>ข้อมูลทีมงานของฉัน

**ทีมงานของฉัน** ทำให้ผู้จัดการสามารถดูและปรับปรุงรายงานโดยตรงและแบบขยายได้ ถ้าต้องการเข้าถึงรายงานแบบขยาย ให้เลือกพนักงานที่มีคำสั่ง และจากนั้น เลือก **ดูทีมงาน** บนไทล์ ตัวเลือกเดียวกันทั้งหมดจะใช้กับรายงานแบบขยายเป็นรายงานโดยตรง 

#### <a name="summary-tab"></a>แท็บสรุป

แท็บ **สรุป** แสดงมุมมองด่วนของรายงานโดยตรงของคุณ นอกจากนี้ ถ้ารายงานโดยตรงยังมีผู้ปฏิบัติงานที่รายงาน บัตรจะแสดงจำนวนของรายงานโดยตรงในส่วนด้านบน พร้อมกับปุ่ม **มุมมองทีม** ตัวเลือกด้านบนของไทล์แต่ละใบจะใช้กับพนักงานที่เลือก ตัวอย่างเช่น ถ้าคุณต้องการป้อนคำขอลางานในนามของพนักงาน ให้คุณเลือกพนักงาน แล้วเลือก **ร้องขอการหยุดพัก** 

ถ้าคุณเลือกปุ่ม **รายละเอียด** หลังจากเลือกพนักงานแล้ว ตัวเลือกดังต่อไปนี้จะแสดงขึ้น:

- **ใบรับรอง**
- **ค่าตอบแทน**
- **หลักสูตร**
- **ตรวจทาน**
- **เวลาหยุดพัก**
- **รายการที่ยืม**
- **เป้าหมายผลการปฏิบัติงาน**
- **หลักสูตรที่ลงทะเบียน**
- **ทักษะ**
- **ส่งคำติชม**

ทั้งนี้ขึ้นอยู่กับการตั้งค่าขององค์กรของคุณ คุณสามารถทำการเปลี่ยนแปลง หรือดู อย่างใดอย่างหนึ่งเท่านั้น

#### <a name="position-tab"></a>แท็บตำแหน่ง

แท็บ **ตำแหน่ง** แสดงมุมมองสรุปของพนักงานในตำแหน่งหลักของพวกเขา ชื่อ ไทล์ และแผนก จะแสดงในพื้นที่หัวข้อของไทล์แต่ละใบ ไทล์นี้รวม:

- **วันที่อายุงานของพนักงาน** - ซึ่งแสดงผลจากส่วนสรุปของผู้ปฏิบัติงานของหน้า **ผู้ปฏิบัติงาน**
- **ปีที่ให้บริการ** - ซึ่งคำนวณตามวันที่เริ่มต้นการจ้างงานของพนักงาน
- **จำนวนของตำแหน่งก่อนหน้านี้** - ซึ่งขึ้นอยู่กับประวัติตำแหน่ง การเลือกหมายเลขนี้จะเปิดมุมมองโดยละเอียดของตำแหน่งที่รับผิดชอบก่อนหน้านี้ทั้งหมด
- **วันเกิด** - เดือนและวันของวันเกิดของพนักงาน

คุณสามารถดูข้อมูลตำแหน่งสำหรับทั้งรายงานโดยตรงและแบบขยาย

#### <a name="compensation-tab"></a>แท็บค่าตอบแทน

แท็บ **ค่าตอบแทน** จะแสดงเงินเดือนประจำปีของพนักงาน ตัวระบุบริษัทจะแสดงภายใต้ยอดเงินเดือน ถ้าพนักงานมีการจ้างงานมากกว่าหนึ่งรายการและได้รับการชำระเงินจากนิติบุคคลหลายราย พนักงานจะมีแผนค่าตอบแทนหลายแผน เมื่อต้องการดูแผนค่าตอบแทนทั้งหมดในนิติบุคคลโดยไม่มีการสลับบริษัท คุณต้องเปิดใช้งานข้ามค่าตอบแทนภายใต้ **ทรัพยากรบุคคล > พารามิเตอร์ที่ใช้ร่วมกัน > การเข้าถึงขั้นสูง > เปิดใช้งานค่าตอบแทนของบริษัทระหว่างบริษัท**

ถ้าต้องการดูประวัติค่าตอบแทน ให้เลือก **ยอดเงินเดือน** เพื่อเปิดหน้า **รายละเอียด** เฉพาะเรกคอร์ดค่าตอบแทนคงที่และค่าตอบแทนผันแปรปัจจุบันและในอดีตเท่านั้นที่แสดงในหน้า **ค่าตอบแทน** ถ้าพนักงานมีการจ้างงานมากกว่าหนึ่งคน คุณสามารถสลับไปมาระหว่างบริษัทเพื่อดูประวัติค่าตอบแทนในบริษัทแต่ละแห่ง หรือเปิดใช้งานค่าตอบแทนของระหว่างบริษัทใน **พารามิเตอร์ที่ใช้ร่วมกันของฝ่ายทรัพยากรบุคคล** เพื่อดูแผนค่าตอบแทนทั้งหมด

คุณสามารถดูค่าตอบแทนสำหรับทั้งรายงานโดยตรงและแบบขยาย

#### <a name="leave-and-absence-tab"></a>แท็บการลางานและการหยุดงาน

แท็บ **การลางานและการขาดงาน** จะแสดงยอดดุลสูงสุดสำหรับพนักงานที่มีกิจกรรม ถ้าต้องการดำเนินการหรือดูรายการกิจกรรมทั้งหมด ให้เลือก **รายละเอียด** แล้วจากนั้น เลือก **เลืเวลาหยุดพัก** บนหน้า **เวลาหยุดพัก** คุณสามารถดูยอดดุล คำขอ เวลาหยุดพักที่อนุมัติแล้ว และยอดดุลการคาดการณ์ เพื่อช่วยให้พนักงานบริหารเวลาได้ดียิ่งขึ้น ทั้งนี้ขึ้นอยู่กับการตั้งค่าขององค์กรของคุณ คุณยังสามารถร้องขอการหยุดพักสำหรับรายงานโดยตรงและแบบขยายของคุณได้ด้วย

#### <a name="performance-goals-tab"></a>แท็บเป้าหมายประสิทธิภาพ

แท็บ **เป้าหมายประสิทธิภาพ** จะสรุปเป้าหมายประสิทธิภาพโดยเรียงตามสถานะ เลือกหมายเลขสำหรับสถานะหรือเลือกเป้าหมายประสิทธิภาพจาก **รายละเอียด** เพื่อดูเป้าหมายทั้งหมดสำหรับพนักงาน ผู้จัดการและพนักงานสามารถปรับปรุงเป้าหมายได้ตามความจำเป็นในช่วงเวลาของเป้าหมาย

ผู้จัดการสามารถเห็นเป้าหมายทั้งหมดสำหรับทีมงานของพวกเขาผ่านไทล์ **เป้าหมายประสิทธิภาพของทีมงาน** ในส่วน **สรุป** ของ **ทีมงานของฉัน**

#### <a name="reviews-tab"></a>แท็บการตรวจทาน

แท็บ **การตรวจทาน** สรุปการตรวจทานที่พนักงานมีในแต่ละสถานะ: **กำลังดำเนินการอยู่** **พร้อมสำหรับการตรวจทาน** และ **การตรวจทานครั้งสุดท้าย** ถ้าต้องการเข้าถึงการตรวจทานของพนักงาน ให้เลือกปุ่ม **รายละเอียด** แล้วเลือกการตรวจทานเพื่อทำงานร่วมกัน โดยขึ้นอยู่กับตำแหน่งที่การตรวจทานอยู่ภายในกระบวนการลำดับงาน คุณสามารถดูได้ว่าการตรวจทานพร้อมใช้งานสำหรับการปรับปรุงหรือไม่ 

คุณสามารถเห็นการตรวจทานทั้งหมดสำหรับทีมงานของคุณผ่านไทล์ **การตรวจทานประสิทธิภาพของทีมงาน** ในส่วน **สรุป** ของ **ทีมงานของฉัน**
