# <a name="job-detail-api"></a>API ของรายละเอียดงาน

หัวข้อนี้แสดงรายละเอียดและตัวอย่างการสอบถามสำหรับเอนทิตีรายละเอียดงานใน Dynamics 365 Human Resources

## <a name="properties"></a>คุณสมบัติ

| คุณสมบัติ<br>**ชื่อทางกายภาพ**<br>**_ชนิด_** | ใช้ | คำอธิบาย |
| --- | --- | --- |
| **รหัสงาน**<br>mshr_jobid<br>*สตริง* | อ่านอย่างเดียว<br>ต้องระบุ | รหัสเฉพาะของงาน |
| **ขีดจำกัดต่ำสุดของราคาตลาด**<br>mshr_marketpricelowthreshold<br>*ทศนิยม* | | ค่า GUID ที่ระบบสร้างขึ้นเพื่อระบุถึงตำแหน่งเฉพาะ  |
