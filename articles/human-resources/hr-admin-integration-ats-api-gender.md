# <a name="gender"></a>เพศ


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

บทความนี้อธิบายชุดตัวเลือกเพศสำหรับ Dynamics 365 Human Resources

ชื่อทางกายภาพ: mshr_hcmpersongender

การแจงนับนี้จะให้ชุดตัวเลือกของเพศสำหรับผู้สมัคร ซึ่งพร้อมใช้งานในชุดตัวเลือก mshr_hcmpersongender

| มูลค่า | ป้ายชื่อ | คำอธิบาย |
| --- | --- | --- |
| 200000000 | None | ไม่ได้ระบุเพศ |
| 200000001 | เพศชาย | เพศชาย |
| 200000002 | เพศหญิง | เพศหญิง |
| 200000003 | ไม่เจาะจง | การเลือกเพศที่ไม่ระบุเพศ |
