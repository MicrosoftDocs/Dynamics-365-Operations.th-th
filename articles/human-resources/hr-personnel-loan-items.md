# <a name="manage-items-that-are-lent-to-workers"></a><span data-ttu-id="d8948-103">จัดการสินค้าที่ให้ยืมแก่ผู้ปฏิบัติงาน</span><span class="sxs-lookup"><span data-stu-id="d8948-103">Manage items that are lent to workers</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="d8948-104">สินค้าที่กู้ยืมมีเรกคอร์ดซึ่งช่วยให้ผู้จัดการติดตามสินค้าทางกายภาพที่บริษัทของคุณให้ผู้ปฏิบัติงานยืม</span><span class="sxs-lookup"><span data-stu-id="d8948-104">Loan items are records that help managers track the physical items that your company lends to its workers.</span></span> 

<span data-ttu-id="d8948-105">เนื้อหาต่อไปนี้แสดงรายการตัวอย่างของสินค้าที่บริษัทอาจให้ผู้ปฏิบัติงานยืม:</span><span class="sxs-lookup"><span data-stu-id="d8948-105">The following points list examples of items that a company might lend to workers:</span></span>
-   <span data-ttu-id="d8948-106">โทรศัพท์มือถือ</span><span class="sxs-lookup"><span data-stu-id="d8948-106">Mobile telephones</span></span>
-   <span data-ttu-id="d8948-107">รถยนต์</span><span class="sxs-lookup"><span data-stu-id="d8948-107">Automobiles</span></span>
-   <span data-ttu-id="d8948-108">อุปกรณ์คอมพิวเตอร์</span><span class="sxs-lookup"><span data-stu-id="d8948-108">Computer equipment</span></span>

<span data-ttu-id="d8948-109">สินค้าที่มีอยู่จริงแต่ละรายการต้องมีสินค้าให้กู้ยืมที่สอดคล้องกัน</span><span class="sxs-lookup"><span data-stu-id="d8948-109">Each physical item must have a corresponding loan item.</span></span> <span data-ttu-id="d8948-110">แต่ละเรกคอร์ดสินค้ากู้ยืมควรอธิบายถึงสิ่งที่ให้ยืม ใครที่รับผิดชอบสำหรับการกู้ยืม และจำนวนวันที่สินค้าสามารถให้กู้ยืมแก่ผู้ปฏิบัติงาน</span><span class="sxs-lookup"><span data-stu-id="d8948-110">Each loan item record should describe what is being loaned, who is responsible for the loan, and the number of days the item can loaned to a worker.</span></span> <span data-ttu-id="d8948-111">คุณสามารถสร้างรายการกู้ยืมหลายรายการ สำหรับรายการต่าง ๆ เช่น กุญแจ บัตรผ่าน หรือ เครื่องแบบ ในเวลาเดียวกัน</span><span class="sxs-lookup"><span data-stu-id="d8948-111">You can create multiple loan items, for items such as keys, access cards or uniforms, at the same time.</span></span> 

<span data-ttu-id="d8948-112">เมื่อให้กู้ยืมสินค้า ป้อนวันที่มีการกู้ยืมสินค้า และวันส่งคืนที่วางแผนไว้</span><span class="sxs-lookup"><span data-stu-id="d8948-112">When loaning an item, enter the date that the item was loaned, and the planned return date.</span></span> <span data-ttu-id="d8948-113">เมื่อมีการส่งคืนสินค้า ป้อนวันส่งคืนจริง</span><span class="sxs-lookup"><span data-stu-id="d8948-113">When the item is returned, enter the actual return date.</span></span>

<span data-ttu-id="d8948-114">พนักงานสามารถดูเรกคอร์ดของสินค้าที่กู้ยืมได้โดยใช้พื้นที่ทำงานการบริการตนเองของพนักงานได้</span><span class="sxs-lookup"><span data-stu-id="d8948-114">Employees can view the records of the items that have been loaned to them using the Employee self-service workspace.</span></span> <span data-ttu-id="d8948-115">พวกเขาสามารถแก้ไขเรกคอร์ดที่มีอยู่ หรือป้อนสินค้ากู้ยืมใหม่ถ้าพวกเขาได้รับสินค้าทางกายภาพที่เพิ่มเติม</span><span class="sxs-lookup"><span data-stu-id="d8948-115">They can also edit the existing records or enter new loan items, if they've received additional physical items.</span></span>  <span data-ttu-id="d8948-116">ลำดับงานสามารถถูกตั้งค่าเป็นการเปลี่ยนแปลงเส้นทางของสินค้าที่ให้กู้ยืมใหม่หรือที่มีอยู่ภายในกระบวนการอนุมัติ</span><span class="sxs-lookup"><span data-stu-id="d8948-116">Workflow can be set up to route changes to new or existing loan items through an approval process.</span></span> 

<span data-ttu-id="d8948-117">ผู้จัดการสามารถดูสินค้าที่ให้ยืมสำหรับรายงานโดยตรงของพวกเขา</span><span class="sxs-lookup"><span data-stu-id="d8948-117">Managers can view loaned items for their direct reports.</span></span> <span data-ttu-id="d8948-118">พวกเขาจะได้รับสิทธิ์ในการเพิ่มสินค้าที่ให้กู้ยืมใหม่ในนามของพนักงานของพวกเขา</span><span class="sxs-lookup"><span data-stu-id="d8948-118">They can also be granted permission to add new loan items on behalf of their employees.</span></span>
