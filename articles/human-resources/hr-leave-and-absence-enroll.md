# <a name="assign-workers-to-a-leave-plan"></a>กำหนดผู้ปฏิบัติงานให้กับแผนการลางาน


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

หลังจากที่คุณสร้างแผนการลางานและขาดงานใน Dynamics 365 Human Resources ให้กำหนดให้กับผู้ปฏิบัติงาน

## <a name="assign-a-plan-to-a-group-of-employees"></a>กำหนดแผนให้กับกลุ่มพนักงาน

1. บนหน้า **การลางานและขาดงาน** ให้เลือกแผนที่คุณต้องการกำหนด

2. ให้เลือก **กำหนดให้พนักงาน**

3. ปรับปรุงการตั้งค่าต่อไปนี้ตามความเหมาะสม:

   - **ละทะเบียนวันที่เริ่มต้นการจ้างงาน**
   - **วันที่ลงทะเบียน**
   - **วันที่สิ้นสุด**
   - **ระดับพื้นฐาน**

4. ภายใต้ **ผู้ปฏิบัติงานที่จะลงทะเบียน** ให้เลือกงานทั้งหมดที่จะกำหนดให้กับแผน

5. เลือก **กำหนดแผน** ที่ด้านล่างของหน้า

## <a name="assign-a-plan-to-an-employee"></a>การกำหนดแผนให้กับพนักงาน

1. บนเรกคอร์ดของพนักงาน ให้เลือก **ลา**

2. ให้เลือก **การลางานและขาดงาน**

3. เลือก **ใหม่**

4. ป้อนวันที่ลงทะเบียน และเลือกแผนการลางาน
