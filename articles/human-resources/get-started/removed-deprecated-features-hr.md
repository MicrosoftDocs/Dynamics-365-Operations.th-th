# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>คุณลักษณะที่ลบออกหรือไม่สนับสนุนใน Dynamics 365 Human resources

บทความนี้อธิบายคุณลักษณะที่ถูกลบออก หรือที่ถูกวางแผนไว้สำหรับการลบจาก Microsoft Dynamics 365 Human resources

- คุณลักษณะที่ *ถูกลบ* จะไม่พร้อมใช้งานในผลิตภัณฑ์อีกต่อไป
- คุณลักษณะที่ *ถูกยกเลิกการใช้* ไม่ได้อยู่ในการพัฒนาที่ใช้งานอยู่ และอาจถูกลบออกในการปรับปรุงในอนาคต

รายการนี้มีไว้เพื่อช่วยคุณพิจารณาการลบออกและการยกเลิกการใช้งานเหล่านี้สำหรับการวางแผนของคุณเอง

## <a name="features-removed-or-deprecated-in-human-resources"></a>คุณลักษณะที่ลบออกหรือไม่สนับสนุนในทรัพยากรบุคคล 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>การสร้างโครงการ LCS ใหม่ในทรัพยากรบุคคล (แบบสแตนด์อโลน) 

[เตรียมใช้งานทรัพยากรบุคคล](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **เหตุผลสำหรับการยกเลิกการใช้/การลบ** | แทนที่ด้วยการสร้างโครงการในโครงสร้างพื้นฐานของการเงินและการดําเนินงาน หากต้องการเรียนรู้เพิ่มเติมเกี่ยวกับการผสานโครงสร้างพื้นฐาน ให้ดูที่ [FAQ การผสานโครงสร้างพื้นฐานของ Dynamics 365 Human resource](/hr-infrastructure-merge-faq) |
| **แทนที่ด้วยลักษณะการทำงานอื่นหรือไม่**   | ใช่ |
| **พื้นที่ของผลิตภัณฑ์ที่ได้รับผล**         | ใบคำขอเปิด |
| **ตัวเลือกการปรับใช้**              | ทั้งหมด |
| **สถานะ**                         | ไม่ได้รับการสนับสนุน: ตั้งแต่กรกฎาคม 2022 เราไม่สนับสนุนการสร้างโครงการใหม่ในทรัพยากรบุคคล (แบบสแตนด์อโลน) อีกต่อไป |

| &nbsp; | &nbsp; |
|------------|--------------------|
