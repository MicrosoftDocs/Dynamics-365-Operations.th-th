# <a name="configure-waiting-days"></a>ตั้งค่าคอนฟิกจำนวนวันที่รอ

ใน Microsoft Dynamics 365 Human Resources วันที่กำลังรอสร้างเหตุการณ์สำคัญเป็นรายเดือนเพื่อใช้สำหรับแผนสวัสดิการ ตัวอย่างเช่น วันที่รอ 15 แทนวันที่ 15 ของเดือนแต่ละเดือน 

1. ในพื้นที่ทำงาน **การจัดการสวัสดิการ** ภายใต้ **การตั้งค่า** ให้เลือก **วันที่กำลังรอ**

2. เลือก **ใหม่**

3. ระบุค่าสำหรับฟิลด์ต่อไปนี้

   | ฟิลด์ | คำอธิบาย |
   | --- | --- |
   | **จำนวนวันที่รอ** | ตัวระบุเฉพาะสำหรับวันที่กำลังรอ |
   | **คำอธิบาย** | คำอธิบายสตริงของวันที่กำลังรอ |
   | **วัน** | วันของเดือนที่ระบบควรตั้งค่าวันที่ของรอบระยะเวลาการรอ |
   
4. เลือก **บันทึก**
