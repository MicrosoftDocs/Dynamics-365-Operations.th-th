# <a name="configure-future-life-events"></a>ตั้งค่าคอนฟิกเหตุการณ์ของชีวิตในอนาคต

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]
[!include [banner](../includes/preview-banner.md)]

คุณสามารถจัดกำหนดการเหตุการณ์ของชีวิตอนาคตใน Dynamics 365 Human Resources

1. ในพื้นที่ทำงาน **การจัดการสวัสดิการ** ภายใต้ **การตั้งค่า** ให้เลือก **เหตุการณ์ของชีวิตในอนาคต**

2. เลือก **ใหม่**

3. ระบุค่าสำหรับฟิลด์ต่อไปนี้

   | ฟิลด์ | คำอธิบาย |
   | --- | --- |
   | วันที่เหตุการณ์ของชีวิต | ระบบประมวลผลเหตุการณ์ทั้งหมดในระหว่างรอบระยะเวลาของการลงทะเบียนที่เกิดขึ้นจนถึงวันนี้ |
   | ลงบันทึกเหตุการณ์ของชีวิตแล้ว | วันที่และเวลาเมื่อมีการบันทึกล็อกเหตุการณ์ของชีวิต |
   | ชนิดการล็อก | แสดงว่าการดำเนินการเป็นอย่างใดอย่างหนึ่งต่อไปนี้:</br></br>- **การอัพเดต** – การเปลี่ยนแปลงที่เกิดขึ้นกับเรกคอร์ดที่ติดตามเหตุการณ์ของชีวิต</br></br>- **การแทรก** – การสร้างเรกคอร์ดเหตุการณ์ของชีวิตใหม่ |
   | รหัสชนิดเหตุการณ์ของชีวิต | ตัวระบุเฉพาะของชนิดเหตุการณ์ของชีวิต |
   | ชนิดเหตุการณ์ของชีวิต | ตัวเร่งเพื่ออัปเดตการลงทะเบียนสวัสดิการของพนักงาน สำหรับรายละเอียดเพิ่มเติม ให้ดูที่ส่วนทริกเกอร์เหตุการณ์ของชีวิต |
   | สถานะ | มีการประมวลผลเหตุการณ์ของชีวิตแล้วหรือไม่ |
   | รายการ | หมายเลขบรรทัดของเหตุการณ์ของชีวิตในอนาคต |

4. เลือก **บันทึก** 

คุณสามารถลบเหตุการณ์ของชีวิตอนาคต ถ้ามีการลบเหตุการณ์อายุการใช้งานในอนาคตที่ประมวลผลแล้ว เรกคอร์ดในอนาคตจะถูกลบด้วย
