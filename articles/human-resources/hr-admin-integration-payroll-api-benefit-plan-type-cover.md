# <a name="benefit-plan-type-cover"></a>ใบปะหน้าชนิดแผนสวัสดิการ


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

บทความนี้อธิบายชุดตัวเลือกความครอบคลุมของชนิดแผนสวัสดิการสำหรับ Dynamics 365 Human Resources

## <a name="option-set-process-type"></a>ชุดตัวเลือก: ชนิดของกระบวนการ

ชื่อทางกายภาพ: mshr_plantypecode

การแจงนับนี้จะให้การกำหนดตัวเลือกสำหรับชนิดของกระบวนการในเอนทิตีเสมือนของ Human Resources

| มูลค่า | ป้ายชื่อ | คำอธิบาย |
| --- | --- | --- |
| 200000000 | การรักษาพยาบาล | ลาป่วย |
| 200000001 | ทันตกรรม | ทันตกรรม |
| 200000002 | การมองเห็น | การมองเห็น |
| 200000003 | ประหยัดได้ | ประหยัด |
| 200000004 | บัญชีการใช้จ่ายที่ยืดหยุ่น | บัญชีการใช้จ่ายแบบยืดหยุ่น |
| 200000005 | ชีวิต | ชีวิต |
| 200000006 | AccidentalDeathDismemberment | AccidentalDeathDismemberment |
| 200000007 | อื่นๆ | อื่นๆ|
| 200000008 | LongTermDisability | LongTermDisability |
| 200000009 | ShortTermDisability | ShortTermDisability |
