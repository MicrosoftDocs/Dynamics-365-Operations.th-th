# <a name="award-type"></a>ชนิดรางวัล


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

หัวข้อนี้อธิบายการกำหนดตัวเลือกชนิดรางวัลสำหรับ Dynamics 365 Human Resources

## <a name="option-set-award-type"></a>ชุดตัวเลือก: ชนิดรางวัล

ชื่อทางกายภาพ: mshr_HrmCompVarAwardEmplType

การแจงนับนี้จะให้การกำหนดตัวเลือกสำหรับชนิดของค่าตอบแทนผันแปรในเอนทิตีเสมือนของทรัพยากรบุคคล

| มูลค่า | ป้ายชื่อ | คำอธิบาย |
| --- | --- | --- |
| 200000000 | รางวัลทั่วไป | รางวัลทั่วไป |
| 200000001 | จำนวนเงินค่าจ้างคงที่รวมทั้งหมด | จำนวนเงินค่าจ้างคงที่รวมทั้งหมด |
