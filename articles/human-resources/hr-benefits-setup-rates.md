# <a name="configure-rates"></a>ตั้งค่าคอนฟิกอัตรา


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

อัตรากำหนดว่านายจ้างและพนักงานมีส่วนร่วมสำหรับสวัสดิการมากน้อยเพียงใด ค่าอาจเป็นจำนวนหรือจำนวนเครดิตแบบยืดหยุ่นเวลาก็ได้ ขึ้นอยู่กับการตั้งค่าคอนฟิกของคุณ

ใช้อัตราเพื่อกำหนดว่าพนักงานและนายจ้างที่ชำระค่าจ้างสำหรับแต่ละสวัสดิการเท่าไหร่ ขึ้นอยู่กับปัจจัยหลายอย่าง อัตราครอบคลุมจะมีผลในวันที่บังคับใช้ ดังนั้นคุณจึงสามารถจัดเก็บบันทึกประวัติของอัตราได้ 

## <a name="set-up-rates"></a>ตั้งค่าอัตรา

1. ในพื้นที่ทำงาน **การจัดการสวัสดิการ** ภายใต้ **การตั้งค่า** ให้เลือก **อัตรา**

2. เลือก **ใหม่**

3. ระบุค่าสำหรับฟิลด์ต่อไปนี้

   | ฟิลด์ | คำอธิบาย |
   | --- | --- |
   | **อัตรา** | ชื่อเฉพาะที่ระบุอัตราสวัสดิการ |
   | **คำอธิบาย** | คำอธิบายย่อเกี่ยวกับอัตราของสวัสดิการ |
   | **มีผลบังคับใช้** | วันที่เมื่ออัตรามีผลบังคับใช้ วันที่ปัจจุบันของระบบคือค่าเริ่มต้น วันที่นี้ควรเป็นหรือก่อนรอบระยะเวลาสวัสดิการของคุณ แนวทางปฏิบัติที่ดีคือกําหนดวันที่นี้จนถึงวันที่ของแผนสวัสดิการ |
   | **การหมดอายุ** | วันที่สิ้นสุดการของอัตรา 12/31/2154 (ซึ่งแสดงว่าไม่มี) เป็นค่าเริ่มต้น |
   | **ใช้ชั้น** |  ใช้ฟิลด์นี้ถ้าคุณมีตรรกะที่ต้องใช้ในการกําหนดอัตรา ตัวอย่างเช่น ถ้าอัตราต้องเพิ่มขึ้นตามอายุ ให้เลือกค่าที่นี่ เลือก **ระดับเดี่ยว** สำหรับหนึ่งระดับอัตราสวัสดิการหรือ **ระดับคู่** สำหรับสองระดับอัตราสวัสดิการ ตัวอย่างของระดับคู่ขึ้นอยู่กับเพศและอายุ หลังจากที่คุณเลือกค่าแล้ว ให้เลือก **การดําเนินการ** แล้วเลือก **อัตราระดับ** ถ้าคุณมีอัตราคงที่ที่ไม่เปลี่ยนแปลง ให้ปล่อยฟิลด์นี้ว่างไว้ |
   | **ความถี่ในการชำระเงิน** | ระบุความถี่ที่ควรชำระค่าสวัสดิการพิเศษสำหรับผู้ให้บริการสวัสดิการ อัตราที่คุณป้อนในหน้าซึ่งอธิบายต่อไปในบทความนี้ จะอิงตามความถี่ในการชําระเงินที่คุณระบุที่นี่ ตัวอย่างเช่น ถ้าคุณป้อน **รายเดือน** ในฟิลด์นี้ และคุณป้อนอัตราพนักงานเป็น **$100** จะถือว่าพนักงานต้องเสียค่าสวัสดิการ $100 ต่อเดือน อย่างไรก็ตาม พนักงานอาจได้รับค่าจ้างสองครั้งต่อเดือน ตามความถี่ในการชำระเงินสวัสดิการที่กําหนดไว้ในเรกคอร์ดพนักงาน ในกรณีนี้ เมื่อพนักงานลงชื่อเข้าใช้ **ระบบพนักงานบริการตนเอง** ยอดเงินที่พวกเขาจะชำระเป็น $50 เนื่องจากอัตราที่ **ระบบพนักงานบริการตนเอง** แสดงขึ้นกับความถี่ในการชำระของพนักงาน |
   | **การปัดเศษอัตราความถี่ในการชำระค่าจ้าง** | วิธีการปัดเศษอัตราได้แก่ มาตรฐาน การตัดให้สั้นลง ปกติ ลง และการปัดเศษขึ้น </br></br><ul><li>**มาตรฐาน** – ปัดเศษขึ้นเสมอ ตัวอย่างเช่น 10.611 จะปัดเศษเป็น 10.62 -10.231 ปัดเศษเป็น -10.23 </li><li>**ตัดให้สั้นลง** – ปัดเศษลงเสมอ ตัวอย่างเช่น 10.619 จะปัดเศษเป็น 10.61 -10.231 ปัดเศษเป็น -10.24 </li><li>**ปกติ** – ค่าทศนิยมที่ลงท้ายด้วยค่าที่มากกว่า 5 จะปัดเศษออกจากศูนย์ ค่าทศนิยมที่สิ้นสุดในหรือน้อยกว่า 4 จะปัดเศษเป็นศูนย์ ตัวอย่างเช่น 10.615 จะปัดเศษเป็น 10.62 -10.235 ปัดเศษเป็น -10.24 10.614 ปัดเศษเป็น 10.61 -10.234 ปัดเศษเป็น -10.23 </li><li>**ลง** – ปัดเศษลงศูนย์ ตัวอย่างเช่น 10.619 จะปัดเศษเป็น 10.61 -10.231 ปัดเศษเป็น -10.23 </li><li>**การปัดเศษขึ้น** – ปัดเศษขึ้นจากศูนย์ ตัวอย่างเช่น 10.619 จะปัดเศษเป็น 10.62 -10.231 ปัดเศษเป็น -10.24 |
   | **จำนวนของพนักงานที่ไม่สูบบุหรี่** | จำนวนเงินค่าธรรมเนียมของผู้ให้บริการสวัสดิการสำหรับพนักงานที่ไม่สูบบุหรี่ นี่คือจำนวนเงินที่นายจ้างชำระให้กับผู้ให้สวัสดิการ และควรยึดตามความถี่ในการชำระเงินสำหรับการตั้งค่าอัตรา |
   | **จำนวนของนายจ้างที่ไม่สูบบุหรี่** | จำนวนเงินค่าธรรมเนียมของผู้ให้บริการสวัสดิการสำหรับพนักงานที่ไม่สูบบุหรี่ นี่คือจำนวนเงินที่นายจ้างชำระให้กับผู้ให้สวัสดิการ และควรยึดตามความถี่ในการชำระเงินสำหรับการตั้งค่าอัตรา |
   | **จำนวนของพนักงานที่ผู้สูบบุหรี่** | จำนวนเงินค่าธรรมเนียมของผู้ให้บริการสวัสดิการสำหรับพนักงานที่สูบบุหรี่ นี่คือจำนวนเงินที่นายจ้างชำระให้กับผู้ให้สวัสดิการ และควรยึดตามความถี่ในการชำระเงินสำหรับการตั้งค่าอัตรา |
   | **จำนวนของนายจ้างที่สูบบุหรี่** | จำนวนเงินค่าธรรมเนียมของผู้ให้บริการสวัสดิการสำหรับพนักงานที่สูบบุหรี่ นี่คือจำนวนเงินที่นายจ้างชำระให้กับผู้ให้สวัสดิการ และควรยึดตามความถี่ในการชำระเงินสำหรับการตั้งค่าอัตรา |
   | **จำนวนเงินสำหรับการดูแล** | ยอดเงินในการดูแลที่เรียกเก็บโดยผู้ดูแลระบบของบุคคลฝ่ายที่สาม นี่คือจำนวนเงินที่นายจ้างชำระให้กับผู้ดูแลระบบของบุคคลฝ่ายที่สาม และควรยึดตามความถี่ในการชำระเงินสำหรับการตั้งค่าอัตรา |
   | **อัตราเครดิตแบบยืดหยุ่น** | จำนวนของเครดิตแบบยืดหยุ่นต้นทุนสวัสดิการ การทำเช่นนี้สามารถใช้ได้กับอัตราที่มีสำหรับแผนสวัสดิการที่เกี่ยวข้องกับโปรแกรมเครดิตแบบยืดหยุ่นเท่านั้น ถ้าคุณใช้อัตราระดับ อัตราระดับเครดิตแบบยืดหยุ่นจะถูกกำหนดไว้ในการดำเนินการ > อัตราระดับ |
   | **วันที่มีผลของการเปลี่ยนแปลง** | วันที่ที่อัตราสวัสดิการเปลี่ยนแปลงจะมีผล ระบบจะเปลี่ยนอัตราสวัสดิการและอัปเดตแผนสวัสดิการทั้งหมดโดยอัตโนมัติที่เกี่ยวข้องกับอัตรานี้ ตราบใดที่คุณดำเนินการเปลี่ยนแปลงการประมวลผลการอัปเดต อย่ากำหนดวันที่นี้ ยกเว้นว่าคุณต้องการให้ระบบอัปเดตแผนสวัสดิการของพนักงานโดยอัตโนมัติตามอัตรานี้ ซึ่งโดยปกติแล้วจะถูกสงวนไว้เพื่อการประมวลผลการเปลี่ยนแปลงอัตราในอนาคตโดยอัตโนมัติ **วันที่มีผลบังคับใช้ของการเปลี่ยนแปลง** ต้องอยู่ภายในวันที่อัตราสวัสดิการมีผลบังคับใช้และวันหมดอายุ |
   | **การเปลี่ยนแปลงอัตราเสร็จสมบูรณ์แล้ว** | กล่องกาเครื่องหมาย **การเปลี่ยนแปลงอัตราเสร็จสมบูรณ์** จะถูกเลือกโดยอัตโนมัติเมื่อการเปลี่ยนแปลงอัตราสวัสดิการเสร็จสมบูรณ์แล้วโดยการประมวลผลการอัปเดตอัตรา |

4. การติดตามและรักษาการเปลี่ยนแปลงการตั้งค่าอัตราสิทธิประโยชน์ ให้เลือก **ดำเนินการ** จากนั้นเลือก **รักษาเวอร์ชัน**

5. เลือก **บันทึก** 

## <a name="set-up-tier-rates"></a>ตั้งค่าอัตราระดับ

คุณสามารถใช้อัตราระดับในการตั้งค่าอัตราของคุณได้ถ้าอัตราแตกต่างกันไปขึ้นอยู่กับปัจจัยต่างๆ ตัวอย่างเช่น คุณสามารถตั้งค่าคอนฟิกอัตราระดับที่จะบอกว่า ถ้าอายุของคุณสูงสุดถึง 34.99 แล้วจำนวนที่ไม่สูบบุหรี่คือ 2  ถ้าอายุของคุณสูงถึง 39.99 แล้วจำนวนที่ไม่สูบบุหรี่คือ 3

นอกจากนี้คุณยังสามารถใช้ระดับคู่ได้ ถ้าคุณเลือก **ระดับคู่** สำหรับค่า **ระดับที่ใช้** ในฟอร์ม **ตั้งค่าอัตรา** คุณสามารถกำหนดอัตราโดยใช้สองมิติ ตัวอย่างเช่น คุณสามารถตั้งค่าคอนฟิกระบบระดับที่คู่จะบอกว่า ถ้าคุณเป็นเพศชายและอายุของคุณสูงสุดถึง 34.99 แล้วจำนวนที่ไม่สูบบุหรี่คือ 2  ถ้าคุณคือเพศชาย และอายุของคุณสูงถึง 39.99 แล้วจำนวนที่ไม่สูบบุหรี่คือ 3 ถ้าคุณคือเพศหญิง และอายุของคุณสูงถึง 34.99 แล้วจำนวนที่ไม่สูบบุหรี่คือ 1.8 ถ้าคุณคือเพศหญิง และอายุของคุณสูงถึง 39.99 แล้วจำนวนที่ไม่สูบบุหรี่คือ 2.8

> [!IMPORTANT]
> บนเรกคอร์ดผู้ปฏิบัติงาน ตัวเลือกภายใต้ **ข้อมูลส่วนบุคคล** มีตัวเลือกที่ใช้เพื่อบ่งชี้ว่าพนักงานเป็นผู้สูบบุหรี่หรือไม่ หากบันทึกพนักงานเป็นผู้สูบบุหรี่ จะใช้อัตราผู้สูบบุหรี่ (ข้อบ่งชี้ของผู้สูบบุหรี่จะไม่แสดงให้พนักงานเห็น)
   
1. ในพื้นที่ทำงาน **การจัดการสวัสดิการ** ภายใต้ **การตั้งค่า** ให้เลือก **อัตรา**

2. เลือกอัตราหนึ่งรายการขึ้นไปจากรายการ เลือก **การดำเนินการ** แล้วเลือก **ระดับอัตรา**

3. เลือก **ใหม่**

3. ระบุค่าสำหรับฟิลด์ต่อไปนี้

   | ฟิลด์ | คำอธิบาย |
   | --- | --- | 
   | **คำอธิบาย** | ค่าฟิลด์ **คำอธิบาย** ใช้จากคำอธิบายในเรกคอร์ดการตั้งค่าอัตรา การทำเช่นนี้จะช่วยให้คุณระบุว่าการตั้งค่าอัตราที่จะเชื่อมโยงกับอัตราระดับ |
   | **รหัสชั้น** | เลือกรหัสระดับ รหัสระดับถูกกำหนดในหน้า **รหัสระดับ** ระบบจะแสดงคำอธิบายของรหัสระดับในกริดทางด้านซ้ายโดยอัตโนมัติ |
   | **ชนิดชั้น** | ระบุฟิลด์ที่ควรใช้เป็นเงื่อนไขการเลือกสำหรับกระบวนการคำนวณอัตราระดับ ตัวอย่างเช่น:</br></br><ul><li>ถ้ามีการใช้ **อายุ** ระบบจะใช้วันเกิดของพนักงานในกระบวนการคำนวณสวัสดิการ</li><li>ถ้ามีการใช้ **เงินเดือน** ระบบจะใช้เงินสวัสดิการรายปีของพนักงานในกระบวนการคำนวณสวัสดิการ</li><li>ถ้ามีการใช้ **ชนิดงาน** เรกคอร์ดข้อมูลตำแหน่งงานปัจจุบันของพนักงานจะใช้เพื่อกำหนดชนิดงานโดยเรกคอร์ดงานที่เชื่อมโยงกับตำแหน่ง</li></ul></br></br>ชนิดของเทียร์ได้แก่ **อายุ**, **เงินเดือน**, **กายภาพ**, **เพศ**, **เทียบเท่าเต็มเวลา**, **ชนิดงาน**, **ภูมิภาคค่าตอบแทน**, และ **ระดับ** | 
   | **ระดับ** | ค่าที่จะใช้กับชนิดของระดับในระหว่างกระบวนการคำนวณอัตราสวัสดิการ ตัวอย่างเช่น:</br></br><ul><li>ถ้าชนิดของระดับเป็น **อายุ** นี่อาจจะเป็นค่าอายุ</li><li>ถ้าชนิดของระดับเป็น **เงินเดือน** นี่อาจจะเป็นจำนวนเงินเดือน</li><li> ถ้าชนิดของระดับเป็น **ประเภทงาน** นี่อาจจะเป็นประเภทงาน</li></ul></br></br>ด้วยชนิดระดับของ **อายุ** หรือ **เงินเดือน** ค่าในฟิลด์ **ระดับ** จะแสดงถึงขอบเขตล่างของระดับ เมื่อประเภทของระดับคือ **ประเภทงาน** วิธีการจับคู่ที่ตรงกันจะใช้ในระหว่างการเลือกอัตราระดับ |
   | **ชนิดการคำนวณ** | ระบุว่าจะใช้ยอดเงินในฟิลด์ยอดเงินที่คำนวณอย่างไร และการคำนวณทางคณิตศาสตร์ใดที่จะดำเนินการ ถ้าจำเป็น ถ้าชนิดการคำนวณเป็นยอดเงินคงที่ ฟิลด์ยอดเงินจะใช้ตามที่เป็นอยู่ ถ้าชนิดการคำนวณเป็นต่อจำนวนหรือครอบคลุมของเงินเดือน ยอดการคำนวณและทิศทางการคำนวณจะใช้ในการคำนวณทางคณิตศาสตร์</br></br>ถ้าชนิดการคำนวณเป็นต่อจำนวนของเงินเดือน สมการคณิตศาสตร์ต่อไปนี้จะถูกใช้:</br></br>เงินเดือนสวัสดิการประจำปีหารด้วยยอดเงินการคำนวณ (การปัดเศษขึ้นหรือลง) คูณด้วยจำนวนคนสูบบุหรี่หรือไม่สูบบุหรี่ของพนักงานหรือนายจ้าง</br></br>ถ้าชนิดการคำนวณเป็นต่อจำนวนของความครอบคลุม สมการคณิตศาสตร์ต่อไปนี้จะถูกใช้:</br></br>เงินครอบคลุมหารด้วยยอดเงินการคำนวณ (การปัดเศษขึ้นหรือลง) คูณด้วยจำนวนคนสูบบุหรี่หรือไม่สูบบุหรี่ของพนักงานหรือนายจ้าง</br></br>ในการคำนวณทั้งสอง ทิศทางการคำนวณจะใช้เพื่อกำหนดว่าจะปัดเศษยอดเงินเดือนหรือยอดเงินความครอบคลุมของสวัสดิการรายปีหารด้วยยอดเงินการคำนวณขึ้นหรือลง |
   | **จำนวนเงินในการคำนวณ** | จำนวนที่จะใช้ระหว่างกระบวนการคำนวณอัตราสวัสดิการ ยอดเงินนี้จะเป็นตัวหารระหว่างการคำนวณทางคณิตศาสตร์ของอัตราระดับ |
   | **ทิศทางการคำนวณ** | ทิศทางที่จำนวนผลลัพธ์ได้ถูกคำนวณควรถูกปัดเศษ ระบบสนับสนุนทิศทางการคำนวณสามประการดังนี้: ว่างเปล่า (วิธีการที่แน่นอน) **เพิ่ม** และ **ลด**</br></br><ul><li>ถ้าว่างเปล่า ระบบจะใช้การคำนวณที่แน่นอนของจำนวนเงินเดือน/ความครอบคลุมหารด้วยยอดเงินการคำนวณ ถ้าค่านี้มีเศษส่วน ค่านี้จะใช้ในการคำนวณ</li><li>ถ้า **เพิ่มขึ้น** การคำนวณทางคณิตศาสตร์จะเพิ่มจำนวนเงินเดือน/ความครอบคลุมหารด้วยยอดเงินการคำนวณเป็นจำนวนเต็มถัดไป ซึ่งหมายความว่า 12.25 จะเพิ่มขึ้นเป็น 13</li><li>ถ้า **ลดลง** การคำนวณทางคณิตศาสตร์จะลดจำนวนเงินเดือน/ความครอบคลุมหารด้วยยอดเงินการคำนวณเป็นจำนวนเต็มในขณะนี้ ซึ่งหมายความว่า 12.25 จะลดลงเป็น 12</li></ul> |
   | **จำนวนของพนักงานที่ไม่สูบบุหรี่** | จำนวนเงินค่าธรรมเนียมของผู้ให้บริการสวัสดิการสำหรับพนักงานที่ไม่สูบบุหรี่ นี่คือจำนวนเงินที่นายจ้างชำระให้กับผู้ให้สวัสดิการ และควรยึดตามความถี่ในการชำระเงินสำหรับการตั้งค่าอัตรา |
   | **จำนวนของนายจ้างที่ไม่สูบบุหรี่** | จำนวนเงินค่าธรรมเนียมของผู้ให้บริการสวัสดิการสำหรับพนักงานที่ไม่สูบบุหรี่ นี่คือจำนวนเงินที่นายจ้างชำระให้กับผู้ให้สวัสดิการ และควรยึดตามความถี่ในการชำระเงินสำหรับการตั้งค่าอัตรา |
   | **จำนวนของพนักงานที่ผู้สูบบุหรี่** | จำนวนเงินค่าธรรมเนียมของผู้ให้บริการสวัสดิการสำหรับพนักงานที่ไม่สูบบุหรี่ นี่คือจำนวนเงินที่นายจ้างชำระให้กับผู้ให้สวัสดิการ และควรยึดตามความถี่ในการชำระเงินสำหรับการตั้งค่าอัตรา |
   | **จำนวนของนายจ้างที่สูบบุหรี่** | จำนวนเงินค่าธรรมเนียมของผู้ให้บริการสวัสดิการสำหรับพนักงานที่ไม่สูบบุหรี่ นี่คือจำนวนเงินที่นายจ้างชำระให้กับผู้ให้สวัสดิการ และควรยึดตามความถี่ในการชำระเงินสำหรับการตั้งค่าอัตรา |
   | **จำนวนเงินสำหรับการดูแล** | ยอดเงินในการดูแลที่เรียกเก็บโดยผู้ดูแลระบบของบุคคลฝ่ายที่สาม นี่คือจำนวนเงินที่นายจ้างชำระให้กับผู้ดูแลระบบของบุคคลฝ่ายที่สาม และควรยึดตามความถี่ในการชำระเงินสำหรับการตั้งค่าอัตรา |
   | **อัตราผู้ไม่สูบบุหรี่สำหรับเครดิตแบบยืดหยุ่น** | จำนวนของต้นทุนสวัสดิการของเครดิตแบบยืดหยุ่น ตามการคำนวณที่กำหนดสำหรับระดับชั้นสำหรับผู้ไม่สูบบุหรี่ ตัวอย่างเช่น ถ้าชนิดการคำนวณเป็น **ต่อยอดเงินของความครอบคลุม** ยอดเงินที่คำนวณได้คือ 10,000 และอัตราเครดิตแบบยืดหยุ่นคือ 6 ซึ่งหมายความว่าพนักงานที่ไม่สูบบุหรี่เลือกความครอบคลุม $10,000 จะมีต้นทุนคือเครดิตแบบยืดหยุ่น 6 เครดิต ถ้ามีการเลือก $20,000 ของความครอบคลุม จะมีต้นทุนเครดตแบบยืดหยุ่น 12 เครดิต |
   | **อัตราผู้สูบบุหรี่สำหรับเครดิตแบบยืดหยุ่น** | จำนวนของต้นทุนสวัสดิการของเครดิตแบบยืดหยุ่น ตามการคำนวณที่กำหนดสำหรับระดับชั้นสำหรับผู้ที่สูบบุหรี่ |
