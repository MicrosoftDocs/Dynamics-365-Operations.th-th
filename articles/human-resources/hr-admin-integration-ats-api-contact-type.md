# <a name="contact-type"></a>ชนิดผู้ติดต่อ


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

บทความนี้อธิบายการกำหนดตัวเลือกชนิดผู้ติดต่อสำหรับ Dynamics 365 Human Resources

ชื่อทางกายภาพ: mshr_logisticselectronicaddressmethodtype

การแจงหมายเลขนี้จะให้ชุดตัวเลือกของชนิดผู้ติดต่อของผู้สมัคร 

| มูลค่า | ป้ายชื่อ | คำอธิบาย |
| --- | --- | --- |
| 200000000 | None | ไม่มีชนิดที่เลือกไว้ |
| 200000001 | โทรศัพท์ | หมายเลขโทรศัพท์ |
| 200000002 | ที่อยู่อีเมล | ที่อยู่อีเมล |
| 200000003 | URL | URL ของเว็บไซต์ |
| 200000004 | เทเล็กซ์ | หมายเลขเทเล็กซ์ |
| 200000005 | โทรสาร | หมายเลขโทรสาร |
| 200000006 | Facebook | บัญชี Facebook ระบุโดยรหัสผู้ใช้ |
| 200000007 | Twitter | บัญชี Twitter ระบุโดย @username |
| 200000008 | LinkedIn | บัญชี LinkedIn ระบุโดยชื่อผู้ใช้ |
