# <a name="blank-yes-no"></a>ว่าง ใช่ ไม่ใช่

หัวข้อนี้อธิบายชุดตัวเลือกว่าง ใช่ ไม่ใช่สำหรับ Dynamics 365 Human Resources

ชื่อทางกายภาพ: mshr_hrmblankyesno

การแจงนับนี้จะให้ชุดตัวเลือกสำหรับคุณสมบัติ ใช่/ไม่ใช่ ที่อนุญาตให้มีช่องว่างด้วย

| มูลค่า | ป้ายชื่อ | คำอธิบาย |
| --- | --- | --- |
| 200000000 | ว่างเปล่า | ไม่ได้เลือกค่า |
| 200000001 | ใช่ | ใช่ |
| 200000002 | หมายเลข | ลำดับที่ |
