# <a name="configure-employee-self-service"></a>ตั้งค่าคอนฟิกระบบบริการตนเองของพนักงาน

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

ใน Microsoft Dynamics 365 Human Resources คุณสามารถตั้งค่าคอนฟิกของไทล์สำหรับการนำทางระดับบนสุดในบริการตนเองของพนักงาน ไทล์แผนสิทธิประโยชน์ครอบคลุมผู้ใช้โดยตรงสำหรับแผนสิทธิประโยชน์ที่พวกเขามีสิทธิ์

## <a name="set-up-a-benefit-plans-tile"></a>ตั้งค่าไทล์ของแผนสิทธิประโยชน์

1. ในพื้นที่ทำงาน **การจัดการสิทธิประโยชน์** ภายใต้ **การตั้งค่า** ให้เลือก **ค่าพารามิเตอร์การบริการตนเองของพนักงาน**

2. เลือกแท็บ **การตั้งค่าไทล์ของแผนสิทธิประโยชน์** จากนั้นเลือก **ใหม่**

3. ระบุค่าสำหรับฟิลด์ต่อไปนี้

   | ฟิลด์ | คำอธิบาย |
   | --- | --- |
   | **รหัสไทล์** | ตัวระบุเฉพาะสำหรับไทล์ |
   | **ข้อความป้ายชื่อของไทล์** | ข้อความที่จะปรากฏในไทล์ของการบริการตนเอง |
   | **คำอธิบาย** | คำอธิบายเกี่ยวกับไทล์ |
   | **ที่อยู่อินเทอร์เน็ต** | ป้อน URL ไปยังหน้าบริการตนเองของพนักงาน |
   | **ขนาดไทล์** | ขนาดของไทล์ เล็ก กลาง หรือใหญ่ |
   | **เป้าหมาย** | ระบุว่าควรเปิดหน้าในหน้าต่างใหม่หรือหน้าต่างปัจจุบันหรือไม่ |
   | **รูปภาพพื้นหลังของไทล์** | URL ของรูปภาพที่จะใช้สำหรับไทล์ (เลือกกำหนดได้) |
   | **เริ่ม** | วันที่และเวลาเริ่มต้นของไทล์ควรพร้อมใช้งาน |
   | **End** | วันที่และเวลาสิ้นสุดของไทล์ควรพร้อมใช้งาน |

4. เลือก **บันทึก**
