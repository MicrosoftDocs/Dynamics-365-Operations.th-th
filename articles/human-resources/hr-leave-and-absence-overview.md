# <a name="overview"></a>ภาพรวม

Dynamics 365 Human Resources ช่วยให้คุณสามารถทำให้ผลประโยชน์การลางานที่ดีแก่ผู้ปฏิบัติงานของคุณ พื้นที่การทำงาน **การลางานและขาดงาน** จะแสดงกรอบความยืดหยุ่นสำหรับการสร้างแผนการลางานให ลำดับงานสำหรับการจัดการการร้องขอ และหน้าบริการตนเองที่ใช้งานง่ายสำหรับพนักงานเพื่อร้องขอเวลาปิด การวิเคราะห์ช่วยให้องค์กรของคุณประเมินและตรวจสอบยอดดุลการลางานและการใช้งานสำหรับแผนการลางานของคุณ

- [สร้างลำดับงานการร้องขอลางาน](hr-leave-and-absence-workflow.md)

## <a name="leave-and-absence-known-issues"></a>ปัญหาเกี่ยวกับการลางานและการขาดงาน

### <a name="rounding-precision"></a>ความแม่นยำในการปัดเศษ

คุณไม่สามารถตั้งค่า **ความแม่นยำในการปัดเศษ** เมื่อคุณตั้งค่า **ชนิดการปัดเศษ** คุณสามารถตั้งค่า **ความแม่นยำในการปัดเศษ** โดยใช้เอนทิตี้ **ชนิดการลางานและการขาดงานชนิดการลางานและการขาดงาน** เท่านั้น 

1. จาก **ชนิดการลางานและการขาดงาน** ให้เลือก **เปิดในExcel** เพื่อเปิดเอนทิตี้ **ชนิดการลางานและการขาดงาน**

2. หลังจากเปิดไฟล์และเปิดใช้งานแล้ว ให้เลือก **การออกแบบ**

3. บนตาราง **ชนิดการลาและการขาดงาน** ให้เลือกตัวเลือกดินสอเพื่อแก้ไข

4. เลือก **RoundingPrecision** และ **RoundingType** แล้วเลือก **เพิ่ม** เพื่อเพิ่มลงในรายการฟิลด์

5. เลือก **อัพเดต** แล้วจากนั้นเลือก **เสร็จสิ้น**

6. ป้อนหรือเลือก **ชนิดการปัดเศษ** สำหรับแต่ละชนิดของการลางาน ถ้ายังไม่ได้ตั้งค่าไว้แล้ว 

7. ป้อน **ความแม่นยำในการปัดเศษ** สำหรับชนิดที่เหมาะสม

8. เลือก **เผยแพร่** เพื่อผลักดันการเปลี่ยนแปลงลงในทรัพยากรบุคคล

## <a name="leave-and-absence-preview-features"></a>คุณลักษณะการแสดงตัวอย่างการลางานและการขาดงาน

คุณสามารถลองใช้คุณลักษณะการแสดงตัวอย่างการลางานและขาดงานในสิ่งแวดล้อม **Sandbox** สำหรับข้อมูลเกี่ยวกับการเปิดใช้งานคุณลักษณะการแสดงตัวอย่าง ให้ดู [จัดการคุณลักษณะ](hr-admin-manage-features.md) 

[!include [banner](includes/preview-feature.md)]

คุณลักษณะการแสดงตัวอย่างได้แก่:

- **การลางานคงค้างสำหรับแต่ละบริษัทหรือแต่ละแผน** - คุณสามารถรันกระบวนการค้างรับสำหรับบริษัททั้งหมดหรือสำหรับบริษัทหนึ่งๆ นอกจากนี้ คุณยังสามารถรันกระบวนการค้างรับสำหรับแผนการลางานและการขาดงานเฉพาะสำหรับบริษัทหนึ่งๆ ได้ด้วย 

- **ซื้อการลางาน** - คุณสามารถเปิดใช้งานและสร้างนโยบายการลางานสำหรับพนักงานที่จะส่งคำขอซื้อ พนักงานสามารถส่งคำขอซื้อและปรับปรุงยอดดุลโดยอัตโนมัติเพื่อแสดงคำขอ  

- **เพิ่มส่วนแนบไปยังคำขอการลางานที่ได้รับการอนุมัติ** - คุณสามารถเพิ่มส่วนแนบไปยังคำขอลางานที่ได้รับอนุมัติแล้ว
