# <a name="create-an-open-ended-question"></a>สร้างคำถามปลายเปิดแบบสิ้นสุด

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



คำถามปลายเปิดช่วยให้ผู้ตอบให้ตอบแสดงการตอบสนองของตนเอง  คุณสามารถควบคุมการตอบสนองที่เป็นตัวเลข ทศนิยม วัน ข้อคิดเห็น และอื่น ๆ  ขั้นตอนนี้นำไปสู่กระบวนการสร้างชนิดของคำถามที่เฉพาะเจาะจง คำถามปลายเปิด  เพื่อเริ่มต้นกระบวนงานนี้ ไปที่แบบสอบถาม > การออกแบบ > คำถาม  ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF

1. คลิก สร้าง
2. เลือกชนิดเพื่อจัดกลุ่มคำถามที่เกี่ยวข้องกัน
3. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
4. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
    * ป้อนชนิดของข้อความ จำนวนเต็ม จำนวนจริง วัน เวลา และหมายเหตุที่จะใช้สำหรับคำถามปลายเปิด  
5. ในฟิลด์ชนิดของข้อมูลป้อนเข้า ให้เลือกตัวเลือกหนึ่งตัวเลือก
    * ชนิดของการป้อนข้อมูลของข้อความหรือหมายเหตุสำหรับคำถามปลายเปิด  
6. ในฟิลด์ข้อความ ให้ป้อนคำถามของคุณ
