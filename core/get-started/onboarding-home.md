# <a name="before-you-buy"></a>ก่อนที่คุณจะซื้อ 

[!include[banner](../includes/banner.md)]

ยังไม่เคยใช้ Dynamics 365 for Finance and Operations, Enterprise Edition ใช่หรือไม่ เราได้รวมคำแนะนำทีละขั้นตอนไว้ไม่ว่าคุณจะยังคงกำลังประเมิน Finance and Operations อยู่หรือพร้อมที่จะทำการซื้อก็ตาม

## <a name="step-one--try-out-finance-and-operations-in-a-free-7-day-trial"></a>ขั้นตอนที่หนึ่ง: ลองใช้ Finance and Operations ในช่วงทดลองใช้ฟรี 7 วัน
คุณสามารถเริ่มต้นช่วงทดลองใช้ฟรี 7 วันสำหรับ Dynamics 365 for Finance and Operations ได้ผ่านการลงทะเบียนอีเมลอย่างง่าย รุ่นทดลองใช้ของ Finance and Operations ประกอบด้วยคำแนะนำการเริ่มต้นใช้งานที่มี คู่มืองานซึ่งให้คำแนะนำทีละขั้นตอน ซึ่งจะช่วยให้คุณสามารถดูสถานการณ์จำลองเฉพาะในการดำเนินการต่าง ๆ ผลิตภัณฑ์จะพร้อมใช้งานสำหรับการสำรวจและการดำเนินการในสถานการณ์จำลองต่าง ๆ มีข้อมูลสาธิตเพื่อช่วยให้การใช้ผลิตภัณฑ์ง่ายขึ้นและเพื่อให้ประสบการณ์มีความหมายมากยิ่งขึ้น อีเมลการแจ้งเตือนจะถูกส่ง 3 วันก่อนที่การทดลองใช้งานจะหมดอายุ สามารถเริ่มต้นการซื้อในขณะนั้นเพื่อให้การซื้อเสร็จสมบูรณ์ ดูรายละเอียดใน [www.microsoft.com/en-us/dynamics365/operations](https://www.microsoft.com/en-us/dynamics365/operations)

## <a name="step-two-buy-and-manage-a-subscription"></a>ขั้นตอนที่สอง: ซื้อและจัดการการบอกรับเป็นสมาชิก
คุณสามารถซื้อการบอกรับเป็นสมาชิกสำหรับ Dynamics 365 for Finance and Operations ผ่านสิทธิ์ใช้งานปริมาตรหรือผู้ให้บริการโซลูชัน Cloud

### <a name="buy-through-volume-licensing"></a>ซื้อผ่านการให้สิทธิ์ใช้งานปริมาตร
หากองค์กรของคุณมีผู้ใช้ Dynamics 365 อย่างน้อย 250 คน คุณอาจสนใจ [ข้อตกลงการให้สิทธิ์ใช้งานปริมาตร](https://www.microsoft.com/en-us/licensing/how-to-buy/how-to-buy.aspx) 

ในการให้สิทธิ์ใช้งานปริมาตร สามารถใช้งาน Finance and Operations ได้ผ่าน:
- ข้อตกลงขององค์กร
- การบอกรับเป็นสมาชิกข้อตกลงขององค์กร
- การลงทะเบียนสำหรับโซลูชันการศึกษา (ภายใต้ข้อตกลงของสถาบันและโรงเรียน)
- ข้อตกลงของผลิตภัณฑ์และการบริการของ Microsoft

### <a name="buy-through-a-cloud-solution-provider"></a>ซื้อผ่านผู้ให้บริการโซลูชัน Cloud
ผู้ให้บริการโซลูชัน Cloud ของ Microsoft สามารถทำงานกับคุณอย่างใกล้ชิดเพื่อทำความเข้าใจถึงความต้องการของธุรกิจหรือองค์กรของคุณ [เรียนรู้วิธีการค้นหาคู่ค้า Dynamics 365](https://partnercenter.microsoft.com/en-us/partner/home) 

### <a name="choose-your-support"></a>เลือกการสนับสนุนของคุณ
Microsoft ให้การสนับสนุนแบบยืดหยุ่นและเป็นผู้นำทางอุตสาหกรรม การบริการ และทรัพยากรที่ช่วยให้ผู้ใช้สามารถระบุปัญหาทางเทคนิคและส่งคืนการลงทุน Dynamics 365 ของคุณให้มากที่สุดได้อย่างรวดเร็ว เลือกแผนที่ตรงกับความต้องการทางธุรกิจของคุณมากที่สุด ข้อมูลเพิ่มเติม:[การสนับสนุน 365 Dynamics](https://www.microsoft.com/en-us/dynamics365/support)

## <a name="step-three-learn-about-fasttrack-and-plan-your-deployment"></a>ขั้นตอนที่สาม: เรียนรู้เกี่ยวกับ FastTrack และแผนการปรับใช้ของคุณ
Microsoft FastTrack สำหรับ Dynamics 365 คือ บริการความสำเร็จของบริษัทลูกค้าที่ได้รับการออกแบบมาเพื่อช่วยให้คุณสามารถย้าย Dynamics 365 ได้อย่างราบรื่น และมั่นใจ เพื่อให้คุณสามารถรับรู้มูลค่าทางธุรกิจได้เร็วขึ้น เมื่อคุณเข้าร่วมในโปรแกรม FastTrack คุณจะได้รับคำแนะนำเกี่ยวกับแนวทางที่พึงปฏิบัติและวิธีการวางแผนสำหรับการเปิดตัวที่ประสบความสำเร็จ คุณจะได้เรียนรู้เกี่ยวกับวิธีการเปิดใช้งานผู้ใช้ใหม่และการขยายความสามารถ – ทั้งหมดตามความเร็วของคุณเอง นอกจากนี้ คุณจะสามารถเข้าถึงทรัพยากรวิศวกรรม Microsoft ที่ถูกกำหนดให้ทำให้ประสบการณ์ Dynamics 365 ของคุณประสบความสำเร็จ ข้อมูลเพิ่มเติม: [Microsoft FastTrack สำหรับภาพรวม Dynamics 365](/dynamics365/unified-operations/fin-and-ops/get-started/fasttrack-dynamics-365-overview) 

### <a name="plan-your-deployment"></a>วางแผนการปรับใช้ของคุณ
การตัดสินใจที่คุณทำเมื่อเริ่มต้นและระหว่างกระบวนการปรับใช้สามารถมีผลกระทบต่อต่อขั้นตอนปลายน้ำ และระยะอื่นๆ ของแผน การพัฒนาแผนการปรับใช้ Finance and Operations มีความสำคัญสำหรับการกำหนดการดำเนินการที่จะใช้ตั้งแต่เริ่มต้นไปจนถึงการใช้งานจริง ปริมาณทรัพยากรที่เกี่ยวข้องกับการวางแผนขึ้นอยู่กับขนาดและขอบเขตของโครงการ เราขอแนะนำให้คุณใช้ระเบียบวิธี Lifecycle Services ที่เหมาะสมเพื่อช่วยในการวางแผนของคุณ 

ข้อมูลเพิ่มเติม: 
- การปรับใช้บนสภาพแวดล้อมที่ใช้ Cloud
- [การปรับใช้ในองค์กร](/dev-itpro/deployment/on-premises-deployment-landing-page)

บ่อยครั้งที่องค์กรจะจ้างงานผู้ขายซอฟต์แวร์อิสระ (ISV), ผู้ขายสินค้ามูลค่าเพิ่ม หรือคู่ค้าเพื่อช่วยในการปรับใช้และรักษาการปรับใช้ Dynamics 365 คู่ค้าจะมอบประสบการณ์ แนวทางปฏิบัติ และความเชี่ยวชาญอุตสาหกรรมมาใช้ให้เป็นประโยชน์ในกระบวนการวางแผนและปรับใช้ [ค้นหาคู่ค้า หากไม่มี](https://partnercenter.microsoft.com/en-us/pcv/search)
