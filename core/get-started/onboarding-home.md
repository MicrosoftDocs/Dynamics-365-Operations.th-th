# <a name="onboarding-home-page"></a>โฮมเพจของการเตรียมความพร้อม

[!include[banner](../includes/banner.md)]


เมื่อต้องการเริ่มต้นใช้งาน Microsoft Dynamics 365 for Operations คุณสามารถเลือกระดับของการบอกรับเป็นสมาชิกที่เหมาะสมกับคุณ ค้นหาคู่ค้าที่มีประสบการณ์ในภูมิภาคหรืออุตสาหกรรมของคุณ และใช้โปรแกรม Microsoft FastTrack เพื่อช่วยคุณในการเริ่มใช้งานและทำงานได้อย่างรวดเร็ว  

ทรัพยากรต่อไปนี้แสดงภาพรวมของ Lifecycle Services และวิธีการทำงานกับหุ้นส่วนของคุณเพื่อเตรียมความพร้อมสำหรับ Dynamics 365 for Operations

-   [ภาพรวม Microsoft FastTrack สำหรับ Dynamics 365](fasttrack-dynamics-365-overview.md)
