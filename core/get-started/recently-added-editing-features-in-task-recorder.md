# <a name="recently-added-editing-features-in-task-recorder"></a>รายการที่เพิ่ง เพิ่มคุณลักษณะการแก้ไขในตัวบันทึกงาน

ถ้าคุณใช้ตัวบันทึกงานเพื่อสร้างเส้นบอกแนวงาน คุณสามารถแก้ไขแฟ้มของคุณอย่างมีประสิทธิภาพโดยใช้ฟังก์ชันการทำงานที่อธิบายไว้ใน wiki นี้

คุณลักษณะเหล่านี้จะพร้อมใช้งานบนตัว**ตั้งค่า&gt;ดงาน&gt;บันทึกการแก้ไข**เมนู

-   แทรกขั้นตอนโดยไม่ต้องบันทึกแฟ้มทั้งหมด
-   ย้ายขั้นตอนภายใต้งานย่อยโดยไม่ต้องบันทึกแฟ้มทั้งหมดใหม่อีกครั้ง
-   ยุบเขตข้อมูลชื่อและคำอธิบายบันทึก

## <a name="insert-steps-without-rerecording-the-entire-file"></a>แทรกขั้นตอนโดยไม่ต้อง rerecording แฟ้มทั้งหมด
คุณสามารถเพิ่มขั้นตอนใดก็ได้ในคำแนะนำของงานโดยไม่เล่น หรือบันทึกแฟ้มทั้งหมด

1.  เลือกขั้นตอนหลังจากที่คุณต้องการให้ขั้นตอนใหม่ที่จะแทรก ตรวจสอบให้แน่ใจว่า ขั้นตอนจะถูกเน้น

ในใบสั่งสำหรับตัวบันทึกงานเพื่อแทรกขั้นตอน คุณต้องมีเปิดเพจที่ถูกต้อง หน้าที่ถูกต้องคือ เพจที่เกิดขั้นตอนใหม่ ตัวบันทึกภารกิจมีกลไกที่เป็นตัวกำหนดสิ่งที่ใช้งานอยู่เพจเป็น และจะปิดการใช้งานถ้าไม่ได้เปิดเพจที่ถูกต้อง 

[![tg1](./media/tg1.png)](./media/tg1.png) 


เมื่อคุณอยู่บนเพจที่ถูกต้อง**ขั้นตอนการแทรก**กลายเป็นพร้อมใช้งาน

[![tg2](./media/tg2-231x300.png)](./media/tg2.png)

2. คลิ**ขั้นตอนการแทรก**

เมื่อคุณคลิ**ขั้นตอนการแทรก**ดงานสลับไปยังโหมดบันทึก การดำเนินการใด ๆ ใน UI จะถูกบันทึกในขณะนี้ และเพิ่มแบบแทนที่เป็นขั้นตอน

3. คลิ**หยุด**

คุณสามารถทำซ้ำกระบวนการ การเพิ่มขั้นตอนมาก หรือย้ายงานย่อยมากเท่าที่จำเป็น (ดูด้านล่างสำหรับงานย่อย)

4. เมื่อคุณทำเสร็จคลิกแก้ไขรายการแนะนำงาน**ทำการแก้ไข**และจากนั้น เลือกหนึ่งในอ็อพชันเมื่อต้องการบันทึก หรือจัดพิมพ์คำแนะนำของงาน

## <a name="move-steps-under-a-subtask-without-rerecording-the-entire-file"></a>ขั้นตอนภายใต้กรอบย้าย โดยไม่มีแฟ้มทั้งหมดที่ rerecording
คุณสามารถเลื่อนขั้นตอนภายใต้งานย่อย โดยไม่มีการเล่น หรือบันทึกแฟ้มทั้งหมด นอกจากนี้คุณสามารถย้ายขั้นตอนงานย่อยหรือขั้นตอนย่อยงานสิ้นสุดถ้าคุณต้องการจัดกลุ่มบล็อกอยู่ของขั้นตอน

1.  เลือกขั้นตอนหรือขั้นตอนงานย่อยที่คุณต้องการย้าย ตรวจสอบให้แน่ใจว่า ขั้นตอนจะถูกเน้น
2.  คลิกจุดไข่ปลา จาก นั้นคลิก**ย้ายขั้นตอนหลังจาก**

[![tg3](./media/tg3.png)](./media/tg3.png)

3. เลือกขั้นตอนหรือขั้นตอนงานย่อยที่คุณต้องการย้ายขั้นตอนหรืองานย่อยหลังจาก ตัวบันทึกภารกิจจะย้ายขั้น

4. เมื่อต้องการย้ายงานย่อยขั้นตอนท้าย เลือก จุดไข่ปลาคลิก คลิก**ย้ายขั้นตอนหลังจาก**แล้ว เลือกขั้นตอนหลังจากที่คุณต้องการให้ขั้นตอนย่อยงานสิ้นสุดเพื่อให้

ถ้าคุณต้องการให้ขั้นตอนแรกในรายการแนะนำงานให้อยู่ในงานย่อย สร้างงานย่อยขั้นตอนเป็นขั้นตอนสอง แล้ว ย้ายขั้นแรกไมโครโฟน คุณสามารถเพิ่ม หรือย้ายขั้นตอนหรืองานย่อยตามความจำเป็นมาก

5. เมื่อคุณทำเสร็จคลิกแก้ไขรายการแนะนำงาน**ทำการแก้ไข**และจากนั้น เลือกหนึ่งในอ็อพชันเมื่อต้องการบันทึก หรือจัดพิมพ์คำแนะนำของงาน

## <a name="collapse-recording-name-and-description"></a>ยุบการบันทึกชื่อและคำอธิบาย
คุณสามารถขยาย และยุบตัว**บันทึกชื่อ**และ**บันทึกคำอธิบาย**เขตข้อมูลได้ เมื่อเขตข้อมูลเหล่านี้ถูกยุบ ขั้นตอนเพิ่มเติมจะปรากฏในโปรแกรมเรกคอร์ดงานการแก้ไขบานหน้าต่าง
