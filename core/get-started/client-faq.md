# <a name="dynamics-365-for-operations-client-faq"></a>Dynamics 365 สำหรับไคลเอ็นต์การดำเนินการบ่อย

บทความนี้ให้คำตอบสำหรับคำถามที่ถามบ่อยเกี่ยวกับ 365 Dynamics ของ Microsoft สำหรับไคลเอ็นต์การดำเนินการ

<a name="why-arent-symbols-loaded-when-i-use-dynamics-365-for-operations"></a>เหตุใดจึงไม่สัญลักษณ์โหลดเมื่อฉันใช้ Dynamics 365 สำหรับการดำเนินงาน
-----------------------------------------------------------------

การตั้งค่าความปลอดภัยในเบราเซอร์ของคุณอาจป้องกันสัญลักษณ์จากการถูกโหลดอย่างถูกต้อง เพื่อแก้ไขปัญหานี้ ทำตามขั้นตอนต่อไปนี้:

-   ถ้าคุณกำลังประสบกับปัญหานี้ใน Internet Explorer คลิ**เครื่องมือ**แล้ว คลิก**ตัวเลือกอินเทอร์เน็ต**  ในกล่องโต้ตอบตัวเลือกอินเทอร์เน็ต บน**ความเป็นส่วนตัว**แท็บ คลิก**ระดับแบบกำหนดเอง**และให้แน่ใจว่าการ**ดาวน์โหลดแบบอักษร**เลือก
-   มิฉะนั้น คุณอาจต้องเพิ่ม 365 Dynamics สำหรับไซต์การดำเนินการกับรายชื่อไซต์ที่เชื่อถือ

## <a name="i-miss-the-ribbon-from-dynamics-ax-2012-can-i-keep-action-pane-tabs-open-all-the-time"></a>ฉันคิดถึง ribbon จาก Dynamics AX 2012 ฉันสามารถทำให้บานหน้าต่างการดำเนินการที่แท็บเปิดตลอดเวลาหรือไม่
เรากำลังวางแผนจะใช้คุณลักษณะนี้ในเร็ว ๆ นี้ ผู้ใช้นั้นจะสามารถเลือกที่จะเก็บแท็บในบานหน้าต่างการดำเนินการเปิดตลอดเวลา มิฉะนั้น แท็บจะสามารถยุบเมื่อไม่ได้กำลังใช้งาน เพื่อเข้าใช้พื้นที่หน้าจอเพิ่มเติมสำหรับหน้านั้น

## <a name="why-do-i-sometimes-see-different-shortcut-menus-when-i-rightclick"></a>เหตุใดบางครั้งเห็นเมนูทางลัดที่แตกต่างกันเมื่อฉัน rightclick หรือไม่
ถ้าคุณคลิกขวาในฟิลด์แก้ไขได้ (หรือ ถ้าข้อความถูกเลือก), เมนูทางลัดของเบราเซอร์จะแสดงขึ้น เมนูนี้ช่วยให้คุณเข้าถึงคำสั่ง **ตัด**, **สำเนา**และ **วาง** เราไม่สามารถฝังคำสั่งเหล่านี้ลงใน 365 Dynamics สำหรับเมนูทางลัดของการดำเนินงานเนื่องจาก สำหรับเหตุผลด้านความปลอดภัย เบราว์เซอร์ไม่ช่วยให้เราสามารถเข้าถึงคลิปบอร์ระบบโดยทางโปรแกรม

ถ้าคุณคลิกขวาที่ป้ายชื่อเขตข้อมูลหรือค่าของตัวควบคุมแบบอ่านอย่างเดียว คุณจะเห็น 365 Dynamics สำหรับเมนูทางลัดของการดำเนินงาน

เพื่อเข้าถึงแป้นพิมพ์ให้ง่ายขึ้น เราจะใช้แป้นพิมพ์ลัดในอนาคตซึ่งจะเปิด 365 Dynamics สำหรับเมนูทางลัดของการดำเนินงาน

## <a name="where-is-the-view-details-functionality-in-dynamics-365-for-operations"></a>ฟังก์ชันดูรายละเอียดใน 365 Dynamics สำหรับการดำเนินงานอยู่ที่ไหน
ตัวเลือก **ดูรายละเอียด** จะพร้อมใช้งานในสองวิธี:

-   หากส่วนควบคุมมีความสามารถในการ **ดูรายละเอียด **และถ้าตัวควบคุมมีค่า ค่านั้นจะแสดงเป็นไฮเปอร์ลิงค์ คุณสามารถคลิกไฮเปอร์ลิงค์เพื่อเปิดหน้าซึ่งประกอบด้วยรายละเอียดเพิ่มเติม
-   **ดูรายละเอียด**เป็นตัวเลือกบน Dynamics 365 สำหรับเมนูทางลัดของการดำเนินงาน สำหรับข้อมูลเพิ่มเติมเกี่ยวกับเวลา Dynamics 365 สำหรับเมนูทางลัดของการดำเนินงานจะแสดงขึ้นเมื่อคุณคลิกขวา ดูส่วนก่อนหน้า
