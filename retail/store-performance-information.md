# <a name="analyze-store-performance"></a>วิเคราะห์ประสิทธิภาพร้านค้า

บทความนี้อธิบายถึงวิธีการที่คุณสามารถใช้วิเคราะห์ในหน่วยความจำ แบบเรียลไทม์เพื่อเข้าถึง สำรวจ และได้รับช่วยเกี่ยวกับประสิทธิภาพการจัดเก็บ ตาม 365 Dynamics ของคุณ Microsoft สำหรับข้อมูลในการดำเนินงาน 

เป็นส่วนหนึ่งของ Dynamics 365 สำหรับการดำเนินการ ผู้ใช้สามารถศึกษาประสิทธิภาพการจัดเก็บในเวลาจริงข้ามลำดับชั้นขององค์กรในระดับต่าง ๆ ระยะเวลาที่เลือก ด้วยการเปิดออกกล่อง**สรุปช่องทาง**รายงานจากที่ตั้งต่อไปนี้:

-   **จัดการร้านค้าปลีก**Dynamics 365 พื้นที่ทำงานสำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สถานี**&gt;**จัดการร้านค้าปลีก**&gt;**รายงาน**&gt;**สถานีรายงานสรุป**
-   **เงินของร้านค้าปลีก**Dynamics 365 พื้นที่ทำงานสำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สถานี**&gt;**เงินของร้านค้าปลีก**&gt;**รายงาน**&gt;**สถานีรายงานสรุป**
-   **การสอบถามและรายงาน**ส่วน 365 Dynamics สำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สอบถามและรายงาน**&gt;**รายงานการขาย**&gt;**สถานีรายงานสรุป**

รายงานนี้แสดงสแนปช็อตของข้อสรุปต่อไปนี้ซึ่งเป็นส่วนหนึ่งของประสิทธิภาพการทำงานของร้านค้า:

-   สรุปยอดขายรวม
-   สรุปชนิดการชำระเงิน
-   สรุปภาษี
-   สรุปการแทนที่ราคา
-   สรุปส่วนลด
