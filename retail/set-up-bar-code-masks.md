# <a name="set-up-bar-code-masks"></a>ตั้งค่าตัวพรางบาร์โค้ด

หัวข้อนี้อธิบายวิธีการตั้งค่าอักขระตัวพรางบาร์โค้ด ตัวพรางบาร์โค้ด และวิธีการกำหนดบาร์โค้ดหน้าที่ปิดบังกับบาร์โค้ด

<a name="set-up-bar-code-mask-characters"></a>ตั้งค่าอักขระตัวพรางบาร์โค้ด
-------------------------------

ใช้ตัวพรางบาร์โค้ด เพื่อสร้างบาร์โค้ด และบาร์โค้ดที่สแกนเข้ากับจุดขาย (POS) ที่ระบุได้อย่างรวดเร็ว ตัวพรางจะประกอบด้วยอักขระที่ทำหน้าที่เป็นตัวยึดที่ระบุรูปแบบสำหรับบาร์โคดที่จะถูกสร้างขึ้น ด้วย การตั้งค่าตัวพรางบาร์โค้ด คุณจำเป็นต้องตั้งค่าอักขระตัวพรางบาร์โค้ด ไปที่**ขายปลีกและพาณิชย์**&gt;**บริหารสินค้าคงคลัง**&gt;**บาร์โค้ด และป้ายชื่อ**&gt;**มาสก์อักขระ** คลิ**สร้าง**เพื่อสร้างอักขระตัวพรางบาร์โค้ด อักขระตัวพรางที่คุณสามารถสร้างเพื่อระบุข้อมูลบาร์โค้ดต่อไปนี้

|                      |                                                                                                                 |
|----------------------|-----------------------------------------------------------------------------------------------------------------|
| **Field**            | **Description**                                                                                                 |
| **Product**          | ตัวยึดตำแหน่งสำหรับรหัสผลิตภัณฑ์                                                                                     |
| **Any number**       | ใช้เพื่อระบุตัวเลขที่จะยากมีโค้ดในบาร์โค้ด                                                  |
| **Check digit**      | บ่งชี้ว่า รูปแบบบาร์โค้ดในตัวพรางบาร์โค้ดใช้ตำแหน่งการตรวจสอบเพื่อยืนยันความถูกต้องของบาร์โค้ด |
| **Size digit**       | บ่งชี้ขนาดเป็นบาร์โค้ดที่สร้างขึ้นสำหรับผลิตภัณฑ์ย่อยซึ่งรวมถึงขนาด                                 |
| **Color digit**      | บ่งชี้สีในบาร์โค้ดที่สร้างขึ้นสำหรับผลิตภัณฑ์ย่อยซึ่งรวมถึงสี                               |
| **Style digit**      | บ่งชี้ลักษณะในบาร์โค้ดที่สร้างขึ้นสำหรับผลิตภัณฑ์ย่อยซึ่งรวมถึงลักษณะ                             |
| **EAN license code** | ตัวยึดตำแหน่งสำหรับลิขสิทธิ์ EAN ออกสำหรับรหัสลิขสิทธิ์ EAN                                                       |
| **Price**            | บ่งชี้ว่า ราคาสำหรับราคาบาร์โค้ดที่ฝังตัว                                                                   |
| **Quantity**         | บ่งชี้ปริมาณในปริมาณ/สุ่มน้ำหนักที่ฝังตัวอยู่อเมริกา                                                |
| **Employee**         | บ่งชี้ว่า เซ็กเมนต์บาร์โค้ดสำหรับหมายเลขรหัสพนักงานที่ใช้สำหรับเข้าสู่ระบบบาร์โค้ด POS                                  |
| **Customer**         | บ่งชี้ว่า เซ็กเมนต์รหัสลูกค้า                                                                                  |
| **ป้อนข้อมูล**       | *ยังไม่ได้ นำมาใช้*                                                                                          |
| **Discount code**    | บ่งชี้ว่า รหัสส่วนลดสำหรับบาร์โคดที่จะใช้ในการเพิ่มส่วนลดไปยังจุดของธุรกรรมการขาย             |
| **Gift card**        | บ่งชี้ว่า หมายเลขบัตรของขวัญเมื่อออก หรือชำระ ด้วยบัตรของขวัญ                                               |
| **Loyalty card**     | เพิ่มลูกค้าสมาชิกให้กับธุรกรรม และถูกใช้เมื่อการชำระเงิน โดยลูกค้าสมาชิก                             |

## <a name="define-bar-code-masks"></a>กำหนดตัวพรางบาร์โค้ด
หลังจากที่มีระบุอักขระตัวพรางบาร์โค้ดสำหรับตัวพรางบาร์โค้ดที่จำเป็น ไป**ขายปลีกและพาณิชย์**&gt;**บริหารสินค้าคงคลัง**&gt;**บาร์โค้ด และป้ายชื่อ**&gt;**การตั้งค่าตัวพรางบาร์โค้ด** ในหน้านี้ คุณสามารถกำหนดตัวพรางบาร์โค้ดที่ใช้อักขระที่ระบุไว้ก่อนหน้านี้ บาร์โค้ดรูปแบบที่จะใช้เมื่อสร้างบาร์โค้ด และจะยังเหล่านี้ช่วยให้คุณระบุบาร์โค้ดสแกนที่ POS

1.  คลิ**สร้าง**เพื่อสร้างตัวพรางบาร์โค้ดใหม่
2.  ป้อนค่าในการ**รหัสตัวพราง**และ**คำอธิบาย**ฟิลด์ และจากนั้น เลือกชนิดตัวพรางบาร์โค้ด**ชนิด**เขตข้อมูล
3.  ในการ**ทั่วไป**ส่วน เลือกค่าในการ**มาตรฐานบาร์โค้ด**ฟิลด์ และจากนั้น ระบุคำนำหน้าบาร์โค หากจำเป็น
4.  ในการ**เซ็กเมนต์ตัวพรางบาร์โค้ด**ส่วน เพิ่มเซ็กเมนต์บาร์โค้ดที่จะใช้ในการสร้างบาร์โค้ด

ตามตัวอย่าง เพื่อสร้างตัวพรางบาร์โค้ดที่ มีรหัสตัวพราง 'ผลิตภัณฑ์' คุณต้องทำต่อไปนี้:

1.  สร้างตัวพรางบาร์โค้ดใหม่ และเลือกชนิด 'ผลิตภัณฑ์'
2.  เลือกมาตรฐานบาร์โคด ตัวอย่างเช่น 'รหัส 39'
3.  ใส่คำนำหน้าจะใช้เพื่อระบุบาร์โค้ดได้อย่างง่ายดาย ตัวอย่างเช่น '22'
4.  เพิ่มเซ็กเมนต์ตัวพราง เซ็กเมนต์ตัวพราง 'ผลิตภัณฑ์' จะถูกเลือก
5.  ระบุความยาวสำหรับเซ็กเมนต์ผลิตภัณฑ์ เช่น '10' ความยาวควรตรงกับความยาวของหมายเลขผลิตภัณฑ์ที่ใช้กันทั่วไปในร้านค้า จะแสดงเป็นตัวอย่างในรูปแบบ**ทั่วไป**ส่วนภายใต้**รูปแบบ**

## <a name="assign-bar-code-masks-to-bar-codes"></a>กำหนดตัวพรางบาร์โค้ดให้กับบาร์โค้ด
ตัวพรางบาร์โค้ดต้องกำหนดบาร์โค้ดก่อนที่จะสามารถใช้ ดำเนินการต่อไป ด้วยตัวอย่างก่อนหน้านี้ เพื่อ กำหนดตัวพรางบาร์โค้ดเป็นบาร์โค ทำสิ่งต่อไปนี้:

1.  ไปที่**จัดการองค์กร**&gt;**ตั้งค่า**&gt;**บาร์โค** คลิ**สร้าง**เพื่อสร้างบาร์โค้ดใหม่
2.  ป้อนค่าในการ**บาร์โค้ด****ตั้งค่า**และ ** เซ็ตอัพ ** เขตข้อมูล
3.  ในการ**ทั่วไป**จากนั้น ใน**บาร์โค้ดชนิด**ฟิลด์ เลือก Code 39 ในการ**รูปแบบ****ID**ฟิลด์ เลือกตัวพราง 'ผลิตภัณฑ์' ที่สร้างไว้ก่อนหน้านี้
4.  ภายใต้**ขนาด**ป้อน '12'
5.  Click **Save**.

ขณะนี้สามารถใช้ตัวพรางบาร์โค้ดเพื่อสร้างบาร์โค้ดสำหรับผลิตภัณฑ์ ขั้นตอนต่าง ๆ ข้างต้นเป็นตัวอย่างของวิธีการสร้างตัวพรางบาร์โค้ดสำหรับผลิตภัณฑ์ แต่พวกเขายังแสดงให้เห็นถึงวิธีการสร้างตัวพรางบาร์โค้ดสำหรับชนิดบาร์โค้ดที่ได้รับการสนับสนุนอื่น ๆ ตัวพรางบาร์โค้ด ชนิด และความยาวที่ควรได้รับการปรับสำหรับใช้ในสภาพแวดล้อมเฉพาะของคุณ
