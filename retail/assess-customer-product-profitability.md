# <a name="assess-customer-and-product-profitability"></a>ประเมินความสามารถในการทำกำไรของลูกค้าและผลิตภัณฑ์

บทความนี้อธิบายถึงวิธีการที่คุณสามารถใช้วิเคราะห์ในหน่วยความจำ แบบเรียลไทม์เพื่อเข้าถึง สำรวจ และได้รับช่วยเกี่ยวกับลูกค้าและกำไรจากผลิตภัณฑ์จาก 365 Dynamics ของคุณ Microsoft สำหรับข้อมูลในการดำเนินงาน 

เป็นส่วนหนึ่งของ Dynamics 365 สำหรับการดำเนินการ ผู้ใช้สามารถศึกษากำไรจากสำหรับลูกค้าด้านบน (10 ถึง 100) ข้ามระดับที่แตกต่างของลำดับชั้นขององค์กร ขึ้นอยู่กับค่าใดค่าหนึ่งในเงื่อนไขต่อไปนี้:

-   ยอดขาย
-   ปริมาณ
-   อัตรากำไรขั้นต้น
-   เปอร์เซ็นต์กำไร

สำหรับการประเมินนี้ คุณสามารถใช้รายงาน **ลูกค้าอันดับแรก** แบบ out-of-box ซึ่งคุณสามารถเปิดได้จากตำแหน่งที่ตั้งต่อไปนี้:

-   **จัดการร้านค้าปลีก**Dynamics 365 พื้นที่ทำงานสำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สถานี**&gt;**จัดการร้านค้าปลีก**&gt;**รายงาน**&gt;**รายงานลูกค้าอันดับแรก**
-   **การสอบถามและรายงาน**ส่วน 365 Dynamics สำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สอบถามและรายงาน**&gt;**รายงานการขาย**&gt;**รายงานลูกค้าอันดับแรก**

ในทำนองเดียวกัน ผู้ใช้สามารถศึกษาผลกำไรสำหรับผลิตภัณฑ์อันดับแรก (10 ถึง 100) ระหว่างระดับต่างๆของลำดับชั้นขององค์กร ที่ขึ้นอยู่กับเงื่อนไขข้อใดข้อหนึ่งต่อไปนี้:

-   ยอดขาย
-   ปริมาณ
-   อัตรากำไรขั้นต้น
-   เปอร์เซ็นต์กำไร

สำหรับการประเมินนี้ คุณสามารถใช้รายงาน **ผลิตภัณฑ์อันดับแรก **แบบ out-of-box ซึ่งคุณสามารถเปิดได้จากตำแหน่งที่ตั้งต่อไปนี้:

-   **จัดการร้านค้าปลีก**Dynamics 365 พื้นที่ทำงานสำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สถานี**&gt;**จัดการร้านค้าปลีก**&gt;**รายงาน**&gt;**อันดับรายงานผลิตภัณฑ์**
-   **จัดการประเภทและผลิตภัณฑ์**Dynamics 365 พื้นที่ทำงานสำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**ผลิตภัณฑ์และประเภท**&gt;**จัดการร้านค้าปลีก**&gt;**รายงาน**&gt;**อันดับรายงานผลิตภัณฑ์**
-   **การสอบถามและรายงาน**ส่วน 365 Dynamics สำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สอบถามและรายงาน**&gt;**รายงานการขาย**&gt;**อันดับรายงานผลิตภัณฑ์**
