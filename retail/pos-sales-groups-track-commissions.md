# <a name="track-commissions-in-pos-using-sales-groups"></a>ติดตามค่าคอมมิชชันใน POS โดยใช้กลุ่มการขาย

เป็นวิธีปฏิบัติการขายปลีกทั่วไปเพื่อติดตามการขายเชื่อมโยงที่ทำงานกับลูกค้าซึ่งให้ความช่วยเหลือ ขายขึ้น ข้าม- และขายในธุรกรรม

การติดตามการขาย โดยพนักงานขายเป็นหน่วยวัดสมาคมการขายความสามารถ ในขณะที่ขายโดยพนักงานเก็บเงินเป็นหน่วยวัดความเร็วและมีประสิทธิภาพ ขายติดตามพนักงานขายนอกจากนี้มักจะใช้ในการคำนวณค่าคอมมิชชันหรือสิ่งจูงใจอื่น ๆ

## <a name="configuring-a-worker-to-be-a-sales-representative-in-pos"></a>การตั้งค่าคอนฟิกผู้ปฏิบัติงานเป็น พนักงานขาย POS
เมื่อมีเพิ่มผู้ปฏิบัติงานให้กับกลุ่มการขาย จะกลายเป็นสิทธิสำหรับค่าคอมมิชชัน และสามารถระบุเป็นพนักงานขายในระบบ ผู้ปฏิบัติงานที่ไม่ได้อยู่ในกลุ่มการขายไม่เหมาะสมสำหรับค่าคอมมิชชัน และจะไม่แสดงในรายการเป็นพนักงานขายในจุดของแอพลิเคชันการขาย (POS) ใน POS รายชื่อของพนักงานขายจะได้รับมาจากกลุ่มการขายทั้งหมดที่ประกอบด้วยอย่างน้อยหนึ่งผู้ปฏิบัติงานที่กำหนดให้กับร้านค้า แสดงรายการใน POS เป็นการรวมกันของการขาย ID กลุ่มและชื่อ (ID: ชื่อ) กลุ่มการขายเริ่มต้นจะกำหนดให้กับผู้ปฏิบัติงานเพื่อสนับสนุนสถานการณ์สมมติซึ่งผู้ค้าปลีกที่เลือกเพื่อกำหนดตัวแทนฝ่ายขายในบรรทัด POS โดยอัตโนมัติ ผู้ใช้สามารถเลือกจากกลุ่มการขายใด ๆ ที่ผู้ปฏิบัติงานที่เป็นสมาชิกของ

## <a name="functionality-profile-settings"></a>ตั้งค่าโพรไฟล์ฟังก์ชัน
มีหมายเลขของหน้าที่การใช้งานการตั้งค่าโพรไฟล์สำหรับร้านค้าที่จะกำหนดขั้นตอนและกระบวนการใน POS ที่เกี่ยวข้องกับพนักงานขาย

|                                       |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Profile**                           | **Description**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Default to cashier when available** | ถ้าเปิดใช้งานตัวเลือกนี้ POS จะรวบรวมธุรกรรมบรรทัด ด้วยกลุ่มการขายเริ่มต้นของพนักงานเก็บเงินปัจจุบันโดยอัตโนมัติ ถ้าพนักงานเก็บเงินไม่มีการเริ่มต้นกลุ่มการขายที่ระบุ จะไม่สามารถตั้งค่า ผู้ใช้ด้วยตนเองยังคงตั้งกลุ่มการขาย โดยใช้ปุ่มกริดปุ่ม POS                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Prompt for sales representative**   | ตัวเลือกนี้มีสามค่าที่เป็นไปได้: ** ไม่ใช่ ** -ถ้ามีเลือกตัวเลือกนี้ ผู้ใช้จะไม่ได้รับพร้อมท์ให้เลือกกลุ่มการขาย ค่ายังคงถูกตั้งค่า โดยใช้กลุ่มการขายเริ่มต้นของพนักงานเก็บเงิน หรือด้วยตนเอง โดยใช้ POS มีปุ่มกริดปุ่ม **จุดเริ่มต้นของธุรกรรม**- ถ้ามี เลือกตัวเลือกนี้ และอย่างใดอย่างหนึ่ง**เริ่มต้นให้เป็นพนักงานเก็บเงิน**อ็อพชันไม่ได้เปิดใช้งาน หรือพนักงานเก็บเงินปัจจุบันไม่มีกลุ่มการขายเริ่มต้น ผู้ใช้จะได้รับพร้อมท์ให้เลือกกลุ่มการขาย ณจุดเริ่มต้นของแต่ละธุรกรรม เลือกกลุ่มการขายจากพร้อมท์นี้จะเริ่มต้นบรรทัดต่อมาทั้งหมดให้กับกลุ่มการขายที่เลือก ผู้ใช้ด้วยตนเองยังคงตั้งกลุ่มการขาย โดยใช้ปุ่มกริดปุ่ม POS **สำหรับแต่ละบรรทัด**- ถ้ามี เลือกตัวเลือกนี้ และอย่างใดอย่างหนึ่ง**เริ่มต้นให้เป็นพนักงานเก็บเงิน**อ็อพชันไม่ได้เปิดใช้งาน หรือพนักงานเก็บเงินปัจจุบันไม่มีกลุ่มการขายเริ่มต้น ผู้ใช้จะได้รับพร้อมท์ให้เลือกกลุ่มการขายหลังจากที่เพิ่มแต่ละบรรทัด ผู้ใช้ด้วยตนเองยังคงตั้งกลุ่มการขาย โดยใช้ปุ่มกริดปุ่ม POS |
| **จำเป็นต้องมี**                           | ตัวเลือกนี้จะใช้ได้เฉพาะเมื่อ POS ถูกกำหนดค่าให้แสดงพร้อมท์สำหรับพนักงานขายเท่านั้น ถ้าเปิดใช้งาน ผู้ใช้จำเป็นต้องเลือกกลุ่มขายก่อนดำเนินการต่อ มิฉะนั้น ผู้ใช้จะได้รับ พร้อมท์ แต่คุณสามารถยกเลิก และดำเนินต่อ โดยไม่มีการเลือก หลังจากที่มีเพิ่มบรรทัด ผู้ใช้ที่ มีสิทธิ์เพียงพอที่ไม่สามารถเอากลุ่มการขายจากบรรทัดยังคง ไม่ได้บังคับใช้ "ต้องพนักงานขาย" ในสถานการณ์นี้                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |

## <a name="displaying-the-sales-representative-information-on-the-pos-transactions-screen"></a>แสดงข้อมูลพนักงานขายบนหน้าจอ POS ธุรกรรม
โครงร่างหน้าจอของธุรกรรม POS และเนื้อหาจะสามารถจัดโครงแบบโดยใช้โครงร่างหน้าจอของตัวออกแบบ และกำหนดโครงร่างหน้าจอร้านค้า ทะเบียน หรือผู้ปฏิบัติงาน การ**พนักงานขาย**สามารถเพิ่มเขตข้อมูลไปยังแท็บบรรทัดของบานหน้าต่างใบเสร็จได้  นี้จะแสดงรหัสของกลุ่มการขายที่ระบุสำหรับแต่ละบรรทัดบนหน้าจอของธุรกรรม

## <a name="adding-sales-representative-operations-to-pos-button-grids"></a>การเพิ่มการขายการดำเนินงานที่พนักงานที่ POS ปุ่มกริด
POS อนุญาตให้ผู้ใช้สามารถตั้งค่าคอนฟิกกริดปุ่ม ซึ่งรวมอยู่ในโครงร่างหน้าจอเพื่อให้เข้ากับการดำเนินงานของ POS การดำเนินการต่อไปนี้ของ POS สามารถกำหนดให้ปุ่มของกริดปุ่มที่เกี่ยวข้องกับพนักงานขาย

|                                           |                                                                                                                                                                                                                                                                                              |
|-------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Operation**                             | **Description**                                                                                                                                                                                                                                                                              |
| ตั้งค่าพนักงานขายในรายการ          | การดำเนินการนี้ POS แสดงรายการของกลุ่มการขายที่เหมาะสม (ID: ชื่อ) สำหรับร้านค้า เลือกกลุ่มการขายจากรายการนี้จะเป็นตั้งค่าบนบรรทัดธุรกรรมปัจจุบัน                                                                                                            |
| ล้างข้อมูลพนักงานขายในรายการ        | การดำเนินการของ POS นี้เอาค่ากลุ่มขายปัจจุบันจากบรรทัดธุรกรรมปัจจุบัน                                                                                                                                                                                                  |
| ชุดพนักงานขายในธุรกรรม   | การดำเนินการนี้ POS แสดงรายการของกลุ่มการขายที่เหมาะสม (ID: ชื่อ) สำหรับร้านค้า เลือกกลุ่มการขายจากรายการนี้จะเป็นตั้งค่าเริ่มต้นบนธุรกรรมปัจจุบัน รายการใด ๆ ที่มีอยู่ โดยไม่มีการกำหนดกลุ่มการขายจะเป็นชุด ตลอดจนบรรทัดใด ๆ เพิ่มเติมในเวลาต่อมา |
| พนักงานขายล้างบนธุรกรรม | การดำเนินการของ POS นี้เอาค่ากลุ่มขายเริ่มต้นปัจจุบันจากธุรกรรมปัจจุบัน จะไม่มีผลกระทบต่อบรรทัดใด ๆ ที่มีอยู่ในธุรกรรม                                                                                                                             |

## <a name="calculating-commissions"></a>การคำนวณค่าคอมมิชชัน
มีคำนวณค่าคอมมิชชันสำหรับผู้ปฏิบัติงานในกลุ่มการขายที่ระบุ ณเวลาลงรายการบัญชีใบแจ้งยอดหรือลงรายการบัญชีใบสั่งขาย ยอดค่าคอมมิชชันจะถูกกำหนดตามส่วนแบ่งค่าคอมมิชชันของผู้ปฏิบัติงาน ตามที่กำหนดไว้ในกลุ่มการขายและการตั้งค่าการคำนวณค่าคอมมิชชันที่เกี่ยวข้องสำหรับลูกค้าและ/หรือผลิตภัณฑ์บนธุรกรรม
