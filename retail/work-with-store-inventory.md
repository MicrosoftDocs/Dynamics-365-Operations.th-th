# <a name="manage-store-inventory"></a>จัดการสินค้าคงคลังของร้านค้า

บทความนี้อธิบายชนิดของเอกสารที่คุณสามารถใช้ในการจัดการสินค้าคงคลัง

คุณสามารถใช้ชนิดเอกสารต่อไปนี้ในการจัดการสินค้าคงคลังขององค์กรของคุณ

## <a name="purchase-orders"></a>ใบสั่งซื้อ
ใบสั่งซื้อถูกสร้างที่สำนักงานใหญ่ ถ้าคลังสินค้าขายปลีกจะรวมอยู่ในใบสั่งซื้อ ใบสั่งจะได้รับที่ร้านค้า โดยใช้ POS สมัยใหม่ (MPOS) หรือ Cloud POS ใน Microsoft Dynamics 365 สำหรับการดำเนินงาน - ขายปลีก หลังจากที่มีป้อนปริมาณที่จะได้รับที่ร้านค้า สามารถบันทึกภายในเครื่องสำหรับการแก้ไขเพิ่มเติม หรือปริมาณจะถูกกำหนด และส่งไปยังสำนักงานใหญ่ ที่สำนักงาน ปริมาณที่ได้รับที่ร้านค้าจะแสดงขึ้นใน 365 Dynamics สำหรับการดำเนินการ ในการ**รับทันที**บนใบสั่งซื้อ
ใบสั่งโอนย้าย
---------------

ใบสั่งโอนย้ายสามารถระบุว่า ร้านค้าเฉพาะใดเป็นสถานที่สินค้าถูกจัดส่ง ในกรณีนี้ ใบสั่งโอนย้ายปรากฏขึ้นที่ร้านค้าเป็นคำขอเบิกสินค้าใน MPOS หรือ Cloud POS หลังจากที่มีการเบิกสินค้าปริมาณที่ร้องขอ พวกเขาจะยอมรับ และส่งไปยังสำนักงานใหญ่ ที่สำนักงาน ปริมาณที่รับของที่ร้านค้าจะแสดงขึ้นใน 365 Dynamics สำหรับการดำเนินการ ในการ**จัดส่งทันที**บนใบสั่งโอนย้าย ใบสั่งโอนย้ายอาจระบุว่าร้านค้าเฉพาะใดเป็นสถานที่สินค้าถูกส่งไป ในกรณีนี้ ใบสั่งโอนย้ายปรากฏขึ้นที่ร้านค้าเป็นคำขอรับใน MPOS หรือ Cloud POS หลังจากที่มีป้อนปริมาณที่จะได้รับที่ร้านค้า สามารถบันทึกภายในเครื่องสำหรับการแก้ไขเพิ่มเติม หรือปริมาณจะถูกกำหนด และส่งไปยังสำนักงานใหญ่ ที่สำนักงาน ปริมาณที่ได้รับที่ร้านค้าจะแสดงขึ้นใน 365 Dynamics สำหรับการดำเนินการ ในการ**รับทันที**บนใบสั่งโอนย้าย

## <a name="stock-counts"></a>การตรวจนับสินค้าคงคลัง
การตรวจนับสินค้าคงคลังสามารถตามกำหนดการ หรือไม่ได้ตามกำหนดการ การตรวจนับสินค้าคงคลังตามกำหนดการเริ่มต้นที่สำนักงานใหญ่ ซึ่งระบุสินค้าที่ต้องถูกตรวจนับ สำนักงานใหญ่สร้างเอกสารการตรวจนับที่สามารถรับได้ที่ร้านค้า ที่มีป้อนปริมาณของสต็อกถือครองที่แท้จริงใน MPOS หรือ Cloud POS การตรวจนับสินค้าคงคลังที่ไม่ได้จัดกำหนดการเริ่มต้นที่ร้านค้า และปริมาณของสต็อกถือครองที่แท้จริงจะถูกอัพเดตใน MPOS หรือ Cloud POS ไม่เหมือนกับการตรวจนับสินค้าคงคลังตามกำหนดการ การตรวจนับสินค้าคงคลังที่ไม่ได้จัดกำหนดการไม่มีรายการสินค้าที่กำหนดไว้ล่วงหน้า เมื่อเสร็จสิ้นการตรวจนับสินค้าคงคลังชนิดใดชนิดหนึ่ง จะถูกกำหนด และส่งไปยังสำนักงานใหญ่ ที่สำนักงานใหญ่ การตรวจนับจะถูกตรวจสอบ และลงรายการบัญชี
