# <a name="pos-hardware-peripherals"></a>อุปกรณ์ต่อพ่วงฮาร์ดแวร์ POS

การขายหน้าร้าน (POS) และ POS ระบบคลาวด์ของ Retail Modern สามารถใช้อุปกรณ์ต่อพ่วงฮาร์ดแวร์ POS ที่หลากหลายที่มีอินเทอร์เฟสและตัวเลือกการปรับใช้ที่หลากหลายเพื่อให้บรรลุตามสถานการณ์ทางธุรกิจต่างๆ ของผู้ค้าปลีก 

เพื่อสนับสนุนการเลือกที่กว้างที่สุดของอุปกรณ์ต่างๆ ระหว่างผู้ผลิตและแบบจำลอง POS ใช้อินเทอร์เฟสมาตรฐาน เช่น OLE สำหรับ Retail POS (OPOS) ไดรเวอร์อุปกรณ์ของ Windows และ Application Program Interface (API) จุดบริการของ Windows โดยทั่วไป POS จะทำงานบนอุปกรณ์เหล่านี้ถ้ามีโปรแกรมควบคุมที่เหมาะสม อย่างไรก็ตาม เนื่องจากผู้ผลิตและนักพัฒนาซอฟต์แวร์แต่ละรายนำมาตรฐานเหล่านี้ไปใช้แตกต่างกัน ความสามารถหรือพฤติกรรมในการสนับสนุนจึงมักจะแตกต่างกัน

รายการต่อไปนี้รวมถึงแบบจำลองอุปกรณ์ในประเภทแต่ละคลาสที่มีการทดสอบภายในโดย Microsoft

-   MSR – HP IDRA-334133, Magtek PN - 21073062
-   LineDisplay – Epson M58DC

-   แป้นกดลายเซ็น – Scriptel ST1550

-   ลิ้นชักเงินสด – Star SMD2-1317BK44
