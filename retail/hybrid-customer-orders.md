# <a name="hybrid-customer-orders"></a>ไฮบริดสลีใบสั่งของลูกค้า

ใบสั่งของลูกค้าไฮบริดสลีเป็นใบสั่งใบเดียว ซึ่งประกอบด้วยผลิตภัณฑ์ที่สามารถทำออกจากเก็บจากลูกค้า ตลอดจนผลิตภัณฑ์ที่จะเบิกสินค้า หรือจัดส่งได้ในภายหลัง

ใน 365 Dynamics Microsoft สำหรับการดำเนินงาน - ขายปลีก คุณสามารถเลือกปฏิบัติอย่างใดอย่างหนึ่งตามข่าวผลิตภัณฑ์ทั้งหมด หรือดำเนินการเลือกผลิตภัณฑ์สำหรับใบสั่งของลูกค้า ผลิตภัณฑ์บรรทัดที่ทำเครื่องหมายไว้ในขณะที่ดำเนินการโดยอัตโนมัติออกอินวอยซ์หลังจากที่สร้างใบสั่ง ในทำนองเดียวกันนี้จะเหมือนกันสำหรับใบสั่งที่จะสามารถรับสายโทรศัพท์หลังจากที่สร้างใบสั่ง ยอดเงินครบกำหนดบนไฮบริดสลี ใบสั่งจะถูกกำหนดโดยการเพิ่มเปอร์เซ็นต์ใบนำฝากธนาคารในการรับบรรทัดการจัดส่งสินค้า ด้วยยอดเงินเต็มของยอดยกบรรทัด สำหรับใบสั่งไฮบริดสลี ระบบสลับไปมาระหว่างโหมดสั่งลูกค้าและยกยอดกำไรเงินสดเป็นดังนี้:

-   ถ้าผลิตภัณฑ์ทั้งหมดในรถเข็นถูกตั้งค่าเป็น**ดำเนินการจัดส่ง**ใบสั่งจะถูกจัดการเป็นธุรกรรมเงินสดและภาระงาน
-   ถ้าบรรทัดใด ๆ หรือทั้งหมดในรถเข็นถูกตั้งค่าเป็นอย่างใดอย่างหนึ่ง**รับ**หรือ**การจัดส่งการจัดส่ง**ใบสั่งจะถูกจัดการเป็นธุรกรรมใบสั่งของลูกค้า

ถ้ามีเลือกรายการรถเข็น และ**เบิกสินค้าที่เลือก**,**การจัดส่งที่เลือก**หรือ**ดำเนินการเลือก**ถูกเลือก บรรทัดเฉพาะรถเข็นตั้ง ด้วยวิธีการจัดส่ง ในกรณีดังกล่าว ขั้นตอนปลายน้ำของการดำเนินงานต่อไปตามปกติ อย่างไรก็ตาม ถ้า**เบิกสินค้าที่เลือก**,**การจัดส่งที่เลือก**หรือ**ดำเนินการเลือก**ไว้ โดยไม่มีรถเข็นของบรรทัดเลือกไว้ การเปิดหน้าใหม่ที่แสดงรายการทั้งหมดในรายการในรถเข็น บนหน้าจอนั้น คุณสามารถเลือกหลายบรรทัดพร้อมกันสำหรับการตั้งค่าวิธีการจัดส่ง เมื่อคุณใช้วิธีนั้นสำหรับการเลือกบรรทัด วิธีการจัดส่งใด ๆ ก่อนหน้านี้ที่ได้ถูกกำหนดให้กับบรรทัดจะถูกแทน
