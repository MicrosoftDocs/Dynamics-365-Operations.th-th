# <a name="create-and-associate-a-device"></a> สร้างและเชื่อมโยงอุปกรณ์

[!include[task guide banner](../includes/task-guide-banner.md)]

กระบวนการนี้นำไปสู่วิธีการสร้างอุปกรณ์ใหม่และเชื่อมโยงกับการลงทะเบียน  สันนิษฐานว่าการลงทะเบียนถูกสร้างขึ้นแล้ว   งานนี้ใช้บริษัท USRT ในข้อมูลสาธิต  งานนี้มีไว้สำหรับผู้ที่มีบทบาทดูแลระบบ, ผู้เชี่ยวชาญด้านไอทีหรือฝ่ายไอทีระบบขายปลีก, และผู้จัดการร้าน

1. ไปที่ การขายปลีกและการค้า > .. > อุปกรณ์
2. คลิก สร้าง
3. ในฟิลด์รหัสอุปกรณ์ ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ชนิดอุปกรณ์ ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
5. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
6. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
7. ในฟิลด์หมายเลขเครื่องบันทึกเงินสด ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
8. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
9. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
10. คลิก บันทึก
