# <a name="retail-home-page"></a>โฮมเพจการขายปลีก

[!include[banner](includes/banner.md)]

Dynamics 365 for Operations – Retail นำเสนอประสบการณ์การเลือกซื้อของที่สมบูรณ์แบบด้วยโซลูชันช่องทาง Omni ที่ไม่มีขอบเขตและมีความแตกต่างที่ทันสมัย แบบเคลื่อนที่ และเป็นสากล ความสามารถที่ครอบคลุม ตั้งแต่ชุดซอฟต์แวร์การขายหน้าร้าน (POS) ที่ทันสมัย การดำเนินงานร้านค้า การจัดซื้อสินค้า พาณิชย์อิเล็กทรอนิกส์ ศูนย์บริการ การตลาด และการดูแลลูกค้า จนถึงห่วงโซ่อุปทาน การเงิน และอื่นๆ สามารถนำมาปรับใช้ในลักษณะที่เหมาะสมที่สุดสำหรับธุรกิจของคุณ

หน้านี้ชี้ไปที่ทรัพยากรที่จะช่วยให้คุณใช้ Microsoft Dynamics 365 for Operations - Retail ได้อย่างมีประสิทธิผลมากยิ่งขึ้น ในบานหน้าต่างด้านซ้าย เลือกพื้นที่คุณลักษณะเพื่อเรียนรู้เพิ่มเติม 

## <a name="set-up-retail-channels"></a>ตั้งค่าช่องทางการขายปลีก
เมื่อคุณตั้งค่าประสบการณ์การขายปลีกของคุณ คุณจะสามารถตัดสินใจเกี่ยวกับวิธีการกำหนดและตั้งค่าคอนฟิกร้านค้า ทั้งร้านค้าจริง ร้านค้าออนไลน์ และศูนย์บริการได้ 

ร้านค้าปลีกแต่ละร้านสามารถมีวิธีการชำระเงิน วิธีการจัดส่ง กลุ่มราคา บัญชีรายได้/ค่าใช้จ่าย เครื่องบันทึกเงินสด และพนักงานเป็นของตนเอง หลังจากที่คุณสร้างร้านค้าปลีกของคุณ คุณสามารถจัดประเภทผลิตภัณฑ์ที่คุณต้องการให้ร้านค้านำเข้ามาได้ คุณยังสามารถกำหนดราคาและส่วนลดเฉพาะร้านค้าเพื่อใช้กับผลิตภัณฑ์ที่จำหน่ายในร้านค้าได้

-   [กำหนดช่องทางการขายปลีก](define-maintain-retail-channels.md)
-   [ภาพรวมร้านค้าออนไลน์](online-stores.md)

## <a name="set-up-a-call-center"></a>ตั้งค่าศูนย์บริการ
คุณสามารถตั้งค่าศูนย์บริการเป็นชนิดของช่องทางการขายปลีก ในศูนย์บริการ ผู้ปฏิบัติงานจะรับใบสั่งจากลูกค้าผ่านทางโทรศัพท์ และสร้างใบสั่งขาย เพื่อช่วยให้พนักงานของคุณรับใบสั่งและจัดการการบริการลูกค้าผ่านกระบวนการเติมสินค้าตามใบสั่ง คุณสามารถตั้งค่าสิ่งที่จะรวมอยู่ในใบสั่งขาย วิธีการควบคุมการกำหนดราคา และการเข้าถึงข้อมูลเกี่ยวกับลูกค้า ผลิตภัณฑ์ และการกำหนดราคา สำหรับแต่ละศูนย์บริการ คุณยังสามารถตั้งค่าผู้ใช้ วิธีการชำระเงิน กลุ่มราคา มิติทางการเงิน และวิธีการจัดส่งในแบบของตนเองได้

-   [ฟังก์ชันข้อมูลศูนย์บริการ](call-center-functionality.md)
-   [สร้างแค็ตตาล็อกของศูนย์บริการ](create-call-center-catalogs.md)

## <a name="define-your-pos-solutions"></a>กำหนดโซลูชัน POS ของคุณ
Retail Modern POS (MPOS) เป็นแอพลิเคชันระบบขายหน้าร้านสำหรับพีซี แท็บเล็ต และโทรศัพท์ พนักงานขายสามารถประมวลผลธุรกรรมการขายและใบสั่งลูกค้า ดำเนินกิจกรรมที่สร้างความสัมพันธ์อันดีกับลูกค้า และดำเนินงานประจำวันและบริหารสินค้าคงคลัง โดยการใช้อุปกรณ์เคลื่อนที่ที่ใด ๆ ในร้านค้า เช่น ที่เครื่องบันทึกเงินสดที่ใช้พีซี Cloud POS คือโปรแกรม Modern POS ที่สามารถเปิดใช้ในเบราว์เซอร์มาตรฐานทั่วไปได้ 

มีส่วนประกอบหลายรายการที่คุณจะตั้งค่าเพื่อกำหนดความสัมพันธ์ระหว่างร้านค้า เครื่องบันทึกเงินสด (POS) ภายในร้านค้า และอุปกรณ์ต่อพ่วงขายปลีกที่เครื่องบันทึกเงินสดใช้ในการประมวลผลธุรกรรม นอกจากนี้คุณยังจะต้องกำหนดลักษณะของอินสแตนซ์เฉพาะของ POS รวมถึงโพรไฟล์ฮาร์ดแวร์หรือการตั้งค่าสำหรับอุปกรณ์ต่อพ่วงขายปลีกที่จะใช้ที่เครื่องบันทึกเงินสด ร้านค้าที่แม็ปไว้กับที่เครื่องบันทึกเงินสด ประสบการณ์ทางภาพสำหรับผู้ใช้ที่ล็อกออนเข้าสู่เครื่องบันทึกเงินสดนั้น และฟังก์ชันการใช้งานที่มีการเปิดใช้งานในเครื่องบันทึกเงินสด คุณยังสามารถกำหนดให้เครื่องบันทึกเงินสดสามารถทำงานในสถานะที่ยกเลิกการเชื่อมต่อหรือออฟไลน์ได้

-   [กำหนดและรักษาไคลเอนต์ช่องทาง และเครื่องบันทึกเงินสดและสถานีฮาร์ดแวร์](define-maintain-channel-clients-registers-hw-stations.md)
-   [ตั้งค่าคอนฟิกและติดตั้ง MPOS](retail-modern-pos-device-activation.md)

## <a name="set-up-a-customer-loyalty-program"></a>ตั้งค่าโปรแกรมตอบแทนลูกค้าสมาชิก

## <a name="set-up-retail-products"></a>ตั้งค่าผลิตภัณฑ์ขายปลีก
ก่อนที่คุณสามารถเสนอผลิตภัณฑ์จำหน่ายในช่องทางการขายปลีก คุณต้องสร้างและตั้งค่าคอนฟิกผลิตภัณฑ์ใน Microsoft Dynamics 365 for Operations คุณสามารถสร้างผลิตภัณฑ์ กำหนดคุณสมบัติของผลิตภัณฑ์และแอททริบิวต์ และกำหนดผลิตภัณฑ์ให้กับลำดับชั้นประเภทการขายปลีก การทำให้ผลิตภัณฑ์พร้อมใช้งานสำหรับช่องทางการขายปลีก และเพิ่มลงในการจัดประเภทที่ใช้งานอยู่ คุณต้องนำผลิตภัณฑ์ออกใช้กับนิติบุคคลที่พร้อมใช้งาน
