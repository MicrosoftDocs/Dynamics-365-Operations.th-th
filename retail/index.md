# <a name="retail-home-page"></a>โฮมเพจขายปลีก

[!include[banner](includes/banner.md)]

ขายปลีก dynamics 365 สำหรับการดำเนินงาน – ส่งแบบสมบูรณ์ซื้อของประสบการณ์การใช้งาน ตัวอย่างราบและ differentiating omni สถานีโซลูชันที่เป็นแบบสมัยใหม่ โทรศัพท์มือถือ และสากล สิ้นสุดเพื่อสิ้นสุดความสามารถ จากจุดสมัยใหม่ของวิธีแก้ไขปัญหาในซอฟต์แวร์ขาย (POS) การดำเนินการจัดเก็บ การจัดการสินค้า อีคอมเมิร์ซ ศูนย์บริการ การตลาด และดูแลเอาใจ ใส่ลูกค้า การใส่ห่วงโซ่ การเงิน และ อื่น ๆ สามารถปรับใช้ในลักษณะที่เหมาะสมสุดสำหรับธุรกิจของคุณ

จุดหน้านี้ไปยังทรัพยากรที่จะช่วยให้คุณใช้ Microsoft Dynamics 365 สำหรับการดำเนินงาน - ขายปลีกอย่างมีประสิทธิภาพมากขึ้น ในบานหน้าต่างด้านซ้าย เลือกพื้นที่ลักษณะการทำงานเพื่อศึกษาเพิ่มเติมเกี่ยวกับ 

## <a name="set-up-retail-channels"></a>ตั้งค่าช่องทางการขายปลีก
เมื่อคุณตั้งค่าประสบการณ์การขายปลีกของคุณ คุณจะตัดสินใจว่า วิธีการกำหนด และการตั้งค่าคอนฟิกลายอิฐและปูนร้านค้าของคุณ (เรียกว่า*ร้านค้าปลีก*ใน 365 Dynamics สำหรับการดำเนินงาน) และการดำเนินการของร้านค้าออนไลน์ 

แต่ละร้านค้าปลีกสามารถมีตัวเองวิธีการชำระเงิน กลุ่มราคา POS โซลูชัน บัญชีกำไรขาดทุน บัญชีค่าใช้จ่าย และพนักงาน หลังจากที่คุณสร้างร้านค้าปลีกของคุณ คุณกำหนดผลิตภัณฑ์ที่คุณต้องการให้ร้านค้าสามารถดำเนินการ กำหนดพนักงาน เครื่องบันทึกเงินสด และลูกค้าไปยังร้านค้า  และสุดท้ายให้คุณเพิ่มร้านค้าใหม่กับลำดับชั้นขององค์กร

เมื่อคุณสร้างร้านค้าออนไลน์ของคุณ คุณต้องตั้งค่าคอมโพเนนต์เช่นจัดประเภท คุณลักษณะ วิธีการชำระเงิน และวิธีการจัดส่ง คุณยังสามารถกำหนดราคา ส่งเสริมการขาย ส่วนลด และเงื่อนไขการจัดส่งที่เฉพาะเจาะจงไปยังร้านค้าออนไลน์

-   [กำหนดช่องทางการขายปลีก](define-maintain-retail-channels.md)
-   [Online store overview](online-stores.md)

## <a name="set-up-a-call-center"></a>ตั้งค่าศูนย์บริการ
คุณสามารถตั้งค่าศูนย์การโทรเป็นชนิดของช่องทางการขายปลีก ในศูนย์บริการ ผู้ปฏิบัติงานจะรับใบสั่งจากลูกค้าผ่านทางโทรศัพท์ และสร้างใบสั่งขาย เพื่อช่วยให้พนักงานของคุณใช้ใบสั่ง และจัดการการบริการลูกค้าตลอดทั้งกระบวนการเติมสินค้าตามใบสั่ง คุณสามารถตั้งค่าอะไรจะประกอบด้วยใบสั่งขาย วิธีการกำหนดราคาจะถูกควบคุม และสามารถเข้าถึงข้อมูลเกี่ยวกับลูกค้า ผลิตภัณฑ์ และการกำหนดราคา สำหรับแต่ละศูนย์โทร คุณสามารถตั้งค่าผู้ใช้ วิธีการชำระเงิน กลุ่มราคา มิติทางการเงิน และวิธีการจัดส่ง

-   [Call center functionality](call-center-functionality.md)
-   [Create call center catalogs](create-call-center-catalogs.md)

## <a name="define-your-pos-and-mpos-solutions"></a>กำหนดโซลูชัน POS และ MPOS ของคุณ
POS สมัยใหม่ของขายปลีก (MPOS) เป็นแอพลิเคชันระบบขายหน้าร้านสำหรับพีซี tablets และโทรศัพท์ พนักงานขายสามารถประมวลผลธุรกรรมการขาย ใบสั่งลูกค้า และดำเนินงานประจำวัน และบริหารสินค้าคงคลังกับอุปกรณ์มือถือที่ใดก็ได้ ในร้านค้า เช่นเดียว กับ ที่เครื่องบันทึกเงินสดที่ใช้พีซี เมฆ POS ได้ POS ที่ใช้เบราว์เซอร์ 

มีคอมโพเนนต์ต่าง ๆ ที่คุณจะตั้งค่าการกำหนดความสัมพันธ์ระหว่างร้านค้า เครื่องบันทึกเงินสด POS หรือสถานีภายในร้านค้า และอุปกรณ์ต่อพ่วงขายปลีกที่ที่ทำการลงทะเบียน หรือสถานีที่ใช้ในการประมวลผลธุรกรรม นอกจากนี้คุณจะกำหนดลักษณะของอินสแตนซ์ที่ระบุของ POS รวมทั้งโพรไฟล์ของฮาร์ดแวร์หรือการตั้งค่าสำหรับอุปกรณ์ต่อพ่วงขายปลีกที่จะใช้ในการลงทะเบียน และร้านค้าที่ลงทะเบียนถูกแมปกับ การแสดงผลทางภาพระดับสำหรับผู้ใช้ที่เข้าสู่ระบบที่มีการลงทะเบียน

-   [ตั้งค่าคอนฟิก และการติดตั้ง MPOS](retail-modern-pos-device-activation.md)

## <a name="set-up-a-customer-loyalty-program"></a>ตั้งค่าโปรแกรมตอบแทนลูกค้าสมาชิกลูกค้า

## <a name="set-up-retail-products"></a>ตั้งค่าผลิตภัณฑ์ขายปลีก
ก่อนที่คุณสามารถเสนอผลิตภัณฑ์สำหรับการขายในช่องทางการขายปลีกของคุณ คุณจำเป็นต้องสร้าง และตั้งค่าคอนฟิกผลิตภัณฑ์ใน 365 Dynamics สำหรับการดำเนินงาน คุณสามารถสร้างผลิตภัณฑ์ กำหนดคุณสมบัติของผลิตภัณฑ์และแอททริบิวต์ และกำหนดผลิตภัณฑ์ให้กับลำดับชั้นประเภทการขายปลีก เมื่อต้องการทำให้ผลิตภัณฑ์พร้อมใช้งานสำหรับช่องทางการขายปลีกของคุณ และเพิ่มลงในการจัดประเภทที่ใช้งานอยู่ คุณจะปล่อยผลิตภัณฑ์ไปยังหน่วยงานตามกฎหมายซึ่งจะมีอยู่แล้ว
