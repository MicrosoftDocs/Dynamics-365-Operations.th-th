# <a name="add-a-recommendations-control-to-the-transaction-page-on-a-pos-device"></a>เพิ่มตัวควบคุมคำแนะนำในหน้าธุรกรรมบนอุปกรณ์ POS

หัวข้อนี้อธิบายวิธีการเพิ่มตัวควบคุมคำแนะนำบนหน้าจอธุรกรรมบนจุดขาย (POS) อุปกรณ์ที่ใช้โปรแกรมออกแบบโครงร่างหน้าจอใน Microsoft Dynamics 365 สำหรับการดำเนินงาน

คุณสามารถแสดงคำแนะนำผลิตภัณฑ์บน POS อุปกรณ์ของคุณเมื่อคุณใช้ Microsoft Dynamics 365 สำหรับการดำเนินงาน *คำแนะนำ*เป็นสินค้าที่ลูกค้าของคุณอาจสนใจโดยยึด บนของประวัติการซื้อ สินค้าในรายการของพวกเขาพร และสินค้าที่ลูกค้ารายอื่นซื้อแบบออนไลน์ และ ในร้านค้าลายอิฐ และปูน เมื่อต้องการแสดงคำแนะนำผลิตภัณฑ์ คุณต้องการเพิ่มตัวควบคุมไปยังหน้าจอธุรกรรมโดยใช้โปรแกรมออกแบบโครงร่างหน้าจอ

## <a name="open-layout-designer"></a>เปิดโปรแกรมออกแบบโครงร่าง
1.  ไปที่**พาณิชย์และผู้ค้าปลีก**&gt;**ตั้งค่าช่องสัญญาณ**&gt;**ตั้งค่า POS**&gt;**POS**&gt;**โครงร่างหน้าจอ**
2.  ใช้ตัวกรองอย่างรวดเร็วเพื่อค้นหาหน้าจอที่คุณต้องการเพิ่มตัวควบคุม ตัวอย่างเช่น ตัวกรองบน**รหัสโครงร่างหน้าจอ**เขตข้อมูลโดยใช้ค่าของ 'F2CP16:9M'
3.  ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ ตัวอย่างเช่น เลือก ' ชื่อ: รหัสโครงร่างหน้าจอ F2CP16:9M: F2CP16:9M'
4.  คลิ**ผู้ออกแบบเค้าโครง**
5.  ทำตามพร้อมท์เพื่อเปิดใช้งานโปรแกรมออกแบบโครงร่าง เมื่อได้รับพร้อมท์สำหรับข้อมูลประจำตัว ป้อนข้อมูลประจำตัวเดียวกันที่ถูกใช้เมื่อมีการเปิดใช้โปรแกรมออกแบบโครงร่างจาก**โครงร่างหน้าจอ**หน้า
6.  เมื่อคุณเข้าสู่ระบบ หน้าคล้ายกับด้านล่างปรากฏขึ้น โครงร่างจะแตกต่างกันโดยขึ้นอยู่กับการกำหนดเองที่ทำไว้สำหรับร้านค้าของคุณ

[![screenlayout pic 1](./media/screenlayout-pic-1.png)](./media/screenlayout-pic-1.png)เลือกตัวเลือกการแสดงผล

มีสองตัวเลือกการตั้งค่าคอนฟิกพร้อมใช้งาน เลือกตัวเลือกที่ดีที่สุดสำหรับร้านค้าของคุณ และทำตามคำแนะนำที่เหลือเพื่อเสร็จสิ้นการตั้งค่าตัวควบคุม มีสองอ็อพชัน:
-   คำแนะนำจะปรากฏขึ้นเสมอ
-   A**คำแนะนำ**แท็บปรากฏขึ้นในกริดทางด้านขวาของหน้าจอ

#### <a name="to-make-recommendations-always-visible"></a>เมื่อต้องการสร้างคำแนะนำที่มองเห็นได้เสมอ

1.  ลดความสูงของพื้นที่รายละเอียดของบรรทัดธุรกรรมเพื่อให้มีความสูงเป็นแผงลูกค้าแสดงที่ด้านซ้าย [](./media/pic-2.png)[![screenlayout-pic-2](./media/screenlayout-pic-2.png)](./media/screenlayout-pic-2.png)
2.  จากเมนูทางด้านซ้าย ลาก และปล่อยตัวควบคุมคำแนะนำทางระหว่างพื้นที่รายละเอียดของบรรทัดธุรกรรมและกริดปุ่มในศูนย์กลางด้านล่างของหน้าจอของธุรกรรม ปรับขนาดตัวควบคุมเพื่อให้พอดีในช่องนั้น [](./media/pic-3.png)[![screenlayout-pic-3](./media/screenlayout-pic-3.png)](./media/screenlayout-pic-3.png)
3.  คลิก**X**เพื่อบันทึก และออกจากโปรแกรมออกแบบโครงร่าง
4.  ใน 365 Dynamics สำหรับการดำเนินงาน ไป**ขายปลีกและพาณิชย์**&gt;**ไอทีขายปลีก**&gt;**กำหนดการกระจาย**
5.  เลือกในรายการ**1090 การลงทะเบียน**
6.  คลิ**เรียกใช้เดี๋ยวนี้**

#### <a name="to-add-a-recommendations-tab-to-the-button-grid-on-the-right-side-of-the-screen"></a>เมื่อต้องการเพิ่มแท็บคำแนะนำเกี่ยวกับกริดปุ่มทางด้านขวาของหน้าจอ

1.  คลิกขวาในพื้นที่ว่างที่อยู่ด้านล่างของแท็บสุดท้ายในกริดปุ่มที่อยู่ทางด้านขวาของหน้า
2.  คลิ**กำหนด**[![pic-5](./media/pic-5.png)](./media/pic-5.png)
3.  คลิ**แท็บใหม่**
4.  ค้นหาแท็บใหม่ที่คุณเพิ่งเพิ่มเข้ามา คุณอาจต้องเลื่อนลง
5.  ในการ**เนื้อหา**หล่นลง เลือก**แนะนำผลิตภัณฑ์** [![pic-6](./media/pic-6.png)](./media/pic-6.png)
6.  ในการ**ป้ายชื่อ**ฟิลด์ พิมพ์ชื่อสำหรับแท็บคำแนะนำ ตัวอย่าง พิมพ์ 'แนะนำผลิตภัณฑ์'
7.  ในการ**อิมเม**ฟิลด์ เลือกรูปที่จะปรากฏบนแท็บ
8.  Click **OK**. แท็บใหม่ปรากฏในกริดปุ่ม
9.  คลิก**X**เพื่อบันทึก และออกจากโปรแกรมออกแบบโครงร่าง
10. ใน 365 Dynamics สำหรับการดำเนินงาน ไป**ขายปลีกและพาณิชย์**&gt;**ไอทีขายปลีก**&gt;**กำหนดการกระจาย**
11. เลือกในรายการ**1090 การลงทะเบียน**
12. คลิ**เรียกใช้เดี๋ยวนี้**
