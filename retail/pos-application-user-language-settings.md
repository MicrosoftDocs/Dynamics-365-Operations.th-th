# <a name="pos-application-and-user-language-settings"></a>แอพลิเคชัน POS และการตั้งค่าภาษาผู้ใช้

หัวข้อนี้อธิบายวิธีการเปลี่ยนการตั้งค่าภาษาใน POS สมัยใหม่ของขายปลีก (MPOS) และ Cloud POS

<a name="overview"></a>ภาพรวม
========

POS สมัยใหม่ของขายปลีก (MPOS) และ POS เมฆสนับสนุนสภาพแวดล้อมที่การตั้งค่าภาษาและการแปลที่แตกต่างกันระหว่างการตั้งค่าร้านค้าและผู้ใช้ ตัวอย่างเช่น ร้านค้าไม่สามารถอยู่ในบริเวณที่ภาษาอังกฤษกันมากสุดสำหรับลูกค้าของพวกเขา แต่ผู้ปฏิบัติงานบางอย่างที่ต้องการใช้แอพลิเคชัน มีการแปลภาษาฝรั่งเศส

## <a name="data-language"></a>ข้อมูลภาษา
โดยไม่คำนึงถึงการตั้งค่าของผู้ใช้ MPOS และ POS เมฆจะเสมอใช้การตั้งค่าภาษาของร้านค้าเพื่อกำหนดคำแปลที่ใช้กับข้อมูล ซึ่งจะให้แน่ใจว่า ผู้ใช้ทั้งหมดและลูกค้าจะมีประสบการณ์ที่สอดคล้องกัน  ตัวอย่างของข้อมูลได้แก่:

-   ผลิตภัณฑ์
-   แอททริบิวต์และมูลค่า
-   ชื่อประเภท
-   ใบเสร็จรับเงินของธุรกรรมที่พิมพ์ออกมาหรือส่งทางอีเมล
-   ชื่อวิธีการชำระเงิน
-   ข้อความแสดงรายการ

ภาษาของร้านค้าจะยังสามารถใช้สำหรับ POS ล็อกหน้าจอหลัก เนื่องจากไม่รู้จักผู้ใช้ก่อนการเข้าสู่ระบบ ถ้าการแปลไม่พร้อมใช้งานสำหรับภาษาของร้านค้า POS จะแปลงกลับไปเป็นภาษาของบริษัท

### <a name="configuring-the-stores-language-setting"></a>การตั้งค่าคอนฟิกการตั้งค่าภาษาของร้านค้า

ตั้งการตั้งค่าภาษาของร้านค้าจาก**ร้านค้าการขายปลีกทั้งหมด**ในการ**ร้านค้าปลีก**หน้าภายใต้ ** ทั่วไป&gt;การตั้งค่าภูมิภาค&gt;ภาษา ** ใช้แบบหล่นลงให้เลือกภาษาสำหรับแต่ละร้านค้า

## <a name="user-interface-language"></a>ภาษาของอินเทอร์เฟสผู้ใช้
การตั้งค่าภาษาของผู้ใช้ POS กำหนดคำแปลที่ใช้ในอินเทอร์เฟซสำหรับผู้ใช้แอพลิเคชัน ข้อมูลนี้รวมถึงป้ายชื่อ เมนู และรายการทั้งหมดที่จะไม่มีพิจารณาข้อมูล ข้อยกเว้นคือ ข้อความที่จะแสดงอยู่ในกริดปุ่ม POS กริดปุ่มไม่สนับสนุนแปล เพื่อให้พวกเขาเสมอจะแสดงข้อความตามที่กำหนดไว้บนปุ่ม เพื่อที่จะสนับสนุนปุ่มแปล คุณจะมีการคัดลอก และรักษากริดปุ่มแยกต่างหาก และกำหนดให้กับผู้ใช้ตามความเหมาะสม

### <a name="configuring-the-users-language-setting"></a>การตั้งค่าคอนฟิกการตั้งค่าภาษาของร้านค้า

ตั้งการตั้งค่าภาษาของผู้ใช้ POS จาก**ผู้ปฏิบัติงานทั้งหมด**ในการ**ผู้ปฏิบัติงาน**หน้าภายใต้**ขายปลีก&gt;ภาษา**  ไม่ได้ถูกตั้งค่าบนแท็บส่วนกำหนดค่าหลัก  ไม่มีใช้การตั้งค่านี้ โดย POS ถ้าภาษาของผู้ใช่ไม่ได้ถูกตั้งค่าหรือมีการตั้งค่าไปยังภาษาที่การแปลไม่พร้อมใช้งาน POS จะเปลี่ยนกลับไปยังภาษาของร้านค้า
