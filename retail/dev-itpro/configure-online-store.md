# <a name="configure-an-online-store"></a>ตั้งค่าคอนฟิกร้านค้าออนไลน์

บทความนี้แสดงลิงค์ไปยังหัวข้อที่จะช่วยให้คุณจากส่วนกลางที่ตั้งค่าคอนฟิก และจัดการ และร้านค้าออนไลน์จาก 365 Dynamics Microsoft สำหรับการดำเนินงาน

หัวข้อที่แสดงในตารางต่อไปนี้ช่วยคุณในการตั้งค่าคอนฟิก 365 Dynamics สำหรับการดำเนินงาน - คอมโพเนนต์ของการขายปลีกและร้านค้าปลีกออนไลน์ใน 365 Dynamics สำหรับไคลเอ็นต์การดำเนินการ
