# <a name="analyze-sales-trends-and-patterns"></a>วิเคราะห์แนวโน้มและรูปแบบการขาย

คุณสามารถศึกษาแนวโน้มการขายและรูปแบบในเวลาจริงใน 365 Dynamics Microsoft สำหรับการดำเนินงาน - ขายปลีก

เป็นส่วนหนึ่งของ Dynamics 365 สำหรับการดำเนินการ ผู้ใช้สามารถศึกษาแนวโน้มการขายและรูปแบบในเวลาจริงข้ามลำดับชั้นขององค์กรในระดับต่าง ๆ ระยะเวลาหนึ่งปี โดยใช้ออกกล่อง**ช่องทางการขายตามปี**รายงานได้ คุณสามารถเปิดรายงานนี้จากตำแหน่งที่ตั้งต่อไปนี้:

-   **จัดการร้านค้าปลีก**Dynamics 365 พื้นที่ทำงานสำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สถานี**&gt;**จัดการร้านค้าปลีก**&gt;**รายงาน**&gt;**ช่องทางการขายตามปีรายงาน**
-   **เงินของร้านค้าปลีก**Dynamics 365 พื้นที่ทำงานสำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สถานี**&gt;**เงินของร้านค้าปลีก**&gt;**รายงาน**&gt;**ช่องทางการขายตามปีรายงาน**
-   **การสอบถามและรายงาน**ส่วน 365 Dynamics สำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สอบถามและรายงาน**&gt;**รายงานการขาย**&gt;**ช่องทางการขายตามปีรายงาน**

ผู้ใช้ยังสามารถศึกษาแนวโน้มและรูปแบบตามชั่วโมงระหว่างลำดับชั้นขององค์กรระดับต่างๆ ณ รอบระยะเวลาที่เลือก โดยใช้รายงาน **รายงานการขายผ่านช่องทางตามชั่วโมง** นอกกรอบ คุณสามารถเปิดรายงานนี้จากตำแหน่งที่ตั้งต่อไปนี้:

-   **จัดการร้านค้าปลีก**Dynamics 365 พื้นที่ทำงานสำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สถานี**&gt;**จัดการร้านค้าปลีก**&gt;**รายงาน**&gt;**ช่องทางการขายตามรายงานชั่วโมง**
-   **เงินของร้านค้าปลีก**Dynamics 365 พื้นที่ทำงานสำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สถานี**&gt;**เงินของร้านค้าปลีก**&gt;**รายงาน**&gt;**ช่องทางการขายตามรายงานชั่วโมง**
-   **การสอบถามและรายงาน**ส่วน 365 Dynamics สำหรับการดำเนินงาน&gt;**ขายปลีกและพาณิชย์**&gt;**สอบถามและรายงาน**&gt;**รายงานการขาย**&gt;**ช่องทางการขายตามรายงานชั่วโมง**
