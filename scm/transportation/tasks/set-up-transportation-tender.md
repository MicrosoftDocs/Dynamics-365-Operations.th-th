# <a name="set-up-a-transportation-tender"></a>ตั้งค่าวิธีการชำระเงินการขนส่ง

[!include[task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงวิธีการตั้งค่าวิธีการชำระเงินการขนส่ง  ซึ่งปกติจะดำเนินการโดยผู้ประสานงานการขนส่ง  คุณสามารถใช้กระบวนงานนี้ในบริษัทข้อมูลสาธิต USMF 


## <a name="select-a-route"></a>เลือก กระบวนการผลิต
1. ไปที่ การจัดการการขนส่ง > การวางแผน > เวิร์กเบนช์การวางแผนการบรรทุก
2. ล้างข้อมูลกล่องกาเครื่องหมาย ซ่อนสินค้าที่จัดส่งและที่รับ
3. เลือกบรรทัดที่มีรหัสจำนวนงานในศูนย์การผลิต 00006
4. คลิก การจัดอันดับและการกำหนดเส้นทาง
5. คลิกกระบวนการผลิต

## <a name="create-the-transportation-tender"></a>สร้างวิธีการชำระเงินการขนส่ง
1. คลิก วิธีการชำระเงินการขนส่ง
2. คลิก สร้าง
3. ขยายส่วน ทั่วไป
4. ในฟิลด์อัตราที่ร้องขอ ให้ป้อนหมายเลข
5. คลิก บันทึก
6. คลิก ปรับปรุงสถานะ
7. คลิก ส่ง 
8. เลือก กระบวนการผลิต
