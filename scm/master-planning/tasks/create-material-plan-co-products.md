# <a name="create-a-material-plan-for-co-products"></a>สร้างแผนวัสดุสำหรับผลิตภัณฑ์ร่วม

[!include[task guide banner](../../includes/task-guide-banner.md)]

ผู้วางแผนการผลิตจะวางแผนความต้องการวัสดุสำหรับสินค้าที่เป็นสูตรผลิตภัณฑ์ร่วม  บริษัทข้อมูลสาธิตที่เคยสร้างขั้นตอนนี้คือ USP2


## <a name="create-requirement-for-a-co-product"></a>สร้างความต้องการสำหรับผลิตภัณฑ์ร่วม
1. ไปที่แดชบอร์ดเริ่มต้น 
2. คลิกที่การประมวลผลและการสอบถามใบสั่งขาย 
3. คลิก สร้าง
4. คลิกที่ใบสั่งขาย 
5. ในฟิลด์บัญชีลูกค้า ให้พิมพ์ค่า 
    * ตัวอย่าง: US-001  
6. คลิก ตกลง
7. ในฟิลด์หมายเลขสินค้า ให้พิมพ์ค่า
    * ตัวอย่าง : P6003  
8. ในฟิลด์ ปริมาณ ให้ป้อนตัวเลข
    * ตัวอย่าง : 50000  
9. คลิก บันทึก

## <a name="create-a-material-plan-for-co-products"></a>สร้างแผนวัสดุสำหรับผลิตภัณฑ์ร่วม
1. คลิกที่งานการวางแผนหลัก 
2. ในฟิลด์แผน ให้คลิกที่ปุ่มดรอปดาวน์เพื่อเปิดการค้นหา 
3. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
    * ตัวอย่าง: แผนหลัก  
4. คลิก เรียกใช้
5. ขยายหรือยุบส่วนของเรกคอร์ดที่จะรวม 
6. คลิกที่ตัวกรองข้อมูล
7. ในรายการ ให้เลือกแถวสำหรับฟิลด์ = หมายเลขสินค้า 
8. ในฟิลด์กรณี ให้ป้อนค่า
    * ตัวอย่าง : P6003  
9. คลิก ตกลง
10. คลิก ตกลง
11. คลิกที่แผนการใบสั่ง 
12. ใช้ตัวกรองข้อมูลด่วนเพื่อค้นหาเรกคอร์ด  ตัวอย่างเช่น กรองฟิลด์หมายเลขสินค้าด้วยค่า 'P6000'
    * กรองด้วยสูตรสินค้าที่เป็นผลิตภัณฑ์ร่วมของสินค้าที่คุณได้สร้างใบสั่งขายให้   
13. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
    * เลือกแถวใดๆที่ถูกตีกลับโดยตัวกรอง   
14. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
15. ขยายหรือยุบส่วนความเชื่อมโยงความต้องการกับการจัดซื้อ 
16. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
    * คำสั่งซื้อที่วางแผนไว้จะเชื่อมโยงกับใบสั่งขายสำหรับสินค้าร่วม  
17. ปิดหน้า
