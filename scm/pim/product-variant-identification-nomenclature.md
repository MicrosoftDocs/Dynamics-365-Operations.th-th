# <a name="product-number-nomenclature"></a>ระบบการตั้งชื่อสำหรับหมายเลขผลิตภัณฑ์

[!include[banner](../includes/banner.md)]


หัวข้อนี้อธิบายวิธีที่คุณสามารถตั้งค่าระบบการตั้งชื่อหมายเลขผลิตภัณฑ์เพื่อแทนที่รูปแบบถาวร [หมายเลขผลิตภัณฑ์หลัก - Configuration - Size - Color - Style] ด้วยรูปแบบเป้าหมายที่มีหมายเลขผลิตภัณฑ์หลัก มิติของผลิตภัณฑ์ที่ใช้งานอยู่ และตัวกำหนดเขตข้อความที่คุณเลือก คุณยังสามารถสร้างระบบการตั้งชื่อเพื่อระบุการจัดโครงแบบที่สร้างขึ้นโดยตัวจัดโครงแบบผลิตภัณฑ์ตามข้อจำกัดได้อีกด้วย ระบบการตั้งชื่อเหล่านี้อาจประกอบด้วยแอททริบิวต์ที่คุณเลือก

ระบบการตั้งชื่อหมายเลขผลิตภัณฑ์ย่อยใหม่ช่วยให้คุณสามารถรวมเซ็กเมนต์ในตัวระบุผลิตภัณฑ์ย่อยของคุณ เซ็กเมนต์เหล่านี้อาจรวมถึงหมายเลขผลิตภัณฑ์หลัก มิติของผลิตภัณฑ์ ลำดับหมายเลข ค่าคงที่ข้อความ และแอททริบิวต์ ฟังก์ชันนี้ช่วยให้คุณสามารถค้นหาผลิตภัณฑ์ย่อยเฉพาะได้อย่างรวดเร็วเมื่อคุณสร้างใบสั่งซื้อหรือใบสั่งขาย

## <a name="nomenclature-of-predefined-product-variants"></a>ระบบการตั้งชื่อสำหรับหมายเลขผลิตภัณฑ์ที่กำหนดไว้ล่วงหน้า
มีการสร้างผลิตภัณฑ์ย่อยสำหรับผลิตภัณฑ์หลักตามค่าใดค่าหนึ่งในเทคโนโลยีการตั้งค่าคอนฟิกสามรายการดังนี้:

-   ผลิตภัณฑ์ย่อยที่กำหนดไว้ล่วงหน้า
-   ตามข้อจำกัด
-   ตามมิติ

ผลิตภัณฑ์ย่อยแต่ละรายการมีหมายเลข และระบบการตั้งชื่อการระบุผลิตภัณฑ์ย่อยช่วยให้คุณสามารถเลือกเซ็กเมนต์ที่จะรวมอยู่ในแต่ละหมายเลขผลิตภัณฑ์ย่อย คุณสามารถเลือกกลุ่มต่อไปนี้ในหน้า **ระบบการตั้งชื่อผลิตภัณฑ์** ได้

-   หมายเลขผลิตภัณฑ์หลัก
-   ค่าของลำดับหมายเลข
-   ค่าคงที่ของข้อความ
-   มิติของผลิตภัณฑ์
    -   โครงแบบ
    -   สี
    -   ขนาด
    -   ลักษณะ

หลังจากที่มีการกำหนดระบบการตั้งชื่อผลิตภัณฑ์ย่อย รายการเหล่านั้นสามารถเชื่อมโยงกับกลุ่มมิติของผลิตภัณฑ์ได้ ดังนั้น ผลิตภัณฑ์หลักทั้งหมดที่อ้างอิงถึงกลุ่มมิติของผลิตภัณฑ์นี้จะถูกกำหนดหมายเลขผลิตภัณฑ์ย่อยตามระบบการตั้งชื่อ นอกจากนี้ยังสามารถกำหนดระบบการตั้งชื่อการระบุผลิตภัณฑ์ย่อยไปยังผลิตภัณฑ์หลักโดยตรงได้ ซึ่งผลิตภัณฑ์ย่อยที่เป็นของผลิตภัณฑ์หลักนี้จะถูกกำหนดหมายเลขผลิตภัณฑ์ย่อยตามระบบการตั้งชื่อ

### <a name="example"></a>ตัวอย่าง

เสื้อยืดคอกลม (TS1234) ถูกผลิตขึ้นโดยมีขนาดที่แตกต่างกัน 3 ขนาด (S, M, L) มีสีที่แตกต่างกัน 4 สี (สีแดง สีเขียว สีน้ำเงิน สีเหลือง) และ 2 ลักษณะ (Polo, V) ซึ่งทำให้รวมแล้วมีผลิตภัณฑ์ย่อยที่เป็นไปได้ทั้งหมด 24 รายการ ระบบการตั้งชื่อการระบุผลิตภัณฑ์ย่อยถูกสร้างขึ้นด้วยเซ็กเมนต์ต่อไปนี้:

1.  หมายเลขผลิตภัณฑ์หลัก
2.  ค่าคงที่ของข้อความ: '-'
3.  สี
4.  ค่าคงที่ของข้อความ: '-'
5.  ขนาด
6.  ค่าคงที่ของข้อความ: '-'
7.  ลักษณะ

หมายเลขผลิตภัณฑ์ย่อยสำหรับสีแดง ขนาดเล็ก แบบ Polo คือ: TS1234-Red-Small-Polo

## <a name="nomenclature-of-constraintbased-configurations"></a>ระบบการตั้งชื่อของการจัดโครงแบบตามข้อจำกัด
สำหรับการจัดโครงแบบตามข้อจำกัด สามารถสร้างระบบการตั้งชื่อเฉพาะสำหรับมิติของผลิตภัณฑ์การจัดการโครงแบบ คุณสามารถเลือกกลุ่มต่อไปนี้ในหน้า **ระบบการตั้งชื่อผลิตภัณฑ์** ได้

-   ค่าของลำดับหมายเลข
-   ค่าคงที่ของข้อความ
-   ค่าแอททริบิวต์ 

แต่ละส่วนประกอบในแบบจำลองการจัดโครงแบบผลิตภัณฑ์สามารถมีระบบการตั้งชื่อการจัดโครงแบบของตนเอง สามารถใช้ได้เฉพาะแอททริบิวต์ของส่วนประกอบ ไม่มีแอททริบิวต์จากส่วนประกอบย่อยหรือความต้องการของผู้ใช้

### <a name="example"></a>ตัวอย่าง

แบบจำลองการจัดโครงแบบผลิตภัณฑ์มีส่วนประกอบรากที่มีแอททริบิวต์สองรายการ

-   วัสดุ (พลาสติก ไม้ เหล็ก)
-   ความยาว (10...100)

ระบบการตั้งชื่อการจัดโครงแบบถูกกำหนดโดยใช้เซ็กเมนต์ต่อไปนี้:

1.  ค่าแอททริบิวต์: วัสดุ
2.  ค่าคงที่ของข้อความ: 'AAA'
3.  ค่าแอททริบิวต์: ความยาว

รหัสการจัดโครงแบบสำหรับวัสดุไม้ที่มีความยาว 78 จะได้รับรหัสการจัดโครงแบบต่อไปนี้: WoodAAA78

## <a name="nomenclature-of-dimensionbased-configurations"></a>ระบบการตั้งชื่อการจัดโครงแบบตามมิติ
สำหรับการจัดโครงแบบตามมิติ สามารถสร้างระบบการตั้งชื่อเฉพาะสำหรับมิติของผลิตภัณฑ์การจัดการโครงแบบ คุณสามารถเลือกกลุ่มต่อไปนี้ในหน้า **ระบบการตั้งชื่อผลิตภัณฑ์** ได้

-   ค่าของลำดับหมายเลข
-   ค่าคงที่ของข้อความ
-   สินค้าในกลุ่มการจัดโครงแบบ

คุณสามารถกำหนดระบบการตั้งชื่อการจัดโครงแบบสำหรับสูตรการผลิต (BOM)

### <a name="example"></a>ตัวอย่าง

สูตรการผลิตมีรายการ BOM 4 รายการที่แบ่งออกเป็นกลุ่มการจัดโครงแบบ 2 กลุ่ม

-   รายการ BOM: M0007 Cabinet มาตรฐาน
    -   กลุ่มการจัดโครงแบบ: Cabinet
-   รายการ BOM: M0008 Cabinet ชั้นสูง
    -   กลุ่มการจัดโครงแบบ: Cabinet
-   รายการ BOM: M0021 ผ้า Grill หน้า
    -   กลุ่มการจัดโครงแบบ: Grill หน้า
-   รายการ BOM: M0022 โลหะ Grill หน้า
    -   กลุ่มการจัดโครงแบบ: Grill หน้า

ระบบการตั้งชื่อการจัดโครงแบบถูกกำหนดโดยใช้เซ็กเมนต์ต่อไปนี้:

1.  กลุ่มการจัดโครงแบบ: Cabinet
2.  ค่าคงที่ของข้อความ: '&'
3.  กลุ่มการจัดโครงแบบ: Grill หน้า

รหัสการจัดโครงแบบสำหรับ Cabinet มาตรฐานที่มีผ้า Grill หน้าคือ: M0007&M0021

## <a name="nomenclature-of-a-combination-of-product-variants-and-configurations"></a>ระบบการตั้งชื่อของชุดผลิตภัณฑ์ย่อยและการจัดโครงแบบ
เมื่อคุณใช้เทคโนโลยีการจัดโครงแบบตามข้อจำกัดหรือตามมิติอย่างใดอย่างหนึ่งเพื่อจัดโครงแบบผลิตภัณฑ์ย่อยสำหรับผลิตภัณฑ์หลัก ผลิตภัณฑ์ย่อยสามารถเรียกดูหมายเลขผลิตภัณฑ์ย่อยที่รวมระบบการตั้งชื่อจากมิติการจัดโครงแบบ ทำตามขั้นตอนเหล่านี้เพื่อจัดโครงแบบผลิตภัณฑ์ย่อย:

1.  กำหนดระบบการตั้งชื่อหมายเลขผลิตภัณฑ์ย่อยที่มีมิติการจัดโครงแบบในหน้า **ระบบการตั้งชื่อผลิตภัณฑ์**
2.  กำหนดระบบการตั้งชื่อนี้ให้กับกลุ่มมิติของผลิตภัณฑ์ที่มีมิติการจัดโครงแบบ
3.  กำหนดระบบการตั้งชื่อการจัดโครงแบบสำหรับส่วนประกอบหรือ BOM ที่จะใช้สำหรับการจัดโครงแบบผลิตภัณฑ์ย่อย

### <a name="example-for-constraint-based-configurations"></a>ตัวอย่างการจัดโครงแบบตามข้อจำกัด

ในตัวอย่างนี้ คุณสามารถใช้ระบบการตั้งชื่อหมายเลขผลิตภัณฑ์ย่อยที่ประกอบด้วยเซ็กเมนต์ต่อไปนี้:

1.  หมายเลขผลิตภัณฑ์หลัก
2.  ค่าคงที่ของข้อความ '\_'
3.  โครงแบบ

ระบบการตั้งชื่อการจัดโครงแบบสามารถประกอบด้วยเซ็กเมนต์ต่อไปนี้:

1.  ค่าแอททริบิวต์: วัสดุ
2.  ค่าคงที่ของข้อความ: 'AAA'
3.  ค่าแอททริบิวต์: ความยาว

คุณสามารถป้อนค่าต่อไปนี้สำหรับเซ็กเมนต์:

-   หมายเลขผลิตภัณฑ์หลัก = M0099
-   วัสดุ = พลาสติก
-   ความยาว = 12

หมายเลขผลิตภัณฑ์ย่อยจะกลายเป็น: M0099\_PlasticAAA12

### <a name="example-for-dimension-based-configurations"></a>ตัวอย่างการจัดโครงแบบตามมิติ

ในตัวอย่างนี้ คุณสามารถใช้ระบบการตั้งชื่อหมายเลขผลิตภัณฑ์ย่อยที่ประกอบด้วยเซ็กเมนต์ต่อไปนี้:

1.  หมายเลขผลิตภัณฑ์หลัก
2.  ค่าคงที่ของข้อความ '//'
3.  โครงแบบ

ระบบการตั้งชื่อการจัดโครงแบบสามารถประกอบด้วยเซ็กเมนต์ต่อไปนี้:

1.  กลุ่มการจัดโครงแบบ: Cabinet
2.  ค่าคงที่ของข้อความ: '&'
3.  กลุ่มการจัดโครงแบบ: Grill หน้า

คุณสามารถป้อนค่าต่อไปนี้สำหรับเซ็กเมนต์:

-   หมายเลขผลิตภัณฑ์หลัก = D0123
-   Cabinet = M0008
-   Grill หน้า = M0022

หมายเลขผลิตภัณฑ์ย่อยจะกลายเป็น: D0123//M0008&M0022

## <a name="numbering-conflicts"></a>การกำหนดหมายเลขความขัดแย้ง
สามารถตั้งค่าระบบการตั้งชื่อหมายเลขผลิตภัณฑ์ย่อยที่ไม่ได้ส่งผลต่อหมายเลขผลิตภัณฑ์ย่อยเฉพาะ ตัวอย่างเช่น อาจเกิดขึ้นถ้าไม่มีมิติของผลิตภัณฑ์ที่ใช้งานอยู่หนึ่งรายการรวมอยู่ในระบบการตั้งชื่อสำหรับผลิตภัณฑ์หลักที่ใช้เทคโนโลยีการจัดโครงแบบผันแปรที่กำหนดไว้ล่วงหน้า มีการจัดการความขัดแย้งแตกต่างกันสำหรับเทคโนโลยีการจัดโครงแบบที่แตกต่างกัน

### <a name="predefined-variants"></a>ผลิตภัณฑ์ย่อยที่กำหนดไว้ล่วงหน้า

ข้อผิดพลาดจะเกิดขึ้นถ้าคุณพยายามสร้างผลิตภัณฑ์ย่อยหนึ่งหรือหลายรายการที่ลงท้ายด้วยหมายเลขผลิตภัณฑ์ย่อยโดยอัตโนมัติหรือด้วยตนเอง เพื่อหลีกเลี่ยงปัญหานี้ คุณควรใช้มิติของผลิตภัณฑ์ที่ใช้งานอยู่ทั้งหมดในกลุ่มมิติของผลิตภัณฑ์ หรือรวมลำดับหมายเลขเพื่อให้แน่ใจว่าหมายเลขผลิตภัณฑ์ย่อยไม่ซ้ำกัน

### <a name="constraint-based-configurations"></a>การจัดโครงแบบตามข้อจำกัด

โดยขึ้นอยู่กับระบบการตั้งชื่อ ระบบอาจพยายามที่จะกำหนดหมายเลขผลิตภัณฑ์ย่อยที่ซ้ำกันให้กับการจัดโครงแบบ ในกรณีนี้ ระบบจะใช้ลำดับหมายเลขสำหรับมิติการจัดโครงแบบตามหมายเลขผลิตภัณฑ์ย่อยแทน ถ้ากรณีนี้เกิดขึ้น คุณจะได้รับคำเตือน เพื่อหลีกเลี่ยงปัญหานี้ คุณควรรวมแอททริบิวต์ที่เพียงพอในระบบการตั้งชื่อเพื่อให้แน่ใจว่าจะไม่ซ้ำกัน และตรวจสอบให้แน่ใจว่าตัวเลือก **นำมาใช้ใหม่** เปิดอยู่สำหรับส่วนประกอบ

### <a name="dimension-based-configurations"></a>การจัดโครงแบบตามมิติ

กระบวนการจัดโครงแบบรวมถึงขั้นตอนที่ระบบจะแนะนำค่าการจัดโครงแบบตามระบบการตั้งชื่อ ในขั้นตอนนี้ คุณสามารถเปลี่ยนค่าการจัดโครงแบบด้วยตนเอง เมื่อคุณบันทึกการจัดโครงแบบ ระบบจะตรวจสอบว่ามีค่าการจัดโครงแบบที่ซ้ำกันหรือไม่ ถ้าไม่ใช่กรณีนี้ ข้อผิดพลาดจะปรากฏขึ้น คุณต้องป้อนค่าการจัดโครงแบบเฉพาะเพื่อบันทึกการจัดโครงแบบ
