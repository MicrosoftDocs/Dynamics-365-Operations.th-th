# <a name="create-a-repeat-purchase-order"></a>สร้างใบสั่งซื้อซ้ำ

[!include[task guide banner](../../includes/task-guide-banner.md)]

กระบวนงานนี้แสดงวิธีการสร้างใบสั่งซื้อ (PO) ที่ทำซ้ำโดยการคัดลอกรายการจากเอกสารใบสั่งซื้อก่อนหน้านี้ไปยังใบสั่งซื้อใหม่หรือใบสั่งซื้อที่มีอยู่  มีสองวิธีในการสร้างใบสั่งที่ทำซ้ำ คุณสามารถใช้การดำเนินการที่พร้อมใช้งานในระดับเอกสารจากบานหน้าต่างการดำเนินการ หรือคุณสามารถใช้การดำเนินการรายละเอียดรายการ โดยส่วนใหญ่แล้วการดำเนินการระดับเอกสารมีไว้สำหรับการสร้างใบสั่งซื้อใหม่โดยการเพิ่มรายการและข้อมูลหัวข้อจากใบสั่งอื่น ในขณะที่การดำเนินการรายละเอียดรายการส่วนใหญ่มีไว้สำหรับการเพิ่มรายการไปยังใบสั่งที่มีอยู่ ตัวอย่างที่แสดงในคำแนะนำนี้สามารถใช้ในบริษัทข้อมูลสาธิต USMF โดยทั่วไปงานเหล่านี้จะดำเนินการโดยเจ้าหน้าที่จัดซื้อ


## <a name="create-a-new-repeat-purchase-order"></a>สร้างใบสั่งซื้อที่ทำซ้ำใหม่
1. ไปที่การจัดซื้อและการจัดหา > ใบสั่งซื้อ > ใบสั่งซื้อทั้งหมด
    * อันดับแรกเราจะลองตัวเลือกสำหรับการคัดลอกข้อมูลไปยังใบสั่งใหม่  
2. คลิก สร้าง
3. ในฟิลด์บัญชีผู้จัดจำหน่าย ให้ป้อน US-101'
4. คลิก ตกลง
5. ในบานหน้าต่างการดำเนินการ คลิกที่ใบสั่งซื้อ
6. คลิกจากทั้งหมด
    * นี่คือหน้าที่คุณสามารถคัดลอกจากใบสั่งที่มีอยู่ไปยังใบสั่งของคุณ  มีตัวเลือกที่แตกต่างกันสำหรับวิธีการคัดลอกรายการ และชนิดเอกสารต่าง ๆ ที่คุณสามารถคัดลอก  ก่อนอื่นเราจะดูตัวเลือกวิธีคัดลอกรายการ   
7. ขยายส่วน พารามิเตอร์
    * ในฟิลด์ตัวคูณปริมาณมีประโยชน์ถ้าคุณจำเป็นต้องใช้ปริมาณที่แตกต่างจากปริมาณในใบสั่งที่คุณกำลังคัดลอก  ตัวอย่างเช่น ถ้าคุณต้องการสั่งซื้อเป็นสองเท่าของปริมาณในรายการที่คุณกำลังคัดลอก ให้พิมพ์ '2' ในฟิลด์นี้ และระบบจะเพิ่มรายการที่มีปริมาณเป็นสองเท่าของปริมาณเดิม  
    * นอกจากนี้ฟิลด์กลับเครื่องหมายยังสนับสนุนการเปลี่ยนปริมาณที่สั่งโดยการเปลี่ยนเครื่องหมายของปริมาณสำหรับรายการใบสั่งที่จะเพิ่ม  สิ่งนี้อาจมีประโยชน์ถ้าคุณต้องการกลับรายการธุรกรรมโดยการสร้างรายการใบสั่งที่ยกเลิกธุรกรรม ตัวเลือกนี้ถูกเลือกโดยอัตโนมัติเมื่อมีการเปิดหน้าจากการดำเนินการสร้างใบลดหนี้  
    * ตัวเลือกคัดลอกค่าธรรมเนียมช่วยให้คุณสามารถคัดลอกค่าธรรมเนียมไปยังใบสั่งใหม่ของคุณจากเอกสารที่คุณกำลังคัดลอกรายการใบสั่ง  
    * ตัวเลือกคำนวณราคาอีกครั้งใช้ราคาปัจจุบันและส่วนลดแทนที่จะคัดลอกจากเอกสารที่คุณกำลังคัดลอกข้อมูลเหล่านี้  
    * ตัวเลือกคัดลอกอย่างแม่นยำสร้างสำเนาที่เหมือนกันของค่าในฟิลด์ทั้งหมดในส่วนหัวและบรรทัดของเอกสารใบสั่ง  ถ้าไม่ได้เลือกตัวเลือกนี้ ค่าเริ่มต้นจะถูกใช้สำหรับฟิลด์หลายรายการที่เกี่ยวข้องกับผู้จัดจำหน่ายและผลิตภัณฑ์เช่นเดียวกับที่คุณสร้างใบสั่งใหม่ด้วยตนเอง ตัวอย่างเช่น ถ้าใบสั่งที่คุณกำลังคัดลอกแทนที่บัญชีใบแจ้งหนี้เริ่มต้นสำหรับผู้จัดจำหน่าย บัญชีใบแจ้งหนี้เดียวกันนั้นจะสามารถคัดลอกไปยังใบสั่งของคุณ ถ้าคุณไม่ได้เลือกตัวเลือกคัดลอกอย่างแม่นยำ บัญชีใบแจ้งหนี้เริ่มต้นสำหรับผู้จัดจำหน่ายจะถูกใช้ในใบสั่งของคุณแทน  
    * ตัวเลือกลบรายการสั่งซื้อจะลบรายการใบสั่งซื้อทั้งหมดที่อยู่ในใบสั่งซื้อที่คุณกำลังคัดลอก ก่อนที่จะใช้รายการใหม่  ใช้ตัวเลือกนี้ด้วยความระมัดระวังเนื่องจากตัวเลือกนี้จะลบบรรทัดที่มีอยู่ทั้งหมดโดยไม่มีคำเตือนต่อไป  
    * ถ้าคุณใช้ตัวเลือกคัดลอกส่วนหัวของใบสั่ง คุณไม่จำเป็นต้องสร้างข้อมูลส่วนหัวของใบสั่งใหม่ของคุณด้วยตนเอง  โปรดสังเกตว่าตัวเลือกนี้จะส่งผลให้ใช้ค่าเริ่มต้นสำหรับฟิลด์เกี่ยวข้องกับผู้จัดจำหน่าย ถ้าเอกสารที่คุณกำลังคัดลอกมีค่าที่ไม่ใช่ค่าเริ่มต้นที่คุณต้องการคัดลอก ให้ใช้ตัวเลือกคัดลอกอย่างแม่นยำด้วยเช่นกัน  
8. ยุบส่วนพารามิเตอร์
    * มีแหล่งข้อมูลเอกสารต่าง ๆ ที่คุณสามารถคัดลอกได้ และแต่ละแหล่งข้อมูลนั้นมีส่วนที่แยกต่างหากบนหน้านี้  ตัวอย่างเช่น ส่วนของใบสั่งซื้อช่วยให้คุณสามารถคัดลอกจากใบสั่งซื้อที่มีอยู่  
9. คลิกที่รายการของใบสั่งซื้อที่มีรหัสเป็น 00015 
10. เลือกรายการโดยคลิกในกล่องกาเครื่องหมาย
11. ล้างข้อมูลกล่องกาเครื่องหมายสำหรับรายการนั้นเพื่อไม่ให้มีการคัดลอกไปยังใบสั่งของคุณ
    * ขณะนี้มีการเลือก 4 รายการที่จะคัดลอกไปยังใบสั่งซื้อของคุณ  สามารถเลือกรายการใบสั่งซื้อจากใบสั่งซื้ออื่น ๆ และคัดลอกไปยังใบสั่งของคุณด้วยเช่นกัน สามารถเพิ่มรายการจากเอกสารการจัดซื้อประเภทอื่นได้ ขั้นตอนถัดไปคือการตรวจทานตัวเลือกต่าง ๆ  
12. ยุบส่วนใบสั่งซื้อ
13. ขยายส่วนการยืนยัน
    * คุณสามารถเลือกการยืนยันใบสั่งซื้อที่จะคัดลอกที่นี่  การยืนยันใบสั่งซื้อจะถูกระบุโดยรหัสสมุดรายวันการซื้อที่เกี่ยวข้องหรือรหัสใบสั่งซื้อ  
14. ยุบส่วนการยืนยัน
15. ขยายส่วนใบรับสินค้า
    * คุณสามารถเลือกสมุดรายวันใบรับสินค้าเพื่อคัดลอกที่นี่  สมุดรายวันใบรับสินค้าจะถูกระบุโดยใบสำคัญการรับสินค้าหรือรหัสใบสั่งซื้อ   
16. ยุบส่วนใบรับสินค้า
17. ขยายส่วนใบแจ้งหนี้
    * คุณสามารถเลือกใบแจ้งหนี้ของผู้จัดจำหน่ายจะคัดลอกที่นี่ ใ บแจ้งหนี้จะถูกระบุโดยใบสำคัญใบแจ้งหนี้หรือรหัสใบสั่งซื้อ   
18. ยุบส่วนใบแจ้งหนี้ 
19. ขยายส่วนรายการหรือหัวข้อที่เลือกเพื่อที่จะคัดลอก
    * มุมมองนี้แสดงสรุปของเอกสารทั้งหมดและรายการที่คุณเลือกที่จะคัดลอกไปยังใบสั่งของคุณ   
20. ยุบส่วนรายการหรือหัวข้อที่เลือกเพื่อที่จะคัดลอก
21. คลิก ตกลง
    * 4 รายการที่คุณเลือกถูกคัดลอกไปยังใบสั่งซื้อใหม่ของคุณแล้ว   

## <a name="copy-lines-to-an-existing-purchase-order"></a>คัดลอกรายการไปยังใบสั่งซื้อที่มีอยู่
    * แทนการคัดลอกใบสั่งทั้งหมด โดยทั่วไปจะสร้างใบสั่งซื้อใหม่และระบุข้อมูลทั้งหมดในส่วนหน้าของใบสั่งซื้อ และจากนั้นจึงคัดลอกแต่ละรายการจากใบสั่งที่มีอยู่  
1. คลิก บรรทัดรายการใบสั่งซื้อ
2. คลิกจากทั้งหมด
    * หน้าที่เปิดอยู่จะเหมือนกับหน้าที่แสดงก่อนหน้านี้ แต่มีเลือกตัวเลือกที่แตกต่างกันเมื่อเปิดจากมุมมองรายการใบสั่ง  มาตรวจทานพารามิเตอร์กัน   
3. ขยายส่วน พารามิเตอร์
    * ไม่มีการเลือกตัวเลือกลบรายการสั่งซื้อ  ซึ่งหมายความว่าคุณสามารถคัดลอกรายการใหม่ไปยังใบสั่งของคุณโดยไม่ต้องลบรายการที่มีอยู่   
    * ไม่มีการเลือกตัวเลือกคัดลอกส่วนหัวของใบสั่งเช่นกัน เนื่องจากเรากำลังเพิ่มรายการเพิ่มเติมไปยังใบสั่งเท่านั้น   
4. ยุบส่วนพารามิเตอร์
    * สำหรับตัวอย่างนี้ เราจะคัดลอกรายการจากใบสั่งซื้อที่มีอยู่   
5. คลิกที่รายการของใบสั่งซื้อที่มีรหัสเป็น 00034 
6. เลือกรายการโดยคลิกในกล่องกาเครื่องหมาย
    * โปรดสังเกตว่ามีการเลือกรายการใบสั่งเดียวที่อยู่ในใบสั่งซื้อนี้  
7. คลิก ตกลง
    * มีการเพิ่มรายการใบสั่งเพิ่มเติมไปยังใบสั่งซื้อของคุณ
