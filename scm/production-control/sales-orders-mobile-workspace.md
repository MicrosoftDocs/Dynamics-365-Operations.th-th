# <a name="sales-orders-mobile-workspace-for-microsoft-dynamics-365-for-operations-app"></a>พื้นที่ทำงานแบบเคลื่อนที่ของใบสั่งขายสำหรับแอพ Microsoft Dynamics 365 for Operations

ด้วยพื้นที่ทำงานแบบเคลื่อนที่ของใบสั่งขาย คุณสามารถคอยติดตามสถานการณ์ปัจจุบันเกี่ยวกับใบสั่งขายของคุณที่ใดก็ได้ และเมื่อใดก็ได้ 

<a name="prerequisites"></a>ข้อกำหนดเบื้องต้น
-------------

| ข้อกำหนดเบื้องต้น                                                         | คำอธิบาย                                                                                                                                                                   |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| อ่านเกี่ยวกับแพลตฟอร์มเคลื่อนที่ของ Microsoft Dynamics 365 for Operations | [แพลตฟอร์มเคลื่อนที่ของ Microsoft Dynamics 365 for Operations](/dynamics365/operations/dev-itpro/mobile-apps/mobile-platform)                                                              |
| Dynamics 365 for Operations                                          | ตรวจสอบให้แน่ใจว่าคุณกำลังใช้สภาพแวดล้อมที่มี Microsoft Dynamics 365 for Operations เวอร์ชัน 1611 และการอัพเดตแพลตฟอร์ม Microsoft Dynamics for Operations 3 (พฤศจิกายน 2016) |

| อุปกรณ์เคลื่อนที่มี 365 Dynamics สำหรับการดำเนินการติดตั้งแอพลิเคชัน | ดาวน์โหลดแอพ Dynamics 365 for Operations จากร้านค้าแอพบนมือถือของคุณ                                                                                                      |

## <a name="overview"></a>ภาพรวม
พื้นที่ทำงานแบบเคลื่อนที่นี้เข้าถึงแอพ Dynamics 365 for Operations และช่วยให้คุณสามารถดูรายละเอียดเกี่ยวกับแต่ละใบสั่งขาย เช่น สถานะใบสั่ง ข้อมูลการติดต่อของลูกค้า และข้อมูลการติดต่อของผู้รับใบสั่ง พื้นที่ทำงานแบบเคลื่อนที่แสดงมุมมองฉับพลันของใบสั่งขาย คุณสามารถดูใบสั่งขายโดยเรียงตามลูกค้า หรือดูใบสั่งขายทั้งหมด หรือดูข้อมูลเกี่ยวกับใบสั่งขายเฉพาะ พื้นที่ทำงานแบบเคลื่อนที่มีสองมุมมองเพื่อช่วยให้คุณวิเคราะห์ใบสั่งขายในเชิงลึก

### <a name="view-all-sales-orders"></a>ดูใบสั่งขายทั้งหมด

มุมมองนี้แสดงรายการใบสั่งขายทั้งหมด

-   ใช้ตัวกรองใดตัวกรองหนึ่งเพื่อเลือกใบสั่งขายที่คุณต้องการดู
    -   ค้นหาตามใบสั่งขาย
    -   ค้นหาตามบัญชีของลูกค้า
    -   ค้นหาตามชื่อของลูกค้า
    -   ค้นหาตามสถานะ
    -   ค้นหาตามสถานะนำออกใช้
    -   ค้นหาตามวันที่และเวลาที่สร้าง

<!-- -->

-   หลังจากที่คุณเลือกใบสั่งขาย คุณสามารถดูรายละเอียดของใบสั่งเฉพาะได้ โดยเฉพาะอย่างยิ่ง คุณสามารถดู:
    -   ข้อมูลชื่อและที่อยู่ของลูกค้า
    -   วันที่ของใบสั่งขายอื่น เช่น วันที่จัดส่งที่ร้องขอและวันที่จัดส่งที่ยืนยัน
    -   ข้อมูลการติดต่อผู้รับใบสั่ง
    -   ข้อมูลติดต่อของลูกค้า
    -   รายการใบสั่ง
    -   การจัดส่งที่แสดงวิธีการและเมื่อมีการจัดส่งใบสั่งขาย

### <a name="view-orders-for-a-customer-"></a>ดูใบสั่งสำหรับลูกค้า** **

มุมมองนี้แสดงรายการใบสั่งขายสำหรับลูกค้าแต่ละราย

-   ใช้ตัวกรองใดตัวกรองหนึ่งต่อไปนี้เพื่อดูใบสั่งสำหรับลูกค้า
    -   ค้นหาตามชื่อ
    -   ค้นหาตามบัญชี

<!-- -->

-   หลังจากที่คุณเลือกลูกค้า คุณสามารถดู:
    -   ชื่อและกลุ่มลูกค้า
    -   ข้อมูลติดต่อของลูกค้า
    -   ใบสั่งขายของลูกค้าและรายละเอียดเกี่ยวกับใบสั่งขาย:
        -   ข้อมูลชื่อและที่อยู่ของลูกค้า
        -   วันที่ในใบสั่งขายอื่น
        -   ข้อมูลการติดต่อผู้รับใบสั่ง
        -   ข้อมูลติดต่อของลูกค้า
        -   รายการใบสั่ง
        -   การจัดส่งที่แสดงวิธีการและเมื่อมีการจัดส่งใบสั่งขาย

## <a name="get-started"></a>เริ่มต้น
ทำตามขั้นตอนเหล่านี้เพื่อเริ่มต้นการใช้พื้นที่ทำงานแบบเคลื่อนที่ของใบสั่งขายบนอุปกรณ์เคลื่อนที่ของคุณ

1.  บนร้านค้าแอพบนมือถือของคุณ ดาวน์โหลดและติดตั้งแอพ Microsoft Dynamics 365 for Operations
2.  เริ่มการทำงานแอพบนอุปกรณ์ของคุณ
3.  ป้อน URL ของ Dynamics 365
4.  ป้อนบริษัทเพื่อลงชื่อเข้าใช้ ตัวอย่างเช่น ป้อน **USMF**
5.  ในครั้งแรกที่คุณเข้าสู่ระบบ คุณจะได้รับพร้อมท์สำหรับชื่อผู้ใช้และรหัสผ่านสำหรับบัญชี Microsoft Dynamics 365 for Operations ของคุณ ป้อนข้อมูลประจำตัวของคุณ หลังจากที่คุณลงชื่อเข้าใช้แล้ว คุณจะสามารถดูพื้นที่ทำงานที่พร้อมใช้งานสำหรับบริษัทของคุณ

เมื่อต้องการดูพื้นที่ทำงานในแอพบนมือถือของคุณ คุณต้องเผยแพร่พื้นที่ทำงานที่ต้องการไปยังแอพ Dynamics 365 for Operations

1.  เริ่มการทำงาน Dynamics 365 for Operations
2.  ไปที่ **การจัดการระบบ** &gt; **การตั้งค่า** &gt; **พารามิเตอร์ระบบ**
3.  เลือก **จัดการแอพบนมือถือ**
4.  เลือกพื้นที่ทำงานที่จะเผยแพร่ไปยังแพลตฟอร์มเคลื่อนที่
5.  เลือก **เผยแพร่พื้นที่ทำงาน**
6.  รีเฟรชอุปกรณ์ของคุณเพื่อดูพื้นที่ทำงานที่เผยแพร่แล้ว

## <a name="view-information-about-sales-orders-for-a-customer"></a>ดูข้อมูลเกี่ยวกับใบสั่งขายสำหรับลูกค้า
1.  บนอุปกรณ์เคลื่อนที่ เลือกพื้นที่ทำงาน **ใบสั่งขาย**
2.  เลือก **ดูใบสั่งสำหรับลูกค้า**
3.  ใช้ข้อมูล **บัญชี** หรือ **ชื่อลูกค้า** เพื่อค้นหาลูกค้าที่ต้องการ
4.  เลือกลูกค้า
5.  เลือก **ข้อมูลติดต่อ** หรือ **ใบสั่งขาย**
6.  ถ้ามีการเลือก **ใบสั่งขาย** รายการใบสั่งขายสำหรับลูกค้าจะปรากฏขึ้น
7.  เลือก **ใบสั่งขาย**
8.  ที่นี่คุณสามารถดูข้อมูลเกี่ยวกับบรรทัดใบสั่งขาย การจัดส่ง ข้อมูลติดต่อของลูกค้า และข้อมูลการติดต่อของผู้รับใบสั่ง
