# <a name="production-order-defaults-in-manufacturing-execution"></a>ค่าเริ่มต้นของใบสั่งผลิตในส่วนการดำเนินการผลิต

[!include[banner](../includes/banner.md)]




คุณควรพิจารณาการตั้งค่าของ **ค่าเริ่มต้นในใบสั่งผลิต** อย่างถี่ถ้วนก่อนที่ผู้ปฏิบัติงานจะทำการลงทะเบียนเพื่อเริ่มงานผลิต ถ้าบริษัทของคุณใช้ฟังก์ชันหลายไซต์ คุณอาจจะต้องตั้งค่าเริ่มต้นของใบสั่งผลิตสำหรับแต่ละไซต์ให้แตกต่างกัน ค่าเริ่มต้นของใบสั่งที่ใช้สำหรับรวมกับการควบคุมการผลิตจะถูกตั้งค่าตามแท็บต่อไปนี้ในหน้า **ค่าเริ่มต้นของใบสั่งผลิต**:

-   **ทั่วไป**– ค่าเริ่มต้นทั่วไปสำหรับงานการผลิตในการดำเนินการผลิตใน
-   **เริ่มต้น**– ค่าเริ่มต้นที่จะใช้เมื่อเริ่มต้นงานการผลิตหรือเริ่มดำเนินการผลิต
-   **การดำเนินงาน**– ค่าเริ่มต้นที่ใช้กับงานการผลิต หรือการดำเนินการผลิต และฟีดแบคเกี่ยวกับการดำเนินการในระหว่างกระบวนการผลิต
-   **รายงานเมื่อเสร็จสิ้น** – ค่าเริ่มต้นที่ใช้กับไอเทมที่ถูกรายงานว่าเสร็จสิ้นแล้ว ในการดำเนินงานขั้นสุดท้ายของใบสั่งผลิต
-   **การตรวจสอบความถูกต้องของปริมาณ**– ค่าเริ่มต้นสำหรับการตรวจสอบการเริ่มต้นและปริมาณฟีดแบคของใบสั่งผลิต

## <a name="types-of-production-jobs"></a>ชนิดของงานการผลิต
ในแท็บ **การดำเนินงาน** คุณต้องเลือกชนิดของงานการผลิตที่จำเป็นต้องมีการลงทะเบียนในหน้า **การลงทะเบียนงาน**. โดยทั่วไป ผู้ปฏิบัติงานจะทำการลงทะเบียนในงานการตั้งค่าและงานกระบวนการ อย่างไรก็ตาม ถ้ามีใช้การจัดตารางงาน คุณจะสามารถเลือกชนิดของงานอื่น ๆ เช่นงานการขนส่ง ซึ่งผู้ปฏิบัติงานต้องทำการลงทะเบียนเมื่อมีการเปิดใบสั่งผลิตเช่นกัน **สิ่งสำคัญ:** ต้องตรวจสอบว่า ชนิดของงานที่เกี่ยวข้องทั้งหมดได้ถูกเลือกแล้ว มิฉะนั้น งานอาจไม่พร้อมให้ใช้ในการลงทะเบียนในหน้า **ลงทะเบียนงาน** ได้ จัดระเบียบตัวเลือกของคุณโดยใช้ตัวเลือกที่ทำไว้ในคอลัมน์ **การจัดการงาน** ในหน้า **กลุ่มกระบวนผลิต**. ถ้าหาก **การจัดการงาน** ถูกเลือกบนกลุ่มกระบวนการผลิต ชนิดงานจะถูกรายงานว่าเสร็จสมบูรณ์แล้วในใบสั่งผลิต รายงานนี้จะเกิดขึ้นก็ต่อเมื่องานถูกรายงานว่าเสร็จสมบูรณ์แล้ว ในการดำเนินการผลิต เมื่อชนิดงานทั้งหมดที่มีการเลือก **การจัดการงาน** ไว้แล้ว ได้ถูกรายงานในการดำเนินงานว่าเสร็จสมบูรณ์แล้ว การดำเนินการผลิตจะแสดงผลการดำเนินงานว่าเสร็จสมบูรณ์แล้วเช่นกัน **หมายเหตุ:** งานบางชนิดสามารถรายงานแบบแมนนวลได้เองผ่านทางสมุดรายวันการผลิต ในกรณีนี้ ให้เลือก **การจัดการงาน** สำหรับชนิดงาน แต่ไม่ต้องเลือกชนิดงานสำหรับการลงทะเบียนในแท็บ **การดำเนินการ** ใน **หน้าค่าเริ่มต้นของใบสั่งผลิต** .

## <a name="bom-consumption-and-picking-list-journals"></a>การใช้งาน BOM และ สมุดรายวันของรายการเบิกสินค้า
สามารถตั้งค่าวัสดุได้ ดังนั้นพวกมันจะถูกใช้โดยอัตโนมัติ หรือถูกใช้แบบแมนนวลในระหว่างการผลิตก็ได้ ปริมาณการใช้วัสดุอัตโนมัติจะถูกควบคุมโดยหลักการตัดจ่ายที่ถูกตั้งค่าในรายการวัสดุและส่วนประกอบ (BOM) และ โดยการตั้งค่าในฟิลด์ **ปริมาณการใช้ BOM อัตโนมัติ** ในค่าเริ่มต้นของใบสั่งผลิต ปริมาณการใช้วัสดุอัตโนมัติสามารถตั้งค่าเพื่อให้ทำงานเมื่อใบสั่งผลิตเริ่มต้นขึ้น หรือเมื่อถูกรายงานว่าเสร็จสิ้นแล้ว หรืออีกทางหนึ่งคือ มันสามารถเกิดขึ้นในระดับงานที่งานได้เริ่มต้นขึ้น หรือเมื่อมันเสร็จสมบูรณ์แล้ว

### <a name="controlling-material-consumption-when-a-production-order-is-started"></a>การควบคุมปริมาณการใช้วัสดุเมื่อเริ่มต้นใบสั่งผลิต

ปริมาณการใช้วัสดุในระหว่างการเริ่มต้นของใบสั่งผลิตจะควบคุมโดยฟิลด์ **ปริมาณการใช้ BOM อัตโนมัติ** บนแท็บ **เริ่มต้น**. ค่าที่พร้อมใช้งานมีดังต่อไปนี้

-   **หลักการตัดจ่าย**– เมื่อใบสั่งผลิตเริ่มขึ้น ปริมาณวัสดุจะถูกใช้ตามหลักการตัดจ่ายที่ถูกกำหนดไว้ในรายการการผลิต BOM เฉพาะรายการวัสดุที่มีการตั้งค่าหลักการตัดจ่ายเป็น **เริ่มต้น** เท่านั้นที่จะถูกใช้ระหว่างการเริ่มต้นผลิต
-   **เสมอ**– ปริมาณของวัสดุที่คิดเป็นสัดส่วนจากปริมาณเริ่มต้นจะถูกใช้เสมอ
-   **ไม่เคย**– ปริมาณของวัสดุจะไม่ถูกใช้เลย

### <a name="controlling-material-consumption-when-a-production-job-is-started-or-completed"></a>การควบคุมปริมาณการใช้วัสดุเมื่องานการผลิตเริ่มต้นขึ้นหรือสิ้นสุดลง

ปริมาณการใช้วัสดุในระหว่างการเริ่มต้นหรือการสิ้นสุดของงานการผลิตจะถูกควบคุมโดยฟิลด์ **ปริมาณการใช้ BOM อัตโนมัติ** บนแท็บ **การปฏิบัติงาน**. ค่าที่พร้อมใช้งานมีดังต่อไปนี้

-   **หลักการตัดจ่าย**– เมื่อปริมาณงานการผลิตเริ่มต้นหรือสิ้นสุด ปริมาณวัสดุจะถูกใช้ตามหลักการตัดจ่ายที่ถูกกำหนดไว้ในรายการการผลิต BOM เฉพาะรายการวัสดุที่มีการตั้งค่าหลักการตัดจ่ายเป็น **เริ่มต้น** หรือ **เสร็จสิ้น** เท่านั้นที่จะถูกใช้ระหว่างการเริ่มต้นผลิต
-   **เสมอ**– ปริมาณของวัสดุที่คิดเป็นสัดส่วนจากปริมาณเริ่มต้นของงานจะถูกใช้เสมอ
-   **ไม่เคย**– ปริมาณของวัสดุจะไม่ถูกใช้เลย

### <a name="controlling-material-consumption-when-a-production-order-is-reported-as-finished"></a>การควบคุมปริมาณการใช้วัสดุเมื่อใบสั่งผลิตถูกรายงานสถานะเป็นเสร็จสิ้น

ปริมาณการใช้วัสดุในระหว่างขั้นตอนรายงานเมื่อเสร็จสิ้นของใบสั่งผลิตจะควบคุมโดยฟิลด์ **ปริมาณการใช้ BOM อัตโนมัติ** บนแท็บ **รายงานเมื่อเสร็จสิ้น**. ค่าที่พร้อมใช้งานมีดังต่อไปนี้

-   **หลักการตัดจ่าย**– เมื่อใบสั่งผลิตเสร็จสิ้น ปริมาณวัสดุจะถูกใช้ตามหลักการตัดจ่ายที่ถูกกำหนดไว้ในรายการการผลิต BOM เฉพาะรายการวัสดุที่มีการตั้งค่าหลักการตัดจ่ายเป็น **เสร็จสิ้น** เท่านั้นที่จะถูกใช้ระหว่างการเริ่มต้นผลิต
-   **เสมอ**– ปริมาณของวัสดุที่คิดเป็นสัดส่วนจากปริมาณที่ถูกรายงานป็นเสร็จสิ้นจะถูกใช้เสมอ
-   **ไม่เคย**– ปริมาณของวัสดุจะไม่ถูกใช้เลย
