# <a name="inventory-on-hand-mobile-workspace-for-microsoft-dynamics-365-for-operations-app"></a>พื้นที่ทำงานแบบเคลื่อนที่ของปริมาณคงคลังคงเหลือสำหรับแอพ Microsoft Dynamics 365 for Operations

พื้นที่ทำงานแบบเคลื่อนที่ของปริมาณคงคลังคงเหลือช่วยให้คุณได้รับข้อมูลเชิงลึกเคลื่อนที่ในสินค้าคงคลังที่จองไว้และพร้อมใช้งานได้ตลอดเวลา และที่ใดก็ได้

## <a name="introduction"></a>คำนำ
โดยทั่วไป บริษัทจะมีการจัดส่งและการรับสินค้าคงคลังหลายครั้งทุกวัน ความเคลื่อนไหวเหล่านี้จะเปลี่ยนสถานะปริมาณคงคลังคงเหลืออย่างสม่ำเสมอ พื้นที่ทำงานแบบเคลื่อนที่ของปริมาณคงคลังคงเหลือช่วยให้คุณเห็นสถานะปริมาณคงคลังคงเหลือระหว่างบริษัท เพื่อที่คุณจะได้รับข้อมูลเชิงลึกล่าสุดในข้อมูลสินค้าคงคลังบนอุปกรณ์เคลื่อนที่ที่คุณเลือก โดยไม่คำนึงถึงว่าคุณทำงานในคลังสินค้า การซื้อ การขาย การผลิต หรือการจัดการ หรือมีบทบาทอื่น คุณสามารถเข้าถึงข้อมูลปริมาณคงคลังคงเหลือได้ทุกเมื่อใด และที่ใดก็ได้ พื้นที่ทำงานแบบเคลื่อนที่แสดงมุมมองฉับพลันที่มีสถานะปริมาณคงคลังคงเหลือคงของสิ่งอำนวยความสะดวกต่างๆ และช่วยให้คุณสามารถดูปริมาณคงคลังคงเหลือคงของสิ่งอำนวยความสะดวกต่างๆ การจองวัตถุดิบปัจจุบัน และปริมาณสินค้าคงคลังคงเหลือที่จองไว้ นอกจากนี้คุณยังสามารถป้อนหมายเลขเพื่อสอบถามปริมาณคงคลังคงเหลือ และทำการค้นหาผลิตภัณฑ์คงคลังคงเหลือหรือตัวแปรที่ถูกกรองข้อมูล โดยเฉพาะอย่างยิ่ง พื้นที่ทำงานแบบเคลื่อนที่จะมีคุณลักษณะเหล่านี้:

-   คุณสามารถค้นหาตามหมายเลขผลิตภัณฑ์หรือชื่อผลิตภัณฑ์เพื่อค้นหาผลิตภัณฑ์เพื่อดูสถานะสำหรับปริมาณคงคลังคงเหลือ
-   สำหรับผลิตภัณฑ์ที่เลือก คุณสามารถดูข้อมูลต่อไปนี้:
    -   ปริมาณคงคลังคงเหลือต่อไซต์
    -   ปริมาณคงคลังคงเหลือต่อคลังสินค้า
    -   ปริมาณคงคลังคงเหลือต่อสถานที่
    -   ปริมาณคงคลังคงเหลือต่อชุดงาน (สำหรับผลิตภัณฑ์ที่มีการควบคุมชุดงาน)
    -   ปริมาณคงคลังคงเหลือต่อสถานะสินค้าคงคลัง

<!-- -->

-   ปริมาณคงคลังคงเหลือของผลิตภัณฑ์จะแสดงอยู่ในวิธีการต่อไปนี้:
    -   ตามสินค้าคงคลังทางกายภาพ (มุมมองนี้แสดงยอดเงินรวม)
    -   ตามสินค้าคงคลังที่จองไว้ (มุมมองนี้แสดงยอดเงินที่จองไว้)
    -   ตามรายการที่มีอยู่จริง (มุมมองนี้แสดงถึงจำนวนที่ใช้ที่ไม่มีการจอง)

## <a name="get-started"></a>เริ่มต้น
เมื่อต้องการเริ่มต้นใช้งานบนอุปกรณ์เคลื่อนที่ของคุณ:

1.  จากร้านค้าแอพบนมือถือของคุณ ดาวน์โหลดและติดตั้งแอพ Microsoft Dynamics 365 for operations
2.  เริ่มการทำงานแอพบนอุปกรณ์ของคุณ
3.  ป้อน URL ของ Dynamics 365
4.  ป้อนบริษัทเพื่อลงชื่อเข้าใช้ ตัวอย่างเช่น ป้อน **USMF**
5.  ในครั้งแรกที่คุณเข้าสู่ระบบ คุณจะได้รับพร้อมท์สำหรับชื่อผู้ใช้และรหัสผ่านสำหรับบัญชี Microsoft Dynamics 365 for Operations ของคุณ ป้อนข้อมูลประจำตัวของคุณ หลังจากที่คุณลงชื่อเข้าใช้แล้ว คุณจะสามารถดูพื้นที่ทำงานที่พร้อมใช้งานสำหรับบริษัทของคุณ

เมื่อต้องการดูพื้นที่ทำงานในแอพบนมือถือของคุณ คุณต้องเผยแพร่พื้นที่ทำงานที่ต้องการไปยังแอพ Dynamics 365 for Operations

1.  เริ่มการทำงาน Dynamics 365 for Operations
2.  ไปที่ **การจัดการระบบ** &gt; **การตั้งค่า** &gt; **พารามิเตอร์ระบบ**
3.  เลือก **จัดการแอพบนมือถือ**
4.  เลือกพื้นที่ทำงานที่จะเผยแพร่ไปยังแพลตฟอร์มเคลื่อนที่
5.  เลือก **เผยแพร่พื้นที่ทำงาน**
6.  รีเฟรชอุปกรณ์ของคุณเพื่อดูพื้นที่ทำงานที่เผยแพร่แล้ว

## <a name="view-the-onhand-inventory-for-a-product"></a>ดูปริมาณคงคลังคงเหลือสำหรับผลิตภัณฑ์
1.  บนอุปกรณ์เคลื่อนที่ เลือกพื้นที่ทำงาน **ปริมาณคงคลังคงเหลือ**
2.  เลือก **ตรวจสอบปริมาณคงคลังคงเหลือสำหรับสินค้า** คุณเห็นรายการของผลิตภัณฑ์ที่ถูกโหลดลงในแอพของคุณสำหรับการใช้งานแบบออฟไลน์ โดยค่าเริ่มต้น จะมีการโหลดไว้ 50 รายการ แต่คุณสามารถเปลี่ยนแปลงจำนวนนี้ได้ สำหรับข้อมูลเพิ่มเติม ดูในคู่มือที่อ่านล่วงหน้า
3.  ถ้ารายการของคุณไม่ได้อยู่ในรายการ เลือก **ค้นหาเพิ่มเติม** เพื่อทำการค้นหาออนไลน์ใน Dynamics 365 for Operations ค้นหาโดยใช้หมายเลขผลิตภัณฑ์ หรือสลับไปยังการค้นหาตามชื่อผลิตภัณฑ์
4.  เลือกผลิตภัณฑ์ >  ถ้าสินค้ามีรูปภาพ รูปภาพจะแสดงขึ้น
5.  เลือกหนึ่งในตัวเลือกต่อไปนี้เพื่อดูสถานะของปริมาณคงคลังคงเหลือ:
    -   ดูปริมาณคงคลังคงเหลือต่อไซต์
    -   ดูปริมาณคงคลังคงเหลือต่อคลังสินค้า
    -   ดูปริมาณคงคลังคงเหลือต่อสถานที่
    -   ดูปริมาณคงคลังคงเหลือต่อชุดงาน (สำหรับผลิตภัณฑ์ที่มีการควบคุมชุดงาน)
    -   ดูปริมาณคงคลังคงเหลือต่อสถานะสินค้าคงคลัง

    ปริมาณคงคลังคงเหลือของผลิตภัณฑ์จะแสดงอยู่ในวิธีการต่อไปนี้:
    -   ตามสินค้าคงคลังทางกายภาพ (มุมมองนี้แสดงยอดเงินรวม)
    -   ตามสินค้าคงคลังที่จองไว้ (มุมมองนี้แสดงยอดเงินที่จองไว้)
    -   ตามรายการที่มีอยู่จริง (มุมมองนี้แสดงถึงจำนวนที่ใช้ที่ไม่มีการจอง)
