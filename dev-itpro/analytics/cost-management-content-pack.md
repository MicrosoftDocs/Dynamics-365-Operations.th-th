# <a name="cost-management-power-bi-content"></a>ต้นทุนการจัดการพลังงาน BI เนื้อหา

หัวข้อนี้อธิบายถึงสิ่งที่จะรวมในการจัดการต้นทุนพลังงาน BI เนื้อหา จะอธิบายถึงวิธีการเข้าถึงรายงานใช้พลังงาน BI และให้ข้อมูลเกี่ยวกับรูปแบบจำลองข้อมูลและเอนทิตีที่ใช้ในการสร้างเนื้อหา

# <a name="overview"></a>ภาพรวม

การ**การจัดการต้นทุน**เนื้อหา BI พลังงานของ Microsoft มีไว้สำหรับผู้จัดทำบัญชีสินค้าคงคลังหรือบุคคลในองค์กรที่รับผิดชอบสำหรับสินค้าคงคลัง การ**การจัดการต้นทุน**เนื้อหาพลังงาน BI ให้จัดการเข้าใจเกี่ยวกับสินค้าคงคลัง และสินค้าคงคลังของงานระหว่างทำ (WIP) และวิธีไหลของต้นทุนผ่านคิวตามประเภทช่วงเวลาหนึ่ง นอกจากนี้ยังสามารถใช้ข้อมูลเป็นรายละเอียดภาคผนวกกับงบการเงิน

## <a name="key-measures"></a>คีย์วัด

+ ยอดดุลต้นงวด
+ ยอดดุลสิ้นงวด
+ การเปลี่ยนแปลงสุทธิ
+ การเปลี่ยนแปลงสุทธิใน%
+ อายุหนี้

## <a name="key-performance-indicators"></a>ดัชนีชี้วัดผลการปฏิบัติงานหลัก
+ รอบการหมุนเวียนสินค้าคงคลัง
+ ความถูกต้องของสินค้าคงคลัง

แหล่งข้อมูลหลักสำหรับ CostAggregatedCostStatementEntryEntity คือ ตาราง CostStatementCache ตารางนี้จะถูกจัดการโดยกรอบแคชุดข้อมูล โดยค่าเริ่มต้น ตารางมีการปรับปรุงทุก 24 ชั่วโมง แต่คุณสามารถเปิดใช้งานการปรับปรุงด้วยตนเองในการตั้งค่าคอนฟิกการแคชของข้อมูล จากนั้นคุณสามารถทำการปรับปรุงด้วยตนเองในการ**จัดการต้นทุน**หรือ**ต้นทุนวิเคราะห์**พื้นที่ทำงานได้ หลังจากที่รันการปรับปรุงของ CostStatementCache คุณต้องปรับปรุงการเชื่อมต่อ OData พลังงาน BI.com เมื่อต้องการดูข้อมูลที่ปรับปรุงบนไซต์ หน่วยวัดผลต่าง (การซื้อ การผลิต) ในเนื้อหานี้ BI พลังงานที่เกี่ยวข้องกับสินค้าที่กำลังมีการตีราคา โดยวิธีสินค้าคงคลังต้นทุนมาตรฐานเท่านั้น ผลต่างการผลิตถูกคำนวณเป็นส่วนต่างระหว่างต้นทุนที่ใช้งานอยู่และต้นทุนที่เกิดขึ้นจริง มีคำนวณผลต่างการผลิตเมื่อใบสั่งผลิตมีสถานะเป็น**จบ** สำหรับข้อมูลเพิ่มเติมเกี่ยวกับชนิดผลต่างการผลิตและวิธีคำนวณแต่ละชนิด ดู[เกี่ยวกับการวิเคราะห์ผลต่างสำหรับใบสั่งผลิตที่เสร็จสมบูรณ์](https://technet.microsoft.com/en-us/library/gg242850.aspx)

## <a name="accessing-the-power-bi-content"></a>การเข้าถึงเนื้อหาพลังงาน BI
การ**ทุนจัดการ**BI พลังงานเนื้อหาจะพร้อมใช้งานจาก PowerBI.com สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการเชื่อมต่อ และโหลด 365 Dynamics ของคุณ Microsoft สำหรับข้อมูลในการดำเนินงาน ดู[BI พลังงานเข้าถึงเนื้อหาจาก PowerBI.com](power-bi-home-page.md)

## <a name="understanding-the-data-model-and-entities"></a>การทำความเข้าใจเกี่ยวกับแบบจำลองข้อมูลและเอนทิตี้
