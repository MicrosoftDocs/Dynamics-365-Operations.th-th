# <a name="recruiting-power-bi-content"></a>การสรรหาบุคลากรเนื้อหาพลังงาน BI

หัวข้อนี้อธิบายถึง 365 Dynamics สำหรับการดำเนินงาน - เนื้อหา BI พลังงานสรรหาบุคลากร จะอธิบายถึงวิธีการเข้าถึงรายงานที่รวมอยู่ในแพเนื้อหา และให้ข้อมูลเกี่ยวกับรูปแบบจำลองข้อมูลและเอนทิตีที่ใช้ในการสร้างแพเนื้อหา

<a name="accessing-the-content-pack"></a>เข้าถึงแพเนื้อหา
--------------------------

คุณสามารถค้นหาชุดเนื้อหาการสรรหาบุคลากรในไลบรารีสินทรัพย์ที่ใช้ร่วมกันใน Microsoft Dynamics วัฏจักรบริการ (LCS) สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการดาวน์โหลดแพคความเนื้อหา และการเชื่อมต่อกับ 365 Dynamics ของคุณ Microsoft สำหรับข้อมูลในการดำเนินงาน ดู[เนื้อหาพลังงาน BI ใน LCS จาก Microsoft และคู่ค้าของคุณ](power-bi-content-microsoft-partners.md)

## <a name="reports-that-are-included-in-the-content-pack"></a>รายงานที่รวมอยู่ในแพเนื้อหา
หลังจากที่คุณได้เชื่อมต่อแพเนื้อหา 365 Dynamics ของคุณสำหรับข้อมูลการดำเนินงาน รายงานแสดงข้อมูลขององค์กรของคุณ ถ้าคุณไม่เคยใช้ BI พลังงาน Microsoft ก่อน คุณก็สามารถเรียนรู้เพิ่มเติมเกี่ยวกับบนการ[หน้าตัวเรียนรู้สำหรับพลังงาน BI](https://powerbi.microsoft.com/en-us/guided-learning/?WT.mc_id=PBIService_GetData)ได้ รายงานที่รวมอยู่ในแพเนื้อหาทั้งแผนภูมิและตารางที่ประกอบด้วยข้อมูลเพิ่มเติมได้ ตารางต่อไปนี้ให้คำอธิบายเกี่ยวกับรายงาน

| รายงาน                       | เนื้อหา                                                                                               |
|------------------------------|--------------------------------------------------------------------------------------------------------|
| วิเคราะห์ผู้สมัคร           | ผู้สมัครตามงาน แหล่งผู้สมัคร ผู้สมัครตามตำแหน่งที่ตั้ง และจำนวนรวมของผู้สมัคร           |
| สถานะของผู้สมัคร             | ผู้สมัครตามชนิด และสถานะ และสถานะของผู้สมัคร                                                    |
| ข้อมูลประชากรของผู้สมัคร       | ผู้สมัครตามอายุและเพศ และผู้สมัครตามระดับการศึกษาและสถานะ                             |
| วิเคราะห์การสรรหาบุคลากร          | อัตราส่วนการจ้างงานสุทธิ วันที่จ้างงาน เปอร์เซ็นต์ของ hires ไม่ถูกต้อง และต้นทุนการสรรหาบุคลากรโดยเฉลี่ย                    |
| วิเคราะห์โครงการสรรหาบุคลากร | จำนวนของโครงการสรรหาบุคลากร เปิดตามโครงการสรรหาบุคลากร และผู้สมัครตามโครงการสรรหาบุคลากร |

คุณสามารถกรองข้อมูลแผนภูมิและในรายงานเหล่านี้ และตรึงแผนภูมิและแบบเรียงต่อกันไปยังแดชบอร์ด สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการใช้ตัวกรองและ pin ในการใช้พลังงาน BI ดู[สร้างและตั้งค่าคอนฟิกแดชบอร์ด A](https://powerbi.microsoft.com/en-us/guided-learning/powerbi-learning-4-2-create-configure-dashboards)

## <a name="understanding-the-data-model-and-entities"></a>การทำความเข้าใจเกี่ยวกับแบบจำลองข้อมูลและเอนทิตี้
Dynamics 365 สำหรับข้อมูลการดำเนินงานถูกใช้เพื่อรวบรวมรายงานในชุดเนื้อหาการสรรหาบุคลากร ตารางต่อไปนี้แสดงเอนทิตีว่า แพเนื้อหาเป็นไปตาม

| เอนทิตี้                          | เนื้อหา                                                         | ความสัมพันธ์กับเอนทิตีอื่น                                                                                                                                                                                                                 |
|---------------------------------|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| สรรหาบุคลากร\_ผู้สมัคร           | ผู้สมัคร ผู้สมัครเข้ อัตราการจ้างงานสุทธิ และต้นทุน          | สรรหาบุคลากร\_ApplicantName สรรหา\_บริษัทสรรหาบุคลากร\_CalendarOffset Recuriting\_สรรหาวัน\_GeographicLocation สรรหา\_ข้อมูลประชากรของการสรรหาบุคลากร\_งานการสรรหาบุคลากร\_สื่อการสรรหาบุคลากร\_RecruitmentProject                                |
| สรรหาบุคลากร\_ApplicantName       | ชื่อผู้สมัคร นามสกุล และชื่อเต็ม                   | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_CalendarOffset      | ปฏิทินออฟเซ็ตเพื่อรายงานชิ้น                                | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_บริษัท             | บริษัทสามารถกรองรายงานโดย                                   | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_วัน                | วัน สัปดาห์ เดือน และปี                                   | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_ข้อมูลประชากร        | วันเกิด เพศ เชื้อชาติ และสถานภาพการสมรส         | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_EmployedApplicant   | ผู้สมัคร ประสิทธิภาพการทำงาน วันเริ่มต้น และชนิดของผู้สมัคร           | สรรหาบุคลากร\_บริษัทสรรหาบุคลากร\_CalendarOffset สรรหา\_สรรหาวัน\_สรรหาบุคลากร GeographicLocation\_ApplicantName สรรหา\_จ้างงานการสรรหาบุคลากร\_สรรหาบุคลากรประสิทธิภาพ\_งานการสรรหาบุคลากร\_สื่อการสรรหาบุคลากร\_RecruitmentProject          |
| สรรหาบุคลากร\_จ้างงาน          | วันเริ่มต้น วันสิ้นสุด และวันเปลี่ยน                        | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_GeographicLocation  | เมือง รหัสไปรษณีย์ ประเทศ และรัฐ หรือจังหวัด                 | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_งาน                 | ฟังก์ชัน ชนิด และชื่อเรื่อง                                        | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_สื่อ               | แหล่งที่มาของผู้สมัคร                                             | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_ประสิทธิภาพการทำงาน         | จัดอันดับ คำอธิบาย และรูปแบบการประเมิน                            | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_RecruitmentProject  | คำอธิบายโครงการ สถานะโครงการ และเปิด                | สรรหาบุคลากร\_สรรหาผู้สมัคร\_สรรหาบุคลากร EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| สรรหาบุคลากร\_TerminatedApplicant | หยุดการทำงานผู้สมัคร เหตุผล ประสิทธิภาพ และวันสิ้นสุด | สรรหาบุคลากร\_บริษัทสรรหาบุคลากร\_CalendarOffset สรรหา\_วันสรรหาบุคลากร\_GeographicLocation สรรหาบุคลากร\_ประสิทธิภาพการสรรหาบุคลากร\_ข้อมูลประชากรของการสรรหาบุคลากร\_จ้างงานการสรรหาบุคลากร\_สื่อการสรรหาบุคลากร\_สรรหาบุคลากร RecruitmentProject\_ApplicantName |

เอนทิตีเหล่านี้ถูกใช้เพื่อสร้างการวัดที่คำนวณ คำนวณเหล่านี้จะมีใช้หน่วยวัดเพื่อคำนวณตัวบ่งชี้ประสิทธิภาพหลัก (Kpi) และรายงานที่ใช้ในแพเนื้อหา ถ้าคุณต้องการรวมการคำนวณเพิ่มเติมในรายงานและแดชบอร์ดของคุณ คุณสามารถดาวน์โหลด และปรับเปลี่ยนแฟ้ม Recruiting.pbix จาก LCS แฟ้มนี้เป็นรูปแบบข้อมูลเริ่มต้นที่ถูกใช้เพื่อสร้างแพเนื้อหา หลังจากที่คุณได้ทำการแก้ไข คุณสามารถสร้างการ pack เนื้อหาขององค์กรและแดชบอร์ดที่ประกอบด้วยข้อมูลที่คุณเพิ่ม
