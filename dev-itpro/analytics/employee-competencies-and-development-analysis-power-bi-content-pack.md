# <a name="employee-competencies-and-development-power-bi-content"></a>ความสามารถของพนักงานและ BI พลังงานพัฒนาเนื้อหา

หัวข้อนี้อธิบายถึง 365 Dynamics สำหรับการดำเนินงาน - ความสามารถของพนักงานและการพัฒนาพลังงาน BI เนื้อหา จะอธิบายถึงวิธีการเข้าถึงรายงานที่รวมอยู่ในแพเนื้อหา และให้ข้อมูลเกี่ยวกับรูปแบบจำลองข้อมูลและเอนทิตีที่ใช้ในการสร้างแพเนื้อหา

<a name="accessing-the-content-pack"></a>เข้าถึงแพเนื้อหา
--------------------------

คุณสามารถค้นหาความสามารถของพนักงานและพัฒนาเนื้อหาบรรจุในไลบรารีสินทรัพย์ที่ใช้ร่วมกันใน Microsoft Dynamics วัฏจักรบริการ (LCS) สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการดาวน์โหลดแพคความเนื้อหา และการเชื่อมต่อกับ 365 Dynamics ของคุณ Microsoft สำหรับข้อมูลในการดำเนินงาน ดู[เนื้อหาพลังงาน BI ใน LCS จาก Microsoft และคู่ค้าของคุณ](power-bi-content-microsoft-partners.md)

## <a name="reports-that-are-included-in-the-content-pack"></a>รายงานที่รวมอยู่ในแพเนื้อหา
หลังจากที่คุณได้เชื่อมต่อแพเนื้อหา 365 Dynamics ของคุณสำหรับข้อมูลการดำเนินงาน รายงานแสดงข้อมูลขององค์กรของคุณ ถ้าคุณไม่เคยใช้ BI พลังงาน Microsoft ก่อน คุณก็สามารถเรียนรู้เพิ่มเติมเกี่ยวกับบนการ[หน้าตัวเรียนรู้สำหรับพลังงาน BI](https://powerbi.microsoft.com/en-us/guided-learning/?WT.mc_id=PBIService_GetData)ได้ รายงานที่รวมอยู่ในแพเนื้อหาทั้งแผนภูมิและตารางที่ประกอบด้วยข้อมูลเพิ่มเติมได้ ตารางต่อไปนี้ให้คำอธิบายเกี่ยวกับรายงาน

| รายงาน                            | เนื้อหา                                               |
|-----------------------------------|--------------------------------------------------------|
| ความสามารถด้าน & วิเคราะห์การพัฒนา | ชนิดของทักษะสมาชิกทีมและทักษะสมาชิกทีมโดยเรียงตามชนิด |
| โพรไฟล์ทักษะ                     | โพรไฟล์ทักษะสำหรับพนักงานที่เลือก                |
| การวิเคราะห์ทักษะ                    | ทักษะโดยเรียงตามชนิดและการจัดอันดับ                              |

คุณสามารถกรองข้อมูลแผนภูมิและในรายงานเหล่านี้ และตรึงแผนภูมิและแบบเรียงต่อกันไปยังแดชบอร์ด สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการใช้ตัวกรองและ pin ในการใช้พลังงาน BI ดู[สร้างและตั้งค่าคอนฟิกแดชบอร์ด A](https://powerbi.microsoft.com/en-us/guided-learning/powerbi-learning-4-2-create-configure-dashboards)

## <a name="understanding-the-data-model-and-entities"></a>การทำความเข้าใจเกี่ยวกับแบบจำลองข้อมูลและเอนทิตี้
Dynamics 365 สำหรับข้อมูลการดำเนินงานถูกใช้เพื่อรวบรวมรายงานในความสามารถของพนักงานและการ pack เนื้อหาของการพัฒนา ตารางต่อไปนี้แสดงเอนทิตีว่า แพเนื้อหาเป็นไปตาม

| เอนทิตี้                            | เนื้อหา                                                                                                   | ความสัมพันธ์กับเอนทิตีอื่น                                                                                                                                                                                                                                                                       |
|-----------------------------------|------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| บุคลากร\_CalendarOffset         | ปฏิทินออฟเซ็ตเพื่อรายงานชิ้น                                                                          |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_บริษัท                | บริษัทสามารถกรองรายงานโดย                                                                             |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_ค่าตอบแทน           | อัตราค่าจ้างและความถี่ในช่วงเวลาหนึ่ง                                                                           |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_CurrentCompensation    | อัตราค่าจ้างและความถี่ ณวันปัจจุบัน                                                              | บุคลากร\_บุคลากรของบริษัท\_บุคลากร CurrentCompensation\_ข้อมูลประชากรบุคลากร\_งานบุคลากร\_ตำแหน่ง                                                                                                                                                                                            |
| บุคลากร\_CurrentPosition        | ตำแหน่งงาน ณวันปัจจุบัน เทียบเท่าเต็มเวลา (FTE), ตำแหน่งที่เปิด และเปิดการกรอกข้อมูลตำแหน่งงาน | แรงงาน\_งานบุคลากร\_ตำแหน่ง                                                                                                                                                                                                                                                                      |
| บุคลากร\_CurrentWorker          | ผู้ปฏิบัติงาน ณวันปัจจุบัน อายุ และจำนวนพนักงาน                                                         | บุคลากร\_บุคลากรของบริษัท\_แรงงานค่าตอบแทน\_บุคลากร GeographicLocation\_แรงงานประสิทธิภาพการทำงาน\_บุคลากร PersonSkill\_WorkerName บุคลากร\_บุคลากร ReportsToWorkerName\_WorkerTitle บุคลากร\_บุคลากรสถิติประชากร\_งานบุคลากร\_แรงงานจ้างงาน\_ตำแหน่ง                     |
| บุคลากร\_วัน                   | วัน สัปดาห์ เดือน และปี                                                                             |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_ข้อมูลประชากร           | วันเกิด เพศ เชื้อชาติ และสถานภาพการสมรส                                                   |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_จ้างงาน             | วันเริ่มต้น วันสิ้นสุด และวันเปลี่ยน                                                                  |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_GeographicLocation     | เมือง รหัสไปรษณีย์ ประเทศ และรัฐ หรือจังหวัด                                                           |                                                                                                                                                                                                                                                                                                         |
| แรงงาน\_งาน                    | ฟังก์ชัน ชนิด และชื่อเรื่อง                                                                                  |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_JobPreferredSkill      | ระดับความสำคัญ จัดอันดับ ทักษะ และทักษะ                                                                 | บุคลากร\_บุคลากรทักษะ\_งาน                                                                                                                                                                                                                                                                         |
| บุคลากร\_PastPositionAssignment | กำหนดเหตุผล วันเริ่มต้น วันสิ้นสุด และงาน                                                           | บุคลากร\_บุคลากร ClaendarOffset\_วันแรงงาน\_งานบุคลากร\_ตำแหน่ง                                                                                                                                                                                                                            |
| บุคลากร\_ประสิทธิภาพการทำงาน            | จัดอันดับ คำอธิบาย และรูปแบบการประเมิน                                                                      |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_PersonSkill            | ระดับ วันของระดับ และทักษะ                                                                               | บุคลากร\_ทักษะ                                                                                                                                                                                                                                                                                        |
| บุคลากร\_PersonSkillAnalysis    | วันได้รับการรับรอง ระดับ ระดับ และทักษะ                                                                    | บุคลากร\_บุคลากร WorkerName\_ทักษะ                                                                                                                                                                                                                                                                  |
| บุคลากร\_ตำแหน่ง               | แผนก FTE ตำแหน่ง ตำแหน่งชนิด และชื่อเรื่อง                                                        |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_PositionTrend          | ตำแหน่งตามช่วงเวลา FTE และงาน                                                                          | บุคลากร\_บุคลากร CalendarOffset\_วันแรงงาน\_งานบุคลากร\_ตำแหน่ง                                                                                                                                                                                                                            |
| บุคลากร\_ReportsToWorkerName    | ชื่อแรก นามสกุล และชื่อเต็ม                                                                       |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_ทักษะ                  | ทักษะ ชนิดของทักษะ และการจัดอันดับ                                                                              |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_TerminatedWorker       | เลิกจ้างผู้ปฏิบัติงาน วันหยุดชะงัก ชื่อ ตำแหน่งงาน และงาน                                             | บุคลากร\_บุคลากรของบริษัท\_แรงงานค่าตอบแทน\_บุคลากร GeographicLocation\_แรงงานประสิทธิภาพ\_บุคลากร WorkerName\_บุคลากร ReportsToWorkerName\_CalendarOffset Workforces\_วันแรงงาน\_บุคลากร WorkerTitle\_บุคลากรสถิติประชากร\_แรงงานจ้างงาน\_งานบุคลากร\_ตำแหน่ง |
| บุคลากร\_WorkerName             | ชื่อแรก นามสกุล และชื่อเต็ม                                                                       |                                                                                                                                                                                                                                                                                                         |
| บุคลากร\_WorkerTitle            | วันชื่อเรื่องและความอาวุโส                                                                                   |                                                                                                                                                                                                                                                                                                         |
| Workorce\_WorkerTrend             | ผู้ปฏิบัติงานช่วงเวลา จำนวนพนักงาน บริษัท และตำแหน่ง                                                        | บุคลากร\_บุคลากรของบริษัท\_แรงงานค่าตอบแทน\_บุคลากร GeographicLocation\_แรงงานประสิทธิภาพ\_บุคลากร WorkerName\_บุคลากร ReportsToWorkerName\_CalendarOffset Workforces\_วันแรงงาน\_บุคลากร WorkerTitle\_ข้อมูลประชากรบุคลากร\_แรงงานจ้างงาน\_งาน                     |

เอนทิตีเหล่านี้ถูกใช้เพื่อสร้างการวัดที่คำนวณในโมเดลข้อมูล คำนวณเหล่านี้จะมีใช้หน่วยวัดเพื่อคำนวณตัวบ่งชี้ประสิทธิภาพหลัก (Kpi) และรายงานที่ใช้ในแพเนื้อหา ถ้าคุณต้องการรวมการคำนวณเพิ่มเติมในรายงานและแดชบอร์ดของคุณ คุณสามารถดาวน์โหลด และปรับเปลี่ยนแฟ้ม CompetenciesandDevelopment.pbix จาก LCS แฟ้มนี้เป็นรูปแบบข้อมูลเริ่มต้นที่ถูกใช้เพื่อสร้างแพเนื้อหา หลังจากที่คุณได้ทำการแก้ไข คุณสามารถสร้างการ pack เนื้อหาขององค์กรและแดชบอร์ดที่ประกอบด้วยข้อมูลที่คุณเพิ่ม
