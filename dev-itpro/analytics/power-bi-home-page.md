## <a name="access-power-bi-content-from-lcs"></a>เข้าถึงเนื้อหาที่ BI พลังงานจาก LCS
สำหรับข้อมูลเกี่ยวกับวิธีการดาวน์โหลดเนื้อหาพลังงาน BI และเชื่อมต่อกับข้อมูลขององค์กรของคุณ ดู[เนื้อหาพลังงาน BI ใน LCS จาก Microsoft และคู่ค้าของคุณ](power-bi-content-microsoft-partners.md)

## <a name="access-power-bi-content-from-powerbicom"></a>เข้าถึงเนื้อหาที่ BI พลังงานจาก PowerBI.com
1.  ล็อกอินเพื่อ[PowerBI.com](https://www.powerbi.com/)
2.  คลิ**รับข้อมูล**
3.  ในการ**บริการ**กล่อง คลิ**รับ**
4.  เลือกแพเนื้อหาที่คุณต้องการ และจากนั้น คลิ**รับ**
5.  ป้อน URL ของ 365 Dynamics ของคุณสำหรับสภาพแวดล้อมของการดำเนินงาน URL ต้องอยู่ในรูปแบบ**https://&lt;YourAOSTenant&gt;cloudax.dynamics.com** Click **Next**.
6.  เลือก**oAuth2**เป็นวิธีการรับรองความถูกต้อง และคลิก**เข้าสู่ระบบ**
7.  เมื่อคุณได้รับพร้อมท์ ป้อนบัญชี Microsoft Office 365 ที่มีสิทธิ์เข้าถึง 365 Dynamics ของคุณสำหรับสภาพแวดล้อมของการดำเนินงาน
8.  หลังจากที่คุณได้เข้าสู่ระบบใน กระบวนการนำเข้าเริ่มต้นโดยอัตโนมัติ เมื่อการนำเข้าเสร็จสมบูรณ์แล้ว รายงานที่รวมอยู่ในแพเนื้อหาปรากฏในบานหน้าต่างนำทาง เลือกรายงานเพื่อดูข้อมูลที่นำเข้าของคุณ
