# <a name="workforce-metrics-power-bi-content"></a>บุคลากรเครื่องมือวัดพลังงาน BI เนื้อหา

หัวข้อนี้อธิบายถึง 365 Dynamics สำหรับการดำเนินงาน - เนื้อหา BI พลังงานวัดแรงงาน จะอธิบายถึงวิธีการเข้าถึงรายงานที่รวมอยู่ในแพเนื้อหา และให้ข้อมูลเกี่ยวกับรูปแบบจำลองข้อมูลและเอนทิตีที่ใช้ในการสร้างแพเนื้อหา

<a name="accessing-the-content-pack"></a>เข้าถึงแพเนื้อหา
--------------------------

คุณสามารถค้นหาแพเนื้อหาการวัดแรงงานในไลบรารีสินทรัพย์ที่ใช้ร่วมกันใน Microsoft Dynamics วัฏจักรบริการ (LCS) สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการดาวน์โหลดแพคความเนื้อหา และการเชื่อมต่อกับ 365 Dynamics ของคุณ Microsoft สำหรับข้อมูลในการดำเนินงาน ดู[เนื้อหาพลังงาน BI ใน LCS จาก Microsoft และคู่ค้าของคุณ](power-bi-content-microsoft-partners.md)

## <a name="reports-that-are-included-in-the-content-pack"></a>รายงานที่รวมอยู่ในแพเนื้อหา
หลังจากที่คุณได้เชื่อมต่อแพเนื้อหา 365 Dynamics ของคุณสำหรับข้อมูลการดำเนินงาน รายงานแสดงข้อมูลขององค์กรของคุณ ถ้าคุณไม่เคยใช้ BI พลังงาน Microsoft ก่อน คุณก็สามารถเรียนรู้เพิ่มเติมเกี่ยวกับบนการ[หน้าตัวเรียนรู้สำหรับพลังงาน BI](https://powerbi.microsoft.com/en-us/guided-learning/?WT.mc_id=PBIService_GetData)ได้ รายงานที่รวมอยู่ในแพเนื้อหาทั้งแผนภูมิและตารางที่ประกอบด้วยข้อมูลเพิ่มเติมได้ ตารางต่อไปนี้ให้คำอธิบายเกี่ยวกับรายงาน

| รายงาน                                           | เนื้อหา                                                                                                                                                                                                            |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ตำแหน่งที่ตั้งของ บริษัทการวิเคราะห์จำนวนพนักงาน แผนก | จำนวนพนักงาน โดยบริษัท จำนวนพนักงานตามแผนก จำนวนพนักงานตามที่ตั้ง และจำนวนคนทั้งหมด                                                                                                                           |
| ตัวจัดการงาน ขั้นตอน การวิเคราะห์จำนวนพนักงาน            | จำนวนพนักงาน โดยงาน จำนวนพนักงาน โดยขั้นตอน จำนวนพนักงานผู้จัดการ และจำนวนคนทั้งหมด                                                                                                                                      |
| การวิเคราะห์แนวโน้มของจำนวนพนักงาน                         | จำนวนพนักงานปีนี้เทียบกับปีที่แล้วโดยบริษัทและจำนวนพนักงานกำลังย้อนสำหรับ 12 เดือนที่ผ่านมา                                                                                                                        |
| ข้อมูลประชากรของแรงงาน                           | จำนวนพนักงานตามอายุ และเพศ จำนวนพนักงาน โดยเชื้อชาติ จำนวนพนักงานโดยเรียงตามสถานะทหารผ่านศึก จำนวนพนักงานตามสถานภาพการสมรส จำนวนนักศึกษาเต็มเวลา อายุงานเฉลี่ย อายุเฉลี่ย และอัตราส่วนของสตรีแก่บุรุษพนักงาน |
| วิเคราะห์ตำแหน่ง                                | เปิดตำแหน่งตามแผนก เปิดการเติมตำแหน่ง ตำแหน่งที่ใช้งานอยู่เพื่อไม่ได้ใช้งาน และตำแหน่งตามแผนก                                                                                                   |
| การวิเคราะห์ attrition                               | Attrition ปีนี้เทียบกับปีที่แล้ว attrition อายุงานเฉลี่ยของคนที่ทิ้ง อายุเฉลี่ยของคนที่ทิ้ง และพนักงานทิ้ง ด้วยเหตุผล                                                                   |
| บุคลากรโดยเรียงตามแผนก                             | พนักงานที่ มีหมายเลขบุคลากรโดยแผนก ตำแหน่ง และการกำหนดค่าเริ่มต้น และวันสิ้นสุด                                                                                                                       |
| การวิเคราะห์อายุงาน                               | ปีเฉลี่ยของการบริการตามรายการของบริษัทและความอาวุโส                                                                                                                                                              |
| วันครบรอบและปีของการบริการ               | พนักงาน ด้วยจำนวนปีของการบริการ และวันครบรอบ                                                                                                                                                                    |

คุณสามารถกรองข้อมูลแผนภูมิและในรายงานเหล่านี้ และตรึงแผนภูมิและแบบเรียงต่อกันไปยังแดชบอร์ด สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการใช้ตัวกรองและ pin ในการใช้พลังงาน BI ดู[สร้างและตั้งค่าคอนฟิกแดชบอร์ด A](https://powerbi.microsoft.com/en-us/guided-learning/powerbi-learning-4-2-create-configure-dashboards)

## <a name="understanding-the-data-model-and-entities"></a>การทำความเข้าใจเกี่ยวกับแบบจำลองข้อมูลและเอนทิตี้
Dynamics 365 สำหรับข้อมูลการดำเนินงานถูกใช้เพื่อรวบรวมรายงานในแพเนื้อหาการวัดแรงงาน ตารางต่อไปนี้แสดงเอนทิตีว่า แพเนื้อหาเป็นไปตาม

| เอนทิตี้                            | เนื้อหา                                                                                                   | ความสัมพันธ์กับเอนทิตีอื่น                                                                                                                                                                                                                                                                                                |
|-----------------------------------|------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| บุคลากร\_CalendarOffset         | ปฏิทินออฟเซ็ตเพื่อรายงานชิ้น                                                                          | บุคลากร\_บุคลากร PastPositionAssignment\_PositionTrend Workorce\_บุคลากร WorkerTrend\_TerminatedWorker                                                                                                                                                                                                                     |
| บุคลากร\_บริษัท                | บริษัทสามารถกรองรายงานโดย                                                                             | บุคลากร\_บุคลากร CurrentCompensation\_บุคลากร CurrentWorker\_TerminatedWorker Workorce\_WorkerTrend                                                                                                                                                                                                                        |
| บุคลากร\_ค่าตอบแทน           | อัตราค่าจ้างและความถี่ในช่วงเวลาหนึ่ง                                                                           | บุคลากร\_บุคลากร CurrentCompensation\_บุคลากร CurrentWorker\_TerminatedWorker Workorce\_WorkerTrend                                                                                                                                                                                                                        |
| บุคลากร\_CurrentCompensation    | อัตราค่าจ้างและความถี่ ณวันปัจจุบัน                                                              | บุคลากร\_บุคลากรของบริษัท\_แรงงานค่าตอบแทน\_ข้อมูลประชากรบุคลากร\_งานบุคลากร\_ตำแหน่ง                                                                                                                                                                                                                            |
| บุคลากร\_CurrentPosition        | ตำแหน่งงาน ณวันปัจจุบัน เทียบเท่าเต็มเวลา (FTE), ตำแหน่งที่เปิด และเปิดการกรอกข้อมูลตำแหน่งงาน | แรงงาน\_งานบุคลากร\_ตำแหน่ง                                                                                                                                                                                                                                                                                               |
| บุคลากร\_CurrentWorker          | ผู้ปฏิบัติงาน ณวันปัจจุบัน อายุ และจำนวนพนักงาน                                                         | บุคลากร\_บุคลากรของบริษัท\_แรงงานค่าตอบแทน\_บุคลากร GeographicLocation\_แรงงานประสิทธิภาพการทำงาน\_บุคลากร WorkerName\_ReportsToWorkerName บุคลากร\_บุคลากร WorkerTitle\_บุคลากรสถิติประชากร\_งานบุคลากร\_แรงงานจ้างงาน\_วางตำแหน่งบุคลากร\_WorkerBenefit                                            |
| บุคลากร\_วัน                   | วัน สัปดาห์ เดือน และปี                                                                             | บุคลากร\_บุคลากร PastPositionAssignment\_บุคลากร PositionTrend\_TerminatedWorker Workorce\_WorkerTrend                                                                                                                                                                                                                     |
| บุคลากร\_ข้อมูลประชากร           | วันเกิด เพศ เชื้อชาติ และสถานภาพการสมรส                                                   | บุคลากร\_บุคลากร CurrentWorker\_TerminatedWorker Workorce\_WorkerTrend                                                                                                                                                                                                                                                       |
| บุคลากร\_จ้างงาน             | วันเริ่มต้น วันสิ้นสุด และวันเปลี่ยน                                                                  | บุคลากร\_บุคลากร CurrentWorker\_TerminatedWorker Workorce\_WorkerTrend                                                                                                                                                                                                                                                       |
| บุคลากร\_GeographicLocation     | เมือง รหัสไปรษณีย์ ประเทศ และรัฐ หรือจังหวัด                                                           | บุคลากร\_บุคลากร CurrentWorker\_TerminatedWorker Workorce\_WorkerTrend                                                                                                                                                                                                                                                       |
| แรงงาน\_งาน                    | ฟังก์ชัน ชนิด และชื่อเรื่อง                                                                                  | บุคลากร\_บุคลากร CurrentPosition\_CurrentWorker                                                                                                                                                                                                                                                                              |
| บุคลากร\_JobPerferredSkill      |                                                                                                            |                                                                                                                                                                                                                                                                                                                                  |
| บุคลากร\_PastPositionAssignment | กำหนดเหตุผล วันเริ่มต้น วันสิ้นสุด และงาน                                                           | บุคลากร\_บุคลากร CalendarOffset\_วันแรงงาน\_งานบุคลากร\_ตำแหน่ง                                                                                                                                                                                                                                                     |
| บุคลากร\_ประสิทธิภาพการทำงาน            | จัดอันดับ คำอธิบาย และรูปแบบการประเมิน                                                                      | บุคลากร\_บุคลากร CurrentWorker\_TerminatedWorker Workorce\_WorkerTrend                                                                                                                                                                                                                                                       |
| บุคลากร\_PersonSkill            | ระดับและทักษะ                                                                                            | บุคลากร\_ทักษะ                                                                                                                                                                                                                                                                                                                 |
| บุคลากร\_PersonSkillAnalysis    | ได้รับการรับรอง ระดับ และทักษะ                                                                                | บุคลากร\_บุคลากรทักษะ\_WorkerName                                                                                                                                                                                                                                                                                           |
| บุคลากร\_ตำแหน่ง               | แผนก FTE ตำแหน่ง ตำแหน่งชนิด และชื่อเรื่อง                                                        | บุคลากร\_บุคลากร CurrentPosition\_CurrentWorker                                                                                                                                                                                                                                                                              |
| บุคลากร\_PositionTrend          | ตำแหน่งตามช่วงเวลา FTE และงาน                                                                          | บุคลากร\_บุคลากร CalendarOffset\_วันแรงงาน\_งานบุคลากร\_ตำแหน่ง                                                                                                                                                                                                                                                     |
| บุคลากร\_ReportsToWorkerName    | ชื่อแรก นามสกุล และชื่อเต็ม                                                                       | บุคลากร\_บุคลากร CurrentWorker\_TerminatedWorker Workorce\_WorkerTrend                                                                                                                                                                                                                                                       |
| บุคลากร\_ทักษะ                  | ทักษะ ชนิดของทักษะ และการจัดอันดับ                                                                              | บุคลากร\_บุคลากร PersonSkill\_PersonSkillAnalysis                                                                                                                                                                                                                                                                            |
| บุคลากร\_TerminatedWorker       | เลิกจ้างผู้ปฏิบัติงาน วันหยุดชะงัก ชื่อ ตำแหน่งงาน และงาน                                             | บุคลากร\_บุคลากรของบริษัท\_แรงงานค่าตอบแทน\_บุคลากร GeographicLocation\_แรงงานประสิทธิภาพการทำงาน\_บุคลากร WorkerName\_บุคลากร ReportsToWorkerName\_CalendarOffset Workforces\_วันแรงงาน\_บุคลากร WorkerTitle\_ข้อมูลประชากรบุคลากร\_แรงงานจ้างงาน\_งานบุคลากร\_วางตำแหน่งบุคลากร\_WorkerBenefit |
| บุคลากร\_WorkerBenefit          | วันมีผลบังคับใช้ ตัวเลือกสวัสดิการ แผนสวัสดิการ และชนิดของสวัสดิการ                                             | บุคลากร\_บุคลากร CurrentWorker\_TerminatedWorker Workorce\_WorkerTrend                                                                                                                                                                                                                                                       |
| บุคลากร\_WorkerName             | ชื่อแรก นามสกุล และชื่อเต็ม                                                                       | บุคลากร\_บุคลากร CurrentWorker\_TerminatedWorker Workorce\_บุคลากร WorkerTrend\_PersonSkillAnalysis                                                                                                                                                                                                                        |
| บุคลากร\_WorkerTitle            | วันชื่อเรื่องและความอาวุโส                                                                                   | บุคลากร\_บุคลากร CurrentWorker\_TerminatedWorker Workorce\_WorkerTrend                                                                                                                                                                                                                                                       |
| Workorce\_WorkerTrend             | ผู้ปฏิบัติงานช่วงเวลา จำนวนพนักงาน บริษัท และตำแหน่ง                                                        | บุคลากร\_บุคลากรของบริษัท\_แรงงานค่าตอบแทน\_บุคลากร GeographicLocation\_แรงงานประสิทธิภาพ\_บุคลากร WorkerName\_บุคลากร ReportsToWorkerName\_CalendarOffset Workforces\_วันแรงงาน\_บุคลากร WorkerTitle\_บุคลากรสถิติประชากร\_แรงงานจ้างงาน\_งานบุคลากร\_WorkerBenefit                     |

เอนทิตีเหล่านี้ถูกใช้เพื่อสร้างการวัดที่คำนวณในโมเดลข้อมูล คำนวณเหล่านี้จะมีใช้หน่วยวัดเพื่อคำนวณตัวบ่งชี้ประสิทธิภาพหลัก (Kpi) และรายงานที่ใช้ในแพเนื้อหา ถ้าคุณต้องการรวมการคำนวณเพิ่มเติมในรายงานและแดชบอร์ดของคุณ คุณสามารถดาวน์โหลด และปรับเปลี่ยนแฟ้ม CompensationandBenefits.pbix จาก LCS แฟ้มนี้เป็นรูปแบบข้อมูลเริ่มต้นที่ถูกใช้เพื่อสร้างแพเนื้อหา หลังจากที่คุณได้ทำการแก้ไข คุณสามารถสร้างการ pack เนื้อหาขององค์กรและแดชบอร์ดที่ประกอบด้วยข้อมูลที่คุณเพิ่ม
