# <a name="organizational-training-power-bi-content"></a>เนื้อหาการฝึกอบรมพลังงาน BI ขององค์กร

หัวข้อนี้อธิบายถึง 365 Dynamics สำหรับการดำเนินงาน - เนื้อหา BI พลังงานฝึกอบรมขององค์กร จะอธิบายถึงวิธีการเข้าถึงแพเนื้อหา และอธิบายถึงรูปแบบข้อมูลและเอนทิตีที่ใช้ในการสร้างแพเนื้อหา

<a name="accessing-the-content-pack"></a>เข้าถึงแพเนื้อหา
--------------------------

คุณสามารถค้นหาแพเนื้อหาฝึกอบรมองค์กรในไลบรารีสินทรัพย์ที่ใช้ร่วมกันใน Microsoft Dynamics วัฏจักรบริการ (LCS) สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการดาวน์โหลดแพคความเนื้อหา และการเชื่อมต่อกับ 365 Dynamics ของคุณ Microsoft สำหรับข้อมูลในการดำเนินงาน ดู[เนื้อหาพลังงาน BI ใน LCS จาก Microsoft และคู่ค้าของคุณ](power-bi-content-microsoft-partners.md)

## <a name="reports-that-are-included-in-the-content-pack"></a>รายงานที่รวมอยู่ในแพเนื้อหา
หลังจากที่คุณได้เชื่อมต่อแพเนื้อหา 365 Dynamics ของคุณสำหรับข้อมูลการดำเนินงาน รายงานแสดงข้อมูลขององค์กรของคุณ ถ้าคุณไม่เคยใช้ BI พลังงาน Microsoft ก่อน คุณก็สามารถเรียนรู้เพิ่มเติมเกี่ยวกับบนการ[หน้าตัวเรียนรู้สำหรับพลังงาน BI](https://powerbi.microsoft.com/en-us/guided-learning/?WT.mc_id=PBIService_GetData)ได้ รายงานที่รวมอยู่ในแพเนื้อหาทั้งแผนภูมิและตารางที่ประกอบด้วยข้อมูลเพิ่มเติมได้ ตารางต่อไปนี้ให้คำอธิบายเกี่ยวกับรายงาน

| รายงาน          | เนื้อหา                                                                    |
|-----------------|-----------------------------------------------------------------------------|
| วิเคราะห์หลักสูตร | ลงทะเบียนตามสถาน ผู้เข้าร่วมหลักสูตรตามสถานะ และรายการลงทะเบียน |
| ชนิดของหลักสูตร    | ชนิดของหลักสูตรตามทักษะ                                                       |

คุณสามารถกรองข้อมูลแผนภูมิและในรายงานเหล่านี้ และตรึงแผนภูมิและแบบเรียงต่อกันไปยังแดชบอร์ด สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการใช้ตัวกรองและ pin ในการใช้พลังงาน BI ดู[สร้างและตั้งค่าคอนฟิกแดชบอร์ด A](https://powerbi.microsoft.com/en-us/guided-learning/powerbi-learning-4-2-create-configure-dashboards)

## <a name="understanding-the-data-model-and-entities"></a>การทำความเข้าใจเกี่ยวกับแบบจำลองข้อมูลและเอนทิตี้
Dynamics 365 สำหรับข้อมูลการดำเนินงานถูกใช้เพื่อรวบรวมรายงานในแพเนื้อหาการฝึกอบรมขององค์กร ตารางต่อไปนี้แสดงเอนทิตีว่า แพเนื้อหาเป็นไปตาม

| เอนทิตี้                    | เนื้อหา                                                         | ความสัมพันธ์กับเอนทิตีอื่น                                                                                                                                                                  |
|---------------------------|------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| การฝึกอบรม\_CalendarOffset  | ปฏิทินออฟเซ็ตเพื่อรายงานชิ้น                                | การฝึกอบรม\_ฝึกอบรม CourseAgenda\_CourseAttendees                                                                                                                                                   |
| การฝึกอบรม\_บริษัท         | บริษัทสามารถกรองรายงานโดย                                   | การฝึกอบรม\_ฝึกอบรม CourseAgenda\_CourseAttendees                                                                                                                                                   |
| การฝึกอบรม\_หลักสูตร          | หลักสูตร คำอธิบาย ชื่อผู้สอน สถาน ห้อง และสถานะ | การฝึกอบรม\_ฝึกอบรม CourseAgenda\_ฝึกอบรม CourseAttendees\_CourseSkill                                                                                                                             |
| การฝึกอบรม\_CourseAgenda    | วาระการประชุม หลักสูตร และเวลาเริ่มต้นและสิ้นสุด                          | การฝึกอบรม\_บริษัทฝึกอบรม\_ฝึกอบรม CalendarOffset\_วันฝึกอบรม\_หลักสูตร                                                                                                                         |
| การฝึกอบรม\_CourseAttendees | วันชื่อ สถานะ งาน และการลงทะเบียน                         | การฝึกอบรม\_บริษัทฝึกอบรม\_ฝึกอบรม CalendarOffset\_วันที่สิ้นสุดการฝึกอบรม\_ข้อมูลประชากรของการฝึกอบรม\_ฝึกอบรมการจ้างงาน\_ฝึกอบรมในหลักสูตร\_ฝึกอบรม WorkerName\_ฝึกอบรม WorkerTitle\_งานฝึกอบรม\_ตำแหน่ง |
| การฝึกอบรม\_CourseSkill     | ทักษะ ชนิดของทักษะ และระดับ                                     | การฝึกอบรม\_หลักสูตร                                                                                                                                                                                   |
| การฝึกอบรม\_วัน            | วัน สัปดาห์ เดือน และปี                                   | การฝึกอบรม\_ฝึกอบรม CourseAgenda\_CourseAttendees                                                                                                                                                   |
| การฝึกอบรม\_ข้อมูลประชากร    | วันเกิด เพศ เชื้อชาติ และสถานภาพการสมรส         | การฝึกอบรม\_ฝึกอบรม CourseAgenda\_CourseAttendees                                                                                                                                                   |
| การฝึกอบรม\_จ้างงาน      | วันเริ่มต้น วันสิ้นสุด และวันเปลี่ยน                        | การฝึกอบรม\_ฝึกอบรม CourseAgenda\_CourseAttendees                                                                                                                                                   |
| การฝึกอบรม\_งาน             | ฟังก์ชัน ชนิด และชื่อเรื่อง                                        | การฝึกอบรม\_ฝึกอบรม CourseAgenda\_CourseAttendees                                                                                                                                                   |
| การฝึกอบรม\_ตำแหน่ง        | ตำแหน่งงาน ชื่อ และเทียบเท่าเต็มเวลา (FTE)                  | การฝึกอบรม\_ฝึกอบรม CourseAgenda\_CourseAttendees                                                                                                                                                   |
| การฝึกอบรม\_WorkerName      | ชื่อแรก นามสกุล และชื่อเต็ม                             | การฝึกอบรม\_CourseAttendees                                                                                                                                                                          |
| การฝึกอบรม\_WorkerTitle     | วันชื่อเรื่องและความอาวุโส                                         | การฝึกอบรม\_CourseAttendees                                                                                                                                                                          |

เอนทิตีเหล่านี้ถูกใช้เพื่อสร้างการวัดที่คำนวณในโมเดลข้อมูล คำนวณเหล่านี้จะมีใช้หน่วยวัดเพื่อคำนวณตัวบ่งชี้ประสิทธิภาพหลัก (Kpi) และรายงานที่ใช้ในแพเนื้อหา ถ้าคุณต้องการรวมการคำนวณเพิ่มเติมในรายงานและแดชบอร์ดของคุณ คุณสามารถดาวน์โหลด และปรับเปลี่ยนแฟ้ม Training.pbix จาก LCS แฟ้มนี้เป็นรูปแบบข้อมูลเริ่มต้นที่ถูกใช้เพื่อสร้างแพเนื้อหา หลังจากที่คุณได้ทำการแก้ไข คุณสามารถสร้างการ pack เนื้อหาขององค์กรและแดชบอร์ดที่ประกอบด้วยข้อมูลที่คุณเพิ่ม
