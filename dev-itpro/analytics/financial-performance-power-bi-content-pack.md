# <a name="financial-performance-power-bi-content"></a>เนื้อหา BI พลังงานประสิทธิภาพทางการเงิน

หัวข้อนี้อธิบายถึง 365 Dynamics ของ Microsoft สำหรับประสิทธิภาพทางการเงินของการดำเนินงานชุดเนื้อหาสำหรับ Microsoft พลังงาน BI จะอธิบายแดชบอร์ดและรายงานที่รวมอยู่ในแพเนื้อหา และให้ข้อมูลเกี่ยวกับรูปแบบจำลองข้อมูลและเอนทิตีที่ใช้ในการสร้างแพเนื้อหา

<a name="accessing-the-content-pack"></a>เข้าถึงแพเนื้อหา
--------------------------

เวอร์ชันสองเวอร์ชันของชุดเนื้อหาประสิทธิภาพทางการเงินจะพร้อมใช้งาน เวอร์ชันหนึ่งจะพร้อมใช้งานจาก Microsoft Dynamics วัฏจักรบริการ (LCS), และอื่น ๆ พร้อมใช้งานจาก PowerBI.com

-   **รุ่นที่พร้อมใช้งานจาก LCS:**ประสิทธิภาพทางการเงินแบบแพ็คเนื้อหาที่พร้อมใช้งานจาก LCS สนับสนุน Microsoft Dynamics 365 สำหรับการดำเนินงานรุ่น 1611 คุณสามารถค้นหาแพเนื้อหาในไลบรารีสินทรัพย์ที่ใช้ร่วมกันใน LCS สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการดาวน์โหลดแพคความเนื้อหา และการเชื่อมต่อกับ 365 Dynamics ของคุณ Microsoft สำหรับข้อมูลในการดำเนินงาน ดู[เนื้อหาพลังงาน BI ใน LCS จาก Microsoft และคู่ค้าของคุณ](power-bi-content-microsoft-partners.md)
-   **รุ่นที่พร้อมใช้งานจาก PowerBI.com:**ประสิทธิภาพทางการเงินแบบแพ็คเนื้อหาที่พร้อมใช้งานจาก PowerBI.com สนับสนุน Microsoft Dynamics AX เวอร์ชัน 7.0 และ 7.0.1 สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการเชื่อมต่อ และโหลด 365 Dynamics ของคุณสำหรับข้อมูลการดำเนินการ ดู[BI พลังงานเข้าถึงเนื้อหาจาก PowerBI.com](power-bi-home-page.md)

## <a name="main-account-setup"></a>ตั้งค่าบัญชีหลัก
เนื่องจากองค์กรต้องหนี้สินและยอดเงินรายได้ที่ปรากฏเป็นจำนวนค่าบวกในรายงาน การตั้งค่าบัญชีหลักใน 365 Dynamics สำหรับการดำเนินงานมีความสำคัญ สำหรับเหล่านี้บัญชีหลักที่จะปรากฏเป็นจำนวนค่าบวก คุณต้องกำหนดชนิดบัญชีหลักเป็น**หนี้สิน**หรือ**รายได้** เมื่อมีใช้บัญชีชนิดเหล่านี้ รายงานผ่าน Microsoft พลังงาน BI จะกลับเครื่องหมายต่าง ๆ และแสดงยอดเงินเป็นค่าบวก

## <a name="dashboard-and-reports-that-are-included-in-the-content-pack"></a>แดชบอร์ดและรายงานที่รวมอยู่ในแพเนื้อหา
หลังจากที่คุณได้เชื่อมต่อชุดเนื้อหากับข้อมูล Dynamics 365 for Operations ของคุณแล้ว แดชบอร์ดและรายงานจะแสดงข้อมูลทางการเงินของคุณ ถ้าคุณไม่เคยใช้พลังงาน BI ก่อน คุณสามารถเรียนรู้เพิ่มเติมเกี่ยวกับบนการ[หน้าตัวเรียนรู้สำหรับพลังงาน BI](https://powerbi.microsoft.com/en-us/guided-learning/?WT.mc_id=PBIService_GetData)ได้ แดชบอร์ดประกอบด้วยไทล์สรุปของข้อมูลที่เป็นไปตามรายงานที่เน้น แต่ละไทล์ประกอบด้วยข้อมูลสรุปของปีปัจจุบันระหว่างบริษัททั้งหมดในองค์กร ต่อไปนี้เป็นบางไพ่:

-   เงินสด
-   รายได้รวมปีนี้
-   ค่าใช้จ่ายรวมปีนี้
-   กำไรสุทธิปีนี้
-   อัตราส่วนเงินทุนหมุนเวียนเร็ว
-   ค่าใช้จ่ายรวมปีนี้โดยเรียงตามประเภทบัญชี
-   อัตราส่วนเงินทุนหมุนเวียน
-   หนี้สินถึงสินทรัพย์รวม
-   รายได้ที่เกิดขึ้นจริงเทียบกับที่คาดการณ์
-   รายได้ที่เรียกเก็บปีนี้
-   อัตราส่วนค่าใช้จ่ายในการทำงานปีนี้
-   กำไรขั้นต้นปีนี้
-   ค่าใช้จ่ายที่เกิดขึ้นจริงเทียบกับงบประมาณ – บริษัททั้งหมด

มีสนับสนุนไพ่แต่ละรายงานสนับสนุน รายงานเหล่านี้ประกอบด้วยทั้งแผนภูมิและตารางที่ให้ข้อมูลเพิ่มเติม ตารางต่อไปนี้ให้คำอธิบายเกี่ยวกับรายงาน

| การวิเคราะห์ต้นทุนขาย | ต้นทุนขาย (COGS) โดยเรียงตามนิติบุคคล COGS ปีนี้และปีที่แล้วโดยเรียงตามไตรมาส COGS ต่อการขายโดยเรียงตามนิติบุคคล COGS รวมและ COGS ต่อเปอร์เซ็นต์การขาย                                                                                                                                                                                   |

สามารถกรองข้อมูลและตรึงแผนภูมิและไทล์ในรายงานเหล่านี้ทั้งหมดไปยังแดชบอร์ดได้ ดูข้อมูลเพิ่มเติมเกี่ยวกับวิธีการกรองข้อมูลและตรึงใน Power BI ได้ที่ [สร้างและตั้งค่าคอนฟิกแดชบอร์ด](https://powerbi.microsoft.com/en-us/guided-learning/powerbi-learning-4-2-create-configure-dashboards)

## <a name="understanding-the-data-model-and-entities"></a>การทำความเข้าใจเกี่ยวกับแบบจำลองข้อมูลและเอนทิตี้
ข้อมูลที่เติมข้อมูลแดชบอร์ดและรายงานในแพเนื้อหาประสิทธิภาพทางการเงิน มี Dynamics 365 สำหรับข้อมูลการดำเนินงาน เอนทิตีต่อไปนี้ถูกใช้เป็นข้อมูลพื้นฐานของชุดเนื้อหา:**รวมเอนทิตีข้อมูล**

-   **GeneralLedgerActivities** – เอนทิตีนี้รวมยอดดุลบัญชีแยกประเภทตามประเภทบัญชี
-   **BudgetActivities** – เอนทิตีนี้รวมยอดดุลงบประมาณตามประเภทบัญชี

**Data entities**

-   FiscalCalendars
-   MainAccounts
-   LegalEntities
-   บัญชีแยกประเภท
-   ChartofAccounts

เอนทิตีเหล่านี้ถูกใช้เพื่อสร้างการวัดที่คำนวณในโมเดลข้อมูล การวัดที่คำนวณจะใช้ในการคำนวณตัวบ่งชี้ประสิทธิภาพหลัก (Kpi) และรายงานที่ใช้ในแพเนื้อหา โดยค่าเริ่มต้น ชุดเนื้อหาจะนำเข้าข้อมูลของสามปีที่ผ่านมาและหนึ่งปีในอนาคต เมื่อต้องการรวมการคำนวณเพิ่มเติมในรายงานและแดชบอร์ดของคุณ คุณสามารถปรับเปลี่ยน [สมุดงาน Microsoft Excel](https://mbs.microsoft.com/customersource/global/AX/downloads/reports/msdaxfinpercontentpowerbi) ได้ สมุดงานนี้เป็นแบบจำลองข้อมูลเริ่มต้นที่ใช้ในการสร้างชุดเนื้อหา หลังจากที่คุณทำการเปลี่ยนแปลงเสร็จเรียบร้อยแล้ว คุณสามารถสร้างชุดเนื้อหาและแดชบอร์ดขององค์กรที่ประกอบด้วยข้อมูลที่คุณเพิ่มไว้ได้
