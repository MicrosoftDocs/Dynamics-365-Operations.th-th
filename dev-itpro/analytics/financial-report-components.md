# <a name="financial-report-components"></a>ส่วนประกอบของรายงานทางการเงิน

[!include[banner](../includes/banner.md)]


บทความนี้อธิบายวิธีการใช้ส่วนประกอบ หรือบล็อคส่วนประกอบของข้อกำหนดของรายงานในการรายงานทางการเงิน บล็อคส่วนประกอบเหล่านี้รวมคำนิยามแถว คำนิยามคอลัมน์ และคำนิยามแผนภูมิรายงาน บทความนี้อธิบายวิธีการจัดระเบียบและล็อคบล็อคส่วนประกอบ และวิธีการทำงานกับกลุ่มบล็อคส่วนประกอบ 

หลักการในการออกแบบภายใต้ผู้ออกแบบรายงานทางการเงินคือการแบ่งข้อมูลออกเป็นส่วนประกอบหรือบล็อคส่วนประกอบที่เล็กที่สุด และผสมและจับคู่กับส่วนประกอบต่างๆ ตามที่กำหนด ดังนั้น การจัดรูปแบบรายงานของคุณจะแยกต่างหากจากข้อมูลทางการเงินของคุณ และคุณสามารถเปลี่ยนแปลงการออกแบบรายงานได้โดยไม่ต้องปรับเปลี่ยนข้อมูลทางการเงินในระบบ Microsoft Dynamics ERP ของคุณ หากใช้วิธีการบล็อคส่วนประกอบนี้ คุณสามารถรวมข้อความ ยอดเงิน และการคำนวณต่างๆ เพื่อจัดทำรายงานที่คุณต้องการได้ นอกจากนั้น ความยืดหยุ่นนี้จะกระตุ้นความคิดสร้างสรรค์ โดยการทำให้คุณสามารถดูการดำเนินงานของคุณในแบบที่แตกต่างออกไปได้อย่างง่ายดายมากขึ้น บล็อคส่วนประกอบแต่ละรายการของข้อกำหนดของรายงานจะมีลักษณะคล้ายกับกระดาษคำนวณสามมิติ แต่มีประสิทธิภาพมากกว่านั้น ข้อกำหนดของรายงานจะระบุคำนิยามแถว คำนิยามคอลัมน์ และคำนิยามแผนภูมิรายงานเพิ่มเติมที่ควรนำมาใช้สำหรับรายงาน ยังมีข้อมูลเกี่ยวกับที่ที่ใช้เก็บรายงานที่สร้างขึ้นและวิธีการจัดรูปแบบรายงาน สำหรับการนำไปใช้ใหม่และการแบ่งปันที่ดีกว่าเดิมนั้น คุณสามารถสร้างกลุ่มบล็อคส่วนประกอบ ซึ่งเป็นชุดของข้อกำหนดของรายงาน คำนิยามแถว คำ นิยามคอลัมน์ คำนิยามแผนภูมิรายงาน และเซ็ตมิติที่มีอยู่ ที่เชื่อมโยงกับบริษัท

## <a name="building-blocks-of-a-report"></a> บล็อคส่วนประกอบของรายงาน
| บล็อคส่วนประกอบ            | คำอธิบาย                                                                                                                                                                                                                                                                              | สำหรับข้อมูลเพิ่มเติม                                                                                                 |
|---------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| คำนิยามแถว            | คำนิยามแถวกำหนดบรรทัดรายการคำอธิบาย (ตัวอย่างเช่น เงินเดือนหรือการขาย) ในรายงาน อีกทั้งยังแสดงรายการค่าเซ็กเมนต์หรือมิติที่ประกอบด้วยค่าสำหรับสินค้าแต่ละบรรทัดรายการ และรวมถึงการจัดรูปแบบแถวและการคำนวณ                                                    | [คำนิยามแถว](row-definitions-financial-reporting.md)                       |
| คำนิยามคอลัมน์         | คำนิยามคอลัมน์กำหนดรอบระยะเวลาที่จะใช้เมื่อข้อมูลถูกแยกออกจากมิติทางการเงิน ทั้งยังรวมถึงการจัดรูปแบบคอลัมน์และการคำนวณ                                                                                                                                 | [คำนิยามคอลัมน์](column-definitions-financial-reports.md)         |
| คำนิยามแผนภูมิรายงาน | คำนิยามของแผนภูมิรายงานคล้ายคลึงกับแผนภูมิองค์กร ซึ่งประกอบด้วยหน่วยรายงานแต่ละรายการที่แสดงถึงแต่ละกล่องในแผนภูมิ หน่วยต่างๆ อาจเป็นแบบแต่ละแผนกจากข้อมูลทางการเงินหรือหน่วยที่มีระดับสูงกว่าที่สรุปข้อมูลจากหน่วยรายงานอื่น | [คำนิยามแผนภูมิรายงาน](financial-reporting-tree-definitions.md) |
| ข้อกำหนดของรายงาน         | ข้อกำหนดของรายงานจะใช้คำนิยามแถว คำนิยามคอลัมน์ และคำนิยามแผนภูมิรายงานเพิ่มเติมเพื่อสร้างรายงาน อีกทั้งยังให้ตัวเลือกและการตั้งค่าเพิ่มเติมที่คุณสามารถใช้ได้ในการกำหนดรายงาน                                                                    | [ข้อกำหนดของรายงาน](design-financial-report-definitions.md)                  |

ถ้าคุณยังใหม่กับการออกแบบรายงาน คุณอาจต้องการใช้วิซาร์ดรายงานเพื่อสร้างข้อกำหนดของรายงานขึ้นมาอย่างรวดเร็ว ซึ่งคุณสามารถกำหนดค่าต่างๆ ได้ในภายหลัง ถ้าคุณมีประสบการณ์ในการออกแบบรายงาน และต้องการความยืดหยุ่นเพิ่มเติมสำหรับการออกแบบรายงาน คุณสามารถรวมบล็อคส่วนประกอบทั้งของใหม่หรือที่มีอยู่แล้ว เพื่อสร้าง เพื่อสร้างข้อกำหนดของรายงานใหม่ได้ คุณไม่จำเป็นต้องเข้าใจถึงตัวเลือกข้อกำหนดของรายงานที่มีอยู่ทั้งหมด ในการจะสร้างรายงานที่มีคุณภาพขึ้นมา เมื่อคุณเริ่มคุ้นเคยกับการออกแบบรายงานแล้ว คุณสามารถขยายข้อกำหนดของรายงานเพื่อใช้ประโยชน์จากคุณลักษณะขั้นสูง หลังจากที่คุณได้สร้างรายงานพื้นฐานแล้ว คุณสามารถกำหนดข้อกำหนดของรายงานและบล็อคส่วนประกอบใดก็ได้ในข้อกำหนดของรายงาน

## <a name="organize-the-building-blocks"></a>จัดระเบียบบล็อคส่วนประกอบ
ใช้โฟลเดอร์เพื่อจัดระเบียบบล็อคส่วนประกอบของคุณในโปรแกรมออกแบบรายงาน โฟลเดอร์ทั้งหมดจะมีความเฉพาะกับชนิดของบล็อคส่วนประกอบที่อยู่ข้างใน ตัวอย่างเช่น โฟลเดอร์ทั้งหมดที่ประกอบด้วยคำนิยามแถวจะอยู่ในบานหน้าต่าง **คำนิยามแถว** ของโปรแกรมออกแบบรายงาน

### <a name="create-a-folder"></a>สร้างโฟลเดอร์

1.  ในโปรแกรมออกแบบรายงาน เลือกชนิดของบล็อคส่วนประกอบเพื่อจัดระเบียบในบานหน้าต่างนำทาง ตัวอย่างเช่น จะเรียงลำดับคำนิยามแถว คลิก **คำนิยามแถว**
2.  ในบานหน้าต่างนำทาง เลือกโฟลเดอร์ที่มีอยู่เพื่อสร้างโฟลเดอร์ใหม่ด้านล่าง และจากนั้นทำตามขั้นตอนเหล่านี้อย่างใดอย่างหนึ่ง:
    -   คลิกขวาที่โฟลเดอร์หลัก และจากนั้นคลิก **โฟลเดอร์ใหม่**
    -   เลือกโฟลเดอร์หลัก คลิก **ไฟล์** แล้วจึงคลิก **โฟลเดอร์ใหม่**

3.  เมื่อโฟลเดอร์ใหม่ปรากฏขึ้น ป้อนชื่อของโฟลเดอร์ใหม่และจากนั้นกด ป้อน

## <a name="lock-a-building-block"></a> ล็อคบล็อคส่วนประกอบ
คุณสามารถสร้างรหัสผ่านเพื่อล็อค และป้องกันบล็อคส่วนประกอบได้ ในวิธีนี้ คุณสามารถเพิ่มระดับความปลอดภัยให้กับองค์ประกอบของรายงานโดยไม่ต้องมีการรักษาความปลอดภัยทั้งระบบ รหัสผ่านสามารถช่วยป้องกันข้อมูลบล็อคส่วนประกอบที่สำคัญที่ใช้ในการประมวลผลรายงานสิ้นเดือนของคุณ ผู้ใช้ในทุกบทบาทสามารถล็อคบล็อคส่วนประกอบได้ อย่างไรก็ตาม ผู้ใช้รายอื่นสามารถเข้าถึงส่วนประกอบที่ถูกล็อคได้แบบอ่านอย่างเดียวเสมอ ผู้ใช้สามารถเปิด เปลี่ยนแปลง และบันทึกส่วนประกอบที่ถูกล็อคด้วยชื่อใหม่ ผู้ใช้ที่มีบทบาทของผู้ดูแลระบบจะสามารถเข้าถึงและเปลี่ยนแปลงบล็อคส่วนประกอบที่ถูกล็อคได้เสมอ
1.  ในโปรแกรมออกแบบรายงาน เปิดส่วนประกอบรายงานเพื่อล็อค เช่น คำนิยามแถว คำนิยามคอลัมน์ ข้อกำหนดของรายงาน หรือคำนิยามแผนภูมิรายงาน
2.  บนเมนู **เครื่องมือ** ให้คลิก **ป้องกัน/ยกเลิกการป้องกัน** คุณสามารถคลิกไอคอน **ป้องกัน/ยกเลิกการป้องกัน** (ไอคอนล็อค) ในแถบเครื่องมือได้
3.  ในกล่องโต้ตอบ **ป้องกัน** ป้อนและยืนยันรหัสผ่าน และจากนั้น คลิก **ตกลง** ไอคอนล็อกในแถบเครื่องมือจะถูกเน้นไว้บล็อคส่วนประกอบที่เปิดไว้ถูกล็อค

หากจะปลดล็อคบล็อคส่วนประกอบที่ถูกล็อคไว้ ให้เปิดบล็อคส่วนประกอบและคลิก **ป้องกัน/ยกเลิกการป้องกัน** บนแถบเครื่องมือ หรือ บนเมนู **เครื่องมือ** ให้คลิก **ยกเลิกการป้องกัน**

## <a name="building-block-groups"></a>กลุ่มบล็อคส่วนประกอบ

บล็อคส่วนประกอบ คือ คำนิยามแถว คำนิยามคอลัมน์ คำนิยามแผนภูมิรายงาน และข้อกำหนดของรายงานที่คุณสร้างสำหรับรายงาน กลุ่มบล็อคส่วนประกอบ คือ ชุดของคำนิยามและเซ็ตมิติที่เกี่ยวข้องกับบริษัท กลุ่มบล็อคส่วนประกอบอาจเป็นแบบเฉพาะบริษัท หรือบริษัทหลายแห่งสามารถใช้ชุดของบล็อคส่วนประกอบที่เหมือนกันร่วมกันได้ ถ้าบริษัทบางแห่งของคุณมีผังบัญชีที่ต่างกันออกไป คุณสามารถใช้กลุ่มบล็อคส่วนประกอบที่แตกต่างกันสำหรับแต่ละบริษัท อีกทางหนึ่งคือ คุณอาจต้องการตั้งชื่อแต่ละบล็อคส่วนประกอบทั้งหมดเพื่อแสดงบริษัทที่สามารถใช้ได้
### <a name="create-a-building-block-group"></a>สร้างกลุ่มบล็อคส่วนประกอบ

1.  ในโปรแกรมออกแบบรายงาน บนเมนู **บริษัท** คลิก **กลุ่มบล็อคส่วนประกอบ**
2.  ในกล่องโต้ตอบ **กลุ่มบล็อคส่วนประกอบ** คลิก **สร้าง**
3.  ป้อนชื่อและคำอธิบายที่ไม่ซ้ำกันสำหรับกลุ่มบล็อคส่วนประกอบ แต่ละฟิลด์สามารถประกอบด้วยอักขระได้สูงสุด 256 อักขระ (จำนวนนี้รวมช่องว่างต่างๆ ด้วย)
4.  คลิก **ตกลง** ตกลงเพื่อสร้างกลุ่มบล็อคส่วนประกอบใหม่

### <a name="assign-a-building-block-group"></a>กำหนดกลุ่มบล็อคส่วนประกอบ

หลังจากที่คุณสร้างกลุ่มบล็อค คุณต้องกำหนดให้กับบริษัทอย่างน้อยหนึ่งแห่ง จากนั้นคุณสามารถสร้างรายงาน แถว คอลัมน์ และคำนิยามแผนภูมิการรายงาน และบันทึกลงไปในกลุ่มบล็อคส่วนประกอบ คุณต้องปิดบล็อคส่วนประกอบทั้งหมดก่อนที่คุณเริ่มต้นกระบวนงานต่อไปนี้
1.  ใน Report Designer บนเมนู **บริษัท** คลิก **บริษัท**
2.  ในกล่องโต้ตอบ **บริษัท** เลือกบริษัทที่จะกำหนดให้กับกลุ่มบล็อคส่วนประกอบ
3.  คลิก **แก้ไข**>
4.  ในกล่องโต้ตอบ **แก้ไขบริษัท** ในฟิลด์ **กลุ่มบล็อคส่วนประกอบ** เลือกกลุ่มบล็อคส่วนประกอบที่จะกำหนดให้กับบริษัท หรือคลิก **ใหม่** เพื่อสร้างกลุ่มบล็อคส่วนประกอบใหม่
5.  คลิก **ตกลง** เพื่อกำหนดกลุ่มบล็อคส่วนประกอบ
6.  คลิก **ปิด** เพื่อปิดการทำงานของกล่องโต้ตอบ **บริษัท** . ขณะนี้กลุ่มบล็อคส่วนประกอบที่คุณเลือกไว้กำลังถูกกำหนดให้กับบริษัท ตอนนี้คำนิยามแถว คำนิยามคอลัมน์ใหม่ทั้งหมด และอื่นๆ ที่สร้างขึ้น จะเป็นส่วนหนึ่งของกลุ่มบล็อคส่วนประกอบที่ถูกกำหนดให้กับบริษัทนี้ คุณยังสามารถนำเข้าไฟล์ .tdbx หรือรายงานจากระบบอื่นได้

### <a name="view-a-building-block-group"></a> ดูกลุ่มบล็อคส่วนประกอบ

หลังจากที่กลุ่มบล็อคส่วนประกอบถูกสร้างขึ้น และถูกนำมาใช้ คุณสามารถดูบล็อคส่วนประกอบทั้งหมดที่กำหนดให้ได้ คุณสามารถส่งออกหรือนำเข้ากลุ่มบล็อคส่วนประกอบ และทำการบำรุงรักษากลุ่มบล็อคส่วนประกอบเพิ่มเติมได้
1.  ในโปรแกรมออกแบบรายงาน บนเมนู **บริษัท** คลิก **กลุ่มบล็อคส่วนประกอบ**
2.  ในกล่องโต้ตอบ **กลุ่มบล็อคส่วนประกอบ** เลือกบล็อคส่วนประกอบเพื่อดู
3.  คลิก **ดู** เพื่อเปิดกล่องโต้ตอบ **ดูกลุ่มบล็อคส่วนประกอบ** ซึ่งคุณสามารถดูเนื้อหาของกลุ่มบล็อคส่วนประกอบได้
4.  คลิก **ปิด** เพื่อปิดกล่องโต้ตอบ

### <a name="save-a-building-block-group-under-a-new-name"></a>บันทึกกลุ่มบล็อคส่วนประกอบโดยใช้ชื่อใหม่

คุณสามารถบันทึกกลุ่มบล็อคส่วนประกอบที่มีอยู่โดยใช้ชื่อใหม่ได้ จากนั้น คุณสามารถแก้ไขกลุ่มบล็อคส่วนประกอบใหม่โดยไม่เปลี่ยนแปลงกลุ่มบล็อคส่วนประกอบต้นฉบับ
1.  ในโปรแกรมออกแบบรายงาน บนเมนู **บริษัท** คลิก **กลุ่มบล็อคส่วนประกอบ**
2.  ในกล่องโต้ตอบ **กลุ่มบล็อคส่วนประกอบ** เลือกกลุ่มบล็อคส่วนประกอบเพื่อบันทึกโดยใช้ชื่อใหม่
3.  คลิก **บันทึกเป็น**.
4.  ป้อนชื่อและคำอธิบายใหม่สำหรับกลุ่มบล็อคส่วนประกอบ
5.  คลิก **ตกลง** ระบบจะนำเข้าข้อมูลการชำระเงิน กลุ่มบล็อคส่วนประกอบใหม่จะปรากฏในกล่องโต้ตอบ **กลุ่มบล็อคส่วนประกอบ**.

### <a name="export-a-building-block-group"></a> ส่งออกกลุ่มบล็อคส่วนประกอบ

คุณสามารถส่งออกกลุ่มบล็อคส่วนประกอบหรือบล็อคส่วนประกอบรายงานที่ระบุในกลุ่มบล็อคส่วนประกอบได้ คุณสามารถใช้กลุ่มบล็อคส่วนประกอบที่ส่งออกเป็นส่วนสำรองได้ คุณยังสามารถคัดลอกข้อมูลที่ส่งออกระหว่างกลุ่มบล็อคส่วนประกอบหรือการติดตั้งของ Dynamics 365 for Operations ได้อีกด้วย ผู้ออกแบบรายงานจะรวมถึงลักษณะตัวอักษรที่อ้างอิง และเซ็ตมิติพร้อมกับกลุ่มบล็อคส่วนประกอบ
1.  ในโปรแกรมออกแบบรายงาน บนเมนู **บริษัท** คลิก **กลุ่มบล็อคส่วนประกอบ**
2.  ในกล่องโต้ตอบ **กลุ่มบล็อคส่วนประกอบ** เลือกกลุ่มบล็อคส่วนประกอบเพื่อส่งออก แล้วจึงคลิก **ส่งออก**
3.  ในกล่องโต้ตอบ **ส่งออก** เลือกข้อกำหนดของรายงานที่จะส่งออก:
    -   เมื่อต้องการส่งออกข้อกำหนดของรายงานและบล็อคส่วนประกอบที่เกี่ยวข้องทั้งหมดของคุณ คลิก **เลือกทั้งหมด**
    -   เมื่อต้องการส่งออกรายงาน แถว คอลัมน์ แผนภูมิ หรือเซ็ตมิติที่เฉพาะ คลิกแท็บที่เหมาะสม และจากนั้นเลือกรายการที่จะส่งออก กดค้างปุ่ม Ctrl เพื่อเลือกหลายรายการในแท็บ **หมายเหตุ:** เมื่อคุณเลือกรายงานที่จะส่งออก แถว คอลัมน์ แผนภูมิ และเซ็ตมิติที่เกี่ยวข้องจะถูกเลือกไว้

4.  เมื่อคุณเสร็จสิ้นการเลือกรายการที่จะส่งออก คลิก **ส่งออก**
5.  ในกล่องโต้ตอบ **บันทึกเป็น** เลือกสถานที่ที่จะส่งออกกลุ่มบล็อคส่วนประกอบไป
6.  ในฟิลด์ **ชื่อไฟล์** ให้ป้อนชื่อสำหรับไฟล์ ผู้ออกแบบรายงานจะเพิ่มนามสกุลของไฟล์เป็น .tdbx โดยอัตโนมัติ
7.  คลิก **บันทึก** กลุ่มบล็อคส่วนประกอบจะถูกบันทึกลงในสถานที่ที่คุณระบุไว้

### <a name="import-a-building-block-group"></a> นำเข้ากลุ่มบล็อคส่วนประกอบ

คุณสามารถนำเข้ากลุ่มบล็อคส่วนประกอบไปยังกลุ่มบล็อคส่วนประกอบที่มีอยู่ หรือคุณสามารถสร้างกลุ่มบล็อคส่วนประกอบสำหรับข้อมูล กลุ่มบล็อคส่วนประกอบที่ถูกนำเข้าทั้งหมดจะยังรักษาลักษณะตัวอักษรเดิมไว้และการอ้างอิงบริษัทและรวมไปถึงเซ็ตมิติที่เกี่ยวข้องด้วย
1.  ในโปรแกรมออกแบบรายงาน บนเมนู **บริษัท** คลิก **กลุ่มบล็อคส่วนประกอบ**
2.  ในกล่องโต้ตอบ **กลุ่มบล็อคส่วนประกอบ** เลือกกลุ่มบล็อคส่วนประกอบเพื่อนำเข้า แล้วจึงคลิก **นำเข้า**
3.  ในกล่องโต้ตอบ **เปิด** เลือกกลุ่มบล็อคส่วนประกอบเพื่อนำเข้า แล้วจึงคลิก **เปิด**
4.  ในกล่องโต้ตอบ **นำเข้า** เลือกข้อกำหนดของรายงานที่จะนำเข้า:
    -   เมื่อต้องการนำเข้าข้อกำหนดของรายงานและบล็อคส่วนประกอบสนับสนุนทั้งหมด คลิก **เลือกทั้งหมด**
    -   การนำเข้ารายงาน แถว คอลัมน์ แผนภูมิ หรือ เซ็ตมิติที่เฉพาะ ให้เลือกรายงาน แถว คอลัมน์ แผนภูมิ หรือเซ็ตมิติที่จะนำเข้า

5.  เมื่อคุณเสร็จสิ้นการเลือกรายการที่จะนำเข้าแล้ว ให้คลิก **นำเข้า**

### <a name="undo-a-checkout-of-a-building-block"></a>ยกเลิกการเช็คเอาท์บล็อคส่วนประกอบ

เมื่อคุณเปิดบล็อคส่วนประกอบ ผู้ใช้รายอื่นสามารถเข้าถึงบล็อคส่วนประกอบได้แค่แบบอ่านอย่างเดียวเท่านั้น ในบางครั้งผู้ใช้ลืมที่จะปิดบล็อคส่วนประกอบ หรือจะปิดการทำงานของระบบโดยไม่ปิดบล็อคส่วนประกอบ ดังนั้น บล็อคส่วนประกอบจะยังคงถูกเช็คเอาท์ และผู้ใช้คนอื่นไม่สามารถเปิดได้ ในสถานการณ์เหล่านี้ ผู้ดูแลการรายงานทางการเงินสามารถใช้กล่องโต้ตอบ **เช็คเอาท์รายการ** เพื่อตรวจสอบบล็อคส่วนประกอบที่ผู้ใช้ปล่อยทิ้งไว้ให้เป็นแบบเช็คเอาท์ **หมายเหตุ**: คุณต้องมีบทบาทของผู้ดูแลระบบเพื่อเช็คอินบล็อคส่วนประกอบโดยใช้กล่องโต้ตอบ **เช็คเอาท์สินค้า**.
1.  ในโปรแกรมออกแบบรายงาน บนเมนู **เครื่องมือ** คลิก **เช็คเอาท์สินค้า**
2.  ในกล่องโต้ตอบ **เช็คเอาท์สินค้า** เลือก **แสดงรายการจากผู้ใช้ทั้งหมด** รายการจะถูกปรับปรุงเพื่อแสดงบล็อคส่วนประกอบทั้งหมดที่ถูกเช็คเอาท์และผู้ใช้ที่เช็คเอาท์บล็อคเหล่านั้น
3.  เลือกบล็อคส่วนประกอบ แล้วจึงคลิก **ยกเลิกการเช็คเอาท์**
4.  คลิก **ใช่** เพื่อเช็คอินบล็อคส่วนประกอบ
