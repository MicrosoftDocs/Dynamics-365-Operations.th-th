# <a name="help-overview"></a>ภาพรวมของวิธีใช้

บทความนี้แสดงภาพรวมของคอมโพเนนต์ของระบบวิธีใช้ Microsoft Dynamics 365 for Operations นอกจากนี้ยังอธิบายวิธีระบุเอกสารแบบกำหนดเองและการฝึกอบรมองค์กรของคุณ 

Dynamics 365 for Operations ประกอบด้วยระบบวิธีใช้ที่ขึ้นอยู่กับสองส่วนประกอบหลัก:

-   ไซต์เอกสาร
-   คู่มืองาน

คุณสามารถเข้าถึงบทความและการแนะนำงานจากบานหน้าต่าง'วิธีใช้'ใน 365 Dynamics สำหรับการดำเนินงานดังที่แสดงในภาพจากหน้าจอต่อไปนี้ [![บานหน้าต่างวิธีใช้](./media/help-pane-ops-task-guides-1024x741.png)](./media/help-pane-ops-task-guides.png)บทความนี้อธิบายถึงระบบวิธีใช้ และอธิบายวิธีที่คุณสามารถสร้างเอกสารแบบกำหนดเองและการฝึกอบรมสำหรับองค์กรของคุณ

## <a name="help-on-docsmicrosoftcom"></a>ความช่วยเหลือเกี่ยวกับ docs.microsoft.com
ไซต์ docs.microsoft.com ([docs.microsoft.com/dynamics365/operations](https://docs.microsoft.com/en-us/dynamics365/#pivot=solutions&panel=solutions_operations)เป็นแหล่งข้อมูลหลักของเอกสารประกอบผลิตภัณฑ์สำหรับ 365 Dynamics สำหรับการดำเนินงาน ไซต์นั้นนำเสนอคุณลักษณะต่อไปนี้:

-   **การเข้าถึงเนื้อหาล่าสุด**– ไซต์ทำให้เรารวดเร็วขึ้น และความยืดหยุ่นในการสร้าง จัดส่ง และการปรับปรุงเอกสารประกอบของผลิตภัณฑ์ ดังนั้น จึงจะช่วยรับประกันว่า คุณมีสิทธิ์เข้าถึงข้อมูลทางเทคนิคล่าสุด
-    **เนื้อหาที่เขียน โดยผู้เชี่ยวชาญ**– ไซต์นำเสนอชุดเอกสารประกอบของผลิตภัณฑ์ที่สามารถปรับปรุง โดยสมาชิกของชุมชนทั้งภายใน และภาย นอกของ Microsoft ดีขึ้น
-   ** เข้าถึงชนิดต่าง ๆ ของเนื้อหา ** – ไซต์ช่วยให้คุณได้อย่างรวดเร็วเข้าถึงเนื้อหาชนิดต่าง ๆ เกี่ยวกับ 365 Dynamics สำหรับการดำเนินงาน เช่น Microsoft Office รวมงานนำเสนอ แนะนำงาน วิดีโอ และบทความ wiki
-    **เนื้อหาที่สนับสนุนกระบวนการทางธุรกิจของคุณ**– ไซต์รวมธุรกิจกระบวนการ – เน้นเนื้อหาที่ใช้ประโยชน์จากของบบจำลองกระบวนการทางธุรกิจ (BPM) ใน Microsoft Dynamics วัฏจักรบริการ (LCS)

เราได้ย้ายเนื้อหาทั้งหมดจาก wiki ช่วยเราก่อนหน้าไปเอกสาร เราตื่นเต้นมากเกี่ยวกับไซต์ใหม่ของเราและหวังว่าเราจะมีมากเกินไป

### <a name="when-can-we-use-it"></a>เมื่อไรที่เราจะสามารถใช้ได้

คุณสามารถอ่านเนื้อหาในเอกสารในขณะนี้--เป็นสาธารณะ และสามารถค้นหาได้ครบทั้งหมดโดยไม่ต้องมีการเข้าสู่ระบบ คุณสามารถใช้โปรแกรมค้นหาใดก็ตามที่คุณชอบเพื่อค้นหาเนื้อหา คุณสามารถข้อคิดเห็นเกี่ยวกับบทความบนไซต์ถ้าคุณเลือก โดยบันทึกด้วยบัญชี GitHub


## <a name="task-guides"></a>คู่มืองาน
คำแนะนำของงานจะควบคุม แนะนำ แบบโต้ตอบประสบการณ์ที่นำคุณผ่านขั้นตอนของงาน หรือกระบวนการทางธุรกิจ คุณสามารถเปิดรายการแนะนำ (เล่น) งานจากบานหน้าต่าง'วิธีใช้' เมื่อคุณต้องคลิกที่คำแนะนำงาน บานหน้าต่างวิธีใช้จะแสดงคำแนะนำทีละขั้นตอนสำหรับงานนั้น ข้อแนะนำของงานที่เป็นภาษาท้องถิ่นจะพร้อมใช้งานในขณะนี้ [![มุมมองการอ่านคำแนะนำของงาน](./media/task-guide-ops-1024x742.png)](./media/task-guide-ops.png)เพื่อเริ่มต้นกับประสบการณ์ใช้งานแบบโต้ตอบ แนะนำ คลิ**แนะนำงานเริ่มต้น**ที่ด้านล่างของบานหน้าต่าง'วิธีใช้' ตัวชี้สีดำเปิดขึ้น และบ่งชี้ถึงการดำเนินการที่คุณจำเป็นต้องทำ ทำตามคำแนะนำที่ปรากฏใน UI และป้อนข้อมูลตามที่กำหนด [![คำสั่งในขั้นตอนที่แนะนำของงาน](./media/task-guide-step-1-ops.png)](./media/task-guide-step-1-ops.png)**สำคัญ:**ข้อมูลที่คุณป้อนเมื่อคุณเล่นคำแนะนำงานเป็นจริง ถ้าคุณอยู่ในสภาพแวดล้อมการผลิต ข้อมูลจะถูกป้อนในบริษัทที่คุณกำลังใช้อยู่

### <a name="it-all-begins-with-task-recorder"></a>ทั้งหมดเริ่มต้นด้วยตัวบันทึกงาน

คำแนะนำของงานจะสร้างขึ้นโดยใช้ตัวบันทึกงาน เมื่อคุณใช้ตัวบันทึกงาน การดำเนินการทั้งหมดของคุณใน Dynamics 365 for Operations UI (เช่น การคลิกที่เมนู การเปลี่ยนแปลงการตั้งค่า และการป้อนข้อมูล) จะถูกบันทึกไว้ ขั้นตอนที่คุณบันทึกจะถูกเรียกรวมกันว่า บันทึกงาน ดังที่เราได้อธิบายไว้ในหัวข้อก่อนหน้า บันทึกงานที่สามารถแสดงในบานหน้าต่าง วิธีใช้ และเล่นเป็นคู่มืองาน อย่างไรก็ตาม มีวิธีอื่น ๆ ที่คุณสามารถใช้การบันทึกงานได้:

-   **บันทึกการบันทึกงานไปยัง BPM** – คุณสามารถบันทึกการบันทึกงานของลำดับชั้นในไลบรารี BPM ใน LCS ได้ เมื่อคุณบันทึกการบันทึกงานไปยัง BPM ไดอะแกรมแผนผังลำดับงานจะถูกสร้างและแสดงร่วมกับขั้นตอนของการบันทึก **หมายเหตุ:** เพื่อแสดงการบันทึกงานในบานหน้าต่าง วิธีใช้ ของ Dynamics 365 for Operations และเล่นเป็นคู่มืองาน คุณต้องบันทึกการบันทึกไปยังไลบรารี BPM
-   **บันทึกเป็นเอกสาร Word** – โดยการบันทึกการบันทึกงานเป็นเอกสาร Microsoft Word คุณสามารถจัดทำคำแนะนำในการฝึกอบรมที่สามารถพิมพ์ได้สำหรับองค์กรของคุณได้อย่างง่ายดาย

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับตัวบันทึกภารกิจ ดู[ดงานใน Dynamics 365 สำหรับการดำเนินงาน](../user-interface/task-recorder.md)

### <a name="creating-customized-task-recordings"></a>สร้างบันทึกภารกิจที่กำหนดเอง

คุณสามารถสร้างบันทึกงานของคุณเอง หรือคุณสามารถดาวน์โหลด และบันทึกงานที่ Microsoft จัดเตรียมให้ ดังนั้น คุณสามารถสร้างวิธีใช้ที่กำหนดเองสำหรับองค์กรของคุณ ซึ้งจะแสดงใช้งาน Dynamics 365 for Operations แบบเฉพาะ แสดงงานบันทึกใน 365 Dynamics สำหรับบานหน้าต่างวิธีใช้การดำเนินงาน และเล่นเป็นแนวงาน คุณจะต้องบันทึกการบันทึกลงในไลบรารี BPM ใน LCS ถ้าคุณกำลังหุ้นส่วน และคุณเลื่อนระดับไลบรารีในไลบรารีของบริษัท และรวมอยู่ในโซลูชัน จะพร้อมใช้งานเพื่อลูกค้าของคุณ สำหรับคำแนะนำที่สมบูรณ์[ใช้งานการบันทึกเพื่อสร้างเอกสารหรือการฝึกอบรม](../user-interface/task-recorder.md)

## <a name="in-product-help"></a>วิธีใช้ภายในผลิตภัณฑ์
เมื่อต้องการเข้าถึงเนื้อหาวิธีใช้ภายใน 365 Dynamics สำหรับการดำเนินงาน อย่างใดอย่างหนึ่งคลิ**ช่วย**(**?**) ไอคอนแล้วเลือก'วิธีใช้' หรือกด Ctrl + Shift + หรือไม่ ในทั้งสองกรณี บานหน้าต่าง วิธีใช้ จะเปิดขึ้น คุณสามารถเข้าถึงบทความหรือแนะนำงานจากบานหน้าต่างวิธีใช้ [![](./media/help-pane-wiki-1024x684.png)](./media/help-pane-wiki.png)

### <a name="accessing-articles-from-the-help-pane"></a>เข้าถึงบทความจากบานหน้าต่างวิธีใช้

จากบานหน้าต่างวิธีใช้ คุณสามารถเข้าถึงบทความที่นำไปใช้กับ 365 Dynamics สำหรับไคลเอ็นต์การดำเนินการ เมื่อคุณก่อนเปิดบานหน้าต่าง'วิธีใช้' แล้วคลิกที่ตัว**Wiki**แท็บ คุณจะเห็นบทความที่นำไปใช้กับเพจที่คุณกำลังอยู่ในขณะนี้ใน Dynamics 365 สำหรับการดำเนินงาน ถ้าพบบทความที่ไม่มี คุณสามารถป้อนคำสำคัญเพื่อคัดสรรการค้นหาของคุณ เมื่อคุณคลิกในบานหน้าต่างวิธีใช้บทความ แท็บใหม่เปิดขึ้นในเบราว์เซอร์ของคุณ และแสดงบทความ 

### <a name="accessing-task-guides-from-the-help-pane"></a>การเข้าถึงงานการแนะนำจากบานหน้าต่างวิธีใช้

ก่อนที่คุณสามารถเข้าถึงงานการแนะนำจากบานหน้าต่างวิธีใช้ มีผู้ดูแลระบบเพื่อไป**พารามิเตอร์ระบบ**หน้าใน 365 Dynamics สำหรับการดำเนินงาน และการตั้งค่าคอนฟิกการตั้งค่าบางอย่าง **หมายเหตุ:**

-   เพื่อกำหนดค่าวิธีใช้ คุณต้องเข้าสู่ระบบด้วยบัญชีผู้ใช้ในผู้เช่าเดียวกันกับผู้เช่าที่ใช้ใน Dynamics 365 for Operations
-   จะไม่สามารถเชื่อมต่อไปยังไลบรารี LCS จากอินสแตนซ์ของ Dynamics 365 for Operations ที่ติดตั้งในฮาร์ดไดรฟ์เสมือนเฉพาะที่ (VHD)

[![แบบฟอร์มพารามิเตอร์ระบบวิธีใช้การตั้งค่า](./media/system-parameters_ops-1024x437.png)](./media/system-parameters_ops.png)ในการ**พารามิเตอร์ระบบ**หน้า ให้ทำตามขั้นตอนเหล่านี้:

1.  **สิ่งสำคัญ: **ในครั้งแรกที่คุณเปิดแท็บ วิธีใช้ คุณต้องเชื่อมต่อกับ Lifecycle Services ตรวจสอบให้แน่ใจว่าคลิกการเชื่อมโยงในกึ่งกลางของฟอร์ม รอสำหรับการเชื่อมต่อ ปิดกล่องโต้ตอบ และจากนั้น คลิกตกลงเพื่อเข้าถึงแบบฟอร์มพารามิเตอร์ [![เชื่อมต่อกับ LCS](./media/connect-to-lcs-crop-1024x365.png)](./media/connect-to-lcs-crop.png)
2.  เลือกโครงการ Lifecycle Services เพื่อเชื่อมต่อ
3.  เลือกไลบรารี BPM (ภายในโครงการที่เลือก) เพื่อดึงข้อมูลการบันทึกงาน
4.  ตั้งค่าลำดับการแสดงผลของไลบรารี BPM ซึ่งจะกำหนดลำดับว่าบันทึกงานใดจะปรากฏขึ้นในบานหน้าต่าง วิธีใช้

หลังจากผู้ดูแลระบบเสร็จสิ้นขั้นตอนเหล่านี้ คุณสามารถเปิดบานหน้าต่าง วิธีใช้ และคลิกแท็บ **คู่มืองาน** ขณะนี้คุณจะเห็นการแนะนำงานที่นำไปใช้กับเพจที่คุณกำลังอยู่ในขณะนี้ใน Dynamics 365 สำหรับการดำเนินงาน อยู่ ถ้าไม่มีการแนะนำงานพบ คุณสามารถป้อนคำสำคัญเพื่อคัดสรรการค้นหาของคุณ หลังจากที่คุณคลิกการแนะนำงานในบานหน้าต่างวิธีใช้ บานหน้าต่างวิธีใช้แสดงคำแนะนำทีละขั้นตอน และคุณสามารถเล่นรายการแนะนำงาน [![มุมมองการอ่านคำแนะนำของงาน](./media/task-guide-ops-1024x742.png)](./media/task-guide-ops.png)

### <a name="where-are-the-translated-task-guides"></a>การแนะนำงานแปลอยู่ที่ไหน

งานที่แนะนำจะถูกนำออกใช้ในไลบรารี "ภาษาทั้งหมด" ในชื่อเรื่องที่แปลแล้ว ใน 365 Dynamics สำหรับการดำเนินงาน เพื่อดูคำแนะนำงานเป็นภาษาท้องถิ่นที่ช่วย ตรวจสอบให้แน่ใจว่า คุณได้เชื่อมต่อไปยังไลบรารี apppropriate ควบคุมภาษาที่คำแนะนำของงานปรากฏในสำหรับผู้ใช้แต่ละราย โดยการตั้งค่าภาษาภายใต้**ตัวเลือก**&gt;**กำหนดลักษณะ** 
-   ถ้าคำแนะนำของงานได้รับการแปล เมื่อคุณเปิดรายการแนะนำงานที่ข้อความทั้งหมดของงานรายการแนะนำจะปรากฏในภาษาที่คุณเลือก
-   ถ้าคำแนะนำของงานได้ไม่ได้รับการแปล เมื่อคุณเปิดฐานข้อมูล เพียงบางส่วนของข้อความ (ข้อความของตัวควบคุม) จะปรากฏในภาษาของคุณเลือก
