# <a name="new-or-updated-task-guides-august-2016"></a>ข้อแนะนำงานใหม่ หรือที่ปรับปรุง (2016 สิงหาคม)

หัวข้อนี้แสดงรายการการใหม่ หรือปรับปรุงงานแนะนำที่ถูกนำออกใช้ไปยัง Microsoft (2016 สิงหาคม - ทุกภาษา) APQC รวมไลบรารีสำหรับ Microsoft Dynamics AX ไม่มีการนำออกใช้การเริ่มต้นใช้งานไลบรารีใหม่

<a name="task-guides-in-the-august-2016---all-languages-apqc-unified-library-for-microsoft-dynamics-ax"></a>[]()งานแนะนำในการ (2016 สิงหาคม - ทุกภาษา) ไลบรารีโดยรวม APQC สำหรับ Microsoft Dynamics AX
---------------------------------------------------------------------------------------------------

(2016 สิงหาคม - ทุกภาษา) รวม APQC ไลบรารีประกอบด้วยการแนะนำงานที่ตั้งใจจะใช้สำหรับวิธีใช้กับ update แพลตฟอร์ม Microsoft Dynamics AX 2 (2016 สิงหาคม), Dynamics AX อาจปรับปรุงโปรแกรมประยุกต์ 2016 และนำออกใช้ใน Microsoft Dynamics AX 2016 กุมภาพันธ์ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับรุ่นเหล่านี้ ดู[ใหม่ หรือที่เปลี่ยนแปลงคืออะไร](whats-new-changed.md) เวอร์ชันที่เป็นภาษาท้องถิ่นของการแนะนำงานจะพร้อมใช้งานในไลบรารีนี้ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการเชื่อมต่อระบบวิธีใช้ไลบรารีรายการแนะนำงาน ดู[เริ่มต้นใช้งานวิธีใช้](help-overview.md)
