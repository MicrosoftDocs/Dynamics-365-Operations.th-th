# <a name="new-or-updated-task-guides-may-2016"></a>ข้อแนะนำงานใหม่ หรือที่ปรับปรุง (2016 พฤษภาคม)

บทความนี้แสดงรายการงานใหม่ หรือปรับปรุงคำแนะนำที่ถูกนำออกใช้การ Microsoft (อาจ 2016 - ทุกภาษา) APQC ไลบรารีโดยรวมสำหรับ Microsoft Dynamics AX ไม่มีการนำออกใช้การเริ่มต้นใช้งานไลบรารีใหม่ 

<a name="task-guides-in-the-may-2016---all-languages-apqc-unified-library-for-microsoft-dynamics-ax"></a>[]()งานแนะนำในการ (2016 พฤษภาคม - ทุกภาษา) ไลบรารีโดยรวม APQC สำหรับ Microsoft Dynamics AX
------------------------------------------------------------------------------------------------

(2016 พฤษภาคม) ไลบรารีรวม APQC ประกอบด้วยข้อแนะนำที่มีจุดมุ่งหมายที่จะใช้สำหรับวิธีใช้ของ Microsoft Dynamics AX อาจปรับปรุง 2016 งาน และนำออกใช้ใน Microsoft Dynamics AX 2016 กุมภาพันธ์ เวอร์ชันที่เป็นภาษาท้องถิ่นของการแนะนำงานที่ได้ถูกนำออกใช้ใน (2016 กุมภาพันธ์) รวม APQC ไลบรารีจะมีอยู่ในไลบรารีนี้ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการเชื่อมต่อระบบวิธีใช้ไลบรารีรายการแนะนำงาน ดู[เริ่มต้นใช้งานวิธีใช้](help-overview.md) **หมายเหตุ:**เครื่องหมายดอกจัน (**\***) อยู่ถัดจากชื่อเรื่องของบบจำลองกระบวนการทางธุรกิจ (BPM) บรรทัดบ่งชี้ว่า รายการแนะนำงานสำหรับกระบวนการทางธุรกิจที่ถูกสร้างขึ้นสำหรับ Dynamics AX 2016 ปรับปรุง และไม่ได้รับการสนับสนุนใน Microsoft Dynamics AX เวอร์ชันก่อนหน้านี้
