# <a name="connect-the-help-system"></a>เชื่อมต่อระบบวิธีใช้

หัวข้อนี้อธิบายส่วนประกอบของระบบ'วิธีใช้'สำหรับ 365 Dynamics Microsoft สำหรับการดำเนินงาน โดยจะให้ภาพรวมของวิธีการเชื่อมต่อส่วนประกอบเหล่านี้ และช่วยให้ข้อมูลสรุปเกี่ยวกับวิธีการสร้างแบบกำหนดเอง 

<a name="help-architecture"></a>สถาปัตยกรรมของวิธีใช้
-----------------

ภาพประกอบต่อไปนี้แสดงส่วนต่าง ๆ ของ 365 Dynamics สำหรับระบบวิธีใช้ของการดำเนินงาน ระบบวิธีใช้ในผลิตภัณฑ์ดึงบทความจาก 365 Dynamics สำหรับไซต์การดำเนินงานบน https://docs.microsoft.com ตลอดจนแนะนำงานที่เก็บไว้ในบบจำลองกระบวนการธุรกิจในวัฏจักรบริการ (LCS) 
**หมายเหตุ:**ลักษณะการทำงานที่แสดงไว้ในไดอะแกรมที่มีเครื่องหมายดอกจัน (\*) มีการวางแผน แต่ยังไม่พร้อมใช้งาน [![Help architecture](./media/help-architecture.png)](./media/help-architecture.png)

## <a name="connecting-the-help-system"></a>การเชื่อมต่อระบบวิธีใช้
ใช้การ**พารามิเตอร์ระบบ**หน้า ผู้ดูแลระบบเชื่อมต่อชิ้นส่วนของระบบวิธีใช้สำหรับการนำไปใช้ [![แบบฟอร์มพารามิเตอร์ระบบวิธีใช้การตั้งค่า](./media/system-parameters_ops-1024x437.png)](./media/system-parameters_ops.png)ในการ**พารามิเตอร์ระบบ**หน้า ให้ทำตามขั้นตอนเหล่านี้:

1.  **สิ่งสำคัญ:**เป็นครั้งแรกที่คุณเปิดใน**ช่วย**แท็บ คุณต้องเชื่อมต่อไปยังบริการวัฏจักร แน่ใจว่าได้คลิกการเชื่อมโยงในกึ่งกลางของฟอร์ม รอสำหรับการเชื่อมต่อ ปิดกล่องโต้ตอบ และจากนั้น คลิ**ตกลง**เพื่อเข้าไป**พารามิเตอร์ระบบ**หน้า [![เชื่อมต่อกับ LCS](./media/connect-to-lcs-crop-1024x365.png "เชื่อมต่อกับ LCS")](./media/connect-to-lcs-crop.png)
2.  เลือกโครงการ Lifecycle Services เพื่อเชื่อมต่อ
3.  เลือกไลบรารี BPM (ภายในโครงการที่เลือก) เพื่อดึงข้อมูลการบันทึกงาน
4.  ตั้งค่าลำดับการแสดงผลของไลบรารี BPM ซึ่งจะกำหนดลำดับว่าบันทึกงานใดจะปรากฏขึ้นในบานหน้าต่าง **วิธีใช้**

หลังจากผู้ดูแลระบบเสร็จสิ้นขั้นตอนเหล่านี้ คุณสามารถเปิดบานหน้าต่าง **วิธีใช้** และคลิกแท็บ **คู่มืองาน** ได้ ขณะนี้คุณจะเห็นคู่มืองานที่นำไปใช้กับเพจที่คุณกำลังอยู่ใน Dynamics 365 for Operations ถ้าไม่พบคู่มืองาน คุณสามารถป้อนคำสำคัญเพื่อจำกัดการค้นหา

### <a name="showing-translated-task-guides"></a>แสดงคำแปลคู่มืองาน

แนะนำการแปลงานแรกได้ถูกจัดส่งในเดือน 2016 พฤษภาคมไลบรารีรวม APQC และไลบรารีเริ่มต้น ใน Dynamics 365 for Operations เมื่อต้องการดูวิธีใช้ของคู่มืองานที่เป็นภาษาท้องถิ่น ให้ตรวจสอบให้แน่ใจว่าคุณได้เชื่อมต่อไปยังไลบรารีเดือนพฤษภาคมแล้ว ควบคุมภาษาที่คำแนะนำของงานปรากฏในสำหรับผู้ใช้แต่ละราย โดยการตั้งค่าภาษาภายใต้**ตัวเลือก**&gt;**กำหนดลักษณะ** **หมายเหตุ:** แม้ว่าจะมีคู่มืองานจำนวนมากที่ได้รับการแปลแล้ว ในขณะนี้ไคลเอนต์ Dynamics 365 for Operations ไม่แสดงชื่อคู่มืองานที่แปลแล้ว นอกจากนี้ เฉพาะการงานแนะนำที่ถูกนำออกใช้ในเดือน 2016 กุมภาพันธ์จะพร้อมใช้งานในการแปลในไลบรารีอาจ เราจะปล่อยไลบรารีที่ปรับปรุงแล้วพร้อมกับคำแปลเพิ่มเติม

-   ถ้าคู่มืองานได้รับการแปลแล้ว เมื่อคุณเปิดคู่มืองานนั้น ข้อความทั้งหมดของคู่มืองานจะปรากฏในภาษาที่คุณเลือก
-   ถ้าคู่มืองานยังไม่ได้รับการแปล เมื่อคุณเปิดคู่มืองานนั้น จะปรากฏข้อความเพียงบางส่วน (ข้อความของการควบคุม) ในภาษาที่คุณเลือก

## <a name="creating-custom-help"></a>การสร้างวิธีใช้ที่กำหนดเอง
คุณสามารถสร้างวิธีใช้ที่กำหนดเองสำหรับการใช้งาน Dynamics 365 for Operations ของคุณ โดยการสร้างบันทึกงานที่สะท้อนถึงการใช้งาน และบันทึกวิธีใช้เหล่านั้นลงในไลบรารีกระบวนการทางธุรกิจมี LCS สำหรับคู่ค้า ถ้าคุณเลื่อนลำดับเป็นไลบรารีของบริษัท และรวมอยู่ในโซลูชัน ไลบรารีนั้นจะพร้อมใช้งานสำหรับลูกค้าของคุณ คุณสามารถทำสำเนาสากล APQC และจากนั้นเปิดสำเนาของคุณ เปิดบันทึกงานจากไลบรารีนั้น ปรับเปลี่ยน และบันทึกบันทึกงานที่ึคุณทำการเปลี่ยนแปลง สำหรับข้อมูลเพิ่มเติม ดู[วิธีการสร้างหรือการบันทึกงานเพื่อใช้เป็นส่วนเอกสารการฝึกอบรม](../user-interface/task-recorder.md)
