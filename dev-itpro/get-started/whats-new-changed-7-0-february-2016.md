# <a name="whats-new-or-changed-in-dynamics-ax-70-february-2016"></a>มีอะไรใหม่หรือการเปลี่ยนแปลงอะไรใน Dynamics AX 7.0 (กุมภาพันธ์ 2016)

[!include[banner](../includes/banner.md)]


บทความนี้อธิบายถึงคุณลักษณะใหม่หรือที่มีการเปลี่ยนแปลงในการนำออกใช้ Microsoft Dynamics AX 7.0 รุ่นนี้ประกอบด้วยคุณลักษณะทั้งแพลตฟอร์มและแอพพลิเคชัน และถูกนำออกใช้ในเดือนกุมภาพันธ์ 2016
