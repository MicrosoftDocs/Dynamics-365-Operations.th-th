# <a name="whats-new-or-changed"></a>มีอะไรใหม่หรือมีการเปลี่ยนแปลง

[!include[banner](../includes/banner.md)]

หน้านี้ชี้ไปยังหัวข้อที่จะช่วยให้คุณเรียนรู้เกี่ยวกับคุณลักษณะใหม่ใน Microsoft Dynamics 365 for Finance and Operations, Enterprise edition เวอร์ชันล่าสุดของคุณ

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีอัพเกรดเป็นเวอร์ชันล่าสุด ดู [กระบวนการสำหรับการอัพเกรดเป็นการอัพเดตล่าสุด](../migration-upgrade/upgrade-latest-update.md)

## <a name="application-releases"></a>การนำออกใช้ของแอพลิเคชัน

เมื่อต้องการเรียนรู้เพิ่มเติมเกี่ยวกับพื้นที่ทำงานแบบเคลื่อนที่และเนื้อหา Power BI ที่นำออกใช้เมื่อเร็ว ๆ นี้ ให้ดูที่ [พื้นที่ทำงานแบบเคลื่อนที่ที่นำออกใช้สำหรับแอพบนมือถือ Dynamics 365 for Finance and Operations](../mobile-apps/mobile-workspaces-released.md) และ [เนื้อหา Power BI ที่นำออกใช้เมื่อเร็วๆ นี้สำหรับ Finance and Operations](../analytics/power-bi-home-page.md)
