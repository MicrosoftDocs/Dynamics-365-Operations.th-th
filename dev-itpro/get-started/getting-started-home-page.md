# <a name="where-to-start"></a>จะเริ่มต้นที่ใด

- [ลงทะเบียนเพื่อสมัครใช้งาน](../dev-tools/sign-up-preview-subscription.md) 
- [Lifecycle Services (LCS) สำหรับลูกค้า Microsoft Dynamics 365 for Operations](../lifecycle-services/lcs-works-lcs.md) 
- [วิธีที่ลูกค้าที่ซื้อ Microsoft Dynamics 365 for Operations ผ่านทางผู้ให้บริการระบบคลาวด์สามารถได้รับเวอร์ชัน On-premises ของ Dynamics 365 for Operations 2012](../deployment/csp-download-customersource.md)

### <a name="task-recordings-and-task-guides"></a>การบันทึกงานและคำแนะนำของงาน

ตัวบันทึกงานเป็นเครื่องมือ Dynamics 365 for Operations ที่คุณสามารถใช้เพื่อบันทึกการดำเนินการ ที่คุณใช้ในอินเทอร์เฟสผู้ใช้ (UI) เมื่อคุณสร้าง *การบันทึกงาน*เหตุการณ์ทั้งหมดที่ดำเนินการกับเซิร์ฟเวอร์จะถูกบันทึกไว้เมื่อคุณดำเนินการดังกล่าวใน UI เหตุการณ์เหล่านี้รวมถึงการเพิ่มค่า การเปลี่ยนการตั้งค่า และการลบข้อมูล *คู่มืองาน* คือประสบการณ์ที่มีการควบคุม มีการแนะนำ และเชิงโต้ตอบที่จะแนะนำคุณถึงขั้นตอนต่าง ๆ ของการบันทึกงาน คุณสามารถเปิด (หรือ *เล่น*) คู่มืองานจากระบบวิธีใช้ได้
-   [การเชื่อมต่อระบบวิธีใช้](help-connect.md)
-   [ตัวบันทึกงานใน Dynamics 365 for Operations](../user-interface/task-recorder.md)
-   [สร้างเอกสารหรือการใช้งานการบันทึกข้อมูลการฝึกอบรม](../user-interface/task-recorder.md)
-   [มีคู่มืองานสำหรับใช้งาน (กุมภาพันธ์ 2016)](new-task-guides-available-february-2016.md)
