# <a name="where-to-start"></a>ตำแหน่งที่จะเริ่มการทำงาน

เริ่มการทำงานที่นี่เมื่อต้องการ onboard อย่างรวดเร็ว และเรียนรู้ 365 Dynamics Microsoft สำหรับ UI การดำเนินงาน

## <a name="onboarding"></a>ปฐมนิเทศ
ทรัพยากรต่อไปนี้แสดงภาพรวมของวิธีการทำงานกับหุ้นส่วนของคุณเพื่อขอรับ onboarded ถึง Dynamics 365 สำหรับการดำเนินงานและวิธีการใช้ Dynamics 365 สำหรับบริการวัฏจักรการดำเนินงาน 

- [Microsoft FastTrack สำหรับภาพรวมของ Dynamics 365](/dynamics365/operations/get-started/fasttrack-dynamics-365-overview) 
- [Sign up for a subscription](../dev-tools/sign-up-preview-subscription.md) 
- [วัฏจักรบริการ (LCS) สำหรับ 365 Dynamics Microsoft สำหรับลูกค้าที่มีการดำเนินงาน](../lifecycle-services/lcs-works-lcs.md) 
- [วิธีลูกค้าที่ซื้อ 365 Dynamics Microsoft สำหรับการดำเนินงานผ่านทางผู้ให้บริการ Cloud สามารถรับรุ่นในสถานที่ของ Dynamics 365 สำหรับ 2012 การดำเนินงาน](../deployment/csp-download-customersource.md)

### <a name="task-recordings-and-task-guides"></a>การบันทึกงานและแนะนำงาน

ตัวบันทึกภารกิจจะมี 365 Dynamics สำหรับเครื่องมือการดำเนินการที่คุณสามารถใช้เพื่อบันทึกการกระทำที่คุณทำในส่วนติดต่อผู้ใช้ (UI) เมื่อคุณสร้าง *การบันทึกงาน*เหตุการณ์ทั้งหมดที่ดำเนินการกับเซิร์ฟเวอร์จะถูกบันทึกไว้เมื่อคุณดำเนินการดังกล่าวใน UI เหตุการณ์เหล่านี้รวมถึงการเพิ่มค่า การเปลี่ยนการตั้งค่า และการลบข้อมูล 
          *คู่มืองาน* คือประสบการณ์ที่มีการควบคุม มีการแนะนำ และเชิงโต้ตอบที่จะแนะนำคุณถึงขั้นตอนต่าง ๆ ของการบันทึกงาน คุณสามารถเปิด (หรือ *เล่น*) คู่มืองานจากระบบวิธีใช้ได้
-   [การเชื่อมต่อระบบวิธีใช้](help-connect.md)
-   [ตัวบันทึกภารกิจใน Dynamics 365 สำหรับการดำเนินงาน](../user-interface/task-recorder.md)
-   [สร้างเอกสารหรือการใช้งานการบันทึกข้อมูลการฝึกอบรม](../user-interface/task-recorder.md)
-   [มีคู่มืองานสำหรับใช้งาน (กุมภาพันธ์ 2016)](new-task-guides-available-february-2016.md)
