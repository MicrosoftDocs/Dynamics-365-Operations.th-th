# <a name="create-new-users"></a>การสร้างผู้ใช้ใหม่

[!include[task guide banner](../../includes/task-guide-banner.md)]

ผู้ใช้คือพนักงานภายในขององค์กรหรือลูกค้าและผู้จัดจำหน่ายภายนอก ซึ่งมีความจำเป็นต้องเข้าถึงระบบเพื่อดำเนินงานของตน ผู้ดูแลระบบสามารถทำขั้นตอนนี้เพื่อเพิ่มผู้ใช้ในระบบ  ข้อมูลบริษัทสาธิตที่ใช้ในการสร้างกระบวนงานนี้คือ USMF 


## <a name="add-a-new-user"></a>เพิ่มผู้ใช้ใหม่
1. ไปที่การดูแลระบบ > ผู้ใช้ > ผู้ใช้
2. คลิก สร้าง
3. ในฟิลด์รหัสผู้ใช้ ให้พิมพ์ค่า
    * ป้อนรหัสเฉพาะสำหรับผู้ใช้  ต้องมีรหัสผู้ใช้  
4. ในฟิลด์ชื่อผู้ใช้ ให้พิมพ์ค่า
    * ป้อนชื่อของผู้ใช้  
5. ในฟิลด์โดเมน ให้พิมพ์ค่าใดค่าหนึ่ง
    * ป้อนโดเมนของผู้ใช้  
6. ในฟิลด์นามแฝง ให้พิมพ์ค่าใดค่าหนึ่ง
    * ป้อนนามแฝงของผู้ใช้  
7. ในฟิลด์บริษัท ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
8. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
9. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
    * เลือกบริษัทของผู้ใช้  
10. คลิกการกำหนดบทบาท
11. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
12. คลิก ตกลง
13. คลิก บันทึก

## <a name="import-users"></a>นำเข้าผู้ใช้
1. คลิกการนำเข้าผู้ใช้
2. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
3. คลิกการนำเข้าผู้ใช้
4. คลิก ปิด
