# <a name="mobile-workspaces-recently-released"></a>พื้นที่ทำงานแบบเคลื่อนที่ที่นำออกใช้เมื่อเร็วๆ นี้

[!include[banner](../includes/banner.md)]



หัวข้อนี้แสดงรายการพื้นที่ทำงานแบบเคลื่อนที่ที่นำออกใช้เมื่อเร็วๆ นี้สำหรับแอพบนมือถือ Microsoft Dynamics 365 for Unified Operations

> [!NOTE]
> แอพบนมือถือก่อนหน้านี้มีชื่อว่า *Microsoft Dynamics 365 for Finance and Operations*


## <a name="new-mobile-workspaces-released-in-june-2017"></a>พื้นที่ทำงานแบบเคลื่อนที่แบบใหม่ที่นำออกใช้เมื่อเดือนมิถุนายน 2017

| พื้นที่ทำงานแบบเคลื่อนที่     | คำอธิบาย   | เรียนรู้เพิ่มเติม   |
|----------------------|---------------|--------------|
|ไดเรกทอรีบริษัท| ช่วยให้ผู้ใช้ดูและติดต่อพนักงานคนอื่นในองค์กร| [พื้นที่ทำงานไดเรกทอรีของบริษัทแบบเคลื่อนที่](company-directory-mobile-workspace.md)|    
|ทีมงานของฉัน| ช่วยให้ผู้ใช้ดูการรายงานและพนักงานโดยรวมได้โดยตรง และส่งการชมเชยสำหรับแต่ละบุคคลในกลุ่มของการรายงานได้| [พื้นที่ทำงานแบบเคลื่อนที่ของทีมงานของฉัน](manager-self-service-mobile-workspace.md)|     
|การอนุมัติใบแจ้งหนี้| แสดงรายการของใบแจ้งหนี้ที่มีการกำหนดให้กับคุณผ่านขั้นตอนลำดับงานส่วนหัวของใบแจ้งหนี้ผู้จัดจำหน่าย|[พื้นที่ทำงานการอนุมัติใบแจ้งหนี้แบบเคลื่อนที่](invoice-approval-mobile-workspace.md)    |    

## <a name="new-mobile-workspaces-released-in-april-2017"></a>พื้นที่ทำงานแบบเคลื่อนที่แบบใหม่ที่นำออกใช้เมื่อเดือนเมษายน 2017

| พื้นที่ทำงานแบบเคลื่อนที่   | คำอธิบาย                                                                                                                                                                                                      | เรียนรู้เพิ่มเติม                                                                                                      |
|--------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| การจัดการค่าใช้จ่าย | ช่วยให้ผู้ใช้สามารถเก็บบันทึกและอัพโหลดใบเสร็จรับเงิน เพื่อให้สามารถแนบกับรายงานค่าใช้จ่ายในภายหลังได้ พื้นที่ทำงานแบบเคลื่อนที่ยังช่วยให้ผู้ใช้สามารถสร้างรายการค่าใช้จ่ายได้อย่างรวดเร็วโดยใช้ใบเสร็จรับเงินที่แนบ | [พื้นที่ทำงานแบบเคลื่อนที่การจัดการค่าใช้จ่าย](/dynamics365/unified-operations/financials/expense-management/expense-management-mobile-workspace) |
| การอนุมัติใบสั่งซื้อ | อนุญาตให้ผู้ใช้ดูและตอบสนองกับใบสั่งซื้อพร้อมกับการดำเนินการ เช่น อนุมัติ หรือปฏิเสธ | [พื้นที่ทำงานแบบเคลื่อนที่สำหรับการอนุมัติใบสั่งซื้อ](/dynamics365/unified-operations/supply-chain/procurement/purchase-order-mobile-workspace) |

## <a name="new-mobile-workspaces-released-in-march-2017"></a>พื้นที่ทำงานแบบเคลื่อนที่แบบใหม่ที่นำออกใช้เมื่อเดือนมีนาคม 2017

| พื้นที่ทำงานแบบเคลื่อนที่   | คำอธิบาย                                                                                   | เรียนรู้เพิ่มเติม                                                                                                                                                                              |
|--------------------|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| รายการข้อมูลเวลาของโครงการ | ช่วยให้ผู้ใช้สามารถป้อนและประหยัดเวลาในแง่ของโครงการโดยการใช้อุปกรณ์เคลื่อนที่ของตน | [พื้นที่ทำงานแบบเคลื่อนที่สำหรับการป้อนข้อมูลเวลาโครงการ](/dynamics365/unified-operations/financials/project-management/project-time-entry-mobile-workspace) |

## <a name="new-mobile-workspaces-released-in-january-2017"></a>พื้นที่ทำงานแบบเคลื่อนที่แบบใหม่ที่นำออกใช้เมื่อเดือนมกราคม 2017

| พื้นที่ทำงานแบบเคลื่อนที่     | คำอธิบาย                                                                                                                                                                         | เรียนรู้เพิ่มเติม                                                                                                                                                        |
|----------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| การควบคุมต้นทุน     | ช่วยให้ผู้จัดการศูนย์ต้นทุนสามารถดูประสิทธิภาพของศูนย์ต้นทุนได้ตลอดเวลาและที่ใดก็ได้                                                                                               | [พื้นที่ทำงานแบบเคลื่อนที่การควบคุมต้นทุน](/dynamics365/unified-operations/financials/cost-accounting/cost-controlling-mobile-workspace)         |
| ปริมาณสินค้าคงคลังคงเหลือ    | ช่วยให้ผู้ใช้งานรับข้อมูลเชิงลึกในสินค้าคงคลังที่จองไว้และพร้อมใช้งานได้ตลอดเวลา และที่ใดก็ได้                                                                                                    | [พื้นที่ทำงานแบบเคลื่อนที่ของปริมาณคงคลังคงเหลือ](/dynamics365/unified-operations/supply-chain/inventory/inventory-on-hand-mobile-workspace)       |
| ใบสั่งขาย         | ช่วยให้ผู้ใช้คอยติดตามสถานการณ์ปัจจุบันเกี่ยวกับใบสั่งขายของคุณได้ตลอดเวลา และที่ใดก็ได้                                                                                                                          | [พื้นที่ทำงานแบบเคลื่อนที่ใบสั่งขาย](/dynamics365/unified-operations/supply-chain/sales-marketing/sales-orders-mobile-workspace)                 |
| การทำงานร่วมกันกับผู้จัดจำหน่าย | ช่วยให้ผู้จัดจำหน่ายของคุณได้ติดตามสถานการณ์ปัจจุบันในใบสั่งซื้อที่ถูกส่งไปขอการอนุมัติ นอกจากนี้ยังสามารถดูข้อมูลเกี่ยวกับใบสั่งซื้อและผู้ติดต่อใหม่และที่อัพเดตแล้ว | [พื้นที่ทำงานแบบเคลื่อนที่ของการทำงานร่วมกันกับผู้จัดจำหน่าย](/dynamics365/unified-operations/supply-chain/procurement/vendor-collaboration-mobile-workspace) |
