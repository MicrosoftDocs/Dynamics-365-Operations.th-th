# <a name="mobile-workspaces-recently-released-for-the-dynamics-365-for-operations-mobile-app"></a>พื้นที่ทำงานแบบเคลื่อนที่ที่นำออกใช้เมื่อเร็วๆ นี้สำหรับแอพบนมือถือ Dynamics 365 for Operations

[!include[banner](../includes/banner.md)]



หัวข้อนี้แสดงรายการพื้นที่ทำงานแบบเคลื่อนที่ที่นำออกใช้เมื่อเร็วๆ นี้สำหรับแอพบนมือถือ Microsoft Dynamics 365 for Operations

<a name="mobile-workspaces-that-were-released-in-april-2017"></a>พื้นที่ทำงานแบบเคลื่อนที่ที่นำออกใช้ในเดือนเมษายน 2017
--------------------------------------------------

| พื้นที่ทำงานแบบเคลื่อนที่   | คำอธิบาย                                                                                                                                                                                                      | เรียนรู้เพิ่มเติม                                                                                                      |
|--------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| การจัดการค่าใช้จ่าย | พื้นที่ทำงานนี้ช่วยให้ผู้ใช้สามารถเก็บบันทึกและอัพโหลดใบเสร็จรับเงิน เพื่อให้สามารถแนบกับรายงานค่าใช้จ่ายในภายหลังได้ พื้นที่ทำงานแบบเคลื่อนที่ยังช่วยให้ผู้ใช้สามารถสร้างรายการค่าใช้จ่ายได้อย่างรวดเร็วโดยใช้ใบเสร็จรับเงินที่แนบ | [พื้นที่ทำงานแบบเคลื่อนที่การจัดการค่าใช้จ่าย](/dynamics365/operations/financials/expense-management/expense-management-mobile-workspace) |
