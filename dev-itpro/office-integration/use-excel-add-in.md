# <a name="use-the-excel-add-in"></a>ใช้ Excel เพิ่มอิน

หัวข้อนี้อธิบายวิธีการเปิดข้อมูลเอนทิตีใน Microsoft Excel และจากนั้น ดู ปรับปรุง และแก้ไขข้อมูล โดยใช้ Microsoft Dynamics Office เพิ่มอินสำหรับ Excel เปิดเอนทิตีข้อมูล คุณสามารถเริ่มจาก Excel หรือ 365 Dynamics Microsoft สำหรับการดำเนินงาน

โดยการเปิดข้อมูลเอนทิตีใน Microsoft Excel คุณสามารถได้อย่างรวดเร็ว และง่ายดายดู และแก้ไขข้อมูล โดยใช้ Microsoft Dynamics Office add-in ของ Excel Add-in นี้จำเป็นต้องมี Microsoft Excel 2016 **หมายเหตุ:**ผู้เช่า Microsoft Azure Active Directory (Azure AD) ของคุณถูกกำหนดค่าให้ใช้บริการสหพันธรัฐไดเรกทอรีที่ใช้งานอยู่ (AD FS), คุณต้องแน่ใจว่า มีการใช้การปรับปรุงเดือน 2016 พฤษภาคม เพื่อให้ Excel เพิ่มอินสามารถได้อย่างถูกต้องคุณเข้าสู่ระบบ

## <a name="open-entity-data-in-excel-when-you-start-from-dynamics-365-for-operations"></a>เปิดข้อมูลเอนทิตีใน Excel เมื่อคุณเริ่มการทำงานจาก 365 Dynamics สำหรับการดำเนินงาน
1.  บนเพจใน 365 Dynamics Microsoft สำหรับการดำเนินงาน คลิ**เปิดใน Microsoft Office** ถ้าแหล่งข้อมูลราก (ตาราง) สำหรับหน้าจะเหมือนกับแหล่งข้อมูลรากสำหรับเอนทิตีใด ๆ**เปิดใน Excel**ตัวเลือกจะถูกสร้างขึ้นสำหรับหน้านั้น **เปิดใน Excel**ตัวเลือกสามารถพบได้บนเพจที่ใช้บ่อย เช่น**ผู้จัดจำหน่ายทั้งหมด**และ**ลูกค้าทั้งหมด**ได้
2.  คลิกการ**เปิดใน Excel**ตัวเลือก และเปิดสมุดงานที่สร้างขึ้น สมุดงานนี้มีการผูกข้อมูลสำหรับเอนทิตี ตัวชี้กับสภาพแวดล้อมของคุณ และตัวชี้ไปยัง Excel add-in ใน
3.  ใน Excel คลิ**การเปิดใช้งานการแก้ไข**เพื่อเปิดใช้งานใน Excel เพิ่มในการเรียกใช้งาน Excel เพิ่มอินทำงานในบานหน้าต่างด้านขวาของหน้าต่าง Excel
4.  ถ้าคุณกำลังเรียกใช้ Excel เพิ่มอินเป็นครั้งแรก คลิ**เชื่อถือ add-in นี้**
5.  ถ้าคุณได้รับการพร้อมท์ให้เข้าสู่ระบบ คลิ**เข้าสู่ระบบ**และจากนั้น เข้าสู่ระบบ โดยใช้ข้อมูลประจำตัวเดียวกับที่คุณใช้เพื่อเข้าสู่ระบบ Dynamics 365 สำหรับการดำเนินงาน Excel เพิ่มอินจะใช้ก่อนหน้าเครื่องหมายในบริบทที่จาก Internet Explorer และจะลงชื่อเข้าใช้คุณใน ถ้าใช้ได้ ดังนั้น ตรวจสอบชื่อผู้ใช้ในมุมขวาด้านบนของ Excel เพิ่มอิน

Excel เพิ่มอินอ่านข้อมูลสำหรับเอนทิตีที่คุณเลือกโดยอัตโนมัติ โปรดสังเกตว่า จะไม่มีข้อมูลในสมุดงานจนกระทั่ง Excel เพิ่มอินอ่านใน

## <a name="open-entity-data-in-excel-when-you-start-from-excel"></a>เปิดข้อมูลเอนทิตีใน Excel เมื่อคุณเริ่มการทำงานจาก Excel
1.  ใน Excel บน**แทรก**แท็บ ในการ**เพิ่มเติม**จัดกลุ่ม คลิ**ร้านค้า**เพื่อเปิดเก็บ Office
2.  ในเก็บ Office ค้นหาโดยใช้คำสำคัญ "Dynamics" และคลิ**เพิ่ม**ถัดจาก**add-in ที่ Microsoft Dynamics Office** (Excel เพิ่มอิน)
3.  ถ้าคุณกำลังเรียกใช้ Excel เพิ่มอินเป็นครั้งแรก คลิ**เชื่อถือ add-in นี้**เพื่อเปิดใช้งานใน Excel เพิ่มในการเรียกใช้งาน Excel เพิ่มอินทำงานในบานหน้าต่างด้านขวาของหน้าต่าง Excel
4.  คลิ**เพิ่มข้อมูลเซิร์ฟเวอร์**เพื่อเปิด**ตัวเลือก**บานหน้าต่างงาน
5.  คัดลอก URL เบราว์เซอร์จาก Dynamics 365 ของคุณเป้าหมายสำหรับอินสแตนซ์ของการดำเนินงาน วางลงใน**URL ของเซิร์ฟเวอร์**ฟิลด์ และลบทุกอย่างหลังชื่อโฮสต์ (ตัวอย่าง ลบ**/ ? cmp = usmf & mi = CustTableListPage**) URL เป็นผลลัพธ์ควรมีเพียงชื่อโฮสต์ (ตัวอย่างเช่น**https://xxx.dynamics.com**)
6.  คลิ**ตกลง**แล้ว คลิก**ใช่**เพื่อยืนยันการเปลี่ยนแปลง Excel เพิ่มรีสตาร์ท และโหลดข้อมูลเมตา การ**ออกแบบ**ปุ่มจะพร้อมใช้งาน ถ้า Excel เพิ่มอินได้**โหลดแอปเพล็ต**ปุ่ม คุณอาจจะไม่ได้เข้าสู่ระบบในฐานะผู้ใช้ที่ถูกต้อง สำหรับข้อมูลเพิ่มเติม ดู "ปุ่มโหลดแอปเพล็ตจะแสดง" ในส่วน "การแก้ไขปัญหาเบื้องต้น" ของหัวข้อนี้
7.  คลิ**ออกแบบ** Excel เพิ่มอินดึงข้อมูลเมตาของเอนทิตี
8.  คลิ**เพิ่มตาราง** รายการของเอนทิตีปรากฏขึ้น เอนทิตีจะแสดงรายการในรูปแบบ "–ป้ายชื่อ"
9.  เลือกเอนทิตีในรายการ เช่น**ลูกค้า - ลูกค้า**แล้ว คลิก**ถัดไป**
10. เมื่อต้องการเพิ่มเขตข้อมูลจากการ**เขตข้อมูลที่มี**รายการ**เลือกเขตข้อมูล**รายการ คลิกฟิลด์ และจากนั้น คลิ**เพิ่ม** อีกวิธีหนึ่งคือ คลิกสองครั้งเขตข้อมูล
11. หลังจากที่คุณได้เพิ่มเขตข้อมูลที่ต้องการการ**เลือกเขตข้อมูล**รายการ โปรดตรวจสอบให้แน่ใจว่า เคอร์เซอร์อยู่ในตำแหน่งที่ถูกต้องในแผ่นงาน (ตัวอย่างเช่น เซลล์ A1), และจากนั้น คลิ**ทำ** แล้ว คลิก**ทำ**เพื่อออกจากตัวออกแบบ
12. คลิ**รีเฟรช**เพื่อดึงชุดข้อมูล

## <a name="view-and-update-entity-data-in-excel"></a>ดู และอัพเดตข้อมูลเอนทิตีใน Excel
หลังจาก Excel add-in ของอ่านข้อมูลเอนทิตีลงในสมุดงาน คุณสามารถปรับปรุงข้อมูลตลอดเวลา โดยการคลิก**รีเฟรช**ใน Excel เพิ่มอินได้

## <a name="edit-entity-data-in-excel"></a>แก้ไขข้อมูลของเอนทิตีใน Excel
คุณสามารถเปลี่ยนแปลงข้อมูลในเอนทิตีคุณจำเป็นต้องมี และเผยแพร่กลับมา ด้วยการคลิก**ประกาศ**ใน Excel เพิ่มอินได้ เมื่อต้องการแก้ไขเรกคอร์ด เลือกเซลล์ในแผ่นงาน และจากนั้น เปลี่ยนค่าในเซลล์ เมื่อต้องการเพิ่มระเบียนใหม่ ให้ทำตามขั้นตอนเหล่านี้:

-   คลิกที่ใดก็ได้ในแผ่นงาน และจากนั้น คลิ**สร้าง**ใน Excel เพิ่มอิน
-   คลิกในแถวสุดท้ายของแผ่นงาน และจากนั้น กดแป้น Tab จนกว่าเคอร์เซอร์ย้ายจากคอลัมน์สุดท้ายของแถวนั้น และแถวใหม่ถูกสร้างขึ้น
-   คลิกในแถวอยู่ด้านล่างแผ่นงาน และเริ่มต้นการป้อนข้อมูลในเซลล์ เมื่อคุณย้ายโฟกัสออกจากเซลล์นั้น แผ่นงานที่ขยายเพื่อรวมแถวใหม่

เมื่อต้องการลบเร็กคอร์ด ให้ทำตามขั้นตอนเหล่านี้:

-   คลิกขวาที่หมายเลขแถวอยู่ถัดจากแถวแผ่นงานเมื่อต้องการลบ และจากนั้น คลิก**ลบ**
-   คลิกขวาในแถวแผ่นงานเมื่อต้องการลบ แล้ว คลิ ก**ลบ**&gt;**แถวตาราง**

## <a name="add-or-remove-columns"></a>เพิ่มหรือเอาคอลัมน์ออก
คุณสามารถใช้ตัวออกแบบเพื่อปรับปรุงคอลัมน์ที่ถูกเพิ่มโดยอัตโนมัติไปแผ่นงาน

1.  เริ่มการทำงานของโปรแกรมออกแบบแหล่งข้อมูลของ Excel เพิ่มอิน ด้วยการคลิก**ตัวเลือก**ปุ่ม (สัญลักษณ์เกียร์) และจากนั้น เลือก**การเปิดใช้งานการออกแบบ**กล่องกาเครื่องหมาย
2.  คลิ**ออกแบบ**ใน Excel เพิ่มอิน แหล่งข้อมูลทั้งหมดแสดงรายการไว้
3.  อยู่ติดกับแหล่งข้อมูล คลิก**แก้ไข**ปุ่ม (สัญลักษณ์ดินสอ)
4.  ปรับปรุงรายการในแบบ**เลือกเขตข้อมูล**ตามที่คุณต้องการแสดงรายการ:
    -   เมื่อต้องการเพิ่มเขตข้อมูลจากการ**เขตข้อมูลที่มี**รายการ**เลือกเขตข้อมูล**รายการ คลิกฟิลด์ และจากนั้น คลิ**เพิ่ม** อีกวิธีหนึ่งคือ คลิกสองครั้งเขตข้อมูล
    -   เมื่อต้องการเอาเขตข้อมูลจากการ**เลือกเขตข้อมูล**รายการ คลิกฟิลด์ และจากนั้น คลิ**เอา** อีกวิธีหนึ่งคือ คลิกสองครั้งเขตข้อมูล
    -   เมื่อต้องการเปลี่ยนลำดับของเขตข้อมูล คลิกฟิลด์ใน**เลือกเขตข้อมูล**รายการ และจากนั้น คลิ**ค่า**หรือ**ลง**

5.  ใช้การเปลี่ยนแปลงของคุณไปยังแหล่งข้อมูล ด้วยการคลิก**เด** แล้ว คลิก**ทำ**เพื่อออกจากตัวออกแบบ ถ้าคุณเพิ่มเขตข้อมูล (คอลัมน์), คลิก**รีเฟรช**เพื่อดึงชุดการปรับปรุงข้อมูล

## <a name="httpspowerappsmicrosoftcomenustutorialsdataplatforminteractiveexceltroubleshootingtroubleshooting"></a>[](https://powerapps.microsoft.com/enus/tutorials/dataplatforminteractiveexcel/#troubleshooting)Troubleshooting
มีปัญหาบางปัญหาที่สามารถแก้ไขขั้นตอนบางอย่างง่าย

-   **โหลดแอปเพล็ตปุ่มจะแสดงขึ้น** ถ้า Excel เพิ่มอินได้**โหลดแอปเพล็ต**ปุ่มหลังจากเข้าสู่ระบบใน คุณอาจไม่ได้เข้าสู่ระบบในฐานะผู้ใช้ที่ถูกต้อง เมื่อต้องการแก้ไขปัญหานี้ โปรดตรวจสอบว่า ชื่อผู้ใช้ที่ถูกต้องปรากฏอยู่ในมุมขวาด้านบนของ Excel เพิ่มอิน ถ้าชื่อผู้ใช้ที่ไม่ถูกต้องปรากฏขึ้น คลิกที่ ลงชื่อออก และจากนั้น เข้าสู่ระบบกลับไป
-   **คุณได้รับข้อความ "Forbidden"** ถ้าคุณได้รับข้อความ "Forbidden" ในขณะ Excel add-in ในกำลังโหลดข้อมูลเมตา บัญชีผู้ใช้ที่ล็อกอินไปยัง Excel add-in ของ ไม่มีสิทธิ์ในการใช้บริการเป้าหมาย อินสแตนซ์ หรือฐานข้อมูล เมื่อต้องการแก้ไขปัญหานี้ โปรดตรวจสอบว่า ชื่อผู้ใช้ที่ถูกต้องปรากฏอยู่ในมุมขวาด้านบนของ Excel เพิ่มอิน ถ้าชื่อผู้ใช้ที่ไม่ถูกต้องปรากฏขึ้น คลิกที่ ลงชื่อออก และจากนั้น เข้าสู่ระบบกลับไป
-   **เว็บเพจที่ว่างเปล่าจะปรากฏเหนือ Excel** ถ้าเว็บเพจที่ว่างเปล่าเปิดขึ้นในระหว่างกระบวนการเข้าสู่ระบบ บัญชีต้องการโฆษณา FS แต่รุ่นของ Excel ที่กำลังเรียกใช้ add-in ที่ไม่ล่าโหลดกล่องโต้ตอบการเข้าสู่ระบบ เมื่อต้องการแก้ไขปัญหานี้ ปรับปรุงรุ่นของ Excel ที่คุณใช้อยู่ เมื่อต้องการปรับปรุงรุ่นของ Excel เมื่อคุณอยู่ในองค์กรที่อยู่ในช่องทางรอการตัดบัญชี ใช้[เครื่องมือการปรับใช้ Office](https://technet.microsoft.com/library/jj219422.aspx)ไป[ย้ายจากช่องทางที่เลื่อนเวลาไปยังสถานีปัจจุบัน](https://technet.microsoft.com/library/mt455210.aspx)
