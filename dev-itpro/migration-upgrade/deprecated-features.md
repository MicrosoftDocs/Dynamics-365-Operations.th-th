# <a name="deprecated-features"></a>ลักษณะการทำงานที่ไม่ได้รับการสนับสนุน

[!include[banner](../includes/banner.md)]

หัวข้อนี้อธิบายถึงคุณลักษณะที่ได้ถูกลบออก หรือที่ได้ถูกวางแผนไว้สำหรับการลบออกจาก Microsoft Dynamics 365 for Finance and Operations, Enterprise Edition

## <a name="features-that-have-been-deprecated-for-all-deployment-types-of-the-july-2017-update-with-platform-update-8"></a>ลักษณะการทำงานที่ได้รับการสนับสนุนสำหรับชนิดของการปรับใช้ทั้งหมดของการอัพเดตเดือนกรกฎาคม 2017 พร้อมกับการอัพเดตแพลตฟอร์ม 8
รายการนี้รวมลักษณะการทำงานที่ได้รับการสนับสนุนสำหรับทั้งการปรับใช้ระบบ Cloud และในองค์กร

### <a name="warehouse-mobile-devices-portal"></a>พอร์ทัลบนอุปกรณ์เคลื่อนที่ของคลังสินค้า

พอร์ทัลบนอุปกรณ์เคลื่อนที่ของคลังสินค้า (WMDP) เป็นส่วนประกอบแบบสแตนด์อโลนที่สร้างขึ้นสำหรับการปรับใช้ในองค์กรของตนเอง ส่วนประกอบนี้จะไม่สนับสนุนใน Microsoft Dynamics 365 for Finance and Operations, Enterprise Edition อีกต่อไป แอพไคลแอนต์ที่ช่วยปรับปรุงประสบการณ์การใช้งานของผู้ใช้ได้แทนที่การทำงานของ WMDP 

|                                  |                                                 |
|----------------------------------|-------------------------------------------------|
| **เหตุผลสำหรับค่าเสื่อมราคา**       | ฟังก์ชันที่ซ้ำกัน                        |
| **แทนที่ด้วยลักษณะการทำงานอื่นหรือไม่** | ใช่ คุณลักษณะนี้ถูกแทนที่โดย Finance and Operations - คลังสินค้า สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการตั้งค่าและข้อกำหนดเบื้องต้น ให้ดูที่ [ติดตั้ง และตั้งค่าคอนฟิก Microsoft Dynamics 365 for Finance and Operations - คลังสินค้า](/dynamics365/unified-operations/supply-chain/warehousing/install-configure-warehousing-app) |
| **โมดูลได้รับผลกระทบ**             | การจัดการคลังสินค้า การจัดการการขนส่ง |

### <a name="advanced-bank-reconciliation-matching-rule-for-manual-matching"></a>กฎการจับคู่การกระทบยอดบัญชีธนาคารขั้นสูงสำหรับการจับคู่ด้วยตนเอง

กฎการจับคู่ถูกใช้ในการเลือกและทำเครื่องหมายเอกสารธนาคารเมื่อเอกสารถูกจับคู่ด้วยตนเองในแผ่นงานการกระทบยอด

|                                  |                                                                                        |
|----------------------------------|----------------------------------------------------------------------------------------|
| **เหตุผลสำหรับค่าเสื่อมราคา**       | การใช้งานที่จำกัด                                                                         |
| **แทนที่ด้วยลักษณะการทำงานอื่นหรือไม่** | หมายเลข ควรใช้คอลัมน์ความสามารถของการกรองข้อมูลในคอลัมน์เพื่อค้นหาเอกสารสำหรับการกระทบยอด |
| **โมดูลได้รับผลกระทบ**             | การจัดการเงินสดและธนาคาร                                                               |

### <a name="windows-8-tablet-app"></a>แอพสำหรับแท็บเล็ตบน Windows 8
