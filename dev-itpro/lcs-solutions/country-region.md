# <a name="localization-and-regulatory-features"></a>แปลและคุณลักษณะตามข้อบังคับ

356 Dynamics Microsoft สำหรับการดำเนินงานรวมถึงหน้าที่การใช้งานสำหรับบางข้อบังคับเฉพาะประเทศ/ภูมิภาคและกระบวนการ ฟังก์ชันนี้จะเปิดใช้งานตามอยู่หลักของนิติบุคคลที่ใช้งานอยู่ หัวข้อนี้รวมถึงรายการของทรัพยากรที่จะช่วยให้คุณเรียนรู้เพิ่มเติมเกี่ยวกับฟังก์ชันการทำงานเฉพาะประเทศ/ภูมิภาค 

## <a name="developer-and-isv-information"></a>ข้อมูลสำหรับนักพัฒนาและ ISV
ทรัพยากรต่อไปนี้ให้ข้อมูลที่สามารถช่วยให้นักพัฒนาและ Isv ที่กำลังสร้างเองเฉพาะประเทศ/ภูมิภาค
-   [แบบจำลองการแปลแยกต่างหาก](separate-localization-models.md)
-   [ใช้บริบทประเทศ/ภูมิภาค](apply-country-context.md)
-   [ใบรับรองตามข้อบังคับ](regulatory-certifications.md)
-   [คุณลักษณะการแปลการจัดประเภท](classify-localization-features.md)
-   [รหัสประเทศ - ISO 3166](https://www.iso.org/iso-3166-country-codes.html)

## <a name="localization-updates"></a>ปรับปรุงการแปล
ทรัพยากรต่อไปนี้ให้ข้อมูลเกี่ยวกับลักษณะการทำงานของการปรับภาษาใหม่ และวางแผนไว้ 
-   [แปลเว็บไซต์](https://mbs.microsoft.com/customersource/northamerica/ax/support/support-news/GFMLocalizationPortalMC)
-   [ดูตามข้อบังคับและการสื่อสารการปรับปรุงตามข้อบังคับ](regulatory-watch-communication.md)
-   [การแจ้งให้ทราบว่า Microsoft เกี่ยวกับการแจ้งเตือนข้อบังคับของประเทศ/ภูมิภาค และติดตามสถานะของคุณลักษณะตามข้อบังคับ](submit-localization-alerts.md)

-   [ดาวน์โหลดการตั้งค่าคอนฟิกการรายงานแบบอิเล็กทรอนิกส์จาก Lifecycle Services](../analytics/download-electronic-reporting-configuration-lcs.md)
