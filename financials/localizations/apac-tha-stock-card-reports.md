# <a name="stock-card-reports"></a>รายงานบัตรสินค้าคงคลัง

หัวข้อนี้ให้ข้อมูลเกี่ยวกับรายงานบัตรสินค้าคงคลังสำหรับนิติบุคคลในประเทศไทย รายงานบัตรสินค้าคงคลังประกอบด้วยรายละเอียดเกี่ยวกับความเคลื่อนไหวของสินค้าคงคลัง ปริมาณ ต้นทุน และสำหรับแต่ละสินค้าและคลังสินค้า และต้องถูกสร้างขึ้นเมื่อรัฐบาลร้องขอ รายงานบัตรสินค้าคงคลังจะถูกสร้างขึ้นก่อน หรือหลัง จากที่กระบวนปิดสินค้าคงคลัง 

คุณสามารถสร้างรายงานบัตรสินค้าคงคลังต่อไปนี้:

-   **บัตรสินค้าคงคลังทางกายภาพ**– ดูข้อมูลที่เกี่ยวข้องกับไซต์เดียวหรือไซต์ทั้งหมด ต่อสถานที่เก็บและคลังสินค้า หลังจากมีการลงรายการบัญชีบันทึกการจัดส่ง
-   **บัตรสินค้าคงคลังทางการเงิน**– ดูข้อมูลที่เกี่ยวข้องกับไซต์เดียวหรือไซต์ทั้งหมด ต่อสถานที่เก็บและคลังสินค้า หลังจากมีการลงรายการบัญชีใบแจ้งหนี้
-   **บัตรสินค้าคงคลังที่สรุป**– ดูสินค้า หรือรายละเอียดตามกลุ่มสินค้า ไซต์ คลังสินค้า ต้นทุน และคลังสินค้า

## <a name="set-up-the-generation-of-stock-card-reports"></a>ตั้งค่าการสร้างรายงานบัตรสินค้าคงคลัง
ทำงานต่อไปนี้ก่อนที่คุณสร้างรายงานบัตรสินค้าคงคลัง:

-   สร้างรายการที่มีข้อมูลที่จำเป็น **หมายเหตุ:**กำหนดสาขาภาษีให้กับไซต์
-   สร้างสาขาภาษี
-   ตั้งค่าสาขาภาษีสำหรับนิติบุคคล
-   ตั้งค่าสาขาภาษีสำหรับผู้จัดจำหน่ายหรือลูกค้า
-   กำหนดสาขาภาษีให้กับไซต์
-   สร้าง และลงรายการบัญชีการซื้อ การขาย และธุรกรรมการโอนย้ายสินค้าคงคลังที่จำเป็น

## <a name="recalculate-inventory-and-generate-the-financial-stock-card-report"></a>คำนวณสต็อกอีกครั้ง และสร้างรายงานบัตรสินค้าคงคลังทางการเงิน
ใช้การ**การปิดและการปรับปรุง**หน้าการคำนวณสต็อกอีกครั้งหลังจากรันกระบวนการปิดสินค้าคงคลัง และสร้างรายงานบัตรสินค้าคงคลังทางการเงิน

## <a name="generate-the-stock-card-report"></a>สร้างรายงานบัตรสินค้าคงคลัง
คุณสามารถจัดกลุ่มรายการตามไซต์ คลังสินค้า ที่ตั้ง หมายเลขสินค้า หรือกลุ่มสินค้า และสร้างรายงานบัตรสินค้าคงคลังแล้ว ใช้การ**บัตรสินค้าคงคลัง – ทางกายภาพ**รายงานเพื่อสร้างรายงานบัตรสินค้าคงคลังทางกายภาพ ใช้การ**บัตรสินค้าคงคลัง – การเงิน**รายงานเพื่อสร้างรายงานบัตรสินค้าคงคลังทางการเงิน คุณสามารถสร้างรายงานสรุป หรือรายละเอียดบัตรสินค้าคงคลัง
