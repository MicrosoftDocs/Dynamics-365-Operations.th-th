# <a name="budget-planning-templates-for-excel"></a>แม่แบบการวางแผนงบประมาณสำหรับ Excel

หัวข้อนี้อธิบายวิธีการสร้างแม่แบบของ Microsoft Excel ที่สามารถใช้ได้กับแผนงบประมาณ

หัวข้อนี้แสดงวิธีการสร้างแม่แบบ Excel ที่จะใช้กับแผนงบประมาณโดยใช้ชุดข้อมูลสาธิตมาตรฐานและล็อกอินผู้ใช้ Admin สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการวางแผนงบประมาณ ดู[วางแผนภาพรวมงบประมาณ](budget-planning-overview-configuration.md) นอกจากนี้คุณสามารถทำตาม[งบประมาณวางแผน 101](budget-plan.md)บทช่วยสอนการเรียนรู้หลักการตั้งค่าคอนฟิกและการใช้โมดูลพื้นฐานได้

## <a name="generate-a-worksheet-using-budget-plan-document-layout"></a>สร้างแผ่นงานโดยใช้เค้าโครงเอกสารของแผนงบประมาณ
เอกสารแผนงบประมาณสามารถดู และแก้ไขการใช้เค้าโครงอย่าง น้อยหนึ่งรายการ แต่ละโครงร่างสามารถมีแม่แบบเอกสารแผนงบประมาณที่เกี่ยวข้องเพื่อดู และแก้ไขข้อมูลในแผ่นงาน Excel แผนงบประมาณ ในหัวข้อนี้ แม่แบบเอกสารแผนงบประมาณจะถูกสร้างขึ้นโดยใช้การกำหนดค่าโครงร่างที่มีอยู่ เปิดตัว**รายการการวางแผนงบประมาณ**(**จัดทำงบประมาณ**&gt;**แผนงบประมาณ**) คลิ**สร้าง**เมื่อต้องการสร้างเอกสารแผนงบประมาณใหม่ [![bpt1](./media/bpt11-1024x552.png)](./media/bpt11.png) 

ใช้การ**เพิ่ม**ตัวเลือกเพื่อเพิ่มบรรทัดบรรทัด คลิ**เค้าโครง**เพื่อดูการกำหนดค่าโครงร่างเอกสารแผนงบประมาณ 
[![bpt2](./media/bpt2-1024x274.png)](./media/bpt2.png) 

คุณสามารถตรวจทานการตั้งค่าคอนฟิกโครงร่าง และปรับตามความจำเป็น ไปที่**แม่**&gt;**สร้าง**เมื่อต้องการสร้างแฟ้ม Excel สำหรับโครงร่างนี้ หลังจากที่มีสร้างแม่แบบ ไป**แม่**&gt;**ดู**เพื่อเปิด และตรวจทานแม่แบบเอกสารแผนงบประมาณ คุณสามารถบันทึกแฟ้ม Excel ไปที่ไดรฟ์ภายในเครื่องของคุณ [![bpt3](./media/bpt3-1024x545.png)](./media/bpt3.png) 

> [!NOTE] 
> ไม่สามารถแก้ไขเค้าโครงของเอกสารแผนงบประมาณหลังจากที่มีการเชื่อมโยงกับเท็มเพลต Excel เมื่อต้องการปรับเปลี่ยนโครงร่าง ลบแฟ้มแม่แบบ Excel เกี่ยวข้อง และสร้างใหม่ดังกล่าว จำเป็นจะเก็บฟิลด์ในโครงร่างนี้ และแผ่นงานการซิงโครไนส์ 

แม่แบบ Excel จะประกอบด้วยองค์ประกอบจากเค้าโครงเอกสารแผนงบประมาณ ทั้งหมดที่**พร้อมใช้งานในแผ่นงาน**คอลัมน์ถูกตั้งค่าเป็น True องค์ประกอบที่ซ้อนทับกันไม่ได้ในแม่แบบ Excel ตัวอย่างเช่น ถ้าโครงร่างประกอบด้วย Q1 ร้องขอ ขอ Q2, Q3 ร้องขอ และคอลัมน์ Q4 ขอ และคำขอรวมคอลัมน์ที่แสดงผลรวมของคอลัมน์ทั้งหมดรายไตรมาส 4 คอลัมน์รายไตรมาสหรือคอลัมน์ผลรวมเท่านั้นจะพร้อมใช้งานสำหรับใช้ในแม่แบบ Excel แฟ้ม Excel ไม่สามารถปรับปรุงคอลัมน์ที่ซ้อนทับกันอยู่ในระหว่างการปรับปรุงเนื่องจากข้อมูลในตารางอาจจะล้าสมัย และไม่ถูกต้อง

[![bpt4](./media/bpt4-1024x615.png)](./media/bpt4.png)

> [!NOTE] 
> เมื่อต้องการหลีกเลี่ยงปัญหาที่อาจเกิดขึ้นกับดู และแก้ไขข้อมูลแผนงบประมาณที่ใช้ Excel ผู้ใช้เดียวกันควรต้องเข้าสู่ระบบทั้ง 365 Dynamics สำหรับการดำเนินงานและ add-in ที่ Microsoft Dynamics Office ข้อมูลเชื่อมต่อ

## <a name="add-a-header-to-budget-plan-document-template"></a>เพิ่มหัวข้อลงในแม่แบบเอกสารของแผนงบประมาณ
เมื่อต้องการเพิ่มข้อมูลหัวข้อ เลือกแถวบนสุดในแฟ้ม Excel และแทรกแถวที่ว่าง คลิ**ออกแบบ**ในการ**การเชื่อมต่อข้อมูล**เมื่อต้องการเพิ่มเขตข้อมูลหัวข้อไปยังแฟ้ม Excel

[![bpt5](./media/bpt5-1024x615.png)](./media/bpt5.png) 

ในการ**ออกแบบ**แท็บ, ** ** คลิ**เพิ่ม**เขตข้อมูลจากนั้นเลือก**BudgetPlanHeader**เป็นแหล่งข้อมูลของเอนทิตี

[![bpt6](./media/bpt6-1024x615.png)](./media/bpt6.png)

ชี้เคอร์เซอร์ไปยังตำแหน่งที่ต้องการในแฟ้ม Excel คลิ**เพิ่มป้าย**เมื่อต้องการเพิ่มป้ายชื่อเขตข้อมูลไปยังตำแหน่งที่เลือก เลือก**เพิ่มค่า**เมื่อต้องการเพิ่มฟิลด์ค่าไปยังตำแหน่งที่เลือก คลิ**ทำ**เมื่อต้องการปิดตัวออกแบบ

<a name="add-a-calculated-column-to-budget-plan-document-template-table"></a>เพิ่มคอลัมน์จากการคำนวณลงในตารางต้นแบบของเอกสารแผนงบประมาณ
--------------------------------------------------------------

จะมีเพิ่มคอลัมน์จากการคำนวณ ถัดไปกับแม่แบบเอกสารของแผนงบประมาณที่สร้างขึ้น A**ขอรวม**คอลัมน์ ซึ่งสรุปขอ Q1: คอลัมน์ขอ Q4 และ**ปรับปรุง**คอลัมน์ ซึ่งคำนวณ**ขอรวม**คอลัมน์ ด้วยตัวคูณที่กำหนดไว้ล่วงหน้า

คลิ**ออกแบบ**ในการ**การเชื่อมต่อข้อมูล**เมื่อต้องการเพิ่มคอลัมน์ลงในตาราง คลิ**แก้ไข**กับ**BudgetPlanWorksheet**แหล่งข้อมูลเพื่อเพิ่มคอลัมน์เริ่มต้น

[![bpt8](./media/bpt8-1024x301.png)](./media/bpt8.png) 

กลุ่มฟิลด์ที่เลือกแสดงในคอลัมน์ที่มีอยู่ในแม่แบบ คลิ**สูตร**เมื่อต้องการเพิ่มคอลัมน์ใหม่ ชื่อคอลัมน์ใหม่ และวางสูตรลงใน**สูตร**เขตข้อมูล คลิ**เด**เมื่อต้องการแทรกคอลัมน์

[![bpt12](./media/bpt12-1024x565.png)](./media/bpt12.png)

> [!NOTE] 
> เพื่อกำหนดสูตร สร้างสูตรในกระดาษคำนวณ และจากนั้น คัดลอกไป**ออกแบบ**หน้าต่าง มี 365 Dynamics สำหรับการดำเนินการตารางที่ถูกผูกไว้จะโดยทั่วไปมีการตั้งชื่อ "AXTable1" ตัวอย่างเช่น จะสรุป Q1 ที่ร้องขอ: Q4 ขอคอลัมน์ในกระดาษคำนวณ สูตร = AxTable1\[ขอ Q1\]+ AxTable1\[ขอ Q2\]+ AxTable1\[Q3 ที่ร้องขอ\]+ AxTable1\[Q4 ร้องขอ\]

ทำซ้ำขั้นตอนเหล่านี้เพื่อแทรกใบ**การปรับปรุง**คอลัมน์ ใช้สูตร = AxTable1\[ขอรวม\]\*$1 สำหรับคอลัมน์นี้ของฉัน นี้จะใช้ค่าในเซลล์ I1 และคูณค่าในการ**ขอรวม**คอลัมน์เพื่อคำนวณยอดการปรับปรุง

บันทึก และปิดแฟ้ม Excel กลับไปยัง Dynamics 365 สำหรับการดำเนินงาน และใน**เค้าโครง**คลิ**แม่&gt;อัปโหลด**เพื่ออัปโหลดแม่แบบ Excel ที่บันทึกที่จะใช้สำหรับแผนงบประมาณ 

[![bpt10](./media/bpt10-1024x352.png)](./media/bpt10.png) 

ปิดการ**เค้าโครง**เลื่อน ใน**แผนงบประมาณ**เอกสาร คลิก**แผ่นงาน**เพื่อดู และแก้ไขเอกสารใน Excel หมายเหตุว่า แม่แบบ Excel ที่ปรับปรุงแล้วถูกใช้เพื่อสร้างแผ่นงานแผนงบประมาณนี้ และคอลัมน์จากการคำนวณจะถูกปรับปรุงโดยใช้สูตรที่กำหนดไว้ในขั้นตอนก่อนหน้านี้

## <a name="tips--tricks-for-creating-budget-plan-templates"></a>เคล็ดลับ & สำหรับการสร้างเท็มเพลตแผนงบประมาณ
### <a name="can-i-add-and-use-additional-data-sources-to-a-budget-plan-template"></a>ฉันสามารถเพิ่ม และใช้แหล่งข้อมูลเพิ่มเติมให้กับแม่แบบของแผนงบประมาณ

ใช่ คุณสามารถใช้การ**ออกแบบ**เมนูเพื่อเพิ่มเอนทิตีที่เพิ่มเติมเหมือนกันหรือแผ่นงานอื่น ๆ ในแม่แบบ Excel ได้ ตัวอย่างเช่น คุณสามารถเพิ่มการ**BudgetPlanProposedProject**แหล่งข้อมูลเพื่อสร้าง และรักษารายการของโครงการที่นำเสนอในเวลาเดียวกันเมื่อทำงานกับงบประมาณวางแผนข้อมูลใน Excel ได้ หมายเหตุว่า รวมทั้งแหล่งข้อมูลปริมาตรสูงอาจส่งผลกระทบต่อประสิทธิภาพการทำงานของสมุดงาน Excel 

คุณสามารถใช้การ**ตัวกรอง**ตัวเลือกในการ**ตัวเชื่อมต่อข้อมูล**เมื่อต้องการเพิ่มตัวกรองที่ต้องการไปยังแหล่งข้อมูลเพิ่มเติมได้

### <a name="can-i-hide-the-design-option-in-the-data-connector-for-other-users"></a>ฉันสามารถซ่อนตัวเลือกการออกแบบในการเชื่อมต่อข้อมูลสำหรับผู้ใช้อื่น ๆ ได้อย่างไร

ใช่ เปิด**การเชื่อมต่อข้อมูล**ตัวเลือกเพื่อซ่อน**ออกแบบ**ตัวเลือกจากผู้ใช้รายอื่น

[![bpt13](./media/bpt13-1024x565.png)](./media/bpt13.png)

ขยาย**ตัวเลือกการเชื่อมต่อข้อมูล**และล้าง**การเปิดใช้งานการออกแบบ**กล่องกาเครื่องหมาย นี้จะซ่อนตัว**ออกแบบ**ตัวเลือกจาก**การเชื่อมต่อข้อมูล**

[![bpt14](./media/bpt14-1024x592.png)](./media/bpt14.png)

### <a name="can-i-prevent-users-from-accidently-closing-the-data-connector-while-working-with-data"></a>ฉันสามารถป้องกันผู้ใช้จากโดยบังเอิญปิดการเชื่อมต่อข้อมูลขณะทำงานกับข้อมูลได้อย่างไร

เราขอแนะนำแม่แบบเพื่อป้องกันไม่ให้ผู้ใช้ปิดการล็อค เมื่อต้องการเปิดการล็อก**การเชื่อมต่อข้อมูล**มีลูกศรปรากฏในมุมขวาด้านบน 

[![bpt15](./media/bpt15-1024x285.png)](./media/bpt15.png) 

คลิกลูกศรสำหรับเมนูที่มีเพิ่มเติม เลือก**ล็อค**

### <a name="bpt16mediabpt16-1024x614pngmediabpt16png"></a>[![bpt16](./media/bpt16-1024x614.png)](./media/bpt16.png)

### <a name="can-i-use-other-excel-features-like-cell-formatting-colors-conditional-formatting-and-charts-with-my-budget-plan-templates"></a>ฉันสามารถใช้คุณลักษณะอื่น ๆ ของ Excel เช่นเดียวกับการจัดรูปแบบเซลล์ สี จัดรูปแบบตามเงื่อนไข และแผนภูมิกับแม่แบบแผนงบประมาณของฉันได้อย่างไร

ใช่ ความสามารถของ Excel มาตรฐานส่วนใหญ่จะทำงานในเท็มเพลตแผนงบประมาณ เราแนะนำให้ใช้สีสำหรับผู้ใช้เพื่อแยกความแตกต่างระหว่างคอลัมน์แบบอ่านอย่างเดียว และสามารถแก้ไขได้ สามารถใช้การจัดรูปแบบตามเงื่อนไขเพื่อเน้นพื้นที่ปัญหาของงบประมาณ ผลรวมคอลัมน์สามารถแสดงงานได้อย่างง่ายดาย โดยใช้สูตร Excel มาตรฐานอยู่เหนือตาราง

นอกจากนี้คุณยังสามารถสร้าง และใช้ตารางสาระสำคัญและแผนภูมิสำหรับการแสดงภาพประกอบเพลงข้อมูลงบประมาณและการจัดกลุ่มเพิ่มเติม บน**ข้อมูล**แท็บ ในการ**เชื่อมต่อ**จัดกลุ่ม คลิ**ฟื้นฟูทั้งหมด**แล้ว คลิก**คุณสมบัติการเชื่อมต่อ** คลิก**การใช้**แท็บ ภายใต้**รีเฟรช**เลือก**ฟื้นฟูข้อมูลเมื่อเปิดแฟ้ม**กล่องกาเครื่องหมาย
