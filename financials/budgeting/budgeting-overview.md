# <a name="budgeting-home-page"></a>โฮมเพจของการจัดงบประมาณ

หัวข้อนี้แสดงภาพรวมของการจัดทำงบประมาณหน้าที่การใช้งานส่วนประกอบ เครื่องมือการจัดทำงบประมาณ และรายงานความสามารถใน 365 Dynamics Microsoft สำหรับการดำเนินงาน

<a name="components-of-budgeting-functionality"></a>ส่วนประกอบของฟังก์ชันการจัดทำงบประมาณ
-------------------------------------

วงจรการวางแผนทรัพยากรสำหรับบริษัทที่โดยทั่วไปจะประกอบด้วยการวางแผน การจัดงบประมาณ และกิจกรรมการคาดการณ์ 
[![Budgeting functionality components](./media/budgeting-functionality-components.jpg)](./media/budgeting-functionality-components.jpg) The processes for both long-term strategic planning and annual budget planning are supported through a budget plan document. เอกสารแผนงบประมาณรวมเข้ากับ Microsoft Excel อย่างเข้มงวด ผู้ใช้สามารถตั้งค่าคอนฟิกสถานการณ์เชิงปริมาณและยอดเงินที่ไม่จำกัด และยังสามารถกำหนดลำดับชั้นขององค์กรการจัดทำงบประมาณ เพื่อสนับสนุนวิธีการจัดทำงบประมาณทั้งแบบบนลงล่าง และล่างขึ้นบน หลังจากงบประมาณถูกสร้าง และอนุมัติใน Microsoft Dynamics 365 for Operations คุณสามารถแปลงแผนงบประมาณเพื่อรายการทะเบียนงบประมาณ รายการทะเบียนงบประมาณให้เครื่องมือ สำหรับการรักษางบประมาณ และ การเก็บยอดเงินที่สามารถติดตามโดยใช้รหัสงบประมาณได้ รายการทะเบียนงบประมาณให้คุณตรวจทานงบประมาณเดิม ดำเนินการโอนย้าย และยกยอดงบประมาณจากปีก่อนหน้านี้ ตามงบประมาณสร้างไว้แล้ว บริษัทสามารถเปิดใช้งานการควบคุมงบประมาณ ระดับของการควบคุมขึ้นอยู่กับวัฒนธรรมขององค์กรและระดับขององค์กรที่ครบกำหนด องค์กรที่มีวันครบกำหนดต่ำอาจปล่อยให้งบประมาณ เป็น "ตามที่เป็นอยู่" และอาจเป็นแบบปฏิกิริยามากกว่าแบบเชิงรุก ถ้างบประมาณไม่ตรงกับความคาดหวังไว้ องค์กรอื่น ๆ อาจเปิดใช้งานนโยบายการควบคุมงบประมาณที่ป้องกันไม่ให้ผู้ใช้จากการซื้อถ้าเงินงบประมาณไม่มีอยู่ ในตอนท้าย องค์กรที่มีการเติบโตมากๆอาจสร้างวัฒนธรรมขององค์กรที่พนักงานได้รับการศึกษาเกี่ยวกับเป้าหมายขององค์กร และทำตามเป้าหมายดังกล่าวผ่านนโยบาย เช่น "ให้พิจารณาการประชุมแบบออนไลน์แทนการเดินทาง"  Dynamics 365 สำหรับการดำเนินงานรวมเป็นกรอบงานการควบคุมงบประมาณที่ช่วยให้การจัดการของบริษัทที่เลือกอย่างใดอย่างหนึ่งตัวควบคุมฮาร์ดดิสก์ (ซึ่งทำให้ไม่สามารถลงรายการบัญชีที่จะไปเหนืองบประมาณ) หรือตัวควบคุมแบบผ่อนปรน (ซึ่งผู้ใช้จะได้รับการเตือนว่า พวกเขาจะเกินเงินงบประมาณที่มีอยู่ แต่สามารถตัดสินใจสำหรับตัวเองวิธีการดำเนินการ) ในตอนท้าย คุณสามารถใช้ย้อนการคาดการณ์ การคาดการณ์ rolling คือ การเปรียบเทียบงบประมาณกับค่าจริงปกติ และถูกใช้เพื่อกำหนดถึงบริษัทดำเนินงานโดยเทียบกับงบประมาณ การคาดการณ์ย้อนหลังยังมีใช้เพื่อระบุแนวโน้ม ใน Microsoft Dynamics 365 for Operations การคาดการณ์ย้อนหลังได้รับการสนับสนุนผ่านเอกสารแผนงบประมาณเป็นกิจกรรมการวางแผนเริ่มต้น การคาดการณ์ย้อนหลังยังสามารถทำได้พร้อมกับการวางแผนสำหรับวงจรงบประมาณที่กำลังจะเกิดขึ้นได้

## <a name="budgeting-tools-in-dynamics-365-for-operations"></a>เครื่องมือการจัดทำงบประมาณใน Dynamics 365 for Operations
[![เครื่องมือการจัดทำงบประมาณ](./media/budgeting-tools.jpg)](./media/budgeting-tools.jpg) 

การวางแผนเพิ่มเติม และความสามารถในการจัดทำงบประมาณจะพร้อมใช้งานทั่วทั้ง Dynamics 365 for Operations และถูกรวมกับงบประมาณบัญชีแยกประเภท

-   **งบประมาณบุคลากร**– บุคลากรที่จัดงบประมาณรวมถึงรายละเอียดส่วนประกอบของต้นทุนงบประมาณที่วางแผนสำหรับตำแหน่ง กลุ่มค่าตอบแทน และอื่น ๆ
-   **งบประมาณสินทรัพย์ถาวร**– ยึดตามข้อมูลสินทรัพย์ถาวร คุณสามารถคำนวณค่าเสื่อมราคาที่วางแผนไว้ และบันทึกธุรกรรมที่วางแผนไว้อื่นๆที่เกี่ยวข้องกับสินทรัพย์ถาวร
-   **งบประมาณโครงการ**– ในโมดูลโครงการ คุณสามารถสร้างการคาดการณ์โครงการโดยละเอียดได้ การคาดการณ์โครงการที่จะรวมรายละเอียดเกี่ยวกับชั่วโมง ค่าใช้จ่าย ค่าธรรมเนียม และสินค้าที่วางแผนไว้
-   ** อุปสงค์การคาดการณ์ ** – ตามข้อมูลธุรกรรมในอดีต คุณสามารถประเมินความต้องการสินค้าคงคลังในอนาคต และสร้างการคาดการณ์ความต้องการได้

สำหรับข้อมูลเกี่ยวกับวิธีการนำข้อมูลการวางแผนจากโมดูลอื่นเข้าไปในแผนงบประมาณ ให้ดู [การรวมการวางแผนงบประมาณกับโมดูลอื่น](budget-planning-integration-other-modules.md)

## <a name="user-interface-and-reporting-capabilities"></a>อินเทอร์เฟสผู้ใช้ และความสามารถในการรายงาน
ใน Dynamics 365 for Operations ผู้ใช้สามารถสร้างแผนงบประมาณทั้งในไคลเอนต์ Dynamics 365 for Operations โดยตรง (โดยใช้หน้าเอกสารแผนงบประมาณสามารถจัดโครงแบบ) หรือ ผ่าน Excel ได้ Excel มีความสามารถในการเพิ่มเติมต่าง ๆ ตัวอย่างเช่น คุณสามารถใช้ข้อมูลภายนอกเป็นแหล่งข้อมูลสำหรับแผนงบประมาณ ทำการคำนวณที่กำหนดเอง และใช้ Microsoft PivotTable และแผนภูมิ ค่าส่วนใหญ่ของตัวแปรในกระบวนการวางแผนงบประมาณสามารถกำหนดได้ ตัวอย่างเช่น คุณสามารถกำหนดว่าใครจัดทำงบประมาณ การถูกจัดทำงบประมาณคืออะไร และกระบวนการที่คล้ายกัน แม้ว่าคุณสามารถใช้ Excel สำหรับการวางแผนงบประมาณ Dynamics 365 for Operations สามารถถูกเก็บเป็นแหล่งเดียวของความถูกต้องและช่วยป้องกันปัญหาการควบคุมงบประมาณ กระบวนสามารถถูกใช้เพื่อนำเข้าข้อมูลเริ่มต้นสำหรับการจัดงบประมาณเข้ามาไว้ในแผนงบประมาณ สำหรับการรายงาน Dynamics 365 for Operations เสนอชุดของหน้าที่การสอบถามมาตรฐานซึ่งช่วยให้คุณสามารถดู และวิเคราะห์ข้อมูลที่จัดทำงบประมาณ ข้อมูลการจัดทำงบประมาณได้ ข้อมูลแผนงบประมาณสามารถเข้าถึงได้ผ่านทางการรายงานการจัดการโปรแกรม และสถานการณ์จำลองของแผนงบประมาณที่แยกต่างหากสามารถถูกแสดงเป็นคอลัมน์ในรายงาน Management Reporter
