# <a name="consolidation-account-groups-and-additional-consolidation-accounts"></a>กลุ่มบัญชีการรวมบัญชีและบัญชีการรวมบัญชีเพิ่มเติม

หัวข้อนี้แสดงข้อมูลเกี่ยวกับกลุ่มบัญชีการรวมบัญชีและบัญชีการรวมบัญชีเพิ่มเติม และอธิบายถึงวิธีการใช้ใน Microsoft Dynamics 365 สำหรับการดำเนินงาน

<a name="consolidation-account-groups"></a>กลุ่มบัญชีการรวมบัญชี
----------------------------

กลุ่มบัญชีการรวมช่วยให้คุณสร้างกลุ่มของบัญชีที่คุณต้องการใช้ในการรวมข้อมูล บ่อยที่สุด กลุ่มบัญชีการรวมบัญชีภายใต้ควบคุมจากรัฐบาลผังบัญชีแสดง หรือแม็ปบัญชีกลุ่มที่ถูกกำหนด โดย headquarters ของบริษัท คุณสามารถค้นหารวมกลุ่มบัญชีในการ**ตั้งค่า**พื้นที่ของ**รวม**โมดูลได้ เมื่อคุณเพิ่มกลุ่มใหม่ คุณสามารถป้อนรหัสเฉพาะสำหรับกลุ่มบัญชีและชื่อ

## <a name="additional-consolidation-accounts"></a>บัญชีการรวมบัญชีเพิ่มเติม
บัญชีการรวมบัญชีเพิ่มเติมช่วยให้คุณกำหนดบัญชีจากผังบัญชีที่มีอยู่ไปยังกลุ่มบัญชีการรวมบัญชี จากนั้นคุณสามารถระบุค่าบัญชีการรวมบัญชีและชื่ออยู่ 

คุณสามารถค้นหาบัญชีการรวมบัญชีเพิ่มเติมในการ**ตั้งค่า**พื้นที่ของการ**รวม**โมดูลได้ เมื่อคุณสร้างการรวมบัญชีใหม่ คุณต้องระบุข้อมูลต่อไปนี้:

-   **บัญชีหลัก**– ฟิลด์นี้เป็นการค้นหาที่แสดงบัญชีหลักทั้งหมดที่ยึดตามผังบัญชีที่คุณเลือกบนหน้า นั้น เมื่อคุณเลือกบัญชีผู้ใช้ ชื่อที่จะถูกป้อนโดยอัตโนมัติในการ**ชื่อบัญชีหลัก**เขตข้อมูล
-   **กลุ่มบัญชีการรวมบัญชี**– ใช้ฟิลด์นี้เพื่อระบุกลุ่มเพื่อกำหนดบัญชีที่จะ ถ้าคุณรวมสองวิธีที่แตกต่างกัน คุณต้องเพิ่มบัญชีเดียวกันกับกลุ่มบัญชีการรวมบัญชีสี่ทั้งหมด
-   **บัญชีการรวมบัญชี**– ป้อนค่าของบัญชีการรวมบัญชี ค่านี้ไม่จำเป็นต้องเป็นบัญชีจากผังบัญชี อาจเป็นค่าใด ๆ ที่คุณต้องการ
-   **ชื่อบัญชีการรวมบัญชี**– ป้อนชื่อของบัญชี ตามที่คุณต้องการให้ปรากฏบนการสอบถามและรายงาน
-   **ระดับ SAT** – ฟิลด์นี้ถูกใช้ในการรายงานใบแจ้งยอดบัญชีไปยังหน่วยงานภาษีของเม็กซิโก 

เมื่อคุณเสร็จสิ้นการสร้างกลุ่มบัญชีการรวมบัญชีและบัญชีการรวมบัญชีเพิ่มเติมของคุณ คุณสามารถเลือกกลุ่มในกระบวนการรวมบัญชีแบบออนไลน์
