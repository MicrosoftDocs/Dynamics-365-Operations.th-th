# <a name="budget-planning"></a>การวางแผนงบประมาณ

วัตถุประสงค์ของการทดสอบในแล็บนี้คือเพื่อ ให้ดูคำแนะนำของ Microsoft Dynamics 365 สำหรับปรับปรุงฟังก์ชันการทำงานของการดำเนินงานในพื้นที่การวางแผนงบประมาณ จุดประสงค์ของห้องปฏิบัติการนี้คือแสดงตัวอย่างการตั้งค่าคอนฟิกด่วนของโมดูลการวางแผนงบประมาณ และแสดงวิธีวางแผนงบประมาณให้สำเร็จโดยใช้การตั้งค่าคอนฟิกนี้  ทดสอบในแล็บนี้จะเน้นเฉพาะในกระบวนการทางธุรกิจหรืองาน - ต่อไปนี้สร้างลำดับชั้นขององค์กรสำหรับงบประมาณที่วางแผน และการตั้งค่าคอนฟิกความปลอดภัยของผู้ใช้ - การกำหนดสถานการณ์จำลองแผนงบประมาณ คอลัมน์แผนงบประมาณ โครงร่าง และแม่ แบบ Excel - การสร้าง และเรียกใช้ข้อมูลงบประมาณวางแผนกระบวนการ -การสร้างเอกสารแผนงบประมาณ โดย pulling ในที่เกิดขึ้นจริงจากบัญชีแยกประเภททั่วไป - ใช้การปันส่วนการปรับปรุงข้อมูลเอกสารแผนงบประมาณ - การแก้ไขงบประมาณแผนเอกสารใน Excel 

<a name="prerequisites"></a>ข้อกำหนดเบื้องต้น 
------------------

สำหรับบทช่วยสอนนี้ คุณจะต้องเข้าถึง 365 Dynamics สำหรับสภาพแวดล้อมของการดำเนินงานที่มีข้อมูลสาธิต Contoso และสามารถเตรียมใช้งานในฐานะผู้ดูแลบนอินสแตนซ์ ใช้โหมดเบราว์เซอร์ในส่วนตัวสำหรับห้องแล็บนี้ - ลงชื่อออกจากบัญชีอื่น ๆ ในเบราว์เซอร์ถ้าจำเป็น และเข้าสู่ระบบ ด้วย Dynamics 365 สำหรับข้อมูลประจำตัวของผู้ดูแลการดำเนินงาน ขณะล็อกเข้าใน 365 Dynamics สำหรับการดำเนินงาน คุณ**ต้อง**กาเครื่องหมายที่ "ให้ฉันเข้าสู่ระบบ" วิธีนี้สร้างคุกกี้ถาวรที่แอพลิเคชัน Excel ต้องการ ณ ขณะนี้ ถ้าคุณเข้าสู่ใน 365 Dynamics สำหรับการดำเนินงานโดยใช้เบราว์เซอร์อื่นที่ไม่ใช่ IE แล้วคุณจะได้รับพร้อมท์การเข้าสู่ระบบภายในโปรแกรมประยุกต์ Excel เมื่อคุณคลิก "ลงชื่อเข้าใช้" ในแอพลิเคชัน Excel หน้าต่างป๊อปอัพ IE จะเปิดและ ขณะคุณกำลังลงชื่อคุณ **ต้อง** กาเครื่องหมายที่กล่องกาเครื่องหมาย "ให้ฉันลงชื่อเข้าใช้" ถ้าคลิก "ลงชื่อข้าใช้" ในแอพลิเคชัน Excel ไม่ดำเนินการใดๆ คุณควรล้างแคชคุกกี้ IE

## <a name="scenario-overview"></a>**Scenario overview**
จูเลียทำงานเป็นผู้จัดการฝ่ายการเงินในระบบความบันเทิง Contoso ในเยอรมนี (DEMF) เมื่อ FY2016 ใกล้ถึง เธอจำเป็นต้องทำงานในการตั้งค่างบประมาณของบริษัทสำหรับปีกำลังมาถึง การจัดเตรียมงบประมาณมีลักษณะดังนี้:

1.  Julia ใช้ยอดเงินที่เกิดขึ้นจริงของปีก่อนหน้านี้เป็นจุดเริ่มต้นเพื่อสร้างงบประมาณ
2.  ตามยอดจริงในปีก่อนหน้า เธอสร้างการประเมิน 12 เดือน สำหรับปีกำลังมาถึง
3.  Julia ตรวจทานงบประมาณที่ ด้วย CFO เมื่อเสร็จเรียบร้อย เธอทำการปรับปรุงเท่าที่จำเป็นสำหรับแผนงบประมาณ และ ดำเนินการจัดเตรียมงบประมาณประจำปีขั้นสุดท้าย

-   บัญชีแยกประเภท– DEMF
-   โครงสร้างทางบัญชีเริ่มต้น – Manufacturing P&L
