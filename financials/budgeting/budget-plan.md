# <a name="budget-planning"></a>การวางแผนงบประมาณ

วัตถุประสงค์ของห้องปฏิบัติการนี้คือแสดงมุมมองที่แนะนำของการอัพเดตฟังก์ชัน Microsoft Dynamics 365 for Operations ในพื้นที่การวางแผนงบประมาณ จุดประสงค์ของห้องปฏิบัติการนี้คือแสดงตัวอย่างการตั้งค่าคอนฟิกด่วนของโมดูลการวางแผนงบประมาณ และแสดงวิธีวางแผนงบประมาณให้สำเร็จโดยใช้การตั้งค่าคอนฟิกนี้  ห้องปฏิบัติการนี้จะเน้นเฉพาะกระบวนการทางธุรกิจหรืองานต่อไปนี้ -    - การสร้างลำดับชั้นขององค์กรสำหรับงบประมาณที่วางแผน และการตั้งค่าคอนฟิกความปลอดภัยผู้ใช้   -  กำหนดสถานการณ์จำลองแผนงบประมาณ คอลัมน์แผนงบประมาณ โครงร่าง และแม่แบบ Excel   -  สร้างและเปิดใช้งานกระบวนการวางแผนงบประมาณ   - สร้างเอกสารแผนงบประมาณโดยดึงจากบัญชีแยกประเภททั่วไป   -  ใช้การปันส่วนการปรับปรุงข้อมูลเอกสารแผนงบประมาณ   - การแก้ไขงบประมาณข้อมูลแผนงบประมาณเอกสารใน Excel 

<a name="prerequisites"></a>ข้อกำหนดเบื้องต้น 
------------------

สำหรับบทสอนนี้ คุณจะต้องเข้าถึงสภาพแวดล้อม Dynamics 365 for Operations ด้วยข้อมูลสาธิต Contoso และเตรียมใช้งานในฐานะผู้ดูแลบนอินสแตนซ์ อย่าใช้โหมดเบราเซอร์ส่วนตัวสำหรับห้องปฏิบัติการนี้ - ลงชื่อออกจากบัญชีอื่น ๆ ในเบราเซอร์ถ้าจำเป็น และลงชื่อเข้าใช้ ด้วยข้อมูลประจำตัวของผู้ดูแลระบบ Dynamics 365 for Operations เมื่อลงชื่อเข้าใช้ Dynamics 365 for Operations คุณ **ต้อง** กาเครื่องหมายที่ "ให้ฉันลงชื่อเข้าใช้" กล่องกาเครื่องหมาย วิธีนี้สร้างคุกกี้ถาวรที่แอพลิเคชัน Excel ต้องการ ณ ขณะนี้ ถ้าคุณเข้าสู่ระบบ Dynamics 365 for Operations โดยใช้เบราเซอร์อื่นนอกเหนือจาก IE แล้วคุณก็จะพร้อมเข้าสู่ระบบภายในแอพลิเคชัน Excel เมื่อคุณคลิก "ลงชื่อเข้าใช้" ในแอพลิเคชัน Excel หน้าต่างป๊อปอัพ IE จะเปิดและ ขณะคุณกำลังลงชื่อคุณ **ต้อง** กาเครื่องหมายที่กล่องกาเครื่องหมาย "ให้ฉันลงชื่อเข้าใช้" ถ้าคลิก "ลงชื่อข้าใช้" ในแอพลิเคชัน Excel ไม่ดำเนินการใดๆ คุณควรล้างแคชคุกกี้ IE

## <a name="scenario-overview"></a>**ภาพรวมของสถานการณ์จำลอง**
จูเลียทำงานเป็นผู้จัดการฝ่ายการเงินในระบบความบันเทิง Contoso ในเยอรมนี (DEMF) เมื่อ FY2016 ใกล้ถึง เธอจำเป็นต้องทำงานในการตั้งค่างบประมาณของบริษัทสำหรับปีกำลังมาถึง การจัดเตรียมงบประมาณมีลักษณะดังนี้:

1.  Julia ใช้ยอดเงินที่เกิดขึ้นจริงของปีก่อนหน้านี้เป็นจุดเริ่มต้นเพื่อสร้างงบประมาณ
2.  ตามยอดจริงในปีก่อนหน้า เธอสร้างการประเมิน 12 เดือน สำหรับปีกำลังมาถึง
3.  Julia ตรวจทานงบประมาณที่ ด้วย CFO เมื่อเสร็จเรียบร้อย เธอทำการปรับปรุงเท่าที่จำเป็นสำหรับแผนงบประมาณ และ ดำเนินการจัดเตรียมงบประมาณประจำปีขั้นสุดท้าย

## <a name="task-2-configure-user-security"></a>งานที่ 2: ตั้งค่าคอนฟิกความปลอดภัยของผู้ใช้

## <a name="task-3-create-scenarios"></a>งานที่ 3: สร้างสถานการณ์จำลอง

## <a name="task-6-create-a-budget-planning-process"></a>งานที่ 6: สร้างกระบวนการวางแผนงบประมาณ
Julia จำเป็นต้องสร้าง และเรียกใช้กระบวนการวางแผนงบประมาณใหม่รวมกับการตั้งค่าข้างต้นเพื่อเริ่มป้อนการวางแผนงบประมาณใหม่ กระบวนการวางแผนงบประมาณกำหนดจัดทำงบประมาณองค์กร ลำดับงาน โครงร่าง และเท็มเพจจะใช้สำหรับการสร้างแผนงบประมาณ 6.1 นำไปยังการจัดทำงบประมาณ &gt; การตั้งค่า &gt; การวางแผนงบประมาณ &gt; กระบวนการวางแผนงบประมาณ และสร้างเรกคอร์ดใหม่

-   บัญชีแยกประเภท– DEMF
-   โครงสร้างทางบัญชีเริ่มต้น – Manufacturing P&L
