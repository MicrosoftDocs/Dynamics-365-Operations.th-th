# <a name="financial-period-close-workspace"></a>พื้นที่ทำงานการปิดรอบระยะเวลาทางการเงิน

บทความนี้แสดงภาพรวมของพื้นที่ทำงานการปิดรอบระยะเวลาทางการเงินและการตั้งค่าคอนฟิกที่เกี่ยวข้อง

พื้นที่ทำงานการปิดรอบระยะเวลาทางการเงิน

การ**รอบระยะเวลาทางการเงินที่ปิด**พื้นที่ทำงานช่วยให้คุณติดตามกระบวนการปิดบัญชีทางการเงินของคุณทั่วทั้งบริษัท พื้นที่ และบุคคล ขึ้นอยู่กับมุมมองของคุณ**รอบระยะเวลาทางการเงินที่ปิด**ในพื้นที่ทำงานคุณจะเห็นงานและสถานะสำหรับกำหนดการปิด หรือเพียงแค่ภารกิจที่กำหนดให้กับคุณทั้งหมดอย่างใดอย่างหนึ่ง 

คุณต้องเลือกตารางเวลาปิดที่ด้านบนของพื้นที่ทำงาน ข้อมูลทั้งหมดที่แสดงอยู่บนพื้นที่ทำงานมีการกรองข้อมูลแล้วตามกำหนดเวลาปิดบัญชีที่เลือก

### <a name="summary-tiles"></a>สรุปไทล์


          **สรุป** ไทล์แสดงภาพรวมของกระบวนการ และตัวบ่งชี้ช่วยให้คุณรักษากระบวนการปิดบัญชีให้เป็นตามที่คาดไว้ คุณสามารถดูงานที่เลยกำหนดชำระ งานที่เหลือสำหรับวันนี้ งานที่ครบกำหนดชำระในวันนี้ แต่จะถูกบล็อกเนื่องจาก มีการอ้างอิง และงานที่เหลือทั้งหมดสำหรับกระบวนการ ข้อมูลนี้สำหรับบริษัททั้งหมดที่รวมอยู่ในการจัดกำหนดการการปิดอยู่

### <a name="tasks-and-status-section"></a>ส่วนงานและสถานะ

ในการ**งานและสถานะ**ส่วน สถานะของโดยรวมกำหนดการปิดจะแบ่งในหลายวิธี: สถานะ โดยบริษัท สถานะตามพื้นที่ และสถานะ โดยบุคคลที่รับผิดชอบ คุณสามารถดูสถานะสำหรับงานทั้งหมดในการปิดกำหนดเวลา งานเพียงอย่างเดียวที่ครบกำหนดชำระในวันนี้ หรืองานที่เลยกำหนดชำระ โดยการเปลี่ยนตัวกรองที่ด้านบนของรายการบัตร คุณยังสามารถเลือกตัวกรองเพื่อดูสถานะสำหรับบริษัทเฉพาะบริษัท แต่ละแท็บสถานะแสดงการแบ่งรายละเอียดตามเปอร์เซ็นต์ที่เสร็จสมบูรณ์และจำนวนงานที่เหลืออยู่ทั้ง คลิกบัตรหรือ**ดูรายละเอียด**การกระทำเมื่อต้องการกรองรายการงานโดยละเอียดตามการ์ดที่เลือก 

แท็บสุดท้ายสำหรับรายการโดยละเอียดของงานได้ รายการนี้แสดงรายการงานทั้งหมด และคุณสามารถกรองข้อมูลเพื่อให้แสดงเฉพาะงานที่คุณสนใจ คุณสามารถกรองรายการงานในหลายวิธี ตัวอย่างเช่น คุณสามารถกรอง โดยงานวันครบกำหนด บริษัทที่เกี่ยวข้อง และพื้นที่ที่เกี่ยวข้อง คุณยังสามารถเลือกที่จะแสดง หรือซ่อนงานในรายการงานที่เสร็จสมบูรณ์ 

ตัวบ่งชี้สองตัวจะถูกใช้สำหรับงาน:

-   ไอคอนเครื่องหมายอัศเจรีย์บ่งชี้ว่า งานที่เลยวันครบกำหนด สำหรับงานที่พ้นกำหนด วันครบกำหนดนอกจากนี้ยังเน้นในสีแดง
-   ไอคอนแม่กุญแจบ่งชี้ว่า งานขึ้นอยู่กับงานอื่น ๆ ที่ยังไม่เสร็จสมบูรณ์ งานที่ถูกบล็อค โดยการอ้างอิงที่ไม่ถูกทำเครื่องหมายเป็นเสร็จสมบูรณ์แล้ว คุณสามารถตั้งค่าการขึ้นต่อกันสำหรับงาน โดยใช้การ**การตั้งค่าการอ้างอิง**ดำเนินการได้

ชื่องานจะเชื่อมโยงไป 365 Dynamics Microsoft สำหรับหน้าของการดำเนินงานหรือเว็บเพจอื่น ๆ ที่ผู้ใช้ต้องไปทำงาน คุณสามารถตั้งค่านี้การเชื่อมโยงหลายมิติ โดยใช้การ**การเชื่อมโยงงาน**เมื่อคุณแก้ไข หรือสร้างงานได้ 

คุณสามารถแนบแฟ้ม บันทึกย่อ รูปภาพ และ Url ให้เป็นงาน โดยใช้การ**สิ่งที่แนบมา**ดำเนินการได้ ตัวอย่างเช่น คุณสามารถระบุหมายเลขสมุดรายวันที่จะถูกใช้เป็นส่วนหนึ่งของงาน เพิ่มข้อคิดเห็นเกี่ยวกับงานที่เฉพาะเจาะจง หรือแนบแฟ้มรายงานที่พิมพ์สำหรับงานนั้น ไอคอนปรากฏขึ้นในการ**สิ่งที่แนบมา**คอลัมน์สำหรับงานถ้ามีสิ่งที่แนบมา 

การ**ทำงาน**ตัวเลือกรายการต้องถูกเลือกด้วยตนเองหลังจากงานเสร็จสมบูรณ์ เมื่องานถูกทำเครื่องหมายเป็นเสร็จสมบูรณ์แล้ว**วันเสร็จสมบูรณ์**ฟิลด์มีการปรับปรุงโดยอัตโนมัติให้เป็นวันปัจจุบันและเวลา ตัวบ่งชี้การอ้างอิงจะถูกปรับปรุงตามความเหมาะสม

## <a name="all-financial-period-close-tasks-list-page"></a>หน้ารายการงานการปิดรอบระยะเวลาทางการเงินทั้งหมด
คุณสามารถดูงานทั้งหมดในปัจจุบัน และก่อนหน้ารอบระยะเวลาปิดจาก**งานปิดรอบระยะเวลาทางการเงินทั้งหมด**หน้ารายการได้ หน้ารายการนี้เหมาะที่จะใช้สำหรับการวิเคราะห์ในอดีตของกระบวนการปิดบัญชีของคุณ เนื่องจากมีข้อมูลเกี่ยวกับการจัดกำหนดการเป็นวันครบกำหนด วันเสร็จสมบูรณ์ที่เกิดขึ้นจริง และบุคคลที่ทำงานให้เสร็จสมบูรณ์ คุณสามารถส่งข้อมูลบนหน้ารายการนี้ไปยัง Microsoft Excel ได้อย่างง่ายดายสำหรับการรายงาน และเพื่อวัตถุประสงค์ในการตรวจสอบ

## <a name="financial-period-close-configuration-page"></a>หน้าการตั้งค่าคอนฟิกการปิดรอบระยะเวลาทางการเงิน
ก่อนที่คุณสามารถใช้**รอบระยะเวลาทางการเงินที่ปิด**พื้นที่ทำงาน คุณต้องกำหนดค่ากระบวนการใน 365 Dynamics Microsoft สำหรับการดำเนินงาน โดยใช้การ**โครงการปิดรอบระยะเวลาทางการเงิน**หน้า (คลิก**บัญชีแยกประเภททั่วไป**&gt;**ปิดรอบระยะเวลา**&gt;**ฟิกปิดรอบระยะเวลาทางการเงิน**)

### <a name="resources"></a>ทรัพยากร

ในการ**ทรัพยากร**แท็บ คุณกำหนดบุคคลที่เกี่ยวข้องกับกระบวนการปิดบัญชี พนักงานที่จะรับผิดชอบงานการปิดบัญชีต้องกำหนดให้ก่อนที่นี่ คุณต้องระบุมุมมองของพนักงานของพื้นที่ทำงานด้วย มีตัวเลือกดังต่อไปนี้

-   **เฉพาะงานที่มอบหมาย**– ผู้ใช้จะเห็นเฉพาะงานที่ถูกกำหนดให้กับเขาหรือเธอ
-   **งานทั้งหมดและสถานะ**– ผู้ใช้จะเห็นงานการปิดบัญชีทั้งหมดและสถานะของกระบวนการโดยรวม

ผู้ใช้ที่มีสิทธิ์ในการดูเฉพาะงานที่ได้รับมอบหมายของพวกเขา จะไม่สามารถเพิ่มงานลงในรายการงาน แก้ไขงาน หรือลบงานออกจากรายการงานได้

### <a name="task-areas"></a>พื้นที่งาน

คุณใช้พื้นที่งานเพื่อจัดกลุ่มงานการปิดบัญชี เป็นพื้นที่ทางตรรกะของความเป็นเจ้าของภายในองค์กรของคุณ ตัวอย่างเช่น บัญชีเจ้าหนี้ บัญชีแยกประเภทลูกหนี้ หรือบัญชีแยกประเภททั่วไปอาจถูกใช้เป็นพื้นที่งาน

### <a name="calendars"></a>ปฏิทิน

สร้าง และแก้ไขปฏิทินปิดทางการเงินโดยใช้แท็บปฏิทิน  นี่คือสถานที่คุณจะกำหนดวันทำงานสำหรับกระบวนการการปิดบัญชี และจะถูกใช้สำหรับการจัดตารางการผลิตระดับงานของการปิดบัญชี  สร้างปฏิทินใหม่ และระบุวันทำงานที่จะใช้สำหรับการจัดตารางการผลิตระดับงาน  เป็นการดีที่สุดในการสร้างปฏิทินสำหรับรอบระยะเวลา เช่นหนึ่งปีหรือหลายปี ยาวเนื่องจากจะสามารถแก้ไขได้หลังจากการสร้าง  หลังจากสร้างปฏิทิน คลิกปุ่มแก้ไขในการปรับปรุงปฏิทินสำหรับวันที่ระบุ เช่นวันหยุด  การปิดงานจะถูกจัดกำหนดการในวันที่เมื่อค่าของตัวควบคุมถูกตั้งค่าเป็นเปิด  ถ้าการปิดงานไม่ควรกำหนดการในวันระบุ วันที่ควรมีค่าของตัวควบคุมตั้งค่าเป็นปิด

### <a name="templates"></a>เท็มเพลต

คุณสามารถใช้เท็มเพลปิดทางการเงินเพื่อกำหนดงานทั้งหมดที่เป็นส่วนหนึ่งของกระบวนการปิดบัญชี งานการปิดบัญชีมีความพยายามทำงานเกิดซ้ำที่กำหนดให้แก่บุคคลเพื่อให้เสร็จสมบูรณ์เป็นส่วนหนึ่งของแต่ละกระบวนการปิดบัญชี ในแม่แบบ ครบกำหนดสัมพันธ์กันต้องมีกำหนดวันที่สำหรับแต่ละงานการปิดบัญชี ครบกำหนดสัมพันธ์วันคือ จำนวนวันก่อน หรือหลังจากวันสิ้นสุดรอบระยะเวลากำหนดไว้ที่ งานจะครบกำหนดรอบระยะเวลาแต่ละครั้ง เวลาครบกำหนดนอกจากนี้ยังได้รับการกำหนดให้กับแต่ละงาน เวลาครบกำหนดถูกตั้งค่า โดยใช้บริบทของโซนเวลาของคุณ และจะถูกแปลงเป็นโซนเวลาของผู้ใช้แต่ละ 

คุณสามารถมอบหมายงานในแม่แบบกับบริษัทอย่าง น้อยหนึ่งรายการที่ใช้งานนั้น ถ้าบุคคลที่แตกต่างกันถูกกำหนดให้กับดำเนินการที่พยายามทำงานในแต่ละบริษัท คุณอาจพบว่ามีประโยชน์เมื่อต้องการสร้างงานหลายสำหรับความพยายามทำงานเดียวกัน สร้างงานหนึ่งงานสำหรับบริษัทแต่ละแห่ง 

การ**การเชื่อมโยงงาน**รายการเมนูที่เกี่ยวข้องกับความพยายามทำงาน และสามารถใช้ตรงไปหน้าที่เกี่ยวข้องจากการเชื่อมโยงในพื้นที่ทำงาน ตัวอย่างเช่น คุณสามารถเชื่อมโยงงานปิดเพื่อรันกระบวนการประเมินค่าใหม่ตามสกุลเงินสำหรับบัญชีเจ้าหนี้เกี่ยวข้องกัน**การประเมินค่าใหม่ตามสกุลเงินต่างประเทศ**เพจใน 365 Dynamics Microsoft สำหรับการดำเนินงานได้ คุณยังสามารถลิงค์ไปยัง URL ภายนอกได้อีกด้วย 

> [! คำแนะนำ] ถ้าคุณต้องการเชื่อมโยงโปรแกรมรายงานการจัดการรายงานเฉพาะงานปิดรอบระยะเวลาทางการเงิน คุณสามารถใช้ URL ของรายงานได้ เมื่อต้องการเข้าถึง URL รายงาน เปิดรายงานในตัวออกแบบรายงาน และจากนั้น คลิก**แฟ้ม**&gt;**ดูรายงาน**เมื่อต้องการเปิดรายงานในเว็บเบราว์เซอร์ จากนั้นคุณสามารถคัดลอก URL ในแถบที่อยู่ของเบราเซอร์ และวางลงในฟิลด์ **ลิงค์งาน** **URL** ได้ 

คุณสามารถกำหนดความสัมพันธ์ของงานในแม่แบบ ถ้างานมีการตั้งค่าจะขึ้นอยู่กับงานอย่าง น้อยหนึ่ง งานนั้นไม่สามารถทำเครื่องหมายเป็นเสร็จสมบูรณ์จนกว่ามีการอ้างอิงทั้งหมดเสร็จเรียบร้อย 

คุณสามารถสร้างแม่แบบปิดทางการเงินหลายครั้ง จากนั้นคุณสามารถใช้แบบต่าง ๆ เพื่อติดตามกระบวนการปิดบัญชีสำหรับรอบระยะเวลาชนิดต่าง ๆ เช่นสิ้นเดือนหรือสิ้นปี หรือติดตามบริษัทที่ใช้ในกระบวนการปิดบัญชีที่แตกต่างกัน หลังจากที่สร้างเท็มเพลตหนึ่ง คุณสามารถคัดลอกไปยังแม่แบบใหม่ และทำการเปลี่ยนแปลงที่จำเป็น คุณสามารถกำหนดแม่แบบเพียงหนึ่งลงในแต่ละตารางเวลาปิด

### <a name="closing-schedules"></a>กำหนดการในการปิด

คุณสามารถใช้กำหนดการปิดการกำหนดแม่แบบปิดทางการเงินเพื่อระบุรอบระยะเวลาทางการเงินที่ต้องถูกปิด งานจากแม่แบบมีแล้วสร้างขึ้นโดยอัตโนมัติสำหรับรอบระยะเวลาที่ระบุ และกำหนดการปิดใหม่ถูกเพิ่มเข้าไปในพื้นที่ทำงาน เมื่อคุณสร้างการจัดกำหนดการปิดบัญชีใหม่ การ**วันสิ้นสุดรอบระยะเวลา**ใช้ฟิลด์เพื่อกำหนดครบกำหนดจริงวันที่สำหรับงานปิด ตามครบกำหนดชำระสัมพัทธ์วันที่ถูกกำหนดในแม่แบบที่มีการปิดทางการเงิน 

กำหนดปฏิทินที่เหมาะสมสำหรับการจัดกำหนดการปิด เพื่อบ่งชี้วันทำงานที่จะใช้ในการจัดตารางการผลิตระดับงาน ถ้าคุณไม่กำหนดเฉพาะปฏิทิน งานครบกำหนดวันจะใช้ทุกวันในสัปดาห์ 

คุณยังต้องกำหนดบริษัทที่จะเชื่อมโยงกับตารางเวลาปิดบัญชี ถ้าแม่แบบงานถูกกำหนดให้กับบริษัทหลายแห่ง จะมีสร้างงานที่แยกต่างหากสำหรับแต่ละบริษัทที่อยู่ในกำหนดการปิด และกำหนดให้กับงานเท็มเพลต 

หลังจากปิดบัญชีกำหนดการเสร็จสมบูรณ์ เลือก**ปิด**ตัวเลือกสำหรับตารางนั้น ประวัติงานจะยังคงมีอยู่จาก**งานปิดรอบระยะเวลาทางการเงินทั้งหมด**หน้ารายการกำหนดการปิด แต่จะถูกเอาออกจากพื้นที่ทำงาน หลังจากที่ถูกกำหนดตารางเวลาการปิดบัญชีเป็น**ปิด**คุณจะไม่สามารถเพิ่มงาน แก้ไขงาน หรือเอางานออก
