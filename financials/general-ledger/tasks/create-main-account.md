# <a name="create-a-main-account"></a>สร้างบัญชีหลัก

[!include[task guide banner](../../includes/task-guide-banner.md)]

คำแนะนำงานนี้ระบุการเพิ่มบัญชีหลักให้กับผังบัญชีที่มีอยู่  การบันทึกข้อมูลนี้ใช้บริษัทสาธิต USMF  

1. ไปที่บัญชีแยกประเภททั่วไป > ผังบัญชี > บัญชี > บัญชีหลัก
2. คลิก สร้าง
3. ในฟิลด์บัญชีหลัก ให้พิมพ์ค่าใดค่าหนึ่ง
4. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
5. ในฟิลด์ชนิดบัญชีหลัก ให้เลือกแสดงชนิดที่เหมาะสมที่สุดในยอดดุลบัญชีและที่ตั้งในงบการเงิน
6. ในรายการ ให้เลือกประเภทบัญชีที่เป็นของบัญชีหลัก 
    * ประเภทบัญชีจะถูกใช้สำหรับการรายงานทางการเงินเริ่มต้นและเนื้อหาแดชบอร์ด Power BI  
7. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
    * เปลี่ยนยอดดุลเดบิตหรือเครดิตเริ่มต้น  
8. ในฟิลด์สกุลเงินเริ่มต้น ให้เลือกค่าจากรายการของสกุลเงิน
9. ในรายการนี้ ให้ค้นหาและเลือกเรกคอร์ดที่ต้องการ
10. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
11. สลับการขยายของการแทนส่วนนิติบุคคล
12. คลิกเพิ่มเพื่อเลือกนิติบุคคล
13. ในรายการ ให้เลือกนิติบุคคล
14. คลิก เพิ่ม
15. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
16. เลือกหรือยกเลิกการเลือกกล่องกาเครื่องหมายระงับแล้ว
17. ขยายแท็บด่วนรายงานทางการเงิน
18. ในฟิลด์ชนิดอัตราแลกเปลี่ยน ให้คลิกปุ่มดรอปดาวน์เพื่อเปิดการค้นหา
19. ในรายการ เลือกชนิดอัตราแลกเปลี่ยนสำหรับบัญชี
20. ในรายการนี้ ให้คลิกลิงค์ในแถวที่เลือก
21. ในฟิลด์ชนิดการแปลงสกุลเงิน ให้เลือกวิธีการคำนวณอัตราแลกเปลี่ยนสำหรับบัญชี
22. ปิดหน้า
