# <a name="view-a-related-intercompany-voucher-from-a-journal"></a>ดูใบสำคัญระหว่างบริษัทที่เกี่ยวข้องจากสมุดรายวัน

[!include[task guide banner](../../includes/task-guide-banner.md)]

หน้าต่างใบสำคัญที่เกี่ยวข้องแสดงใบสำคัญจากบริษัทตรงข้าม เมื่อมีการลงรายการบัญชีธุรกรรมระหว่างบริษัทจากสมุดรายวันทั่วไป


## <a name="post-an-intercompany-journal"></a>ลงรายการบัญชีสมุดรายวันระหว่างบริษัท
1. ไปที่สมุดรายวันทั่วไป
2. คลิก สร้าง
3. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
4. ในฟิลด์ชื่อ ให้ป้อนหรือเลือกชื่อสมุดรายวันระหว่างบริษัท
5. คลิกรายการ
6. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
7. ในฟิลด์บัญชี ให้ระบุค่าที่ต้องการ
8. ในฟิลด์คำอธิบาย ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
9. ในฟิลด์ คำอธิบาย ให้พิมพ์ค่า
10. ปิดหน้า
11. ในฟิลด์เดบิต ให้ป้อนตัวเลข
12. ในฟิลด์บริษัทตรงข้าม ให้พิมพ์หรือเลือกบริษัทตรงข้าม
13. ในฟิลด์บริษัทตรงข้าม ให้ป้อนหรือเลือกค่าใดค่าหนึ่ง
14. ปิดหน้า
15. ในฟิลด์บัญชีตรงข้าม ให้ระบุค่าที่ต้องการ
16. คลิก ลงรายการบัญชี

## <a name="view-related-intercompany-voucher"></a>ดูใบสำคัญระหว่างบริษัทที่เกี่ยวข้อง
1. คลิกใบสำคัญ
2. คลิกใบสำคัญที่เกี่ยวข้อง
3. ในรายการนี้ ให้ทำเครื่องหมายแถวที่เลือก
4. คลิกใบสำคัญ
