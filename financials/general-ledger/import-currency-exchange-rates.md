# <a name="import-currency-exchange-rates"></a>นำเข้าอัตราแลกเปลี่ยนสกุลเงิน

ถ้าเป็นนิติบุคคลได้รับใบแจ้งหนี้ในสกุลเงินต่างประเทศ จำเป็นต้องแปลงเป็นสกุลเงินต่างประเทศเป็นสกุลเงินท้องถิ่น ซึ่งหมายความ ว่า อัตราแลกเปลี่ยนปัจจุบันสำหรับสกุลเงินที่แตกต่างกันจำเป็น หัวข้อนี้แสดงภาพรวมของการตั้งค่าที่จำเป็นและประมวลผลสำหรับการนำเข้าอัตราแลกเปลี่ยนเงินตราต่างประเทศอ้างอิงที่เผยแพร่บนอินเทอร์เน็ต โดยผู้ให้บริการอัตราแลกเปลี่ยน เช่นธนาคารกลางยุโรปและธนาคารกลางของรัสเซีย

ส่วนต่อไปนี้อธิบายลำดับของข้อมูลที่จะใช้สำหรับการตั้งค่า และการประมวลผลการนำเข้าอัตราแลกเปลี่ยนเงินตราต่างประเทศ

## <a name="configure-an-exchange-rate-provider"></a>การตั้งค่าคอนฟิกตัวให้บริการอัตราแลกเปลี่ยน
ก่อนที่คุณสามารถนำเข้าอัตราแลกเปลี่ยน คุณต้องตั้งค่าข้อมูลที่จำเป็นจากผู้ให้บริการที่เสนออัตราแลกเปลี่ยน ใช้การ**การตั้งค่าคอนฟิกตัวให้บริการอัตราแลกเปลี่ยน**หน้าเพื่อเลือกผู้ให้บริการอัตราแลกเปลี่ยน บางตัวให้บริการอัตราแลกเปลี่ยนมีอยู่ในข้อมูลสาธิตใน 365 Dynamics Microsoft สำหรับการดำเนินงาน ตารางต่อไปนี้ให้คำอธิบายเกี่ยวกับตัวควบคุมต่าง ๆ บนเพจนี้

|           |                                                                                                                                                                                                                             |
|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Field** | **Description**                                                                                                                                                                                                             |
| **Name**  | ชื่อตัวให้บริการอัตราแลกเปลี่ยน                                                                                                                                                                                     |
| **คีย์**   | ตัวระบุเฉพาะสำหรับข้อมูลการตั้งค่าคอนฟิกแต่ละชิ้นที่ตัวให้บริการต้องการ  ข้อมูลนี้ถูกเพิ่มโดยอัตโนมัติสำหรับแต่ละตัวให้บริการอัตราแลกเปลี่ยนที่ คุณเพิ่มเมื่อคุณคลิก**เพิ่ม**ปุ่ม |
| **Value** | ข้อมูลสำหรับแต่ละคีย์  ข้อมูลนี้ถูกเพิ่มสำหรับแต่ละตัวให้บริการอัตราแลกเปลี่ยนที่ คุณเพิ่มเมื่อคุณคลิก**เพิ่ม**ปุ่ม                                                                                         |

## <a name="import-currency-exchange-rates"></a>นำเข้าอัตราแลกเปลี่ยนสกุลเงิน
คุณสามารถนำเข้าอัตราแลกเปลี่ยนจากแหล่งผู้ให้บริการอัตราแลกเปลี่ยน และตั้งค่าในการ**อัตราแลกเปลี่ยนสกุลเงิน**หน้าได้ ใช้การ**นำเข้าอัตราแลกเปลี่ยนสกุลเงิน**หน้าการนำเข้าอัตราแลกเปลี่ยน ตารางต่อไปนี้ให้คำอธิบายเกี่ยวกับฟิลด์ที่จำเป็นต้องทำกระบวนการนำเข้าเสร็จเรียบร้อยแล้ว

|                                        |                                                                                                                                                                                                                                                                                                                                                                             |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Field**                              | **Description**                                                                                                                                                                                                                                                                                                                                                             |
| **Exchange rate type**                 | ชนิดอัตราแลกเปลี่ยนได้                                                                                                                                                                                                                                                                                                                                                      |
| **Exchange rate provider**             | ตัวให้บริการอัตราแลกเปลี่ยน                                                                                                                                                                                                                                                                                                                                                  |
| **Import as of**                       | พารามิเตอร์นี้จัดการว่าจะนำเข้า ณวันของวันนี้ หรือ สำหรับช่วงวัน ถ้าคุณต้องการใช้ช่วงวัน ป้อน หรือเลือกวันเริ่มต้น และวันสิ้นสุด                                                                                                                                                                                                                |
| **Create necessary currency pairs**    | กล่องกาเครื่องหมายนี้จัดการการสร้างอัตโนมัติของคู่สกุลเงิน ถ้าไม่มีคู่สกุลเงินที่นำเข้า ตัวเลือกนี้อาจไม่พร้อมใช้งานสำหรับผู้ให้บริการบางอย่าง                                                                                                                                                                                               |
| **Override existing exchange rates**   | กล่องกาเครื่องหมายนี้จัดการการปรับปรุงอัตราแลกเปลี่ยนที่มีอยู่สำหรับคู่สกุลเงินเมื่อมีอัตราแลกเปลี่ยนสำหรับวันระบุอยู่แล้ว ถ้าคุณไม่ได้เลือกกล่องกาเครื่องหมายนี้ อัตราแลกเปลี่ยนสำหรับวันเฉพาะเจาะจงจะไม่นำเข้าถ้าอัตราแลกเปลี่ยนอื่นอยู่แล้ว                                                                                       |
| **Prevent import on national holiday** | กล่องกาเครื่องหมายนี้จัดการการนำเข้าอัตราแลกเปลี่ยนสำหรับวันที่เป็นวันหยุดสาธารณะ ตัวอย่างเช่น ถ้าคุณเลือกกล่องกาเครื่องหมายนี้ และใช้ธนาคารกลางยุโรปเป็นตัวให้บริการอัตราแลกเปลี่ยน ระบบจะไม่ปรับปรุงอัตราแลกเปลี่ยนในวันหยุดสาธารณะที่เกี่ยวข้องกับนิติบุคคลปัจจุบัน ตัวเลือกนี้อาจไม่พร้อมใช้งานสำหรับผู้ให้บริการบางอย่าง |
