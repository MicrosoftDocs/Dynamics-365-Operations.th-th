# <a name="financial-dimensions"></a>มิติทางการเงิน

[!include[banner](../includes/banner.md)]


บทความนี้อธิบายถึงมิติทางการเงินชนิดต่างๆ และวิธีการตั้งค่า

ใช้หน้ามิติทางการเงินเพื่อสร้างมิติทางการเงินที่คุณสามารถใช้เป็นเซ็กเมนต์บัญชีสำหรับผังบัญชีของบัญชี มิติทางการเงินมีสองชนิด มิติที่กำหนดเอง และมิติที่สำรองไว้ในเอนทิตี้ มิติที่กำหนดเองถูกใช้ร่วมกับนิติบุคคล และค่าจะถูกป้อนไว้และรักษาไว้โดยผู้ใช้ มิติที่สำรองไว้ในเอนทิตี้เป็นมิติที่มีค่าที่ถูกหนดไว้ที่อื่นในระบบ เช่น ลูกค้าหรือร้านค้า บางมิติที่สำรองไว้ในเอนทิตี้ถูกใช้ร่วมกับนิติบุคคล และบางมิติที่สำรองไว้ในเอนทิตี้เป็นเฉพาะบริษัท 

หลังจากที่คุณได้สร้างมิติทางการเงินแล้ว ใช้หน้าค่ามิติทางการเงินเพื่อกำหนดคุณสมบัติเพิ่มเติมให้กับแต่ละมิติทางการเงิน 

ถึงแม้ว่าคุณจะสามารถใช้มิติทางการเงินเพื่อแสดงถึงนิติบุคคลโดยไม่ต้องสร้างนิติบุคคลใน Microsoft Dynamics 365 for Operations แต่มิติทางการเงินก็ไม่ได้รับการออกแบบมาให้ตอบสนองความต้องการในการดำเนินงานหรือธุรกิจของนิติบุคคล ฟังก์ชันการบัญชีระหว่างหน่วยใน Microsoft Dynamics 365 for Operations ได้รับการออกแบบมาเพื่อจัดการเฉพาะรายการบัญชีที่สร้างขึ้นโดยธุรกรรมแต่ละรายการ 

ก่อนที่คุณจะตั้งค่ามิติทางการเงินเป็นนิติบุคคล ให้ประเมินกระบวนการทางธุรกิจของคุณในขอบเขตต่อไปนี้เพื่อกำหนดว่าการตั้งค่านี้จะเหมาะกับองค์กรของคุณหรือไม่:

-   สินค้าคงคลัง
-   การขายและการซื้อระหว่างมิติทางการเงินและนิติบุคคล
-   คำนวณภาษีขายและการรายงาน
-   การรายงานการดำเนินงาน

ตัวอย่างของข้อจำกัดบางประการมีดังต่อไปนี้:

-   คุณสามารถใช้ฟังก์ชันภาษีขายกับนิติบุคคลเท่านั้น ไม่สามารถใช้กับมิติทางการเงิน
-   บางรายงานจะไม่รวมมิติทางการเงิน ดังนั้นคุณจึงไม่สามารถรายงานมิติทางการเงินได้เสมอ ยกเว้นว่าจะมีการแก้ไขรายงานเหล่านั้น

**มิติที่กำหนดเอง** 

เพื่อสร้างมิติทางการเงินที่กำหนดโดยผู้ใช้ ในฟิลด์ใช้ค่าจาก ให้เลือก &lt; มิติที่กำหนดเอง &gt; คุณยังสามารถระบุตัวพรางบัญชีเพื่อจำกัดจำนวนและชนิดของข้อมูลที่คุณสามารถป้อนสำหรับค่ามิติ คุณสามารถป้อนอักขระที่เหมือนกันสำหรับแต่ละค่ามิติ เช่นตัวอักษรหรือเครื่องหมายยัติภังค์  นอกจากนี้ คุณยังสามารถป้อนเครื่องหมายตัวเลข (\#) กับเครื่องหมายและ (&) เป็นตัวยึดตำแหน่งสำหรับตัวอักษรและตัวเลขที่จะเปลี่ยนแปลงทุกครั้งที่มีการสร้างค่ามิติได้ด้วย ใช้เครื่องหมายตัวเลข (\#) เป็นตัวยึดตำแหน่งสำหรับตัวเลข และเครื่องหมายและ (&) เป็นตัวยึดตำแหน่งสำหรับตัวอักษร 

**ตัวอย่าง** 

เมื่อต้องการจำกัดให้ค่ามิติใช้แค่ตัวอักษร CC, ยัติภังค์ และตัวเลขสามหลัก ให้คุณป้อน CC-\#\#\# เป็นตัวกำหนดรูปแบบ ฟิลด์นี้สามารถใช้ได้ก็ต่อเมื่อคุณเลือก &lt; มิติที่กำหนดเอง &gt; ในฟิลด์ใช้ค่าจาก 

**มิติที่สำรองไว้ในเอนทิตี้** 

เพื่อสร้างมิติทางการเงินสำรองในเอนทิตี้ ในฟิลด์ใช้ค่าจาก ให้เลือกเอนทิตี้ที่กำหนดโดยระบบเพื่อเป็นฐานของมิติทางการเงิน ค่ามิติทางการเงินถูกสร้างขึ้นจากการเลือกนี้ ตัวอย่างเช่น เพื่อสร้างค่ามิติสำหรับโครงการ ให้เลือกโครงการ จะมีการสร้างค่ามิติสำหรับแต่ละชื่อโครงการ หน้าค่ามิติหน้าแสดงค่าสำหรับเอนทิตีและถ้าเป็นบริษัทเฉพาะ บริษัทสำหรับค่า 

**การเรียกใช้งานมิติ** 

การเรียกใช้มิติทางการเงินอัพเดตตาราง มีชื่อมิติทางการเงิน และลบมิติที่ถูกลบออก  คุณสามารถป้อนค่ามิติก่อนที่คุณจะเรียกใช้มิติทางการเงิน แต่มิติทางการเงินไม่สามารถใช้งานจนกว่าจะถูกเรียกใช้ ตัวอย่างเช่น คุณไม่สามารถเพิ่มมิติทางการเงินไปยังโครงสร้างทางบัญชีจนกว่าจะมีการเรียกใช้งานมิติทางการเงิน การคลิกเรียกใช้จะอัพเดตมิติทั้งหมดด้วยการเปลี่ยนแปลงสถานะ 

**การแปล** 

หน้าการแปลข้อความช่วยให้คุณสามารถป้อนข้อความที่จะแสดงในภาษาต่างๆสำหรับมิติทางการเงินที่เลือก หน้าการแปลบัญชีหลักเป็นที่ที่คุณสามารถป้อนข้อความที่จะแสดงในภาษาต่างๆสำหรับบัญชีหลักได้ 

**การแทนที่นิติบุคคล** 

ไม่ใช่มิติทั้งหมดจะถูกต้องสำหรับทุกนิติบุคคล และบางมิติเท่านั้นอาจเกี่ยวข้องสำหรับรอบระยะเวลาเฉพาะ ในสถานการณ์การแทนที่นิติบุคคลนี้สามารถนำมาใช้เพื่อระบุว่าบริษัทใดที่มิติควรถูกหยุดชั่วคราว ใครคือเจ้าของ และรอบระยะเวลาที่มิติที่ใช้งานอยู่
