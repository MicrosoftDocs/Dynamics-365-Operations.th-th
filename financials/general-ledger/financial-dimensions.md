# <a name="financial-dimensions"></a>มิติทางการเงิน

บทความนี้อธิบายถึงมิติทางการเงินชนิดต่างๆ และวิธีการตั้งค่า

ใช้หน้ามิติทางการเงินเพื่อสร้างมิติทางการเงินที่คุณสามารถใช้เป็นเซ็กเมนต์บัญชีสำหรับผังบัญชีของบัญชี มิติทางการเงินมีสองชนิด มิติที่กำหนดเอง และมิติที่สำรองไว้ในเอนทิตี้ มิติที่กำหนดเองถูกใช้ร่วมกับนิติบุคคล และค่าจะถูกป้อนไว้และรักษาไว้โดยผู้ใช้ มิติที่สำรองไว้ในเอนทิตี้เป็นมิติที่มีค่าที่ถูกหนดไว้ที่อื่นในระบบ เช่น ลูกค้าหรือร้านค้า บางมิติที่สำรองไว้ในเอนทิตี้ถูกใช้ร่วมกับนิติบุคคล และบางมิติที่สำรองไว้ในเอนทิตี้เป็นเฉพาะบริษัท 

หลังจากที่คุณได้สร้างมิติทางการเงินแล้ว ใช้หน้าค่ามิติทางการเงินเพื่อกำหนดคุณสมบัติเพิ่มเติมให้กับแต่ละมิติทางการเงิน 

ถึงแม้ว่าคุณสามารถใช้มิติทางการเงินเพื่อแทนนิติบุคคลโดยไม่ต้องสร้างนิติบุคคลใน 365 Dynamics Microsoft สำหรับการดำเนินงาน มิติทางการเงินไม่ได้ถูกออกแบบมาเพื่อเน้นการดำเนินงาน หรือธุรกิจที่จำเป็นของนิติบุคคล ฟังก์ชันการลงบัญชีระหว่างหน่วยใน 365 Dynamics Microsoft สำหรับการดำเนินงานถูกออกแบบมาเพื่อให้เฉพาะรายการบัญชีที่สร้างขึ้น โดยธุรกรรมแต่ละที่อยู่ 

ก่อนที่คุณจะตั้งค่ามิติทางการเงินเป็นนิติบุคคล ให้ประเมินกระบวนการทางธุรกิจของคุณในขอบเขตต่อไปนี้เพื่อกำหนดว่าการตั้งค่านี้จะเหมาะกับองค์กรของคุณหรือไม่:

-   สินค้าคงคลัง
-   การขายและการซื้อระหว่างมิติทางการเงินและนิติบุคคล
-   คำนวณภาษีขายและการรายงาน
-   การรายงานการดำเนินงาน

ตัวอย่างของข้อจำกัดบางประการมีดังต่อไปนี้:

-   คุณสามารถใช้ฟังก์ชันภาษีขายกับนิติบุคคลเท่านั้น ไม่สามารถใช้กับมิติทางการเงิน
-   บางรายงานจะไม่รวมมิติทางการเงิน ดังนั้นคุณจึงไม่สามารถรายงานมิติทางการเงินได้เสมอ ยกเว้นว่าจะมีการแก้ไขรายงานเหล่านั้น

**Custom dimensions** 

การสร้างผู้ใช้กำหนดมิติทางการเงิน ในการใช้ค่าจากฟิลด์&lt;ขนาดแบบกำหนดเอง&gt; คุณยังสามารถระบุตัวพรางบัญชีเพื่อจำกัดจำนวนและชนิดของข้อมูลที่คุณสามารถป้อนสำหรับค่ามิติ คุณสามารถป้อนอักขระที่เหมือนกันสำหรับแต่ละค่ามิติ เช่นตัวอักษรหรือเครื่องหมายยัติภังค์  คุณยังสามารถป้อนเครื่องหมายตัวเลข (\#) และ ampersands (&) เป็นตัวยึดสำหรับตัวอักษรและตัวเลขที่จะเปลี่ยนทุกครั้งที่มีสร้างค่ามิติ ได้ ใช้เครื่องหมายเลขที่ (\#) เป็นตัวยึดสำหรับตัวเลขและเครื่องหมาย ampersand (&) เป็นตัวยึดสำหรับตัวอักษรนั้น 

**ตัวอย่าง** 

เมื่อต้องการจำกัดค่ามิติที่จะสำเนาตัวอักษรและตัวเลขสาม คุณป้อน CC -\#\#\#เป็นรูปแบบการจัดรูปแบบ ฟิลด์นี้จะพร้อมใช้งานเฉพาะเมื่อคุณเลือก&lt;ขนาดแบบกำหนดเอง&gt;ในการใช้ค่าจากฟิลด์นั้น 

**มิติของเอนทิตี้สำรอง** 

เมื่อต้องการสร้างมีขนาดเอนทิตี้สำรองข้อมูลทางการเงิน ในการใช้ค่าจากฟิลด์ เลือกเอนทิตีแบบกำหนดเองระบบเมื่อต้องการสร้างมิติทางการเงินบน ค่ามิติทางการเงินถูกสร้างขึ้นจากการเลือกนี้ ตัวอย่างเช่น เพื่อสร้างค่ามิติสำหรับโครงการ ให้เลือกโครงการ จะมีการสร้างค่ามิติสำหรับแต่ละชื่อโครงการ หน้าค่ามิติหน้าแสดงค่าสำหรับเอนทิตีและถ้าเป็นบริษัทเฉพาะ บริษัทสำหรับค่า 

**เรียกใช้มิติ** 

การเรียกใช้มิติทางการเงินอัพเดตตาราง มีชื่อมิติทางการเงิน และลบมิติที่ถูกลบออก  คุณสามารถป้อนค่ามิติก่อนที่คุณจะเรียกใช้มิติทางการเงิน แต่มิติทางการเงินไม่สามารถใช้งานจนกว่าจะถูกเรียกใช้ ตัวอย่างเช่น คุณไม่สามารถเพิ่มมิติทางการเงินไปยังโครงสร้างทางบัญชีจนกว่าจะมีการเรียกใช้งานมิติทางการเงิน การคลิกเรียกใช้จะอัพเดตมิติทั้งหมดด้วยการเปลี่ยนแปลงสถานะ 

**Translations** 

หน้าข้อความที่แปลช่วยให้คุณสามารถป้อนข้อความที่จะแสดงในภาษาที่แตกต่างกันสำหรับมิติทางการเงินที่เลือก หน้าการแปลบัญชีหลักเป็นที่ที่คุณสามารถป้อนข้อความที่จะแสดงในภาษาต่างๆสำหรับบัญชีหลักได้ 

**Legal entity overrides** 

มิติทั้งหมดไม่ถูกต้องสำหรับนิติบุคคลทั้งหมด และบางส่วนเท่านั้นซึ่งอาจเกี่ยวข้องสำหรับรอบระยะเวลาที่ระบุ ในสถานการณ์การแทนที่นิติบุคคลนี้สามารถนำมาใช้เพื่อระบุว่าบริษัทใดที่มิติควรถูกหยุดชั่วคราว ใครคือเจ้าของ และรอบระยะเวลาที่มิติที่ใช้งานอยู่
