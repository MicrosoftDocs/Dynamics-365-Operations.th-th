# <a name="financial-reporting"></a>การรายงานทางการเงิน

หัวข้อนี้อธิบายถึงตำแหน่งที่สามารถเข้าถึงรายงานทางการเงินใน 365 Dynamics Microsoft สำหรับการดำเนินงานและวิธีการใช้ความสามารถในการรายงานทางการเงิน ซึ่งประกอบด้วยคำอธิบายของรายงานทางการเงินเริ่มต้นที่กำหนดไว้

<a name="accessing-financial-reporting"></a>การเข้าถึงรายงานทางการเงิน
-----------------------------

คุณสามารถค้นหา**การรายงานทางการเงิน**เมนูในต่อไปนี้ใส่ใน 365 Dynamics สำหรับการดำเนินงาน:

-   **บัญชีแยกประเภททั่วไป**&gt;**สอบถามและรายงาน**
-   **การจัดงบประมาณ**&gt;**Inquires และรายงาน**&gt;**การจัดงบประมาณพื้นฐาน**
-   **การจัดงบประมาณ**&gt;**สอบถามและรายงาน**&gt;**วางแผนงบประมาณ**
-   **การจัดงบประมาณ**&gt;**สอบถามและรายงาน**&gt;**ควบคุมงบประมาณ**
-   การรวมบัญชี

เมื่อต้องการสร้างรายงานทางการเงินสำหรับนิติบุคคล คุณต้องตั้งค่าข้อมูลต่อไปนี้สำหรับนิติบุคคลนั้น:

-   ปฏิทินทางการเงิน
-   บัญชีแยกประเภท
-   ผังบัญชี
-   สกุลเงิน

ฟังก์ชันการรายงานทางการเงินพร้อมใช้งานสำหรับผู้ใช้ที่มีสิทธิ์ที่เหมาะสม และถูกกำหนดหน้าที่ให้โดยบทบาทความปลอดภัย ส่วนต่อไป แสดงรายการสิทธิ์และหน้าที่เหล่านี้ ร่วมกับบทบาทเกี่ยวข้อง

## <a name="opening-financial-reports"></a>การเปิดรายงานทางการเงิน
เมื่อคุณคลิกเมนู **รายงานทางการเงิน** รายการเริ่มต้นรายงานทางการเงินสำหรับบริษัทจะปรากฏขึ้น คุณสามารถเปิด หรือปรับเปลี่ยนรายงาน เมื่อต้องการเปิดหนึ่งในรายงานเริ่มต้น เลือกชื่อรายงาน ในครั้งแรกที่มีการเปิดรายงาน รายงานจะถูกสร้างโดยอัตโนมัติสำหรับเดือนก่อนหน้านี้ ตัวอย่างเช่น ถ้าคุณเปิดการรายงานเป็นครั้งแรกในเดือน 2016 สิงหาคม รายงานถูกสร้างสำหรับเดือน 31 กรกฎาคม 2016 หลังจากที่มีการเปิดรายงาน คุณสามารถเริ่มสำรวจ โดยการดูรายละเอียดแนวลึกในส่วนของข้อมูลเฉพาะเจาะจง และการเปลี่ยนตัวเลือกรายงาน

## <a name="creating-and-modifying-financial-reports"></a>การสร้างและแก้ไขรายงานทางการเงิน
คุณสามารถสร้างรายงานใหม่ หรือแก้ไขรายงานที่มีอยู่จากรายการรายงานทางการเงิน ถ้าคุณมีสิทธิ์ที่เหมาะสม คุณสามารถสร้างรายงานทางการเงินใหม่ ด้วยการคลิก**สร้าง**บนบานหน้าต่างการดำเนินการได้ โปรแกรมออกแบบรายงานจะถูกดาวน์โหลดไปยังอุปกรณ์ของคุณ หลังจากที่ตัวออกแบบรายงานเริ่มต้น คุณสามารถสร้างรายงานใหม่ หลังจากคุณบันทึกรายงานใหม่ จะปรากฏอยู่ในรายการรายงานทางการเงิน รายชื่อจะแสดงรายงานที่สร้างขึ้นสำหรับบริษัทที่คุณกำลังใช้อยู่ใน Dynamics 365 สำหรับการดำเนินงาน สำหรับข้อมูลเพิ่มเติมเกี่ยวกับกระบวนการของการสร้าง และปรับเปลี่ยนรายงานทางการเงินใน 365 Dynamics สำหรับการดำเนินงาน การอ้างอิงถึงสิ่งเหล่านี้[โพสต์บล็อก](https://blogs.msdn.microsoft.com/dynamics_financial_reporting/tag/learning/)จากการแปลงเงินรายงานบล็อก **หมายเหตุ:**คอมพิวเตอร์ที่คุณกำลังดาวน์โหลดรายงานออกแบบไคลเอนต์บนต้องมีรุ่น 4.6.2 การ Microsoft .NET Framework ติดตั้งอยู่ สามารถดาวน์โหลด และติดตั้งจาก Microsoft .NET Framework รุ่นนี้[นี่](https://www.microsoft.com/en-us/download/details.aspx?id=53345)ได้ ถ้าคุณกำลังใช้โครเมียม คุณต้องติดตั้งส่วนขยาย ClickOnce เพื่อดาวน์โหลดไคลเอ็นต์ตัวออกแบบรายงาน ถ้าคุณกำลังทำงานในโหมด incognito ทำให้แน่ใจว่า เปิดใช้งานส่วนขยาย ClickOnce สำหรับโหมด incognito คุณยังสามารถแก้ไขรายงานที่ปรากฏอยู่ในรายการรายงานทางการเงิน เลือกพื้นที่โดยรอบชื่อรายงานถูก คลิก **แก้ไข** บนบานหน้าต่างการดำเนินการ เริ่มต้นโปรแกรมออกแบบรายงาน
