# <a name="advanced-bank-reconciliation-overview"></a>ภาพรวมการกระทบยอดบัญชีธนาคารขั้นสูง

กระทบยอดบัญชีธนาคารขั้นสูงช่วยให้คุณสามารถนำเข้าใบแจ้งยอดจากธนาคารอิเล็กทรอนิกส์ และกระทบยอดโดยอัตโนมัติกับธุรกรรมธนาคารใน 365 Dynamics Microsoft สำหรับการดำเนินงาน  บทความนี้จะอธิบายถึงการตั้งค่ากระบวนการสำหรับการกระทบยอด  

จำนวนชิ้นที่ต้องตั้งค่าก่อนที่จะใช้ฟังก์ชันการกระทบยอดบัญชีธนาคารขั้นสูง ได้ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการตั้งค่าการนำเข้าใบแจ้งยอดจากธนาคาร ดู[ตั้งค่ากระบวนการนำเข้าใบแจ้งยอดจากธนาคาร](set-up-advanced-bank-reconciliation-import-process.md)  ความต้องการสำหรับการตั้งค่าของกระบวนการกระทบยอดมีรายละเอียดด้านล่าง

## <a name="transaction-codes"></a>รหัสธุรกรรม
รหัสธุรกรรมที่สามารถใช้เป็นส่วนหนึ่งของการกระทบยอดธนาคารที่ตรงกับกฎ  รหัสธุรกรรมจะช่วยให้คุณจับคู่เพียงชนิดเดียวกันธุรกรรมระหว่าง 365 Dynamics สำหรับการดำเนินงานและใบแจ้งยอดจากธนาคารของคุณ  ทำการจับคู่ชนิดนี้ คุณต้องก่อนกำหนดชนิดของธุรกรรมที่ใช้สำหรับธุรกรรมธนาคารจาก 365 Dynamics สำหรับการดำเนินงาน จึงแมปชนิดเหล่านั้นไปยังรหัสธุรกรรมใบแจ้งยอดที่ใช้ โดยธนาคารของคุณ  มีกำหนดชนิดของธุรกรรมสำหรับ Dynamics 365 สำหรับธุรกรรม ธนาคารที่ดำเนินการในการ**ชนิดธุรกรรมธนาคาร**หน้า  นี่คือที่คุณกำหนดบัญชีหลักที่จะใช้สำหรับการลงรายการบัญชีที่เกี่ยวข้องกับธุรกรรมชนิดนั้นด้วย 

หลังจากที่มีกำหนด 365 Dynamics ของคุณสำหรับรหัสธุรกรรม ธนาคารในการดำเนินงาน คุณแล้วแม็ปกับรหัสธุรกรรมที่ใช้ในใบแจ้งยอดจากธนาคารอิเล็กทรอนิกส์ของคุณ  ขั้นตอนการแมปนี้เสร็จเรียบร้อยแล้วโดยใช้การ**แม็ปรหัสธุรกรรม**หน้า  การแม็ปรหัสธุรกรรมเสร็จสมบูรณ์แล้วโดยแยกต่างหากสำหรับแต่ละบัญชีธนาคาร

## <a name="matching-rules-and-matching-rule-sets"></a>กฎการจับคู่และชุดกฎการจับคู่
กฎการจับคู่ช่วยให้คุณสามารถกำหนดเงื่อนไขสำหรับการกระทบยอดอัตโนมัติระหว่าง 365 Dynamics สำหรับธุรกรรม ธนาคารที่ดำเนินการกับธุรกรรมใบแจ้งยอดจากธนาคาร  ตั้งค่ากฎการจับคู่เสร็จแล้ว**กฎการจับคู่การกระทบยอด**หน้า  สำหรับข้อมูลเพิ่มเติม ดู[ตั้งค่ากฎการจับคู่การกระทบยอดบัญชีธนาคาร](set-up-bank-reconciliation-matching-rules.md) 

ชุดกฎการจับคู่จะถูกใช้เพื่อกำหนดกลุ่มของกฎการจับคู่ที่ทำงานตามลำดับในระหว่างกระบวนการกระทบยอดบัญชีธนาคาร  มีการกำหนดค่าชุดกฎการจับคู่ในแบบ**ชุดกฎการจับคู่การกระทบยอด**หน้า

## <a name="cash-and-bank-management-parameters"></a>พารามิเตอร์การจัดการเงินสดและธนาคาร
มีจำนวนพารามิเตอร์เฉพาะสำหรับกระบวนการกระทบยอดบัญชีธนาคารขั้นสูงในการ**พารามิเตอร์การจัดการเงินสดและธนาคาร**หน้า  การ**แสดงยอดในบรรทัดใบแจ้งยอดในเดบิต/เครดิต**เปลี่ยนมุมมองของยอดเงินในใบ**ใบแจ้งยอดจากธนาคาร**หน้า  ถ้ามีเลือกตัวเลือกนี้ จะแสดงยอดเงินธุรกรรมของใบแจ้งยอดจากธนาคารในใบเพิ่มหนี้ที่แยกต่างหากและคอลัมน์สินเชื่อ  ถ้าไม่ได้เลือก ยอดเงินธุรกรรมของใบแจ้งยอดจากธนาคารจะแสดงขึ้นในคอลัมน์ยอดเงินเดียวที่มีเครื่องหมายที่เหมาะสม 

ตัวตรวจสอบการตั้งค่าในพารามิเตอร์หน้าแทนการเลือกที่กำหนดในกฎการจับคู่  ตัวอย่างเช่น คุณไม่ด้วยตนเอง หรือโดยอัตโนมัติตรงกับเอกสารนอกเหนือจากความแตกต่างของวันที่ที่ตั้งบนหน้าพารามิเตอร์  นอกจากนี้ ถ้าตัวเลือกในการ**ตรวจสอบการแม็ปชนิดธุรกรรม**ถูกเลือก ชนิดธุรกรรมต้องแม็ประหว่าง 365 Dynamics สำหรับธุรกรรม ธนาคารที่ดำเนินการกับธุรกรรมใบแจ้งยอดจากธนาคารในใบสั่งสำหรับธุรกรรมโดยอัตโนมัติ หรือด้วยตนเองตรงกัน 

คุณยังต้องตั้งค่าคอนฟิกลำดับหมายเลขที่จำเป็นในการ**พารามิเตอร์การจัดการเงินสดและธนาคาร**หน้า  ในการ**ลำดับหมายเลข**แท็บ รหัสลำดับหมายเลขชุดสำหรับการดาวน์โหลด**ID รหัสใบแจ้งยอด กระทบยอด ID และธนาคารการกระทบยอด**อ้างอิง

## <a name="bank-account-reconciliation-options"></a>ตัวเลือกการกระทบยอดบัญชีธนาคาร
คุณต้องเปิดใช้งานการกระทบยอดบัญชีธนาคารขั้นสูงสำหรับบัญชีธนาคารก่อน  จำนวนของตัวเลือกเพิ่มเติมจะพร้อมใช้งานบนตัว**บัญชีธนาคาร**หน้าหลังจากที่เปิดใช้งานฟังก์ชันการกระทบยอดบัญชีธนาคารขั้นสูง 

**ใบแจ้งยอดจาก ธนาคารที่ใช้เป็นการยืนยันการชำระเงินทางอิเล็กทรอนิกส์**ฟังก์ชันรวมฟังก์ชันการกระทบยอดบัญชีธนาคารที่ มีสถานะการชำระเงินทางอิเล็กทรอนิกส์  เมื่อนี้เปิดใช้งาน เอกสารธนาคารจะถูกสร้างขึ้นโดยอัตโนมัติสำหรับสถานะการชำระเงินทางอิเล็กทรอนิกส์ถูกตั้งค่าเป็น**"ส่งแล้ว"**  นอกจากนี้ จะปรับปรุงสถานะของการชำระเงินทางอิเล็กทรอนิกส์จาก**"ส่งแล้ว"**ไป**ได้รับ**หลังจากจับคู่การชำระเงิน การกระทบยอด และลงรายการบัญชี 

การ**ชื่อบัญชีธนาคารในใบแจ้งยอด**เขตข้อมูลเป็นชื่อที่ใช้สำหรับบัญชีธนาคารในใบแจ้งยอดจากธนาคารอิเล็กทรอนิกส์ของคุณ  ชื่อนี้จะใช้เมื่อตรวจสอบธุรกรรมใดที่จะนำเข้าบัญชีธนาคารจากใบแจ้งยอดที่อาจประกอบด้วยข้อมูลสำหรับบัญชีธนาคารหลายบัญชี 

ตัวเลือกในการ**กระทบยอดหลังจากนำเข้า**จะตรวจสอบใบแจ้งยอดจากธนาคาร สร้าง และโดยอัตโนมัติการกระทบยอดบัญชีธนาคารใหม่และแผ่นงาน รันชุดกฎการจับคู่เริ่มต้น  ฟังก์ชันนี้คล้ายกระบวนการจนถึงจุดของธุรกรรมที่ต้องตรงกันด้วยตนเอง  การตั้งค่าบัญชีธนาคารที่จะเริ่มต้นเมื่อมีการนำเข้า
