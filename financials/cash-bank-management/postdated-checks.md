# <a name="postdated-checks"></a>เช็คลงวันที่ล่วงหน้า

บทความนี้แสดงข้อมูลเกี่ยวกับการสนับสนุนสำหรับเช็คใน 365 Dynamics Microsoft สำหรับการดำเนินงาน เช็คลงวันที่ล่วงหน้าเป็นเช็คที่ถูกออก เพื่อทำและรับการชำระเงินในวันที่ในอนาคต  ดังนั้นจึงไม่สามารถจ่ายเช็คได้จนกว่าถึงวันที่ระบุ

365 Dynamics Microsoft สำหรับการดำเนินการสนับสนุนวงจรการจัดการเต็มรูปแบบสำหรับเช็คในบัญชีลูกหนี้และบัญชีเจ้าหนี้ ดังที่แสดงในตารางต่อไปนี้
