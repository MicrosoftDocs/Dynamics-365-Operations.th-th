# <a name="set-up-bills-of-exchange"></a>ตั้งค่าตั๋วแลกเงิน

หัวข้อนี้อธิบายขั้นตอนสำหรับการตั้งค่าตั๋วแลกเงิน

ตั๋วแลกเงินคือ ใบสั่งที่เป็นลายลักษณ์อักษร หรืออิเล็กทรอนิกส์จากลูกค้าซึ่งระบุว่า ผู้อื่น มักจะเป็นธนาคาร ควรชำระเงินไปยังบริษัท เมื่อคุณใช้ตั๋วแลกเงินชำระเงินให้กับใบแจ้งหนี้ของใบสั่งขายหรือใบแจ้งหนี้ข้อความอิสระ คุณจะเครดิตบัญชีของลูกค้า  เครดิตดังกล่าวจะถูกเก็บรักษาไว้โดยตั๋วแลกเงินจนกว่าลูกค้าจะชำระเงินตั๋วแลกเงินให้กับธนาคาร  โดยทั่วไปแล้ว คุณจะชำระใบแจ้งหนี้ ด้วยตั๋วแลกเงินในวันครบกำหนด เมื่อคุณได้รับการแจ้งเตือนจากธนาคารของคุณว่าตั๋วแลกเงินได้รับการจ่ายเงินแล้ว คุณสามารถปิดตั๋วแลกเงินได้ คุณสามารถออกตั๋วแลกเงินผ่านธนาคารของคุณในเวลาต่อไปนี้อย่างใดอย่างหนึ่ง:

-   ในวันครบกำหนดชำระ วิธีการนี้เรียกว่าการชำระเงินผ่านธนาคารสำหรับการเรียกเก็บเงิน
-   ก่อนวันครบกำหนด โดยทั่วไปในวันส่วนลด ที่มีระบุเงื่อนไขการชำระเงินที่ตั้งค่าไว้สำหรับลูกค้า เมื่อคุณลงรายการบัญชีธุรกรรม ยอดส่วนลดจะถูกลงรายการบัญชีไปยังบัญชีค่าใช้จ่าย  โดยยอดเงินคงเหลือยังคงเป็นหนี้สินของคุณจนกว่าธนาคารจะได้รับการชำระเงินจากลูกค้า  วิธีการนี้เรียกว่าการชำระเงินผ่านธนาคารสำหรับส่วนลด

## <a name="set-up-posting-profiles-for-bills-of-exchange"></a>การตั้งค่าโพรไฟล์การลงบัญชีสำหรับตั๋วแลกเงิน
ใช้การ**โพรไฟล์การลงบัญชีลูกค้า**หน้าการตั้งค่าส่วนกำหนดค่าที่คุณสามารถใช้กับตั๋วแลกเงิน ปฏิเสธการจ่ายตั๋วแลกเงิน การชำระเงินผ่านธนาคารสำหรับการเรียกเก็บเงิน และการชำระเงินผ่านธนาคารสำหรับส่วนลด ในการลงรายการบัญชี ในการ**บัญชีสรุป**ฟิลด์ เลือกบัญชีสรุปเพื่อลงรายการบัญชีตั๋วแลกเงิน บัญชีผู้ใช้นี้เป็นเดบิต หรือ เครดิต ขึ้นอยู่กับชนิดของธุรกรรมตั๋วแลกเงิน:
-   สำหรับตั๋วแลกเงิน บัญชีนี้ถูกเดบิตเมื่อลงรายการบัญชีตั๋วแลกเงิน และจะถูกเครดิตเมื่อมีการลงรายการบัญชีการชำระเงินผ่านธนาคารสำหรับส่วนลดหรือการชำระเงินผ่านธนาคารสำหรับการเรียกเก็บเงิน
-   สำหรับการปฏิเสธการจ่ายตั๋วแลกเงิน บัญชีนี้จะถูกเดบิตเมื่อมีการลงรายการบัญชีการปฏิเสธการจ่ายตั๋วแลกเงิน
-   สำหรับจ่ายเงินสำหรับจดหมายเรียกเก็บเงินผ่านธนาคาร บัญชีนี้จะถูกเดบิตเมื่อมีการลงรายการบัญชีการชำระเงินสำหรับจดหมายเรียกเก็บผ่านธนาคาร
-   สำหรับการชำระเงินผ่านธนาคารสำหรับส่วนลด บัญชีนี้จะถูกเดบิตเมื่อการชำระเงินผ่านธนาคารสำหรับส่วนลดได้รับการลงรายการบัญชี

ในการ**ชำระบัญชี**ฟิลด์ เลือกบัญชีเงินสดเพื่อลงรายการบัญชีตั๋วแลกเงิน บัญชีนี้จะถูกเดบิตเมื่อมีการจ่ายเงินตั๋วแลกเงิน ในการ**การชำระเงินภาษีขายล่วงหน้า**ฟิลด์ เลือกบัญชีสรุปเพื่อลงรายการบัญชียอดเงินภาษีขายเมื่อใช้ตั๋วแลกเงินสำหรับการชำระเงินล่วงหน้า ในการ**หนี้สินสำหรับบัญชีส่วนลด**ฟิลด์ เลือกบัญชีที่จะลงรายการบัญชียอดส่วนลดสำหรับการชำระเงินผ่านธนาคารสำหรับส่วนลดจะ บัญชีนี้จะถูกเครดิต เมื่อมีการลงรายการบัญชีการชำระเงินผ่านธนาคารสำหรับส่วนลด

## <a name="set-up-accounts-receivable-parameters-for-bills-of-exchange"></a>ตั้งค่าพารามิเตอร์ลูกหนี้สำหรับตั๋วแลกเงินของบัญชี
ในการ**พารามิเตอร์ลูกหนี้**หน้า ค่าเริ่มต้นโพรไฟล์การลงรายการบัญชีสำหรับตั๋วแลกเงินจะถูกป้อนใน**บัญชีแยกประเภทและภาษีขาย**แท็บ มีระบุลำดับหมายเลขไว้**ลำดับหมายเลข**แท็บ
ตั้งค่าชื่อสมุดรายวันสำหรับตั๋วแลกเงิน
------------------------------------------

ในการ**ชื่อสมุดรายวัน**หน้า สร้างชื่อสมุดรายวันอย่างน้อยห้าเพื่อใช้สำหรับตั๋วแลกเงิน ต่อไปนี้เป็นชนิดสมุดรายวัน:
-   **ตั๋วแลกเงินที่ลูกค้าออกตั๋วเงิน**– สร้างชื่อสมุดรายวันสำหรับสมุดรายวันการออกตั๋วแลกเงิน
-   **ตั๋วแลกเงินที่ลูกค้าปฏิเสธการจ่ายเงิน**– สร้างชื่อสมุดรายวันสำหรับสมุดรายวันการปฏิเสธการจ่ายตั๋วแลกเงิน
-   **ตั๋วแลกเงินที่ลูกค้าออกเงิน**– สร้างชื่อสมุดรายวันสำหรับสมุดรายวันการออกตั๋วแลกเงิน
-   **ชำระเงินผ่าน ธนาคารของลูกค้า**– สร้างชื่อสมุดรายวันสำหรับสมุดรายวันการชำระเงินผ่านธนาคาร
-   **ตั๋วแลกเงินที่ลูกค้าชำระเงิน**– สร้างชื่อสมุดรายวันสำหรับสมุดรายวันการชำระตั๋วแลกเงิน

บนหน้าใบสำคัญสมุดรายวันสำหรับแต่ละสมุดรายวันตั๋วแลกเงิน ป้อนข้อมูลเกี่ยวกับตั๋วแลกเงิน**ตั๋วแลกเงิน**แท็บ หลังจากที่ลงรายการบัญชีบรรทัดสมุดรายวันตั๋วแลกเงิน คุณสามารถดูเอกสารในการ**การสอบถามเกี่ยวกับสมุดรายวันตั๋วแลกเงิน**หน้าและ**สถิติตั๋วแลกเงิน**หน้าได้
ตั้งค่าวิธีของการชำระเงินสำหรับตั๋วแลกเงิน
-----------------------------------------------

ในการ**วิธีการชำระเงิน**หน้า ตั้งค่าอย่างน้อยหนึ่งวิธีการชำระเงินสำหรับตั๋วแลกเงิน ถ้าคุณทำธุรกิจกับธนาคารมากกว่าหนึ่ง ตั้งค่าวิธีการชำระเงินที่สอดคล้องกับรูปแบบการชำระเงินผ่านธนาคารที่ธนาคารแต่ละแห่งที่จำเป็นต้องใช้สำหรับตั๋วแลกเงิน
ตั้งค่าค่าธรรมเนียมการชำระเงินสำหรับตั๋วแลกเงิน
-----------------------------------------

ค่าธรรมเนียมการชำระเงินเป็นค่าธรรมเนียมที่เชื่อมโยงกับกระบวนการเรียกเก็บการชำระเงินจากลูกค้า  ตั้งค่าค่าธรรมเนียมการชำระเงินหลายบรรทัดอาจมีค่าธรรมเนียมการชำระเงินแต่ละที่สัมพันธ์กัน คุณสามารถใช้บรรทัดการตั้งค่าเพื่อควบคุมวิธีคำนวณยอดเงินเริ่มต้นสำหรับค่าธรรมเนียมการชำระเงิน ตัวอย่างเช่น คุณสามารถสร้างรายการการตั้งค่าสำหรับวิธีการชำระเงิน ข้อมูลจำเพาะเกี่ยวกับการชำระเงิน สกุลเงิน และรอบระยะเวลา  คุณยังสามารถสร้างบรรทัดการตั้งค่าสำหรับเปอร์เซ็นต์หรือยอดเงินที่เป็นไปตามช่วงเวลาของวัน ตัวอย่างเช่น คุณสามารถตั้งค่าเป็นเปอร์เซ็นต์ดอกเบี้ยที่จะขึ้นอยู่กับระยะเวลาการชำระเงินจะเกินกำหนด ถ้าธนาคารธรรมเนียมที่ต่างกันสำหรับชนิดการชำระเงินผ่านธนาคารที่แตกต่างกัน เช่น**คอลเลกชัน**หรือ**ส่วนลด**ตั้งค่าบรรทัดค่าธรรมเนียมการชำระเงินแยกต่างหากสำหรับแต่ละชนิดการชำระเงินผ่านธนาคาร
ตั้งค่าค่าธรรมเนียมการชำระเงินผ่านธนาคารสำหรับไฟล์การชำระเงินผ่านธนาคาร
------------------------------------------------

ในการ**บัญชีธนาคาร**หน้า คุณสามารถตั้งค่าค่าธรรมเนียมการชำระเงินผ่านธนาคารที่ธนาคารค่าธรรมเนียมสำหรับแต่ละไฟล์การชำระเงินผ่านธนาคารที่สร้างขึ้น ได้ จะมีการลงรายการบัญชีค่าธรรมเนียมการชำระเงินผ่านธนาคาร เมื่อการชำระเงินผ่านธนาคารได้รับการยืนยันและทราบยอดเงินค่าธรรมเนียมที่รับรู้  ค่าธรรมเนียมการชำระเงินผ่านธนาคารที่แตกต่างจากค่าธรรมเนียมการชำระเงิน ซึ่งจะเรียกเก็บจากลูกค้า และแนบอยู่กับบรรทัดสมุดรายวัน
ตั้งค่าโครงร่างเอกสารสำหรับตั๋วแลกเงิน
---------------------------------------------

บน**บัญชีธนาคาร**หน้า คลิ**ตั้งค่า**และระบุโครงร่างเอกสารที่จำเป็นสำหรับบัญชีธนาคารแต่ละบัญชีที่คุณจะสร้างตั๋วแลกเงินที่พิมพ์เอกสารสำหรับ งาน
ตั้งค่าลูกค้าสำหรับตั๋วแลกเงิน
--------------------------------------

ในการ**ลูกค้า**หน้า สำหรับลูกค้าแต่ละรายที่คุณได้ตกลงที่จะชำระเงิน โดยใช้ตั๋วแลกเงิน คุณสามารถตั้งค่าวิธีการเริ่มต้นของการชำระเงินสำหรับตั๋วแลกเงินในการ**ค่าเริ่มต้นของการชำระเงิน**แท็บ
