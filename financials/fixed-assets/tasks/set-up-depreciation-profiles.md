# <a name="set-up-and-create-depreciation-profiles"></a>ตั้งค่าและสร้างโพรไฟล์ค่าเสื่อมราคา

[!include[task guide banner](../../includes/task-guide-banner.md)]

โพรไฟล์การกำหนดชนิดค่าเสื่อมราคาและความถี่ของค่าเสื่อมราคาสำหรับสินทรัพย์    บันทึกนี้ใช้บริษัท USMF สาธิตและบทบาทของนักบัญชี


## <a name="create-a-depreciation-profile"></a>การสร้างโพรไฟล์การคิดค่าเสื่อมราคา
1. ไปที่ สินทรัพย์ถาวร > การตั้งค่า > โพรไฟล์ค่าเสื่อมราคา
2. คลิก สร้าง
3. ในฟิลด์โพรไฟล์ค่าเสื่อมราคา พิมพ์ค่า
4. ในฟิลด์ชื่อ ให้พิมพ์ค่า 
5. ในฟิลด์วิธีการ ให้เลือกหนึ่งตัวเลือก
    * ถ้าเลือกยอดดุลที่ลดลง คุณจะต้องป้อนเปอร์เซ็นต์ในฟิลด์เปอร์เซ็นต์  
6. ในฟิลด์ปีค่าเสื่อมราคา ให้เลือกหนึ่งตัวเลือก
7. ในฟิลด์ความถี่ระยะเวลา ให้เลือกหนึ่งตัวเลือก
8. ปิดแบบฟอร์ม
