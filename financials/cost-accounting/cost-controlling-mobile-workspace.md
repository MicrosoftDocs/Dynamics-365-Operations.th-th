# <a name="cost-controlling-mobile-workspace-for-microsoft-dynamics-365-for-operations-app"></a>พื้นที่ทำงานแบบเคลื่อนที่ของการควบคุมต้นทุนสำหรับแอพ Microsoft Dynamics 365 for Operations

## <a name="introduction"></a>คำนำ
พื้นที่ทำงานแบบเคลื่อนที่การควบคุมต้นทุนแสดงมุมมองโต้ตอบแบบทันทีของประสิทธิภาพการทำงานปัจจุบันของศูนย์ต้นทุน โดยการเปรียบเทียบต้นทุนจริงกับต้นทุนที่จัดงบประมาณ คุณสามารถดูรายละเอียดแนวลึกของสถานะขององค์ประกอบต้นทุนแต่ละรายการได้

### <a name="example"></a>ตัวอย่าง

พนักงานได้รับการเชิญไปยังการประชุมนานาชาติ องค์กรจะต้องครอบคลุมค่าใช้จ่ายการเดินทางทั้งหมด พนักงานถามผู้จัดการของตนว่าเขาสามารถเข้าร่วมการประชุมได้หรือไม่ ผู้จัดการรีบเปิดดูพื้นที่การทำงานเคลื่อนที่การควบคุมต้นทุนบนโทรศัพท์มือถือของเขาเพื่อดูว่าเขามีงบประมาณสำหรับพนักงานเพื่อที่จะเข้าร่วมการประชุมหรือไม่

### <a name="data-security"></a>ความปลอดภัยของข้อมูล

ข้อมูลในพื้นที่ทำงานการควบคุมต้นทุนได้รับการป้องกันโดยข้อมูลประจำตัวผู้ใช้ อนุญาตให้ผู้จัดการศูนย์ต้นทุนสามารถดูได้เฉพาะข้อมูลสำหรับศูนย์ต้นทุนของตนเองเท่านั้น มีการจัดการความปลอดภัยระดับการเข้าถึงภายในโมดูลการบัญชีต้นทุน ผู้จัดทำบัญชีต้นทุนกำหนดการตั้งค่าคอนฟิกพื้นที่ทำงานแบบเคลื่อนที่การควบคุมต้นทุนในโมดูลการบัญชีต้นทุน หลังจากที่พื้นที่ทำงานถูกเผยแพร่ไปยังแอพ Microsoft Dynamics 365 for Operations พื้นที่ทำงานดังกล่าวจะพร้อมใช้งานในแอพบนมือถือ Microsoft Dynamics 365 for Operations ซึ่งช่วยให้มั่นใจว่าผู้จัดการศูนย์ต้นทุนทั้งหมดในองค์กรจะได้ดูข้อมูลในรูปแบบเดียวกัน

### <a name="actions-views-and-links"></a>การดำเนินการ มุมมอง และการเชื่อมโยง

พื้นที่ทำงานแบบเคลื่อนที่การควบคุมต้นทุนสำหรับแอพ Dynamics 365 for Operations มีการดำเนินการ มุมมอง และการเชื่อมโยงดังต่อไปนี้:

-   การดำเนินการ 
    -   เลือก **การตั้งค่าคอนฟิก** เพื่อเลือกเค้าโครง
    -   เลือก **ออบเจ็กต์ต้นทุน** เพื่อเลือกศูนย์ต้นทุนที่คุณต้องการกรองข้อมูล **หมายเหตุ:** รายการจะถูกแสดงตามการเข้าถึงที่อนุญาตในโมดูลการบัญชีต้นทุน

<!-- -->

-   ขึ้นอยู่กับสิ่งที่เลือกภายใต้ **การดำเนินการ** และสิ่งที่ถูกตั้งค่าคอนฟิกในโมดูลการบัญชีต้นทุน คุณสามารถดูข้อมูลต่อไปนี้ในบัตรได้ หมายเหตุว่ายอดเงินถูกแสดงในรูปแบบเดียวกัน: ยอดเงินจริง งบประมาณ ผลต่าง และ % ผลต่าง 
    -   ยอดเงินจริงเทียบกับงบประมาณ (รอบระยะเวลาปัจจุบัน)
    -   งบประมาณจริงเทียบกับงบประมาณที่ปรับปรุงแล้ว (รอบระยะเวลาปัจจุบัน)
    -   ยอดเงินจริงเทียบกับงบประมาณ (รอบระยะเวลาก่อนหน้านี้)
    -   งบประมาณจริงเทียบกับงบประมาณที่ปรับปรุงแล้ว (รอบระยะเวลาก่อนหน้านี้)
    -   ยอดเงินจริงเทียบกับงบประมาณ (ปีถึงปัจจุบัน)
    -   งบประมาณจริงเทียบกับงบประมาณที่ปรับปรุงแล้ว (ปีถึงปัจจุบัน)

<!-- -->

-   ลิงก์
    -   รายละเอียดสำหรับรอบระยะเวลาปัจจุบัน
    -   รายละเอียดสำหรับรอบระยะเวลาก่อนหน้านี้
    -   รายละเอียดสำหรับปีถึงปัจจุบัน

เมื่อคุณเลือกการเชื่อมโยงใดการเชื่อมโยงหนึ่งแล้ว บัตรต่อองค์ประกอบต้นทุนจะแสดงขึ้น ยอดเงินในบัตรจะแสดงอยู่ในรูปแบบต่อไปนี้: ยอดเงินจริง งบประมาณ ผลต่างงบประมาณ % ผลต่างงบประมาณ งบประมาณที่ปรับปรุงแล้ว ผลต่างงบประมาณที่ปรับปรุงแล้ว และ % ผลต่างงบประมาณที่ปรับปรุงแล้ว  [![การควบคุมต้นทุน](./media/cost-controlling.png)](./media/cost-controlling.png)

## <a name="get-started"></a>เริ่มต้น
ทำตามขั้นตอนเหล่านี้เพื่อเริ่มต้นการใช้แอพบนมือถือการควบคุมต้นทุนบนอุปกรณ์เคลื่อนที่ของคุณ

1.  บนร้านค้าแอพบนมือถือของคุณ ดาวน์โหลดและติดตั้งแอพ Microsoft Dynamics 365 for Operations
2.  เริ่มการทำงานแอพบนอุปกรณ์ของคุณ
3.  ป้อน URL ของ Dynamics 365
4.  ป้อนบริษัทเพื่อลงชื่อเข้าใช้ ตัวอย่างเช่น ป้อน **USMF**
5.  ในครั้งแรกที่คุณเข้าสู่ระบบ คุณจะได้รับพร้อมท์สำหรับชื่อผู้ใช้และรหัสผ่านสำหรับบัญชี Microsoft Dynamics 365 for Operations ของคุณ ป้อนข้อมูลประจำตัวของคุณ หลังจากที่คุณลงชื่อเข้าใช้แล้ว คุณจะสามารถดูพื้นที่ทำงานที่พร้อมใช้งานสำหรับบริษัทของคุณ

เมื่อต้องการดูพื้นที่ทำงานในแอพบนมือถือของคุณ คุณต้องเผยแพร่พื้นที่ทำงานที่ต้องการไปยังแอพ Dynamics 365 for Operations

1.  เริ่มการทำงาน Dynamics 365 for Operations
2.  ไปที่ **การจัดการระบบ** &gt; **การตั้งค่า** &gt; **พารามิเตอร์ระบบ**
3.  เลือก **จัดการแอพบนมือถือ**
4.  เลือกพื้นที่ทำงาน **การควบคุมต้นทุน** เพื่อเผยแพร่ไปยังแพลตฟอร์มเคลื่อนที่
5.  เลือก **เผยแพร่พื้นที่ทำงาน**
6.  รีเฟรชอุปกรณ์ของคุณเพื่อดูพื้นที่ทำงานที่เผยแพร่แล้ว

## <a name="view-the-performance-of-your-cost-center"></a>ดูประสิทธิภาพการทำงานของศูนย์ต้นทุนของคุณ
1.  บนอุปกรณ์เคลื่อนที่ เลือกพื้นที่ทำงาน **การควบคุมต้นทุน**
2.  เลือก**การควบคุมออบเจ็กต์ต้นทุน**
3.  คลิก **การดำเนินการ**
4.  คลิก **เลือกการตั้งค่าคอนฟิก** เพื่อเลือกเค้าโครงการควบคุมต้นทุน
5.  คลิก **เสร็จสิ้น**
6.  คลิก **การดำเนินการ**
7.  คลิก **เลือกออบเจ็กต์ต้นทุน** เพื่อเลือกศูนย์ต้นทุนที่คุณได้รับอนุญาตให้เข้าถึง
8.  คลิก **เสร็จสิ้น**
9.  ดูประสิทธิภาพการทำงานโดยรวมของศูนย์ต้นทุนของคุณ
10. คลิก **รายละเอียดสำหรับรอบระยะเวลาปัจจุบัน**
11. ดูประสิทธิภาพการทำงานขององค์ประกอบต้นทุนแต่ละรายการ
12. คุณยังสามารถค้นหาองค์ประกอบต้นทุนเฉพาะได้อีกด้วย
